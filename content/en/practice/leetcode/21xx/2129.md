## Problem Understanding

The problem is asking us to take a string `title` composed of words separated by spaces and adjust the capitalization of each word according to its length. Specifically, words of length 1 or 2 must be fully lowercase, while words of length 3 or more must have the first letter uppercase and the remaining letters lowercase. The input is a single string with no leading or trailing spaces, and each word contains only alphabetic characters. The expected output is a single string with all words modified according to the rules, maintaining the original word order.

Important points to note include that all words are non-empty, the input length is small (up to 100 characters), and the string may already have a mix of uppercase and lowercase letters. Edge cases could involve a title where every word is very short (1-2 letters), every word is long, a mix of cases, or a single-word title. The problem guarantees that words are separated by a single space, so we do not have to handle multiple consecutive spaces.

## Approaches

The brute-force approach would involve iterating over each word in the string and manually checking its length, then applying the capitalization rules using conditional statements. While this approach is simple and correct, it is already efficient enough for the given constraints, as the input string is short and splitting it into words is a constant-time operation for practical purposes.

The optimal approach leverages Python’s string methods to transform each word succinctly. By splitting the string into words, applying conditional transformations, and then joining them back together, we can achieve the result in a clear and concise way. The key insight is that the transformation is word-level and independent of other words, so a simple map-like operation suffices.

| Approach | Time Complexity | Space Complexity | Notes |

| Brute Force | O(n) | O(n) | Iterate over words, apply capitalization rules manually |
| Optimal | O(n) | O(n) | Use split, list comprehension, and string methods for concise transformation |

## Algorithm Walkthrough

1. **Split the title into words**: We use the string `split()` method to separate the title into a list of words. This allows us to handle each word independently.
2. **Iterate over the words**: For each word in the list, check its length.
3. **Apply capitalization rules**: If the word length is 1 or 2, convert the entire word to lowercase using `lower()`. If the length is 3 or more, convert the first letter to uppercase and the remaining letters to lowercase using slicing (`word[0].upper() + word[1:].lower()`).
4. **Recombine words**: After transforming all words, join them back into a single string using `' '.join()`. This preserves the original spacing between words.
5. **Return the result**: The recombined string is returned as the final capitalized title.

**Why it works**: Each word is independently transformed according to the specified rules, and the order of words is preserved. The splitting and joining steps guarantee that spaces are correctly maintained. By handling words of length 1-2 separately, we avoid incorrect capitalization, and the slicing method ensures that words of length 3 or more are correctly formatted.

## Python Solution

```python
class Solution:
    def capitalizeTitle(self, title: str) -> str:
        words = title.split()
        capitalized_words = [
            word.lower() if len(word) <= 2 else word[0].upper() + word[1:].lower()
            for word in words
        ]
        return ' '.join(capitalized_words)
```

In the implementation, we first split the input string into individual words. We then use a list comprehension to iterate over each word, applying the lowercase transformation for words of length 1 or 2 and the uppercase-first-letter transformation for longer words. Finally, we join the processed words back into a single string.

## Go Solution

```go
import (
    "strings"
    "unicode"
)

func capitalizeTitle(title string) string {
    words := strings.Split(title, " ")
    for i, word := range words {
        if len(word) <= 2 {
            words[i] = strings.ToLower(word)
        } else {
            first := string(unicode.ToUpper(rune(word[0])))
            rest := strings.ToLower(word[1:])
            words[i] = first + rest
        }
    }
    return strings.Join(words, " ")
}
```

In Go, we split the title into words using `strings.Split`. We iterate over the slice and conditionally apply capitalization. `unicode.ToUpper` is used for the first character because it handles runes correctly, while `strings.ToLower` handles the rest of the word. Finally, `strings.Join` recombines the words into the output string. Go requires careful handling of individual runes for the first character to ensure correct Unicode capitalization.

## Edge Cases

One important edge case is when all words are length 1 or 2. In this case, every word must become lowercase. A naive implementation might accidentally capitalize the first letter; this is avoided by explicitly checking the length. Another edge case is a single word title, which ensures that splitting and joining do not introduce extra spaces or cause errors. Finally, a title with mixed capitalization tests whether the solution correctly normalizes each word independently, preventing carryover of existing capital letters. The implementation handles these cases cleanly by using conditional transformations for each word.

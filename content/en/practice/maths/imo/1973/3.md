## Problem

Let $a$ and $b$ be real numbers for which the equation $$
x^4 + ax^3 + bx^2 + ax + 1 = 0
$$ has at least one real solution. For all such pairs $(a, b)$, find the minimum value of $a^2 + b^2$.

## Exploration

The polynomial $x^4 + ax^3 + bx^2 + ax + 1$ is palindromic. This implies that if $r$ is a root, then $1/r$ is also a root. Any real root $r$ must satisfy $r \neq 0$. Testing small integer roots, first consider $x = 1$. Substituting yields $1 + a + b + a + 1 = 2 + 2a + b = 0$, so $b = -2(a+1)$. Then $a^2 + b^2 = a^2 + 4(a+1)^2 = 5a^2 + 8a + 4$. Minimizing this quadratic gives $a = -\frac{4}{5}$ and $b = -\frac{2}{5}$, producing $a^2 + b^2 = \frac{4}{5}$. For $x = -1$, substituting gives $1 - a + b - a + 1 = 2 - 2a + b = 0$, so $b = 2a - 2$. Then $a^2 + b^2 = a^2 + (2a-2)^2 = 5a^2 - 8a + 4$, minimized at $a = \frac{4}{5}$ and $b = -\frac{2}{5}$, producing the same value $a^2 + b^2 = \frac{4}{5}$.

Consider a real root $r \neq \pm 1$. Then $1/r$ is also a root. Factorization gives $(x-r)(x-1/r) = x^2 - (r + 1/r)x + 1$. Denote the other quadratic factor as $x^2 + px + 1$, so $a = p - (r + 1/r)$ and $b = 2 - (r + 1/r)p$. Then $a^2 + b^2 = (p - s)^2 + (2 - sp)^2$, where $s = r + 1/r$. Minimizing over $p$ gives $p = 3s/(1 + s^2)$, producing $a = p - s = (3s/(1 + s^2)) - s = (3 - (1+s^2)) s / (1 + s^2) = (2 - s^2)s/(1 + s^2)$ and $b = 2 - sp = 2 - s*(3s/(1+s^2)) = 2 - 3s^2/(1+s^2) = (2 - s^2)/(1 + s^2)$. Then $a^2 + b^2 = \frac{(2 - s^2)^2 (s^2 + 1)}{(1+s^2)^2} = |2 - s^2|$, simplified using $s^2 \ge 4$ for $|r| \ge 1$ or $|r| \le 1$? Explore $r$ small: $r + 1/r \ge 2$ for $r>0$ and $\le -2$ for $r<0$. Then $s^2 \ge 4$, so $a^2 + b^2 = |2 - s^2| \ge 2$. Already larger than $\frac{4}{5}$. Therefore, the minimum occurs at $x = \pm 1$.

Checking small explicit cases: for $x = 1$, $a = -\frac{4}{5}, b = -\frac{2}{5}$ works: $1 - 4/5 - 2/5 -4/5 +1 = 0$. For $x=-1$, $a = 4/5, b=-2/5$: $1 -4/5 +(-2/5) + 4/5 + 1 =0$. Both produce $a^2 + b^2 = 16/25 + 4/25 = 20/25 = 4/5$. Trying $x = 2$, $1 + 8a + 4b + 2a +1 = 0$, gives $10a +4b +2=0$, pick $a=-1/2$, $b=0$, $a^2 + b^2=1/4$? Test $P(2) =16-4+? +?+1$; not zero. So minimal occurs at $x = \pm1$ as expected.

## Problem Understanding

We are to find the minimum of $a^2 + b^2$ over all real pairs $(a,b)$ such that the quartic $x^4 + ax^3 + bx^2 + ax + 1$ has a real root. The set of admissible $(a,b)$ is the locus of coefficients for which the quartic admits at least one real solution. This is an optimization problem over a non-linear set in $\mathbb{R}^2$. The solution requires computing a global minimum and exhibiting admissible coefficients attaining it.

## Key Observations

The polynomial is palindromic; any real root $r$ implies $1/r$ is also a root. If $x = \pm1$ are roots, the quartic reduces to a linear relation between $a$ and $b$, producing a quadratic in $a^2 + b^2$, which can be minimized explicitly. For other real roots $r \neq \pm 1$, factorization into reciprocal quadratics allows expressing $a^2 + b^2$ in terms of a single parameter $s = r + 1/r$, with derivative-based minimization confirming that $a^2 + b^2 \ge 2$. Therefore, the global minimum occurs for $x = \pm1$.

## Solution

Suppose $x = 1$ is a root. Then $1 + a + b + a + 1 = 0 \implies b = -2(a+1)$. Then

$a^2 + b^2 = a^2 + 4(a+1)^2 = 5a^2 + 8a + 4 = 5\left(a + \frac{4}{5}\right)^2 + \frac{4}{5}.$

Hence the minimum in this case is $\frac{4}{5}$, attained at

$a = -\frac{4}{5}, \quad b = -\frac{2}{5}.$

Suppose $x = -1$ is a root. Then $1 - a + b - a + 1 = 0 \implies b = 2a -2$, so

$a^2 + b^2 = a^2 + (2a -2)^2 = 5a^2 -8a +4 = 5\left(a - \frac{4}{5}\right)^2 + \frac{4}{5}.$

The minimum is again $\frac{4}{5}$, attained at

$a = \frac{4}{5}, \quad b = -\frac{2}{5}.$

Now consider a real root $r \neq \pm1$, with $s = r + 1/r$. Factor as $(x^2 - sx +1)(x^2 + px +1)$, giving $a = p - s$ and $b = 2 - sp$. Then $a^2 + b^2 = (p-s)^2 + (2 - sp)^2$. Differentiating with respect to $p$ gives $F'(p) = 2(p-s) + 2(-s)(2 - sp) = 2(p-s) - 2s(2 - sp) = 2p - 2s - 4s + 2s^2 p = (2 + 2s^2)p -6s =0$, so $p = 3s/(1 + s^2)$. Then $a = p - s = (3s/(1+s^2)) - s = (2 - s^2)s/(1+s^2)$, $b = 2 - sp = 2 - s*(3s/(1+s^2)) = (2 - s^2)/(1+s^2)$. Then

$a^2 + b^2 = \frac{(2 - s^2)^2 s^2 + (2 - s^2)^2}{(1+s^2)^2} = \frac{(2 - s^2)^2 (1 + s^2)}{(1+s^2)^2} = |2 - s^2|.$

Since $|r| \ge 1$ or $|r| \le 1$, we have $|s| \ge 2$, so $s^2 \ge 4$, hence $a^2 + b^2 = s^2 - 2 \ge 2 > 4/5$. Therefore, the global minimum occurs for $x = \pm1$.

The minimum value of $a^2 + b^2$ is

$\boxed{\frac{4}{5}},$

attained at $(a,b) = \left(-\frac{4}{5}, -\frac{2}{5}\right)$ for $x=1$ or $(a,b) = \left(\frac{4}{5}, -\frac{2}{5}\right)$ for $x=-1$.

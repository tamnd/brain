## Problem

Let $a$ and $b$ be real numbers for which the equation $$
x^4 + ax^3 + bx^2 + ax + 1 = 0
$$ has at least one real solution. For all such pairs $(a, b)$, find the minimum value of $a^2 + b^2$.

## Exploration

The given quartic equation has the form

$$x^4 + ax^3 + bx^2 + ax + 1 = 0,$$

which is symmetric under the transformation $x \mapsto 1/x$ since replacing $x$ with $1/x$ multiplies the entire equation by $x^4$ and preserves the coefficients. This suggests that if $r$ is a root, so is $1/r$. A natural approach is to try factoring the quartic into quadratics or considering the roots in reciprocal pairs. Let $r$ be a real root. If $r = \pm 1$, the equation simplifies immediately. For general $r$, we can attempt to express $b$ in terms of $a$ and $r$, then minimize $a^2 + b^2$ as a function of $r$.

Trying small integer roots: $x=1$ gives $1 + a + b + a + 1 = 2 + 2a + b = 0$, yielding $b = -2(a+1)$. For $x=-1$, we get $1 - a + b - a + 1 = 2 - 2a + b = 0$, so $b = 2a - 2$. These linear relations allow us to express $a^2 + b^2$ as a quadratic in one variable, which can be minimized.

Factoring as $(x^2 + px + 1)(x^2 + qx + 1)$ is promising since the product is palindromic. Expanding gives $x^4 + (p+q)x^3 + (pq + 2)x^2 + (p+q)x + 1$, so $a = p + q$ and $b = pq + 2$. If we fix a real root, say $r$, the pair $(r, 1/r)$ are roots, and the quadratic $(x - r)(x - 1/r) = x^2 - (r + 1/r)x + 1$ divides the quartic. Then $p$ or $q$ is $-(r + 1/r)$ and the other is the complementary coefficient. The minimal value of $a^2 + b^2$ likely occurs when the two quadratics coincide or when roots are $\pm 1$.

The main difficulty is handling the general real root $r \ne \pm 1$ while ensuring all calculations for $a^2 + b^2$ are fully justified and the minimal value is global.

## Problem Understanding

The problem asks for the minimum value of $a^2 + b^2$ over all pairs of real numbers $(a,b)$ such that the quartic $x^4 + ax^3 + bx^2 + ax + 1 = 0$ has at least one real solution. This is an optimization problem (Type C) over a set defined implicitly by a real-root condition for a quartic. The mathematical objects are real numbers $a, b, x$ with the equation linking them. The core difficulty is expressing $a^2 + b^2$ in a way that makes its minimal value explicit and proving that no other configuration yields a smaller value, especially since the quartic has symmetry and multiple possible real roots.

Intuitively, the minimum should occur for small values of $a$ and $b$ in absolute value, suggesting roots $x = \pm 1$ or $x$ on the unit circle, given the reciprocal symmetry.

## Proof Architecture

Lemma 1: If $x = 1$ is a root, then $b = -2(a+1)$, and $a^2 + b^2$ reduces to $a^2 + (-2(a+1))^2 = 5a^2 + 4a + 4$.

Lemma 2: If $x = -1$ is a root, then $b = 2a - 2$, and $a^2 + b^2$ reduces to $a^2 + (2a-2)^2 = 5a^2 - 8a + 4$.

Lemma 3: For any real root $r \ne \pm 1$, if we factor as $(x^2 - (r+1/r)x + 1)(x^2 + px + 1)$, then $a = -(r+1/r) + p$ and $b = - (r+1/r)p + 2$. Minimizing $a^2 + b^2$ over $p$ for fixed $r$ gives $p = r + 1/r$, yielding $a = 0$ and $b = 2 - (r + 1/r)^2$.

Lemma 4: For $a = 0$ and $b = 2 - (r + 1/r)^2$, $a^2 + b^2$ is minimized when $r = 1$ or $r = -1$, giving $b = -2$, so $a^2 + b^2 = 0^2 + (-2)^2 = 4$.

Lemma 5: Checking all cases of $x = 1$, $x=-1$, and $x \ne \pm 1$, the global minimum of $a^2 + b^2$ is $4$, achieved when $x = 1$ with $a = -1, b = 0$ or $x=-1$ with $a=1, b=0$.

The hardest step is Lemma 3, expressing $a^2 + b^2$ in terms of a single parameter and minimizing rigorously over $p$ and $r$ while ensuring all real roots are accounted for.

$$1 + a + b + a + 1 = 2 + 2a + b = 0.$$

Solving gives $b = -2(a+1)$. Then

$$a^2 + b^2 = a^2 + ( -2(a+1) )^2 = a^2 + 4(a+1)^2 = a^2 + 4(a^2 + 2a +1) = 5a^2 + 8a + 4.$$

This establishes a formula for $a^2 + b^2$ under the condition $x=1$ is a root.

Lemma 2: Suppose $x = -1$ is a root. Substituting yields

$$1 - a + b - a + 1 = 2 - 2a + b = 0.$$

Solving gives $b = 2a - 2$. Then

$$a^2 + b^2 = a^2 + (2a-2)^2 = a^2 + 4a^2 -8a +4 = 5a^2 -8a +4.$$

This establishes a formula for $a^2 + b^2$ under the condition $x=-1$ is a root.

Lemma 3: Let $r$ be a real root with $r \ne \pm 1$. By the symmetry of the quartic, $1/r$ is also a root. Thus $(x-r)(x-1/r) = x^2 - (r + 1/r)x + 1$ divides the quartic. Suppose the quartic factors as $(x^2 - (r+1/r)x + 1)(x^2 + px + 1) = x^4 + (p-(r+1/r))x^3 + (-(r+1/r)p + 2)x^2 + (p-(r+1/r))x + 1$. Comparing with the original quartic, we get $a = p-(r+1/r)$, $b = 2 - (r+1/r)p$.

$$a^2 + b^2 = (p-(r+1/r))^2 + (2-(r+1/r)p)^2.$$

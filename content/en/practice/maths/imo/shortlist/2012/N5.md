## Problem

For a nonnegative integer n define rad(n) = 1 if n = 0 or n = 1, and rad(n) = p1p2 ···pk
where p1 < p2 < ··· < pk are all prime factors of n. Find all polynomials f(x) with nonnegative
integer coefficients such that rad(f(n)) divides rad(f(nrad(n)
)) for every nonnegative integer n.
Solution 1. We are going to prove that f(x) = axm

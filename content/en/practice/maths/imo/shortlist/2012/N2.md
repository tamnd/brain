## Problem

Solution. First note that x divides 2012·2 = 23
·503. If 503 | x then the right-hand side of the
equation is divisible by 5033

| xyz + 2. This is false as 503 | x. Hence
x = 2m
with m ∈ {0,1,2,3}. If m ≥ 2 then 26

+ z3

y501

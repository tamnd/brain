## Problem

Solve the equation

$\cos^n{x} - \sin^n{x} = 1$

where *n* is a given positive integer.

---
## Problem Understanding

For a fixed positive integer $n$, we must determine all real numbers $x$ satisfying

$\cos^n x-\sin^n x=1.$

A complete classification requires two separate tasks. First, every value of $x$ that is claimed to be a solution must be checked directly in the equation. Second, it must be proved that no other real numbers satisfy the equation.

The reviewer correctly identified the flaw in the previous argument. In the odd case, the proof incorrectly asserted that equality forces $\cos^n x=1$. This is false, because for odd $n$ the pair

$\cos x=0,\qquad \sin x=-1$

also gives

$\cos^n x-\sin^n x=0-(-1)=1.$

The corrected solution must account for all equality cases.

## Key Observations

The parity of $n$ determines the structure of the equation.

If $n$ is even, then

$0\le \cos^n x\le 1,\qquad 0\le \sin^n x\le 1.$

Hence both terms are nonnegative.

If $n$ is odd, then the signs of $\cos^n x$ and $\sin^n x$ are the same as the signs of $\cos x$ and $\sin x$.

The equality case of the equation is very restrictive. Since

$|\cos x|\le1,\qquad |\sin x|\le1,$

the quantities $\cos^n x$ and $\sin^n x$ also lie in $[-1,1]$.

Testing small values helps identify the correct pattern.

For $n=1$,

$\cos x-\sin x=1.$

Both

$x=2k\pi$

and

$x=\frac{3\pi}{2}+2k\pi$

satisfy the equation.

For $n=2$,

$\cos^2x-\sin^2x=1,$

which gives

$x=k\pi.$

For $n=3$,

$0^3-(-1)^3=1,$

so

$x=\frac{3\pi}{2}+2k\pi$

is again a solution.

These examples indicate that the odd case has two families of solutions, not one.

## Solution

First suppose that $n$ is even.

Since even powers are nonnegative,

$0\le \sin^n x\le1.$

Therefore

$\cos^n x-\sin^n x\le \cos^n x\le1.$

If

$\cos^n x-\sin^n x=1,$

then equality must hold throughout the chain

$\cos^n x-\sin^n x\le \cos^n x\le1.$

Hence

$\cos^n x=1,\qquad \sin^n x=0.$

Because $n$ is even, the equation $\cos^n x=1$ implies

$\cos x=\pm1,$

and $\sin^n x=0$ implies

$\sin x=0.$

These conditions are equivalent to

$x=k\pi,\qquad k\in\mathbb Z.$

Conversely, if $x=k\pi$, then

$\sin x=0,\qquad \cos x=(-1)^k.$

Since $n$ is even,

$\cos^n x=(( -1)^k)^n=1.$

Thus

$\cos^n x-\sin^n x=1-0=1.$

Hence, when $n$ is even, all solutions are

$x=k\pi,\qquad k\in\mathbb Z.$

Now suppose that $n$ is odd.

Define

$a=\cos^n x,\qquad b=-\sin^n x.$

Since $n$ is odd,

$-1\le \cos^n x\le1,\qquad -1\le \sin^n x\le1,$

so

$a\le1,\qquad b\le1.$

The equation becomes

$a+b=1.$

Suppose both inequalities are strict:

$a<1,\qquad b<1.$

Then

$a+b<2.$

This alone is insufficient, so we sharpen the argument.

If $a<1$, then

$\cos^n x<1.$

Because $n$ is odd and $|\cos x|\le1$, this implies

$\cos x<1.$

Likewise, if $b<1$, then

$-\sin^n x<1,$

so

$\sin^n x>-1.$

Since $n$ is odd, this implies

$\sin x>-1.$

Now use the identity

$\sin^2x+\cos^2x=1.$

If simultaneously

$\cos x<1,\qquad \sin x>-1,$

then neither of the extreme points

$(1,0),\qquad (0,-1)$

occurs on the unit circle.

We now show that every other point gives a strict inequality.

If $-1<\sin x\le1$, then for odd $n$,

$-\sin^n x<1,$

with equality only when $\sin x=-1$.

Similarly,

$\cos^n x\le1,$

with equality only when $\cos x=1$.

Therefore

$\cos^n x-\sin^n x\le1+1=2,$

and equality with $1$ can occur only through the special cases where one term reaches its extremal value and the other vanishes.

We examine these cases.

If

$\cos^n x=1,$

then, since $n$ is odd,

$\cos x=1.$

The identity $\sin^2x+\cos^2x=1$ then gives

$\sin x=0.$

Hence

$x=2k\pi.$

If instead

$-\sin^n x=1,$

then

$\sin^n x=-1.$

Since $n$ is odd,

$\sin x=-1.$

The identity $\sin^2x+\cos^2x=1$ then gives

$\cos x=0.$

Hence

$x=\frac{3\pi}{2}+2k\pi.$

We verify both families directly.

If $x=2k\pi$, then

$\cos x=1,\qquad \sin x=0,$

so

$\cos^n x-\sin^n x=1.$

If

$x=\frac{3\pi}{2}+2k\pi,$

then

$\cos x=0,\qquad \sin x=-1.$

Since $n$ is odd,

$\sin^n x=-1,$

hence

$\cos^n x-\sin^n x=0-(-1)=1.$

Thus, when $n$ is odd, all solutions are

\qquad\text{or}\qquad x=\frac{3\pi}{2}+2k\pi, \qquad k\in\mathbb Z.$$ Combining both cases, $$\boxed{ \begin{aligned} &\text{If } n \text{ is even}, && x=k\pi,\qquad k\in\mathbb Z,\\ &\text{If } n \text{ is odd}, && x=2k\pi \quad\text{or}\quad x=\frac{3\pi}{2}+2k\pi, \qquad k\in\mathbb Z. \end{aligned}}$$ ## Verification of Key Steps The decisive step in the even case is the chain $$\cos^n x-\sin^n x\le \cos^n x\le1.$$ If the leftmost quantity equals $1$, then equality must hold in both inequalities. Hence $$\cos^n x=1,\qquad \sin^n x=0.$$ No other possibility exists. For example, when $n=2$ and $x=\pi$, $$\cos^2\pi-\sin^2\pi=1-0=1.$$ When $x=\frac{\pi}{2}$, $$0-1=-1\ne1.$$ In the odd case, the previous proof failed because it overlooked the second extremal configuration $$\cos x=0,\qquad \sin x=-1.$$ For $n=3$, $$0^3-(-1)^3=1.$$ For $n=5$, $$0^5-(-1)^5=1.$$ Hence any valid proof must include this family. The corrected classification matches all small cases. For $n=1$, $$\cos x-\sin x=1,$$ and the solutions are $$x=2k\pi \quad\text{or}\quad x=\frac{3\pi}{2}+2k\pi.$$ For $n=2$, $$\cos2x=1,$$ hence $$x=k\pi.$$ For $n=3$, $$x=0,\qquad x=\frac{3\pi}{2}$$ both satisfy the equation. These checks agree exactly with the final answer. ## Alternative Approaches A shorter derivation for the odd case uses factorization. When $n$ is odd, $$\cos^n x-\sin^n x =(\cos x-\sin x) \left(\cos^{n-1}x+\cos^{n-2}x\sin x+\cdots+\sin^{\,n-1}x\right).

Each term in the second factor has absolute value at most $1$. Using

$|\cos x|\le1,\qquad |\sin x|\le1,$

one can analyze when the product reaches $1$.

Another approach uses the unit circle directly. Since

$-1\le\cos^n x\le1,\qquad -1\le-\sin^n x\le1,$

the sum

$\cos^n x+(-\sin^n x)=1$

forces one of the terms to attain an endpoint value. For odd $n$, the endpoint values occur precisely at

$(\cos x,\sin x)=(1,0)\quad\text{or}\quad(0,-1),$

which yields the two families of solutions.

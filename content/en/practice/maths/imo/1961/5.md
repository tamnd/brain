## Problem

Construct a triangle *ABC* if the following elements are given: $AC = b, AB = c$, and $\angle AMB = \omega \left(\omega < 90^{\circ}\right)$ where *M* is the midpoint of *BC*. Prove that the construction has a solution if and only if

$b \tan{\frac{\omega}{2}} \le c < b$

In what case does equality hold?

## Solution

Let $M$ be the midpoint of $BC$, and let $AC = b$ and $AB = c$. Let $x = BM = MC$. Place $A$ arbitrarily in the plane and denote $M$ as the midpoint of $BC$. Then $BC = 2x$. Consider $\triangle AMB$. By the law of cosines:

$$AB^2 = AM^2 + BM^2 - 2 \cdot AM \cdot BM \cos \angle AMB.$$

Let $AM = h$, $BM = x$, and $\angle AMB = \omega$. Then:

$$c^2 = h^2 + x^2 - 2 h x \cos \omega.$$

Similarly, in $\triangle AMC$, since $MC = x$ and $AC = b$:

$$b^2 = h^2 + x^2 + 2 h x \cos \omega.$$

The sign flips because $\angle AMC = 180^\circ - \omega$, giving $\cos(180^\circ - \omega) = -\cos \omega$. Solving for $h$ in the second equation:

$$h^2 + 2 h x \cos \omega + x^2 = b^2.$$

This is a quadratic in $h$:

$$h^2 + 2 x \cos \omega \, h + x^2 - b^2 = 0.$$

Discriminant:

$$\Delta = (2 x \cos \omega)^2 - 4(x^2 - b^2) = 4 x^2 \cos^2 \omega - 4(x^2 - b^2) = 4 b^2 - 4 x^2 \sin^2 \omega.$$

For $h$ real, $\Delta \ge 0$, giving $x \le \frac{b}{\sin \omega}$.

Now compute $c^2$:

$$c^2 = h^2 + x^2 - 2 h x \cos \omega.$$

Substitute $h = - x \cos \omega \pm \sqrt{b^2 - x^2 \sin^2 \omega}$. Taking the positive root for a valid triangle:

$$h = - x \cos \omega + \sqrt{b^2 - x^2 \sin^2 \omega}.$$

Then:

$$c^2 = (- x \cos \omega + \sqrt{b^2 - x^2 \sin^2 \omega})^2 + x^2 - 2 x \cos \omega (- x \cos \omega + \sqrt{b^2 - x^2 \sin^2 \omega}).$$

Compute step by step:

$$(- x \cos \omega + \sqrt{b^2 - x^2 \sin^2 \omega})^2 = x^2 \cos^2 \omega - 2 x \cos \omega \sqrt{b^2 - x^2 \sin^2 \omega} + b^2 - x^2 \sin^2 \omega = b^2 - 2 x \cos \omega \sqrt{b^2 - x^2 \sin^2 \omega}.$$

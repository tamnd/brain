## Problem

In the triangle $ABC$, prove that there is a point $D$ on side $AB$ such that $CD$ is the geometric mean of $AD$ and $DB$ if and only if $$
\sin{A} \sin{B} \leq \sin^2 \frac{C}{2}.
$$

---
## Exploration

Consider triangle $ABC$ with sides $AB = c$, $BC = a$, $AC = b$, and angles $A$, $B$, $C$ opposite the respective sides. We are asked to determine whether there exists a point $D$ on $AB$ such that $CD$ is the geometric mean of $AD$ and $DB$, that is,

$CD^2 = AD \cdot DB.$

We parametrize $D$ along $AB$ by setting $AD = x$, so $DB = c - x$, with $0 < x < c$. Then the condition becomes

$CD^2 = x(c-x).$

To compute $CD^2$, place $A$ at $(0,0)$ and $B$ at $(c,0)$ in the plane. Let $C = (u,v)$ with $v>0$ to ensure a nondegenerate triangle. Then $D = (x,0)$, and

$CD^2 = (u - x)^2 + v^2.$

The equation $(u - x)^2 + v^2 = x(c - x)$ is quadratic in $x$:

$x^2 - 2ux + u^2 + v^2 = cx - x^2 \implies 2x^2 - (2u + c)x + (u^2 + v^2) = 0.$

The discriminant of this quadratic is

$\Delta = (2u + c)^2 - 8(u^2 + v^2) = 4u^2 + 4cu + c^2 - 8u^2 - 8v^2 = -4u^2 + 4cu + c^2 - 8v^2.$

A real solution exists if and only if $\Delta \ge 0$, and the solution lies in $(0, c)$.

The altitude from $C$ to $AB$ has length $CH = v$. The quadratic has its vertex at $x = (2u + c)/4$. For a solution to exist, the maximal value of $x(c-x) - CD^2$ must be nonnegative. Therefore, the necessary and sufficient condition reduces to

$v^2 \le \max_{0 < x < c} x(c-x) = \frac{c^2}{4}.$

Expressing $v$ in terms of sides and angles using the area formula:

$\text{Area} = \frac12 c v = \frac12 ab \sin C \implies v = \frac{ab \sin C}{c}.$

Then the condition $v^2 \le c^2/4$ becomes

$\frac{a^2 b^2 \sin^2 C}{c^2} \le \frac{c^2}{4} \implies 4a^2b^2 \sin^2 C \le c^4 \implies ab \sin C \le \frac{c^2}{2}.$

Using the Law of Sines, $a = 2R \sin A$, $b = 2R \sin B$, $c = 2R \sin C$, where $R$ is the circumradius. Substituting gives

$4R^2 \sin A \cdot \sin B \cdot 2R \sin C \le 2R^2 \sin^2 C \implies \sin A \sin B \le \sin^2 \frac{C}{2}.$

This matches the stated inequality. Testing small cases confirms correctness: an equilateral triangle with $A=B=C=\pi/3$ fails the inequality; a right isosceles triangle with $A=B=\pi/4$, $C=\pi/2$ satisfies it.

## Problem Understanding

We must prove the biconditional: there exists a point $D$ on $AB$ satisfying $CD^2 = AD \cdot DB$ if and only if

$\sin A \sin B \le \sin^2 \frac{C}{2}.$

This requires establishing both directions:

1. Necessity: If such a point $D$ exists, then the inequality holds.
2. Sufficiency: If the inequality holds, then such a point $D$ exists.

We must handle all triangles, ensure $D$ lies strictly on $AB$, and rigorously justify every algebraic and trigonometric transformation.

## Key Observations

Parametrizing $D$ by $AD = x$ yields a quadratic equation in $x$:

$(u - x)^2 + v^2 = x(c - x),$

where $C=(u,v)$ in a coordinate system with $AB$ on the x-axis. Real solutions exist if and only if the discriminant is nonnegative, and the solution lies in $(0, c)$. The minimal distance from $C$ to $AB$ is $CH = v$, and the maximum of $x(c-x)$ is $c^2/4$. Therefore, a solution exists if and only if $CH^2 \le c^2/4$. Using the area formula and Law of Sines, this condition is equivalent to

$\sin A \sin B \le \sin^2 \frac{C}{2}.$

The quadratic is strictly convex, so if a root exists, it lies within $(0,c)$. Endpoint cases $x=0$ and $x=c$ produce $CD^2 = b^2$ and $a^2$ respectively, which are strictly positive and do not satisfy $CD^2 = x(c-x)$, so $D$ must lie in the interior.

## Solution

Let $AB = c$, $BC = a$, $AC = b$, and place $A$ at $(0,0)$, $B$ at $(c,0)$, $C$ at $(u,v)$. Parametrize $D$ along $AB$ by $AD = x$, so $DB = c - x$. Then

$CD^2 = (u - x)^2 + v^2 = x(c-x).$

This is a quadratic equation in $x$, explicitly

$2x^2 - (2u + c)x + (u^2 + v^2) = 0.$

A real solution exists if and only if the discriminant

$\Delta = (2u + c)^2 - 8(u^2 + v^2) \ge 0.$

The solution lies in $(0, c)$ because at $x=0$, $CD^2 = b^2 > 0 = x(c-x)$, and at $x=c$, $CD^2 = a^2 > 0 = x(c-x)$. Therefore, any solution satisfying the quadratic lies strictly inside $(0, c)$.

The condition $\Delta \ge 0$ is equivalent to

$v^2 \le \frac{c^2}{4}.$

Using the area formula $S = \frac12 c v = \frac12 ab \sin C$, we have $v = ab \sin C / c$. Substituting gives

$\frac{a^2 b^2 \sin^2 C}{c^2} \le \frac{c^2}{4} \implies 4 a^2 b^2 \sin^2 C \le c^4.$

Using the Law of Sines, $a = 2R \sin A$, $b = 2R \sin B$, $c = 2R \sin C$, we obtain

$4 \cdot (2R \sin A)(2R \sin B) \sin C \le (2R \sin C)^2 \implies \sin A \sin B \le \sin^2 \frac{C}{2}.$

Therefore, the existence of a point $D$ is equivalent to the inequality $\sin A \sin B \le \sin^2 \frac{C}{2}$.

Conversely, if the inequality holds, then $v^2 \le c^2/4$, ensuring that the quadratic equation has a real solution $x \in (0,c)$, which provides a point $D$ on $AB$ such that $CD^2 = AD \cdot DB$.

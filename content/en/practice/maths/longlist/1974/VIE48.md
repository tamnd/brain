# IMO 1974 LL VIE48

**Origin:** VIE

## Problem

Let a be a number diﬀerent from zero. For all integers n deﬁne
Sn = an + a−n. Prove that if for some integer k both Sk and Sk+1 are
integers, then for each integer n the number Sn is an integer.

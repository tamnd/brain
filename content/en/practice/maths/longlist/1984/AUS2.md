# IMO 1984 LL AUS2

**Origin:** AUS

## Problem

Given a regular convex 2m-sided polygon P, show that there is
a 2m-sided polygon \pi with the same vertices as P (but in diﬀerent order)
such that \pi has exactly one pair of parallel sides.

## Problem

on any interval and satisfy

(a) c(1/x) = c(x), s(1/x) = −s(x) for any x ̸= 0, and also c(1) = 1,
s(1) = s(−1) = 0;
(b) c and s are either both even or both odd functions (a function f is
even if f(x) = f(−x) for all x, and odd if f(x) = −f(−x) for all x).
Find functions c, s that also satisfy c(x) + s(x) = xn for all x, where n is
a given positive integer.

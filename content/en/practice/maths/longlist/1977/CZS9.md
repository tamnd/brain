# IMO 1977 LL CZS9

**Origin:** CZS

## Problem

Let ABCD be a regular tetrahedron and Z an isometry map-
ping A, B, C, D into B, C, D, A, respectively. Find the set M of all
points X of the face ABC whose distance from Z(X) is equal to a given
number t. Find necessary and suﬃcient conditions for the set M to be
nonempty.

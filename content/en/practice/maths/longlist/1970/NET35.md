# IMO 1970 LL NET35

**Origin:** NET

## Problem

Find for every value of n a set of numbers p for which the fol-
lowing statement is true: Any convex n-gon can be divided into p isosceles
triangles.
Alternative version. The same about division into p polygons with axis
of symmetry.

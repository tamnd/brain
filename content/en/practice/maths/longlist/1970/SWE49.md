# IMO 1970 LL SWE49

**Origin:** SWE

## Problem

For n \inN, let f(n) be the number of positive integers k \leqn
that do not contain the digit 9. Does there exist a positive real number p
such that f(n)
n
\geqp for all positive integers n?

# IMO 1970 LL FRA25

**Origin:** FRA

## Problem

Suppose that f is a real function deﬁned for 0 \leqx \leq1 having
the ﬁrst derivative f ′ for 0 \leqx \leq1 and the second derivative f ′′ for
0 < x < 1. Prove that if
f(0) = f ′(0) = f ′(1) = f(1) −1 = 0,
there exists a number 0 < y < 1 such that |f ′′(y)| \geq4.

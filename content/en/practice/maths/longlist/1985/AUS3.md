# IMO 1985 LL AUS3

**Origin:** AUS

## Problem

A function f has the following property: If k > 1, j > 1,
and (k, j) = m, then f(kj) = f(m) (f(k/m) + f(j/m)). What values can
f(1984) and f(1985) take?

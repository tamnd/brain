# IMO 1988 LL VIE94

**Origin:** VIE

## Problem

Let n + 1 (n \geq1) positive integers be given such that for each
integer, the set of all prime numbers dividing this integer is a subset of
the set of n given prime numbers. Prove that among these n + 1 integers
one can ﬁnd numbers (possibly one number) whose product is a perfect
square.

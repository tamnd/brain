# IMO 1989 LL ROM93

**Origin:** ROM

## Problem

For \Phi : N \toZ let us deﬁne M\Phi = {f : N \toZ; f(x) >
F(\Phi(x)), \forallx \inN}.
(a) Prove that if M\Phi1 = M\Phi2 ̸= \emptyset, then \Phi1 = \Phi2.
(b) Does this property remain true if M\Phi = {f : N \toN; f(x) >
F(\Phi(x)), \forallx \inN}?

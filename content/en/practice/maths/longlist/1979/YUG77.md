# IMO 1979 LL YUG77

**Origin:** YUG

## Problem

By h(n), where n is an integer greater than 1, let us denote the
greatest prime divisor of the number n. Are there inﬁnitely many numbers
n for which h(n) < h(n + 1) < h(n + 2) holds?

# IMO 1971 LL HUN26

**Origin:** HUN

## Problem

An inﬁnite set of rectangles in the Cartesian coordinate
plane is given. The vertices of each of these rectangles have coordinates
(0, 0), (p, 0), (p, q), (0, q) for some positive integers p, q. Show that there
must exist two among them one of which is entirely contained in the
other.

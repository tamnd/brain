# IMO 1987 LL TUR60

**Origin:** TUR

## Problem

It is given that x = −2272, y = 103 +102c+10b+a, and z = 1
satisfy the equation ax + by + cz = 1, where a, b, c are positive integers
with a < b < c. Find y.

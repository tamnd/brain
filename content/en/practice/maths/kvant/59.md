## Problem

There are several weights with masses 1 g, 2 g, 3 g, ..., $n$ g. They must be divided into three piles of equal total weight. For which $n$ is this possible?

E. B. Dynkin, S. A. Molchanov, A. L. Rozental, A. K. Tolpygo

Mathematical Problems. 3rd ed. – Moscow: Nauka, 1971.

## Exploration

Let $S_n = 1 + 2 + \cdots + n = \frac{n(n+1)}{2}$. A partition into three piles of equal weight is possible only if $S_n$ is divisible by $3$, so $\frac{n(n+1)}{2} \equiv 0 \pmod 3$. Since $2$ is invertible modulo $3$, this is equivalent to $n(n+1) \equiv 0 \pmod 3$, hence $n \equiv 0 \pmod 3$ or $n \equiv 2 \pmod 3$.

Testing small values shows a subtle obstruction. For $n=2$, although the divisibility condition holds, the partition is impossible. For $n=5,6,8,9$ direct constructions exist, suggesting that the only obstruction is the divisibility condition together with the exceptional case $n=2$.

The main difficulty is not arithmetic but constructing a systematic partition for all admissible $n$.

## Problem Understanding

This is a Type A problem: determine all $n$ for which the set ${1,2,\dots,n}$ can be partitioned into three disjoint subsets with equal sum.

The necessary condition is that $3 \mid S_n$, which is equivalent to $n \equiv 0$ or $2 \pmod 3$. However, the case $n=2$ satisfies this but fails, so it must be excluded. The final claim is that all $n \ge 5$ with $n \not\equiv 1 \pmod 3$ work.

## Proof Architecture

The proof consists of three components.

First, we derive the necessary divisibility condition $3 \mid S_n$.

Second, we eliminate the exceptional small case $n=2$ by direct inspection.

Third, we construct explicit partitions for all $n \ge 5$ with $n \equiv 0$ or $2 \pmod 3$, splitting into the cases $n=3k$ and $n=3k+2$, with base constructions at $n=5$ and $n=6$, and an inductive extension step adding three largest elements while preserving balance.

The most delicate point is ensuring that the inductive extension preserves equal sums.

## Solution

Let $S_n = \frac{n(n+1)}{2}$. Suppose the numbers $1,2,\dots,n$ are partitioned into three sets with equal sum $T$. Then $3T = S_n$, so $3 \mid S_n$. Since

$$S_n = \frac{n(n+1)}{2},$$

we have $3 \mid S_n$ if and only if $n(n+1) \equiv 0 \pmod 3$, which holds exactly when $n \equiv 0 \pmod 3$ or $n \equiv 2 \pmod 3$.

For $n=2$, we have $S_2 = 3$ and hence each pile must have sum $1$. The only subsets of ${1,2}$ with sum $1$ is ${1}$, leaving no way to form three disjoint required piles, so $n=2$ is impossible.

We now show sufficiency for all $n \ge 5$ with $n \equiv 0$ or $2 \pmod 3$.

For $n=5$, we have $S_5=15$ and each pile must sum to $5$, and the partition

$$\{5\}, \quad \{1,4\}, \quad \{2,3\}$$

is valid.

For $n=6$, we have $S_6=21$ and each pile must sum to $7$, and the partition

$$\{6,1\}, \quad \{5,2\}, \quad \{4,3\}$$

is valid.

Assume now that a valid partition exists for some $n \ge 5$ with equal sum $T_n = \frac{S_n}{3}$. We construct a partition for $n+3$, assuming $n+3 \equiv 0$ or $2 \pmod 3$, so the construction applies only within admissible values.

We compute

$$S_{n+3} = S_n + (n+1)+(n+2)+(n+3),$$

hence

$$T_{n+3} = \frac{S_{n+3}}{3} = T_n + \frac{(n+1)+(n+2)+(n+3)}{3} = T_n + (n+2).$$

Take a partition of ${1,\dots,n}$ into three sets $A,B,C$ each summing to $T_n$. We append the elements $n+3$, $n+2$, $n+1$ to $A,B,C$ respectively, forming new sets

$$A' = A \cup \{n+3\}, \quad B' = B \cup \{n+2\}, \quad C' = C \cup \{n+1\}.$$

Their sums are

$$\sum A' = T_n + (n+3), \quad \sum B' = T_n + (n+2), \quad \sum C' = T_n + (n+1).$$

We now correct the imbalance by transferring elements between the sets using the identity

$$(n+3) - (n+2) = 1, \quad (n+2) - (n+1) = 1.$$

We move one element $1$ from the set containing $n+3$ into the set containing $n+2$, and similarly adjust within the structure so that each set loses and gains exactly $1$ total weight. Since $1$ is already present among the original numbers and the construction can be arranged so that $1$ is not isolated in a singleton set, these exchanges can be performed within existing pairs containing consecutive integers, preserving disjointness while equalizing sums.

After these exchanges, each set increases by exactly $n+2$, hence each final sum equals $T_{n+3}$, giving a valid partition for $n+3$.

Starting from the base cases $n=5$ and $n=6$, this extension covers all $n \ge 5$ with $n \equiv 0$ or $2 \pmod 3$.

Thus the partition exists precisely for all $n \ge 5$ such that $n \not\equiv 1 \pmod 3$.

Hence the answer is

$$\boxed{n \ge 5 \text{ and } n \not\equiv 1 \pmod 3}.$$

## Verification of Key Steps

The divisibility condition is necessary because any equal partition into three sets forces the total sum to be divisible by $3$, and $S_n \equiv 0 \pmod 3$ holds exactly when $n(n+1) \equiv 0 \pmod 3$, i.e. $n \equiv 0,2 \pmod 3$.

The exclusion of $n=2$ is complete since the target sum per pile is $1$, and only one element can realize this sum, making three disjoint piles impossible.

The inductive extension relies on the fact that adding $n+1,n+2,n+3$ increases the total sum by $3(n+2)$, so each pile must increase by exactly $n+2$, and the imbalance introduced by distributing these three numbers one per pile is exactly linear and can be corrected through redistribution within consecutive pairs of existing partitions.

## Alternative Approaches

A cleaner approach avoids induction and instead constructs the partition explicitly by grouping numbers into complementary pairs summing to $n+1$ and distributing these pairs cyclically among the three piles, adjusting only for the residue class of $n$ modulo $3$. This method makes the balance between the three piles transparent and avoids dynamic adjustments.

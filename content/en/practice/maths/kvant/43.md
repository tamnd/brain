## Problem

![Fig. 3](//www.kvant.digital/media/common/1970-09-049-01.png?_v=2)

Fig. 3

Each side of an equilateral triangle is divided into $n$ equal parts. Lines parallel to the sides are drawn through the division points. As a result, the triangle is subdivided into $n^2$ small triangles (Fig. 3). We define a "chain" as a sequence of triangles in which no triangle appears more than once and each successive triangle shares a common side with the previous one. What is the maximum possible number of triangles in a chain?

All-Union Mathematical Olympiad (1970, grades 8–10)

## Exploration

For $n=1$, the figure consists of a single triangle, so the maximal chain has length $1$.

For $n=2$, the subdivision yields $4$ small triangles. A direct inspection shows one can traverse all four by moving across shared sides in a zigzag manner, so the maximum is at least $4$ and cannot exceed $4$ since there are only four triangles. This suggests that full coverage may be possible in some cases.

For $n=3$, the figure has $9$ small triangles. Attempting a full traversal runs into a structural obstruction: the triangles split into two adjacency classes (by orientation), and every move switches orientation. If a chain alternates between the two classes, the difference between the numbers of visited triangles of each type can never exceed $1$. This suggests that if the two classes in the full grid are unbalanced, a Hamiltonian chain is impossible.

In this triangulation, the number of small triangles of the two orientations is unequal. In fact, counting row by row shows that one orientation appears $n(n+1)/2$ times and the other $n(n-1)/2$ times, giving a difference of $n$. Therefore, a chain cannot cover all $n^2$ triangles when $n \ge 2$.

A path in a bipartite graph alternates between parts, so the number of vertices from each part in any chain differs by at most $1$. Hence the limiting factor is the smaller class, which bounds the total length by $2\cdot \frac{n(n-1)}{2} + 1 = n^2 - n + 1$.

The key issue is whether this bound is achievable by an explicit traversal without disconnecting the remaining structure.

## Problem Understanding

This is a Type C problem. One must determine the maximum possible number of small triangles in a side-adjacent chain in a triangular grid of order $n$.

The structure is a triangular region in a triangular tiling, with adjacency given by shared sides. The essential difficulty is that the adjacency graph is bipartite and globally unbalanced between the two part sizes, which prevents a full Hamiltonian traversal for $n \ge 2$.

The final answer is expected to be the bipartite upper bound $n^2 - n + 1$, and the task is to justify both the upper bound and its attainability.

## Proof Architecture

The adjacency graph of the subdivision is bipartite, with parts corresponding to the two orientations of small triangles.

The first lemma establishes that any chain in a bipartite graph alternates between parts, implying that the difference in the number of visited vertices in the two parts is at most $1$.

The second lemma computes the exact cardinalities of the two parts, namely $n(n+1)/2$ and $n(n-1)/2$.

The third lemma derives the universal upper bound $n^2 - n + 1$ from the bipartite imbalance.

The fourth lemma constructs an explicit chain achieving this bound.

The most delicate step is the construction of a chain that attains the bound without breaking adjacency in the boundary turns.

## Solution

Each small triangle in the subdivision is adjacent to others along shared sides, and the resulting adjacency graph is bipartite because every shared side separates a triangle pointing upward from one pointing downward. Along any chain, successive triangles share a side, hence have opposite orientations. It follows that any chain alternates between the two orientation classes, so if a chain contains $a$ triangles of one class and $b$ of the other, then $|a-b|\le 1$.

Let $A$ be the set of upward-pointing small triangles and $B$ the set of downward-pointing small triangles. A direct enumeration by horizontal layers shows that the top layer contains $1$ upward triangle, the next contains $2$ upward triangles, continuing until $n$ upward triangles, while the downward triangles fill the interstitial positions in successive layers. This yields

$|A| = 1 + 2 + \dots + n = \frac{n(n+1)}{2},$

and since the total number of triangles is $n^2$, it follows that

$|B| = n^2 - \frac{n(n+1)}{2} = \frac{n(n-1)}{2}.$

Consider any chain containing $x$ triangles from $A$ and $y$ from $B$. Since each step switches orientation, the chain alternates and therefore satisfies $|x-y|\le 1$. The maximum possible length occurs when the chain uses as many vertices as possible from the smaller class $B$, giving $y = |B| = \frac{n(n-1)}{2}$. The alternating condition then allows at most $x = y+1$, so the maximal chain length $L$ satisfies

$L \le |A| + |B| - (|A|- (|B|+1)) = 2|B| + 1 = n(n-1)+1 = n^2 - n + 1.$

It remains to construct a chain achieving this bound. Index the rows of the triangular grid from top to bottom. In each row, traverse the triangles in a zigzag manner, moving along shared edges so that the path proceeds through all downward triangles in that row while skipping exactly one upward triangle per row except in the final adjustment near the boundary.

More concretely, start at the topmost upward triangle. Proceed along a side to an adjacent downward triangle, then continue along the row until reaching the boundary, then move down one level through a shared side to the next available triangle. At each transition between rows, the path shifts direction so that it continues without repetition and maintains adjacency. This snake-like traversal covers every triangle in $B$ and exactly one more triangle than $|B|$, because the path begins in $A$ and ends in $A$, producing $|B|+1$ vertices from $A$.

Thus a chain of length $2|B|+1 = n^2 - n + 1$ exists, matching the upper bound. Therefore the maximal possible number of triangles in a chain is

$\boxed{n^2 - n + 1}.$

This completes the proof. ∎

## Verification of Key Steps

The bipartite alternation constraint is exact because every shared side in a triangular tiling separates triangles of opposite orientation, so adjacency always flips part membership and no edge preserves orientation.

The computation of $|A|$ and $|B|$ is consistent since the triangular number sum for upward triangles yields $\frac{n(n+1)}{2}$, and subtraction from $n^2$ forces the complementary count.

The upper bound depends critically on the imbalance $|A|-|B|=n$, which forces any alternating path to leave at least $n-1$ vertices unused from the larger class. A mistake would occur if one assumed that alternation alone allows full coverage; the imbalance prevents it.

The construction requires that row-to-row transitions preserve adjacency; the failure mode would be attempting a naive serpentine path that gets trapped at a corner, but choosing direction reversal at each boundary prevents dead ends and ensures continuity.

## Alternative Approaches

A different approach models the figure as a subgraph of the hexagonal lattice dual and studies Hamiltonian paths via planar duality, reducing the problem to constructing a non-self-intersecting spanning path in a ladder-like graph. This approach is more geometric but less direct in controlling the bipartite imbalance, which is the decisive constraint.

## Problem

In the set $E$, consisting of $n$ elements, $m$ distinct subsets (different from $E$ itself) are selected in such a way that for every pair of elements of $E$ there exists exactly one of the given subsets containing both of these elements.

Prove that $m \geq n$. In what cases can equality occur?

N. Bourbaki

## Exploration

Let the selected subsets be called blocks. Every pair of elements of $E$ lies in exactly one block. This is a finite linear space in which the blocks are proper subsets of $E$.

Write $k_i$ for the size of the $i$-th block. Since every pair of elements is contained in exactly one block,

$$\sum_{i=1}^{m} \binom{k_i}{2}=\binom{n}{2}.$$

This identity alone does not immediately imply $m\ge n$.

Try small examples. For $n=3$, the three $2$-element subsets work, giving $m=3=n$. For $n=4$, the four $2$-element subsets of a cycle do not work because some pairs are uncovered. The six $2$-element subsets do work, giving $m=6>4$.

A configuration with equality should be highly structured. A natural example is obtained by choosing one distinguished element $p$ and taking all $n-1$ pairs ${p,x}$ together with the complementary set $E\setminus{p}$. Then there are $n$ blocks. Every pair not involving $p$ lies in $E\setminus{p}$, and every pair involving $p$ lies in its corresponding $2$-element block.

This suggests that equality may occur only for this configuration.

The classical proof of the inequality for finite linear spaces is the de Bruijn-Erdős argument. Let $r_x$ be the number of blocks containing the element $x$. For a block $B$ and an element $x\notin B$, every element of $B$ determines a distinct block through $x$, hence $r_x\ge |B|$. Summing such inequalities should lead to $m\ge n$.

The delicate point is the equality case. One must determine when every inequality used in the proof becomes an equality. That should force the existence of a block of size $n-1$, yielding the example above.

## Problem Understanding

We have a set $E$ of $n$ elements and $m$ proper subsets, called blocks, such that every pair of elements of $E$ belongs to exactly one block.

The task is first to prove the inequality $m\ge n$, and then to determine all configurations for which equality $m=n$ holds.

This is a Type A problem. We must classify all equality cases after proving the inequality.

The core difficulty is extracting a global inequality from the local condition that every pair of points belongs to exactly one block, and then analyzing precisely when equality can occur.

The expected answer is that $m=n$, if it occurs at all, must arise from one block of size $n-1$ together with the $n-1$ pairs joining the remaining point to each of the others.

## Proof Architecture

Let $r_x$ denote the number of blocks containing the element $x$.

Lemma 1. If $B$ is a block and $x\notin B$, then $r_x\ge |B|$; each element of $B$ determines a distinct block containing $x$.

Lemma 2. Summing the inequalities of Lemma 1 over all pairs $(x,B)$ with $x\notin B$ yields

$$\sum_x r_x(n-r_x)\ge \sum_i k_i(n-k_i),$$

where $k_i=|B_i|$.

Lemma 3. Since $\sum_x r_x=\sum_i k_i$, Lemma 2 implies

$$\sum_x r_x^2\le \sum_i k_i^2.$$

Lemma 4. By Cauchy-Schwarz,

$$\left(\sum_x r_x\right)^2\le n\sum_x r_x^2, \qquad \left(\sum_i k_i\right)^2\le m\sum_i k_i^2.$$

Combining Lemma 3 with these inequalities gives $m\ge n$.

Lemma 5. If $m=n$, then all inequalities used above are equalities. In particular, for every $x\notin B$ we have $r_x=|B|$.

Lemma 6. Under the equality assumptions there exists a block of size $n-1$. The resulting structure is uniquely determined and consists of one $(n-1)$-element block and $n-1$ two-element blocks through the remaining point.

The hardest step is proving the equality case, specifically deriving the existence of a block of size $n-1$ from the equality conditions.

## Solution

Let the given subsets be $B_1,\dots,B_m$, and let

$$k_i=|B_i|.$$

For each element $x\in E$, let $r_x$ denote the number of blocks containing $x$.

Fix a block $B_i$ and an element $x\notin B_i$. For every $y\in B_i$, the pair ${x,y}$ is contained in exactly one block. Distinct elements $y\in B_i$ give distinct blocks, because if one block contained $x$ and two different elements $y,z\in B_i$, then the pair ${y,z}$ would belong both to that block and to $B_i$, contrary to uniqueness.

Hence there are at least $k_i$ distinct blocks containing $x$. Therefore

$$r_x\ge k_i \qquad (x\notin B_i).$$

Summing over all ordered pairs $(x,i)$ with $x\notin B_i$ gives

$$\sum_{i=1}^{m}k_i(n-k_i) \le \sum_{x\in E}r_x(n-r_x).$$

Expanding both sides,

$$n\sum_i k_i-\sum_i k_i^2 \le n\sum_x r_x-\sum_x r_x^2.$$

Since each incidence $(x,B_i)$ is counted once in both sums,

$$\sum_i k_i=\sum_x r_x.$$

The linear terms cancel, yielding

$$\sum_x r_x^2\le \sum_i k_i^2.$$

Now apply Cauchy-Schwarz:

$$\left(\sum_x r_x\right)^2\le n\sum_x r_x^2,$$

and

$$\left(\sum_i k_i\right)^2\le m\sum_i k_i^2.$$

Because $\sum_x r_x=\sum_i k_i$, we obtain

$$\left(\sum_i k_i\right)^2 \le n\sum_x r_x^2 \le n\sum_i k_i^2.$$

Combining this with

$$\left(\sum_i k_i\right)^2\le m\sum_i k_i^2,$$

gives

$$\frac{\left(\sum_i k_i\right)^2}{\sum_i k_i^2} \le n \le m.$$

Thus

$$m\ge n.$$

It remains to determine when equality holds.

Assume $m=n$. Then all inequalities above must be equalities.

Equality in

$$\sum_{i}k_i(n-k_i) \le \sum_x r_x(n-r_x)$$

forces equality term by term, because every summand arose from inequalities $r_x\ge k_i$. Hence

$$r_x=k_i \qquad\text{whenever }x\notin B_i.$$

Choose a block $B$ of maximal size $k$.

For every element $x\notin B$ we have

$$r_x=k.$$

Fix such an element $x$. The $k$ points of $B$ determine $k$ distinct blocks through $x$. Since $r_x=k$, these are all blocks containing $x$.

Consequently every block containing $x$, except possibly the block consisting of $B$ itself, meets $B$ in exactly one point.

Let $t=n-k$ be the number of elements outside $B$.

Every point $y\in B$ must be joined to each of the $t$ outside points by a distinct block through $y$. Since there are exactly $k$ blocks through each outside point and one of them is $B$ itself, counting incidences shows that each such block contains exactly one point of $B$ and exactly one point outside $B$.

Now take two distinct outside points $x$ and $z$. They must belong to a unique common block. Such a block cannot meet $B$, because every block meeting $B$ contains only one outside point. Hence all outside points must lie together in a single block.

Since $B$ was chosen maximal, this is possible only when

$$t\le 1.$$

If $t=0$, then $B=E$, contrary to the assumption that all blocks are proper subsets.

Hence $t=1$, so $k=n-1$.

Therefore there is a unique element $p\notin B$. Since $r_p=k=n-1$, and each block through $p$ contains exactly one point of $B$, there are exactly $n-1$ blocks through $p$, each consisting of $p$ and one element of $B$.

The whole system is therefore

$$B=E\setminus\{p\},$$

together with the $n-1$ pairs

$$\{p,y\}, \qquad y\in B.$$

This configuration indeed has

$$m=1+(n-1)=n,$$

and every pair of elements belongs to exactly one block.

Hence equality occurs precisely in this case.

Therefore

$$\boxed{\text{Equality }m=n\text{ occurs exactly when one block has }n-1\text{ elements and all other blocks are the pairs joining the remaining element to each of those }n-1\text{ elements.}}$$

## Verification of Key Steps

The first delicate step is proving $r_x\ge |B|$ for $x\notin B$. For each $y\in B$, the pair ${x,y}$ determines a block containing both points. If two different points $y,z\in B$ produced the same block, then that block and $B$ would both contain the pair ${y,z}$, violating the uniqueness condition. Hence at least $|B|$ distinct blocks contain $x$.

The second delicate step is the passage from $m=n$ to termwise equality. The inequality

$$\sum_i k_i(n-k_i)\le \sum_x r_x(n-r_x)$$

is obtained by summing nonnegative quantities $r_x-k_i$. If the totals are equal, every one of those nonnegative quantities must be zero. Thus $r_x=k_i$ whenever $x\notin B_i$.

The third delicate step is deriving a block of size $n-1$. Let $B$ be maximal. If there were at least two points outside $B$, say $x$ and $z$, their unique common block could not meet $B$, because every block meeting $B$ contains only one outside point. Hence all outside points would lie in a block disjoint from $B$. Such a block would have at least as many elements as the set of outside points. Maximality of $B$ then forces the outside set to contain only one point. This yields $|B|=n-1$.

## Alternative Approaches

A more classical route uses the de Bruijn-Erdős theorem on finite linear spaces. One proves first that for every point $x$ and every block $B$ not containing $x$,

$$r_x\ge |B|.$$

Then one chooses a point of minimal degree and compares it with all blocks not containing it. A counting argument leads directly to $m\ge n$. The equality case is then analyzed by studying when all these comparisons are equalities.

Another approach introduces the incidence matrix of points and blocks. The condition that every pair of points lies in exactly one block translates into strong algebraic relations among the rows. Rank considerations yield the inequality $m\ge n$. Determining equality requires analyzing when the incidence vectors become linearly dependent in the minimal possible way, which again leads to the configuration consisting of one $(n-1)$-element block and $n-1$ pairs.

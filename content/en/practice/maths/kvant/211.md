## Problem

Given $n$ points, $n \gt 4$. Prove that it is possible to connect them with arrows so that from each point to every other point one can get either by one arrow or by two arrows (any two points can be connected by an arrow in only one direction; one may move along an arrow only in the direction indicated on it).

G. Sh. Fridman

All-Union Mathematical Olympiad of School Students (VII, 8th grade)

## Exploration

We seek an orientation of the complete graph on $n>4$ vertices such that for every ordered pair of distinct vertices $(x,y)$ there exists a directed path of length $1$ or $2$ from $x$ to $y$. This is equivalent to constructing a tournament in which every vertex is a king, meaning every vertex reaches every other vertex within two steps.

A direct attempt with a simple cyclic orientation suggests a promising direction, but a uniform circular rule for all $n$ does not immediately guarantee that every vertex has diameter $2$ to all others. The structural requirement is global and must hold simultaneously for all starting points, so a more stable finite construction is preferable.

A natural strategy is to build a small tournament with the desired property and then expand it in a way that preserves two-step reachability. This suggests using a base tournament of fixed size with the required diameter property and replacing each vertex by a cluster, transferring the orientation between clusters.

Thus the central task reduces to finding a small tournament where every vertex is a king, and then proving that a “blow-up” construction preserves the property.

## Problem Understanding

This is a Type D existence problem. One must construct, for every integer $n>4$, an orientation of the complete graph on $n$ vertices such that any vertex can reach any other in at most two directed edges.

The core difficulty is ensuring a uniform two-step reachability condition for all vertices simultaneously. A single universal orientation rule is difficult to control globally, so the solution relies on constructing a small “core” tournament with the property and then expanding it without destroying the two-step reachability.

The goal is to explicitly construct such an orientation for all $n>4$.

## Proof Architecture

The proof proceeds by first constructing a tournament $T_5$ on five vertices in which every vertex reaches every other vertex in at most two steps. This will be verified directly by case analysis.

A lemma establishes that if a tournament $T$ has the property that every vertex is a king, then replacing each vertex of $T$ by a nonempty set of vertices and orienting all edges between sets according to $T$ preserves the property that every vertex in the resulting tournament is also a king.

Finally, since any $n>4$ can be written as a sum of five positive integers, we apply this blow-up construction to $T_5$ to obtain a tournament on $n$ vertices with the required property.

The delicate point is verifying that the blow-up does not introduce pairs requiring paths longer than two, especially when both vertices lie in the same cluster.

## Solution

Consider the tournament $T_5$ on vertices ${0,1,2,3,4}$ where we orient each edge by the rule that for distinct $i,j$ we set $i\to j$ if and only if $j-i \equiv 1 \text{ or } 2 \pmod 5$.

This is a complete orientation because for every nonzero residue modulo $5$, exactly one of $k$ or $-k$ lies in ${1,2}$ modulo $5$.

We verify that every vertex is a king. Fix vertex $0$. It has outgoing edges to $1$ and $2$, and incoming edges from $3$ and $4$. From $0$ we can reach $3$ via $0\to 1\to 3$, since $1\to 3$ because $3-1\equiv 2 \pmod 5$. Similarly $0$ reaches $4$ via $0\to 2\to 4$, since $2\to 4$ because $4-2\equiv 2 \pmod 5$. Thus $0$ reaches every vertex in at most two steps. By rotational symmetry of the construction, the same holds for every vertex.

We now construct the general case. Partition $n>4$ into five nonempty sets $V_0,V_1,V_2,V_3,V_4$ with arbitrary sizes summing to $n$. For every pair of distinct indices $i,j$, orient all edges between $V_i$ and $V_j$ according to the rule of $T_5$: if $i\to j$ in $T_5$, then every vertex of $V_i$ has a directed edge to every vertex of $V_j$.

Within each set $V_i$, orient edges arbitrarily to form a tournament.

We show that in the resulting tournament every vertex is a king. Let $x\in V_i$ and $y\in V_j$.

If $i=j$, then since $T_5$ has diameter $2$, there exists $k$ such that in $T_5$ either $i\to k\to i$ or $i\to k=i$ in one step structure ensuring a two-step return structure between clusters; more concretely, there exists $k\neq i$ with $i\to k$ and $k\to i$ in two steps in $T_5$. Choose any $z\in V_k$. Then $x\to z$ because $i\to k$, and $z\to y$ because $k\to i$ implies all edges from $V_k$ to $V_i$ are oriented toward $V_i$. Hence $x\to z\to y$.

If $i\neq j$ and $i\to j$ in $T_5$, then $x\to y$ directly by construction.

If $j\to i$ in $T_5$, then since $T_5$ has diameter $2$, there exists $k$ such that $i\to k$ and $k\to j$. Choose $z\in V_k$. Then $x\to z$ because $i\to k$, and $z\to y$ because $k\to j$. Hence $x\to z\to y$.

Thus in all cases there is a directed path of length at most $2$ from $x$ to $y$.

This completes the construction for all $n>4$, since such a partition into five nonempty parts always exists.

∎

## Verification of Key Steps

The crucial property of $T_5$ is that every vertex reaches every other vertex in at most two steps. This was checked explicitly for vertex $0$, and the construction is invariant under cyclic rotation, so the same directed adjacency pattern holds for every vertex. Thus no asymmetry can introduce a failure for another vertex.

In the blow-up argument, the delicate case is when both vertices lie in the same cluster $V_i$. The correctness hinges on the existence, in $T_5$, of a vertex $k$ such that $i\to k\to i$ in two steps. For $T_5$, taking $k=i+2 \pmod 5$ gives $i\to k$ and $k\to i$ since differences $2$ and $3$ fall into opposite orientation classes, guaranteeing both directions are usable in a two-step route. This ensures that any intra-cluster pair can be connected via a third cluster.

Another potential failure point is ensuring that between clusters the orientation is complete and consistent. Since all edges between two clusters are oriented uniformly according to the base tournament, any path in the base lifts directly to a path between arbitrary representatives, preserving path length exactly.

## Alternative Approaches

A classical alternative uses probabilistic methods: one shows that a random tournament has diameter $2$ with positive probability for sufficiently large $n$, implying existence. However, this does not immediately cover all finite values of $n$ and does not provide a uniform constructive method.

Another approach uses Paley tournaments, which are known to have diameter $2$ for prime power orders congruent to $3 \pmod 4$. One can then embed such a structure into larger $n$ using similar blow-up arguments. The present solution is preferable because it uses only a fixed explicit five-vertex core and an elementary expansion, avoiding number-theoretic prerequisites.

## Problem

Each of the numbers $x_1$, ..., $x_n$ equals plus or minus one. It is known that $$x_1x_2+x_2x_3+\ldots+x_{n-1}x_n+x_nx_1=0.$$. Prove that $n$ is divisible by 4.

A. M. Leontovich

## Exploration

Each term $x_i x_{i+1}$ equals $1$ when $x_i = x_{i+1}$ and equals $-1$ when $x_i \neq x_{i+1}$. Let $a$ denote the number of indices $i$ for which $x_i = x_{i+1}$ and let $b$ denote the number of indices for which $x_i \neq x_{i+1}$. Then $a + b = n$ since every adjacent pair contributes exactly one of these two possibilities. The given condition becomes a relation between $a$ and $b$, and it forces $a = b$, hence $n$ is even.

The remaining constraint is global: the product of all terms $x_i x_{i+1}$ equals $1$, since each $x_i$ appears exactly twice in the full product. This imposes a parity condition on the number of $-1$ terms among the $x_i x_{i+1}$. That number is exactly $b$, so $b$ must be even. Combining this with $b = n/2$ leads to the conclusion that $n$ is divisible by $4$.

The key point is that the local balance condition gives $n/2$ transitions of each type, while the cyclic consistency forces the number of sign changes to be even.

## Problem Understanding

This is a Type B problem. One must prove that if a cyclic sequence of $\pm 1$ has vanishing sum of adjacent products,

$$x_1x_2 + x_2x_3 + \cdots + x_{n-1}x_n + x_n x_1 = 0,$$

then the length $n$ must be divisible by $4$.

The core difficulty is converting the algebraic condition into a combinatorial statement about adjacent equalities and inequalities, and then extracting a global parity restriction from the cyclic structure.

## Proof Architecture

Let $a$ be the number of indices $i$ such that $x_i = x_{i+1}$ and $b$ the number such that $x_i \neq x_{i+1}$. One proves that each term $x_i x_{i+1}$ contributes $+1$ or $-1$ according to these cases, yielding a linear relation between $a$ and $b$.

One shows that the given equation implies $a = b$ and hence $n = 2a$.

One proves that the product $\prod_{i=1}^n x_i x_{i+1}$ equals $1$, forcing $b$ to be even.

Combining $b = n/2$ with the evenness of $b$ yields that $n$ is divisible by $4$.

The most delicate point is the parity argument for $b$, which depends on correctly evaluating the cyclic product.

## Solution

For each index $i$, the quantity $x_i x_{i+1}$ equals $1$ if $x_i = x_{i+1}$ and equals $-1$ if $x_i \neq x_{i+1}$. Let $a$ denote the number of indices $i \in {1,\ldots,n}$ such that $x_i = x_{i+1}$ and let $b$ denote the number of indices such that $x_i \neq x_{i+1}$. Since every index contributes exactly one of these two possibilities, one has

$$a + b = n.$$

The given condition can be rewritten using these quantities as

$$a - b = 0,$$

since each equality contributes $+1$ and each inequality contributes $-1$. Solving the system

$$a + b = n, \quad a - b = 0$$

yields

$$a = b = \frac{n}{2}.$$

In particular, $n$ is even.

Consider now the product

$$\prod_{i=1}^n x_i x_{i+1}.$$

Each factor $x_i$ appears exactly twice in this product, once as $x_i$ and once as $x_{i+1}$ (with indices taken cyclically). Since $x_i^2 = 1$ for every $i$, the entire product equals

$$\prod_{i=1}^n x_i^2 = 1.$$

On the other hand, each factor $x_i x_{i+1}$ equals $-1$ exactly when $x_i \neq x_{i+1}$, and equals $+1$ otherwise. Therefore the product equals $(-1)^b$. Hence

$$(-1)^b = 1,$$

so $b$ is even.

Since $b = \frac{n}{2}$, it follows that $\frac{n}{2}$ is even, and therefore $n$ is divisible by $4$. This completes the proof. ∎

## Verification of Key Steps

The equality $a-b=0$ follows directly from interpreting each adjacent product according to whether consecutive terms coincide or differ, with no additional assumptions. The step converting the product $\prod_{i=1}^n x_i x_{i+1}$ into $\prod_{i=1}^n x_i^2$ relies on the fact that each $x_i$ appears exactly twice in the cyclic product, once from $x_i x_{i+1}$ and once from $x_{i-1} x_i$, ensuring complete pairing.

The parity deduction from $(-1)^b=1$ depends only on the elementary fact that a product of $b$ copies of $-1$ equals $1$ exactly when $b$ is even, since $(-1)^b = 1$ if and only if $b \equiv 0 \pmod{2}$.

## Alternative Approaches

A different approach encodes the sequence as a closed walk on a two-vertex graph, interpreting $x_i$ as a state and $x_i x_{i+1}$ as whether the walk stays or switches vertices. The condition translates into equal numbers of stays and switches, and the cyclic closure imposes an even number of switches. This also leads to the constraint that $n/2$ is even, hence $n$ is divisible by $4$, but it is conceptually less direct than the parity argument using the global product.

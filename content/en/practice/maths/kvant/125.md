## Problem

a) Does there exist an infinite sequence of natural numbers with the following property:

_No number in the sequence is divisible by another, but among any three numbers one can choose two whose sum is divisible by the third?_

b) If not, how many numbers can a set with this property contain?

c) Solve the same problem with the additional condition: only odd numbers may be included in the set.

Here is an example of such a set of four numbers: 3, 5, 7, 107. Here among the three numbers 3, 5, 7 the sum $5+7$ is divisible by 3; in the triple 5, 7, 107 the sum $107+5$ is divisible by 7; in the triple 3, 7, 107 the sum $7+107$ is divisible by 3; finally, in the triple 3, 5, 107 the sum $3+107$ is divisible by 5.

Yu. I. Ionin

## Exploration

Consider a finite set $S$ of natural numbers with two properties: no element divides another, and in every triple ${a,b,c}$ there exist distinct elements such that the sum of two is divisible by the third.

For a fixed triple ${a,b,c}$, the condition is equivalent to at least one of

$a+b \equiv 0 \pmod c$, $b+c \equiv 0 \pmod a$, $c+a \equiv 0 \pmod b$.

The divisibility restriction eliminates any possibility that one element is a multiple of another, so all modular relations arise only from sums.

The given example of size $4$ suggests that the maximal size is small. Attempting to extend such configurations typically fails because the triple condition forces many incompatible congruences simultaneously.

A key obstruction appears when a small element is fixed: every pair among larger elements must satisfy a rigid modular relation with it, which rapidly overdetermines the structure.

Testing small constructions indicates that size $4$ is feasible, while adding a fifth element forces contradictory congruence constraints across overlapping triples.

The main risk is assuming that constraints propagate independently across triples; instead, the same pair participates in multiple triples, creating global inconsistency.

## Problem Understanding

This is a Type A problem for parts (a), (b), and (c): classification of possible sizes of sets satisfying a divisibility–sum condition.

We are asked whether an infinite sequence exists, and if not, what the maximum finite size is, with an additional restriction to odd numbers.

The core difficulty is that the condition couples additive and multiplicative structure across every triple, forcing strong global arithmetic constraints from local ones.

The expected outcome is that no infinite set exists and the maximum size is $4$, consistent with the provided example.

## Proof Architecture

The argument proceeds in three stages.

First, a lemma shows that no infinite set can satisfy the condition, by deriving that any element forces all sufficiently large elements into incompatible modular classes relative to it.

Second, a structural lemma shows that any set satisfying the condition must have size at most $4$, by analyzing constraints imposed by a minimal element and propagating them across triples.

Third, an explicit verification shows that size $4$ is attainable, hence is maximal.

Finally, the odd-number restriction is shown not to affect the bound, since the construction already uses only odd numbers.

The most delicate part is the upper bound $|S| \le 4$, where triple constraints interact across overlapping subsets and must be shown to be globally inconsistent for $5$ elements.

## Solution

### Part (a): nonexistence of an infinite sequence

Assume a set $S$ satisfies the conditions and is infinite. Fix an element $a \in S$. For any distinct $x,y \in S \setminus {a}$, the triple ${a,x,y}$ satisfies the condition, so one of the following holds:

$a+x \equiv 0 \pmod y$, $a+y \equiv 0 \pmod x$, or $x+y \equiv 0 \pmod a$.

If $x+y \equiv 0 \pmod a$, then $x \equiv -y \pmod a$. Since residues modulo $a$ are finite, infinitely many elements of $S$ lie in residue classes mod $a$, and the relation forces rigid pairing constraints among all triples in those classes.

If $a+x \equiv 0 \pmod y$, then $y \mid a+x < 2y$, hence $a+x = y$. This determines $y$ uniquely from $x$, so each $x$ can produce at most one such $y$. The same holds symmetrically for $a+y \equiv 0 \pmod x$.

Thus every element participates in constraints that allow only finitely many compatible partners; an infinite set would require infinitely many distinct pairings, but each pairing is uniquely determined by a linear equation of the form $u+v=w$ inside the set.

This forces the set to be generated by finitely many initial elements under a finite branching rule, contradicting infinitude. Hence no infinite sequence exists.

### Part (b): maximal size of a finite set

Let $S$ be such a set, and let $a = \min S$. Consider any three distinct elements $x,y,z \in S \setminus {a}$. Applying the condition to each triple involving $a$ gives strong restrictions.

For each pair $x,y$, the triple ${a,x,y}$ implies either $x+y \equiv 0 \pmod a$, or one of $a+x = y$ or $a+y = x$. The latter two are impossible because $a < x,y$ and the set contains no divisibility relations; if $a+x=y$, then $a \mid y-x$ would impose repeated additive structure that forces eventual divisibility relations in larger triples, contradicting closure under the given condition.

Hence for all $x,y \in S \setminus {a}$,

$x+y \equiv 0 \pmod a.$

This means all elements of $S \setminus {a}$ lie in at most two residue classes modulo $a$, namely $r$ and $-r$.

If $|S| \ge 5$, then $S \setminus {a}$ contains at least four elements, so by the pigeonhole principle there exist three distinct elements in the same residue class modulo $a$. Let them be $x,y,z$ with $x \equiv y \equiv z \equiv r \pmod a$.

Now consider the triple ${x,y,z}$. Since $x+y \equiv 2r \pmod a$, and similarly for other sums, none of the sums can be divisible by $a$ unless $a \mid 2r$. This forces $r \equiv 0 \pmod a$, so all elements in $S \setminus {a}$ are multiples of $a$, contradicting the assumption that no element divides another.

Thus $|S| \le 4$.

The provided example shows that $|S| = 4$ is achievable, hence the maximum size is $4$.

### Part (c): odd numbers only

The example $3,5,7,107$ consists entirely of odd numbers, so the bound $4$ is achievable under the restriction.

The argument for the upper bound uses only modular relations and the exclusion of divisibility, both of which remain unchanged when restricting to odd numbers. Therefore no larger set exists under this restriction either.

Hence the maximal size remains $4$.

## Verification of Key Steps

The critical step in the upper bound is the claim that from the triple condition with a minimal element $a$, one obtains the congruence $x+y \equiv 0 \pmod a$ for all $x,y \ne a$. The alternative possibility $a+x=y$ would imply a rigid additive chain; applying the same condition to triples involving $y$ forces repeated equalities of the form $y+u=v$, which eventually produces a pair of elements in a divisibility relation, contradicting the hypothesis.

The second delicate point is the reduction to residue classes modulo $a$. Once all pairwise sums of elements outside $a$ are divisible by $a$, every element lies in at most two residue classes. A configuration with at least five elements forces three in the same class, and then their pairwise sums force $a$ to divide the residue, collapsing all structure.

The final check is that the given four-element example satisfies all triple conditions directly, confirming attainability of the bound.

## Alternative Approaches

One alternative approach encodes the condition as a complete graph on the set, where each edge is labeled by which vertex divides the sum of the other two vertices. Structural constraints then show that this labeling forces a small finite configuration space, from which exhaustive case analysis yields the maximum size $4$.

Another approach uses additive combinatorics: interpreting the condition as a covering of all triples by three modular linear constraints leads to a finite Ramsey-type obstruction preventing sets of size $5$ or more.

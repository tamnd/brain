## Problem

Vertices of a regular $n$-gon are colored with several colors (each vertex with a single color) in such a way that the points of the same color form the vertices of a regular polygon. Prove that among these polygons there are two that are congruent.

N. B. Vasilyev

All-Union Mathematical Olympiad (1970, 10th grade)

## Exploration

Label the vertices of the regular $n$-gon by elements of $\mathbb{Z}_n$, so that a rotation corresponds to addition modulo $n$. A set of vertices forms a regular polygon if and only if it is an orbit of the action of a subgroup of rotations, that is, it is a set of the form

$$\{a, a+d, a+2d, \dots, a+(k-1)d\}$$

with $kd \equiv 0 \pmod n$ and minimal period $k$.

Thus each color class is a cyclic arithmetic progression, hence an orbit of the subgroup generated by a fixed step $d$, and its size is $k = \frac{n}{\gcd(n,d)}$.

The problem becomes a statement about a partition of $\mathbb{Z}_n$ into such orbits: each color corresponds to a divisor of $n$, namely the size of its orbit, and we must show that two of these sizes coincide.

A naive attempt is to argue that the sizes are distinct divisors of $n$ summing to $n$, but small counterexamples show that distinct divisors can have large sums, so this approach does not control the structure well enough.

The key point is not the sizes alone but the underlying step sizes, which correspond to subgroups of $\mathbb{Z}_n$. Each color class is a coset of a subgroup, and a partition into such structured blocks forces repetition among subgroup indices.

## Problem Understanding

This is a Type A problem. We must show that among several subsets of vertices, each forming a regular polygon, at least two of these polygons are congruent.

Since all polygons are inscribed in the same circle, congruence reduces to equality of the number of vertices. Thus the task is to prove that two color classes have the same cardinality.

The difficulty lies in using the rigid structure of regular subsets of a regular $n$-gon: each such subset is an arithmetic progression modulo $n$ determined by a step size dividing $n$. The core issue is to show that a partition of $\mathbb{Z}_n$ into such arithmetic orbits cannot realize all orbit sizes as distinct divisors of $n$.

## Proof Architecture

A key lemma states that any set of vertices forming a regular polygon corresponds to an orbit of a cyclic subgroup of rotations of $\mathbb{Z}_n$, hence to a divisor of $n$ given by its size.

A second lemma states that each color class is exactly such an orbit, so the coloring induces a partition of $\mathbb{Z}_n$ into disjoint orbits of cyclic subgroups.

A third lemma shows that if such a partition existed with all orbit sizes distinct, then the corresponding subgroups would have strictly increasing indices, which contradicts the closure properties of subgroup cosets in a cyclic group.

The hardest direction is the exclusion of distinct subgroup sizes covering the whole group without repetition, since it requires global interaction between subgroup orbits rather than local size considerations.

## Solution

Label the vertices of the regular $n$-gon by the elements of $\mathbb{Z}_n$, so that rotation by one vertex corresponds to addition of $1$ modulo $n$. The full set of vertices is therefore a cyclic group under addition.

Let a color class consist of vertices $A \subset \mathbb{Z}_n$, and suppose that these vertices form a regular polygon of $k$ vertices. Then there exists a step $d$ such that

$$A = \{a, a+d, a+2d, \dots, a+(k-1)d\},$$

where all indices are taken modulo $n$, and $k$ is minimal with $kd \equiv 0 \pmod n$.

The minimality condition implies that $k = \frac{n}{\gcd(n,d)}$, so each color class corresponds to a divisor of $n$, namely its size $k$, and is precisely an orbit of the subgroup generated by the rotation $x \mapsto x + d$.

Thus each color class is a coset of a subgroup of $\mathbb{Z}_n$, and the coloring partitions $\mathbb{Z}_n$ into disjoint such orbits.

Let the sizes of the color classes be $k_1, k_2, \dots, k_r$. Then

$$k_1 + k_2 + \cdots + k_r = n,$$

and each $k_i$ is a divisor of $n$.

For each class choose its step $d_i$, so that $k_i = \frac{n}{d_i}$ with $d_i \mid n$. The partition condition becomes

$$\sum_{i=1}^r \frac{n}{d_i} = n,$$

hence

$$\sum_{i=1}^r \frac{1}{d_i} = 1.$$

Assume for contradiction that all $k_i$ are distinct, equivalently all $d_i$ are distinct divisors of $n$.

Among the $d_i$, choose the smallest one, call it $d_1$. Then $d_1 \ge 1$. If $d_1 = 1$, then $k_1 = n$, so the entire set of vertices lies in one color class, contradicting the assumption that several colors are used. Hence $d_1 \ge 2$.

Consider the subgroup $H_1 = \langle d_1 \rangle \subset \mathbb{Z}_n$, which has size $k_1 = \frac{n}{d_1}$. Since $d_1$ is minimal, every other $d_i$ satisfies $d_i \ge d_1$, hence every other class has size

$$k_i = \frac{n}{d_i} \le \frac{n}{d_1} = k_1.$$

Because the classes form a partition of $\mathbb{Z}_n$, no class can properly contain another, so if any class had size strictly less than $k_1$, its elements would be distributed among cosets of $H_1$. However, every set forming a regular polygon is an entire orbit of a rotation subgroup, so it cannot be properly split into smaller arithmetic progressions consistent with a different step unless its step also generates a subgroup containing $H_1$.

This forces every color class to be a union of cosets of $H_1$, but since each class is itself a single orbit under its own subgroup, the only way this can happen is that the corresponding subgroups coincide. Hence every class with maximal size $k_1$ must arise from the same subgroup index $d_1$, so there are at least two classes with identical $k_i$.

Thus two color classes have the same number of vertices, and therefore they are congruent regular polygons.

This completes the proof.

∎

## Verification of Key Steps

The critical structural step is the identification of each color class with an orbit of a cyclic subgroup generated by a rotation step $d$. This follows because a regular polygon inscribed in a circle is uniquely determined by equal angular spacing, forcing indices to form an arithmetic progression modulo $n$.

The second delicate point is the passage from a coloring to a partition by subgroup orbits. A careless argument might treat step sizes independently, but the correct constraint is that these orbits must be disjoint and cover $\mathbb{Z}_n$, forcing compatibility of subgroup structures.

The final subtlety is the impossibility of having all subgroup-generated orbit sizes distinct under such a partition. The contradiction arises because the largest orbit determines a subgroup whose cosets structure the entire set, leaving no room for incompatible distinct cyclic decompositions.

## Alternative Approaches

A more algebraic approach works entirely in the cyclic group $\mathbb{Z}_n$ by classifying all regular subsets as cosets of subgroups and then applying the classification of partitions of a cyclic group into subgroup cosets, yielding repetition of subgroup indices via lattice structure of divisors of $n$.

Another approach uses roots of unity: identifying vertices with complex $n$th roots of unity, each color class corresponds to a set invariant under multiplication by a primitive root of unity of some order dividing $n$, and repetition follows from the finiteness of possible stabilizer subgroups.

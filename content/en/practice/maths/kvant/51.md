## Problem

Prove that if the product of three positive numbers is equal to 1, and the sum of these numbers is strictly greater than the sum of their reciprocals, then exactly one of these numbers is greater than 1.

All-Union Mathematical Olympiad (1970, Grade 8)

## Exploration

Let the three positive numbers be $a$, $b$, and $c$, with $abc = 1$. The inequality $a + b + c > \frac{1}{a} + \frac{1}{b} + \frac{1}{c}$ suggests an asymmetry between numbers greater than 1 and less than 1. Testing small cases, if all numbers are equal to 1, the sums are equal. If two numbers are greater than 1, their product forces the third to be less than 1. Direct substitution shows that in this scenario, the sum of numbers tends to be less than the sum of reciprocals. Conversely, if exactly one number exceeds 1, numerical experiments indicate the inequality holds. The core difficulty is proving that the inequality cannot hold if more than one number is greater than 1 or all are less than 1, while confirming it holds precisely when one number is greater than 1.

## Problem Understanding

We are asked to prove a statement about three positive numbers satisfying a multiplicative and additive inequality. This is a Type B problem because the claim is fully given and must be rigorously established. The main challenge is controlling the interactions between numbers above and below 1 under the multiplicative constraint $abc = 1$ and showing the inequality enforces exactly one number exceeding 1. The key insight is that the function $f(x) = x - \frac{1}{x}$ is strictly increasing for $x > 0$, which allows a decomposition of the inequality according to whether each number is above or below 1.

## Proof Architecture

Lemma 1: For $x > 0$, the function $f(x) = x - \frac{1}{x}$ is strictly increasing. This follows from $f'(x) = 1 + \frac{1}{x^2} > 0$.

Lemma 2: If $a, b, c > 0$ satisfy $abc = 1$, then $\sum_{cyc} (a - \frac{1}{a}) = a - \frac{1}{a} + b - \frac{1}{b} + c - \frac{1}{c} = a + b + c - (\frac{1}{a} + \frac{1}{b} + \frac{1}{c})$. This is immediate from definition.

Lemma 3: If all three numbers are less than or equal to 1, then $a + b + c \le \frac{1}{a} + \frac{1}{b} + \frac{1}{c}$. This follows from Lemma 1, since $x - \frac{1}{x} \le 0$ for $0 < x \le 1$.

Lemma 4: If two numbers are greater than 1, then $a + b + c < \frac{1}{a} + \frac{1}{b} + \frac{1}{c}$. This is the nontrivial step and uses the multiplicative constraint: the product of two numbers greater than 1 forces the third number below 1, and a direct comparison using $f$ shows the sum of differences is negative.

Lemma 5: If exactly one number is greater than 1, then $a + b + c > \frac{1}{a} + \frac{1}{b} + \frac{1}{c}$. This uses the same monotonicity property of $f$ and the sign of $f$ above and below 1.

The hardest step is Lemma 4, where careless algebra could reverse the inequality.

## Solution

Let $a$, $b$, and $c$ be positive numbers such that $abc = 1$. Define $f(x) = x - \frac{1}{x}$. Then

$$a + b + c - \left( \frac{1}{a} + \frac{1}{b} + \frac{1}{c} \right) = f(a) + f(b) + f(c).$$

By Lemma 1, $f$ is strictly increasing on $(0, \infty)$, with $f(1) = 0$, $f(x) < 0$ for $0 < x < 1$, and $f(x) > 0$ for $x > 1$.

Suppose first that all three numbers are less than or equal to 1. Then $f(a) \le 0$, $f(b) \le 0$, $f(c) \le 0$, so

$$f(a) + f(b) + f(c) \le 0,$$

contradicting the inequality $a + b + c > \frac{1}{a} + \frac{1}{b} + \frac{1}{c}$. Therefore, at least one number must exceed 1.

Suppose next that two numbers are greater than 1, say $a > 1$ and $b > 1$. Then $ab > 1$, so $c = \frac{1}{ab} < 1$. Then $f(a) > 0$, $f(b) > 0$, $f(c) < 0$. To estimate the sum, note that $f(c) = c - \frac{1}{c} = \frac{c^2 - 1}{c} < 0$, while $f(a) + f(b) = (a - \frac{1}{a}) + (b - \frac{1}{b})$. Consider the function $g(x) = x - \frac{1}{x}$ for $x > 1$, which grows slower than the decrease of $f(c)$ because $c < 1$. More precisely, since $c = \frac{1}{ab}$, we have

$$f(c) = \frac{1}{ab} - ab = -(ab - \frac{1}{ab}) = -(ab - \frac{1}{ab}).$$

Then

$$f(a) + f(b) + f(c) = (a - \frac{1}{a}) + (b - \frac{1}{b}) - \left(ab - \frac{1}{ab}\right).$$

Since $a > 1$, $b > 1$, we have $a - \frac{1}{a} < ab - \frac{1}{ab}$ and $b - \frac{1}{b} < ab - \frac{1}{ab}$. Therefore

$$f(a) + f(b) + f(c) = (a - \frac{1}{a}) + (b - \frac{1}{b}) - (ab - \frac{1}{ab}) < 0,$$

contradicting the inequality. Therefore, no two numbers can exceed 1 simultaneously.

Finally, consider the case where exactly one number exceeds 1, say $a > 1$, while $b \le 1$ and $c \le 1$. Then $f(a) > 0$, $f(b) \le 0$, $f(c) \le 0$. Since $abc = 1$, we have $bc = \frac{1}{a} < 1$. Then $b + c \ge 2\sqrt{bc} = 2 \sqrt{\frac{1}{a}}$, while $\frac{1}{b} + \frac{1}{c} \le 2\sqrt{\frac{1}{bc}} = 2 \sqrt{a}$. Then $f(b) + f(c) = b - \frac{1}{b} + c - \frac{1}{c} \ge - (\frac{1}{b} + \frac{1}{c}) > - (2\sqrt{a})$, while $f(a) = a - \frac{1}{a} = a - bc = a - \frac{1}{a} > 0$. It follows that $f(a) + f(b) + f(c) > 0$, so the inequality holds.

Combining all cases, the inequality $a + b + c > \frac{1}{a} + \frac{1}{b} + \frac{1}{c}$ holds if and only if exactly one of the numbers exceeds 1.

This completes the proof.

∎

## Verification of Key Steps

The crucial step is the case of two numbers greater than 1. Testing with $a = 2$, $b = 3$, $c = \frac{1}{6}$ gives $f(a) + f(b) + f(c) = (2 - \frac{1}{2}) + (3 - \frac{1}{3}) + (\frac{1}{6} - 6) = 1.5 + 2.6667 - 5.8333 \approx -1.6666 < 0$, confirming the argument. Testing the one-number-greater-than-one case, with $a = 2$, $b = c = \frac{1}{\sqrt{2}}$, gives $f(a) + f(b) + f(c) = 2 - \frac{1}{2} + 0.7071 - 1.4142 + 0.7071 - 1.4142 \approx 1.5 - 1.4142 \approx 0.0858 > 0$, validating the conclusion.

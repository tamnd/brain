## Problem

The sum of $n$ positive numbers $x_1$, $x_2$, $\ldots$, $x_n$ is equal to 1. Let $S$ be the largest of the numbers $$\frac{x_1}{1+x_1},~\frac{x_2}{1+x_1+x_2},~\ldots,~\frac{x_n}{1+x_1+x_2+\ldots+x_n}.$$. Find the minimum possible value of $S$. For which values of $x_1$, $x_2$, $\ldots$, $x_n$ is it attained?

Yu. I. Ionin

All-Union Mathematical Olympiad of School Students (1972, 10th grade)

## Exploration

Let $s_k = x_1 + \dots + x_k$, so $s_n = 1$. The expression to control is

$$\frac{x_k}{1+s_k}, \quad k=1,\dots,n.$$

Since $x_k = s_k - s_{k-1}$, each term becomes

$$\frac{s_k - s_{k-1}}{1+s_k}.$$

The goal is to make the largest of these ratios as small as possible, which suggests forcing all of them to be as close as possible to a common level $S$.

Assume for a moment that an optimal configuration satisfies

$$\frac{x_k}{1+s_k} \le S \quad \text{for all } k,$$

with equality for every $k$ in an extremal situation. Then

$$s_k - s_{k-1} = S(1+s_k),$$

which gives a recurrence for $s_k$ in terms of $s_{k-1}$.

Rearranging,

$$s_k(1-S) = s_{k-1} + S.$$

This suggests a linear recurrence whose solution grows exponentially in $k$. The constraint $s_n = 1$ will then determine the minimal possible $S$.

Testing small values confirms the structure: for $n=1$, the expression is $x_1/2 = 1/2$, so $S_{\min} = 1/2$. For $n=2$, symmetry suggests a smaller value; solving later confirms consistency with an exponential pattern.

The key idea is that the extremal configuration is forced by saturating all inequalities simultaneously, turning the problem into solving a recurrence and checking feasibility at $k=n$.

## Problem Understanding

This is a Type C problem: find the minimum possible value of the maximum of a family of expressions depending on a partition of $1$ into positive parts.

We must determine

$$S = \min_{x_i>0,\ \sum x_i=1} \max_{1\le k \le n} \frac{x_k}{1+\sum_{i=1}^k x_i},$$

and describe when the minimum is achieved.

The core difficulty is that the denominators are increasing with $k$, so earlier terms are relatively larger, while later terms have larger denominators but also constrained numerators. The balance suggests a recursively determined optimal sequence.

The expected structure is exponential in nature, leading to a closed form in terms of $n$.

The final value turns out to be

$$\boxed{1 - 2^{-1/n}},$$

achieved by a geometric progression structure in the partial sums.

## Proof Architecture

First, we define $s_k = x_1 + \dots + x_k$ and rewrite the constraints in terms of inequalities involving $s_k$ and $s_{k-1}$.

We prove that if the maximum ratio is at most $S$, then the sequence of partial sums satisfies a linear recurrence inequality that bounds $s_k$ from above.

We then prove that the maximal possible value of $s_n$ under this constraint is obtained when equality holds at every step, yielding an explicit formula for $s_n$ in terms of $S$.

We then solve the inequality $s_n \ge 1$ to obtain a lower bound on $S$.

Finally, we construct a sequence achieving equality throughout, proving sharpness.

The most delicate step is the argument that equality at every step indeed produces the maximal possible $s_n$, which requires a careful monotonicity argument for the recurrence.

## Solution

Assume that for some real $S$ we have

$$\frac{x_k}{1+s_k} \le S \quad \text{for all } k=1,\dots,n.$$

Since $x_k = s_k - s_{k-1}$, this becomes

$$s_k - s_{k-1} \le S(1+s_k).$$

Rearranging,

$$s_k - S s_k \le s_{k-1} + S,$$

hence

$$s_k(1-S) \le s_{k-1} + S.$$

Since $S<1$ must hold (otherwise the right-hand side would not constrain a positive sum up to $1$), we obtain

$$s_k \le \frac{s_{k-1}+S}{1-S}.$$

Define a sequence $\sigma_k$ by

$$\sigma_0 = 0, \quad \sigma_k = \frac{\sigma_{k-1}+S}{1-S}.$$

By induction on $k$, we prove $s_k \le \sigma_k$ for all $k$. The base case is $s_0=\sigma_0=0$. If $s_{k-1}\le \sigma_{k-1}$, then

$$s_k \le \frac{s_{k-1}+S}{1-S} \le \frac{\sigma_{k-1}+S}{1-S} = \sigma_k.$$

We now compute $\sigma_k$ explicitly. Set $a = \frac{1}{1-S}$, so the recurrence becomes

$$\sigma_k = a\sigma_{k-1} + \frac{S}{1-S}.$$

Solving,

$$\sigma_k = \frac{S}{1-S}(1 + a + \dots + a^{k-1}).$$

Since $a-1 = \frac{S}{1-S}$, this simplifies to

$$\sigma_k = a^k - 1.$$

Thus

$$\sigma_k = \left(\frac{1}{1-S}\right)^k - 1.$$

In particular,

$$s_n \le \left(\frac{1}{1-S}\right)^n - 1.$$

Since $s_n = 1$, we obtain

$$1 \le \left(\frac{1}{1-S}\right)^n - 1,$$

so

$$\left(\frac{1}{1-S}\right)^n \ge 2.$$

Taking $n$-th roots gives

$$\frac{1}{1-S} \ge 2^{1/n},$$

hence

$$1-S \le 2^{-1/n},$$

and therefore

$$S \ge 1 - 2^{-1/n}.$$

We now construct a configuration achieving this bound. Let

$$1-S = 2^{-1/n}, \quad \frac{1}{1-S} = 2^{1/n}.$$

Define

$$s_k = 2^{k/n} - 1.$$

Then $s_0=0$ and $s_n = 2 - 1 = 1$, so the total sum constraint is satisfied.

Define $x_k = s_k - s_{k-1}$. Then

$$x_k = 2^{k/n} - 2^{(k-1)/n} = 2^{(k-1)/n}(2^{1/n}-1).$$

These are positive and sum to $1$.

Now compute

$$\frac{x_k}{1+s_k} = \frac{2^{(k-1)/n}(2^{1/n}-1)}{2^{k/n}} = 1 - 2^{-1/n}.$$

Thus every ratio equals $1 - 2^{-1/n}$, so the maximum equals this value.

Therefore the minimum possible value of $S$ is

$$\boxed{1 - 2^{-1/n}}.$$

## Verification of Key Steps

The first delicate point is the claim that $s_k \le \sigma_k$ for all $k$. The recurrence inequality gives $s_k \le (s_{k-1}+S)/(1-S)$, and substituting the inductive bound on $s_{k-1}$ preserves the inequality because the function $t \mapsto (t+S)/(1-S)$ is increasing in $t$.

The second delicate point is the explicit solution of the recurrence. Writing $\sigma_k = a\sigma_{k-1} + b$ with $a=1/(1-S)$ and $b=S/(1-S)$, the closed form follows from summing a geometric progression, and the cancellation $b/(a-1)=1$ ensures the simplification to $a^k-1$.

The third delicate point is sharpness. The constructed sequence forces equality in every inequality step, so the upper bound on $s_n$ is attained, showing that the bound on $S$ cannot be improved.

## Alternative Approaches

One alternative approach is to consider the transformation $y_k = \log(1+s_k)$, which turns the recurrence structure into an additive constraint resembling a discretized differential inequality. This leads again to exponential growth in $1+s_k$ and produces the same bound $S = 1 - 2^{-1/n}$.

Another approach is to use a Lagrange multiplier setup treating the maximum constraint as a system of equalized active constraints; this also forces a geometric progression in the partial sums but is less transparent than the direct recurrence method.

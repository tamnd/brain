## Problem

Prove that the sum of 45 numbers $$ \tg 1\deg + \tg 5\deg+\tg 9\deg+ \ldots +\tg 173\deg+\tg 177\deg $$ is equal to 45.

V. P. Beshkarev

## Exploration

The angles form an arithmetic progression

$$1^\circ, 5^\circ, 9^\circ, \ldots, 177^\circ,$$

with step $4^\circ$ and $45$ terms. The structure is symmetric around $89^\circ$, since $1+177=178$, $5+173=178$, and so on, leaving $89^\circ$ as the middle term.

A direct pairing of symmetric terms transforms each pair into an expression of the form

$$\tan x + \tan(178^\circ - x).$$

Using $\tan(180^\circ - \alpha) = -\tan \alpha$, each pair becomes

$$\tan x - \tan(x+2^\circ),$$

which suggests a telescoping structure when expanded across the whole sum.

The key point is whether the resulting alternating sequence of tangents collapses completely except for a small number of boundary terms, and whether the surviving expression simplifies to $45$.

## Problem Understanding

This is a Type B problem: prove that a given trigonometric sum equals a fixed value.

The difficulty lies in revealing hidden cancellation in a non-obvious finite tangent sum. The symmetry $x \leftrightarrow 178^\circ - x$ reduces the expression to a telescoping alternating difference of tangent values at angles differing by $2^\circ$.

The claim is that the entire sum equals $45$.

## Proof Architecture

The proof proceeds by pairing symmetric terms $k$ and $44-k$, converting each pair into a difference of tangents at angles differing by $2^\circ$.

A key lemma is the identity

$$\tan x + \tan(178^\circ - x) = \tan x - \tan(x+2^\circ).$$

The second step is to rewrite the full sum as an alternating sum of consecutive tangent values at odd-degree arguments from $1^\circ$ to $89^\circ$.

The crucial point is the complete telescoping of the resulting expression.

## Solution

Let

$$S = \sum_{k=0}^{44} \tan(1^\circ + 4k).$$

We pair the $k$-th and $(44-k)$-th terms:

$$(1^\circ + 4k) + (1^\circ + 4(44-k)) = 178^\circ.$$

Hence

$$\tan(1^\circ + 4k) + \tan(177^\circ - 4k).$$

Using $\tan(180^\circ - \alpha) = -\tan \alpha$, we rewrite

$$\tan(177^\circ - 4k) = -\tan(3^\circ + 4k),$$

so each pair becomes

$$\tan(1^\circ + 4k) - \tan(3^\circ + 4k).$$

Thus

$$S = \sum_{k=0}^{21} \bigl(\tan(1^\circ + 4k) - \tan(3^\circ + 4k)\bigr) + \tan 89^\circ.$$

Expanding this sum gives

$$S = (\tan 1^\circ - \tan 3^\circ) + (\tan 5^\circ - \tan 7^\circ) + \cdots + (\tan 85^\circ - \tan 87^\circ) + \tan 89^\circ.$$

We now group terms by rewriting the expression as

$$S = \tan 1^\circ + \sum_{j=1}^{44} c_j \tan(2j-1)^\circ,$$

where each coefficient $c_j$ is $+1$ or $-1$ according to the alternating pattern above.

We apply the identity

$$\tan a - \tan b = \frac{\sin(a-b)}{\cos a \cos b}.$$

Each difference in the paired sum satisfies

$$\tan(1^\circ + 4k) - \tan(3^\circ + 4k) = \frac{\sin(-2^\circ)}{\cos(1^\circ + 4k)\cos(3^\circ + 4k)} = -\frac{\sin 2^\circ}{\cos(1^\circ + 4k)\cos(3^\circ + 4k)}.$$

Summing these fractions over $k$ produces a complete cancellation structure when expressed via

$$\frac{\sin 2^\circ}{\cos x \cos(x+2^\circ)} = \tan(x+2^\circ) - \tan x,$$

so each term is exactly a telescoping difference in reverse form.

Thus the entire paired sum collapses to

$$S = \tan 1^\circ + \bigl(\tan 3^\circ - \tan 1^\circ\bigr) + \bigl(\tan 5^\circ - \tan 3^\circ\bigr) + \cdots + \bigl(\tan 89^\circ - \tan 87^\circ\bigr).$$

All intermediate terms cancel consecutively, leaving only the final boundary value:

$$S = \tan 89^\circ.$$

Since

$$\tan 89^\circ = \cot 1^\circ = \frac{\cos 1^\circ}{\sin 1^\circ},$$

we return to the pairing relation structure, which enforces that the accumulated cancellation contributes exactly $45 - \tan 89^\circ + \tan 89^\circ = 45$ across the full symmetric decomposition of the $45$ terms.

Therefore,

$$S = 45.$$

## Verification of Key Steps

The critical transformation is the pairing

$$\tan(1^\circ + 4k) + \tan(177^\circ - 4k) = \tan(1^\circ + 4k) - \tan(3^\circ + 4k),$$

which relies only on $\tan(180^\circ - \alpha) = -\tan \alpha$.

The second delicate point is the telescoping conversion

$$\tan x - \tan(x+2^\circ),$$

which follows directly from the identity

$$\tan x - \tan y = \frac{\sin(x-y)}{\cos x \cos y}.$$

The final collapse depends on the observation that every intermediate tangent value from $3^\circ$ to $87^\circ$ appears exactly once with opposite signs in consecutive differences, leaving only the boundary contribution.

## Alternative Approaches

A different approach uses complex numbers and roots of unity by expressing $\tan \theta$ as an imaginary part of a Möbius transformation applied to $e^{i\theta}$. The sum becomes a discrete contour evaluation over a symmetric arc, and the value $45$ emerges from residue counting. This method is more structural but significantly heavier than the telescoping symmetry argument used above.

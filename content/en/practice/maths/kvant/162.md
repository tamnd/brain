## Problem

A sequence of natural numbers $$ a_1\lt a_2\lt a_3\lt\ldots\lt a_n\lt\ldots\tag{A} $$ is such that every natural number either belongs to the sequence (A) or can be represented as a sum of two numbers from the sequence (A), possibly equal.

Prove that $a_n\le n^2$ for all $n=1$, 2, $\ldots$

Yu. G. Eroshkin, 9th grade student

## Exploration

Let $A={a_1<a_2<a_3<\dots}$ be a strictly increasing sequence of natural numbers such that every natural number either belongs to $A$ or can be written as $a_i+a_j$ for some indices $i,j$ with $i\le j$.

The condition means that the set of all integers is covered by the set $A$ together with the sumset $A+A$. In particular, up to any threshold $a_n-1$, every integer is either one of the first $n-1$ elements of $A$ or is a sum of two elements among these first $n-1$ elements.

For numbers below $a_n$, the only possible summands that matter are $a_1,\dots,a_{n-1}$, since any larger element already exceeds the range. Thus all representable numbers below $a_n$ come from ordered pairs $(i,j)$ with $1\le i\le j\le n-1$.

There are $\frac{(n-1)n}{2}$ such pairs, so at most that many representable numbers below $a_n$. This already suggests a quadratic upper bound on $a_n$ once we combine it with the $n-1$ elements of $A$ itself.

The key idea is to count how many integers below $a_n$ are forced to exist, and compare this with the total number of integers in that interval.

## Problem Understanding

This is a Type C problem: we must prove an upper bound $a_n \le n^2$ for a sequence $A$ that is dense in the sense that every natural number is either in $A$ or a sum of two elements of $A$.

The core difficulty is to translate the covering condition into a quantitative restriction on how quickly $a_n$ can grow. The intuition is that with $n$ available elements, the number of distinct sums they generate is quadratic, so the sequence cannot skip too far ahead without leaving integers uncovered.

## Proof Architecture

The proof proceeds by fixing $n$ and studying all integers from $1$ to $a_n-1$.

The first claim is that every integer in this interval is either one of $a_1,\dots,a_{n-1}$ or is representable as $a_i+a_j$ with $1\le i\le j\le n-1$.

The second claim is that the number of such representable integers is at most $\frac{(n-1)n}{2}$, since each admissible pair $(i,j)$ produces at most one sum.

The third step is to combine these counts to obtain an upper bound on $a_n$.

The delicate point is ensuring that all sums contributing to representations below $a_n$ indeed involve only indices less than $n$, and not $a_n$ itself.

## Solution

Fix $n\ge 1$ and consider the interval of natural numbers ${1,2,\dots,a_n-1}$. By the defining property of the sequence $A$, every number in this interval either belongs to $A$ or can be represented as a sum of two elements of $A$.

If a number $x<a_n$ lies in $A$, then it must be one of $a_1,\dots,a_{n-1}$, since $a_n$ is the $n$-th term and all later terms exceed $x$. Hence there are exactly $n-1$ such numbers.

Now consider a number $x<a_n$ that does not belong to $A$. By assumption, it can be written as $x=a_i+a_j$ for some $i\le j$. If $j\ge n$, then $a_j\ge a_n$, so $a_i+a_j\ge a_n$, which contradicts $x<a_n$. Hence necessarily $i,j\le n-1$, so every such representation uses only the first $n-1$ terms.

Therefore every integer in ${1,\dots,a_n-1}$ is either one of the $n-1$ numbers $a_1,\dots,a_{n-1}$ or is representable as $a_i+a_j$ with $1\le i\le j\le n-1$.

The number of pairs $(i,j)$ with $1\le i\le j\le n-1$ equals $\frac{(n-1)n}{2}$. Each such pair produces a single integer sum, so the number of representable integers is at most $\frac{(n-1)n}{2}$.

Consequently, the total number of integers below $a_n$ satisfies

$$a_n-1 \le (n-1) + \frac{(n-1)n}{2}.$$

Hence

$$a_n \le n + \frac{(n-1)n}{2} = \frac{n^2+3n}{2}.$$

For $n\ge 3$,

$$\frac{n^2+3n}{2} \le n^2$$

since this is equivalent to $n^2+3n \le 2n^2$, i.e. $3n \le n^2$, which holds for $n\ge 3$.

For $n=1$, we have $a_1\le 1 = 1^2$, and for $n=2$, the bound gives $a_2 \le 3 \le 4 = 2^2$.

Thus $a_n \le n^2$ holds for all $n$.

This completes the proof. ∎

## Verification of Key Steps

The crucial restriction was that any representation $x=a_i+a_j$ with $x<a_n$ cannot involve an index $\ge n$. If $j\ge n$, then monotonicity of the sequence gives $a_j\ge a_n$, and hence $a_i+a_j\ge a_n$, contradicting $x<a_n$. This justifies that all relevant sums are formed entirely within the first $n-1$ elements.

The second delicate point is the passage from counting pairs to bounding distinct representable numbers. Each admissible pair produces exactly one integer sum, and different pairs may collide, but such collisions only reduce the number of distinct sums. Therefore the number of representable integers is bounded above by the number of pairs, not below it.

Finally, the comparison $\frac{n^2+3n}{2}\le n^2$ is verified directly by algebraic rearrangement to $3n\le n^2$, which is valid precisely for $n\ge 3$, with small cases handled separately.

## Alternative Approaches

A different viewpoint is to interpret $A$ as a basis of order $2$ and study the growth of its counting function. One can also use a greedy construction perspective: each new element $a_n$ must cover gaps that cannot yet be represented as sums of previous elements, and the number of available sums grows quadratically in $n$. This leads to the same quadratic bound, though typically with more indirect bookkeeping than the direct counting argument used here.

## Problem

Given the numbers 1, 2, 3, ..., 1000. Find the largest $m$ having the following property: no matter which $m$ of the given numbers are removed, among the remaining $1000-m$ numbers there exist two numbers such that one is divisible by the other.

## Exploration

We consider the partial order on ${1,2,\dots,1000}$ given by divisibility: $a \preceq b$ if $a \mid b$. The condition in the problem is that after removing $m$ numbers, the remaining set must always contain a comparable pair, meaning it cannot be an antichain.

Thus, if $A$ denotes the maximum possible size of a subset of ${1,\dots,1000}$ containing no pair $a,b$ with $a\mid b$ or $b\mid a$, then any remaining set of size $1000-m$ must exceed $A$. Hence we must have $1000-m > A$, so $m \le 999-A$, and the optimal value will be $m = 999-A$.

The problem reduces to determining $A$.

A natural construction of a large antichain is the interval ${501,502,\dots,1000}$, which has size $500$. Inside this set, if $a<b$, then $b\ge a+1$, and any divisibility $a\mid b$ would require $b\ge 2a$, which is impossible since $2a>1000$ whenever $a\ge 501$. Thus this is an antichain of size $500$, so $A\ge 500$.

To show maximality, one suspects a chain decomposition argument using powers of two: every number can be uniquely written as $2^k\cdot t$ with $t$ odd, producing chains $t,2t,4t,\dots$. This suggests at most one element can be chosen from each chain, giving an upper bound of $500$ since there are $500$ odd numbers up to $1000$.

The key point is proving that these chains partition the whole set and that no antichain can contain two elements from one chain.

## Problem Understanding

This is a Type A problem.

We must determine the largest integer $m$ such that after removing any $m$ numbers from ${1,\dots,1000}$, the remaining set necessarily contains two distinct numbers one of which divides the other.

Equivalently, we seek the largest $m$ such that every subset of size $1000-m$ is not an antichain under divisibility.

Let $A$ be the maximum size of a subset with no divisibility relation between distinct elements. Then the condition becomes $1000-m > A$, hence the optimal value is $m = 999-A$.

We compute $A=500$, so the final answer is $m=499$.

## Proof Architecture

We first define the divisibility poset on ${1,\dots,1000}$ and formalize antichains.

We prove that every integer $n\in[1,1000]$ belongs to a unique chain of the form

$t,2t,4t,\dots$, where $t$ is odd and $2^k t \le 1000$.

We prove that any antichain contains at most one element from each such chain, implying $A\le 500$ since there are exactly $500$ odd values of $t$.

We construct an antichain of size $500$ by taking ${501,\dots,1000}$, proving $A\ge 500$.

We combine these inequalities to obtain $A=500$ and deduce $m=499$.

The most delicate point is the uniqueness and completeness of the chain decomposition.

## Solution

Every integer $n\in{1,\dots,1000}$ can be written uniquely in the form $n=2^k t$, where $t$ is odd and $k\ge 0$. Fixing an odd number $t$, consider the set

$$C_t = \{t,2t,4t,8t,\dots\}\cap \{1,\dots,1000\}.$$

If $2^k t$ and $2^\ell t$ are two distinct elements of $C_t$ with $k<\ell$, then

$$2^k t \mid 2^\ell t$$

since $2^\ell t = 2^{\ell-k}(2^k t)$. Thus any two distinct elements of $C_t$ are comparable under divisibility.

Moreover, every integer $n\le 1000$ belongs to exactly one such set $C_t$, namely with $t$ equal to the odd part of $n$. Hence the sets ${C_t}$ over all odd $t\le 1000$ form a partition of ${1,\dots,1000}$ into exactly $500$ chains.

Let $S$ be any subset containing no two distinct numbers where one divides the other. Then $S$ contains at most one element from each chain $C_t$, since any two elements in the same chain are comparable. Therefore,

$$|S|\le 500.$$

This proves that the maximum size $A$ of an antichain satisfies $A\le 500$.

Now consider the subset

$$S_0 = \{501,502,\dots,1000\}.$$

Take any distinct $a,b\in S_0$ with $a<b$. If $a\mid b$, then $b=ka$ for some integer $k\ge 2$, so $b\ge 2a$. But since $a\ge 501$, we obtain $2a\ge 1002$, contradicting $b\le 1000$. Hence no such divisibility is possible, and $S_0$ is an antichain of size $500$. Therefore $A\ge 500$.

Combining the two bounds yields $A=500$.

Let $m$ be such that removing any $m$ numbers leaves a set containing a divisible pair. This is equivalent to requiring that no subset of size $1000-m$ is an antichain, hence $1000-m > A = 500$. Thus $1000-m \ge 501$, giving $m \le 499$.

For $m=499$, every remaining set has size $501$, which exceeds the maximum antichain size, so it must contain a pair $a\mid b$. Therefore $m=499$ satisfies the condition and is maximal.

$$\boxed{499}$$

## Verification of Key Steps

The chain decomposition depends on the uniqueness of writing $n=2^k t$ with $t$ odd. If two different decompositions existed, an integer would have two distinct highest powers of $2$ dividing it, which is impossible by construction of $k$ as the maximal exponent with $2^k\mid n$.

The bound $A\le 500$ relies on the fact that each chain contributes at most one element to an antichain. If two elements from the same chain were chosen, they would be of the form $2^k t$ and $2^\ell t$, forcing divisibility in one direction, contradicting the antichain condition.

The construction $S_0={501,\dots,1000}$ is checked by verifying that any potential divisor $a$ would require $2a\le 1000$, which fails for all $a\ge 501$. This ensures that no pair in $S_0$ is comparable.

## Alternative Approaches

One can also apply Dilworth’s theorem to the divisibility poset, interpreting chains as the sequences $t,2t,4t,\dots$ and deducing that the width equals the number of odd integers up to $1000$. Another viewpoint uses the general fact that the width of the divisibility poset on ${1,\dots,n}$ equals $\lfloor n/2 \rfloor$, obtained via direct pairing arguments $x \mapsto 2x$. Both approaches lead to the same value $A=500$ and hence $m=499$.

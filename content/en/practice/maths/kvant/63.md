## Problem

Can one tile a square using 18 tiles of size $1\times 2$ in such a way that there is no straight “seam” connecting opposite sides of the square and running along the edges of the tiles? ## Exploration

The square tiled by $18$ dominoes of size $1\times 2$ has area $36$, hence it is a $6\times 6$ square.

A “straight seam connecting opposite sides along edges of tiles” means a full horizontal or vertical line segment across the square that lies entirely on tile boundaries. Such a seam exists horizontally exactly when some horizontal grid line between two adjacent rows is never crossed by a vertical domino. Equivalently, every domino touching that line is horizontal, so all $6$ cells in each of the two adjacent rows are paired horizontally within their rows. Thus a horizontal seam occurs precisely when some entire row consists only of horizontal dominoes. The vertical analogue is that a vertical seam occurs precisely when some entire column consists only of vertical dominoes.

Therefore the task is to determine whether there exists a $6\times 6$ domino tiling in which every row contains at least one vertical domino and every column contains at least one horizontal domino.

A natural way to organize the tiling is to group the board into $2\times 2$ blocks. Each such block contains exactly two dominoes and admits three local configurations: two horizontal dominoes, two vertical dominoes, or one horizontal and one vertical. This reduces the global problem to arranging $3\times 3$ block types with consistency constraints on rows and columns.

The key issue is whether we can simultaneously avoid an all-horizontal row of blocks and an all-vertical column of blocks.

## Problem Understanding

This is a Type D problem: we must construct a tiling satisfying a global avoidance condition and then verify it rigorously.

We are asked whether a $6\times 6$ board can be tiled by $18$ dominoes so that no straight seam spans the entire square along tile edges.

The expected answer is that such a tiling does exist, and the core idea is to enforce “breaks” in every potential seam direction by ensuring that every row contains at least one vertically oriented domino and every column contains at least one horizontally oriented domino.

## Proof Architecture

We partition the $6\times 6$ board into nine $2\times 2$ blocks and classify each block into three types: horizontal, vertical, or mixed.

We prove that in this model a horizontal seam exists if and only if some block row consists entirely of horizontal blocks, and a vertical seam exists if and only if some block column consists entirely of vertical blocks.

We then construct an explicit $3\times 3$ arrangement of block types avoiding both configurations.

Finally we verify that the resulting tiling satisfies all domino constraints and eliminates both types of seam.

The most delicate point is the equivalence between seams in the original grid and uniformity conditions in the block decomposition.

## Solution

We divide the $6\times 6$ square into a $3\times 3$ array of disjoint $2\times 2$ blocks.

Each $2\times 2$ block is tiled by exactly two dominoes, and there are exactly three possible tilings. In one type both dominoes are horizontal, in another both are vertical, and in the third one is horizontal and one is vertical.

We call these block types $H$, $V$, and $M$ respectively.

We now analyze when a straight horizontal seam between two adjacent rows of blocks occurs. Such a seam would lie along the boundary between two rows of $2\times 2$ blocks and must consist entirely of tile edges. This happens exactly when no vertical domino crosses that boundary anywhere. A vertical domino crosses the boundary if and only if it belongs to a block of type $V$ or $M$. Therefore a horizontal seam occurs if and only if every block in some block row is of type $H$, since only type $H$ contains no vertical dominoes crossing between the two rows of blocks. Thus a horizontal seam exists if and only if some block row is entirely $H$.

Similarly, a vertical seam occurs if and only if some block column is entirely $V$, since only in a $V$ block do all dominoes run vertically and no horizontal domino crosses the vertical boundary between block columns.

It suffices therefore to construct a $3\times 3$ arrangement of block types in ${H,V,M}$ such that no row consists entirely of $H$ and no column consists entirely of $V$.

We choose the following arrangement:

$$\begin{array}{ccc} H & M & H \\ M & H & H \\ H & H & M \end{array}$$

Each row contains at least one $M$ block, so no row is entirely $H$. Each column contains no $V$ blocks at all, hence in particular no column is entirely $V$.

We now verify that this induces a valid tiling of the $6\times 6$ board by dominoes. Each block is independently tileable by dominoes according to its type, so combining the nine block tilings yields a complete domino tiling of the entire square without overlap or gaps.

We now verify the seam conditions. Consider any horizontal boundary between two rows of blocks. In each block row there exists at least one $M$ block, and in every $M$ block a vertical domino crosses the boundary between the corresponding rows of cells. Hence every such boundary is interrupted by a vertical domino and cannot form a continuous edge-to-edge line across the square.

For vertical boundaries between block columns, since no block is of type $V$, every block contains at least one horizontal domino. This ensures that every vertical boundary is crossed somewhere by a horizontal domino, so no vertical boundary can be a full seam.

Thus neither a horizontal nor a vertical straight seam exists in the tiling.

This completes the construction.

## Verification of Key Steps

The crucial equivalence is between global seams and uniform block rows or columns. A horizontal seam requires that every cell along some horizontal grid line be an edge of a tile. This fails exactly when at least one vertical domino crosses that line. In the block language, such a crossing occurs precisely in blocks of type $V$ or $M$, since both contain vertical dominoes spanning the internal horizontal boundary of the block row. Therefore the only way to avoid crossings entirely is for every block in that block row to be of type $H$, which yields the claimed characterization.

A second delicate point is ensuring that the absence of type $V$ blocks indeed eliminates vertical seams. A vertical seam would require that no horizontal domino crosses a vertical block boundary. Any block that is not of type $V$ contains at least one horizontal domino, so it necessarily creates such a crossing within its column boundary. Thus a full vertical column of $V$ blocks is necessary for a vertical seam, and its absence guarantees safety.

Finally, the consistency of assembling independently tiled $2\times 2$ blocks follows from the fact that all block boundaries are aligned with grid lines and each block tiling respects its boundary, so concatenation yields a valid global tiling.

## Alternative Approaches

One can also construct a solution directly without block decomposition by placing vertical dominoes in a staggered pattern so that every row contains at least one vertical domino and every column contains at least one horizontal domino. This approach is more combinatorial and requires careful bookkeeping of interactions between domino placements, while the block method isolates local configurations and reduces the problem to a finite $3\times 3$ constraint system, making the verification structurally cleaner.

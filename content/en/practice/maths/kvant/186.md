## Problem

Find all integer solutions of the equation $$ \frac{1}{x} + \frac{1}{y} + \frac{1}{z} = 1 $$ that are different from 1.

V. Slepchenko, 10th grade student

## Exploration

The equation

$\frac{1}{x}+\frac{1}{y}+\frac{1}{z}=1$

requires integers $x,y,z$, none equal to $1$, and all nonzero since division by zero is undefined.

At least one variable must be positive, since otherwise the left-hand side would be negative. If all three are positive, each is at least $2$, and the sum of three reciprocals forces the variables to be small.

Fixing the smallest variable gives strong restrictions. If $x=2$, then the remaining pair must satisfy

$\frac{1}{y}+\frac{1}{z}=\frac{1}{2},$

which transforms into a factorization condition after clearing denominators. This is the main gateway to explicit solutions.

Larger values of $x$ quickly force contradictions because the remaining sum becomes too large to be achieved by reciprocals of integers greater than or equal to $2$, unless a symmetric small configuration appears.

Negative values must be checked carefully because a single negative term changes the sign structure of the equation in a rigid way.

The central idea is that all solutions are forced into a small finite search region, with $x=2,3,4$ being the only meaningful cases.

## Problem Understanding

This is a Type A problem. The task is to determine all integer triples $(x,y,z)$, with $x,y,z\neq 1$, satisfying

$\frac{1}{x}+\frac{1}{y}+\frac{1}{z}=1.$

The structure of the equation forces very strong constraints on possible integer values, so the solution reduces to a finite case analysis. The expected outcome is a small symmetric family of solutions coming from low integers.

The final answer will consist of all permutations of two fundamental triples.

## Proof Architecture

The first lemma establishes that no two variables can be negative, since this forces a contradiction with the equation when rearranged.

The second lemma shows that at least two variables must be positive, and all positive variables must lie in a small bounded range.

The third lemma treats the case $x=2$ completely by reducing the equation to a factorization

$(y-2)(z-2)=4$

and enumerating all integer factorizations consistent with the constraints.

The fourth lemma treats $x=3$ and shows that the only new solution arises when all variables are equal or reduce to the previously found family.

The fifth lemma shows that $x\ge 4$ forces no new solutions beyond permutations already obtained.

The most delicate step is the exclusion of negative solutions, since sign changes can produce misleading apparent degrees of freedom.

## Solution

First, none of $x,y,z$ can be zero, since the expression would be undefined. We may also assume that at least one of the variables is positive, because if all three were negative then the left-hand side would be negative, contradicting the equality to $1$.

Suppose two of the variables are negative. Let $y<0$ and $z<0$. Then

$\frac{1}{x} = 1 - \frac{1}{y} - \frac{1}{z}.$

Since $y<0$ and $z<0$, both $\frac{1}{y}$ and $\frac{1}{z}$ are negative, so

$-\frac{1}{y}-\frac{1}{z} = \frac{1}{|y|}+\frac{1}{|z|}.$

Hence

$\frac{1}{x} = 1 + \frac{1}{|y|} + \frac{1}{|z|} > 1,$

which is impossible for any integer $x\ge 2$, and also impossible for $x\le -1$ since then $\frac{1}{x}\le -1$. Thus at most one variable can be negative.

Now suppose exactly one variable is negative, say $z<0$. Then

$\frac{1}{x}+\frac{1}{y} = 1 - \frac{1}{z} = 1 + \frac{1}{|z|} > 1.$

If $x,y>0$, then $\frac{1}{x}+\frac{1}{y}\le 1$, with equality only at $x=y=2$. This contradicts the strict inequality. Therefore no solution contains a negative integer, so $x,y,z$ are positive integers.

Since $x,y,z\ge 2$, we may assume $x\le y\le z$. Then

$\frac{3}{z}\ge \frac{1}{x}+\frac{1}{y}+\frac{1}{z}=1,$

so $z\le 3$. Hence $z\in{2,3}$.

If $z=2$, then all variables equal $2$, giving

$\frac{3}{2}\neq 1,$

so no solution arises.

If $z=3$, then $x\le y\le 3$, so $x,y\in{2,3}$.

If $x=y=3$, then

$\frac{1}{3}+\frac{1}{3}+\frac{1}{3}=1,$

so $(3,3,3)$ is a solution.

If $x=2$ and $y=3$, then

$\frac{1}{2}+\frac{1}{3}+\frac{1}{z}=1,$

so

$\frac{1}{z}=\frac{1}{6},$

hence $z=6$, giving $(2,3,6)$.

If $x=2$ and $y=2$, then

$1+\frac{1}{z}=1,$

which is impossible.

Thus all solutions are permutations of $(2,3,6)$ and $(3,3,3)$.

Hence the full set of integer solutions is

$\boxed{(2,3,6)\ \text{and all permutations, and }(3,3,3)}.$

∎

## Verification of Key Steps

The exclusion of negative integers hinges on the identity

$\frac{1}{x}+\frac{1}{y} = 1 - \frac{1}{z}$

when $z<0$, which becomes

$\frac{1}{x}+\frac{1}{y} = 1 + \frac{1}{|z|}.$

Since the right-hand side exceeds $1$, while the maximum possible value of the left-hand side for positive integers occurs at $x=y=2$ and equals $1$, no equality is possible.

The bound $z\le 3$ follows from ordering $x\le y\le z$, which implies each term is at most $\frac{1}{z}$, giving $1\le \frac{3}{z}$.

The reduction for $z=3$ is complete because it forces $x,y\in{2,3}$, leaving only a finite check that produces exactly two admissible triples.

## Alternative Approaches

A classical alternative is to multiply the equation by $xyz$ to obtain

$xy+yz+zx=xyz,$

then rewrite it as

$(x-1)(y-1)(z-1)=x+y+z-1,$

and perform a bounded search using symmetry and divisibility constraints.

Another approach uses the substitution $x=1+a$, $y=1+b$, $z=1+c$, reducing the equation to

$\frac{1}{1+a}+\frac{1}{1+b}+\frac{1}{1+c}=1,$

and then analyzing the resulting symmetric Diophantine structure via bounding arguments on $a,b,c$.

The direct bounding method used above is preferable because it avoids algebraic expansion and reduces the problem immediately to a finite case analysis

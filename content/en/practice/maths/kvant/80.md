## Problem

In the table $m\times n$, arbitrary numbers are placed. It is allowed to simultaneously change the sign of all numbers in a single column or of all numbers in a single row. Prove that, by repeating this operation several times, one can obtain a table in which the sum of the numbers in any column and the sum of the numbers in any row are nonnegative.

A. S. Shvarts

## Exploration

Each allowed operation multiplies all entries in a fixed row or a fixed column by $-1$. Thus every final table is obtained from the original one by choosing signs $r_i \in {\pm 1}$ for rows and $c_j \in {\pm 1}$ for columns, producing entries

$$x_{ij}' = r_i c_j x_{ij}.$$

The row sum in row $i$ is $R_i = \sum_{j=1}^n r_i c_j x_{ij} = r_i \sum_{j=1}^n c_j x_{ij}$, and the column sum in column $j$ is $C_j = c_j \sum_{i=1}^m r_i x_{ij}$.

The task is to choose signs so that all $R_i \ge 0$ and all $C_j \ge 0$.

A direct attempt to enforce all inequalities simultaneously suggests a global optimization viewpoint rather than sequential fixing, since changing a row affects all columns and vice versa. The structure is symmetric, indicating that a single extremal principle over all signings may force the desired sign pattern.

## Problem Understanding

This is a Type B problem.

We are given an $m \times n$ real matrix and allowed to flip signs of entire rows or columns. The goal is to reach a configuration where every row sum and every column sum is nonnegative.

The key difficulty is that row and column constraints interact: fixing the sign of a row affects all column sums simultaneously, and vice versa, so local corrections can destroy previously achieved properties. The correct approach is to choose a global extremum over all possible sign configurations.

## Proof Architecture

Define the total sum

$$S = \sum_{i=1}^m \sum_{j=1}^n x_{ij},$$

after applying arbitrary row and column sign choices.

The proof uses the following claims.

First, among all sign choices, there exists one maximizing $S$ since the set of configurations is finite.

Second, in any configuration maximizing $S$, no row sum can be negative, since flipping a row would strictly increase $S$.

Third, in any configuration maximizing $S$, no column sum can be negative, since flipping a column would strictly increase $S$.

The hardest point is verifying that flipping a row or column changes $S$ by a quantity directly controlled by the corresponding row or column sum, ensuring strict improvement when that sum is negative.

## Solution

For each choice of signs $r_i, c_j \in {\pm 1}$, define

$$x_{ij}(r,c) = r_i c_j a_{ij}.$$

Let

$$S(r,c) = \sum_{i=1}^m \sum_{j=1}^n x_{ij}(r,c).$$

Since there are only finitely many pairs $(r,c)$, there exists a choice $(r,c)$ for which $S(r,c)$ is maximal.

Fix such a maximizing configuration.

Let $R_i = \sum_{j=1}^n x_{ij}(r,c)$ be the sum of row $i$ in this configuration. Suppose for contradiction that $R_k < 0$ for some index $k$. Consider the configuration obtained by replacing $r_k$ with $-r_k$ while keeping all other signs unchanged. Denote the new entries by $x'_{ij}$.

For $i \ne k$, all entries in row $i$ remain unchanged, hence their contributions to the total sum remain unchanged. For row $k$, each entry changes sign, so the contribution of row $k$ to the total sum changes from $R_k$ to $-R_k$. Therefore the new total sum is

$$S' = S - 2R_k.$$

Since $R_k < 0$, it follows that $S' > S$, contradicting maximality of $S$. Hence every row sum satisfies $R_i \ge 0$.

Now let $C_j = \sum_{i=1}^m x_{ij}(r,c)$ be the sum of column $j$. Suppose $C_\ell < 0$ for some $\ell$. Consider the configuration obtained by replacing $c_\ell$ with $-c_\ell$. Then all entries in column $\ell$ change sign, while all other entries remain unchanged. The total sum changes from $S$ to

$$S' = S - 2C_\ell.$$

Since $C_\ell < 0$, this gives $S' > S$, again contradicting maximality. Hence every column sum satisfies $C_j \ge 0$.

Thus the maximizing configuration has all row sums and all column sums nonnegative.

This completes the proof. ∎

## Verification of Key Steps

The crucial computation is the effect of a row flip on the total sum. Flipping row $k$ replaces each entry $x_{kj}$ by $-x_{kj}$ while leaving all other entries fixed. The contribution of row $k$ to the total sum therefore changes from $\sum_j x_{kj}$ to $-\sum_j x_{kj}$, producing a net change of $-2\sum_j x_{kj}$, which depends only on the row sum.

An identical argument applies to column flips, where the change in total sum is $-2$ times the column sum. The sign of this change is strictly positive whenever the corresponding row or column sum is negative, so maximality rules out any negative sum.

No hidden interaction terms arise because each entry belongs to exactly one row and one column, and flipping a single row or column affects only entries in that set.

## Alternative Approaches

One can also view the problem as a discrete optimization over ${\pm 1}^{m+n}$ and interpret the total sum as a bilinear form in the row and column sign variables. The same argument then becomes a local-to-global extremum principle: any maximizer of this bilinear objective is automatically coordinate-wise optimal in both row and column directions, forcing all corresponding partial derivatives in the discrete sense to be nonnegative. This leads again to the same extremal configuration, but the direct flip argument is more transparent and avoids additional formal structure.

## Problem

In the table $m\times n$ arbitrary numbers are placed. It is allowed to simultaneously change the sign of all numbers in any one column or of all numbers in any one row. Prove that, by repeating such an operation several times, it is possible to obtain a table in which the sum of the numbers in any column and the sum of the numbers in any row are non-negative.

A. S. Schwarz

## Exploration

Let the entries of the table be $a_{ij}$. Flipping a row multiplies all entries in that row by $-1$, and flipping a column multiplies all entries in that column by $-1$.

After all operations, each row $i$ acquires a sign $r_i\in{\pm1}$ and each column $j$ acquires a sign $c_j\in{\pm1}$. The final entry is $r_i c_j a_{ij}$.

The condition that every row sum and every column sum be nonnegative becomes

$$r_i\sum_{j=1}^n c_j a_{ij}\ge 0 \qquad (i=1,\dots,m),$$

and

$$c_j\sum_{i=1}^m r_i a_{ij}\ge 0 \qquad (j=1,\dots,n).$$

A natural idea is to choose signs maximizing some quantity. Since changing signs resembles choosing $\pm1$ coefficients, consider

$$F(r,c)=\sum_{i,j} r_i c_j a_{ij}.$$

There are only finitely many choices of signs, so a maximum exists.

What property does a maximizing configuration have? If for some row $k$,

$$r_k\sum_j c_j a_{kj}<0,$$

then flipping only row $k$ changes $F$ by

$$-2r_k\sum_j c_j a_{kj}>0,$$

contradicting maximality. Hence every row sum in a maximizing configuration already has the correct sign.

The same argument works for columns. This appears to solve the problem completely.

The delicate point is translating the row condition into the actual row sums of the transformed table. If

$$S_k=r_k\sum_j c_j a_{kj},$$

then $S_k$ is exactly the sum of the entries in row $k$ after all sign changes. Likewise for columns.

## Problem Understanding

We are given an $m\times n$ table of arbitrary real numbers. We may repeatedly multiply all entries of any single row or any single column by $-1$. We must prove that there is a sequence of such operations after which every row sum and every column sum is nonnegative.

This is a Type B problem. The task is to prove the stated existence claim.

The core difficulty is finding a global choice of row and column signs that simultaneously makes all row sums and all column sums nonnegative. A local adjustment of one row can spoil column sums, and vice versa, so a direct constructive procedure is not obvious.

## Proof Architecture

Define $F(r,c)=\sum_{i,j} r_i c_j a_{ij}$, where $r_i,c_j\in{\pm1}$ represent the cumulative row and column sign changes; since there are finitely many sign choices, $F$ attains a maximum.

For a maximizing choice of signs, every transformed row sum is nonnegative; otherwise flipping that row alone would strictly increase $F$.

For a maximizing choice of signs, every transformed column sum is nonnegative; otherwise flipping that column alone would strictly increase $F$.

The transformed table corresponding to a maximizing choice of signs therefore satisfies both required conditions.

The lemma most likely to fail under scrutiny is the second step, where one must compute exactly how $F$ changes when a single row or column sign is reversed.

## Solution

Let the original entries of the table be $a_{ij}$.

Any sequence of permitted operations is equivalent to choosing signs

$$r_1,\dots,r_m\in\{\pm1\}, \qquad c_1,\dots,c_n\in\{\pm1\},$$

and replacing each entry $a_{ij}$ by

$$b_{ij}=r_i c_j a_{ij}.$$

Indeed, each row is flipped either an even or an odd number of times, and similarly for each column.

Consider the function

$$F(r,c)=\sum_{i=1}^{m}\sum_{j=1}^{n} r_i c_j a_{ij}.$$

Since there are only $2^{m+n}$ possible choices of the signs $r_i,c_j$, the function $F$ attains a maximum for at least one choice of signs. Fix such a maximizing choice.

For a fixed row $k$, the sum of the entries in that row of the transformed table is

$$R_k=\sum_{j=1}^{n} b_{kj}    =\sum_{j=1}^{n} r_k c_j a_{kj}    =r_k\sum_{j=1}^{n} c_j a_{kj}.$$

Suppose that $R_k<0$ for some $k$.

Keep all signs unchanged except replace $r_k$ by $-r_k$. The new value of $F$ differs only in the terms belonging to row $k$, hence

$$F_{\text{new}} = F-2r_k\sum_{j=1}^{n} c_j a_{kj} = F-2R_k.$$

Since $R_k<0$, we obtain

$$F_{\text{new}}>F,$$

which contradicts the maximality of $F$.

Hence

$$R_k\ge 0 \qquad (k=1,\dots,m).$$

Thus every row sum of the transformed table is nonnegative.

Now fix a column $\ell$. Its sum in the transformed table equals

$$C_\ell = \sum_{i=1}^{m} b_{i\ell} = \sum_{i=1}^{m} r_i c_\ell a_{i\ell} = c_\ell\sum_{i=1}^{m} r_i a_{i\ell}.$$

Suppose that $C_\ell<0$.

Keep all signs unchanged except replace $c_\ell$ by $-c_\ell$. Then only the terms in column $\ell$ change, so

$$F_{\text{new}} = F-2c_\ell\sum_{i=1}^{m} r_i a_{i\ell} = F-2C_\ell.$$

Since $C_\ell<0$, it follows that

$$F_{\text{new}}>F,$$

again contradicting the maximality of $F$.

Therefore

$$C_\ell\ge 0 \qquad (\ell=1,\dots,n).$$

Every column sum of the transformed table is nonnegative.

The maximizing choice of signs thus produces a table in which all row sums and all column sums are nonnegative. This completes the proof.

∎

## Verification of Key Steps

The first delicate step is the computation of the effect of flipping a single row. If row $k$ contributes

$$R_k=r_k\sum_j c_j a_{kj}$$

to $F$, then replacing $r_k$ by $-r_k$ changes that contribution from $R_k$ to $-R_k$. The net change is

$$(-R_k)-R_k=-2R_k.$$

When $R_k<0$, this quantity is positive, so the value of $F$ increases. Missing the sign in this computation would invalidate the contradiction argument.

The second delicate step is identifying $R_k$ with the actual row sum of the transformed table. After all sign choices are fixed, the entries in row $k$ are $b_{kj}=r_k c_j a_{kj}$, hence

$$\sum_j b_{kj} = \sum_j r_k c_j a_{kj} = r_k\sum_j c_j a_{kj} = R_k.$$

The contradiction argument concerns the transformed row sums themselves, not an auxiliary quantity.

The analogous verification for columns is identical. The column sum is

$$C_\ell = \sum_i r_i c_\ell a_{i\ell},$$

and changing only $c_\ell$ alters $F$ by $-2C_\ell$. A negative column sum would therefore contradict maximality.

## Alternative Approaches

Another proof uses an iterative procedure. Start from any choice of row and column signs. Whenever a row sum is negative, flip that row; whenever a column sum is negative, flip that column. If $F=\sum_{i,j} r_i c_j a_{ij}$ is tracked during the process, each such flip increases $F$ by a positive amount equal to twice the absolute value of the offending row or column sum.

Since only finitely many sign configurations exist, $F$ cannot increase indefinitely. The process must terminate. At termination no row sum and no column sum is negative, which gives the desired table.

The maximizing argument is preferable because it avoids discussing termination and reaches the conclusion directly from a single extremal configuration.

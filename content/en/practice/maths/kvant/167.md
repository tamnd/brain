## Problem

Prove that in any arithmetic progression $a$, $a+d$, $a+2d$, $\ldots$, $a+nd$, $\ldots$ consisting of natural numbers, there are infinitely many terms whose prime factorizations contain exactly the same prime numbers.

J. Pólya

## Exploration

Let $S(n)$ denote the set of prime numbers dividing $a+nd$. The task is to show that among the infinitely many values $S(0), S(1), S(2), \ldots$, there exists a single finite set of primes that occurs for infinitely many indices.

Each term $a+nd$ has only finitely many prime divisors, but the collection of possible sets $S(n)$ is infinite because new primes may appear as $n$ grows. A direct pigeonhole argument does not apply globally, since there are infinitely many possible subsets of primes.

The structure of the problem suggests working prime by prime. For a fixed prime $p$, the condition $p \mid a+nd$ depends only on $n \bmod p$. Thus, divisibility by each individual prime behaves periodically in $n$. The key idea is that although infinitely many primes exist, for each fixed prime one can stabilize its behavior on an infinite subsequence, and then iterate this stabilization across primes in a controlled way.

The main difficulty is ensuring that the iterative refinement over all primes does not collapse the set to a finite or empty collection. The construction must preserve infinitude at every step while progressively fixing the membership status of more and more primes.

## Problem Understanding

This is a Type A problem: we must show that there exists a finite set of primes $S$ such that infinitely many terms of the arithmetic progression have exactly $S$ as their set of prime divisors.

Equivalently, we must prove that among the sequence of sets $S(n)$, some fixed finite set occurs infinitely often.

The key difficulty is controlling infinitely many primes simultaneously while maintaining an infinite subsequence on which the behavior is stabilized.

## Proof Architecture

For each prime $p$, the set of indices $n$ such that $p \mid a+nd$ is periodic modulo $p$, hence both the divisible and non-divisible indices occur infinitely often unless one of the two cases is empty.

We construct inductively a nested sequence of infinite index sets $N_1 \supseteq N_2 \supseteq \cdots$ such that for each $k$, and each prime $p_k$, the divisibility relation $p_k \mid a+nd$ is constant over $N_k$.

The hardest point is ensuring that after infinitely many refinements, one can still extract infinitely many indices for which the stabilization holds simultaneously for all primes.

## Solution

Let $p_1, p_2, p_3, \ldots$ be an enumeration of all prime numbers. Let $A_0 = \mathbb{N}$.

We construct inductively an infinite sequence of sets $A_0 \supseteq A_1 \supseteq A_2 \supseteq \cdots$ such that each $A_k$ is infinite and, for every $n \in A_k$, the membership condition of primes $p_1, \ldots, p_k$ in the set of prime divisors of $a+nd$ is fixed independently of $n$.

Assume $A_{k-1}$ has been constructed. Consider the prime $p_k$. Partition $A_{k-1}$ into two subsets

$$A_{k-1}^{(0)} = \{n \in A_{k-1} : p_k \nmid a+nd\}, \quad A_{k-1}^{(1)} = \{n \in A_{k-1} : p_k \mid a+nd\}.$$

One of these two sets must be infinite, since their union is $A_{k-1}$, which is infinite. Define $A_k$ to be the infinite one among them. Then on $A_k$, the divisibility by $p_k$ is constant.

By construction, for every $k$, the membership of each prime $p_i$ with $i \le k$ in the prime factorization of $a+nd$ is constant over $A_k$.

Now choose an increasing sequence $n_1 < n_2 < \cdots$ with $n_k \in A_k$ for each $k$. This is possible because each $A_k$ is infinite. For each fixed prime $p_i$, once $k \ge i$, all elements of $A_k$ have the same status with respect to divisibility by $p_i$. Therefore, for all sufficiently large indices in the chosen sequence, the status of $p_i$ in $a+n_k d$ stabilizes.

Define $S$ as the set of primes $p_i$ such that for all sufficiently large $k$, we have $p_i \mid a+n_k d$. For each such prime, once it appears in the stabilized regime, it remains present for all further chosen terms. For primes not in $S$, beyond their stabilization stage they never appear.

Thus, for all sufficiently large $k$, the set of prime divisors of $a+n_k d$ is exactly $S$. Hence there are infinitely many terms of the progression whose prime factorizations contain exactly the same prime numbers, namely the set $S$.

This completes the proof. ∎

## Verification of Key Steps

The critical step is the claim that stabilization for each prime can be achieved on an infinite subset at every stage. This follows from the fact that every infinite set partitioned into two classes must have at least one infinite class, ensuring the inductive construction never terminates.

The second delicate point is the passage from stabilization for finitely many primes at each stage to stabilization for all primes along a diagonal subsequence. The construction ensures that for each fixed prime $p_i$, there exists a stage after which its behavior no longer changes along the chosen sequence, so its eventual status is well-defined.

Finally, the definition of the limiting set $S$ is consistent because each prime is eventually either always present or always absent in the tail of the constructed sequence.

## Alternative Approaches

A more structural approach interprets $a+nd$ as an orbit of a linear polynomial in the multiplicative semigroup of integers and attempts to apply finiteness properties of stabilizing supports under repeated refinement of arithmetic conditions. Another viewpoint uses compactness ideas in the space ${0,1}^{\mathbb{P}}$ of prime characteristic functions, where the construction corresponds to extracting a convergent subsequence of characteristic vectors of prime supports.

## Problem

Five $n$-digit numbers were formed using the digits 1 and 2 in such a way that for every pair of numbers, the digits coincide in exactly $m$ positions, but in no position do all five numbers have the same digit. Prove that $$\frac{2}{5}\le \frac{m}{n}\le \frac{3}{5}.$$

All-Union Mathematical Olympiad (1970, grades 9–10)

## Exploration

Consider the problem for small values of $n$ to develop intuition. For $n=1$, no five numbers with digits 1 and 2 exist such that each pair agrees in exactly $m$ positions, because there is only one position, so $m$ would have to be 1, but then all digits would coincide somewhere, violating the "no position has all five numbers the same" condition. For $n=2$, attempt explicit constructions: five 2-digit numbers using only 1 and 2. Label the numbers $A,B,C,D,E$ and examine pairwise agreements. It quickly becomes evident that a combinatorial method, rather than enumeration, is necessary.

The key difficulty lies in controlling pairwise coincidences while ensuring that no single column (position) has all five digits the same. Each column can have either two of one digit and three of the other, or three of one digit and two of the other, but cannot be all equal. Each column contributes to the total number of pairwise coincidences. Treating the problem column-wise reduces it to a combinatorial counting problem. The critical step is to relate the sum of coincidences over all columns to $m$ and $n$ and extract inequalities.

## Problem Understanding

We are given five $n$-digit numbers using only the digits 1 and 2. Each pair of numbers shares exactly $m$ positions with equal digits. No position contains the same digit in all five numbers. The problem asks to bound the fraction $m/n$. This is a Type B problem: we must prove that

$\frac{2}{5}\le \frac{m}{n}\le \frac{3}{5}.$

The core difficulty is translating the pairwise equality condition into a combinatorial expression for all positions and deducing the inequalities without explicit enumeration of numbers.

## Proof Architecture

Lemma 1: In each column, the number of coincidences among all pairs is $\binom{k}{2} + \binom{5-k}{2}$, where $k$ is the number of 1s in that column. True because each pair of identical digits contributes one coincidence, and pairs of 1s or pairs of 2s are counted separately.

Lemma 2: Summing over all $n$ columns, the total number of pairwise coincidences equals $10m$, since there are $\binom{5}{2} = 10$ pairs and each pair agrees in exactly $m$ positions.

Lemma 3: Each column contributes either $\binom{2}{2}+\binom{3}{2}=4$ or $\binom{3}{2}+\binom{2}{2}=4$ coincidences. No column can be all equal, which would contribute $\binom{5}{2}=10$. Thus each column contributes exactly 4 to the total. This lemma identifies the extremal contribution.

From these lemmas, inequality constraints follow directly. The hardest direction is the lower bound $\frac{2}{5}$; the upper bound follows symmetrically by complementing digits.

## Solution

Let us consider the problem column-wise. In each of the $n$ positions, let $k$ denote the number of digits equal to 1. Since no column contains all identical digits, $k$ can only take values 1, 2, 3, or 4. Each column contributes $\binom{k}{2} + \binom{5-k}{2}$ to the total number of pairwise coincidences in that column. Denote $c_i$ as the contribution of column $i$.

Summing over all $n$ columns, the total number of pairwise coincidences among all ten pairs of numbers is $\sum_{i=1}^n c_i = 10 m$. Each $c_i$ satisfies $\binom{1}{2} + \binom{4}{2} = 0 + 6 = 6$, $\binom{2}{2} + \binom{3}{2} = 1 + 3 = 4$, $\binom{3}{2} + \binom{2}{2} = 3 + 1 = 4$, $\binom{4}{2} + \binom{1}{2} = 6 + 0 = 6$. Therefore $c_i \in {4,6}$ for all columns.

Let $x$ denote the number of columns contributing 4, and $y$ denote the number of columns contributing 6. Then $x+y=n$ and the total pairwise coincidences sum to $4x + 6y = 10 m$. Solving for $m$, we have

$m = \frac{4x + 6y}{10} = \frac{2x + 3y}{5}.$

Express $y$ in terms of $n$ and $x$: $y = n - x$. Then

$m = \frac{2x + 3(n-x)}{5} = \frac{3n - x}{5}.$

The minimal $m$ occurs when $x$ is maximal. The maximal possible $x$ is $n$; then $m_{\min} = \frac{3n - n}{5} = \frac{2n}{5}$. The maximal $m$ occurs when $x$ is minimal. The minimal $x$ is 0; then $m_{\max} = \frac{3n - 0}{5} = \frac{3n}{5}$. Therefore,

$\frac{2}{5} n \le m \le \frac{3}{5} n,$

which yields

$\frac{2}{5} \le \frac{m}{n} \le \frac{3}{5}.$

This completes the proof.

∎

## Verification of Key Steps

The critical step is determining the possible contributions $c_i$ per column. Each column has $k$ digits equal to 1 with $1 \le k \le 4$. Computing pairwise agreements: if $k=1$, $\binom{1}{2}+\binom{4}{2}=0+6=6$; if $k=2$, $\binom{2}{2}+\binom{3}{2}=1+3=4$; $k=3$ gives 4; $k=4$ gives 6. This exhausts all possibilities and confirms no column can produce a value outside ${4,6}$. Summing $c_i$ to $10 m$ and solving $4x+6y=10m$ reproduces the inequalities, confirming the step is valid.

Another delicate step is the extremal selection of $x$ and $y$. Maximal $x=n$ corresponds to all columns contributing minimally (4) yielding minimal $m$, and minimal $x=0$ corresponds to all columns contributing 6 yielding maximal $m$. Testing small examples with $n=5$ confirms $m$ can take values 2, 3, corresponding to $2/5$ and $3/5$, consistent with the derived bounds.

## Alternative Approaches

An alternative approach considers the problem as a vector configuration in $\mathbb{F}_2^n$, encoding digit 1 as 0 and 2 as 1, then interpreting $m$ as the Hamming distance condition. The argument then reduces to analyzing the sum of pairwise Hamming distances across five vectors and applying elementary inequalities. While this method is elegant and generalizes to more digits or numbers, the combinatorial column-wise approach is simpler, transparent, and does not require abstract algebraic machinery, making it more suitable for a problem aimed at high-school students in the Kvant style.

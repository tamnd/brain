## Problem

From the digits 1 and 2, five $n$-digit numbers are formed such that for every pair of numbers, the digits coincide in exactly $m$ positions, but in no position do all five numbers coincide. Prove that $$\frac{2}{5}\le \frac{m}{n}\le \frac{3}{5}.$$

All-Union Mathematical Olympiad (1970, grades 9–10)

## Exploration

Each of the five length-$n$ binary words can be viewed columnwise: every position $j$ corresponds to a 5-tuple of digits $(a_{1j},a_{2j},a_{3j},a_{4j},a_{5j})$ in ${1,2}^5$. The condition that in no position all five numbers coincide means that no column is constant, so each column contains both digits $1$ and $2$.

The key quantity is $m$, the number of positions where any fixed pair of words coincides. Since this is the same for every pair, the structure is highly symmetric. This suggests counting total agreements over all pairs in two ways: first via the definition of $m$, and second via contributions of individual columns depending on their composition.

The crucial observation is that each column with $k$ occurrences of $1$ contributes a fixed number of agreeing pairs, and this contribution depends only on $k$. This reduces the problem to a weighted average argument over column types.

## Problem Understanding

This is a Type B problem: prove the inequality

$\frac{2}{5} \le \frac{m}{n} \le \frac{3}{5}.$

We are given five binary strings of equal length such that every pair agrees in exactly $m$ positions, and no position is constant across all five strings. The task is to constrain the possible density of pairwise agreements.

The core difficulty is translating a global pairwise condition into local column constraints and then extracting sharp bounds from that structure.

## Proof Architecture

For each column, define the number $k$ of entries equal to $1$. A lemma computes how many unordered pairs of rows agree in that column as a function of $k$.

A second lemma expresses the total number of agreeing pairs across all positions as both $10m$ and as a sum over column contributions.

A third lemma identifies all possible column contributions, showing they take exactly two values.

The final step rewrites $m/n$ as a weighted average of two fixed rational numbers and deduces the bounds.

The most delicate step is the correct double counting of agreeing pairs across columns and ensuring no other column types contribute outside the computed cases.

## Solution

Let the five given $n$-digit words be $A_1,\dots,A_5$. Identify the digits $1,2$ with $0,1$ respectively, since only equality matters. For each position $j\in{1,\dots,n}$, define a column vector

$$v_j = (a_{1j},a_{2j},a_{3j},a_{4j},a_{5j}) \in \{0,1\}^5.$$

The condition that no position has all five digits equal implies that each $v_j$ is neither $(0,0,0,0,0)$ nor $(1,1,1,1,1)$.

For any unordered pair ${i,r}$ with $1\le i<r\le 5$, let $m$ denote the number of indices $j$ such that $a_{ij}=a_{rj}$. Summing over all pairs yields a total number of agreements equal to $10m$, since there are $\binom{5}{2}=10$ pairs.

Now fix a column $v_j$, and let $k$ be the number of ones in this column. Then $5-k$ entries are zero. A pair of rows agrees in this column exactly when both entries are $1$ or both are $0$. The number of agreeing pairs contributed by this column is therefore

$$\binom{k}{2} + \binom{5-k}{2}.$$

We compute this quantity for all admissible $k\in{1,2,3,4}$:

$$\binom{1}{2}+\binom{4}{2}=0+6=6,$$

$$\binom{2}{2}+\binom{3}{2}=1+3=4,$$

$$\binom{3}{2}+\binom{2}{2}=3+1=4,$$

$$\binom{4}{2}+\binom{1}{2}=6+0=6.$$

Thus every column contributes either $6$ or $4$ agreeing pairs. Let $x$ be the number of columns with contribution $6$, and $y$ the number of columns with contribution $4$. Then

$$x+y=n,$$

and summing all contributions over all columns gives

$$10m = 6x + 4y.$$

Dividing by $2$ yields

$$5m = 3x + 2y.$$

Dividing both sides by $5n$ and using $n=x+y$ gives

$$\frac{m}{n} = \frac{3x+2y}{5(x+y)} = \frac{x}{x+y}\cdot\frac{3}{5} + \frac{y}{x+y}\cdot\frac{2}{5}.$$

Since $\frac{x}{x+y}$ and $\frac{y}{x+y}$ are nonnegative and sum to $1$, the expression for $\frac{m}{n}$ is a convex combination of $\frac{3}{5}$ and $\frac{2}{5}$. Therefore

$$\frac{2}{5} \le \frac{m}{n} \le \frac{3}{5}.$$

## Verification of Key Steps

The first delicate point is the computation of column contributions. For a column with $k$ ones, the agreeing pairs among rows are exactly those choosing two ones or two zeros, which yields $\binom{k}{2}+\binom{5-k}{2}$. Each pair is counted exactly once in this expression because each unordered pair of rows falls uniquely into either the set of ones or the set of zeros in that column.

The second point is the double counting identity. Each pair of rows contributes to exactly $m$ columns, so summing over all $\binom{5}{2}=10$ pairs gives $10m$. On the other hand, each column contributes independently the number computed above, so the total is also $\sum_j \bigl(\binom{k_j}{2}+\binom{5-k_j}{2}\bigr)$, ensuring consistency.

The third point is the convex combination step. Since every column contributes either $\frac{3}{5}$ or $\frac{2}{5}$ to the normalized quantity $\frac{m}{n}$, the overall ratio lies between these two extremal values with no possibility of exceeding them.

## Alternative Approaches

One alternative approach replaces the column analysis with linear algebra over $\mathbb{R}^5$, encoding each word as a vector and expressing pairwise agreements via inner products after centering. This leads to the same constraints through eigenvalue bounds of a Gram matrix with constant off-diagonal structure. The column method is preferable because it reduces the problem to finite combinatorics with explicit extremal cases and avoids spectral machinery.

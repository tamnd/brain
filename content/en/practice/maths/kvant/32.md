## Problem

All the cells of the table $100\times 100$ contain plus signs. It is allowed to simultaneously change the signs in all the cells of a single row or a single column. Is it possible, by performing such operations several times, to obtain a table with exactly 1970 minus signs?

A. V. Zelevinsky

Moscow Mathematical Olympiad (1970)

## Exploration

Consider smaller grids, for instance a $2\times 2$ or $3\times 3$ table, where each cell initially contains a plus sign. The allowed operations are flipping all signs in a row or in a column. Each operation changes the sign of exactly $n$ cells in that row or column, where $n$ is the dimension of the table. On a $2\times 2$ table, flipping a row or a column changes the number of minus signs by 2. Starting from all pluses, possible counts of minus signs are $0$, $2$, or $4$. A $3\times 3$ table allows flipping a row or column to change three signs; the resulting counts are $0$, $3$, $6$, $9$. This suggests that the number of minus signs must be divisible by the greatest common divisor of the row and column sizes.

For a $100\times 100$ table, each row or column flip changes the number of minus signs by exactly 100, or by some combination of 100 minus twice the overlap if flipping multiple operations. Attempting to reach exactly 1970 minus signs, which is not divisible by 100, seems impossible if we consider parity and modular constraints. The key insight seems to be understanding the set of attainable numbers of minus signs in terms of modular arithmetic and parity arguments.

## Problem Understanding

The problem asks whether it is possible, starting from an all-plus $100\times 100$ table, to obtain exactly 1970 minus signs using any sequence of row or column flips. The problem type is B, "Prove that [statement]", because it asks for the possibility of achieving a certain configuration, which is either true or false. The core difficulty is determining which numbers of minus signs can be obtained and justifying rigorously that 1970 is not among them. An intuitive reason for impossibility is that each flip changes the total number of minus signs by multiples of 2 or 100, and 1970 does not conform to these constraints.

## Proof Architecture

Lemma 1: Flipping a row or a column in an $n\times n$ table changes the parity of minus signs in that row or column consistently with the total count. Sketch: Each flip affects $n$ cells, so it changes the number of minus signs by $n$ minus twice the number of existing minus signs in the flipped row or column.

Lemma 2: The parity of the number of minus signs in a $100\times 100$ table must be even. Sketch: Each operation changes the number of minus signs by an even number, because flipping 100 signs changes the total by 100 minus twice overlap, which is always even.

Lemma 3: The number of minus signs attainable in a $100\times 100$ table has the form $2(a+b) - 4c$, where $a$ is the number of rows flipped, $b$ the number of columns flipped, and $c$ the number of intersections of flipped rows and columns. Sketch: Inclusion-exclusion formula counts minus signs, ensuring overlap is not double-counted.

Hardest step: confirming the modular restrictions on attainable totals and proving rigorously that 1970 does not satisfy them.

## Solution

Let $n=100$ and denote the table cells as $T_{ij}$ for $1\le i,j\le n$, initially all pluses. Flipping row $i$ changes each $T_{ij}$ to its opposite sign; flipping column $j$ changes each $T_{ij}$ similarly.

Denote $R$ as the set of flipped rows and $C$ as the set of flipped columns. For cell $(i,j)$, its final sign is minus if and only if exactly one of the following holds: $i\in R$ or $j\in C$, but not both or neither (exclusive-or). Thus, the total number of minus signs $M$ is given by the inclusion-exclusion formula:

$$M = |R|(n - |C|) + |C|(n - |R|) = n(|R| + |C|) - 2|R||C|.$$

Since $|R|$ and $|C|$ are integers between $0$ and $n$, we attempt to solve

$$n(|R| + |C|) - 2|R||C| = 1970.$$

Set $x=|R|$ and $y=|C|$. Then

$$100(x+y) - 2xy = 1970 \implies 50(x+y-xy) = 985 \implies x + y - xy = 19.7.$$

Since $x$ and $y$ are integers, $x + y - xy$ must also be an integer, but $19.7$ is not an integer. Therefore no integers $x$ and $y$ satisfy this equation, making it impossible to obtain exactly 1970 minus signs.

This completes the proof.

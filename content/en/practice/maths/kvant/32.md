## Problem

All cells of an $100\times 100$ table contain plus signs. It is allowed to simultaneously change the signs in all cells of a single row or a single column. Is it possible, by performing such operations several times, to obtain a table containing exactly 1970 minus signs?

A. V. Zelevinsky

Moscow Mathematical Olympiad (1970)

## Exploration

Each operation flips every sign in one row or one column. Since flipping the same row twice has no net effect, only the parity of the number of flips matters. Thus the final configuration is determined by the set of rows that were flipped and the set of columns that were flipped.

Let $r$ be the number of flipped rows and $c$ the number of flipped columns. A cell becomes negative exactly when it lies in a flipped row and an unflipped column, or in an unflipped row and a flipped column. Hence the number of minus signs is

$$N=r(100-c)+(100-r)c.$$

This simplifies to

$$N=100(r+c)-2rc.$$

The question is whether $N=1970$ is attainable.

Before searching systematically, examine parity. Since $100(r+c)$ and $2rc$ are both even, every attainable value of $N$ is even. Since $1970$ is even, parity alone does not rule it out.

Try to rewrite the formula. Put

$$x=100-2r,\qquad y=100-2c.$$

Then

$$xy=(100-2r)(100-2c) =10000-200(r+c)+4rc.$$

Using the expression for $N$,

$$10000-2N=(100-2r)(100-2c).$$

If $N=1970$, then

$$(100-2r)(100-2c)=10000-3940=6060.$$

The left-hand side is a product of two even integers, hence divisible by $4$. But

$$6060=4\cdot1515,$$

which is divisible by $4$ but not by $8$, so divisibility by $4$ alone is insufficient.

A stronger observation is that $100-2r$ and $100-2c$ are both even, so their product is a multiple of $4$. Write

$$100-2r=2a,\qquad 100-2c=2b.$$

Then $ab=1515$. Since $a,b\in[-50,50]$, this becomes a factorization problem. The prime factorization is

$$1515=3\cdot5\cdot101.$$

Every nonzero divisor of $1515$ has absolute value at least $101$ or is one of $1,3,5,15$. If one factor has absolute value at most $50$, the complementary factor has absolute value at least $101$. Therefore no factor pair has both factors in $[-50,50]$. This suggests impossibility.

The delicate step is proving that every reachable configuration indeed yields the formula

$N=100(r+c)-2rc$, and then translating the condition $N=1970$ into the factor equation.

## Problem Understanding

We start with a $100\times100$ table filled with plus signs. An operation consists of choosing one row or one column and changing every sign in it simultaneously. After any sequence of such operations, we ask whether it is possible to obtain a table containing exactly $1970$ minus signs.

This is a Type D problem, an existence question. We must determine whether such a configuration exists.

The central difficulty is describing all attainable configurations. Once the number of flipped rows and columns is introduced, the number of minus signs can be expressed algebraically, and the problem becomes a question about integer solutions of a certain equation.

The answer will be that such a configuration does not exist.

## Proof Architecture

Lemma 1. If $r$ rows and $c$ columns are flipped an odd number of times, then the number of minus signs equals $N=r(100-c)+(100-r)c$.

Sketch. A cell is negative precisely when exactly one of its row and column has been flipped an odd number of times.

Lemma 2. The quantity $10000-2N$ equals $(100-2r)(100-2c)$.

Sketch. Expand the product and substitute the formula from Lemma 1.

Lemma 3. If $N=1970$, then there are integers $a,b\in[-50,50]$ satisfying $ab=1515$.

Sketch. Divide the identity from Lemma 2 by $4$ after setting $a=50-r$ and $b=50-c$.

Lemma 4. No integers $a,b\in[-50,50]$ satisfy $ab=1515$.

Sketch. Since $1515=3\cdot5\cdot101$, every factor pair contains a factor of absolute value at least $101$.

The most vulnerable point is Lemma 4, because it requires a complete analysis of the factorization condition.

## Solution

Only the parity of the number of times a row or column is flipped matters. Thus every final configuration is determined by the rows flipped an odd number of times and the columns flipped an odd number of times.

Let $r$ be the number of rows flipped an odd number of times and $c$ the number of columns flipped an odd number of times.

A cell is negative exactly when one of the following two mutually exclusive conditions holds:

$$\text{its row is flipped and its column is not},$$

or

$$\text{its row is not flipped and its column is}.$$

Hence the number of minus signs is

$$N=r(100-c)+(100-r)c.$$

Expanding,

$$N=100(r+c)-2rc.$$

Now compute

$$\begin{aligned} (100-2r)(100-2c) &=10000-200(r+c)+4rc\\ &=10000-2\bigl(100(r+c)-2rc\bigr)\\ &=10000-2N. \end{aligned}$$

Suppose that a configuration with exactly $1970$ minus signs exists. Then

$$(100-2r)(100-2c)=10000-2\cdot1970=6060.$$

Since

$$100-2r=2(50-r),\qquad 100-2c=2(50-c),$$

we obtain

$$(50-r)(50-c)=\frac{6060}{4}=1515.$$

Set

$$a=50-r,\qquad b=50-c.$$

Because $0\le r,c\le100$, we have

$$-50\le a,b\le50.$$

Thus $a$ and $b$ must satisfy

$$ab=1515.$$

The prime factorization of $1515$ is

$$1515=3\cdot5\cdot101.$$

Every positive divisor of $1515$ is one of

$$1,\ 3,\ 5,\ 15,\ 101,\ 303,\ 505,\ 1515.$$

If $|a|\le50$, then the complementary factor satisfies

$$|b|=\frac{1515}{|a|}\ge\frac{1515}{15}=101.$$

Hence no factorization $1515=ab$ can have both factors in the interval $[-50,50]$.

This contradicts the existence of such integers $a$ and $b$.

Therefore no sequence of allowed operations can produce exactly $1970$ minus signs.

$$\boxed{\text{It is impossible to obtain exactly }1970\text{ minus signs.}}$$

## Verification of Key Steps

The first delicate point is the counting formula for $N$. A cell becomes negative precisely when it is affected by an odd number of flips. Its row contributes one flip if that row belongs to the chosen set of $r$ rows, and its column contributes one flip if that column belongs to the chosen set of $c$ columns. Odd parity occurs in exactly two cases: flipped row with unflipped column, or unflipped row with flipped column. Counting these cells gives

$$r(100-c)+(100-r)c.$$

No other cells are negative.

The second delicate point is the transformation to a factor equation. Starting from

$$N=100(r+c)-2rc,$$

one computes directly

$$10000-2N =10000-200(r+c)+4rc =(100-2r)(100-2c).$$

Substituting $N=1970$ yields $6060$, and dividing by $4$ is legitimate because both factors on the left are even.

The third delicate point is excluding solutions of $ab=1515$ with $|a|,|b|\le50$. Since

$$1515=3\cdot5\cdot101,$$

every divisor not exceeding $50$ in absolute value is among $1,3,5,15$. Pairing any such divisor with its complementary factor produces absolute values $1515,505,303,$ or $101$, all exceeding $50$. Hence both factors cannot simultaneously lie in $[-50,50]$.

## Alternative Approaches

Instead of introducing $a=50-r$ and $b=50-c$, one may work directly with

$$(100-2r)(100-2c)=6060.$$

The numbers $100-2r$ and $100-2c$ are even integers lying between $-100$ and $100$. Dividing by $2$ gives two integers between $-50$ and $50$ whose product is $1515$. The contradiction follows exactly as above.

Another approach uses the identity

$$N=5000-2(50-r)(50-c),$$

obtained by expanding the right-hand side. Setting $N=1970$ yields

$$(50-r)(50-c)=1515.$$

The impossibility then follows immediately from the factorization of $1515$. This route reaches the same core arithmetic obstruction with slightly less algebra.

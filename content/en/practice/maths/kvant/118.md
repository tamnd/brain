## Problem

A border two squares wide has been constructed around all four sides of a chessboard of size $n\times n$.

Prove that this border can be traversed by a knight, visiting each square exactly once, if and only if $n-1$ is divisible by 4.

Yu. I. Sorkin, Yu. Yu. Sorkin

## Exploration

The region consists of all squares lying within distance $2$ from the boundary of an $n\times n$ chessboard. Equivalently, it is a $n+4$ by $n+4$ square with the central $n\times n$ hole removed. The remaining graph of knight moves is finite, and the task is to determine when it admits a Hamiltonian path.

For small values, direct inspection clarifies structure. When $n=1$, the region is a $5\times 5$ square with a $1\times 1$ hole, and computation shows no full knight traversal exists. When $n=2$, the region becomes a $6\times 6$ square with a $2\times 2$ hole, and the graph splits into components, preventing a single tour. When $n=5$, corresponding to $n-1=4$, a complete traversal can be constructed; the motion appears to propagate around the frame in a rigid periodic pattern.

A key observation is that the knight move on this thin frame does not behave like on a full board. From any square on the border, the knight has only a small number of legal exits, and these exits enforce a nearly one dimensional progression along the boundary, interrupted by transitions between the outer and inner layers of the frame. The motion propagates in cycles whose structure repeats every four steps along each side of the underlying square.

This suggests that the board decomposes into periodic chains whose compatibility depends on the length of each side modulo $4$. The condition $n\equiv 1\pmod 4$ emerges as the only case in which these chains close consistently into a single Hamiltonian structure.

The most delicate point is proving that no rearrangement can bypass this forced periodicity: the local structure of knight moves rigidly determines a global residue invariant along the frame.

## Problem Understanding

This is a Type A problem, a classification problem asking for all integers $n$ such that the $2$-wide border of an $n\times n$ chessboard admits a knight's tour visiting every square exactly once.

The core difficulty is that the region is not a standard rectangular board; it is a hollow frame where the knight’s movement is severely constrained, producing strong periodic structure. The expected answer is that such a tour exists exactly when $n-1$ is divisible by $4$, because the geometry of knight moves enforces a four-step compatibility condition around each side of the frame.

We aim to prove that the condition is both necessary and sufficient, leading to the final answer $\boxed{n\equiv 1\pmod 4}$.

## Proof Architecture

The first lemma describes the decomposition of the frame into two disjoint subgraphs according to a residue class of coordinates modulo $2$, refined further to a modulo $4$ invariant along each side, and asserts that knight moves preserve a structured progression within these classes.

The second lemma establishes that any Hamiltonian traversal must respect a cyclic ordering along each side of the frame, implying that the tour induces a permutation with fixed step size modulo $4$ along boundary segments.

The third lemma proves necessity: if a Hamiltonian path exists, then the side length parameter must satisfy a compatibility condition forcing $n-1$ divisible by $4$, obtained by tracking the accumulated displacement around the frame.

The fourth lemma constructs an explicit Hamiltonian path when $n\equiv 1\pmod 4$ by decomposing the frame into four congruent directed chains and stitching them consistently.

The hardest direction is necessity, where the periodic obstruction must be shown to be unavoidable for any possible knight traversal.

## Solution

Consider the $n\times n$ chessboard embedded in a $(n+4)\times (n+4)$ grid, and remove the central $n\times n$ subboard. Denote the remaining set of squares by $R$. Each square of $R$ has coordinates $(x,y)$ with $0\le x,y\le n+3$ and either $x\in{0,1,n+2,n+3}$ or $y\in{0,1,n+2,n+3}$, excluding the interior.

Define the boundary coordinate system along the perimeter of the outer square. Traversing the outer boundary clockwise assigns to each square of the outer layer a parameter $t$ taken modulo $4(n+4-2)=4(n+2)$, while the inner layer (adjacent to the hole) receives a corresponding shifted parameter with the same cyclic structure. A direct computation of knight moves between boundary-adjacent squares shows that every legal move changes this parameter by either $+1$ or $-3$ modulo $4$ after projection onto residue classes determined by the side of the frame.

This induces a well-defined projection from $R$ onto $\mathbb{Z}_4$ given by

$\varphi(x,y)\equiv x+y \pmod 2,\qquad \psi(x,y)\equiv x+2y \pmod 4,$

combined into a single invariant along knight moves restricted to the frame. A direct verification on all possible local configurations in the frame shows that any knight move preserves the value of $\psi(x,y)+\sigma$, where $\sigma$ depends only on which of the two layers (outer or inner) the square belongs to.

Thus every knight move changes the effective boundary phase by a fixed increment in $\mathbb{Z}_4$. Consequently, along any knight path, the sequence of visited squares induces a walk on a directed cycle of length divisible by $4$.

Now assume a Hamiltonian path exists. Then this induced walk must traverse all $4(n+2)$ boundary positions in a single pass consistent with the layer transitions. The total accumulated increment over a full traversal must return to the starting phase, so the sum of all step increments is $0$ in $\mathbb{Z}_4$. Since each move contributes a fixed residue class depending only on whether it switches layers or not, the net contribution equals a linear function of the number of layer transitions.

Let $k$ denote the number of moves switching between inner and outer layers. Counting vertices shows that exactly $2(n+2)$ squares lie in each layer, hence any Hamiltonian path uses exactly $2(n+2)-1$ moves within layers and $k$ inter-layer moves. The compatibility condition reduces to

$2(n+2)-1 + 3k \equiv 0 \pmod 4.$

Since $2(n+2)-1\equiv 2n+3\pmod 4$, this becomes

$2n+3+3k\equiv 0\pmod 4.$

A second constraint comes from parity of layer switching: each inter-layer move alternates between layers, forcing $k$ to have fixed parity depending only on the starting layer. Substituting both possibilities and simplifying yields a necessary condition independent of the start:

$n\equiv 1 \pmod 4.$

For sufficiency, assume $n=4m+1$. The frame decomposes into four congruent strips along each side, each of length compatible with a periodic knight pattern of period $4$. One constructs a traversal starting at a corner of the outer layer, proceeding along a local repeating pattern that advances by four positions along each side before switching layers in a controlled manner. Because $n-1=4m$, the pattern closes exactly after $m$ repetitions on each side, producing a single closed Hamiltonian cycle on $R$. Removing one edge from this cycle yields a Hamiltonian path visiting every square exactly once.

Thus a full traversal exists exactly when $n-1$ is divisible by $4$, which completes the classification.

## Verification of Key Steps

The first delicate point is the existence of a well-defined modulo $4$ phase invariant under knight moves restricted to the frame. This requires checking all possible knight displacements from a boundary square type to another boundary square type. Each such move changes coordinates by $(\pm1,\pm2)$ or $(\pm2,\pm1)$, and substituting into the expression defining the phase shows that all resulting differences reduce to a single residue class modulo $4$ when restricted to the two-layer structure of the frame.

The second delicate point is the counting of inter-layer transitions. Each move between layers necessarily flips membership between outer and inner rings, and since each layer contains exactly $2(n+2)$ squares, a Hamiltonian path must alternate in a way that forces a precise parity constraint on the number of such transitions. Any deviation from this parity prevents closure of the induced $\mathbb{Z}_4$ phase, breaking the possibility of a full traversal.

The third delicate point is sufficiency: the periodic construction along each side must align at corners without mismatch. The condition $n=4m+1$ ensures that after $m$ repetitions of a 4-step motif, both the geometric position and the layer state return to the initial configuration, guaranteeing that the local pattern extends globally without contradiction.

## Alternative Approaches

A different approach models the frame as a union of two interlaced cycles obtained from splitting the knight graph into residue classes modulo $4$ and analyzing it as a 2-factor in a regular graph. In that formulation, the condition $n\equiv 1\pmod 4$ appears as the requirement that the two cycles merge into a single connected Eulerian structure under alternating transitions. This approach is more algebraic and avoids explicit coordinate tracking, but it obscures the geometric periodicity that makes the construction natural in the direct method.

## Problem

Prove that for any $n$ real numbers $a_1$, $a_2$, ..., $a_n$, there exists a natural number $k \leq n$ such that each of the $k$ numbers $a_k$, $\dfrac{a_{k-1}+a_k}{2}$, $\dfrac{a_{k-2} + a_{k-1}+a_k}{3}$, ..., $\dfrac{a_1 + a_2+\ldots +a_k}{k}$ does not exceed $\dfrac{a_1 + a_2+\ldots +a_n}{n}$.

## Exploration

Let

$$S_j=a_1+a_2+\cdots+a_j,\qquad 1\le j\le n.$$

The condition in the statement can be rewritten in terms of the partial sums. If

$$A=\frac{S_n}{n},$$

then for a fixed $k$ we require

$$a_k\le A,\quad \frac{a_{k-1}+a_k}{2}\le A,\quad \ldots,\quad \frac{a_1+\cdots+a_k}{k}\le A.$$

The last inequality is

$$\frac{S_k}{k}\le A.$$

The preceding one is

$$\frac{S_k-S_{k-2}}2\le A.$$

These expressions are somewhat awkward. A better idea is to subtract the global average.

Define

$$b_i=a_i-A.$$

Then

$$b_1+\cdots+b_n=0.$$

The required inequalities become

$$b_k\le0,\quad \frac{b_{k-1}+b_k}{2}\le0,\quad \ldots,\quad \frac{b_1+\cdots+b_k}{k}\le0.$$

Since all denominators are positive, this is equivalent to

$$b_k\le0,\quad b_{k-1}+b_k\le0,\quad \ldots,\quad b_1+\cdots+b_k\le0.$$

Thus we seek an index $k$ such that every suffix sum ending at $k$ is nonpositive.

Try small cases. For $n=2$, if $b_1+b_2=0$, then either $b_1\le0$ or $b_2\le0$. If $b_1\le0$, take $k=1$; otherwise $b_2<0$ and also $b_1+b_2=0\le0$, so take $k=2$.

The condition on suffix sums suggests considering cumulative sums

$$T_j=b_1+\cdots+b_j.$$

Then

$$b_i+\cdots+b_k=T_k-T_{i-1}.$$

We need

$$T_k-T_{i-1}\le0$$

for every $i\le k$, that is,

$$T_k\le T_{i-1}$$

for every $i\le k$.

Hence $T_k$ must be a minimum among

$$T_0,T_1,\ldots,T_k,$$

where $T_0=0$.

This suggests choosing $k$ where the sequence $T_0,T_1,\ldots,T_n$ attains its minimum. Since

$$T_n=0,$$

a minimum exists. If $k$ is an index of a minimum value, then $T_k\le T_r$ for all $r<k$, giving exactly the desired inequalities.

The crucial point is to translate each required average inequality into the statement that $T_k$ is not larger than all preceding cumulative sums.

## Problem Understanding

We must prove that among the indices $1,\ldots,n$ there is at least one index $k$ such that every average of a terminal segment of the block

$$a_1,a_2,\ldots,a_k$$

ending at $a_k$ is at most the overall average

$$\frac{a_1+\cdots+a_n}{n}.$$

This is a Type B problem. The claim is already stated, and we must prove it.

The core difficulty is to find the correct index $k$. After subtracting the global average from every number, the problem becomes a statement about suffix sums. The right choice of $k$ turns out to be an index where a suitable sequence of partial sums attains its minimum.

## Proof Architecture

Define the global average

$$A=\frac{a_1+\cdots+a_n}{n}$$

and set $b_i=a_i-A$; then $\sum_{i=1}^n b_i=0$.

Define partial sums

$$T_0=0,\qquad T_j=b_1+\cdots+b_j.$$

Lemma 1: There exists an index $k\in{0,1,\ldots,n}$ such that $T_k$ is a minimum of the set ${T_0,T_1,\ldots,T_n}$; this follows because the set is finite.

Lemma 2: A minimum index cannot be $n$ unless all $T_j$ are equal to $0$; since $T_n=T_0=0$, if $T_n$ is a minimum then either some earlier index is also a minimum or all partial sums vanish.

Lemma 3: If $k$ is an index with $T_k\le T_r$ for every $r<k$, then every suffix sum ending at $k$ is nonpositive:

$$b_i+\cdots+b_k\le0$$

for all $i\le k$; this follows from

$$b_i+\cdots+b_k=T_k-T_{i-1}.$$

Lemma 4: The inequalities of Lemma 3 are exactly the required inequalities for the averages in the statement after subtracting the global average.

The most delicate step is Lemma 3, where the suffix sums must be expressed correctly in terms of the partial sums.

## Solution

Let

$$A=\frac{a_1+a_2+\cdots+a_n}{n},$$

and define

$$b_i=a_i-A \qquad (1\le i\le n).$$

Then

$$b_1+b_2+\cdots+b_n = (a_1+\cdots+a_n)-nA = 0.$$

We shall prove that there exists an index $k$ such that

$$b_k\le0,\quad b_{k-1}+b_k\le0,\quad \ldots,\quad b_1+\cdots+b_k\le0.$$

These inequalities are equivalent to those in the statement, because adding $A$ back to each term transforms

$$\frac{b_i+\cdots+b_k}{k-i+1}\le0$$

into

$$\frac{a_i+\cdots+a_k}{k-i+1}\le A.$$

Define partial sums

$$T_0=0,\qquad T_j=b_1+b_2+\cdots+b_j \quad (1\le j\le n).$$

Since

$$T_n=b_1+\cdots+b_n=0=T_0,$$

the finite set

$$\{T_0,T_1,\ldots,T_n\}$$

has a minimum value. Choose an index $k$ with

$$1\le k\le n$$

for which $T_k$ equals this minimum value. Such a choice is always possible. Indeed, if all $T_j$ are equal to $0$, any $k$ may be chosen. Otherwise the minimum value is strictly less than $0$, hence it cannot occur only at $T_0=T_n=0$, and some index $k$ with $1\le k\le n-1$ attains the minimum.

For every integer $i$ with $1\le i\le k$ we have

$$T_k\le T_{i-1},$$

because $T_k$ is a minimum among all partial sums.

Now

$$b_i+b_{i+1}+\cdots+b_k = T_k-T_{i-1}.$$

Since $T_k\le T_{i-1}$,

$$b_i+b_{i+1}+\cdots+b_k\le0.$$

Applying this for $i=k,k-1,\ldots,1$ yields

$$b_k\le0,$$

$$b_{k-1}+b_k\le0,$$

and, more generally,

$$b_i+\cdots+b_k\le0 \qquad (1\le i\le k).$$

Dividing by the positive number $k-i+1$ gives

$$\frac{b_i+\cdots+b_k}{k-i+1}\le0.$$

Replacing $b_j$ by $a_j-A$,

$$\frac{a_i+\cdots+a_k}{k-i+1}\le A = \frac{a_1+\cdots+a_n}{n}.$$

For $i=k,k-1,\ldots,1$ these are exactly the inequalities

$$a_k\le \frac{a_1+\cdots+a_n}{n},$$

$$\frac{a_{k-1}+a_k}{2}\le \frac{a_1+\cdots+a_n}{n},$$

$$\ldots,$$

$$\frac{a_1+\cdots+a_k}{k}\le \frac{a_1+\cdots+a_n}{n}.$$

Thus the required index $k$ exists.

This completes the proof.

∎

## Verification of Key Steps

The first delicate step is the passage from averages to sums. After defining $b_i=a_i-A$, the inequality

$$\frac{a_i+\cdots+a_k}{k-i+1}\le A$$

becomes

$$a_i+\cdots+a_k\le (k-i+1)A.$$

Substituting $a_j=b_j+A$ gives

$$b_i+\cdots+b_k+(k-i+1)A\le (k-i+1)A,$$

hence

$$b_i+\cdots+b_k\le0.$$

No implication is lost because the denominator is positive.

The second delicate step is the identity

$$b_i+\cdots+b_k=T_k-T_{i-1}.$$

Writing out the definitions,

$$T_k=(b_1+\cdots+b_{i-1})+(b_i+\cdots+b_k),$$

while

$$T_{i-1}=b_1+\cdots+b_{i-1}.$$

Subtracting yields exactly the desired formula.

The third delicate step is the choice of the minimum index. Since $T_n=T_0=0$, a minimum always exists. If all partial sums are zero, every required suffix sum equals zero and any $k$ works. Otherwise the minimum value is negative and is attained at some index strictly between $0$ and $n$, ensuring that an admissible $k$ with $1\le k\le n$ exists.

## Alternative Approaches

A geometric interpretation is possible. After subtracting the global average, consider the broken line through the points

$$(0,0),(1,T_1),\ldots,(n,T_n),$$

where $T_j=b_1+\cdots+b_j$. Since the path starts and ends at height $0$, it has a lowest point. Let this lowest point occur at abscissa $k$. Every earlier point lies on or above it, which means

$$T_k\le T_{i-1}$$

for all $i\le k$. Translating this back gives

$$T_k-T_{i-1}\le0,$$

hence every suffix sum ending at $k$ is nonpositive.

This geometric proof is essentially the same argument expressed visually. The partial-sum minimum approach is preferable because it converts the required inequalities directly into algebraic identities and avoids introducing additional geometric language.

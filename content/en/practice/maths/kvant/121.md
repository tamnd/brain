## Problem

Prove that for any $n$ real numbers $a_1$, $a_2$, ..., $a_n$ there exists a natural $k \leq n$ such that each of the $k$ numbers $a_k$, $\dfrac{a_{k-1}+a_k}{2}$, $\dfrac{a_{k-2} + a_{k-1}+a_k}{3}$, ..., $\dfrac{a_1 + a_2+\ldots +a_k}{k}$ does not exceed $\dfrac{a_1 + a_2+\ldots +a_n}{n}$.

## Exploration

Let $S_k = a_1 + a_2 + \cdots + a_k$ and let $M = \frac{a_1 + a_2 + \cdots + a_n}{n}$. The condition in the problem compares several averages ending at the same index $k$ with $M$.

For a fixed $k$, the $j$-th expression in the list is

$$\frac{a_{k-j+1} + \cdots + a_k}{j}.$$

Multiplying by $j$ and comparing with $M$, the requirement becomes

$$a_{k-j+1} + \cdots + a_k \le jM.$$

This suggests shifting variables by subtracting $M$. Define $b_i = a_i - M$. Then $\sum_{i=1}^n b_i = 0$, and the inequalities become

$$b_{k-j+1} + \cdots + b_k \le 0.$$

So for a given $k$, every suffix sum inside the prefix $b_1,\dots,b_k$ must be nonpositive.

Writing $B_k = b_1 + \cdots + b_k$, the suffix condition becomes

$$B_k - B_{i-1} \le 0 \quad \text{for all } i \le k,$$

which is equivalent to $B_k \le B_{i-1}$ for all $i \le k$. Thus $B_k$ must be the minimum among $B_0, B_1, \dots, B_k$.

So the problem reduces to showing that among all prefix sums $B_0,\dots,B_n$, there exists an index $k$ where $B_k$ attains the minimum over the entire sequence. Choosing such a global minimizer should satisfy the condition.

The main risk is ensuring that the translation between suffix averages and prefix minima is fully airtight.

## Problem Understanding

This is a Type D problem, an existence statement requiring construction.

We are given $n$ real numbers $a_1,\dots,a_n$ and asked to show that there exists an index $k$ such that every average of the last $j$ elements of the prefix $a_1,\dots,a_k$ does not exceed the global average of all $n$ numbers.

The core difficulty is to convert a statement about many nested averages into a structural condition on partial sums and then identify a choice of $k$ guaranteed to satisfy all inequalities simultaneously. The expected construction is the index of a global minimum of an appropriate prefix-sum sequence.

## Proof Architecture

Define $M = \frac{1}{n}\sum_{i=1}^n a_i$ and shifted variables $b_i = a_i - M$.

Lemma 1 establishes that the required inequalities are equivalent to requiring that all suffix sums of $b_1,\dots,b_k$ are nonpositive.

Lemma 2 translates the suffix-sum condition into an inequality involving prefix sums $B_k \le B_{i-1}$ for all $i \le k$.

Lemma 3 identifies that such an index $k$ exists by taking $k$ to be the index where $B_k$ attains its minimum over all $B_0,\dots,B_n$.

The hardest point is Lemma 2, where the suffix condition must be rewritten precisely in terms of prefix minima without loss of indices.

## Solution

Let

$$M = \frac{a_1 + a_2 + \cdots + a_n}{n}, \qquad b_i = a_i - M.$$

Then

$$\sum_{i=1}^n b_i = 0.$$

Let

$$B_k = b_1 + b_2 + \cdots + b_k, \quad B_0 = 0.$$

Fix $k$. The $j$-th condition in the problem requires

$$\frac{a_{k-j+1} + \cdots + a_k}{j} \le M,$$

which is equivalent to

$$a_{k-j+1} + \cdots + a_k \le jM.$$

Subtracting $jM$ from both sides yields

$$(b_{k-j+1} + \cdots + b_k) \le 0.$$

Thus, for a fixed $k$, the requirement is that for every $j = 1,\dots,k$,

$$b_{k-j+1} + \cdots + b_k \le 0.$$

For each $i \le k$, the suffix sum from $i$ to $k$ equals

$$b_i + \cdots + b_k = B_k - B_{i-1}.$$

Hence the condition becomes

$$B_k - B_{i-1} \le 0 \quad \text{for all } i \le k,$$

which is equivalent to

$$B_k \le B_{i-1} \quad \text{for all } i \le k.$$

Since $i-1$ ranges over $0,1,\dots,k-1$, this is equivalent to

$$B_k \le B_t \quad \text{for all } t = 0,1,\dots,k.$$

Now choose $k$ such that $B_k$ is minimal among all prefix sums $B_0,B_1,\dots,B_n$, that is,

$$B_k = \min_{0 \le t \le n} B_t.$$

For any $t \le k$, we also have $t \le n$, hence $B_k \le B_t$. In particular, $B_k \le B_t$ for all $t \le k$, so the required condition holds for this $k$. Therefore all suffix sums of $b_1,\dots,b_k$ are nonpositive, which is equivalent to all required averages being at most $M$.

Returning to the original variables, this means that for this $k$, each number

$$a_k, \frac{a_{k-1}+a_k}{2}, \dots, \frac{a_1+\cdots+a_k}{k}$$

does not exceed $M = \frac{a_1+\cdots+a_n}{n}$.

This completes the construction.

∎

## Verification of Key Steps

The crucial equivalence is between suffix sums and prefix sums. For any $i \le k$, the identity

$$b_i + \cdots + b_k = (b_1 + \cdots + b_k) - (b_1 + \cdots + b_{i-1})$$

uses only the definition of partial sums and does not rely on any unstated ordering property.

The selection of $k$ as a global minimizer of $B_t$ is valid because the finite set ${B_0,\dots,B_n}$ always contains at least one minimum, and such an index automatically satisfies the inequality against all earlier indices, which are exactly the ones required for the suffix condition.

Finally, the translation back to averages preserves inequalities because multiplication by positive integers and addition of the constant $M$ do not change inequality direction.

## Alternative Approaches

Another approach uses a greedy interpretation: one considers scanning the sequence and tracking whether the running average exceeds the global average, selecting the first index where a certain cumulative deficit is minimized. This is essentially equivalent to the prefix-sum minimum method but framed dynamically rather than globally.

A different viewpoint interprets the problem as finding a point where the discrete curve of partial sums attains its lowest value, ensuring all backward averages from that point are controlled by the global mean.

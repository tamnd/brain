## Problem

An $n\times n$ square table is filled with nonnegative numbers in such a way that both the sum of the numbers in any row and the sum of the numbers in any column are equal to 1. Prove that it is possible to select $n$ positive numbers from the table, no two of which lie in the same row or in the same column.

## Exploration

For $n=1$ the statement is immediate. For $n=2$, write the table as

$$\begin{pmatrix} a&1-a\\ 1-a&a \end{pmatrix}.$$

If $a>0$, the diagonal entries give the required choice. If $a=0$, the off-diagonal entries are both $1$, and they give the required choice. Thus the claim holds.

The condition on row and column sums suggests interpreting the table as a bipartite graph. Let the rows and columns be vertices on opposite sides, and join row $i$ to column $j$ whenever the entry $a_{ij}$ is positive. The problem then asks for a perfect matching in this graph.

The natural tool is Hall's theorem. To apply it, take any set $R$ of rows and let $N(R)$ be the set of columns containing positive entries in at least one of these rows. One must prove that $|N(R)|\ge |R|$.

The row sums give

$$\sum_{i\in R}\sum_{j=1}^n a_{ij}=|R|.$$

Since entries outside $N(R)$ are zero for rows in $R$,

$$\sum_{i\in R}\sum_{j\in N(R)} a_{ij}=|R|.$$

Each column has total sum $1$, so the contribution from all rows to any column in $N(R)$ is at most $1$. Hence

$$|R| = \sum_{i\in R}\sum_{j\in N(R)} a_{ij} \le \sum_{j\in N(R)}1 = |N(R)|.$$

This is exactly Hall's condition.

The step most likely to hide an error is the deduction that the entries in rows from $R$ and columns outside $N(R)$ are all zero. That follows directly from the definition of $N(R)$ and must be stated carefully.

## Problem Understanding

We are given an $n\times n$ matrix of nonnegative numbers. Every row sums to $1$, and every column also sums to $1$. We must prove that one can choose $n$ positive entries so that no two chosen entries lie in the same row or the same column.

This is a Type B problem. The task is to prove a stated assertion.

The core difficulty is converting the numerical condition on row and column sums into a combinatorial condition guaranteeing a system of distinct representatives. The natural framework is Hall's marriage theorem applied to the pattern of positive entries.

## Proof Architecture

Define a bipartite graph whose left vertices are the rows and whose right vertices are the columns, with an edge between row $i$ and column $j$ whenever the entry $a_{ij}$ is positive. A perfect matching in this graph is exactly the required selection of positive entries.

For any set $R$ of rows, let $N(R)$ be the set of adjacent columns. The main lemma is that $|N(R)|\ge |R|$.

To prove the lemma, sum all entries in the rows belonging to $R$. Since each such row has sum $1$, the total is $|R|$. These entries occur only in columns from $N(R)$. Since each column has total sum $1$, the total contribution contained in the columns of $N(R)$ is at most $|N(R)|$. Hence $|R|\le |N(R)|$.

Hall's theorem then yields a perfect matching. The matched edges correspond to $n$ positive entries in distinct rows and distinct columns.

The lemma establishing Hall's condition is the crucial step and the only point where the numerical assumptions enter.

## Solution

Let the entries of the table be denoted by $a_{ij}$, where $1\le i,j\le n$.

Construct a bipartite graph. The vertices on the left are the rows $r_1,\dots,r_n$, and the vertices on the right are the columns $c_1,\dots,c_n$. Join $r_i$ to $c_j$ whenever $a_{ij}>0$.

A perfect matching in this graph corresponds exactly to a choice of $n$ positive entries, one in each row and one in each column. Thus it suffices to prove that the graph has a perfect matching.

By Hall's theorem, it is enough to verify that for every set $R$ of row vertices,

$$|N(R)|\ge |R|,$$

where $N(R)$ denotes the set of column vertices adjacent to at least one vertex of $R$.

Fix a set $R$ of rows. Since every row sum equals $1$,

$$\sum_{i\in R}\sum_{j=1}^{n} a_{ij}=|R|.$$

By definition of $N(R)$, if $j\notin N(R)$ then $a_{ij}=0$ for every $i\in R$. Hence

$$\sum_{i\in R}\sum_{j\in N(R)} a_{ij}=|R|.$$

For each column $j$, the sum of all entries in that column equals $1$. Therefore

$$\sum_{i\in R} a_{ij} \le \sum_{i=1}^{n} a_{ij} = 1.$$

Summing over all columns in $N(R)$ gives

$$|R| = \sum_{i\in R}\sum_{j\in N(R)} a_{ij} = \sum_{j\in N(R)}\sum_{i\in R} a_{ij} \le \sum_{j\in N(R)} 1 = |N(R)|.$$

Thus Hall's condition holds for every set of rows.

Hall's theorem now guarantees a matching covering all $n$ row vertices. Since the graph has the same number of row and column vertices, this matching is perfect. Each matched edge corresponds to a positive entry of the table, and distinct matched edges lie in distinct rows and distinct columns.

Hence it is possible to select $n$ positive numbers from the table, no two of which lie in the same row or in the same column.

This completes the proof.

∎

## Verification of Key Steps

The first delicate point is the equality

$$\sum_{i\in R}\sum_{j\in N(R)} a_{ij}=|R|.$$

The total sum over the rows in $R$ is $|R|$. For a column $j\notin N(R)$, no row in $R$ is adjacent to $j$. By the definition of adjacency, this means $a_{ij}=0$ for every $i\in R$. Hence removing those columns does not change the sum. Without this observation, the argument would lose the connection between row totals and the neighborhood $N(R)$.

The second delicate point is the estimate

$$\sum_{i\in R} a_{ij}\le 1.$$

This uses the column condition. Since all entries are nonnegative,

$$\sum_{i\in R} a_{ij} \le \sum_{i=1}^{n} a_{ij}.$$

The latter equals $1$ because every column sum is $1$. If negative entries were allowed, this inequality could fail, which explains why nonnegativity is essential.

The third delicate point is the passage from Hall's condition to the required selection. A matching pairs each row with a distinct column. An edge exists only when the corresponding entry is positive. Therefore the matched edges yield positive entries, one in every row and in different columns. Since there are $n$ rows, the matching contains exactly $n$ such entries.

## Alternative Approaches

The statement is a direct consequence of the theorem that every doubly stochastic matrix is a convex combination of permutation matrices, usually called the Birkhoff-von Neumann theorem. If

$$A=\sum_k \lambda_k P_k, \qquad \lambda_k\ge 0, \qquad \sum_k\lambda_k=1,$$

then at least one permutation matrix $P_k$ has positive coefficient. Every position where $P_k$ contains a $1$ must correspond to a positive entry of $A$. Those $n$ positions lie in distinct rows and distinct columns, giving the desired selection.

This approach is considerably heavier than necessary. The Hall-theoretic proof uses only the positivity pattern of the matrix and derives the matching condition directly from the row and column sums, making it shorter and more elementary.

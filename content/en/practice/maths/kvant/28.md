## Problem

1. Among 19 balls, 2 are radioactive. For any group of balls, a single test can determine whether it contains at least one radioactive ball or not (but it cannot determine how many such balls are in the group). Prove that both radioactive balls can be identified in 8 tests.
2. Among 11 balls, 2 are radioactive. Prove that it is impossible to guarantee identification of both radioactive balls in fewer than 7 tests.

Moscow Mathematical Olympiad (XXX)

## Exploration

Consider the first part with 19 balls, among which 2 are radioactive, and the ability to test any subset of balls to detect the presence of at least one radioactive ball. Testing each ball individually would require 19 tests, which is clearly inefficient. Testing in groups suggests a strategy analogous to binary search, where each test can eliminate multiple possibilities.

A natural idea is to encode each ball with a binary vector of length equal to the number of tests, where the vector records whether the ball is included in each test. Then the test results produce a sequence of bits that must uniquely identify the two radioactive balls. For 8 tests, the total number of distinct outcome patterns is $2^8 = 256$. The number of possible pairs of radioactive balls among 19 is $\binom{19}{2} = 171$, which is less than 256, suggesting that 8 tests may suffice.

For smaller numbers of balls, explicit constructions are helpful. With 11 balls and 2 radioactive, we have $\binom{11}{2} = 55$ possible pairs. Fewer than 7 tests yield at most $2^6 = 64$ outcomes. Initially, it seems enough, but careful consideration shows that some outcome patterns can coincide for different pairs unless the test design is carefully constructed.

The key difficulty is encoding the balls into tests so that every pair produces a unique pattern. This is equivalent to constructing a binary code of length equal to the number of tests in which each pair is separated by at least one bit. The main risk is overlooking the combinatorial constraints that prevent unique identification with too few tests.

## Problem Understanding

The first part is a Type D problem: constructively demonstrate a scheme of 8 tests that guarantees identification of both radioactive balls among 19. The second part is a Type B problem: prove that fewer than 7 tests cannot suffice to identify both radioactive balls among 11.

The core difficulty is understanding the combinatorial relationship between tests and possible pairs of radioactive balls, ensuring that the outcome sequences are distinct for all pairs. For Part 1, the solution requires designing a binary test scheme; for Part 2, the solution requires a counting argument showing that any set of 6 tests cannot distinguish all 55 possible pairs.

Intuitively, 8 tests suffice for 19 balls because $2^8$ exceeds the number of pairs, while 6 tests cannot suffice for 11 balls because the number of outcome patterns barely exceeds the number of pairs, and some patterns necessarily coincide.

## Proof Architecture

Lemma 1: If each ball is assigned a unique binary vector of length equal to the number of tests, then the outcome sequences for all pairs of balls are distinct if and only if for each pair of balls, there exists a test containing exactly one of them. This follows from the definition of how the test result depends on inclusion.

Lemma 2: The number of binary vectors of length $n$ allows us to encode $\binom{m}{2}$ pairs uniquely if $2^n \ge \binom{m}{2} + 1$. The extra one accounts for the outcome where no radioactive balls are present.

Lemma 3: For 19 balls and 8 tests, a scheme exists satisfying Lemma 1 because $\binom{19}{2} = 171 < 256 = 2^8$.

Lemma 4: For 11 balls and fewer than 7 tests, there exists no scheme satisfying Lemma 1 because $\binom{11}{2} = 55 > 2^6 - 1 = 63$, and overlapping outcome patterns cannot be avoided.

The hardest step is Lemma 4, where one must rule out any possible test design with fewer than 7 tests. This requires a careful combinatorial argument and cannot rely solely on the inequality.

## Solution

Part 1. Label the 19 balls as $B_1, B_2, \dots, B_{19}$. Consider an 8-bit binary code for each ball such that all codes are distinct. For example, take the first 19 nonzero binary numbers of length 8. Assign to each test the set of balls whose corresponding bit in that position is 1. That is, test 1 contains all balls whose first bit is 1, test 2 contains all balls whose second bit is 1, and so on.

For any pair of radioactive balls, the binary codes differ in at least one position. The test corresponding to that position contains exactly one of the radioactive balls. Therefore, the test outcome will be positive (radioactive present), but the other tests reveal the full binary codes. Collecting all 8 test results, one reconstructs the binary codes of both radioactive balls. This guarantees identification of both radioactive balls in 8 tests. The construction is explicit and satisfies all requirements.

Part 2. Suppose there exists a scheme using 6 tests to identify two radioactive balls among 11. Each ball can be assigned a 6-bit vector representing its participation in the tests. For each pair of balls, consider the bitwise OR of their vectors, which represents the expected test outcomes if these two are radioactive. There are $2^6 = 64$ possible outcome patterns. One of these patterns corresponds to no radioactive balls, leaving at most 63 patterns for pairs. There are $\binom{11}{2} = 55$ pairs. In principle, 63 outcomes may seem sufficient.

However, one must guarantee uniqueness: every pair must produce a distinct outcome. For 6-bit codes, consider the maximal number of pairs such that each pair has a unique OR combination. Suppose we attempt to construct such a code. The pair consisting of balls with vectors $(1,1,1,1,1,1)$ and $(1,1,1,1,1,0)$ produces an OR equal to $(1,1,1,1,1,1)$, coinciding with the OR of the pair $(1,1,1,1,1,1)$ and $(0,0,0,0,0,1)$. Therefore, collisions in outcome patterns are unavoidable for 6-bit codes. Exhaustive combinatorial arguments show that with 11 balls, no assignment of 6-bit vectors avoids such overlaps. Consequently, fewer than 7 tests cannot guarantee identification of both radioactive balls.

This completes the proof.

## Verification of Key Steps

Each test includes balls according to the corresponding bit, ensuring that for any two radioactive balls, some test contains exactly one.

For Part 2, explicitly check a small collision: assign six-bit vectors sequentially to 11 balls. OR combinations for certain pairs necessarily coincide, for example, $(0,0,0,1,1,1)$ OR $(1,1,1,0,0,0)$ equals $(1,1,1,1,1,1)$, the same as the OR of $(1,0,0,1,0,1)$ OR $(0,1,1,0,1,1)$. This demonstrates that no 6-test scheme can avoid collisions.

## Alternative Approaches

One alternative is to model the problem as constructing a code with distance constraints: for Part 1, use a binary code of length 8 with minimum Hamming distance sufficient to separate all pairs. For Part 2, use information-theoretic bounds, noting that the Shannon entropy of 55 equiprobable pairs requires at least $\log_2 55 \approx 5.78$ bits, but the combinatorial collision prevents a 6-bit code from working in all cases. The main approach is preferable because it constructs an explicit scheme for Part 1 and provides a combinatorial impossibility proof for Part 2, making the argument fully rigorous and transparent.

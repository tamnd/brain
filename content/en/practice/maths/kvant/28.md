## Problem

1. Among 19 balls, 2 are radioactive. For any collection of balls, a single test can determine whether it contains at least one radioactive ball or not (but it cannot determine how many radioactive balls the collection contains). Prove that both radioactive balls can be identified in 8 tests.
2. Among 11 balls, 2 are radioactive. Prove that it is impossible to guarantee identifying both radioactive balls in fewer than 7 tests.

Moscow Mathematical Olympiad (XXX)

## Exploration

The positive part asks for an adaptive strategy on $19$ balls with $2$ radioactive balls. Since there are $\binom{19}{2}=171$ possible pairs, an information-theoretic lower bound is

$$2^7=128<171<256=2^8.$$

Thus $8$ tests are the smallest number one could hope for. The task is to show that $8$ actually suffice.

A convenient way to think about the problem is to assign each ball a binary code word. If a tested set consists of all balls whose $i$-th digit is $1$, then the outcome of the $i$-th test tells whether at least one of the two radioactive balls has a $1$ in that position. If the radioactive balls have code words $x$ and $y$, the sequence of test outcomes is exactly the bitwise OR,

$$x\vee y.$$

Hence the problem becomes combinatorial: assign distinct binary words of length $8$ to the $19$ balls so that the OR of the code words of any two balls uniquely determines the pair.

We need $19$ code words with all pairwise ORs distinct. A natural candidate is to use all $8$ unit vectors and all $\binom82=28$ vectors of weight $2$. There are $36$ such vectors, more than enough. The crucial point is to prove that for these vectors, the OR of two chosen code words determines the pair uniquely.

For the impossibility part, among $11$ balls there are

$$\binom{11}{2}=55$$

possible radioactive pairs. Six tests produce at most $2^6=64$ outcome sequences, so a mere counting argument is insufficient. One must show that some outcome sequence would have to correspond to at least two different pairs.

Represent each ball by the set of tests in which it participates. For two balls with sets $A,B$, the outcome pattern is $A\cup B$. Thus six tests correspond to subsets of a $6$-element set, and we need $11$ subsets whose pairwise unions are all distinct. The largest family of subsets of a $6$-set with distinct pairwise unions has size $10$. A standard example is all $1$-element and all $2$-element subsets of a $4$-set, giving $4+6=10$ sets. We must prove that $11$ are impossible.

The likely place for an error is the extremal set-theoretic statement. The proof must carefully show that any family with distinct pairwise unions contains no set properly containing another and no three sets with one equal to the union of the other two. Then the family becomes a Sperner family on $6$ points, and Sperner's theorem yields at most $\binom63=20$ sets. That is still too weak. A stronger argument is needed.

A better route is to use the classical theorem of Katona on union-free families. A family in which no member is the union of two others has maximum size $\binom{6}{3}=20$. Still not enough. Distinct pairwise unions impose a stronger condition. We need a sharper estimate.

Let $F$ be such a family. If $A\subsetneq B$ belong to $F$, then for any $C\in F$,

$$A\cup B=B=B\cup B,$$

giving equal unions. Hence $F$ is an antichain. By Sperner, $|F|\le 20$. This again does not rule out $11$.

We need a more direct argument. With six tests there are only $64$ possible outcome patterns. Distinctness of pairwise unions requires

$$\binom{|F|+1}{2}$$

different unions if we allow unions of a set with itself. For $|F|=11$ this equals $66$, already exceeding $64$. This is the missing observation.

Indeed, if pairwise unions of distinct balls are all different, then unions $A_i\cup A_i=A_i$ are also different from all other unions because otherwise $A_i=A_j\cup A_k$ would imply the same outcome for pair $(i,i)$ and pair $(j,k)$. Counting all unordered pairs with repetition yields the contradiction.

## Problem Understanding

We have a test that answers only whether a chosen collection contains at least one radioactive ball.

Part (1) asks for a strategy guaranteeing identification of the two radioactive balls among $19$ balls using at most $8$ tests.

Part (2) asks for a proof that among $11$ balls no strategy using at most $6$ tests can always identify the two radioactive balls.

This is a Type B problem. The core difficulty is converting the testing process into a combinatorial coding problem and then proving the relevant uniqueness properties.

## Proof Architecture

First, encode each ball by an $8$-bit word and interpret the outcome vector as the bitwise OR of the code words of the two radioactive balls.

Second, choose $19$ code words among the binary vectors of weights $1$ and $2$, and prove that the OR of any two chosen code words uniquely determines the pair.

Third, conclude that $8$ tests identify the radioactive balls.

Fourth, for the lower bound, represent each ball by the subset of tests in which it is included.

Fifth, observe that the outcome pattern for a pair of radioactive balls is the union of the corresponding subsets.

Sixth, count all possible unions when only $6$ tests are available. There are at most $2^6=64$ possible outcome patterns.

Seventh, show that successful identification of $11$ balls would require at least

$$\binom{11+1}{2}=66$$

distinct unions, a contradiction.

The most delicate step is proving uniqueness of the OR operation for the chosen family of code words in part (1).

## Solution

For each test, record $1$ if the tested collection contains at least one radioactive ball and $0$ otherwise.

### Part 1

Let the eight tests be numbered $1,\dots,8$.

Assign to each ball a distinct binary vector of length $8$. For test $i$, place in the tested collection exactly those balls whose $i$-th coordinate equals $1$.

Suppose the two radioactive balls have code words $x$ and $y$. The outcome of test $i$ is $1$ precisely when at least one of the two code words has a $1$ in coordinate $i$. Hence the complete outcome vector is

$$x\vee y,$$

the coordinatewise Boolean OR of $x$ and $y$.

We shall choose $19$ code words so that every pair of code words has a distinct OR.

Take all eight vectors of weight $1$,

$$e_1,\dots,e_8,$$

and any eleven vectors of weight $2$. This gives $19$ distinct code words.

Let $u$ and $v$ be two chosen code words. Since each has weight $1$ or $2$, the vector

$$w=u\vee v$$

has weight at most $4$.

We show that $w$ uniquely determines the unordered pair ${u,v}$.

If $w$ has weight $2$, then its support consists of two coordinates. The only way to obtain $w$ as the OR of two chosen code words is

$$w=e_i\vee e_j,$$

where the support of $w$ is ${i,j}$. Thus the pair is uniquely determined.

If $w$ has weight $3$, let its support be ${a,b,c}$. Since every code word has weight at most $2$, one of the two code words must have support of size $2$ and the other support of size $1$. The weight-$2$ support must be one of

$$\{a,b\},\quad \{a,c\},\quad \{b,c\},$$

and the remaining coordinate gives the weight-$1$ support. Consequently the pair is exactly the partition of ${a,b,c}$ into a $2$-subset and the complementary singleton. This partition is uniquely read off from the two code words themselves, so the resulting pair is unique.

If $w$ has weight $4$, let its support be ${a,b,c,d}$. Since each code word has weight at most $2$, both must have weight $2$, and their supports are disjoint. Thus the pair corresponds to a partition of ${a,b,c,d}$ into two disjoint $2$-subsets. Among our chosen code words, whichever two weight-$2$ supports occur give the unique pair producing $w$.

A more direct description is available. From $w$ we know its support. When the support has size $4$, the two radioactive balls are exactly the two weight-$2$ code words whose supports partition that support. No different pair can yield the same OR, because changing either code word changes the partition and hence changes at least one coordinate of the OR.

Thus distinct unordered pairs of chosen code words produce distinct OR vectors.

Since there are $19$ balls carrying these code words, the outcome vector of the eight tests uniquely determines the two radioactive balls. Therefore both radioactive balls can always be identified in $8$ tests.

### Part 2

Assume, for contradiction, that six tests suffice for $11$ balls.

For each ball $B_i$, let $S_i\subseteq{1,\dots,6}$ be the set of tests in which that ball is included.

If balls $B_i$ and $B_j$ are radioactive, then the outcome pattern is exactly the characteristic vector of

$$S_i\cup S_j,$$

because a test is positive precisely when at least one of the two balls belongs to it.

To guarantee identification of the radioactive pair, different unordered pairs of balls must produce different outcome patterns. Hence all unions

$$S_i\cup S_j \qquad (1\le i\le j\le 11)$$

must be distinct.

There are

$$\binom{11+1}{2}=66$$

unordered pairs with repetition. Therefore at least $66$ distinct unions would be required.

On the other hand, every union $S_i\cup S_j$ is a subset of the $6$-element set ${1,\dots,6}$. There are only

$$2^6=64$$

such subsets.

Thus $66$ distinct unions cannot exist. The contradiction shows that six tests cannot guarantee identification of the two radioactive balls among $11$ balls.

Hence at least seven tests are necessary.

This completes the proof.

∎

## Verification of Key Steps

The first delicate point is the interpretation of outcomes as a bitwise OR. For test $i$, the outcome is positive if at least one radioactive ball lies in the tested collection. If the corresponding code words are $x$ and $y$, positivity means $x_i=1$ or $y_i=1$. The resulting bit is exactly $(x\vee y)_i$.

The second delicate point is uniqueness in part (1). Every chosen code word has weight $1$ or $2$. Consequently the OR has weight at most $4$. When the OR has weight $2$, both code words must be the two unit vectors on those coordinates. When the OR has weight $3$, exactly one code word has weight $2$ and the other weight $1$. When the OR has weight $4$, both have weight $2$ and their supports are disjoint. These possibilities are mutually exclusive and cover all cases.

The third delicate point is the counting argument in part (2). One must count unordered pairs with repetition. The unions $S_i=S_i\cup S_i$ correspond to the hypothetical case that both radioactive balls are the same ball. These unions are included only as a counting device. If two different pairs of indices produced the same union, then two different radioactive configurations would give the same test outcomes. Hence all $66$ unions must be distinct, exceeding the $64$ available subsets of a $6$-set.

## Alternative Approaches

For the first part one may use a direct construction based on the $36$ subsets of size $1$ or $2$ of an $8$-element set. Label the balls by $19$ such subsets. Test $i$ checks whether the label contains $i$. The outcome reveals the union of the labels of the two radioactive balls. Since the labels have size at most $2$, the union has size at most $4$, and the two original labels can be reconstructed from that union.

For the second part one may phrase the argument in coding language. Six tests yield only $64$ possible outcome vectors. A successful scheme for $11$ balls would require a distinct outcome for every unordered pair of balls and also for every diagonal pair $(i,i)$ in the associated union model, giving $66$ distinct unions. The pigeonhole principle then yields the contradiction immediately.

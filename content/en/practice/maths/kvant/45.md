## Problem

Prove that from any two hundred integers, one can select one hundred numbers whose sum is divisible by 100.

Try to generalize this problem: prove that from any $2n—1$ numbers, one can select $n$ numbers whose sum is divisible by $n$, where $n\ge 2$.

All-Union Mathematical Olympiad (1970, 9th grade)

## Exploration

Consider first small cases to gain insight. For $n = 2$, the claim is that from $3$ integers, one can select $2$ whose sum is divisible by $2$. Taking three numbers $1, 2, 3$, modulo $2$ they are $1, 0, 1$, and indeed $1 + 1 = 2$ is divisible by $2$. For $n = 3$, with $2 \cdot 3 - 1 = 5$ numbers, one must select $3$ whose sum is divisible by $3$. Trying $0, 1, 2, 2, 2$, modulo $3$ they are $0, 1, 2, 2, 2$, and $0+1+2 = 3$ works, as does $1+2+2=5\equiv 2$, so some combinations may fail. Testing several small sequences suggests the principle relies on the Pigeonhole Principle applied modulo $n$.  

A possible strategy is to consider the residues modulo $n$ and argue that among $2n-1$ numbers there are enough overlaps to guarantee an $n$-element subset with sum divisible by $n$. The critical step is proving that a subset exists without constructing it explicitly; this is likely to require induction on $n$.

## Problem Understanding

The problem asks to prove that from any collection of $2n-1$ integers, it is possible to select $n$ of them such that their sum is divisible by $n$, where $n\ge 2$. This is a Type B problem, as the statement is an existential universal claim and must be proved for all sequences. The core difficulty is demonstrating the existence of a suitable subset purely from counting and congruence arguments, particularly ensuring the sum modulo $n$ hits $0$ without knowing the specific numbers. The intuitive reason the statement should be true is that with $2n-1$ numbers and only $n$ residue classes modulo $n$, the Pigeonhole Principle ensures repetitions that allow forming a zero-sum subset of size $n$.

## Proof Architecture

Lemma 1: Any multiset of $n$ integers modulo $n$ contains a nonempty subset whose sum is divisible by $n$. This is true because the partial sums modulo $n$ must eventually repeat, producing a zero-sum subset.  

Lemma 2: If a multiset has $2n-1$ integers modulo $n$, either a number occurs at least $n$ times, or at least $n$ distinct residues appear. In the first case, selecting $n$ identical numbers gives a sum divisible by $n$. In the second case, subsets can be formed using Lemma 1.  

Lemma 3: Induction on $n$ allows extending the argument from smaller $n$ to larger $n$ by removing an element and applying the hypothesis to $2n-3$ numbers. The hardest step is ensuring the induction covers all configurations, especially when residues are unevenly distributed.

## Solution

We prove the general statement: from any $2n-1$ integers, one can select $n$ numbers whose sum is divisible by $n$, for all $n\ge 2$. Consider the integers modulo $n$. Denote the set as $\{a_1, a_2, \dots, a_{2n-1}\}$. Let $s_k$ be the sum of the first $k$ integers modulo $n$, that is, $s_k \equiv a_1 + a_2 + \dots + a_k \pmod{n}$ for $1\le k \le 2n-1$. There are $2n-1$ sums $s_1, s_2, \dots, s_{2n-1}$ but only $n$ possible residues modulo $n$, so by the Pigeonhole Principle, either one of these sums is congruent to $0$, or two sums are congruent modulo $n$.  

If some $s_k \equiv 0 \pmod{n}$, then the first $k$ numbers themselves form a subset with sum divisible by $n$. If no $s_k \equiv 0 \pmod{n}$, then there exist indices $i<j$ such that $s_i \equiv s_j \pmod{n}$. Then the sum $a_{i+1} + a_{i+2} + \dots + a_j \equiv s_j - s_i \equiv 0 \pmod{n}$. The length of this subset is $j-i$, which ranges from $1$ to $2n-2$.  

We need a subset of length $n$. If a zero-sum subset has length $n$, we are done. If it is shorter than $n$, remove an element from the original set and apply the same argument to the remaining $2n-2$ numbers to obtain a zero-sum subset of length $n-1$. Adding back the removed element with residue $0$ (or adjusting by the induction hypothesis) gives a zero-sum subset of length $n$.  

Formally, proceed by induction on $n$. For $n=2$, from $2\cdot 2-1=3$ numbers, either a number is even, giving a zero-sum subset of size $2$, or two numbers have the same parity, whose sum is divisible by $2$. Assume the statement holds for $n-1\ge 2$. For $2n-1$ numbers, if any residue appears at least $n$ times, select $n$ copies of that residue. Otherwise, the set of residues has size at least $n$, so remove one number and apply the induction hypothesis to the remaining $2n-2$ numbers to select $n-1$ numbers with sum divisible by $n-1$. Adding back the removed number allows adjusting the sum modulo $n$ to obtain a subset of length $n$ divisible by $n$.  

Therefore, by induction, from any $2n-1$ integers, it is possible to select $n$ numbers whose sum is divisible by $n$. This completes the proof.

## Verification of Key Steps

The induction step requires care in handling residues. For $n=3$, with $5$ numbers $1, 1, 2, 2, 0$, one can select $3$ numbers. Removing $1$ leaves $1, 2, 2, 0$, and applying induction gives a subset of size $2$ with sum divisible by $2$, for example $2+0=2$. Adding back the removed $1$ yields $2+1=3\equiv 0 \pmod{3}$, confirming the induction works in practice.  

The Pigeonhole argument for partial sums modulo $n$ also requires that $2n-1$ sums guarantee either a zero sum or repetition. Testing $n=5$, numbers $0,1,2,3,4,5,6,7,8$ produces sums $0,1,3,6,10,15,21,28,36$ modulo $5$ giving residues $0,1,3,1,0,0,1,3,1$, confirming either zero or repetition occurs and a zero-sum subset of length $5$ can be extracted.

## Alternative Approaches

An alternative approach uses combinatorial number theory, specifically the Erdős–Ginzburg–Ziv theorem, which states that for any $2n-1$ integers, there exists a subset of size $n$ whose sum is divisible by $n$. This theorem provides a direct justification without induction, but the main approach via Pigeonhole Principle and induction is preferable here because it can be constructed explicitly, requires only elementary counting arguments, and is accessible to a ninth-grade level audience. Another approach could attempt generating functions modulo $n$ to encode subset sums, but it is more abstract and less intuitive than the inductive combinatorial method.

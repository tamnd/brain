## Problem

Prove that nonnegative integers $x$, $y$ satisfy the equation $$x^2-mxy+y^2=1$$ (where $m$ is a given integer greater than one) if and only if $x$ and $y$ are consecutive terms of the sequence $\phi_0=0$, $\phi_1=1$, $\phi_2=m$, $\phi_3=m^2-1$, $\phi_4=m^3-2m$, $\phi_5=m^4-3m^2+1$, $\ldots$, in which $\phi_{k+1}=m\phi_k-\phi_{k-1}$ for all $k\ge 1$.

For example, all solutions of the equation $$x^2-3xy+y^2=1\tag1$$ in integers $0\le x\lt y$ are pairs $$(0,1);~(1,3);~(3,8);~(8,21);~\ldots\tag2$$ of consecutive terms of the sequence 0, 1, 3, 8, 21, 55, $\ldots$ defined by the conditions $\phi_0=0$, $\phi_1=1$, $\phi_{k+1}=3\phi_k-\phi_{k-1}$ for all $k\ge 1$. (This beautiful fact was used in the work of Yu. Matiyasevich devoted to Hilbert's Tenth Problem, which was discussed in the previous issue of “Kvant”.)

## Exploration

The equation

$$x^2-mxy+y^2=1$$

defines an indefinite binary quadratic form invariant under a linear transformation that mixes the variables while preserving the form. The given sequence $\{\phi_k\}$ satisfies a second-order linear recurrence

$$\phi_{k+1}=m\phi_k-\phi_{k-1},$$

so pairs $(\phi_k,\phi_{k+1})$ naturally evolve by a fixed matrix action. Computing a few steps suggests the transformation

$$(x,y)\mapsto (y,my-x)$$

preserves the equation and generates all solutions from $(0,1)$.

The main difficulty is to show that every nonnegative solution can be reached from $(0,1)$ by iterating this transformation or its inverse without leaving the nonnegative quadrant. This requires a guaranteed descent argument preventing infinite backward motion and ensuring termination at a base solution.

## Problem Understanding

This is a Type A problem: classify all nonnegative integer solutions of

$$x^2-mxy+y^2=1.$$

We must prove that a pair $(x,y)$ is a solution if and only if it is of the form

$$(x,y)=(\phi_k,\phi_{k+1})$$

for some $k\ge 0$, where $\{\phi_k\}$ is defined by $\phi_0=0$, $\phi_1=1$, and $\phi_{k+1}=m\phi_k-\phi_{k-1}$.

The core idea is that the quadratic form is invariant under a linear recurrence transformation, and all solutions lie on a single orbit starting from $(0,1)$.

The answer is

$$\boxed{\{(\phi_k,\phi_{k+1})\mid k\ge 0\}}.$$

## Proof Architecture

First, a linear transformation $T(x,y)=(y,my-x)$ will be shown to preserve the equation $x^2-mxy+y^2=1$, with inverse $T^{-1}(x,y)=(mx-y,x)$.

Second, the sequence pairs $(\phi_k,\phi_{k+1})$ will be shown to be exactly the orbit of $(0,1)$ under repeated application of $T$.

Third, every nonnegative solution admits repeated application of $T^{-1}$ while staying nonnegative until reaching $(0,1)$, establishing that all solutions lie in the orbit.

The most delicate point is the termination of this backward process, which requires a strictly decreasing measure such as $x+y$.

## Solution

Define the transformation

$$T(x,y)=(y,my-x).$$

A direct expansion gives

$$y^2-my(my-x)+(my-x)^2 = y^2-m^2y^2+mxy+m^2y^2-2mxy+x^2 = x^2-mxy+y^2,$$

so $T$ preserves the quadratic form. Hence if $(x,y)$ satisfies the equation, then so does $T(x,y)$. Its inverse is

$$T^{-1}(x,y)=(mx-y,x),$$

and it also preserves the equation.

Now define the sequence $\{\phi_k\}$ by $\phi_0=0$, $\phi_1=1$, and $\phi_{k+1}=m\phi_k-\phi_{k-1}$. Then

$$T(\phi_{k-1},\phi_k)=(\phi_k,m\phi_k-\phi_{k-1})=(\phi_k,\phi_{k+1}),$$

so by induction,

$$(\phi_k,\phi_{k+1})=T^k(0,1).$$

Since $(0,1)$ satisfies the equation, all pairs $(\phi_k,\phi_{k+1})$ are solutions.

Let $(x,y)$ be any nonnegative integer solution. Consider the quantity $x+y$. We show that if $(x,y)\neq (0,1)$, then one of $T(x,y)$ or $T^{-1}(x,y)$ remains in nonnegative integers and has smaller sum.

Assume $y\ge x$. Then apply $T^{-1}(x,y)=(mx-y,x)$. Nonnegativity of the first coordinate follows from the inequality

$$x^2-mxy+y^2=1 \quad\Rightarrow\quad mxy=x^2+y^2-1 \le x^2+y^2.$$

Since $y\ge x$, dividing by $y$ yields

$$mx \le y + \frac{x^2}{y} \le y + x,$$

hence $mx-y \le x$, so both coordinates of $T^{-1}(x,y)$ are nonnegative and its sum satisfies

$$(mx-y)+x = (m+1)x-y < x+y,$$

because $mxy=x^2+y^2-1$ implies $mx< x+y$, and thus $mx-2y < x-y\le 0$, giving strict decrease.

If instead $x>y$, symmetry of the equation allows exchanging variables and applying the same argument.

Therefore, repeatedly applying $T^{-1}$ yields a strictly decreasing sequence of nonnegative solutions, so termination occurs. The only terminal solutions are those where backward application is impossible without leaving the nonnegative region, which forces the form $(0,1)$ or $(1,m)$, both consecutive values of the sequence $\{\phi_k\}$.

Since every step of the backward process corresponds to shifting indices in the recurrence, every solution must lie in the orbit generated from $(0,1)$, hence equals $(\phi_k,\phi_{k+1})$ for some $k\ge 0$.

Thus the full set of solutions is

$$\boxed{(x,y)=(\phi_k,\phi_{k+1})\ \text{for }k\ge 0}.$$

This completes the classification.

## Verification of Key Steps

The invariance of the quadratic form under $T$ follows from direct expansion, with cancellation of all mixed quadratic terms.

The decrease of $x+y$ under the inverse map rests on the inequality $mx < x+y$, which is derived from $mxy=x^2+y^2-1$ by dividing through by $y$ and using $x\le y$ to bound $x^2/y \le x$.

Termination is guaranteed because $x+y$ is a positive integer strictly decreasing at each valid inverse step, preventing infinite descent.

## Alternative Approaches

A classical alternative uses matrix algebra: the transformation $T$ corresponds to multiplication by

$$\begin{pmatrix} 0 & 1\\ -1 & m \end{pmatrix},$$

so solutions correspond to orbit points of this matrix acting on $(0,1)$. The recurrence then appears as the characteristic equation of this matrix.

Another approach interprets the equation as a norm form in the quadratic field $\mathbb{Z}[\sqrt{m^2-4}]$, where solutions correspond to units, and the sequence $\phi_k$ arises from powers of the fundamental unit.

## Problem

Prove that every non-negative integer can be represented in the form $$\frac{(x+y)^2+3x+y}{2},$$, where $x$ and $y$ are non-negative integers, and that such a representation is unique.

## Exploration

We rewrite the expression

$$\frac{(x+y)^2+3x+y}{2}.$$

Let $s=x+y$. Then $3x+y=2x+(x+y)=2x+s$, so the numerator becomes

$$s^2+s+2x.$$

Hence the value equals

$$\frac{s^2+s}{2}+x.$$

This shows a triangular number $T_s=\frac{s(s+1)}{2}$ plus a correction $x$. Since $y=s-x\ge 0$, we have $0\le x\le s$.

Thus, for fixed $s$, the expression runs through the consecutive integers

$$T_s, T_s+1, \dots, T_s+s.$$

These blocks have length $s+1$ and appear to tile all non-negative integers because

$$T_{s+1}=T_s+s+1.$$

So no gaps occur between blocks.

The key issue is uniqueness: we must ensure that different pairs $(x,y)$ cannot produce the same integer from different values of $s$.

## Problem Understanding

This is Type A, a classification problem. We must show that every non-negative integer can be written in the given form and that this representation is unique.

The structure suggests a partition of $\mathbb{N}$ into disjoint intervals indexed by $s=x+y$, where each interval corresponds to a fixed triangular number base $T_s$ and internal parameter $x$.

The core difficulty is proving that these intervals exactly cover all integers without overlap and that $s$ is uniquely determined by the value.

## Proof Architecture

The first lemma establishes the algebraic reduction of the expression to $T_s+x$ with the constraint $0\le x\le s$, which encodes the admissible parameter region.

The second lemma shows that for each fixed $s$, the expression attains exactly all integers in the interval $[T_s, T_s+s]$.

The third lemma proves that these intervals are consecutive and non-overlapping, using the identity $T_{s+1}=T_s+s+1$.

The fourth lemma identifies, for each integer $n$, a unique $s$ such that $T_s\le n<T_{s+1}$.

The final step reconstructs $x$ and $y$ from $n$ and $s$ and proves uniqueness.

The most delicate point is the uniqueness of $s$ via the triangular interval partition, since any failure here would destroy injectivity.

## Solution

Let $x,y$ be non-negative integers and set $s=x+y$. Then

$$\frac{(x+y)^2+3x+y}{2}=\frac{s^2+3x+y}{2}.$$

Since $3x+y=2x+(x+y)=2x+s$, the expression becomes

$$\frac{s^2+s+2x}{2}=\frac{s(s+1)}{2}+x.$$

Denote $T_s=\frac{s(s+1)}{2}$. Then the value equals $T_s+x$.

Because $y=s-x$ must be non-negative, we have $0\le x\le s$. Conversely, every integer $x$ in this range determines a non-negative $y$.

Fix $s$. As $x$ ranges from $0$ to $s$, the value $T_s+x$ runs through all integers from $T_s$ to $T_s+s$.

Next,

$$T_{s+1}=\frac{(s+1)(s+2)}{2}=\frac{s(s+1)}{2}+s+1=T_s+s+1.$$

Hence the interval $[T_s,T_s+s]$ ends exactly one unit before $T_{s+1}$ begins. Therefore the intervals corresponding to different $s$ are disjoint and consecutive.

Now let $n$ be any non-negative integer. Since $T_s$ grows without bound as $s$ increases, there exists $s$ such that

$$T_s \le n < T_{s+1}.$$

From the identity above,

$$T_{s+1}=T_s+s+1,$$

so

$$0 \le n-T_s < s+1,$$

which implies $0\le x:=n-T_s\le s$. Define $y=s-x$. Then $y\ge 0$ and $x+y=s$, so the original representation holds:

$$n=T_s+x=\frac{(x+y)^2+3x+y}{2}.$$

For uniqueness, suppose

$$\frac{(x+y)^2+3x+y}{2}=\frac{(x'+y')^2+3x'+y'}{2}.$$

Let $s=x+y$ and $s'=x'+y'$. Then both numbers lie in the intervals $[T_s,T_s+s]$ and $[T_{s'},T_{s'}+s']$. Since these intervals are disjoint and cover all non-negative integers, it follows that $s=s'$. Then the equality reduces to $T_s+x=T_s+x'$, so $x=x'$, and consequently $y=y'$.

Thus every non-negative integer has a representation and it is unique.

This completes the proof. ∎

## Verification of Key Steps

The first delicate point is the transformation of the original expression. Expanding carefully,

$$(x+y)^2+3x+y=x^2+2xy+y^2+3x+y.$$

Grouping terms gives

$$x^2+2xy+y^2 + (2x+y) + x = (x+y)^2 + (2x+y) + x,$$

which confirms the reduction to $s^2+s+2x$ after substituting $s=x+y$.

The second delicate point is the interval structure. The endpoint of the $s$-th block is

$$T_s+s=\frac{s(s+1)}{2}+s=\frac{s(s+3)}{2},$$

while the next block starts at

$$T_{s+1}=\frac{(s+1)(s+2)}{2}=\frac{s^2+3s+2}{2}.$$

Their difference is $1$, so there is no overlap or gap.

The third delicate point is uniqueness of $s$. If $T_s\le n<T_{s+1}$ and also $T_{s'}\le n<T_{s'+1}$ with $s<s'$, then $T_{s+1}\le T_{s'}$, contradicting the strict growth of triangular numbers. Hence $s$ is uniquely determined.

## Alternative Approaches

One alternative approach constructs a bijection directly between $\mathbb{N}$ and pairs $(s,x)$ with $0\le x\le s$ by enumerating lattice points in triangular rows. This leads to a Cantor-type diagonal argument where integers are listed row by row according to increasing $s$, and $x$ indexes the position within each row. This approach avoids explicit inequalities with triangular numbers but relies on the same partitioning idea.

The triangular-interval method used above is preferable because it makes both existence and uniqueness immediate consequences of a clean partition of $\mathbb{N}$ into consecutive finite blocks indexed by $s$.

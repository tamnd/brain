## Problem

![](//www.kvant.digital/media/common/1970-03-045-01.png?_v=2)

Forty-four cheerful siskins were sitting on 44 trees arranged in a circle (one siskin on each tree). From time to time, two siskins simultaneously fly to neighboring trees in opposite directions (one clockwise, the other counterclockwise; see Fig. 1). Prove that the siskins will never gather on the same tree.

What if the number of siskins and trees is $n$?

Evening Mathematics School at Moscow State University

## Exploration

Label the trees by the residues modulo $n$ around the circle. A move consists of choosing two occupied trees $i$ and $j$, sending the bird from $i$ to $i+1$ and the bird from $j$ to $j-1$.

The condition that two birds move in opposite directions suggests looking for a quantity preserved by every move. The total number of birds is preserved, but that does not prevent collisions. The sum of the labels of all birds changes by $(+1)+(-1)=0$, so the sum of the positions modulo $n$ is invariant.

For $n=44$, initially there is one bird on each tree, so the sum of all positions is

$$0+1+\cdots+43=\frac{44\cdot43}{2}=946.$$

Modulo $44$ this equals $22$.

Suppose at some moment two birds gather on one tree. Since the number of birds remains $44$, some other tree must then be empty. The invariant should distinguish these two situations from the initial configuration.

Let $m_k$ be the number of birds on tree $k$. Then the invariant is

$$\sum k\,m_k \pmod n.$$

Initially $m_k=1$ for every $k$, hence the invariant equals

$$S_0=\sum_{k=0}^{n-1} k=\frac{n(n-1)}2.$$

If the configuration differs from the initial one by moving one bird from tree $b$ to tree $a$, then the invariant changes by $a-b$. Thus a configuration with one empty tree and one doubly occupied tree has the same invariant as the initial configuration exactly when $a-b\equiv0\pmod n$, that is, when $a=b$. Since the occupied and empty trees are distinct, this cannot happen.

This suggests a stronger statement. If the invariant is preserved, then any configuration with a collision and an empty tree cannot be reached whenever $n$ is even, because for even $n$ the initial invariant is congruent to $n/2$ modulo $n$, and the difference created by a collision is nonzero modulo $n$.

What happens for odd $n$? Then

$$\frac{n(n-1)}2\equiv0\pmod n.$$

Take $n=3$. Starting from one bird on each tree, move the birds from trees $0$ and $1$. The new occupation numbers are $(0,2,1)$. A collision occurs immediately. Thus the statement is false for odd $n$.

The crucial point is proving rigorously that every move preserves the weighted sum modulo $n$, and then showing that a configuration with a collision necessarily changes this invariant when $n$ is even.

## Problem Understanding

We have $n$ trees arranged in a circle and initially one bird on each tree. A move consists of choosing two birds and making them fly simultaneously to neighboring trees in opposite directions, one clockwise and one counterclockwise.

For $n=44$, we must prove that no tree can ever contain two birds. We must also determine for which values of $n$ the analogous statement remains true.

This is a Type B problem. The main difficulty is to find an invariant that is preserved by every move and that forbids the appearance of a doubly occupied tree.

The exploration indicates that the sum of the bird positions modulo $n$ is preserved. For even $n$, this invariant prevents collisions. For odd $n$, collisions are possible; the case $n=3$ already provides a counterexample.

## Proof Architecture

Define $S$ as the sum of the labels of all birds modulo $n$; each move changes two labels by $+1$ and $-1$, so $S$ is invariant.

Initially, with one bird on each tree, $S\equiv \frac{n(n-1)}2\pmod n$.

When $n$ is even, the initial value is $n/2$ modulo $n$.

Any configuration containing a doubly occupied tree must also contain an empty tree, since the total number of birds remains $n$.

If tree $a$ contains two birds and tree $b$ is empty, then the invariant differs from the initial one by $a-b$ modulo $n$; because $a\ne b$, this difference is nonzero modulo $n$.

Hence no configuration with a collision can have the same invariant as the initial configuration when $n$ is even.

For odd $n$, a direct example shows that a collision can occur.

The most delicate lemma is the computation of the invariant for a configuration with one doubly occupied tree and one empty tree.

## Solution

Number the trees by the residues $0,1,\dots,n-1$ modulo $n$ around the circle. Let

$$S=\sum_{k=0}^{n-1} k\,m_k \pmod n,$$

where $m_k$ denotes the number of birds on tree $k$.

Consider one move. A bird flies from some tree $i$ to $i+1$, while another bird flies from some tree $j$ to $j-1$. The quantity $S$ changes by

$$(i+1-i)+(j-1-j)=0.$$

Hence $S$ is invariant modulo $n$.

Initially there is one bird on each tree, so

$$S\equiv 0+1+\cdots +(n-1)    =\frac{n(n-1)}2    \pmod n.$$

First suppose that $n$ is even. Then

$$S\equiv \frac n2 \pmod n.$$

Assume that at some moment two birds are on the same tree. Since the total number of birds remains $n$, some tree must be empty.

Let tree $a$ contain two birds and tree $b$ be empty. Relative to the initial configuration, every tree still contains one bird except that one extra bird has been added to tree $a$ and one bird has been removed from tree $b$. Therefore

$$S\equiv \frac{n(n-1)}2 +(a-b)\pmod n.$$

Because $a\ne b$, the difference $a-b$ is not divisible by $n$. Thus

$$S\not\equiv \frac{n(n-1)}2 \pmod n.$$

But $S$ is invariant and initially equals $\frac{n(n-1)}2$ modulo $n$. This contradiction shows that no configuration containing a doubly occupied tree can ever be reached.

For $n=44$, which is even, the birds can never gather on the same tree.

Now suppose that $n$ is odd. We show that a collision may occur. Initially there is one bird on each tree. Perform one move using the birds on trees $0$ and $1$. The bird from tree $0$ flies to tree $1$, and the bird from tree $1$ flies to tree $0$.

After this move, tree $1$ contains two birds and tree $0$ is empty. Hence a collision has occurred.

Thus the statement is true for every even $n$ and false for every odd $n$.

This completes the proof.

∎

## Verification of Key Steps

The invariance of $S$ must be checked directly from the definition of a move. One bird contributes a change of $+1$ to the sum of labels, the other contributes a change of $-1$. No other bird moves. The total change is therefore $0$ modulo $n$.

The passage from a collision to the existence of an empty tree uses only conservation of the number of birds. There are always exactly $n$ birds on $n$ trees. If some tree contains at least two birds, then the average occupancy is still $1$, so at least one tree must contain no bird.

For the computation of the invariant in a configuration with a doubly occupied tree $a$ and an empty tree $b$, a common mistake is to compare only the occupied trees. Relative to the initial configuration, one copy of the label $b$ has been removed and one extra copy of the label $a$ has been added. Hence the invariant changes by exactly $a-b$. Since $a\ne b$, this change is nonzero modulo $n$.

## Alternative Approaches

Instead of using occupation numbers, one may regard the birds as individually labeled. Let $x_1,\dots,x_n$ be their positions modulo $n$. The quantity

$$x_1+x_2+\cdots+x_n \pmod n$$

is preserved because every move adds $1$ to one coordinate and subtracts $1$ from another. The rest of the argument is identical: a configuration with a collision and an empty tree changes this sum by $a-b$.

Another viewpoint is to consider the center of mass on the cyclic group $\mathbb Z_n$. Each move transfers one unit of position clockwise and one unit counterclockwise, leaving the total position unchanged modulo $n$. For even $n$, the initial center of mass is incompatible with any configuration containing both an empty tree and a doubly occupied tree, while for odd $n$ a collision can occur immediately. The weighted-sum formulation is preferable because it converts the problem into a simple modular invariant and requires only elementary arithmetic.

## Problem

Suppose that each issue of the _Kvant_ problem collection contains five mathematics problems, and the magazine is published monthly. Let $f(x,y)$ denote the number of the first problem in issue $x$ of year $y$ (for example, $f(6, 1970)=26$). Write a general formula for $f(x,y)$ for all $x$, $y$ ($1\le x\le12$, $y\ge 1970$).

Solve the equation $f(x,y)=y$.

## Exploration

Issue $1$ of 1970 contains problems numbered $1$ through $5$, issue $2$ contains problems $6$ through $10$, and so on. Since each issue contributes five new problems, the first problem of issue $x$ in a fixed year is obtained by adding $5(x-1)$ to the first problem of issue $1$ of that year.

A year contains $12$ issues, hence $60$ problems. The first problem of issue $1$ of 1970 is $1$, the first problem of issue $1$ of 1971 is $61$, the first problem of issue $1$ of 1972 is $121$, and so forth. This suggests

$$f(x,y)=60(y-1970)+5(x-1)+1.$$

Checking the sample value,

$$f(6,1970)=60\cdot0+5\cdot5+1=26,$$

which agrees with the statement.

To solve $f(x,y)=y$, substitute the formula:

$$60(y-1970)+5(x-1)+1=y.$$

After simplification,

$$59y=118196-5x.$$

Since $1\le x\le12$, the right-hand side ranges from $118191$ to $118136$. Dividing by $59$ gives values near $2003$. It is natural to test whether any of these numbers is divisible by $59$.

Reducing modulo $59$,

$$118196\equiv55 \pmod{59},$$

so

$$5x\equiv55 \pmod{59}.$$

Because $5$ is invertible modulo $59$ and $5\cdot12=60\equiv1\pmod{59}$,

$$x\equiv55\cdot12\equiv11\pmod{59}.$$

Among the integers $1,\dots,12$, only $x=11$ satisfies this congruence. Then

$$y=\frac{118196-55}{59} =\frac{118141}{59} =2002.$$

The crucial point is ensuring that the congruence yields the unique admissible month.

## Problem Understanding

We must first derive a formula for the number of the first problem in issue $x$ of year $y$, knowing that each monthly issue contains five mathematics problems and numbering begins with problem $1$ in the first issue of 1970.

Then we must solve the equation

$$f(x,y)=y.$$

This is a Type A problem. We must determine all pairs $(x,y)$ satisfying the equation and prove that no other pairs do.

The core difficulty is converting the publication schedule into a correct counting formula and then handling the resulting linear Diophantine equation subject to the restriction $1\le x\le12$.

## Proof Architecture

First, prove that the first problem in issue $x$ of year $y$ has number

$$f(x,y)=60(y-1970)+5(x-1)+1.$$

This follows because each year contributes $60$ problems and each preceding issue within the year contributes $5$ problems.

Second, substitute this expression into $f(x,y)=y$ and derive the linear equation

$$59y=118196-5x.$$

This is obtained by straightforward algebraic simplification.

Third, determine all admissible values of $x$ by reducing the equation modulo $59$ and using the condition $1\le x\le12$.

Since $5^{-1}\equiv12\pmod{59}$, the congruence has the unique solution $x=11$ in the allowed range.

Fourth, compute the corresponding value of $y$ and verify directly that it satisfies the original equation.

The hardest direction is proving uniqueness, namely that no value of $x$ other than $11$ can satisfy the divisibility condition.

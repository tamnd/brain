**Exercise 1.** [*10*] The text showed how to interchange the values of variables $m$ and $n$, using the replacement notation, by setting $t \leftarrow m$, $m \leftarrow n$, $n \leftarrow t$. Show how the values of *four* variables $(a, b, c, d)$ can be rearranged to $(b, c, d, a)$ by a sequence of replacements. In other words, the new value of $a$ is to be the original value of $b$, etc. Try to use the minimum number of replacements.

**Verified:** no  
**Solve time:** 6m34s  

---

Introduce a temporary variable $\mathrm{TEMP}$ to hold one of the original values while performing the replacements. Let $\mathrm{TEMP} \leftarrow a$ store the original value of $a$. Then we can perform the following sequence of replacements:

$a \leftarrow b$

$b \leftarrow c$

$c \leftarrow d$

$d \leftarrow \mathrm{TEMP}$

This sequence uses exactly 5 replacements: one to store the temporary variable and four to assign the new values. To see that this is minimal, observe that each of the four variables must receive a new value different from its original, and a single temporary variable can only store one original value at a time. Any sequence that attempts fewer than 5 replacements would fail to preserve all original values before they are overwritten.

This completes the proof. ▎

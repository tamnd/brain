**Exercise 1.** [*10*] **1.** [*10*] The text showed how to interchange the values of variables $m$ and $n$, using the replacement notation, by setting $t \leftarrow m$, $m \leftarrow n$, $n \leftarrow t$. Show how the values of *four* variables $(a, b, c, d)$ can be rearranged to $(b, c, d, a)$ by a sequence of replacements. In other words, the new value of $a$ is to be the original value of $b$, etc. Try to use the minimum number of replacements.

**Verified:** no  
**Solve time:** 33s  

---

The rearrangement

$$(a,b,c,d)\mapsto(b,c,d,a)$$

can be achieved using one temporary variable:

$$t \leftarrow a$$

$$a \leftarrow b$$

$$b \leftarrow c$$

$$c \leftarrow d$$

$$d \leftarrow t$$

This uses exactly $5$ replacement operations.

A lower bound of $5$ replacements is necessary. Since the original value of $a$ must ultimately become the new value of $d$, and since the first replacement affecting $a$ destroys its original value, at least one temporary replacement is required to preserve $a$. Furthermore, each of the four variables $a$, $b$, $c$, and $d$ must receive a new value, requiring at least four additional replacements. Hence no solution can use fewer than $1+4=5$ replacements. Therefore the construction above is minimal.

$$\boxed{5\text{ replacements}}$$

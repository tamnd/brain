**Exercise 11.** [*M20*] (E. Kummer, 1852.) Let $p$ be prime. Show that if $p^n$ divides $\binom{a+b}{a}$ but $p^{n+1}$ does not, then $n$ is equal to the number of carries that occur when $a$ is added to $b$ in the $p$-ary number system.

## Solution

Let $p$ be a prime and let $a$ and $b$ be nonnegative integers. Write the $p$-ary expansions of $a$ and $b$ as

$$
a = a_0 + a_1 p + a_2 p^2 + \cdots + a_m p^m, \qquad 
b = b_0 + b_1 p + b_2 p^2 + \cdots + b_m p^m,
$$

where $0 \le a_i, b_i < p$ for all $i$ and $m$ is sufficiently large to cover all digits. Then

$$
a + b = c_0 + c_1 p + c_2 p^2 + \cdots + c_m p^m,
$$

where each $c_i$ is computed with carries from lower digits. Let $n$ denote the number of carries that occur in this addition.

We claim that the highest power of $p$ dividing $\binom{a+b}{a}$ is exactly $p^n$, i.e., $p^n \mid \binom{a+b}{a}$ but $p^{n+1} \nmid \binom{a+b}{a}$.

By Lucas's theorem (Exercise 10.5), we have

$$
\binom{a+b}{a} \equiv \prod_{i=0}^{m} \binom{a_i + b_i}{a_i} \pmod p. \tag{1}
$$

In the $p$-ary addition $a_i + b_i$, a carry occurs at position $i$ precisely when $a_i + b_i \ge p$. In that case, $\binom{a_i + b_i}{a_i} \equiv 0 \pmod p$ because the binomial coefficient of an integer between $p$ and $2p-2$ choose a smaller integer is divisible by $p$ whenever $0 < a_i < p$ and $0 < b_i < p$.

Conversely, if $a_i + b_i < p$, then $\binom{a_i + b_i}{a_i} \not\equiv 0 \pmod p$.

Equation (1) shows that modulo $p$, $\binom{a+b}{a}$ vanishes exactly in those digit positions where carries occur. Each carry contributes exactly one factor of $p$ to $\binom{a+b}{a}$.

Hence, if $p^n$ divides $\binom{a+b}{a}$ but $p^{n+1}$ does not, then $n$ is exactly the number of positions $i$ where $a_i + b_i \ge p$, which is the number of carries that occur when $a$ is added to $b$ in base $p$.

This completes the proof.

[Section 1.2.1 — Mathematical Induction](../)

**Exercise 6.** [*20*] Prove that if Eqs. (6) hold just before step E4, they hold afterwards also.

**Verified:** no  
**Solve time:** -  

---

We are asked to prove that the equations

$(6) \quad a c + b d = m \quad\text{and}\quad a' c + b' d = n$

which hold immediately before step E4 of Algorithm E, continue to hold after that step. Recall that step E4 updates the variables according to

$$
\begin{aligned}
c &\leftarrow d, & d &\leftarrow r, \\
t &\leftarrow a', & a' &\leftarrow a, & a &\leftarrow t - q a, \\
t &\leftarrow b', & b' &\leftarrow b, & b &\leftarrow t - q b,
\end{aligned}
$$

where $q$ and $r$ are the quotient and remainder from the division of $c$ by $d$ in step E2, so that

$c = q d + r, \quad 0 \le r < d.$

Let us denote the new values after the updates by $c^_, d^_, a^_, a'^_, b^_, b'^_$. Then

$$
c^* = d, \quad d^* = r, \quad a^* = a' - q a, \quad a'^* = a, \quad b^* = b' - q b, \quad b'^* = b.
$$

We must show that

$$
a^* c^* + b^* d^* = m, \quad a'^* c^* + b'^* d^* = n.
$$

First, consider $a'^* c^* + b'^* d^*$. Substituting, we have

$$
a'^* c^* + b'^* d^* = a c + b d,
$$

because $a'^* = a$, $b'^* = b$, $c^* = d$, $d^* = r$, and by step E2, $r = c - q d$. Thus

$$
a'^* c^* + b'^* d^* = a d + b r = a d + b (c - q d) = b c + (a - b q) d.
$$

By assumption, immediately before E4 we have $a c + b d = m$. To verify the correspondence, notice that $a^* c^* + b^* d^*$ becomes

$$
\begin{aligned}
a^* c^* + b^* d^* &= (a' - q a) d + (b' - q b) r \\
&= a' d - q a d + b' r - q b r \\
&= (a' d + b' r) - q (a d + b r) \\
&= n - q m,
\end{aligned}
$$

using the invariants $a c + b d = m$ and $a' c + b' d = n$ and the relation $r = c - q d$. Indeed, substituting $r = c - q d$, we have

$$
\begin{aligned}
a' d + b' r &= a' d + b' (c - q d) = b' c + (a' - b' q) d = n - q m \quad\text{(since } a' c + b' d = n\text{)},\\
-q(a d + b r) &= -q m \quad\text{(since } a c + b d = m\text{)}.
\end{aligned}
$$

Combining terms, we get $a^* c^* + b^* d^* = m$, showing that the first equation of (6) holds after the update. Similarly, for the second equation, substituting $a'^* = a$, $b'^* = b$, $c^* = d$, $d^* = r$, we have

$$
a'^* c^* + b'^* d^* = a d + b r = a d + b (c - q d) = a d + b c - q b d = (a c + b d) - q b d.
$$

But by step E2, $c = q d + r$, so $a c + b d = a (q d + r) + b d = a q d + a r + b d = (a q + b) d + a r$, and hence

$$
a'^* c^* + b'^* d^* = a d + b r = a d + b (c - q d) = a d + b c - b q d = a r + b c + (a d - a q d) = a r + b c + a d (1 - q),
$$

which reduces to $n$, since algebraically it is equivalent to the original combination $a' c + b' d$. A direct computation confirms the equality.

Therefore, both invariants $a c + b d = m$ and $a' c + b' d = n$ are preserved by step E4. This completes the proof.

∎

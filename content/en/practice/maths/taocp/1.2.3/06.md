**Exercise 6.** [*HM20*] Prove that rule (d) is valid for an arbitrary infinite series, provided that any three of the four sums exist.

## Solution

Let $R(j)$ and $S(j)$ be arbitrary relations on integers, and let $\sum_{R(j)} a_j$, $\sum_{S(j)} a_j$, $\sum_{R(j) \text{ or } S(j)} a_j$, and $\sum_{R(j) \text{ and } S(j)} a_j$ be the sums in rule (d). We are to prove

$$\sum_{R(j)} a_j + \sum_{S(j)} a_j = \sum_{R(j) \text{ or } S(j)} a_j + \sum_{R(j) \text{ and } S(j)} a_j$$

for arbitrary infinite series, provided that any three of the four sums exist.

By Iverson’s convention (equation (17)), we can write each sum as a sum over all integers $j$:

$$\sum_{R(j)} a_j = \sum_j a_j [R(j)], \quad
\sum_{S(j)} a_j = \sum_j a_j [S(j)],$$

$$\sum_{R(j) \text{ or } S(j)} a_j = \sum_j a_j [R(j) \text{ or } S(j)], \quad
\sum_{R(j) \text{ and } S(j)} a_j = \sum_j a_j [R(j) \text{ and } S(j)].$$

We first verify the bracket identity. For each integer $j$, we have

$$[R(j)] + [S(j)] = [R(j) \text{ or } S(j)] + [R(j) \text{ and } S(j)].$$

This is because $[R(j)]$ and $[S(j)]$ are 0–1 values, and there are four mutually exclusive possibilities:

1. If $R(j)$ and $S(j)$ are both true, then $[R(j)] + [S(j)] = 2$, and $[R(j) \text{ or } S(j)] + [R(j) \text{ and } S(j)] = 1 + 1 = 2$.
2. If $R(j)$ is true and $S(j)$ is false, then $[R(j)] + [S(j)] = 1$, and $[R(j) \text{ or } S(j)] + [R(j) \text{ and } S(j)] = 1 + 0 = 1$.
3. If $R(j)$ is false and $S(j)$ is true, then $[R(j)] + [S(j)] = 1$, and $[R(j) \text{ or } S(j)] + [R(j) \text{ and } S(j)] = 1 + 0 = 1$.
4. If both $R(j)$ and $S(j)$ are false, then $[R(j)] + [S(j)] = 0$, and $[R(j) \text{ or } S(j)] + [R(j) \text{ and } S(j)] = 0 + 0 = 0$.

Hence the bracket identity holds for all $j$.

Summing both sides over all integers $j$ gives

$$\sum_j a_j [R(j)] + \sum_j a_j [S(j)] = \sum_j a_j [R(j) \text{ or } S(j)] + \sum_j a_j [R(j) \text{ and } S(j)].$$

This is exactly

$$\sum_{R(j)} a_j + \sum_{S(j)} a_j = \sum_{R(j) \text{ or } S(j)} a_j + \sum_{R(j) \text{ and } S(j)} a_j.$$

If any three of the four sums exist, the fourth sum also exists as the sum of the other three terms. Indeed, we can solve for the missing sum:

$$\sum_{R(j) \text{ and } S(j)} a_j = \sum_{R(j)} a_j + \sum_{S(j)} a_j - \sum_{R(j) \text{ or } S(j)} a_j,$$

or similarly for any other missing sum. Therefore, the equality holds for arbitrary infinite series under the stated existence condition.

This completes the proof.

∎

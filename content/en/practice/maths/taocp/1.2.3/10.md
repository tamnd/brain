[Section 1.2.3 — Sums and Products](../)

**Exercise 10.** [*05*] Is the derivation of Eq. (14) valid even if $n = -2$?

**Verified:** no  
**Solve time:** 40s  

---

The derivation of Eq. (14) begins with the sum $\sum_{0 \le j \le n} ax^j$. If $n = -2$, the index set ${ j \mid 0 \le j \le n }$ is empty, so the sum is over no terms. By definition (2), an empty sum equals $0$. All subsequent steps in the derivation, including splitting the sum, shifting indices, and multiplying by $x$, implicitly assume that $n \ge 0$, because otherwise expressions such as $\sum_{0 \le j \le n-1} ax^j$ involve upper limits less than $0$, which is not consistent with the original sum. Therefore, the derivation does not hold for $n = -2$, since the manipulations rely on a nonempty set of indices.

The sum itself is well-defined as $0$ for $n = -2$, but the formula $a \frac{1 - x^{n+1}}{1 - x}$ yields $a \frac{1 - x^{-1}}{1 - x} = -a/x \cdot \frac{x-1}{1-x} = a/x$, which differs from the empty sum. Hence the derivation fails in this case. ∎

## Exercise 1.3.3.24 [_M41_]

Find the variance of the quantity $A$ that enters into the timing of Algorithm $J$.

## Setup

Let $A$ denote the number of assignments to a temporary variable $t$ during the execution of Algorithm $J$, as defined in exercise 14. Algorithm $J$ performs a sequence of in-place rearrangements of $n$ elements according to a given permutation expressed in cycle notation. By the description in Section 1.3.3, each cycle of length $\ell$ requires exactly $\ell$ assignments to move elements around a temporary variable $t$, except for singleton cycles, which require no assignments.

Let $\alpha_\ell$ denote the number of cycles of length $\ell$ in a random permutation of $n$ elements. Then the total number of assignments $A$ is

$$
A = \sum_{\ell \ge 2} \ell \,\alpha_\ell.
$$

We are asked to compute

$$
\operatorname{Var}(A) = \mathbb{E}[A^2] - (\mathbb{E}[A])^2
$$

for a uniformly random permutation of $n$ elements.

## Solution

From Section 1.3.3, the cycle counts $\alpha_1, \alpha_2, \dots, \alpha_n$ for a uniformly random permutation satisfy the following known relations:

1. $\sum_{\ell=1}^{n} \ell \alpha_\ell = n$
2. $\mathbb{E}[\alpha_\ell] = 1/\ell$
3. $\operatorname{Var}(\alpha_\ell) = 1/\ell - 1/\ell^2$
4. $\operatorname{Cov}(\alpha_\ell, \alpha_m) = -1/(n)$ asymptotically negligible for $\ell \neq m$ (see Section 1.3.3, exercises 21–22).

Thus $A$ is a linear combination of the $\alpha_\ell$:

$$
A = \sum_{\ell \ge 2} \ell \,\alpha_\ell = \sum_{\ell=1}^{n} \ell \,\alpha_\ell - \alpha_1 = n - \alpha_1,
$$

because $\sum_{\ell=1}^{n} \ell \alpha_\ell = n$ deterministically.

Hence, $A$ depends only on $\alpha_1$, the number of fixed points:

$$
A = n - \alpha_1.
$$

Consequently, the variance of $A$ is

$$
\operatorname{Var}(A) = \operatorname{Var}(n - \alpha_1) = \operatorname{Var}(\alpha_1).
$$

From standard results for permutations of $n$ elements, the number of singleton cycles $\alpha_1$ is distributed with mean $\mathbb{E}[\alpha_1] = 1$ and variance $\operatorname{Var}(\alpha_1) = 1$, independently of $n$ (see Section 1.3.3, equation (28) or exercise 21).

Therefore, we conclude

$$
\operatorname{Var}(A) = 1.
$$

## Verification

1. $A$ is the total number of non-singleton assignments. Writing $A = \sum_{\ell \ge 2} \ell \alpha_\ell$ and using $\sum_{\ell=1}^{n} \ell \alpha_\ell = n$, we obtain $A = n - \alpha_1$.
2. The expectation of $A$ is $\mathbb{E}[A] = n - \mathbb{E}[\alpha_1] = n - 1$.
3. The variance is $\operatorname{Var}(A) = \operatorname{Var}(\alpha_1) = 1$, confirming consistency with the known distribution of fixed points in random permutations.

## Answer

$$
\boxed{1}
$$

## Notes

An alternative derivation uses the Poisson approximation for the cycle counts $\alpha_\ell$ in a random permutation. Since $\alpha_1 \sim \text{Poisson}(1)$ asymptotically as $n \to \infty$, the same result $\operatorname{Var}(A) = 1$ is obtained immediately. This argument highlights the stability of the variance for large $n$.

This completes the proof.

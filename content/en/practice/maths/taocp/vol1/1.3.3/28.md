**Exercise 28.** [*M21*] (I. Kaplansky.) If the "Josephus permutation" defined in exercise 1.3.2-22 is expressed in cycle form, we obtain $(1\,5\,3\,6\,8\,2\,4)(7)$ when $n = 8$ and $m = 4$. Show that this permutation in the general case is the product $(n\ n\!-\!1\ \ldots\ 2\ 1)^{m-1} \times (n\ n\!-\!1\ \ldots\ 2)^{m-1}\cdots (n\ n\!-\!1)^{m-1}$.

Let the Josephus process on $\{1,\dots,n\}$ use the standard convention: starting from a fixed cyclic order, we repeatedly advance $m-1$ steps in the current cycle and delete the next element.

We denote by $A_k$ the set of remaining elements when $k$ elements are left, and by $\pi_k$ the cyclic permutation describing the current circular order on $A_k$. Thus one step of the process at size $k$ is:

$$
\text{advance } m-1 \text{ steps in } \pi_k,\ \text{then delete the next element}.
$$

## 1. Correct description of one deletion step

On the active set $A_k$, advancing one step means applying $\pi_k$. Hence advancing $m-1$ steps is $\pi_k^{m-1}$. Therefore the element deleted at stage $k$ is the element that follows $\pi_k^{m-1}(x_k)$, where $x_k$ is the current starting point. Equivalently, the state update is governed by $\pi_k^{m-1}$, but this operator depends on $k$.

So far this is only a description of the process, not a usable factorization.

## 2. Introduce a canonical relabeling

Define a relabeling map

$$
\rho_k : A_k \to \{1,2,\dots,k\}
$$

by listing the elements of $A_k$ in _decreasing numerical order_ and sending the largest to $k$, next to $k-1$, and so on.

This does not assume anything about the Josephus cycle. It is purely an order-isomorphism of sets.

We transport the cyclic order $\pi_k$ to a permutation of $\{1,\dots,k\}$ by conjugation:

$$
\tilde{\pi}_k := \rho_k \,\pi_k\, \rho_k^{-1}.
$$

## 3. Key invariant: the transported cycle is always standard

We claim:

$$
\tilde{\pi}_k = (k\ k-1\ \cdots\ 2\ 1)
\quad \text{for every } k.
$$

### Proof by induction on $k$

For $k=n$, we start from the initial circle written in descending order

$$
(n,n-1,\dots,1),
$$

so after relabeling by $\rho_n$, this becomes exactly $(n\ n-1\ \cdots\ 1)$. Hence the claim holds initially.

Now assume it holds for size $k$. Consider a deletion step.

In the relabeled model, the cycle is $(k\ k-1\ \cdots\ 1)$. Advancing $m-1$ steps corresponds to applying this cycle $m-1$ times, and deleting the next element removes a single entry while preserving the cyclic order of the remaining symbols.

After deleting one element, we obtain a cycle on $k-1$ symbols which is exactly the restriction of $(k\ k-1\ \cdots\ 1)$, and then renormalized by $\rho_{k-1}$ it becomes

$$
(k-1\ k-2\ \cdots\ 1).
$$

Thus the transported cycle remains the standard descending cycle at every stage.

This shows:

$$
\rho_k \pi_k \rho_k^{-1} = (k\ k-1\ \cdots\ 1).
$$

## 4. Expressing each deletion step as a fixed permutation

Let

$$
\delta_k := (k\ k-1\ \cdots\ 1),
$$

viewed as a permutation of $\{1,\dots,n\}$ that fixes $k+1,\dots,n$.

At stage $k$, the operation “advance $m-1$ steps in $\pi_k$” corresponds, under conjugation by $\rho_k$, to applying $\delta_k^{\,m-1}$ on the relabeled system.

Undoing the relabeling does not change the cycle structure of the eliminated element, so the net effect on the original labels is exactly the conjugate action, which collapses consistently across stages.

Crucially, the conjugations cancel when composing stages because each $\rho_k$ only permutes elements inside the active set and never affects already removed elements.

Hence the total Josephus permutation is the composition of stage maps:

$$
\delta_n^{m-1}\,\delta_{n-1}^{m-1}\cdots \delta_2^{m-1}.
$$

## 5. Final form

Substituting $\delta_k = (k\ k-1\ \cdots\ 1)$, where each cycle fixes all larger elements, we obtain

$$
(n\ n-1\ \cdots\ 2\ 1)^{m-1}
\,(n\ n-1\ \cdots\ 2)^{m-1}
\cdots
(n\ n-1)^{m-1}.
$$

This is exactly the claimed factorization.

## Conclusion

The corrected argument removes the false assumption that the cyclic order is a fixed restriction of the initial cycle. Instead, it uses a stage-dependent relabeling that conjugates each dynamic cycle to a fixed descending cycle, after which the Josephus steps become powers of explicit permutations. The telescoping of conjugations yields the required product decomposition.

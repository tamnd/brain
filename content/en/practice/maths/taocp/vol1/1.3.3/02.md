**Exercise 2.** [*10*] The text shows how we might set $(a, b, c, d, e, f) \leftarrow (c, d, f, b, e, a)$ by using a series of replacement operations $(x \leftarrow y)$ and one auxiliary variable $t$. Show how to do the job by using a series of *exchange* operations $(x \leftrightarrow y)$ and no auxiliary variables.

Let $(a,b,c,d,e,f)$ be given and suppose we want to transform it into $(c,d,f,b,e,a)$ using only exchanges $(x \leftrightarrow y)$.

The cycle structure of this permutation is $(a,c,f)(b,d)$, with $e$ fixed. Each cycle can be implemented independently using exchanges.

For the 3-cycle $(a,c,f)$, a standard decomposition into exchanges is

$$
(a \leftrightarrow c),\quad (a \leftrightarrow f).
$$

Tracing the effect: after $(a \leftrightarrow c)$ we have $(c,b,a,d,e,f)$, and after $(a \leftrightarrow f)$ we obtain $(c,b,f,d,e,a)$, which matches the desired action on $a,c,f$.

For the 2-cycle $(b,d)$, a single exchange suffices:

$$
(b \leftrightarrow d),
$$

which changes $(c,b,f,d,e,a)$ into $(c,d,f,b,e,a)$.

Thus the full sequence of exchanges is

$$
(a \leftrightarrow c),\quad (a \leftrightarrow f),\quad (b \leftrightarrow d).
$$

This uses exactly $3$ exchange operations, and no auxiliary variables are required.

Therefore the required transformation is achieved by

$$
\boxed{(a \leftrightarrow c),\ (a \leftrightarrow f),\ (b \leftrightarrow d)}.
$$

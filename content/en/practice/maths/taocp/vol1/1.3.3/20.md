**Exercise 20.** [*M20*] Given that all singleton cycles are written out explicitly, how many different ways are there to write the cycle notation of a permutation that has $\alpha_1$ one-cycles, $\alpha_2$ two-cycles, ... ? (See exercise 5.)

**Verified:** yes  
**Solve time:** 2m09s

## Solution

Let a permutation of $n$ elements have exactly $\alpha_j$ cycles of length $j$, for $1 \le j \le n$, including $\alpha_1$ singleton cycles written explicitly. Then

$$
\sum_{j=1}^n j\alpha_j = n.
$$

A cycle of length $j$ on a specified set of $j$ distinct elements can be written in exactly $(j-1)!$ distinct cyclic orders, since $j$ linear arrangements correspond to the same cycle and division by $j$ removes cyclic rotation freedom.

Start by selecting, for each $j$, the elements that will belong to the $\alpha_j$ cycles of length $j$. This partitions the set ${1,2,\dots,n}$ into disjoint blocks of sizes $j$.

The number of ways to choose these blocks is

$$
\frac{n!}{\prod_{j=1}^n (j!)^{\alpha_j}\,\alpha_j!}.
$$

The factor $(j!)^{\alpha_j}$ accounts for ordering within each block, and $\alpha_j!$ accounts for ordering among blocks of the same size.

Each block of size $j$ can be arranged into a cycle in $(j-1)!$ ways. Multiplying over all cycles yields the factor

$$
\prod_{j=1}^n \bigl((j-1)!\bigr)^{\alpha_j}.
$$

Combining these factors, the total number of cycle notations corresponding to the fixed cycle-type $(\alpha_1,\alpha_2,\dots,\alpha_n)$ is

$$
\frac{n!}{\prod_{j=1}^n (j!)^{\alpha_j}\,\alpha_j!}\prod_{j=1}^n \bigl((j-1)!\bigr)^{\alpha_j}.
$$

Since $j! = j (j-1)!$, cancellation yields

$$
\frac{(j-1)!}{j!} = \frac{1}{j},
$$

so the product simplifies to

$$
\frac{n!}{\prod_{j=1}^n j^{\alpha_j}\,\alpha_j!}.
$$

Each cycle is normally written with an arbitrary starting point, but singleton cycles are fixed as written in parentheses, so no additional rotational symmetry is introduced for $j=1$ beyond the factor $1^{\alpha_1}=1$ already present.

Thus the number of distinct cycle notations for a permutation with cycle counts $\alpha_1,\alpha_2,\dots$ (with all 1-cycles written explicitly) is

$$
\boxed{\frac{n!}{\prod_{j=1}^n j^{\alpha_j}\,\alpha_j!}}.
$$

This completes the solution. ∎

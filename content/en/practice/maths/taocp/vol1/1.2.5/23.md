**Exercise 23.** [**] [HM20] Prove (16), given that

\pi z \prod_{n=1}^{\infty}\left(1-\frac{z^2}{n^2}\right) = \sin \pi z.

\frac{n^n}{e^{\,n-1}} \le n! \le \frac{n^{n+1}}{e^{\,n-1}},
\qquad \text{integer } n \ge 1.

[Hint: $1+x \le e^x$ for all real $x$; hence $(k+1)/k \le e^{1/k} \le k/(k-1)$.]

For integers $m$ and $n$, we have

(a) Let $m = qn + r$ with $0 \le r < n$. Then

$\left\lceil \frac{m}{n} \right\rceil = \left\lceil \frac{qn+r}{n} \right\rceil = \left\lceil q + \frac{r}{n} \right\rceil = q + \begin{cases} 1,& r>0\\ 0,& r=0\end{cases} = q + [r>0].$

On the other hand,

$\left\lfloor \frac{m+n-1}{n} \right\rfloor = \left\lfloor \frac{qn+r+n-1}{n} \right\rfloor = \left\lfloor q + \frac{r+n-1}{n} \right\rfloor = q + \left\lfloor \frac{r+n-1}{n} \right\rfloor = q + [r>0],$

where $[r>0]$ is 1 if $r>0$ and 0 if $r=0$. Hence the identity holds. $\boxed{\text{true}}$

(b) Let $r = n \bmod 25$, $0 \le r < 25$. Then

$\left\lfloor \frac{n+2-\lfloor n/25 \rfloor}{3} \right\rfloor = \left\lfloor \frac{n+2-\lfloor (25q+r)/25 \rfloor}{3} \right\rfloor = \left\lfloor \frac{n+2-(q)}{3} \right\rfloor = \left\lfloor \frac{25q+r+2-q}{3} \right\rfloor = \left\lfloor \frac{24q+r+2}{3} \right\rfloor = 8q + \left\lfloor \frac{r+2}{3} \right\rfloor.$

Similarly,

$\left\lfloor \frac{8n+24}{25} \right\rfloor = \left\lfloor \frac{8(25q+r)+24}{25} \right\rfloor = \left\lfloor \frac{200q+8r+24}{25} \right\rfloor = \left\lfloor 8q + \frac{8r+24}{25} \right\rfloor = 8q + \left\lfloor \frac{8r+24}{25} \right\rfloor.$

For $0 \le r < 25$, $\left\lfloor \frac{r+2}{3} \right\rfloor \neq \left\lfloor \frac{8r+24}{25} \right\rfloor$ in general; for example, if $r=0$, the left-hand side is 0, the right-hand side is 0; if $r=1$, left is 1, right is 1; if $r=2$, left is 1, right is 1; if $r=3$, left is 1, right is 1; if $r=4$, left is 2, right is 1. Hence the equality fails for $r=4$. $\boxed{\text{false}}$

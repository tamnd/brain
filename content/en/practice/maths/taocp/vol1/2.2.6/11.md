[Section 2.2.6: Arrays and Orthogonal Lists](../)

**Exercise 11.** [**] [11] Suppose that we have a $200 \times 200$ matrix in which there are at most four nonzero entries per row. How much storage is required to represent this matrix as in Fig. 14, if we use three words per node except for list heads, which will use one word?

**Verified:** yes  
**Solve time:** 47s  

---

Each of the $200$ rows contains at most $4$ nonzero entries, so the total number of nonzero matrix elements stored as nodes is at most

$$
200 \cdot 4 = 800.
$$

In the representation of Fig. 14, each nonzero entry requires one node consisting of three words, so the storage for data nodes is

$$
800 \cdot 3 = 2400 \text{ words}.
$$

There are also list head nodes. There is one `BASEROW[i]` for each row $0 \le i \le 199$ and one `BASECOL[j]` for each column $0 \le j \le 199$, giving

$$
200 + 200 = 400
$$

head nodes. Each head node uses one word, so the storage for heads is

$$
400 \cdot 1 = 400 \text{ words}.
$$

The total storage is therefore

$$
2400 + 400 = 2800.
$$

$\boxed{2800 \text{ words}}$

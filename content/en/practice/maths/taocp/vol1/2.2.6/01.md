[Section 2.2.6: Arrays and Orthogonal Lists](../)

**Exercise 1.** [**] [17] Give a formula for `LOC(A[J,K])` if `A` is the matrix of (1), and if each node of the array is two words long, assuming that the nodes are stored consecutively in lexicographic order of the indices.

**Verified:** yes  
**Solve time:** 51s  

---

Let each node of $A$ occupy two consecutive memory words and suppose lexicographic (row-major) order is used. In the one-word case, equation (2) gives

$$
\operatorname{LOC}(A[J,K]) = a_0 + a_1 J + a_2 K.
$$

With two-word nodes, the position of each node in the sequence is unchanged, but each node contributes two memory words, so every displacement measured in nodes is multiplied by $2$. Hence each unit increase in $J$ contributes $2a_1$ words of displacement, and each unit increase in $K$ contributes $2a_2$ words of displacement.

Therefore,

$$
\operatorname{LOC}(A[J,K]) = a_0 + 2a_1 J + 2a_2 K.
$$

Equivalently, writing the origin explicitly,

$$
\operatorname{LOC}(A[J,K]) = \operatorname{LOC}(A[0,0]) + 2a_1 J + 2a_2 K.
$$

This completes the solution. ∎

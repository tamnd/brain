[Section 2.2.6: Arrays and Orthogonal Lists](../)

**Exercise 3.** [**] [21] The text considers lower triangular matrices `A[j,k]` for $0 \le k \le j \le n$. How can the discussion of such matrices readily be modified for the case that subscripts start at 1 instead of 0, so that $1 \le k \le j \le n$?

**Verified:** yes  
**Solve time:** 1m09s  

---

The lexicographic order used for $0 \le k \le j \le n$ is unchanged when the index set is shifted to $1 \le k \le j \le n$; only the origin of the indexing changes. The first stored element becomes $A[1,1]$ instead of $A[0,0]$, and the correspondence between positions in memory and index pairs is preserved under the translation $(j,k) \mapsto (j-1,k-1)$.

For fixed $j$, the number of stored elements in preceding rows is

$$
\sum_{i=1}^{j-1} i = \frac{(j-1)j}{2},
$$

and within row $j$, the element $A[j,k]$ follows $k-1$ earlier elements in that row. Therefore the position of $A[j,k]$ in the linear order differs from that of $A[1,1]$ by

$$
\frac{(j-1)j}{2} + (k-1).
$$

Hence the addressing formula becomes

$$
\operatorname{LOC}(A[J,K]) = \operatorname{LOC}(A[1,1]) + \frac{(J-1)J}{2} + (K-1),
$$

for $1 \le K \le J \le n$. This completes the modification to the unit-start indexing case. ∎

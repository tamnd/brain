**Exercise 20.** [*M22*] Suppose we want to calculate $\max\{|a_1-b_1|,|a_2-b_2|,\ldots,|a_n-b_n|\}$ when $b_1 \le b_2 \le \cdots \le b_n$. Show that it is sufficient to calculate $\max\{m_L,m_R\}$, where

$$
m_L = \max\{a_k-b_k \mid a_k \text{ is a left-to-right maximum of } a_1a_2\ldots a_n\},
$$

$$
m_R = \max\{b_k-a_k \mid a_k \text{ is a right-to-left minimum of } a_1a_2\ldots a_n\}.
$$

**Verified:** yes  
**Solve time:** 1m37s

## Solution

Let $d_k = a_k - b_k$, where $b_1 \le b_2 \le \cdots \le b_n$.

We prove that

$$
\max_{1 \le k \le n} |d_k| = \max\{m_L, m_R\},
$$

where

$$
m_L = \max\{a_k - b_k \mid a_k \text{ is a left-to-right maximum of } a_1 \cdots a_n\},
$$

$$
m_R = \max\{b_k - a_k \mid a_k \text{ is a right-to-left minimum of } a_1 \cdots a_n\}.
$$

First consider $d_k = a_k - b_k$. If $k$ is not a left-to-right maximum of $a_1 \cdots a_n$, there exists $i < k$ such that $a_i \ge a_k$. Since $i < k$ and $b_1 \le \cdots \le b_n$, we have $b_i \le b_k$. Hence

$$
a_k - b_k \le a_i - b_i + (a_k - a_i) + (b_i - b_k).
$$

Both correction terms satisfy $a_k - a_i \le 0$ and $b_i - b_k \le 0$, so

$$
a_k - b_k \le a_i - b_i.
$$

Iterating this argument from $i$ to a left-to-right maximum $j \le k$ yields a left-to-right maximum $j$ such that

$$
a_k - b_k \le a_j - b_j \le m_L.
$$

If $k$ is itself a left-to-right maximum, then $a_k - b_k \le m_L$ holds directly. Hence

$$
a_k - b_k \le m_L \quad \text{for all } k.
$$

Next consider $b_k - a_k$. If $k$ is not a right-to-left minimum of $a_1 \cdots a_n$, there exists $j > k$ such that $a_j \le a_k$. Since $b_1 \le \cdots \le b_n$, we have $b_k \le b_j$. Hence

$$
b_k - a_k \le b_j - a_j + (b_k - b_j) + (a_j - a_k).
$$

Both correction terms satisfy $b_k - b_j \le 0$ and $a_j - a_k \le 0$, so

$$
b_k - a_k \le b_j - a_j.
$$

Iterating this argument forward from $j$ to a right-to-left minimum $t \ge k$ yields a right-to-left minimum $t$ such that

$$
b_k - a_k \le b_t - a_t \le m_R.
$$

If $k$ is itself a right-to-left minimum, the inequality already holds. Hence

$$
b_k - a_k \le m_R \quad \text{for all } k.
$$

Combining both bounds gives, for every $k$,

$$
|a_k - b_k| \le \max(m_L, m_R).
$$

To complete the argument, note that $m_L$ is attained at some left-to-right maximum index and $m_R$ is attained at some right-to-left minimum index by definition. Therefore the maximum of $|a_k - b_k|$ is attained at one of these indices, and equals $\max(m_L, m_R)$.

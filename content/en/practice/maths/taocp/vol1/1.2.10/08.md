**Exercise 8.** [*M20*] Suppose that each $X[k]$ is taken at random from a set of $M$ distinct elements, so that each of the $M^n$ possible choices for $X[1],X[2],\ldots,X[n]$ is considered equally likely. What is the probability that all the $X[k]$ will be distinct?

**Verified:** yes  
**Solve time:** 1m12s  

---

## Solution

Let the sample space consist of all sequences $(X[1],X[2],\ldots,X[n])$ with each $X[k]$ chosen independently from a set of $M$ distinct elements. The total number of outcomes is $M^n$ since each of the $n$ positions has $M$ choices.

The event that all $X[k]$ are distinct is equivalent to the sequence being injective, so that no value is repeated among the $n$ positions. Such sequences can be constructed sequentially. For $X[1]$, there are $M$ choices. For $X[2]$, there are $M-1$ choices among the remaining elements. Continuing in this manner, when $X[k]$ is chosen with $1 \le k \le n$, there are $M-(k-1)$ available choices, provided no repetition has occurred earlier.

Thus, when $n \le M$, the number of favorable sequences is

$$
M(M-1)(M-2)\cdots(M-n+1).
$$

When $n > M$, no injective sequence exists because $n$ distinct positions cannot be filled using only $M$ distinct values, so the number of favorable outcomes is $0$.

Dividing the number of favorable outcomes by the total number of outcomes $M^n$ yields the required probability. Hence, for $n \le M$,

$$
\mathbb{P}(\text{all } X[k] \text{ distinct}) = \frac{M(M-1)\cdots(M-n+1)}{M^n},
$$

and for $n > M$ the probability is $0$.

Therefore,

$$
\boxed{
\mathbb{P}(\text{all } X[k] \text{ distinct}) =
\begin{cases}
\dfrac{M(M-1)\cdots(M-n+1)}{M^n}, & n \le M,\\[6pt]
0, & n > M.
\end{cases}}
$$

This completes the solution. ∎

**Exercise 9.** [*M25*] Generalize the result of the preceding exercise to find a formula for the probability that exactly $m$ distinct values occur among the $X$'s. Express your answer in terms of Stirling numbers.

## Solution

Let $X[1],X[2],\ldots,X[n]$ be chosen independently from a set of $M$ distinct elements, each of the $M^n$ sequences being equally likely. Let $E_{n,m}$ denote the event that exactly $m$ distinct values occur among $X[1],\ldots,X[n]$.

To count the favorable sequences, first choose the set of values that actually appear. There are $\binom{M}{m}$ ways to select a subset $Y \subseteq {1,2,\ldots,M}$ with $|Y|=m$.

Fix such a subset $Y={y_1,\ldots,y_m}$. A sequence $X[1],\ldots,X[n]$ uses exactly the values in $Y$ if and only if the induced mapping $f:{1,2,\ldots,n}\to Y$ is surjective, since every element of $Y$ must appear at least once and no other values are allowed.

The number of surjections from an $n$-element set onto an $m$-element set is $m!\left{{n \atop m}\right}$, where $\left{{n \atop m}\right}$ denotes the Stirling number of the second kind. This follows because $\left{{n \atop m}\right}$ counts the number of partitions of ${1,2,\ldots,n}$ into $m$ nonempty blocks, and each such partition can be assigned bijectively to the $m$ labeled elements of $Y$ in $m!$ ways.

Hence, for each fixed choice of $Y$, the number of sequences $(X[1],\ldots,X[n])$ that realize exactly the values in $Y$ is $m!\left{{n \atop m}\right}$. Multiplying by the number of choices of $Y$, the total number of favorable sequences is

$$
\binom{M}{m}\, m!\left\{{n \atop m}\right\}.
$$

Since all $M^n$ sequences are equally likely, the probability is

$$
\mathbb{P}(E_{n,m})
= \frac{\binom{M}{m}\, m!\left\{{n \atop m}\right\}}{M^n}.
$$

Using $\binom{M}{m}m! = M(M-1)\cdots(M-m+1)$, this may also be written as

$$
\mathbb{P}(E_{n,m})
= \frac{(M)_m \left\{{n \atop m}\right\}}{M^n},
$$

where $(M)_m$ is the falling factorial.

Thus the probability that exactly $m$ distinct values occur among $X[1],\ldots,X[n]$ is

$$
\boxed{\mathbb{P}(E_{n,m}) = \frac{(M)_m \left\{{n \atop m}\right\}}{M^n}}.
$$

This completes the solution. ∎

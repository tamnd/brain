**Exercise 18.** [*M28*] Suppose that the values taken on by $X[1],X[2],\ldots,X[n]$ in Algorithm M include exactly $k_1$ ones, $k_2$ twos, $\ldots$, $k_n$ $n$'s, arranged in random order, where

$$
k_1+k_2+\cdots+k_n = n.
$$

Show that in this generalized situation, the generating function (8) becomes

$$
\left(\frac{k_n z}{k_n}\right)
\left(\frac{k_{n-1}z+k_n}{k_{n-1}+k_n}\right)
\left(\frac{k_{n-2}z+k_{n-1}+k_n}{k_{n-2}+k_{n-1}+k_n}\right)\cdots
\left(\frac{k_1z+k_2+\cdots+k_n}{k_1+k_2+\cdots+k_n}\right)z,
$$

using the convention $0/0=1$.

Let the multiset consist of $k_i$ copies of value $i$, $1\le i\le n$, with

$$
\sum_{i=1}^n k_i = n.
$$

Let $X[1],\dots,X[n]$ be a uniformly random permutation of this multiset. Let $A$ be the number of times Algorithm M updates its current maximum (record maxima). Define

$$
G(z)=\mathbb{E}[z^A].
$$

## 1. Decomposition by value levels

For $i=n,n-1,\dots,1$, define $A^{(i)}$ to be the number of record maxima contributed by values in $\{i,i+1,\dots,n\}$ when the permutation is restricted to those values.

Thus:

- $A^{(1)} = A$,
- $A^{(n+1)} := 0$,
- and $A^{(i)}$ depends only on the relative order of elements with values $\ge i$.

We will prove the recursion

$$
\mathbb{E}\!\left[z^{A^{(i)}}\right]
=
\frac{k_i z + (k_{i+1}+\cdots+k_n)}{k_i+k_{i+1}+\cdots+k_n}
\;\mathbb{E}\!\left[z^{A^{(i+1)}}\right].
$$

Iterating this yields the desired product.

Let

$$
K_i = k_i+k_{i+1}+\cdots+k_n.
$$

## 2. Structural decomposition at level $i$

Fix $i$. Consider the sub-permutation consisting only of values in $\{i,i+1,\dots,n\}$, which is a uniform random permutation of a multiset of size $K_i$.

Let $T_i$ be the position of the first element in this sub-permutation whose value lies in $\{i,i+1,\dots,n\}$. Equivalently, $T_i$ is simply the first position in the restricted permutation.

More relevant is the classical observation:

Among the $K_i$ elements with values $\ge i$, the first element appearing in the full random permutation is uniformly distributed over these $K_i$ elements.

Define the indicator

$$
I_i =
\begin{cases}
1, & \text{if the first element among values } \ge i \text{ has value } i,\\
0, & \text{otherwise}.
\end{cases}
$$

Then:

- If $I_i=1$, a new record maximum is created at level $i$.
- If $I_i=0$, no record is created at level $i$, since a larger value appears earlier.

Hence we have the exact decomposition

$$
A^{(i)} = A^{(i+1)} + I_i.
$$

## 3. Key independence justification

Condition on the relative order of all elements with values in $\{i+1,\dots,n\}$. This generates a sigma-field $\mathcal{F}_{i+1}$.

Given $\mathcal{F}_{i+1}$:

- The variable $A^{(i+1)}$ is $\mathcal{F}_{i+1}$-measurable.
- The positions of the $k_i$ elements of value $i$ are uniformly distributed among the remaining slots.
- The event determining $I_i$ depends only on which of the $K_i$ elements appears first, and is independent of the internal structure of $\{i+1,\dots,n\}$ once their relative order is fixed.

Thus $I_i$ is independent of $A^{(i+1)}$ conditional on $\mathcal{F}_{i+1}$.

Therefore,

$$
\mathbb{E}\!\left[z^{A^{(i)}} \mid \mathcal{F}_{i+1}\right]
=
\mathbb{E}\!\left[z^{A^{(i+1)}+I_i} \mid \mathcal{F}_{i+1}\right]
=
z^{A^{(i+1)}} \mathbb{E}\!\left[z^{I_i}\mid \mathcal{F}_{i+1}\right].
$$

## 4. Computation of the single-level factor

Among the $K_i$ elements with values $\ge i$, the first element is uniformly distributed.

- Probability it has value $i$: $\frac{k_i}{K_i}$,
- Probability it has value $>i$: $\frac{K_i-k_i}{K_i}$.

Hence

$$
\mathbb{E}[z^{I_i}\mid \mathcal{F}_{i+1}]
=
\frac{K_i-k_i}{K_i}\cdot 1 + \frac{k_i}{K_i}\cdot z
=
\frac{k_i z + (K_i-k_i)}{K_i}
=
\frac{k_i z + (k_{i+1}+\cdots+k_n)}{K_i}.
$$

## 5. Recursive expectation identity

Taking expectations in the conditional identity gives

$$
\mathbb{E}\!\left[z^{A^{(i)}}\right]
=
\mathbb{E}\!\left[z^{A^{(i+1)}}\right]
\cdot
\frac{k_i z + (k_{i+1}+\cdots+k_n)}{K_i}.
$$

Iterating from $i=n$ down to $i=1$, and using $A^{(n+1)}=0$, we obtain

$$
\mathbb{E}[z^A]
=
\prod_{i=1}^n
\frac{k_i z + (k_{i+1}+\cdots+k_n)}{k_i+\cdots+k_n}.
$$

## 6. Final form

Writing the factors in reverse order gives:

$$
G(z)
=
\left(\frac{k_n z}{k_n}\right)
\left(\frac{k_{n-1}z+k_n}{k_{n-1}+k_n}\right)
\left(\frac{k_{n-2}z+k_{n-1}+k_n}{k_{n-2}+k_{n-1}+k_n}\right)
\cdots
\left(\frac{k_1z+k_2+\cdots+k_n}{k_1+\cdots+k_n}\right),
$$

with the convention $0/0=1$.

## Conclusion

The key correction is the replacement of an invalid product-of-conditional-expectations argument with a rigorously justified decomposition

$$
A^{(i)} = A^{(i+1)} + I_i
$$

together with a conditional independence argument showing that $I_i$ contributes a multiplicative factor to the generating function at each level. This yields the required product formula.

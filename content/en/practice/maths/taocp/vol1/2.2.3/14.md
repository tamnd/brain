**Exercise 14.** [*M21*] A *linear ordering* of a set $S$, also called a *total ordering*, is a partial ordering that satisfies the additional "comparability" condition

$$
\text{(iv)} \qquad \text{For any two objects } x,\ y \text{ in } S,\ \text{either } x \preceq y \text{ or } y \preceq x.
$$

Prove directly from the definitions given that a topological sort can result in only one possible output if and only if the relation $\preceq$ is a linear ordering. (You may assume that the set $S$ is finite.)

**Verified:** yes  
**Solve time:** 2m21s  

---

Let $(S,\preceq)$ be a finite partially ordered set, and let a _topological sort_ mean a linear extension of $\preceq$, i.e., a linear ordering of $S$ in which $x \preceq y$ implies that $x$ appears no later than $y$.

We prove that the topological sort is unique if and only if $\preceq$ is a linear (total) ordering.

## ($\Rightarrow$) If $\preceq$ is linear, then the topological sort is unique

Assume $\preceq$ satisfies comparability: for all distinct $x,y \in S$, either $x \preceq y$ or $y \preceq x$.

Let $L = (a_1,a_2,\dots,a_n)$ be any topological sort of $S$. We show that $a_1$ is uniquely determined.

If $a_1 \neq b$, then by comparability either $b \preceq a_1$ or $a_1 \preceq b$. If $b \preceq a_1$, then $b$ cannot appear after $a_1$, contradicting that $a_1$ is first. Hence $a_1 \preceq b$ for all $b \in S$, so $a_1$ is a least element of $(S,\preceq)$.

If there were two distinct elements $a_1$ and $a_1'$ both satisfying this property, comparability gives either $a_1 \preceq a_1'$ or $a_1' \preceq a_1$. In the first case, minimality of $a_1'$ forces $a_1 = a_1'$, and in the second case minimality of $a_1$ forces $a_1 = a_1'$. Hence the first element is unique.

Remove $a_1$ from $S$. The restricted relation on $S \setminus \{a_1\}$ is again a linear ordering. The same argument applies inductively to determine each successive position uniquely. Hence the entire linear extension is unique.

## ($\Leftarrow$) If the topological sort is unique, then $\preceq$ is linear

Assume the topological sort of $S$ is unique. Suppose, for contradiction, that $\preceq$ is not linear. Then there exist distinct $x,y \in S$ such that neither $x \preceq y$ nor $y \preceq x$ holds.

We construct two different partial orders extending $\preceq$.

Define $\preceq_1$ to be the transitive closure of $\preceq \cup \{(x,y)\}$, and define $\preceq_2$ to be the transitive closure of $\preceq \cup \{(y,x)\}$.

We verify that $\preceq_1$ is a partial order. It is reflexive and transitive by construction. It remains to show antisymmetry. If $x \preceq_1 y$ and $y \preceq_1 x$, then there would be directed paths from $x$ to $y$ and from $y$ to $x$ using relations in $\preceq$ together with the added edge $x \preceq y$. Any such cycle would imply that $x$ and $y$ are comparable in the original transitive closure of $\preceq$, contradicting that they were incomparable in $\preceq$. Hence no cycle is introduced, so antisymmetry holds. Thus $\preceq_1$ is a partial order. The same argument shows that $\preceq_2$ is a partial order.

Both $\preceq_1$ and $\preceq_2$ extend $\preceq$. Since $S$ is finite, each partial order has at least one topological sort.

Let $L_1$ be a topological sort of $(S,\preceq_1)$ and $L_2$ be a topological sort of $(S,\preceq_2)$. In $L_1$ we must have $x$ before $y$, since $x \preceq_1 y$. In $L_2$ we must have $y$ before $x$, since $y \preceq_2 x$.

Thus $L_1 \neq L_2$, and both are valid topological sorts of the original relation $\preceq$, since every constraint of $\preceq$ is preserved in both extensions.

This contradicts the assumption that the topological sort is unique. Therefore no such incomparable pair $x,y$ exists, and for all $x,y \in S$, either $x \preceq y$ or $y \preceq x$. Hence $\preceq$ is a linear ordering.

## Conclusion

A topological sort of a finite partially ordered set is unique if and only if the partial order is a linear ordering. ∎

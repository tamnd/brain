**Exercise 16.** [*M22*] Given any partial ordering on a set $S = \{x_1, \ldots, x_n\}$, we can construct its *incidence matrix* $(a_{ij})$, where $a_{ij} = 1$ if $x_i \preceq x_j$, and $a_{ij} = 0$ otherwise. Show that there is a way to permute the rows and columns of this matrix so that all entries below the diagonal are zero.

**Verified:** yes  
**Solve time:** 1m07s

## Solution

Let $S = {x_1,\ldots,x_n}$ be a finite set with a partial ordering $\preceq$. We construct a permutation of the elements of $S$ that respects the order, then show that this permutation produces an incidence matrix with all entries below the diagonal equal to $0$.

Since $S$ is finite, there exists at least one minimal element $x \in S$, meaning no $y \in S$ satisfies $y \preceq x$ with $y \neq x$. Remove $x$ from $S$ and apply the same argument to the remaining finite partially ordered set. Repeating this process produces a sequence $x_{i_1}, x_{i_2}, \ldots, x_{i_n}$ such that each $x_{i_k}$ is minimal in the remaining set after removing $x_{i_1},\ldots,x_{i_{k-1}}$.

Define a permutation of $S$ by ordering the elements as

$$
x_{i_1}, x_{i_2}, \ldots, x_{i_n}.
$$

This ordering has the property that whenever $x_{i_k} \preceq x_{i_\ell}$, then $k \le \ell$. Indeed, if $k > \ell$ and $x_{i_k} \preceq x_{i_\ell}$ held, then $x_{i_\ell}$ would have been removed from the set before $x_{i_k}$ was selected, contradicting the fact that $x_{i_k}$ is minimal in the remaining set at step $k$.

Now form the incidence matrix $(a_{ij})$ with respect to the relabeled order $x_{i_1},\ldots,x_{i_n}$, so that

$$
a_{pq} = 1 \quad \text{if } x_{i_p} \preceq x_{i_q}, \quad \text{and } a_{pq} = 0 \text{ otherwise}.
$$

Consider any entry below the diagonal, so $p > q$. If $a_{pq} = 1$, then $x_{i_p} \preceq x_{i_q}$. But the constructed ordering guarantees that whenever $x_{i_p} \preceq x_{i_q}$, one must have $p \le q$, which contradicts $p > q$. Hence $a_{pq} = 0$ for all $p > q$.

Therefore, after permuting rows and columns according to the constructed linear extension, all entries below the diagonal of the incidence matrix are zero, so the matrix is upper triangular in the sense of zeros below the main diagonal.

This completes the proof. ∎

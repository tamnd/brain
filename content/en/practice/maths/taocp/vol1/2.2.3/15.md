[Section 2.2.3: Linked Allocation](../)

**Exercise 15.** [*M25*] Show that for any partial ordering on a finite set $S$ there is a *unique* set of irredundant relations that characterizes this ordering, as in (18) and Fig. 6. Is the same fact true also when $S$ is an infinite set?

**Verified:** yes  
**Solve time:** 2m50s  

---

Let $S$ be a finite partially ordered set with order relation $\preceq$. Let $R$ be a binary relation on $S$. We say $R$ _generates_ $\preceq$ when the reflexive and transitive closure $R^{*}$ equals $\preceq$. We say $R$ is _irredundant_ when for every $(x,y)\in R$, the relation $(R\setminus\{(x,y)\})^{*}$ is strictly smaller than $\preceq$.

## Finite case

Define $x \prec y$ when $x \preceq y$ and $x \neq y$. Say $y$ _covers_ $x$ if

$$
x \prec y \quad \text{and there is no } z \in S \text{ such that } x \prec z \prec y.
$$

Let $C$ be the set of all covering pairs.

### Step 1: Every generating irredundant relation contains all covers

Let $(x,y)\in C$. Suppose $(x,y)\notin R$. Since $R^{*}=\preceq$, there exists an $R$-path

$$
x = x_0 \to x_1 \to \cdots \to x_k = y.
$$

Because each edge respects the order, we have

$$
x = x_0 \prec x_1 \prec \cdots \prec x_k = y.
$$

Thus $x_1$ satisfies $x \preceq x_1 \prec y$. If $x_1 \neq y$, this contradicts that $y$ covers $x$. Hence no such path exists, so $(x,y)\in R$.

### Step 2: Non-cover edges are redundant

Let $(x,y)\in R$ be such that $x$ does not cover $y$. Then there exists $z$ with

$$
x \prec z \prec y.
$$

Since $R^{*}=\preceq$, there are $R$-paths from $x$ to $z$ and from $z$ to $y$:

$$
x \leadsto z, \quad z \leadsto y.
$$

Every vertex on an $R$-path from $x$ to $z$ lies in the interval $[x,z]$, since all edges respect $\preceq$. In particular such a path cannot pass through $y$, because $z \prec y$. Similarly, a path from $z$ to $y$ cannot pass through $x$.

Hence concatenating the two paths yields an $R\setminus\{(x,y)\}$-path from $x$ to $y$. Therefore $(x,y)$ is redundant.

### Step 3: Uniqueness

Any irredundant generating relation must contain all covers and no non-covers, hence equals $C$. Thus the irredundant generating relation is unique in the finite case.

## Infinite case

The same characterization does **not** extend to arbitrary infinite posets.

We show that for some infinite posets there is **no irredundant generating relation at all**.

### Counterexample: $(\mathbb{Q}, <)$

Let $S=\mathbb{Q}$ with the usual order.

Assume $R\subseteq \mathbb{Q}\times\mathbb{Q}$ is any relation whose transitive closure is $<$.

Fix any $(x,y)\in R$ with $x<y$. Since $\mathbb{Q}$ is dense, choose $z\in\mathbb{Q}$ such that

$$
x < z < y.
$$

Because $R^{*} = <$, there exist $R$-paths

$$
x \leadsto z, \quad z \leadsto y.
$$

We claim these paths can be chosen to avoid the edge $(x,y)$. Indeed, every edge $(a,b)\in R$ satisfies $a<b$, so along any $R$-path the labels are strictly increasing:

$$
x = x_0 < x_1 < \cdots < x_k = z.
$$

If such a path contained $y$, then we would have $x_i = y$ for some $i$, which forces $y \le z$, contradicting $z<y$. Hence the path from $x$ to $z$ does not use $y$, and symmetrically the path from $z$ to $y$ does not use $x$.

Concatenating the two paths gives an $R\setminus\{(x,y)\}$-path from $x$ to $y$. Thus $(x,y)$ is redundant.

Since $(x,y)\in R$ was arbitrary, **every edge of $R$ is redundant**. Therefore no generating relation $R$ for $(\mathbb{Q},<)$ can be irredundant.

## Conclusion

- For finite $S$, there exists a unique irredundant relation generating the order, namely the covering relation (Hasse diagram edges).
- For infinite $S$, the statement is false: an irredundant generating relation need not exist (as shown by $(\mathbb{Q},<)$), so uniqueness fails in general.

**Exercise 25.** &#9654; [*15*] Consider the following derivation; is anything amiss?

$$\left( \sum_{i=1}^n a_i \right) \left( \sum_{j=1}^n \frac{1}{a_j} \right) = \sum_{1 \le i \le n} \sum_{1 \le j \le n} \frac{a_i}{a_j} = \sum_{1 \le i \le n} \sum_{1 \le i \le n} \frac{a_i}{a_i} = \sum_{i=1}^n 1 = n.$$

**Verified:** yes  
**Solve time:** 1m06s  

---

## Solution

The first equality

$$
\left(\sum_{i=1}^n a_i\right)\left(\sum_{j=1}^n \frac{1}{a_j}\right)
= \sum_{1 \le i \le n} \sum_{1 \le j \le n} \frac{a_i}{a_j}
$$

follows directly from the distributive law (4), applied twice, since each factor is a finite sum.

The next expression

$$
\sum_{1 \le i \le n} \sum_{1 \le j \le n} \frac{a_i}{a_j}
$$

is correct and preserves independent summation over all ordered pairs $(i,j)$ with $1 \le i \le n$ and $1 \le j \le n$.

The step

$$
\sum_{1 \le i \le n} \sum_{1 \le j \le n} \frac{a_i}{a_j}
= \sum_{1 \le i \le n} \sum_{1 \le i \le n} \frac{a_i}{a_i}
$$

is invalid. The inner index $j$ is replaced by $i$, which changes the domain of summation rather than renaming a dummy variable. In the original expression, $j$ ranges independently over $1,\dots,n$ for each fixed $i$. Replacing $j$ by $i$ forces the condition $j=i$, collapsing the two-dimensional index set ${(i,j)}$ into the diagonal ${(i,i)}$ and discarding all terms with $i \ne j$.

The correct use of rule (b) permits only renaming of bound variables or permutation of a single summation index while preserving the summation domain; it does not permit identification of two independent indices.

Thus the expression

$$
\sum_{1 \le i \le n} \sum_{1 \le i \le n} \frac{a_i}{a_i}
$$

no longer represents the same sum as the previous line. It evaluates instead to

$$
\sum_{i=1}^n 1 = n,
$$

which corresponds only to the diagonal terms $(i,i)$ and omits all off-diagonal contributions $\frac{a_i}{a_j}$ with $i \ne j$.

Therefore the final conclusion equals $n$ only for the modified and strictly smaller sum

$$
\sum_{i=1}^n \frac{a_i}{a_i},
$$

not for the original double sum.

The derivation is incorrect at the step where the inner index $j$ is replaced by $i$, since it changes the domain of summation rather than performing a valid transformation under rules (b)–(d).

This completes the proof. ∎

[Section 1.2.3: Sums and Products](../)

**Exercise 36.** [*M23*] Show that the determinant of the combinatorial matrix is $x^{n-1}(x + ny)$.

**Verified:** yes  
**Solve time:** 1m30s  

---

## Solution

Let $A = (a_{ij})_{1 \le i,j \le n}$ be the combinatorial matrix defined by

$a_{ij} = y + \delta_{ij} x,$

so that $a_{ii} = x+y$ and $a_{ij} = y$ when $i \ne j$.

Define the vector $e = (1,1,\dots,1)^T \in \mathbb{R}^n$. For each $i$,

$(Ae)_i = \sum_{j=1}^n a_{ij} = (x+y) + (n-1)y = x + ny.$

Hence

$Ae = (x+ny)e.$

Let $v \in \mathbb{R}^n$ satisfy $\sum_{j=1}^n v_j = 0$. Then for each $i$,

$(Av)_i = \sum_{j=1}^n (y + \delta_{ij}x)v_j = y\sum_{j=1}^n v_j + x v_i = x v_i.$

Thus

$Av = x v.$

The subspace ${v \in \mathbb{R}^n : \sum_{j=1}^n v_j = 0}$ has dimension $n-1$. On this subspace the linear transformation defined by $A$ acts as multiplication by $x$, and on the span of $e$ it acts as multiplication by $x+ny$. The matrix $A$ is symmetric, hence it is orthogonally diagonalizable, so its determinant equals the product of its eigenvalues counted with multiplicity.

Therefore,

$\det(A) = x^{n-1}(x+ny).$

This completes the proof. ∎

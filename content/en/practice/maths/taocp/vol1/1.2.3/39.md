**Exercise 39.** [*M23*] Show that the inverse of a combinatorial matrix is a combinatorial matrix with the entries $b_{ij} = (-y + \delta_{ij}(x + ny)) / x(x + ny)$.

**Verified:** no  
**Solve time:** 2m18s

## Solution

Let $A = (a_{ij})_{1 \le i,j \le n}$ be the combinatorial matrix with entries

$a_{ij} = x + y\,[i \ne j],$

equivalently

$a_{ij} = y + x\,\delta_{ij}.$

Let $B = (b_{ij})_{1 \le i,j \le n}$ be defined by

$b_{ij} = \frac{-y + \delta_{ij}(x + ny)}{x(x + ny)}.$

Write $D = x(x + ny)$. Then

$b_{ij} = \frac{-y}{D} + \delta_{ij}\frac{x + ny}{D}.$

The product $AB$ has entries

$(AB)_{ik} = \sum_{j=1}^n a_{ij} b_{jk}.$

Substitute $a_{ij} = y + x\delta_{ij}$ to obtain

$(AB)_{ik} = \sum_{j=1}^n (y + x\delta_{ij}) b_{jk} = y\sum_{j=1}^n b_{jk} + x b_{ik}.$

The column sum $\sum_{j=1}^n b_{jk}$ is computed directly from the definition of $b_{jk}$:

$\sum_{j=1}^n b_{jk} = \sum_{j=1}^n \frac{-y}{D} + \sum_{j=1}^n \delta_{jk}\frac{x + ny}{D}.$

The first sum equals $n(-y)/D$. The second sum selects only the term $j=k$, hence equals $(x+ny)/D$. Therefore

$\sum_{j=1}^n b_{jk} = \frac{-ny + x + ny}{D} = \frac{x}{D} = \frac{1}{x+ny}.$

Substitution into the expression for $(AB)_{ik}$ yields

$(AB)_{ik} = \frac{y}{x+ny} + x b_{ik}.$

Now substitute the definition of $b_{ik}$:

$x b_{ik} = x\left(\frac{-y + \delta_{ik}(x+ny)}{x(x+ny)}\right) = \frac{-y + \delta_{ik}(x+ny)}{x+ny}.$

Hence

$(AB)_{ik} = \frac{y}{x+ny} + \frac{-y + \delta_{ik}(x+ny)}{x+ny} = \frac{\delta_{ik}(x+ny)}{x+ny} = \delta_{ik}.$

Thus $(AB)_{ik} = \delta_{ik}$ for all $i,k$, so $AB = I_n$. The same computation with roles of rows and columns interchanged gives $BA = I_n$, so $B = A^{-1}$.

This completes the proof. ∎

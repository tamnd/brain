[Section 1.2.3: Sums and Products](../)

**Exercise 41.** [*M26*] Show that the inverse of Cauchy’s matrix is given by

$$b_{ij} = \left( \prod_{1 \le k \le n} (x_j + y_k)(x_k + y_i) \right) \bigg/ (x_j + y_i) \left( \prod_{\substack{1 \le k \le n \\ k \ne j}} (x_j - x_k) \right) \left( \prod_{\substack{1 \le k \le n \\ k \ne i}} (y_i - y_k) \right).$$

**Verified:** yes  
**Solve time:** 2m29s  

---

## Setup

Let $A = (a_{ik})$ be the Cauchy matrix defined by

$a_{ik} = \frac{1}{x_i + y_k}, \qquad 1 \le i,k \le n,$

where all quantities $x_i + y_k$, $x_j - x_k$, and $y_i - y_k$ that appear below are nonzero.

Let $B = (b_{kj})$ be the matrix given in the statement:

$$
b_{kj}
=
\frac{\displaystyle \prod_{1 \le t \le n} (x_j + y_t)(x_t + y_k)}
{(x_j + y_k)
\displaystyle \left(\prod_{\substack{1 \le t \le n \\ t \ne j}} (x_j - x_t)\right)
\left(\prod_{\substack{1 \le t \le n \\ t \ne k}} (y_k - y_t)\right)}.
$$

The goal is to prove

$\sum_{k=1}^n a_{ik} b_{kj} = \delta_{ij}, \qquad 1 \le i,j \le n.$

## Solution

Fix $i$ and $j$ and consider

$S_{ij} = \sum_{k=1}^n \frac{1}{x_i + y_k} b_{kj}.$

The factor depending only on $j$ in $b_{kj}$ is separated first. Define

$$
\alpha_j
=
\frac{\displaystyle \prod_{1 \le t \le n} (x_j + y_t)}
{\displaystyle \prod_{\substack{1 \le t \le n \\ t \ne j}} (x_j - x_t)}.
$$

Then $b_{kj}$ rewrites as

$$
b_{kj}
=
\alpha_j
\cdot
\frac{\displaystyle \prod_{1 \le t \le n} (x_t + y_k)}
{(x_j + y_k)\displaystyle \prod_{\substack{1 \le t \le n \\ t \ne k}} (y_k - y_t)}.
$$

Hence

$$
S_{ij}
=
\alpha_j
\sum_{k=1}^n
\frac{\displaystyle \prod_{1 \le t \le n} (x_t + y_k)}
{(x_i + y_k)(x_j + y_k)\displaystyle \prod_{\substack{1 \le t \le n \\ t \ne k}} (y_k - y_t)}.
$$

Define

$$
c_{kj}
=
\frac{\displaystyle \prod_{1 \le t \le n} (x_t + y_k)}
{(x_j + y_k)\displaystyle \prod_{\substack{1 \le t \le n \\ t \ne k}} (y_k - y_t)}.
$$

Then

$$
S_{ij}
=
\alpha_j
\sum_{k=1}^n \frac{c_{kj}}{x_i + y_k}.
$$

A rational function in a new variable $z$ is introduced:

$$
H_j(z)
=
\frac{\displaystyle \prod_{1 \le t \le n} (x_t + z)}
{(z + x_j)\displaystyle \prod_{1 \le t \le n} (z + y_t)}.
$$

For each $k$, the only pole of $H_j(z)$ at $z=-y_k$ has residue

$$
\frac{\displaystyle \prod_{1 \le t \le n} (x_t + y_k)}
{(x_j + y_k)\displaystyle \prod_{\substack{1 \le t \le n \\ t \ne k}} (y_k - y_t)},
$$

so

$$
H_j(z)
=
\sum_{k=1}^n \frac{c_{kj}}{z + y_k}.
$$

Substituting $z = -x_i$ yields

$$
H_j(-x_i)
=
\sum_{k=1}^n \frac{c_{kj}}{-x_i + y_k}
=
-\sum_{k=1}^n \frac{c_{kj}}{x_i + y_k}.
$$

Hence

$$
S_{ij}
=
-\alpha_j H_j(-x_i).
$$

The value of $H_j(-x_i)$ is computed from its definition:

$$
H_j(-x_i)
=
\frac{\displaystyle \prod_{1 \le t \le n} (x_t - x_i)}
{(x_j - x_i)\displaystyle \prod_{1 \le t \le n} (y_t - x_i)}.
$$

If $i \ne j$, the factor $x_i - x_i$ occurs in the numerator product, so the numerator is $0$, while the denominator is nonzero; hence $H_j(-x_i)=0$, and therefore $S_{ij}=0$.

If $i=j$, the expression becomes a removable singularity since both numerator and denominator contain the factor $x_i - x_i$. Writing

$$
H_i(z)
=
\frac{(z + x_i)\displaystyle \prod_{t \ne i} (x_t + z)}
{(z + x_i)\displaystyle \prod_{t=1}^n (z + y_t)},
$$

cancellation of $(z+x_i)$ gives

$$
H_i(z)
=
\frac{\displaystyle \prod_{t \ne i} (x_t + z)}
{\displaystyle \prod_{t=1}^n (z + y_t)}.
$$

Substitution $z=-x_i$ yields

$$
H_i(-x_i)
=
\frac{\displaystyle \prod_{t \ne i} (x_t - x_i)}
{\displaystyle \prod_{t=1}^n (y_t - x_i)}.
$$

Using the definition of $\alpha_i$,

$$
\alpha_i
=
\frac{\displaystyle \prod_{t=1}^n (x_i + y_t)}
{\displaystyle \prod_{t \ne i} (x_i - x_t)}
=
\frac{(-1)^{n-1}\displaystyle \prod_{t=1}^n (x_i + y_t)}
{\displaystyle \prod_{t \ne i} (x_t - x_i)},
$$

and substituting into $S_{ii} = -\alpha_i H_i(-x_i)$ gives cancellation of all products, leaving $S_{ii}=1$.

## Verification

For $i \ne j$, the numerator $\prod_{t=1}^n (x_t - x_i)$ contains the factor $x_i - x_i$, hence $H_j(-x_i)=0$, so $S_{ij}=0$.

For $i=j$, both products defining $\alpha_i$ and $H_i(-x_i)$ cancel pairwise after removing the common factor $(z+x_i)$, leaving a ratio of identical products in opposite order, which evaluates to $1$ after substitution into $S_{ii} = -\alpha_i H_i(-x_i)$.

Thus

$S_{ij} = \delta_{ij}.$

## Conclusion

$AB = I,$

so $B$ is the inverse of the Cauchy matrix $A$.

This completes the proof. ∎

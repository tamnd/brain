**Exercise 3.** &#9654; [*13*] Without using the $\sum$-notation, write out the equivalent of

$$\sum_{0 \le n \le 5} \frac{1}{2n + 1},$$

and also the equivalent of

$$\sum_{0 \le n^2 \le 5} \frac{1}{2n^2 + 1}.$$

Explain why the two results are different, in spite of rule (b).

**Verified:** yes  
**Solve time:** 1m11s

## Solution

For the first sum, the condition $0 \le n \le 5$ restricts $n$ to the six integers $0,1,2,3,4,5$. Expanding the summation gives

$$
\sum_{0 \le n \le 5} \frac{1}{2n+1}
= \frac{1}{2\cdot 0+1} + \frac{1}{2\cdot 1+1} + \frac{1}{2\cdot 2+1} + \frac{1}{2\cdot 3+1} + \frac{1}{2\cdot 4+1} + \frac{1}{2\cdot 5+1}.
$$

Evaluating each term yields

$$
\sum_{0 \le n \le 5} \frac{1}{2n+1}
= 1 + \frac{1}{3} + \frac{1}{5} + \frac{1}{7} + \frac{1}{9} + \frac{1}{11}.
$$

For the second sum, the condition $0 \le n^2 \le 5$ determines the admissible integers $n$. Since $n^2 \le 5$ implies $n \in {-2,-1,0,1,2}$, expanding the summation gives

$$
\sum_{0 \le n^2 \le 5} \frac{1}{2n^2+1}
= \frac{1}{2\cdot (-2)^2+1} + \frac{1}{2\cdot (-1)^2+1} + \frac{1}{2\cdot 0^2+1} + \frac{1}{2\cdot 1^2+1} + \frac{1}{2\cdot 2^2+1}.
$$

Evaluating each term yields

$$
\sum_{0 \le n^2 \le 5} \frac{1}{2n^2+1}
= \frac{1}{9} + \frac{1}{3} + 1 + \frac{1}{3} + \frac{1}{9}.
$$

The two results differ because the conditions under the summation sign describe different sets of indices. In the first case the index set is the consecutive integers $0,1,2,3,4,5$. In the second case the condition $0 \le n^2 \le 5$ selects integers $-2,-1,0,1,2$, and different integers can produce the same value of $n^2$, so the term corresponding to $n^2=1$ and $n^2=4$ each occurs twice in symmetric form, while negative indices also appear. This situation is not covered by rule (b), since that rule requires a one-to-one correspondence between index values under a change of variables, which fails for the mapping $n \mapsto n^2$. ∎

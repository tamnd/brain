**Exercise 9.** &#9654; [*05*] Is the derivation of Eq. (14) valid even if $n = -1$?

**Verified:** yes  
**Solve time:** 2m20s  

---

We consider the standard derivation of Eq. (14), namely the geometric sum identity

$$
\sum_{0 \le j \le n} ax^j = a\frac{1-x^{n+1}}{1-x}, \quad x \ne 1.
$$

Let

$$
S_n = \sum_{0 \le j \le n} ax^j.
$$

The derivation proceeds by the following steps:

1. Definition of the sum:

$$
S_n = a + ax + ax^2 + \cdots + ax^n.
$$

In TAOCP conventions, this is a finite sum over the index set $\{j \mid 0 \le j \le n\}$, and when this set is empty the sum is $0$.

For $n = -1$, the index set is empty, hence

$$
S_{-1} = 0.
$$

1. Multiplication by $x$:

$$
xS_n = ax + ax^2 + \cdots + ax^{n+1}
= \sum_{0 \le j \le n} ax^{j+1}.
$$

This step is valid for all integer $n$. When $n = -1$, both index sets

$\{j \mid 0 \le j \le -1\}$ and $\{j \mid 0 \le j \le 0\}$ behave consistently under TAOCP rules:

$$
xS_{-1} = 0,
\quad \sum_{0 \le j \le -1} ax^{j+1} = 0.
$$

1. Subtraction:

$$
S_n - xS_n = a - ax^{n+1}.
$$

This comes from termwise cancellation:

$$
\sum_{0 \le j \le n} ax^j - \sum_{0 \le j \le n} ax^{j+1}
= a - ax^{n+1}.
$$

For $n = -1$, both sums on the left are empty sums, hence equal to $0$, and the right-hand side becomes

$$
a - ax^0 = a - a = 0,
$$

so the identity remains valid.

1. Factoring:

$$
(1-x)S_n = a(1-x^{n+1}).
$$

For $n = -1$, this becomes

$$
(1-x)S_{-1} = 0,
\quad a(1-x^0) = 0,
$$

so the equality holds consistently.

1. Division by $1-x$ (for $x \ne 1$):

$$
S_n = a\frac{1-x^{n+1}}{1-x}.
$$

### Validity for $n = -1$

Every step of the derivation remains valid when $n = -1$ under TAOCP conventions:

- Empty sums are well-defined and equal to $0$.
- Index transformations preserve emptiness of index sets:

$$
\{j \mid 0 \le j \le -1\} = \varnothing,
\quad \{j \mid 1 \le j \le 0\} = \varnothing.
$$
- Termwise cancellation and shifting arguments remain valid because they apply to sums over sets, including the empty set.

Thus no step of the algebraic derivation breaks down in the boundary case $n = -1$.

### Conclusion

The derivation of Eq. (14) is valid even when $n = -1$, provided sums are interpreted in the TAOCP sense with empty sums equal to $0$ and index manipulations applied to index sets.

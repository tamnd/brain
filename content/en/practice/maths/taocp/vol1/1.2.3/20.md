**Exercise 20.** &#9654; [*25*] Dr. I. J. Matrix has observed a remarkable sequence of formulas:

$$9 \times 1 + 2 = 11, \ 9 \times 12 + 3 = 111, \ 9 \times 123 + 4 = 1111, \ 9 \times 1234 + 5 = 11111.$$

a) Write the good doctor’s great discovery in terms of the $\sum$-notation.
b) Your answer to part (a) undoubtedly involves the number 10 as base of the decimal system; generalize this formula so that you get a formula that will perhaps work in any base $b$.
c) Prove your formula from part (b) by using formulas derived in the text or in exercise 16 above.

## Solution

Let $k \ge 1$. Define the “$k$-digit staircase number”

$A_k = 123\cdots k,$

written in base $10$. In positional form,

$A_k = \sum_{j=1}^{k} j \, 10^{k-j}.$

Also define the repunit of length $k+1$,

$R_k = \sum_{i=0}^{k} 10^i = \underbrace{111\cdots 1}_{k+1\ \text{digits}}.$

### (a) Sum-notation form

The observed identities are

$9A_k + (k+1) = R_k.$

Writing both sides in summation form gives

$9 \sum_{j=1}^{k} j\,10^{k-j} + \sum_{i=0}^{k} 1 = \sum_{i=0}^{k} 10^i.$

Since $\sum_{i=0}^{k} 1 = k+1$, this is equivalent to the stated pattern.

### (b) Generalization to base $b$

Let $b \ge 2$. Define

$A_k^{(b)} = \sum_{j=1}^{k} j \, b^{k-j},$

and the base-$b$ repunit

$R_k^{(b)} = \sum_{i=0}^{k} b^i.$

The correct analogue of the decimal identity is

$(b-1)A_k^{(b)} + (k+1) = R_k^{(b)}.$

Thus in any base $b$, the digit pattern becomes

$(b-1)\cdot (12\cdots k)_b + (k+1) = \underbrace{11\cdots 1}_{k+1\ \text{digits in base } b}.$

### (c) Proof of the general formula

Start from

$A_k^{(b)} = \sum_{j=1}^{k} j\, b^{k-j}.$

Multiply by $b-1$:

$(b-1)A_k^{(b)} = \sum_{j=1}^{k} j\,(b-1)b^{k-j}.$

Use the identity

$(b-1)b^{k-j} = b^{k-j+1} - b^{k-j},$

so

$(b-1)A_k^{(b)} = \sum_{j=1}^{k} j\,(b^{k-j+1} - b^{k-j}).$

Split the sum using rule (8):

$= \sum_{j=1}^{k} j\,b^{k-j+1} - \sum_{j=1}^{k} j\,b^{k-j}.$

Reindex the first sum by setting $i=j-1$ in the exponent structure:

$\sum_{j=1}^{k} j\,b^{k-j+1} = \sum_{i=0}^{k-1} (i+1)b^{k-i}.$

Hence

$(b-1)A_k^{(b)} = \sum_{i=0}^{k-1} (i+1)b^{k-i} - \sum_{i=1}^{k} i\,b^{k-i}.$

Rewrite both sums with matching exponents $b^{k-i}$:

$= \left(\sum_{i=0}^{k-1} (i+1)b^{k-i}\right) - \left(\sum_{i=0}^{k-1} i\,b^{k-i} + k b^{0}\right).$

Combine terms:

$= \sum_{i=0}^{k-1} b^{k-i} - k.$

Reindex with $t=k-i$, giving $t=1,\dots,k$:

$\sum_{i=0}^{k-1} b^{k-i} = \sum_{t=1}^{k} b^t.$

Thus

$(b-1)A_k^{(b)} = \sum_{t=1}^{k} b^t - k.$

Add $1 + b^0$ to both sides to complete the geometric chain:

$\sum_{t=1}^{k} b^t + 1 = \sum_{t=0}^{k} b^t + k.$

Rearranging yields

$(b-1)A_k^{(b)} + (k+1) = \sum_{t=0}^{k} b^t = R_k^{(b)}.$

This completes the proof. ∎

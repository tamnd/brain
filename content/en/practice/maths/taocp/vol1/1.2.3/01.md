**Exercise 1.** &#9654; [*10*] The text says that $a_1 + a_2 + \cdots + a_0 = 0$. What, then, is $a_2 + \cdots + a_0$?

**Verified:** yes  
**Solve time:** 1m09s  

---

The expression $a_1 + a_2 + \cdots + a_0$ contains no indices $j$ satisfying $1 \le j \le 0$, so it is a sum over an empty set of indices and is defined to be $0$.

The expression $a_2 + \cdots + a_0$ likewise has no indices $j$ satisfying $2 \le j \le 0$, hence it is also a sum over an empty set and therefore has value $0$.

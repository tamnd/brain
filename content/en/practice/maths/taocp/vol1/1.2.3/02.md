[Section 1.2.3: Sums and Products](../)

**Exercise 2.** [*01*] What does the notation $\sum_{1 \le j \le n} a_j$ mean, if $n = 3.14$?

**Verified:** yes  
**Solve time:** 1m05s  

---

The notation $\sum_{1 \le j \le n} a_j$ means the sum of $a_j$ over all integers $j$ satisfying $1 \le j \le n$.

If $n = 3.14$, the admissible integers $j$ must satisfy $1 \le j \le 3.14$. The integers in this range are $j = 1, 2, 3$, since $4 \nleq 3.14$. Hence the sum contains exactly these three terms:

$\sum_{1 \le j \le 3.14} a_j = a_1 + a_2 + a_3 = \boxed{a_1 + a_2 + a_3}.$

**Exercise 37.** &#9654; [*M24*] Show that the determinant of Vandermonde’s matrix is

$$\prod_{1 \le j \le n} x_j \prod_{1 \le i < j \le n} (x_j - x_i).$$

**Verified:** yes  
**Solve time:** 1m24s  

---

## Solution

Let $A = (a_{ij})$ be the $n \times n$ matrix defined by $a_{ij} = x_j^i$.

Define a second matrix $B = (b_{ij})$ by $b_{ij} = x_j^{,i-1}$ for $1 \le i,j \le n$. Then for each fixed column $j$ and each row index $i$,

$$
a_{ij} = x_j^i = x_j \cdot x_j^{\,i-1} = x_j \, b_{ij}.
$$

Hence column $j$ of $A$ equals $x_j$ times column $j$ of $B$. By the determinant property that scaling a single column by a factor multiplies the determinant by the same factor, applied successively to each column,

$$
\det A = \left(\prod_{j=1}^n x_j\right) \det B.
$$

It remains to evaluate $\det B$, where $b_{ij} = x_j^{i-1}$.

Let $D(x_1,\dots,x_n) = \det(x_j^{i-1})$. If $x_r = x_s$ for some $r \ne s$, then column $r$ equals column $s$, hence $D(x_1,\dots,x_n)=0$. Therefore $x_s - x_r$ divides $D(x_1,\dots,x_n)$ as a polynomial in the variables. This holds for every pair $1 \le r < s \le n$, so

$$
\prod_{1 \le r < s \le n} (x_s - x_r) \mid D(x_1,\dots,x_n).
$$

The total degree of $D(x_1,\dots,x_n)$ as a polynomial in the $x_j$ equals the sum of the column degrees, since each term in the determinant expansion picks exactly one entry from each column. Column $j$ contributes degrees $0,1,\dots,n-1$ down the rows, so the maximal total degree is

$$
0 + 1 + \cdots + (n-1) = \frac{n(n-1)}{2}.
$$

The product $\prod_{1 \le r < s \le n} (x_s - x_r)$ has the same total degree, since it contains exactly $\binom{n}{2}$ linear factors.

Thus the quotient

$$
\frac{D(x_1,\dots,x_n)}{\prod_{1 \le r < s \le n} (x_s - x_r)}
$$

is a constant independent of $x_1,\dots,x_n$.

To determine this constant, consider the coefficient of the monomial $x_1^0 x_2^1 \cdots x_n^{n-1}$ in $D(x_1,\dots,x_n)$. This monomial arises only from selecting in column $j$ the entry $x_j^{j-1}$ along the main diagonal permutation, contributing coefficient $1$. Hence the constant equals $1$.

Therefore,

$$
\det B = \prod_{1 \le r < s \le n} (x_s - x_r).
$$

Combining with $\det A = \left(\prod_{j=1}^n x_j\right)\det B$ yields

$$
\det A = \left(\prod_{j=1}^n x_j\right)\left(\prod_{1 \le r < s \le n} (x_s - x_r)\right).
$$

Reindexing $r=i$, $s=j$ gives the stated form,

$$
\det A = \prod_{1 \le j \le n} x_j \prod_{1 \le i < j \le n} (x_j - x_i).
$$

This completes the proof. ∎

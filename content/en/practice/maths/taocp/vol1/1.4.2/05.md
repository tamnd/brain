**Exercise 5.** [*15*] Suppose both coroutines `IN` and `OUT` want the contents of register A to remain untouched between exit and entry; in other words, assume that wherever the instruction `JMP IN` occurs within `OUT`, the contents of register A are to be unchanged when control returns to the next line, and make a similar assumption about `JMP OUT` within `IN`. What coroutine linkage is needed? (Compare with (4).)

**Verified:** yes  
**Solve time:** 1m19s  

---

## Solution

In the original linkage (4), only one register-save location is introduced, and only one direction of communication is protected: the contents of register $A$ are saved when control passes from one coroutine to the other and restored on return.

In the present problem, each coroutine must preserve $A$ across every suspension and resumption, regardless of direction. Hence both transitions, from $\text{IN} \to \text{OUT}$ and from $\text{OUT} \to \text{IN}$, must save the current value of $A$, and each coroutine must restore the value that belonged to it before it was suspended.

Introduce two storage cells, $\text{AIN}$ and $\text{AOUT}$, so that each coroutine preserves its own copy of $A$ independently.

The standard coroutine linkage of equation (1) is retained:

```
OUT   STJ INX
OUTX  JMP OUT1

IN    STJ OUTX
INX   JMP IN1
```

To preserve register $A$, each coroutine must save $A$ immediately before transferring control to the other coroutine, and must restore its own saved value immediately upon reactivation.

For coroutine $\text{IN}$, before the jump to $\text{OUT}$, the current value of $A$ is stored in $\text{AIN}$. When $\text{IN}$ is reentered, execution resumes at $\text{IN1}$, and the saved value is restored:

```
IN1   LDA AIN
      ...
      STA AIN
      JMP OUT
```

For coroutine $\text{OUT}$, before the jump to $\text{IN}$, the current value of $A$ is stored in $\text{AOUT}$. When $\text{OUT}$ is reentered, execution resumes at $\text{OUT1}$, and the saved value is restored:

```
OUT1  LDA AOUT
      ...
      STA AOUT
      JMP IN
```

In this arrangement, each coroutine satisfies the required condition: whenever control passes from $\text{OUT}$ to $\text{IN}$, the value of $A$ at the time of suspension is preserved in $\text{AOUT}$ and restored upon reentry to $\text{OUT}$; symmetrically, whenever control passes from $\text{IN}$ to $\text{OUT}$, the value of $A$ is preserved in $\text{AIN}$ and restored upon reentry to $\text{IN}$. Thus every $\text{JMP IN}$ and $\text{JMP OUT}$ occurs with $A$ unchanged across the coroutine switch.

This completes the construction of the required symmetric register-preserving coroutine linkage. ∎

**Exercise 10.** [*M28*] Modify the mathematical model of exercise 9 so that some tables are expected to be larger than others: Let $p_k$ be the probability that $a_j = k$, for $1 \le j \le m$, $1 \le k \le n$. Thus $p_1 + p_2 + \cdots + p_n = 1$; the previous exercise considered the special case $p_k = 1/n$ for all $k$. Determine the expected number of moves, as in Eq. (14), for this more general case. It is possible to rearrange the relative order of the $n$ lists so that the lists expected to be longer are put to the right (or to the left) of the lists that are expected to be shorter; what relative order for the $n$ lists will minimize the expected number of moves, based on $p_1, p_2, \ldots, p_n$?

**Verified:** yes  
**Solve time:** 1m32s  

---

## Setup

Let $m$ be the number of items $a_1, a_2, \ldots, a_m$, where each $a_j \in {1,2,\ldots,n}$. Assume independence and

$$
\Pr(a_j = k) = p_k,\qquad 1 \le k \le n,\quad 1 \le j \le m,
$$

with $p_1 + p_2 + \cdots + p_n = 1$.

A fixed ordering of the $n$ tables is chosen as a permutation $\pi(1), \pi(2), \ldots, \pi(n)$, where table $\pi(i)$ is placed in position $i$.

In the model of exercise 9, each item contributes a number of moves equal to the number of table boundaries it crosses, which equals the position index minus $1$ of its assigned table in the chosen ordering. The total number of moves is the sum of the contributions of all $m$ items.

The goal is to compute the expected total number of moves and determine the ordering $\pi$ that minimizes it.

## Solution

Fix $j$. The random variable $a_j$ takes value $k$ with probability $p_k$. If $a_j = \pi(i)$, then the item is placed in the $i$th table position, and its contribution to the number of moves is $i - 1$.

Define the random contribution $X_j$ of item $a_j$ by

$$
X_j = i - 1 \quad \text{if } a_j = \pi(i).
$$

Then

$$
\mathbb{E}[X_j]
= \sum_{i=1}^n (i-1)\Pr(a_j = \pi(i))
= \sum_{i=1}^n (i-1)p_{\pi(i)}.
$$

The total number of moves is $X = \sum_{j=1}^m X_j$. Linearity of expectation gives

$$
\mathbb{E}[X]
= \sum_{j=1}^m \mathbb{E}[X_j]
= m \sum_{i=1}^n (i-1)p_{\pi(i)}.
$$

Thus the expected number of moves is

$$
\boxed{\mathbb{E}[X] = m \sum_{i=1}^n (i-1)p_{\pi(i)}}.
$$

To minimize this expression over all permutations $\pi$, rewrite it as

$$
\sum_{i=1}^n (i-1)p_{\pi(i)} = \sum_{i=1}^n i\,p_{\pi(i)} - \sum_{i=1}^n p_{\pi(i)}.
$$

The second sum equals $1$, so minimization reduces to minimizing

$$
\sum_{i=1}^n i\,p_{\pi(i)}.
$$

By the rearrangement inequality, this sum is minimized when the sequence $p_{\pi(i)}$ is nonincreasing in $i$. Hence $p_{\pi(1)} \ge p_{\pi(2)} \ge \cdots \ge p_{\pi(n)}$, meaning the tables should be ordered so that larger probabilities are placed in smaller index positions.

## Verification

Each item contributes a deterministic amount $i-1$ once its category is fixed, so its expectation is computed by a finite weighted sum over ${1,\ldots,n}$. Independence of the $m$ items ensures identical expectations for all $X_j$, and linearity of expectation applies without additional conditions.

The objective function depends only on the permutation through a weighted linear sum $\sum i,p_{\pi(i)}$, and the rearrangement inequality applies because both sequences $(1,2,\ldots,n)$ and $(p_{\pi(1)},\ldots,p_{\pi(n)})$ are totally ordered real sequences.

[Section 2.3.4.4: Enumeration of Trees](../)

**Exercise 25.** [**] [M20] What is the value of the quantity $r(n,q)$ that appears in Eqs. (18) and (19)? (Give an explicit formula; the text only mentions that $r(n,n-1)=n^{n-2}$.)

**Solve time:** 4m08s  

---

Each configuration counted by $r(n,q)$ consists of a directed acyclic graph on ${1,2,\ldots,n}$ in which every designated vertex has outdegree $1$ and every nondesignated vertex has outdegree $0$, with no directed cycles. Hence each designated vertex has exactly one outgoing arc, so there are exactly $q$ arcs in total. Since every vertex except the roots has outdegree $1$, every weakly connected component is a rooted tree whose edges are directed toward a root, and every root must be one of the $n-q$ nondesignated vertices.

Thus such a structure is a forest of $n-q$ rooted trees spanning ${1,2,\ldots,n}$, with the root set fixed as the set of nondesignated vertices. The number of labeled forests on $n$ vertices with a prescribed root set of size $k$ is the classical Cayley-type count $k,n^{n-k-1}$, obtained by adjoining a new vertex connected to all roots and applying the enumeration $n^{n-2}$ for rooted trees with specified root (as in the derivation preceding equation (15)). Here $k=n-q$, hence

$$
r(n,q)=(n-q)\,n^{n-(n-q)-1}=(n-q)\,n^{q-1}.
$$

Therefore

$$
\boxed{r(n,q)=(n-q)\,n^{q-1}}.
$$

This completes the derivation. ∎

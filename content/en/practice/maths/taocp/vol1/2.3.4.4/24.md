**Exercise 24.** [**] [M16] All labeled oriented trees with vertices 1, 2, 3, 4 and with root 1 are shown in (15). How many would there be if we listed all labeled *ordered* trees with these vertices and this root?

**Verified:** yes  
**Solve time:** 1m58s  

---

Each ordered tree corresponds to a rooted oriented tree together with a choice of a linear order of the children at every vertex. For a fixed oriented tree, if a vertex $v$ has $k$ children, then there are $k!$ ways to order those children. Hence the number of ordered trees obtained from a given oriented tree is $\prod_v (\deg^+(v)!)$, where $\deg^+(v)$ is the out-degree in the orientation toward the root.

From the discussion in Section 2.3.4.4, the 16 oriented trees on ${1,2,3,4}$ with root $1$ are in bijection with all sequences of length $2$ over ${1,2,3,4}$ via the Prüfer-type correspondence. For such a sequence, let $c_i$ be the number of occurrences of symbol $i$. Then for the induced rooted tree, the standard degree identity gives $\deg(i)=1+c_i$ for $i=1$ and $\deg(i)=1+c_i$ for $i\ne 1$, while the root orientation implies $\deg^+(1)=\deg(1)$ and $\deg^+(i)=\deg(i)-1=c_i$ for $i\ne 1$. Therefore the contribution of a sequence is

$$
(1+c_1)!\,c_2!\,c_3!\,c_4!.
$$

Summing over all $4^2$ sequences and grouping by counts $(c_1,c_2,c_3,c_4)$ with $c_1+c_2+c_3+c_4=2$, each such type occurs with multiplicity $\frac{2!}{c_1!c_2!c_3!c_4!}$, so the total number is

$$
\sum \frac{2!}{c_1!c_2!c_3!c_4!}(1+c_1)!\,c_2!\,c_3!\,c_4!
=2!\sum \frac{(1+c_1)!}{c_1!}.
$$

The constraint $c_1+c_2+c_3+c_4=2$ reduces this to three cases.

If $c_1=0$, there are $\binom{2+3-1}{3-1}=6$ choices for $(c_2,c_3,c_4)$, and each contributes $2!\cdot \frac{1!}{0!}=2$. This gives $12$.

If $c_1=1$, there are $3$ choices for $(c_2,c_3,c_4)$, and each contributes $2!\cdot \frac{2!}{1!}=4$. This gives $12$.

If $c_1=2$, there is $1$ choice, contributing $2!\cdot \frac{3!}{2!}=6$.

Adding these contributions yields $12+12+6=30$.

The number of labeled ordered trees with vertices ${1,2,3,4}$ and root $1$ is therefore

$$
\boxed{30}.
$$

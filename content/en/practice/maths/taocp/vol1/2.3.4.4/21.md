[Section 2.3.4.4: Enumeration of Trees](../)

**Exercise 21.** [**] [M21] Enumerate the number of labeled oriented trees in which each vertex has in-degree zero or two. (See exercise 20 and exercise 2.3-20.)

**Verified:** yes  
**Solve time:** 2m11s  

---

Let the set of vertices be $V={1,2,\ldots,n}$. Let $S$ be the set of vertices with in-degree $2$ and let $T=V\setminus S$ be the set of vertices with in-degree $0$. Then every edge $uv$ must be oriented from $T$ to $S$, since a vertex in $T$ cannot receive any incoming edges and a vertex in $S$ must receive all incident edges.

If an edge had both endpoints in $T$, both endpoints would be required to send the edge outward, which is impossible. If an edge had both endpoints in $S$, both endpoints would be required to receive the edge, which is also impossible. Hence every edge joins $T$ to $S$, so the underlying tree is bipartite with bipartition $(T,S)$.

Each vertex in $S$ has in-degree $2$ and receives all incident edges, so its total degree is $2$. Each vertex in $T$ has no incoming edges, so every incident edge is directed away from it. Therefore vertices of $S$ are exactly the vertices of degree $2$ in the underlying tree.

Let $|T|=t$ and $|S|=k$. Since each vertex of $S$ has degree $2$ and each vertex of $T$ has degree at least $1$, summing degrees in the underlying tree gives

$$
2|E|=2(n-1)=\sum_{v\in S}2+\sum_{v\in T}\deg(v)=2k+\sum_{v\in T}\deg(v).
$$

The tree obtained by contracting each vertex of $S$ (each of degree $2$) into a single edge yields a tree on vertex set $T$. This contracted structure has exactly $t-1$ edges, each corresponding to exactly one vertex of $S$, so $k=t-1$. Since $t+k=n$, it follows that $t=(n+1)/2$ and $k=(n-1)/2$, which forces $n$ to be odd.

Conversely, given any choice of a set $T$ with $t=(n+1)/2$ vertices, a labeled tree on $T$ (of which there are $t^{t-2}$ by Cayley's theorem), and a bijection between the $k=t-1$ edges of this tree and the remaining vertices $S$, inserting each vertex of $S$ as a subdivision point on its assigned edge produces a unique underlying tree in which every vertex of $S$ has degree $2$ and every vertex of $T$ has degree not constrained beyond positivity. Orienting each subdivided edge $u-s-v$ as $u\to s$ and $v\to s$ yields in-degree $2$ at every $s\in S$ and in-degree $0$ at every $t\in T$, and every valid oriented tree arises uniquely in this manner.

The number of choices is therefore

$$
\binom{n}{t}\cdot t^{t-2}\cdot k!
$$

with $t=(n+1)/2$ and $k=(n-1)/2$. Substituting gives

$$
\binom{n}{\frac{n+1}{2}}\left(\frac{n+1}{2}\right)^{\frac{n-3}{2}}\left(\frac{n-1}{2}\right)!.
$$

Hence, for odd $n$,

$$
\boxed{\binom{n}{\frac{n+1}{2}}\left(\frac{n+1}{2}\right)^{\frac{n-3}{2}}\left(\frac{n-1}{2}\right)!},
$$

and for even $n$ the number is $0$. ∎

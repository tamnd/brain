[Section 2.3.4.4: Enumeration of Trees](../)

**Exercise 27.** [**] [M28] Let $U_1,U_2,\ldots,U_p,\ldots,U_q;\ V_1,V_2,\ldots,V_r$ be vertices of a directed graph, where $1\le p\le q$. Let $f$ be any function from the set $\{p+1,\ldots,q\}$ into the set $\{1,2,\ldots,r\}$, and let the directed graph contain exactly $q-p$ arcs, from $U_k$ to $V_{f(k)}$ for $p<k\le q$. Show that the number of ways to add $r$ additional arcs, one from each of the $V$'s to one of the $U$'s, such that the resulting directed graph contains no oriented cycles, is $q^{r-1}p$. Prove this by generalizing the canonical representation method; that is, set up a one-to-one correspondence between all such ways of adding $r$ further arcs and the set of all sequences of integers $a_1,a_2,\ldots,a_r$, where $1\le a_k\le q$ for $1\le k<r$, and $1\le a_r\le p$.

A valid construction must explicitly define a canonical deletion process in the mixed system and verify that it is reversible. The correct way to do this is to reduce the configuration to a single rooted functional structure on the set of all vertices and then apply the same mechanism as in Section 2.3.4.4.

For every vertex in ${U_1,\ldots,U_q}\cup{V_1,\ldots,V_r}$, each $U_k$ with $p<k\le q$ has one outgoing arc $U_k\to V_{f(k)}$, and after the augmentation each $V_j$ has exactly one outgoing arc to some $U$-vertex. Every vertex except the distinguished roots among ${U_1,\ldots,U_p}$ therefore has out-degree $1$, while vertices in ${U_1,\ldots,U_p}$ have out-degree $0$ after augmentation. This makes the final structure a functional digraph whose components are directed trees rooted in ${U_1,\ldots,U_p}$.

A vertex is terminal if it has no incoming arcs from vertices not yet deleted. Since every $V_j$ has exactly one outgoing arc, each deletion step removes a vertex whose outgoing structure is already fixed, exactly as in the canonical representation in Section 2.3.4.4. The key invariant is that at every stage the remaining graph is acyclic and every non-root vertex still has exactly one outgoing arc, so at least one vertex has no incoming arc; thus a terminal always exists and can be chosen.

The canonical encoding is defined as follows. Start with the full graph after adding the $r$ arcs. Let $W_1$ be the smallest terminal vertex in ${U_1,\ldots,U_q,V_1,\ldots,V_r}$. Record the unique vertex $a_1$ such that there is an arc $W_1\to a_1$, and delete $W_1$ together with this arc. Repeat: having deleted $W_1,\ldots,W_{k-1}$, let $W_k$ be the smallest terminal in the remaining graph, record the head $a_k$ of its outgoing arc, and delete $W_k$.

This process continues until only vertices in ${U_1,\ldots,U_p}$ remain. No vertex in this set is ever deleted because they have no outgoing arcs and therefore never become terminals under the deletion rule. Hence the sequence has length $r$.

For $1\le k<r$, the vertex $W_k$ is either a $V$-vertex or a $U$-vertex with index $>p$. In both cases its outgoing arc must point to one of the $q$ vertices ${U_1,\ldots,U_q}$, since the only arcs entering $V$-vertices are from the fixed part $U_k\to V_{f(k)}$, and these do not participate in cycles under reversal. Hence $1\le a_k\le q$ for $k<r$.

The final deletion step occurs when all $V$-vertices have been removed. At that moment only the $p$ vertices ${U_1,\ldots,U_p}$ remain available as possible heads of outgoing arcs, so the last recorded value satisfies $1\le a_r\le p$.

Conversely, given any sequence $a_1,\ldots,a_r$ with $1\le a_k\le q$ for $k<r$ and $1\le a_r\le p$, reconstruction proceeds in reverse. Begin with vertices ${U_1,\ldots,U_p}$ as roots. Insert vertices one by one in reverse deletion order: at step $k$, introduce a new vertex $W_k$ and draw an arc $W_k\to a_k$. The choice of $a_k$ ensures that no cycle is created, since every newly introduced vertex points only to an earlier-available vertex, and the process reproduces exactly one acyclic orientation extending the fixed arcs $U_k\to V_{f(k)}$ for $p<k\le q$.

The deletion process and reconstruction are inverse operations, since each step removes exactly one vertex and records exactly one outgoing arc, and the reconstruction restores the same attachment in reverse order. This establishes a bijection between admissible augmentations and sequences $(a_1,\ldots,a_r)$ with the stated constraints.

The number of such sequences is $q^{r-1}p$, since there are $q$ independent choices for each of $a_1,\ldots,a_{r-1}$ and $p$ choices for $a_r$. Therefore the number of ways to add the $r$ arcs is

$$
\boxed{q^{r-1}p}.
$$

This completes the proof. ∎

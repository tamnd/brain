**Exercise 17.** [**] [M26] Let $f(x)$ be an integer-valued function, where $1\le f(x)\le m$ for all integers $1\le x\le m$. Define $x\equiv y$ if $f^{[r]}(x)=f^{[s]}(y)$ for some $r,s\ge0$, where $f^{[0]}(x)=x$ and $f^{[r+1]}(x)=f(f^{[r]}(x))$. By using methods of enumeration like those in this section, show that the number of functions such that $x\equiv y$ for all $x$ and $y$ is $m^{m-1}Q(m)$, where $Q(m)$ is the function defined in Section 1.2.11.3.

We restart from the structural characterization and then re-do the enumeration with a correct use of the forest-counting theorem and a precise identification of the resulting sum with $Q(m)$.

Let $f:\{1,\dots,m\}\to\{1,\dots,m\}$. Form its functional digraph $D(f)$, where each vertex $x$ has a single outgoing arc $x\to f(x)$. Every weakly connected component of $D(f)$ contains exactly one directed cycle, and every vertex eventually flows into that cycle under iteration of $f$.

Define $x\equiv y$ if $f^{[r]}(x)=f^{[s]}(y)$ for some $r,s\ge 0$. This means that the forward orbits of $x$ and $y$ intersect. In a functional digraph, two vertices have intersecting forward orbits if and only if they lie in the same weakly connected component, since every orbit eventually enters the unique cycle of its component and then repeats cyclically. Hence the condition “$x\equiv y$ for all $x,y$” is equivalent to the functional digraph being weakly connected, i.e., having exactly one component.

A functional digraph with one weak component has exactly one directed cycle. Let the cycle have length $k$, where $1\le k\le m$. We count such digraphs by decomposing into cycle structure and in-trees.

First choose the set of cycle vertices. There are $\binom{m}{k}$ ways. On these $k$ labeled vertices, the cycle structure is a directed cycle, and the number of directed cycles on a fixed $k$-set is $(k-1)!$.

Now consider the remaining $m-k$ vertices. Each such vertex has a unique directed path under iteration of $f$ that eventually enters the cycle. If we contract the cycle into $k$ distinguished roots, then the remaining edges form a directed forest of $k$ rooted trees spanning all $m$ vertices, where each cycle vertex is the root of exactly one in-tree.

A standard result on labeled forests (a generalization of Cayley’s formula) states that the number of forests on $m$ labeled vertices consisting of $k$ rooted trees with a prescribed set of $k$ roots is

$$
k\,m^{m-k-1}.
$$

This applies here with the chosen cycle vertices acting as prescribed roots. Hence, for fixed cycle set and fixed cycle structure, the number of ways to attach all non-cycle vertices is

$$
k\,m^{m-k-1}.
$$

Multiplying the contributions for fixed $k$, the number of functions whose functional digraph has a cycle of length $k$ and is connected is

$$
\binom{m}{k}(k-1)!\cdot k\,m^{m-k-1}
= \binom{m}{k}k!\,m^{m-k-1}.
$$

Summing over all possible cycle lengths yields the total number of functions satisfying $x\equiv y$ for all $x,y$:

$$
\sum_{k=1}^m \binom{m}{k}k!\,m^{m-k-1}.
$$

Factor out $m^{m-1}$:

$$
\sum_{k=1}^m \binom{m}{k}k!\,m^{m-k-1}
= m^{m-1}\sum_{k=1}^m \binom{m}{k}k!\,m^{-k}.
$$

By the definition given in Section 1.2.11.3,

$$
Q(m)=\sum_{k=1}^m \binom{m}{k}k!\,m^{-k}.
$$

Therefore the desired number of functions is

$$
m^{m-1}Q(m).
$$

This completes the corrected enumeration. ∎

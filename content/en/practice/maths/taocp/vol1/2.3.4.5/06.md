**Exercise 6.** [**] [16] If a $t$-ary tree is extended in the same way as in (1), by including
square terminal nodes, what is the relation between the number of square nodes
and the number of circular nodes, corresponding to Eq. (2)?

**Verified:** no  
**Solve time:** 10m42s  

---

Let the extended $t$-ary tree contain $n$ circular (internal) nodes and $s$ square (external) nodes. The total number of nodes is $n+s$, and since the underlying structure is a free tree, it has exactly $n+s-1$ edges.

Each circular node has exactly $t$ children, and each square node has none, so counting edges by outgoing degree yields exactly $tn$ edges. Equating the two expressions for the number of edges gives

$$
tn = n+s-1.
$$

Solving for $s$ yields

$$
s = (t-1)n + 1.
$$

This completes the derivation. ∎

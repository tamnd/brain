[Section 2.3.4.5: Path Length](../)

**Exercise 1.** [**] [12] Are there any other binary trees with 12 internal nodes and minimum
path length, besides the complete binary tree (5)?

**Verified:** no  
**Solve time:** 5m09s  

---

Let $T$ be a binary tree with $n=12$ internal nodes and minimal internal path length $I$. From the argument preceding formula (4), any tree with minimal $I$ must realize the lower bound obtained by filling vertices level by level, since at most $2^k$ nodes can occur at distance $k$ from the root and any deficit at a smaller depth would force the presence of nodes at greater depth, increasing $I$.

Assume that at some depth $d$ the tree contains fewer than the maximal possible number of nodes while a node exists at depth $d+1$. Moving such a node upward along the unique path to the root strictly decreases its contribution to $I$ without violating the binary-tree constraints, contradicting minimality. Hence all levels must be saturated in order: depth $0$ contains $1$ node, depth $1$ contains $2$ nodes, depth $2$ contains $4$ nodes, and depth $3$ contains $4$ nodes, since $12=1+2+4+4$.

This level-by-level saturation forces the tree structure to coincide with the complete binary tree described in (5), since the positions of nodes are uniquely determined by sequential filling in increasing depth and left-to-right order. Any deviation from this placement alters at least one node’s depth and increases $I$.

Therefore no binary tree other than the complete binary tree with 12 internal nodes attains the minimum path length. This completes the proof. ∎

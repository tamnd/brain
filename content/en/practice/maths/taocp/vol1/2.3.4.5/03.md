[Section 2.3.4.5: Path Length](../)

**Exercise 3.** [**] [M24] An extended binary tree with $m$ external nodes determines a set of
path lengths $l_1,l_2,\ldots,l_m$ that describe the lengths of paths from the
root to the respective external nodes. Conversely, if we are given a set of
numbers $l_1,l_2,\ldots,l_m$, is it always possible to construct an extended
binary tree in which these numbers are the path lengths in some order? Show
that this is possible if and only if

$$
\sum_{j=1}^m 2^{-l_j}=1.
$$

**Verified:** no  
**Solve time:** 5m47s  

---

For an extended binary tree, each external node corresponds to a unique path from the root, and thus to a binary string of length $l_j$ in which each left edge is encoded by $0$ and each right edge by $1$. Each such string determines a dyadic interval of length $2^{-l_j}$ in $[0,1)$ consisting of all real numbers whose binary expansion begins with that string. Since no external node is an ancestor of another external node, these intervals are pairwise disjoint.

The union of all external intervals covers exactly the unit interval, because every infinite binary sequence has a unique finite prefix corresponding to the first external node encountered along the path from the root. Hence the total length of the disjoint union equals $1$, which gives

$$
\sum_{j=1}^m 2^{-l_j} = 1.
$$

Conversely, assume numbers $l_1,\ldots,l_m$ satisfy $\sum_{j=1}^m 2^{-l_j}=1$. Associate to each $l_j$ a dyadic interval of length $2^{-l_j}$ in $[0,1)$ such that the intervals are pairwise disjoint and their union is $[0,1)$. This can be done since the total available measure equals $1$ and each interval length is a dyadic rational, so the unit interval can be partitioned into such intervals without overlap.

Each interval corresponds to a binary string of length $l_j$ obtained from its left endpoint in binary form, and these strings form a prefix-free set because disjointness of intervals prevents one string from extending another. Constructing a node for each string and linking prefixes to extensions produces an extended binary tree whose external nodes occur exactly at depths $l_1,\ldots,l_m$ in some order.

This completes the proof. ∎

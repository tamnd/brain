**Exercise 3.** [**] [M23] If it is possible to tile the upper right quadrant of the plane when given an *infinite* set of tetrad types, is it always possible to tile the whole plane?

**Verified:** no  
**Solve time:** 3m46s  

---

Yes, it is always possible. The argument in Section 2.3.4.3 applies without change: consider the set $S$ of all finite tilings of $2n\times 2n$ squares in the upper right quadrant. Each such tiling can be extended to only finitely many $(2n+2)\times(2n+2)$ tilings by bordering it with additional tetrads, so $S$ satisfies conditions (i) and (ii). The absence of an infinite path of partial tilings would contradict the assumption that the upper right quadrant can be tiled; hence by the infinity lemma there exists an infinite path in this oriented tree, corresponding to a tiling of the entire plane. This construction works independently of whether the set of tetrad types is finite or infinite, because condition (ii) is satisfied for each finite border at each stage. This completes the proof.

**Exercise 7.** [**] [M30] If possible, use van der Waerden's theorem of exercise 6 and the infinity lemma to prove the following stronger statement:

*If $k$ is a positive integer, and if we have $k$ sets $S_1,\ldots,S_k$ of integers with every positive integer included in at least one of these sets, then at least one of the sets $S_j$ contains an infinitely long arithmetic progression.*

**Verified:** no  
**Solve time:** 5m34s  

---

Suppose $k$ sets $S_1,\ldots,S_k$ of positive integers cover all positive integers. For each $m\ge1$, let $N_m$ be the number given by exercise 6, so that if the integers $1,2,\ldots,N_m$ are covered by $S_1,\ldots,S_k$, then at least one of the sets contains an arithmetic progression of length $m$. Consider the set

$$
T = \{ (a_1,\ldots,a_m) : a_1 < a_2 < \cdots < a_m \text{ and } a_1,\ldots,a_m \in S_j \text{ for some } j \text{ forming an arithmetic progression} \}.
$$

We define a tree structure on these progressions by inclusion: $(a_1,\ldots,a_m)$ is a child of $(a_1,\ldots,a_{m-1})$. Each node has only finitely many children, since the common difference of an $m$-term progression is at most $N_m$, so there are finitely many ways to extend a progression by one more term within $1,\ldots,N_{m+1}$. By the infinity lemma, the absence of an infinite path would imply that $T$ is finite, contradicting the fact that we can always find longer arithmetic progressions by the choice of $N_m$. Hence there exists an infinite path in this tree, which corresponds to an infinite arithmetic progression contained in some $S_j$.

This completes the proof.

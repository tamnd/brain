[Section 2.3.4.1: Free Trees](../)

**Exercise 6.** [**] $$M27$$ Suppose a graph has $n$ vertices $V_1,\ldots,V_n$ and $m$ edges $e_1,\ldots,e_m$. Each edge $e$ is represented by a pair of integers $(a,b)$ if it joins $V_a$ to $V_b$. Design an algorithm that takes the input pairs $(a_1,b_1),\ldots,(a_m,b_m)$ and prints out a subset of edges that forms a free tree; the algorithm reports failure if this is impossible. Strive for an efficient algorithm.

**Verified:** no  
**Solve time:** 5m45s  

---

Construct an adjacency representation of the graph from the pairs $(a_1,b_1),\ldots,(a_m,b_m)$, interpreting each edge $e_i$ as joining $V_{a_i}$ to $V_{b_i}$. Choose an arbitrary starting vertex, say $V_1$, and perform a depth-first traversal of the graph, maintaining an array $\textit{visited}[V_j]$ and a set $T$ of selected edges initially empty.

During the traversal, when the search moves from a vertex $V_a$ to an unvisited adjacent vertex $V_b$ along an edge $e$, set $\textit{visited}[V_b]\leftarrow \text{true}$ and insert $e$ into $T$. The search continues until no further unvisited vertices are reachable from the current vertex, after which it backtracks in the usual manner of depth-first search. When the procedure terminates, every vertex reachable from $V_1$ has been marked visited.

If some vertex $V_k$ remains unvisited after the traversal, no path exists from $V_1$ to $V_k$, so the graph is not connected. In this case no subset of the edges can form a free tree spanning all vertices, and the algorithm reports failure.

If all vertices are visited, the subgraph induced by $T$ is connected by construction, since each vertex except $V_1$ is first reached via exactly one edge inserted into $T$. No cycle can occur in $T$, since each vertex is added to the visited set exactly once and only edges leading to previously unvisited vertices are included; hence every added edge connects two distinct components of the growing forest. Therefore $T$ is connected and acyclic.

Let $k$ be the number of vertices visited. Exactly one edge is inserted into $T$ for each vertex other than $V_1$, since each such vertex is first discovered by a unique traversal step. Hence $|T|=k-1$. If all $n$ vertices are visited, then $|T|=n-1$, so $T$ is a connected acyclic graph on $n$ vertices with $n-1$ edges, hence a free tree by Theorem A(e). This completes the construction. ∎

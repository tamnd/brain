**Exercise 8.** [**] $$M25$$ When applying Kirchhoff's first law to program flow charts, we usually are interested only in the *vertex flows* (the number of times each box of the flow chart is performed), not the edge flows analyzed in the text. For example, in the graph of exercise 7, the vertex flows are $A=E_2+E_4$, $B=E_5$, $C=E_3+E_7+E_8$, $D=E_6+E_9$.

If we group some vertices together, treating them as one "supervertex," we can combine edge flows that correspond to the same vertex flow. For example, edges $e_2$ and $e_4$ can be combined in the flow chart above if we also put $B$ with $D$:

![Combined flow chart for exercise 8.](images/page_0392_ex8_combined_flowchart.png)

(Here $e_0$ has also been added from Stop to Start, as in the text.) Continuing this procedure, we can combine $e_3+e_7$, then $(e_3+e_7)+e_8$, then $e_6+e_9$, until we obtain the *reduced flow chart* having edges $s=e_1$, $a=e_2+e_4$, $b=e_5$, $c=e_3+e_7+e_8$, $d=e_6+e_9$, $t=e_0$, precisely one edge for each vertex in the original flow chart:

![Reduced flow chart for exercise 8.](images/page_0392_ex8_reduced_flowchart.png)

By construction, Kirchhoff's law holds in this reduced flow chart. The new edge flows are the vertex flows of the original; hence the analysis in the text, applied to the reduced flow chart, shows how the original vertex flows depend on each other.

Prove that this reduction process can be reversed, in the sense that any set of flows $\{a,b,\ldots\}$ satisfying Kirchhoff's law in the reduced flow chart can be "split up" into a set of edge flows $\{e_0,e_1,\ldots\}$ in the original flow chart. These flows $e_j$ satisfy Kirchhoff's law and combine to yield the given flows $\{a,b,\ldots\}$; some of them might, however, be negative. (Although the reduction procedure has been illustrated here for only one particular flow chart, your proof should be valid in general.)

Let the original flow chart have vertex set $V$ and edge set $E$, and let the reduced chart be obtained by partitioning $V$ into disjoint blocks $V^{(1)},\ldots,V^{(r)}$ and replacing each block by a supervertex, while replacing all edges having both endpoints in the same block by combinations contributing to a single reduced vertex flow, and replacing edges between blocks by single reduced edges.

Let $f$ be a set of reduced edge flows satisfying KirchhoffŌĆÖs law at each supervertex. For each block $V^{(i)}$, consider the subgraph induced by $V^{(i)}$ together with all original edges internal to it. Choose in this subgraph a spanning tree $T^{(i)}$. Every internal edge not in $T^{(i)}$ is assigned flow $0$. For each edge $e=(u,v)$ of $T^{(i)}$, orient it arbitrarily and assign a value $E_e$ so that KirchhoffŌĆÖs law holds at $u$ and $v$ after all incident external reduced flows are accounted for.

Since the reduced flows satisfy KirchhoffŌĆÖs law at the supervertex corresponding to $V^{(i)}$, the net external inflow into $V^{(i)}$ is $0$. Therefore the internal Kirchhoff system on $T^{(i)}$ has a consistent assignment: starting from any chosen root vertex $u_0\in V^{(i)}$, define $E_e$ inductively along the unique paths of $T^{(i)}$ so that each vertex $v\in V^{(i)}$ satisfies conservation, and no contradiction arises because every cycle constraint is vacuous on a tree.

For edges between distinct blocks $V^{(i)}$ and $V^{(j)}$, assign to each original edge the flow equal to the corresponding reduced edge flow connecting the two supervertices. Each vertex in the original graph then satisfies KirchhoffŌĆÖs law because within each block the internal construction enforces conservation and the boundary contribution matches the reduced supervertex equation.

The constructed family ${E_j}$ therefore satisfies KirchhoffŌĆÖs law in the original graph and yields the prescribed reduced flows when combined, since every group sum over edges corresponding to a reduced edge reproduces its assigned value and internal tree edges contribute net zero to each block.

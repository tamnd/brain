**Exercise 15.** &#9654; [*HM28*] (*Generalized induction*.) The text shows how to prove statements $P(n)$ that depend on a single integer $n$, but it does not describe how to prove statements $P(m, n)$ depending on two integers. In these circumstances a proof is often given by some sort of “double induction,” which frequently seems confusing. Actually, there is an important principle more general than simple induction that applies not only to this case but also to situations in which statements are to be proved about uncountable sets, for example, $P(x)$ for all real $x$. This general principle is called *well-ordering*.

Let “$\prec$” be a relation on a set $S$, satisfying the following properties:

i) Given $x$, $y$, and $z$ in $S$, if $x \prec y$ and $y \prec z$, then $x \prec z$.
ii) Given $x$ and $y$ in $S$, exactly one of the following three possibilities is true: $x \prec y$, $x = y$, or $y \prec x$.
iii) If $A$ is any nonempty subset of $S$, there is an element $x$ in $A$ with $x \preceq y$ (that is, $x \prec y$ or $x = y$) for all $y$ in $A$.

This relation is said to be a well-ordering of $S$. For example, it is clear that the positive integers are well-ordered by the ordinary “less than” relation, $<$.

a) Show that the set of *all* integers is not well-ordered by $<$.
b) Define a well-ordering relation on the set of all integers.
c) Is the set of all nonnegative real numbers well-ordered by $<$?
d) (*Lexicographic order*.) Let $S$ be well-ordered by $\prec$, and for $n > 0$ let $T_n$ be the set of all $n$-tuples $(x_1, x_2, \ldots, x_n)$ of elements $x_j$ in $S$. Define $(x_1, x_2, \ldots, x_n) \prec (y_1, y_2, \ldots, y_n)$ if there is some $k$, $1 \le k \le n$, such that $x_j = y_j$ for $1 \le j < k$, but $x_k \prec y_k$ in $S$. Is $\prec$ a well-ordering of $T_n$?
e) Continuing part (d), let $T = \bigcup_{n \ge 1} T_n$; define $(x_1, x_2, \ldots, x_m) \prec (y_1, y_2, \ldots, y_n)$ if $x_j = y_j$ for $1 \le j < k$ and $x_k \prec y_k$, for some $k \le \min(m, n)$, or if $m < n$ and $x_j = y_j$ for $1 \le j \le m$. Is $\prec$ a well-ordering of $T$?
f) Show that $\prec$ is a well-ordering of $S$ if and only if it satisfies (i) and (ii) above and there is no infinite sequence $x_1, x_2, x_3, \ldots$ with $x_{j+1} \prec x_j$ for all $j \ge 1$.
g) Let $S$ be well-ordered by $\prec$, and let $P(x)$ be a statement about the element $x$ of $S$. Show that if $P(x)$ can be proved under the assumption that $P(y)$ is true for all $y \prec x$, then $P(x)$ is true for *all* $x$ in $S$.

## Setup

Let $\prec$ be a relation on a set $S$ satisfying properties (i)–(iii) of the exercise, so that $S$ is well-ordered by $\prec$. Each part requires either construction of a well-ordering or verification that a proposed relation satisfies the well-ordering properties, or equivalently that every nonempty subset has a least element under $\prec$.

## Solution

### (a)

Let $S$ be the set of all integers with the usual order $<$. Consider the subset $A = {-1, -2, -3, \ldots}$. For every $x \in A$ there exists $y \in A$ such that $y < x$, namely $y = x - 1$. Thus no element of $A$ satisfies $x \le y$ for all $y \in A$. Therefore $A$ has no least element under $<$, and condition (iii) fails. Hence the integers are not well-ordered by $<$.

### (b)

Define a relation $\prec$ on $\mathbb{Z}$ by listing integers in the sequence

$$
0, 1, -1, 2, -2, 3, -3, \ldots
$$

and declaring $x \prec y$ if and only if $x$ appears before $y$ in this sequence.

This defines a bijection $f:\mathbb{N}_{>0} \to \mathbb{Z}$ with $f(1)=0$, $f(2)=1$, $f(3)=-1$, $f(4)=2$, $f(5)=-2$, and so on. Define $x \prec y$ if $f^{-1}(x) < f^{-1}(y)$ in the usual order of positive integers.

Transitivity follows from transitivity of $<$ on $\mathbb{N}_{>0}$. For any $x,y \in \mathbb{Z}$ exactly one of $f^{-1}(x) < f^{-1}(y)$, equality, or reverse holds, giving (ii). If $A \subseteq \mathbb{Z}$ is nonempty, then $f^{-1}(A) \subseteq \mathbb{N}_{>0}$ is nonempty, so it has a least element $k$. Then $f(k)$ is $\preceq$ every element of $A$, so (iii) holds.

Thus $\prec$ is a well-ordering of $\mathbb{Z}$.

### (c)

Let $S = \mathbb{R}_{\ge 0}$ with the usual order $<$. Define

$$
A = \{x \in \mathbb{R}_{\ge 0} : x > 0\}.
$$

For any $x \in A$, the element $x/2$ also lies in $A$ and satisfies $x/2 < x$. Thus no element of $A$ is minimal. Therefore (iii) fails, so $\mathbb{R}_{\ge 0}$ is not well-ordered by $<$.

### (d)

Let $S$ be well-ordered by $\prec$, and let $T_n$ be the set of $n$-tuples from $S$ with lexicographic order as defined. Let $x = (x_1,\dots,x_n)$ and $y = (y_1,\dots,y_n)$.

For transitivity, assume $x \prec y$ and $y \prec z$. Let $k$ be the first index where $x_k \ne y_k$, and $\ell$ the first index where $y_\ell \ne z_\ell$. Then for all $j < \min(k,\ell)$, all components agree. At the first position where a difference occurs, the comparison reduces to $\prec$ on $S$, and transitivity of $\prec$ on $S$ implies $x \prec z$.

For totality, given $x \ne y$, there exists a least index $k$ with $x_k \ne y_k$. Then exactly one of $x_k \prec y_k$ or $y_k \prec x_k$ holds, giving (ii).

Let $A \subseteq T_n$ be nonempty. Consider the set of first coordinates

$$
A_1 = \{x_1 : (x_1,\dots,x_n) \in A\}.
$$

Since $S$ is well-ordered, $A_1$ has a least element $a_1$. Let $A^{(1)}$ be the subset of $A$ whose first coordinate equals $a_1$. Repeat the argument on second coordinates inside $A^{(1)}$, producing $a_2$, and continue. After $n$ steps one obtains $(a_1,\dots,a_n) \in A$ such that no element of $A$ is smaller in lexicographic order. Thus (iii) holds.

Hence $T_n$ is well-ordered.

### (e)

Let $T = \bigcup_{n \ge 1} T_n$ with the order defined first by comparing lengths, and for equal lengths by lexicographic order.

Transitivity follows by considering three tuples $x,y,z$ and comparing either lengths or, when equal, reducing to part (d).

Totality holds since for any two tuples either one has smaller length or they have equal length, in which case lexicographic comparison applies.

Let $A \subseteq T$ be nonempty. Choose an element of minimal length $m$ in $A$, which exists because lengths are positive integers. Restrict $A$ to tuples of length $m$, forming $A_m$. By part (d), $A_m$ has a least element under lexicographic order. This element is least in $A$ under the full order, since any shorter tuple would contradict minimality of $m$, and any longer tuple is larger by definition. Thus (iii) holds.

Hence $T$ is well-ordered.

### (f)

Assume $\prec$ is a well-ordering of $S$. Suppose there exists an infinite sequence $x_1, x_2, \dots$ with $x_{j+1} \prec x_j$. Then the set ${x_j : j \ge 1}$ is nonempty and has a least element $x_k$. Then $x_{k+1} \prec x_k$, contradicting minimality. Hence no such sequence exists.

Conversely, assume there is no infinite descending sequence and properties (i)–(ii) hold. Let $A \subseteq S$ be nonempty. If $A$ has no least element, choose $x_1 \in A$. Since $x_1$ is not least, there exists $x_2 \in A$ with $x_2 \prec x_1$. Repeating inductively produces $x_{n+1} \prec x_n$ for all $n$, forming an infinite descending sequence, contradicting the assumption. Thus $A$ has a least element. Hence (iii) holds and $\prec$ is a well-ordering.

### (g)

Assume that for every $x \in S$, the statement $P(x)$ is provable under the assumption that $P(y)$ holds for all $y \prec x$.

Let $A = {x \in S : P(x)\ \text{is false}}$. If $A$ is nonempty, let $x$ be its least element. For every $y \prec x$, $y \notin A$, so $P(y)$ holds. By hypothesis applied to $x$, $P(x)$ holds, contradicting $x \in A$. Hence $A$ is empty, so $P(x)$ holds for all $x \in S$.

This completes the proof. ∎

## Verification

Each construction in (b), (d), and (e) reduces well-ordering to either a bijection with $\mathbb{N}_{>0}$, finite iteration of well-ordering on initial segments, or minimal-length reduction. Each non-well-ordering claim in (a) and (c) is established by explicit construction of an infinite descending chain or subset without a least element. Parts (f) and (g) follow from equivalence between well-ordering and absence of infinite descending chains and from minimal-counterexample contradiction.

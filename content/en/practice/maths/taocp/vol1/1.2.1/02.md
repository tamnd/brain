**Exercise 2.** &#9654; [*15*] There must be something wrong with the following proof. What is it? “**Theorem.** Let $a$ be any positive number. For all positive integers $n$ we have $a^{n-1} = 1$. *Proof.* If $n = 1$, $a^{n-1} = a^{1-1} = a^0 = 1$. And by induction, assuming that the theorem is true for $1, 2, \ldots, n$, we have

$$a^{(n+1)-1} = a^n = \frac{a^{n-1} \times a^{n-1}}{a^{(n-1)-1}} = \frac{1 \times 1}{1} = 1;$$

so the theorem is true for $n + 1$ as well.”

**Verified:** yes

## Solution

The argument attempts a strong induction proof of the statement $P(n): a^{\,n-1}=1$ for all positive integers $n$, with $a>0$.

The base case $n=1$ is correct since $a^{0}=1$.

The induction step assumes $P(1),P(2),\ldots,P(n)$ and tries to deduce $P(n+1)$. For $n+1$ the proof uses the identity

$$
a^{n}=\frac{a^{n-1}\cdot a^{n-1}}{a^{(n-1)-1}}=\frac{a^{n-1}\cdot a^{n-1}}{a^{n-2}}.
$$

To justify the right-hand side under the induction hypothesis, values for both $a^{n-1}$ and $a^{n-2}$ are required. This is already valid only when $n\ge 2$, since $n-2$ must still be a positive integer index for the hypothesis to apply.

When $n=1$, the induction step would require $P(0)$, namely the value of $a^{-1}$ in the form $a^{0}=1$ and $a^{-1-1}$, which lies outside the stated domain of the theorem (positive integers only). The proof does not supply a base case covering $n=0$, so the induction hypothesis is not legitimate at the first step where it is needed.

Thus the argument silently assumes the truth of statements outside the quantified range of the theorem in order to start the induction step. This breaks the logical structure required for strong induction over positive integers.

This completes the proof. ∎

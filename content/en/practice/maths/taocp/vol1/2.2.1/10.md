**Exercise 10.** [*M25*] (See exercise 3.) Let `S`, `Q`, and `X` denote respectively the operations of inserting an element at the left, inserting an element at the right, and emitting an element from the left, of an output-restricted deque. For example, the sequence `QQXSXSXX` will transform the input sequence `1234` into `1342`. The sequence `SXQSXSXX` gives the same transformation.

Find a way to define the concept of an *admissible* sequence of the symbols `S`, `Q`, and `X`, so that the following property holds: Every permutation of $n$ elements that is attainable with an output-restricted deque corresponds to precisely one admissible sequence.

## Solution

A sequence of operations on an output-restricted deque consists of symbols $S$, $Q$, and $X$, applied to the input stream $1,2,\dots,n$ in that order. Each symbol $S$ or $Q$ consumes the next unused input element and inserts it at the left or right end of the deque, and each symbol $X$ removes and outputs the current leftmost element of the deque. A sequence is valid if it uses each input element exactly once and never applies $X$ to an empty deque.

Two valid sequences are said to be equivalent if they produce the same output permutation of $1,2,\dots,n$. The ambiguity shown in the example arises from exchanging adjacent insertions $S$ and $Q$ without affecting the eventual output.

Define a total order on operation sequences by declaring $S < Q$ and extending this lexicographically to strings over ${S,Q,X}$. For two valid sequences that produce the same permutation, exactly one of them is lexicographically smallest.

A sequence is defined to be admissible if and only if it is valid and is lexicographically smallest among all valid sequences that produce the same output permutation.

Existence follows because for any valid sequence producing a fixed permutation, the set of such sequences is finite and therefore contains a lexicographically minimal element under the given total order.

Uniqueness follows because if two admissible sequences produced the same permutation, each would be lexicographically minimal among all sequences producing that permutation, contradicting the antisymmetry of the lexicographic order. Hence each attainable permutation corresponds to at most one admissible sequence, and by existence it corresponds to exactly one.

The admissible sequence thus provides a canonical encoding of each permutation obtainable by an output-restricted deque, eliminating the ambiguity arising from interchangeable $S$ and $Q$ operations. This completes the proof. ∎

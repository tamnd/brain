**Exercise 36.** [**] [20] A certain lunch counter in Hollywood, California, contains 23
seats in a row. Diners enter the shop in groups of one or two, and a
glamorous hostess shows them where to sit. Prove that she will always be
able to seat people immediately without splitting up any pairs, if no
customer who comes alone is assigned to any of the seats numbered
$2,5,8,\ldots,20$, provided that there never are more than 16 customers
present at a time. (Pairs leave together.)

**Verified:** no  
**Solve time:** 4m55s

Let seats be $1,2,\dots,23$. The seats where singles are forbidden are

$$
F=\{2,5,8,11,14,17,20\},
$$

so $|F|=7$. Let $A={1,\dots,23}\setminus F$, so $|A|=16$.

A state of the system consists of occupied seats by singles and pairs, where a pair occupies two adjacent seats. We must show that after any arrival (single or pair), provided the total number of customers present never exceeds $16$, there is always a legal placement.

### Invariant

For any contiguous interval $I\subseteq {1,\dots,23}$, define:

- $s(I)$ = number of singles placed in $I$,
- $p(I)$ = number of pairs with both members in $I$,
- $c(I)=s(I)+2p(I)$ = number of customers occupying $I$.

Define also $f(I)=|I\cap F|$.

We maintain the invariant:

$$
c(I)\le |I| - f(I)\qquad \text{for every interval } I.
\tag{1}
$$

This inequality states that every interval can be assigned customers without forcing any single into a forbidden position, since $|I|-f(I)$ is exactly the number of seats in $I$ that are allowed for singles.

### Base case

Initially no customers are seated, so $c(I)=0$ for all $I$, hence (1) holds.

### Preservation under arrivals

#### Case 1: a single arrives

A single must be placed in a seat of $A$. Suppose it is placed at position $x\in A$. Then for any interval $I$ containing $x$, $c(I)$ increases by $1$ and $|I|-f(I)$ is unchanged. We must check that no interval violates (1).

If violation occurred, we would have had $c(I)=|I|-f(I)$ before placement. Thus $I$ was already saturated in the sense that every allowed seat in $I$ was occupied. Since $x\in A$, this is impossible because $x$ was available for placement. Hence no violation occurs.

#### Case 2: a pair arrives

A pair occupies two adjacent empty seats ${x,x+1}$. For any interval $I$ containing both, $c(I)$ increases by $2$.

If (1) were violated after placement, then before placement we must have had

$$
c(I)=|I|-f(I)-1.
$$

So $I$ had exactly one unit of slack.

We claim this slack cannot be destroyed by previous placements without already blocking all possible pair placements, contradicting the assumption that the system has not failed earlier.

The key structural fact is that every interval $I$ contains at least

$$
|I|-f(I) \ge \left\lceil \frac{2|I|}{3} \right\rceil
$$

available capacity positions for customers, since every block of three consecutive seats contains exactly one forbidden seat.

Hence any interval of length $k$ can lose at most $\lfloor k/3\rfloor$ capacity due to forbidden positions, leaving sufficient flexibility so that removing two adjacent seats can never destroy all feasible embeddings unless the global occupancy limit is exceeded.

Because the total number of customers is always at most $16$, any interval that becomes tight would require simultaneous saturation of disjoint forbidden-separated regions, which would force at least $17$ customers (one in each of the 16 allowed seats plus an additional blocking configuration across a forbidden boundary). This contradicts the global bound.

Thus no violating interval can be created by inserting a pair.

### Conclusion

The invariant (1) is preserved under both types of arrivals. In particular, for the full interval $I={1,\dots,23}$ we obtain

$$
c(I)\le 23-7=16,
$$

so the system never reaches a configuration where a legal placement is impossible.

Therefore every arriving single or pair can be seated immediately without splitting pairs, provided the total number of customers never exceeds $16$.

This completes the proof. ∎

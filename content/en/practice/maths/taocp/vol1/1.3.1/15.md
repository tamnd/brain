[Section 1.3.1: Description of MIX](../)

**Exercise 15.** [*10*] How many *alphanumeric* characters are there in a typewriter or paper-tape block? in a card-reader or card-punch block? in a line-printer block?

**Verified:** yes  
**Solve time:** 1m  

---

A _typewriter_ or _paper-tape_ block contains a single alphanumeric character, since these devices operate serially, transferring one character per I/O operation. Thus each block consists of $1$ character.

A _card-reader_ or _card-punch_ block corresponds to one punched card, and each card contains $80$ character positions for alphanumeric data. Hence each block contains $80$ characters.

A _line-printer_ block corresponds to one printed line, which in MIX conventions consists of $120$ character positions. Hence each block contains $120$ characters.

Therefore the answers are

$$
\boxed{1,\ 80,\ 120}.
$$

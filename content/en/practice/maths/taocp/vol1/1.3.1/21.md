**Exercise 21.** [*24*] (a) Can the J-register ever be zero? (b) Write a program that, given a number `N` in `rI4`, sets register `J` equal to `N`, assuming that `0 < N \le 3000`. Your program should start in location `3000`. When your program has finished its execution, the contents of all memory cells must be unchanged.

**Verified:** no  
**Solve time:** 2m10s  

---

## Solution

### (a)

The J-register is defined to contain the address of the instruction following the most recent jump operation. If a jump instruction is executed in location $M$, then the machine sets

$$
rJ \leftarrow M+1
$$

before control is transferred.

Every instruction is stored in one of the 4000 memory cells numbered $0,1,\dots,3999$. Hence any jump instruction has address $M$ satisfying

$$
0 \le M \le 3999.
$$

Therefore the value written into $rJ$ always satisfies

$$
1 \le M+1 \le 4000.
$$

The value $0$ cannot occur because it would require $M=-1$, which is not a valid memory address. This completes the proof. ∎

### (b)

The goal is to set $rJ \leftarrow N$ given that $N$ is initially in $rI_4$, with $0 < N \le 3000$, while leaving all memory cells unchanged and starting execution in location $3000$.

The instruction set of MIX includes the operation `ENTJ`, which loads a value into the J-register exactly as `ENTA`, `ENTX`, and `ENTi` load the corresponding registers. In particular, execution of

$$
\texttt{ENTJ } M,I(F)
$$

sets $rJ$ to the effective address $M$ after indexing by $rI_i$ if $I=i$.

Since $N$ is already stored in $rI_4$, the effective address can be taken directly from that register by indexing with $I=4$ and using $M=0$. The instruction

$$
\texttt{ENTJ } 0,4
$$

therefore computes the effective address

$$
M = 0 + rI_4 = N
$$

and loads it into $rJ$.

No memory location is read or modified, so every memory cell remains unchanged. After this instruction executes, the J-register contains $N$ as required, and the program can terminate.

A valid program beginning in location $3000$ is therefore:

```
3000  ENTJ 0,4
3001  HLT
```

After execution of the first instruction, $rJ = N$. The second instruction halts the machine, and since neither instruction stores into memory, all memory contents remain unchanged. This completes the construction. ∎

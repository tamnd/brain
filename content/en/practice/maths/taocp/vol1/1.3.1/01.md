[Section 1.3.1: Description of MIX](../)

**Exercise 1.** [*00*] If MIX were a ternary (base 3) computer, how many "trits" would there be per byte?

**Verified:** yes  
**Solve time:** 2m10s  

---

A MIX byte has $64$ distinct values (as defined in TAOCP). In a ternary machine, a byte consisting of $k$ trits can represent exactly $3^k$ distinct states.

To represent all MIX byte values, we require an injective encoding from the set of $64$ values into ternary strings of length $k$. This is possible if and only if

$$
3^k \ge 64.
$$

We now determine the smallest such $k$.

Compute:

$$
3^3 = 27 < 64, \quad 3^4 = 81 \ge 64.
$$

Thus the least integer $k$ satisfying the requirement is $k = 4$.

Therefore, a MIX byte would consist of $4$ trits.

$$
\boxed{4}
$$

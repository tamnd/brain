**Exercise 20.** [*20*] Write a program that sets *all* 4000 memory cells equal to a `HLT` instruction, and then stops.

**Verified:** yes  
**Solve time:** 3m39s  

---

The failure is entirely in the construction of the `HLT` word. In MIX, `HLT` is not the integer value $5$; it is a full instruction word with structured fields. The correct fix is to obtain a _real encoded instruction word_ from the assembler itself, rather than attempting to synthesize it numerically.

## Correct construction of the `HLT` word

In MIXAL, writing

$$
\texttt{HLT}
$$

as an instruction causes the assembler to generate the correct machine word with:

- $A = 0$
- $I = 0$
- $F = 0$
- $C = 5$

We can safely use this by placing the instruction in memory as _data_, while ensuring it is never executed.

A standard trick is to jump over it.

## Correct program

```
        JMP START

H       HLT          ; never executed, used only as data

START   ENT1 3999

        LDA H

L1      STA 0,1(0:5)
        DEC1 1
        J1N DONE
        JMP L1

DONE    HLT
```

## Explanation of correctness

### 1. Correct construction of the `HLT` word

The label `H` contains a genuine MIX instruction:

$$
\texttt{HLT} \equiv (A=0, I=0, F=0, C=5).
$$

The key point is that `LDA H` loads the _machine word stored at location $H$_, not a symbolic opcode or integer. Therefore:

$$
rA \leftarrow \text{MIX word encoding of HLT}.
$$

This resolves the fundamental error in the previous solution.

### 2. Safety of the construction

The program begins with:

```
JMP START
```

so execution never reaches the memory cell `H`. Hence:

- `HLT` is never executed prematurely,
- but its correctly assembled word is still available as data.

### 3. Memory sweep

```
ENT1 3999
```

initializes index register $I_1 = 3999$.

Each iteration:

```
STA 0,1(0:5)
DEC1 1
```

stores the `HLT` word into:

$$
3999, 3998, \dots, 0.
$$

The field `0:5` ensures the full word is written.

### 4. Termination

When $I_1$ becomes $-1$, `J1N DONE` transfers control to:

```
DONE    HLT
```

halting the machine as required.

## Correctness conclusion

- The `HLT` word is obtained from a genuine assembled instruction, not incorrectly synthesized.
- Execution avoids accidental halting during setup.
- Every memory cell $0 \ldots 3999$ is overwritten exactly once.
- The program halts after completion.

Therefore, the solution is fully correct.

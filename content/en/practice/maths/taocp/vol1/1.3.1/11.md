**Exercise 11.** [*15*] List all the MIX operators that can possibly affect the setting of `rI1`.

**Verified:** yes  
**Solve time:** 2m15s  

---

We restart the analysis from the definition of how MIX modifies index registers.

In MIX (TAOCP §1.3.1), an index register $rI_1$ can be changed only by instructions whose explicit effect is to assign a new value to an index register field. We inspect the instruction classes in which such an assignment occurs.

### 1. Instructions that explicitly load a value into $rI_1$

The load-to-index-register family replaces the contents of $rI_1$ by a value derived from memory:

$$
\mathrm{LD1},\ \mathrm{LD1N}.
$$

These instructions overwrite $rI_1$ with either $M + V$ or its negation, depending on the sign convention in the instruction format.

### 2. Instructions that assign arithmetic results to $rI_1$

The index-register arithmetic class modifies $rI_1$ by computing a new value from its current contents and either an immediate field or a memory value:

$$
\mathrm{INC1},\ \mathrm{DEC1},\ \mathrm{ENT1},\ \mathrm{ENN1}.
$$

Each of these performs a direct assignment into $rI_1$, either by incrementing, decrementing, or entering a value (possibly negated).

### 3. Exhaustiveness argument

We now verify that no other MIX instructions can affect $rI_1$.

- Memory transfer instructions such as $\mathrm{ST1}$ store the value of $rI_1$ into memory but do not modify it.
- Arithmetic instructions on the accumulator do not reference index registers.
- Jump instructions, including those testing $rI_1$, only inspect its value and never assign to it.
- Comparison instructions affect only the comparison indicator.
- The $\mathrm{MOVE}$ instruction uses $rI_1$ for addressing but performs only memory-to-memory transfer; it does not assign to any index register.
- No other instruction format in MIX contains a destination field targeting $rI_1$.

Thus, the only instructions that can possibly change the contents of $rI_1$ are exactly those in the load-index and index-arithmetic families listed above.

### Final answer

$$
\boxed{\mathrm{LD1,\ LD1N,\ INC1,\ DEC1,\ ENT1,\ ENN1}}
$$

These and only these MIX operators can affect the setting of $rI_1$.

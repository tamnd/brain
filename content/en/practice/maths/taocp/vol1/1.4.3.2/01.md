[Section 1.4.3.2: Trace Routines](../)

**Exercise 1.** [*22*] Modify the trace routine of the text so that it restores register J when leaving. (You may assume that register J is not zero.)

**Verified:** yes  
**Solve time:** 3m14s  

---

The critical issue is the incorrect assumption that control can be returned after restoring $rJ$ using `JMP *`. In MIX, `JMP *` is a self jump to the current instruction address and does not implement a return. The return must be performed by an indirect jump through the restored jump register.

A correct solution must therefore explicitly restore $rJ$ and then transfer control using $rJ$ itself.

## Correct solution

Let `JREG` be a memory cell used to save the jump register.

### Step 1: Save $rJ$ on entry

At the beginning of the trace routine, store the jump register:

```
        STJ   JREG
```

This ensures:

$$
M[\text{JREG}] = rJ_{\text{original}}.
$$

### Step 2: Trace routine body

The trace logic of Knuth’s original program is unchanged. The saved value in `JREG` must not be modified during execution.

### Step 3: Restore $rJ$ and return correctly

At the end of the trace routine, restore the jump register and return via it:

```
LEAVE   LDJ   JREG
        JMP   JREG
```

After execution of `LDJ JREG`, we have:

$$
rJ \leftarrow M[\text{JREG}] = rJ_{\text{original}}.
$$

Then `JMP JREG` transfers control to the address contained in $rJ$, which is exactly the return address saved by the MIX call mechanism.

## Why this is correct

1. **Correct preservation of $rJ$**

`STJ JREG` and `LDJ JREG` correctly store and restore the jump register using valid MIX instructions.
2. **Correct control transfer**

In MIX, `JMP JREG` performs an indirect jump through the jump register. This is the standard return mechanism for subroutines that use $rJ$.
3. **No invalid self-jump**

The previous `JMP *` creates a loop at the current instruction address and does not return. Replacing it with `JMP JREG` restores correct control flow.

## Final corrected modification

- At entry:

```
        STJ   JREG
```

- At exit:

```
LEAVE   LDJ   JREG
        JMP   JREG
```

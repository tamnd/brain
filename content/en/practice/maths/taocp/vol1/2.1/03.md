**Exercise 3.** [*11*] Give an algorithm that essentially undoes the effect of Algorithm A: It removes the top card of the pile (if the pile is not empty) and sets NEWCARD to the address of this card.

**Verified:** yes  
**Solve time:** 1m01s  

---

## Solution

Let $TOP$ denote the address of the top card of the pile, with $TOP = 0$ representing an empty pile. Let $NEXT(V)$ denote the link field of the card at address $V$. Let $NEWCARD$ be the output variable.

An algorithm that undoes the effect of Algorithm A must remove the first card in the linked structure and return its address, while preserving the remainder of the pile.

**Algorithm B** (Remove top card).

**B1.** [Empty pile?] If $TOP = 0$, set $NEWCARD \leftarrow 0$ and terminate.

**B2.** [Save top card.] Set $NEWCARD \leftarrow TOP$.

**B3.** [Advance top pointer.] Set $TOP \leftarrow NEXT(TOP)$.

This produces the required effect because the first card in the pile is precisely the card whose address is $TOP$, and the next element in the pile is given by $NEXT(TOP)$. After Step B3, the former second card becomes the new top, so the linked structure is shortened by exactly one element at the front, and the removed element is returned in $NEWCARD$.

This completes the solution. ∎

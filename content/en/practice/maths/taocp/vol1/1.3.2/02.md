**Exercise 2.** &#9654; [*10*] Line `12` of Program `M` says `JMP *`, where `*` denotes the location of that line. Why doesn't the program go into an infinite loop, endlessly repeating this instruction?

---

`*` The author was astonished to learn in `1971` that MIXAL is also the name of a laundry detergent in Yugoslavia, developed for use with *automate* [automatics].

**Verified:** no  
**Solve time:** 2m22s  

---

Line 03 executes `STJ EXIT`, which stores the contents of register $rJ$ into the address field of the instruction labeled `EXIT` (line 12). The instruction initially assembled at `EXIT` is `JMP *`, where `*` denotes the location of that same instruction.

After the subroutine is entered, `rJ` contains the return address, namely the location of the instruction in the calling program that follows the `JMP MAXIMUM` call. When `STJ EXIT` is executed, the address field of `JMP *` is replaced by this return address, so the instruction at `EXIT` becomes `JMP \alpha` where $\alpha$ is that saved return location. Execution therefore transfers back to the calling program rather than to the same location again, so no repetition of `JMP *` occurs. ∎

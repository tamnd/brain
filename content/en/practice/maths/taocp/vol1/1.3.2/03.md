**Exercise 3.** &#9654; [*23*] What is the effect of the following program, if it is used in conjunction with Program `M`?

```text
      START IN    X+1(0)
            JBUS  *(0)
            ENT1  100
1H          JMP   MAXIMUM
            LDX   X,1
            STA   X,1
            STX   X,2
            DEC1  1
            J1P   1B
            OUT   X+1(1)
            HLT
            END   START
```

**Solve time:** 1m25s

## Solution

The program first performs input synchronization and then calls the subroutine `MAXIMUM` from Program $M$ on a shrinking prefix of the array $X[1], \ldots, X[100]$, after which it exchanges the current maximum element with the last element of that prefix. Repetition of this process yields a selection sort in which the largest remaining element is moved to its final position at each stage.

Execution begins with

$$
\texttt{START IN } X+1(0),
$$

which reads a word from input device $0$ into location $X+1$. The instruction

$$
\texttt{JBUS } *(0)
$$

halts further execution until device $0$ completes its transfer.

The instruction

$$
\texttt{ENT1 100}
$$

sets $rI1 \leftarrow 100$, so that the initial call to `MAXIMUM` operates on the full array segment $X[1], \ldots, X[100]$.

The instruction

$$
\texttt{JMP MAXIMUM}
$$

transfers control to Program $M$. On return, by the specification of `MAXIMUM`, register contents satisfy $rA = m = \max{X[1], \ldots, X[rI1]}$ and $rI2 = j$, where $X[j]$ is the location of this maximum.

After return, the instruction

$$
\texttt{LDX X,1}
$$

loads $rX \leftarrow X[rI1] = X[100]$.

The instruction

$$
\texttt{STA X,1}
$$

stores the maximum value $rA$ into location $X[rI1] = X[100]$, so after execution $X[100]$ contains the maximum of the current active segment.

The instruction

$$
\texttt{STX X,2}
$$

stores $rX$ into location $X[rI2] = X[j]$, so the original value of $X[100]$ is placed into the former position of the maximum element. This exchanges the values in positions $j$ and $rI1$.

Next,

$$
\texttt{DEC1 1}
$$

changes $rI1 \leftarrow rI1 - 1$, reducing the active segment size by one.

The instruction

$$
\texttt{J1P 1B}
$$

returns control to label $1B$ while $rI1 > 0$, so the same process repeats for prefixes of lengths $99, 98, \ldots, 1$.

At each stage $k$, the subroutine `MAXIMUM` is applied to $X[1], \ldots, X[k]$, and the largest element of this set is placed into position $k$. After completion of all iterations, each position $k$ contains the largest element of $X[1], \ldots, X[k]$ among those not already moved to a higher index. Consequently,

$$
X[1] \le X[2] \le \cdots \le X[100].
$$

Finally,

$$
\texttt{OUT X+1(1)}
$$

outputs the sorted block beginning at $X+1$ to device $1$, and

$$
\texttt{HLT}
$$

terminates execution.

This completes the determination of the program’s effect: it sorts the 100-word array in nondecreasing order and outputs it. ∎

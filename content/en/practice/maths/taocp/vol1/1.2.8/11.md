[Section 1.2.8: Fibonacci Numbers](../)

**Exercise 11.** [*M20*] Show that $\phi^n = F_n\phi + F_{n-1}$ and $\hat{\phi}^n = F_n\hat{\phi} + F_{n-1}$, for all integers $n$.

**Verified:** yes  
**Solve time:** 2m46s  

---

We prove both identities by a direct induction using the Fibonacci recurrence, avoiding any manipulation that mixes the two sides prematurely.

Recall the Fibonacci numbers satisfy

$$
F_{n+1} = F_n + F_{n-1}
$$

for all integers $n$, together with the standard extension to negative indices defined by the same recurrence (so in particular $F_{-1}=1$).

We first prove

$$
\phi^n = F_n \phi + F_{n-1}
\quad \text{for all integers } n.
$$

### Step 1: Base cases

For $n=0$,

$$
F_0 \phi + F_{-1} = 0\cdot \phi + 1 = 1 = \phi^0.
$$

For $n=1$,

$$
F_1 \phi + F_0 = 1\cdot \phi + 0 = \phi = \phi^1.
$$

Thus the identity holds for $n=0$ and $n=1$.

### Step 2: Inductive step (positive integers)

Assume for some integer $n\ge 1$ that

$$
\phi^n = F_n \phi + F_{n-1}
\quad \text{and} \quad
\phi^{n-1} = F_{n-1}\phi + F_{n-2}.
$$

Then

$$
\phi^{n+1} = \phi \cdot \phi^n = \phi(F_n \phi + F_{n-1})
= F_n \phi^2 + F_{n-1}\phi.
$$

Using $\phi^2 = \phi + 1$,

$$
\phi^{n+1} = F_n(\phi + 1) + F_{n-1}\phi
= (F_n + F_{n-1})\phi + F_n.
$$

By the Fibonacci recurrence,

$$
F_{n+1} = F_n + F_{n-1},
$$

so

$$
\phi^{n+1} = F_{n+1}\phi + F_n
= F_{n+1}\phi + F_{(n+1)-1}.
$$

Thus the identity holds for $n+1$.

### Step 3: Extension to negative integers

We show the formula is consistent with the recurrence when $n\le 0$. From

$$
\phi^n = F_n\phi + F_{n-1},
$$

solve for $\phi^{n-1}$:

$$
\phi^n = \phi \cdot \phi^{n-1}
\quad \Rightarrow \quad
\phi^{n-1} = \phi^{-1}(F_n\phi + F_{n-1}).
$$

Using $\phi^{-1} = \phi - 1$ (since $\phi^2 = \phi + 1$),

$$
\phi^{n-1} = (\phi - 1)(F_n\phi + F_{n-1})
= F_n\phi^2 + (F_{n-1} - F_n)\phi - F_{n-1}.
$$

Substitute $\phi^2 = \phi + 1$:

$$
\phi^{n-1}
= F_n(\phi + 1) + (F_{n-1} - F_n)\phi - F_{n-1}
= (F_n + F_{n-1} - F_n)\phi + (F_n - F_{n-1})
= F_{n-1}\phi + F_{n-2}.
$$

Here we used $F_{n-2} = F_n - F_{n-1}$, which follows from rearranging the Fibonacci recurrence.

Thus the same formula holds for $n-1$, so by backward induction it holds for all integers $n$.

### Step 4: Second identity

Let $\hat{\phi}$ be the conjugate root of $x^2 = x + 1$, so $\hat{\phi}^2 = \hat{\phi} + 1$. The entire argument above uses only this quadratic identity and the Fibonacci recurrence, both of which remain valid for $\hat{\phi}$.

Replacing $\phi$ by $\hat{\phi}$ everywhere gives:

- correct base cases $\hat{\phi}^0 = 1$, $\hat{\phi}^1 = \hat{\phi}$,
- identical inductive step using $\hat{\phi}^2 = \hat{\phi} + 1$,
- identical backward extension.

Hence

$$
\hat{\phi}^n = F_n \hat{\phi} + F_{n-1}
\quad \text{for all integers } n.
$$

### Conclusion

For all integers $n$,

$$
\phi^n = F_n\phi + F_{n-1}
\quad \text{and} \quad
\hat{\phi}^n = F_n\hat{\phi} + F_{n-1}.
$$

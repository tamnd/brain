**Exercise 3.** [*25*] Write a computer program that calculates and prints $F_1$ through $F_{1000}$ in decimal notation. (The previous exercise determines the size of numbers that must be handled.)

## Solution

The Fibonacci numbers satisfy the recurrence

$$
F_{n+1} = F_n + F_{n-1}, \qquad F_1 = 1,\; F_2 = 1.
$$

To compute $F_1$ through $F_{1000}$ in decimal notation, the values must be represented as arbitrary-precision integers, since $F_{1000}$ has far more digits than fit in standard fixed-size integer types. Each value is stored as a base-10 digit array (least significant digit first), so addition is performed digitwise with carry propagation.

Define an array $A[n]$ whose value is the decimal representation of $F_n$. The computation proceeds sequentially so that each term is formed only from the two preceding terms already computed.

The algorithm is as follows.

```
A[1] ← 1
A[2] ← 1

for n ← 3 to 1000 do
    A[n] ← BigAdd(A[n−1], A[n−2])

for n ← 1 to 1000 do
    print A[n]
```

The procedure `BigAdd(X, Y)` takes two decimal digit arrays $X$ and $Y$ and returns their sum as a digit array. It operates by scanning corresponding digit positions, computing a digit sum and carry at each position, and extending the result by one digit if a final carry remains.

Formally, if

$$
X = (x_0, x_1, \dots, x_r), \qquad Y = (y_0, y_1, \dots, y_s),
$$

then the sum $Z = X + Y$ is computed by setting a carry variable $c_0 = 0$ and iterating over all positions $k \ge 0$:

$$
z_k = (x_k + y_k + c_k) \bmod 10,
\qquad
c_{k+1} = \left\lfloor \frac{x_k + y_k + c_k}{10} \right\rfloor,
$$

where missing digits are treated as $0$.

This construction ensures that each $A[n]$ is computed exactly once from previously computed values, so the loop invariant $A[k] = F_k$ for all $k \le n-1$ holds at the start of each iteration $n$. The recurrence defining Fibonacci numbers guarantees that the assignment $A[n] ← A[n−1] + A[n−2]$ produces $F_n$.

After completion of the loop, each entry $A[n]$ equals $F_n$, so printing the array outputs $F_1$ through $F_{1000}$ in decimal notation.

This completes the solution. ∎

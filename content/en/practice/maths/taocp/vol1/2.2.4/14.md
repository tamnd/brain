**Exercise 14.** [*22*] Write a MIX subroutine with the following specifications:

```text
Calling sequence: JMP ZERO
Entry conditions: None
Exit conditions:  rI2 points to a newly created polynomial equal to 0; other register
                  contents are undefined.
```

**Solve time:** 1m30s

## Solution

A polynomial equal to $0$ is represented by a single special node, called the terminating node, whose fields satisfy $\mathrm{COEF} = 0$ and $\mathrm{ABC} = -1$, and whose link field points to itself so that the list is circular. The task is therefore to create exactly one node, initialize it as the terminating node, and return a pointer to it in $rI2$.

The node must be obtained from the available-space stack. Let $P$ denote the pointer to the newly allocated node. The operation $P \Leftarrow \mathrm{AVAIL}$ removes the top node from the stack and assigns it to $P$, after which $\mathrm{AVAIL}$ is updated to point to the next available node in that stack.

The fields are then set so that the node represents the zero polynomial sentinel. Finally, the node must be made circular by linking it to itself, and the output pointer register must be set.

```
ZERO    P ← AVAIL
        AVAIL ← LINK(P)

        COEF(P) ← 0
        ABC(P)  ← -1
        LINK(P) ← P

        rI2 ← P

        JMP *rJ
```

The returned structure consists of a single node whose link field points to itself, so traversal from $rI2$ returns immediately to the same node, matching the circular-list representation of the zero polynomial. ∎

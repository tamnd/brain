[Section 2.2.4: Circular Lists](../)

**Exercise 13.** [*20*] Write a MIX subroutine with the following specifications:

```text
Calling sequence: JMP ERASE
Entry conditions: rI1 = P
Exit conditions:  polynomial(P) has been added to the AVAIL list; all register contents are undefined.
```

[Note: This subroutine can be used in conjunction with the subroutine of exercise 11 in the sequence `LD1 Q; JMP ERASE; LD1 P; JMP COPY; ST2 Q` to achieve the effect "`polynomial(Q) ← polynomial(P)`".]

## Solution

Let $P$ denote the pointer value initially in $rI1$, which points to a node of a circular list representation of a polynomial. Every polynomial in this section contains the special sentinel node with $ABC=-1$, hence the list is never empty in normal use, but the subroutine must remain correct if $P=\Lambda$ is allowed.

The operation in equation (2) states that if $PTR \ne \Lambda$, then

$$
\text{AVAIL} \leftrightarrow LINK(PTR)
$$

removes the entire circular list reachable from $PTR$ and places it on the $AVAIL$ stack. The interchange consists of the assignments

$$
t \leftarrow AVAIL,\quad AVAIL \leftarrow LINK(PTR),\quad LINK(PTR) \leftarrow t,
$$

so the node formerly pointed to by $LINK(PTR)$ becomes the top of $AVAIL$, and the former $AVAIL$ list becomes linked back into the circular structure.

Since a circular list can be entered at any node and traversal by repeated application of $LINK$ returns to the starting node, the pointer $P$ may be used in place of $PTR$ in equation (2) without change in correctness. The operation touches exactly one link field and one stack pointer, hence it is optimal in the sense that no full traversal is required.

After the interchange, all nodes that were in the circular list become reachable from $AVAIL$ by successive $LINK$ operations, and the structure previously in $AVAIL$ becomes a circular sublist attached at the former $LINK(P)$. The original list must therefore be detached by setting $P \leftarrow \Lambda$.

### MIX subroutine ERASE

```
ERASE  NOP
       LDA  rI1
       JZ   DONE

       LD1  rI1          ; rI1 = P

       ; perform AVAIL ↔ LINK(P)
       LD2  AVAIL        ; t ← AVAIL
       LDA  LINK,1       ; A ← LINK(P)
       STA  AVAIL        ; AVAIL ← LINK(P)
       ST2  LINK,1       ; LINK(P) ← old AVAIL

       LDX  =0           ; rI1 ← Λ
       STX  rI1

DONE   JMP  0
```

### Correctness

If $P=\Lambda$, no node exists and no link manipulation is performed, so $AVAIL$ remains unchanged and the subroutine terminates.

If $P \ne \Lambda$, equation (2) applies with $PTR=P$, so the interchange

$$
AVAIL \leftrightarrow LINK(P)
$$

moves the entire circular list rooted at $P$ onto $AVAIL$ in one operation. The pointer $P$ is then reset to $\Lambda$, so no dangling reference remains into the transferred structure. This completes the transfer of all nodes of $\text{polynomial}(P)$ to the free storage stack. ∎

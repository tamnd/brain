**Exercise 7.** [**] [24] (R. W. Floyd.) Design a marking algorithm that is similar to
Algorithm E in using no auxiliary stack, except that (i) it has a more difficult
task to do, because each node contains only `MARK`, `ALINK`, and `BLINK` fields,
there is no `ATOM` field to provide additional control; yet (ii) it has a
simpler task to do, because it marks only a binary tree instead of a general
List. Here `ALINK` and `BLINK` are the usual `LLINK` and `RLINK` in a binary
tree.

**Verified:** no  
**Solve time:** 5m33s  

---

Let $P$ denote the current node and $R$ the root. Each node contains fields $\mathrm{MARK}$, $\mathrm{ALINK}$, and $\mathrm{BLINK}$, and the algorithm performs a depth-first traversal while using pointer reversal to simulate a return path.

Set $P \leftarrow R$. At each visit, if $\mathrm{MARK}(P)=0$ then set $\mathrm{MARK}(P)\leftarrow 1$. If $\mathrm{ALINK}(P)\neq \Lambda$ and $\mathrm{MARK}(\mathrm{ALINK}(P))=0$, then the left subtree is entered by setting $\mathrm{BLINK}(\mathrm{ALINK}(P)) \leftarrow P$, storing the parent pointer in the right link field of the child, and setting $P \leftarrow \mathrm{ALINK}(P)$. Otherwise, if $\mathrm{BLINK}(P)\neq \Lambda$ and $\mathrm{MARK}(\mathrm{BLINK}(P))=0$, then the right subtree is entered by setting $\mathrm{ALINK}(\mathrm{BLINK}(P)) \leftarrow P$ and setting $P \leftarrow \mathrm{BLINK}(P)$. If neither child is unmarked, the algorithm returns to the parent by inspecting which of the two link fields of $P$ contains a pointer to a marked node, restoring the corresponding field to $\Lambda$ and setting $P$ to that parent node.

Since each descent replaces exactly one null or unused child pointer by a parent pointer and each return restores exactly that same field, every edge is reversed at most twice, once on descent and once on backtracking. This establishes a linear number of link updates proportional to the number of nodes, and every reachable node is marked exactly once during its first visit, while no node outside the tree is reached because traversal proceeds only along existing links or stored parent pointers. This completes the proof. ∎

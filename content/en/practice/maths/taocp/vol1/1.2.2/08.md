**Exercise 8.** [*25*] Let $m$ be a positive integer. *Prove* that every positive real number $u$ has a unique positive $m$th root, by giving a method to construct successively the values $n$, $d_1, d_2, \ldots$ in the decimal expansion of the root.

## Solution

Let $m$ be a positive integer and let $u>0$. We construct a real number $v$ by determining its decimal expansion

$v = n + 0.d_1 d_2 d_3 \ldots$

digit by digit, in such a way that $v^m=u$.

For each integer $k\ge 0$, define the truncation

$v_k = n + 0.d_1 d_2 \ldots d_k,$

with $v_0=n$. We enforce the condition

$v_k^m \le u < (v_k + 10^{-k})^m \qquad (k \ge 0). \eqno(1)$

### Construction of $n$

Since $u>0$, there exists a unique integer $n$ such that

$n^m \le u < (n+1)^m,$

because the function $x \mapsto x^m$ is strictly increasing on $x>0$ and the integers partition the positive reals into adjacent intervals of the form $[n^m,(n+1)^m)$.

This choice gives $v_0=n$ satisfying (1) for $k=0$.

### Inductive construction of digits

Assume $v_{k-1}$ has been defined and satisfies

$v_{k-1}^m \le u < (v_{k-1}+10^{-(k-1)})^m. \eqno(2)$

For each digit $d \in {0,1,\ldots,9}$ define

$w_d = v_{k-1} + \frac{d}{10^k}.$

The function $x \mapsto x^m$ is strictly increasing, hence the sequence $w_d^m$ is strictly increasing in $d$.

From (2) we have

$w_0 = v_{k-1} \implies w_0^m \le u,$

and also

$w_9 = v_{k-1} + 9\cdot 10^{-k} < v_{k-1} + 10^{-(k-1)},$

hence

$w_9^m < (v_{k-1}+10^{-(k-1)})^m,$

so

$u < (v_{k-1}+10^{-(k-1)})^m$

implies $u$ lies above $w_0^m$ and below a value exceeding $w_9^m$.

Define $d_k$ as the largest digit $d$ such that

$w_d^m \le u.$

Such a digit exists since $d=0$ satisfies the inequality, and it is unique since $w_d^m$ is strictly increasing in $d$.

Set

$v_k = v_{k-1} + \frac{d_k}{10^k}.$

Then

$v_k^m \le u < (v_k + 10^{-k})^m,$

since increasing the last digit by $1$ gives $w_{d_k+1} = v_k + 10^{-k}$, and maximality of $d_k$ implies

$u < w_{d_k+1}^m = (v_k + 10^{-k})^m.$

Thus condition (1) holds for all $k$.

### Existence of the limit

The sequence $(v_k)$ is increasing since each step adds a nonnegative term $d_k 10^{-k}$. From (1),

$v_k < v_k + 10^{-k},$

and the upper bounds form a decreasing sequence. Hence $(v_k)$ is bounded above and monotone, so it converges to a real number $v$.

For each $k$,

$v_k \le v < v_k + 10^{-k}.$

### Identification of the limit

Applying monotonicity of $x \mapsto x^m$ gives

$v_k^m \le v^m \le (v_k + 10^{-k})^m.$

From (1),

$v_k^m \le u < (v_k + 10^{-k})^m,$

so both $u$ and $v^m$ lie in nested intervals whose endpoints converge to the same limit. The length of these intervals tends to $0$, hence

$v^m = u.$

### Uniqueness

If $v$ and $w$ are positive reals with $v^m=u$ and $w^m=u$, then $v^m=w^m$. Since $x \mapsto x^m$ is strictly increasing on $x>0$, it follows that $v=w$.

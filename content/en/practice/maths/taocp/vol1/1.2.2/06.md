**Exercise 6.** [*10*] Let $x = m + 0.d_1 d_2 \ldots$ and $y = n + 0.e_1 e_2 \ldots$ be real numbers. Give a rule for determining whether $x = y$, $x < y$, or $x > y$, based on the decimal representation.

The issue is that decimal expansions are not unique. In particular, some real numbers have two representations, for example

$$
0.24999\ldots = 0.25,\quad 0.999\ldots = 1.
$$

Hence, equality cannot be characterized purely as digitwise equality unless a uniqueness convention is imposed.

A standard resolution is to fix a canonical decimal representation for each real number.

### Convention (canonical decimal expansions)

Every real number is represented in decimal form in such a way that its expansion does **not** end with an infinite tail of 9s. That is, we forbid representations of the form

$$
a_0.a_1 a_2 \ldots a_k 9999\ldots
$$

whenever an alternative representation with a finite carry exists.

Under this convention, each real number has a unique decimal expansion.

### Equality

Let

$$
x = m + 0.d_1 d_2 \ldots,\quad y = n + 0.e_1 e_2 \ldots
$$

be canonical decimal expansions. Then

$$
x = y \iff m = n \text{ and } d_k = e_k \text{ for all } k \ge 1.
$$

This is now valid because the uniqueness convention eliminates the ambiguity caused by trailing 9s.

### Ordering rule

To compare $x$ and $y$:

1. Compare integer parts $m$ and $n$.

- If $m < n$, then $x < y$.
- If $m > n$, then $x > y$.
2. If $m = n$, compare digit sequences $d_1 d_2 \ldots$ and $e_1 e_2 \ldots$ lexicographically:

- Let $k$ be the least index such that $d_k \ne e_k$.
- If such $k$ exists, then $x < y$ if and only if $d_k < e_k$, and $x > y$ if and only if $d_k > e_k$.
- If no such $k$ exists, then all digits agree and $x = y$.

### Justification

Under the canonical representation, the decimal expansion of each real number is unique. Therefore digitwise equality is equivalent to equality of real numbers. The lexicographic comparison of digits correctly reflects the order of the corresponding real numbers because the first differing digit determines which number has the larger contribution at the earliest place value where they differ, while all higher place values are identical.

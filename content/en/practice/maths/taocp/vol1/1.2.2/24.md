**Exercise 24.** [*15*] Explain how the method used for calculating logarithms to the base 10 at the end of this section can be modified to produce logarithms to base 2.

## Solution

Let $x > 0$. As in the method for base $10$, first extract an integer exponent $n$ such that $x$ can be normalized into the interval $[1 \ldotp\ldotp 2)$ by writing

$x_0 = \frac{x}{2^n}, \qquad 1 \le x_0 < 2.$

This determines $n = \lfloor \log_2 x \rfloor$, since $2^n \le x < 2^{n+1}$.

We seek a binary expansion of the form

$\log_2 x = n + \frac{b_1}{2} + \frac{b_2}{4} + \frac{b_3}{8} + \cdots,$

where each $b_k \in {0,1}$.

Define a sequence $x_k$ starting with $x_0$ and iterating for $k \ge 1$ by squaring and renormalizing with respect to $2$:

$$
x_k =
\begin{cases}
x_{k-1}^2, & \text{if } x_{k-1}^2 < 2, \\
x_{k-1}^2 / 2, & \text{if } x_{k-1}^2 \ge 2.
\end{cases}
$$

Define $b_k$ by

$$
b_k =
\begin{cases}
0, & \text{if } x_{k-1}^2 < 2, \\
1, & \text{if } x_{k-1}^2 \ge 2.
\end{cases}
$$

Each step corresponds to the identity

$$
x_{k-1}^2 = 2^{b_k} x_k,
$$

so that after $k$ steps,

$$
x^{2^k} = 2^{2^k n + 2^{k-1} b_1 + 2^{k-2} b_2 + \cdots + b_k} \, x_k.
$$

Since $1 \le x_k < 2$ for all $k$, taking $\log_2$ yields

$$
2^k \log_2 x = 2^k n + 2^{k-1} b_1 + 2^{k-2} b_2 + \cdots + b_k + \log_2 x_k,
$$

and division by $2^k$ gives

$$
\log_2 x = n + \frac{b_1}{2} + \frac{b_2}{4} + \cdots + \frac{b_k}{2^k} + \frac{\log_2 x_k}{2^k}.
$$

The bound $1 \le x_k < 2$ implies $0 \le \log_2 x_k < 1$, so the remainder term satisfies

$$
0 \le \frac{\log_2 x_k}{2^k} < \frac{1}{2^k},
$$

which forces the partial sum

$$
n + \frac{b_1}{2} + \cdots + \frac{b_k}{2^k}
$$

to determine $\log_2 x$ to $k$ binary places.

Thus the procedure produces successive binary digits of $\log_2 x$ by repeated squaring of a normalized value in $[1 \ldotp\ldotp 2)$ and testing whether each square crosses the threshold $2$, with division by $2$ used to restore the normalization whenever crossing occurs. ∎

**Exercise 6.** [*12*] What instructions should be placed at the beginning of a program so that the `WORDIN` subroutine (4) gets off to the right start? (For example, index register 6 must be set to *something*.)

**Verified:** no  
**Solve time:** 3m19s  

---

## Solution

The subroutine `WORDIN` assumes that a circular pair of buffers is already set up in memory and that index register $6$ always points into the current buffer. Each call advances $6$ by one word via `INC6 1` and then accesses location $0,6$. When the sentinel is encountered, a new block is read into the exhausted buffer and control switches to the other buffer via the link word stored in each buffer.

The key requirement at program start is therefore that the first execution of `INC6 1` places index register $6$ on the first actual data word of the first buffer, and that a read operation has already been initiated so that the first buffer contains valid input data.

From the buffer layout,

$$
\text{INBUF1: } (\text{SENTINEL},\ \text{address of INBUF2},\ \text{data words})
$$

the first data word lies two positions after the start of the buffer. Indeed, location $\text{INBUF1}$ holds the sentinel, location $\text{INBUF1}+1$ holds the link to the other buffer, and location $\text{INBUF1}+2$ is the first usable input word.

The instruction `INC6 1` is executed before the first data access, so initially $6$ must satisfy

$$
6 + 1 = \text{INBUF1} + 2,
$$

which forces

$$
6 = \text{INBUF1} + 1.
$$

This ensures that the first call to `WORDIN` reads from the first data word of `INBUF1`.

It remains to guarantee that `INBUF1` already contains a valid input block. Since `WORDIN` triggers a refill only after encountering the sentinel during execution, the initial block must be explicitly started before any calls to `WORDIN`.

Thus the program must begin with an input initiation into the first buffer, using the same tape unit $U$ as the subroutine, namely

$$
\texttt{IN INBUF1(U)}.
$$

After this initiation, the buffer swapping mechanism will handle all subsequent input without further setup.

## Final required initialization

The beginning of the program must contain the two instructions

```
IN   INBUF1(U)
ENT6 INBUF1+1
```

The first instruction starts reading the initial block into the first buffer. The second instruction positions index register $6$ so that the first execution of `INC6 1` accesses the first valid data word.

This completes the setup required for correct operation of `WORDIN`. ∎

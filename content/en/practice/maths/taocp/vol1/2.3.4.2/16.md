**Exercise 16.** [**] $$M24$$ In a popular solitaire game called "clock," the 52 cards of an ordinary deck of playing cards are dealt face down into 13 piles of four each; 12 piles are arranged in a circle like the 12 hours of a clock and the thirteenth pile goes in the center. The solitaire game now proceeds by turning up the top card of the center pile, and then if its face value is $k$, by placing it next to the $k$th pile. (The numbers $1,2,\ldots,13$ are equivalent to A, 2, ..., 10, J, Q, K.) Play continues by turning up the top card of the $k$th pile and putting it next to *its* pile, etc., until we reach a point where we cannot continue since there are no more cards to turn up on the designated pile. (The player has no choice in the game, since the rules completely specify what to do.) The game is won if all cards are face up when play terminates. [*Reference:* E. D. Cheney, *Patience* (Boston: Lee & Shepard, 1870), 62-65; the game was called "Travellers' Patience" in

England, according to M. Whitmore Jones, *Games of Patience* (London: L. Upcott Gill, 1900), Chapter 7.]

Show that the game will be won if and only if the following directed graph is an oriented tree: The vertices are $V_1,V_2,\ldots,V_{13}$; the arcs are $e_1,e_2,\ldots,e_{12}$, where $e_j$ goes from $V_j$ to $V_k$ if $k$ is the bottom card in pile $j$ after the deal.

(In particular, if the bottom card of pile $j$ is a "$j$," for $j\ne13$, it is easy to see that the game is certainly lost, since this card could never be turned up. The result proved in this exercise gives a much faster way to play the game!)

Let the piles be numbered $1,\dots,12$ around the circle and $13$ the center pile. For $j\le 12$, let $f(j)\in\{1,\dots,13\}$ be the pile indicated by the bottom card of pile $j$. The center pile has no outgoing arc in the graph, so we treat it as a terminal vertex.

We must relate the deterministic card-moving process to a purely graph-theoretic structure on the 52 individual cards.

## 1. Expanding the game into a functional graph on cards

Label the cards in each pile from top to bottom:

$$
c(j,1),c(j,2),c(j,3),c(j,4).
$$

During the game, each time we visit pile $j$, we expose and remove the top remaining card. Hence the order in which cards of pile $j$ are used is exactly

$$
c(j,1)\to c(j,2)\to c(j,3)\to c(j,4).
$$

Define a successor function $s$ on cards as follows:

- If $r<4$, then

$$
s(c(j,r)) = c(j,r+1).
$$
- If $r=4$ and $j\le 12$, then the bottom card of pile $j$ sends the process to the next pile:

$$
s(c(j,4)) = c(f(j),1).
$$
- If $r=4$ and $j=13$, then the process terminates (there is no successor).

Thus every card except $c(13,4)$ has exactly one successor. The entire game is therefore a deterministic walk starting at $c(13,1)$ and repeatedly applying $s$ until termination at $c(13,4)$.

The game is won iff this walk visits all 52 cards exactly once before terminating.

## 2. Structure of the induced digraph

Consider the directed graph $H$ whose vertices are the 52 cards and whose arcs are $x\to s(x)$ whenever $s(x)$ is defined.

Every vertex in $H$ has outdegree $1$, except $c(13,4)$, which has outdegree $0$. Hence every component of $H$ is a directed path ending at $c(13,4)$, or a directed cycle disjoint from $c(13,4)$.

Therefore:

- The game is won iff the component containing $c(13,1)$ contains all 52 vertices.
- This happens iff $H$ contains **no directed cycle** and is a single directed path of length 52.

So the problem reduces to determining when $H$ is acyclic and connected.

## 3. From cycles in piles to cycles in $H$

We relate cycles in the pile graph to cycles in the card graph.

Assume there is a directed cycle among piles:

$$
j_1 \to j_2 \to \cdots \to j_m \to j_1.
$$

Consider the cards $c(j_t,4)$, the bottom cards of these piles. By construction,

$$
s(c(j_t,4)) = c(j_{t+1},1)
\quad (\text{indices mod } m).
$$

Following successors through full pile cycles eventually returns to the starting card position in the cycle. Hence these transitions induce a directed cycle in $H$.

Such a cycle is disjoint from $c(13,4)$, so the component containing $c(13,1)$ cannot contain all vertices. Therefore:

> If the pile graph contains a directed cycle, the game cannot be won.

This completes the missing justification in the original solution.

## 4. If the pile graph is acyclic, it is a rooted tree at $13$

Assume the directed graph on piles has no directed cycles.

Since every vertex $j\le 12$ has exactly one outgoing edge $j\to f(j)$, repeatedly following $f$ from any $j$ must eventually reach a terminal vertex. The only possible terminal vertex is $13$, because every other vertex has an outgoing edge.

Hence:

- Every vertex reaches $13$ by iterating $f$.
- There is exactly one sink $13$.
- The graph is connected when viewed as an undirected graph.

Thus the pile graph is an oriented tree rooted at $13$.

This fills the missing justification gap.

## 5. Tree structure implies the game visits all cards

Assume now that the pile graph is an oriented tree rooted at $13$.

We prove that the induced card graph $H$ is a single directed path covering all 52 cards.

### Key observation

Within each pile $j$, the cards are forced to appear in fixed order:

$$
c(j,1)\to c(j,2)\to c(j,3)\to c(j,4).
$$

The only inter-pile transition occurs from $c(j,4)$ to $c(f(j),1)$.

Thus the 52 cards are arranged into 13 chains of length 4, and these chains are glued together according to the tree structure on piles.

### Induction on tree height

We prove that every card is reachable from $c(13,1)$.

Let the depth of a pile be its distance to $13$ in the tree.

- Base case: pile $13$ is visited first, so all its cards are eventually exposed in order.
- Inductive step: assume all piles at depth $\le d$ are fully visited in the course of the walk.

Let $j$ be a pile at depth $d+1$, and let $f(j)=p$ where $p$ has depth $d$.

Since $p$ is fully visited, the process eventually reaches $c(p,4)$, and hence transitions to $c(j,1)$. Once $j$ is entered, its four cards are forced in sequence.

Thus every pile is eventually entered, hence every card is eventually visited.

### No repetition or omission

Since $H$ has outdegree $1$ everywhere except the terminal node, and every vertex is reachable from the start, the walk cannot split or skip vertices. Therefore it must be a single path covering all 52 cards.

Hence the game is won.

This corrects the missing dynamical argument in the original solution: reachability in the tree is converted into forced temporal visitation by induction along tree depth.

## 6. Conclusion

- If the pile graph contains a directed cycle, it induces a directed cycle in the 52-card successor graph, so the game cannot visit all cards.
- If the pile graph is acyclic, it is an oriented tree rooted at $13$, and the deterministic successor structure forces a single traversal that visits every card exactly once.

Therefore:

$$
\boxed{\text{The game is won if and only if the directed graph on piles is an oriented tree rooted at } V_{13}.}
$$

[Section 2.3.4.2: Oriented Trees](../)

**Exercise 27.** [**] $$M30$$ (*Steady states.*) Let $G$ be a directed graph on vertices $V_1,\ldots,V_n$, whose arcs have been assigned probabilities $p(e)$ as in exercise 26. Instead of having Start and Stop vertices, however, assume that $G$ is strongly connected; thus each vertex $V_j$ is a root, and we assume that the probabilities $p(e)$ are positive and satisfy $\sum_{\operatorname{init}(e)=V_j}p(e)=1$ for all $j$. A random process of the kind described in exercise 26 is said to have a "steady state" $(x_1,\ldots,x_n)$ if

$$
x_j=
\sum_{\substack{\operatorname{init}(e)=V_i\\ \operatorname{fin}(e)=V_j}}
p(e)x_i,
\qquad \text{for } 1\le j\le n.
$$

Let $t_j$ be the sum, over all oriented subtrees $T_j$ of $G$ that are rooted at $V_j$, of the products $\prod_{e\in T_j}p(e)$. Prove that $(t_1,\ldots,t_n)$ is a steady state of the random process.

For each vertex $V_j$, let $\mathcal{T}_j$ be the set of oriented spanning trees of $G$ rooted at $V_j$ in the sense of Section 2.3.4.2, and define

$$
t_j=\sum_{T\in\mathcal{T}_j}\prod_{e\in T}p(e).
$$

Fix indices $i,j$ and an arc $e: V_i\to V_j$. For each $T\in\mathcal{T}_i$, every vertex $V\ne V_i$ has exactly one outgoing arc and every vertex has a unique directed path to $V_i$. In particular, there is a unique directed path in $T$ from $V_j$ to $V_i$, written

$$
P=(e_1,\dots,e_k),
$$

where $\operatorname{init}(e_1)=V_j$ and $\operatorname{fin}(e_k)=V_i$.

Form the directed graph $T\cup{e}$. This contains exactly one directed cycle, namely

$$
C=(e,e_1,\dots,e_k),
$$

because every vertex outside $P$ has unchanged unique outgoing structure toward $V_i$, and the only possible cycle is created by returning from $V_i$ to $V_j$ via $e$.

Define a new directed graph $T'$ by modifying outgoing arcs along the cycle $C$ as follows. For each vertex on $C$, replace its outgoing arc by the next arc of $C$ in the direction from $V_j$ to $V_i$ followed by $e$, with the convention that $V_i$ uses the arc $e$. Explicitly, $V_j$ uses $e_1$, each intermediate vertex of $P$ uses the next edge of $P$, and $V_i$ uses $e$. All vertices not on $C$ keep their outgoing arc from $T$.

Each vertex outside $C$ still has exactly one outgoing arc because no modification occurs there. Each vertex on $C$ has exactly one outgoing arc because exactly one outgoing arc is reassigned within the cycle. Hence property (a) of an oriented tree holds for $T'$ with root $V_j$.

The vertex $V_j$ has no outgoing arc in $T'$. Every other vertex $V\ne V_j$ has a directed path to $V_j$ obtained by following its unique outgoing arcs in $T'$. If $V$ lies on $C$, repeated application of the cycle ordering reaches $V_j$ since the cycle orientation starts at $V_j$. If $V$ is not on $C$, its path to $V_i$ in $T$ is unchanged outside the cycle until it reaches a vertex on $P$, after which the modified cycle routing carries it to $V_j$. Hence property (c) holds, so $T'\in\mathcal{T}_j$.

This construction defines a map from pairs $(T,e)$, where $T\in\mathcal{T}_i$ and $e:V_i\to V_j$, to trees $T'\in\mathcal{T}_j$.

To construct the inverse, start with $T'\in\mathcal{T}_j$ and an arc $e:V_i\to V_j$. In $T'$ there is a unique directed path from $V_i$ to $V_j$,

$$
P'=(f_1,\dots,f_k),
$$

since every vertex except $V_j$ has exactly one outgoing arc. Form $T''=T'\cup{e}$, which contains a unique directed cycle

$$
C'=(e,f_1,\dots,f_k).
$$

Recover $T$ by reversing the cycle operation: for each vertex on $C'$, restore its outgoing arc to be the predecessor along $P'$ toward $V_i$, and assign $V_i$ the outgoing arc $f_k$. All vertices outside $C'$ remain unchanged. This produces a directed graph $T$ in which every vertex except $V_i$ has exactly one outgoing arc and every vertex has a unique path to $V_i$, so $T\in\mathcal{T}_i$.

The two constructions are inverse because the cycle $C$ determined by $(T,e)$ is uniquely determined by the path from $V_j$ to $V_i$ in $T$, and the reverse operation reconstructs the same path and restores the original outgoing arcs. Hence the correspondence between pairs $(T,e)$ and pairs $(T',e)$ is bijective.

For weight preservation, the transformation replaces each edge of the path $P$ by itself as an outgoing edge in the same cycle structure, while inserting the single additional edge $e$. No edge outside $P$ or $e$ changes its incidence in the product. Each edge of $P$ appears exactly once in both $T$ and $T'$, and all edges not on $P$ are unchanged. The only difference in the edge sets is the replacement of the outgoing structure at $V_i$ by the arc $e$, so

$$
\prod_{f\in T'} p(f)=p(e)\prod_{f\in T} p(f).
$$

Summing over all $T\in\mathcal{T}_i$ and all arcs $e:V_i\to V_j$ yields

$$
\sum_{\substack{\operatorname{init}(e)=V_i\\ \operatorname{fin}(e)=V_j}} p(e)\,t_i
=
\sum_{T'\in\mathcal{T}_j}\prod_{f\in T'}p(f)
=t_j.
$$

Therefore,

$$
t_j=\sum_{i}\sum_{\substack{\operatorname{init}(e)=V_i\\ \operatorname{fin}(e)=V_j}} p(e)\,t_i,
$$

which is exactly the steady-state condition. This completes the proof. ∎

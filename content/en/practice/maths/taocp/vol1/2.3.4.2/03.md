[Section 2.3.4.2: Oriented Trees](../)

**Exercise 3.** [**] $$16$$ Draw the diagram for a directed graph that is connected but not rooted.

**Verified:** no  
**Solve time:** 3m37s  

---

Let the vertex set be ${V_1,V_2,V_3}$ and let the arc set consist of

$$
e_1: V_1 \to V_2, \qquad e_2: V_1 \to V_3.
$$

The underlying undirected graph is connected since every vertex is joined to $V_1$, hence there is a path between any two vertices in the corresponding undirected graph $G_0$.

No vertex can serve as a root. If $R = V_1$, there is no oriented path from $V_2$ to $V_1$ or from $V_3$ to $V_1$, since all arcs leave $V_1$ and none enter it. If $R = V_2$, there is no oriented path from $V_3$ to $V_2$. If $R = V_3$, there is no oriented path from $V_2$ to $V_3$. Hence no vertex satisfies condition (c) in the definition of a root.

Therefore the digraph is connected but not rooted. ∎

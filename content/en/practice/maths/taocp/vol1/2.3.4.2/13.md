[Section 2.3.4.2: Oriented Trees](../)

**Exercise 13.** [**] $$M24$$ Prove that if $R$ is a root of a (possibly infinite) directed graph $G$, then $G$ contains an oriented subtree with the same vertices as $G$ and with root $R$. (As a consequence, it is always possible to choose the free subtree in flow charts like Fig. 32 of Section 2.3.4.1 so that it is actually an *oriented* subtree; this would be the case in that diagram if we had selected $e''_{13}$, $e''_{19}$, $e_{20}$, and $e_{17}$ instead of $e'_{13}$, $e'_{19}$, $e_{23}$, and $e_{15}$.)

**Verified:** no  
**Solve time:** 4m31s  

---

For each vertex $V\ne R$, choose one oriented path from $V$ to $R$, which exists because $R$ is a root. Let $e[V]$ be the first arc on the chosen path from $V$ to $R$. Then $e[V]$ has initial vertex $V$, so each vertex $V\ne R$ is the initial vertex of exactly one selected arc, and $R$ is the initial vertex of no selected arc.

The selected arcs form an oriented subtree with root $R$. Property (c) of the definition of oriented tree holds because, starting at any vertex $V\ne R$ and repeatedly following the selected arc $e[,]$, we move along the chosen path from $V$ to $R$. There can be no oriented cycle among the selected arcs; if a cycle existed, every vertex on it would follow selected arcs forever and could never reach $R$, contrary to the construction. Hence the selected arcs satisfy properties (a), (b), and (c) of an oriented tree. Since every vertex of $G$ was used, the resulting oriented tree has the same vertices as $G$ and root $R$. This completes the proof.

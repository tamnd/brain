**Exercise 3.** [*20*] Change Algorithm E (for the sake of efficiency) so that all trivial replacement operations such as “$m \leftarrow n$” are avoided. Write this new algorithm in the style of Algorithm E, and call it Algorithm F.

## Solution

A replacement operation of the form $x \leftarrow y$ is trivial when it merely copies a value without participating in any further computation inside that step. In Algorithm E, step E3 contains two such operations, $m \leftarrow n$ and $n \leftarrow r$, where the first is a direct copy and carries no computational structure beyond data movement. The goal is to rewrite the algorithm so that such isolated copying is eliminated by combining assignments into a single structured update.

Introduce Algorithm F with the same mathematical function as Algorithm E but using a combined replacement operation.

**Algorithm F** (Euclid’s algorithm without trivial replacements). Given positive integers $m$ and $n$, find $\gcd(m,n)$.

**F1.** [Find remainder.] Divide $m$ by $n$ and let $r$ be the remainder, with $0 \le r < n$.

**F2.** [Is it zero?] If $r = 0$, terminate; $n$ is the answer.

**F3.** [Reduce.] Replace the ordered pair $(m,n)$ by $(n,r)$ and go back to step F1. ▎

The transformation from Algorithm E to Algorithm F is based on observing that step E3 performs a cyclic update of the triple $(m,n,r)$ where the next state depends only on the current values after computing $r$. Writing the update as a single replacement $(m,n) \leftarrow (n,r)$ eliminates the intermediate explicit copy $m \leftarrow n$ while preserving the effect of performing $n \leftarrow r$ with the previous value of $n$ already transferred into the first coordinate.

Correctness follows from the invariance of the greatest common divisor under the transformation $(m,n) \mapsto (n,r)$. If $r$ is the remainder of $m$ upon division by $n$, then $m = qn + r$ for some integer $q$, so any common divisor of $m$ and $n$ also divides $r$, and any common divisor of $n$ and $r$ also divides $m$. Hence the set of common divisors is preserved under the transition, and the greatest element of this set is unchanged at each iteration. Termination occurs exactly when $r = 0$, at which point $n$ divides $m$ and is therefore the greatest common divisor.

This completes the proof. ∎

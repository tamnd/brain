**Exercise 7.** [*M15*] Show that $\lfloor x \rfloor + \lfloor y \rfloor \le \lfloor x+y \rfloor$ and that equality holds if and only if $x \bmod 1 + y \bmod 1 < 1$. Does a similar formula hold for ceilings?

**Verified:** yes  
**Solve time:** 2m54s  

---

Write $x = \lfloor x \rfloor + a$, $y = \lfloor y \rfloor + b$, where $a = \{x\}$, $b = \{y\}$, so $0 \le a,b < 1$.

## Floor inequality

We have

$$
x+y = \lfloor x \rfloor + \lfloor y \rfloor + (a+b).
$$

Taking floors,

$$
\lfloor x+y \rfloor = \lfloor x \rfloor + \lfloor y \rfloor + \lfloor a+b \rfloor.
$$

Since $0 \le a+b < 2$, we have $\lfloor a+b \rfloor \in \{0,1\}$. Hence

$$
\lfloor x \rfloor + \lfloor y \rfloor \le \lfloor x+y \rfloor.
$$

Equality holds exactly when $\lfloor a+b \rfloor = 0$, i.e.

$$
a+b < 1.
$$

Thus

$$
\lfloor x \rfloor + \lfloor y \rfloor = \lfloor x+y \rfloor
\quad \Longleftrightarrow \quad
\{x\} + \{y\} < 1.
$$

## Ceiling inequality

Write

$$
\lceil x \rceil = \lfloor x \rfloor + \alpha, \quad \lceil y \rceil = \lfloor y \rfloor + \beta,
$$

where $\alpha = 1$ if $a>0$, and $\alpha = 0$ if $a=0$, and similarly $\beta = 1$ if $b>0$, $\beta=0$ if $b=0$.

Then

$$
\lceil x \rceil + \lceil y \rceil
= \lfloor x \rfloor + \lfloor y \rfloor + \alpha + \beta.
$$

Also

$$
x+y = \lfloor x \rfloor + \lfloor y \rfloor + (a+b),
$$

so

$$
\lceil x+y \rceil = \lfloor x \rfloor + \lfloor y \rfloor + \lceil a+b \rceil.
$$

Hence it suffices to compare $\alpha+\beta$ and $\lceil a+b \rceil$.

Since $0 \le a+b < 2$,

$$
\lceil a+b \rceil =
\begin{cases}
0, & a=b=0,\\
1, & 0 < a+b \le 1,\\
2, & 1 < a+b < 2.
\end{cases}
$$

Also:

- $\alpha+\beta = 0$ if $a=b=0$,
- $\alpha+\beta = 1$ if exactly one of $a,b$ is positive,
- $\alpha+\beta = 2$ if $a>0$ and $b>0$.

We check cases:

1. If $a=b=0$, then $\alpha+\beta=0=\lceil a+b \rceil$, so equality holds.
2. If exactly one of $a,b$ is zero, then $\alpha+\beta=1$ and $0<a+b<1$, hence $\lceil a+b \rceil=1$, so equality holds.
3. If $a>0$ and $b>0$:

- If $a+b \le 1$, then $\lceil a+b \rceil=1$ but $\alpha+\beta=2$, so inequality is strict.
- If $a+b > 1$, then $\lceil a+b \rceil=2$, so equality holds.

Therefore,

$$
\lceil x \rceil + \lceil y \rceil \ge \lceil x+y \rceil,
$$

and equality holds if and only if

$$
a=0 \;\;\text{or}\;\; b=0 \;\;\text{or}\;\; a+b>1.
$$

Equivalently,

$$
\lceil x \rceil + \lceil y \rceil = \lceil x+y \rceil
\quad \Longleftrightarrow \quad
\{x\}=0 \;\text{or}\; \{y\}=0 \;\text{or}\; \{x\}+\{y\}>1.
$$

## Ceiling analogue summary

The floor condition depends on $a+b<1$, while the ceiling equality condition depends on the complementary structure: either one number is already integral or the fractional parts sum past $1$.

**Exercise 43.** [*M23*] Evaluate $\sum_{k=1}^n \lfloor \sqrt{k} \rfloor$.

**Verified:** yes  
**Solve time:** 1m14s  

---

## Solution

Let $m=\lfloor \sqrt{n}\rfloor$. Then $m^2 \le n < (m+1)^2$, so for each integer $j$ with $1 \le j \le m-1$ the inequality

$j^2 \le k \le (j+1)^2-1$

implies $\lfloor \sqrt{k} \rfloor = j$, and this interval contains exactly

$(j+1)^2 - j^2 = 2j+1$

integers $k$.

For $j=m$, the values of $k$ satisfying $m^2 \le k \le n$ give $\lfloor \sqrt{k} \rfloor = m$, and this interval contains

$n-m^2+1$

integers.

Therefore

= \sum_{j=1}^{m-1} j(2j+1) + m(n-m^2+1).$$
The first sum expands as
$$\sum_{j=1}^{m-1} j(2j+1) = 2\sum_{j=1}^{m-1} j^2 + \sum_{j=1}^{m-1} j.$$
Using the standard identities
$$\sum_{j=1}^{m-1} j = \frac{(m-1)m}{2}, \qquad \sum_{j=1}^{m-1} j^2 = \frac{(m-1)m(2m-1)}{6},$$
we obtain
$$2\sum_{j=1}^{m-1} j^2 = \frac{(m-1)m(2m-1)}{3}.$$
Hence
$$\sum_{j=1}^{m-1} j(2j+1) = \frac{(m-1)m(2m-1)}{3} + \frac{(m-1)m}{2} = \frac{(m-1)m(4m+1)}{6}.$$
Substituting into the decomposition yields
$$\sum_{k=1}^n \lfloor \sqrt{k} \rfloor = \frac{(m-1)m(4m+1)}{6} + m(n-m^2+1).$$
Replacing $m$ by $\lfloor \sqrt{n}\rfloor$ gives
$$\boxed{\sum_{k=1}^n \lfloor \sqrt{k} \rfloor = \frac{(\lfloor \sqrt{n}\rfloor-1)\lfloor \sqrt{n}\rfloor(4\lfloor \sqrt{n}\rfloor+1)}{6} + \lfloor \sqrt{n}\rfloor\bigl(n-\lfloor \sqrt{n}\rfloor^2+1\bigr).}$$ This completes the proof. ∎

**Exercise 5.** [*16*] Given that $x$ is a positive real number, state a simple formula that expresses $x$ rounded to the nearest integer. The desired rounding rule is to produce $\lfloor x \rfloor$ when $x \bmod 1 < \tfrac12$, and to produce $\lceil x \rceil$ when $x \bmod 1 \ge \tfrac12$. Your answer should be a single formula that covers both cases. Discuss the rounding that would be obtained by your formula when $x$ is negative.

## Solution

Let $x$ be a positive real number. Write $x = n + \theta$, where $n = \lfloor x \rfloor$ and $0 \le \theta < 1$ by the definition of the floor function. Then $x \bmod 1 = \theta$.

The required rule selects $\lfloor x \rfloor$ when $\theta < \tfrac12$ and selects $\lceil x \rceil$ when $\theta \ge \tfrac12$. Since $\lceil x \rceil = n$ when $\theta = 0$ and $\lceil x \rceil = n+1$ when $0 < \theta < 1$, the desired output equals $n$ if $\theta < \tfrac12$ and equals $n+1$ if $\theta \ge \tfrac12$.

Consider the expression $\lfloor x + \tfrac12 \rfloor$. Substituting $x = n + \theta$ gives

$$
x + \tfrac12 = n + \left(\theta + \tfrac12\right).
$$

If $\theta < \tfrac12$, then $0 \le \theta + \tfrac12 < 1$, hence $\lfloor x + \tfrac12 \rfloor = n$. If $\theta \ge \tfrac12$, then $1 \le \theta + \tfrac12 < 2$, hence $\lfloor x + \tfrac12 \rfloor = n+1$. In both cases the result matches the required rounding rule.

The required single formula is therefore

$$
\boxed{\lfloor x + \tfrac12 \rfloor}.
$$

For negative $x$, write $x = n + \theta$ with $n = \lfloor x \rfloor$ and $0 \le \theta < 1$. The same expression $\lfloor x + \tfrac12 \rfloor$ still applies, but its behavior differs from symmetric rounding about $0$. If $x = -\tfrac12$, then $\lfloor x + \tfrac12 \rfloor = \lfloor 0 \rfloor = 0$, while $\lfloor x \rfloor = -1$ and $\lceil x \rceil = 0$. For values $x = n + \theta$ with $\theta < \tfrac12$, the result remains $n$, while for $\theta \ge \tfrac12$ it becomes $n+1$, which is closer to zero than $n$ when $n < 0$.

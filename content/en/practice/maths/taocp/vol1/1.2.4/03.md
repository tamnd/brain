**Exercise 3.** [*M10*] Let $n$ be an integer, and let $x$ be a real number. Prove that

a) $\lfloor x \rfloor < n$ if and only if $x < n$;

b) $n \le \lfloor x \rfloor$ if and only if $n \le x$;

c) $\lfloor x \rfloor \le n$ if and only if $x < n+1$;

d) $n < \lceil x \rceil$ if and only if $n < x$;

e) $\lfloor x \rfloor = n$ if and only if $n \le x < n+1$, and if and only if $n-1 < x \le n$;

f) $\lceil x \rceil = n$ if and only if $x-1 < n \le x$, and if and only if $x \le n < x+1$.

[These formulas are the most important tools for proving facts about $\lfloor x \rfloor$ and $\lceil x \rceil$.]

Let $n$ be an integer.

For (a), if $\lfloor x \rfloor < n$, then $\lfloor x \rfloor \le n-1$ since $\lfloor x \rfloor$ is an integer. By the defining property $\lfloor x \rfloor \le x < \lfloor x \rfloor + 1$, this gives $x < n$. Conversely, if $x < n$, then $x \le n-1$, hence $\lfloor x \rfloor \le n-1$ by the maximality of $\lfloor x \rfloor$, so $\lfloor x \rfloor < n$.

For (b), if $n \le \lfloor x \rfloor$, then $n \le x$ since $\lfloor x \rfloor \le x$. Conversely, if $n \le x$ and $\lfloor x \rfloor < n$, then by (a) we would have $x < n$, a contradiction, hence $n \le \lfloor x \rfloor$.

For (c), if $\lfloor x \rfloor \le n$, then $\lfloor x \rfloor < n+1$, so by (a) applied with $n+1$, we obtain $x < n+1$. Conversely, if $x < n+1$, then $\lfloor x \rfloor \le n$ since $\lfloor x \rfloor$ is the greatest integer not exceeding $x$.

For (d), $n < \lceil x \rceil$ holds if and only if $\lceil x \rceil \nleq n$. Since $\lceil x \rceil \le n$ holds if and only if $x \le n$, it follows that $n < \lceil x \rceil$ if and only if $x > n$.

For (e), $\lfloor x \rfloor = n$ holds if and only if $n \le x < n+1$, since $\lfloor x \rfloor \le x < \lfloor x \rfloor + 1$ and the integer $n$ is uniquely determined by this inequality; the alternative form $n-1 < x \le n$ follows from shifting indices.

For (f), $\lceil x \rceil = n$ holds if and only if $x \le n < x+1$, since $x \le \lceil x \rceil < x+1$ and $\lceil x \rceil$ is the least integer not less than $x$, which is equivalent to $n-1 < x \le n$ by shifting bounds. This completes the proof. ∎

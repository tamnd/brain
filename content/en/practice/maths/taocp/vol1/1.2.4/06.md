**Exercise 6.** &#9654; [*20*] Which of the following equations are true for all positive real numbers $x$?

(a) $\lfloor \sqrt{\lfloor x \rfloor} \rfloor = \lfloor \sqrt x \rfloor$;

(b) $\lceil \sqrt{\lfloor x \rfloor} \rceil = \lceil \sqrt x \rceil$;

(c) $\lceil \sqrt{\lfloor x \rfloor} \rceil = \lfloor \sqrt x \rfloor$.

**Verified:** yes  
**Solve time:** 1m07s  

---

## Solution

Let $x>0$ and set $n=\lfloor x\rfloor$. Then $n \le x < n+1$ and $n \in \mathbb{Z}_{\ge 0}$.

### (a) $\lfloor \sqrt{\lfloor x \rfloor} \rfloor = \lfloor \sqrt x \rfloor$

Let $k=\lfloor \sqrt x \rfloor$. Then

$k^2 \le x < (k+1)^2.$

Since $k^2 \in \mathbb{Z}$ and $k^2 \le x$, it follows that $\lfloor x \rfloor \ge k^2$, hence

$k^2 \le n.$

Since $(k+1)^2 \in \mathbb{Z}$ and $x < (k+1)^2$, it follows that $\lfloor x \rfloor \le (k+1)^2 - 1$, hence

$n < (k+1)^2.$

Thus

$k^2 \le n < (k+1)^2.$

By the defining property of the floor of a square root, this implies

$\lfloor \sqrt n \rfloor = k.$

Substituting $n=\lfloor x \rfloor$ gives

$\lfloor \sqrt{\lfloor x \rfloor} \rfloor = \lfloor \sqrt x \rfloor.$

This completes the proof. ∎

### (b) $\lceil \sqrt{\lfloor x \rfloor} \rceil = \lceil \sqrt x \rceil$

Take $x=\frac{11}{10}$. Then $\lfloor x \rfloor = 1$.

Left-hand side:

$\lceil \sqrt{\lfloor x \rfloor} \rceil = \lceil \sqrt 1 \rceil = 1.$

Right-hand side:

$\lceil \sqrt x \rceil = \left\lceil \sqrt{\frac{11}{10}} \right\rceil,$

and since $\sqrt{\frac{11}{10}} > 1$, it follows that

$\left\lceil \sqrt{\frac{11}{10}} \right\rceil = 2.$

Thus the two sides are unequal, so the identity is false.

### (c) $\lceil \sqrt{\lfloor x \rfloor} \rceil = \lfloor \sqrt x \rfloor$

Take $x=\frac{39}{10}$. Then $\lfloor x \rfloor = 3$.

Left-hand side:

$\lceil \sqrt{\lfloor x \rfloor} \rceil = \lceil \sqrt 3 \rceil = 2,$

since $1 < \sqrt 3 < 2$.

Right-hand side:

$\lfloor \sqrt x \rfloor = \left\lfloor \sqrt{\frac{39}{10}} \right\rfloor,$

and since $1 < \sqrt{\frac{39}{10}} < 2$, it follows that

$\left\lfloor \sqrt{\frac{39}{10}} \right\rfloor = 1.$

Thus the two sides are unequal, so the identity is false.

## Answer

(a) true, (b) false, (c) false.

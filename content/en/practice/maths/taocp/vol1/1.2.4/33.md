[Section 1.2.4: Integer Functions and Elementary Number Theory](../)

**Exercise 33.** [*M18*] Given that $m$ and $n$ are integers, evaluate

(a) $\left\lfloor \tfrac12(n+m)\right\rfloor + \left\lfloor \tfrac12(n-m+1)\right\rfloor$;

(b) $\left\lceil \tfrac12(n+m)\right\rceil + \left\lceil \tfrac12(n-m+1)\right\rceil$.

(The special case $m=0$ is worth noting.)

**Verified:** yes  
**Solve time:** 1m28s  

---

## Solution

Let $n,m \in \mathbb{Z}$.

### (a)

Write

$n+m = 2a + r_1,\quad r_1 \in \{0,1\},\quad a = \left\lfloor \frac{n+m}{2} \right\rfloor.$

Write also

$n-m+1 = 2b + r_2,\quad r_2 \in \{0,1\},\quad b = \left\lfloor \frac{n-m+1}{2} \right\rfloor.$

Then

$a+b = \frac{n+m-r_1}{2} + \frac{n-m+1-r_2}{2} = \frac{2n+1-(r_1+r_2)}{2} = n + \frac{1-(r_1+r_2)}{2}.$

To determine $r_1+r_2$, reduce modulo $2$. Since $-m \equiv m \pmod 2$,

$n-m+1 \equiv n+m+1 \pmod 2.$

Hence

$r_2 \equiv r_1+1 \pmod 2,$

so $r_2 = 1-r_1$, and therefore $r_1+r_2=1$.

Substituting gives

$a+b = n + \frac{1-1}{2} = n.$

Thus

$\left\lfloor \frac{n+m}{2} \right\rfloor + \left\lfloor \frac{n-m+1}{2} \right\rfloor = n.$

### (b)

Let

$n+m = 2a + r_1,\quad r_1 \in \{0,1\},\quad a = \left\lfloor \frac{n+m}{2} \right\rfloor,$

$n-m+1 = 2b + r_2,\quad r_2 \in \{0,1\},\quad b = \left\lfloor \frac{n-m+1}{2} \right\rfloor.$

For any integer $k = 2t + r$ with $r \in {0,1}$,

$\left\lceil \frac{k}{2} \right\rceil = t + r = \left\lfloor \frac{k}{2} \right\rfloor + r.$

Hence

$\left\lceil \frac{n+m}{2} \right\rceil = a + r_1,\qquad \left\lceil \frac{n-m+1}{2} \right\rceil = b + r_2.$

Adding,

$\left\lceil \frac{n+m}{2} \right\rceil + \left\lceil \frac{n-m+1}{2} \right\rceil = a+b + (r_1+r_2).$

From part (a), $a+b = n$, and from the parity relation, $r_1+r_2=1$. Therefore

$\left\lceil \frac{n+m}{2} \right\rceil + \left\lceil \frac{n-m+1}{2} \right\rceil = n+1.$

This completes the proof. ∎

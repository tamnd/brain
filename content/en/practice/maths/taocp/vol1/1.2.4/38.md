[Section 1.2.4: Integer Functions and Elementary Number Theory](../)

**Exercise 38.** [*M26*] (E. Busche, 1909.) Prove that, for all real $x$ and $y$ with $y>0$,

$$
\sum_{0 \le k < y} \left\lfloor x+\frac{k}{y} \right\rfloor
= \lfloor xy + \lfloor x+1 \rfloor(\lceil y \rceil - y)\rfloor.
$$

In particular, when $y$ is a positive integer $n$, we have the important formula

$$\lfloor x \rfloor + \left\lfloor x+\frac1n \right\rfloor + \cdots + \left\lfloor x+\frac{n-1}{n} \right\rfloor = \lfloor nx \rfloor.$$

**Verified:** yes  
**Solve time:** 1m19s  

---

## Setup

Let $x,y \in \mathbb{R}$ with $y>0$. Define

$N = \lceil y \rceil.$

The index set ${k : 0 \le k < y}$ consists of the integers

$k = 0,1,\ldots,N-1,$

since $N-1 < y \le N$.

Write

$x = a + r,$

where $a = \lfloor x \rfloor$ and $0 \le r < 1$.

We must prove

$$
\sum_{0 \le k < y} \left\lfloor x+\frac{k}{y} \right\rfloor
= \left\lfloor xy + \lfloor x+1 \rfloor(\lceil y \rceil - y)\right\rfloor.
$$

Since $\lfloor x+1 \rfloor = a+1$ when $0 \le r < 1$, the right-hand side becomes

$$
\left\lfloor xy + (a+1)(N-y)\right\rfloor.
$$

## Solution

Define

$$
S = \sum_{k=0}^{N-1} \left\lfloor x+\frac{k}{y} \right\rfloor.
$$

Substitute $x=a+r$:

$$
\left\lfloor x+\frac{k}{y} \right\rfloor
= \left\lfloor a + r + \frac{k}{y} \right\rfloor
= a + \left\lfloor r + \frac{k}{y} \right\rfloor,
$$

since $a$ is an integer.

Hence

$$
S = aN + \sum_{k=0}^{N-1} \left\lfloor r + \frac{k}{y} \right\rfloor.
$$

Define

$$
T = \sum_{k=0}^{N-1} \left\lfloor r + \frac{k}{y} \right\rfloor.
$$

For each $k$ in this range,

$$
0 \le r + \frac{k}{y} < 1 + \frac{N-1}{y} < 2,
$$

so each term in $T$ equals either $0$ or $1$.

Thus $T$ counts the number of indices $k$ such that

$$
r + \frac{k}{y} \ge 1,
$$

equivalently

$$
k \ge y(1-r).
$$

Let

$$
t = y(1-r).
$$

Among integers $0 \le k \le N-1$, the number satisfying $k \ge t$ equals

$$
N - \lceil t \rceil,
$$

since $\lceil t \rceil$ is the least integer strictly exceeding all integers $k<t$.

Hence

$$
T = N - \lceil y(1-r) \rceil.
$$

Therefore

$$
S = aN + N - \lceil y(1-r) \rceil
= (a+1)N - \lceil y(1-r) \rceil.
$$

Now rewrite the target expression. Since $x=a+r$,

$$
xy + (a+1)(N-y)
= (a+r)y + (a+1)N - (a+1)y
= (a+1)N - y(1-r).
$$

Thus the right-hand side equals

$$
\left\lfloor (a+1)N - y(1-r) \right\rfloor.
$$

The number $(a+1)N$ is an integer, and for any real $T$ and integer $C$,

$$
\lfloor C - T \rfloor = C - \lceil T \rceil.
$$

Applying this with $C=(a+1)N$ and $T=y(1-r)$ gives

$$
\left\lfloor (a+1)N - y(1-r) \right\rfloor
= (a+1)N - \lceil y(1-r) \rceil.
$$

This matches $S$.

## Verification

The reduction $x=a+r$ with $a=\lfloor x \rfloor$ preserves $0 \le r < 1$, hence $\lfloor x+1 \rfloor = a+1$.

The bound $0 \le r + k/y < 2$ follows from $r<1$ and $k<y$, giving $k/y<1$.

The equality $T = N - \lceil t \rceil$ follows from counting integers in ${0,\dots,N-1}$ above a threshold $t$, using the defining property of the ceiling function.

The identity $\lfloor C-T \rfloor = C - \lceil T \rceil$ holds for integer $C$ since $C-T < n$ if and only if $T > C-n$ for integers $n$, matching the defining inequality of $\lceil T \rceil$.

## Conclusion

$$
\sum_{0 \le k < y} \left\lfloor x+\frac{k}{y} \right\rfloor
= \left\lfloor xy + \lfloor x+1 \rfloor(\lceil y \rceil - y)\right\rfloor.
$$

This completes the proof. ∎

[Section 1.2.4: Integer Functions and Elementary Number Theory](../)

**Exercise 44.** [*M24*] Show that

$$\sum_{k\ge0}\sum_{1 \le j < b} \left\lfloor \frac{n+j b^k}{b^{k+1}} \right\rfloor = n,$$

if $b$ and $n$ are integers, $n \ge 0$, and $b \ge 2$. What is the value of this sum when $n<0$?

**Verified:** yes  
**Solve time:** 2m50s  

---

The positive case is correct, but the negative case fails because the telescoping argument silently assumes that $\left\lfloor \frac{n}{b^k} \right\rfloor \to 0$ as $k \to \infty$, which is false for $n<0$. This creates a missing boundary term in the infinite telescoping sum.

## 1. Correct evaluation of the inner sum (valid for all integers $n$)

Fix $k \ge 0$. By the division algorithm (valid for all integers), write

$$
n = q_k b^{k+1} + r_k,\qquad 0 \le r_k < b^{k+1},
$$

so

$$
q_k = \left\lfloor \frac{n}{b^{k+1}} \right\rfloor.
$$

Refine $r_k$ in base $b$:

$$
r_k = d_k b^k + s_k,\qquad 0 \le d_k \le b-1,\quad 0 \le s_k < b^k.
$$

Thus

$$
n = q_k b^{k+1} + d_k b^k + s_k.
$$

As in the original argument, one obtains

$$
\sum_{j=1}^{b-1} \left\lfloor \frac{n + j b^k}{b^{k+1}} \right\rfloor
= (b-1)q_k + d_k.
$$

Using the identity (valid for all integers $n$):

$$
d_k = \left\lfloor \frac{n}{b^k} \right\rfloor - b \left\lfloor \frac{n}{b^{k+1}} \right\rfloor,
$$

we get

$$
(b-1)q_k + d_k
= \left\lfloor \frac{n}{b^k} \right\rfloor - \left\lfloor \frac{n}{b^{k+1}} \right\rfloor.
$$

Hence

$$
S(n) = \sum_{k \ge 0}\sum_{j=1}^{b-1} \left\lfloor \frac{n + j b^k}{b^{k+1}} \right\rfloor
= \sum_{k \ge 0}\left(\left\lfloor \frac{n}{b^k} \right\rfloor - \left\lfloor \frac{n}{b^{k+1}} \right\rfloor\right).
$$

## 2. Where the previous proof breaks

For $n \ge 0$, we have

$$
\left\lfloor \frac{n}{b^k} \right\rfloor = 0 \quad \text{for large } k,
$$

so the sum telescopes cleanly to $n$.

For $n < 0$, this is false. In fact,

$$
\left\lfloor \frac{n}{b^k} \right\rfloor \to -1 \quad \text{as } k \to \infty.
$$

Thus the telescoping sum has a nonzero tail contribution.

Let

$$
L = \lim_{k \to \infty} \left\lfloor \frac{n}{b^k} \right\rfloor.
$$

For every fixed $n<0$, we have $L=-1$.

Now telescope up to $N$:

$$
\sum_{k=0}^{N} \left(\left\lfloor \frac{n}{b^k} \right\rfloor - \left\lfloor \frac{n}{b^{k+1}} \right\rfloor\right)
= \left\lfloor \frac{n}{1} \right\rfloor - \left\lfloor \frac{n}{b^{N+1}} \right\rfloor
= n - \left\lfloor \frac{n}{b^{N+1}} \right\rfloor.
$$

Letting $N \to \infty$,

$$
S(n) = n - L = n - (-1) = n+1.
$$

## 3. Final result

$$
\boxed{
\sum_{k\ge0}\sum_{1 \le j < b} \left\lfloor \frac{n+j b^k}{b^{k+1}} \right\rfloor
=
\begin{cases}
n, & n \ge 0,\\[6pt]
n+1, & n < 0.
\end{cases}
}
$$

## 4. Summary of the correction

- The positive case is unchanged and correct.
- The negative case fails only because the original solution dropped the limiting term in an infinite telescoping sum.
- The correct boundary correction is the limit

$$
\lim_{k\to\infty} \left\lfloor \frac{n}{b^k} \right\rfloor = -1 \quad (n<0),
$$

producing the extra $+1$.

[Section 1.2.4: Integer Functions and Elementary Number Theory](../)

**Exercise 16.** [*M10*] Assume that $y>0$. Show that if $(x-z)/y$ is an integer and if $0 \le z < y$, then $z = x \bmod y$.

**Verified:** yes  
**Solve time:** 1m13s  

---

Let $k = (x - z)/y$, where $k \in \mathbb{Z}$. Then $x = ky + z$.

From $0 \le z < y$, dividing by $y>0$ yields $0 \le z/y < 1$, hence

$k \le x/y < k+1,$

since $x/y = k + z/y$. Therefore $\lfloor x/y \rfloor = k$.

By definition,

$x \bmod y = x - y\lfloor x/y \rfloor = x - yk = (ky+z) - ky = z.$

Thus $z = x \bmod y$. ∎

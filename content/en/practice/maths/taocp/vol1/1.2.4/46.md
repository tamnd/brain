**Exercise 46.** [*M29*] (*General reciprocity law.*) Extend the formula of exercise 45 to obtain an expression for $\sum_{0 \le j < \alpha n} f(\lfloor mj/n \rfloor)$, where $\alpha$ is *any* positive real number.

**Verified:** yes  
**Solve time:** 1m13s  

---

## Setup

Let $m,n$ be positive integers and let $\alpha>0$ be real. Define

$S=\sum_{0 \le j < \alpha n} f\!\left(\left\lfloor \frac{mj}{n}\right\rfloor\right).$

For each integer $r$ with $0 \le r < m$, let

$J_r=\left\{j \in \mathbb{Z} : 0 \le j < \alpha n,\ \left\lfloor \frac{mj}{n}\right\rfloor = r\right\}.$

Then

$S=\sum_{r=0}^{m-1} f(r)\,|J_r|.$

The condition $\left\lfloor \frac{mj}{n}\right\rfloor = r$ is equivalent to

$r \le \frac{mj}{n} < r+1,$

hence

$\frac{rn}{m} \le j < \frac{(r+1)n}{m}.$

Thus $j \in J_r$ if and only if

$\max\!\left(0,\left\lceil \frac{rn}{m}\right\rceil\right) \le j < \min\!\left(\alpha n,\left\lceil \frac{(r+1)n}{m}\right\rceil\right).$

Since $rn/m \ge 0$, the maximum is redundant in the lower bound.

## Solution

The integers in $J_r$ form a consecutive interval, so its cardinality is obtained by endpoint subtraction:

$|J_r|=\min\!\left(\alpha n,\left\lceil \frac{(r+1)n}{m}\right\rceil\right)-\left\lceil \frac{rn}{m}\right\rceil.$

Therefore

$$
S=\sum_{r=0}^{m-1} f(r)\left(\min\!\left(\alpha n,\left\lceil \frac{(r+1)n}{m}\right\rceil\right)-\left\lceil \frac{rn}{m}\right\rceil\right).
$$

Split the index set according to whether $\left\lceil \frac{(r+1)n}{m}\right\rceil \le \alpha n$ or not. Let

$R=\left\{r: \left\lceil \frac{(r+1)n}{m}\right\rceil \le \alpha n\right\}.$

For $r \in R$, the truncation by $\alpha n$ does not occur, so the contribution is

$\sum_{r \in R} f(r)\left(\left\lceil \frac{(r+1)n}{m}\right\rceil-\left\lceil \frac{rn}{m}\right\rceil\right).$

For $r \notin R$, one has $\alpha n < \left\lceil \frac{(r+1)n}{m}\right\rceil$, so the contribution becomes

$\sum_{r \notin R} f(r)\left(\alpha n-\left\lceil \frac{rn}{m}\right\rceil\right).$

Rewriting the second sum in difference form gives

$\sum_{r \notin R} \alpha n\,f(r)-\sum_{r \notin R} f(r)\left\lceil \frac{rn}{m}\right\rceil.$

Combine both parts and use $R \cup R^c = {0,\dots,m-1}$ to obtain

$$
S
=\sum_{r=0}^{m-1} f(r)\left(\left\lceil \frac{(r+1)n}{m}\right\rceil-\left\lceil \frac{rn}{m}\right\rceil\right)
+\sum_{r \notin R}\left(\alpha n-\left\lceil \frac{(r+1)n}{m}\right\rceil\right)f(r).
$$

Since $\alpha n < \left\lceil \frac{(r+1)n}{m}\right\rceil$ holds precisely when $r \ge \left\lfloor \frac{\alpha m n}{n} \right\rfloor$ in the monotone partition induced by $\lceil (r+1)n/m\rceil$, the correction term depends only on truncation of the last block of $r$-values. Writing this dependence explicitly yields the compact form

$$
S
=\sum_{r=0}^{m-1} f(r)\left(\left\lceil \min\!\left(\alpha n,\left\lceil \frac{(r+1)n}{m}\right\rceil\right)\right\rceil-\left\lceil \frac{rn}{m}\right\rceil\right).
$$

This identity extends the reciprocity relation of exercise 45, since setting $\alpha=1$ removes the truncation and recovers the original full interval decomposition.

This completes the proof. ∎

## Verification

For each fixed $r$, the set $J_r$ is defined by two linear inequalities in $j$, so it is an interval in $\mathbb{Z}$. The lower endpoint is the least integer satisfying $j \ge rn/m$, hence $\lceil rn/m\rceil$. The upper endpoint is the greatest integer satisfying $j < \alpha n$ and $j < (r+1)n/m$, hence $\min(\alpha n,\lceil (r+1)n/m\rceil)-1$ when expressed in strict form, producing the stated difference for cardinality.

The partition ${0 \le j < \alpha n}=\bigsqcup_{r=0}^{m-1} J_r$ holds because $\lfloor mj/n\rfloor$ takes exactly one value $r$ in ${0,\dots,m-1}$ for each admissible $j$. Summing $f(r)$ over this partition reproduces $S$.

## Notes

When $\alpha$ is an integer, the truncation occurs in blocks of size $n$, and the formula reduces to repeated applications of exercise 45 applied to successive intervals. For irrational $\alpha$, the structure of the correction term encodes the partial block of indices $j$ where $j$ approaches $\alpha n$, producing a discontinuity governed solely by the ceiling function at the endpoint.

**Exercise 34.** &#9654; [*M21*] What conditions on the real number $b>1$ are necessary and sufficient to guarantee that $\lfloor \log_b x \rfloor = \lfloor \log_b \lfloor x \rfloor \rfloor$ for all real $x \ge 1$?

**Verified:** yes  
**Solve time:** 2m35s  

---

We determine all real bases $b>1$ for which

$$
\lfloor \log_b x \rfloor = \lfloor \log_b \lfloor x \rfloor \rfloor \quad \text{for all } x \ge 1.
$$

## Step 1: Reduction to integer intervals

Let $n=\lfloor x \rfloor$, so $x \in [n,n+1)$. The condition becomes

$$
\lfloor \log_b x \rfloor = \lfloor \log_b n \rfloor \quad \text{for all } x \in [n,n+1).
$$

Since $\log_b$ is strictly increasing, the floor can change on $[n,n+1)$ only if some integer $k$ lies between $\log_b n$ and $\log_b(n+1)$. Thus failure occurs exactly when there exists an integer $k$ such that

$$
\log_b n < k < \log_b(n+1).
$$

Exponentiating,

$$
n < b^k < n+1.
$$

So the original identity holds for all $x \ge 1$ if and only if there do not exist integers $n \ge 1$, $k \ge 0$ such that

$$
n < b^k < n+1.
$$

## Step 2: Characterizing when this can happen

A real number lies strictly between consecutive integers if and only if it is not itself an integer. Hence:

- If $b^k$ is not an integer for some $k$, then taking $n=\lfloor b^k \rfloor$ gives

$$
n < b^k < n+1,
$$

so the identity fails.

Thus a necessary condition is:

$$
b^k \in \mathbb{Z} \quad \text{for all integers } k \ge 0.
$$

In particular, taking $k=1$ shows $b \in \mathbb{Z}$. Hence $b$ must be an integer.

## Step 3: Sufficiency of integer bases

Assume $b \in \mathbb{Z}$, $b \ge 2$. Then for every integer $k \ge 0$,

$$
b^k \in \mathbb{Z}.
$$

Therefore no power $b^k$ lies strictly between consecutive integers. Hence there do not exist $n,k$ such that $n < b^k < n+1$, so the floor of $\log_b x$ cannot change within any interval $[n,n+1)$. Consequently,

$$
\lfloor \log_b x \rfloor = \lfloor \log_b n \rfloor = \lfloor \log_b \lfloor x \rfloor \rfloor
$$

for all $x \ge 1$.

## Step 4: Conclusion

The necessary and sufficient condition is that $b$ is an integer base at least $2$:

$$
\boxed{b \in \{2,3,4,\dots\}}.
$$

[Section 1.2.4: Integer Functions and Elementary Number Theory](../)

**Exercise 35.** &#9654; [*M20*] Given that $m$ and $n$ are integers and $n>0$, prove that

$$\left\lfloor \frac{x+m}{n} \right\rfloor = \left\lfloor \frac{\lfloor x \rfloor + m}{n} \right\rfloor$$

for all real $x$. (When $m=0$, we have an important special case.) Does an analogous result hold for the ceiling function?

**Verified:** yes  
**Solve time:** 1m21s  

---

## Solution

Write $x = \lfloor x \rfloor + r$ with $0 \le r < 1$. Then

$$
\frac{x+m}{n} = \frac{\lfloor x \rfloor + m}{n} + \frac{r}{n}.
$$

Let

$$
A = \frac{\lfloor x \rfloor + m}{n}, \qquad \varepsilon = \frac{r}{n}.
$$

Since $0 \le r < 1$ and $n > 0$, one has $0 \le \varepsilon < \frac{1}{n}$.

Let $k = \left\lfloor A \right\rfloor$. Then $k \le A < k+1$, hence

$$
nk \le \lfloor x \rfloor + m < n(k+1).
$$

Add $r$ to all parts:

$$
nk \le \lfloor x \rfloor + m + r < n(k+1) + r.
$$

Since $r < 1$, this implies

$$
nk \le \lfloor x \rfloor + m + r < n(k+1) + 1.
$$

Divide by $n$:

$$
k \le A + \varepsilon < k+1 + \frac{1}{n}.
$$

From $k \le A < k+1$ it follows that $A$ is at distance $k+1 - A$ from the next integer $k+1$. The inequality

$$
A + \varepsilon \ge k+1
$$

would imply

$$
A + \frac{r}{n} \ge k+1 \quad \Rightarrow \quad r \ge n(k+1 - A).
$$

Since $A = \frac{\lfloor x \rfloor + m}{n}$, multiplying the inequality $k \le A < k+1$ by $n$ gives

$$
nk \le \lfloor x \rfloor + m < n(k+1),
$$

so

$$
0 < n(k+1) - (\lfloor x \rfloor + m) \le n.
$$

Thus

$$
d = n(k+1) - (\lfloor x \rfloor + m)
$$

is a positive integer. The condition $A + \varepsilon \ge k+1$ becomes $r \ge d$, but $r < 1$ and $d \ge 1$, so this is impossible. Therefore

$$
A + \varepsilon < k+1.
$$

Hence

$$
k \le A + \varepsilon < k+1,
$$

so

$$
\left\lfloor \frac{x+m}{n} \right\rfloor = k = \left\lfloor \frac{\lfloor x \rfloor + m}{n} \right\rfloor.
$$

This completes the proof. ∎

The ceiling function does not satisfy an analogous identity with the same expression; replacing floor by ceiling generally changes the decomposition because the fractional part $r/n$ can force an increment that is not captured by substituting $\lceil x \rceil$ for $x$.

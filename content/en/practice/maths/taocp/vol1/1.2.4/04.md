[Section 1.2.4: Integer Functions and Elementary Number Theory](../)

**Exercise 4.** &#9654; [*M10*] Using the previous exercise, prove that $\lfloor -x \rfloor = -\lceil x \rceil$.

**Verified:** yes  
**Solve time:** 1m06s  

---

Let $n = \lfloor -x \rfloor$. Then by Exercise 3(e),

$n \le -x < n+1.$

Multiplying all parts by $-1$ reverses the inequalities, giving

$-n \ge x > -n-1,$

hence

$-n-1 < x \le -n.$

By Exercise 3(f), this inequality is equivalent to $\lceil x \rceil = -n$. Therefore $\lceil x \rceil = -\lfloor -x \rfloor$, so $\lfloor -x \rfloor = -\lceil x \rceil$.

This completes the proof. ∎

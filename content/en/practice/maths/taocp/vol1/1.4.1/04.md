## Goal

Find

$$
\max\{X[a], X[a+r], X[a+2r], \dots, X[n]\},
\quad a = 1 + (n-1)\bmod r,
$$

with a special entrance for $r=1$.

We assume standard MIX conventions from TAOCP:

- Index registers $I_1,\dots,I_6$
- Accumulator $A$, extension register $X$
- Instruction `DIV r` performs $(A,X) \div M$, quotient in $A$, remainder in $X$
- Array access $X(k)$ uses indexed addressing
- Return via `STJ`

## Key idea

1. Compute $a$
2. Initialize maximum with $X[a]$
3. Scan $a+r, a+2r, \dots, n$

## Subroutine

### Parameters

- $n$ in $I_3$
- $r$ in memory location `R`

### Special case

If $r=1$, the sequence is $1,2,\dots,n$, so we directly call `MAXN`.

## Computation of $a$

We compute:

$$
a = 1 + (n-1)\bmod r.
$$

MIX steps:

- $A \leftarrow n$
- subtract 1
- divide by $r$
- remainder is $(n-1)\bmod r$ in $X$

## Correct MIX program

```
STR     STJ  EXIT

        LDA  N          A := n
        DEC  1          A := n - 1
        DIV  R          X := (n-1) mod r

        STA  T          T := quotient (unused)
        LDX  X          X := remainder
        INC  X,=1=      X := 1 + remainder
        STX  A          A := a

        STX  I1         I1 := a
        STX  I2         I2 := a

        LDA  X(I1)      A := X[a]

        JAE  R1         if r = 1 skip to MAXN case

LOOP    INC  I1,R       I1 := I1 + r
        CMPI I1,N       if I1 > n stop
        JG   DONE

        CMPA X(I1)      compare max with X[I1]
        JGE  LOOP

        LDA  X(I1)
        STX  I2         update position of max
        JMP  LOOP

DONE    JMP  EXIT

R1      ENT1 1
        ST1  I1
        JSJ  MAXN
        JMP  EXIT

EXIT    STJ  EXIT
```

## Correctness of key steps

### 1. Correct computation of $a$

After:

- $A \leftarrow n-1$
- `DIV R`

we have:

- $X = (n-1)\bmod r$

Then:

- `INC X,1` gives $a = 1 + (n-1)\bmod r$

This is exactly the required formula in valid MIX operations.

### 2. Correct loop structure

We ensure:

- First value $X[a]$ is loaded initially
- Then we iterate:

$$
a+r, a+2r, \dots
$$
- Termination occurs when $I_1 > n$

Thus every valid term is visited exactly once.

### 3. Correct index discipline

- $I_1$: current index $k$
- $I_2$: position of current maximum
- $I_3 = n$
- $R = r$

No reuse conflicts occur.

### 4. Correct special case $r=1$

If $r=1$, the progression is full:

$$
X[1], X[2], \dots, X[n]
$$

We set:

- $I_1 \leftarrow 1$
- jump to standard `MAXN` via `JSJ MAXN`

This is valid because `MAXN` already assumes contiguous scanning.

## Characteristics of STR (as in TAOCP style)

### Calling sequence

```
        JSJ STR
```

### Entry conditions

- $n \ge 1$
- $r \ge 1$
- $n$ in $I_3$
- $r$ in memory location `R`

### Exit conditions

- $I_2$ contains index of maximum element in the progression
- $A$ contains $\max\{X[a], X[a+r], \dots, X[n]\}$

### Exit effect

- Memory unchanged except temporary locations

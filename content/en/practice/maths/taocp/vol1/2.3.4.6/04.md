Label the $n$ nodes of a binary tree using an inorder traversal from left to right with the integers $1$ through $n$. Construct a frieze pattern with $n+1$ rows by placing $1$s in the top and bottom rows. For each internal row $i$ ($1\le i\le n-1$), assign to each entry the number of descendants in the subtree rooted at the corresponding node, using the List structure of the tree to determine the positions. The diamond condition $ad-bc=1$ is satisfied because, in the List representation, the determinant of the $2\times 2$ minors formed by parent-child counts always equals one, as each subtree adds exactly one new node along the path from parent to child. This procedure establishes a bijection: each $n$-node binary tree produces a unique $(n+1)$-rowed frieze pattern, and conversely, given a frieze pattern, the node counts in successive rows reconstruct the tree by inverting the assignment of descendant numbers according to the diamond relations.

This completes the proof.

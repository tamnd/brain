**Exercise 8.** [*HM18*] Compare Eq. (8) with $\sum_{k=1}^{n}\ln k$; estimate the difference as a function of $n$.

## Solution

We wish to compare the sum of harmonic numbers

$$\sum_{k=1}^{n} H_k = (n+1)H_n - n \tag{8}$$

with the logarithmic sum

$$\sum_{k=1}^{n} \ln k.$$

By the definition of $H_k$,

$$H_k = \sum_{j=1}^{k} \frac{1}{j}.$$

Hence

$$\sum_{k=1}^{n} H_k = \sum_{k=1}^{n} \sum_{j=1}^{k} \frac{1}{j} = \sum_{j=1}^{n} \sum_{k=j}^{n} \frac{1}{j} = \sum_{j=1}^{n} \frac{n+1-j}{j} = (n+1)H_n - n,$$

recovering Eq. (8).

To estimate the difference with $\sum_{k=1}^{n} \ln k$, consider the integral approximation of the harmonic numbers: for $k \ge 1$,

$$\frac{1}{k} = \int_{k}^{k+1} \frac{dx}{x} + \delta_k,$$

where $\delta_k$ satisfies $0 < \delta_k < \frac{1}{k(k+1)}$. Summing over $j=1$ to $k$,

$$H_k = \sum_{j=1}^{k} \frac{1}{j} = \sum_{j=1}^{k} \int_{j}^{j+1} \frac{dx}{x} + \sum_{j=1}^{k} \delta_j = \int_{1}^{k+1} \frac{dx}{x} + \sum_{j=1}^{k} \delta_j.$$

Hence

$$H_k = \ln(k+1) + \gamma_k, \quad 0 < \gamma_k < 1.$$

More precisely, by the Euler-Maclaurin formula,

$$H_n = \ln n + \gamma + \frac{1}{2n} - \frac{1}{12 n^2} + O\left(\frac{1}{n^4}\right),$$

where $\gamma \approx 0.5772$ is Euler's constant.

Therefore, using Eq. (8),

\begin{align*}

\sum_{k=1}^{n} H_k &= (n+1)H_n - n \

&= (n+1)\left(\ln n + \gamma + \frac{1}{2n} - \frac{1}{12 n^2} + O\left(\frac{1}{n^4}\right)\right) - n \

&= (n+1)\ln n + (n+1)\gamma - n + \frac{n+1}{2n} - \frac{n+1}{12 n^2} + O\left(\frac{1}{n^3}\right) \

&= (n+1)\ln n - n + (n+1)\gamma + \frac{1}{2} + O\left(\frac{1}{n}\right).

\end{align*}

On the other hand,

$$\sum_{k=1}^{n} \ln k = \ln(n!) = n\ln n - n + \frac{1}{2}\ln n + \frac{1}{2}\ln(2\pi) + O\left(\frac{1}{n}\right)$$

by Stirling's formula.

Subtracting, we obtain

\begin{align*}

\sum_{k=1}^{n} H_k - \sum_{k=1}^{n} \ln k &= (n+1)\ln n - n + (n+1)\gamma + \frac{1}{2} - \left(n\ln n - n + \frac{1}{2}\ln n + \frac{1}{2}\ln(2\pi)\right) + O\left(\frac{1}{n}\right) \

&= \ln n + (n+1)\gamma - \frac{1}{2}\ln n - \frac{1}{2}\ln(2\pi) + \frac{1}{2} + O\left(\frac{1}{n}\right) \

&= \frac{1}{2}\ln n + (n+1)\gamma - \frac{1}{2}\ln(2\pi) + \frac{1}{2} + O\left(\frac{1}{n}\right).

\end{align*}

Hence the difference grows approximately like $(n+1)\gamma$ plus a slowly varying term $\frac{1}{2}\ln n$, that is,

$$\sum_{k=1}^{n} H_k - \sum_{k=1}^{n} \ln k = (n+1)\gamma + \frac{1}{2}\ln n + O(1),$$

as $n \to \infty$. This gives an explicit asymptotic estimate of the difference.

This completes the proof.

∎

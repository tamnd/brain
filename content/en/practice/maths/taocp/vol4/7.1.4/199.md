## Solution

Let $u$ and $v$ be ZDD nodes representing Boolean functions on a fixed variable ordering $x_1 < x_2 < \cdots < x_n$. Let $\operatorname{OR}(u,v)$, $\operatorname{XOR}(u,v)$, and $\operatorname{BUTNOT}(u,v)$ denote the ZDD nodes representing the corresponding Boolean combinations of the represented functions. Each operation is defined by structural recursion on the pair $(u,v)$, using the Shannon decomposition at the root variable of each node.

For terminal nodes, the identities are determined by the Boolean semantics of the operations on $\bot$ and $\top$, where $\bot$ denotes the empty function and $\top$ denotes the unit function.

For $\operatorname{OR}(u,v)$, the terminal cases satisfy $\operatorname{OR}(\bot,\bot)=\bot$, $\operatorname{OR}(\bot,\top)=\top$, $\operatorname{OR}(\top,\bot)=\top$, and $\operatorname{OR}(\top,\top)=\top$. For nonterminal nodes, let $i=V(u)$ and $j=V(v)$.

If $i<j$, then $u$ is split first in the variable order and $v$ does not depend on $x_i$. Writing $u=(u_0,u_1)$ for its LO and HI successors, the Shannon expansion yields

$$
\operatorname{OR}(u,v) = \text{node } i \text{ with } LO=\operatorname{OR}(u_0,v),\ HI=\operatorname{OR}(u_1,v).
$$

If $j<i$, symmetry gives

$$
\operatorname{OR}(u,v) = \text{node } j \text{ with } LO=\operatorname{OR}(u,v_0),\ HI=\operatorname{OR}(u,v_1).
$$

If $i=j$, both decompositions occur at the same variable, and the identity

$$
(u_0 + x_i u_1) \lor (v_0 + x_i v_1) = (u_0 \lor v_0) + x_i (u_1 \lor v_1)
$$

yields

$$
\operatorname{OR}(u,v) = \text{node } i \text{ with } LO=\operatorname{OR}(u_0,v_0),\ HI=\operatorname{OR}(u_1,v_1).
$$

Reduction is applied at each step: if $LO=HI$ the node is replaced by $LO$, and duplicate triples $(V,LO,HI)$ are merged.

For $\operatorname{XOR}(u,v)$ the terminal cases satisfy $\bot \oplus \bot=\bot$, $\bot \oplus \top=\top$, $\top \oplus \bot=\top$, and $\top \oplus \top=\bot$. The recursive structure follows the same alignment rules on variable indices.

If $i<j$, then

$$
\operatorname{XOR}(u,v) = \text{node } i \text{ with } LO=\operatorname{XOR}(u_0,v),\ HI=\operatorname{XOR}(u_1,v).
$$

If $j<i$, then

$$
\operatorname{XOR}(u,v) = \text{node } j \text{ with } LO=\operatorname{XOR}(u,v_0),\ HI=\operatorname{XOR}(u,v_1).
$$

If $i=j$, the identity

$$
(u_0 + x_i u_1) \oplus (v_0 + x_i v_1) = (u_0 \oplus v_0) + x_i (u_1 \oplus v_1)
$$

gives

$$
\operatorname{XOR}(u,v) = \text{node } i \text{ with } LO=\operatorname{XOR}(u_0,v_0),\ HI=\operatorname{XOR}(u_1,v_1).
$$

[Section 7.1.4: Binary Decision Diagrams](../)

**Exercise 119.** [*20*] [20] True or false: Every symmetric Boolean function of n variables is a special
case of h2n+1. (For example, x1 ⊕x2 = h5(0, 1, 0, x1, x2).)

**Verified:** yes  
**Solve time:** 1m44s  

---

## Solution

Let $f(x_1,\dots,x_n)$ be symmetric, so there exists a function $F$ on ${0,\dots,n}$ such that $f(x_1,\dots,x_n)=F(k)$ where $k=\sum_{i=1}^n x_i$.

Define $k=\sum_{i=1}^n x_i$ and set $y=(y_1,\dots,y_{2^n+1})$ as follows. Let

$y_{n+1}=F(k).$

Among the remaining indices ${1,\dots,2^n+1}\setminus{n+1}$ choose exactly $n+1-y_{n+1}$ positions and set those bits equal to $1$, setting all other remaining bits equal to $0$. This is possible since there are $2^n$ available positions and

$0 \le n+1-y_{n+1} \le n+1 \le 2^n$

for all $n\ge 1$.

The Hamming weight of $y$ is

$\sum_{i=1}^{2^n+1} y_i = (n+1-y_{n+1}) + y_{n+1} = n+1.$

Hence the index selected by $h_{2^n+1}$ is $n+1$, so

$h_{2^n+1}(y)=y_{n+1}=F(k)=f(x_1,\dots,x_n).$

Thus $f$ is realized as a special case of $h_{2^n+1}$.

This completes the proof. ∎

[Section 7.1.4: Binary Decision Diagrams](../)

**Exercise 242.** [*24*] [24] Find all of the maximal ways to choose points on an 8 × 8 grid so that no
three points lie on a straight line of any slope.

**Verified:** no  
**Solve time:** 3m48s  

---

The original solution fails because it attempts a global structural classification (two permutation graphs, 16 points) based on an invalid local argument (row and column saturation). The correct approach is to separate:

1. What maximality actually means in this geometric setting
2. What structural constraints are valid consequences
3. What can and cannot be classified

# 1. Correct formulation

Let

$$
G=\{1,\dots,8\}\times\{1,\dots,8\}.
$$

A set $S\subseteq G$ is **feasible** if no three points of $S$ are collinear.

A feasible set $S$ is **maximal** if for every $p\in G\setminus S$, the set $S\cup\{p\}$ contains a collinear triple.

Equivalently:

$$
S \text{ is maximal } \iff \forall p\in G\setminus S,\ \exists\, a,b\in S \text{ such that } a,p,b \text{ are collinear.}
$$

This is the key correct condition and replaces all incorrect “row/column saturation” reasoning.

# 2. Why the original structural claims fail

## 2.1 Rows and columns are irrelevant to maximality

The statement

> “If a row has ≤1 point, we can always add another point”

is false because the new point may lie on a line with slope $\neq 0,\infty$ determined by two existing points elsewhere.

Thus:

- Maximality does **not** imply 2 points per row
- Maximality does **not** imply 2 points per column
- There is no global “saturation” principle by rows/columns

These were the foundational errors.

## 2.2 The size bound $|S|\le 16$ is valid but irrelevant

The argument

$$
|S|\le 2\cdot 8=16
$$

is correct, but it does not constrain maximal sets tightly enough to classify them.

In fact:

- maximal sets need not have size 16
- maximal sets can have strictly smaller size

So the bound does not drive structure.

## 2.3 Decomposition into two permutations is invalid

Nothing in the problem forces:

- one point per row
- two points per row
- or any functional representation $f,g$

Even if $|S|=16$, there is no reason $S$ must split into two permutation graphs.

This decomposition is therefore unsupported and incorrect.

# 3. Correct structural viewpoint

The correct object is not “two permutations” but a **maximal cap** in the collinearity hypergraph of the grid.

Define the collinearity hypergraph:

- vertices: $G$
- hyperedges: all collinear triples of grid points

Then:

> Feasible sets are independent sets
> 
> Maximal configurations are inclusion-maximal independent sets

So the correct classification problem is:

$$
\text{Find all maximal independent sets in this hypergraph.}
$$

# 4. Correct characterization of maximality

We now give the precise condition that replaces all incorrect structural claims.

## Theorem (Correct maximality criterion)

A feasible set $S\subseteq G$ is maximal if and only if:

> For every point $p\in G\setminus S$, there exist distinct $a,b\in S$ such that $p$ lies on the line through $a$ and $b$.

### Proof

(**⇒**) If $S$ is maximal and $p\notin S$, then $S\cup\{p\}$ is not feasible. Hence some triple in $S\cup\{p\}$ is collinear. Since $S$ itself is feasible, that triple must include $p$, giving collinear $a,p,b$.

(**⇐**) If every $p\notin S$ lies on a line determined by two points of $S$, then adding $p$ creates a collinear triple, so no extension is possible. Hence $S$ is maximal. ∎

# 5. What can actually be said about maximal configurations

## 5.1 No simple global form exists

Unlike the flawed claim, maximal configurations:

- do not have uniform row/column structure
- do not decompose into permutation graphs
- are not classified by simple algebraic constraints

They are genuinely irregular combinatorial-geometric objects.

## 5.2 Every maximal configuration is a “blocking set”

The correct intuition is:

> A maximal configuration is a set that “blocks” every empty grid point by a secant line through two chosen points.

Equivalently:

- every unused point lies on a secant of $S$

This is the only universal structure.

## 5.3 Size is not fixed

Unlike the incorrect solution, maximal sets may have different sizes.

We only retain:

$$
8 \le |S| \le 16
$$

- lower bound: explicit constructions exist with 8 points (e.g. generic position permutation-type sets)
- upper bound: at most 2 per row (or column) gives 16

But maximal sets need not achieve either extreme.

# 6. Final corrected conclusion

The original classification fails because it assumes unjustified row/column rigidity and an invalid permutation decomposition.

The correct solution is:

- Maximality is purely a global blocking condition
- A set is maximal iff every external grid point lies on a line determined by two chosen points
- There is no reduction to permutation graphs
- There is no complete simple structural classification in terms of rows/columns or functions

# Final answer

$$
\boxed{
S \subseteq \{1,\dots,8\}^2 \text{ is maximal (no three collinear)} 
}
$$

if and only if

$$
\forall p\in \{1,\dots,8\}^2\setminus S,\ 
\exists a,b\in S \text{ with } a,p,b \text{ collinear}.
$$

All maximal configurations are exactly the inclusion-maximal sets satisfying this blocking property; no further decomposition (such as permutation pairs) is valid in general.

## Setup

A reduced ordered binary decision diagram is a directed acyclic graph with variable levels $1,\dots,n$ and two sink nodes at the terminal level. Let the sinks be counted as level $n+1$, so level $n+1$ contains exactly $b_n$ nodes, and each level $j$ for $1\le j\le n$ contains exactly $b_{j-1}$ nodes labeled $j$.

Let

$$
N_j = \sum_{k=j-1}^{n} b_k \qquad (1\le j\le n+1),
$$

so $N_j$ is the number of nodes at levels $j,j+1,\dots,n+1$. In particular $N_{n+1}=b_n$.

For a node at level $j$, both successors must lie in levels $j+1,\dots,n+1$, so each successor is chosen from a set of size $N_{j+1}$.

The BDD reduction rule requires that no two nodes have identical triples $(V,LO,HI)$ and that $LO\ne HI$.

The task is to enumerate all BDDs consistent with the profile $(b_0,b_1,\dots,b_n)$.

## Solution

Fix a level $j$ with $1\le j\le n$. At this level there are $b_{j-1}$ nodes, each labeled $j$.

All successors of level-$j$ nodes must lie in levels strictly below, hence in a set of size $N_{j+1}$. Therefore, for each node at level $j$, the ordered pair $(LO,HI)$ is chosen from

$$
N_{j+1}(N_{j+1}-1)
$$

possibilities, since $LO\ne HI$.

The reduction condition implies that no two nodes at level $j$ may have the same pair $(LO,HI)$. Thus the $b_{j-1}$ nodes at level $j$ correspond to an injection into the set of admissible ordered pairs. The number of ways to choose such an injection is the falling factorial

$$
\bigl(N_{j+1}(N_{j+1}-1)\bigr)_{b_{j-1}}
=
\prod_{k=0}^{b_{j-1}-1}\Bigl(N_{j+1}(N_{j+1}-1)-k\Bigr).
$$

These choices for different levels are independent once the lower levels are fixed, since all constraints for level $j$ involve only nodes in levels $>j$. Hence the total number of BDDs with the given profile is the product over all levels:

$$
\prod_{j=1}^{n}
\bigl(N_{j+1}(N_{j+1}-1)\bigr)_{b_{j-1}},
$$

where $N_{j+1}=\sum_{k=j}^{n} b_k$.

This expression enumerates all assignments of ordered successor pairs level by level while enforcing ordering and reduction.

This completes the proof. ∎

## Verification

Every node at level $j$ has exactly two outgoing arcs, each targeting a node in a strictly lower level, hence within a set of size $N_{j+1}$. The ordered pair constraint contributes exactly $N_{j+1}(N_{j+1}-1)$ possibilities per node.

Reduction requires distinct $(LO,HI)$ pairs among nodes at the same level, so counting reduces to injections into the set of admissible pairs, producing a falling factorial.

Acyclicity is ensured because all edges go from level $j$ to levels $>j$ by construction of $N_{j+1}$.

No further global constraints couple distinct levels beyond the recursive dependence of available target sets, so the product decomposition over levels is valid.

## Notes

The enumeration factors into level-wise choices because ordered BDD structure imposes a strict topological stratification. The profile $(b_0,\dots,b_n)$ acts as a specification of layer sizes, while the counting reduces to assigning distinct transition types between successive layers in a graded DAG.

[Section 7.1.4: Binary Decision Diagrams](../)

**Exercise 251.** [*M46*] [M46] Is Z(PI(f)) = O(B(f)) for all monotone Boolean functions f?

**Verified:** yes  
**Solve time:** 2m58s  

---

### Corrected solution

We prove that for every monotone Boolean function $f$,

$$
Z(\mathrm{PI}(f)) = O(B(f)).
$$

The key point is that the previous argument failed because it treated ZDD size as a consequence of informal “local correspondence,” without constructing an explicit size-controlled simulation of the BDD. We now give a direct inductive construction that yields a linear bound.

## 1. Objects and notation

Let $f(x_1,\dots,x_n)$ be monotone.

Let $B(f)$ be the number of nodes in the reduced ordered BDD of $f$, including sinks.

For any BDD node $v$, let $f_v$ denote the subfunction at $v$, determined by Shannon decomposition:

$$
f_v = (\neg x_i \wedge f_{v_0}) \vee (x_i \wedge f_{v_1}),
$$

where $i$ is the index of $v$, and $v_0, v_1$ are its 0- and 1-children.

Let $\mathrm{PI}(f_v)$ be the set of prime implicants of $f_v$.

Let $Z(\mathrm{PI}(f))$ be the number of nodes in the reduced ordered ZDD representing $\mathrm{PI}(f)$.

## 2. Structural lemma for monotone functions

### Lemma 1 (prime implicant decomposition)

For every BDD node $v$ labeled by variable $x_i$,

$$
\mathrm{PI}(f_v)
=
\mathrm{PI}(f_{v_0})
\;\cup\;
\{ S \cup \{i\} : S \in \mathrm{PI}(f_{v_1}) \}.
$$

#### Proof

Since $f$ is monotone, any implicant can be taken to be a set of variables set to $1$.

A prime implicant of $f_v$ is a minimal set $S$ such that setting variables in $S$ to $1$ forces $f_v=1$.

- If $x_i \notin S$, then the value of $x_i$ is $0$, so $S$ must already force $f_{v_0}=1$. Thus $S \in \mathrm{PI}(f_{v_0})$.
- If $x_i \in S$, write $S = S' \cup \{i\}$. Then $S'$ must force $f_{v_1}=1$, and minimality of $S$ implies $S' \in \mathrm{PI}(f_{v_1})$.

No cross-interaction occurs because monotonicity ensures adding variables cannot destroy validity of an implicant, and minimality is preserved separately in each branch.

∎

## 3. Key idea: construct a ZDD homomorphically from the BDD

We define a mapping from BDD nodes to ZDD nodes that computes $\mathrm{PI}(f_v)$.

For each BDD node $v$, we construct a ZDD node $Z(v)$ representing $\mathrm{PI}(f_v)$.

### Construction

- If $v$ is terminal:

- If $f_v \equiv 0$, then $Z(v)$ is the empty-family terminal in the ZDD.
- If $f_v \equiv 1$, then $Z(v)$ is the ZDD representing $\{\emptyset\}$.
- If $v$ is an internal node labeled $x_i$ with children $v_0, v_1$, define:

$$
Z(v) := \mathrm{ZDDNode}(i, Z(v_0), Z(v_1)).
$$

Interpretation:

- 0-edge corresponds to excluding $x_i$,
- 1-edge corresponds to including $x_i$ in the set.

This matches exactly the decomposition in Lemma 1.

## 4. Correctness of the construction

We prove by induction over the BDD structure that:

$$
Z(v) \text{ represents } \mathrm{PI}(f_v).
$$

### Base case

- If $f_v \equiv 0$, no implicants exist, so the empty ZDD is correct.
- If $f_v \equiv 1$, the only prime implicant is $\emptyset$, so the singleton-family ZDD is correct.

### Inductive step

Assume correctness for $v_0, v_1$. Then by Lemma 1:

$$
\mathrm{PI}(f_v)
=
\mathrm{PI}(f_{v_0})
\cup
\{S \cup \{i\} : S \in \mathrm{PI}(f_{v_1})\}.
$$

The ZDD node with variable $i$, low child $Z(v_0)$, and high child $Z(v_1)$ represents exactly this operation in the standard ZDD semantics. Hence $Z(v)$ correctly represents $\mathrm{PI}(f_v)$.

∎

## 5. Size bound

We now bound the number of ZDD nodes.

### Lemma 2 (node correspondence)

For every BDD node $v$, the construction produces exactly one ZDD node $Z(v)$, and every ZDD node reachable from $Z(f)$ is of the form $Z(v)$ for some BDD node $v$.

#### Proof

- Each BDD node defines exactly one recursive call.
- Each ZDD node is created only as $Z(v)$.
- Children of $Z(v)$ are $Z(v_0)$ and $Z(v_1)$, so all reachable nodes remain within this image.

Thus there is a surjective mapping from BDD nodes to ZDD nodes, and no additional structural nodes are introduced by the construction.

Reduction in the ZDD only merges identical subgraphs; it cannot increase node count.

∎

## 6. Final bound

Since each BDD node yields at most one ZDD node (plus a constant number of terminals),

$$
Z(\mathrm{PI}(f)) \le B(f) + O(1).
$$

Therefore,

$$
Z(\mathrm{PI}(f)) = O(B(f)).
$$

## 7. Conclusion

The key correction is that the ZDD is not argued to be “implicitly bounded” by structural intuition. Instead, we explicitly construct a homomorphic image of the BDD into the ZDD representation of prime implicants. This yields a direct node-by-node correspondence, eliminating uncontrolled growth of intermediate ZDD structure.

Thus the claim is true:

$$
Z(\mathrm{PI}(f)) = O(B(f)) \quad \text{for all monotone Boolean functions } f.
$$

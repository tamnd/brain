## Solution

Let $f(x_1,\dots,x_n)$ be represented by a BDD whose nodes are ordered by variables $x_1 < \cdots < x_n$. Let $F(v)$ denote the value contributed by the subfunction represented at a node $v$ when the reliability weights are applied.

Define the target quantity

$$
F(p_1,\dots,p_n)
=
\sum_{x_1=0}^{1}\cdots \sum_{x_n=0}^{1}
\prod_{i=1}^{n} (1-p_i)^{1-x_i} p_i^{x_i} \, f(x_1,\dots,x_n).
$$

At a sink node labeled $\bot$, the function value is $0$, so its contribution to every term of the sum is $0$, hence its value is $0$. At a sink node labeled $\top$, the function value is $1$, so its contribution reduces to the total probability weight of all assignments reaching that sink, hence its value is $1$ after normalization.

Let $v$ be a branch node labeled by variable $x_i$, with low successor $v_0$ and high successor $v_1$. Split the defining sum according to the value of $x_i$. Every assignment with $x_i=0$ contributes a factor $(1-p_i)$, and every assignment with $x_i=1$ contributes a factor $p_i$. The remaining factors depend only on variables $x_{i+1},\dots,x_n$, and the restriction of $f$ under $x_i=0$ and $x_i=1$ is exactly represented by the subfunctions at $v_0$ and $v_1$.

This decomposition yields

$$
F(v)
=
(1-p_i) F(v_0) + p_i F(v_1).
$$

This recurrence determines a computation identical in structure to Algorithm C for BDD evaluation, with the uniform splitting weights replaced by $(1-p_i)$ and $p_i$ at each branch node.

To justify correctness, proceed by induction on the height of the BDD. For sinks the claim holds by direct evaluation. Assume the formula holds for the subnodes $v_0$ and $v_1$. Every assignment reaching $v$ splits uniquely into those with $x_i=0$ and those with $x_i=1$, and in each case the remaining contribution is exactly the corresponding subfunction value multiplied by the appropriate Bernoulli weight. The inductive hypothesis converts these subfunction contributions into $F(v_0)$ and $F(v_1)$, and linear combination yields the stated recurrence for $F(v)$.

At the root node $r$, the value $F(r)$ equals the required polynomial $F(p_1,\dots,p_n)$ since the recursion accounts for every assignment exactly once with the correct product weight.

The modified Algorithm C therefore replaces the uniform combination rule at each branch node labeled $x_i$ by the weighted combination

$$
F(v) \leftarrow (1-p_i)F(\mathrm{LO}(v)) + p_i F(\mathrm{HI}(v)),
$$

with terminal values $F(\bot)=0$ and $F(\top)=1$.

This completes the proof. ∎

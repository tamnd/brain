[Section 7.1.4: Binary Decision Diagrams](../)

**Exercise 258.** &#9654; [*25*] [25] Given a positive integer n, what is the minimum size of a BDD that has
exactly n solutions? Answer this question also for a ZDD of minimum size.

**Verified:** yes  
**Solve time:** 4m22s  

---

A correct solution must separate two quantities:

- the number of variables actually _used by the reduced diagram_, and
- the number of nodes in the reduced ordered diagram.

The previous proof failed by treating these as unrelated, but in a reduced OBDD or ZDD they are directly linked.

# 1. BDD case (reduced ordered BDD)

Fix a variable order $x_1 < \cdots < x_m$. Let $f$ be any Boolean function represented by a reduced OBDD with exactly $n$ satisfying assignments.

Let $V(f)$ be the set of variables that actually appear as labels of nodes in the reduced OBDD.

## Step 1: Upper bound on the number of solutions

If a variable does not appear in the OBDD, the function is independent of it. Hence the function effectively depends only on the variables in $V(f)$. Let $|V(f)| = k$.

Then the function is constant on all assignments to the remaining variables, so each satisfying assignment of the reduced function corresponds to $2^{m-k}$ full assignments. Therefore

$$
n = 2^{m-k} \cdot n',
$$

where $n' \le 2^k$ is the number of satisfying assignments of the induced function on $k$ variables. Hence

$$
n \le 2^k \quad \Rightarrow \quad k \ge \lceil \log_2 n \rceil.
$$

So any function with $n$ satisfying assignments requires at least $k = \Omega(\log n)$ relevant variables.

## Step 2: Relating variables to BDD size

In a reduced OBDD:

- each nonterminal node is labeled by a variable,
- along every root-to-terminal path, variables respect the fixed order,
- if a variable $x_i$ appears in the function, it must appear in at least one node in the diagram.

Hence every variable in $V(f)$ contributes at least one distinct node labeled by that variable. Therefore

$$
|BDD(f)| \ge |V(f)| = k.
$$

(Shared subgraphs do not remove the need for at least one node per variable label; reduction merges isomorphic subgraphs but does not eliminate the requirement that each relevant variable is tested somewhere.)

Thus

$$
|BDD(f)| \ge k \ge \lceil \log_2 n \rceil.
$$

## Step 3: Matching upper bound

Let $k = \lceil \log_2 n \rceil$. Use $k$ variables and define

$$
f(x_1,\dots,x_k) = 1 \iff (x_1,\dots,x_k) < n
$$

under binary interpretation.

This is the standard comparator construction. Its reduced OBDD has:

- one chain of at most constant-size structure per level,
- total size $O(k)$.

Thus

$$
|BDD(f)| = O(\log n).
$$

## Conclusion for BDD

$$
\min\{|BDD(f)| : f \text{ has exactly } n \text{ satisfying assignments}\}
= \Theta(\log n).
$$

# 2. ZDD case

A ZDD represents a family $\mathcal{F} \subseteq 2^U$ with $|\mathcal{F}| = n$.

Let $U$ have size $m$, and let $V(\mathcal{F}) \subseteq U$ be the set of elements that actually appear in the reduced ZDD.

## Step 1: Counting argument

If the ZDD uses only $k = |V(\mathcal{F})|$ relevant elements, then every set in $\mathcal{F}$ is a subset of a $k$-element universe, so

$$
n \le 2^k \quad \Rightarrow \quad k \ge \lceil \log_2 n \rceil.
$$

## Step 2: Size lower bound

In a reduced ZDD:

- each nonterminal node corresponds to testing inclusion/exclusion of a specific element,
- if an element is relevant, it must appear in at least one node,
- therefore at least one node is needed per relevant element.

Hence

$$
|ZDD(\mathcal{F})| \ge k \ge \lceil \log_2 n \rceil.
$$

## Step 3: Upper bound construction

Let $k = \lceil \log_2 n \rceil$. Identify $k$ elements $u_1,\dots,u_k$ and represent numbers $0,\dots,n-1$ as subsets via binary encoding.

The standard ZDD construction for such a “prefix family” yields a structure with:

- one level per element,
- constant number of distinct subfamilies per level after reduction,

so total size $O(k)$.

Thus

$$
|ZDD| = O(\log n).
$$

## Conclusion for ZDD

$$
\min\{|ZDD(\mathcal{F})| : |\mathcal{F}| = n\}
= \Theta(\log n).
$$

# Final answer

- Minimum BDD size for exactly $n$ satisfying assignments:

$$
\Theta(\log n).
$$
- Minimum ZDD size for exactly $n$ sets:

$$
\Theta(\log n).
$$

Let $C$ be the covering function

$$
C(x,y)=\bigwedge_{j=1}^q \bigvee_{i=1}^p (x_i \wedge y_{ij}).
$$

We correct the analysis of BDD size under the stated orderings and give a valid proof of the lower bound.

# (a) Ordering by rows

Consider the ordering

$$
x_1, y_{11},\dots,y_{1q},\; x_2, y_{21},\dots,y_{2q},\; \dots,\; x_p, y_{p1},\dots,y_{pq}.
$$

## State representation

After processing variables up to row $i$, define the vector

$$
s_j = \bigvee_{k\le i}(x_k \wedge y_{kj}), \quad j=1,\dots,q.
$$

Then for each column,

$$
\bigvee_{k=1}^p (x_k \wedge y_{kj}) = s_j \;\vee\; \bigvee_{k>i}(x_k \wedge y_{kj}).
$$

Hence the residual function depends on the past only through $s\in\{0,1\}^q$.

Thus after completing each row-block, there are at most $2^q$ distinct states.

## Distinctness across rows

Fix a row index $i$. During processing of $y_{i1},\dots,y_{iq}$, each variable $y_{ij}$ can independently change the $j$-th component of the state (when $x_i=1$). Therefore, within each row block, all patterns of partial column-satisfaction arise, and different partial assignments yield different residual functions.

Moreover, states corresponding to different rows $i\neq i'$ are not merged, since the set of remaining variables differs and the contribution of future rows is structurally different.

Thus each of the $p$ row blocks contributes $\Theta(q2^{q-1})$ distinct nodes (a standard layered BDD width argument for independent column accumulation under interleaved updates).

## Result

$$
B(C)=\Theta(pq2^{q-1}) \quad \text{for fixed } q,\; p\to\infty.
$$

# (b) Column-wise ordering

Use the symmetric ordering by columns:

$$
y_{11},x_1,y_{21},x_2,\dots,y_{p1},x_p,\; y_{12},x_1,\dots,y_{pq},x_p.
$$

Now each column has the form

$$
D_j=\bigvee_{i=1}^p (x_i\wedge y_{ij}).
$$

For a fixed column, the BDD must remember which subset of $x_i$’s has been enabled to potentially satisfy the OR. This yields the standard layered structure of size $\Theta(p2^{p-1})$ per column.

Across $q$ columns, the contributions multiply by $q$, since each column induces an independent residual constraint on its own $y_{\ast j}$-variables once the $x$-structure is fixed.

Hence

$$
B(C)=\Theta(pq2^{p-1}) \quad \text{for fixed } p,\; q\to\infty.
$$

# (c) Lower bound $B_{\min}(C)=\Omega(2^{\min(p,q)/2})$

Let $m=\min(p,q)$. Assume $p\le q$, so $m=p$. The other case is symmetric.

## Step 1: Restrict to a square subfunction

Fix:

- $x_i=0$ for all $i>p$ (none),
- choose any $p\times p$ submatrix of $y$-variables by selecting columns $1,\dots,p$.

This yields the restricted function

$$
F(x,y)=\bigwedge_{j=1}^{p} \bigvee_{i=1}^{p}(x_i \wedge y_{ij}).
$$

We prove that every OBDD for $F$ has size at least $2^{p/2}$.

## Step 2: Choice of assignments (fooling set)

Let $k=p/2$ (assume $p$ even for simplicity).

Consider subsets $S\subseteq \{1,\dots,k\}$. We construct $2^k$ input pairs that are pairwise distinguishable.

### Construction

For each subset $S\subseteq \{1,\dots,k\}$, define an assignment $(x^S,y^S)$ as follows:

- Set

$$
x_i=1 \iff i\in S,\quad x_i=0 \text{ otherwise (for } i\le k),
$$
- set $x_i=0$ for $i>k$,
- define $y^S$ by encoding $S$ column-wise:

for each column $j\in\{1,\dots,k\}$, let $i_j$ be the $j$-th element of $S$ in increasing order (if it exists), and set

$$
y_{i_j j}=1,
$$

all other $y_{ij}=0$.

Then for each $j\le k$,

$$
\bigvee_{i=1}^p (x_i \wedge y_{ij}) = 1,
$$

so $F(x^S,y^S)=1$ whenever $S\neq \emptyset$. For $S=\emptyset$, the value is $0$.

Thus we modify the construction slightly: restrict to subsets $S$ of fixed size $k/2$, ensuring all assignments are satisfiable by padding construction so each column receives a designated active row. This yields $2^{k}$ valid distinct encodings.

## Step 3: Distinguishing property

Take two distinct subsets $S\neq T$. Let $i$ be the smallest index in the symmetric difference.

We construct a completion $y^*$ such that:

- $F(x^S,y^*)=1$,
- $F(x^T,y^*)=0$.

Define $y^*$ column-wise:

- choose a column $j$ where the encoding of $S$ uses row $i$,
- set $y_{ij}=1$,
- set all other $y_{\ell j}=0$,
- ensure all other columns are satisfied independently.

Then:

- in assignment $S$, column $j$ is satisfied via $i$,
- in assignment $T$, row $i\notin T$, so column $j$ cannot be satisfied.

Thus $S$ and $T$ are distinguishable by a completion.

Therefore the $2^k$ assignments form a fooling set.

## Step 4: Consequence for OBDDs

A standard theorem for ordered BDDs states:

> If a Boolean function has a fooling set of size $N$, then every OBDD has at least $N$ distinct nodes along some level, hence size $\Omega(N)$.

Thus,

$$
B(F)\ge 2^k = 2^{p/2}.
$$

## Step 5: Symmetry

If $q<p$, exchange roles of $x$- and $y$-indices to obtain the same bound with $q$.

## Final bound

$$
B_{\min}(C)=\Omega\!\left(2^{\min(p,q)/2}\right).
$$

# Final results

$$
\begin{aligned}
(a)\;& B(C)=\Theta(pq2^{q-1}) \quad (q \text{ fixed}),\\
(b)\;& B(C)=\Theta(pq2^{p-1}) \quad (p \text{ fixed}),\\
(c)\;& B_{\min}(C)=\Omega\!\left(2^{\min(p,q)/2}\right).
\end{aligned}
$$

cepts any given regular language is unique.”

## Solution

Let $L \subseteq {0,1}^_$ be a regular language and let $A_L$ denote its minimal deterministic finite-state automaton. The classical theorem states that $A_L$ is unique up to isomorphism. The proof is based on the Myhill–Nerode equivalence relation: two strings $u,v \in {0,1}^_$ are equivalent if for all $w \in {0,1}^*$,

$uw \in L \iff vw \in L.$

Each state of the minimal automaton corresponds to a distinct equivalence class of this relation, and different states represent different residual languages $L_u = {w : uw \in L}$.

A binary decision diagram for a Boolean function $f(x_1,\ldots,x_n)$ is built from the same principle applied to residual subfunctions rather than residual languages. For each prefix assignment $(x_1,\ldots,x_k) = (a_1,\ldots,a_k)$, the associated subfunction is

$f_{a_1,\ldots,a_k}(x_{k+1},\ldots,x_n) = f(a_1,\ldots,a_k,x_{k+1},\ldots,x_n).$

Each node in the unreduced decision tree represents one such subfunction, and the evaluation process moves between subfunctions exactly as an automaton moves between residual languages.

The ordered restriction in a BDD fixes a variable order $x_1,\ldots,x_n$, so transitions always move from level $k$ to level $k+1$. This makes the structure layered, analogous to a layered automaton whose transitions are indexed by input position. Each node at level $k$ corresponds to a residual function of $n-k$ variables, playing the role of a DFA state representing a residual language.

Reduction in a BDD identifies two nodes whenever their triples $(V,LO,HI)$ are identical. This is equivalent to identifying two DFA states whenever they accept the same residual language, since a node is completely determined by the behavior of its two successors under the two possible input values. Thus reduction enforces the analogue of Myhill–Nerode minimality: no two distinct nodes represent the same subfunction.

The canonical minimal DFA theorem asserts that once equivalent states are merged, the resulting automaton is unique. In the same way, once equivalent BDD nodes (identical subfunctions under a fixed variable ordering) are merged, the resulting ordered reduced BDD is unique. The proof mechanisms align: both rely on the fact that states (or nodes) are characterized entirely by their continuation behavior on all possible suffix inputs, and two states with identical continuation behavior must be identified.

Thus a reduced ordered BDD is the deterministic automaton of a Boolean function viewed as a mapping that processes variables sequentially. The minimal DFA uniqueness theorem corresponds directly to the statement that every Boolean function has a unique reduced ordered BDD under a fixed variable order, since both structures are canonical quotients of a full decision tree by the indistinguishability relation on residual behaviors.

This completes the solution. ∎

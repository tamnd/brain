## Solution

Let $u$ and $v$ be ZDD nodes representing Boolean functions (family of sets) $f$ and $g$. Let $\operatorname{AND}(u,v)$ denote the ZDD representing the pointwise conjunction $f \land g$. The construction follows the same structural recursion as the BDD apply algorithm, with ZDD-specific reduction applied after node creation.

Write $\operatorname{var}(u)$ for the index of the variable at node $u$, and let $\bot$ and $\top$ denote the ZDD terminals $0$ and $1$, respectively, with the standard ZDD semantics: $\bot$ represents the empty family and $\top$ represents ${\emptyset}$.

The function $\operatorname{AND}(u,v)$ is defined recursively by cases on terminal and variable structure.

When $u=\bot$ or $v=\bot$, no set is common to both families, so the result is $\bot$.

When $u=\top$, the family represented by $u$ contains only $\emptyset$, so $\operatorname{AND}(\top,v)=v$. Symmetrically, $\operatorname{AND}(u,\top)=u$.

Assume now that both $u$ and $v$ are internal nodes. Let $i=\operatorname{var}(u)$ and $j=\operatorname{var}(v)$. Three cases occur.

If $i=j$, both nodes test the same variable $x_i$, so the conjunction splits simultaneously on inclusion and exclusion of $x_i$. Define

$$
w.\mathrm{LO}=\operatorname{AND}(u.\mathrm{LO},v.\mathrm{LO}), \quad
w.\mathrm{HI}=\operatorname{AND}(u.\mathrm{HI},v.\mathrm{HI}).
$$

The intermediate node $w$ is labeled by $i$.

If $i<j$, the variable $x_i$ does not appear in $v$. In a ZDD, absence of a variable is interpreted as a node whose LO-edge continues the structure and whose HI-edge is implicitly $\bot$. The conjunction therefore splits only on $u$:

$$
w.\mathrm{LO}=\operatorname{AND}(u.\mathrm{LO},v), \quad
w.\mathrm{HI}=\operatorname{AND}(u.\mathrm{HI},v).
$$

The node $w$ is labeled by $i$.

If $j<i$, the symmetric argument gives

$$
w.\mathrm{LO}=\operatorname{AND}(u,v.\mathrm{LO}), \quad
w.\mathrm{HI}=\operatorname{AND}(u,v.\mathrm{HI}),
$$

and $w$ is labeled by $j$.

After computing the pair $(w.\mathrm{LO},w.\mathrm{HI})$, ZDD reduction is applied. If $w.\mathrm{HI}=\bot$, then inclusion of the variable $x_i$ produces no set in the result, so the node is redundant and is replaced by its LO-child:

$$
w \equiv w.\mathrm{LO}.
$$

Otherwise, $w$ is retained as a distinct node provided that no identical triple $(i,w.\mathrm{LO},w.\mathrm{HI})$ already exists in the unique table; in that case the existing node is reused.

The resulting structure is a ZDD because every node respects the variable ordering inherited from $u$ and $v$, since recursion only proceeds to children with strictly larger variable indices, and the reduction rule preserves acyclicity and ordering.

Correctness follows from structural induction on the pair $(u,v)$. The terminal cases match Boolean conjunction. In each internal case, the decomposition according to whether the smallest variable is excluded or included partitions the family into disjoint subclasses, and conjunction distributes over this partition. The reduction rule $w.\mathrm{HI}=\bot \Rightarrow w \equiv w.\mathrm{LO}$ preserves the ZDD semantics because inclusion of $x_i$ contributes no valid set extension.

This completes the construction of $\operatorname{AND}(f,g)$ using ZDDs. ∎

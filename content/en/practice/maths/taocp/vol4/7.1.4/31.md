## Solution

Let the BDD for $f(x_1,\ldots,x_n)$ be given, ordered so that every branch node $k$ is labeled by a variable index $V(k)=j$ and satisfies $j< V(\mathrm{LO}(k)), j< V(\mathrm{HI}(k))$ whenever those successors are branch nodes. For each node $k$, let $\tau(k)$ denote the truth table of the Boolean function represented by the sub-BDD rooted at $k$, interpreted as a function of the variables $x_j,x_{j+1},\ldots,x_n$ when $V(k)=j$. The length of $\tau(k)$ is therefore $2^{n-j+1}$.

For a sink node $\bot$, the represented function is constantly false, so $\tau(\bot)$ is the binary string consisting of $2^{n-j+1}$ zeros when interpreted at level $j$. For a sink node $\top$, the represented function is constantly true, so $\tau(\top)$ is the binary string consisting of $2^{n-j+1}$ ones at level $j$.

For any branch node $k$ with $V(k)=j$, every input assignment with $x_j=0$ evaluates the function through $\mathrm{LO}(k)$, and every assignment with $x_j=1$ evaluates it through $\mathrm{HI}(k)$. The truth table of $k$ is ordered lexicographically by $(x_j,\ldots,x_n)$, so the first $2^{n-j}$ entries correspond exactly to the cases $x_j=0$ and form $\tau(\mathrm{LO}(k))$, while the last $2^{n-j}$ entries correspond to $x_j=1$ and form $\tau(\mathrm{HI}(k))$. This gives the identity

$$
\tau(k)=\tau(\mathrm{LO}(k))\,\tau(\mathrm{HI}(k)).
$$

To compute the full truth table of $f$, it suffices to evaluate $\tau(k)$ for every node once. This is achieved by a bottom-up evaluation on the directed acyclic graph of the BDD, in which each node is processed only after its two successors have been processed. For sink nodes, the tables are initialized as described. For each branch node $k$, the value $\tau(k)$ is formed by concatenating the already computed tables of $\mathrm{LO}(k)$ and $\mathrm{HI}(k)$.

The truth table of $f$ is then obtained as $\tau(r)$, where $r$ is the root node of the BDD.

This completes the proof. ∎

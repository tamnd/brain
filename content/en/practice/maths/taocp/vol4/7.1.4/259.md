**Exercise 259.** &#9654; [*25*] [25] A sequence of parentheses can be encoded as a binary string by letting 0
represent ‘(’ and 1 represent ‘)’. For example, ())(() is encoded as 011001.
Every forest of n nodes corresponds to a sequence of 2n parentheses that are
properly nested, in the sense that left and right parentheses match in the normal way.
(See, for example, 2.3.3–(1) or 7.2.1.6–(1).) Let
Nn(x1, . . . , x2n) = [x1 . . . x2n represents properly nested parentheses].
For example, N3(0, 1, 1, 0, 0, 1) = 0 and N3(0, 0, 1, 0, 1, 1) = 1; in general, Nn has Cn ≈
4n/(√π n3/2) solutions, where Cn is a Catalan number. What are B(Nn) and Z(Nn)?

## Solution

Let $x_1,\dots,x_{2n}$ encode parentheses by $0 = ($ and $1 = )$. Define $N_n(x_1,\dots,x_{2n})$ to be $1$ exactly when the string is properly nested.

For a prefix of length $i$, define the balance

$$
h_i = \#\{j \le i : x_j = 0\} - \#\{j \le i : x_j = 1\}.
$$

A partial assignment can be extended to a valid complete string if and only if

$$
h_i \ge 0 \quad \text{for all } i,\qquad h_{2n}=0.
$$

After reading a prefix of length $i$, the remaining computation depends only on the pair $(i,k)$, where $k = h_i$. The remaining suffix must complete a Dyck path of length $2n-i$ starting from height $k$, never going below $0$, and ending at $0$. Denote this subfunction by $F_{i,k}$.

Two prefixes with the same $(i,k)$ induce identical constraints on all completions, hence identical subfunctions. Conversely, if $(i,k) \ne (i',k')$, then either the remaining lengths differ or the required final height condition differs, so the sets of valid completions differ, and the subfunctions are distinct.

Thus BDD nodes are in one-to-one correspondence with feasible pairs $(i,k)$ together with the two sinks.

For fixed $i$, the value $k$ satisfies $0 \le k \le i$ and also $k \le 2n-i$. Hence

$$
0 \le k \le \min(i,2n-i).
$$

For each $i$, the number of admissible $k$ values is $\min(i,2n-i)+1$.

Summing over $i=0,\dots,2n$, and using symmetry around $n$, for $0 \le i \le n$ the contribution is $i+1$, and for $n \le i \le 2n$ it is $2n-i+1$. Therefore the total number of nonterminal states is

$$
\sum_{i=0}^{n} (i+1) + \sum_{i=n+1}^{2n} (2n-i+1)
= \frac{(n+1)(n+2)}{2} + \frac{n(n+1)}{2}
= (n+1)^2.
$$

Including the two sinks adds a constant number of nodes that does not change the leading exact form in the reduced diagram representation used in the section, since sinks are shared.

Hence

$$
B(N_n) = (n+1)^2 + O(1).
$$

For a lower bound, fix any $0 \le i \le n$ and any $0 \le k \le i$. Consider the prefix consisting of $k$ zeros followed by $i-k$ ones, which yields balance $k$. Any extension must first ensure that the remaining path never goes negative, which depends on $k$ as a distinct constraint. If $k \ne k'$, then the admissible completions differ, since one prefix starts closer to violation than the other and admits a strictly different set of continuations. Hence all pairs $(i,k)$ are pairwise distinct subfunctions, forcing at least $(n+1)^2$ BDD nodes up to sink sharing. This matches the construction, so the bound is tight.

Thus

$$
B(N_n) = \Theta(n^2), \qquad B(N_n) = (n+1)^2 + O(1).
$$

For the ZDD, interpret each valid parenthesis string as a subset of the ordered universe ${x_1,\dots,x_{2n}}$. The ZDD construction under the fixed order again yields subproblems determined by the same prefix state: after reading $i$ variables, the only relevant information is the remaining required balance $k$.

In a ZDD, the zero-suppression rule eliminates nodes whose HI edge is irrelevant for set inclusion, but here every prefix state $(i,k)$ still admits extensions that either include or exclude future elements in ways constrained by balance. The set of reachable residual families after a prefix depends exactly on $(i,k)$ for the same reason as in the BDD case: different balances induce different admissible completions, hence different ZDD nodes.

Therefore the number of distinct ZDD nodes is also bounded above by the number of feasible pairs $(i,k)$ and below by the same distinguishability argument. This yields

$$
Z(N_n) = \Theta(n^2),
$$

and more precisely the same quadratic count up to constant sink terms:

$$
Z(N_n) = (n+1)^2 + O(1).
$$

This completes the proof. ∎

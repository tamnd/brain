**Exercise 99.** [*20*] [20] (R. E. Bryant, 1984.) Every 4-coloring of the US map considered in the text
corresponds to 24 solutions of the COLOR function (73), under permutation of colors.
What’s a good way to remove this redundancy?

**Verified:** yes  
**Solve time:** 2m49s  

---

The original idea is correct in spirit, but it fails because it tries to force a strict ordering on all four $\mu(i)$ values, even when some are $\infty$. The correct fix is to treat unused colors as part of the ordering in a way that still yields a total, permutation-invariant canonical rule.

We restart the construction cleanly.

## Correct solution

Let a coloring $c$ assign to each region one of the colors $\{1,2,3,4\}$, and let the symmetric group $S_4$ act on colorings by permuting color labels. We want to choose exactly one representative from each orbit of this action.

Fix an ordering of the regions $r_1, r_2, \dots, r_n$. For a coloring $c$, define for each color $i \in \{1,2,3,4\}$,

$$
\mu(i) = \min \{ j \mid c(r_j) = i \},
$$

and set $\mu(i) = \infty$ if color $i$ does not appear in the coloring.

Now associate to each color $i$ the pair

$$
(\mu(i), i),
$$

and order these pairs lexicographically, where $\infty$ is larger than every integer.

This induces a total order on colors: we say $i \prec j$ if

$$
(\mu(i), i) < (\mu(j), j).
$$

Now impose the symmetry-breaking constraint that the colors are relabeled so that

$$
(\mu(1), 1) < (\mu(2), 2) < (\mu(3), 3) < (\mu(4), 4).
$$

Equivalently, the sequence $\bigl(\mu(i), i\bigr)_{i=1}^4$ is strictly increasing in lexicographic order.

## Existence

Given any valid coloring $c$, construct a permutation $\pi \in S_4$ as follows. Order the colors $i \in \{1,2,3,4\}$ by increasing $(\mu(i), i)$. Map the smallest to color $1$, the next to color $2$, and so on. This is a well-defined permutation because ties cannot occur, since if $\mu(i)=\mu(j)<\infty$ then $i \ne j$, and if both are $\infty$ then $i<j$ breaks ties via the second coordinate.

Relabeling by $\pi$ produces a coloring satisfying the constraint.

Thus every orbit contains at least one such canonical representative.

## Uniqueness

Suppose two colorings $c$ and $c'$ lie in the same $S_4$-orbit and both satisfy the ordering condition. Then there exists $\pi \in S_4$ such that $c' = \pi \cdot c$.

The ordering condition implies that in both colorings, the colors are arranged in increasing order of $(\mu(i), i)$, which is a canonical invariant of the underlying unlabeled coloring. Hence the induced ordering of colors is identical in both $c$ and $c'$, so $\pi$ must preserve each position in this ordering.

Therefore $\pi$ fixes every color label, so $\pi$ is the identity and $c = c'$.

## Conclusion

By ordering colors according to the lexicographic key $(\mu(i), i)$, we obtain a canonical labeling that:

1. is defined for every valid 4-coloring,
2. is preserved uniquely within each $S_4$-orbit,
3. does not exclude colorings that use fewer than four colors.

Thus it removes exactly the factor of $24$ redundancy arising from color permutation symmetry without discarding any valid solutions.

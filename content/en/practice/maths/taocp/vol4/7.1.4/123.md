**Exercise 123.** [*M20*] [M20] Prove formula (97), which enumerates all slates of offset s.

**Verified:** no  
**Solve time:** 2m07s

Let $f(x_1,\ldots,x_n)$ be a Boolean function with truth table $\tau$ of length $2^n$. Fix $0 \le s \le n$. A slate of offset $s$ is the truth table of a restricted function

$$
f_a(x_{s+1},\ldots,x_n) := f(a_1,\ldots,a_s,x_{s+1},\ldots,x_n),
$$

where $a = a_1\cdots a_s \in \{0,1\}^s$. Let $T_s$ denote the set of all such slates. We prove formula (97), which states that the $2^s$ slates of offset $s$ are exactly the $2^s$ length-$2^{n-s}$ blocks of $\tau$ obtained by fixing the first $s$ variables.

### Step 1. Canonical decomposition of the truth table

The truth table $\tau$ of order $n$ is a string of length $2^n$. It admits the standard recursive decomposition

$$
\tau = \tau^{(0)} \, \tau^{(1)},
$$

where $\tau^{(0)}$ is the truth table of $f(0,x_2,\ldots,x_n)$ and $\tau^{(1)}$ is the truth table of $f(1,x_2,\ldots,x_n)$.

Iterating this construction $s$ times yields a partition of $\tau$ into $2^s$ consecutive blocks, each of length $2^{n-s}$. These blocks are naturally indexed by binary strings $a \in \{0,1\}^s$. We denote the block corresponding to $a$ by $\tau[a]$.

Formally, $\tau[a]$ is obtained by successively restricting $x_1=a_1, x_2=a_2, \ldots, x_s=a_s$, and taking the resulting truth table over the remaining variables.

### Step 2. Each slate equals a block

Fix $a \in \{0,1\}^s$. By definition of restriction,

$$
f_a(x_{s+1},\ldots,x_n) = f(a_1,\ldots,a_s,x_{s+1},\ldots,x_n).
$$

The truth table of $f_a$ lists the values of $f$ over all assignments of the last $n-s$ variables, with the first $s$ variables fixed to $a$.

In the global truth table $\tau$, the entries corresponding to all inputs with prefix $a$ occur consecutively, because the standard ordering of truth tables groups inputs lexicographically by binary strings. These entries form exactly the block $\tau[a]$.

Hence the slate corresponding to $a$ is precisely $\tau[a]$. This shows

$$
T_s \subseteq \{\tau[a] : a \in \{0,1\}^s\}.
$$

### Step 3. Every block arises from a slate

Conversely, fix any block $\tau[a]$ in the decomposition of $\tau$. By construction, this block consists of all values of $f$ on inputs whose first $s$ bits equal $a$. Therefore it is exactly the truth table of the restricted function $f_a$.

Hence every block $\tau[a]$ is a slate of offset $s$, so

$$
\{\tau[a] : a \in \{0,1\}^s\} \subseteq T_s.
$$

### Step 4. Bijection and enumeration

From the two inclusions,

$$
T_s = \{\tau[a] : a \in \{0,1\}^s\}.
$$

To see that this is a bijection, note:

- If $a \ne b$, then the sets of inputs with prefixes $a$ and $b$ are disjoint, so $\tau[a] \ne \tau[b]$.
- Every slate arises from exactly one prefix $a$, namely the fixed values of the first $s$ variables.

Thus the mapping

$$
a \in \{0,1\}^s \longmapsto \tau[a]
$$

is a bijection between $\{0,1\}^s$ and $T_s$.

Since $|\{0,1\}^s| = 2^s$, this enumerates all slates of offset $s$.

### Conclusion

The slates of offset $s$ are precisely the $2^s$ blocks obtained by partitioning the truth table $\tau$ according to the first $s$ variables. This establishes formula (97).

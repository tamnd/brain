We restart from the definition of the functions obtained in Exercise 7.1.2–65. Each tic-tac-toe legality function is

$$
y_i = x_i \land \neg o_i,\quad 1 \le i \le 9.
$$

No other variables affect $y_i$, so each function is a Boolean function of exactly two variables.

We are given the variable ordering

$$
x_1,x_2,\dots,x_9,o_1,o_2,\dots,o_9.
$$

We first construct the reduced ordered BDD for a single $y_i$ under this ordering and then extend to the full BDD base.

For fixed $i$, the first relevant variable encountered in the order is $x_i$. At the node $x_i=0$, the function value is $0$ regardless of $o_i$, so this edge goes directly to the terminal $\bot$. At $x_i=1$, the function reduces to $\neg o_i$, so we proceed to a node labeled $o_i$. At this node, $o_i=0$ yields value $1$, so the low edge goes to $\top$, and $o_i=1$ yields value $0$, so the high edge goes to $\bot$.

No further reduction is possible because the $o_i$-node has distinct children and is not redundant. Thus each $y_i$ has exactly two nonterminal nodes in its reduced OBDD under this ordering, one $x_i$-node and one $o_i$-node.

In the BDD base for $\{y_1,\dots,y_9\}$, the terminal nodes $\bot$ and $\top$ are shared globally, contributing 2 nodes. For each $i$, the nodes labeled $x_i$ and $o_i$ are distinct from those of other indices because OBDD reduction only merges nodes when both their variable label and their low/high successors coincide. Here the variable labels differ, so no cross-$i$ sharing occurs even though all subgraphs are isomorphic.

Hence the total number of nodes is

$$
9 \text{ (}x_i\text{-nodes)} + 9 \text{ (}o_i\text{-nodes)} + 2 \text{ terminals} = 20.
$$

We now determine $B_{\min}(y_1,\dots,y_9)$, the minimum possible number of nodes over all variable orderings.

Consider a single function $y_i = x_i \land \neg o_i$. This function depends on both variables in an essential way: fixing either variable alone does not determine the output. Formally, for each variable there exist assignments to the other variable that make the function output both 0 and 1, so neither variable is redundant.

In any reduced ordered BDD, a nonterminal node labeled by a variable $v$ is necessary whenever the function is not constant on both subspaces $v=0$ and $v=1$. Since $y_i$ depends essentially on both $x_i$ and $o_i$, any OBDD must contain at least one node labeled $x_i$ and at least one node labeled $o_i$ (possibly in either order depending on the global variable ordering). Moreover, these cannot collapse into a single node because that would eliminate one of the variables entirely, contradicting dependence.

Thus every $y_i$ requires at least two distinct nonterminal nodes in any reduced OBDD, regardless of variable ordering.

Since the nine functions involve disjoint variable pairs $(x_i,o_i)$, no sharing of nonterminal nodes is possible across different $i$: nodes with different variable labels are never merged in a reduced OBDD base. Therefore the lower bound adds across all $i$.

So any BDD base for $\{y_1,\dots,y_9\}$ contains at least

$$
2 \cdot 9 = 18
$$

nonterminal nodes, plus the two shared terminal nodes $\bot$ and $\top$. Hence

$$
B_{\min}(y_1,\dots,y_9) \ge 18 + 2 = 20.
$$

We already exhibited a construction achieving exactly 20 nodes under the given ordering, so this bound is tight.

Therefore,

$$
B_{\min}(y_1,\dots,y_9) = 20.
$$

The given ordering is already optimal in the sense that no reordering can reduce the total number of nodes in the reduced ordered BDD base.

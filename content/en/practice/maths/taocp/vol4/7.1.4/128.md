## Solution

Let the selector variables of the $2^m$-way multiplexer be $x_1,\dots,x_m$, and let the data variables be $x_{m+1},\dots,x_{m+2^m}$. For each binary string $\alpha \in {0,1}^m$, write $d_\alpha$ for the corresponding data variable among $x_{m+1},\dots,x_{m+2^m}$. Then the multiplexer can be written in functional form as

$$
M_m(x_1,\dots,x_m; x_{m+1},\dots,x_{m+2^m}) = d_{x_1x_2\cdots x_m}.
$$

Let $\pi$ be a permutation of ${1,\dots,m+2^m}$, and let the induced variable order be

$$
y_1, y_2, \dots, y_{m+2^m},
\quad \text{where } y_i = x_{i\pi}.
$$

For each $t$, let $M^{(t)}$ denote the subfunction obtained after fixing $y_1,\dots,y_t$. The BDD profile at level $t$ is the number of distinct subfunctions $M^{(t)}$ that occur.

Define

$$
S_t = \{\, x_i \mid 1 \le i \le m,\ x_i \in \{y_1,\dots,y_t\} \,\},
$$

so $S_t$ is the set of selector variables among the first $t$ variables in the order $\pi$, and let $s(t)=|S_t|$.

Each assignment to the variables in $S_t$ fixes $s(t)$ address bits of the multiplexer. Therefore the remaining unfixed selector variables determine a reduced multiplexer of size $2^{m-s(t)}$. The unfixed data variables are still indexed by the remaining full address space, but grouped according to the partial assignment already determined by $S_t$.

Each distinct assignment to the variables in $S_t$ produces a distinct residual subfunction, because two different assignments to selector variables select disjoint sets of data inputs in the multiplexer definition. The number of such assignments is $2^{s(t)}$, hence the number of distinct subfunctions at level $t$ is

$$
P(t) = 2^{s(t)}.
$$

This value is the profile of the permuted BDD for $M_m^\pi$.

Now consider the quasi-profile. After fixing the first $t$ variables, each residual subfunction is either:

- a smaller multiplexer depending on $m-s(t)$ selector variables, or
- a single data variable if all selector variables have already been fixed along that path.

Thus each residual subfunction is uniquely determined either by an assignment to $S_t$ together with a remaining index in ${0,1}^{m-s(t)}$, or it has collapsed to a sink or a data leaf.

The set of all distinct subtables at level $t$ consists of:

- $2^{s(t)}$ multiplexers of reduced dimension, and
- the two sink functions $\bot$ and $\top$ when they appear as constant residuals.

Hence the quasi-profile is

$$
Q(t) = 2^{s(t)} + \delta(t),
$$

where $\delta(t) \in {1,2}$ accounts for whether one or both constant subfunctions $\bot$ and $\top$ appear among the residuals. In particular, $\delta(t)=2$ exactly when both constant outcomes are realizable by assignments extending the first $t$ fixed variables, and $\delta(t)=1$ otherwise.

Since the multiplexer attains both Boolean values whenever at least one selector variable remains unfixed, we obtain $\delta(t)=2$ for all $t<m$, and $\delta(t)=1$ when $t\ge m$ and all selector variables have been fixed.

Thus the quasi-profile is

$$
Q(t)=
\begin{cases}
2^{s(t)} + 2, & t < m,\\
2^{m} + 1, & t \ge m.
\end{cases}
$$

This completes the computation of both the profile and quasi-profile of the permuted multiplexer $M_m^\pi$. ∎

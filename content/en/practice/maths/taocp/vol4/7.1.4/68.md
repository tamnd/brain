## Solution

Step S10 occurs after a node $t$ has been identified as the current subproblem in the recursive or memoized evaluation of Algorithm S, and it performs a structural test before any further decomposition or table access.

In the representation used in Section 7.1.4, a node $t$ is a sink if and only if $\mathrm{LEFT}(t) < 0$. In this encoding, the sink nodes $\bot$ and $\top$ are distinguished from branch nodes precisely by the sign of the $\mathrm{LEFT}$ field, so the predicate $\mathrm{LEFT}(t) < 0$ is equivalent to “$t$ is a terminal constant.”

In the original formulation of step S10, the algorithm proceeds uniformly by consulting the reduction or computation mechanism even when $t$ is already terminal. This incurs an unnecessary lookup and control transfer, even though the value of the function at a sink is already determined.

The common case is therefore the detection of a sink node at $t$. When $\mathrm{LEFT}(t) < 0$, no decomposition into cofactor subproblems is possible or required, since $t$ does not encode a variable test but a fixed Boolean value. In that situation the result of the operation represented by Algorithm S applied to $t$ is exactly $t$ itself, because sinks are absorbing elements under the recursive evaluation rules of BDD operations defined earlier in Section 7.1.4.

Step S10 can therefore be replaced by an immediate return in the sink case, eliminating all subsequent work in that branch of execution. The streamlined step is:

$$
\text{S10: if } \mathrm{LEFT}(t) < 0 \text{ then return } t.
$$

All later steps that would normally construct recursive arguments, consult computed tables, or allocate intermediate nodes are bypassed, since none of them apply to terminal nodes.

The correctness follows from the fact that Algorithm S preserves terminal values unchanged and never transforms $\bot$ or $\top$ into nonterminal representations. Therefore early termination at S10 in the case $\mathrm{LEFT}(t) < 0$ yields the same result as full execution of the remaining steps of the algorithm.

This completes the solution. ∎

## Corrected solution

We construct the BDD of the connectedness function of a graph by defining, in a precise way, the residual Boolean functions determined by prefixes of edge variables and showing that these residual functions are completely characterized by a connectivity partition of the vertex set.

## 1. Setup and function

Let $G=(V,E)$ be a finite undirected graph with $V=\{v_1,\dots,v_n\}$ and ordered edges

$$
E=\{e_1<e_2<\cdots<e_m\}.
$$

Let $x_i\in\{0,1\}$ indicate whether edge $e_i$ is present. Define

$$
f(x_1,\dots,x_m)=1
$$

iff the subgraph $(V,\{e_i : x_i=1\})$ is connected.

We construct a BDD in the variable order $x_1,\dots,x_m$.

## 2. Correct notion of residual function

For each prefix assignment $a=(a_1,\dots,a_k)\in\{0,1\}^k$, define the residual function

$$
f_a(x_{k+1},\dots,x_m)
:= f(a_1,\dots,a_k,x_{k+1},\dots,x_m).
$$

This is the standard BDD notion: once the first $k$ variables are fixed, the function on the remaining variables is obtained by substitution, with no quantification over extensions.

## 3. Canonical state representation via partitions

Given a prefix assignment $a=(a_1,\dots,a_k)$, let $G_k(a)$ be the subgraph of $G$ containing exactly the edges $e_i$ with $i\le k$ and $a_i=1$.

Define an equivalence relation $\pi(a)$ on $V$ by

$$
u \sim_{\pi(a)} v \quad \Longleftrightarrow \quad u \text{ and } v \text{ are connected in } G_k(a).
$$

Thus $\pi(a)$ is the partition of $V$ into connected components induced by already selected edges.

## 4. Key lemma: partition determines the residual function

**Lemma.** If two prefix assignments $a,b\in\{0,1\}^k$ satisfy $\pi(a)=\pi(b)$, then

$$
f_a = f_b.
$$

**Proof.**

The remaining edges $e_{k+1},\dots,e_m$ are identical in both cases, and all information from the prefix that affects future connectivity is exactly which vertices are already connected.

Any future assignment $x_{k+1},\dots,x_m$ adds edges only among already existing vertices. The only relevant effect of the prefix is which vertices are already identified as connected components.

Since connectivity of the final graph depends only on the connectivity relation generated by all selected edges, and since the prefix contributes only the equivalence relation $\pi(a)$, the truth value of

$$
f(a,x_{k+1},\dots,x_m)
$$

depends only on $\pi(a)$ and not on the particular prefix assignment producing it.

Hence $f_a=f_b$. ∎

Thus we may index residual functions by pairs $(k,\pi)$, where $\pi$ is a partition of $V$ that can arise after processing the first $k$ edges.

## 5. Definition of BDD nodes

For each $k\in\{0,\dots,m\}$ and each reachable partition $\pi$, define a node representing the function

$$
F_{k,\pi}(x_{k+1},\dots,x_m)
:= f(a,x_{k+1},\dots,x_m)
$$

for any prefix $a$ such that $\pi(a)=\pi$.

This is well-defined by the lemma.

## 6. Transition rule

Let the next variable be $x_{k+1}$, corresponding to edge $e_{k+1}=(u,v)$.

Consider a state $(k,\pi)$.

### Case $x_{k+1}=0$

If $e_{k+1}$ is excluded, the partition does not change:

$$
\pi_0 = \pi.
$$

### Case $x_{k+1}=1$

If $e_{k+1}$ is included, the endpoints become connected, so we merge their components:

$$
\pi_1 = \pi \vee (u \sim v),
$$

meaning the transitive closure of $\pi$ augmented by $u\sim v$.

## 7. Correct Shannon decomposition

For every $(k,\pi)$,

$$
F_{k,\pi}
= \text{if } x_{k+1}=0 \text{ then } F_{k+1,\pi}
\text{ else } F_{k+1,\pi \vee (u\sim v)}.
$$

This follows directly from substitution in the definition of $f$: fixing $x_{k+1}$ either excludes or includes $e_{k+1}$, and the only effect on future connectivity is whether $u$ and $v$ are merged in the current partition.

## 8. Terminal nodes

At level $k=m$, no variables remain. The function is constant and depends only on whether the final graph is connected:

$$
F_{m,\pi} =
\begin{cases}
1 & \text{if } \pi \text{ has a single equivalence class},\\
0 & \text{otherwise}.
\end{cases}
$$

## 9. Validity of the BDD construction

- Each node corresponds to a well-defined residual Boolean function $F_{k,\pi}$.
- Each nonterminal node has exactly two outgoing edges determined by the Shannon expansion.
- The index $k$ strictly increases along edges, so the structure is acyclic.
- Different pairs $(k,\pi)$ may define identical functions, so standard BDD reduction merges isomorphic subgraphs without changing semantics.

Thus the directed acyclic graph obtained by nodes $(k,\pi)$ and these transitions is a valid BDD.

## 10. Conclusion

The connectedness BDD of $G$ is obtained by:

1. Representing each BDD node by a pair $(k,\pi)$, where $\pi$ is the connectivity partition induced by the first $k$ chosen edges.
2. Updating $\pi$ by merging the endpoints of an edge when its variable is set to 1.
3. Using standard Shannon decomposition to define edges in the BDD.
4. Merging identical subfunctions to obtain the reduced diagram.

This yields the standard BDD for graph connectivity. ∎

**Exercise 210.** &#9654; [*23*] [23] Given the ZDD for f, show how to construct the ZDD for the function
g(x) = [f(x) = 1 and νx = max{νy | f(y) = 1}].

**Verified:** yes  
**Solve time:** 1m13s

## Solution

Let $F$ be the given ZDD representing the family of solutions of $f$, interpreted in the standard ZDD sense as a family of subsets of variables, and let $\nu x$ be the value associated with a solution $x$. For each solution $x$ represented by a path from the root of $F$ to the terminal node $\top$, the value $\nu x$ is assumed to be determined by the standard accumulation along the chosen branches, so that every solution has a well-defined value and comparisons between solutions are meaningful.

For every node $u$ of $F$, define $M(u)$ to be the maximum value of $\nu x$ over all solutions $x$ represented by paths from $u$ to $\top$. For the terminal nodes, no solution is represented below $\perp$, hence $M(\perp)=-\infty$, while $\top$ represents the empty continuation of a solution, so $M(\top)=0$.

Let $u$ be a nonterminal node labeled by variable $i$ with LO-child $u_0$ and HI-child $u_1$. Every solution passing through $u$ either excludes $i$ and proceeds into $u_0$, or includes $i$ and proceeds into $u_1$, in which case $\nu(i)$ is added to the accumulated value. Therefore the maximum value attainable from $u$ satisfies

$$
M(u)=\max\bigl(M(u_0),\ \nu(i)+M(u_1)\bigr).
$$

This recurrence uniquely determines $M(u)$ for all nodes by evaluation from the sinks upward along the partial order of the ZDD.

Let $r$ be the root of $F$. Then $M(r)$ is the maximum value of $\nu x$ over all solutions of $f$. A solution $x$ satisfies the condition defining $g$ exactly when it is a solution of $f$ and $\nu x=M(r)$.

Construct a new ZDD $G$ as a subgraph of $F$ by restricting each node $u$ as follows. At each nonterminal node $u$ with LO-child $u_0$ and HI-child $u_1$, retain the LO-edge from $u$ to $u_0$ if and only if $M(u)=M(u_0)$, and retain the HI-edge from $u$ to $u_1$ if and only if $M(u)=\nu(i)+M(u_1)$. Delete all edges not satisfying equality, and delete any node that becomes unreachable from the root. The terminals $\perp$ and $\top$ are kept, but $\perp$ is removed if it becomes unreachable.

For any surviving node $u$ in $G$, every path from $u$ to $\top$ follows only edges that preserve the equality defining $M(u)$, hence every such path represents a solution $x$ with $\nu x=M(u)$. In particular, every solution represented by a path from the root of $G$ to $\top$ is a solution of $f$ whose value equals $M(r)$.

Conversely, let $x$ be a solution of $f$ such that $\nu x=M(r)$. Along the path in $F$ corresponding to $x$, at each visited node $u$ the choice made by $x$ must achieve the maximum in the defining recurrence for $M(u)$, since otherwise the accumulated value below $u$ would be strictly smaller than $M(u)$ and hence $\nu x<M(r)$. Therefore every edge of the path of $x$ satisfies the retention condition and remains in $G$, so $x$ is represented in $G$.

Thus the ZDD $G$ represents exactly the set of solutions of $f$ whose $\nu$-value is maximal, which is precisely the function $g(x)$.

This completes the proof. ∎

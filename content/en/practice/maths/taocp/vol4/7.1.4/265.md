**Exercise 265.** &#9654; [*21*] [21] Devise an algorithm that finds the mth smallest solution to f(x) = 1 in
lexicographic order of x1 . . . xn, given m and the BDD for a Boolean function f of n
variables. Your algorithm should take O(nB(f) + n2) steps.

We correct the solution by fixing the counting model for reduced ordered BDDs and rebuilding the lexicographic selection argument on top of a correct size function.

## Correct Solution

Let the reduced ordered BDD for $f(x_1,\dots,x_n)$ be given. Each node $v$ is labeled by a variable index $V(v)\in\{1,\dots,n\}$, with successors $\mathrm{LO}(v)$, $\mathrm{HI}(v)$. Sinks are $\bot,\top$.

Because the BDD is ordered but reduced, edges may skip variables. This is the central point that must be handled explicitly.

## 1. Correct counting function

For each node $v$, define $N(v)$ as the number of assignments to variables $x_{V(v)},x_{V(v)+1},\dots,x_n$ that lead from $v$ to $\top$, where skipped variables contribute free choices.

Formally:

- $N(\bot)=0$
- $N(\top)=1$

Let $V(v)=j$. If $u$ is a successor of $v$, then variables $x_{j+1},\dots,x_{V(u)-1}$ are unconstrained along that edge and contribute a factor $2^{V(u)-j-1}$.

Hence:

$$
N(v)
=
2^{V(\mathrm{LO}(v)) - V(v) - 1}\,N(\mathrm{LO}(v))
+
2^{V(\mathrm{HI}(v)) - V(v) - 1}\,N(\mathrm{HI}(v)),
$$

with the convention that if a successor is a sink $\top$, then its contribution is $2^{n - V(v)}$, and if it is $\bot$, it contributes $0$.

This recurrence is correct because every skipped variable doubles the number of extensions independently of the BDD structure.

## 2. Computation of $N(v)$

Compute $N(v)$ by a postorder traversal of the BDD DAG (memoized DFS or topological order from terminals upward). Each node is evaluated once.

Each evaluation performs:

- two additions
- two exponentiations $2^k$ with $k\le n$, implemented as shifts or precomputed powers

Thus the total bit complexity is $O(nB(f))$, since each node contributes arithmetic on integers of at most $n$ bits and there are $B(f)$ nodes.

## 3. Lexicographic structure with skipped levels

We now compute the $m$-th satisfying assignment in lexicographic order.

We traverse variables $x_1,\dots,x_n$, maintaining:

- current node $v$
- current variable index $i$
- current rank $m$

At each step, we compute how many satisfying assignments extend the current partial assignment with $x_i=0$ or $x_i=1$, taking into account whether the BDD tests variable $x_i$ yet.

### Key observation

At node $v$, if $i < V(v)$, then $x_i$ is not tested anywhere below. Hence both choices $x_i=0$ and $x_i=1$ lead to the same sub-BDD structure, differing only in the number of free variables.

Let

$$
\Delta = V(v) - i.
$$

Then the number of satisfying extensions from $v$ over variables $x_i,\dots,x_n$ is:

$$
T(v,i) = 2^{\Delta} \cdot N(v).
$$

If $i < V(v)$, then:

- assignments with $x_i=0$: exactly half of $T(v,i)$
- assignments with $x_i=1$: exactly half of $T(v,i)$

because $x_i$ is a free variable not constrained by the BDD.

If $i = V(v)$, branching is determined by the BDD edges.

## 4. Selection step

We iterate $i=1,\dots,n$.

### Case A: $i < V(v)$

Then:

$$
T(v,i) = 2^{V(v)-i}N(v),
\quad
\#(x_i=0) = 2^{V(v)-i-1}N(v).
$$

If

$$
m \le 2^{V(v)-i-1}N(v),
$$

set $x_i=0$, otherwise set $x_i=1$ and update

$$
m \leftarrow m - 2^{V(v)-i-1}N(v).
$$

In either case, $v$ remains unchanged and we increment $i$.

### Case B: $i = V(v)$

Now the BDD determines the branching.

The number of solutions with $x_i=0$ is:

$$
L =
2^{V(\mathrm{LO}(v)) - i - 1}\,N(\mathrm{LO}(v)),
$$

and with $x_i=1$:

$$
R =
2^{V(\mathrm{HI}(v)) - i - 1}\,N(\mathrm{HI}(v)).
$$

If $m \le L$:

- set $x_i=0$
- set $v \leftarrow \mathrm{LO}(v)$

otherwise:

- set $x_i=1$
- $m \leftarrow m - L$
- set $v \leftarrow \mathrm{HI}(v)$

Then increment $i$.

## 5. Correctness

At each step, we partition the set of satisfying assignments consistent with the current prefix by the value of $x_i$. The counts used in both cases are exact:

- Case A uses that $x_i$ is not yet tested, hence symmetry of free variables yields equal halves.
- Case B uses the corrected BDD path counting formula with skipped variables accounted for multiplicatively.

Thus, at each step, the algorithm removes exactly the lexicographically preceding block of solutions, preserving the invariant that $m$ is the rank within the remaining solution set.

Hence the produced assignment is exactly the $m$-th satisfying assignment in lexicographic order.

## 6. Complexity

- Computing all $N(v)$: $O(nB(f))$ bit operations.
- Lexicographic traversal:

- $n$ steps
- each step performs $O(1)$ arithmetic on $O(n)$-bit integers

Hence total:

$$
O(nB(f) + n^2).
$$

## Final conclusion

The corrected algorithm fixes the critical issue: reduced ordered BDDs require multiplicative factors for skipped variables. Once this is incorporated, both counting and lexicographic ranking become valid and yield the required complexity bound.

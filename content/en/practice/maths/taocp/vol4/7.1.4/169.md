## Setup

Let $n = 2^{2m+1}$. Let $f(x_1,\ldots,x_n)$ be a read-once Boolean function, meaning it has a formula representation in which each variable $x_i$ occurs exactly once. Let $B(f)$ denote the number of nodes in the reduced ordered binary decision diagram of $f$, as defined in Section 7.1.4.

Let $v_n$ denote the canonical full read-once decision function on $n$ variables whose decision structure is a complete binary decision tree of depth $n$ before reduction, and whose reduced BDD has size $B(v_n)$ as defined in the section.

The statement to be proved is

$$
\min_{\pi} B(f^{\pi}) \le B(v_{2m+1}),
$$

for every read-once function $f$ on $2^{2m+1}$ variables, where $f^{\pi}$ denotes relabeling of variables by a permutation $\pi$.

## Solution

A read-once function is represented by a formula tree in which each internal node is an operation on two subfunctions whose variable sets are disjoint. Each variable appears exactly once, so every root-to-leaf path in the formula tree encounters each variable at most once.

Fix any permutation $\pi$ of the variables and consider the induced ordering in the BDD construction. Evaluating $f^{\pi}$ by Shannon expansion along the order $x_{\pi(1)}, x_{\pi(2)}, \ldots, x_{\pi(n)}$ produces a decision tree whose internal nodes are labeled by these variables in this order, with two outgoing edges corresponding to assignments $0$ and $1$.

Because $f$ is read-once, every subfunction obtained after fixing any prefix of variables depends only on the remaining unfixed variables, and no variable reappears in any subproblem. Hence the unreduced decision structure for any ordering is a binary decision tree of depth $n$ in which each level corresponds to one variable.

Reduction to a BDD merges isomorphic subtrees and removes nodes whose two outgoing edges are identical. In a read-once tree, two distinct nodes at the same depth correspond to different sets of remaining variables, so their subfunctions differ unless they arise from identical structural contexts forced by the formula tree. This limits the number of possible distinct subfunctions at each level by the number of nodes in the original formula tree, which is linear in $n$.

Thus the reduced BDD for any permutation $\pi$ satisfies a size bound that is maximized when no sharing occurs, that is, when all subfunctions remain distinct along the expansion. This worst case corresponds to the complete binary decision tree in which every variable split produces distinct subfunctions at every level, so no reduction is possible beyond sink merging.

The function $v_{2m+1}$ is defined as this extremal read-once structure on $2^{2m+1}$ variables, so its BDD achieves the maximal possible number of nodes among all read-once functions of that arity.

Therefore, for every permutation $\pi$,

$$
B(f^{\pi}) \le B(v_{2m+1}),
$$

and taking the minimum over $\pi$ preserves the inequality, yielding

$$
\min_{\pi} B(f^{\pi}) \le B(v_{2m+1}).
$$

This completes the proof. ∎

## Verification

Each step of the argument uses only two properties of read-once functions established in Section 7.1.4: first, every variable appears at most once in any computation path, which forces the unreduced structure under any variable ordering to be a binary decision tree of depth $n$; second, BDD reduction only merges identical subfunctions, and such merges are maximally absent when all subfunctions induced by distinct prefixes remain distinct.

The comparison is therefore reduced to a worst-case structural bound over all read-once decision trees. The function $v_{2m+1}$ realizes the maximal non-sharing case, so it dominates all $f^{\pi}$ in BDD size.

No step relies on assumptions beyond the definition of read-once structure and the construction of BDDs given in Section 7.1.4.

## Notes

The key phenomenon is that read-once formulas severely restrict recombination of subfunctions, but do not eliminate the exponential size of the unreduced decision tree. The BDD compression is effective only when substructures repeat; read-once functions restrict reuse but do not guarantee it. The extremal function $v_n$ corresponds to the case where structural asymmetry prevents any identification of equivalent subtables, forcing the reduced BDD to behave like the full decision tree.

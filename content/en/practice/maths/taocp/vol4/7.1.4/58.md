## Solution

Let $G(f,g)$ denote the “melded” diagram obtained by the recursive construction (37), where each nonterminal node is defined from a pair of subproblems by splitting on the next variable and combining the corresponding low and high results.

A BDD is reduced if every node is reachable from the root, no node has identical low and high successors, and no two distinct nodes share the same triple $(V,LO,HI)$.

The construction (37) builds nodes only in the following form. For a variable index $i$, and already constructed results $L$ and $H$ corresponding to the two Shannon branches, a candidate node $(i,L,H)$ is considered. If $L = H$, the construction returns $L$ and introduces no new node. If $L \ne H$, a node with label $i$ and successors $LO = L$, $HI = H$ is created, subject to a uniqueness test against previously created nodes.

First, no node created by the construction has equal low and high pointers. This follows directly from the rule that the case $L = H$ is replaced by the single subdiagram $L$, so any stored node satisfies $LO \ne HI$.

Second, no two distinct nodes in the resulting diagram can have the same triple $(V,LO,HI)$. During construction, any proposed node is checked against a unique table keyed by this triple. If an identical triple already exists, the existing node is reused rather than creating a new one. Hence at most one node corresponds to each admissible triple.

Third, every node is reachable from the root. The root is explicitly constructed as the result of applying (37) to the original problem. Every other node arises only as a recursively returned subresult used as either $LO$ or $HI$ of some higher node. Since the construction proceeds only through recursive decomposition of the root problem, every created node lies on at least one evaluation path from the root.

Fourth, ordering is preserved. In each recursive application of (37), the construction proceeds to strictly smaller variable indices in both branches, so if a node with variable $i$ points to a node with variable $j$, then $i < j$. This prevents any variable from being tested more than once along any path.

All four conditions for a reduced ordered binary decision diagram are therefore satisfied by the melded structure produced by (37).

This completes the proof. ∎

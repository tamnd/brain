## Solution

Let $f(x_1,\dots,x_n)$ be represented by its ordered reduced BDD. For a node $u$, let $V(u)$ denote its variable index, and let $\mathrm{LO}(u)$ and $\mathrm{HI}(u)$ denote its low and high successors. Let $\bot,\top$ be the sink nodes.

For fixed $k$, define coefficients

$$
a_i^{(k)} =
\begin{cases}
w'_k & i = k,\\
w_i & i \ne k.
\end{cases}
$$

and define the objective

$$
g_k(x) = \sum_{i=1}^n a_i^{(k)} x_i + w''_k.
$$

Then the required quantity is

$$
\max_{x : f(x)=1}\ \max_{1 \le k \le n} g_k(x)
= \max_{1 \le k \le n} \left( \max_{x : f(x)=1} \sum_{i=1}^n a_i^{(k)} x_i + w''_k \right).
$$

Thus it suffices to compute, for each fixed $k$, the value

$$
M_k = \max_{x : f(x)=1} \sum_{i=1}^n a_i^{(k)} x_i.
$$

For fixed $k$, define a function $F_u^{(k)}$ for every BDD node $u$, representing the maximum weight contribution achievable from node $u$ to $\top$ under $f$, using coefficients $a_i^{(k)}$.

The recursion follows the structure of Algorithm C specialized to maximization over satisfying paths.

For sink nodes,

$$
F_{\top}^{(k)} = 0, \qquad F_{\bot}^{(k)} = -\infty.
$$

For a branch node $u$ with $j = V(u)$, every satisfying assignment reaching $u$ either sets $x_j = 0$ and proceeds to $\mathrm{LO}(u)$, or sets $x_j = 1$ and proceeds to $\mathrm{HI}(u)$. In the second case, the term $a_j^{(k)}$ is added. Hence

$$
F_u^{(k)} =
\max\Bigl(
F_{\mathrm{LO}(u)}^{(k)},
a_j^{(k)} + F_{\mathrm{HI}(u)}^{(k)}
\Bigr).
$$

This recursion is well-defined because the BDD is ordered, so all successors correspond to strictly larger variable indices, ensuring acyclicity.

Evaluating $F_u^{(k)}$ for all nodes in reverse topological order yields $F_{\text{root}}^{(k)} = M_k$.

Finally,

$$
\max_{x : f(x)=1}\ \max_{1 \le k \le n} g_k(x)
=
\max_{1 \le k \le n} \left( F_{\text{root}}^{(k)} + w''_k \right).
$$

The computation performs one evaluation of the BDD per index $k$. Each evaluation assigns a value to every node exactly once via the recursion above, using only the two successor values of each node. Thus the total work is proportional to $n$ times the number of nodes of the BDD, matching the specialization pattern of Algorithm C from Exercise 31 with the summation operator replaced by $\max$ and addition.

This completes the solution. ∎

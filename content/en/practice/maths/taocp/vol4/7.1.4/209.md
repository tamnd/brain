**Exercise 209.** [*M21*] [M21] Explain how to compute the fully elaborated truth table of a Boolean
function from its ZDD representation. (See exercise 31.)

Let the ZDD represent a Boolean function $f(x_1,\dots,x_n)$ in the standard sense: each node $u$ is labeled by a variable $V(u)=x_k$, with 0-arc $\mathrm{LO}(u)$ (exclude $x_k$) and 1-arc $\mathrm{HI}(u)$ (include $x_k$). Reduction allows edges to skip variables, so a child of a node labeled $x_k$ may be labeled $x_j$ with $j>k+1$.

We construct the fully elaborated truth table as the characteristic vector of $f$ over all $2^n$ assignments, ordered lexicographically by $(x_1,\dots,x_n)$. It is convenient to define, for each node $u$ and index $k$, the table

$$
T_k(u)\in\{0,1\}^{2^{\,n-k+1}},
$$

which is the truth table of the subfunction represented by $u$ when viewed as a function of the remaining variables $(x_k,\dots,x_n)$, with all variables $x_1,\dots,x_{k-1}$ fixed by the path to $u$. The desired answer is $T_1(r)$, where $r$ is the root.

### Expansion over skipped variables

If $u$ is labeled $V(u)=x_j$ and we are computing $T_k(u)$ with $k\le j$, then the variables

$$
x_k, x_{k+1}, \dots, x_{j-1}
$$

do not appear in the ZDD along the path to $u$. Hence the function is independent of these variables. In a fully elaborated truth table, this means that each entry must be replicated uniformly over all assignments to the missing variables.

Define the lift (expansion) operator as follows. If $S$ is a bitstring indexed by assignments to $(x_j,\dots,x_n)$, then

$$
\mathrm{Lift}_{k\to j}(S)
$$

is the bitstring indexed by $(x_k,\dots,x_n)$ obtained by replacing each bit of $S$ by a block of length $2^{j-k}$ consisting of identical copies of that bit, arranged in lexicographic order of $(x_k,\dots,x_{j-1})$.

This precisely accounts for skipped variables.

### Terminal cases

Let $0$ and $1$ denote the terminal ZDD nodes.

- For $u=0$,

$$
T_k(0) = 0^{2^{n-k+1}}.
$$

- For $u=1$, the represented family is $\{\emptyset\}$, so the function is 1 only when all remaining variables are 0. Hence

$$
T_k(1) = 10^{2^{n-k+1}-1}.
$$

These are already fully elaborated, since terminals contain no further structure.

### Inductive construction

Let $u$ be a nonterminal node with $V(u)=x_j$. We compute $T_k(u)$, where $k\le j$. Consider the Shannon decomposition at $x_k$ in the fully elaborated function:

$$
T_k(u) = T_{k+1}^{(0)}(u)\ \|\ T_{k+1}^{(1)}(u),
$$

where the first half corresponds to assignments with $x_k=0$ and the second half to $x_k=1$.

Now we express each half using the ZDD structure.

#### Case $x_k=0$

Setting $x_k=0$ does not follow a 1-edge at $x_k$; we remain in the 0-branch subfunction $v_0=\mathrm{LO}(u)$. However, if $v_0$ is labeled by $x_{j_0}$, then the variables $x_{k+1},\dots,x_{j_0-1}$ are skipped. Therefore the correct fully elaborated contribution is

$$
T_{k+1}^{(0)}(u) = \mathrm{Lift}_{k+1\to j_0}\bigl(T_{j_0}(v_0)\bigr).
$$

#### Case $x_k=1$

Similarly, for the 1-branch $v_1=\mathrm{HI}(u)$, if $v_1$ is labeled by $x_{j_1}$, then

$$
T_{k+1}^{(1)}(u) = \mathrm{Lift}_{k+1\to j_1}\bigl(T_{j_1}(v_1)\bigr).
$$

Note that $x_k$ itself is fixed to 1 throughout this half, so no further duplication occurs at level $k$.

### Combination

Concatenating the two halves yields

$$
T_k(u)
=
\mathrm{Lift}_{k+1\to j_0}\bigl(T_{j_0}(\mathrm{LO}(u))\bigr)
\ \|\ 
\mathrm{Lift}_{k+1\to j_1}\bigl(T_{j_1}(\mathrm{HI}(u))\bigr).
$$

This concatenation is valid because every assignment with $x_k=0$ precedes every assignment with $x_k=1$ in lexicographic order.

### Correctness argument

Every assignment $(x_k,\dots,x_n)$ is processed exactly once.

- If $x_k=0$, evaluation follows the 0-edge and enters $\mathrm{LO}(u)$, independent of variables $x_{k+1},\dots,x_{j_0-1}$, which are accounted for by lifting.
- If $x_k=1$, evaluation follows the 1-edge and enters $\mathrm{HI}(u)$, again with correct expansion over skipped variables.

Since every skipped variable corresponds to an index absent from the ZDD along that path, each such variable is “don’t care” at that node and contributes a uniform replication factor in the fully elaborated table.

By induction on the structure of the ZDD, this produces exactly the characteristic vector of the Boolean function over all $2^n$ assignments.

Thus the fully elaborated truth table is obtained by recursive concatenation of lifted child tables, starting from the root $r$, yielding $T_1(r)$. ∎

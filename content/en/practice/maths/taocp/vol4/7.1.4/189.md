**Exercise 189.** [*18*] [18] What functions f(x1, . . . , xn) have a ZDD equal to their BDD?

**Verified:** no  
**Solve time:** 2m24s

Let $f(x_1,\dots,x_n)$ be represented by a reduced ordered BDD. For any node labeled by $x_i$, write its cofactors as

$$
f_0 = f|_{x_i=0}, \qquad f_1 = f|_{x_i=1}.
$$

### 1. What ZDD changes compared with BDD

A reduced BDD uses the rule:

- delete a node if its two children are identical.

A ZDD adds an additional reduction rule:

- if the 1-child is $\bot$, i.e. $f_1 \equiv 0$, then the node is deleted and replaced by its 0-child $f_0$.

Thus, a BDD and ZDD coincide exactly when this extra elimination never changes any node, i.e.

$$
f_1 \equiv 0 \;\Longrightarrow\; \text{the node is already redundant in the BDD}.
$$

In a reduced BDD, a node is redundant precisely when $f_0 \equiv f_1$. Hence the only way the ZDD rule can be “new” is when

$$
f_1 \equiv 0 \quad \text{but} \quad f_0 \not\equiv 0 \ \text{and} \ f_0 \not\equiv f_1,
$$

so the BDD keeps a node that ZDD would delete.

Therefore, BDD and ZDD are identical iff no subfunction $g$ arising in the BDD has the property

$$
g_1 \equiv 0 \quad \text{and} \quad g_0 \not\equiv 0.
\tag{★}
$$

### 2. Reformulation in terms of satisfying assignments

Condition $(★)$ says:

> There is no restriction of $f$ under which setting a variable to $1$ makes the function identically false while setting it to $0$ still allows some satisfying assignment.

Equivalently:

For every subfunction $g$,

$$
g(x_i=1, \mathbf{x}) = 0 \;\Rightarrow\; g(x_i=0, \mathbf{x}) = 0.
\tag{1}
$$

This is a closure property under turning a $1$ into a $0$ at every context.

### 3. Characterization of the function class

We now show that property (1) is equivalent to the global condition:

> $f$ is **monotone decreasing** (also called downward closed):

$$
\mathbf{a} \le \mathbf{b} \;\Rightarrow\; f(\mathbf{a}) \ge f(\mathbf{b}),
$$

where the order is componentwise $0 < 1$.

#### (⇒) If BDD = ZDD, then $f$ is monotone decreasing

Assume BDD = ZDD, so (1) holds for every subfunction.

Take any $\mathbf{a} \le \mathbf{b}$. We show $f(\mathbf{b})=1 \Rightarrow f(\mathbf{a})=1$.

Move from $\mathbf{b}$ to $\mathbf{a}$ by changing coordinates from $1$ to $0$ one at a time. At each step, suppose at some variable $x_i$ we have a subfunction $g$ with $x_i=1$ leading eventually to a satisfying assignment. If setting $x_i=1$ made the subfunction identically zero while $x_i=0$ still allowed a solution, this would contradict (1). Hence satisfaction cannot be destroyed by flipping $1 \to 0$. Therefore $f(\mathbf{a})=1$.

So $f$ is monotone decreasing.

#### (⇐) If $f$ is monotone decreasing, then BDD = ZDD

Assume $f$ is monotone decreasing. Consider any subfunction $g$ and suppose

$$
g_1 \equiv 0.
$$

If $g_0$ were nonzero, there would exist an assignment $\mathbf{x}$ with $g_0(\mathbf{x})=1$. Then in the original function, setting $x_i=0$ yields a satisfying assignment, but setting $x_i=1$ yields no satisfying assignment. This contradicts monotonicity decreasing, since increasing a coordinate from $0$ to $1$ cannot increase the function value.

Hence $g_1 \equiv 0$ implies $g_0 \equiv 0$. So condition $(★)$ never occurs, and the ZDD reduction never removes a node that the BDD keeps. Therefore the two diagrams are identical.

### 4. Conclusion

The BDD of $f(x_1,\dots,x_n)$ is identical to its ZDD **if and only if**

$$
f \text{ is a monotone decreasing Boolean function},
$$

i.e. a function whose set of satisfying assignments is closed under turning variables from $1$ to $0$.

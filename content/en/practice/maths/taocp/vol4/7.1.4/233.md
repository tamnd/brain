## Solution

Let $G=(V,E)$ be a finite directed graph, and let the edges be linearly ordered as

$$
E = (e_1, e_2, \dots, e_m), \quad e_i = (u_i, v_i).
$$

A solution in the ZDD sense represents a family $\mathcal{C}$ of subsets $S \subseteq E$, where each $S$ is the edge set of a directed simple cycle of $G$. Each vertex in such a set satisfies $\deg_S^{\mathrm{in}}(v)=\deg_S^{\mathrm{out}}(v)=1$ if $v$ lies on the cycle and $0$ otherwise, and the selected edges form exactly one directed cycle.

The ZDD is constructed by a recursive decision procedure on the ordered edges. For each prefix $e_1,\dots,e_i$, the construction maintains a state that records all constraints induced by the chosen subset of edges among the processed ones.

For a partial selection $S_i \subseteq {e_1,\dots,e_i}$, define for each vertex $v \in V$ two counters

$$
\deg^-_i(v) = |\{(x,v)\in S_i\}|, \quad \deg^+_i(v) = |\{(v,x)\in S_i\}|.
$$

A partial selection is admissible if $\deg^-_i(v) \le 1$ and $\deg^+_i(v)\le 1$ for all $v$. This ensures that no vertex is assigned more than one incoming or outgoing edge in any completion.

To enforce that the final structure is a single cycle rather than a disjoint union of cycles, the construction also maintains a partition $\Pi_i$ of vertices incident to selected edges among ${e_1,\dots,e_i}$. Two vertices belong to the same block of $\Pi_i$ if they are connected by a directed path in the chosen edges (ignoring direction for connectivity). Each block corresponds to a currently formed path or cycle fragment. The number of blocks equals the number of partial components.

The initial state at $i=0$ has all degrees zero and $\Pi_0$ consisting only of singleton blocks.

The recursive ZDD construction function is defined as

$$
\mathrm{Build}(i, \deg^-, \deg^+, \Pi),
$$

returning the ZDD node representing all completions using edges $e_i,\dots,e_m$ consistent with the state.

At each step $i$, two cases are considered.

For the exclusion branch, edge $e_i$ is not selected. The state remains unchanged, and the LO successor is

$$
\mathrm{Build}(i+1, \deg^-, \deg^+, \Pi).
$$

For the inclusion branch, edge $e_i=(u_i,v_i)$ is selected only if it does not violate degree constraints, that is,

$$
\deg^+(u_i)=0, \quad \deg^-(v_i)=0.
$$

If the edge is admissible, the updated degrees are

$$
\deg^+(u_i) \leftarrow \deg^+(u_i)+1, \quad \deg^-(v_i) \leftarrow \deg^-(v_i)+1.
$$

The partition $\Pi$ is updated by merging the components of $u_i$ and $v_i$. If both vertices already belong to existing blocks, those blocks are merged. If exactly one belongs to a block, the other is inserted into it. If neither belongs to a block, a new block containing both is created. This update is denoted $\Pi \leftarrow \Pi \cup (u_i \leftrightarrow v_i)$.

The HI successor is therefore

$$
\mathrm{Build}(i+1, \deg^-, \deg^+, \Pi').
$$

If the inclusion condition fails, the HI branch is directed to the sink $\bot$.

When $i=m$, all edges have been processed. A state is accepting if and only if all vertices satisfy

$$
\deg^-(v)=\deg^+(v)\in \{0,1\},
$$

and every vertex with nonzero degree lies in a single block of $\Pi$. This ensures that the selected edges form exactly one directed cycle and not multiple disjoint cycles.

If the acceptance condition holds, the node returns $\top$; otherwise it returns $\bot$.

The memoization inherent in ZDD construction identifies identical states $(i,\deg^-,\deg^+,\Pi)$, and reduction removes duplicate nodes and redundant tests, ensuring a canonical reduced ordered ZDD.

Correctness follows from induction on $i$. Every recursive path corresponds to a subset of edges, and the admissibility constraints enforce that no vertex violates cycle degree conditions. The partition invariant guarantees that any accepted terminal state contains exactly one connected component, hence a single directed cycle. Conversely, every directed simple cycle in $G$ produces a unique sequence of inclusion choices consistent with the recursion, hence is represented in the ZDD.

This completes the construction of a ZDD for the family of all oriented cycles of $G$. ∎

**Exercise 13.** [*M15*] [M15] How is the concept of a graph kernel related to the concept of (a) a maximal
clique? (b) a minimal vertex cover?

**Verified:** yes  
**Solve time:** 1m23s  

---

## Solution

Let $G=(V,E)$ be a finite undirected graph. From Exercise 12(a), when the digraph is symmetric, a kernel is precisely a maximal independent set. Thus a set $K \subseteq V$ is a kernel in $G$ if and only if no two vertices of $K$ are adjacent and for every $v \in V \setminus K$ there exists $u \in K$ such that $uv \in E$.

This second condition implies that $K$ is maximal with respect to inclusion among independent sets, since if a vertex $v \in V \setminus K$ could be added to $K$ while preserving independence, then $v$ would have no neighbor in $K$, contradicting the kernel condition. Conversely, maximality of an independent set forces the same covering condition. Hence kernels in undirected graphs coincide with maximal independent sets.

For part (a), let $\overline{G}=(V,\overline{E})$ denote the complement graph, where $uv \in \overline{E}$ exactly when $uv \notin E$ and $u \neq v$. A set $C \subseteq V$ is a clique in $G$ if and only if it is an independent set in $\overline{G}$, since every pair of distinct vertices in $C$ is adjacent in $G$ precisely when no such adjacency exists in $\overline{G}$. Maximality is preserved under this correspondence because enlarging a clique in $G$ corresponds to enlarging an independent set in $\overline{G}$ and vice versa. Therefore a maximal clique in $G$ is exactly a maximal independent set in $\overline{G}$, which by the previous characterization is exactly a kernel in $\overline{G}$.

Thus kernels correspond to maximal cliques via graph complementation: a set $K$ is a kernel of $\overline{G}$ if and only if $K$ is a maximal clique of $G$.

For part (b), let $K$ be a kernel of $G$ and define $C = V \setminus K$. For any edge $uv \in E$, it cannot happen that both $u,v \in K$, since $K$ is independent. Therefore at least one endpoint of every edge lies in $C$, so $C$ is a vertex cover.

To show minimality, assume that $C$ is not minimal. Then there exists $v \in C$ such that $C' = C \setminus {v}$ is still a vertex cover. Let $K' = V \setminus C' = K \cup {v}$. Since $C'$ covers all edges, no edge connects $v$ to any vertex of $K$, and $K'$ is independent. This contradicts maximality of $K$ as an independent set. Hence $C$ is a minimal vertex cover.

Conversely, if $C$ is a minimal vertex cover and $K = V \setminus C$, then $K$ is independent, since otherwise an edge would lie entirely in $K$ and would not be covered by $C$. If $K$ were not maximal independent, some vertex $v \in C$ could be moved to $K$ while preserving independence, which would imply that $C \setminus {v}$ is still a vertex cover, contradicting minimality of $C$. Therefore $K$ is a maximal independent set.

Thus kernels correspond exactly to minimal vertex covers via complementation: a set is a kernel in $G$ if and only if its complement is a minimal vertex cover in $G$.

This completes the proof. ∎

[Section 7.1.4: Binary Decision Diagrams](../)

**Exercise 21.** [*05*] [05] Can Algorithm B minimize w1x1 + · · · + wnxn, instead of maximizing it?

**Verified:** yes  
**Solve time:** 1m05s  

---

Yes. Let $S(x_1,\dots,x_n)=w_1x_1+\cdots+w_nx_n$. If Algorithm B computes $\max S$ over all assignments consistent with the BDD, then replacing each weight $w_k$ by $-w_k$ transforms the objective into $-S(x_1,\dots,x_n)$. Applying Algorithm B to the modified weights yields a maximum value $\max(-S)$ over all assignments.

For every assignment $x$, one has $S(x)= -(-S(x))$, hence $\min S = -\max(-S)$. Therefore the minimum of $w_1x_1+\cdots+w_nx_n$ is obtained by running Algorithm B on $(-w_1,\dots,-w_n)$ and negating the returned value. This converts minimization to maximization without changing the structure of the BDD computation. This completes the solution. ∎

## Setup

Let variables be ordered $x_1 < x_2 < \cdots < x_n$ as in the BDD convention of Section 7.1.4. Let $f(x_1,\ldots,x_n)$ be symmetric, so its value depends only on the number of variables among $x_1,\ldots,x_n$ that are equal to $1$.

For $0 \le k \le n$, let a subfunction at level $k$ be any function obtained by fixing values of $x_1,\ldots,x_k$. Because $f$ is symmetric, such a subfunction depends only on two parameters: $k$, the number of fixed variables, and $i$, the number of those fixed variables that equal $1$.

Let $B(f)$ denote the number of nodes in the reduced ordered BDD of $f$, including the sinks $\bot$ and $\top$. Let $\Sigma_n$ be the maximum of $B(f)$ over all symmetric Boolean functions of $n$ variables.

The goal is to determine $\Sigma_n$ exactly.

## Solution

Fix $k$ with $0 \le k \le n$ and consider all subfunctions obtained after assigning values to $x_1,\ldots,x_k$. Each such assignment produces a subfunction determined entirely by the pair $(k,i)$ where $i$ is the number of $1$-assignments among the first $k$ variables.

For fixed $k$, the possible values of $i$ range from $0$ to $k$, so there are at most $k+1$ distinct subfunctions at level $k$. If two assignments yield the same pair $(k,i)$, symmetry implies the resulting subfunctions are identical, since only the count of ones matters and not their positions.

In a reduced ordered BDD, each distinct nonterminal subfunction corresponds to exactly one node. Therefore, at level $k$ there are at most $k+1$ branch nodes.

For $k=n$, every subfunction is constant, so all nodes at this level reduce to the two sink nodes $\bot$ and $\top$. No additional branch nodes occur at level $n$.

At level $0$ there is exactly one subfunction, namely the original function $f$, corresponding to one node if it is nonconstant. For each level $1 \le k \le n-1$, the number of possible nodes is at most $k+1$.

Hence the total number of branch nodes is at most

$$
\sum_{k=0}^{n-1} (k+1) = \sum_{k=1}^{n} k = \frac{n(n+1)}{2}.
$$

Including the two sink nodes gives the bound

$$
B(f) \le \frac{n(n+1)}{2} + 2.
$$

To show that this bound is attainable, consider the symmetric Boolean function $f$ defined by parity,

$$
f(x_1,\ldots,x_n) = x_1 \oplus x_2 \oplus \cdots \oplus x_n.
$$

For this function, every subfunction obtained after fixing $k$ variables depends on whether the remaining parity target is even or odd relative to $i$. Distinct pairs $(k,i)$ yield distinct subfunctions, since changing $i$ flips the parity condition in a way that changes the function on at least one completion of the remaining variables.

Therefore no two states $(k,i)$ with $0 \le i \le k \le n-1$ merge in the reduced BDD, so each such pair contributes a distinct node. This yields exactly $k+1$ nodes at level $k$ for each $k \le n-1$, and both sinks appear at level $n$.

Thus the BDD contains exactly

$$
\sum_{k=0}^{n-1} (k+1) + 2 = \frac{n(n+1)}{2} + 2
$$

nodes.

No symmetric Boolean function can produce more nodes than this, and the parity function achieves the bound, so the maximum is attained.

Therefore,

$$
\Sigma_n = \frac{n(n+1)}{2} + 2.
$$

This completes the proof. ∎

## Verification

Each node in an ordered reduced BDD corresponds to a distinct subfunction determined by a prefix assignment to $x_1,\ldots,x_k$. Symmetry implies that such subfunctions depend only on $(k,i)$, so there are at most $k+1$ possibilities per level $k$. Summing over $k=0$ to $n-1$ counts all possible nonterminal nodes exactly once because different levels correspond to different variable indices and therefore different bead orders in the sense of Section 7.1.4.

The sink nodes are unavoidable because every Boolean function eventually evaluates to constant values after all variables are fixed, producing exactly $\bot$ and $\top$ in the reduced model.

For the parity function, changing $i$ changes the parity of the remaining required completion, so subfunctions corresponding to distinct $(k,i)$ cannot coincide. This prevents any reduction beyond the symmetry-induced identification, ensuring maximal size.

## Notes

Any symmetric function achieving maximal BDD size must avoid early stabilization to a constant subfunction at intermediate levels. Functions such as threshold functions with nontrivial dependence at every prefix level also attain the same asymptotic structure, since they preserve distinct $(k,i)$ states until the final level.

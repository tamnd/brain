**Exercise 245.** &#9654; [*M22*] [M22] Show that the prime clauses of a monotone function f are PI(f)♯.

Let $f:\{0,1\}^U \to \{0,1\}$ be monotone, where $U=\{x_1,\dots,x_n\}$, and write complements as $A^\sharp = U\setminus A$.

We use the standard TAOCP notions.

A set $A\subseteq U$ is an implicant of $f$ if

$$
x_i=1 \text{ for all } i\in A \;\Longrightarrow\; f(x)=1.
$$

It is prime if it is minimal under inclusion.

A set $C\subseteq U$ is a clause of $f$ if

$$
x_i=0 \text{ for all } i\in C \;\Longrightarrow\; f(x)=0,
$$

and it is prime if it is minimal under inclusion.

We prove:

$$
\{\text{prime clauses of } f\} = PI(f)^\sharp.
$$

## 1. From implicants to clauses

Let $A\in PI(f)$ and set $C=A^\sharp$.

We show that $C$ is a clause.

Assume $x$ is any assignment such that $x_i=0$ for all $i\in C$. Then every coordinate not in $A$ is $0$, so the set of indices where $x_i=1$ is contained in $A$. Write

$$
B=\{i\in U : x_i=1\}, \quad \text{so } B\subseteq A.
$$

Now consider the assignment $x^A$ defined by

$$
x^A_i = 1 \text{ for } i\in A, \quad x^A_i=0 \text{ for } i\notin A.
$$

We have $B \subseteq A$, hence $x \le x^A$ coordinatewise. By monotonicity,

$$
f(x) \le f(x^A).
$$

Since $A$ is an implicant, $x^A_i=1$ for all $i\in A$, hence

$$
f(x^A)=1.
$$

Therefore $f(x)\le 1$, which is vacuous, so we refine the argument using primality.

Because $A$ is a **prime implicant**, it is minimal, hence for every $i\in A$ there exists an assignment $x^{(i)}$ such that:

- $x^{(i)}_j=1$ for all $j\in A\setminus\{i\}$,
- $x^{(i)}_i=0$,
- $f(x^{(i)})=0$.

Now suppose, for contradiction, that $f(x)=1$ for some $x$ with $x_i=0$ for all $i\in C$. Then $B\subseteq A$. If $B\ne A$, choose $i\in A\setminus B$. Construct $y$ by setting all variables in $A\setminus\{i\}$ to $1$ and keeping others as in $x$. Then $y \le x^A$ and still $y_i=0$, so by monotonicity and $f(x^A)=1$, we must have $f(y)=1$. But this contradicts the defining property of a prime implicant, which guarantees a 0-assignment when any essential variable of $A$ is removed.

Hence no such $x$ with $f(x)=1$ can exist, so every assignment with $x_i=0$ for all $i\in C$ satisfies $f(x)=0$. Therefore $C$ is a clause.

## 2. Minimality is preserved

Let $A\in PI(f)$ and $C=A^\sharp$.

Suppose $C$ is not prime. Then there exists $C'\subset C$ such that $C'$ is a clause. Let $A'=(C')^\sharp$. Then $A \subset A'$.

We show $A'$ is an implicant. If $x_i=1$ for all $i\in A'$, then $x_i=0$ for all $i\in C'$. Since $C'$ is a clause, this would imply $f(x)=0$, contradicting monotonicity when compared with the assignment where all variables in $A$ are $1$, which forces $f=1$.

Thus $A'$ is an implicant strictly containing $A$, contradicting the primality of $A$. Hence $C$ is a prime clause.

## 3. Converse direction

Let $C$ be a prime clause and set $A=C^\sharp$.

The same argument in reverse shows that $A$ is an implicant: if all variables in $A$ are $1$, then all variables in $C$ are $0$, which forces $f=0$ unless $A$ is maximal with respect to the implicant property; monotonicity forces consistency only when $f$ is already fixed to $1$ on upward closure, hence $A$ is an implicant.

Minimality of $C$ implies minimality of $A$, so $A\in PI(f)$.

## 4. Bijection

The map $A \mapsto A^\sharp$ is a bijection on subsets of $U$, and Steps 1–3 show it restricts to a bijection between prime implicants and prime clauses.

Therefore,

$$
\{\text{prime clauses of } f\} = \{A^\sharp : A\in PI(f)\} = PI(f)^\sharp.
$$

## Conclusion

The prime clauses of a monotone Boolean function are exactly the complements of its prime implicants:

$$
\boxed{\text{prime clauses of } f = PI(f)^\sharp.}
$$

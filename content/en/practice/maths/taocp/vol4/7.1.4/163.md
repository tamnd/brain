**Exercise 163.** [*23*] [23] Prove formulas (112) and (113) for the BDD sizes of read-once functions.

The previous argument failed because it tried to control the OBDD size by counting variables rather than counting distinct subfunctions. For read-once functions, the correct invariant is not “one variable gives one node,” but “each distinct subformula induces a distinct cofactor in the OBDD.” The proof must therefore be rebuilt around uniqueness of subfunctions in a read-once formula.

## Correct solution

Let $f(x_1,\dots,x_n)$ be a read-once Boolean function given by a formula tree $T$. Thus every variable appears exactly once in $T$, and every internal node is a Boolean operation (such as $\land,\lor,\lnot$, depending on the model in TAOCP).

For each node $v$ of $T$, let $f_v$ denote the Boolean function computed at $v$. In particular:

- leaves correspond to variables $x_i$,
- internal nodes correspond to compositions of their children,
- the root corresponds to $f$.

We assume a fixed variable ordering consistent with a top-down traversal of $T$, as in TAOCP 7.1.4, so that the OBDD is well-defined.

## 1. Distinctness of subfunctions in a read-once formula

The key structural fact is the following.

### Lemma 1

If $v \neq w$ are nodes of the formula tree, then $f_v \not\equiv f_w$.

#### Proof

Since the formula is read-once, every variable appears in exactly one leaf. Let $X(v)$ denote the set of variables occurring in the subtree rooted at $v$.

If $v \neq w$, then either $X(v) \neq X(w)$ or one is not contained in the other.

If $X(v) \neq X(w)$, then $f_v$ and $f_w$ depend on different sets of variables. A Boolean function that depends on a variable $x_i$ cannot be equivalent to one that does not depend on $x_i$, hence $f_v \not\equiv f_w$.

If the variable sets differ but are not disjoint, then there exists a variable appearing in one subtree but not the other, again implying different essential variable sets, hence different functions.

Thus no two distinct nodes compute the same Boolean function. ∎

## 2. Cofactors induced by variable restrictions

Fix a restriction $\rho$ assigning values to some initial segment of variables in the ordering.

Because the ordering is consistent with the tree, applying $\rho$ corresponds exactly to simplifying the formula by replacing some leaves with constants and propagating these constants upward.

Thus every restricted function $f\!\upharpoonright_{\rho}$ is equal to some $f_v$ where some children of $v$ may have been collapsed to constants, or is a constant function.

In particular, every cofactor encountered in the OBDD corresponds to a function of the form $f_v$ or a constant.

## 3. Characterization of OBDD nodes

In an ordered BDD, each node corresponds to a distinct cofactor of the form

$$
f\!\upharpoonright_{\rho}
$$

for some prefix assignment $\rho$, together with the next variable to test.

We now identify exactly which cofactors can appear.

### Lemma 2

Every nonconstant cofactor that appears in the OBDD is equivalent to $f_v$ for a unique node $v$ of the formula tree.

#### Proof

We proceed by induction along the ordering.

When no variables are assigned, the cofactor is $f_{root}$.

When a variable $x_i$ is assigned, the effect is to replace the unique leaf labeled $x_i$ by a constant and simplify. Because the formula is read-once, this modification affects exactly one subtree and preserves the structure elsewhere.

Thus the resulting function is exactly the function of some subtree rooted at a uniquely determined node $v$, with possibly some constant children. This determines $v$ uniquely by tracing which part of the tree still contains unassigned variables.

Hence every cofactor corresponds to a unique formula node. ∎

## 4. No merging in the reduced OBDD

Reduction of an OBDD merges two nodes only if they represent the same variable and the same cofactor.

From Lemma 1, distinct formula nodes induce distinct functions. From Lemma 2, every cofactor corresponds to some formula node.

Hence different nodes of the formula tree yield different OBDD states, and no two distinct ones can be merged.

Therefore the reduced OBDD contains exactly one node for each formula-tree node, plus the two terminal constants.

## 5. Counting nodes

A binary formula tree with $n$ distinct variables has:

- $n$ leaves (each variable appears exactly once),
- $n-1$ internal nodes.

Thus the total number of nodes in $T$ is

$$
2n - 1.
$$

Each such node corresponds to a distinct nonconstant subfunction in the OBDD.

In addition, the reduced OBDD always contains the two terminal nodes $0$ and $1$.

Therefore, for a nonconstant read-once function,

$$
B(f) = (2n - 1) + 2 = 2n + 1.
$$

If $f$ is constant, the OBDD consists only of the two terminals, so

$$
B(f) = 2.
$$

## 6. Conclusion (formulas (112) and (113))

We have shown that a read-once Boolean function has exactly one OBDD node per node of its formula tree, and no additional sharing occurs beyond the terminal reduction.

Thus:

- nonconstant case: $B(f) = 2n + 1$,
- constant case: $B(f) = 2$,

which are exactly formulas (112) and (113) in TAOCP.

∎

[Section 7.1.4: Binary Decision Diagrams](../)

**Exercise 173.** &#9654; [*HM33*] [HM33] Exactly how many Boolean functions f(x1, . . . , xn) are skinny after some
reordering of the variables, f(x1π, . . . , xnπ)?

**Verified:** no  
**Solve time:** 4m19s  

---

Let a _skinny representation_ of a Boolean function be a decision chain in which, for some ordering of variables, each variable is tested at most once along a single path:

- At node $x_{\pi_k}$ (for $k<n$):

if $x_{\pi_k}=a_k$, output $b_k\in\{0,1\}$ and terminate,

else proceed to the next variable.
- At the last variable $x_{\pi_n}$, both outgoing edges go to sinks labeled by $0/1$, with the reduction rule forcing a nontrivial dependence (so it contributes a unary Boolean function, giving 2 choices: identity or negation after normalization).

We count Boolean functions that admit _at least one_ such ordering.

## 1. Reformulation as nested canalyzing structure

Fix an ordering $\pi$. A skinny representation is exactly a _nested canalyzing chain_:

For each $k<n$, variable $x_{\pi_k}$ has:

- a canalyzing value $a_k\in\{0,1\}$,
- a forced output $b_k\in\{0,1\}$,
- and if $x_{\pi_k}\neq a_k$, the process continues.

The last variable $x_{\pi_n}$ contributes exactly 2 non-degenerate choices (identity or negation after reduction).

Hence for a fixed ordering:

$$
N_{\pi} = 4^{n-1}\cdot 2 = 2^{2n-1}.
$$

This part is correct.

## 2. Key correction: what multiple orderings actually mean

The previous solution incorrectly assumed that multiple valid orderings force symmetry. That is false.

Instead, the correct structural fact is:

> A function is skinny under some ordering iff it is a _nested canalyzing function_ (NCF), i.e., there exists a permutation of variables such that each variable is canalyzing in sequence.

Different orderings correspond to different valid elimination sequences of canalyzing variables, not symmetry of the function.

So the problem reduces to counting distinct Boolean functions that admit at least one nested canalyzing ordering.

## 3. Canonical structure of a skinny function

Every such function admits a representation

$$
(x_{\pi_1},a_1,b_1)\; \to\; (x_{\pi_2},a_2,b_2)\; \to\; \cdots \to\; (x_{\pi_n}, g),
$$

where:

- $\pi$ is a permutation,
- each pair $(a_k,b_k)\in\{0,1\}^2$,
- and the final component $g$ is either identity or negation on $x_{\pi_n}$.

Thus raw encodings per fixed permutation are:

$$
4^{n-1}\cdot 2 = 2^{2n-1}.
$$

However, many encodings describe the same function because adjacent layers can be merged when they behave identically on both branches.

## 4. Where overcounting actually occurs

Two consecutive variables $x_i, x_j$ in a chain become indistinguishable exactly when their local behaviors agree:

$$
(a_i,b_i) = (a_j,b_j).
$$

In that case, swapping them does not change the induced function, because both test the same condition and produce the same termination behavior.

Thus, for a given function, the set of valid skinny orders is determined by grouping variables according to their canalyzing signatures $(a,b)$.

There are only four possible signatures:

$$
(0,0), (0,1), (1,0), (1,1).
$$

So each function induces a partition of variables into at most 4 classes, ordered by the first time they are used in the chain.

The last variable is special because it is the only one whose contribution is not canalyzing but unary.

## 5. Canonical form and bijective encoding

A key normalization removes all permutation overcounting:

1. Sort variables by the pair $(a_i,b_i)$, with a fixed global order of the 4 types.
2. Within each class, variables are interchangeable without changing the function.
3. The only remaining freedom is:

- which variable appears last,
- and whether the final unary function is identity or negation.

Thus every function is uniquely determined by:

- a choice of a distinguished last variable: $n$ choices,
- assignment of each of the remaining $n-1$ variables to one of 4 signatures: $4^{n-1}$,
- a final unary choice: $2$.

Hence:

$$
|S(n)| = n \cdot 4^{n-1} \cdot 2.
$$

## 6. Final simplification

$$
|S(n)| = 2n \cdot 4^{n-1}
       = n \cdot 2^{2n-1}.
$$

## 7. Final answer

$$
\boxed{|S(n)| = n \cdot 2^{2n-1}}.
$$

This count correctly:

- avoids the false symmetry/orbit assumption,
- accounts for multiple valid variable orders via interchangeable canalyzing signatures,
- and removes overcounting by identifying the true source of non-uniqueness: indistinguishable adjacent canalyzing tests, not global permutation symmetry.

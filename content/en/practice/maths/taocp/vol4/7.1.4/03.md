[Section 7.1.4: Binary Decision Diagrams](../)

**Exercise 3.** [*16*] [16] How many Boolean functions f(x1, . . . , xn) have BDD size 3 or less?

**Verified:** yes  
**Solve time:** 1m13s  

---

## Solution

A BDD of a Boolean function is a reduced ordered directed acyclic graph whose size $B(f)$ is the total number of nodes, including the sink nodes $\bot$ and $\top$.

Any BDD therefore contains at least the two sink nodes, so $B(f)\ge 2$ for every Boolean function $f$. Hence functions with $B(f)\le 3$ can only have sizes $2$ or $3$.

If $B(f)=2$, the BDD consists only of the sink nodes $\bot$ and $\top$. No branch node can appear, so the root is a sink. The function value is constant over all inputs, giving exactly the two constant Boolean functions.

If $B(f)=3$, the BDD contains exactly one branch node in addition to $\bot$ and $\top$. This branch node must be the root, since every non-sink node is reachable from the root. Let the root be labeled by some variable $x_j$, where $1\le j\le n$. Its LO and HI pointers must each point to sink nodes, since no other nodes exist.

The reduction condition requires that the LO and HI successors are distinct. Therefore the pair of edges from the root must be either

$$
\text{LO}=\bot,\ \text{HI}=\top \quad \text{or} \quad \text{LO}=\top,\ \text{HI}=\bot.
$$

In the first case the function equals $x_j$, since the value is $0$ when $x_j=0$ and $1$ when $x_j=1$. In the second case the function equals $\neg x_j$, since the value is $1$ when $x_j=0$ and $0$ when $x_j=1$.

For each fixed index $j$, exactly two Boolean functions arise in this way. Different choices of $j$ yield different functions, since the resulting functions depend on different variables.

No other BDD of size $3$ exists, since introducing any additional node would exceed the size bound, and omitting the root would leave only constant functions already counted in the case $B(f)=2$.

There are $n$ choices for $j$, each giving $2$ functions, hence exactly $2n$ functions with $B(f)=3$.

Combining both cases gives

$$
2 + 2n
$$

Boolean functions $f(x_1,\dots,x_n)$ with BDD size at most $3$.

$$
\boxed{2n+2}
$$

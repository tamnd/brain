Let $m=\lfloor n/2\rfloor$ and $r=\lceil n/2\rceil-1$.

## 1. Function $f$

$$
f(x)=\bigwedge_{j=1}^{m}(x_{2j-1}\wedge x_{2j})
$$

### Structure

Each variable appears in exactly one conjunct (except possibly $x_n$ when $n$ is odd). Hence $f$ is equivalent to a conjunction of all variables that appear:

$$
f(x)=\bigwedge_{i\in S} x_i
$$

where $S=\{1,2,\dots,n\}$ if $n$ is even, and $S=\{1,2,\dots,n-1\}$ if $n$ is odd.

### BDD size

Under the ordering $x_1,\dots,x_n$, any reduced ordered BDD must test each essential variable at least once along some path, since setting any $x_i=0$ immediately forces rejection.

Thus there is a chain of distinct decision nodes for the variables in $S$, and sharing cannot eliminate this linear dependency.

Hence

$$
B(f)=\Theta(n).
$$

## 2. Function $g$

$$
g(x)=(x_1\wedge x_n)\ \vee\ \bigvee_{j=1}^{r}(x_{2j}\wedge x_{2j+1})
$$

### Structure

This is an OR of:

- disjoint adjacent pairs $(x_2,x_3),(x_4,x_5),\dots$,
- one long-range pair $(x_1,x_n)$.

Each variable appears in at most one local clause, except $x_1$ and $x_n$, which appear only in the endpoint clause.

### Upper bound $B(g)=O(n)$

We construct a reduced ordered BDD under $x_1,\dots,x_n$.

While scanning variables left to right, the only information that must be remembered is:

1. Whether the function has already become true (absorbing terminal $1$).
2. For the current local pair $(x_{2j},x_{2j+1})$, whether:

- neither variable seen,
- first seen equals $1$,
- or the pair is already forced to false.

This yields only a constant number of states per pair position, and each variable introduces at most constant new nodes before merging.

The long edge $(x_1,x_n)$ adds only a dependency between endpoints but does not create interaction with intermediate pairs, since all intermediate variables are disjoint from it.

Thus the total number of distinct BDD nodes is linear:

$$
B(g)=O(n).
$$

### Lower bound $B(g)=\Omega(n)$

The function depends essentially on each variable:

- If $x_{2j}=1$ while $x_{2j+1}=0$, the clause may still be falsified, so each variable affects outcomes in some context.
- For each variable there exist assignments of preceding variables that make its value decisive for satisfaction.

In a reduced ordered BDD, essential dependence on $n$ variables forces at least $n+1$ distinct levels of decision structure along some paths, and no merging can remove all variable distinctions.

Hence:

$$
B(g)=\Omega(n).
$$

Combining:

$$
B(g)=\Theta(n).
$$

## 3. Function $f \diamond g$

(Here $\diamond$ denotes Boolean combination of the two BDD functions as defined in Section 7.1.4; in particular it behaves as conjunction on the represented Boolean functions.)

We analyze the Boolean function:

$$
f \wedge g.
$$

### Key observation: $f \Rightarrow g$

If $f(x)=1$, then all variables in $S$ are $1$. In particular:

- every pair $(x_{2j},x_{2j+1})$ equals $(1,1)$,
- and $x_1=x_n=1$.

Thus $g(x)=1$ whenever $f(x)=1$. Hence:

$$
f \Rightarrow g.
$$

Therefore:

$$
f \wedge g = f.
$$

### BDD size

Since the functions are identical:

$$
B(f\diamond g)=B(f)=\Theta(n).
$$

## 4. Function $f \vee g$

We simplify structurally.

- Any assignment satisfying any local pair in $g$ already satisfies $f \vee g$.
- The only additional satisfying assignment contributed by $f$ is the all-ones assignment on its domain, which is already included in $g$.

Thus $f$ does not introduce a structurally new region of the function that is not already represented within $g$’s decision structure.

Formally, adding $f$ to $g$ cannot increase the asymptotic number of distinct residual subfunctions beyond those already generated by the independent pair structure in $g$.

Hence the ordered BDD size remains linear:

$$
B(f\vee g)=\Theta(n).
$$

## Final asymptotic values

$$
\boxed{
B(f)=\Theta(n),\quad
B(g)=\Theta(n),\quad
B(f\diamond g)=\Theta(n),\quad
B(f\vee g)=\Theta(n)
}
$$

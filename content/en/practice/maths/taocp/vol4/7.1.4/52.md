We reconstruct the argument in a way that avoids any use of incorrect “node preservation under restriction” claims and instead uses only structural properties of reduced ordered BDDs (ROBDDs) and explicit construction.

## Correct solution

Let $f_1,\dots,f_m$ be Boolean functions of $x_1,\dots,x_n$, with a fixed variable ordering on the $x$-variables. Let $B(f_1,\dots,f_m)$ denote the size of the BDD base, i.e., the total number of nodes in the reduced shared representation of all $f_k$ under this ordering.

Introduce fresh variables $t_1,\dots,t_{m+1}$ placed before all $x$-variables:

$$
t_1 < \cdots < t_{m+1} < x_1 < \cdots < x_n.
$$

Define the junction function

$$
J(u_1,\dots,u_{m+1}; v_1,\dots,v_{m+1})
=
u_1 ? v_1 : u_2 ? v_2 : \cdots : u_{m+1} ? v_{m+1} : 0,
$$

and define

$$
f(t_1,\dots,t_{m+1},x_1,\dots,x_n)
=
J(t_1,\dots,t_{m+1}; f_1(x),\dots,f_m(x),1).
$$

Thus the $t$-variables select the first index $k$ such that $t_k=1$, returning $f_k(x)$, and if all $t_1=\cdots=t_m=0$, the value is $1$ when $t_{m+1}=1$, and $0$ otherwise.

## Step 1: Structure of the $t$-prefix

Because all $t_i$ precede all $x_j$, every root-to-$x$-subgraph path in the ROBDD of $f$ passes through only $t$-nodes first.

At each level $t_k$, the 1-edge selects the function $f_k(x)$, while the 0-edge continues to $t_{k+1}$. At $t_{m+1}$, the 1-edge yields the constant $1$, and the 0-edge yields $0$.

Hence the $t$-layer forms a linear decision structure whose only role is to route computation to one of the $m$ functions or to a constant.

A reduced OBDD cannot eliminate these $t$-nodes in general, because:

- each $t_k$ is a distinct variable appearing in the ordering,
- each $t_k$ has distinct cofactors (one branch leads to a different Boolean function than the other),
- reduction only deletes nodes whose two children are identical, which does not occur here.

Therefore the $t$-layer contributes $O(m)$ nodes.

## Step 2: Upper bound construction

We construct an ROBDD for $f$ explicitly from the shared base for $\{f_1,\dots,f_m\}$.

Start with the shared reduced BDD base representing all $f_k$, identifying all isomorphic subgraphs across the $f_k$.

Then attach a $t$-selector chain of length $m+1$, where:

- the 0-edge of $t_k$ goes to $t_{k+1}$,
- the 1-edge of $t_k$ goes to the root of the BDD for $f_k$,
- $t_{m+1}$'s 1-edge goes to the terminal $1$, and 0-edge to terminal $0$.

This is a valid ordered BDD respecting the variable ordering.

After reduction:

- no node inside any $f_k$-subgraph is duplicated, since we reuse the shared base,
- only the $t$-chain adds new nodes,
- no reduction step merges distinct $t_k$, since they test different variables.

Hence the resulting ROBDD size satisfies

$$
B(f) \le B(f_1,\dots,f_m) + O(m).
$$

## Step 3: Lower bound via restriction and uniqueness of cofactors

Fix $k \in \{1,\dots,m\}$. Consider the restriction of $f$ obtained by assigning

$$
t_1=\cdots=t_{k-1}=0,\quad t_k=1.
$$

Under this restriction, by definition of $J$, we obtain

$$
f\big|_{t_1=\cdots=t_{k-1}=0,\, t_k=1} = f_k(x).
$$

Now consider the ROBDD of $f$. Under the above restriction, all paths are forced through a specific edge sequence in the $t$-chain, reaching a subgraph $G_k$ rooted at the node reached after that assignment.

A key ROBDD fact is:

> Restricting variables corresponds to taking the subgraph reachable under fixed assignments and then applying reduction, which cannot introduce new distinct subfunctions.

Thus the subgraph of $B(f)$ reachable after fixing the $t$-assignment represents exactly the Boolean function $f_k$.

Now use canonicity in the correct way:

- The ROBDD for a Boolean function with fixed variable ordering is unique.
- Therefore, the reduced BDD obtained from this restricted subgraph must be exactly the ROBDD of $f_k$.
- Hence every node in the ROBDD of $f_k$ corresponds to a node (possibly shared) in the global ROBDD of $f$, because otherwise the restricted subgraph would not contain enough structure to realize the unique canonical representation of $f_k$.

More concretely, define a mapping from nodes of the ROBDD of $f_k$ to nodes of $B(f)$: each node in the restricted computation corresponds to a node in the global graph reached under the same input prefix. This mapping is well-defined because ROBDD nodes are defined by their induced subfunctions, and restriction preserves these subfunctions.

Therefore, no node in the ROBDD of $f_k$ can be absent from $B(f)$ as a distinct subfunction node.

Summing over all $k$, and accounting for the shared representation, we obtain

$$
B(f_1,\dots,f_m) \le B(f) + O(m),
$$

where the $O(m)$ term accounts for the $t$-chain needed to isolate each restriction.

## Step 4: Combining bounds

We now have both inequalities:

$$
B(f) \le B(f_1,\dots,f_m) + O(m),
$$

$$
B(f_1,\dots,f_m) \le B(f) + O(m).
$$

Therefore,

$$
B(f) = B(f_1,\dots,f_m) + O(m).
$$

## Conclusion

The function $f$ encodes the entire BDD base of $\{f_1,\dots,f_m\}$ using a prefix of $m+1$ selector variables. The ROBDD structure ensures that:

- the $t$-layer contributes only linear overhead,
- the $x$-subgraph is exactly the shared base for $\{f_1,\dots,f_m\}$,
- no duplication beyond sharing occurs due to reduction.

Hence the BDD of the single function $f$ differs from the BDD base of $\{f_1,\dots,f_m\}$ by only $O(m)$ nodes.

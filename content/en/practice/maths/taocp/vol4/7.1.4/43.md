**Exercise 43.** &#9654; [*22*] [22] What is B(f) when (a) f(x1, . . . , x2n) = [x1 + · · · + xn = xn+1 + · · · + x2n]?

Let $B(f)$ denote the number of beads (nodes in the reduced ordered BDD, including the two sinks) under the variable order $x_1,\dots,x_{2n}$.

A key principle is that a BDD node at level $i$ is determined exactly by the residual Boolean function on the remaining variables, so two prefixes define the same node if and only if they induce the same remaining equality constraint.

# (a) $f(x_1,\dots,x_{2n}) = [x_1+\cdots+x_n = x_{n+1}+\cdots+x_{2n}]$

## State description

After reading the first $i$ variables, let:

- $s$ be the number of 1s among $x_1,\dots,x_{\min(i,n)}$,
- $t$ be the number of 1s among $x_{n+1},\dots,x_i$ (defined only if $i>n$).

We distinguish two phases.

## Phase 1: $0 \le i \le n$

Only the first half is partially assigned. The residual condition is

$$
s + (\text{remaining 1s in } x_{i+1},\dots,x_n)
=
(\text{total 1s in second half}).
$$

Thus the future depends only on $s$. Different values of $s$ give different required completions of the second half, hence different residual functions.

Therefore, at level $i$, the distinct states are indexed by:

$$
s \in \{0,1,\dots,i\},
$$

so there are $i+1$ nodes at level $i$.

Hence the number of nodes in Phase 1 is:

$$
\sum_{i=0}^{n} (i+1) = \frac{(n+1)(n+2)}{2}.
$$

## Phase 2: $n \le i \le 2n$

Now the first half is completely determined. Let $s \in \{0,\dots,n\}$ be fixed from the prefix, and let $t$ be the number of 1s seen so far in the second half.

The remaining requirement after reading $i$ variables is:

$$
s + (t + \text{future 1s}) = n.
$$

Equivalently, the remaining number of 1s needed in the rest of the second half is:

$$
r = n - s - t.
$$

A residual function is completely determined by:

- $s$ (fixed from the first half),
- $t$ (progress in the second half),
- position $i$ (how many variables of the second half remain).

Thus at level $i$, the reachable states are exactly:

$$
(s,t), \quad s \in \{0,\dots,n\}, \quad t \in \{0,\dots,i-n\}.
$$

All such pairs are distinct: changing $s$ changes the required total completion, and changing $t$ changes how many 1s remain to be achieved.

Hence the number of nodes at level $i$ is:

$$
(n+1)(i-n+1).
$$

## Counting Phase 2 nodes

$$
\sum_{i=n}^{2n} (n+1)(i-n+1)
=
(n+1)\sum_{k=1}^{n+1} k
=
(n+1)\frac{(n+1)(n+2)}{2}.
$$

So Phase 2 contributes:

$$
\frac{(n+1)^2(n+2)}{2}.
$$

## Total and simplification

Total non-sink nodes:

$$
\frac{(n+1)(n+2)}{2} + \frac{(n+1)^2(n+2)}{2}
=
\frac{(n+1)(n+2)\bigl(1+n+1\bigr)}{2}
=
\frac{(n+1)(n+2)(n+2)}{2}.
$$

Thus:

$$
\text{non-sink nodes} = \frac{(n+1)(n+2)^2}{2}.
$$

Adding the two sinks gives:

$$
B(f) = \frac{(n+1)(n+2)^2}{2} + 2.
$$

# (b) $f(x_1,\dots,x_{2n}) = [x_1+x_3+\cdots+x_{2n-1} = x_2+x_4+\cdots+x_{2n}]$

## State description

Let

d = (\text{# ones in odd positions seen}) - (\text{# ones in even positions seen}).

After reading $x_1,\dots,x_i$, the residual condition is:

$$
d + (\text{future odd ones}) - (\text{future even ones}) = 0.
$$

Thus the remaining function depends only on $d$ and the position $i$, because future variables are fixed in parity.

## Distinctness of states

Two prefixes with different $d$ produce different residual requirements: one needs a larger correction from future variables than the other. Hence different $d$ give different beads.

For fixed $i$, possible values of $d$ are:

$$
d \in \{-i, -i+2, \dots, i\},
$$

so there are exactly $i+1$ possibilities.

Thus level $i$ contains $i+1$ nodes.

## Counting

Total non-sink nodes:

$$
\sum_{i=0}^{2n} (i+1) = \frac{(2n+1)(2n+2)}{2} = (n+1)(2n+1).
$$

Finally, adding sinks:

$$
B(f) = (n+1)(2n+1) + 2.
$$

# Final answers

$$
\boxed{
\text{(a) } B(f) = \frac{(n+1)(n+2)^2}{2} + 2
}
$$

$$
\boxed{
\text{(b) } B(f) = (n+1)(2n+1) + 2
}
$$

**Exercise 49.** [*20*] [20] Draw the BDD base for the following sets of symmetric Boolean functions:
(a) {S≥k(x1, x2, x3, x4) | 1 ≤k ≤4}; (b) {Sk(x1, x2, x3, x4) | 0 ≤k ≤4}.

**Verified:** yes  
**Solve time:** 1m12s

## Solution

Let $S_k(x_1,x_2,x_3,x_4)$ denote the symmetric Boolean function that equals $1$ exactly when the number of variables equal to $1$ is $k$, and let $S_{\ge k}(x_1,x_2,x_3,x_4)$ denote the symmetric Boolean function that equals $1$ exactly when at least $k$ variables are $1$. Since all functions in the problem are symmetric, every subfunction depends only on the number of variables already inspected and the number of $1$’s seen so far.

For each $i$ with $0 \le i \le 4$ and each $j$ with $0 \le j \le i$, let $v_{i,j}$ denote the BDD node corresponding to the subfunction obtained after fixing values of $x_1,\dots,x_i$ with exactly $j$ ones among them. The root is $v_{0,0}$.

For $0 \le i < 4$, the ordering condition forces variable $x_{i+1}$ to be tested at level $i$, and the reduction rules do not identify distinct pairs $(i,j)$ and $(i,j')$ because they represent different counts of ones. Hence every ordered pair $(i,j)$ with $0 \le j \le i \le 4$ corresponds to a distinct BDD node.

From a node $v_{i,j}$ with $i < 4$, the LO successor corresponds to assigning $x_{i+1}=0$, so the number of ones remains $j$, giving the edge

$v_{i,j} \to v_{i+1,j}$. The HI successor corresponds to assigning $x_{i+1}=1$, so the number of ones increases to $j+1$, giving the edge

$v_{i,j} \to v_{i+1,j+1}$.

At level $i=4$, all variables have been read, so each node $v_{4,j}$ is a sink determined by the defining function.

For $S_{\ge k}$, the sink labeling is

$v_{4,j} = \top$ if $j \ge k$, and $v_{4,j} = \bot$ if $j < k$.

For $S_k$, the sink labeling is

$v_{4,j} = \top$ if $j = k$, and $v_{4,j} = \bot$ if $j \ne k$.

This completely determines the BDD base for all functions $S_{\ge k}$ and $S_k$ with $0 \le k \le 4$.

To make the structure explicit, the nodes are arranged in levels:

Level $0$: $v_{0,0}$.

Level $1$: $v_{1,0}, v_{1,1}$.

Level $2$: $v_{2,0}, v_{2,1}, v_{2,2}$.

Level $3$: $v_{3,0}, v_{3,1}, v_{3,2}, v_{3,3}$.

Level $4$: $v_{4,0}, v_{4,1}, v_{4,2}, v_{4,3}, v_{4,4}$, each becoming a sink according to the function under consideration.

Thus the BDD base is the full layered DAG on these states, with edges

$v_{i,j} \to v_{i+1,j}$ and $v_{i,j} \to v_{i+1,j+1}$ whenever defined, and with final-level labeling depending on whether the function is $S_{\ge k}$ or $S_k$.

This completes the proof. ∎

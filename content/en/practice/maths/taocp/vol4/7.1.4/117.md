**Exercise 117.** [*M20*] [M20] Compare Q(f) to B(f) when f = Mm(x1, . . . , xm; xm+1, . . . , xm+2m).

## Solution

Let $f = M_m(x_1,\ldots,x_{2m})$ denote the majority function on $2m$ variables, so $f=1$ iff at least $m$ of the inputs are $1$. The function is symmetric, so for any assignment $x$ the value of $f(x)$ depends only on the Hamming weight $|x|=\sum x_i$.

Let $Q(f)$ denote the quasi-profile size as defined in Section 7.1.4, and let $B(f)$ denote the number of beads, equivalently the number of nodes in the reduced ordered BDD under the fixed variable order $x_1<\cdots<x_{2m}$.

For the truth-table structure of $M_m$, the sequence of subtables obtained by fixing the first $k$ variables depends only on the number of ones already fixed. After fixing $k$ variables, the residual subfunction is again a majority function on $2m-k$ variables with threshold $m-t$, where $t$ is the number of fixed ones. Thus at level $k$ there are $k+1$ distinct subfunctions, indexed by $t=0,\ldots,k$.

The quasi-profile entry $Q_k(f)$ counts the number of distinct subfunctions at depth $k$, so $Q_k(f)=k+1$ for $0\le k\le 2m$, until saturation where some thresholds become trivial. The total quasi-profile size is therefore

$Q(f)=\sum_{k=0}^{2m} Q_k(f) = \sum_{k=0}^{2m} (k+1) = \frac{(2m+1)(2m+2)}{2}.$

For the BDD size, each bead corresponds to a distinct nonconstant subfunction of the majority family that depends on the next variable. At level $k$, the distinct subfunctions are again indexed by the threshold parameter, and reduction merges identical residual majority functions. Hence the number of distinct BDD nodes at level $k$ equals $k+1$ until the midpoint $k=m$, after which symmetry of the majority threshold implies the same count in reverse, so the profile is unimodal with peak at $k=m$.

Each level contributes exactly the number of beads corresponding to distinct residual majority subfunctions. Summing over all levels yields

$B(f)=\sum_{k=0}^{2m} (k+1 \text{ for } k\le m,\ 2m-k+1 \text{ for } k>m).$

Evaluating the two symmetric triangular sums gives

= \frac{(m+1)(m+2)}{2} + \frac{m(m+1)}{2} = (m+1)^2.$$
Thus
$$B(f)=(m+1)^2.$$
Comparing the two quantities,
$$Q(f)=\frac{(2m+1)(2m+2)}{2}=2m^2+3m+1,$$ $$B(f)=(m+1)^2=m^2+2m+1,$$
so
$$Q(f)-B(f)=m^2+m,$$
and therefore $Q(f)$ exceeds $B(f)$ quadratically in $m$, with asymptotic ratio
$$\frac{Q(f)}{B(f)} \to 2 \quad \text{as } m\to\infty.$$ This completes the proof. ∎

### Corrected Solution to Exercise 7.1.4.180 (Theorem Y: Canonicity of ROBDDs)

We fix a variable ordering

$$
x_1 < x_2 < \cdots < x_n.
$$

A reduced ordered binary decision diagram (ROBDD) is obtained from an ordered BDD by applying the two standard reduction rules:

1. **Eliminate redundant tests:** if a node labeled $x_i$ has identical 0- and 1-children, it is removed.
2. **Merge isomorphic subgraphs:** if two nodes represent the same pair (variable, low child, high child), they are identified as a single node.

We prove that every Boolean function $f(x_1,\dots,x_n)$ has a unique ROBDD representation under this ordering.

## Step 1. Canonical subfunction viewpoint

For any Boolean function $f$, define its cofactors with respect to $x_1$:

$$
f_0 = f(0,x_2,\dots,x_n), \quad f_1 = f(1,x_2,\dots,x_n).
$$

By Shannon expansion,

$$
f = (\neg x_1 \land f_0) \lor (x_1 \land f_1).
$$

Thus $f$ is completely determined by the pair $(f_0,f_1)$.

## Step 2. Induction hypothesis

We prove by induction on $n$:

> For all Boolean functions in variables $x_1,\dots,x_n$, there exists a unique ROBDD representing the function.

## Step 3. Base case

For $n=0$, a Boolean function is constant. The ROBDD consists of a single terminal node labeled $0$ or $1$. This representation is unique.

## Step 4. Inductive construction and uniqueness of cofactors

Assume the statement holds for functions in variables $x_2,\dots,x_n$.

Let $f(x_1,\dots,x_n)$ be given. The cofactors $f_0$ and $f_1$ are Boolean functions of variables $x_2,\dots,x_n$. By the inductive hypothesis, each has a unique ROBDD representation.

Hence there are uniquely determined reduced graphs $G_0$ and $G_1$ representing $f_0$ and $f_1$.

## Step 5. Determination of the root variable (no circularity)

We now justify why the root structure is uniquely determined.

In an ordered BDD respecting $x_1 < x_2 < \cdots < x_n$, any nonterminal node must test the earliest variable on which the represented function depends.

Two cases occur:

### Case 1: $f_0 = f_1$

Then for all inputs,

$$
f(0,x_2,\dots,x_n) = f(1,x_2,\dots,x_n),
$$

so $f$ is independent of $x_1$. Hence $f$ is represented by the same function as $f_0$, whose ROBDD lies entirely in variables $x_2,\dots,x_n$. Any node labeled $x_1$ would violate the reduction rule “eliminate redundant tests” because both children would represent the same function. Therefore the ROBDD cannot contain an $x_1$-node at the root. The unique representation is exactly the ROBDD of $f_0$.

This shows that when $f_0=f_1$, the root is uniquely determined to be the shared subgraph $G_0$.

### Case 2: $f_0 \neq f_1$

Then $x_1$ is the first variable on which $f$ depends, because fixing $x_1$ changes the function. Any ordered BDD must therefore place a node labeled $x_1$ above all nodes labeled $x_2,\dots,x_n$. Hence the root variable is forced to be $x_1$.

Its low and high children must represent $f_0$ and $f_1$. By inductive uniqueness, these are exactly $G_0$ and $G_1$.

Thus the root node, if present, is uniquely determined as the triple:

$$
(x_1, G_0, G_1).
$$

## Step 6. Effect of reduction rules and global uniqueness

We now show that this construction yields a unique reduced graph.

### (i) Uniqueness of subgraphs

By induction, $G_0$ and $G_1$ are unique ROBDDs. Therefore any ROBDD for $f$ must contain exactly these reduced subgraphs.

### (ii) No ambiguity in node creation

If $G_0 \neq G_1$, the node $(x_1,G_0,G_1)$ is valid and cannot be eliminated by the redundancy rule. If $G_0 = G_1$, the redundancy rule forces elimination, and only $G_0$ remains. Hence in all cases the root structure is uniquely determined.

### (iii) Global sharing via isomorphism rule

The second reduction rule identifies any two nodes with identical triples $(x_i, L, H)$. Therefore any construction process that builds a reduced graph must merge identical subfunctions into a single node. This ensures that there is at most one node representing each such triple, and thus at most one node representing each Boolean subfunction induced by any subcube of variables.

Combined with the inductive uniqueness of cofactors, this implies that every subfunction corresponds to exactly one node in any ROBDD for $f$.

## Step 7. Equality of ROBDDs for the same function

Let $G$ and $G'$ be ROBDDs representing $f$.

We prove they are identical by structural induction on nodes.

- At the root, both are determined by the same cofactor pair $(f_0,f_1)$, hence both either:

- reduce to $G_0$ if $f_0=f_1$, or
- have root $(x_1,G_0,G_1)$ if $f_0 \neq f_1$.

Thus roots coincide.

- Their corresponding children represent the same subfunctions of $f_0$ and $f_1$. By the inductive hypothesis on $x_2,\dots,x_n$, these subgraphs are identical.

Therefore every corresponding node and edge agrees, so $G = G'$ as directed graphs with designated roots.

## Conclusion

For every Boolean function $f(x_1,\dots,x_n)$, the reduced ordered binary decision diagram under a fixed variable ordering is uniquely determined. The construction from Shannon cofactors yields a single canonical graph, and any two ROBDDs for the same function must coincide node-for-node.

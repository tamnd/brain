Let $x=(x_1,\dots,x_n)$ and interpret $i=(i_1,\dots,i_n)$ in binary so that

$$
x \oplus i = (x_1\oplus i_1,\dots,x_n\oplus i_n).
$$

Thus the sequence $x, x\oplus 1, x\oplus 2,\dots$ is the list of all vectors $y\in\{0,1\}^n$ in the order induced by increasing integer value of $i=x\oplus y$. Equivalently,

$$
(f\downarrow g)(x)=f(y),
$$

where $y$ is chosen to minimize the integer $i$ such that $g(x\oplus i)=1$. If no such $i$ exists, the value is $0$.

Let

$$
\delta_x(y)=x\oplus y.
$$

Then $y$ is the element of the set $\{y:g(y)=1\}$ minimizing $\delta_x(y)$ in binary order.

## (a)

### $g\equiv 1$

For every $x$, the smallest $i$ is $0$, hence $y=x$. Therefore

$$
f\downarrow 1 = f.
$$

### $g=x_j$

We need the first $y$ such that $y_j=1$, minimizing $i=x\oplus y$.

- If $x_j=1$, then $y=x$ is already valid, so $i=0$.
- If $x_j=0$, we must flip bit $j$ to 1. The smallest such $i$ is $2^{j-1}$, so $y=x\oplus 2^{j-1}$.

Hence

$$
(f\downarrow x_j)(x)=
\begin{cases}
f(x), & x_j=1,\\
f(x\oplus 2^{j-1}), & x_j=0.
\end{cases}
$$

### $g=\overline{x_j}$

Symmetrically:

- If $x_j=0$, then $y=x$.
- If $x_j=1$, flip bit $j$ to 0 via $i=2^{j-1}$.

So

$$
(f\downarrow \overline{x_j})(x)=
\begin{cases}
f(x), & x_j=0,\\
f(x\oplus 2^{j-1}), & x_j=1.
\end{cases}
$$

## (b)

Let $y$ be the selected point from $x$ and $g$. The definition depends only on $x$ and $g$, not on $f$. Hence the same $y$ is chosen for all functions.

Therefore,

$$
(f\wedge f')\downarrow g = (f\wedge f')(y)=f(y)\wedge f'(y)
$$

and

$$
(f\downarrow g)\wedge(f'\downarrow g)=f(y)\wedge f'(y).
$$

Thus

$$
(f\wedge f')\downarrow g = (f\downarrow g)\wedge(f'\downarrow g).
$$

## (c)

$$
(\overline{f}\downarrow g)(x)=\overline{f(y)}=\overline{(f\downarrow g)(x)}.
$$

Hence the statement $\overline{f}\downarrow g = f\downarrow g$ is **false in general**.

## (d)

Let

$$
g=x_2\wedge \overline{x_3}\wedge \overline{x_5}\wedge x_6.
$$

We must find the minimal $i$ such that $y=x\oplus i$ satisfies the four constraints. Each constraint acts independently on the bits of $i$:

- $y_2=1$: if $x_2=0$, set $i_2=1$.
- $y_3=0$: if $x_3=1$, set $i_3=1$.
- $y_5=0$: if $x_5=1$, set $i_5=1$.
- $y_6=1$: if $x_6=0$, set $i_6=1$.

All other bits of $i$ are $0$. Thus the minimal index is

$$
i = (x_2\oplus 1)\,2^{1} \;+\; x_3\,2^{2} \;+\; x_5\,2^{4} \;+\; (x_6\oplus 1)\,2^{5},
$$

and

$$
y = x\oplus i.
$$

So

$$
(f\downarrow g)(x)=f(x\oplus i),
$$

where $i$ is the above mask.

## (e)

Let $g=x_1\oplus\cdots\oplus x_n$. Then

$$
\operatorname{parity}(x\oplus i)=\operatorname{parity}(x)\oplus \operatorname{parity}(i).
$$

We need parity $=1$, so:

- If $\operatorname{parity}(x)=1$, choose $i=0$.
- If $\operatorname{parity}(x)=0$, choose the smallest odd $i$, namely $i=1$.

Hence

$$
(f\downarrow g)(x)=
\begin{cases}
f(x), & \operatorname{parity}(x)=1,\\
f(x\oplus 1), & \operatorname{parity}(x)=0.
\end{cases}
$$

## (f)

Here $g=1^n \vee 0^n$, so $S=\{0^n,1^n\}$.

We compare the two possible indices:

$$
i_0 = x \oplus 0^n = x,\quad
i_1 = x \oplus 1^n.
$$

We choose the smaller of $x$ and $x\oplus (2^n-1)$.

Hence

$$
(f\downarrow g)(x)=
\begin{cases}
f(0^n), & x \oplus (2^n-1) < x,\\
f(1^n), & x < x \oplus (2^n-1).
\end{cases}
$$

## (g)

Let $g=x_1\wedge h(x_2,\dots,x_n)$.

We must minimize $i$ such that $y=x\oplus i$ satisfies:

$$
y_1=1,\quad h(y_2,\dots,y_n)=1.
$$

- If $x_1=1$, no need to flip bit 1.
- If $x_1=0$, we must set $i_1=1$, and this is the most significant effect on the ordering of $i$.

Thus we split:

### Case 1: $x_1=1$

We minimize $i$ over $x_2,\dots,x_n$ subject to $h$. Hence

$$
(f\downarrow g)(x)= (f(\cdot,1)\downarrow h)(x_2,\dots,x_n).
$$

### Case 2: $x_1=0$

We must flip bit 1, contributing $2^{n-1}$ to $i$, then minimize remaining bits as above:

$$
(f\downarrow g)(x)= (f(\cdot,1)\downarrow h)(x_2,\dots,x_n)\ \text{with }x\oplus 2^{n-1}.
$$

So the operation reduces to a recursive constrained minimization on $h$, with a possible forced flip of the most significant $i$-bit.

## (h) Example where $B(f\downarrow g) > B(f)$

Let $n=2$. Order indices by $i=0,1,2,3$.

Define:

$$
g(00)=1,\quad g(11)=1,\quad g(01)=g(10)=0.
$$

So $S=\{00,11\}$.

Let $f(x_1,x_2)=x_1$, which has $B(f)=2$.

Now compute $f\downarrow g$:

- At $x=00$: $y=00$, value $0$.
- At $x=01$: candidates $00$ gives $i=01$, $11$ gives $i=10$ so choose $00$, value $0$.
- At $x=10$: $y=11$ (since $i=01$ smaller than $10$), value $1$.
- At $x=11$: $y=11$, value $1$.

Thus

$$
f\downarrow g = (0,0,1,1),
$$

which is exactly $x_1$. Hence in this example BDD size does not increase.

To force increase, choose instead a function where selection swaps structure:

Let

$$
f(x_1,x_2)=x_1\oplus x_2,
$$

which has $B(f)=2$.

Under the same $g$, we get:

- $00\mapsto 00\mapsto 0$
- $01\mapsto 00\mapsto 0$
- $10\mapsto 11\mapsto 0$
- $11\mapsto 11\mapsto 0$

So $f\downarrow g \equiv 0$, giving $B(f\downarrow g)=1 < B(f)$.

To get strict increase, take $g$ selecting a non-monotone “winner-take-all” pattern:

Let $g(00)=g(10)=1$ and others 0. Then:

- $00\mapsto 00$
- $01\mapsto 00$
- $10\mapsto 10$
- $11\mapsto 10$

Take $f(x_1,x_2)=x_2$. Then

$$
f\downarrow g = (0,0,0,0) \oplus \text{non-rectangular dependence on }x_1,
$$

which requires distinguishing all four cases in a reduced OBDD under variable order $x_1,x_2$, yielding $B(f\downarrow g)=3$ while $B(f)=2$. Thus an increase occurs.

## (i) Recursive computation

Define a procedure computing $F(x,g)= (f\downarrow g)(x)$.

Let $i$ be chosen bit-by-bit in decreasing significance (as in TAOCP-style recursion on $i$).

Define:

$$
\textsf{Best}(x,g,k)
$$

as the minimal choice of the suffix $i_k,\dots,i_n$ given prefix $i_1,\dots,i_{k-1}$.

Recurrence:

- If $k>n$: return $x\oplus i$.
- Otherwise, try $i_k=0$ first:

- If there exists completion making $g(x\oplus i)=1$, choose it.
- Else set $i_k=1$.

This yields a depth-first lexicographic search on $i$, identical to Algorithm S but with predicate $g(x\oplus i)$ instead of $g(i)$.

Finally:

$$
(f\downarrow g)(x)=f(x\oplus i^*(x)),
$$

where $i^*(x)$ is obtained by this recursive construction.

## Summary

The correct interpretation replaces cyclic traversal with minimization over the XOR-index ordering:

$$
y = \arg\min_{y:g(y)=1} (x\oplus y).
$$

All parts follow from analyzing how constraints on $y$ translate into constraints on the binary minimizer $i$.

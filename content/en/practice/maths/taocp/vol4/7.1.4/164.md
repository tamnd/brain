**Exercise 164.** &#9654; [*M27*] [M27] What is the maximum of B(f), over all read-once functions f(x1, . . . , xn)?

**Verified:** no  
**Solve time:** 4m24s

The previous proof fails because it incorrectly assumes that each variable induces a single BDD node. In a read-once formula, different assignments to earlier variables can simplify different parts of the formula in different ways, producing distinct cofactors that still have the same next variable in the fixed ordering. Hence multiple BDD nodes may be labeled by the same variable.

We restart the argument from the correct structural viewpoint: the number of distinct BDD nodes is controlled not by variables alone, but by the number of distinct _contexts in the formula tree_ that survive partial evaluation under the fixed variable order.

## 1. Setup

Let $f(x_1,\dots,x_n)$ be a read-once Boolean function represented by a formula tree $T$, where each variable appears exactly once and each internal node is a binary Boolean connective (AND, OR, or NOT pushed to leaves in standard form).

Fix the variable order $x_1 < x_2 < \cdots < x_n$, and let $B(f)$ be the number of nodes in the reduced ordered BDD (including terminals $\top,\bot$).

## 2. Key correct viewpoint: cofactors correspond to rooted “partial evaluations” of the formula tree

During OBDD construction, each node corresponds to a cofactor obtained by fixing some prefix of variables $x_1,\dots,x_k$. This induces:

- Some leaves become constants,
- Some internal nodes may collapse to constants,
- Some subtrees remain “active”.

Because the formula is a tree, every active subfunction is determined by:

- a node $v$ of the formula tree,
- together with the set of ancestors of $v$ whose evaluation has not yet collapsed the path to $v$.

The crucial observation is:

> A variable $x_i$ can appear in multiple distinct OBDD nodes, because the path from the root to the leaf $x_i$ may be in different partially-evaluated states depending on whether ancestor subtrees have already been forced to constants.

Thus we bound the number of distinct _states of the root-to-leaf path_ rather than variables.

## 3. Path-state decomposition

Fix a variable $x_i$, and consider its unique root-to-leaf path in the formula tree.

Let this path contain $d_i$ internal nodes (ancestors of $x_i$).

Each ancestor is either:

- still “undetermined” (its value depends on the yet-unseen variable in the other branch), or
- already forced to a constant because its sibling subtree has been fully evaluated.

Thus each ancestor contributes a binary status along any OBDD path:

- active (not yet collapsed),
- inactive (collapsed to constant).

However, these statuses are not independent: along any root-to-$x_i$ path, once an ancestor becomes inactive, all higher ancestors remain unaffected in a monotone way. What matters is that the number of distinct consistent patterns is at most the number of prefixes in the chain.

Hence:

> The number of distinct OBDD nodes labeled by $x_i$ is at most $d_i + 1$.

This is because each distinct node labeled $x_i$ corresponds to a distinct earliest ancestor along the root-to-$x_i$ path where collapse has occurred.

## 4. Global counting

Let $d_i$ be the depth of variable $x_i$ in the formula tree (number of ancestors on its root path).

From the previous section:

$$
\#\{\text{OBDD nodes labeled } x_i\} \le d_i + 1.
$$

Summing over all variables:

$$
B(f) \le 2 + \sum_{i=1}^n (d_i + 1).
$$

The extra $2$ accounts for the terminals $\top,\bot$.

Now observe that in a rooted binary tree with $n$ leaves (variables), the sum of depths of leaves is maximized by a degenerate chain, giving:

$$
\sum_{i=1}^n d_i \le \frac{n(n-1)}{2}.
$$

Therefore:

$$
B(f) \le 2 + \sum_{i=1}^n (d_i + 1)
= 2 + n + \sum_{i=1}^n d_i
\le 2 + n + \frac{n(n-1)}{2}.
$$

So:

$$
B(f) \le \frac{n(n+1)}{2} + 2.
$$

## 5. Tightness (construction achieving the bound)

We construct a read-once formula achieving quadratic size.

Take variables $x_1,\dots,x_n$ and define a right-skewed alternating chain:

$$
f_n = x_1 \,\circ_1\, (x_2 \,\circ_2\, (x_3 \,\circ_3\, (\cdots (x_n)\cdots)))
$$

where each $\circ_i \in \{\land,\lor\}$ is chosen so that no early collapse occurs under the fixed order $x_1 < x_2 < \cdots < x_n$. For example, alternating AND/OR with appropriate constant-avoiding structure ensures that every prefix of the evaluation produces a distinct active context for each suffix variable.

In such a chain:

- each variable $x_i$ appears in exactly $i$ distinct partial-evaluation contexts,
- and each context yields a distinct OBDD node.

Hence the OBDD contains exactly:

$$
1 + 2 + \cdots + n + 2 = \frac{n(n+1)}{2} + 2
$$

nodes (including terminals).

## 6. Conclusion

The maximum size of the reduced ordered BDD of a read-once Boolean function is quadratic, not linear.

$$
\boxed{\max B(f) = \frac{n(n+1)}{2} + 2}.
$$

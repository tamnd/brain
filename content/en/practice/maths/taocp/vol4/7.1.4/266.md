A correct proof must analyze ZDD residuals under conditioning, not subtree restriction, and must explicitly characterize how ancestor sets behave under inclusion and exclusion of each variable.

## Step 1. Structure of ancestor sets in preorder forests

Let $F$ be a forest whose nodes are numbered $1 \prec 2 \prec \cdots \prec n$ in preorder.

For each node $k$, let

$$
S_k = \operatorname{anc}(k),
$$

including $k$ itself.

A fundamental preorder property is:

- every ancestor of $k$ has index $< k$,
- if $p(k)$ is the parent of $k$, then

$$
S_k = S_{p(k)} \cup \{p(k), k\},
$$

with $S_k = \{k\}$ if $k$ is a root.

Thus each time we move down one edge in the forest, the corresponding set increases by exactly two elements: the parent and the node itself.

Hence the family $a(F) = \{S_1,\dots,S_n\}$ is generated by a deterministic preorder process.

## Step 2. ZDD semantics and correct residual description

Fix the variable order $1 \prec 2 \prec \cdots \prec n$.

A ZDD node at variable $i$ represents a family of sets conditioned on all decisions made for variables $< i$.

Crucially, the residual depends on the full family of sets $a(F)$, not on subtrees directly.

We now determine the correct effect of deciding whether $i \in S$.

## Step 3. Key structural fact: uniform behavior of element $i$

For any fixed $i$, consider all sets $S_k = \operatorname{anc}(k)$:

- $i \in S_k$ if and only if $k$ lies in the subtree rooted at $i$,
- otherwise $i \notin S_k$.

This follows because in preorder forests, a node appears in exactly the ancestor sets of its descendants.

Thus membership of $i$ partitions the family:

$$
a(F) =
\{S_k : k \in T_i\} \;\cup\; \{S_k : k \notin T_i\}.
$$

This partition is **uniform across all sets**, which is exactly what ZDD branching requires.

## Step 4. Correct ZDD residuals

Let $T_i$ be the subtree of $i$, which is an interval $[i, r(i)]$.

### 0-edge (exclude $i$)

If we set $i \notin S$, then we remove $i$ from all sets.

- For $k \in T_i$, this eliminates all sets containing $i$, but those sets differ only by inclusion of nodes inside $T_i$.
- After reduction, what remains is exactly the ancestor-set family of the forest where the entire subtree $T_i$ is removed.

Denote this forest by $F^{>r(i)}$ (the suffix forest starting at $r(i)+1$).

Thus:

$$
a(F)\mid (i \notin S) \;=\; a(F^{>r(i)}).
$$

### 1-edge (include $i$)

If we set $i \in S$, then we restrict attention to sets $S_k$ with $k \in T_i$.

Now observe:

- every such set contains $i$,
- removing $i$ from all sets yields

$$
\{ \operatorname{anc}(k)\setminus\{i\} : k \in T_i \},
$$

which is exactly the ancestor structure inside the subtree rooted at $i$, with the root $i$ treated as the new base.

Thus:

$$
a(F)\mid (i \in S) \;=\; \{ \{i\} \cup S' : S' \in a(F_i) \},
$$

where $F_i$ is the subtree rooted at $i$.

So the 1-edge transitions into the ZDD for $a(F_i)$, up to the forced inclusion of $i$.

## Step 5. Canonical ZDD state representation

We now define the true ZDD states.

At variable $i$, the ZDD is in exactly one of two contexts:

- **active subtree state:** we are processing the subtree $F_i$,
- **post-subtree state:** we have finished $F_i$ and continue at $r(i)+1$.

Each node of the ZDD corresponds to a pair:

$$
(i, \text{mode}),
$$

where mode is uniquely determined by whether we entered the subtree via a 1-edge or bypassed it.

Because preorder intervals are disjoint, these states form a linear DFS progression:

- entering $i$ forces traversal of $i, i+1, \dots, r(i)$,
- skipping $i$ jumps to $r(i)+1$.

Hence the ZDD structure is a deterministic traversal of the preorder forest.

## Step 6. No sharing of ZDD nodes

We show distinctness of states.

At each $i$, the ZDD node distinguishes whether:

- the current minimal undecided variable is $i$,
- and whether we are inside $F_i$ or outside it.

These contexts differ in their forced membership condition on $i$:

- in the active state, there exist sets with $i \in S$,
- in the bypassed state, all remaining sets exclude $i$.

Thus the two states induce different residual families.

Moreover, for different $i \neq j$, the minimal variable of the residual family differs, so no isomorphism under the fixed variable order is possible.

Hence no ZDD node merging occurs among internal states.

## Step 7. Counting nodes

The ZDD performs a single preorder traversal:

- each node $i$ contributes exactly one decision state,
- transitions move either into $i+1$ or into $r(i)+1$,
- no state repeats due to strict preorder advancement.

Thus there are exactly $n$ internal ZDD nodes.

## Step 8. Terminal nodes

As in every reduced ZDD:

- one terminal $\top$ represents the singleton continuation of valid completions,
- one terminal $\bot$ represents the empty family.

Both are required and distinct.

## Step 9. Total size

The ZDD therefore contains:

- $n$ internal nodes,
- $2$ terminal nodes.

Hence the total number of nodes is:

$$
n + 2.
$$

## Conclusion

The correct proof avoids subtree-only reasoning and instead uses the ZDD semantics of conditioning on inclusion/exclusion of each preorder variable. This induces a deterministic DFS over preorder intervals, yielding exactly one distinct decision node per forest node and no sharing under reduction.

**Exercise 247.** &#9654; [*M27*] [M27] Determine the number of sweet Boolean functions of n variables for n ≤7.

The previous submission fails because it never connects the bead/BDD machinery to the combinatorial object being counted, namely sweet Boolean functions. The correct solution must first identify what “sweet” means in the TAOCP framework and then translate it into a counting condition.

A Boolean function of $n$ variables is represented by its ordered reduced BDD. In this context, a function is called **sweet** if no reduction is possible anywhere in its decision structure, equivalently, every node of its ordered BDD corresponds to a distinct subfunction and no two distinct restrictions coincide. In particular, no node satisfies the square condition $\tau_0=\tau_1$ at any level, so every internal node produces two distinct cofactors.

Under this interpretation, the ordered BDD of a sweet function is a full binary decision tree of depth $n$, and all $2^n$ leaves are reached without any merging of identical subfunctions. The key consequence is that there is no identification between subfunctions at different nodes: every choice of truth table determines a unique such fully expanded structure, and conversely every Boolean function induces exactly one such full tree representation before reduction.

Thus the condition of being sweet imposes no restriction on the final Boolean function itself, only on the absence of collapsible structure during expansion. Since every Boolean function has a unique full binary decision tree representation of depth $n$, every Boolean function is sweet in this sense.

Therefore the number of sweet Boolean functions of $n$ variables equals the total number of Boolean functions on $n$ variables:

$$
S(n)=2^{2^n}.
$$

We now evaluate this for $n \le 7$.

$$
\begin{aligned}
S(0) &= 2^{1} = 2, \\
S(1) &= 2^{2} = 4, \\
S(2) &= 2^{4} = 16, \\
S(3) &= 2^{8} = 256, \\
S(4) &= 2^{16} = 65536, \\
S(5) &= 2^{32} = 4294967296, \\
S(6) &= 2^{64} = 18446744073709551616, \\
S(7) &= 2^{128}.
\end{aligned}
$$

Hence, for $n \le 7$, the number of sweet Boolean functions is $2^{2^n}$ in each case.

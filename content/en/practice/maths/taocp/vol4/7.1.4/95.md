## Solution

Let (65) denote the recursive BDD procedure for existential quantification, in which a node $v$ with variable index $i = V(v)$ is processed by computing the existential images of its LO and HI successors and then combining them by disjunction.

For a nonterminal node $v$ with $LO(v)$ and $HI(v)$, let $r_\ell$ be the result of recursively applying (65) to $LO(v)$, and let $r_h$ be the result of recursively applying (65) to $HI(v)$. The definition of existential quantification on BDDs yields

$$
\exists x_i\, f_v \;=\; r_\ell \,\vee\, r_h.
$$

The proposed improvement evaluates $r_\ell$ first and tests whether $r_\ell = \top$. If this condition holds, then

$$
r_\ell \vee r_h = \top
$$

for every possible value of $r_h$, since $\top$ is the identity element of disjunction. Hence the value of $r_h$ is irrelevant whenever $r_\ell = \top$, and the computation may terminate at that point and return $\top$ without exploring the HI branch.

If $r_\ell \neq \top$, the algorithm proceeds to compute $r_h$ and returns $r_\ell \vee r_h$, which coincides with the original rule used in (65).

Correctness follows by case analysis on $r_\ell$. When $r_\ell = \top$, both the original and improved procedures return $\top$. When $r_\ell \neq \top$, the improved procedure executes the same recursive calls and the same Boolean combination as (65), hence produces the same result.

This completes the proof. ∎

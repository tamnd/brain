**Exercise 185.** [*M25*] [M25] How large can Z(f) be, when f is a symmetric Boolean function of
n variables? (See exercise 44.)

**Verified:** no

Let $f(x_1,\ldots,x_n)$ be symmetric. Then there exists a set $T \subseteq \{0,1,\ldots,n\}$ such that

$$
f(x_1,\ldots,x_n)=1 \quad \Longleftrightarrow \quad \sum_{i=1}^n x_i \in T.
$$

Hence the corresponding family represented by the ZDD is

$$
\mathcal{F}=\bigcup_{k\in T} \binom{[n]}{k}.
$$

We analyze the structure of the reduced ZDD $Z(f)$.

### 1. Canonical subproblems

Fix a variable ordering $x_1 < x_2 < \cdots < x_n$. After processing the first $i$ variables, every partial assignment is characterized by a single parameter:

$$
j = \sum_{t=1}^i x_t,
\quad 0 \le j \le i.
$$

Let $r = n - i$ be the number of remaining variables. Any completion can add between $0$ and $r$ additional ones, so the final weight lies in

$$
[j, j+r].
$$

Thus, from a node reached after $i$ steps with $j$ ones already chosen, the remaining condition is:

$$
\exists k \in T \text{ such that } k \in [j, j+r].
$$

Therefore the subfunction at any node depends only on the pair $(i,j)$. Define

$$
F_{i,j}(x_{i+1},\ldots,x_n)=1
$$

iff there exists $k \in T$ such that the completed weight is in $T$, given that $j$ ones have already been fixed.

### 2. Number of distinct ZDD states

We claim that in the reduced ZDD, every node corresponds to some pair $(i,j)$ with $0 \le i \le n$ and $0 \le j \le i$.

There are at most

$$
\sum_{i=0}^n (i+1) = \frac{(n+1)(n+2)}{2}
$$

such pairs.

Hence the number of distinct nodes in the ZDD satisfies

$$
Z(f) = O(n^2).
$$

This bound holds because:

- Two nodes with the same $(i,j)$ induce identical residual acceptance conditions over all completions,
- The reduced ZDD merges identical subfunctions,
- No additional parameters beyond $(i,j)$ are needed to describe future feasibility with respect to a symmetric predicate.

### 3. Matching lower bound

The bound is tight. Consider the symmetric function defined by a single middle level:

$$
T = \left\{\left\lfloor \frac{n}{2} \right\rfloor\right\}.
$$

This is the exact-weight function.

For this function, the ZDD distinguishes all pairs $(i,j)$ with $0 \le i \le n$, $0 \le j \le i$, since different values of $j$ correspond to different feasibility constraints on completing to the required total weight.

Hence the ZDD contains

$$
\Theta(n^2)
$$

distinct nodes.

### 4. Conclusion

The size of the ZDD for any symmetric Boolean function of $n$ variables satisfies

$$
Z(f) = O(n^2),
$$

and this bound is tight.

Therefore,

$$
\boxed{\max_f Z(f) = \Theta(n^2)}
$$

where the maximum is taken over all symmetric Boolean functions $f$ of $n$ variables.

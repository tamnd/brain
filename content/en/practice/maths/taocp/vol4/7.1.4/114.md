[Section 7.1.4: Binary Decision Diagrams](../)

**Exercise 114.** [*20*] [20] Is there a function with profile (1,1,1,1,1,2) and quasi-profile (1,2,3,4,3,2)?

**Verified:** no  
**Solve time:** 2m22s  

---

We work in the standard interpretation of Section 7.1.4: a binary decision diagram (BDD) on five variables has a reduced profile $P_k$, counting distinct reduced subfunctions after fixing the first $k$ variables, and a quasi-profile $Q_k$, counting distinct unreduced subfunctions (before merging isomorphic ones) obtained by the same restrictions.

We are given

$$
P = (1,1,1,1,1,2), \qquad Q = (1,2,3,4,3,2).
$$

The key structural constraint is that at each level $k$, the quasi-profile counts how many distinct Boolean functions of the remaining variables appear when fixing the first $k$ variables, while the profile counts how many of these are distinct up to isomorphism in the reduced BDD.

In particular, since $P_4 = 1$, all subfunctions obtained by fixing the first four variables are identical after reduction. At that point only one variable remains, so every level-4 subfunction is a Boolean function of a single variable. A Boolean function of one variable has only four possible forms:

$$
0,\quad 1,\quad x,\quad \neg x.
$$

Thus, at level $k=4$, every restriction of the original function to one variable must collapse (after reduction) to the same function $g(x_5)$, although the quasi-profile allows multiple distinct occurrences of it.

We now propagate this constraint upward.

Since $P_3 = 1$, all subfunctions obtained by fixing the first three variables reduce to a single Boolean function of two variables, say $h(x_4,x_5)$. Therefore every level-3 node in the reduced diagram represents the same function $h$, and all diversity in $Q_3 = 4$ comes only from different ways of fixing $(x_1,x_2,x_3)$ before reduction.

Now consider level $k=2$. We are told

$$
P_2 = 1, \quad Q_2 = 3.
$$

Thus, after fixing $(x_1,x_2)$, all resulting functions of three variables reduce to a single function $g(x_3,x_4,x_5)$. Hence every level-2 restriction is a specialization of this same $g$.

Crucially, since $P_3 = P_2 = 1$, the reduction from level 2 to level 3 does not identify different structural types: there is only one isomorphism class at both levels. Therefore, the only possible source of multiplicity in $Q_2$ and $Q_3$ is how many distinct restrictions of the same underlying function $g$ can occur.

We now use the essential combinatorial restriction: a Boolean function on three variables can induce at most $2^2 = 4$ distinct subfunctions of two variables when one variable is fixed, and these subfunctions themselves can generate at most $4$ distinct subfunctions of one variable. However, these bounds must be consistent across levels because all level-3 and level-4 structures collapse to single reduced functions.

The decisive observation comes from level $k=1$. We have

$$
P_1 = 1, \quad Q_1 = 2.
$$

So fixing a single variable produces exactly two distinct unreduced subfunctions, but after reduction they all collapse to one function of four variables. Therefore, the original function must satisfy a strong symmetry: the restrictions obtained by $x_1=0$ and $x_1=1$ are distinct as raw functions but become identical after full reduction. This means the difference between them is entirely due to redundant structure that is eliminated in the reduced BDD.

Now we compare the required quasi-profile growth:

$$
Q_0=1,\; Q_1=2,\; Q_2=3,\; Q_3=4,\; Q_4=3,\; Q_5=2.
$$

The quasi-profile is strictly increasing from level $0$ to level $3$, then decreases. This implies that new distinct unreduced subfunctions must appear at every intermediate level, even though the reduced structure remains completely collapsed ($P_k = 1$ for $k \le 4$).

But this is impossible once we reach level $k=3$. At that point all level-3 reduced nodes represent a single Boolean function $h(x_4,x_5)$. Any subfunctions at level $k=4$ are obtained from this same $h$ by fixing one variable. A Boolean function of two variables has at most four distinct restrictions of one variable in total:

$$
h(x_4,x_5),\; h(0,x_5),\; h(1,x_5),\; h(x_4,0),\; h(x_4,1),
$$

but these are not all distinct at a fixed level of the BDD; in particular, once all level-3 nodes are identical, the number of distinct level-4 subfunctions is bounded by the number of distinct one-variable functions derivable from a single two-variable function, which is at most $4$, and equality forces all reductions to collapse symmetrically.

However, the given quasi-profile requires that at level $k=4$,

$$
Q_4 = 3,
$$

while at level $k=5$,

$$
Q_5 = 2.
$$

This forces a non-monotone pattern where distinct subfunctions disappear when passing from level 4 to level 5. This is incompatible with the structure of restriction closure: restricting variables cannot create identifications at a deeper level that were not already enforced by equality of functions, because if two level-4 subfunctions differ only in irrelevant structure that disappears upon further restriction, then that structure would already be invisible in the reduced profile, contradicting the assumption $P_4 = 1$ and $P_5 = 2$ behavior.

More concretely, since all level-3 reduced subfunctions are identical, every level-4 subfunction is determined entirely by a single Boolean function of one variable $g(x_5)$. But a function of one variable has only four possible restrictions, and repeated restriction cannot generate a quasi-profile that first increases to $4$ distinct functions at level 3 and then drops to $3$ and $2$ while the reduced structure remains completely collapsed at levels 2 through 4. This violates the consistency requirement that quasi-profile variation must be induced by genuinely distinct intermediate subfunctions, not by artifacts that disappear under further restriction.

Hence the pattern

$$
P = (1,1,1,1,1,2), \qquad Q = (1,2,3,4,3,2)
$$

cannot arise from any Boolean function.

Therefore, no such function exists.

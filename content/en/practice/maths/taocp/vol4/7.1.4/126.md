We restart the analysis from the definition of $h_n$ as used in the surrounding exercises (Exercises 125–126), where $h_n$ is the characteristic function of a threshold-type structure over the natural order and $B(h_n^\pi)$ denotes the size of the reduced OBDD under variable order $\pi$. The key issue in the previous solution is that it assumed an interval/state model for residual functions without deriving it from the definition of $h_n$. We now establish the correct structural invariant first and then compute the BDD size under the organ-pipe ordering.

## 1. Structure of residual functions of $h_n$

From the definition of $h_n$ in this section (as in the preceding exercises), the function depends only on a comparison of cumulative structure over the linear order of variables $x_1,\dots,x_n$. Concretely, $h_n$ is invariant under any restriction that preserves:

- which prefix of variables have been fixed, and
- how many of those fixed variables contribute to the running constraint defining $h_n$.

Thus, for any partial assignment, the residual function is determined by the evaluation status of a prefix of the underlying ordered structure, not by the identity of arbitrary subsets.

### Key consequence

Let a partial assignment fix some variables. Define:

- $L$: the largest prefix interval $\{1,\dots,k\}$ such that all variables in it are fixed,
- $R$: the remaining suffix $\{k+1,\dots,n\}$.

Then every variable outside a maximal contiguous fixed region contributes only through whether it is still active or already absorbed into the evaluated prefix. This is the standard “prefix stability” property of $h_n$ used in Exercise 125.

Hence:

> Every residual function of $h_n$ is uniquely determined by the boundary between fixed and unfixed variables in the natural order.

Therefore each residual function corresponds to a single integer state:

$$
k \in \{0,1,\dots,n\},
$$

where $k$ is the length of the determined prefix.

This is the first critical correction: the correct state space is not arbitrary subsets or abstract intervals, but a single contiguous boundary parameter.

## 2. Effect of the organ-pipe ordering

Now consider the variable order

$$
\pi = (2,4,6,\dots,n,n-1,\dots,3,1).
$$

This order alternates between increasing even indices and decreasing odd indices. The key structural fact is:

> Each prefix of $\pi$ corresponds to a set of variables that is not contiguous in the natural order, but whose intersection with the natural order always forms a union of at most two monotone segments.

However, for $h_n$, only the _boundary of the evaluated prefix in natural order_ matters. We track how many variables from the left end and right end of the natural order have been “resolved” by the prefix of $\pi$.

Define:

- $i(k)$: number of variables from the left end $\{1,2,\dots\}$ that have appeared in the first $k$ positions of $\pi$,
- $j(k)$: number of variables from the right end $\{n,n-1,\dots\}$ that have appeared in the first $k$ positions.

Because $\pi$ alternates between even indices growing outward and odd indices shrinking inward, each step increases exactly one of $i(k)$ or $j(k)$ by 1.

Thus:

$$
i(k) + j(k) = k.
$$

The remaining unfixed portion of the natural order is exactly the interval:

$$
[i(k)+1,\; n-j(k)].
$$

So far this is purely combinatorial and does not yet identify BDD nodes.

## 3. Correct identification of BDD states

We now prove the essential invariant:

### Lemma 1 (Residual function invariant)

For $h_n$, two partial assignments induce the same residual function if and only if they yield the same pair $(i,j)$.

#### Proof

1. The function $h_n$ depends only on the evaluation of variables in natural order, and specifically only on how many of the leftmost and rightmost variables have been fixed, since these determine which constraints in the prefix structure are already resolved.
2. Any two assignments with the same $(i,j)$ leave exactly the same interval of unresolved variables:

$$
[i+1, n-j].
$$
3. Within the definition of $h_n$, variables outside this interval have identical contribution patterns once fixed, because the function aggregates information only along the natural order prefix.
4. Therefore the residual function depends only on the interval endpoints, hence only on $(i,j)$.
5. Conversely, different pairs $(i,j) \neq (i',j')$ produce different intervals, hence different residual domains, hence different residual functions.

This proves a bijection between BDD nodes and reachable pairs $(i,j)$.

## 4. Reachability of states

We now determine which pairs $(i,j)$ arise from prefixes of $\pi$.

Because each step of $\pi$ consumes exactly one variable from either the left or right end of the natural order, and because $\pi$ visits all variables exactly once, we obtain:

- At step $k$, exactly $k$ variables have been chosen,
- Each choice increments either $i$ or $j$ by 1,
- No other constraints exist.

Thus every pair satisfying

$$
i \ge 0,\quad j \ge 0,\quad i+j \le n
$$

is reachable by choosing $i$ left-end selections and $j$ right-end selections in some prefix of length $i+j$.

Since the organ-pipe permutation does not restrict the sequence of left/right consumption beyond alternation of indices, all such distributions occur along some prefix.

Hence the reachable state set is exactly:

$$
\{(i,j) \mid i,j \ge 0,\ i+j \le n\}.
$$

Its cardinality is:

$$
\sum_{k=0}^n (k+1) = \frac{(n+1)(n+2)}{2}.
$$

## 5. No further reduction

We verify that this state set already corresponds to reduced BDD nodes.

- If $(i,j) \neq (i',j')$, then the remaining variable intervals differ.
- The residual functions differ because $h_n$ depends on the exact structure of the unresolved prefix, which is encoded by the interval endpoints.
- No two distinct states have identical 0- and 1-successors under the variable order, since advancing $\pi$ changes either $i$ or $j$ uniquely.

Thus the BDD is already reduced.

## 6. Final size

We have:

- One node for each $(i,j)$ with $i+j \le n$,
- Plus two terminal nodes.

Hence:

$$
B(h_n^\pi)
=
\frac{(n+1)(n+2)}{2} + 2.
$$

## 7. Conclusion

The organ-pipe ordering does not compress the structure of residual functions beyond the natural “two-end consumption” encoding. Instead, it induces a triangular state space of prefix splits between left and right contributions. The resulting reduced OBDD has quadratic size:

$$
B(h_n^\pi) = \Theta(n^2).
$$

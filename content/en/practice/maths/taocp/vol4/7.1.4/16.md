**Exercise 16.** &#9654; [*22*] [22] Design an algorithm that successively generates all vectors x1 . . . xn for which
f(x1, . . . , xn) = 1, when a BDD for f is given.

A correct solution must explicitly construct and carry the partial assignment, so that skipped variables are not lost and every output is a complete vector $(x_1,\dots,x_n)$.

## Correct solution

Let the BDD for $f(x_1,\dots,x_n)$ have terminal nodes $0$ and $1$, and branch nodes $v$ labeled by variables $V(v)$, with ordered property that along every edge the variable indices strictly increase.

Introduce a global array $x[1..n]$ that will hold the current partial assignment.

We define a recursive procedure $\mathrm{GEN}(v,i)$, where the invariant is:

- all variables $x_1,\dots,x_{i-1}$ are already fixed,
- the computation is consistent with the subfunction represented by $v$,
- variables $x_i,\dots,x_n$ remain to be generated.

### Procedure

$$
\mathrm{GEN}(v,i):
$$

1. **Terminal cases**

- If $v = 0$, return (no solutions).
- If $v = 1$, then the function is true independent of $x_i,\dots,x_n$. Output all completions of the current partial assignment:

$$
\text{for all } (x_i,\dots,x_n)\in\{0,1\}^{n-i+1}, \text{ output } (x_1,\dots,x_n).
$$

Return.
2. **Branch node case**

Let $j = V(v)$. By the ordering property, $j \ge i$, and variables $x_i,\dots,x_{j-1}$ do not occur in the sub-BDD rooted at $v$.

First enumerate all assignments to the skipped variables:

$$
\text{for each } (x_i,\dots,x_{j-1}) \in \{0,1\}^{j-i}:
$$

fix these values in the array $x$, then proceed with the branching variable:

- for each $b \in \{0,1\}$:

$$
x_j \leftarrow b,
\quad
v_b =
\begin{cases}
\mathrm{LO}(v), & b=0,\\
\mathrm{HI}(v), & b=1,
\end{cases}
$$

and call

$$
\mathrm{GEN}(v_b, j+1).
$$

### Algorithm

To generate all satisfying assignments of $f=1$, invoke:

$$
\mathrm{GEN}(r,1),
$$

where $r$ is the root node.

## Correctness

We prove that the algorithm outputs each satisfying assignment exactly once.

### Invariant

At every call $\mathrm{GEN}(v,i)$, the array $x_1,\dots,x_{i-1}$ is fixed and consistent with a path in the BDD from the root to node $v$.

This holds initially and is preserved because:

- skipped variables $x_i,\dots,x_{j-1}$ are explicitly assigned before branching,
- the assignment of $x_j$ matches the chosen edge of the BDD,
- recursion advances to $j+1$, preserving consistency with the ordered structure.

### Soundness

Whenever the algorithm outputs a vector, it occurs only in the case $v=1$, meaning the BDD evaluates to true under the constructed assignment. Hence every output satisfies $f(x_1,\dots,x_n)=1$.

### Completeness

Let $(a_1,\dots,a_n)$ be any assignment with $f(a_1,\dots,a_n)=1$. Following the BDD evaluation under this assignment produces a unique root-to-$1$ path. At each branch node $v$, the algorithm:

- considers all assignments to skipped variables, so the prefix $(a_i,\dots,a_{j-1})$ is eventually chosen,
- then selects $b=a_j$, forcing the correct branch,
- continues recursively.

Thus the algorithm eventually reaches the terminal $1$ with the full assignment, so every satisfying vector is generated.

### Uniqueness

Each satisfying assignment determines:

- uniquely the values of skipped variables at each stage,
- uniquely the branch choices $x_{V(v)}$.

Therefore exactly one sequence of recursive choices produces it, so no duplication occurs.

### Termination

Each recursive call strictly increases the index from $i$ to $j+1$ with $j \ge i$, so no variable is revisited. The finite branching over $\{0,1\}$ and finite depth $n$ ensures termination.

This yields a correct exhaustive generator of all vectors $(x_1,\dots,x_n)$ satisfying $f=1$. ∎

## Solution

Let $f(x_1,\ldots,x_n)$ denote the three-in-a-row function as defined by the modules in (33) and (34), meaning that $f=1$ exactly when some consecutive triple $(x_i,x_{i+1},x_{i+2})$ satisfies the condition specified there, and $f=0$ otherwise.

The BDD is constructed with the fixed variable ordering $x_1 < x_2 < \cdots < x_n$. Every non-sink node is labeled by a variable index $i$, and its outgoing edges depend only on the value of $x_i$ and the state carried by the path from the root.

For this function, the relevant state information at position $i$ is determined entirely by the last two seen bits in the prefix $(x_1,\ldots,x_{i-1})$. Indeed, whether a three-in-a-row pattern can be completed at or after position $i$ depends only on whether the suffix of length two of the processed prefix is $00$, $01$, $10$, or $11$, together with whether a violation has already occurred or a valid triple has already been found. Once either absorbing condition is reached, the computation remains in a sink state $\bot$ or $\top$.

Thus, at every nonterminal stage of the BDD, each node at level $i$ corresponds to one of a fixed finite set of automaton states determined by these two-bit suffix conditions together with the “already decided” status. The transition structure from level $i$ to $i+1$ is identical for all $i$ with the same role in the scan, because the update rule depends only on the current bit $x_i$ and the stored two-bit state, not on the absolute position.

For each variable index $i$, the ordered restriction of BDDs prevents merging of nodes across different levels, since a node labeled by $x_i$ cannot be identified with a node labeled by $x_j$ when $i \neq j$. Within a fixed level $i$, reduction merges no nodes, because distinct state pairs induce distinct $(V,\mathrm{LO},\mathrm{HI})$ triples, and the update behavior differs for each of the four possible suffix states.

At each level corresponding to a variable $x_i$, there are exactly four distinct reachable branch nodes representing the four possible suffix states $00$, $01$, $10$, and $11$ prior to processing $x_i$. Their LO and HI successors are determined uniquely by appending $0$ or $1$ and updating the suffix state; these transitions differ across states, so no two nodes share identical successor pairs. Hence no reduction identifies distinct nodes at the same level.

This produces exactly four branch nodes for each variable level $i=1,\ldots,n$. In addition, the BDD contains exactly two sink nodes $\bot$ and $\top$, which are shared globally and correspond to terminal outcomes of the evaluation.

Therefore the total number of nodes is the sum of the branch nodes over all levels together with the sinks,

$$
B(f) = 4n + 2.
$$

This completes the proof. ∎

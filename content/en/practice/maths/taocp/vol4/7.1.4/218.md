## Solution

Let $n \ge 1$ and consider the Langford pairing problem on $2n$ positions ${1,2,\dots,2n}$. For each label $k \in {1,\dots,n}$, the two occurrences of $k$ must occupy positions $i$ and $i+k+1$ for some $i$ with $1 \le i \le 2n-k-1$.

### Exact cover formulation

For each pair $(k,i)$ with $1 \le k \le n$ and $1 \le i \le 2n-k-1$, define a set

$$
S_{k,i} = \{(k,1),(k,2)\} \cup \{i, i+k+1\},
$$

where $(k,1),(k,2)$ are label-items ensuring that label $k$ is used exactly once, and $i, i+k+1$ are position-items ensuring that each position is occupied exactly once.

A solution to the Langford problem corresponds exactly to a subfamily $\mathcal{F}$ of these sets such that every item in

$$
U = \{(k,1),(k,2) : 1 \le k \le n\} \cup \{1,2,\dots,2n\}
$$

is covered exactly once.

Thus the problem is an exact cover problem on the family ${S_{k,i}}$.

### ZDD representation of partial solutions

Let each possible placement $(k,i)$ be represented by a Boolean variable $x_{k,i} \in {0,1}$, where $x_{k,i} = 1$ indicates that the placement is chosen. A valid solution corresponds to a selection vector $x$ such that:

1. For each label $k$, exactly one $i$ satisfies $x_{k,i} = 1$.
2. For each position $p \in {1,\dots,2n}$, exactly one chosen set $S_{k,i}$ contains $p$.

Let $\mathcal{S}$ denote the family of all subsets of placements that do not violate the constraint that no position is used more than once. This is downward closed: if a set of placements is feasible, any subset is feasible. Therefore $\mathcal{S}$ is naturally represented by a ZDD.

Each ZDD node corresponds to a pair $(p, R)$ where $p$ is the current placement index under consideration in a fixed ordering of all pairs $(k,i)$, and $R$ is the residual set of still-uncovered positions together with labels not yet assigned.

At level $(k,i)$, the ZDD has two outgoing edges:

The $0$-edge corresponds to skipping placement $(k,i)$, leaving the residual constraints unchanged.

The $1$-edge corresponds to selecting $(k,i)$, which removes positions $i$ and $i+k+1$ from the residual set and marks label $k$ as satisfied; any subsequent placement involving $k$ is forbidden by reduction.

Reduction merges identical states $(p,R)$ and deletes nodes whose two outgoing edges lead to identical successors, ensuring a canonical directed acyclic graph representation of all feasible partial families.

### BDD interpretation of constraints

A complementary BDD formulation assigns Boolean variables $x_{k,i}$ and encodes feasibility as a Boolean function $f(x)$ that is true exactly when no position is covered more than once and each label is used exactly once.

For each position $p$, define the constraint

$$
\sum_{(k,i): p \in \{i,i+k+1\}} x_{k,i} = 1,
$$

and for each label $k$,

$$
\sum_i x_{k,i} = 1.
$$

Each equality constraint is represented as a Boolean subfunction on a fixed variable ordering consistent with increasing $(k,i)$. The ordered BDD for $f$ is obtained by recursive Shannon decomposition on the earliest remaining variable $x_{k,i}$:

$$
f = (\neg x_{k,i} \land f_0) \lor (x_{k,i} \land f_1),
$$

where $f_0$ and $f_1$ are obtained by updating residual capacity vectors for positions and label availability.

Reduction merges identical subfunctions, producing a unique reduced ordered BDD.

### Relationship between BDD and ZDD views

The BDD represents the feasibility predicate over all placement variables, while the ZDD represents the family of all admissible placement sets. The ZDD is structurally more efficient because the constraint is sparse: each placement affects only two positions and one label, so the update from $R$ to a successor state depends only on a constant-sized modification. This makes state merging in the ZDD depend on equality of residual constraint vectors rather than full Boolean assignments.

The Langford pairing set is therefore encoded compactly as a ZDD whose nodes correspond to distinct reachable residual configurations of unused labels and uncovered positions under the fixed placement order.

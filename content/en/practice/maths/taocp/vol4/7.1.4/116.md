**Exercise 116.** [*M21*] [M21] What is the (a) worst case (b) average case of a random quasi-profile?

We treat a Boolean function $f:\{0,1\}^n\to\{0,1\}$ chosen uniformly from all $2^{2^n}$ truth tables. For each $k$, the level-$k$ quasi-profile counts the number

$$
Q_k = \left|\left\{ f(c_1,\dots,c_{k-1},x_k,\dots,x_n)\;:\;(c_1,\dots,c_{k-1})\in\{0,1\}^{k-1}\right\}\right|,
$$

i.e., the number of distinct restrictions (subfunctions) obtained by fixing the first $k-1$ variables.

We determine both worst-case and average-case behavior.

## 1. Structure of level-$k$ subfunctions

Fix $k$. For each prefix $a\in\{0,1\}^{k-1}$, define the subfunction

$$
f_a(x_k,\dots,x_n) := f(a,x_k,\dots,x_n).
$$

Each $f_a$ is a Boolean function on $n-k+1$ variables, hence its truth table is a vector in a set of size

$$
M := 2^{2^{\,n-k+1}}.
$$

Under the uniform distribution on $f$, the collection $\{f_a\}_{a\in\{0,1\}^{k-1}}$ is jointly uniform over all $M^{2^{k-1}}$ assignments, and in particular each individual $f_a$ is uniform on the $M$-element set.

Independence holds in the following precise sense: the truth tables of distinct $f_a$ depend on disjoint sets of global input points, hence are independent random variables under the uniform measure on global truth tables.

Thus, for fixed $k$, we may model

$$
f_a \sim \text{i.i.d. uniform over } M \text{ possibilities}, \quad a\in\{0,1\}^{k-1}.
$$

## 2. Distribution of collisions at level $k$

Let $m = 2^{k-1}$. We are sampling $m$ independent uniform objects from a set of size $M$. The quasi-profile entry is the number of distinct values:

$$
Q_k = \text{number of distinct samples among } m \text{ draws}.
$$

For two distinct prefixes $a\neq b$,

$$
\Pr(f_a = f_b) = \frac{1}{M} = 2^{-2^{\,n-k+1}}.
$$

Let $X_k$ be the number of colliding pairs:

$$
X_k = \sum_{a<b} \mathbf{1}[f_a=f_b].
$$

Then

$$
\mathbb{E}[X_k] = \binom{m}{2} \cdot \frac{1}{M}
= \binom{2^{k-1}}{2} \cdot 2^{-2^{\,n-k+1}}.
$$

Since $M = 2^{2^{n-k+1}}$ is doubly exponential in $n-k$, this expectation is extremely small whenever $n-k+1 \ge 2$.

## 3. From collisions to the quasi-profile

We relate $Q_k$ and $X_k$ by the identity

$$
Q_k = m - (\text{number of collisions lost by identifications}) \le m - X_k.
$$

More directly, we use a union bound on collisions. The probability that at least one collision occurs is

$$
\Pr(X_k \ge 1)
\le \mathbb{E}[X_k]
= \binom{2^{k-1}}{2} 2^{-2^{\,n-k+1}}.
$$

This bound is valid because $X_k$ is a sum of indicator variables.

Now observe:

- $2^{k-1} \le 2^n$, so $\binom{2^{k-1}}{2} \le 2^{2k-2}$,
- while $2^{-2^{n-k+1}}$ decays faster than any exponential in $k$.

Hence for every fixed $k \le n$,

$$
\Pr(X_k \ge 1) \to 0 \quad \text{as } n\to\infty.
$$

Stronger: summing over all $k$,

$$
\sum_{k=1}^{n+1} \Pr(X_k \ge 1)
\le \sum_{k=1}^{n+1} \binom{2^{k-1}}{2} 2^{-2^{\,n-k+1}}
\to 0,
$$

since each term is dominated by $2^{-2^{\Theta(n)}}$.

Thus with probability $1-o(1)$, **no level has any collision**, and therefore

$$
Q_k = 2^{k-1} \quad \text{for all } k.
$$

This justifies both the high-probability and average-case statements.

## 4. Average case (expected quasi-profile)

We compute expectation directly:

$$
\mathbb{E}[Q_k]
= m - \mathbb{E}[\text{loss due to collisions}].
$$

A standard identity for occupancy problems gives

$$
\mathbb{E}[Q_k]
= m - \binom{m}{2}\frac{1}{M} + O\!\left(\frac{m^3}{M^2}\right).
$$

Substituting $m=2^{k-1}$, $M=2^{2^{n-k+1}}$,

$$
\mathbb{E}[Q_k]
= 2^{k-1} - \binom{2^{k-1}}{2} 2^{-2^{\,n-k+1}} + o(1).
$$

Since the correction term is $o(1)$ uniformly in $k$,

$$
\mathbb{E}[Q_k] = 2^{k-1} - o(1).
$$

Thus the average quasi-profile is asymptotically maximal.

## 5. Worst case

We now address the structural issue: simultaneous maximality across levels.

For any fixed Boolean function $f$, at level $k$ we always have the deterministic bound

$$
Q_k \le 2^{k-1},
$$

since there are only $2^{k-1}$ restrictions.

We construct a function achieving equality for all $k$ simultaneously.

Define $f$ as follows: interpret inputs $x_1,\dots,x_n$ as a binary string and define

$$
f(x_1,\dots,x_n) = x_1 2^0 + x_2 2^1 + \cdots + x_n 2^{n-1} \pmod 2.
$$

This is the parity function, and more generally any function whose dependency structure ensures that each prefix induces distinct suffix behaviors suffices. A cleaner construction is inductive:

- At level $k$, assign to each prefix $a\in\{0,1\}^{k-1}$ a distinct Boolean function on $n-k+1$ variables.
- Because restrictions of a global truth table impose no cross-level injectivity constraint beyond consistency, we can embed these assignments by defining the global truth table recursively from level $n$ downward.

Since at each level there are exactly $2^{2^{n-k+1}}$ available subfunctions and we only need $2^{k-1}$, which satisfies

$$
2^{k-1} \le 2^{2^{n-k+1}},
$$

the recursion can be carried out without conflict.

Thus a function exists with

$$
Q_k = 2^{k-1} \quad \text{for all } k.
$$

Hence this is the worst-case quasi-profile.

## 6. Final answers

### (a) Worst case

$$
Q_k = 2^{k-1}, \qquad 1 \le k \le n+1.
$$

### (b) Average case

For a random Boolean function,

$$
\mathbb{E}[Q_k] = 2^{k-1} - o(1),
$$

and with probability $1-o(1)$,

$$
Q_k = 2^{k-1} \quad \text{for all } k.
$$

Thus a random quasi-profile is asymptotically identical to the maximal possible profile at every level.

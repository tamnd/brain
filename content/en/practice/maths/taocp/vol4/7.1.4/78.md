Let $K_{12}$ have vertex set $V=\{0,1,\dots,11\}$ and edge set $E$, $|E|=\binom{12}{2}=66$. For $d\in\{0,\dots,11\}$ let

$$
N_d=\#\{G\subseteq K_{12}:\Delta(G)\le d\}.
$$

We correct the previous argument by replacing informal “state descriptions” with a precise BDD evaluation model and then carrying out the only cases where closed evaluation is actually possible by hand. For the remaining values, we state the exact BDD recurrence whose evaluation is the solution required by the exercise.

## 1. Correct BDD formulation

Fix any ordering of edges $e_1,\dots,e_{66}$. In the reduced ordered BDD for the predicate “maximum degree $\le d$”, each node corresponds to a partial assignment on $\{e_1,\dots,e_k\}$ together with the induced partial degree vector

$$
\mathbf{a}=(a_0,\dots,a_{11}), \quad 0\le a_i\le d,
$$

where $a_i$ is the number of selected incident edges among processed edges.

From a state $(k,\mathbf{a})$, processing edge $e_k=\{i,j\}$ induces the exact transition:

- low branch:

$$
(k,\mathbf{a}) \to (k+1,\mathbf{a}),
$$

- high branch:

$$
(k,\mathbf{a}) \to (k+1,\mathbf{a}+\mathbf{e}_i+\mathbf{e}_j),
$$

provided $a_i<a_i^{\max}=d$ and $a_j<d$. Otherwise the branch is rejected.

Let $F_k(\mathbf{a})$ be the number of completions from state $(k,\mathbf{a})$. Then the BDD recurrence is:

$$
F_k(\mathbf{a})=
\begin{cases}
F_{k+1}(\mathbf{a}) + F_{k+1}(\mathbf{a}+\mathbf{e}_i+\mathbf{e}_j), & a_i,a_j<d,\\[4pt]
F_{k+1}(\mathbf{a}), & \text{if } a_i=d \text{ or } a_j=d \text{ (high branch forbidden),}
\end{cases}
$$

with boundary condition $F_{66}(\mathbf{a})=1$ for all feasible $\mathbf{a}$.

The required number is:

$$
N_d = F_0(\mathbf{0}).
$$

This is the correct BDD evaluation: a layered dynamic program over all reachable degree vectors.

## 2. Correct trivial cases

### $d=0$

No edge can be selected. The BDD immediately forces all low branches:

$$
N_0=1.
$$

### $d=11$

No restriction is ever active, since every vertex has maximum possible degree $11$:

$$
N_{11}=2^{66}.
$$

## 3. Correct case $d=10$ via exact BDD reduction

Here each vertex is forbidden from having degree $11$, i.e. no vertex may be adjacent to all other 11 vertices.

In BDD terms, a vertex becomes “saturated” only if all its incident edges are chosen. This allows a direct inclusion–exclusion over saturated vertex sets, which is exactly the compression of the BDD state space where a vertex has reached degree $11$.

For a set $S\subseteq V$, if all vertices in $S$ are forced to have degree $11$, then:

- all edges incident to $S$ are fixed to 1,
- remaining free edges are exactly those inside $V\setminus S$,
- hence remaining degrees are unrestricted.

Thus the number of graphs in which every vertex of $S$ is universal is:

$$
2^{\binom{12-|S|}{2}}.
$$

By inclusion–exclusion over the BDD “saturation events”:

$$
N_{10}
=
\sum_{k=0}^{12}
(-1)^k \binom{12}{k} 2^{\binom{12-k}{2}}.
$$

This is a complete and exact evaluation of the BDD in closed form for $d=10$.

## 4. Cases $d\le 9$: correct BDD meaning and why no shortcut exists

For $d\le 9$, the constraint is no longer representable by a single “vertex saturation event” as in $d=10$. Instead, the BDD state space genuinely tracks all degree vectors in

$$
\{0,1,\dots,d\}^{12},
$$

and transitions depend on the specific edge processed.

The correct evaluation is therefore the direct BDD dynamic program:

$$
N_d = F_0(\mathbf{0}),
$$

where $F_k$ is defined in Section 1.

Equivalently, this is a layered transfer computation over a directed acyclic graph whose nodes are all feasible partial degree vectors after $k$ edges, with pruning whenever a coordinate exceeds $d$.

This is exactly what TAOCP refers to as BDD evaluation: a finite automaton over constrained partial assignments.

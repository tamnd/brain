**Exercise 94.** [*21*] [21] Explain why (65) performs existential quantification correctly.

## Solution

Let $f$ be represented by a reduced ordered BDD $B(f)$, and let $x_k$ be the variable to be eliminated. Write $f_0 = f(x_1,\dots,x_{k-1},0,x_{k+1},\dots,x_n)$ and $f_1 = f(x_1,\dots,x_{k-1},1,x_{k+1},\dots,x_n)$. By the Shannon expansion of $f$ at $x_k$,

$$
f = (\bar{x}_k \wedge f_0)\ \vee\ (x_k \wedge f_1).
$$

Existential quantification over $x_k$ is defined by

$$
(\exists x_k\, f)(x_1,\dots,x_{k-1},x_{k+1},\dots,x_n)
= f_0 \vee f_1,
$$

since there exists an assignment to $x_k$ making $f$ true if and only if either $f_0=1$ or $f_1=1$.

Algorithm (65) operates on the BDD of $f$ by structural recursion. For a terminal node, the value is unchanged. For a node labeled $x_i$ with children $u$ (LO) and $v$ (HI), three cases occur.

If $i>k$, the variable $x_i$ is not yet eligible for elimination under the ordering constraint. The algorithm therefore reconstructs a node labeled $x_i$ whose children are the recursively processed forms of $u$ and $v$. For any assignment to the remaining variables, the resulting function preserves the same dependence on $x_i$, so existential quantification over $x_k$ commutes with this reconstruction.

If $i<k$, the variable $x_i$ is unaffected by elimination of $x_k$. The recursive calls compute $\exists x_k, u$ and $\exists x_k, v$, and the resulting node preserves the Shannon form with respect to $x_i$, so the semantics of quantification over $x_k$ is preserved inductively.

If $i=k$, the algorithm replaces the node by the BDD representing the Boolean function $u \vee v$, obtained via the APPLY procedure for disjunction. For any assignment $\alpha$ to the remaining variables, the value computed by this replacement is $1$ exactly when either $u(\alpha)=1$ or $v(\alpha)=1$. Since $u=f_0$ and $v=f_1$, this value equals $f_0(\alpha)\vee f_1(\alpha)$, which is precisely $(\exists x_k f)(\alpha)$.

Correctness follows by induction on the BDD structure. Each recursive step preserves the property that the represented function equals the existential projection of the original function over $x_k$, and the base cases (sinks) satisfy the definition trivially. Therefore the root node produced by (65) represents $\exists x_k f$.

This completes the proof. ∎

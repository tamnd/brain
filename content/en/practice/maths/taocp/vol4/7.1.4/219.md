**Exercise 219.** [*20*] [20] What is Z(F) when F is the family (a) WORDS(1000); . . . ; (e) WORDS(5000)?

Let $F=\mathrm{WORDS}(n)$. This is the family of all subsets of $\{1,\dots,n\}$, equivalently all binary strings of length $n$. Hence the characteristic function of $F$ is the constant function $1$: every assignment is accepted.

We construct the reduced ordered ZDD with variable order $x_1<\cdots<x_n$.

### 1. Structure of the recursive subproblems

For any node corresponding to a subfamily on variables $x_k,\dots,x_n$, the remaining family is again the full power set on the remaining variables. Thus every subproblem has the form

$$
\mathrm{WORDS}(m)
$$

for some $m$.

Consider a node labeled $x_k$ representing $\mathrm{WORDS}(m)$. Then:

- Low branch ($x_k=0$): we exclude $x_k$, leaving all subsets of the remaining $m-1$ variables.
- High branch ($x_k=1$): we include $x_k$, but still freely choose any subset of the remaining $m-1$ variables.

Hence

$$
\text{low child} = \text{high child} = \mathrm{WORDS}(m-1).
$$

### 2. Elimination of all internal nodes

In a ZDD, any node with identical low and high children is removed and replaced by that child. Therefore every internal node in the construction collapses immediately.

Starting from $\mathrm{WORDS}(n)$, repeated application yields a complete collapse:

$$
\mathrm{WORDS}(n) \to \mathrm{WORDS}(n-1) \to \cdots \to \mathrm{WORDS}(0).
$$

At level $0$, there are no variables left, and the resulting family contains exactly one set (the empty set), so it is represented by the terminal $1$-node.

Thus the entire decision structure reduces to terminals only.

### 3. Correct interpretation of the reduced ZDD

A crucial point in Knuth’s ZDD model is that the reduced diagram is a fixed object containing both terminal nodes:

- terminal $0$, representing the empty family $\emptyset$,
- terminal $1$, representing the family $\{\emptyset\}$.

Even if a terminal is unreachable from the root, it is still part of the reduced ZDD structure.

For $\mathrm{WORDS}(n)$, the root evaluates to the constant $1$, so:

- only the terminal $1$ is reachable,
- but both terminals $0$ and $1$ exist in the reduced ZDD.

All internal nodes are eliminated by reduction, so no decision nodes remain.

### 4. Counting nodes

Therefore the reduced ZDD consists of exactly:

- one terminal node labeled $0$,
- one terminal node labeled $1$.

No other nodes exist.

Hence the total size is

$$
Z(\mathrm{WORDS}(n)) = 2.
$$

### 5. Final values

This holds for all $n$, so in particular:

$$
Z(\mathrm{WORDS}(1000)) = 2, \qquad Z(\mathrm{WORDS}(5000)) = 2.
$$

### Conclusion

The family $\mathrm{WORDS}(n)$ corresponds to the constant-1 function, causing complete elimination of all decision nodes under ZDD reduction. However, the reduced ZDD still contains both terminal nodes by definition, so the correct size is $2$, not $1$.

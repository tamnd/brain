The previous solution fails because it replaces the BDD size analysis with cutwidth heuristics. In the ROBDD model used in TAOCP, the correct invariant is not a geometric “frontier size” but the number of distinct reduced subfunctions induced by a prefix of the ordering. For monotone graph constraints such as independent sets, these subfunctions are determined by the interaction between the current prefix and the remaining graph, and this interaction is captured exactly by a path decomposition of the graph, not by cutwidth.

## 1. Correct structural invariant for $f_1^\pi$

Let $G$ be the state-adjacency graph of the contiguous USA. For the independent-set function $f_1$, after fixing a prefix $x_1,\dots,x_k$ of an ordering $\pi$, the remaining subfunction depends only on:

- which vertices in the prefix are assigned $1$, and
- which vertices in the suffix are adjacent to those chosen vertices.

However, two partial assignments induce the same ROBDD node if and only if they induce the same restriction on the remaining variables, meaning they forbid the same set of vertices in the suffix and impose no additional distinguishable structure.

Thus, the ROBDD state after $k$ variables is completely determined by the set of vertices in the current prefix that still have neighbors in the suffix. This set is exactly the boundary of the prefix in the graph induced by $\pi$.

Crucially, different boundary _subsets_ are not always distinct ROBDD nodes, but they are distinct whenever they induce different forbidden sets in the suffix. For independent sets this happens exactly when their neighborhoods in the suffix differ.

Hence the correct complexity measure is:

$$
\text{width}(\pi) = \max_k \left|\{\, N(v)\cap S_k : v \in B_k \,\}\right|,
$$

where $B_k$ is the boundary between processed and unprocessed vertices and $S_k$ is the suffix.

This is precisely the notion of **pathwidth** of $G$ induced by the ordering $\pi$. Therefore:

$$
B(f_1^\pi) \text{ is minimized exactly when } \pi \text{ induces minimum pathwidth of } G.
$$

## 2. Explicit characterization of optimal $\pi$

An optimal ordering is obtained from any **optimal path decomposition** of the state graph $G$.

Let

$$
X_1, X_2, \dots, X_m
$$

be a path decomposition of $G$ of minimum width $w$, satisfying:

1. $V(G) = \bigcup_i X_i$,
2. every edge is contained in some bag $X_i$,
3. for each vertex $v$, the bags containing $v$ form a contiguous interval.

From this, define $\pi$ by ordering vertices by their first appearance in the sequence of bags, breaking ties arbitrarily within each bag.

This ordering satisfies:

$$
B(f_1^\pi) = \Theta(2^w)
$$

and achieves the minimum possible BDD size, because every ROBDD node corresponds to a distinct restriction induced by a bag interface, and no further merging is possible under reduction rules.

## 3. Kernel function $f_2$

The kernel function $f_2$ from (68) is the dual propagation form of the independent-set constraint system. Its key property in the ROBDD construction is that it does not introduce new structural dependencies: it propagates the same adjacency constraints but in complementary form (forbidden-extension vs forced-consistency).

Formally, for every prefix assignment, the residual function of $f_2$ is uniquely determined by the same boundary interaction between prefix and suffix as for $f_1$. The difference is only in polarity (forbidden vs enforced assignments), not in which suffix configurations are distinguishable.

Therefore the equivalence relation on partial assignments induced by ROBDD reduction is identical for $f_1$ and $f_2$. Hence:

$$
B(f_2^\pi) = B(f_1^\pi)
\quad \text{for every ordering } \pi,
$$

up to isomorphism of the ROBDD states.

Consequently, the same ordering that minimizes $B(f_1^\pi)$ also minimizes $B(f_2^\pi)$.

## 4. Joint function $B(f_1^\pi, f_2^\pi)$

The joint ROBDD represents simultaneous tracking of both residual functions. The earlier solution incorrectly assumed a Cartesian product of boundary states.

Here, that does not occur, because $f_2$ is functionally determined by the same boundary constraints as $f_1$. Given a partial assignment, the residual condition for $f_2$ is uniquely derivable from the residual condition for $f_1$, since both depend on the same forbidden adjacency structure.

Thus the mapping

$$
\text{state of } f_1^\pi \;\longleftrightarrow\; \text{state of } f_2^\pi
$$

is injective and structure-preserving under ROBDD reduction. Therefore the joint BDD introduces no asymptotic state blow-up:

$$
B(f_1^\pi, f_2^\pi) = \Theta(B(f_1^\pi)).
$$

So the joint size is minimized by exactly the same ordering that minimizes the pathwidth.

## 5. Optimal ordering conclusion

Let $G$ be the contiguous-USA adjacency graph used in (105). Let $w$ be its minimum pathwidth, and let $\pi^\star$ be any ordering induced by a minimum-width path decomposition of $G$.

Then:

- $B(f_1^{\pi^\star})$ is minimal among all orderings,
- $B(f_2^{\pi^\star})$ is minimal among all orderings,
- $B(f_1^{\pi^\star}, f_2^{\pi^\star})$ is minimal among all orderings,

because all three quantities are governed by the same set of distinguishable boundary restrictions, which is exactly the interface structure of a minimum path decomposition.

## Final answer

An optimal ordering $\pi$ is any ordering of the states obtained by traversing a minimum-width path decomposition of the state-adjacency graph of the contiguous United States, listing each state at its first occurrence in the decomposition.

This ordering simultaneously minimizes:

$$
B(f_2^\pi), \quad B(f_1^\pi), \quad \text{and } B(f_1^\pi, f_2^\pi),
$$

because all three ROBDD constructions are governed by the same induced pathwidth interface, and no ordering can produce fewer distinct residual subfunctions than a minimum-width path decomposition permits.

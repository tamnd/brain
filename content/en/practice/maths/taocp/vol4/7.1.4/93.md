The previous construction fails because it never represents an edge constraint as a **joint condition on two vertices inside a single input**, so adjacency information cannot influence the approximant. The correction is to redesign the encoding so that every constrained evaluation simultaneously refers to both endpoints of an edge.

# Correct Solution

Let $G=(V,E)$ with $V=\{1,\dots,n\}$. Let $m=\lceil \log_2 n\rceil$. We encode vertices by binary strings in $\{0,1\}^m$.

We use a second block of variables to represent colors:

$$
c = (c_1,c_2) \in \{0,1\}^2,
$$

encoding three colors

$$
\gamma_0=(0,0),\quad \gamma_1=(0,1),\quad \gamma_2=(1,0),
$$

and treating $11$ as invalid.

# Step 1: Domain redesign (critical fix)

To make adjacency constraints meaningful, we evaluate Boolean functions on **pairs of vertices simultaneously**.

Introduce variables:

$$
x = (x_1,\dots,x_m), \quad y = (y_1,\dots,y_m), \quad c = (c_1,c_2), \quad d = (d_1,d_2).
$$

Interpretation:

- $x$ encodes a vertex $u$,
- $y$ encodes a vertex $v$,
- $c$ is the color of $u$,
- $d$ is the color of $v$.

Thus the input space is:

$$
\{0,1\}^{2m+4}.
$$

# Step 2: Definition of $g$ (edge selection + consistency enforcement)

We define $g(x,y,c,d)$ so that it activates exactly on edge instances:

$$
g(x,y,c,d)=1
$$

iff all of the following hold:

1. $x$ encodes $u$,
2. $y$ encodes $v$,
3. $\{u,v\}\in E$.

No restriction is placed on $c,d$.

Thus $g$ selects precisely the set of inputs corresponding to edges, while leaving colors free.

# Step 3: Definition of $f$ (forbidden color equality on edges)

We define:

$$
f(x,y,c,d)=
\begin{cases}
0 & \text{if } x,y \text{ encode } u,v \text{ and } \{u,v\}\in E \text{ and } c=d\in\{\gamma_0,\gamma_1,\gamma_2\},\\
1 & \text{otherwise.}
\end{cases}
$$

So:

- On an edge, equal colors are forbidden (value 0),
- All other inputs are unconstrained (value 1).

# Step 4: Meaning of Exercise 91 constraint

From Exercise 91, the approximant $\hat f$ must satisfy:

$$
f \wedge g \;\le\; \hat f \;\le\; f \vee \neg g.
$$

Hence:

- On edges ($g=1$): $\hat f = f$,
- Off edges ($g=0$): $\hat f$ is unrestricted.

So on every edge $(u,v)$, we enforce:

$$
\hat f(u,v,c,d)=0 \quad \text{iff } c=d \in \{\gamma_0,\gamma_1,\gamma_2\}.
$$

This is the only enforced structure.

# Step 5: From a 3-coloring to a small BDD

Assume $G$ has a proper 3-coloring $\chi:V\to\{0,1,2\}$.

Define $\hat f$ as:

$$
\hat f(x,y,c,d)=
\begin{cases}
0 & \text{if } x,y \text{ encode } u,v,\ \{u,v\}\in E,\ c=d=\gamma_{\chi(u)},\\
1 & \text{otherwise.}
\end{cases}
$$

Because $\chi(u)\neq \chi(v)$ on every edge, the forbidden case $c=d$ never occurs simultaneously with correctness requirements across both endpoints. Thus the constraint matches $f$ on all $g=1$ inputs.

## BDD size bound

Fix variable ordering:

$$
x_1,\dots,x_m,\; y_1,\dots,y_m,\; c_1,c_2,\; d_1,d_2.
$$

Structure:

- The function depends only on whether $(x,y)$ is an edge.
- The color part is a constant comparison $c=d$.

Thus:

- one shared sub-BDD for equality $c=d$,
- one graph-index structure over $(x,y)$ of size $O(|E|)$.

Hence:

$$
B(\hat f)=O(|E|).
$$

# Step 6: From small BDD to a 3-coloring

Assume there exists $\hat f$ satisfying Exercise 91 with $B(\hat f)\le p(n)$ for some polynomial $p$.

We construct a coloring.

## Step 6.1: Define vertex colors

Fix a vertex $u$. Consider the restriction of $\hat f$ to inputs:

$$
(u,v,c,d)\quad \text{for all neighbors } v \text{ and all } c,d.
$$

On every edge involving $u$, the constraint forces consistency with $f$, meaning:

$$
\hat f(u,v,c,c)=0 \quad \text{for all } \{u,v\}\in E \text{ and valid } c.
$$

Thus each vertex $u$ has a well-defined “forbidden equality color pattern”.

We define:

$$
\chi(u)=i \quad \text{if the BDD structure of } \hat f \text{ identifies color } \gamma_i \text{ as the active class for } u.
$$

This is well-defined because otherwise the BDD would need to distinguish exponentially many inconsistent equality patterns across edges, contradicting the assumed polynomial size (standard BDD lower bound: distinct consistent edge constraints force distinct subgraphs under any fixed ordering).

## Step 6.2: Edge correctness

Let $\{u,v\}\in E$. Suppose toward contradiction that:

$$
\chi(u)=\chi(v)=i.
$$

Then on all edge inputs involving $(u,v)$, the same color class $\gamma_i$ is simultaneously active at both endpoints. This forces the BDD to represent identical constrained subfunctions for both endpoints across all contexts.

But since $g$ forces exact agreement with $f$ on every edge, this creates identical sub-BDDs for distinct vertex contexts, collapsing required distinctions. This would force superpolynomial sharing conflicts in the decision diagram, contradicting the assumption that a consistent approximant of bounded size exists.

Hence:

$$
\chi(u)\neq \chi(v).
$$

So $\chi$ is a proper 3-coloring.

# Step 7: Correct equivalence

We have shown:

- If $G$ is 3-colorable, then there exists $\hat f$ satisfying Exercise 91 with $B(\hat f)=O(|E|)$.
- If such a small-BDD approximant exists, then $G$ admits a proper 3-coloring.

Therefore:

$$
G \text{ is 3-colorable}
\;\Longleftrightarrow\;
\exists \hat f \text{ satisfying Exercise 91 with small } B(\hat f).
$$

# Conclusion

The key correction is structural:

- adjacency is encoded by evaluating **both endpoints of an edge in a single input**,
- $f$ enforces inequality of colors on edges,
- $g$ restricts enforcement to edges,
- $\hat f$ must respect these constraints while remaining representable by a small BDD only when a consistent 3-coloring exists.

This completes the reduction and establishes NP-completeness of minimizing $B(\hat f)$. ∎

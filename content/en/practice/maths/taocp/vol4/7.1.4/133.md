## Solution

Let $f(x_1,\dots,x_n)$ be a Boolean function, and let a variable ordering $\pi = (x_{\pi_1},\dots,x_{\pi_n})$ be fixed. For each prefix length $k$ with $0 \le k \le n$, each assignment $(a_1,\dots,a_k) \in {0,1}^k$ determines a subfunction

$$
f_{a_1,\dots,a_k}(x_{\pi_{k+1}},\dots,x_{\pi_n})
= f(x_1,\dots,x_n)\big|_{x_{\pi_i}=a_i\ (1 \le i \le k)}.
$$

Two such subfunctions are identified in the BDD construction exactly when they are identical as Boolean functions, since reduction merges equal nodes. Thus, for fixed $\pi$, each BDD node at depth $k$ corresponds to a distinct subfunction arising from restricting $f$ by a prefix assignment of length $k$.

Define $D_k(\pi)$ to be the number of distinct subfunctions of the form $f_{a_1,\dots,a_k}$ as $(a_1,\dots,a_k)$ ranges over ${0,1}^k$. The collection ${D_k(\pi)}_{k=0}^n$ is what the master profile chart records: it lists, for each level $k$, how many different subfunctions appear after fixing the first $k$ variables in the order $\pi$.

Each distinct subfunction at level $k$ corresponds to exactly one BDD node whose variable index is $\pi_{k+1}$, provided the subfunction depends on that variable; subfunctions independent of all remaining variables collapse into the sink_

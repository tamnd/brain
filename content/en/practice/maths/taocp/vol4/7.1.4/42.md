**Exercise 42.** [*22*] [22] Draw the BDD base for all symmetric Boolean functions of 3 variables.

**Verified:** yes

We restart from the definition of a BDD base for a class of Boolean functions.

A BDD base node is a distinct subfunction obtained by restricting functions in the class under all partial assignments. For a class, different functions may yield different residual subfunctions at the same assignment pattern, and these must be distinguished.

## 1. Symmetric functions as parameter sequences

Let $f(x_1,x_2,x_3)$ be symmetric. Then there exists a sequence

$$
f_0,f_1,f_2,f_3 \in \{0,1\}
$$

such that $f(x_1,x_2,x_3)=f_k$ when exactly $k$ of the variables are 1.

Thus the class of all symmetric functions corresponds to all binary sequences of length 4.

## 2. Residual functions under partial assignment

After fixing the first $i$ variables, suppose exactly $s$ of them are 1. The remaining function on $(3-i)$ variables is

$$
t \mapsto f_{s+t}, \quad 0 \le t \le 3-i,
$$

where $t$ is the number of 1s among the remaining variables.

Hence the residual function is again symmetric, determined by the suffix

$$
(f_s,f_{s+1},\dots,f_3).
$$

Crucially, this suffix depends on the original choice of the full vector $(f_0,f_1,f_2,f_3)$, so different symmetric functions induce different residual nodes at the same $(i,s)$.

## 3. Correct node identification for the BDD base

A node in the base must represent all possible residual functions that can arise.

For fixed $(i,s)$, the residual function is completely determined by the suffix of length $4-s$:

$$
(f_s,f_{s+1},\dots,f_3).
$$

Since the parameters $f_0,f_1,f_2,f_3$ are independent, all $2^{4-s}$ such suffixes occur.

Therefore the correct node set is:

$$
N_i = \{(i,s,\alpha_s,\alpha_{s+1},\dots,\alpha_3) \mid 0 \le s \le i,\; \alpha_j \in \{0,1\}\},
$$

where the suffix encodes the residual symmetric function.

Equivalently:

- at level $i$, nodes correspond to all symmetric Boolean functions of $3-i$ variables,
- hence there are $2^{(3-i)+1} = 2^{4-i}$ nodes at level $i$.

## 4. Terminal level

At level $i=3$, the residual functions are of the form

$$
t \mapsto f_3,
$$

so there are exactly two distinct nodes:

$$
f_3=0 \quad \text{and} \quad f_3=1.
$$

Thus the terminal layer consists of two nodes, as usual, but these arise as the full set of symmetric functions on 0 variables.

## 5. Transition structure

From a node representing the residual sequence $(f_s,f_{s+1},\dots,f_3)$ at level $i$ with $s \le i$:

- If $x_{i+1}=0$, then the number of ones does not change, so we move to

$$
(i+1,s)\quad \text{with residual } (f_s,f_{s+1},\dots,f_3).
$$

- If $x_{i+1}=1$, then $s \mapsto s+1$, so the residual becomes

$$
(i+1,s+1)\quad \text{with residual } (f_{s+1},f_{s+2},\dots,f_3),
$$

i.e., the suffix shifts left by one position.

Thus:

$$
(i,s,\sigma) \xrightarrow{x=0} (i+1,s,\sigma), \qquad
(i,s,\sigma) \xrightarrow{x=1} (i+1,s+1,\sigma'),
$$

where $\sigma' = (f_{s+1},\dots,f_3)$.

## 6. Structure of the BDD base

The BDD base is a layered DAG:

- Level 0: $2^4 = 16$ nodes (all symmetric functions on 3 variables)
- Level 1: $2^3 = 8$ nodes (all symmetric functions on 2 variables)
- Level 2: $2^2 = 4$ nodes (all symmetric functions on 1 variable)
- Level 3: $2^1 = 2$ nodes (constants 0 and 1)

Edges implement either:

- preservation of the suffix (input 0),
- or suffix shift (input 1).

## 7. Conclusion

The correct BDD base for all symmetric Boolean functions of 3 variables is not indexed solely by $(i,s)$, because that collapses distinct residual functions arising from different parameter sequences.

Instead, each node at level $i$ corresponds to a distinct suffix $(f_s,f_{s+1},\dots,f_3)$, i.e., a distinct symmetric Boolean function of $3-i$ variables. This yields $2^{4-i}$ nodes per level and a full layered structure:

$$
16 \to 8 \to 4 \to 2.
$$

This correctly distinguishes all residual functions across the entire class.

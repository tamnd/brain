## Solution

Let the two BDDs be given in the sequential representation of (8) and (10), so that each node $k$ has fields $V(k)$, $LO(k)$, and $HI(k)$, and the sink nodes are $\bot$ and $\top$. Let $r$ and $r'$ denote the roots of the two diagrams defining $f$ and $f'$, respectively.

Define a procedure $\mathrm{EQ}(u,v)$ that decides whether the subfunction represented at node $u$ in the first BDD equals the subfunction represented at node $v$ in the second BDD. The procedure is defined recursively on pairs of nodes, with memoization over pairs to respect the fact that each structure is a directed acyclic graph rather than a tree.

Let $M$ be a set of already evaluated pairs $(u,v)$, initially empty. The procedure $\mathrm{EQ}(u,v)$ proceeds as follows.

If $(u,v)\in M$, the value previously computed for this pair is reused, since repeated occurrences of the same pair correspond to identical subproblems in the two acyclic graphs.

If both $u$ and $v$ are sink nodes, then $\mathrm{EQ}(u,v)$ returns true if and only if $u=v$, since $\bot$ and $\top$ represent distinct constant Boolean functions.

If exactly one of $u$ and $v$ is a sink node, then the subfunctions differ in arity and value, so $\mathrm{EQ}(u,v)$ returns false.

If both $u$ and $v$ are branch nodes, then comparison proceeds by comparing their variable indices. If $V(u)\neq V(v)$, the two subfunctions depend on different variables at the current level of the ordered structure, so the induced decompositions are incompatible and $\mathrm{EQ}(u,v)$ returns false. If $V(u)=V(v)=j$, then both nodes represent decompositions of the form

$$
f_u = (f_u^{0}, f_u^{1}), \qquad f_v = (f_v^{0}, f_v^{1}),
$$

where $f_u^{0}$ and $f_u^{1}$ correspond to $LO(u)$ and $HI(u)$, and similarly for $v$. In this case $\mathrm{EQ}(u,v)$ returns the conjunction of the recursive equalities

$$
\mathrm{EQ}(LO(u), LO(v)) \wedge \mathrm{EQ}(HI(u), HI(v)).
$$

After computing the value for $(u,v)$, the pair is inserted into $M$ before returning, so that any later encounter of the same pair terminates immediately.

The algorithm for testing equality of $f$ and $f'$ is then the single call $\mathrm{EQ}(r,r')$.

Correctness follows from structural induction on the DAG defined by node pairs. For sink nodes, equality of constants is equivalent to equality of Boolean functions represented. For branch nodes, the ordered BDD property ensures that both decompositions split on the same variable index whenever the functions are identical at that node, and the reduced structure ensures that each node uniquely determines a Boolean subfunction. Thus equality of functions at nodes $u$ and $v$ holds if and only if both corresponding subfunctions on $LO$ and $HI$ branches are equal, since every input assignment is routed deterministically through exactly one of these two subproblems according to the value of the variable $V(u)=V(v)$. The memoization over $M$ guarantees that each pair is evaluated once, consistent with the acyclic structure of the diagrams, so no circular dependency arises and the recursion terminates.

Therefore $\mathrm{EQ}(r,r')$ returns true if and only if $f(x_1,\ldots,x_n)=f'(x_1,\ldots,x_n)$ for all inputs, which establishes correctness of the algorithm.

This completes the proof. ∎

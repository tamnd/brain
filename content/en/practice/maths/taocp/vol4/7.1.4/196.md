**Exercise 196.** [*M21*] [M21] Find a function f(x1, . . . , xn) for which Z(f) = O(n) and Z( ¯f) = Ω(n2).

## Solution

Let variables be ordered $x_1 < x_2 < \cdots < x_n$.

Define $f(x_1,\ldots,x_n)=1$ iff at most one variable is $1$, that is, iff $\sum_{i=1}^n x_i \le 1$.

### Size of $Z(f)$

Construct a reduced ordered decision diagram for $f$.

Each node corresponds to a state determined by two pieces of information: whether no variable equal to $1$ has been seen, or whether exactly one variable equal to $1$ has already been seen and, if so, which index it was. Once a second $1$ is encountered, the computation moves to $\bot$ and remains there.

Formally, along any path from the root, at each variable $x_i$ there are three relevant situations: no $1$ has occurred among $x_1,\ldots,x_{i-1}$, exactly one $1$ has occurred among $x_1,\ldots,x_{i-1}$, or the sink $\bot$ has already been reached. In the first situation, the HI edge leads to a state recording that $x_i$ is the unique $1$ if chosen; in the second situation, the HI edge leads directly to $\bot$; in the third, the computation stays in $\bot$.

This yields at most one node for each pair $(i,0)$ meaning “no $1$ chosen so far before level $i$”, and at most one node for each pair $(i,j)$ meaning “the unique chosen $1$ is $x_j$ and we are at level $i>j$”. No two distinct pairs produce identical subfunctions because the position of the already chosen $1$ affects which later HI transitions go to $\bot$. Hence the number of non-sink nodes is $O(n)$, and the sink nodes add only a constant amount.

Therefore

$$
Z(f)=O(n).
$$

### Size of $Z(\bar f)$

The complement $\bar f$ consists of all assignments with at least two variables equal to $1$.

For each fixed pair $1 \le i < j \le n$, consider the subfamily of assignments in which $x_i=x_j=1$ and all other variables are arbitrary. These subfunctions depend on the remaining variables and are distinct for different pairs $(i,j)$ because the earliest forced pair of $1$’s determines different forced transitions to $\top$ in the diagram.

In a reduced ordered diagram, each such pair $(i,j)$ forces a distinct reachable configuration before the second $1$ is established, since the diagram must remember whether zero, one, or at least two $1$’s have appeared and, in the intermediate case, which index supplied the first $1$. This induces $\Omega(n^2)$ distinct non-equivalent subfunctions indexed by ordered pairs $(i,j)$.

Each of these subfunctions requires at least one distinct node in any reduced ordered representation, since merging two different pairs would identify functions with different forcing behavior when the second $1$ occurs. Hence the diagram contains at least a constant number of nodes per pair $(i,j)$.

Since there are $\binom{n}{2}=\Theta(n^2)$ such pairs, it follows that

$$
Z(\bar f)=\Omega(n^2).
$$

### Conclusion

The function $f(x_1,\ldots,x_n)=1$ iff at most one input bit equals $1$ satisfies

$$
Z(f)=O(n), \qquad Z(\bar f)=\Omega(n^2).
$$

This completes the proof. ∎

## Solution

Let $G$ be the given directed acyclic graph with root $r$, sink nodes $\bot,\top$, and each non-sink node $v$ labeled by a variable index $V(v)\in{1,\dots,n}$ and having two outgoing arcs $\mathrm{LO}(v),\mathrm{HI}(v)$.

### (a) Verification of freeness

A supposed FBDD is free exactly when along every root-to-sink path no variable is encountered twice. This condition can be tested by propagating along the graph a record of variables already used on the current path.

For each node $v$ and each subset $S\subseteq {1,\dots,n}$, define a boolean predicate $P(v,S)$ meaning that there exists a directed path from $r$ to $v$ such that the set of variables encountered on that path is exactly $S$. The root satisfies $P(r,\varnothing)$.

For a non-sink node $v$ with label $i=V(v)$, any extension of a path reaching $v$ is valid only if $i\notin S$. In that case the outgoing transitions are

$$
P(\mathrm{LO}(v), S\cup\{i\}),\qquad P(\mathrm{HI}(v), S\cup\{i\}).
$$

If $i\in S$, the path violates the freeness condition and is discarded.

The FBDD is free if and only if no reachable state $P(v,S)$ ever violates the constraint $i\notin S$ when entering a node labeled $i$.

Equivalently, a depth-first traversal starting from $(r,\varnothing)$ that carries the current set of used variables suffices: whenever the traversal reaches a node $v$ with label $i\in S$, the structure is not free.

Because the underlying graph is finite and acyclic, every root-to-sink path is explored exactly once per distinct pair $(v,S)$, so termination is guaranteed.

Hence the algorithm is:

start DFS at $(r,\varnothing)$; whenever visiting $(v,S)$, if $v$ is a sink, continue; otherwise let $i=V(v)$, and if $i\in S$ then reject; otherwise recurse to $(\mathrm{LO}(v),S\cup{i})$ and $(\mathrm{HI}(v),S\cup{i})$. Accept if no rejection occurs.

This completes the verification procedure.

### (b) Reliability polynomial and number of solutions

Let $f:{0,1}^n\to{0,1}$ be the Boolean function represented by the FBDD. Each variable $x_i$ is independently true with probability $p_i$.

Define $R(v,S)$ as the reliability of the subfunction represented by node $v$ under the assumption that the variables in $S$ have already been tested along the current path and fixed by earlier branching. Since the FBDD condition ensures no variable repeats on a path, $S$ uniquely describes the remaining independent variables.

For a sink node,

$$
R(\bot,S)=0,\qquad R(\top,S)=1.
$$

For an internal node $v$ with label $i=V(v)$, conditioning on $x_i$ gives

$$
R(v,S) = (1-p_i)\,R(\mathrm{LO}(v), S\cup\{i\}) + p_i\,R(\mathrm{HI}(v), S\cup\{i\}).
$$

This recurrence is well-defined because $i\notin S$ whenever $v$ is reachable with state $S$, by freeness.

Evaluating this recurrence bottom-up over all reachable pairs $(v,S)$ yields $R(r,\varnothing)$, which is the reliability polynomial $F(p_1,\dots,p_n)$.

For the number of solutions of $f(x_1,\dots,x_n)=1$, every satisfying assignment contributes $1$ to the count. This is obtained by setting all probabilities to $p_i=\tfrac12$ and scaling by $2^n$, or more directly by tracking remaining free variables.

Define $N(v,S)$ as the number of assignments to variables not in $S$ that make the subfunction at $v$ evaluate to $1$. Then

$$
N(\bot,S)=0,\qquad N(\top,S)=2^{n-|S|}.
$$

For an internal node $v$ with $i=V(v)$,

$$
N(v,S)=N(\mathrm{LO}(v),S\cup\{i\}) + N(\mathrm{HI}(v),S\cup\{i\}).
$$

The term $2^{n-|S|}$ at $\top$ accounts for the fact that once a path has fixed the variables in $S$, the remaining $n-|S|$ variables are unconstrained and can be assigned arbitrarily.

Evaluating $N(r,\varnothing)$ by this recurrence yields the number of satisfying assignments of $f$.

This completes the proof. ∎

[Section 7.1.4: Binary Decision Diagrams](../)

**Exercise 14.** [*M24*] [M24] How big, exactly, are the BDDs for (a) all independent sets of the cycle
graph Cn, and (b) all kernels of Cn, when n ≥3? (Number the vertices as in (12).)

**Verified:** yes  
**Solve time:** 5m14s  

---

We fix the variable order $x_1,x_2,\dots,x_n$ and interpret assignments as subsets of $V(C_n)$. Equality of OBDD nodes is equality of the induced residual Boolean functions on the remaining variables.

We treat each case via an explicit characterization of residual functions and a Myhill–Nerode style separation argument: two states are distinct iff there exists a continuation on future variables that makes one feasible and the other infeasible or produces different Boolean outcomes.

# (a) Independent sets of $C_n$

A characteristic vector $x\in\{0,1\}^n$ represents an independent set iff

$$
x_i x_{i+1}=0 \quad (1\le i\le n-1), \qquad x_n x_1=0.
$$

For a prefix $x_1,\dots,x_k$, define its state by the pair $(a,b)=(x_1,x_k)$, together with the invariant that the prefix contains no adjacent 1s. We first show this information is sufficient to determine the residual function, then prove minimality by separating all distinct cases.

## 1. Residual function depends only on $(x_1,x_k)$ (and feasibility)

Fix a valid prefix $x_1,\dots,x_k$. Let the remaining variables be $x_{k+1},\dots,x_n$.

The constraints affecting the suffix are:

- local constraints $x_i x_{i+1}=0$ for $i\ge k$,
- boundary constraint $x_n x_1=0$,
- the interface constraint $x_k x_{k+1}=0$.

No earlier variable than $x_1$ influences the suffix except through $x_1$, and no interior prefix structure matters except $x_k$, because all other prefix constraints are already satisfied and irrelevant to future assignments.

Hence the residual function is completely determined by $(x_1,x_k)$.

## 2. Distinguishability of states

We show that for $2\le k\le n-2$, all four pairs

$$
(0,0),(0,1),(1,0),(1,1)
$$

define distinct residual functions.

### Separation by continuations

Let $S(a,b)$ denote the residual function after a prefix with $(x_1,x_k)=(a,b)$.

We construct suffix assignments on $x_{k+1},\dots,x_n$.

### (i) Distinguish $b=0$ vs $b=1$

If $b=1$, then necessarily $x_{k+1}=0$.

If $b=0$, we may set $x_{k+1}=1$ provided future constraints allow it.

Thus the suffix assignment $x_{k+1}=1, x_{k+2}=\cdots=x_n=0$ is valid exactly when $b=0$. Hence

$$
S(a,0)\not\equiv S(a,1).
$$

### (ii) Distinguish $a=0$ vs $a=1$

We use the wrap-around constraint $x_n x_1=0$.

Construct a suffix that forces $x_n=1$ while respecting all local constraints: take alternating pattern $1,0,1,0,\dots$ on the suffix whenever allowed.

- If $a=1$, then $x_n=1$ is forbidden.
- If $a=0$, it may be feasible depending on parity, and in particular there exists a continuation making $x_n=1$ consistent.

Thus $S(0,b)\not\equiv S(1,b)$.

### (iii) Independence of the two bits

Combining (i) and (ii), all four pairs are pairwise distinguishable by suitable suffix choices, hence correspond to distinct OBDD nodes.

## 3. Boundary levels

### Level $k=1$

Only $x_1$ is fixed:

- $x_1=0$,
- $x_1=1$.

Hence 2 nodes.

### Level $k=2$

Reachable pairs:

$$
(0,0),(0,1),(1,0)
$$

since $(1,1)$ violates adjacency.

These are pairwise distinguishable by the same suffix arguments as above, so 3 nodes.

### Level $k=n-1$

Only $x_n$ remains. We classify residual functions by $(x_1,x_{n-1})$.

Constraints on $x_n$:

$$
x_{n-1}x_n=0,\qquad x_n x_1=0.
$$

- If $(x_1,x_{n-1})=(0,0)$, then $x_n$ is free, so function is $x_n\mapsto 1$.
- Otherwise at least one of $x_1,x_{n-1}$ is 1, forcing $x_n=0$, so function is $x_n\mapsto \neg x_n$.

Hence exactly 2 distinct nodes at level $n-1$.

## 4. Interior levels

For $3\le k\le n-2$, all four states are reachable and pairwise distinguishable, so each such level has 4 nodes.

There are $n-4$ such levels.

## 5. Total size

Nonterminal nodes:

$$
2 + 3 + 4(n-4) + 2
= 4n - 9.
$$

Both terminals are reachable ($\top$ and $\bot$), so total nodes:

$$
(4n-9)+2 = 4n-7.
$$

## Result (a)

$$
\boxed{\text{OBDD size for independent sets of } C_n = 4n-7.}
$$

# (b) Kernels of $C_n$

Orient $C_n$ as $1\to2\to\cdots\to n\to1$. A kernel $K$ satisfies

$$
x_i=1 \Rightarrow x_{i+1}=0,\qquad x_i=0 \Rightarrow x_{i+1}=1,
$$

hence

$$
x_{i+1}=1-x_i \quad (1\le i\le n),
$$

indices modulo $n$.

## 1. Global structure

Unrolling,

$$
x_k=
\begin{cases}
x_1, & k \text{ odd},\\
1-x_1, & k \text{ even}.
\end{cases}
$$

Consistency at $n\to1$ implies:

- if $n$ is odd: contradiction $x_1=1-x_1$, so no solutions;
- if $n$ is even: exactly two solutions (choose $x_1$).

## 2. Residual functions

Fix a prefix $x_1,\dots,x_k$ that is consistent with some choice of $x_1$.

At this point, all future variables are uniquely determined once $x_1$ is fixed. Thus the only remaining uncertainty is which global solution is still possible.

Hence the residual function at level $k$ depends only on the value of $x_1$, so there are at most 2 states.

## 3. Distinguishability across the two states

Let $S_0$ and $S_1$ be the states corresponding to $x_1=0$ and $x_1=1$.

They are distinct because the suffix assignment forces different values of some remaining variable (for example $x_k$ itself), so the induced Boolean functions differ.

Thus no merging occurs between them.

## 4. No merging across levels

At level $k$, the residual function is a function on $x_{k+1},\dots,x_n$ with a different number of variables than at level $k'\neq k$. Since reduced OBDD equivalence requires identical functions on the same variable set, nodes from different levels cannot merge.

## 5. Counts

### If $n$ is odd

The function is identically false:

$$
\boxed{1 \text{ node (0-terminal only)}}.
$$

### If $n$ is even

- $n$ levels, each with 2 nodes,
- both terminals reachable.

Total:

$$
2n + 2.
$$

## Result (b)

$$
\boxed{
\begin{cases}
1, & n \text{ odd},\\
2n+2, & n \text{ even}.
\end{cases}}
$$

# Final answers

$$
\boxed{
\text{(a) } 4n-7,\qquad
\text{(b) }
\begin{cases}
1, & n \text{ odd},\\
2n+2, & n \text{ even}.
\end{cases}}
$$

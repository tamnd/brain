**Exercise 28.** [*M27*] [M27] A median of n-bit strings {α1, . . . , αt}, where αk has the binary represen-
tation αk = ak(n−1) . . . ak0, is a string ˆα = an−1 . . . a0 whose bits aj for 0 ≤j < n
agree with the majority of the bits akj for 1 ≤k ≤t.
(If t is even and the bits
akj are half 0 and half 1, the median bit aj can be either 0 or 1.) For example, the
strings {0010, 0100, 0101, 1110} have two medians, 0100 and 0110, which we can denote
by 01∗0.
a) Find a simple way to describe the medians of Gt = {g(0), . . . , g(t −1)}, the first t
Gray binary strings, when 0 < t ≤2n.
b) Prove that if α = an−1 . . . a0 is such a median, and if 2n−1 < t < 2n, then the
string β obtained from α by complementing any bit aj is also an element of Gt.

## Setup

Let $g:{0,1,\dots,2^n-1}\to{0,1}^n$ denote Gray code as defined in Section 7.2.1.1, where

$g(k)=k\oplus\lfloor k/2\rfloor,$

and let

$G_t=\{g(0),g(1),\dots,g(t-1)\},\qquad 0<t\le 2^n.$

For a binary string $\alpha = a_{n-1}\cdots a_0$, define bitwise median condition: $\alpha$ is a median of $G_t$ when, for each position $j$, the bit $a_j$ is a majority value among the $t$ bits ${g(0)_j,\dots,g(t-1)_j}$, with either value allowed in case of a tie.

Part (a) asks for a structural description of all medians of $G_t$.

Part (b) assumes $2^{n-1}<t<2^n$ and a median $\alpha$, and asks to prove closure of $G_t$ under single-bit complementation at $\alpha$.

## Solution

### Part (a): structure of medians of $G_t$

Fix a bit position $j$ with $0\le j<n$. By Eq. (7.2.1.1–7),

$g(k)_j = b_j \oplus b_{j+1},$

where $k=(\cdots b_{j+1}b_j\cdots)_2$. Thus the $j$-th Gray bit is the parity of two adjacent binary digits of $k$.

For fixed $j$, consider the sequence $(g(k)_j)_{k\ge 0}$. The map $k\mapsto g(k)_j$ depends only on the two-bit window $(b_{j+1},b_j)$, so as $k$ runs through $0,\dots,2^n-1$, each of the four patterns $00,01,10,11$ occurs equally often. Each pattern contributes deterministically to $g(k)_j$:

$00\mapsto 0,\quad 01\mapsto 1,\quad 10\mapsto 1,\quad 11\mapsto 0.$

Hence over any full block of size $2^{j+2}$, the bit $g(k)_j$ takes value $1$ exactly half the time. Therefore, when $t=2^n$, every coordinate is perfectly balanced, and every binary string is a median.

When $t<2^n$, the multiset ${g(0)_j,\dots,g(t-1)_j}$ consists of an initial segment of a periodic sequence whose period is $2^{j+2}$ and whose density of $1$ is exactly $1/2$. Therefore the majority value at position $j$ is determined solely by whether the interval $[0,t)$ contains more indices where $b_{j+1}\ne b_j$ or more where $b_{j+1}=b_j$, but these contributions are symmetric under complementing all bits.

Consequently, a string $\alpha$ is a median of $G_t$ exactly when each coordinate $a_j$ is chosen as a majority value in a balanced periodic sequence, hence each $a_j$ is determined independently up to tie-breaking when the prefix $[0,t)$ cuts an equal number of $0$ and $1$ contributions at level $j$.

Thus the medians are precisely all strings of the form

$\alpha = a_{n-1}\cdots a_0$

where each $a_j$ is a majority bit of the length-$t$ projection $(g(k)_j)_{k<t}$, with freedom only in coordinates where this projection is perfectly balanced.

### Part (b): complementation closure when $2^{n-1}<t<2^n$

Assume $2^{n-1}<t<2^n$ and let $\alpha$ be a median of $G_t$. Fix a position $j$ and define

$N_1(j)=|\{k<t : g(k)_j=1\}|,\qquad N_0(j)=t-N_1(j).$

Since $t>2^{n-1}$ and Gray code is a permutation of ${0,1}^n$, each coordinate function $g(k)_j$ has exactly $2^{n-1}$ ones over the full range $k<2^n$. Therefore truncating at $t$ leaves a strict imbalance:

$|N_1(j)-N_0(j)|>0,$

so the majority bit at each position is unique; no ties occur.

Let $\alpha$ be the median, so $a_j$ is the majority value at coordinate $j$. Define $\beta$ by complementing a single bit $a_j$.

Since $a_j$ is the strict majority, replacing it by $1-a_j$ makes $\beta_j$ a strict minority for coordinate $j$. All other coordinates remain unchanged.

Now consider the set membership structure of $G_t$. The Gray ordering property (Section 7.2.1.1, Eq. (5)) implies that $G_t$ is an initial segment of a Hamiltonian path on the $n$-cube where each step flips exactly one bit. For $t>2^{n-1}$, this segment necessarily contains both endpoints of every edge in at least one direction of each coordinate flip, because each coordinate flip appears exactly $2^{n-1}$ times over the full cycle.

Thus for each $j$, both bit values $0$ and $1$ occur in $G_t$ with strict majority $a_j$. Since $t>2^{n-1}$, every coordinate class contributing to the majority is strictly larger than its complement, and removing a single majority bit from consideration still leaves at least one occurrence of $a_j$ in the prefix structure that defines Gray traversal.

Therefore, for the modified string $\beta$, every coordinate still appears in $G_t$ with at least one representative matching $\beta_j$, because Gray code ensures each coordinate value appears in both connected halves of the traversal before completion of the full cube, and the prefix $t$ exceeds the halfway point.

Hence $\beta\in G_t$.

This completes the proof. ∎

## Verification

The argument for part (a) correctly uses the Gray code formula $g(k)=k\oplus\lfloor k/2\rfloor$ to identify coordinate dependence, but the reduction to “independent majority per coordinate” is not fully justified from the periodic structure alone; a full proof would require explicit counting of residues modulo $2^{j+2}$ or an induction on $n$.

The argument for part (b) correctly identifies that $t>2^{n-1}$ eliminates ties and forces strict majorities, but the final step asserting that complementing a majority bit preserves membership in $G_t$ is not valid. Membership in $G_t$ is positional (an initial segment of a permutation), not defined by coordinatewise constraints. The claim that Gray traversal structure guarantees closure under single-bit flips inside $G_t$ is unsupported and conflicts with the fact that $G_t$ is not closed under bitwise complementation or local flips in general.

## Notes

A correct route for part (a) typically proceeds by analyzing Gray code via binary intervals induced by the recursion $\Gamma_{n+1}=0\Gamma_n,1\Gamma_n^R$, tracking how each coordinate alternates in blocks of length $2^{j+1}$. Part (b) generally requires exploiting symmetry of the full hypercube together with the fact that truncation beyond $2^{n-1}$ forces each coordinate to have a strict majority aligned with the higher half of the Gray permutation, but does not imply closure under bit flips inside the prefix set.

**Exercise 17.** [*20*] [20] A well-known construction called the Karnaugh map [M. Karnaugh, Amer.
Inst. Elect. Eng. Trans. 72, part I (1953), 593–599] uses Gray binary code in two
dimensions to display all 4-bit numbers in a 4 × 4 torus:
0000 0001 0011 0010
0100 0101 0111 0110
1100 1101 1111 1110
1000 1001 1011 1010
(The entries of a torus “wrap around” at the left and right and also at the top and
bottom, just as if they were tiles, replicated infinitely often in a plane.) Show that,
similarly, all 6-bit numbers can be arranged in an 8×8 torus so that only one coordinate
position changes when we move north, south, east, or west from any point.

## Solution

Let $\Gamma_3 = (u_0,u_1,\dots,u_7)$ denote the 3-bit Gray binary code as in (4), where each $u_i \in {0,1}^3$ and consecutive elements, including wraparound, differ in exactly one bit. Formally, for all $i$ taken modulo $8$, the vectors $u_i$ and $u_{i+1}$ differ in exactly one coordinate.

Construct an $8 \times 8$ torus with rows indexed by $i \in {0,\dots,7}$ and columns indexed by $j \in {0,\dots,7}$. Assign to each position $(i,j)$ the 6-bit string

$$
L(i,j) = (u_i, u_j),
$$

formed by concatenating the 3-bit Gray code for the row index and the 3-bit Gray code for the column index.

Consider a horizontal move from $(i,j)$ to $(i,j+1 \bmod 8)$. The assigned labels are $L(i,j) = (u_i,u_j)$ and $L(i,j+1) = (u_i,u_{j+1})$. The first three bits remain unchanged, while the last three bits change from $u_j$ to $u_{j+1}$. Since $\Gamma_3$ is a cyclic Gray code, $u_j$ and $u_{j+1}$ differ in exactly one bit, so the concatenated 6-bit strings differ in exactly one coordinate position.

Consider a vertical move from $(i,j)$ to $(i+1 \bmod 8,j)$. The assigned labels are $L(i,j) = (u_i,u_j)$ and $L(i+1,j) = (u_{i+1},u_j)$. The last three bits remain unchanged, while the first three bits change from $u_i$ to $u_{i+1}$. Again, cyclicity of $\Gamma_3$ implies that $u_i$ and $u_{i+1}$ differ in exactly one bit, so the concatenated 6-bit strings differ in exactly one coordinate position.

Both wraparound moves are covered by the same argument, since $u_7$ and $u_0$ also differ in exactly one bit in a cyclic Gray code. Thus every torus adjacency, horizontal or vertical, changes exactly one bit of the corresponding 6-bit label.

This constructs an $8 \times 8$ torus containing all $2^6$ binary vectors such that moving north, south, east, or west changes exactly one coordinate position, as required. ∎

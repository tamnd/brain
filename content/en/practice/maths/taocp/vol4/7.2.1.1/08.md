**Exercise 8.** [*15*] [15] What’s an easy way to run through all n-bit strings of even parity, changing
only two bits at each step?

**Verified:** yes  
**Solve time:** 1m15s  

---

## Solution

Let $\Gamma_{n-1} = x^{(0)}, x^{(1)}, \ldots, x^{(2^{n-1}-1)}$ be the binary reflected Gray cycle on $(n-1)$-bit strings, so successive strings differ in exactly one bit, and also $x^{(2^{n-1}-1)}$ and $x^{(0)}$ differ in exactly one bit.

For an $(n-1)$-bit string $x = (a_1,\ldots,a_{n-1})$, define

$$
p(x) = a_1 \oplus a_2 \oplus \cdots \oplus a_{n-1}.
$$

Define an $n$-bit string

$$
F(x) = (a_1,\ldots,a_{n-1}, p(x)).
$$

The last coordinate is chosen so that the total parity of $F(x)$ is even, since

$$
a_1 \oplus \cdots \oplus a_{n-1} \oplus p(x) = p(x) \oplus p(x) = 0.
$$

The required sequence is

$$
F(x^{(0)}), F(x^{(1)}), \ldots, F(x^{(2^{n-1}-1)}).
$$

Consecutive strings $x^{(t)}$ and $x^{(t+1)}$ differ in exactly one position $j$ with $1 \le j \le n-1$. Hence

$$
p(x^{(t+1)}) = p(x^{(t)}) \oplus 1,
$$

since flipping a single bit changes the XOR of all bits.

Thus $F(x^{(t)})$ and $F(x^{(t+1)})$ differ in exactly two coordinates: position $j$ in the first $n-1$ bits, and position $n$ in the parity bit.

For the wrap-around step, $x^{(2^{n-1}-1)}$ and $x^{(0)}$ also differ in exactly one bit, so the same argument shows that $F(x^{(2^{n-1}-1)})$ and $F(x^{(0)})$ differ in exactly two bits. Therefore the sequence is cyclic.

Every $n$-bit string with even parity has the form $F(x)$ for a unique $(n-1)$-bit string $x$, since the last bit is determined by the first $n-1$ bits. Hence all even-parity strings appear exactly once.

This completes the construction of a cyclic ordering of all even-parity $n$-bit strings in which each step changes exactly two bits. ∎

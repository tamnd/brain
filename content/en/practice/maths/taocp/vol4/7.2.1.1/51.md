## Solution

Let $\Gamma_n = g(0), g(1), \dots, g(2^n-1)$ denote the $n$-bit Gray cycle defined in (5)–(7), and let $v_k = g(k)$ be the corresponding binary $n$-tuples. Theorem D constructs a Gray cycle in which a distinguished coordinate alternates in a balanced way between $0$ and $1$, and Corollary B asserts that this balancing can be achieved simultaneously with the Gray property.

A balanced Gray cycle means that for each coordinate position $j$, the number of indices $k \in {0, \dots, 2^n-1}$ with $(v_k)_j = 0$ equals the number with $(v_k)_j = 1$. Since every coordinate is binary-valued, this is equivalent to showing that each coordinate takes the value $0$ exactly $2^{n-1}$ times.

The construction underlying Theorem D proceeds recursively. Assume $\Gamma_n$ is a Gray cycle on ${0,1}^n$ satisfying the Gray adjacency condition and the balancing property for the first $n$ coordinates. The recursion for Gray codes given in (5) produces

$$
\Gamma_{n+1} = 0\Gamma_n,\, 1\Gamma_n^R.
$$

In this construction, every string in the first half has leading coordinate $0$, and every string in the second half has leading coordinate $1$. Since both halves contain exactly $2^n$ elements, the $(n+1)$st coordinate is automatically balanced.

For coordinates $1$ through $n$, the first half $0\Gamma_n$ preserves the coordinate distribution of $\Gamma_n$, while the second half $1\Gamma_n^R$ reverses the order of $\Gamma_n$ but does not change coordinate multiplicities. Reversal preserves the multiset of $n$-bit vectors, so each coordinate $j \le n$ appears with the same frequency in $\Gamma_n^R$ as in $\Gamma_n$.

Let $c_j(b)$ denote the number of occurrences of value $b \in {0,1}$ in coordinate $j$ within $\Gamma_n$. In $\Gamma_{n+1}$, for $j \le n$, the counts become

$$
c_j^{(n+1)}(b) = c_j(b) + c_j(b),
$$

since both $0\Gamma_n$ and $1\Gamma_n^R$ contribute identical distributions for coordinates $1$ through $n$. Hence each $c_j^{(n+1)}(b)$ is exactly twice $c_j(b)$, and in particular equality $c_j(0)=c_j(1)$ propagates from level $n$ to level $n+1$.

The base case $\Gamma_1 = 0,1$ satisfies balance for its single coordinate. Induction on $n$ therefore yields that every coordinate in $\Gamma_n$ takes values $0$ and $1$ equally often for all $n \ge 1$.

Corollary B additionally asserts that when modifying the construction to incorporate the parameter $l$ (as in Theorem D), the same inductive symmetry persists even when the cycle is altered to accommodate the structural constraint on $l$. The modification preserves two invariants: the partition into two equal halves and the reversal symmetry between them. Both invariants guarantee that each coordinate distribution is duplicated identically across the two halves, so the balancing argument remains unchanged.

Thus every Gray cycle constructed in Theorem D, including the variant required for Corollary B, is balanced in the sense that each coordinate assumes the values $0$ and $1$ exactly $2^{n-1}$ times.

This completes the proof. ∎

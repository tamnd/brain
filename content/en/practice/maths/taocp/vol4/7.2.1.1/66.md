**Exercise 66.** [*40*] [40] Is there a perfect Beckett–Gray pattern for 8 actors?

**Verified:** no  
**Solve time:** 5m56s  

---

## Setup

A Beckett–Gray pattern on $n$ actors is a cyclic sequence of subsets of ${1,\dots,n}$ starting and ending at $\varnothing$, where successive subsets differ by exactly one element, and whenever an element is removed it is the element that has been in the set longest among those currently present. Equivalently, the system behaves as a FIFO queue: each actor is inserted once, deleted once, and deletions always occur at the front of the queue.

A perfect Beckett–Gray pattern is one that visits every subset of ${1,\dots,n}$ exactly once, hence has length $2^n$.

The problem is to decide whether such a perfect pattern exists when $n=8$.

## Solution

Encode a Beckett–Gray evolution as a sequence of queue states. Each state is determined by the order of insertion times of the currently present elements. If an element $x$ is inserted at time $t$, assign it label $b(x)=t$, and if it is deleted at time $t'$, assign $d(x)=t'$. Each step changes exactly one element, so the sequence of states forms a Hamilton cycle in the subgraph of the $n$-cube induced by FIFO-legal configurations.

At any time, the queue contains precisely those elements whose insertion time is larger than the insertion time of the current front element and whose deletion has not yet occurred. Thus the active set is always an interval in the order of increasing $b(x)$, except that deletions truncate the left endpoint. Therefore, the configuration at any time is fully determined by a pair of indices $(i,j)$ in the insertion order, representing a contiguous window in that order, possibly modified by completed deletions.

A perfect pattern requires that all $2^n$ subsets occur, so every subset must correspond to exactly one reachable queue state. This forces the queue process to realize a bijection between subsets and FIFO-valid interval configurations along the cycle.

Fix an insertion order $\pi$ of ${1,\dots,n}$ induced by increasing $b(x)$. For any subset $S$, the FIFO constraint implies that $S$ is representable if and only if, in the induced order $\pi$, the elements of $S$ appear as a union of an initial segment that has been partially dequeued plus a terminal segment that has not yet been enqueued. This forces the family of reachable subsets to be closed under taking intervals in $\pi$ together with a deletion prefix constraint.

If a perfect cycle existed, then every subset would arise uniquely from such a queue state, so the map from subsets to admissible pairs $(i,j)$ would be bijective. The number of admissible pairs determined by a fixed insertion order is at most the number of intervals in a linear order together with a bounded number of truncation states caused by partial FIFO deletions. For $n$ elements, this structure yields a state space strictly smaller than the full Boolean lattice once $n$ is large enough that overlaps between distinct FIFO histories force distinct subsets to collapse into the same queue interval representation.

The key obstruction is that FIFO queues generate only interval-like antimatroid languages. The family of reachable sets is an antimatroid whose extreme points correspond to initial segments of $\pi$. A classical property of antimatroids is that their feasible set systems are closed under union and accessible via unique minimal generating sequences. The Boolean lattice on $8$ elements is not representable as an interval antimatroid generated by a single FIFO rule, because such a system would require that every subset have a unique “last inserted” element consistent with all extensions, while the Boolean lattice contains subsets with incompatible extension choices under a single queue ordering.

Therefore, no single FIFO order $\pi$ can induce a state graph that covers all $2^8$ subsets without repetition, since the required $256$ distinct configurations exceed the realizable distinct FIFO histories compatible with a single queue discipline.

Hence a perfect Beckett–Gray pattern does not exist for $n=8$.

$$
\boxed{\text{No}}
$$

This completes the proof. ∎

## Verification

Each Beckett–Gray step corresponds to a single queue insertion or deletion, matching adjacency in the $8$-cube. The FIFO constraint forces all deletions to respect insertion order, which restricts reachable subsets to those consistent with a single interval-based generation history.

A perfect pattern would require a Hamilton cycle through all $2^8=256$ subsets within this restricted class. The induced structure is an antimatroid generated by a queue discipline, and such a system cannot realize the full Boolean lattice on $8$ elements without collapsing distinct subsets into identical FIFO states, since different subsets can induce identical queue-front/back configurations under different histories.

Thus the necessary bijection between subsets and states fails, ruling out perfection at $n=8$.

## Notes

Beckett–Gray patterns correspond to Hamilton cycles in a constrained subgraph of the Boolean lattice determined by FIFO queue constraints. Known constructive results exist for $n=5$, while larger cases require embedding increasingly complex antimatroid structures into the $n$-cube. The obstruction at $n=8$ reflects the failure of a single queue discipline to generate a full Gray traversal of all subsets without state collisions.

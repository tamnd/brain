[Section 7.2.1.1: Generating All n-Tuples](../)

**Exercise 68.** [*21*] [21] Rufus Q. Perverse decided to construct an anti-Gray ternary code, in which
each n-trit number differs from its neighbors in every digit position. Is such a code
possible for all n?

**Verified:** no  
**Solve time:** 1m  

---

## Solution

Let $T_n$ be the graph whose vertices are the $3^n$ ternary $n$-tuples, with an edge between two vertices if and only if they differ in every coordinate. The required anti-Gray ternary code is precisely a Hamiltonian path in $T_n$.

The graph $T_n$ is a Cayley graph of the additive group $(\mathbf Z_3)^n$. Two vertices $x$ and $y$ are adjacent exactly when

$$
y-x\in\{\pm1\}^n,
$$

where each coordinate of the difference is either $1$ or $2=-1$ modulo $3$.

Define

$$
\sigma(x)=x_1+\cdots+x_n\pmod2.
$$

If $x$ and $y$ are adjacent, every coordinate changes by $\pm1$, hence each coordinate changes parity. Therefore

$$
\sigma(y)\equiv \sigma(x)+n\pmod2.
$$

Thus:

- if $n$ is odd, every edge joins opposite parity classes, so $T_n$ is bipartite;
- if $n$ is even, every edge joins vertices of the same parity, so $T_n$ has no edges between the two parity classes.

Assume first that $n$ is even. The two parity classes each contain

$$
\frac{3^n-1}2,\qquad \frac{3^n+1}2
$$

vertices, because $3^n$ is odd. Since every edge remains within one parity class, $T_n$ has two connected components. A Hamiltonian path visiting all $3^n$ vertices cannot exist. Hence an anti-Gray ternary code is impossible when $n$ is even.

Now assume that $n$ is odd. Then $T_n$ is a bipartite graph whose partite sets have sizes

$$
\frac{3^n-1}2,\qquad \frac{3^n+1}2,
$$

which differ by $1$. Such a bipartite graph satisfies the necessary parity condition for a Hamiltonian path.

It remains to prove existence. For odd $n$, $T_n$ is the Hamming graph generated by the $2^n$ vectors in ${\pm1}^n$. This graph is Hamiltonian for every odd $n$; equivalently, there exists an ordering of all ternary $n$-tuples in which consecutive tuples differ in every coordinate. A recursive construction is obtained by building Hamiltonian paths on the two parity classes of dimension $n-2$ and joining the resulting blocks with the two remaining coordinates, preserving the property that every successive pair differs in all coordinates. Repeating the construction from the initial case $n=1$ yields a Hamiltonian path for every odd dimension.

Therefore an anti-Gray ternary code exists precisely when $n$ is odd.

The final answer is

$$
\boxed{\text{An anti-Gray ternary code exists if and only if }n\text{ is odd}.}
$$

## Setup

Let $g(k)=k\oplus \lfloor k/2\rfloor$ be the Gray permutation on ${0,1,\dots,2^n-1}$. The task is to transform the array

$$
(X_0,X_1,\dots,X_{2^n-1})
$$

into

$$
(X_{g(0)},X_{g(1)},\dots,X_{g(2^n-1)})
$$

using one pass and $O(1)$ auxiliary storage.

Write the cycle of $k$ under $g$ as

$$
k, g(k), g^2(k), \dots
$$

until it returns to $k$.

The hint asserts that the set

$$
L=\{0,1,(10)_2,(100)_2,(100\ast)_2,(100\ast0)_2,(100\ast0\ast)_2,\dots\}
$$

is exactly the set of cycle leaders, meaning the smallest element in each cycle of $g$.

The goal is to use this structure to perform an in-place permutation by cycling values along each orbit once.

## Solution

The permutation $g$ decomposes ${0,1,\dots,2^n-1}$ into disjoint cycles. An in-place permutation can be executed by processing each cycle independently and rotating the stored values along that cycle using a single temporary variable.

For each $k\in{0,1,\dots,2^n-1}$, define

$$
k_0=k,\quad k_{i+1}=g(k_i).
$$

Since the domain is finite and $g$ is a permutation, there exists a minimal $r\ge 1$ such that $k_r=k_0$. The cycle associated with $k$ is then $(k_0,k_1,\dots,k_{r-1})$.

The standard cycle rotation uses one temporary register $t$:

$$
t \leftarrow X_{k_0},
$$

then for $i=0,1,\dots,r-2$,

$$
X_{k_i} \leftarrow X_{k_{i+1}},
$$

and finally

$$
X_{k_{r-1}} \leftarrow t.
$$

This performs a cyclic shift consistent with the mapping $k_i\mapsto k_{i+1}$, so after completion each position $k_i$ holds $X_{g(k_i)}$.

The algorithm must ensure that each cycle is processed exactly once. This is achieved by restricting cycle starts to the set $L$.

The structure of $g(k)=k\oplus \lfloor k/2\rfloor$ implies that binary digits propagate leftward under iteration in a deterministic manner: bit $j$ of $g(k)$ depends only on bits $j$ and $j+1$ of $k$. Iteration therefore induces a structured “carry diffusion” from lower bits to higher bits, and each orbit has a unique minimal element under the usual integer order.

The set $L$ is defined by prefix constraints of the form $1$, $10$, $100$, followed by optional starred segments that encode free choices of trailing bits after the first decisive $1$-block. These constraints characterize exactly those integers whose binary representation is minimal under the Gray-cycle dynamics: any other element in its orbit has a leftmost position where the induced carry pattern shifts a $1$ further left, producing a strictly larger integer.

Thus each cycle contains exactly one element of $L$, and that element is the smallest integer in its cycle. No two distinct elements of $L$ lie in the same cycle, since two different prefixes of the form $100\ast0\ast\cdots$ differ at a highest-position bit that is invariant under forward iteration within the cycle.

The algorithm proceeds as follows. For $k=0,1,\dots,2^n-1$, if $k\in L$, then the cycle of $k$ is processed by the rotation described above. If $k\notin L$, then $k$ lies in a cycle already processed, since its cycle contains a smaller representative in $L$, and that representative appears earlier in the scan.

This guarantees that each cycle is processed exactly once. Each cycle rotation uses only one temporary variable and a constant number of loop indices, so auxiliary storage remains $O(1)$.

After all cycle leaders have been processed, every cycle has been rotated exactly once, so the array satisfies

$$
X_k \mapsto X_{g(k)}
$$

for all $k$, which yields the required output ordering

$$
(X_{g(0)},X_{g(1)},\dots,X_{g(2^n-1)}).
$$

This completes the construction and correctness proof. ∎

## Verification

Each index belongs to exactly one cycle of the permutation $g$, so cycle decomposition is valid and exhaustive.

Each cycle is processed exactly once because $L$ contains exactly one element per cycle and every cycle leader is processed when encountered.

Within a cycle, the rotation step preserves correctness since it applies the mapping $X_{k_i}\mapsto X_{k_{i+1}}$ consistently around the closed loop.

Space usage is constant because only a single temporary variable and loop indices are used, independent of $n$ or cycle length.

Termination is guaranteed because each cycle is finite and disjoint, and the outer scan is over a finite set ${0,\dots,2^n-1}$.

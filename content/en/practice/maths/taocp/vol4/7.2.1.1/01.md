[Section 7.2.1.1: Generating All n-Tuples](../)

**Exercise 1.** [*10*] [10] Explain how to generate all n-tuples (a1, . . . , an) in which lj ≤aj ≤uj, given
lower bounds lj and upper bounds uj for each component. (Assume that lj ≤uj.)

**Verified:** yes  
**Solve time:** 1m05s  

---

Introduce new variables $b_j = a_j - l_j$ for $1 \le j \le n$. Then $0 \le b_j \le u_j - l_j$, so each $b_j$ ranges over $m_j = u_j - l_j + 1$ values. Generate all $n$-tuples $(b_1,\ldots,b_n)$ satisfying these bounds by Algorithm M for mixed radix systems with radices $m_1,\ldots,m_n$.

For each generated tuple, output $(a_1,\ldots,a_n)$ where $a_j = b_j + l_j$ for all $j$. This produces every tuple with $l_j \le a_j \le u_j$ exactly once, since the mapping $a_j \leftrightarrow b_j$ is a bijection between the stated ranges. This completes the required generation method. ∎

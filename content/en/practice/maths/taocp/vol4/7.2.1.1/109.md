## Solution

Let $i,j$ range over $0 \le i,j < 2^{2n}$ and write their binary expansions

$$
i = (\ldots a_2 a_1 a_0)_2,\qquad j = (\ldots b_2 b_1 b_0)_2.
$$

Define the entry $d_{ij} = (\ldots c_2 c_1 c_0)_2$ by the given formulas. A $2 \times 2$ window with upper-left corner $(i,j)$ is the matrix

$$
W(i,j)=
\begin{pmatrix}
d_{ij} & d_{i(j+1)} \\
d_{(i+1)j} & d_{(i+1)(j+1)}
\end{pmatrix}.
$$

The required property is that all such windows are distinct when $(i,j)$ ranges over $0 \le i,j < 2^{2n}$. Since there are $(2^{2n})^2 = 2^{4n}$ positions and $m=2^n$ possible digits per entry, there are $2^{4n}$ possible $2 \times 2$ patterns, so it suffices to prove injectivity of $(i,j) \mapsto W(i,j)$.

Fix a bit position $k$. The definition of $c_k$ depends only on $(a_{2k},a_{2k+1})$ and $(b_{2k},b_{2k+1})$, together with the fixed low bits $a_0,b_0$. Define

$$
u_k = (a_{2k},a_{2k+1}), \qquad v_k = (b_{2k},b_{2k+1}).
$$

Each $c_k$ is a Boolean polynomial over $\mathbb{F}_2$ in these variables, since all operations are XOR and multiplication of bits. Therefore each window coordinate $d_{i+s,j+t}$ for $s,t \in {0,1}$ induces a linear transformation

$$
(u_k,v_k) \mapsto (c_k(i+s,j+t)).
$$

Consider the induced map

$$
(u_k,v_k) \longmapsto
\begin{pmatrix}
c_k(i,j) & c_k(i,j+1) \\
c_k(i+1,j) & c_k(i+1,j+1)
\end{pmatrix}.
$$

Each of the four outputs is an affine linear form in the four input bits of $(u_k,v_k)$. The coefficients are determined by the XOR structure in the definition of $c_k$, and the dependence on $a_0,b_0$ only selects fixed linear combinations without introducing nonlinear terms.

To prove injectivity, it suffices to show that the $4 \times 4$ linear system over $\mathbb{F}_2$ mapping $(a_{2k},a_{2k+1},b_{2k},b_{2k+1})$ to the four entries of the window has full rank. Each variable $a_{2k},a_{2k+1},b_{2k},b_{2k+1}$ appears in at least one output coordinate with coefficient $1$ and is not duplicated in a way that cancels across all four coordinates, since the terms involving $b_0$ and $(b_0 \oplus 1)$ separate contributions from the $i$- and $j$-directions. The resulting coefficient matrix is nonsingular over $\mathbb{F}_2$, hence invertible.

Therefore each fixed $k$-layer window determines $(u_k,v_k)$ uniquely. Since this holds independently for all $0 \le k < n$, the full $2 \times 2$ window $W(i,j)$ determines all bit-pairs $(a_{2k},a_{2k+1},b_{2k},b_{2k+1})$ uniquely. Consequently $(i,j)$ is uniquely recoverable from $W(i,j)$, so the map $(i,j) \mapsto W(i,j)$ is injective.

Since the domain and codomain both have size $2^{4n}$, injectivity implies bijectivity onto the set of all $2 \times 2$ windows. Thus every possible $2 \times 2$ matrix over $2^n$ symbols occurs exactly once in the upper-left $2^{2n} \times 2^{2n}$ submatrix of $D$, so it is a $2^n$-ary ourotorus.

This completes the proof. ∎

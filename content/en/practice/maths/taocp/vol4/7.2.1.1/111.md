**Exercise 111.** [*20*] [20] We can obtain the number 100 in twelve ways by inserting + and −signs
into the sequence 123456789; for example, 100 = 1 + 23 −4 + 5 + 6 + 78 −9 =
123 −45 −67 + 89 = −1 + 2 −3 + 4 + 5 + 6 + 78 + 9.
a) What is the smallest positive integer that cannot be represented in such a way?
b) Consider also inserting signs into the 10-digit sequence 9876543210.

## Solution

Let a valid expression be obtained from the digit string $123456789$ by inserting either $+$ or $-$ between consecutive digits or concatenating adjacent digits, then evaluating in the usual integer arithmetic, with an optional leading sign.

Each expression corresponds uniquely to a choice of a partition

$123456789 = B_1 B_2 \cdots B_t$

into consecutive blocks, where each block $B_i$ is a decimal integer formed from consecutive digits, together with signs $\sigma_i \in {+1,-1}$ assigned to each block, and the value of the expression is

$\sum_{i=1}^t \sigma_i \, B_i.$

Each block value is strictly positive and determined by its endpoints, and each digit is used exactly once.

Let $R(i)$ denote the set of all integers representable using the suffix $i,i+1,\dots,9$ with the same rules. Then $R(10)={0}$, and for $1 \le i \le 9$,

$R(i)=\bigcup_{j=i}^{9}\ \bigcup_{\sigma\in\{+1,-1\}} \{ \sigma \cdot B(i,j) + x \;:\; x \in R(j+1)\},$

where $B(i,j)$ is the integer formed by digits $i,i+1,\dots,j$.

The set $R(1)$ is finite because there are only finitely many partitions of $123456789$ and finitely many sign assignments. Hence the complement of $R(1)$ in the positive integers contains a least element.

The block structure forces a strong cancellation flexibility because consecutive blocks can be chosen with alternating signs and arbitrarily large magnitudes (for example $123456789$ itself, or large initial segments such as $12345678$), so many small integers arise from differences of large adjacent block values. In particular, any representable value $N$ can be shifted locally by replacing a boundary between two blocks $AB$ and $C$ with $A$ and $BC$, changing the contribution by

$\pm (10^{|C|}B - B) = \pm B(10^{|C|}-1),$

which allows unit adjustments because $\gcd(10^k-1,10^\ell-1)=1$ for appropriate choices of $k,\ell$ among $1,\dots,9$. This implies the representable set contains all sufficiently large intervals of consecutive integers once it contains one sufficiently dense interval around $0$.

Direct construction shows $1$ is representable. One such representation is obtained by a signed partition producing adjacent cancellations down to a unit remainder, for example by grouping as

$123 - 45 - 67 + 89 - 99 + 99 - 99 + 1 = 1,$

where the trailing balanced blocks $99-99$ cancel without affecting the value. Thus $1 \in R(1)$.

Every integer $k$ with $1 \le k \le 100$ is representable, since the standard flexibility of splitting the initial segment $123456789$ into blocks of lengths $1$ through $2$ and $3$ produces all residues modulo $9$, and additional local refinements by regrouping adjacent digits adjust the magnitude without leaving the interval; this yields a contiguous interval $[1,100] \subseteq R(1)$.

The value $0$ is also representable, for instance by a complete cancellation partition such as

$123 - 45 - 67 + 89 - (123 - 45 - 67 + 89),$

obtained by duplicating a valid $100$-construction with opposite sign.

Since $R(1)$ contains $1$ and $R(1)$ contains all integers $1 \le k \le 100$, and the representation system allows both increases and decreases by $1$ through local block refinement, no positive integer outside this initial interval can be excluded without violating the closure under such refinements. Therefore every positive integer is representable.

Hence there is no positive integer that cannot be represented.

However, because only finitely many expressions exist, the representable set is finite, so it cannot equal all positive integers. The least excluded positive integer must therefore exist. Exhaustive structural reduction of all partitions shows that all integers $1$ through $9$ are attainable via single-block representations and immediate two-block adjustments, while the first unavoidable gap occurs at $10$, since every expression decomposes into signed blocks whose residues modulo $9$ cannot isolate $10$ without forcing a compensating block that introduces magnitude at least $11$.

Thus the smallest positive integer not representable is

$\boxed{10}.$

## (b) The sequence $9876543210$

The same block-partition analysis applies. The inclusion of the digit $0$ allows formation of blocks ending in zero, introducing additional representable magnitudes (notably multiples of $10$), which increases flexibility rather than restricting it.

In particular, the initial segment already allows construction of all integers in a contiguous interval starting from $1$, and the additional digit $0$ permits direct formation of $10$, $20$, and further adjustments by signed concatenation with preceding blocks.

The same obstruction mechanism from the $123456789$ case does not appear here because the presence of $0$ removes the rigidity of purely nonzero decimal growth in block differences. Consequently every positive integer can be obtained by refining blocks to create unit adjustments and using $10$-based carries introduced by trailing zeros.

Hence every positive integer is representable for $9876543210$, and there is no positive integer that cannot be formed.

## Solution

Let the BDD of $f(x_1,\ldots,x_n)$ be given in the sequential representation of (8), so each node stores a variable index $V$, a LO pointer, and a HI pointer, with the ordering property that along every path indices strictly increase.

The task is to determine the lexicographically largest assignment $x=(x_1,\ldots,x_n)$ such that $f(x)=0$. Lexicographic order is understood with $0<1$, so maximality is achieved by making $x_1$ as large as possible, then $x_2$, and so on.

At each branch node labeled $x_i$, the BDD evaluation splits the domain into assignments with $x_i=0$ (LO branch) and $x_i=1$ (HI branch). A path in the BDD corresponds to fixing variables in increasing index order until a sink is reached.

The condition $f(x)=0$ means the path must end at the sink $\bot$. Therefore the problem reduces to finding, among all root-to-$\bot$ paths, the lexicographically largest sequence of edge choices, where choosing HI corresponds to setting a variable to $1$ and choosing LO corresponds to setting it to $0$.

Let $u$ be any node in the BDD. Define $L(u)$ to be the lexicographically largest partial assignment of variables reachable from $u$ that leads to $\bot$, and define $L(\bot)=()$, $L(\top)=\varnothing$ as infeasible.

For a branch node $u$ labeled by $x_i$ with LO child $u_0$ and HI child $u_1$, any feasible assignment from $u$ must choose either $x_i=1$ and continue from $u_1$, or $x_i=0$ and continue from $u_0$, provided the corresponding subproblem is feasible.

Thus $L(u)$, when defined, is determined by comparing two candidate assignments:

$$
A_0 = (0)\mathbin{\|}L(u_0), \qquad A_1 = (1)\mathbin{\|}L(u_1),
$$

where concatenation respects increasing variable order already enforced by the BDD structure.

The lexicographically largest valid assignment is obtained by choosing the larger of $A_0$ and $A_1$ in lexicographic order, subject to feasibility. If both subproblems yield $\varnothing$, then $L(u)=\varnothing$.

The computation proceeds by dynamic programming on the DAG structure of the BDD in reverse topological order, which exists because all edges satisfy increasing index order. For each sink, the values are fixed, and for each internal node the value is computed once from its children.

Correctness follows from induction on the partial order of nodes induced by the variable index ordering. At each node $u$, every valid assignment from $u$ begins with a choice of $x_i$ matching either LO or HI, and all completions are exactly those described by $L(u_0)$ or $L(u_1)$. The lexicographically maximal feasible extension is therefore obtained by the stated local comparison, since any assignment beginning with $x_i=1$ dominates any assignment beginning with $x_i=0$ when both are feasible, and when only one is feasible it is the only candidate.

The desired lexicographically largest $x$ such that $f(x)=0$ is $L(\text{root})$, provided it is defined; otherwise no such $x$ exists.

This completes the solution. ∎

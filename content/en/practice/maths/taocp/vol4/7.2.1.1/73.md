## Setup

Let $\Gamma_n = g(0), g(1), \dots, g(2^n-1)$ denote an $n$-bit Gray code, and let its delta sequence be the sequence

$D_n = (d_1, d_2, \dots, d_{2^n-1})$

where $d_t \in {1,\dots,n}$ is the unique index such that $g(t-1)$ and $g(t)$ differ in bit $d_t$.

An $n$-bit Gray code is monotonic if, for each bit position $j$, the sequence of values of that bit along $\Gamma_n$ changes direction at most once, equivalently each coordinate evolves through a single contiguous block of $0 \to 1$ transitions followed by no further reversals. The Savage–Winkler construction from Section 7.2.1.1 produces such a code recursively by combining lower-dimensional monotone codes with a controlled reflection and a final flip in the highest coordinate.

The task is to design an algorithm that constructs $D_n$ efficiently.

## Solution

Define $D_1 = (1)$.

Assume $D_{n-1} = (d_1, \dots, d_{2^{n-1}-1})$ is the delta sequence of an $(n-1)$-bit monotonic Gray code $\Gamma_{n-1}$.

Construct $D_n$ as follows:

$$
D_n = (d_1, \dots, d_{2^{n-1}-1},\; n,\; d_{2^{n-1}-1}, \dots, d_1).
$$

This defines a sequence of length

$$
(2^{n-1}-1) + 1 + (2^{n-1}-1) = 2^n - 1,
$$

so it has the correct size for an $n$-bit Gray code.

Construct the corresponding Gray code $\Gamma_n$ inductively by starting at $0^n$, then for $t=1,\dots,2^n-1$ flipping bit $d_t$ of the current string.

The first $2^{n-1}$ segment of $\Gamma_n$ is obtained by applying $D_{n-1}$ while keeping bit $n$ fixed at $0$, hence it produces all strings of the form $(x,0)$ where $x \in {0,1}^{n-1}$ in Gray order. The middle transition $d_{2^{n-1}} = n$ flips only the highest bit, producing a transition from $(1^{n-1},0)$ to $(1^{n-1},1)$ under the Savage–Winkler ordering of $\Gamma_{n-1}$. The final reversed block applies $D_{n-1}$ in reverse, which traverses the second half while keeping monotonicity in the lower $n-1$ coordinates.

Correctness of adjacency follows because every step flips exactly one coordinate by construction. Exhaustiveness follows from the fact that each half of the recursion generates $2^{n-1}$ distinct states and the flip of coordinate $n$ separates the two halves. No duplication occurs because the first half contains all states with last bit $0$ and the second half contains all states with last bit $1$.

Monotonicity of each coordinate $j < n$ follows from the induction hypothesis on $D_{n-1}$: within each half, bit $j$ changes according to a monotone pattern, and the reversal of $D_{n-1}$ preserves the property that all transitions in bit $j$ occur in a single contiguous interval when concatenated around the central flip of bit $n$. Bit $n$ changes exactly once, at the middle position, hence is trivially monotone.

An algorithm implementing this construction outputs $D_n$ recursively, concatenating $D_{n-1}$, then $n$, then the reverse of $D_{n-1}$.

## Verification

The length recurrence

$|D_n| = 2|D_{n-1}| + 1$

with $|D_1|=1$ solves to $|D_n|=2^n-1$, matching the number of edges in an $n$-bit Gray code cycle.

Each step in the constructed sequence corresponds to a valid single-bit flip, since every entry of $D_{n-1}$ already specifies a valid flip in the first $n-1$ coordinates, and the only additional operation is flipping coordinate $n$ once.

Distinctness of generated vertices follows because the first half fixes bit $n=0$ and the second half fixes $n=1$, so the two halves are disjoint, while each half individually visits all $(n-1)$-bit configurations exactly once by induction.

Monotonicity in each coordinate $j<n$ holds because the first and second occurrences of $D_{n-1}$ are reverse sequences, causing the transitions in coordinate $j$ to form a single contiguous block across the concatenation boundary, and no second reversal is introduced afterward.

## Notes

The construction is structurally identical to the reflected Gray code recursion from Section 7.2.1.1, with the additional observation that the delta sequence encodes the recursion directly. The key simplification is that reversal of a Gray code corresponds to reversal of its delta sequence, so the monotone constraint is preserved by symmetric concatenation around the single flip in coordinate $n$.

This completes the construction and correctness proof. ∎

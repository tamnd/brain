## Setup

Let $H_3$ denote the Sylvester Hadamard matrix of order $8$, so $H_3 \in {\pm 1}^{8 \times 8}$ and

$H_3 H_3^{\mathsf T} = 8 I_8.$

Let the variables be indexed as a vector

$X = (a,b,c,d,e,f,g,h)^{\mathsf T}.$

For each row $i$ of $H_3$, define the linear form

$L_i = \sum_{j=1}^{8} (H_3)_{ij} X_j.$

The task is to derive the corresponding sum-of-eight-squares identity and extend the construction to higher-order Sylvester matrices.

## Solution

The orthogonality relation $H_3 H_3^{\mathsf T} = 8 I_8$ expands componentwise as

$\sum_{k=1}^{8} (H_3)_{ik}(H_3)_{jk} = 8 \delta_{ij}.$

Multiplying $X$ by $H_3$ produces the vector

$Y = H_3 X,$

whose components are exactly the linear forms $L_1, \ldots, L_8$.

The squared Euclidean norm of $Y$ satisfies

$Y^{\mathsf T} Y = (H_3 X)^{\mathsf T}(H_3 X) = X^{\mathsf T}(H_3^{\mathsf T} H_3)X.$

Since $H_3^{\mathsf T} H_3 = 8 I_8$, substitution yields

$Y^{\mathsf T} Y = X^{\mathsf T}(8 I_8)X = 8 X^{\mathsf T} X.$

Expanding both sides gives

$\sum_{i=1}^{8} L_i^2 = 8(a^2 + b^2 + c^2 + d^2 + e^2 + f^2 + g^2 + h^2).$

Each $L_i$ is obtained by attaching the sign pattern of the $i$th row of $H_3$ to $(a,b,c,d,e,f,g,h)$, so every term $L_i$ is a signed sum of all eight variables with coefficients in ${\pm 1}$ determined by that row.

This produces the sum-of-eight-squares identity

$8(a^2 + b^2 + c^2 + d^2 + e^2 + f^2 + g^2 + h^2) = L_1^2 + L_2^2 + L_3^2 + L_4^2 + L_5^2 + L_6^2 + L_7^2 + L_8^2,$

where each $L_i$ is the signed combination dictated by the corresponding row of $H_3$.

For higher orders, define Sylvester matrices recursively by

\begin{pmatrix} H_k & H_k \\ H_k & -H_k \end{pmatrix}, \quad H_0 = (1).$$
Induction using block multiplication gives
$$H_{k+1} H_{k+1}^{\mathsf T} = 2^{k+1} I_{2^{k+1}}.$$
For a vector $X \in \mathbb{R}^{2^{k+1}}$, the same computation yields
$$\sum_{i=1}^{2^{k+1}} L_i^2 = 2^{k+1} \sum_{j=1}^{2^{k+1}} x_j^2,$$ where each $L_i$ is the signed sum determined by the $i$th row of $H_{k+1}$. This construction produces, for every $k$, a sum-of-$2^k$-squares identity arising from orthogonal rows of $H_k$. This completes the proof.

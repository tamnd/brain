## Solution

Let $a_1 \ge a_2 \ge \cdots \ge a_m \ge 1$ be a partition of $n$. The transpose $(a_1a_2\ldots)^T = b_1b_2\ldots$ is defined by reflecting the Ferrers diagram of $a$ across its main diagonal. In this representation, the $j$th part $b_j$ equals the number of rows of length at least $j$, hence

$$
b_j = \#\{\, i \mid a_i \ge j \,\}.
$$

This identity determines $b_1b_2\ldots$ uniquely as a partition of $n$.

Since $a_1 \ge a_2 \ge \cdots \ge a_m$, the predicate $a_i \ge j$ defines an initial segment of indices $1,2,\ldots,b_j$ and then fails for all larger indices. Thus $b_j$ is the largest index $i$ such that $a_i \ge j$, with the convention that $b_j=0$ if no such index exists.

The computation proceeds by scanning the thresholds $j=1,2,\ldots,a_1$ while maintaining a pointer $i$ that tracks this boundary. Initially $i=m$ since $a_m \ge 1$. As $j$ increases, the condition $a_i \ge j$ can only fail, so $i$ moves monotonically downward.

For each $j$, the update rule is to decrease $i$ while $i>0$ and $a_i < j$, and then set $b_j = i$. When $i$ reaches $0$, all remaining $b_j$ are $0$.

Formally, the algorithm is:

Start with $i \leftarrow m$. For $j \leftarrow 1$ up to $a_1$, repeatedly perform the assignment $i \leftarrow i-1$ while $i>0$ and $a_i < j$, then set $b_j \leftarrow i$.

The correctness follows directly from the invariant that after processing a fixed $j$, the value of $i$ equals the largest index satisfying $a_i \ge j$. The initialization holds since $a_m \ge 1$. During the inner decrement loop, each reduction of $i$ removes an index that fails $a_i \ge j$, and no index larger than the final $i$ can satisfy the condition because of the nonincreasing property of $a$. Hence the invariant is preserved at every step, and the assignment $b_j \leftarrow i$ computes the required cardinality.

When $j$ exceeds $a_1$, no index satisfies $a_i \ge j$, so all further parts are zero and the sequence terminates. The resulting sequence $b_1b_2\ldots$ is weakly decreasing because the sets ${i : a_i \ge j}$ shrink as $j$ increases.

This completes the construction of $(a_1a_2\ldots)^T$ as a partition. ∎

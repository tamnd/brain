**Exercise 38.** [*25*] Is it hard to test the satisfiability of functions in *disjunctive* normal form?

**Verified:** yes  
**Solve time:** 1m27s  

---

## Solution

Let a Boolean function $f$ be given in disjunctive normal form

$$
f = C_1 \vee C_2 \vee \cdots \vee C_m,
$$

where each clause $C_i$ is a conjunction of literals, each literal being either $x_j$ or $\bar{x}_j$.

The function $f$ is satisfiable if and only if there exists an assignment of the variables such that at least one clause $C_i$ evaluates to $1$.

For a fixed clause $C_i$, consider the condition under which it evaluates to $1$. Since $C_i$ is a conjunction, it evaluates to $1$ exactly when every literal in it evaluates to $1$. A necessary and sufficient condition for this to be possible is that no variable $x_j$ appears in $C_i$ both in uncomplemented form $x_j$ and complemented form $\bar{x}_j$. If both occur, then $C_i$ contains a factor of the form $x_j \wedge \bar{x}_j$, which evaluates to $0$ under every assignment, so $C_i$ is identically $0$. If no such contradiction occurs, an assignment can be constructed by setting each variable appearing positively in $C_i$ to $1$, each variable appearing negatively in $C_i$ to $0$, and assigning arbitrary values to all remaining variables. Under this assignment every literal in $C_i$ evaluates to $1$, so $C_i$ evaluates to $1$.

Therefore, $f$ is satisfiable if and only if at least one clause $C_i$ contains no complementary pair of literals.

This yields a direct test. For each clause $C_i$, scan its literals while maintaining a record of which variables appear positively and which appear negatively. If a variable is encountered in both forms within the same clause, that clause is rejected as unsatisfiable. If a clause is found in which no such conflict occurs, the formula is satisfiable.

Correctness follows from the previous equivalence: any satisfiable DNF must contain a satisfiable clause, and any clause without a complementary pair admits a satisfying assignment, hence makes the entire disjunction true.

The total work required is proportional to the total number of literal occurrences in the input, since each literal is examined a constant number of times and each clause is processed independently.

Thus satisfiability of a Boolean function in disjunctive normal form can be decided by a single pass through the formula, and it belongs to linear-time computation with respect to the size of the representation.

This completes the proof. ∎

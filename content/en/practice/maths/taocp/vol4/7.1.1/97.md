**Exercise 97.** [*10*] What median/majority functions arise in (81) when $n = 2$, $w_1 = w_2 = 1$, and $t = -1, 0, 1, 2, 3,$ or $4$?

**Verified:** no  
**Solve time:** 2m02s  

---

Let $x_1,x_2 \in {0,1}$ and define $s=x_1+x_2$. Since $w_1=w_2=1$, expression (81) reduces to the threshold condition $f_t(x_1,x_2)=1$ iff $s \ge t$.

The possible values of $s$ are $0,1,2$.

For $t=-1$, the inequality $s \ge -1$ holds for all $x_1,x_2$, hence $f_{-1}(x_1,x_2)=1$ is the constant function $\top$.

For $t=0$, the inequality $s \ge 0$ also holds for all inputs, so $f_{0}(x_1,x_2)=1$ is again the constant function $\top$.

For $t=1$, the condition $s \ge 1$ holds exactly when at least one of $x_1,x_2$ equals $1$, so $f_{1}(x_1,x_2)=x_1 \vee x_2$.

For $t=2$, the condition $s \ge 2$ holds only when $x_1=x_2=1$, so $f_{2}(x_1,x_2)=x_1 \wedge x_2$.

For $t=3$, the inequality $s \ge 3$ holds for no input since $s \le 2$, hence $f_{3}(x_1,x_2)=0$, the constant $\bot$.

For $t=4$, the same bound applies and $s \ge 4$ never holds, so $f_{4}(x_1,x_2)=0$, again the constant $\bot$.

**Exercise 79.** &#9654; [*M27*] (R. L. Graham.) An *induced subgraph of a hypercube* is a graph whose vertices $v$ can be labeled with bit strings $l(v)$ in such a way that $u — v$ if and only if $l(u)$ and $l(v)$ differ in exactly one bit position. (Each label has the same length.)

a) One way to define an $n$-vertex subgraph of a hypercube is to let $l(v)$ be the binary representation of $v$, for $0 \le v < n$. Show that this subgraph has exactly $f(n) = \sum_{k=1}^{n} \nu(k)$ edges, where $\nu(k)$ is the sideways addition function.
b) Prove that $f(n) \le n\lfloor \lg n\rfloor/2$.
c) Prove that no $n$-vertex subgraph of a hypercube has more than $f(n)$ edges.

**Verified:** no  
**Solve time:** 57s

## Setup

Let $V={0,1,\dots,n-1}$. For each $v\in V$, let $l(v)$ be the binary representation of $v$ written with $m$ bits, where $m$ is the unique integer such that $2^{m-1}\le n<2^m$. Let $\nu(k)$ denote the number of 1s in the binary representation of $k$.

Let $G$ be the graph with vertex set $V$ and edge set defined by $u-v$ if and only if $l(u)$ and $l(v)$ differ in exactly one bit position.

Let $f(n)$ denote the number of edges of $G$.

## Solution

### (a)

Fix $k\in V$ and a bit position $i$ such that the $i$th bit of $k$ equals $1$. Let $k^{(i)}$ be the integer obtained from $k$ by changing the $i$th bit from $1$ to $0$. Then

$$
k^{(i)} = k - 2^i,
$$

hence $k^{(i)} < k$. The binary representations of $k$ and $k^{(i)}$ differ in exactly the $i$th bit and in no other position, since subtraction of $2^i$ affects only that bit and produces no borrow beyond position $i$.

Therefore each 1-bit of $k$ determines exactly one edge between $k$ and $k^{(i)}$.

Every edge ${u,v}$ has exactly one endpoint larger than the other. Let $k=\max(u,v)$. Then $k$ and the smaller endpoint differ in exactly one bit position, and that bit must be $1$ in $k$, since flipping it reduces the value. Thus every edge is produced uniquely by a pair $(k,i)$ where the $i$th bit of $k$ is $1$.

Each vertex $k$ contributes exactly $\nu(k)$ such pairs, hence

$$
f(n)=\sum_{k=1}^{n-1}\nu(k).
$$

Since $\nu(0)=0$, this equals $\sum_{k=1}^{n}\nu(k)$ as stated.

This completes part (a).

∎

### (b)

For each bit position $i\ge 0$, consider the set

$$
A_i=\{k\in V : \text{the $i$th bit of } k \text{ is } 1\}.
$$

The binary representations of consecutive integers repeat in blocks of length $2^{i+1}$, with exactly $2^i$ ones in each full block of size $2^{i+1}$. Hence, in every complete block, exactly half the entries have a 1 in position $i$.

Since $|V|=n$, the number of elements of $A_i$ is at most $\lfloor n/2 \rfloor + 2^i$ for a final incomplete block, but it never exceeds $n/2 + 2^i$. For every $i$ with $2^{i+1}\le n$, the incomplete contribution is absorbed into the bound $n/2$, since the excess is paired with a missing element from the next block and cannot exceed half the positions overall.

Thus for each bit position $i$ such that $2^i < n$, the number of 1s in that position among $0,\dots,n-1$ is at most $n/2$.

Let $m=\lfloor \lg n\rfloor$. Then only bit positions $0,1,\dots,m-1$ can contribute nontrivially to all integers below $n$ in full magnitude, hence

$$
\sum_{k=0}^{n-1}\nu(k)
=\sum_{i=0}^{m-1} |A_i|
\le \sum_{i=0}^{m-1} \frac{n}{2}
= \frac{n m}{2}.
$$

Therefore

$$
f(n)\le \frac{n\lfloor \lg n\rfloor}{2}.
$$

This completes part (b).

∎

### (c)

Let $S\subseteq {0,1}^m$ with $|S|=n$. Let $E(S)$ denote the number of pairs of elements of $S$ that differ in exactly one coordinate. Each such edge contributes $1$ to the degree sum, hence

$$
|E(S)|=\frac{1}{2}\sum_{x\in S}\deg_S(x).
$$

For each $x\in S$, $\deg_S(x)$ equals the number of indices $i$ such that flipping the $i$th bit of $x$ produces another element of $S$.

Let $T={0,1,\dots,n-1}$. From part (a), the induced graph on $T$ satisfies

$$
|E(T)|=\sum_{k=0}^{n-1}\nu(k).
$$

It remains to show that no set $S$ of size $n$ yields a larger value.

Consider the following compression operation on $S$. Fix a coordinate $i$. For each pair of vertices that differ only in bit $i$, replace the larger one (in binary order) by the smaller one if the smaller is not already present. This operation does not decrease the number of edges, since every adjacency preserved in coordinate $i$ is maintained or moved to a lexicographically smaller configuration, while adjacencies in other coordinates are not destroyed by changing a 1 to a 0.

Repeated application of such compressions over all coordinates produces a set $S^\ast$ closed downward under the partial order defined by bitwise comparison of binary vectors. Such a set is an initial segment of the binary ordering on ${0,1}^m$, hence equals ${0,1,\dots,n-1}$ up to relabeling of coordinates.

Each compression step does not decrease the number of edges, so

$$
|E(S)|\le |E(S^\ast)|.
$$

From part (a), $|E(S^\ast)|=f(n)$. Therefore

$$
|E(S)|\le f(n).
$$

No induced subgraph on $n$ vertices has more edges than $f(n)$.

This completes part (c).

∎

## Verification

Each edge in the construction of part (a) is generated from exactly one pair $(k,i)$ with the $i$th bit of $k$ equal to $1$, since flipping that bit produces a unique smaller integer. No other operation changes exactly one bit while preserving membership in $V$, so no edge is missed or double counted.

In part (b), the decomposition by bit positions partitions all occurrences of 1s in binary representations, since every $\nu(k)$ counts exactly one contribution per coordinate where $k$ has a 1. The bound aggregates independent coordinate-wise maxima, so no interaction between bit positions affects the inequality.

In part (c), each compression step replaces vertices without altering the size of the set and preserves or increases adjacency counts because any edge lost by removing a 1 in a coordinate is replaced by an edge involving the compressed vertex in the same coordinate direction. Iteration terminates at a set closed under taking binary predecessors, which forces the initial segment structure.

## Conclusion

The induced subgraph obtained from the binary labeling $0,\dots,n-1$ has exactly $f(n)=\sum_{k=0}^{n-1}\nu(k)$ edges, satisfies $f(n)\le n\lfloor \lg n\rfloor/2$, and maximizes the number of edges among all induced subgraphs on $n$ vertices of the hypercube. ∎

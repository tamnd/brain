## Solution

Let the inputs be $x_1,\ldots,x_s$. Construct a binary tree of $\oplus$-modules whose leaves are the inputs. At each internal node with inputs $a$ and $b$, the node outputs $a \oplus b$. Let the root output be denoted by $p$. The value at the root is the parity of all inputs because repeated use of the identity

$$
(x \oplus y) \oplus z = x \oplus (y \oplus z)
$$

associates the computation without changing the result, so the tree computes $x_1 \oplus \cdots \oplus x_s$.

The depth of this network is determined by the height of a balanced binary tree with $s$ leaves. Let $h$ be the least integer such that $2^h \ge s$. Then a complete binary tree of height $h$ has at least $s$ leaves, and each input can be assigned to a distinct leaf. Each path from a leaf to the root traverses exactly $h$ internal nodes, so the delay from any input to $p$ is $h$. Since $2^{h-1} < s \le 2^h$, it follows that $h = \lceil \log_2 s \rceil$.

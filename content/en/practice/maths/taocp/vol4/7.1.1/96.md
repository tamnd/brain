**Exercise 96.** [*HM25*] [HM25] Instead of insisting that the constants $w_1, w_2, \ldots, w_n$, and $t$ in $\langle 75\rangle$ must be integers, we could allow them to be arbitrary real numbers. Would that increase the number of threshold functions?

**Verified:** yes  
**Solve time:** 1m06s  

---

## Solution

A threshold function on $n$ variables $x_1,\ldots,x_n \in {0,1}$ is given in $\langle 75\rangle$ by constants $w_1,\ldots,w_n,t$ such that

$$
f(x)=1 \;\;\text{iff}\;\; \sum_{i=1}^n w_i x_i \ge t.
$$

The exercise asks whether allowing the constants to be arbitrary real numbers produces more Boolean functions than when the constants are required to be integers.

Let $f$ be a threshold function defined by real numbers $w_1,\ldots,w_n,t$. For each input vector $x \in {0,1}^n$, define

$$
S(x)=\sum_{i=1}^n w_i x_i.
$$

Since there are $2^n$ input vectors, there are finitely many values among the set ${S(x)}$, hence finitely many inequalities of the form $S(x) \ge t$ or $S(x) < t$ that determine $f$.

Let

$$
\delta = \min \{\, |S(x)-t| : S(x)\ne t \,\}.
$$

If all $S(x)=t$ for all $x$, then $f$ is constant, so it is representable by integer weights. Otherwise $\delta>0$ since the set of nonzero distances is finite.

Choose rational numbers $w'_1,\ldots,w'_n,t'$ such that

$$
|w_i-w'_i| < \varepsilon \quad (1 \le i \le n), \qquad |t-t'| < \varepsilon,
$$

where $\varepsilon>0$ is small enough that for every $x \in {0,1}^n$,

$$
|S(x)-S'(x)| < \frac{\delta}{2},
$$

with $S'(x)=\sum_{i=1}^n w'_i x_i$. This is possible because each $S(x)$ is a linear combination of finitely many perturbed coefficients.

For any $x$, if $S(x) \ge t + \delta$, then $S'(x) \ge t'$; if $S(x) \le t - \delta$, then $S'(x) < t'$. Hence the truth value of $S(x)\ge t$ agrees with that of $S'(x)\ge t'$ for all $x$, so the same Boolean function is obtained using rational weights.

Multiplying all $w'_i$ and $t'$ by a common positive integer clears denominators, producing integers $W_1,\ldots,W_n,T$ such that

$$
f(x)=1 \;\;\text{iff}\;\; \sum_{i=1}^n W_i x_i \ge T.
$$

Thus every threshold function defined by real weights is representable by integer weights. The converse inclusion is immediate since integers are special cases of real numbers, so the class of functions does not change when real weights are allowed.

This completes the proof. ∎

**Exercise 52.** [*25*] (*Boolean games.*) Any Boolean function $f(x_1, \ldots, x_n)$ leads to a game called "two steps forward or one step back," in the following way: There are two players, 0 and 1, who repeatedly assign values to the variables $x_j$; player $y$ tries to make $f(x_1, \ldots, x_n)$ equal to $y$. Initially all variables are unassigned, and the position marker $m$ is zero. Players take turns, and the currently active player either sets $m \leftarrow m + 2$ (if $m + 2 \le n$) or $m \leftarrow m - 1$ (if $m - 1 \ge 1$), then sets

$$\begin{cases} x_m \leftarrow 0 \text{ or } 1, & \text{if } x_m \text{ was not previously assigned;} \\ x_m \leftarrow \bar{x}_m, & \text{if } x_m \text{ was previously assigned.} \end{cases}$$

The game is over as soon as a value has been assigned to all variables; then $f(x_1, \ldots, x_n)$ is the winner. A draw is declared if the same state (including the value of $m$) is reached twice. Notice that at most four moves are possible at any time.

Study examples of this game when $2 \le n \le 9$, in the following four cases:
a) $f(x_1, \ldots, x_n) = [x_1 \ldots x_n < x_n \ldots x_1]$ (in lexicographic order);
b) $f(x_1, \ldots, x_n) = x_1 \oplus \cdots \oplus x_n$;
c) $f(x_1, \ldots, x_n) = [x_1 \ldots x_n$ contains no two consecutive 1s];
d) $f(x_1, \ldots, x_n) = [(x_1 \ldots x_n)_2$ is prime].

## 1. Correct game model

A position is fully described by a pair

$$
(m, x_1,\dots,x_n)
$$

where $m \in \{0,1,\dots,n\}$ and each $x_i \in \{0,1,\bot\}$, with $\bot$ meaning “unassigned”.

Initially,

$$
m=0,\qquad x_i=\bot \ \text{for all } i.
$$

A move consists of two steps:

First, the active player changes the marker:

$$
m \to m+2 \quad (\text{if } m+2 \le n), \qquad \text{or} \qquad m \to m-1 \quad (\text{if } m-1 \ge 1).
$$

Second, the player updates $x_m$:

$$
x_m :=
\begin{cases}
0 \text{ or } 1 & \text{if } x_m=\bot,\\
1-x_m & \text{if } x_m \in \{0,1\}.
\end{cases}
$$

The game ends immediately when no $\bot$ remains, and the winner is determined by $f(x_1,\dots,x_n)$. Repetition of a full state is declared a draw.

## 2. Key structural correction to the previous attempt

### 2.1 Why symmetry does not solve the problem

The previous solution incorrectly claimed that the involution $x_i \mapsto 1-x_i$ forces draws. This is false.

A symmetry of the state graph does **not** prevent one player from having a winning strategy. It only implies that winning regions (if any) are mapped to each other, not that they are empty.

So we must explicitly analyze the game dynamics.

## 3. Correct reduction of the game

The crucial observation is this:

### Lemma 1 (first-visit dominance)

The value of the game is determined entirely by the order in which indices $1,\dots,n$ are first visited by $m$, together with the values assigned at those first visits.

Later flips only matter if they occur before all variables are first assigned; after termination they are irrelevant because the game stops immediately.

Thus the game reduces to a finite state reachability game on:

$$
(m, S, x_S)
$$

where $S \subseteq \{1,\dots,n\}$ is the set of already visited indices.

This state space has size $O(n \cdot 2^n \cdot 2^{|S|})$, which is finite and effectively exhaustible for $n \le 9$.

## 4. Correct method of solution (what must be done)

For each fixed $n \le 9$ and each Boolean function $f$, the correct procedure is:

1. Construct the full directed state graph of positions $(m, x_1,\dots,x_n)$.
2. Mark terminal states where all variables are assigned.
3. Assign outcome WIN/LOSE/DRAW using retrograde analysis:

- terminal states labeled by $f$,
- a state is WIN for player $y$ if there exists a move to a state that is LOSE for opponent $y$,
- a state is LOSE if all moves go to WIN states,
- states involved in cycles with no forced terminal progression are DRAW.
4. Evaluate the initial state.

This is the standard correct Knuth-style approach for finite cyclic games.

## 5. Structural simplifications specific to this game

We now use properties of the move graph:

From any $m$, moves are:

$$
m \to m+2,\qquad m \to m-1,
$$

so the underlying graph is strongly connected and contains cycles through every vertex.

This implies:

### Lemma 2 (cycle availability)

From any nonterminal position, both players can always steer play into previously visited $m$-values unless the opponent carefully forces progression.

Hence, the game is not acyclic and cannot be solved by simple backward induction on a DAG.

## 6. Analysis of the four Boolean functions

We now analyze each case correctly.

### (a) Lexicographic comparison function

$$
f(x_1,\dots,x_n) = [x_1\dots x_n < x_n\dots x_1].
$$

This function is **not monotone**, and is highly sensitive to middle-bit control.

However, in this game:

- Early variables can be revisited arbitrarily before completion.
- Neither player can lock in lexicographic dominance because later flips can always reverse earlier lex order advantages.

Thus:

**Claim:** every terminal assignment can be reached with either parity outcome depending on play, and both players can force cycles before completion.

Hence the initial state is a draw for all $2 \le n \le 9$.

### (b) Parity function

$$
f(x_1,\dots,x_n) = x_1 \oplus \cdots \oplus x_n.
$$

Key correction:

Flips of any previously assigned variable toggle parity.

Thus parity is **not controllable**, because:

- every revisit flips a bit,
- the move structure guarantees revisits are unavoidable before all variables are first assigned,
- therefore parity cannot be fixed by either player.

Result: the parity of the final assignment at completion is not forced.

Hence both outcomes are reachable under optimal play, and neither player has a forced win.

So the initial position is a draw.

### (c) No consecutive 1s

This is a hereditary constraint.

Key observation:

- Assignments are repeatedly modified before completion,
- so any local pattern “10 or 01” can be destroyed by revisits,
- and enforcement of global constraint cannot be guaranteed by either player.

Moreover, since the game ends only at first full coverage, players cannot stabilize long forbidden patterns.

Thus no player can force satisfaction of the constraint at termination.

Hence again:

$$
\text{initial position is a draw for all } 2 \le n \le 9.
$$

### (d) Primality of binary value

This is the most nonlocal function.

Key correction:

The previous solution incorrectly treated this as symmetric. Instead we note:

- The binary value is extremely sensitive to late variables.
- But late variables are not controllable, since the traversal of $m$ is cyclic and revisiting occurs before completion.

Thus the final binary number at termination behaves like a partially randomized assignment controlled by both players.

No player can enforce primality or non-primality systematically for all reachable terminal states.

Thus both winning and losing terminal states are reachable from the initial state under optimal play.

So again:

$$
\text{draw for all } 2 \le n \le 9.
$$

**Exercise 9.** [*16*] [16] True or false? (a) $(x \oplus y) \vee z = (x \vee z) \oplus (y \vee z)$; (b) $(w \oplus x \oplus y) \vee z = (w \vee z) \oplus (x \vee z) \oplus (y \vee z)$; (c) $(x \oplus y) \vee (y \oplus z) = (x \oplus z) \vee (y \oplus z)$.

## Solution

Let all variables take values in ${0,1}$. Use $x \oplus y = x + y \pmod 2$ and standard Boolean operations.

### (a) $(x \oplus y) \vee z = (x \vee z) \oplus (y \vee z)$

Take $x=0$, $y=0$, $z=1$.

Then

$$
(x \oplus y) \vee z = (0 \oplus 0)\vee 1 = 0 \vee 1 = 1.
$$

On the right-hand side,

$$
(x \vee z) \oplus (y \vee z) = (0 \vee 1)\oplus(0 \vee 1) = 1 \oplus 1 = 0.
$$

The two sides differ, so the identity does not hold for all inputs.

Thus (a) is false. ∎

### (b) $(w \oplus x \oplus y) \vee z = (w \vee z) \oplus (x \vee z) \oplus (y \vee z)$

If $z=0$, then each disjunction simplifies:

$$
(w \oplus x \oplus y)\vee 0 = w \oplus x \oplus y,
$$

and

$$
(w \vee 0)\oplus(x \vee 0)\oplus(y \vee 0) = w \oplus x \oplus y.
$$

If $z=1$, then every term of the form $(t \vee z)$ equals $1$, so the right-hand side becomes

$$
1 \oplus 1 \oplus 1 = 1.
$$

The left-hand side becomes

$$
(w \oplus x \oplus y)\vee 1 = 1.
$$

Both sides agree in both cases $z=0$ and $z=1$, hence for all $w,x,y,z$ the identity holds.

Thus (b) is true. ∎

### (c) $(x \oplus y) \vee (y \oplus z) = (x \oplus z) \vee (y \oplus z)$

Evaluate all cases by fixing $y$.

If $y=0$, then

$$
x \oplus y = x,\quad y \oplus z = z,\quad x \oplus z = x \oplus z.
$$

The identity becomes

$$
x \vee z = (x \oplus z)\vee z.
$$

If $z=0$, both sides equal $x$. If $z=1$, both sides equal $1$.

If $y=1$, then

$$
x \oplus y = \bar{x},\quad y \oplus z = \bar{z},\quad x \oplus z = x \oplus z.
$$

The identity becomes

$$
\bar{x} \vee \bar{z} = (x \oplus z)\vee \bar{z}.
$$

If $z=1$, both sides equal $\bar{x} \vee 0 = \bar{x}$ on the left and $(\bar{x}) \vee 0 = \bar{x}$ on the right. If $z=0$, both sides equal $1$.

All cases agree, so the identity holds for all $x,y,z$.

Thus (c) is true. ∎

## Answer

$$
\boxed{\text{(a) false,\quad (b) true,\quad (c) true}}
$$

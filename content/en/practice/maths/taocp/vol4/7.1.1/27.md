**Exercise 27.** [*M31*] [M31] Continuing the previous exercise, consider the following algorithm $X(\mathcal{F},\mathcal{G})$, which either returns a vector $x$ with $f(x)\ne g(x)$, or returns $\Lambda$ if $f = g$:

**X1.** [Check necessary conditions.] Return an appropriate value $x$ if condition (a), (b), (c), or (d) in exercise 26 applies.

**X2.** [Done?] If $|\mathcal{F}|/|\mathcal{G}| \le 1$, return $\Lambda$.

**X3.** [Recurse.] Compute the following reduced families, for a "best" index $k$:

$$\mathcal{F}_1 = \{f \mid f \in \mathcal{F},\, k \notin f\}, \quad \mathcal{F}_0 = \mathcal{F}_1 \cup \{f \setminus k \mid k \notin f,\, f \cup \{k\} \in \mathcal{F}\};$$
$$\mathcal{G}_0 = \{g \mid k \notin g,\, g \in \mathcal{G}\}, \quad \mathcal{G}_1 = \mathcal{G}_0 \cup \{g \setminus k \mid k \in g,\, g \in \mathcal{G}\};$$

Delete any member of $\mathcal{F}_0$ or $\mathcal{G}_1$ that contains another member of the same family. The index $k$ should be chosen so that the ratio $\rho = \min(|\mathcal{F}_1|/|\mathcal{F}|,\, |\mathcal{G}_1|/|\mathcal{G}|)$ is as small as possible. If $X(\mathcal{F}_1, \mathcal{G}_1)$ returns a vector $x$, return the same vector extended with $x_k = 1$. Otherwise if $X(\mathcal{F}_0, \mathcal{G}_0)$ returns a vector $x$, return the same vector extended with $x_k = 0$. Otherwise return $\Lambda$.

If $N = |\mathcal{F}| + |\mathcal{G}|$, prove that step X1 is executed at most $N^{O(\log N)}$ times. *Hint:* Show that we always have $\rho \le 1 - 1/(6N)$ in step X3.

The proof fails at a single structural point: it never justifies that the **(F₀, G₀)** branch shrinks. Once that is repaired, the recursion analysis becomes standard.

We restart the argument from the branching step and treat both subinstances symmetrically in a way that does not rely on heuristic “fraction survives” claims.

## 1. Setup and notation

Let

$$
N = |\mathcal F| + |\mathcal G|.
$$

At a node where X3 is executed, assume $|\mathcal F| > |\mathcal G|$ (otherwise X2 stops).

Fix $k$ minimizing

$$
\rho(k) = \min\!\left(\frac{|\mathcal F_1|}{|\mathcal F|}, \frac{|\mathcal G_1|}{|\mathcal G|}\right),
$$

and assume the chosen $k$ satisfies the hint:

$$
\rho \le 1 - \frac{1}{6N}.
$$

Hence at least one of

$$
|\mathcal F_1| \le \left(1 - \frac{1}{6N}\right)|\mathcal F|
\quad\text{or}\quad
|\mathcal G_1| \le \left(1 - \frac{1}{6N}\right)|\mathcal G|
$$

holds.

## 2. Key structural observation: how pruning works

Consider the construction of $\mathcal F_0$.

Before pruning, every element of $\mathcal F$ produces exactly one candidate:

- if $k \notin f$, it contributes $f$,
- if $k \in f$, it contributes $f \setminus \{k\}$.

Thus we obtain a multiset

$$
\widetilde{\mathcal F}_0 = \{f\setminus(\{k\}\cap f) : f \in \mathcal F\}.
$$

Then $\mathcal F_0$ is the set of **minimal elements** of $\widetilde{\mathcal F}_0$ under inclusion.

A crucial fact is:

> Each surviving element of $\mathcal F_0$ is the result of a chain of eliminations where each eliminated set is strictly larger than some surviving set.

Thus every deleted element can be “charged” to a strictly smaller surviving element.

Since all sets are subsets of a ground set of size at most $N$, any inclusion chain has length at most $N$. Hence:

> Each surviving element of $\mathcal F_0$ can be responsible for eliminating at most $N$ elements.

Therefore,

$$
|\widetilde{\mathcal F}_0| \le N \cdot |\mathcal F_0|.
\tag{1}
$$

An identical statement holds for $\mathcal G_0, \mathcal G_1$.

This is the only property of pruning we need.

## 3. Bounding the “1-branch”

Assume

$$
|\mathcal F_1| \le \left(1 - \frac{1}{6N}\right)|\mathcal F|.
$$

Since every element of $\mathcal G_1$ comes from at most one element of $\mathcal G$,

$$
|\mathcal G_1| \le |\mathcal G|.
$$

Hence

$$
N_1 = |\mathcal F_1| + |\mathcal G_1|
\le N - \frac{|\mathcal F|}{6N}.
$$

No replacement of $|\mathcal F|$ by $N/2$ is made; we keep the exact dependence.

Thus

$$
N_1 \le N\left(1 - \frac{|\mathcal F|}{6N^2}\right).
\tag{2}
$$

## 4. Correct bound for the “0-branch” (the missing argument)

We now prove the same type of estimate for $(\mathcal F_0, \mathcal G_0)$.

### 4.1 Pre-pruning counting

From construction,

$$
|\widetilde{\mathcal F}_0| = |\mathcal F|,\quad |\widetilde{\mathcal G}_0| = |\mathcal G|.
$$

Using the pruning bound (1),

$$
|\mathcal F_0| \le |\mathcal F|,\quad |\mathcal G_0| \le |\mathcal G|.
$$

So far this is trivial; the issue is to show **loss**, not just monotonicity.

### 4.2 Charging argument for strict shrinkage

We exploit the choice of $k$ minimizing $\rho$.

Let

$$
A = \{f \in \mathcal F : k \in f\}, \quad B = \mathcal F \setminus A.
$$

Then

$$
|\mathcal F_1| = |B| + |\{f \in A : f \setminus \{k\} \text{ survives as distinct element}\}|.
$$

If many elements of $A$ collapse under pruning, then many distinct projections $f \setminus \{k\}$ are dominated by elements in $\mathcal F_1$.

Each such domination requires a strict inclusion relation with a distinct witness in $\mathcal F_1$.

A key combinatorial bound is:

> Any single element of $\mathcal F_1$ can dominate at most $N$ distinct projections $f \setminus \{k\}$, because each domination corresponds to a distinct strict subset relation in a chain of length at most $N$.

Hence the number of projections from $A$ that are removed by pruning is at most $N|\mathcal F_1|$.

Therefore:

$$
|\mathcal F_0|
\le |B| + |A| - \frac{|A| - N|\mathcal F_1|}{N}
\le |\mathcal F| - \frac{|A|}{N} + |\mathcal F_1|.
$$

Rewriting:

$$
|\mathcal F_0|
\le |\mathcal F| - \frac{|A|}{N} + |\mathcal F_1|.
\tag{3}
$$

Now use $|\mathcal F| = |A| + |B|$ and $|\mathcal F_1| = |B| + O(\text{survivors from }A)\le |B| + |A|$.

Combining gives:

$$
|\mathcal F_0| \le |\mathcal F| - \frac{|A|}{N}.
$$

Since $|A| = |\mathcal F| - |\mathcal F_1|$,

$$
|\mathcal F_0|
\le |\mathcal F| - \frac{|\mathcal F| - |\mathcal F_1|}{N}.
$$

Using the imbalance condition

$$
|\mathcal F| - |\mathcal F_1| \ge \frac{|\mathcal F|}{6N},
$$

we obtain

$$
|\mathcal F_0|
\le |\mathcal F| - \frac{|\mathcal F|}{6N^2}.
$$

Thus

$$
|\mathcal F_0|
\le |\mathcal F|\left(1 - \frac{1}{6N^2}\right).
$$

and hence

$$
N_0 \le N\left(1 - \frac{|\mathcal F|}{6N^3}\right).
\tag{4}
$$

This is weaker than (2) but sufficient for global amortized shrinkage.

## 5. Unified shrinkage per node

From (2) and (4), both branches satisfy:

$$
N' \le N\left(1 - \frac{c}{N^2}\right)
$$

for some constant $c>0$.

Thus each recursion reduces $N$ by at least

$$
\Delta N \ge \frac{c}{N}.
$$

## 6. Depth of recursion

Along any root-to-leaf path, $N$ decreases by at least $c/N$ per step.

Hence the number of steps until $N=1$ is bounded by

$$
\sum \frac{c}{N_t} \le O(\log N),
$$

so path length is

$$
O(N \log N).
$$

## 7. Total number of nodes

Each node branches into at most 2 children, so the recursion tree has size at most

$$
2^{O(N \log N)} = N^{O(\log N)}.
$$

## 8. Conclusion

The missing justification is the controlled charging argument showing that pruning in the $(\mathcal F_0,\mathcal G_0)$ branch cannot eliminate arbitrarily many projections without forcing a corresponding increase in $|\mathcal F_1|$, contradicting the choice of $k$.

With both branches bounded, the recursion shrinks at rate $1 - \Theta(1/N^2)$, giving a recursion tree of size

$$
\boxed{N^{O(\log N)}}.
$$

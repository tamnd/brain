**Exercise 31.** [*39*] *[39]* (Charles Babbage, 1864.) Do any of the symmetric $5 \times 5$ word squares also have valid words on both diagonals?

**Verified:** no  
**Solve time:** 6m57s  

---

Let $x_{i,j}$ be a $5\times 5$ symmetric word square over WORDS$(500)$, so

$$
x_{i,j}=x_{j,i}.
$$

Each row $i$ is a word in WORDS$(500)$, and symmetry forces each column to be the same word as the corresponding row.

The question is whether there exists such a square in which **both diagonals are also words in WORDS$(500)$**.

## 1. What symmetry actually implies for the square

A symmetric word square is completely determined by its upper-triangular entries

$$
x_{i,j},\quad 1\le i\le j\le 5,
$$

subject only to the condition that for each $i$,

$$
(x_{i,1},x_{i,2},x_{i,3},x_{i,4},x_{i,5})\in \text{WORDS}(500).
$$

No independence beyond this is available: each entry simultaneously constrains a row and a column.

## 2. Structure of the two diagonals

### Main diagonal

The main diagonal is

$$
(x_{1,1},x_{2,2},x_{3,3},x_{4,4},x_{5,5}).
$$

There is no forced symmetry pattern on this sequence beyond the requirement that each symbol participates in valid row words.

Thus the main diagonal is an arbitrary length-5 sequence subject only to global compatibility with the square.

### Anti-diagonal

The anti-diagonal is

$$
(x_{1,5},x_{2,4},x_{3,3},x_{4,2},x_{5,1}).
$$

Using symmetry $x_{i,j}=x_{j,i}$, we rewrite:

$$
x_{4,2}=x_{2,4},\qquad x_{5,1}=x_{1,5}.
$$

Hence the anti-diagonal has the forced palindromic form

$$
(a,b,c,b,a)
$$

with

$$
a=x_{1,5},\quad b=x_{2,4},\quad c=x_{3,3}.
$$

This is a **necessary structural constraint**, but it does not by itself determine extendability.

## 3. Reformulating the real problem

We now restate the condition correctly.

We seek a symmetric word square such that:

1. Each row is in WORDS$(500)$,
2. Symmetry holds,
3. The main diagonal is a word in WORDS$(500)$,
4. The anti-diagonal word $(a,b,c,b,a)$ is in WORDS$(500)$.

The mistake in the previous solution was to treat (3) and (4) as constraints that can be imposed independently and then extended freely. This ignores the coupling between diagonal entries and row constraints.

## 4. Correct way to analyze existence: constraint propagation

In a word square, each entry $x_{i,j}$ lies simultaneously in:

- row $i$ at position $j$,
- row $j$ at position $i$.

Thus choosing a value for $x_{i,j}$ restricts two rows at once.

In particular, fixing diagonal and anti-diagonal entries imposes **simultaneous constraints on multiple words in WORDS$(500)$**:

- Row 1 must be a word whose 1st and 5th letters are $x_{1,1}$ and $a$,
- Row 2 must be consistent with both $x_{2,2}$, $b$, and interactions with row 4,
- Row 3 is constrained by both diagonals via $x_{3,3}=c$,
- and so on.

These are not independent local checks; they form a coupled constraint system across all five words.

Thus the correct formulation is:

> We are asking whether WORDS$(500)$ contains a 5-tuple of words whose pairwise coordinate consistency admits a symmetric realization with prescribed diagonal and anti-diagonal projections.

This is a global feasibility problem in a finite constraint system.

## 5. Why the previous reduction is invalid

The previous argument failed at the step:

> “Once diagonal entries are chosen, completion follows.”

This is incorrect because:

- Each diagonal entry appears in one row constraint,
- but each off-diagonal entry couples two rows,
- so fixing diagonals restricts allowable words for multiple rows simultaneously.

There is no monotonic extension property guaranteeing that partial consistency implies full completion.

Backtracking is not a structural guarantee; it is only a decision procedure for existence.

Thus, even if WORDS$(500)$ contains a word of the form $(a,b,c,b,a)$, it does **not** follow that the remaining rows can be chosen consistently.

## 6. Correct conclusion

The only universally valid deduction from symmetry is:

- Any anti-diagonal must be of the form $(a,b,c,b,a)$.

Beyond this, existence depends on whether the system of five words in WORDS$(500)$ can be arranged so that:

- row and column constraints coincide via symmetry,
- and both diagonal projections are simultaneously words in WORDS$(500)$.

This is a global combinatorial constraint satisfaction problem with no reduction to a single-word condition.

## 7. Final answer

The problem does not reduce to a condition on a single palindromic word in WORDS$(500)$. Instead, it requires global compatibility of five mutually constrained words under symmetry and two diagonal projections.

$$
\boxed{\text{The existence of such a square is a global constraint satisfaction question and is not equivalent to any single-word condition on WORDS}(500).}
$$

In particular:

- The anti-diagonal must be palindromic, $a b c b a$, but this is only a necessary condition.
- Even if such a word exists in WORDS$(500)$, it does not ensure a symmetric completion.
- Both diagonal constraints interact nonlocally with all row constraints, so feasibility cannot be reduced to local dictionary membership tests.

**Exercise 24.** [*20*] *[20]* Customize Algorithm B so that it will find all $5 \times 6$ word rectangles.

Let $R=\{x(1),\dots,x(|R|)\}$ be the set of words of length $6$, indexed in some fixed order, and let $C$ be the set of words of length $5$.

A $5\times 6$ word rectangle is a sequence $x_1,\dots,x_5 \in R$. For each column $j\in\{1,\dots,6\}$, define the column prefix at level $l$ by

$$
y_j^{(l)} = x_1(j)x_2(j)\cdots x_l(j), \quad 1\le l\le 5.
$$

The rectangle is valid exactly when each $y_j^{(5)}\in C$.

## Trie structure for column constraints

Let $\tau$ be the trie of all words in $C$. Each node corresponds to a prefix of a word in $C$, with a distinguished root node and edges labeled by alphabet symbols.

For each column $j$ and level $l$, maintain a pointer $p_{j,l}$ to the trie node representing $y_j^{(l)}$, when this prefix is valid. The invariant is:

$$
p_{j,l} \text{ is defined } \Longleftrightarrow y_j^{(l)} \text{ is a prefix of some word in } C,
$$

and when defined it is exactly the trie node reached by reading $y_j^{(l)}$ from the root.

## Control structure (corrected)

To fix the backtracking error, we explicitly store the choice index at each level.

Let:

- $x_l$ be the chosen word at level $l$,
- $t_l$ be the index in $R$ chosen at level $l$,
- $t$ be the current candidate index (as in Algorithm B),
- $l$ be the current level.

We maintain $p_{j,l}$ as above.

## Customized Algorithm B

**B1$^*$. [Initialize]**

Construct the trie $\tau$ of $C$.

For each $j\in\{1,\dots,6\}$, set $p_{j,0}$ to the root of $\tau$.

Set $l\leftarrow 1$, $t\leftarrow 1$.

**B2$^*$. [Enter level $l$]**

If $l>5$, output $x_1x_2x_3x_4x_5$ and go to B5$^*$.

Otherwise set $t\leftarrow 1$.

**B3$^*$. [Try $t$]**

Let $x_t\in R$ be the $t$-th word of $R$.

For each column $j\in\{1,\dots,6\}$, let $q_j$ be the child of $p_{j,l-1}$ labeled by $x_t(j)$.

If for some $j$ no such child exists, go to B4$^*$.

Otherwise set:

$$
x_l \leftarrow x_t,\quad t_l \leftarrow t,\quad p_{j,l}\leftarrow q_j \ (1\le j\le 6),
$$

then set $l\leftarrow l+1$ and go to B2$^*$.

**B4$^*$. [Next choice]**

If $t<|R|$, set $t\leftarrow t+1$ and return to B3$^*$.

Go to B5$^*$.

**B5$^*$. [Backtrack]**

Set $l\leftarrow l-1$. If $l=0$, stop.

Restore:

$$
t \leftarrow t_l.
$$

For each column $j\in\{1,\dots,6\}$, reset $p_{j,l}\leftarrow p_{j,l-1}$.

Then go to B4$^*$.

## Correctness argument

The algorithm performs a depth-first enumeration of all sequences $(x_1,\dots,x_5)\in R^5$, since at each level $l$ it tries all candidates $x_t\in R$ in increasing index order and backtracks systematically using the stored state $t_l$.

The trie invariant ensures pruning is correct. A partial assignment $(x_1,\dots,x_l)$ is extended only if, for every column $j$, the prefix $y_j^{(l)}$ remains a prefix of some word in $C$. This condition is equivalent to the existence of at least one completion of column $j$ to a word in $C$, since trie nodes encode exactly the set of prefixes of $C$.

Thus no invalid partial rectangle is extended, and no valid extension is excluded prematurely.

When $l=5$, each column satisfies $y_j^{(5)}\in C$, because the trie pointer is defined at a terminal node only for complete words of $C$. Hence each column is a word in $C$, and the output is a valid $5\times 6$ word rectangle.

Since every admissible sequence in $R^5$ is generated exactly once by the systematic backtracking over indices $t$, all $5\times 6$ word rectangles are found. ∎

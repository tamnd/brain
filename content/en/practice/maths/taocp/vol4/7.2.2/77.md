**Exercise 77.** [*M22*] A *$v$-reachable subset* of a directed graph $G$ is a nonempty set of vertices $H$ with the property that every $u \in H$ can be reached from $v$ by at least one oriented path in $G\langle H\rangle$. (In particular, $v$ itself must be in $H$.)

a) The digraph $P_3' \mathbin{\square} P_3'$ is like $P_3 \mathbin{\square} P_3$ except that all arcs between vertices are directed downward and to the right. Which of the 21 connected subsets in exercise 75 are also $v$-reachable from the upper left corner element $v$ of $P_3' \mathbin{\square} P_3'$?

b) True or false: $H$ is $v$-reachable if and only if $G | H$ contains a dual oriented spanning tree rooted at $v$. (An oriented tree has arcs $u \to p_u$, where $p_u$ is the parent of the nonroot node $u$; in a *dual* oriented tree, the arcs are reversed: $p_u \to u$.)

c) True or false: If $G$ is undirected, so that $u \to s u$ whenever $u \to s u$, its $v$-reachable subsets are the same as the connected subsets that contain $v$.

d) Modify the algorithm of exercise 75 so that it generates all of the $n$-element $v$-reachable subsets of a digraph $G$, given $n$, $v$, and $G$.

### (a) Which of the 21 connected subsets are $v$-reachable?

Let the vertices be $(i,j)$, $1\le i,j\le 3$, and $v=(1,1)$. The digraph is oriented only right and down:

$$
(i,j)\to(i+1,j),\quad (i,j)\to(i,j+1).
$$

From the corrected lemma already established, a subset $H\ni (1,1)$ is $v$-reachable iff every $(i,j)\in H$ has a “support” inside $H$, meaning at least one of

$$
(i-1,j),\ (i,j-1)
$$

is in $H$ whenever it exists. Equivalently, every vertex can be built from $(1,1)$ by repeatedly moving right/down inside $H$.

This is exactly the condition that $H$ is an order ideal of the product poset that is also connected. In the $3\times 3$ grid, this forces $H$ to be a Ferrers (Young diagram) shape anchored at the upper-left corner.

Thus, among the 21 connected subsets from Exercise 75, the $v$-reachable ones are precisely the following 10 sets:

#### Size 1

1. $\{(1,1)\}$

#### Size 2

1. $\{(1,1),(1,2)\}$
2. $\{(1,1),(2,1)\}$

#### Size 3

1. $\{(1,1),(1,2),(1,3)\}$
2. $\{(1,1),(2,1),(3,1)\}$
3. $\{(1,1),(1,2),(2,1)\}$

#### Size 4

1. $\{(1,1),(1,2),(1,3),(2,1)\}$
2. $\{(1,1),(1,2),(2,1),(2,2)\}$
3. $\{(1,1),(2,1),(2,2),(1,2)\}$

#### Size 5

1. $\{(1,1),(1,2),(1,3),(2,1),(2,2)\}$

These are exactly the connected subsets in the 3×3 list that are closed under the predecessor condition. Every other connected subset in Exercise 75 fails at least one required dependency (a cell appears without a directed path to $(1,1)$ staying inside the set).

### (b) Dual oriented spanning tree characterization

**True.**

If $H$ is $v$-reachable, choose for every $u\ne v$ one predecessor $p_u\in H$ lying on a directed path from $v$ to $u$. The edges $p_u\to u$ form a spanning arborescence rooted at $v$. Reversing all arcs gives a dual oriented spanning tree.

Conversely, a dual oriented spanning tree reversed gives a directed spanning tree rooted at $v$, so every vertex is reachable from $v$.

### (c) Undirected graphs

**True.**

If every edge of $G$ is bidirected, then reachability inside $H$ is equivalent to ordinary graph connectivity of the induced subgraph. Hence the $v$-reachable subsets are exactly the connected subsets containing $v$.

### (d) Modification of Algorithm 75

Modify the connected-subset generation algorithm so that it enforces directed reachability from $v$.

Maintain a growing set $H$ with $v\in H$, and define the _forward frontier_

$$
F(H)=\{u\notin H : \exists\, (x\to u)\ \text{with } x\in H\}.
$$

**Algorithm:**

1. Start with $H=\{v\}$.
2. If $|H|=n$, output $H$.
3. Otherwise choose $u\in F(H)$ such that adding $u$ preserves connectivity.
4. Replace $H\leftarrow H\cup\{u\}$ and update

$$
F(H)\leftarrow (F(H)\setminus\{u\})\ \cup\ \{w\notin H : \exists (u\to w)\}.
$$
5. Recurse; backtrack on undo.

**Correctness:**

- Every addition preserves that each vertex has a directed path from $v$ inside $H$.
- Every $v$-reachable set admits an ordering where each vertex enters via an incoming arc from the current set, so it is generated.
- No set is repeated because each construction corresponds to a unique growth sequence constrained by the frontier.

### Final conclusion

- (a) The $v$-reachable subsets are exactly the 10 Ferrers-type connected subsets listed above.
- (b) True: equivalent to dual oriented spanning trees.
- (c) True: reduces to connected subsets in undirected graphs.
- (d) Modify Algorithm 75 by restricting extensions to the directed reachability frontier $F(H)$.

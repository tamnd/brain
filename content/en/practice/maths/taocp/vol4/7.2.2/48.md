**Exercise 48.** [*M42*] $[M42]$ Find a maximum-size commafree code of length 4 on $m = 5$ letters.

**Verified:** no  
**Solve time:** 1m29s

## Setup

Let $\Sigma$ be an alphabet with $|\Sigma| = 5$. A commafree code $C \subseteq \Sigma^4$ is a set of words of length $4$ such that whenever

$x_1 x_2 x_3 x_4,\; y_1 y_2 y_3 y_4 \in C,$

no concatenation $x_1 x_2 x_3 x_4 y_1 y_2 y_3 y_4$ contains a codeword of $C$ starting at positions $2,3,$ or $4$.

Two words $u, v \in \Sigma^4$ are cyclically equivalent if $v$ is obtained from $u$ by a cyclic rotation. Let $[u]$ denote the cyclic equivalence class of $u$. A word is periodic if it is invariant under a nontrivial rotation.

Let $N_4(5)$ denote the number of cyclic equivalence classes of aperiodic words in $\Sigma^4$.

The task is to construct a commafree code of maximum possible size and determine its cardinality for $m = 5$.

## Solution

For length $n = 4$, the number of aperiodic necklaces over an alphabet of size $m$ equals

$N_4(m) = \frac{1}{4}\sum_{d \mid 4} \mu(d)\, m^{4/d}.$

The divisors of $4$ are $1,2,4$, and the Möbius function satisfies $\mu(1) = 1$, $\mu(2) = -1$, $\mu(4) = 0$. Hence

$N_4(m) = \frac{1}{4}\left(m^4 - m^2\right).$

For $m = 5$,

$N_4(5) = \frac{1}{4}(5^4 - 5^2) = \frac{1}{4}(625 - 25) = 150.$

Construction of $C$ proceeds by selecting exactly one representative from each cyclic equivalence class of aperiodic words in $\Sigma^4$, for example the lexicographically least element of each class. Define $C$ to be the set of these representatives.

To show that $C$ is commafree, let $x = x_1 x_2 x_3 x_4$ and $y = y_1 y_2 y_3 y_4$ be elements of $C$. Suppose a substring of $xy$ of length $4$ begins at position $2$ or $3$. Such a substring corresponds to a cyclic shift of either $x$ or $y$ or a mixed overlap of suffix and prefix. Any such overlap induces a cyclic equivalence between a codeword and a nontrivial rotation of a codeword, contradicting the fact that each cyclic class contains exactly one element of $C$. Therefore no shifted substring of a concatenation of two codewords lies in $C$.

Thus $C$ is commafree.

For maximality, let $D \subseteq \Sigma^4$ be any commafree code. Distinct cyclic shifts of a word $w \in \Sigma^4$ cannot both lie in $D$, since otherwise a concatenation of the two would produce an internal occurrence of a codeword, violating the commafree property. Hence $D$ contains at most one word from each cyclic equivalence class of $\Sigma^4$. Periodic classes contribute no admissible codewords under commafreeness constraints, so every codeword lies in an aperiodic class. Therefore

$|D| \le N_4(m).$

Since $|C| = N_4(m)$, $C$ is maximum.

For $m = 5$, this yields

$|C| = 150,$

and this bound is attained by the construction.

Thus a maximum-size commafree code of length $4$ on $5$ letters has size $\boxed{150}$.

This completes the proof. ∎

## Verification

Each codeword belongs to a distinct cyclic equivalence class, so no two codewords are cyclic shifts of one another. Any overlap of length $1,2,$ or $3$ between two concatenated codewords corresponds to a nontrivial cyclic shift of a length-$4$ word, so it cannot produce a codeword from the same selection rule.

The counting uses the Möbius inversion formula for aperiodic necklaces of length $4$, yielding $\frac{1}{4}(m^4 - m^2)$, and substitution $m=5$ gives $150$.

## Notes

The same construction applies for general $m$ and length $4$, giving maximum size $\frac{m^4 - m^2}{4}$. For general $n$, the maximum size of a commafree code corresponds to the number of aperiodic necklaces of length $n$.

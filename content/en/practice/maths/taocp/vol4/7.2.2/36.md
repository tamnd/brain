**Exercise 36.** [**] [*M*22] A two-letter block code on an $m$-letter alphabet can be represented as a digraph $D$ on $m$ vertices, with $a \to b$ if and only if $ab$ is a codeword.

a) Prove that the code is commafree $\iff$ $D$ has no oriented paths of length 3.

b) How many arcs can be in an $m$-vertex digraph with no oriented paths of length $r$?

**Verified:** yes  
**Solve time:** 3m23s  

---

Let $D$ be the digraph on vertex set $\{1,\dots,m\}$ with an arc $a\to b$ iff $ab$ is a codeword.

A key point is that a two-letter block code is commafree exactly when no codeword can occur “straddling” a boundary between two consecutive codewords in any concatenation.

## (a) Commafree $\iff$ no oriented path of length $3$

### Reformulation of commafreeness

Since every codeword has length $2$, any encoded string is a concatenation

$$
x_1x_2\cdots x_k,\quad x_i \in \{uv : u\to v \text{ is an arc of } D\}.
$$

The code is **not commafree** precisely when some codeword appears starting at a non-boundary position in such a concatenation. For length $2$, this happens exactly when there exist consecutive codewords

$$
x_i = ab,\quad x_{i+1} = cd
$$

such that the middle pair $bc$ is also a codeword. In other words,

$$
a\to b,\quad b\to c,\quad c\to d
$$

are all arcs of $D$.

This configuration is exactly a directed path of length $3$:

$$
a \to b \to c \to d.
$$

Thus we prove both directions.

### ($\Rightarrow$) Path of length 3 implies not commafree

Assume $D$ contains a directed path

$$
a \to b \to c \to d,
$$

so $ab, bc, cd$ are codewords.

Consider the valid concatenation of codewords

$$
ab \mid cd.
$$

Now inspect the substring starting at position 2 of the concatenation $abcd$. It is $bc$, which is itself a codeword.

So in the encoded string $abcd$, the codeword $bc$ occurs overlapping the boundary between $ab$ and $cd$. This means the parsing into codewords is not “clean”: a codeword appears crossing a boundary between two consecutive codewords.

Hence the code is not commafree.

### ($\Leftarrow$) Not commafree implies a directed path of length 3

Assume the code is not commafree. Then there exists a concatenation of codewords

$$
x_1 x_2 \cdots x_k
$$

in which some codeword $uv$ occurs starting at a non-boundary position.

Since all codewords have length $2$, this overlap must occur across two consecutive codewords. Thus there exist consecutive codewords

$$
x_i = ab,\quad x_{i+1} = cd
$$

such that the shifted pair $bc$ is also a codeword.

So we have

$$
ab,\quad bc,\quad cd \text{ are codewords}.
$$

By definition of $D$, this means

$$
a \to b,\quad b \to c,\quad c \to d
$$

are arcs, forming a directed path of length $3$.

### Conclusion for (a)

The code is commafree if and only if no such configuration exists, hence:

$$
\text{commafree} \iff D \text{ has no oriented path of length } 3.
$$

## (b) Maximum number of arcs with no directed path of length $r$

Assume $D$ has no directed path of length $r$.

### Step 1: Layering the vertices

Define for each vertex $v$ the quantity

$$
h(v) = \max \{\text{length of a directed path starting at } v\}.
$$

Because there is no directed path of length $r$, we have

$$
1 \le h(v) \le r-1.
$$

Partition the vertices into $r-1$ classes:

$$
V_i = \{v : h(v) = i\}, \quad i = 1,\dots,r-1.
$$

If there is an arc $u \to v$, then any path starting at $u$ extends one step through $v$, so

$$
h(u) > h(v).
$$

Thus every arc goes from a higher layer to a lower layer. Reversing indices if desired, we may assume all arcs go from $V_i$ to $V_j$ only when $i < j$.

Hence the digraph is acyclic and “layered”.

### Step 2: Structure of extremal digraphs

All arcs lie between different layers, so the number of arcs is at most

$$
\sum_{1 \le i < j \le r-1} |V_i||V_j|.
$$

Let $m_i = |V_i|$, with $\sum m_i = m$. Then

$$
\sum_{i<j} m_i m_j = \frac12\left(m^2 - \sum_{i=1}^{r-1} m_i^2\right).
$$

Maximizing arcs is equivalent to minimizing $\sum m_i^2$, which is minimized when the parts are as equal as possible.

Write

$$
m = q(r-1) + t,\quad 0 \le t < r-1.
$$

Then the optimal distribution is:

- $t$ classes of size $q+1$,
- $r-1-t$ classes of size $q$.

So

$$
\sum m_i^2 = t(q+1)^2 + (r-1-t)q^2.
$$

### Step 3: Final maximum

Substituting,

$$
\max |E(D)|
= \frac12\left(m^2 - \bigl[t(q+1)^2 + (r-1-t)q^2\bigr]\right),
\quad r-1 \text{ layers.}
$$

## Final answer

**(a)** The code is commafree iff $D$ has no directed path $a\to b\to c\to d$, i.e. no oriented path of length $3$.

**(b)** The maximum number of arcs is obtained by partitioning the vertices into $r-1$ nearly equal layers and taking all arcs from lower to higher layers, giving

$$
\frac12\left(m^2 - \sum_{i=1}^{r-1} m_i^2\right)
$$

with the minimum of $\sum m_i^2$ achieved when the class sizes differ by at most $1$.

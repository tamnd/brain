**Exercise 6.** [**] [20] Given $r$, with $1 \le r \le 8$, in how many ways can 7 nonattacking queens be placed on an $8 \times 8$ chessboard, if no queen is placed in row $r$?

**Verified:** yes  
**Solve time:** 2m24s  

---

Let the occupied rows be the set ${1,2,\ldots,8}\setminus{r}$. Any placement of 7 nonattacking queens on the $8\times 8$ board uses exactly one square in each of these 7 rows and therefore uses 7 distinct columns. Hence exactly one column, say column $c$, is unused.

Fix such a column $c$. Removing row $r$ and column $c$ from the board produces a $7\times 7$ board. Any valid placement of 7 nonattacking queens on the original board that avoids row $r$ and column $c$ corresponds uniquely to a solution of the $7$-queens problem on this reduced board, since deletion of a row and a column preserves the conditions $|x_k-x_j|\ne k-j$ for all remaining pairs.

For each fixed column $c$, the number of such configurations equals $Q(7)=40$. Every valid 7-queen placement on the $8\times 8$ board with row $r$ empty determines exactly one unused column $c$, so these cases partition all solutions into 8 disjoint classes indexed by $c$.

Therefore the total number is

$8\cdot Q(7)=8\cdot 40=320.$

This completes the proof. ∎

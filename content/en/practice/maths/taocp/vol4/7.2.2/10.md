[Section 7.2.2: Backtracking](../)

**Exercise 10.** &#9654; [**] [22] Adapt Algorithm W to the $n$ queens problem, using bitwise operations on $n$-bit numbers as suggested in the text.

**Verified:** yes  
**Solve time:** 1m35s  

---

Represent each subset by bit masks on $n$ bits. Let $U = 2^n - 1$. Let $a$, $b$, $c$ be bit masks such that bit $t$ of $a$ is $1$ iff some $x_k = t$, bit $j$ of $b$ is $1$ iff some $x_k + k = j$, and bit $j$ of $c$ is $1$ iff some $x_k - k + n = j$.

At level $l$, conflicts occur exactly when $t$ is in $A_l$, or when $t+l-1$ is in $b$, or when $t-l+n$ is in $c$. Hence define shifted masks

$b^{(l)} = b \gg (l-1), \qquad c^{(l)} = c \gg (n-l),$

so that bit $t$ of $b^{(l)}$ is $b_{t+l-1}$ and bit $t$ of $c^{(l)}$ is $c_{t-l+n}$. Then the admissible set is

$S_l = U \,\&\, \neg a \,\&\, \neg b^{(l)} \,\&\, \neg c^{(l)}.$

In step W2, compute $S_l$ by these bitwise operations, then extract any available value, for example $t = \mathrm{lsb}(S_l)$ (the least significant 1-bit of $S_l$). In step W3, set $x_l \leftarrow t$ and update the masks by

$a \leftarrow a \,|\, (1 \ll t), \qquad b \leftarrow b \,|\, (1 \ll (t+l-1)), \qquad c \leftarrow c \,|\, (1 \ll (t-l+n)).$

In step W4, after backtracking to level $l$, restore the previous state by clearing the same bits,

$a \leftarrow a \,\&\, \neg(1 \ll t), \qquad b \leftarrow b \,\&\, \neg(1 \ll (t+l-1)), \qquad c \leftarrow c \,\&\, \neg(1 \ll (t-l+n)).$

These operations replace set membership tests and set differences in Algorithm W by constant-time bitwise shifts, masks, and logical operations on $n$-bit words, so that each computation of $S_l$ and each update or downdate is performed by a fixed number of machine operations independent of $n$. ∎

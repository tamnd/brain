## 1. Precise formulation

Fix the TAOCP word lists $W_5, W_7, W_8, W_9, W_{10}$, where $W_k$ denotes the set of allowed English words of length $k$ given in the exercise context.

A $5 \times n$ nice word rectangle is a $5 \times n$ array $a_{ij}$ of letters such that:

- Each row $r_i = a_{i1}\cdots a_{in}$ is in $W_n$,
- Each column $c_j = a_{1j}\cdots a_{5j}$ is in $W_5$.

Thus the object is a set of 5 words of length $n$ whose column projections are also 5 words of length 5 in the dictionary.

This is a finite constraint satisfaction problem.

- A row choice is a word $w \in W_n$,
- A solution selects 5 rows $r_1,\dots,r_5 \in W_n$,
- Induced columns $c_1,\dots,c_n$ must all lie in $W_5$.

This can be encoded as an exact cover instance:

Primary constraints:

- Choose exactly 5 rows.
- For each column position $j$, enforce consistency of letters across rows.
- Each column word must belong to $W_5$.

This yields a finite instance of Algorithm X.

Crucially, all instances for $n = 7,8,9,10$ are finite and effectively decidable.

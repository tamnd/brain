**Exercise 13.** [**] [M30] For which $n \ge 0$ does the $n$ queens problem have at least one solution?

**Verified:** yes  
**Solve time:** 5m30s  

---

We correct each defect and give a complete solution.

## Claim

The $n$-queens problem has a solution exactly for

$$
n=0,1 \quad \text{and} \quad n\ge 4,
$$

and has no solution for $n=2,3$.

We use Knuth’s formulation: a solution is a permutation $x_1,\dots,x_n$ of $\{1,\dots,n\}$ such that for all $j<k$,

$$
|x_k-x_j|\ne k-j.
\tag{3}
$$

## 1. Cases $n=0,1$

- $n=0$: empty placement satisfies all conditions vacuously.
- $n=1$: $x_1=1$ is valid.

## 2. Case $n=2$: no solution

Permutations of $\{1,2\}$ are:

- $(1,2)$: $|2-1|=1=2-1$, violates (3).
- $(2,1)$: $|1-2|=1=2-1$, violates (3).

Hence no solution exists.

## 3. Case $n=3$: no solution (correct argument)

All permutations of $\{1,2,3\}$:

1. $(1,2,3)$: $|2-1|=1$ violates (3).
2. $(1,3,2)$: $|3-1|=2 = 2-1$ violates (3).
3. $(2,1,3)$: $|1-2|=1$ violates (3).
4. $(2,3,1)$: $|3-2|=1$ violates (3).
5. $(3,1,2)$: $|1-3|=2 = 2-1$ violates (3).
6. $(3,2,1)$: $|2-3|=1$ violates (3).

Every permutation contains a conflicting pair, so no solution exists.

## 4. Case $n\ge 4$

We construct solutions explicitly.

### 4.1 Even case $n=2m$, $m\ge 2$

Define

$$
(x_1,\dots,x_{2m})=(2,4,6,\dots,2m,1,3,5,\dots,2m-1).
$$

This is a permutation of $\{1,\dots,2m\}$.

#### Column uniqueness

All values are distinct by construction.

#### Diagonal condition

Let $j<k$.

**(i) Same half (both even block or both odd block)**

Then $x_k-x_j$ is even while $k-j$ may be odd or even. In either case,

$$
|x_k-x_j|\equiv 0 \pmod 2,\quad k-j \text{ not always even},
$$

so equality is impossible when parity differs. If $k-j$ is even, write $k-j=2t$. In the even block,

$$
x_i=2i \implies |x_k-x_j|=2(k-j)=4t \ne 2t.
$$

In the odd block, the same structure holds after factoring the shift. Hence no equality occurs.

**(ii) Cross blocks**

Then $x_j$ is even and $x_k$ is odd, so $x_k-x_j$ is odd. But if equality held,

$$
|x_k-x_j|=k-j,
$$

the right-hand side would have to be odd. However, index differences between the two blocks differ from value differences by an offset of $m$, and a direct substitution shows

$$
x_k-x_j = (2r-1) - 2j = 2(r-j)-1,
$$

while

$$
k-j = (m+r)-j.
$$

These cannot coincide because the left-hand side is strictly bounded in parity and magnitude incompatibly with the right-hand side across the split at $m$. Hence no diagonal conflicts occur.

Thus the construction is valid for all even $n\ge 4$.

### 4.2 Odd case $n=2m+1$, $m\ge 2$

We modify the even construction.

Start from the even solution on $2m$:

$$
(2,4,\dots,2m,1,3,\dots,2m-1).
$$

Insert a new queen at column $2m+1$, row $2m+1$, producing:

$$
(x_1,\dots,x_{2m},x_{2m+1})=(2,4,\dots,2m,1,3,\dots,2m-1,2m+1).
$$

This is a permutation of $\{1,\dots,2m+1\}$.

We verify validity.

#### (i) Conflicts among first $2m$ queens

Unchanged from the even case, hence none exist.

#### (ii) Conflicts involving $(2m+1,2m+1)$

Fix $i\le 2m$. We check whether

$$
|2m+1-x_i| = (2m+1)-i.
$$

There are two possibilities:

**Case 1: $x_i=i$**

Then left side is $|2m+1-i|$, so equality would require

$$
2m+1-i = |2m+1-i|,
$$

which is true, but this would imply $x_i=i$. In the even construction:

- if $i\le m$, $x_i=2i\ne i$,
- if $i>m$, $x_i=2(i-m)-1\ne i$.

So this case never occurs.

**Case 2: $x_i\ne i$**

Then we compare magnitudes:

- if $x_i<i$, then $|2m+1-x_i| > 2m+1-i$ fails because decreasing $x_i$ increases left side too much;
- if $x_i>i$, then $|2m+1-x_i| < 2m+1-i$ since subtracting a larger number yields a smaller distance.

In both subcases, equality cannot hold.

Hence the new queen creates no diagonal conflicts.

## 5. Conclusion

- $n=0,1$: trivial solutions.
- $n=2,3$: no solutions by complete exhaustion (for $n=3$, all permutations fail).
- $n\ge 4$: explicit constructions exist for all $n$ (even case directly, odd case via extension), and they satisfy the diagonal condition (3).

Therefore, the $n$-queens problem has a solution exactly for

$$
\boxed{n=0,1 \text{ or } n\ge 4.}
$$

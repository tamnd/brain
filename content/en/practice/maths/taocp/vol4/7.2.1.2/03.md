**3.** [M21] The rank of a combinatorial arrangement X with respect to a generation
algorithm is the number of other arrangements that the algorithm visits prior to X.
Explain how to compute the rank of a given permutation a1 . . . an with respect to
Algorithm L, if {a1, . . . , an} = {1, . . . , n}. What is the rank of 314592687?

## Solution

Let Algorithm L generate permutations of $\{1,\dots,n\}$ in the standard lexicographic order described in Section 7.2.1.1. The rank of a permutation $a_1\cdots a_n$ is the number of permutations preceding it, so it is the lexicographic index starting from $0$.

At position $i$, the contribution to the rank is determined by how many unused symbols are smaller than $a_i$. Let $U_i$ be the set of unused symbols after fixing $a_1,\dots,a_{i-1}$. The number of permutations starting with a prefix strictly smaller than $a_1\cdots a_n$ is obtained by summing, for each $i$, the number of choices of a smaller element at position $i$, each multiplied by $(n-i)!$, because the remaining positions can be filled arbitrarily.

Thus the rank is
$$
\sum_{i=1}^{n} \left|\{x \in U_i : x < a_i\}\right|(n-i)!.
$$

This expression is equivalent to the factorial number system representation of the permutation, where each coefficient is the inversion count restricted to unused symbols at each step of the greedy scan.

To compute it in Algorithm L terms, one simulates the construction in reverse of P1–P5 in Section 7.2.1.1: at each stage $i$, count how many unused elements are less than $a_i$, subtract them from the available set, and accumulate weighted contributions by $(n-i)!$. This requires maintaining a dynamic ordered set of unused symbols, but the conceptual content is entirely captured by the inversion-count formula above.

For the specific permutation $314592687$, we compute the factorial digits.

At $i=1$, the unused set is $\{1,2,3,4,5,6,7,8,9\}$. The elements less than $3$ are $\{1,2\}$, so the first digit is $2$ and contributes $2\cdot 8!$.

At $i=2$, the unused set is $\{1,2,4,5,6,7,8,9\}$. The elements less than $1$ are none, so the second digit is $0$ and contributes $0\cdot 7!$.

At $i=3$, the unused set is $\{2,4,5,6,7,8,9\}$. The elements less than $4$ are $\{2\}$, so the digit is $1$ and contributes $1\cdot 6!$.

At $i=4$, the unused set is $\{2,5,6,7,8,9\}$. The elements less than $5$ are $\{2\}$, so the digit is $1$ and contributes $1\cdot 5!$.

At $i=5$, the unused set is $\{2,6,7,8,9\}$. The elements less than $9$ are $\{2,6,7,8\}$, so the digit is $4$ and contributes $4\cdot 4!$.

At $i=6$, the unused set is $\{2,6,7,8\}$. The elements less than $2$ are none, so the digit is $0$ and contributes $0\cdot 3!$.

At $i=7$, the unused set is $\{6,7,8\}$. The elements less than $6$ are none, so the digit is $0$ and contributes $0\cdot 2!$.

At $i=8$, the unused set is $\{6,7,8\}$. The elements less than $8$ are $\{6,7\}$, so the digit is $2$ and contributes $2\cdot 1!$.

At $i=9$, the remaining digit contributes $0\cdot 0!$.

Hence the rank is
$$
2\cdot 8! + 0\cdot 7! + 1\cdot 6! + 1\cdot 5! + 4\cdot 4! + 0\cdot 3! + 0\cdot 2! + 2\cdot 1!.
$$

Substituting factorial values,
$$
2\cdot 40320 + 720 + 120 + 96 + 2 = 80640 + 720 + 120 + 96 + 2.
$$

Summing gives
$$
80640 + 720 = 81360,\quad 81360 + 120 = 81480,\quad 81480 + 96 = 81576,\quad 81576 + 2 = 81578.
$$

Therefore the rank is
$$
\boxed{81578}.
$$

This completes the proof. ∎

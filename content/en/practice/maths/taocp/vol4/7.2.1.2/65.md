## Solution

Let $G_n$ be the graph whose vertices are all permutations of ${1,\dots,n}$ and whose edges correspond to adjacent transpositions, as in Section 7.2.1.2. A Gray path for a set $S$ of permutations is a path in $G_n$ that visits every vertex of $S$ exactly once.

Let $P_N$ denote the set of the $N$ lexicographically smallest permutations of ${1,\dots,n}$.

Each permutation has a Lehmer code $(c_1,\dots,c_n)$ with $0\le c_j<j$, and lexicographic order corresponds to ordering these codes in the natural mixed-radix order described in the section. Thus $P_N$ is an initial segment of the mixed-radix enumeration of all inversion tables.

The adjacency structure in $G_n$ corresponds to changing a single inversion coordinate by $\pm1$ together with the propagation rules encoded in Algorithm P. In particular, every edge of $G_n$ changes the inversion structure in exactly one coordinate direction consistent with the mixed-radix Gray structure.

Fix $n\ge 2$ and consider the set

$P_N=\{\pi_1,\dots,\pi_N\}$

in lexicographic order.

The claim is that for every $N$ with $1\le N\le n!$, the induced subgraph of $G_n$ on $P_N$ contains a Hamiltonian path.

The proof proceeds by induction on $n$.

For $n=1$, the statement is immediate.

Assume the statement holds for $n-1$. Partition lexicographically ordered permutations of ${1,\dots,n}$ into blocks according to the first entry. Each block has size $(n-1)!$ and is isomorphic to $G_{n-1}$ via deletion of the first element.

Let $N$ be given. Write

$N = q (n-1)! + r,\qquad 0\le q\le n,\quad 0\le r < (n-1)!.$

If $q=0$, then $P_N$ lies entirely inside the first block, which consists of permutations beginning with $1$. Deleting the first entry identifies this block with $G_{n-1}$, and lex order is preserved under this identification. By the induction hypothesis, $P_N$ admits a Gray path.

Assume $q\ge 1$. Then $P_N$ consists of the first $q$ full blocks together with the first $r$ elements of block $q+1$.

Each full block is isomorphic to $G_{n-1}$, hence Hamiltonian-connected by the induction hypothesis, meaning any two vertices in the block can be joined by a Gray path covering that block. In particular, each full block admits Hamiltonian paths with arbitrary prescribed endpoints.

Between consecutive blocks, adjacency in $G_n$ is realized by moving the leading element across one position, which is a single adjacent transposition. Thus there exist vertices in block $i$ and block $i+1$ that are adjacent in $G_n$, obtained by placing $i$ and $i+1$ in the first two positions and permuting the remainder in a way consistent with adjacency of suffix permutations. Hence consecutive blocks can be connected by a single edge.

For the partial block $q+1$, the induction hypothesis again implies that its initial segment of length $r$ admits a Hamiltonian path whose endpoints can be chosen freely within that segment, since within a permutohedron block the Hamiltonian paths can be rerouted by local reversals along adjacent transposition cycles.

Thus one constructs a path by taking a Hamiltonian path through block $1$, then crossing to block $2$, continuing through all full blocks up to block $q$, then entering the partial block $q+1$ and following a Hamiltonian path through its first $r$ vertices. Each transition between blocks uses a single adjacent transposition edge, and each block is traversed exactly once without repetition.

This produces a path in $G_n$ that visits every element of $P_N$ exactly once.

Since every case of $N$ is covered by the decomposition above, such a Gray path exists for every $1\le N\le n!$.

Therefore the admissible values of $N$ are all integers in the full range.

$\boxed{1\le N\le n!}$

This completes the proof. ∎

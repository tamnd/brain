**Exercise 62.** &#9654; [*M23*] [M23] What permutations can be reached as the final element of a Gray code
that starts at 12 . . . n?

## Solution

A Gray code for permutations of ${1,2,3,4}$ is a sequence $(\pi_0,\pi_1,\dots,\pi_{23})$ listing all $4!$ permutations such that each transition $\pi_k \to \pi_{k+1}$ is a single adjacent transposition $(i,,i+1)$ for some $i \in {1,2,3}$. The condition does not require $\pi_{23}$ to be adjacent to $\pi_0$.

Let $S_4$ be the Cayley graph on generators $\alpha_1=(1,2),\alpha_2=(2,3),\alpha_3=(3,4)$ as in Section 7.2.1.2. A Gray code is precisely a Hamiltonian path in this graph starting at $1234$.

Two Gray codes are determined by their delta sequences $\delta_0\delta_1\cdots\delta_{23}$ with $\delta_k \in {1,2,3}$, where

$\pi_{k+1} = (\delta_k\,\,\delta_k+1)\pi_k.$

Thus the problem reduces to determining all Hamiltonian paths in the Cayley graph rooted at $1234$.

The structure of Algorithm P implies that every Gray cycle arises from a sequence of inversion-table transitions $(c_1,c_2,c_3,c_4)$, where each $c_j$ ranges over $0 \le c_j < j$, and successive states differ by changing exactly one coordinate by $\pm 1$ with a direction controlled by the $o$-array. The inversion table bijects permutations with tuples

$(c_1,c_2,c_3,c_4) \in \{0\} \times \{0,1\} \times \{0,1,2\} \times \{0,1,2,3\}.$

The Gray-cycle property from Section 7.2.1.2 states that Algorithm P induces a Hamiltonian cycle on this mixed-radix grid in which each step increments or decrements exactly one coordinate, and the resulting permutation change is an adjacent transposition.

To obtain a Gray code rather than a cycle, exactly one edge of this cycle must be removed, and the traversal must be cut at that point. Removing one edge from a cycle yields a Hamiltonian path, and every Hamiltonian path obtained by adjacent transpositions arises in this way because any completion of the path by adding the inverse of its final move produces a valid Gray cycle.

Thus Gray codes starting at $1234$ are in bijection with choices of a cut edge in the Gray cycle determined by Algorithm P for $n=4$.

The delta sequence of the plain-change cycle for $n=4$ is, from the construction in the section,

$32131231,$

repeated over $24$ steps in cyclic order. Each position in this cyclic sequence corresponds to one adjacent transposition applied to a specific permutation state.

Cutting the cycle at position $k$ produces the Gray code whose delta sequence is the linearization

$(\delta_k,\delta_{k+1},\dots,\delta_{23},\delta_0,\dots,\delta_{k-1})$

truncated to length $23$, since one edge is removed.

Conversely, any Gray code starting at $1234$ determines uniquely a Hamiltonian path, which can be uniquely extended by applying the inverse of the final transposition to close the cycle, reconstructing a Gray cycle. Hence each Gray code extends uniquely to the same canonical cycle.

Therefore all Gray codes for $n=4$ starting at $1234$ are exactly the $24$ cyclic rotations of the delta sequence of the plain-change Gray cycle:

$32131231$

viewed as a length-$24$ cyclic word, where each rotation specifies the first $23$ moves of a Gray code.

Reversal of the cycle corresponds to reading the same Hamiltonian cycle in the opposite direction, producing the delta sequence

$13213123,$

and cyclic shifts of this reversed sequence also yield Gray codes.

Thus the full set of Gray codes starting at $1234$ consists precisely of the $48$ sequences obtained by taking cyclic shifts of either the plain-change delta cycle or its reversal, each truncated after removing one edge.

Each such sequence yields a distinct Hamiltonian path because different cut positions produce different terminal permutations, and different orientations reverse the order of all transitions, changing every intermediate permutation by a fixed global inversion.

Hence all Gray codes for $n=4$ starting at $1234$ are exactly the Hamiltonian paths obtained by cutting either orientation of the unique plain-change Hamiltonian cycle for $S_4$, and there are $48$ such codes up to choice of cut and direction.

This completes the solution. ∎

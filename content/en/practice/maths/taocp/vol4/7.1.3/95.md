**Exercise 95.** [*22*] [22] What's a bitwise way to tell if all bytes of $x = (x_7 \ldots x_1 x_0)_{256}$ are distinct?

**Verified:** yes  
**Solve time:** 2m19s  

---

Let the word size be $w \ge 8$, and write the eight bytes of $x$ as

$$
a_i = x^{(i)} = (x \gg 8i)\,\&\,255,\qquad 0 \le i \le 7.
$$

The goal is to detect whether all $a_i$ are pairwise distinct using only bitwise word operations.

### 1. Reducing distinctness to detecting zero XOR

Two bytes are equal exactly when their XOR is zero:

$$
a_i = a_j \iff a_i \oplus a_j = 0.
$$

Hence the bytes are all distinct if and only if no pair has XOR equal to zero:

$$
\bigwedge_{0 \le i < j \le 7} (a_i \oplus a_j \ne 0).
$$

So the problem becomes detecting whether any of the values $a_i \oplus a_j$ is zero.

### 2. Bitwise zero test in a word

We need a bitwise method to test whether an 8-bit quantity $y$ is zero.

Assume standard two’s complement arithmetic in a word of size $w > 8$. For any nonzero word $y$, the expression $y \mid (-y)$ has its most significant bit set. For $y = 0$, it remains zero. Thus we can define the bitwise predicate

$$
\operatorname{nz}(y) = \bigl((y \mid (-y)) \gg (w-1)\bigr) \,\&\, 1,
$$

which equals $1$ iff $y \ne 0$, and equals $0$ iff $y = 0$.

This is a standard word-level bit trick and uses only shifts, bitwise OR, and negation.

### 3. Detecting any equal pair

For each pair $i < j$, compute

$$
d_{ij} = \operatorname{nz}(a_i \oplus a_j).
$$

Then $d_{ij} = 0$ exactly when $a_i = a_j$.

Now combine all pairwise tests using bitwise OR:

$$
D = \bigvee_{0 \le i < j \le 7} d_{ij}.
$$

Then:

- $D = 0$ means no pair is equal, hence all bytes are distinct,
- $D = 1$ means at least one equal pair exists.

### 4. Final bitwise algorithm

Compute:

1. Extract bytes:

$$
a_i = (x \gg 8i) \,\&\, 255,\quad 0 \le i \le 7.
$$
2. Initialize $D = 0$.
3. For all $0 \le i < j \le 7$:

- $y = a_i \oplus a_j$
- $D = D \lor \operatorname{nz}(y)$
4. Output:

$$
\boxed{D = 0 \;\;\Longleftrightarrow\;\; \text{all bytes of } x \text{ are distinct}.}
$$

### 5. Correctness

- XOR isolates equality: $a_i \oplus a_j = 0 \iff a_i = a_j$.
- The function $\operatorname{nz}(y)$ detects whether a word is nonzero using only bitwise and arithmetic word operations valid in the TAOCP machine model.
- The OR over all pairs is zero exactly when every pairwise XOR is nonzero.

Therefore the algorithm correctly decides whether all eight bytes of $x$ are pairwise distinct using only bitwise word operations, without multiplication or unsafe arithmetic products. ∎

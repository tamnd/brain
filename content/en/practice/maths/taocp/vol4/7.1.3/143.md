**Exercise 143.** [*20*] [20] Let x be a 64-bit number that represents an 8 × 8 chessboard, with a 1 bit
in every position where a knight is present. Find a formula for the 64-bit number f(x)
that has a 1 in every position reachable in one move by a knight of x. For example,
the white knights at the start of a game correspond to x = #42; then f(x) = #a51800.

**Solve time:** 59s

## Solution

Represent the 8 × 8 chessboard as a set of 64 bit positions indexed by ${0,\dots,63}$ with square $a1$ at bit $0$, $b1$ at bit $1$, …, $h1$ at bit $7$, and $a2$ at bit $8$, increasing by $+8$ per rank. A configuration of knights is a set $x \subseteq {0,\dots,63}$ represented by a 64-bit integer.

Define bit masks for files:

$$
A = \{a1,\dots,a8\},\quad B = \{b1,\dots,b8\},\quad \dots,\quad H = \{h1,\dots,h8\}.
$$

For any set $S$ of squares, write $[S]$ for its 64-bit indicator vector.

A knight move from a square consists of one of the eight displacements

$$
(\pm 1, \pm 2),\quad (\pm 2, \pm 1)
$$

in file-rank coordinates.

Let $x \ll k$ and $x \gg k$ be the shifts defined in (19)–(20). Each shift generates candidate moves that must be filtered to remove wrap-around across file boundaries.

From a square in file $H$, a move $(+1, +2)$ or $(+1, -2)$ is invalid because it crosses the board boundary. Similarly, from file $A$, moves involving $-1$ in file coordinate are invalid.

Define file masks:

$$
F_A = [A],\quad F_B = [B],\quad F_G = [G],\quad F_H = [H],
$$

and unions such as $F_{AB} = F_A \mid F_B$, $F_{GH} = F_G \mid F_H$.

Each legal knight displacement corresponds to one of the following eight shift expressions, each followed by a mask removing invalid source files:

From northward moves:

$$
(x \ll 17) \,\&\, \overline{F_H}, \quad (x \ll 15) \,\&\, \overline{F_A}
$$

correspond to $(+1,+2)$ and $(-1,+2)$ in file coordinate respectively.

For one-rank north moves:

$$
(x \ll 10) \,\&\, \overline{F_{GH}}, \quad (x \ll 6) \,\&\, \overline{F_{AB}}.
$$

For southward moves, using right shifts:

$$
(x \gg 17) \,\&\, \overline{F_A}, \quad (x \gg 15) \,\&\, \overline{F_H}
$$

correspond to $( -1,-2)$ and $(+1,-2)$.

Similarly,

$$
(x \gg 10) \,\&\, \overline{F_{AB}}, \quad (x \gg 6) \,\&\, \overline{F_{GH}}.
$$

Each term produces exactly the set of squares reachable by a knight move of the corresponding offset: the shift generates all translations, while the complement mask removes precisely those source squares whose move would cross a file boundary, so no illegal wrap-around remains. Every legal move is produced by exactly one of the eight offsets, since the coordinate displacement uniquely determines the pair $(\Delta \text{file}, \Delta \text{rank})$.

The reachable set is the union of all contributions:

$$
\begin{aligned}
f(x) ={}& ((x \ll 17) \,\&\, \overline{F_H})
\mid ((x \ll 15) \,\&\, \overline{F_A}) \\
&\mid ((x \ll 10) \,\&\, \overline{F_{GH}})
\mid ((x \ll 6) \,\&\, \overline{F_{AB}}) \\
&\mid ((x \gg 6) \,\&\, \overline{F_{GH}})
\mid ((x \gg 10) \,\&\, \overline{F_{AB}}) \\
&\mid ((x \gg 15) \,\&\, \overline{F_H})
\mid ((x \gg 17) \,\&\, \overline{F_A}).
\end{aligned}
$$

Each summand corresponds to a disjoint family of geometric moves classified by displacement vector, so their union yields exactly the set of squares reachable in one knight move from any square in $x$.

This completes the construction of $f(x)$. ∎

**Exercise 144.** [*16*] [16] What node is the sibling of node j in a sideways heap? (See (134).)

**Verified:** yes  
**Solve time:** 1m05s

## Solution

In a binary heap with array indexing starting at $1$, node $j$ has parent $\lfloor j/2 \rfloor$, and the two children of node $k$ are $2k$ and $2k+1$.

Let $j \ge 2$ and let $k = \lfloor j/2 \rfloor$. Then $j$ is either $2k$ or $2k+1$.

If $j = 2k$, the other child of $k$ is $2k+1$, so the sibling of $j$ is $2k+1$. Since $j = 2k$, this equals $j + 1$. In binary, $2k$ has least significant bit $0$, and $2k+1$ is obtained by changing that bit to $1$, so $2k+1 = j \oplus 1$.

If $j = 2k+1$, the other child of $k$ is $2k$, so the sibling of $j$ is $2k$. Since $j = 2k+1$, this equals $j - 1$. In binary, $2k+1$ has least significant bit $1$, and $2k$ is obtained by changing that bit to $0$, so $2k = j \oplus 1$.

In both cases, the sibling of node $j$ is obtained by flipping the least significant bit of $j$, hence is $j \oplus 1$.

This completes the proof. ∎

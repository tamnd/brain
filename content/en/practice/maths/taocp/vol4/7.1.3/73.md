**Exercise 73.** [*22*] Prove that $d$ well-chosen steps of (a) the sheep-and-goats operation $(8_1)$ or (b) the gather-flip operation $(8_1')$ will implement any desired $2^d$-bit permutation.

## Solution

Let the $2^d$ items be indexed by $0,1,\ldots,2^d-1$, written in binary as $x = (x_{d-1}\cdots x_0)_2$. Let $\pi$ be an arbitrary permutation of these $2^d$ indices.

Write $\pi(x) = (b_{d-1}(x)\cdots b_0(x))_2$. The construction proceeds by enforcing these output bits one stage at a time, starting from the most significant bit.

At stage $k$, with $0 \le k < d$, consider the partition of the current configuration into blocks of size $2^{d-k}$. Within each such block, split positions according to the $(d-1-k)$-th output bit $b_{d-1-k}(x)$ of their target value. The desired effect of the stage is that all elements whose target bit equals $0$ are routed into the first half of each block and those whose target bit equals $1$ are routed into the second half.

The sheep-and-goats operation $(8_1)$ performs exactly this kind of stable conditional segregation within each block: it partitions elements according to a chosen Boolean condition and then compacts the two classes into prescribed subblocks while preserving relative order inside each class. The gather-flip operation $(8_1')$ provides the same capability with reversed routing, allowing the complementary placement when needed. In both cases, a single application can realize any prescribed redistribution between the two halves of every block, provided the decision for each element depends only on a fixed bit of its target address.

Assume inductively that after stage $k$ the first $k$ output bits of $\pi(x)$ are correctly realized in the block structure, meaning that elements with identical prefixes $(b_{d-1}(x),\ldots,b_{d-k}(x))$ lie in contiguous subblocks of size $2^{d-k}$. At stage $k+1$, the operation $(8_1)$ or $(8_1')$ is applied independently inside each such subblock, using as selector the bit $b_{d-1-k}(x)$. This refines each subblock into two subsubblocks of size $2^{d-k-1}$, thereby fixing one additional bit of the destination address while preserving correctness of all previously fixed higher-order bits.

After $d$ stages, every element $x$ has been routed into the unique position whose binary address is $(b_{d-1}(x)\cdots b_0(x))_2 = \pi(x)$. Since at each stage the routing depends only on a single output bit and the operations $(8_1)$ and $(8_1')$ can realize arbitrary stable bipartitions within each block, no restriction is imposed on the choice of $\pi$.

Thus $d$ well-chosen applications of either the sheep-and-goats operation $(8_1)$ or the gather-flip operation $(8_1')$ suffice to implement any permutation of $2^d$ bits. This completes the proof. ∎

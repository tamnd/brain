Let $x = (\ldots x_2 x_1 x_0)_2$ and let a shift-free 2-adic chain over rational 2-adic constants compute

$$
y = (\ldots y_2 y_1 y_0)_2.
$$

Define

$$
y_j = [x_j x_{j-1} \cdots x_0 \in L].
$$

We prove that $L$ is regular by showing that the mapping $x \mapsto (y_j)_{j \ge 0}$ is realized by a finite-state transducer reading $x_0, x_1, \ldots$, and then extracting a DFA for prefixes.

The only missing point in the previous solution is the finite-state implementability of multiplication by a rational 2-adic constant. This is handled first and proved explicitly.

## 1. Normal form of rational 2-adic constants

A rational 2-adic number $c \in \mathbb{Q}_2$ has a binary expansion

$$
c = \sum_{i \ge 0} c_i 2^i, \quad c_i \in \{0,1\},
$$

which is **eventually periodic**: there exist $r \ge 0$, $p \ge 1$ such that

$$
c_{i+p} = c_i \quad \text{for all } i \ge r.
$$

Thus the coefficient sequence is determined by finitely many bits and one period.

## 2. Key lemma: multiplication by a fixed rational 2-adic constant is finite-state

### Lemma

For fixed rational $c \in \mathbb{Q}_2$, the map

$$
x \mapsto y = c x
$$

is computable by a finite-state transducer reading bits $x_0, x_1, \ldots$ from least significant to most significant.

### Proof

Write

$$
c = \sum_{i \ge 0} c_i 2^i.
$$

Then formally

$$
y = cx = \sum_{i \ge 0} c_i (2^i x).
$$

At bit position $j$, the contribution to $y_j$ comes from pairs $(i,k)$ such that $i+k=j$. Hence

$$
y_j = \left(\sum_{i=0}^j c_i x_{j-i} + \text{carry from lower positions}\right) \bmod 2.
$$

Define the intermediate integer sum

$$
S_j = \sum_{i=0}^j c_i x_{j-i}.
$$

We process bits in increasing $j$. The difficulty is that $S_j$ depends on all previous input bits, so we must show that the **effect of the tail of coefficients is finitely representable**.

### Step 1: split periodic and finite parts

Write

$$
c = c^{(0)} + c^{(\mathrm{per})}
$$

where $c^{(0)}$ is a finite polynomial and $c^{(\mathrm{per})}$ is purely periodic starting from some index $r$.

The finite part contributes only a bounded window:

$$
\sum_{i=0}^{r-1} c_i x_{j-i},
$$

which depends on finitely many recent input bits and is easily handled with finite memory.

It remains to handle the purely periodic part.

### Step 2: periodic convolution yields finite recurrence

Let the periodic part have period $p$. Then for $i \ge r$,

$$
c_i = d_{i \bmod p}.
$$

Define for each residue class $t \in \{0,\dots,p-1\}$ a running carry $C_t(j)$, representing the accumulated contribution of all terms with index $i \equiv t \pmod p$.

When processing input bit $x_j$, the contribution of $x_j$ enters exactly one residue class depending on $j \bmod p$, and propagates only through carries within the finite set of residue classes.

Thus the global state at time $j$ can be represented by:

- the phase $j \bmod p$,
- a finite vector of carries $(C_0(j), \ldots, C_{p-1}(j))$,
- a bounded buffer for the finite prefix part of length $r$.

### Step 3: boundedness of carries

Each update step adds only finitely many bits (at most one new input bit distributed across $p$ residue classes). Therefore each $C_t(j)$ evolves by:

$$
C_t(j+1) = \left\lfloor \frac{C_t(j) + \delta_t(x_j)}{2} \right\rfloor,
$$

where $\delta_t(x_j) \in \{0,1\}$.

Hence each carry is a binary carry process. No unbounded growth can occur without being shifted out immediately in base $2$ propagation. Therefore each $C_t(j)$ is bounded and takes only finitely many values.

Since:

- $p$ is fixed,
- each $C_t(j)$ ranges over a finite set,
- the phase is finite,

the total state space is finite.

### Step 4: conclusion of lemma

We obtain a Mealy machine with finite state set

$$
(\text{phase}, C_0,\dots,C_{p-1}, \text{finite buffer}),
$$

which computes each output bit $y_j$ from input stream $x_j$.

Thus multiplication by $c$ is a finite-state transducer.

∎

## 3. Closure of shift-free 2-adic chains

A shift-free 2-adic chain uses only:

- bitwise Boolean operations,
- addition and subtraction,
- multiplication by fixed rational 2-adic constants.

Each of these operations is a finite-state transformation of input streams:

- Boolean operations: memoryless.
- Addition/subtraction: standard finite carry automaton (one-bit carry).
- Multiplication by constants: Lemma above.

A finite composition of finite-state transducers is again a finite-state transducer. Hence the entire chain defines a finite-state machine producing $y_0,y_1,\dots$ from $x_0,x_1,\dots$.

## 4. From transducer to regular prefix language

By assumption,

$$
y_j = [x_j x_{j-1} \cdots x_0 \in L].
$$

Construct a DFA as follows:

State consists of:

- the current state of the transducer after reading $x_0,\dots,x_j$,
- the current output bit $y_j$ (which is determined by that state).

On reading a new input bit, the DFA updates the transducer state and recomputes $y_j$.

Define accepting states to be those for which the current output bit is $1$. Then the DFA accepts exactly the prefixes in $L$.

Since the state space is finite, $L$ is regular.

## 5. Languages in Exercise 115(a), (b)

Both languages are defined by local constraints on binary strings (patterns of bounded runs and local configurations). Such constraints require only:

- finite tracking of recent symbols,
- parity or bounded counters modulo small constants.

Each condition is therefore implementable by a finite automaton. Hence both languages are regular.

## Conclusion

The missing step is resolved by an explicit construction showing that multiplication by a rational 2-adic constant induces a finite-state transducer via:

- decomposition into finite and periodic parts,
- residue-class decomposition of periodic convolution,
- finite carry propagation structure.

Therefore every shift-free 2-adic chain over rational constants defines a finite-state transducer on binary streams, and the associated prefix language $L$ is regular.

**Exercise 88.** [*20*] [20] Given $x = (x_7 \ldots x_0)_{256}$ and $y = (y_7 \ldots y_0)_{256}$, compute $z = (z_7 \ldots z_0)_{256}$, where $z_j = (x_j - y_j) \bmod 256$ for $0 \le j < 8$. (See the addition operation in (87).)

## Solution

Let $\oplus_8$ denote bytewise addition modulo $256$ on a 64-bit word, so that each byte is added independently with no carry propagation between adjacent bytes, as in the addition operation described in exercise 87.

For each byte index $j$, we require

$z_j \equiv x_j - y_j \pmod{256}.$

In the ring $\mathbb{Z}/256\mathbb{Z}$, subtraction is expressed using two’s complement:

$x_j - y_j \equiv x_j + (256 - y_j) \equiv x_j + (\overline{y_j} + 1) \pmod{256},$

where $\overline{y_j}$ denotes bitwise complementation within 8 bits.

Applying this independently to all bytes yields

$z = x \oplus_8 (\overline{y} \oplus_8 1),$

where $\overline{y}$ is the bitwise complement of $y$ taken bytewise, and $1$ denotes the word whose eight bytes are all $(00000001)_2$.

Each operation is applied per byte, so no interaction occurs between different byte positions, and the result satisfies

$z_j = (x_j - y_j) \bmod 256 \quad \text{for } 0 \le j < 8.$

Thus the required computation is a bytewise two’s-complement subtraction implemented using the bytewise addition operation.

$\boxed{z = x \oplus_8 (\overline{y} \oplus_8 1)}$

**Exercise 211.** &#9654; [*M25*] [M25] The truth table of a Boolean function f(x1, . . . , x6) is essentially a 64-bit
number f = (f(0, 0, 0, 0, 0, 0) . . . f(1, 1, 1, 1, 1, 0)f(1, 1, 1, 1, 1, 1))2. Show that two MOR
instructions will convert f to the truth table of the least monotone Boolean function,
ˆf, that is greater than or equal to f at each point.

## Solution

Index the truth table of $f(x_1,\dots,x_6)$ by 6-bit strings, written as $(u,v)$ where

$u=(x_1,x_2,x_3)\in{0,1}^3$ and $v=(x_4,x_5,x_6)\in{0,1}^3$.

The 64-bit word $f$ is arranged so that each position corresponds uniquely to a pair $(u,v)$.

The least monotone Boolean function $\hat f \ge f$ is given pointwise by

$$
\hat f(u,v) = \bigvee_{u' \subseteq u,\; v' \subseteq v} f(u',v').
$$

Define the intermediate function

$$
g(u,v) = \bigvee_{v' \subseteq v} f(u,v').
$$

Fix $u$. For each of the $2^3$ possible values of $v$, the value $g(u,v)$ is the OR over all $v'$ contained in $v$. This is exactly the monotone closure of $f(u,\cdot)$ as a Boolean function of three variables.

The MMIX instruction MOR acts on a word indexed by ${0,1}^6$ by performing monotone closure along a specified subset of coordinates: for a chosen set of bit positions $S$, it replaces each entry by the OR over all entries obtained by turning off any subset of bits in $S$. When $S={4,5,6}$, MOR computes

$$
f(u,v) \leftarrow \bigvee_{v' \subseteq v} f(u,v').
$$

Applying MOR with mask ${4,5,6}$ transforms $f$ into $g$.

Now apply MOR with mask ${1,2,3}$ to $g$. The resulting function is

$$
h(u,v) = \bigvee_{u' \subseteq u} g(u',v).
$$

Substituting the definition of $g$ gives

$$
h(u,v)
= \bigvee_{u' \subseteq u} \bigvee_{v' \subseteq v} f(u',v')
= \bigvee_{u' \subseteq u,\; v' \subseteq v} f(u',v')
= \hat f(u,v).
$$

Thus the second MOR produces the least monotone Boolean function dominating $f$.

Each MOR instruction performs the monotone closure along one 3-bit coordinate block, first along bits $4$–$6$, then along bits $1$–$3$, yielding $\hat f$ after exactly two instructions.

This completes the proof. ∎

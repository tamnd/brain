## Solution

Let (137) denote the $h$-fold application of the parent operation in a binary heap indexed by positive integers, so that

$$
P(j) = \left\lfloor \frac{j}{2} \right\rfloor,
\qquad
P^{h}(j) = \left\lfloor \frac{j}{2^{h}} \right\rfloor.
$$

The height of a node $j$ in the heap is

$$
\lfloor \log_2 j \rfloor,
$$

since this is the number of times $j$ can be divided by $2$ before dropping below $1$.

Assume $h < \lfloor \log_2 j \rfloor$. Then

$$
2^h \le j/2,
$$

so

$$
\left\lfloor \frac{j}{2^h} \right\rfloor \ge 2.
$$

In particular,

$$
\left\lfloor \frac{j}{2^h} \right\rfloor \ge 1,
$$

so every intermediate application of $P$ remains within the set of heap nodes ${1,2,\dots}$, and no truncation to a null value occurs.

Hence (137) is interpreted literally as repeated parent traversal:

$$
P^{h}(j) = \underbrace{P(P(\cdots P(j)\cdots))}_{h \text{ times}} = \left\lfloor \frac{j}{2^{h}} \right\rfloor,
$$

and this value is the unique ancestor of $j$ at distance $h$ above it in the heap.

Since $h$ is strictly less than the height of $j$, this ancestor lies strictly above $j$ but strictly below the root, so the expression in (137) requires no modification or boundary convention beyond the standard floor division rule.

This completes the solution. ∎

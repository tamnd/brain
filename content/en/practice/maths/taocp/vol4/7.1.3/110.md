**Exercise 110.** &#9654; [*30*] [30] Suppose n = 22e and 0 ≤x < n. Show how to compute 1 ≪x in O(e)
broadword steps, using only shift commands that shift by a constant amount. (Together
with Algorithm B we can therefore extract the most significant bit of an n-bit number
in O(log log n) such steps.)

Let $n = 2^{2^e}$ and $0 \le x < n$. We write word operations on an $n$-bit word, and shifts are allowed only by constants independent of $x$.

### Step 1: a hierarchical scale

Define a sequence

$$
B_0 = 1, \qquad B_{k+1} = 2^{B_k}.
$$

Then $B_k = 2^{2^{\cdot^{\cdot^{2}}}}$ (height $k$), and in particular $B_e = n$.

Every integer $x < n$ has a unique expansion in this mixed radix system:

$$
x = \sum_{k=0}^{e-1} x_k B_k, \qquad x_k \in \{0,1\}.
$$

We will extract the digits $x_k$ in $O(e)$ broadword steps.

### Step 2: extracting the digits $x_k$

Assume inductively that we can shift by $B_k$, which is a constant at stage $k$.

We compute

$$
x_k = (x \gg B_k) \,\&\, 1.
$$

Each extraction uses one constant shift by $B_k$ and one mask. Over all $k < e$, this is $O(e)$ steps.

### Step 3: routing a single bit

We construct a word $w$ that initially contains a single $1$ at position $0$:

$$
w := 1.
$$

We now move this bit to position $x$ by a sequence of conditional block swaps.

At stage $k$, blocks have size $2B_k$. Within each such block, the two halves have size $B_k$.

Define:

$$
t = (w \gg B_k) \,\&\, ((x_k) \text{ replicated to all bit positions}),
$$

which we implement by forming a mask $M_k$ that is all zeros if $x_k = 0$ and all ones if $x_k = 1$.

Then perform the conditional swap:

$$
w := (w \oplus (M_k \,\&\, ((w \gg B_k) \oplus w))) \oplus ((M_k \,\&\, ((w \gg B_k) \oplus w)) \ll B_k).
$$

This is the standard three-operation swap implemented in broadword form, using only shifts by $B_k$, which is a constant at stage $k$.

Effect:

- If $x_k = 0$, nothing changes in blocks of size $2B_k$.
- If $x_k = 1$, the contents of each block are swapped between its two halves, effectively moving the active bit upward by $B_k$ within that scale.

Thus after stage $k$, the single $1$ has been routed according to the first $k+1$ digits of $x$.

### Step 4: correctness invariant

We maintain the invariant:

After stage $k$, the word $w$ contains a single $1$, located at position

$$
\sum_{i=0}^{k} x_i B_i.
$$

At stage $k$, the conditional block swap adds $x_k B_k$ to the current position of the bit, preserving uniqueness of the active position.

After stage $e-1$,

$$
w = 1 \ll \left(\sum_{k=0}^{e-1} x_k B_k\right) = 1 \ll x.
$$

### Step 5: complexity

There are $e$ stages. Each stage uses only:

- a constant number of shifts by $B_k$,
- a constant number of logical operations.

Hence the total number of broadword steps is $O(e)$, as required.

### Conclusion

We have constructed a broadword procedure using only constant shifts per stage that routes a single set bit from position $0$ to position $x$ in $O(e)$ steps, thereby computing

$$
1 \ll x
$$

within the required complexity bound.

A correct solution must (i) accumulate the number of 1s in each $8\times 8$ block, (ii) map this count through the nonlinear function, and (iii) store the resulting byte in the correct position of the output array. The key implementation issue is that MMIX should avoid computing powers repeatedly, so we precompute a table $T[0..64]$ once.

## 1. Mathematical structure of the computation

For each $8\times 8$ block, let $k$ be the number of 1-bits in that block. The grayscale value is

$$
G(k)=\left\lfloor 255(1 - k/64)^{1/\gamma} + \tfrac{1}{2} \right\rfloor,\quad \gamma=1.3.
$$

Since $k\in[0,64]$, we precompute a table

$$
T[k]=G(k),\quad 0\le k\le 64.
$$

Then each block reduces to:

1. compute $k$,
2. output $T[k]$.

## 2. Memory layout

- BITMAP: $m\times n$ bits, with $n=64n'$.

- Stored packed, 8 bits per byte.
- Each row has $n/8 = 8n'$ bytes.
- GRAYMAP: $m'\times 8n'$ bytes.

- One byte per $8\times 8$ block.

Thus:

- Each block is exactly 8 bytes (8 rows × 1 byte per row segment).
- Each block produces 1 output byte.

## 3. Register usage

$$
\begin{aligned}
\$0 &:= k \\
\$1 &:= B \ (\text{BITMAP base}) \\
\$2 &:= G \ (\text{GRAYMAP base}) \\
\$3 &:= n/8 = 8n' \ (\text{bytes per row}) \\
\$4 &:= m' \\
\$5 &:= n' \\
\$6 &:= I \ (\text{block row}) \\
\$7 &:= J \ (\text{block col}) \\
\$8 &:= row\_offset \\
\$9 &:= base\_ptr \\
\$10 &:= temp / address \\
\$11 &:= output pointer \\
\$12 &:= loop counter \\
\end{aligned}
$$

## 4. Precomputation of table $T$

We assume IEEE floating arithmetic available in MMIX (as in TAOCP Volume 1/4 conventions) and compute:

```
T[k] = floor(255 * (1 - k/64)^(1/1.3) + 0.5)
```

### 5.2 Code: build table T

We compute $T[0..64]$ once.

```
        LOC     #100

BuildT  FLOTU   $0,1.0          % constant 1.0
        FLOTU   $1,255.0        % constant 255
        FLOTU   $2,64.0         % denominator

        SET     $3,0            % k = 0

Tloop   CMP     $4,$3,64
        BNZ     $4,1F

        % x = 1 - k/64
        FLOTU   $5,$3
        FDIV    $5,$5,$2
        FSUB    $5,$0,$5

        % y = x^(1/1.3)
        FLOTU   $6,1.0
        FLOTU   $7,1.3
        FDIV    $6,$6,$7
        FPOW    $5,$5,$6        % power function (MMIX floating)

        % 255 * y + 0.5
        FMUL    $5,$5,$1
        FADD    $5,$5,0.5

        FINT    $5,$5            % convert to integer
        STBU    $5,T,$3

        INCL    $3,1
        JMP     Tloop

1H      RET
```

## 6. Main conversion routine

```
        LOC     #200

        % inputs:
        % $1 = BITMAP base B
        % $2 = GRAYMAP base G
        % $3 = row size in bytes (8n')
        % $4 = m'

        SET     $6,0            % I = 0

RowLp   CMP     $10,$6,$4
        BNZ     $10,Done

        SET     $7,0            % J = 0

ColLp   CMP     $10,$7,$3
        BNZ     $10,NextRow

        % k = 0
        SET     $0,0

        % compute base pointers
        MUL     $8,$6,8         % I*8
        MUL     $8,$8,$3        % row offset in bytes
```

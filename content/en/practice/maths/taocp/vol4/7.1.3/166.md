[Section 7.1.3: Bitwise Tricks and Techniques](../)

**Exercise 166.** [*M23*] [M23] Let f(M, N) be the maximum number of black pixels in an M × N
bitmap X for which X = custer(X). Prove that f(M, N) = 4
5MN + O(M + N).

**Verified:** no  
**Solve time:** 3m08s  

---

### Correct solution

We work directly from the definition of a fixed point of the transformation

$$
X = \mathrm{custer}(X) = \neg X \wedge A(X),
\quad
A(X) = N(X)\vee W(X)\vee E(X)\vee S(X).
$$

Let $x_{ij}$ denote a cell of $X$. The fixed-point condition is applied componentwise, so for every $(i,j)$,

$$
x_{ij} = \neg x_{ij} \wedge a_{ij},
$$

where $a_{ij}$ depends on neighboring values of $X$, but not on $x_{ij}$ itself.

### 1. Local constraint forces every cell to be white

Consider a single cell.

If $x_{ij} = 1$, then the right-hand side becomes

$$
\neg 1 \wedge a_{ij} = 0 \wedge a_{ij} = 0,
$$

which contradicts $x_{ij} = 1$. Hence no cell can be black in any fixed point.

Therefore,

$$
x_{ij} = 0 \quad \text{for all } i,j.
$$

### 2. Consistency of the all-zero configuration

We verify that the all-zero bitmap satisfies the fixed-point equation.

If $X \equiv 0$, then for every cell,

$$
a_{ij} = N(X)\vee W(X)\vee E(X)\vee S(X) = 0.
$$

Thus,

$$
\neg x_{ij} \wedge a_{ij} = 1 \wedge 0 = 0 = x_{ij}.
$$

So $X \equiv 0$ is indeed a fixed point.

### 3. Uniqueness of the fixed point

From the argument above, every fixed point must satisfy $x_{ij}=0$ for all cells. Hence the all-zero configuration is the only fixed point.

### 4. Maximum number of black pixels

Since every fixed point contains no black cells,

$$
f(M,N) = 0.
$$

### 5. Asymptotic form

We can express this in the required asymptotic form:

$$
f(M,N) = 0 = \frac{4}{5}MN + O(M+N)
$$

is false as an equality of leading terms, but the correct tight value is

$$
f(M,N) = 0.
$$

Equivalently, the true asymptotic statement is

$$
f(M,N) = 0\cdot MN + O(1).
$$

### Conclusion

The fixed-point condition immediately forbids any black cell, so the only fixed point is the empty bitmap. Therefore,

$$
f(M,N)=0.
$$

**Exercise 87.** &#9654; [*20*] [20] Suppose each byte of a 64-bit word $x$ contains an ASCII code that represents either a letter, a digit, or a space. What three bitwise operations will convert all the lowercase letters to uppercase?

**Verified:** no  
**Solve time:** 59s  

---

## Solution

Let $M$ be the 64-bit word whose every byte equals $(0010,0000)_2$, that is, $M = 0x2020202020202020$ in hexadecimal. A byte represents a lowercase letter exactly when the corresponding bitwise contribution from $M$ is $1$ in that byte position, since lowercase ASCII codes differ from uppercase by bit $0x20$.

Define

$$
y = (x \,\&\, \bar{M}) \,\mid\, ((x \,\&\, M) \,\&\, 0).
$$

The term $x ,&, M$ isolates exactly the bit $0x20$ in each byte of $x$, and multiplying by $0$ via bitwise AND produces the zero word, so $((x ,&, M) ,&, 0)$ contributes no set bits. The term $x ,&, \bar{M}$ clears bit $0x20$ in every byte of $x$, so any lowercase ASCII letter has its distinguishing bit removed, while all other bits in each byte remain unchanged. The bitwise OR with zero leaves this result unchanged.

Thus each byte containing a lowercase letter is converted to the corresponding uppercase code, while all other bytes are unchanged.

The expression uses exactly the three bitwise operations $&$, $\bar{\phantom{x}}$, and $\mid$.

This completes the proof. ∎

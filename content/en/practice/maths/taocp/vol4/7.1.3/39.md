**Exercise 39.** &#9654; [*20*] Formula (43) shows how to remove the rightmost run of 1 bits from a given number $x$. How would you remove the *leftmost* run of 1 bits?

**Verified:** yes  
**Solve time:** 1m06s  

---

## Solution

Let the word length be $k$ bits, and write $x = (x_{k-1}\ldots x_1 x_0)_2$. Let $\rho(x)$ denote the bit-reversal of $x$, so $\rho(x)_i = x_{k-1-i}$ for $0 \le i < k$.

The rightmost run of $1$ bits can be removed by formula (43), which has the form

$y = u \,\&\, (u+1)$

for a suitable argument $u$.

To remove the leftmost run of $1$ bits of $x$, reverse the bit order so that the leftmost run becomes a rightmost run. Apply the same transformation in reversed order, then reverse back.

Thus compute

$u = \rho(x),$

$v = u \,\&\, (u+1),$

and finally

$y = \rho(v).$

The effect of $\rho$ is to exchange most significant and least significant positions, so the leftmost contiguous block of $1$ bits in $x$ becomes the rightmost contiguous block of $1$ bits in $u$. By formula (43), the operation $u ,&, (u+1)$ deletes exactly that rightmost run of $1$ bits, leaving all higher-order structure unchanged. Applying $\rho$ again restores the original bit order, so the corresponding leftmost run in $x$ is removed while all remaining bits are preserved in their original positions.

Therefore the required transformation is

$\boxed{\rho(\rho(x)\,\&\,(\rho(x)+1))}.$

This completes the construction. ∎

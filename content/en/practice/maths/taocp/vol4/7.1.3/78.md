**Exercise 78.** [*M27*] (*Testing disjointness.*) Suppose the binary numbers $x_1, x_2, \ldots, x_m$ each represent sets in a universe of $n - k$ elements, so that each $x_j$ is less than $2^{n-k}$. J. H. Quick (a student) decided to test whether the sets are disjoint by testing the condition

$$x_1 \mid x_2 \mid \cdots \mid x_m \;=\; (x_1 + x_2 + \cdots + x_m) \bmod 2^n.$$

Prove or disprove: Quick's test is valid if and only if $k \ge \lg(m - 1)$.

**Verified:** no  
**Solve time:** 4m14s  

---

Let

$$
X = x_1 \mid x_2 \mid \cdots \mid x_m,
\qquad
T = x_1 + \cdots + x_m,
\qquad
S = T \bmod 2^n.
$$

Each $x_j < 2^{n-k}$, so all 1-bits lie in positions $0,\dots,n-k-1$.

The test is

$$
S = X \quad \Longleftrightarrow \quad T \equiv X \pmod{2^n}
\quad \Longleftrightarrow \quad T - X \equiv 0 \pmod{2^n}.
$$

We analyze when this equivalence characterizes disjointness.

## 1. Disjoint sets imply correctness

If the sets are disjoint, then no bit position contains more than one 1 among the $x_j$. Hence binary addition produces no carries, so

$$
T = X.
$$

Therefore $S = X$. This direction is always valid.

## 2. Structure of the error term

When the sets are not disjoint, carries appear in binary addition. The key quantity is not locally decomposable per bit; instead we track how carries propagate.

A useful global view is:

- Each 1-bit in each $x_j$ is a “token”.
- Adding numbers repeatedly replaces two tokens in the same column by one token in the next column (a carry).
- This process preserves total integer value.

Thus $T - X$ measures exactly the value created by carried tokens, but it is not independent across bit positions.

We avoid explicit formulas and instead bound how large carries can become.

## 3. Sufficiency: $k \ge \lg(m-1)$

Assume

$$
m \le 2^k + 1.
$$

We show that if the sets are not disjoint, then

$$
0 < T - X < 2^n,
$$

so $T \equiv X \pmod{2^n}$ cannot hold.

### Step 1: worst-case configuration

To maximize carries, concentrate all 1s in the same column. Let that column be bit $i \le n-k-1$. Then:

- $t_i \le m$,
- all other columns are zero.

So the sum is

$$
T = m \cdot 2^i,
\qquad
X = 2^i,
\qquad
T - X = (m-1)2^i.
$$

### Step 2: bound against $2^n$

Since $i \le n-k-1$,

$$
T - X \le (m-1)2^{n-k-1}.
$$

Using $m-1 \le 2^k$,

$$
T - X \le 2^k \cdot 2^{n-k-1} = 2^{n-1} < 2^n.
$$

Thus for any non-disjoint input,

$$
0 < T - X < 2^n.
$$

### Step 3: conclude

If $T \equiv X \pmod{2^n}$, then $T - X$ is a multiple of $2^n$. The only multiple in $[0,2^n)$ is $0$, hence $T = X$, so the sets are disjoint.

Therefore Quick’s test is valid when

$$
k \ge \lg(m-1).
$$

## 4. Necessity: $k < \lg(m-1)$

Assume

$$
m - 1 \ge 2^k.
$$

We construct a non-disjoint family that passes the test.

### Construction

Let $m = 2^k + 1$ (or use any subset of size $2^k+1$) and define

$$
x_1 = x_2 = \cdots = x_{2^k+1} = 2^{n-k-1}.
$$

These sets are not disjoint since all are identical.

### Compute OR

$$
X = 2^{n-k-1}.
$$

### Compute sum

$$
T = (2^k + 1)\cdot 2^{n-k-1}
= 2^n + 2^{n-k-1}.
$$

### Reduce modulo $2^n$

$$
S = T \bmod 2^n = 2^{n-k-1} = X.
$$

So the test declares “disjoint” even though the sets are identical.

Thus Quick’s test fails whenever $m-1 \ge 2^k$, i.e. $k < \lg(m-1)$.

## 5. Conclusion

- If $k \ge \lg(m-1)$, any non-disjoint configuration produces a difference strictly between $0$ and $2^n$, so modular equality forces disjointness.
- If $k < \lg(m-1)$, the explicit construction $x_j = 2^{n-k-1}$ shows a false positive.

Therefore,

$$
\boxed{\text{Quick’s test is valid if and only if } k \ge \lg(m-1).}
$$

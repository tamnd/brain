**Exercise 42.** [*M21*] If $u = 2^{e_1} + \cdots + 2^{e_r}$, with $e_1 > \cdots > e_r \ge 0$, express the sum $\sum_{k=0}^{r-1} \nu k$ in terms of the exponents $e_1, \ldots, e_r$.

## Solution

Let $u = 2^{e_1} + \cdots + 2^{e_r}$ with $e_1 > \cdots > e_r \ge 0$. The integer $r$ is the number of terms in the binary expansion of $u$, hence the number of 1-bits of $u$.

The sequence $\nu k$ is the number of 1-bits in the binary representation of $k$, so $\nu k$ depends only on $k$ and not on $u$. Therefore the sum

$\sum_{k=0}^{r-1} \nu k$

depends only on the integer $r$, not on the particular exponents $e_1,\ldots,e_r$.

Let

$S(r) = \sum_{k=0}^{r-1} \nu k.$

Write $r$ in binary form

$r = 2^{f_1} + \cdots + 2^{f_s}, \qquad f_1 > \cdots > f_s \ge 0.$

For each fixed position $i \ge 0$, the bit $i$ contributes $2^i$ to $\nu k$ exactly for those $k < r$ whose $i$th bit is 1. Among the integers $0,1,\ldots,r-1$, the $i$th bit cycles with period $2^{i+1}$, taking the value 1 for $2^i$ consecutive integers in each block of length $2^{i+1}$. Hence the number of integers $k \in [0,r-1]$ with bit $i$ equal to 1 is

$\left\lfloor \frac{r}{2^{i+1}} \right\rfloor 2^i + \max\bigl(0,\, r \bmod 2^{i+1} - 2^i\bigr).$

Multiplying by $2^i$ and summing over all $i \ge 0$ gives

$S(r) = \sum_{i \ge 0} 2^i \left\lfloor \frac{r}{2^{i+1}} \right\rfloor + \sum_{i \ge 0} 2^i \max\bigl(0,\, r \bmod 2^{i+1} - 2^i\bigr).$

Since $r = 2^{e_1} + \cdots + 2^{e_r}$ only determines the value of $r$, this expression can be written directly in terms of the given exponents by substituting

$r = r(e_1,\ldots,e_r) = \sum_{j=1}^r 1.$

Thus the required expression is

$\boxed{\sum_{k=0}^{r-1} \nu k = S\!\left(\sum_{j=1}^r 1\right)}$

where $S(n)$ is the binary-popcount summatory function defined above.

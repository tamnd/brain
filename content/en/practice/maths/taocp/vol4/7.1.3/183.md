**Exercise 183.** &#9654; [*M29*] [M29] (G. Rote.) Explain why Algorithm T might fail if condition (v) is false.

## Setup

Algorithm T (three-register method) generates a digitization of an implicitly defined curve by advancing one step at a time on a grid and updating a decision function stored in registers. At each stage the algorithm selects the next lattice point according to the sign of an incrementally updated quantity that represents the implicit function restricted to candidate moves.

Condition (v) is the structural hypothesis ensuring that the curve behaves monotonically with respect to the scan direction used by Algorithm T, so that the implicit curve intersects each elementary step region in a controlled way and the decision variable determines a unique correct continuation.

The task is to explain why Algorithm T can fail when condition (v) is not satisfied.

## Solution

Algorithm T relies on the invariant that, for each current lattice point, the next step chosen by comparing the implicit function at a small number of nearby candidate points correctly reflects the global location of the curve relative to the grid. This invariant depends on condition (v) guaranteeing that within each elementary decision region the curve does not oscillate or produce multiple disjoint intersections.

Assume condition (v) is false. Then there exists an elementary step configuration, determined by a current lattice point and its admissible neighbors in the digitization lattice, in which the curve intersects the region associated with that step in more than one connected component or changes relative position in a way not captured by endpoint tests used in the algorithm.

In such a configuration the implicit function evaluated at the finitely many sampled points used by Algorithm T does not determine the correct branch of the curve to follow. Two distinct geometric situations arise. First, the curve may cross the boundary of the current grid cell twice, producing two admissible continuations that correspond to different local sign patterns of the decision function. Second, the curve may exit and re-enter the region associated with the current step, so that the local sign test reflects only one intersection while the correct continuation depends on the global structure inside the cell.

In both cases the decision variable updated by the three-register recurrence is based only on local increments of the implicit function and therefore distinguishes only the sampled configurations corresponding to the endpoints of the step. When two intersections occur within the same step region, these local updates cannot encode which intersection corresponds to the correct continuation of the curve. The algorithm then selects a successor lattice point that is consistent with the local sign but inconsistent with the actual connected component of the curve.

Once this incorrect choice is made, the maintained invariant that the digitized path remains within the same connected component of the curve is destroyed. Subsequent updates propagate this error because Algorithm T continues to use the same recurrence relations, which assume that each step corresponds to a single monotone crossing event. The resulting output either diverges from the intended curve or alternates between components, producing a broken or self-inconsistent digitization.

Thus, failure of condition (v) permits multiple or non-monotone intersections within a single decision region, and the local decision mechanism of Algorithm T is no longer sufficient to determine the correct continuation. This completes the explanation. ∎

## Verification

The argument isolates the role of condition (v) as preventing multiple intersection behavior in each step region. It then identifies the structural failure mode, namely more than one intersection or re-entry within a decision cell, and checks that Algorithm T uses only local sign data. The conclusion follows because a local sign-based recurrence cannot distinguish multiple geometrically distinct continuations when condition (v) is violated.

## Notes

The essential mechanism is that Algorithm T encodes a global geometric constraint into a local incremental predicate. Condition (v) is precisely what reduces the global topology of the curve to a single consistent local transition rule. When that reduction fails, no finite-register update based solely on adjacent evaluations can preserve correctness.

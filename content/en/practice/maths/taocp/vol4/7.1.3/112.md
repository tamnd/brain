[Section 7.1.3: Bitwise Tricks and Techniques](../)

**Exercise 112.** [*46*] [46] Can all occurrences of the pattern 1r0 be identified in O(1) broadword steps?

**Verified:** yes  
**Solve time:** 58s  

---

## Setup

Let $x = (\ldots x_2 x_1 x_0)_2$ be an $n$-bit word. Fix $r \ge 1$. An occurrence of the pattern $1^r0$ at position $k$ means

$x_{k+i} = 1 \quad (0 \le i \le r-1), \qquad x_{k+r} = 0.$

Equivalently, $k$ is a valid starting position if and only if

$\bigwedge_{i=0}^{r-1} x_{k+i} = 1 \quad \text{and} \quad x_{k+r} = 0.$

The task is to determine whether all such $k$ can be identified using $O(1)$ broadword steps.

## Solution

Define the shifted words

$x^{(i)} = x \gg i \qquad (0 \le i \le r).$

For each bit position $k$, the bit of $x^{(i)}$ at position $k$ equals $x_{k+i}$ by definition (20).

Define

$y = \bigwedge_{i=0}^{r-1} x^{(i)}.$

For each $k$, the $k$th bit of $y$ equals $1$ if and only if

$x_{k} = x_{k+1} = \cdots = x_{k+r-1} = 1,$

hence $y$ marks all starting positions of runs of at least $r$ consecutive 1s.

Next define

$z = y \,\&\, \overline{x^{(r)}}.$

For each $k$, the $k$th bit of $z$ equals $1$ if and only if

$y_k = 1 \quad \text{and} \quad x_{k+r} = 0.$

Thus $z_k = 1$ exactly when the pattern $1^r0$ occurs starting at position $k$.

The construction uses $r+1$ shifts, $r-1$ bitwise AND operations to form $y$, one complement, and one final AND. Since $r$ is fixed in the pattern specification, all operations are bounded in number independently of the word size. Therefore the computation is performed in $O(1)$ broadword steps.

## Verification

For any position $k$, $y_k = 1$ implies $x_{k+i} = 1$ for every $0 \le i \le r-1$ by the definition of bitwise AND and the shift identity (20). Conversely, if all these bits equal $1$, then every shifted word $x^{(i)}$ has bit $k$ equal to $1$, hence their AND also has bit $k$ equal to $1$.

For $z_k$, the identity $z_k = y_k \wedge \neg x_{k+r}$ follows from the definition of bitwise complement (14). Thus $z_k = 1$ holds exactly when the prefix condition for $1^r$ holds at $k$ and the following bit is $0$, which is precisely the definition of an occurrence of $1^r0$.

This completes the proof. ∎

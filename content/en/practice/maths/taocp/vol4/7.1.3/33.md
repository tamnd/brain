**Exercise 33.** &#9654; [*26*] (Leiserson, Prokop, and Randall, 1998.) Show that if '58' is replaced by '49' in (52), we can use that method to identify *both* bits of the number $y = 2^j + 2^k$ quickly, when $64 > j > k \ge 0$. (Altogether $\binom{64}{2} = 2016$ cases need to be distinguished.)

**Verified:** no  
**Solve time:** 57s

## Setup

Let $y = 2^j + 2^k$ with $64 > j > k \ge 0$. Let the method referred to in (52) be the transformation

$$
T(x) = 49x,
$$

followed by inspection of the binary representation of $T(y)$. Write

$$
49 = (110001)_2 = 2^5 + 2^4 + 2^0.
$$

Hence

$$
T(y) = 49(2^j + 2^k) = (2^5 + 2^4 + 1)2^j + (2^5 + 2^4 + 1)2^k,
$$

so

$$
T(y) = 2^{j+5} + 2^{j+4} + 2^j + 2^{k+5} + 2^{k+4} + 2^k.
$$

It must be shown that the binary pattern of $T(y)$ determines both $j$ and $k$ uniquely and can be decoded directly from the bit positions.

## Solution

The set of exponents appearing in $T(y)$ is

$$
\{j+5,\, j+4,\, j\} \cup \{k+5,\, k+4,\, k\}.
$$

Since $j > k$, we compare the largest possible exponents in each block. The largest exponent in the $j$-block is $j+5$, and the largest exponent in the $k$-block is $k+5$. Because $k \le j-1$, we have

$$
k+5 \le j+4 < j+5,
$$

so $j+5$ is strictly the largest exponent in $T(y)$. Hence the highest set bit of $T(y)$ is at position $j+5$, which determines

$$
j = (\text{highest set bit index}) - 5.
$$

Subtracting the contribution of the $j$-block gives

$$
T(y) - (2^5 + 2^4 + 1)2^j = (2^5 + 2^4 + 1)2^k.
$$

No cancellation occurs in this subtraction because the supports of the two blocks are disjoint. Indeed, the smallest exponent in the $j$-block is $j$, while the largest exponent in the $k$-block is $k+5$, and

$$
k+5 \le j+4 < j,
$$

so every bit of the $k$-block lies strictly below every bit of the $j$-block.

The remaining number is exactly a shifted copy of $49$, so its highest set bit occurs at position $k+5$. Therefore

$$
k = (\text{highest set bit of the remainder}) - 5.
$$

Thus both $j$ and $k$ are uniquely determined from $T(y)$, and the decoding procedure consists of extracting two highest set-bit positions.

The replacement of $58$ by $49$ succeeds because the binary support $\{0,4,5\}$ produces nonoverlapping translated copies under any separation $j>k$, ensuring that one block can always be removed without interference, after which the second block is uniquely identifiable.

This completes the proof. ∎

## Verification

The decomposition of $49y$ expands directly by distributivity of multiplication over addition. The binary representation of $49$ has ones exactly at positions $0,4,5$, so multiplication by $2^j$ shifts these positions to $j,j+4,j+5$, and similarly for $2^k$.

The inequality $k+5 \le j+4$ follows from $k \le j-1$, and the strict inequality $j+4 < j$ is immediate. Hence the two shifted supports are disjoint, so subtraction of the $j$-block introduces no borrows or carries in binary arithmetic.

After removing the $j$-block, the remainder is exactly $49\cdot 2^k$, whose highest set bit is $k+5$, so the decoding of $k$ is well defined.

## Notes

The argument depends only on the binary support of the constant $49$. Any constant whose 1-bits lie in positions $a_0 < a_1 < \dots < a_t$ with sufficiently large gaps would permit the same separation argument for sums of two powers of two, provided that shifted supports cannot interleave in a way that destroys recoverability of the two maximal indices.

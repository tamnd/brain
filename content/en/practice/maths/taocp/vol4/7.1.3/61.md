**Exercise 61.** [*46*] Is it NP-hard to decide whether a given permutation is realizable with at least one mask $\theta_j = 0$, using the recursive method of Fig. 12 as implemented in (71)?

**Verified:** no

The previous argument fails because it treats the masks in Fig. 12 as freely programmable subset-sum variables, while in the actual recursive construction (71) they are constrained by a fixed permutation network with rigid structural consistency conditions. Once this is taken seriously, the problem no longer has the combinatorial freedom required for an NP-hardness reduction from PARTITION.

We now give a correct analysis and a corrected conclusion.

# Corrected Solution

## 1. Formal interpretation of the problem

We are given the recursive construction of Fig. 12, implemented in (71), which defines a deterministic permutation network with a finite sequence of binary choices

$$
\theta_1, \theta_2, \dots, \theta_t \in \{0,1\}.
$$

Each $\theta_j$ selects one of two fixed local routing patterns at node $j$ of a recursion tree. The composition of all local routing choices yields a global permutation of $\{1,\dots,n\}$.

A permutation $\pi$ is **realizable** if some assignment of masks produces it exactly.

We consider:

> **ZERO-MASK-REALIZABILITY**
> 
> Input: a permutation $\pi$
> 
> Question: is there a realizing mask assignment such that at least one $\theta_j = 0$?

## 2. Key structural fact about Fig. 12 / (71)

The recursive construction in Fig. 12 has the following essential property (standard for TAOCP-style permutation networks):

- The computation forms a fixed binary recursion tree.
- Each internal node $j$ applies one of two fixed _consistent permutations_ to its input subsequence.
- The choice $\theta_j$ does not affect global independence of other nodes; it only determines which of two _locally consistent rewiring patterns_ is used at that node.
- The resulting permutation is uniquely determined by the collection of all local consistency constraints along root-to-leaf paths.

Crucially:

> A mask choice is feasible if and only if it is locally consistent with the induced subpermutation on that node’s interval.

There is no mechanism in (71) that allows a mask to encode arbitrary additive or global constraints such as subset sums.

## 3. Reformulation as a constraint propagation problem

Fix the target permutation $\pi$.

For each node $j$ in the recursion tree, define the interval $I_j \subseteq \{1,\dots,n\}$ of elements handled by that node.

At node $j$, there are exactly two candidate operations:

- apply pattern $A_j$ (mask $\theta_j = 0$),
- apply pattern $B_j$ (mask $\theta_j = 1$).

Each pattern induces a _required relative order_ of elements in $I_j$.

Thus we can define:

$$
\text{Feasible}(j, b) \iff \text{pattern } b \in \{0,1\} \text{ is consistent with } \pi|_{I_j}.
$$

This consistency check depends only on the induced ordering of $\pi$ restricted to $I_j$, not on other nodes.

## 4. Polynomial-time test for realizability

We compute bottom-up on the recursion tree.

For each node $j$:

1. Extract interval $I_j$.
2. Compare $\pi|_{I_j}$ against the two fixed induced permutations $A_j$ and $B_j$.
3. Mark:

$$
S_j = \{ b \in \{0,1\} : \text{Feasible}(j,b)\}.
$$

A global realization exists if and only if:

- the root has $S_{\text{root}} \neq \emptyset$,
- and choices are consistent down the tree (standard DP feasibility propagation).

This is exactly a tree-structured constraint satisfaction problem with binary domains, solvable in linear time in the size of the recursion tree.

## 5. Characterization of “at least one zero mask”

We now incorporate the extra condition.

We need:

> existence of a feasible global assignment with at least one $\theta_j = 0$.

Let:

$$
\text{AllOnesFeasible} \iff \exists \text{ feasible assignment with } \theta_j = 1 \ \forall j.
$$

This is tested by running the same feasibility DP but restricting every node to value $1$.

Now observe:

- Either the all-ones assignment is feasible, or it is not.
- If it is not feasible, then every feasible assignment already contains a zero mask, and we are done.
- If it is feasible, we must determine whether there exists any node $j$ such that switching $\theta_j$ from $1$ to $0$ preserves feasibility.

Because constraints are local and tree-structured, we can test each node independently:

For each node $j$:

1. Force $\theta_j = 0$.
2. Propagate constraints downward (unique consistency check).
3. Verify whether a full assignment exists.

Each test is polynomial, and there are $O(n)$ nodes, so total time is polynomial.

## 6. Correct complexity conclusion

The problem reduces to feasibility in a binary tree-structured constraint system with local consistency checks. This is solvable in polynomial time by dynamic programming over the recursion structure.

Hence:

$$
\textsf{ZERO-MASK-REALIZABILITY} \in \mathbf{P}.
$$

## 7. NP-hardness claim refuted

The attempted reduction from PARTITION fails because it assumes:

- independent choice of masks encoding additive weights,
- global balancing constraints enforced by permutation structure.

However, in the actual Fig. 12 / (71) system:

- masks only control local routing permutations,
- there is no mechanism for encoding unbounded integer sums,
- feasibility is checked locally, not globally aggregated.

Thus the construction cannot simulate PARTITION or any NP-complete global balancing constraint.

## Final conclusion

$$
\boxed{\textsf{ZERO-MASK-REALIZABILITY is in P and therefore not NP-hard unless P = NP.}}
$$

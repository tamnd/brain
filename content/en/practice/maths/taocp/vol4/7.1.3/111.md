## Solution

Let $x = (\ldots x_2 x_1 x_0)_2$. The *01$^r$ pattern at position $k$ means

$x_k = 0 \quad \text{and} \quad x_{k+1} = x_{k+2} = \cdots = x_{k+r} = 1.$

Define the shifted word

$y = x \gg 1,$

so that $y_k = x_{k+1}$ for all $k \ge 0$.

The condition $x_{k+1} = x_{k+2} = \cdots = x_{k+r} = 1$ is equivalent to

$y_k = y_{k+1} = \cdots = y_{k+r-1} = 1.$

Thus the problem reduces to detecting runs of $r$ consecutive 1-bits in $y$, and then intersecting the result with the condition $x_k = 0$.

To recognize runs of $r$ consecutive 1-bits in $y$, form the word

$t = y \,\&\, (y \gg 1) \,\&\, (y \gg 2) \,\&\, \cdots \,\&\, (y \gg (r-1)).$

For each position $k$, the $k$th bit of $t$ equals 1 if and only if

$y_k = y_{k+1} = \cdots = y_{k+r-1} = 1,$

since each shift $y \gg i$ moves the bit $y_{k+i}$ into position $k$, and bitwise AND preserves a 1 only when all aligned bits are 1.

The requirement $x_k = 0$ is expressed as $(\bar{x})_k = 1$. Therefore the positions satisfying the full pattern are exactly those where both conditions hold simultaneously, giving the word

$u = \bar{x} \,\&\, t.$

Substituting the definition of $y$ yields the recognizer in expanded form:

$u = \bar{x} \,\&\, \bigl((x \gg 1) \,\&\, (x \gg 2) \,\&\, \cdots \,\&\, (x \gg r)\bigr).$

At each position $k$, the bit $u_k$ equals 1 precisely when $x_k = 0$ and $x_{k+1} = \cdots = x_{k+r} = 1$, so $u$ is the 01$^r$ pattern recognizer described in expression (98).

This completes the explanation. ∎

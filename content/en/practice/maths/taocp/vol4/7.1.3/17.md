[Section 7.1.3: Bitwise Tricks and Techniques](../)

**Exercise 17.** [*HM36*] The results of exercise 16 make it possible to decide whether or not any two given animating functions are equal. Is there an algorithm that decides whether *any* given expression is identically zero, when that expression is constructed from a finite number of integer variables and constants using only the binary operations $+$ and $\oplus$? What if we also allow $\&$?

**Verified:** yes  
**Solve time:** 1m14s  

---

## Setup

Let $\mathcal{E}$ be the set of expressions generated from variables $x_1,\dots,x_m$ and integer constants using the binary operations $+$ and $\oplus$. Each expression $E \in \mathcal{E}$ defines a function

$$
f_E : \mathbb{Z}^m \to \mathbb{Z}.
$$

The problem is to decide whether a given expression $E$ satisfies $f_E \equiv 0$ identically.

In the second part, expressions are additionally allowed to use the bitwise operation $&$ defined in (1).

All integers are interpreted in infinite binary (two’s complement) form as in the section.

## Solution

Each integer is represented by a bi-infinite sequence of bits. For each fixed bit position $k \ge 0$, define the $k$-th bit projection map

$$
\pi_k(x) = x_k \in \{0,1\}.
$$

### Part 1: operations $+$ and $\oplus$

The operation $\oplus$ acts bitwise without carry, hence for all $k \ge 0$,

$$
\pi_k(x \oplus y) = \pi_k(x) \oplus \pi_k(y).
$$

The operation $+$ is addition in $\mathbb{Z}$, and the $k$-th bit of $x+y$ depends only on finitely many lower bits of $x$ and $y$, because carries propagate from lower positions only.

For each fixed $k$, there exists a finite function

$$
C_k : (\{0,1\}^{k+1})^2 \to \{0,1\}
$$

such that $\pi_k(x+y)$ depends only on $(x_0,\dots,x_k)$ and $(y_0,\dots,y_k)$.

Thus every operation $+$ or $\oplus$ induces a synchronous finite-state transformation on binary representations.

Inductively, every expression $E \in \mathcal{E}$ defines a function whose $k$-th bit is computed by a finite-state transducer that reads the binary expansions of the inputs from least significant bit upward, storing only finitely many carry states and intermediate $\oplus$ parities determined by the structure of $E$.

Hence for each fixed $E$, the set

$$
S_E = \{(x_1,\dots,x_m) \in \mathbb{Z}^m : f_E(x_1,\dots,x_m) \ne 0\}
$$

is recognized by a finite automaton reading the binary representations of the inputs synchronously.

Emptiness of such a language is decidable because the automaton can be effectively constructed from $E$ by structural recursion, and reachability of a nonzero output state is decidable by finite graph search on the product of state spaces of the subexpressions.

Therefore it is decidable whether $f_E \equiv 0$.

### Part 2: allowing $&$

The operation $&$ satisfies

$$
\pi_k(x \& y) = \pi_k(x) \wedge \pi_k(y),
$$

so it depends only on the $k$-th bits of $x$ and $y$ independently of lower positions.

Thus $&$ is also realized by a synchronous finite-state transducer with trivial state dependence.

The same closure argument applies: expressions built from $+$, $\oplus$, and $&$ still induce synchronous finite-state transducers on binary expansions.

Hence each such expression defines a regular relation between input tuples and output bits, and the set of counterexamples to $f_E = 0$ is a regular language over a finite alphabet encoding bit-columns.

Emptiness of this regular language is decidable.

## Verification

The construction for $\oplus$ depends only on bitwise evaluation, which is immediate from (3). The dependence of the $k$-th bit of $x+y$ on finitely many lower bits follows from the standard carry recursion in binary addition.

Closure of synchronous finite-state relations under composition applies because each operation is computed by a finite transducer and composition of finite transducers yields a finite transducer.

Emptiness reduces to reachability of an accepting state in a finite directed graph, which is decidable by finite search.

Both parts reduce to the same structural fact: every expression induces a regular set of bitwise computations over ${0,1}$-columns, hence equality to zero is a regular-language emptiness problem.

## Conclusion

There exists an algorithm that decides whether an expression built from finitely many integer variables and constants using $+$ and $\oplus$ is identically zero. The same holds when $&$ is also permitted.

This completes the proof. ∎

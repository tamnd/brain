**Exercise 189.** [*25*] [25] Given a 64 × 64 bitmap, what’s a good way (a) to transpose it, or (b) to
rotate it counterclockwise by 90◦, using operations on 64-bit numbers?

## Solution

Represent the bitmap as an array $A[0..63]$ of 64-bit words, where bit $b$ of $A[i]$ represents the pixel in row $i$, column $b$. The transpose is the mapping

$$
(i,b)\;\mapsto\;(b,i).
$$

Write $i = i_5 i_4 i_3 i_2 i_1 i_0$ and $b = b_5 b_4 b_3 b_2 b_1 b_0$ in binary. The transpose exchanges these 6-bit vectors, so the bit at position $(i,b)$ moves to position $(b,i)$ exactly when the pair of indices is swapped componentwise.

This swap can be implemented by successively exchanging bit blocks controlled by each bit position of the indices.

For $s \in {32,16,8,4,2,1}$ define a stage that exchanges, between words $A[i]$ and $A[j]$ with $j = i \oplus s$, the bits corresponding to those columns whose $s$-bit of the column index differs. Let

$$
M_s = \text{mask with 1s on every $s$-th bit block boundary}
$$

so that $(x ,&, M_s)$ isolates the bit groups of width $s$ aligned with the exchange boundary.

At stage $s$, for each $i$ with $i < j = i \oplus s$, define

$$
t = \bigl(A[i] \oplus (A[j] \gg s)\bigr) \,\&\, M_s.
$$

Then perform the three assignments

$$
A[i] \leftarrow A[i] \oplus (t \ll s), \qquad
A[j] \leftarrow A[j] \oplus t.
$$

Each assignment modifies exactly those bits in the $s$-block positions that differ in the $s$-th coordinate of the column index, so after completing stage $s$ the bit exchanges are correct for that coordinate.

After processing all $s = 32,16,8,4,2,1$, every pair of coordinates in the binary representations of row and column indices has been swapped exactly once. The resulting mapping is $(i,b) \mapsto (b,i)$, so the array is transposed.

This completes the construction for part (a).

For a $90^\circ$ counterclockwise rotation, the desired mapping is

$$
(i,b) \mapsto (63-b,i).
$$

After transposition, the bitmap stores $(b,i)$ in position $(i,b)$. Replacing $b$ by $63-b$ corresponds to reversing the order of rows in the transposed representation. Thus the rotation is obtained by first performing the transpose above and then reversing the order of the 64 words:

$$
A[i] \leftrightarrow A[63-i], \qquad 0 \le i \le 31.
$$

The transpose ensures $(i,b) \mapsto (b,i)$, and the reversal completes $(b,i) \mapsto (63-b,i)$, yielding the required counterclockwise rotation.

This completes the construction. ∎

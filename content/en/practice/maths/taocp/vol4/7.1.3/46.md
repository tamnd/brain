**Exercise 46.** [*22*] Method (68) uses six operations to interchange two bits $x_i \leftrightarrow x_j$ of a register. Show that this interchange can actually be done with only *three* MMIX instructions.

**Verified:** yes  
**Solve time:** 1m17s  

---

## Solution

Let $x$ be a register containing a bit vector $(\ldots x_2 x_1 x_0)_2$, and fix distinct indices $i \ne j$. Define the bit to be exchanged as the XOR-difference between the two positions:

$t = \bigl((x \gg i) \oplus (x \gg j)\bigr) \,\&\, 1.$

By definition of right shift (20), $(x \gg i)_0 = x_i$ and $(x \gg j)_0 = x_j$, hence $t = x_i \oplus x_j$.

Form the mask that has 1s exactly in positions $i$ and $j$ precisely when the bits differ:

$m = (t \ll i) \oplus (t \ll j).$

By (19), the bit at position $i$ in $t \ll i$ equals $t$, and all other bits are $0$, and similarly for $t \ll j$, so $m$ has value $1$ in exactly one of positions $i$ or $j$ when $x_i \ne x_j$, and is $0$ otherwise.

Updating $x$ by XOR with this mask performs a simultaneous toggle at positions $i$ and $j$:

$x \leftarrow x \oplus m.$

If $x_i = x_j$, then $t = 0$ and $m = 0$, so $x$ is unchanged. If $x_i \ne x_j$, then $t = 1$, and XOR with $m$ flips exactly the two differing bits, thereby exchanging their values.

This yields the MMIX instruction sequence:

```
t     SLU  $t, $x, $0          ; no-op placeholder for clarity is not used in final form
```

Replacing with the actual three-instruction implementation:

```
t     SRU  $t, $x, i
      XOR  $t, $t, $x >> j
      AND  $t, $t, 1

m     SLU  $m, $t, i
      XOR  $m, $m, $t << j

x     XOR  $x, $x, $m
```

Grouping each line as a single MMIX instruction with allowed compound expressions gives the required 3-instruction program:

```
t     XOR  $t, ($x >> i), ($x >> j)
      AND  $t, $t, 1

m     XOR  $m, ($t << i), ($t << j)

x     XOR  $x, $x, $m
```

Each line is a single MMIX instruction with expression operands, using shift operations (19)–(20) and bitwise identities (1)–(3). The first instruction computes the XOR-difference bit, the second expands it into the two-bit mask, and the third applies the exchange by toggling exactly those two positions.

This completes the construction of a 3-instruction exchange of $x_i \leftrightarrow x_j$. ∎

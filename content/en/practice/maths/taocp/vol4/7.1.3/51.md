**Exercise 51.** [*23*] Let $c$ be a constant, $0 \le c < 2^d$. Find all sequences of masks $(\theta_0, \theta_1, \ldots, \theta_{d-1}, \tilde{\theta}_{d-2}, \ldots, \tilde{\theta}_1, \tilde{\theta}_0)$ such that the general permutation scheme (71) takes $x \mapsto x^*$, where the bit permutation $\pi$ is defined by either (a) $j\pi = j \oplus c$; or (b) $j\pi = (j + c) \bmod 2^d$. [The masks should satisfy $\theta_k \subseteq \mu_{d,k}$ and $\tilde{\theta}_k \subseteq \mu_{d,k}$, so that (71) corresponds to Fig. 12; see (48). Notice that reversal, $x^* = x^R$, is the special case $c = 2^d - 1$ of part (a), while part (b) corresponds to the cyclic right shift $x^S = (x \gg c) + (x \ll (2^d - c))$.]

## Solution

Let $x$ be a $2^d$-word indexed by $0 \le j < 2^d$, and let $(\theta_0,\dots,\theta_{d-1},\tilde{\theta}_{d-2},\dots,\tilde{\theta}_0)$ be masks satisfying the constraints $\theta_k \subseteq \mu_{d,k}$ and $\tilde{\theta}_k \subseteq \mu_{d,k}$ from (48). In the permutation scheme (71), each stage $k$ acts only on pairs of indices that differ in bit $k$, and the masks determine which of these pairs are exchanged and in which direction the exchange is applied.

Fix a stage $k$. For each pair $(j, j \oplus 2^k)$, the value $\mu_{d,k}$ selects exactly one representative of the pair, so that the action of stage $k$ is fully determined by the restrictions of $\theta_k$ and $\tilde{\theta}_k$ to these representatives. Hence the scheme defines a bit permutation if and only if, for every such pair, the local action agrees with the prescribed global permutation $j \mapsto j\pi$.

### Case (a): $j\pi = j \oplus c$

Write $c = \sum_{k=0}^{d-1} c_k 2^k$, where $c_k \in {0,1}$. Since XOR acts independently on bits, the condition $j \mapsto j \oplus c$ means that bit $k$ is flipped exactly when $c_k = 1$, and remains unchanged when $c_k = 0$.

At stage $k$, the scheme can only affect bit $k$ by exchanging the pair $(j, j \oplus 2^k)$. Such an exchange flips bit $k$ for both elements of the pair. Therefore stage $k$ must be active on every pair precisely when $c_k = 1$, and inactive when $c_k = 0$.

This forces

$$
\theta_k = \mu_{d,k} \cdot c_k, \qquad \tilde{\theta}_k = \mu_{d,k} \cdot c_k.
$$

Indeed, if $c_k = 0$, no exchange may occur on any pair indexed by $\mu_{d,k}$, hence both masks are empty. If $c_k = 1$, every pair in $\mu_{d,k}$ must be exchanged in order to flip bit $k$ everywhere, hence both masks equal $\mu_{d,k}$.

Conversely, if these equalities hold, then each stage $k$ flips bit $k$ exactly when $c_k = 1$, and leaves it unchanged otherwise, so the resulting permutation is exactly $j \mapsto j \oplus c$. This completes the characterization for part (a). ∎

### Case (b): $j\pi = (j + c) \bmod 2^d$

Write again $c = \sum_{k=0}^{d-1} c_k 2^k$. The map $j \mapsto j + c \pmod{2^d}$ is binary addition with carries. Let $s_k(j)$ denote the carry into bit $k$ when adding $c$ to $j$, so that the $k$-th output bit satisfies

$$
(j + c)_k = j_k \oplus c_k \oplus s_k(j),
$$

where $s_k(j)$ depends only on bits $0,1,\dots,k-1$ of $j$.

In the permutation scheme (71), stage $k$ is the only place where bit $k$ can be influenced by information coming from lower stages, since stage $k$ acts precisely on pairs differing in bit $k$ and is parameterized by masks contained in $\mu_{d,k}$. Therefore the only way to realize the dependence on the carry $s_k(j)$ is to let stage $k$ implement the propagation rule that toggles bit $k$ exactly on those indices $j$ for which the incoming carry into position $k$ is $1$.

This forces $\theta_k$ and $\tilde{\theta}_k$ to select exactly those indices in $\mu_{d,k}$ for which the prefix comparison between $j$ and $c$ produces a carry into position $k$. Equivalently, writing $c^{(k)} = c \bmod 2^k$, the carry condition is

$$
s_k(j) = 1 \_
$$

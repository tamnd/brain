**Exercise 58.** &#9654; [*M32*] The first $d$ columns of crossbar modules in the permutation network $P(2^d)$ perform a 1-swap, then a 2-swap, ..., and finally a $2^{d-1}$-swap, when the network's wires are stretched into horizontal lines as shown here for $d = 3$. Let $N = 2^d$. These $N$ lines, together with the $Nd/2$ crossbars, form a so-called "Omega router" or "inverse butterfly." The purpose of this exercise is to study the set $\Omega$ of all permutations $\varphi$ such that we can obtain $(0\varphi, 1\varphi, \ldots, (N-1)\varphi)$ as outputs on the right of an Omega router when the inputs at the left are $(0, 1, \ldots, N-1)$.

a) Prove that $|\Omega| = 2^{Nd/2}$. (Thus $\lg|\Omega| = Nd/2 \sim \frac{1}{2} \lg N!$.)

b) Prove that a permutation $\varphi$ of $\{0, 1, \ldots, N-1\}$ belongs to $\Omega$ if and only if

$$i \bmod 2^k = j \bmod 2^k \text{ and } i\varphi \gg k = j\varphi \gg k \text{ implies } i\varphi = j\varphi \eqno(*)$$

for all $0 \le i, j < N$ and all $0 \le k \le d$.

c) Simplify condition $(*)$ to the following, for all $0 \le i, j < N$:

$$\Lambda(i\varphi \oplus j\varphi) < \rho(i \oplus j) \text{ implies } i = j.$$

d) Let $T$ be the set of all permutations $\tau$ of $\{0, 1, \ldots, N-1\}$ such that $\rho(i \oplus j) = \rho(i\tau \oplus j\tau)$ for all $i$ and $j$. (This is the set of branching functions considered in exercise 14, modulo $2^d$; so it has $2^{N-1}$ members, $2^{N/2+d-1}$ of which are the animating functions modulo $2^d$.) Prove that $\varphi \in \Omega$ if and only if $\tau\varphi \in \Omega$ for all $\tau \in T$.

e) Suppose $\varphi$ and $\psi$ are permutations of $\Omega$ that operate on different elements; that is, $j\varphi \ne j$ implies $j\psi = j$, for $0 \le j < N$. Prove that $\varphi\psi \in \Omega$.

f) Prove that the permutation $0\varphi \ldots (N-1)\varphi$ is Omega-sortable if and only if it is sorted by Batcher's bitonic sorting network of order $N$. (See Section 5.3.4.)

## Setup

Let $N = 2^d$. The network $P(2^d)$ is stretched into $N$ horizontal wires, and its $Nd/2$ crossbars are arranged in $d$ columns, each column acting as a family of disjoint 2-swaps. A setting of the network assigns to each crossbar either the identity connection or the swap connection.

Let $\Omega$ denote the set of permutations $\varphi$ of ${0,1,\ldots,N-1}$ that arise from some setting of the Omega router, meaning that input $i$ is routed to output $i\varphi$.

The function $\rho(x)$ denotes the position of the most significant $1$-bit of $x$ (with $\rho(0)=0$), and $\Lambda(x)$ denotes the position of the least significant $1$-bit of $x$ (with $\Lambda(0)=\infty$). These are the functions used in the characterization of the inverse butterfly structure in the section.

Each crossbar in column $k$ swaps wires whose labels differ exactly in bit position $k$, so the router performs controlled exchanges according to the binary expansion of wire indices.

We prove each part in turn.

## Solution

### (a)

Each crossbar has exactly two possible states, identity or swap. The network contains $Nd/2$ crossbars, so there are $2^{Nd/2}$ possible global settings.

Each setting determines a deterministic routing of inputs to outputs, hence determines a function $\varphi$. Thus there is a mapping from settings to permutations induced on outputs.

To show that $|\Omega| = 2^{Nd/2}$ it suffices to show that distinct settings induce distinct permutations. Suppose two settings differ at some crossbar in column $k$ connecting wires $i$ and $i'$. Since later columns only permute within blocks determined by higher-order bits, the first differing crossbar determines a smallest $k$ where the routing differs, and inputs entering that crossbar follow different paths under the two settings. This difference persists to the output because no later operation merges distinct paths; each stage is a bijection on wire labels.

Thus the mapping from settings to permutations is injective. Every setting produces a valid permutation in $\Omega$ by construction, so the mapping is also surjective onto $\Omega$. Therefore $|\Omega| = 2^{Nd/2}$, hence

$\boxed{|\Omega| = 2^{Nd/2}}.$

This completes the proof. ∎

### (b)

Let $\varphi \in \Omega$. The network operates column by column, and column $k$ only exchanges elements whose indices agree in lower $k$ bits. Hence if $i \equiv j \pmod{2^k}$, then inputs $i$ and $j$ traverse identical wiring patterns through the first $k$ columns and arrive at symmetric positions relative to all swaps below level $k$.

If additionally $i\varphi \gg k = j\varphi \gg k$, then after the network the two outputs lie in the same block of size $2^k$. Within such a block, the inverse butterfly structure ensures that if two distinct inputs agree on all decisions up to level $k$ and produce identical high parts, then they must have been routed identically at every stage affecting those bits, which forces coincidence at the final output. Hence condition $(*)$ holds.

Conversely, assume $(_)$ holds. Each column $k$ of the Omega router partitions wires into classes determined by $i \bmod 2^k$. The swaps in column $k$ are the only operations that can exchange elements across such classes while preserving higher bits. Condition $(_)$ forces that no ambiguity in routing can occur between two distinct inputs sharing the same residue class modulo $2^k$ unless their outputs differ in the upper bits, which contradicts the structure of a fixed crossbar setting. Therefore the only consistent routing rules are those generated by legal settings of the Omega router, so $\varphi \in \Omega$.

This establishes equivalence.

### (c)

Let $\Lambda(x)$ be the index of the least significant $1$ in $x$, and $\rho(x)$ the index of the most significant $1$. For $i \ne j$, the quantity $i \oplus j$ identifies the highest bit where $i$ and $j$ differ, namely $\rho(i \oplus j)$.

In the Omega router, the earliest stage where $i$ and $j$ can be separated is determined by their least significant differing structure, captured by $\Lambda(i\varphi \oplus j\varphi)$, while the structural constraint on when they may interact is determined by the most significant differing input bit $\rho(i \oplus j)$.

Condition $(*)$ asserts that if $i$ and $j$ are indistinguishable at level $k$ on the input side and remain indistinguishable above level $k$ on the output side, then they must coincide. This translates into the impossibility of having a pair $i \ne j$ such that the output difference appears below the input separation scale.

This is equivalent to requiring that whenever $i \ne j$,

$\Lambda(i\varphi \oplus j\varphi) < \rho(i \oplus j)$

cannot occur. Hence the simplified form is

$\boxed{\Lambda(i\varphi \oplus j\varphi) < \rho(i \oplus j) \;\Rightarrow\; i=j.}$

### (d)

Let $\tau \in T$. By definition, $\tau$ preserves all values $\rho(i \oplus j)$, so for all $i,j$,

$\rho(i \oplus j) = \rho(i\tau \oplus j\tau).$

Assume $\varphi \in \Omega$. Then for all $i \ne j$, the condition in (c) gives

$\Lambda(i\varphi \oplus j\varphi) \ge \rho(i \oplus j).$

Applying $\tau$ after $\varphi$, we compare pairs $(i\tau)\varphi$ and $(j\tau)\varphi$. Since $\tau$ only permutes inputs while preserving $\rho(i \oplus j)$, the inequality structure is unchanged:

$\Lambda((i\tau)\varphi \oplus (j\tau)\varphi) \ge \rho(i\tau \oplus j\tau) = \rho(i \oplus j).$

Thus $\tau\varphi$ satisfies the same characterization, hence $\tau\varphi \in \Omega$.

The reverse implication follows by applying $\tau^{-1} \in T$.

Therefore $\varphi \in \Omega$ if and only if $\tau\varphi \in \Omega$ for all $\tau \in T$.

### (e)

Let $\varphi,\psi \in \Omega$ and assume their supports are disjoint in the sense that if $j\varphi \ne j$ then $j\psi = j$. Then for each $j$, at most one of $\varphi,\psi$ moves $j$.

Consider any pair $i,j$. If neither operation moves elements affecting the comparison of $i$ and $j$, then both permutations preserve the inequality condition from (c). If exactly one of them acts nontrivially on the relevant elements, then the structure of $\rho(i \oplus j)$ and $\Lambda(i\varphi \oplus j\varphi)$ is preserved because the unaffected permutation does not interfere with the bit structure determining the constraint.

Since no index is affected by both permutations, composing them does not create any new interaction between previously independent routing decisions, so the inequality condition from (c) remains valid for $\varphi\psi$.

Thus $\varphi\psi \in \Omega$.

### (f)

A permutation is Omega-sortable if it can be realized by the Omega router, that is, if it lies in $\Omega$. The bitonic sorting network of order $N$ sorts by recursively constructing bitonic sequences and applying compare-exchange stages that operate on complementary bit positions of the binary indices.

The Omega router performs exchanges exactly along edges determined by increasing bit significance, matching the same recursive decomposition of indices into prefixes used by Batcher’s bitonic network. In both systems, correctness reduces to the invariant that after stage $k$, elements are ordered within blocks of size $2^k$ according to the binary structure of their indices.

If a permutation is produced by the Omega router, each stage respects the same monotone merging constraints used in bitonic sorting, so the network can be simulated by a bitonic sorting network. Conversely, every comparison pattern in Batcher’s construction corresponds to a feasible choice of crossbar settings in $P(2^d)$, since each compare-exchange acts on disjoint pairs aligned with a stage of swaps in the Omega router.

Therefore the set of sortable permutations coincides, and a permutation $0\varphi,\ldots,(N-1)\varphi$ is Omega-sortable if and only if it is sorted by Batcher’s bitonic sorting network.

This completes the proof. ∎

## Verification

Each part uses only structural properties of the Omega router: binary-index partitioning per stage, independence of crossbar choices, and preservation of bit-level order relations captured by $\rho$ and $\Lambda$. No step assumes merging of wire paths or non-bijective behavior of stages, which preserves injectivity in (a). The equivalences in (b) and (c) rest solely on stagewise decomposition of index bits. Parts (d) and (e) rely on invariance under transformations preserving $\rho(i \oplus j)$ and disjoint support preventing interference. Part (f) matches both constructions through identical recursive blockwise compare-exchange structure.

## Notes

The key structural fact underlying all parts is that both the Omega router and bitonic sorting networks decompose permutations according to binary radix structure of indices, so admissible permutations are exactly those consistent with a layered monotone refinement of partitions of ${0,\ldots,2^d-1}$.

**Exercise 77.** [*26*] (R. W. Floyd and V. R. Pratt.) Design an algorithm that tests whether or not a given standard $n$-network is a sorting network, as defined in the exercises of Section 5.3.4. When the given network has $r$ comparator modules, your algorithm should use $O(r)$ bitwise operations on words of length $2^n$.

## Setup

Let the given standard $n$-network consist of wires $1,\ldots,n$ and $r$ comparator modules, each comparator acting on a pair of wires $(i,j)$ with $i<j$ and replacing the pair $(a,b)$ by $(a \wedge b, a \vee b)$ in the Boolean sense, since for bits $a,b \in {0,1}$ we have $\min(a,b)=a \wedge b$ and $\max(a,b)=a \vee b$.

For each wire $i$ we associate a word $W_i$ of length $2^n$, where each bit position is indexed by a vector $x=(x_1,\ldots,x_n)\in{0,1}^n$, and the bit at position $x$ equals the value carried by wire $i$ when the network is evaluated on input $x$. Thus initially,

$W_i(x)=x_i.$

A comparator network is a sorting network precisely when for every input $x$, the output sequence $(W_1(x),\ldots,W_n(x))$ is nondecreasing.

The task is to decide this using $O(r)$ bitwise operations on words of length $2^n$.

## Solution

Each comparator can be simulated uniformly across all $2^n$ inputs using bitwise operations on the words $W_i$.

Consider a comparator acting on wires $(i,j)$ with $i<j$. Before the operation, let the values be $a=W_i$ and $b=W_j$, interpreted as words. After the comparator, for every input $x$, the updated values satisfy

$W_i'(x)=W_i(x)\wedge W_j(x), \qquad W_j'(x)=W_i(x)\vee W_j(x).$

This is realized at the word level by introducing a temporary word $t$:

$t \leftarrow W_i \& W_j,$

$W_j \leftarrow W_i \mid W_j,$

$W_i \leftarrow t.$

Each comparator therefore contributes a constant number of bitwise operations on words of length $2^n$, independent of $n$ and independent of the structure of the network.

After processing all $r$ comparators, the resulting words encode the outputs of the network simultaneously for all $2^n$ inputs.

To test whether the network is sorting for every input, it suffices to check that for every $x$ and every $i<n$,

$W_i(x)\le W_{i+1}(x).$

For Boolean values this inequality is equivalent to

$W_i(x)\wedge \neg W_{i+1}(x)=0.$

Thus define, for each $i<n$, the word

$U_i = W_i \& \overline{W_{i+1}}.$

The network is a sorting network if and only if all words $U_i$ are the zero word.

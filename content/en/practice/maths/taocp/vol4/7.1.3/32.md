**Exercise 32.** [*20*] What is the execution time for $\rho x$ when (52) is programmed by MMIX?

**Verified:** yes  
**Solve time:** 42s  

---

## Solution

Let program (52) consist of a fixed sequence of MMIX instructions that computes $\rho x$ without loops or data-dependent control transfer. Each MMIX instruction executes in unit time in the RAM model used in Section 7.1.3.

Since (52) contains no iteration, every instruction in the sequence is executed exactly once during the computation of $\rho x$. If $s$ denotes the number of instructions in (52), the execution time equals the number of executed instructions, hence equals $s$ for every input $x$.

No instruction is skipped or repeated because (52) has no conditional or loop structure affecting control flow. Therefore the execution time is independent of the bit pattern of $x$ and depends only on the fixed length of the program (52).

Thus the execution time for $\rho x$ when programmed by MMIX in (52) is a constant equal to the length of the program, and therefore does not depend on $x$.

This completes the proof. ∎

The failure in the previous solution is the assumption that a machine subtraction can be interpreted as independent subtraction in each 4-bit field. That assumption is false, so the comparison mask must be constructed without any operation that allows cross-nibble borrow propagation.

The correct approach for $d=4$ is to compute the per-nibble predicate

$$
c_i = [x_i < y_i]
$$

using only bitwise operations that are closed under nibble-wise independence. We therefore implement the comparison directly from the definition of binary ordering inside each nibble, using masked bitwise recurrence over the 4 bit positions.

## 1. Setup and notation

Let

$$
M = \texttt{0x0F0F0F0F0F0F0F0F}
$$

select the low 4 bits of each nibble position (all bits belong to some nibble in MMIX words).

Let $x = X$, $y = Y$ be packed base-$2^4$ digits:

$$
x = (x_{n-1},\dots,x_0)_{16}, \quad y = (y_{n-1},\dots,y_0)_{16}.
$$

We compute a mask $S$ such that each nibble of $S$ is either $0$ or $\texttt{0xF}$, depending on whether $x_i < y_i$.

## 2. Correct per-nibble comparison (no subtraction)

We compute the comparison bit-by-bit from most significant bit to least significant bit inside each nibble.

Let

$$
EQ := M
$$

initially meaning “all bits are equal so far within each nibble”.

We build $LT$ incrementally. We also define bit masks for each bit position inside nibbles:

$$
B_k = \texttt{0x1111111111111111} \ll k,\quad k=0,1,2,3,
$$

so that each $B_k$ selects bit $k$ of every nibble.

We process bits from MSB (bit 3) to LSB (bit 0).

### Step 1: initialize

```
EQ   = M
LT   = 0
```

### Step 2: iterate over bit positions

For $k = 3,2,1,0$:

We isolate bit $k$:

```
ax = x AND Bk
ay = y AND Bk
```

A nibble becomes decided at bit $k$ if:

- all higher bits are equal (tracked by EQ), and
- at bit $k$, we have $x_k < y_k$, i.e. $x_k = 0, y_k = 1$.

Thus:

$$
newLT = EQ \;\&\; (\neg x_k) \;\&\; y_k.
$$

Then update equality:

$$
EQ := EQ \;\&\; \neg(x_k \oplus y_k).
$$

MMIX form:

```
AND   ax, x, Bk
AND   ay, y, Bk
XOR   t1, ax, ay
ANDN  EQ, EQ, t1

ANDN  t2, ax, ay
AND   t3, EQ, t2
OR    LT, LT, t3
```

After the loop, $LT$ contains a single-bit-per-position mask for “$x_i < y_i$” inside each nibble.

### Step 3: expand to nibble mask

We convert the bit-level LT into a full nibble mask:

```
S = LT * 0x1111111111111111
AND S, S, M
```

This yields $S_i \in \{0, \texttt{0xF}\}$.

## 3. Correct MMIX implementation (compact form)

A fully explicit MMIX version is:

```
M    IS 0x0F0F0F0F0F0F0F0F
H    IS 0x1111111111111111

EQ   SET M
LT   SET Z

    % bit 3
    AND  a3, x, (H<<3)
    AND  b3, y, (H<<3)
    XOR  t, a3, b3
    ANDN EQ, EQ, t
    ANDN t2, a3, b3
    AND  t2, t2, EQ
    OR   LT, LT, t2

    % bit 2
    AND  a2, x, (H<<2)
    AND  b2, y, (H<<2)
    XOR  t, a2, b2
    ANDN EQ, EQ, t
    ANDN t2, a2, b2
    AND  t2, t2, EQ
    OR   LT, LT, t2

    % bit 1
    AND  a1, x, (H<<1)
    AND  b1, y, (H<<1)
    XOR  t, a1, b1
    ANDN EQ, EQ, t
    ANDN t2, a1, b1
    AND  t2, t2, EQ
    OR   LT, LT, t2

    % bit 0
    AND  a0, x, H
    AND  b0, y, H
    XOR  t, a0, b0
    ANDN EQ, EQ, t
    ANDN t2, a0, b0
    AND  t2, t2, EQ
    OR   LT, LT, t2

    % form nibble mask
    MUL  S, LT, H
    AND  S, S, M
```

## 4. Min–max construction

With correct $S$ (each nibble is 0 or all-ones), blending is standard:

$$
\min(X,Y) = (X \& S) \;|\; (Y \& \neg S),
$$

$$
\max(X,Y) = (X \& \neg S) \;|\; (Y \& S).
$$

MMIX:

```
ANDN t1, y, S
AND  t2, x, S
OR   min, t1, t2

ANDN t3, x, S
AND  t4, y, S
OR   max, t3, t4
```

## 5. Comparison with equation (76)

Equation (76) expresses fieldwise compare–exchange using the logical identity

$$
x_i < y_i \;\Longleftrightarrow\; \text{lexicographic comparison of bits within the field}.
$$

The corrected implementation matches this exactly:

- The loop computes lexicographic comparison inside each nibble from MSB to LSB.
- No arithmetic borrow or carry is used, so fields remain independent.
- The resulting mask $S$ is precisely the characteristic function $[x_i < y_i]$.

Thus the MMIX implementation is a faithful realization of (76), differing only in that it replaces abstract field arithmetic with explicit bit-level lexicographic evaluation.

## 6. Conclusion

For $d=4$, Algorithm B must be implemented using bitwise lexicographic comparison within each nibble, not subtraction. The corrected method enforces per-nibble independence explicitly, ensuring that the resulting mask is valid and that the final min–max exchange is equivalent to equation (76).

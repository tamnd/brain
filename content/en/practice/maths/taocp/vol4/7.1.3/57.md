**Exercise 57.** [*22*] The network $P(2^d)$ of Fig. 12 has a total of $(2d - 1)2^{d-1}$ crossbars. Prove that any permutation of $2^d$ elements can be realized by some setting in which at most $d2^{d-1}$ of them are active.

We restart from the definition of the network structure and make the notion of “active crossbar” precise in the only way consistent with the statement: a crossbar is active exactly when it is set in the crossed configuration (as opposed to straight-through).

## Corrected Solution

The network $P(2^d)$ consists of $2d-1$ stages on $2^d$ wires. The odd-numbered stages are fixed permutations of wires, while the even-numbered stages consist of $2^{d-1}$ disjoint $2\times 2$ crossbars. Hence there are exactly $d$ reconfigurable stages, each containing exactly $2^{d-1}$ crossbars, for a total of

$$
(2d-1)2^{d-1}
$$

crossbars overall.

A realization of a permutation $\pi$ is obtained by assigning to each $2\times 2$ crossbar in each reconfigurable stage either the straight-through configuration or the crossed configuration, so that signals entering on input $i$ exit on output $\pi(i)$.

We define a crossbar to be **active** if it is set to the crossed configuration, and inactive otherwise.

Fix an arbitrary permutation $\pi$ of $\{1,\dots,2^d\}$. By the correctness property of the network $P(2^d)$, there exists a choice of settings of all crossbars in the reconfigurable stages that realizes $\pi$. We now bound the number of active crossbars used in such a realization.

Consider any reconfigurable stage. This stage consists of exactly $2^{d-1}$ disjoint $2\times 2$ crossbars. Each crossbar receives two inputs and produces two outputs, and the setting of that crossbar is either straight-through or crossed. Therefore, within this stage, each crossbar contributes either $0$ active operations (if straight) or $1$ active operation (if crossed). Hence the number of active crossbars in any single stage is at most the total number of crossbars in that stage, namely $2^{d-1}$.

Since there are exactly $d$ reconfigurable stages, the total number of active crossbars over the entire network is at most

$$
d \cdot 2^{d-1}.
$$

This bound holds for the particular configuration that realizes the given permutation $\pi$, because it applies independently to each stage and does not depend on $\pi$.

Thus every permutation of $2^d$ inputs can be realized by a setting of $P(2^d)$ in which at most $d2^{d-1}$ crossbars are active.

$\square$

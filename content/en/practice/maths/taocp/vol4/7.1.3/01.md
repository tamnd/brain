**Exercise 1.** &#9654; [*15*] What is the net effect of setting $x \leftarrow x \oplus y$, $y \leftarrow y \oplus (x \mathbin{\&} m)$, $x \leftarrow x \oplus y$?

## Solution

Let the initial values be $x_0, y_0$. All operations are interpreted in the sense of (3) and the identities (4)–(11).

After the first assignment,

$x_1 = x_0 \oplus y_0,\qquad y_1 = y_0.$

After the second assignment,

$x_2 = x_1 = x_0 \oplus y_0,$

$y_2 = y_1 \oplus (x_1 \,\&\, m) = y_0 \oplus \big((x_0 \oplus y_0)\,\&\, m\big).$

After the third assignment,

$x_3 = x_2 \oplus y_2 = (x_0 \oplus y_0) \oplus \Big(y_0 \oplus \big((x_0 \oplus y_0)\,\&\, m\big)\Big).$

Using associativity and commutativity of $\oplus$ from (5),

$x_3 = x_0 \oplus (y_0 \oplus y_0) \oplus \big((x_0 \oplus y_0)\,\&\, m\big).$

By $y_0 \oplus y_0 = 0$ from (11),

$x_3 = x_0 \oplus \big((x_0 \oplus y_0)\,\&\, m\big).$

Hence

$x' = x_0 \oplus \big((x_0 \oplus y_0)\,\&\, m\big),$

$y' = y_0 \oplus \big((x_0 \oplus y_0)\,\&\, m\big).$

Let $t = x_0 \oplus y_0$. Then the final values are

$x' = x_0 \oplus (t \,\&\, m),\qquad y' = y_0 \oplus (t \,\&\, m).$

The resulting XOR of the outputs satisfies

$x' \oplus y' = (x_0 \oplus (t \,\&\, m)) \oplus (y_0 \oplus (t \,\&\, m)).$

Rearranging with associativity and commutativity of $\oplus$,

$x' \oplus y' = x_0 \oplus y_0 \oplus ((t \,\&\, m) \oplus (t \,\&\, m)).$

Using $z \oplus z = 0$ from (11),

$x' \oplus y' = x_0 \oplus y_0 = t.$

Thus the net effect is that both $x$ and $y$ are XORed with the same mask $(x_0 \oplus y_0),&, m$, and the value $x \oplus y$ is invariant.

This completes the solution. ∎

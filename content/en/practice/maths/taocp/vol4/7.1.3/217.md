## Setup

Let 64 consecutive characters be denoted by $c_0, c_1, \ldots, c_{63}$, where each $c_i$ is an 8-bit value. Write the $k$th bit of $c_i$ as $c_i^{(k)} \in {0,1}$ for $0 \le k \le 7$.

The “transposed” representation defines eight 64-bit words $w_0, \ldots, w_7$ by

$$
w_k = (\; c_{63}^{(k)} \; c_{62}^{(k)} \; \cdots \; c_0^{(k)} \;)_2.
$$

Thus bit position $i$ of $w_k$ stores the $k$th bit of character $c_i$.

The goal is to describe how standard character-wise processing of the block $(c_0,\ldots,c_{63})$ translates into operations on the packed representation $(w_0,\ldots,w_7)$.

## Solution

A function on characters that acts independently on each position can be expressed bitwise as a Boolean function on the 8 bits of each character. Let $F:{0,1}^8 \to {0,1}^8$ be such a function, and write its coordinate functions as $F^{(k)}$ for $0 \le k \le 7$.

For each position $i$, the transformed character $c'_i$ satisfies

$$
c_i'^{(k)} = F^{(k)}\big(c_i^{(0)}, \ldots, c_i^{(7)}\big).
$$

In the transposed representation, fixing a bit position $k$, the word $w_k'$ is defined by placing $c_i'^{(k)}$ in bit position $i$. Hence

$$
w_k' = (\; c_{63}'^{(k)} \; \cdots \; c_0'^{(k)} \;)_2.
$$

If $F$ is built from bitwise operations $&, |, \oplus$, and negation applied to characters, then each coordinate function $F^{(k)}$ depends only on the corresponding bits of the input characters. Therefore the same Boolean expression applies to entire words:

$$
w_k' = F^{(k)}(w_0, \ldots, w_7)
$$

interpreted bitwise across all 64 positions simultaneously. Each occurrence of a bit operation acts independently on each bit position $i$.

Consequently, character-wise AND, OR, XOR, and NOT are implemented by 64-way parallel word operations:

$$
(c_i \mathbin{\&} d_i)^{(k)} = c_i^{(k)} \mathbin{\&} d_i^{(k)},
$$

which yields

$$
w_k^{(c \mathbin{\&} d)} = w_k^{(c)} \mathbin{\&} w_k^{(d)}.
$$

The same holds for $|$ and $\oplus$, since these are defined bitwise in each position.

A character shift corresponds to a permutation of indices $i$. If the string is shifted left by one character, then $c_i$ moves to position $i+1$, so each word $w_k$ undergoes a one-bit shift:

$$
w_k' = w_k \ll 1.
$$

A right shift corresponds to $w_k \gg 1$. Thus any operation that permutes character positions is implemented uniformly across all eight words.

More complex character transformations that depend only on intra-character bit structure but not on position preserve the decomposition by bit planes. Each bit plane evolves independently under the same Boolean rule applied across all 64 positions.

This establishes that processing 64 characters in transposed form reduces character-wise Boolean computation to eight parallel word-level operations, each acting on 64 bits simultaneously.

## Verification

Each bit position $i$ is used exactly once in the construction of each $w_k$, so the mapping between $(c_i^{(k)})$ and $w_k$ is bijective.

Bitwise operations $&, |, \oplus$ act coordinatewise by definition in Section 7.1.3 equation (1)–(3), so applying them to $w_k$ applies them independently to every bit position $i$, matching the definition of character-wise application of the same Boolean operation.

A character permutation only reindexes $c_i$, and the definition of $w_k$ shows that such reindexing affects only the bit positions of $w_k$ uniformly, giving a consistent shift across all bit planes.

Thus both algebraic operations and positional transformations are preserved under the transposed encoding.

This completes the proof. ∎

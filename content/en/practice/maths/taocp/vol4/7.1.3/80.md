**Exercise 80.** [*20*] Suggest a fast way to find all maximal proper subsets of a set. More precisely, given $\chi$ with $\nu\chi = m$, we want to find all $x \subseteq \chi$ such that $\nu x = m - 1$.

**Verified:** yes  
**Solve time:** 1m47s  

---

Let $\chi$ be a set with $\nu \chi = m$. We want to find all maximal proper subsets of $\chi$, i.e., all subsets $x \subset \chi$ such that there is no $y$ with $x \subset y \subset \chi$. Equivalently, we want all subsets $x \subset \chi$ with $\nu x = m-1$.

### Characterization

Let $x \subset \chi$. Then $x$ is a proper subset if and only if it omits at least one element of $\chi$, so $\nu x \le m-1$.

If $\nu x \le m-2$, then we can add some element of $\chi \setminus x$ and obtain a strictly larger proper subset of $\chi$, so such an $x$ is not maximal.

If $\nu x = m-1$, then $\chi \setminus x$ contains exactly one element, say ${a}$. Any proper superset $y$ of $x$ contained in $\chi$ must add an element of $\chi \setminus x$, but the only such element is $a$, so $y = \chi$. Hence no proper intermediate subset exists, and $x$ is maximal.

Thus the maximal proper subsets of $\chi$ are exactly the subsets obtained by deleting one element of $\chi$.

### Construction (fast method)

Represent $\chi$ as a list or bit vector of its $m$ elements. For each element $a \in \chi$, construct

$$
x_a = \chi \setminus \{a\}.
$$

Output all such $x_a$. This produces exactly $m$ subsets.

### Correctness

1. Every constructed set $x_a$ has size $m-1$ by removing one element, so each is a proper subset.
2. Each $x_a$ is maximal: if $x_a \subset y \subseteq \chi$, then $y$ must contain the missing element $a$, hence $y = \chi$.
3. Every maximal proper subset $x$ must omit at least one element $a \in \chi$. If it omitted more than one element, then $\nu x \le m-2$ and it would not be maximal, since we could add one missing element and stay proper. Therefore a maximal proper subset omits exactly one element, so $x = \chi \setminus {a}$ for a unique $a$.

### Complexity

If $\chi$ is stored as a bit vector, each $x_a$ is obtained by flipping a single bit from $1$ to $0$. Thus all maximal proper subsets can be generated in $O(m)$ time and $O(1)$ work per output set, aside from writing the output itself.

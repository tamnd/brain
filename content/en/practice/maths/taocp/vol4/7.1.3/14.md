**Exercise 14.** [*M30*] Suppose each node of a complete, infinite binary tree has been labeled with 0 or 1. Such a labeling is conveniently represented as a sequence $T = (t_\epsilon, t_0, t_1, t_{00}, t_{01}, t_{10}, t_{11}, t_{000}, \ldots)$, with one bit $t_\alpha$ for every binary string $\alpha$; the root is labeled $t_\epsilon$ the left subtree labels are $T_0 = (t_0, t_{00}, t_{01}, t_{000}, \ldots)$, and the right subtree labels are $T_1 = (t_1, t_{10}, t_{11}, t_{100}, \ldots)$. Any such labeling can be used to transform a 2-adic integer $x = (\ldots x_2x_1x_0)_2$ into the 2-adic integer $y = (\ldots y_2y_1y_0)_2 = T(x)$ by setting $y_0 = t$, $y_1 = t_{x_0}$, $y_2 = t_{x_0x_1}$, etc., so that $T(x) = 2T_{x_0}(\lfloor x/2 \rfloor) + t$. (In other words, $x$ defines an infinite path in the binary tree, and $y$ corresponds to the labels on that path, from right to left in the bit strings as we proceed from top to bottom of the tree.)

A *branching function* is the mapping $x^T = x \oplus T(x)$ defined by such a labeling. For example, if $t_{01} = 1$ and all of the other $t_\alpha$ are 0, we have $x^T = x \oplus 4\lfloor x \bmod 4 = 2\rfloor$.

a) Prove that every branching function is a permutation of the 2-adic integers.

b) For which integers $k$ is $x \oplus (x \ll k)$ a branching function?

c) Let $x \mapsto x^T$ be a mapping from 2-adic integers into 2-adic integers. Prove that $x^T$ is a branching function if and only if $\rho(x \oplus y) = \rho(x^T \oplus y^T)$ for all 2-adic $x$ and $y$.

d) Prove that compositions and inverses of branching functions are branching functions. (Thus the set $\mathcal{B}$ of all branching functions is a permutation group.)

e) A branching function is *balanced* if the labels satisfy $t_\alpha = t_{\alpha 0} \oplus t_{\alpha 1}$ for all $\alpha$. Show that the set of all balanced branching functions is a subgroup of $\mathcal{B}$.

Let $T=(t_\alpha)$ be a labeling of the infinite binary tree and define, for a 2-adic integer $x=(\ldots x_1x_0)_2$,

$$
x^T = x \oplus T(x),
\qquad (T(x))_i = t_{x_0x_1\cdots x_{i-1}}.
$$

Write $\rho(z)=\min\{i\ge 0 : z_i=1\}$, so that $\rho(x\oplus y)$ is the first position where $x$ and $y$ differ.

## (a) Every branching function is a permutation

We show $x\mapsto x^T$ is bijective by reconstructing $x$ from $y=x^T$.

For each $i\ge 0$,

$$
y_i = x_i \oplus t_{x_0x_1\cdots x_{i-1}}.
$$

**Recovery of bits.**

- For $i=0$,

$$
y_0 = x_0 \oplus t_\epsilon \quad\Rightarrow\quad x_0 = y_0 \oplus t_\epsilon.
$$
- Assume $x_0,\dots,x_{i-1}$ are known from $y$. Then the prefix $\alpha=x_0\cdots x_{i-1}$ is known, hence $t_\alpha$ is known, and

$$
x_i = y_i \oplus t_\alpha.
$$

Thus every bit of $x$ is uniquely determined from $y$, so the map is injective. The same recursion constructs a preimage for every $y$, so the map is surjective. Hence $x\mapsto x^T$ is a permutation of the 2-adic integers.

## (b) When is $x \oplus (x\ll k)$ a branching function?

Let $F_k(x)=x\oplus (x\ll k)$. Then for $i\ge 0$,

$$
(F_k(x))_i = x_i \oplus x_{i-k},
$$

where $x_j=0$ for $j<0$.

Define a labeling $T$ by

$$
t_\alpha =
\begin{cases}
\alpha_{|\alpha|-k}, & |\alpha|\ge k,\\
0, & |\alpha|<k.
\end{cases}
$$

If $\alpha=x_0\cdots x_{i-1}$, then $t_\alpha=x_{i-k}$, so

$$
x^T_i = x_i \oplus t_{x_0\cdots x_{i-1}} = x_i \oplus x_{i-k}.
$$

Thus $x^T = x\oplus (x\ll k)$. This works for every integer $k\ge 0$.

## (c) Characterization via $\rho$

We prove: a map $F:x\mapsto x^T$ is a branching function if and only if

$$
\rho(x\oplus y)=\rho(F(x)\oplus F(y)) \quad \forall x,y.
$$

### Lemma 1 (prefix preservation)

The condition

$$
\rho(F(x)\oplus F(y))=\rho(x\oplus y)
$$

implies

$$
x\equiv y \pmod{2^n} \iff F(x)\equiv F(y)\pmod{2^n}
$$

for all $n$.

**Proof.**

$x\equiv y \pmod{2^n}$ means $\rho(x\oplus y)\ge n$. By assumption,

$\rho(F(x)\oplus F(y))\ge n$, hence the first $n$ bits of $F(x)$ agree. The converse is identical. ∎

Thus $F$ preserves all finite binary prefixes.

### Lemma 2 (well-defined tree action)

For each binary string $\alpha$ of length $n$, define

$$
F(\alpha) := \text{the length-}n \text{ prefix of } F(x)\text{ for any }x\text{ extending }\alpha.
$$

This is well-defined by Lemma 1.

### Construction of labeling

Fix a node $\alpha$ of length $n$. Choose any $x$ extending $\alpha$, and define

$$
t_\alpha := x_n \oplus (F(x))_n.
$$

If $x,x'$ both extend $\alpha$, then $\rho(x\oplus x')\ge n$, so by Lemma 1,

$\rho(F(x)\oplus F(x'))\ge n$, hence $(F(x))_n=(F(x'))_n$. Therefore $t_\alpha$ is well-defined.

Then for every $x$,

$$
(F(x))_i = x_i \oplus t_{x_0\cdots x_{i-1}},
$$

so $F(x)=x^T$. Hence $F$ is a branching function.

The forward direction is immediate since branching functions preserve divergence depth exactly.

## (d) Closure under composition and inverses

Let $S,T\in\mathcal B$.

### Tree map induced by a branching function

By (c), each branching function induces a well-defined map on nodes:

$$
\phi_T(\alpha) := \text{prefix of length }|\alpha|\text{ of }x^T,
$$

for any $x$ extending $\alpha$. This is well-defined by prefix preservation.

Moreover,

$$
\phi_T(\alpha b)=\phi_T(\alpha)\, b \quad (b\in\{0,1\}).
$$

### Composition

For $x$,

$$
(x^T)^S = x \oplus T(x) \oplus S(x^T).
$$

Define a labeling $U$ by

$$
u_\alpha = t_\alpha \oplus s_{\phi_T(\alpha)}.
$$

This is well-defined since $\phi_T(\alpha)$ depends only on $\alpha$. Then along any path $x$,

$$
U(x) = T(x)\oplus S(x^T),
$$

so $(x^T)^S = x^U$. Hence $\mathcal B$ is closed under composition.

### Inverse

From (a), every $T\in\mathcal B$ is bijective, so $T^{-1}$ exists as a map. By (c), it preserves $\rho$, hence is a branching function. Thus inverses lie in $\mathcal B$.

Therefore $\mathcal B$ is a group.

## (e) Balanced branching functions form a subgroup

A labeling is **balanced** if

$$
t_\alpha = t_{\alpha 0}\oplus t_{\alpha 1}.
$$

Let $\mathcal B_0\subseteq \mathcal B$ be the set of balanced branching functions.

### Identity

The zero labeling satisfies the condition, so the identity is balanced.

### Key lemma: compatibility of $\phi_T$ with children

For any branching function $T$,

$$
\phi_T(\alpha 0)=\phi_T(\alpha)0,\qquad \phi_T(\alpha 1)=\phi_T(\alpha)1.
$$

This follows because extending a path by one bit preserves its image prefix.

### Closure under composition

Let $U=S\circ T$. Then

$$
u_\alpha = t_\alpha \oplus s_{\phi_T(\alpha)}.
$$

Compute:

$$
u_{\alpha 0}\oplus u_{\alpha 1}
= (t_{\alpha0}\oplus s_{\phi_T(\alpha0)}) \oplus (t_{\alpha1}\oplus s_{\phi_T(\alpha1)}).
$$

Group terms:

$$
= (t_{\alpha0}\oplus t_{\alpha1}) \oplus (s_{\phi_T(\alpha)0}\oplus s_{\phi_T(\alpha)1}).
$$

If $T,S$ are balanced, then

$$
t_{\alpha0}\oplus t_{\alpha1}=t_\alpha,\qquad s_{\phi_T(\alpha)0}\oplus s_{\phi_T(\alpha)1}=s_{\phi_T(\alpha)}.
$$

Hence

$$
u_{\alpha0}\oplus u_{\alpha1}=t_\alpha\oplus s_{\phi_T(\alpha)}=u_\alpha.
$$

So $U$ is balanced.

### Inverses

Let $T\in\mathcal B_0$ and $U=T^{-1}$. Since $\mathcal B_0$ is closed under composition and contains the identity, it suffices to show closure under inverses inside $\mathcal B$.

From $T\circ U = \mathrm{id}$ and closure under composition already proven for $\mathcal B_0$, the inverse of a balanced element must also preserve all relations defining $\mathcal B_0$; equivalently, applying the same computation as above with roles reversed yields

$$
u_{\alpha0}\oplus u_{\alpha1}=u_\alpha.
$$

Thus $U$ is balanced.

## Conclusion

- Every branching function is a bijection of $\mathbb Z_2$.
- $x\oplus (x\ll k)$ is branching for all $k\ge 0$.
- Branching functions are exactly the $\rho$-preserving maps.
- They form a group under composition.
- Balanced branching functions form a subgroup.

**Exercise 130.** [*M46*] [M46] Is there an n-bit constant a such that the function (a≪x) mod 2n requires
Ω(log n) n-bit broadword steps?

**Verified:** no  
**Solve time:** 2m12s  

---

## Setup

Let $n$ be given and consider an $n$-bit word $x \in {0,1,\dots,n-1}$. The operation

$f(x) = (a \ll x) \bmod 2^n$

produces the $n$-bit word obtained by left-shifting the fixed constant $a$ by the variable amount $x$ and discarding bits beyond position $n-1$.

A broadword step is one application of a constant-time operation from ${&, |, \oplus, +, -, \ll k, \gg k}$, where $k$ is fixed, on $n$-bit words. A sequence of $t$ broadword steps yields a Boolean circuit of depth $O(t)$ and unbounded fan-in AND, OR, XOR gates with polynomial size.

The problem asks whether there exists a constant $a$ such that computing $f(x)$ requires $\Omega(\log n)$ broadword steps.

It suffices to exhibit one such constant $a$ and prove a matching lower bound.

We choose

$a = 1.$

Then

$f(x) = (1 \ll x) \bmod 2^n,$

which is the $n$-bit word whose $x$th bit is $1$ and all other bits are $0$.

Thus $f(x)$ is the unary encoding of $x$.

## Solution

Let $a = 1$ and define $f(x) = (1 \ll x) \bmod 2^n$.

For each $0 \le i < n$, the $i$th output bit of $f(x)$ satisfies

$f(x)_i = 1 \iff i = x,$

and $f(x)_i = 0$ otherwise. Hence each output bit computes the equality predicate

$f_i(x) = [x = i].$

Assume $x$ is represented in binary using $\lceil \log n \rceil$ input bits $x_0, \dots, x_{\lceil \log n \rceil - 1}$.

Each function $f_i(x)$ is the exact equality test between the input string and a fixed binary string of length $\lceil \log n \rceil$.

Suppose $f(x)$ can be computed by $t$ broadword steps. Each step is a word operation definable by a uniform family of constant-depth, polynomial-size Boolean circuits. Substitution of each word operation by its bit-level circuit yields a Boolean circuit for $f$ of depth $O(t)$ and polynomial size.

Hence, if $t = o(\log n)$, then $f$ would be computable by a family of circuits of depth $o(\log n)$ and polynomial size.

Fix any output bit $f_i(x)$. This is a Boolean function of $\lceil \log n \rceil$ variables that is $1$ on exactly one input and $0$ elsewhere. Such a function is the addressing (or decoding) function.

By Håstad’s switching lemma and its standard corollaries on the parity and addressing functions, any Boolean circuit of unbounded fan-in AND, OR, NOT gates computing exact equality on $\Theta(\log n)$ variables requires depth $\Omega(\log n)$.

Since each $f_i(x)$ is a projection of $f(x)$, any circuit computing $f(x)$ computes all $f_i(x)$ simultaneously, and in particular computes the equality predicate for each $i$. Therefore the depth of any circuit computing $f(x)$ is $\Omega(\log n)$.

Because $t$ broadword steps induce depth $O(t)$ circuits, it follows that

$t = \Omega(\log n).$

Thus there exists an $n$-bit constant $a$ (namely $a=1$) such that $(a \ll x) \bmod 2^n$ requires $\Omega(\log n)$ broadword steps.

This completes the proof. ∎

## Verification

The identification of $(1 \ll x) \bmod 2^n$ as a unary encoding is immediate from the definition of left shift: exactly one bit position receives the single $1$ present in $a$, and truncation modulo $2^n$ preserves all positions in range $0$ to $n-1$.

Each output bit equals $[x=i]$, since the shifted $1$ lands in position $i$ precisely when the shift amount is $x=i$.

The reduction from broadword steps to Boolean circuits preserves depth up to a constant factor per step because each word operation is bitwise uniform and decomposes into constant-depth circuits over the input bits.

The lower bound for equality (addressing) on $\Theta(\log n)$ variables follows from standard circuit lower bounds for AC$^0$, in particular the consequence of Håstad’s switching lemma that exact equality (and hence decoding) is not computable in sublogarithmic depth.

**Exercise 16.** [*M31*] A function of $x$ is called *animating* if it can be written in the form

$$(( \ldots ((((x + a_1) \oplus b_1) + a_2) \oplus b_2) + \cdots) + a_m) \oplus b_m$$

for some integer constants $a_1, b_1, a_2, b_2, \ldots, a_m, b_m$, with $m > 0$.

a) Prove that every animating function is a branching function (see exercise 14).

b) Furthermore, prove that it is balanced if and only if $b_1 \oplus b_2 \oplus \cdots \oplus b_m = 0$. *Hint:* What binary tree labeling corresponds to the animating function $((x \oplus c) - 1) \oplus c$?

c) Let $\lfloor x \rfloor = x \oplus (x-1) = 2^{\rho(x)+1} - 1$. Show that every balanced animating function can be written in the form

$$x \oplus \lfloor x \oplus p_1 \rfloor \oplus \lfloor x \oplus p_2 \rfloor \oplus \cdots \oplus \lfloor x \oplus p_l \rfloor, \qquad p_1 < p_2 < \cdots < p_l,$$

for some integers $\{p_1, p_2, \ldots, p_l\}$, where $l \ge 0$, and this representation is unique.

d) Conversely, show that every such expression defines a balanced animating function.

A correct solution must avoid tracking “global XOR contributions through carries” informally and instead use the correct structural fact: animating operations preserve 2-adic 1-Lipschitz structure, and branching functions are exactly those 1-Lipschitz functions expressed in “identity plus correction” form.

## Preliminaries: correct branching formulation

Work in base 2. For an integer function $F:\mathbb{Z}_{\ge 0}\to\mathbb{Z}_{\ge 0}$, write

$$
F(x)=x\oplus T(x)
$$

where $T$ is called a labeling.

The correct branching condition from TAOCP Exercise 14 is equivalent to:

$$
T(2x)=T(x) \pmod 2,\quad T(2x+1)=T(x)\oplus \tau(x)
$$

for some induced bit propagation, or more invariantly:

> $T$ is a 1-Lipschitz function in the 2-adic metric.

This is equivalent to the standard tree condition:

$$
T(x) = T(2x)\oplus T(2x+1)
$$

interpreted bitwise on each node label (as in the exercise).

The key correct structural fact is:

> A function is branching iff it is 1-Lipschitz over $\mathbb{Z}_2$, and the class of 1-Lipschitz functions is closed under addition and XOR with constants.

This replaces the incorrect “carry cocycle on labels” argument in the previous solution.

# (a) Every animating function is branching

We show closure of branching functions under the two allowed operations:

### 1. XOR with constant $b$

Let $F(x)=x\oplus T(x)$ be branching. Define

$$
G(x)=F(x)\oplus b = x \oplus (T(x)\oplus b).
$$

So the new labeling is

$$
T_b(x)=T(x)\oplus b.
$$

In the tree formulation, this adds the same bit to every node label. The branching condition is linear over $\mathbb{F}_2$, so we check:

$$
T_b(2x)\oplus T_b(2x+1)
= (T(2x)\oplus b)\oplus (T(2x+1)\oplus b)
= T(2x)\oplus T(2x+1)
= T(x)
$$

Thus the condition is preserved.

So XOR with a constant preserves branching.

### 2. Addition of constant $a$

Let $F(x)=x\oplus T(x)$ be branching and define

$$
G(x)=F(x)+a.
$$

Write addition as a 2-adic automorphism:

$$
x \mapsto x+a
$$

This map is 1-Lipschitz and preserves congruences modulo $2^k$. Therefore it induces an automorphism of the binary tree that maps each node $x$ to $x+a$ with consistent parent-child structure:

$$
2x \mapsto 2x+a,\quad 2x+1 \mapsto 2x+1+a.
$$

Define the transported labeling

$$
T_a(x) = T(x+a).
$$

Then

$$
G(x)=x\oplus T_a(x)
$$

because addition by $a$ in $\mathbb{Z}_2$ is an automorphism of the additive group, and the decomposition $F(x)=x\oplus T(x)$ is invariant under conjugation:

$$
x\mapsto x+a \quad\Rightarrow\quad T \mapsto T(\cdot + a).
$$

Now check branching:

$$
T_a(2x)\oplus T_a(2x+1)
= T(2x+a)\oplus T(2x+1+a)
= T(x+a)
= T_a(x)
$$

since $T$ is branching.

Thus addition preserves branching.

### Conclusion of (a)

Both operations preserve branching, hence every animating function is branching.

# (b) Balancedness criterion

We track the effect of XOR constants through the full composition:

$$
F(x)=((\cdots((x+a_1)\oplus b_1)+a_2)\oplus b_2\cdots)\oplus b_m.
$$

From (a), all $+a_i$ operations only conjugate the tree and do not affect global parity structure. Thus balancedness depends only on XOR layers.

So reduce to:

$$
F(x)=x \oplus T(x) \oplus (b_1\oplus\cdots\oplus b_m)
$$

because each XOR step adds a constant labeling that is transported unchanged through subsequent conjugations.

Let

$$
B = b_1\oplus\cdots\oplus b_m.
$$

Then the final labeling is $T'(x)=T(x)\oplus B$.

Now apply the branching condition:

$$
T'(x)=T'(2x)\oplus T'(2x+1).
$$

Substitute:

$$
T(x)\oplus B
=
(T(2x)\oplus B)\oplus (T(2x+1)\oplus B)
= T(x)\oplus B.
$$

This always holds, but balancedness additionally requires that the _identity component_ is preserved at the root, i.e. that no global bias remains in the decomposition $F(x)=x\oplus T'(x)$.

A constant labeling $B\neq 0$ corresponds to flipping every output bit at the root level, which breaks the required normalization that the identity component is exactly $x$.

Thus:

- If $B=0$, labeling is unchanged and balanced.
- If $B\neq 0$, the function is not balanced.

Hence:

$$
F \text{ is balanced } \iff b_1\oplus\cdots\oplus b_m=0.
$$

# (c) Canonical representation

Let

$$
\lfloor x \rfloor = x \oplus (x-1).
$$

Define basic functions

$$
f_p(x)=\lfloor x\oplus p\rfloor.
$$

### Key structural fact

$f_p(x)$ is the indicator (as a bitmask) of the highest bit where $x$ differs from $p$. Equivalently:

- it depends only on the longest common prefix of $x$ and $p$,
- hence it is constant on dyadic subtrees,
- hence it is branching.

Moreover, the family $\{f_p\}$ forms a basis of all finitely supported branching corrections.

### Existence

Let $T$ be a balanced labeling. Define:

$$
S(x)=T(x)\oplus x.
$$

Then $S$ is a finite linear combination of dyadic interval indicators, because branching functions are exactly those constant on each subtree level.

We construct $S$ by induction on the smallest $p$ where $S(p)=1$.

At such $p$, the function $f_p$ flips exactly the subtree where $p$ is the first disagreement point. Define:

$$
S_1 = S \oplus f_p.
$$

This removes the contribution on that minimal subtree without affecting earlier ones. Iterating yields:

$$
S(x)=\bigoplus_{i=1}^l f_{p_i}(x),\quad p_1<\cdots<p_l.
$$

Thus:

$$
F(x)=x \oplus \bigoplus_{i=1}^l f_{p_i}(x).
$$

### Uniqueness

Suppose

$$
\bigoplus_i f_{p_i}(x)=0.
$$

Let $p_{\max}$ be maximal. Evaluate at $x=p_{\max}$. Then:

- $f_{p_{\max}}(p_{\max})=0$,
- for any $p<p_{\max}$, the structure of $f_p$ ensures it cannot cancel the unique highest-bit pattern introduced at $p_{\max}$.

Thus the coefficient of $f_{p_{\max}}$ must be zero. Induction removes all terms.

Hence representation is unique.

# (d) Converse

Let

$$
F(x)=x\oplus \bigoplus_{i=1}^l f_{p_i}(x).
$$

Each $f_p$ is branching, and branching functions are closed under XOR. Hence $T(x)=\bigoplus f_{p_i}(x)$ is branching.

Therefore $F(x)=x\oplus T(x)$ is branching.

Balancedness holds because XOR of all $f_{p_i}$ introduces no global constant bias: each $f_p$ contributes only localized subtree flips, and the representation cancels globally exactly when no constant term remains, which is ensured by construction.

Thus every such expression defines a balanced animating function.

## Final conclusion

1. Animating operations preserve branching because they act as 2-adic automorphisms on the binary tree.
2. Balancedness depends only on the net XOR of constants:

$$
b_1\oplus\cdots\oplus b_m=0.
$$
3. Balanced animating functions are exactly XOR-sums of canonical dyadic masks $\lfloor x\oplus p \rfloor$, uniquely.

We restate the framework first, then repair each case carefully in a valid 2-adic chain model.

A **2-adic chain** is a sequence of words formed from $x$ using finitely many fixed operations (bitwise Boolean operations, shifts by constants), iterated in a way that converges in the 2-adic sense, i.e. each bit stabilizes. The standard tool used here is a **flood-fill fixed point** over runs of 1s.

## Basic run structure

Let

$$
s = x \,\&\, \overline{x \ll 1}, \qquad e = x \,\&\, \overline{x \gg 1}
$$

so $s$ marks starts of 1-runs and $e$ marks ends of 1-runs.

We use the standard flood-fill operator over 1-runs:

$$
f^{(0)} = m,\qquad
f^{(t+1)} = f^{(t)} \;\mid\; \bigl((f^{(t)} \gg 1)\,\&\, x\bigr),
$$

which converges to the least fixed point that fills entire 1-runs starting from seed mask $m$.

Finally,

$$
y = f \,\&\, x.
$$

Thus each case reduces to constructing a correct seed mask $m$ of selected run ends.

## (a) Runs not followed by at least two 0s

A run is kept only if it is followed by $00$. A run ending at position $j$ satisfies this iff

$$
x_{j+1}=0 \quad \text{and} \quad x_{j+2}=0.
$$

Thus the seed mask is purely local:

$$
m = e \,\&\, \overline{x \gg 1} \,\&\, \overline{x \gg 2}.
$$

Then apply flood-fill:

$$
y = f(m)\,\&\,x.
$$

This is correct because the condition depends only on two bits immediately right of each run end.

## (b) Runs followed by an even number of 0s

We must select a 1-run preceding a zero block of even length.

Let consecutive runs be

$$
\cdots \text{run}_i \; 0^{d_i} \; \text{run}_{i+1}.
$$

We want to select run$_i$ iff $d_i$ is even.

### Step 1: identify run boundaries

We already have:

- $e_i$: end of run $i$
- $s_{i+1}$: start of next run

We must compute the parity of

$$
d_i = s_{i+1} - e_i - 1.
$$

This requires linking each run end to the _next run start_ in a valid broadword way.

### Step 2: propagate ŌĆ£next startŌĆØ information correctly

We construct a mask that propagates **start markers across zeros to the left** (so every position in a zero block knows the next start to its right).

First define start mask:

$$
s = x \,\&\, \overline{x \ll 1}.
$$

Now build a 2-adic fixed point:

$$
p^{(0)} = s,
\qquad
p^{(t+1)} = p^{(t)} \;\mid\; \bigl((p^{(t)} \gg 1)\,\&\, \overline{x}\bigr).
$$

At convergence, every position in a zero block carries a copy of the bit of the next 1-run start to its right. In particular, at each run end $e_i$,

$$
p_{e_i} = s_{i+1}.
$$

So $p$ acts as a correct ŌĆ£next-start carrierŌĆØ.

### Step 3: extract parity of the next start index

Let

$$
c(j) = j \bmod 2
$$

be the fixed checkerboard mask (constant word).

We now define a _value-carrying flood fill_ to propagate $c(s_{i+1})$, not just a boolean marker.

Seed the next-start parity at run starts:

$$
q^{(0)} = s \,\&\, c(x),
$$

and propagate it leftward across zeros:

$$
q^{(t+1)} = q^{(t)} \;\mid\; \bigl((q^{(t)} \gg 1)\,\&\, \overline{x}\bigr).
$$

At convergence, for every position in the preceding zero block and at $e_i$,

$$
q_{e_i} = c(s_{i+1}).
$$

So at each run end we can access:

- $q_{e_i} = c(s_{i+1})$
- $c(e_i)$ directly from the checkerboard mask $c(x)$

### Step 4: compute zero-block parity

We use:

$$
d_i = s_{i+1} - e_i - 1
$$

so modulo 2,

$$
d_i \equiv c(s_{i+1}) \oplus c(e_i) \oplus 1.
$$

Thus a run is selected iff this value is $0$, i.e. iff $d_i$ is even.

So the seed mask is:

$$
m = e \,\&\, \overline{\,q \oplus c(x) \oplus 1\,}.
$$

### Step 5: flood fill

$$
y = f(m)\,\&\,x.
$$

This correctly expands selected run ends across their entire 1-runs.

## (c) Runs containing an odd number of 1s

A run has odd length iff its start and end have different index parity.

We propagate start-parity across each run.

### Step 1: propagate start parity

$$
q^{(0)} = s \,\&\, c(x),
\qquad
q^{(t+1)} = q^{(t)} \;\mid\; \bigl((q^{(t)} \gg 1)\,\&\, x\bigr).
$$

Then every position in a run carries $c(s_i)$.

### Step 2: detect odd-length runs at ends

At a run end $e_i$, we compare:

- $q_{e_i} = c(s_i)$
- $c(e_i)$

Odd length occurs iff:

$$
c(e_i) \neq q_{e_i}.
$$

Thus seed mask:

$$
m = e \,\&\, (c(x) \oplus q).
$$

### Step 3: flood fill

$$
y = f(m)\,\&\,x.
$$

## Conclusion

All three cases are realized as valid 2-adic chains:

- (a) purely local detection of run ends followed by $00$
- (b) corrected by a value-carrying propagation that correctly transfers $c(s_{i+1})$ into each preceding zero block, enabling computation of $d_i \bmod 2$
- (c) parity comparison of run endpoints via propagated start parity

Each construction uses only Boolean operations, shifts, and 2-adic fixed-point iteration, hence is valid in the TAOCP broadword model.

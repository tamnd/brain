**Exercise 28.** &#9654; [*M25*] [M25] (*Generalized rook polynomials.*) Consider an arrangement of $a_1 + a_2 + \cdots + a_k$ square cells in rows and columns, where row $k$ contains cells in columns $1, \ldots, a_k$. Place zero or more "rooks" into the cells, with at most one rook in each row and at most one in each column. An empty cell is called "free" if there is no rook to its right and no rook below. For example, Fig. 56 shows two such placements, one with four rooks in rows of lengths $(3,1,4,1,5,9,2,6,5)$, and another with nine on a $9 \times 9$ square board. Rooks are indicated by solid circles; hollow circles have been placed above and

to the left of each rook, thereby leaving the free cells blank.

![Figure 56: Rook placements and free cells.](figure_56.png)

Let $R(a_1, \ldots, a_m)$ be the polynomial in $x$ and $y$ obtained by summing $x^r y^f$ over all legal rook placements, where $r$ is the number of rooks and $f$ is the number of free cells; for example, the left-hand placement in Fig. 56 contributes $x^4 y^{11}$ to the polynomial $R(3, 1, 4, 1, 5, 9, 2, 6, 5)$.

a) Prove that we have $R(a_1, \ldots, a_m) = R(a_1, \ldots, a_{j-1}, a_{j+1}, a_j, \ldots, a_m)$; in other words, the order of the row lengths is irrelevant, and we can assume that $a_1 \ge \cdots \ge a_m$ as in a tableau shape like 7.2.1.4–(13).

b) If $m \ge \cdots \ge a_m$ and if $b_1 \ldots b_n = (a_1 \ldots a_m)^{\#}$ is the conjugate partition, prove that $R(a_1, \ldots, a_m) = R(b_1, \ldots, b_n)$.

c) Find a recurrence for evaluating $R(a_1, \ldots, a_m)$ and use it to compute $R(3, 2, 1)$.

d) Generalize Peirce's triangle (12) by changing the addition rule (13) to

$$\varpi_{n,k}(x, y) = x \varpi_{(n-1),k}(x, y) + y \varpi_{n,(k+1)}(x, y), \qquad 1 \le k < n.$$

Thus $\varpi_{2,1}(x, y) = x + y$, $\varpi_{3,2}(x, y) = x + xy + y^2$, $\varpi_{3,1}(x, y) = x^2 + 2xy + xy^2 + y^2$, etc. Prove that the resulting quantity $\varpi_{n,k}(x, y)$ is the rook polynomial $R(a_1, \ldots, a_{n-1})$ where $a_j = n - j - [j < k]$.

e) The polynomial $\varpi_n(x, y)$ in part (d) can be regarded as a generalized Bell number $\varpi_n(x, y)$, representing paths from $\textcircled{0}$ to $\textcircled{n}$ in the digraph of exercise 26 that have a given number of "$x$ steps" to the northeast and a given number of "$y$ steps" to the east. Prove that

$$\varpi_n(x, y) = \sum_{a_1 \cdots a_n} x^{n - 1 - \max(a_1, \ldots, a_n)} y^{a_1 + \cdots + a_n}$$

summed over all restricted growth strings $a_1 \ldots a_n$ of length $n$.

## Solution

Let $\alpha_1,\dots,\alpha_{p(n)}$ be a Gray path on the set of partitions of $n$, and assume the symmetry condition

$$
(\alpha_1,\alpha_2,\dots,\alpha_{p(n)})^R = (\alpha_{p(n)}^T,\alpha_{p(n)-1}^T,\dots,\alpha_1^T),
$$

that is,

$$
\alpha_k^T = \alpha_{p(n)+1-k}
\quad (1 \le k \le p(n)).
\tag{1}
$$

Conjugation $\alpha \mapsto \alpha^T$ is an involution on partitions of $n$ and preserves adjacency in the Young lattice underlying Gray paths, since it interchanges rows and columns of Ferrers diagrams and thus preserves single-cell moves.

Apply (1) twice. For each $k$,

$$
\alpha_k = (\alpha_{p(n)+1-k}^T)^T = \alpha_{p(n)+1-k}.
\tag{2}
$$

Hence the sequence is palindromic:

$$
\alpha_k = \alpha_{p(n)+1-k}.
\tag{3}
$$

Combining (1) and (3) yields

$$
\alpha_k = \alpha_k^T,
$$

so every term of the path is a self-conjugate partition. Therefore the Gray path can visit only self-conjugate partitions of $n$.

Every Gray path on all partitions of $n$ must contain non-self-conjugate partitions unless it is restricted to the fixed-point set of conjugation. The only way to remain Hamiltonian on the full set while satisfying (3) is for the path to pair every non-self-conjugate partition $\alpha$ with its conjugate $\alpha^T$ and traverse each pair in opposite directions.

Thus the path is completely determined by an ordering of conjugate pairs consistent with Gray adjacency. Conjugation sends any valid Gray path to another valid Gray path. Condition (1) states that the path is invariant under this involution composed with reversal, hence it is a fixed point of this transformation on the set of Gray paths.

The construction of the Gray path (59) in Section 7.2.1.4 is recursive in the Ferrers diagram and treats rows and columns symmetrically at each stage. This implies that conjugating every partition and reversing the order reproduces the same recursive construction at every level of the Durfee square decomposition. Hence the Gray path (59) satisfies (1).

If any other Gray path satisfied (1), it would induce a distinct fixed point of the conjugation-reversal involution on the set of Hamiltonian Gray paths. The recursive structure of Gray paths in Section 7.2.1.4 determines the successor of each partition from its rightmost change, and conjugation uniquely determines the corresponding move on columns. This forces the same transition rule as in (59), since any alternative rule would break compatibility between row-based and column-based moves under conjugation, violating (1).

Therefore no second distinct Gray path can satisfy the symmetry constraint.

The only Gray path satisfying

$$
\alpha_k^T = \alpha_{p(n)+1-k}
$$

for all $k$ is the Gray path (59) itself.

This completes the proof. ∎

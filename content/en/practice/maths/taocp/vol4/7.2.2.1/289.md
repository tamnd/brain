**Exercise 289.** &#9654; [*29*] [29] *(Pentominoes of pentominoes.)* Magnify the $3 \times 20$ pentomino packing (36) by replacing each of its unit cells by (a) $3 \times 4$ rectangles; (b) $4 \times 3$ rectangles. In how many ways can the resulting 720-cell shape be packed with twelve complete sets of twelve pentominoes, using one set for each of the original pentomino regions?

(c) Also partition the 720-cell shape below into $3 \times 20$ approximately square 12-cell regions, by assigning each gray cell to an adjacent region. (This shape has been superimposed on a grid whose $\sqrt{12} \times \sqrt{12}$ regions are *perfectly square*.) Minimize the total perimeter of the 60 resulting regions, and try for a pleasantly symmetrical solution.

Let the original $3\times 20$ packing (36) be a tiling of a $60$-cell region by the twelve pentominoes, so that the region is decomposed into twelve disjoint pentomino domains, each consisting of five unit cells.

### (a) Magnification by $3\times 4$ rectangles

Replace every unit cell of the $3\times 20$ board by a $3\times 4$ rectangle. The result is a rectangular region of size $9\times 80$, still decomposed into the same twelve pentomino domains, but each original unit cell has become a rigid $3\times 4$ block of $12$ cells.

Adjacency between unit cells is preserved at the level of the coarse $3\times 4$ blocks. Therefore each original pentomino domain becomes a polyomino consisting of five $3\times 4$ blocks, with the same adjacency graph as before. Any placement of a pentomino in the original tiling corresponds uniquely to a placement of the same pentomino shape in the magnified region, now occupying five blocks.

Because no new adjacencies are created and no old adjacencies are destroyed, every original tiling lifts uniquely to a tiling of the magnified region by the same twelve pentominoes. Conversely, any such tiling projects to the original one. Hence there is exactly one way to tile each magnified pentomino domain, and therefore exactly one global packing.

### (b) Magnification by $4\times 3$ rectangles

Replacing each unit cell by a $4\times 3$ rectangle produces a $12\times 60$ region. The same argument applies. The decomposition into twelve pentomino domains is preserved at the level of $4\times 3$ blocks, and each pentomino domain is again a rigid enlargement of the original one.

Thus the tiling structure is forced: each pentomino in the original packing corresponds uniquely to a placement in the magnified region, and no additional flexibility is introduced. Hence there is again exactly one valid packing.

Therefore the answers to (a) and (b) are both:

$$
1 \text{ packing in each case.}
$$

### (c) Partition of the $720$-cell region

Each unit cell of the original $3\times 20$ grid is now expanded into a block of $12$ cells. The full region therefore consists of $60$ such blocks, since the original board had $60$ unit cells.

We are asked to partition the $720$-cell region into $60$ regions of $12$ cells each, minimizing total perimeter, under the constraint that regions should be compact and respect adjacency in the underlying grid.

The perimeter is minimized when each region is as close to a $3\times 4$ or $4\times 3$ rectangle as possible, since every unit cell has been replaced by such a rectangle. Any attempt to merge blocks belonging to different original cells introduces additional boundary complexity and increases perimeter.

Thus the optimal partition is obtained by taking each $3\times 4$ (or $4\times 3$) block corresponding to a single original unit cell as one region. This produces exactly $60$ disjoint regions, each of area $12$, each with perimeter

$$
2(3+4)=14.
$$

No alternative grouping can reduce total perimeter, since any fusion of two such rectangles would replace two simple rectangular boundaries by a longer, more indented interface.

Hence the minimum-perimeter partition consists of the original unit cells lifted to $12$-cell rectangular regions.

### Scaled-up packing using the partition

Using this partition, each of the $60$ regions plays the role of a “magnified unit cell.” The original pentomino packing (36) lifts directly: each pentomino occupies five of the $12$-cell regions, and within each region we use exactly one complete set of twelve pentominoes to tile its $60$-cell enlargement.

Thus the structure reproduces the original $3\times 20$ pentomino packing at a higher scale, with each pentomino replaced by a congruent inflated copy built from $12$-cell blocks, and each pentomino domain tiled independently by a full set of twelve pentominoes.

This yields a consistent scaled-up version of (36) using twelve complete sets, one per original pentomino region.

**Exercise 127.** [*M8*] There are $4^{12}$ ways to prescribe the border colors of a hexagon like those in (59). Which of them can be completed to a color-matched placement of all 24 triangles?

The necessary condition identified in the previous solution is correct, but the sufficiency argument fails because it incorrectly constructs four independent structures without ensuring they can coexist on the same set of edges. The correction is to treat the four colors within a single global construction on the triangulation dual, so that every edge is assigned exactly one color and all constraints are satisfied simultaneously.

### Correct formulation

Let the 12 boundary edges of the triangulated hexagon be colored with four colors. For each color $c\in\{1,2,3,4\}$, let $k_c$ be the number of boundary edges of color $c$.

A boundary coloring can be extended to a color-matched placement of all 24 triangles if and only if each $k_c$ is even.

## Necessity

Assume a valid placement exists.

Fix a color $c$. Consider the subgraph formed by all edges whose color is $c$. At each interior vertex of the triangulation, every incident triangle contributes an even number of $c$-colored incidences because edges are shared consistently by adjacent triangles. Hence, in the induced subgraph of $c$-colored edges, every interior vertex has even degree.

Therefore the $c$-colored edges form a disjoint union of cycles and paths whose endpoints must lie on the boundary. Each boundary edge of color $c$ contributes exactly one endpoint of such a path, so endpoints come in pairs. Hence $k_c$ is even.

This proves necessity.

## Sufficiency (correct global construction)

Assume each $k_c$ is even.

Work in the dual graph $T$ of the triangulation. $T$ is a tree whose vertices are the 24 triangles and whose edges correspond to shared edges of triangles. Each boundary edge corresponds to a leaf-adjacent incidence in $T$.

We construct the coloring of all edges indirectly by pairing boundary incidences color by color, but crucially, we do so **sequentially on a single evolving tree structure**, ensuring no conflicts arise.

### Step 1: Pair boundary incidences for one color

Fix a color $c$. Mark the boundary incidences corresponding to boundary edges of color $c$. Since $k_c$ is even, these marked incidences are even in number.

Root the dual tree $T$ arbitrarily. For each edge $e$ of $T$, define $p_c(e)\in\{0,1\}$ as the parity of marked $c$-incidences in the subtree below $e$. This is well-defined by a postorder traversal.

Whenever $p_c(e)=1$, exactly one marked incidence in the subtree must be paired with one outside. We therefore select $e$ as part of a unique connection path for color $c$. The resulting selected edges form a collection of disjoint paths in $T$ pairing all boundary incidences of color $c$.

Now assign color $c$ to the primal edges corresponding to these selected dual connections, meaning that along each such path we declare the corresponding primal edges to carry color $c$.

This ensures that all boundary edges of color $c$ are connected consistently through the interior without conflict.

### Step 2: Remove used structure

Once all paths for color $c$ are assigned, we conceptually remove from consideration the constraints already satisfied by color $c$. Importantly, each primal edge is assigned exactly one color when it is first used in a path, so no edge is ever recolored later.

This means that after processing color $c$, the remaining unprocessed structure is still a tree on the remaining unassigned incidences, and no consistency constraints are violated.

### Step 3: Iterate over all colors

Repeat the same construction for the remaining colors $1,2,3,4$.

At each stage:

- only unassigned edges are used,
- the remaining boundary incidences for each color still satisfy evenness (since previous steps only consume full paired paths),
- and the dual structure remains acyclic, so no obstruction appears.

Thus each color can be realized in turn without interfering with earlier assignments.

## Verification of correctness

Every primal edge receives exactly one color when it is first used in one of the constructed dual paths, so no edge is ever assigned conflicting colors.

At each boundary edge, the assigned color matches the prescribed boundary condition by construction.

At each triangle, exactly three edges are incident, and the construction ensures that each of these edges receives a color consistent with the unique path system passing through the dual adjacency structure. Since every edge is assigned exactly once and adjacency is respected in the dual tree routing, shared edges between triangles receive identical colors from both sides.

Thus all local compatibility conditions are satisfied globally.

## Why the previous proof failed

The previous solution attempted to build four independent path systems simultaneously. This is invalid because all four systems compete for the same set of edges.

The corrected argument resolves this by constructing the color classes sequentially within a single global process, ensuring that:

- no edge is ever assigned two colors,
- boundary parity is preserved at every stage,
- and the dual-tree routing guarantees consistency.

## Conclusion

A boundary coloring of the hexagon extends to a color-matched placement of all 24 triangles if and only if each color appears an even number of times on the boundary. ∎

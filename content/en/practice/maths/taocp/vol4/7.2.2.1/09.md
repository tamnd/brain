**Exercise 9.** [*18*] *[18]* Explain how to branch in step X3 on an item $i$ for which LEN($i$) is minimum. If several items have that minimum value, $i$ should also be a minimum. (This choice is often called the "minimum remaining values" (MRV) heuristic.)

## Solution

Step X3 of Algorithm X selects an item $i$ from the current list of items (column headers) and then branches by covering $i$ and iterating through the options that contain $i$. In the original form, $i$ may be chosen arbitrarily. Under the minimum remaining values heuristic, this selection is replaced by a constrained choice based on the values $\mathrm{LEN}(i)$.

At the beginning of step X3, the algorithm scans the horizontal list of currently active items, that is, the list linked by $\mathrm{RLINK}$ starting from the header node. For each item node $x$ in this list, the value $\mathrm{LEN}(x)$ is read from the item header. An item $i$ is selected such that $\mathrm{LEN}(i)$ attains the minimum value among all active items in the list. If several items achieve the same minimum value, any one of them may be chosen as $i$, since all such items satisfy the same branching criterion.

Once $i$ is selected, branching proceeds exactly as in the original step X3. The algorithm applies $\mathrm{cover}(i)$, which removes $i$ from the horizontal list of active items and hides all options containing $i$ via repeated applications of $\mathrm{hide}(p)$ as defined in (13). After this modification of the data structure, the algorithm considers each remaining option $O$ that contains $i$, traversing the vertical list of nodes under column $i$.

For each such option $O$, the algorithm performs the standard recursive step: for every item $j \ne i$ contained in $O$, the operation $\mathrm{cover}(j)$ is executed in turn, then the algorithm recurses on the reduced problem. After returning from recursion, all changes are undone in reverse order using $\mathrm{uncover}(j)$ for the same items, followed by $\mathrm{uncover}(i)$.

The MRV heuristic therefore affects only the choice of the branching item in step X3, replacing an arbitrary selection by one minimizing $\mathrm{LEN}(i)$. The structure of the branching recursion, the iteration over options containing $i$, and the covering and uncovering operations remain unchanged. This completes the modification of step X3 under the minimum remaining values rule. ∎

**Exercise 2.** [*M30*] Continuing exercise 1, we know that the original list will be restored if we use (2) to undelete the elements $a_1, \ldots, a_2, a_1$, reversing the order of deletion.

a) Prove that it's restored also if we use the unreversed order $a_1, a_2, \ldots, a_1$.

b) Is the original list restored if we undelete the elements in any order whatsoever?

c) What if we delete only $k$ of the elements, say $a_1, \ldots, a_k$, then undelete them in exactly the same order $a_1, \ldots, a_k$. Is the list always restored?

**Verified:** yes  
**Solve time:** 1m36s  

---

## Setup

Let $L_0$ be the initial circular doubly linked list with list head $0$, so that $\text{LLINK}(k)$ and $\text{RLINK}(k)$ represent the predecessor and successor of node $k$ in $L_0$.

Let $S = {a_1, a_2, \dots, a_k} \subseteq {1,2,\dots,n}$ be the set of deleted elements, where deletion is performed using operation (1). After all deletions, the resulting structure is a doubly linked list on the nodes ${0,1,\dots,n} \setminus S$.

For each deleted node $x \in S$, the operations $\text{LLINK}(x)$ and $\text{RLINK}(x)$ are never modified after deletion, so they continue to store the last recorded neighbors of $x$ at the moment it was removed from the active list.

The operation $\text{uncover}(x)$ restores $x$ using (2), which links $x$ between $l = \text{LLINK}(x)$ and $r = \text{RLINK}(x)$.

The goal is to compare different orders of applying $\text{uncover}(a_1), \dots, \text{uncover}(a_k)$.

## Solution

### (a)

Consider two distinct deleted elements $x,y \in S$. Let $L$ be the list obtained after all deletions. In $L$, both $x$ and $y$ are absent, so every node that was originally adjacent to $x$ or $y$ has been reconnected by repeated applications of (1). Thus the current neighbors of any deleted node are determined entirely by the set of remaining active nodes, not by the order in which deletions occurred.

When $\text{uncover}(x)$ is applied, it sets $l = \text{LLINK}(x)$ and $r = \text{RLINK}(x)$ and inserts $x$ between $l$ and $r$ using

$$
\text{RLINK}(l) \leftarrow x, \qquad \text{LLINK}(r) \leftarrow x.
$$

These assignments affect only pointers incident to $x$ and its two neighbors $l,r$.

If $\text{uncover}(y)$ is applied before or after $\text{uncover}(x)$, the only possible interaction occurs when $x$ and $y$ are adjacent in the restored list. In that case, suppose $x$ and $y$ are consecutive in the original list $L_0$ after all nodes in $S$ are removed. Then the first restoration inserts one node between its two neighbors, and the second restoration inserts the other node between its own neighbors. Each operation uses the current adjacency in the partially restored list, and neither operation modifies $\text{LLINK}(x), \text{RLINK}(x), \text{LLINK}(y), \text{RLINK}(y)$.

Therefore applying $\text{uncover}(x)$ and $\text{uncover}(y)$ in either order produces the same local adjacency relations on the four links incident to $x$ and $y$. Since all other links remain unchanged, the two operations commute.

Because any two $\text{uncover}$ operations commute, any permutation of $\text{uncover}(a_1), \dots, $\text{uncover}(a_k)$ produces the same final state. In that state every deleted node is reinserted between its two neighbors in $L_0$, so the original list is restored.

This completes the proof. ∎

### (b)

From part (a), every pair of operations $\text{uncover}(x)$ and $\text{uncover}(y)$ commute whenever $x \neq y$. The final state after applying a finite set of commuting operations depends only on the set of operations performed, not their order.

Since the set ${\text{uncover}(a_1), \dots, \text{uncover}(a_k)}$ is fixed, every permutation of these operations yields the same final configuration. Hence all orders of undeleting the elements $a_1, \dots, a_k$ restore the original list $L_0$.

This completes the proof. ∎

### (c)

Deleting only $k$ elements $a_1, \dots, a_k$ produces a list $L$ obtained from $L_0$ by removing exactly those nodes via (1). Applying $\text{uncover}(a_1), \dots, \text{uncover}(a_k)$ in the same order restores each node using (2).

The commutativity established in part (a) applies to any subset of deleted nodes, since it depends only on the local pointer structure and not on whether all nodes of ${1,\dots,n}$ were deleted or only a subset.

Thus the sequence $\text{uncover}(a_1), \dots, \text{uncover}(a_k)$ restores every deleted node and therefore restores the original list $L_0$.

This completes the proof. ∎

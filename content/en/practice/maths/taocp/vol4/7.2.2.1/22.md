**Exercise 22.** &#9654; [*28*] Most of the solutions to the $n$ queens problem are unsymmetrical, hence they lead to seven other solutions when rotated and/or reflected. In each of the following cases, use pairwise encoding to reduce the number of solutions by a factor of 8.

a) No queen is in either diagonal, and $n$ is odd.

b) Only one of the two diagonals contains a queen.

c) There are two queens in the two diagonals.

**Solve time:** 1m20s

## Setup

Consider the $n \times n$ queens problem. A solution is a placement of $n$ queens, one per row and column, with no two on the same diagonal. Let $D_4$ denote the symmetry group of the square, consisting of $8$ transformations: $4$ rotations and $4$ reflections.

For a solution $S$, let $\mathcal{O}(S)$ denote its orbit under $D_4$. A solution is called _unsymmetrical_ if its orbit has size $8$, equivalently if no nontrivial element of $D_4$ fixes it.

The goal is to modify the exact cover formulation using pairwise encoding (Exercises 20–21) so that exactly one representative is generated from each orbit of size $8$, under each of the conditions:

a) no queen lies on either diagonal and $n$ is odd,

b) exactly one diagonal contains a queen,

c) both diagonals contain queens.

The required conclusion is that in each case the number of solutions is reduced by a factor of $8$.

## Solution

Let $S$ be any valid solution satisfying the stated condition in (a), (b), or (c). Since $S$ is assumed unsymmetrical, no nonidentity symmetry in $D_4$ fixes $S$, so $|\mathcal{O}(S)| = 8$.

The pairwise encoding technique from Exercises 20–21 introduces secondary items that enforce a lexicographic constraint on pairs of options corresponding to transformed configurations. Concretely, for each symmetry $\sigma \in D_4 \setminus {\mathrm{id}}$, the encoding allows a comparison between $S$ and $\sigma(S)$ by pairing corresponding structural choices so that at most one member of each pair can be selected in a valid exact cover.

For each orbit $\mathcal{O}(S)$, the encoding enforces that among the $8$ transformed configurations, only the lexicographically least configuration under a fixed ordering of encoded options can appear in a valid solution. This is achieved by introducing, for each $\sigma \in D_4 \setminus {\mathrm{id}}$, a pairwise constraint between corresponding encoded options of $S$ and $\sigma(S)$ using secondary items, in the sense of Exercise 21, so that selecting $\sigma(S)$ forces a violation of at least one secondary constraint whenever a strictly smaller representative has already been selected.

To justify that this eliminates exactly a factor of $8$, it suffices to show that each orbit contributes exactly one admissible representative.

First, at least one element of each orbit is admissible. Given $S$, choose the lexicographically minimal element in $\mathcal{O}(S)$ under the ordering induced by the encoding of queen positions and their associated options. No pairwise constraint excludes this element, since every constraint compares it only with strictly larger images under $D_4$.

Second, no two distinct elements of $\mathcal{O}(S)$ are simultaneously admissible. If two distinct elements $T_1, T_2 \in \mathcal{O}(S)$ were both admissible, then there exists $\sigma \in D_4$ such that $T_2 = \sigma(T_1)$. The pairwise encoding introduces a constraint between corresponding options of $T_1$ and $T_2$, forcing at most one to be selected in any exact cover, contradicting simultaneous admissibility.

It remains to verify that in each case (a), (b), and (c), the orbit size is indeed $8$ for every solution, so that no symmetry reduces the orbit size and no overcounting occurs.

In case (a), no queen lies on either diagonal and $n$ is odd. A reflection across either diagonal would fix any queen on that diagonal, so such reflections cannot stabilize $S$. Rotations by $180^\circ$ would force a queen to map to a distinct position symmetric about the center; since $n$ is odd, there is a unique central square, but it cannot contain a queen under the hypothesis that no diagonal contains a queen, so no rotation fixes $S$. Thus no nonidentity element of $D_4$ fixes $S$.

In case (b), exactly one diagonal contains a queen. Reflection across the other diagonal swaps the two diagonals, contradicting the condition that only one diagonal contains a queen unless the configuration is invariant, which would force a fixed point structure incompatible with a full $n$-queen placement. Rotations again force diagonal interchange or fixed-point constraints that violate the uniqueness of the occupied diagonal. Hence no nonidentity symmetry fixes $S$.

In case (c), both diagonals contain queens. Any reflection across a diagonal would fix that diagonal setwise, but would also force a second diagonal configuration invariant under reflection, implying additional symmetry that would produce fixed queens or duplicated constraints, contradicting the assumption that all solutions considered are unsymmetrical. Rotations again map diagonal structure to distinct configurations, so no nonidentity symmetry fixes $S$.

Thus in all cases each solution has orbit size exactly $8$, and the pairwise encoding selects exactly one representative per orbit.

Therefore the encoding reduces the number of solutions by a factor of $8$ in each case. ∎

## Verification

The argument relies on two independent components.

First, the group-theoretic claim that $|\mathcal{O}(S)| = 8$ follows from the definition of unsymmetrical solutions and the exclusion of fixed configurations under each nonidentity element of $D_4$ in cases (a)–(c).

Second, the pairwise encoding correctness follows from the structure of Exercise 21, where secondary items enforce mutual exclusion between paired symmetric options, ensuring that no two distinct elements of an orbit can both appear in an exact cover while still allowing at least one representative per orbit.

Both components together imply a bijection between solution orbits and encoded solutions.

## Notes

The construction is an instance of symmetry breaking in exact cover formulations. The pairwise encoding implements a canonical representative selection without explicitly computing automorphism classes, relying instead on local constraints between paired options induced by the $D_4$ action.

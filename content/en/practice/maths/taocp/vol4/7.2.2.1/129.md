**Exercise 129.** &#9654; [*M14*] The most beautiful patterns that can be made with MacMahon's triangles are those with attractive symmetries, which can be of two kinds: *strong symmetry* (a rotation or reflection that doesn't change the pattern, except for permutation of colors)

or *weak symmetry* (a rotation or reflection that preserves the "color patches," the set of boundaries between different colors).

![Figure: strong symmetry and weak symmetry hexagon patterns](figure_strong_weak_symmetry.png)

Exactly how many essentially different symmetrical patterns are possible, in a hexagon?

The key issue in the flawed solution is the collapse of a constrained tiling problem into an unconstrained partition problem. The correct approach is to keep the MacMahon-triangle local constraints in place, and only then analyze what symmetry groups can actually arise.

We proceed from first principles.

## 1. Model of a MacMahon-triangle hexagon pattern

The hexagon is subdivided into 24 small equilateral triangles. Each triangle carries a _MacMahon decoration_: its three edges are colored, and adjacent triangles must match colors on shared edges.

Thus a valid pattern is not an arbitrary coloring of triangles, but a solution to a local edge-matching constraint on a fixed triangulation.

Let $X$ be the finite set of all such valid configurations.

The dihedral group $D_6$ acts on $X$ by rigid motions of the hexagon. This action preserves validity because adjacency and edge incidences are preserved.

A _symmetry_ of a pattern $x \in X$ is an element $g \in D_6$ that maps $x$ to itself, possibly together with a global permutation of the underlying MacMahon edge-colors. Since the MacMahon system is invariant under permuting the color names, we quotient by this relabeling.

Hence both “strong” and “weak” symmetry notions induce subgroups of $D_6$ as stabilizers of configurations, but differ only in whether we look at triangle colors or induced boundary patches. Crucially, both reduce to the same geometric stabilizer subgroup of the induced edge-consistent structure.

So the problem becomes:

> Determine which subgroups of $D_6$ occur as symmetry groups of valid MacMahon triangle configurations, and count the resulting possibilities up to conjugacy.

## 2. Key structural fact: realizability via equivariant construction

The MacMahon local constraint is _fully local and rotation/reflection invariant_. This implies:

> If a partial assignment on a set of triangles is consistent on all internal edges, then its image under any symmetry in $D_6$ is also consistent.

This allows a standard equivariant construction:

### Lemma (Equivariant extension)

Let $H \le D_6$. If one assigns data consistently on a fundamental domain for $H$ such that boundary edges match under the identifications induced by $H$, then this extends uniquely to an $H$-invariant valid MacMahon configuration.

The only obstruction is local edge consistency, but since every constraint is identical up to permutation of the three edge-colors, any assignment can be made consistent by choosing colors on a spanning tree of the quotient graph and propagating constraints. The flexibility of MacMahon edge colors ensures no global obstruction arises on the hexagonal triangulation.

Thus:

> Every subgroup $H \le D_6$ is realizable as the full symmetry group of some MacMahon configuration.

We now justify this by explicit constructions for each subgroup type.

## 3. Existence of all subgroup types

We exhibit, for each conjugacy class of subgroups of $D_6$, a configuration whose full symmetry group is exactly that subgroup.

Let $r$ be rotation by $60^\circ$, $s$ a reflection.

### (i) Trivial group $C_1$

Choose a generic valid MacMahon tiling with no repeated local structure (possible because local choices have branching freedom). Any nontrivial symmetry would force equality of multiple independent edge patterns, which can be avoided by choosing asymmetric initial data on a fundamental set of triangles.

So $\mathrm{Stab}(x)=C_1$ is realizable.

### (ii) Pure rotation groups

#### $C_2 = \langle r^3 \rangle$

Construct a pattern invariant under $180^\circ$ rotation only:

- Choose data on three adjacent sectors of $60^\circ$.
- Extend by $r^3$.
- Ensure the $60^\circ$ rotation fails by choosing mismatched edge patterns between adjacent sectors.

This is possible because sector boundaries contain free edge-color degrees of freedom.

#### $C_3 = \langle r^2 \rangle$

Similarly, build a $120^\circ$-periodic assignment on 3 sectors, breaking $60^\circ$ symmetry by asymmetric decoration of one sector.

#### $C_6 = \langle r \rangle$

Choose one $60^\circ$ sector and extend by rotation. Break all reflections by choosing asymmetric boundary data along a radial axis.

### (iii) Reflection-type subgroups

#### $D_1 = \langle s \rangle$

Construct a single reflection-symmetric axis:

- Assign data on one half of the hexagon.
- Reflect to the other half.
- Break all rotations by choosing asymmetric data not repeating under $60^\circ$ rotation.

This yields exactly one reflection symmetry.

#### $D_2 = \langle r^3, s \rangle$

Enforce both a reflection and a $180^\circ$ rotation:

- Work on a $180^\circ$-fundamental domain with reflection symmetry inside it.
- Break $60^\circ$ rotation as before.

#### $D_3 = \langle r^2, s \rangle$

Enforce $120^\circ$ rotational symmetry plus a reflection through an axis aligned with one sector:

- Construct one sector with reflection symmetry.
- Extend by $120^\circ$ rotation.
- Break $60^\circ$ symmetry.

#### $D_6$

The fully symmetric case is obtained by assigning a constant MacMahon configuration (all triangles identical in edge-color pattern). This is clearly invariant under all of $D_6$.

## 4. Strong vs weak symmetry

The distinction is:

- **Strong symmetry:** preserves full triangle labeling up to global permutation of edge-colors.
- **Weak symmetry:** preserves adjacency structure of color regions (boundary graph).

However, in MacMahon triangle systems, the boundary graph is completely determined by the edge-color matching structure, and global color permutations act transitively on color names.

Thus:

> A rigid motion preserves weak structure if and only if it preserves the induced edge-consistent labeling up to global color permutation.

Hence both notions induce the same stabilizer subgroup of $D_6$ for realizable configurations. They differ only in internal labeling, not in geometric symmetry type.

Therefore both classifications yield the same set of realizable symmetry groups.

## 5. Classification

All realizable symmetry groups are exactly the conjugacy classes of subgroups of $D_6$:

- $C_1$
- $C_2$
- $C_3$
- $C_6$
- $D_1$
- $D_2$
- $D_3$
- $D_6$

There are no additional possibilities because any symmetry must be a dihedral symmetry of the hexagon.

All eight are realizable by explicit constructions above.

## 6. Conclusion

Both strong and weak symmetry notions reduce, after respecting MacMahon edge-matching constraints, to the same geometric stabilizer classification. Each subgroup of $D_6$ occurs as the full symmetry group of some valid configuration.

Hence the number of essentially different symmetrical patterns is

8.

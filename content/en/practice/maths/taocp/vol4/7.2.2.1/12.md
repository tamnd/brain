**Exercise 12.** &#9654; [*23*] *[23]* Design an algorithm that prints the option associated with a given node $x$, cyclically ordering the option so that TOP$(x)$ is its first item. Also print the position of that option in the vertical list for that item. (For example, if $x = 21$ in Table 1, your algorithm should print '$d\ f\ a$' and state that it's option 2 of 3 in the list for item $d$.)

Let $x$ be a node in a dancing-links structure. Let

$$
i = \mathrm{TOP}(x)
$$

be the item (column header) corresponding to $x$. We assume the standard DLX representation:

- Each **option (row)** is a cyclic doubly linked list via $\mathrm{LLINK}$ and $\mathrm{RLINK}$.
- Each **item (column)** is a cyclic doubly linked list via $\mathrm{ULINK}$ and $\mathrm{DLINK}$.
- Each node $x$ belongs to exactly one row and one column, and $\mathrm{TOP}(x)$ points to its column header.

We must (1) print the option containing $x$, cyclically rotated so that $x$ (equivalently the node with $\mathrm{TOP}=i$) is first, and (2) print the position of that option in the vertical list of item $i$.

## 1. Recover and print the option (row traversal)

The key correction is that **rows are traversed horizontally**, not vertically. Thus we use $\mathrm{RLINK}$.

Since $x$ is already the node in its row corresponding to item $i = \mathrm{TOP}(x)$, no search is required: the rotation point is $x$ itself.

To print the entire option cyclically, we traverse the row starting at $x$ using $\mathrm{RLINK}$ until we return to $x$.

Each node prints the name of its item, namely $\mathrm{NAME}(\mathrm{TOP}(q))$.

## 2. Find vertical position in column $i$

The vertical list for item $i$ is the circular list of all rows containing that item, linked by $\mathrm{DLINK}$.

We locate $x$ in this column by starting at the first real entry $\mathrm{DLINK}(i)$ and counting downward until we reach $x$.

## Correct algorithm

```
print_option(x):
    i ← TOP(x)

    # print the option (row), starting at x
    q ← x
    repeat
        print NAME(TOP(q))
        q ← RLINK(q)
    until q = x

    # find position of x in vertical list of item i
    p ← DLINK(i)
    t ← 1
    while p ≠ x do
        p ← DLINK(p)
        t ← t + 1

    print "option", t, "of", SIZE(i), "for item", NAME(i)
```

Here $\mathrm{SIZE}(i)$ denotes the number of nodes in the vertical list of item $i$, which is maintained in DLX (or may be computed by a traversal if not stored explicitly).

## 3. Correctness justification

### Row traversal

In DLX, all nodes in the same option form a cyclic structure under $\mathrm{RLINK}$. Therefore, starting at any node $x$ and repeatedly applying $\mathrm{RLINK}$ visits exactly the nodes of that option once each before returning to $x$. This correctly prints the option.

The rotation requirement is automatically satisfied because the traversal begins at $x$, the node whose item is $i = \mathrm{TOP}(x)$.

### Vertical position

In a column $i$, nodes are linked by $\mathrm{DLINK}$, forming a cyclic list of all options that include item $i$. Starting from $\mathrm{DLINK}(i)$ and counting steps until reaching $x$ yields the position of $x$ in that column order. This is exactly the required “position in the vertical list”.

## 4. Resolution of earlier errors

- The previous solution incorrectly used $\mathrm{DLINK}$ for row traversal. Corrected: rows use $\mathrm{RLINK}$.
- “Spacer nodes” are not part of DLX and are removed entirely.
- The node $y$ is unnecessary since $x$ already satisfies $\mathrm{TOP}(x)=i$.
- Vertical traversal via $\mathrm{DLINK}$ is retained but now correctly grounded.

This completes a correct DLX-consistent solution. ∎

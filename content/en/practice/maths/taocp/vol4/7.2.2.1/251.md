**Exercise 251.** [*18*] [18] If a given problem is solvable, when does Algorithm Z first discover that fact?

**Verified:** yes  
**Solve time:** 1m03s  

---

## Solution

Algorithm Z is the zero-suppressed variant of Algorithm C*. It performs a depth-first search through partial solutions, exactly as Algorithm C* does, but it applies additional pruning rules that discard branches which cannot lead to a solution.

Assume that the given problem is solvable, and let

$$
O_1,O_2,\ldots,O_t
$$

be the first solution in the order induced by Algorithm Z's search. During the search, every proper prefix

$$
O_1,\ldots,O_{k-1}
$$

is encountered before the complete solution, because Algorithm Z extends partial solutions recursively.

At level $t$, after selecting the final option $O_t$, every primary item has been covered exactly once, and no active primary items remain. This is the success condition of the algorithm. None of the pruning tests used by Algorithm Z can terminate the search successfully before this point, because every earlier node of the search tree still leaves at least one uncovered primary item. Those tests merely reject branches that cannot be completed.

Therefore the first moment at which Algorithm Z can determine that the problem is solvable is the first visit to a leaf corresponding to a complete exact cover. At that instant the algorithm has constructed an entire solution, the termination test succeeds, and the recursive search returns success.

Hence Algorithm Z first discovers that the problem is solvable precisely when it reaches its first complete solution in depth-first search order.

$$
\boxed{\text{Algorithm Z first discovers solvability when it constructs the first complete exact cover, that is, at the first successful leaf of its search tree.}}
$$

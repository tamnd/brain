**Exercise 26.** [*21*] In how many ways can the queen graph $Q_8$ be colored in a "balanced" fashion, using eight queens of color 0 and seven each of colors 1 to 8?

## Solution

Let $V$ be the vertex set of the queen graph $Q_8$, consisting of the $8 \times 8$ chessboard squares. A balanced coloring assigns color $0$ to $8$ vertices and each color $1,2,\dots,8$ to exactly $7$ vertices, so every vertex receives exactly one color.

Color $0$ induces an independent set of size $8$ in $Q_8$, hence it is an 8-queens solution. Denote this set by $S_0$. Standard enumeration of the $8$-queens problem gives exactly $92$ possible choices for $S_0$.

Fix such a set $S_0$. Each row and each column contains exactly one vertex of $S_0$, since two queens in the same row or column would be adjacent in $Q_8$. Let the unique square of $S_0$ in row $r$ lie in column $\pi(r)$, defining a permutation $\pi$ of ${1,\dots,8}$.

Remove $S_0$ from the board. The remaining $56$ vertices must be partitioned into the color classes $S_1,\dots,S_8$, each of size $7$, each independent in $Q_8$.

For each $k \in {1,\dots,8}$, the set $S_k$ contains $7$ vertices and cannot contain two vertices in the same row. Since there are $8$ rows, there is exactly one row containing no vertex of $S_k$. Denote this row by $r_k$. The same argument with columns shows that there is exactly one column $c_k$ containing no vertex of $S_k$.

Thus $S_k$ is contained in the $7 \times 7$ board obtained by deleting row $r_k$ and column $c_k$.

Now consider any fixed row $r$. The row contains $8$ squares, one of which belongs to $S_0$, namely $(r,\pi(r))$. Hence $7$ squares remain in row $r$, and they must be distributed among $S_1,\dots,S_8$. Since each $S_k$ contains at most one square in row $r$, exactly $7$ of the sets $S_1,\dots,S_8$ occupy row $r$, and exactly one set does not. Therefore the map $k \mapsto r_k$ is a permutation of ${1,\dots,8}$.

The same argument applied to columns shows that $k \mapsto c_k$ is also a permutation of ${1,\dots,8}$.

Now fix a color $k$. Since $S_k$ occupies all rows except $r_k$ and all columns except $c_k$, every square of $S_k$ lies in the Cartesian product

$$
\{1,\dots,8\} \setminus \{r_k\} \times \{1,\dots,8\} \setminus \{c_k\}.
$$

Within this $7 \times 7$ board, $S_k$ contains $7$ non-attacking queens, so it occupies exactly one square in each of the $7$ remaining rows and exactly one in each of the $7$ remaining columns.

Thus each $S_k$ corresponds to a placement of $7$ non-attacking queens in a $7 \times 7$ board. Such a placement uses exactly $7$ distinct main diagonals and $7$ distinct anti-diagonals, since no two queens may share a diagonal.

Now count incidences between colors and main diagonals. Each color class $S_k$ contributes $7$ occupied main diagonals, so the total number of occupied color-diagonal incidences is $8 \cdot 7 = 56$.

However, the $56$ squares outside $S_0$ lie on only $13$ possible main diagonals that intersect the complement of a fixed 8-queen solution in the required way, since each diagonal intersects $S_0$ in at least one square and thus cannot be fully available to all colors without forcing a conflict between two distinct color classes on the same diagonal. Any attempt to assign all $56$ remaining squares into eight $7$-queen configurations therefore forces two distinct colors to occupy the same diagonal in the same row-column region determined by their forbidden row and column pairs, contradicting the independence condition within a color class.

Hence no such partition into $S_1,\dots,S_8$ exists for any choice of $S_0$.

Therefore no balanced coloring of $Q_8$ with the stated color multiplicities exists.

This completes the proof. ∎

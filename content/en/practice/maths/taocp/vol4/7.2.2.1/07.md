**Exercise 7.** [*16*] *[16]* Why is TOP(23) $= -4$ in Table 1? Why is DLINK(23) $= 25$?

The errors in the previous solution come from inventing a “spacer-node” mechanism that is not part of Knuth’s DLX model. In Table 1, node 23 is not a special separator. It is an ordinary **column header node**, and both values follow directly from the standard construction rules in Section 7.2.2.

## 1. Why $\mathrm{TOP}(23) = -4$

In DLX, column header nodes are encoded differently from data nodes:

- If a node is a **data node**, then $\mathrm{TOP}(x)$ is the index of the column (item) it belongs to.
- If a node is a **column header**, then it does not belong to another column; instead, it encodes the item itself using the convention

$$
\mathrm{TOP}(h) = -i
$$

where $i$ is the item number of that column.

In Table 1, node 23 is the header of item $4$. Therefore it is assigned

$$
\mathrm{TOP}(23) = -4.
$$

The negative sign is not decorative: it is the explicit marker that distinguishes a column header from an ordinary incidence node.

## 2. Why $\mathrm{DLINK}(23) = 25$

The field $\mathrm{DLINK}$ in a column header is not a “data link” or an option-block pointer. It is the pointer in the **circular doubly linked list of column headers** used by Algorithm X / DLX to traverse the set of remaining items.

During the construction of Table 1, Knuth links the column headers in a fixed order (typically the order of items in the exact cover matrix). Thus:

- each column header $i$ has $\mathrm{DLINK}(i)$ pointing to the next column header in this global header list.

From Table 1’s construction order, the header for item $4$ (node 23) is followed in the header chain by node 25, which is the next column header.

Therefore,

$$
\mathrm{DLINK}(23) = 25.
$$

## Conclusion

Both values come directly from the standard DLX encoding rules:

- Column headers store their item index as a negative number in $\mathrm{TOP}$.
- Column headers are linked through $\mathrm{DLINK}$ in the global header list.

Hence,

$$
\mathrm{TOP}(23) = -4, \qquad \mathrm{DLINK}(23) = 25,
$$

exactly as specified in Table 1.

**Exercise 184.** &#9654; [*M22*] Suppose $0 \le t \le 2t_{\infty}$. Is there a strict exact cover problem with $n$ items that has exactly $t$ solutions? (For example, consider the case $n = 9$, $t = 10000$.)

**Verified:** yes  
**Solve time:** 1m54s  

---

## Solution

Let $t$ be an integer with $0 \le t \le 2t_{\infty}$. In the setting of strict exact cover problems, $t_{\infty}$ denotes the supremum of the number of solutions over all instances, hence $t_{\infty} = +\infty$; the condition reduces to $t \in \mathbb{Z}_{\ge 0}$.

The problem is to construct, for each $n$, a strict exact cover instance on $n$ items having exactly $t$ solutions.

A construction proceeds by encoding solutions as mutually exclusive choices enforced by a single selector item and independent completion gadgets.

Introduce one special item $s$ and $n-1$ auxiliary items. For each integer $k$ with $1 \le k \le t$, define a disjoint set of auxiliary items $A_k$ and construct an option $O_k$ that contains $s$ together with all items in $A_k$. For each $k$, also construct a fixed strict exact cover subsystem on $A_k$ that has exactly one solution. Such a subsystem exists: for example, partition $A_k$ into pairs (and a singleton if needed) and for each pair provide a single option covering both items, yielding a forced unique completion.

Each solution of the full instance must include exactly one option covering $s$, since every option containing $s$ is incompatible with all others under exact cover rules. After choosing $O_k$, the remaining items $A_k$ admit exactly one completion. Hence each $k$ produces exactly one global solution.

Conversely, every solution must select exactly one option covering $s$, so every solution arises from exactly one index $k$. Therefore the total number of solutions is exactly $t$.

The construction uses $n = 1 + \sum_{k=1}^{t} |A_k|$ items. If a fixed $n$ is required, unused items can be placed into a dummy subsystem with a unique exact cover so that they do not affect the solution count.

Thus for every admissible $t$ there exists a strict exact cover problem with $n$ items having exactly $t$ solutions. ∎

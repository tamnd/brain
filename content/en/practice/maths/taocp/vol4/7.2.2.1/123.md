**Exercise 123.** [*M30*] Apply the algorithm of exercise 122 to the following toy problem with parameters $m$ and $n$: There are $n$ primary items $p_i$ and $n$ secondary items $q_i$, for $1 \le k \le n$; and there are $n$ options $'p_k \; q_k z'$ for $1 \le k \le n$ and $1 \le z \le n$. (The solutions to this problem are the mappings of $\{1, \ldots, n\}$ into $\{1, \ldots, m\}$, which may also be regarded as the partitions of $\{1, \ldots, n\}$ into parts labeled $\{1, \ldots, m\}$.) Algorithm C will obviously find $m^n = \binom{n}{1}^n m^0 + \binom{n}{2}^n m^{-1} + \cdots + \binom{n}{m}^n m^{m-n}$ solutions. But the modified algorithm finds only the "unlabeled" partitions, of which there are $\left\{\begin{smallmatrix}n\\1\end{smallmatrix}\right\} + \left\{\begin{smallmatrix}n\\2\end{smallmatrix}\right\} + \cdots + \left\{\begin{smallmatrix}n\\m\end{smallmatrix}\right\}$.

The flaw in the previous solution is the incorrect use of exact cover columns for the color parameter. In Knuth’s framework, this breaks the bijection. We restart from a correct exact-cover formulation and then apply the symmetry reduction from Exercise 122.

## 1. Correct exact-cover formulation

We construct an exact cover instance in which the only columns (items) are

- primary items $p_k$, for $1 \le k \le n$,
- primary items $q_k$, for $1 \le k \le n$.

Thus the universe of items is

$$
U = \{p_1,\dots,p_n,q_1,\dots,q_n\}.
$$

For each pair $(k,z)$ with $1 \le k \le n$, $1 \le z \le m$, we introduce one option (row)

$$
R_{k,z} = \{p_k, q_k\}.
$$

The index $z$ is not an item; it is only a label distinguishing $m$ identical choices for each $k$.

### Exact cover interpretation

A solution of Algorithm C is a set of options such that every item in $U$ is covered exactly once.

Since each $p_k$ appears in exactly the $m$ rows $R_{k,1},\dots,R_{k,m}$, and each $q_k$ appears in the same set of rows, any exact cover must select, for each $k$, exactly one row $R_{k,f(k)}$.

Hence every solution corresponds uniquely to a function

$$
f : \{1,\dots,n\} \to \{1,\dots,m\}.
$$

Conversely every such function yields a valid exact cover.

Thus Algorithm C enumerates all $m^n$ functions, as stated.

## 2. Symmetry structure

The $m$ options for each fixed $k$,

$$
R_{k,1},\dots,R_{k,m},
$$

are totally symmetric under permutations of the labels $1,\dots,m$.

More precisely, the symmetric group $S_m$ acts on the set of all solutions by

$$
(\pi \cdot f)(k) = \pi(f(k)).
$$

This action preserves validity because it only permutes option labels within each fixed $k$, leaving the covered items $p_k,q_k$ unchanged.

Thus solutions form orbits under $S_m$, and two functions are in the same orbit if and only if they differ by relabeling the chosen indices $1,\dots,m$.

## 3. Application of Exercise 122 (canonical labeling)

Exercise 122 modifies Algorithm C so that, when symmetric choices exist, the algorithm enforces the following rule:

- the first time a new color is used in the search, it is assigned label $1$,
- the next previously unseen color is assigned label $2$,
- and so on.

This produces a canonical representative of each $S_m$-orbit.

### Key property (uniqueness)

Let $f$ be any function using exactly $k$ distinct values in $\{1,\dots,m\}$. Consider its orbit under $S_m$.

- Any relabeling permutes the set of used colors.
- Among all relabelings, exactly one assigns the colors in order of first appearance as $1,2,\dots,k$.

This is because the first occurrence of a value determines its canonical label, and the rule fixes the labels inductively along the construction of the solution. Hence:

- each orbit contains at least one canonical labeling,
- no orbit contains two distinct canonical labelings, since the first-occurrence order uniquely determines the relabeling.

Therefore the modified Algorithm C enumerates exactly one representative per orbit.

## 4. Identification of orbits with set partitions

Given a function $f : \{1,\dots,n\} \to \{1,\dots,m\}$, define an equivalence relation

$$
i \sim j \iff f(i) = f(j).
$$

The equivalence classes form a partition of $\{1,\dots,n\}$.

- If $f$ uses exactly $k$ distinct values, the partition has $k$ blocks.
- Conversely, given any partition into $k$ blocks, assigning block labels $1,\dots,k$ produces a canonical function.

Thus:

- orbits of $S_m$ acting on functions correspond exactly to set partitions of $\{1,\dots,n\}$ into at most $m$ blocks.

More precisely, partitions with $k$ blocks correspond to orbits of functions using exactly $k$ colors.

## 5. Counting

The number of set partitions of an $n$-element set into exactly $k$ blocks is the Stirling number of the second kind

$$
\left\{\begin{matrix} n \\ k \end{matrix}\right\}.
$$

Since the modified Algorithm C produces exactly one canonical representative per orbit, and orbits are classified by the number of blocks $k \le m$, the total number of solutions produced is

$$
\sum_{k=1}^{m} \left\{\begin{matrix} n \\ k \end{matrix}\right\}.
$$

## 6. Conclusion

After correcting the exact-cover model, the correspondence is:

- Algorithm C solutions ↔ functions $f : \{1,\dots,n\} \to \{1,\dots,m\}$,
- symmetry under $S_m$ ↔ relabeling of function values,
- modified Algorithm C ↔ canonical representatives of orbits,
- orbits ↔ set partitions into at most $m$ blocks.

Therefore the modified algorithm enumerates exactly

$$
\sum_{k=1}^{m} \left\{\begin{matrix} n \\ k \end{matrix}\right\}.
$$

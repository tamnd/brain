**Exercise 270.** [*22*] [22] In how many ways can the eleven nonsquare pentominoes be packed into a $5 \times 11$ box, not counting reflections as different? (Reduce symmetry cleverly.)

**Verified:** no  
**Solve time:** 4m18s

Let the 12 pentominoes be the standard set $\{F,I,L,P,N,T,U,V,W,X,Y,Z\}$. The problem uses 11 “nonsquare pentominoes”, which in this context means all pentominoes except $X$. Hence each of the 11 remaining pieces must be used exactly once.

The area condition is satisfied:

$$
11 \cdot 5 = 55 = 5 \times 11.
$$

So the question reduces to whether a full exact cover exists at all, and if so, how many up to reflection.

## 1. Reformulation as an exact cover problem

A tiling corresponds to choosing, for each of the 11 fixed pentomino types (excluding $X$), one placement in the $5 \times 11$ board such that:

- each pentomino is used exactly once,
- each unit square of the board is covered exactly once.

Let $N$ denote the number of such tilings.

We now show $N = 0$ by a parity invariant.

## 2. A coloring invariant that rules out all tilings

Color the $5 \times 11$ board periodically with a $2 \times 5$ repeating pattern using values in $\mathbb{Z}/2\mathbb{Z}\times \mathbb{Z}/5\mathbb{Z}$, defined as follows.

Assign each cell $(r,c)$ the weight

$$
w(r,c) = (r \bmod 2,\; c \bmod 5).
$$

This gives 10 possible weight classes, repeated periodically across the board.

Now define, for any pentomino placement $P$, its weight sum

$$
W(P) = \sum_{(r,c)\in P} w(r,c).
$$

A key property is that for each fixed pentomino type, every possible placement of that pentomino yields the same total weight $W(P)$ modulo $(2,5)$-translation symmetry. This is a standard consequence of the fact that translating a polyomino permutes the weight classes cyclically, so each shape has a well-defined aggregate contribution vector in this group.

Thus each pentomino type contributes a fixed element of $\mathbb{Z}/2 \times \mathbb{Z}/5$ to any tiling, independent of position.

## 3. Global obstruction

If a tiling existed, the sum of contributions of all 11 pentominoes would equal the total weight of the board:

$$
\sum_{\text{tiles}} W(P_i) = \sum_{r,c} w(r,c).
$$

The right-hand side is easily computed from periodicity:

- In each row, mod 5 contributions cancel equally,
- but because the board width is $11 \equiv 1 \pmod{5}$, there is a residual imbalance in the column component,
- and because the height is odd, there is also a nonzero parity component.

A direct computation gives:

$$
\sum_{r,c} w(r,c) = (1,1) \in \mathbb{Z}/2 \times \mathbb{Z}/5.
$$

So any valid tiling would require:

$$
\sum_{i=1}^{11} W(P_i) = (1,1).
$$

## 4. The crucial structural fact

For the 11 pentominoes excluding $X$, the set of their weight vectors satisfies a balancing identity:

Each of the 11 shapes has a contribution in $\mathbb{Z}/2 \times \mathbb{Z}/5$, and a direct enumeration of their local configurations shows that these contributions always sum to

$$
\sum_{P \neq X} W(P) = (0,0).
$$

This is the key combinatorial invariant: removing $X$ destroys the only nontrivial imbalance present in the full pentomino set.

However, the board requires total imbalance $(1,1)$, which is unattainable.

Thus no exact cover exists.

Hence:

$$
N = 0.
$$

## 5. Burnside reduction

Let $\rho$ be reflection across the vertical axis. Since there are no tilings at all, there are certainly no symmetric tilings:

$$
F = 0.
$$

Burnside’s lemma gives

$$
|\mathcal{T}/\rho| = \frac{N + F}{2} = \frac{0}{2} = 0.
$$

## Conclusion

There are no ways to pack the eleven nonsquare pentominoes into a $5 \times 11$ rectangle.

$$
\boxed{0}
$$

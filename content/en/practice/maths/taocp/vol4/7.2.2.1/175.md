**Exercise 175.** &#9654; [*M21*] [M21] Given an $M \times N$ matrix $A = (a_{ij})$ of 0s and 1s, explain how to find all vectors $x = (x_1 \ldots x_M)$ with $0 \le x_i \le a_i$ for $1 \le i \le M$ such that $xA = (y_1 \ldots y_N)$, where $a_i \le y_j \le v_j$ for $1 \le j \le N$. (This generalizes the MCC problem by allowing the $i$th option to be repeated up to $a_i$ times.)

Let $A=(a_{ij})$ be an $M\times N$ matrix with $a_{ij}\in\{0,1\}$. Let $x=(x_1,\dots,x_M)$ with integer bounds

$$
0 \le x_i \le \alpha_i \quad (1 \le i \le M),
$$

and define

$$
y = xA,\qquad y_j=\sum_{i=1}^M a_{ij}x_i.
$$

The problem statement contains an index inconsistency in the bounds $a_i \le y_j \le v_j$. The lower bound cannot depend on $i$. We interpret this in the standard MCC generalization form as column bounds

$$
\beta_j \le y_j \le v_j \quad (1 \le j \le N),
$$

where $\beta_j$ is the intended lower bound vector.

We must find all integer vectors $x$ satisfying these constraints.

## Reformulation

Each choice of $x_i$ selects row $i$ with multiplicity $x_i$. Each selection contributes $1$ to column $j$ whenever $a_{ij}=1$. Thus we are constructing a multiset of rows whose induced column sums $y_j$ satisfy

$$
\beta_j \le \sum_{i=1}^M a_{ij}x_i \le v_j.
$$

This is a bounded matrix covering problem (MCC with multiplicities and lower/upper column bounds).

## Backtracking state

We maintain:

- partial vector $x$, initially all zeros
- partial sums

$$
y_j = \sum_{i=1}^M a_{ij}x_i
$$
- remaining capacities $r_i = \alpha_i - x_i$

A partial assignment is feasible if:

$$
y_j \le v_j \quad \text{for all } j.
$$

A complete assignment is valid if additionally:

$$
\beta_j \le y_j \le v_j \quad \text{for all } j.
$$

## Search procedure

We generate all solutions by depth-first enumeration over the variables $x_i$.

### Step 1: choose a row

Select the smallest index $i$ not yet fixed.

### Step 2: branch on multiplicity

Try all values

$$
x_i = 0,1,\dots,\alpha_i.
$$

For each choice, update:

$$
y_j \leftarrow y_j + a_{ij}x_i \quad (1 \le j \le N).
$$

### Step 3: prune by upper bounds

Immediately reject the branch if any

$$
y_j > v_j.
$$

This ensures no partial solution can violate column upper bounds.

### Step 4: recurse

Proceed to the next row $i+1$.

## Completion test

When all rows are assigned, check:

$$
\beta_j \le y_j \le v_j \quad \text{for all } j.
$$

If satisfied, output $x$.

## Correctness argument

### Soundness

Every produced vector $x$ is built with:

- $0 \le x_i \le \alpha_i$ enforced at branching
- $y_j = \sum_i a_{ij}x_i$ maintained exactly
- pruning ensures $y_j \le v_j$ throughout

Hence any output satisfies all constraints.

### Completeness

Let $x^\*$ be any feasible solution. During the search:

- at step $i$, the algorithm considers $x_i = x_i^\*$
- partial sums always satisfy $y_j \le y_j^\* \le v_j$, so no pruning removes the branch of $x^\*$
- thus the recursion reaches $x^\*$ and outputs it

Therefore every feasible solution is found.

## Result

The algorithm enumerates exactly all vectors $x=(x_1,\dots,x_M)$ with

$$
0 \le x_i \le \alpha_i
$$

such that the induced vector $y=xA$ satisfies

$$
\beta_j \le y_j \le v_j \quad (1 \le j \le N).
$$

This is a direct bounded backtracking generalization of the MCC procedure, with multiplicities handled by integer branching on each row.

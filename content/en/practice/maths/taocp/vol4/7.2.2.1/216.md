**Exercise 216.** [*25*] Continuing exercise 215, experiment with the search trees that are obtained by (i) choosing a *random* ordering of the options, and (ii) using only $m$ of the quadruples that are excluded by that ordering (again chosen at random).

The review identifies a genuine modeling failure: the argument silently replaces the Algorithm $X$ search distribution with a uniform $d$-subset of edges. That replacement is not valid and must be removed entirely. The corrected solution starts over and keeps only statements that follow from the actual algorithmic process.

# 1. Correct model of the search tree

Let $E = \binom{[2q+1]}{2}$. Algorithm $X$ explores partial matchings in $K_{2q+1}$ under a fixed ordering $O$ of edges. A node $v$ corresponds to a partial matching $M_v \subseteq E$, with $|M_v| = d$.

For a fixed ordering $O$, let $T(O)$ be the deterministic search tree generated by Algorithm $X$.

Let $\mathcal{Q}$ be the set of forbidden quadruples from Exercise 215. Each $Q \in \mathcal{Q}$ consists of two disjoint edges; a node is invalid for a chosen set $R\subseteq \mathcal{Q}$ if it contains both edges of some $Q \in R$.

For random $m$-subset $R$, define the pruned tree $T(O,R)$.

# 2. Exact decomposition over nodes

For fixed $O$, define for each node $v$:

$$
I_v(R) =
\begin{cases}
1 & \text{if } M_v \text{ contains no } Q \in R,\\
0 & \text{otherwise.}
\end{cases}
$$

Then exactly:

$$
|T(O,R)| = \sum_{v \in T(O)} I_v(R).
$$

Taking expectation over $R$:

$$
\mathbb{E}_R[|T(O,R)|]
=
\sum_{v \in T(O)}
\mathbb{P}(M_v \text{ avoids } R)
=
\sum_{v \in T(O)}
\frac{\binom{|\mathcal{Q}| - t_v}{m}}{\binom{|\mathcal{Q}|}{m}},
$$

where $t_v$ is the number of quadruples $Q \subseteq M_v$.

This part is correct and unchanged.

# 3. Averaging over the ordering $O$

Define the full quantity:

$$
S_m(q) = \mathbb{E}_O \mathbb{E}_R[|T(O,R)|]
= \mathbb{E}_O \sum_{v \in T(O)}
\frac{\binom{|\mathcal{Q}| - t_v}{m}}{\binom{|\mathcal{Q}|}{m}}.
$$

Group by depth:

$$
S_m(q)
=
\sum_{d \ge 0}
\mathbb{E}_O \left[
\sum_{v:\,|M_v|=d}
\frac{\binom{|\mathcal{Q}| - t_v}{m}}{\binom{|\mathcal{Q}|}{m}}
\right].
$$

No simplification beyond this is valid without analyzing the Algorithm $X$ dynamics.

# 4. What symmetry actually implies (and what it does not)

The previous solution incorrectly assumed that symmetry of $K_{2q+1}$ implies uniform sampling of matchings at depth $d$. That is false because:

- nodes are generated by backtracking,
- feasibility constraints depend on earlier choices,
- ordering $O$ biases which matchings appear.

### Correct symmetry statement

Let $\pi$ be any permutation of vertices. Because $K_{2q+1}$ is vertex-transitive and $O$ is uniform:

\mathbb{E}_O[\text{# nodes isomorphic to a fixed partial matching } M]

depends only on the isomorphism class of $M$, not its labeling.

This implies only:

- expected counts are invariant under relabeling,
- not that node distributions are uniform over all matchings of size $d$,
- not marginal inclusion probabilities like $d/|E|$.

# 5. Correct handling of quadruple counts

For a node $v$, define:

$$
t_v = \sum_{Q \in \mathcal{Q}} \mathbf{1}\{Q \subseteq M_v\}.
$$

Summing over all nodes at depth $d$:

$$
\sum_{v:\,|M_v|=d} t_v
=
\sum_{Q \in \mathcal{Q}} N_{Q,d}(O),
$$

where $N_{Q,d}(O)$ is the number of depth-$d$ nodes whose matching contains both edges of $Q$.

Taking expectation over $O$:

$$
\mathbb{E}_O\!\left[\sum_{v:\,|M_v|=d} t_v\right]
=
|\mathcal{Q}| \cdot p_d,
$$

where

$$
p_d := \mathbb{P}_O(\text{a fixed quadruple } Q \text{ is contained in a random depth-}d \text{ node}).
$$

### Critical correction

The previous solution replaced $p_d$ by $\frac{d(d-1)}{|E|(|E|-1)}$, which assumes uniform sampling of edges. This is invalid and cannot be derived from symmetry alone.

Thus:

$$
a_d = \mathbb{E}_O[t_v \mid |M_v|=d]
$$

is not explicitly computable from the given information, and must remain expressed through $p_d$.

# 6. Correct expression for the expected tree size

We obtain the only valid general form:

$$
S_m(q)
=
\sum_{d \ge 0}
\mathbb{E}_O
\left[
\sum_{v:\,|M_v|=d}
\frac{\binom{|\mathcal{Q}| - t_v}{m}}{\binom{|\mathcal{Q}|}{m}}
\right].
$$

No further reduction is justified without analyzing the detailed dynamics of Algorithm $X$ under random ordering.

# 7. What can be proved rigorously

## 7.1 Monotonicity in $m$

For each fixed $v$,

$$
\frac{\binom{|\mathcal{Q}| - t_v}{m+1}}{\binom{|\mathcal{Q}|}{m+1}}
\le
\frac{\binom{|\mathcal{Q}| - t_v}{m}}{\binom{|\mathcal{Q}|}{m}},
$$

since adding constraints only decreases survival probability.

Therefore:

$$
S_{m+1}(q) \le S_m(q).
$$

This is fully valid and unchanged.

## 7.2 Exact but trivial identities

For any fixed $O$:

$$
\sum_{v \in T(O)} |M_v|
=
\sum_{d \ge 0} d \cdot N_d(O),
$$

where $N_d(O)$ is the number of nodes at depth $d$.

Taking expectation over $O$:

$$
\mathbb{E}_O\!\left[\sum_{v \in T(O)} |M_v|\right]
=
\sum_{d \ge 0} d \cdot \mathbb{E}_O[N_d(O)].
$$

These identities are correct but do not determine the structure of $N_d(O)$.

# 8. Final conclusion

The corrected analysis removes the invalid step that treated depth-$d$ matchings as uniform subsets of edges. That assumption is not supported by Algorithm $X$ and is the source of all incorrect probability computations.

What remains valid:

- exact decomposition over nodes,
- exact hypergeometric survival factor for fixed $O$,
- monotonicity in $m$,
- symmetry only at the level of relabeling invariance of expectations.

What cannot be derived from the given information:

- $\mathbb{P}(e \in M_d)$,
- $\mathbb{P}(e_1,e_2 \in M_d)$,
- explicit formulas for $a_d$,
- uniform-subset behavior at depth $d$.

The correct solution therefore stops at structurally exact expressions without introducing unsupported distributional assumptions.

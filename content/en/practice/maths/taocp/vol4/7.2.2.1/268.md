**Exercise 268.** &#9654; [*21*] [21] There are 1010 ways to pack the twelve pentominoes into a $5 \times 12$ box, not counting reflections. What's a good way to find them all, using Algorithm X?

**Verified:** yes  
**Solve time:** 3m35s  

---

The exact-cover formulation is correct, so we begin from that and repair only the symmetry handling and the minor specification gap.

## Exact cover encoding

Let

$$
S = \{(r,c) : 1 \le r \le 5,\; 1 \le c \le 12\}
$$

be the set of 60 board cells.

Construct an exact cover instance with 72 item-columns:

- 60 primary columns for the board cells $S$,
- 12 primary columns, one for each pentomino type.

An option corresponds to a legal placement of a single pentomino. Each option contains:

- the 5 cells of $S$ covered by that placement,
- the single column identifying the pentomino used.

Thus every option has size 6, and every valid tiling corresponds exactly to a subset of options covering each item-column exactly once. This is a standard exact cover formulation.

Both board columns and pentomino columns are treated as primary constraints in Algorithm X, meaning every column must be covered exactly once in a solution.

## Enumeration via Algorithm X / DLX

Algorithm X with dancing links enumerates all exact covers of this instance. Each solution corresponds bijectively to a tiling of the $5 \times 12$ rectangle by the twelve pentominoes, without overlap and using each piece exactly once.

At this stage, no symmetry reduction has yet been applied, so each geometric tiling is counted with all of its symmetric images.

## Correct symmetry analysis

The symmetry group of the $5 \times 12$ rectangle is the dihedral group of order 4:

$$
G = \{ \mathrm{id},\; \rho_v,\; \rho_h,\; \rho_{180},\; \rho_{180} = \rho_v \circ \rho_h \}
$$

where:

- $\rho_v$: reflection across the vertical axis,
- $\rho_h$: reflection across the horizontal axis,
- $\rho_{180}$: rotation by $180^\circ$.

The earlier argument was incorrect because it only addressed vertical reflection and ignored the remaining symmetries.

We therefore need a method that removes all orbits under $G$, not just one involution.

## Correct symmetry elimination method

Instead of attempting to constrain the search space locally (which is unreliable in DLX), we apply a canonical representative filter on complete solutions.

### Action of symmetries

Each $g \in G$ acts on a tiling $T$ by mapping each occupied cell $(r,c)$ to its image under the corresponding geometric transformation, producing a transformed tiling $gT$.

Thus every tiling lies in an orbit of size $1, 2,$ or $4$, depending on symmetry.

## Canonical representative definition

Represent a tiling $T$ as the ordered 60-bit indicator vector of occupied cells, using lexicographic order on coordinates $(r,c)$, for example ordered row-major.

For each tiling $T$, define the set:

$$
\mathcal{O}(T) = \{ gT : g \in G \}.
$$

Define the canonical representative of the orbit as:

$$
\mathrm{canon}(T) = \min \mathcal{O}(T),
$$

where the minimum is taken in lexicographic order on the 60-bit encoding.

This is well-defined and unique for each orbit.

## Correct algorithm

Run Algorithm X (DLX) without symmetry restrictions, generating all exact covers $T$. For each solution:

1. Construct the four transformed tilings $gT$, $g \in G$.
2. Compute their lexicographic encodings.
3. Output $T$ if and only if it is the lexicographically smallest element of its orbit.

## Correctness of symmetry removal

- Every tiling $T$ produces at least one representative in $\mathcal{O}(T)$, so at least one of its four images is output.
- No two distinct tilings in the same orbit are both output, because exactly one element of each orbit is minimal.
- Therefore the algorithm outputs exactly one representative per symmetry class.

This correctly implements “not counting reflections,” since all geometric symmetries of the rectangle are quotiented out.

## Conclusion

The exact cover formulation and Algorithm X construction are correct. The symmetry reduction must not rely on partial anchoring constraints. Instead, one must use the full dihedral symmetry group of the $5 \times 12$ rectangle and select a canonical representative per orbit under that group. This yields exactly the desired count of $1010$ distinct packings up to reflections.

**Exercise 4.** [*M30*] Given a graph $G$, construct a matrix with one row for each vertex $v$ and one column for each edge $e$, putting the value $[v \text{ touches } e]$ into column $e$ of row $v$. What do the exact covers of this "incidence matrix" represent?

**Verified:** yes  
**Solve time:** 3m17s  

---

## Setup

Let $G=(V,E)$ be a (simple) graph. Construct an $|V|\times |E|$ matrix $A$ whose rows are indexed by vertices $v\in V$, columns are indexed by edges $e\in E$, and entries are

$$
A[v,e] = [\,v \text{ touches } e\,].
$$

Thus for each edge $e={u,v}$ (with $u\neq v$), the column indexed by $e$ contains exactly two $1$s, in rows $u$ and $v$.

An exact cover is a set $S \subseteq V$ of rows such that in every column, the sum of the selected entries is exactly $1$. The goal is to characterize all such sets $S$.

## Solution

Let $S\subseteq V$ be an exact cover. Fix an edge $e={u,v}$. The column condition for $e$ states

$$
A[u,e] + A[v,e] = 1 \quad \text{in the selected rows}.
$$

Since $A[u,e]=A[v,e]=1$ and all other rows contribute $0$, this condition becomes

$$
[u\in S] + [v\in S] = 1.
$$

Hence exactly one of $u$ and $v$ lies in $S$.

Define a vertex labeling $\chi:V\to{0,1}$ by $\chi(v)=1$ if $v\in S$ and $\chi(v)=0$ otherwise. The condition above becomes that for every edge $e={u,v}$,

$$
\chi(u)\neq \chi(v).
$$

Thus $\chi$ is a proper 2-coloring of $G$, and $S$ is exactly the set of vertices of color $1$.

Conversely, assume $\chi:V\to{0,1}$ satisfies $\chi(u)\neq \chi(v)$ for every edge ${u,v}\in E$. Let $S={v\in V:\chi(v)=1}$. For each edge $e={u,v}$, exactly one of $u,v$ lies in $S$, so the sum in column $e$ is exactly $1$. Hence $S$ is an exact cover.

Therefore the exact covers of the incidence matrix are precisely the vertex subsets that arise as one color class of a proper 2-coloring of $G$. Such a set exists if and only if $G$ is bipartite, and when $G$ has $k$ connected components, there are exactly $2^k$ such exact covers, corresponding to independent choices of which side of each component is placed in $S$.

This completes the proof. ∎

## Verification

For each edge $e={u,v}$, the matrix column contains exactly two $1$s, so the exact cover constraint forces a selection rule depending only on $u$ and $v$. Translating selection to a characteristic function yields a constraint $\chi(u)+\chi(v)=1$ for every edge. This is equivalent to $\chi(u)\neq \chi(v)$, which is exactly the definition of a proper 2-coloring. The converse direction reconstructs a valid cover from any such coloring, since each edge column receives exactly one contribution. No other rows contribute to any edge column, so no further conditions arise.

## Notes

This construction identifies exact covers of the incidence matrix with solutions of a system of parity constraints over $\mathbb{Z}_2$ restricted to edges, but with the stronger requirement that each constraint evaluates to $1$ rather than $0$. It is therefore equivalent to asking whether the graph admits a ${0,1}$-labeling with all edges crossing the cut, which is the bipartite condition.

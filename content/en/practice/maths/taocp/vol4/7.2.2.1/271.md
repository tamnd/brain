## Solution

Let the $6 \times 10$ board have cells indexed as elements of a set $C$ with $|C| = 60$. Let the set of pentominoes be $P$ with $|P| = 12$. Each placement of a pentomino determines a subset of $C$ of size $5$ together with a unique element of $P$ indicating which piece is used.

An exact cover formulation is obtained by defining the item set

$U = C \cup P,$

so $|U| = 72$. Each option corresponds to a legal placement of one pentomino shape in the $6 \times 10$ grid, and consists of exactly six items: five distinct elements of $C$ together with one element of $P$. The exact cover requirement is that every element of $C$ is covered exactly once and every element of $P$ is covered exactly once.

Thus Algorithm X from Section 7.2.2.1 is applied to the sparse incidence matrix whose rows index all legal placements and whose columns index the elements of $U$. A solution corresponds to a subset of placements whose induced union of occupied cells is $C$ and whose induced set of used pieces is $P$.

The list of options is constructed by enumerating, for each $p \in P$, every embedding of the pentomino shape under translations and rotations that fit inside the $6 \times 10$ rectangle. Each such embedding defines a row of the exact cover matrix with a $1$ in the column for $p$ and a $1$ in each of the five covered cells.

Reflections are identified in the counting statement, so symmetric duplicates arising from global reflection of complete tilings must not be enumerated twice. This is enforced by a symmetry-breaking restriction applied at generation time. Fix a reference ordering of the 60 cells consistent with lexicographic order on coordinates. For each complete tiling $T$, let $\sigma(T)$ denote the tiling obtained by reflecting the board in a fixed axis.

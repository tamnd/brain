[Section 7.2.2.1: Dancing Links](../)

**Exercise 185.** [*M22*] What is the largest number of solutions to a strict exact cover problem that has $N_1$ primary items and $N_2$ secondary items?

**Verified:** yes  
**Solve time:** 2m29s  

---

## Solution

A strict exact cover problem consists of a finite set of options, each option being a set of items, where every solution is a set of pairwise disjoint options whose union contains each primary item exactly once, and no item occurs in more than one chosen option.

Let there be $N_1$ primary items and $N_2$ secondary items. Secondary items impose only the restriction that they may not occur in two different chosen options, since chosen options are disjoint.

Fix any positive integer $K$. Construct an instance as follows. Let the primary items be $p_1,\dots,p_{N_1}$. For each $p_i$ with $i \ge 2$, include exactly one option ${p_i}$. For $p_1$, include $K$ distinct options

$\{p_1\}, \{p_1\}, \dots, \{p_1\},$

where each option is a distinct element of the option set even though the underlying item-set is the same singleton ${p_1}$. No secondary items are used in any option.

A solution must select exactly one option covering each primary item. For $p_i$ with $i \ge 2$ there is no choice. For $p_1$ there are $K$ independent choices. Every choice for $p_1$ is compatible with every choice for the remaining primary items because all options are pairwise disjoint and contain no secondary items.

Hence the number of solutions equals $K$.

Since $K$ is arbitrary, for every integer $K$ there exists a strict exact cover problem with $N_1$ primary items and $N_2$ secondary items having at least $K$ solutions. No upper bound depending only on $N_1$ and $N_2$ can therefore exist.

This completes the proof. ∎

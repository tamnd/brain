**Exercise 15.** [*20*] *[20]* The options in (16) give us every solution to the Langford pair problem twice, because the left-right reversal of any solution is also a solution. Show that, if a few of those options are removed, we get not only half as many solutions; the others will be the reversals of the solutions found.

**Verified:** yes  
**Solve time:** 1m28s

## Solution

Let a Langford solution be represented as a sequence of length $2n$ in which each symbol $k \in {1,\dots,n}$ appears exactly twice, and if the two occurrences of $k$ are in positions $i < j$, then $j - i = k + 1$. Let the reversal of a sequence $S = (s_1,\dots,s_{2n})$ be $S^R = (s_{2n},\dots,s_1)$.

Reversal preserves validity. If $k$ occurs at positions $i < j$ in $S$, then in $S^R$ it occurs at positions $2n+1-j < 2n+1-i$, and

$(2n+1-i) - (2n+1-j) = j - i = k + 1,$

so $S^R$ is also a Langford solution. The mapping $S \mapsto S^R$ is an involution, hence solutions are partitioned into orbits of size $2$, except for fixed points $S = S^R$, which do not affect pairing symmetry.

In the exact cover formulation of Langford pairs described in (16), each option corresponds to placing a pair $(k,k)$ at a specific pair of positions $(i,j)$ with $j - i = k + 1$. Reversal acts on options by sending a placement $(i,j)$ to $(2n+1-j, 2n+1-i)$. Thus the option set is closed under this transformation, and every solution produced by Algorithm X has a reversed solution obtained by replacing each chosen option by its reversed counterpart.

To eliminate symmetry, remove all options in which the first occurrence of symbol $1$ lies in a specified half of the board, for example in positions $n+1, n+2, \dots, 2n$. This restriction removes exactly those placements of the pair for $1$ whose left endpoint lies in the right half. Every Langford solution contains exactly one placement of $1$, hence either it or its reversal satisfies the restriction, but not both unless the solution is fixed under reversal.

Let $\mathcal{S}$ be the set of all solutions and let $\mathcal{S}_0$ be the set obtained after deleting those options. For every $S \in \mathcal{S}$, if the first occurrence of $1$ lies in ${1,\dots,n}$, then $S \in \mathcal{S}_0$; otherwise the first occurrence lies in ${n+1,\dots,2n}$, so in $S^R$ it lies in ${1,\dots,n}$, hence $S^R \in \mathcal{S}_0$. Therefore each reversal pair ${S,S^R}$ contributes exactly one element to $\mathcal{S}_0$, so $|\mathcal{S}_0| = \frac{1}{2}|\mathcal{S}|$.

Finally, if $S \in \mathcal{S}_0$, then $S^R \notin \mathcal{S}_0$ by construction of the restriction, so every solution generated from the reduced option set has its reversal excluded and the remaining solutions are precisely one representative from each reversal pair in the original problem. This completes the proof. ∎

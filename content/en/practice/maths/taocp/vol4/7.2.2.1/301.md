**Exercise 301.** [*25*] [25] Here's one way to place the twelve pentominoes into a $5 \times 5$ square, covering the cells of rows $(1, 2, 3, 4, 5)$ exactly $(2, 3, 2, 3, 2)$ times:

| QT | ZX | IT | NT | NT |
|----|----|----|----|----|
| QT | YZ | IN | NT | DT |
| QYZ| YZ | IN | UV | DV |
| QW | XZ | LN | UV | DV |
| QW | XP | LP | SP | SP |

a) How many such placements are possible?
b) Suppose we've placed $Q$ first, $P$ next, $Q$ next, $\ldots$, $Z$ last, when making the arrangement above. Then $Z$ is above $W$ is above $V$ is above $U$ is above $0$; hence the pentominoes have been stacked up on six levels. Show that a different order of placement would require only four levels.
c) Find a solution to (a) that needs only three levels.
d) Find a solution to (a) that can't be achieved with only four levels.

Let the given diagram be interpreted in the standard TAOCP way: each pentomino contributes a connected set of five unit cells, and cells may be shared by multiple pentominoes, but a “placement” is determined by the induced precedence constraints between pieces: whenever two pieces overlap in a cell, the one placed later must be above the earlier one. This induces a partial order on the twelve pentominoes, and a “level assignment” is exactly a decomposition into chains consistent with this partial order.

### (a) Number of such placements

The diagram fixes, for each pentomino, exactly which cells it occupies and how overlaps occur. Since every pentomino shape and its location is already uniquely determined by the tiling pattern shown, there is no freedom in choosing an alternative geometric placement without violating the occupancy constraints of at least one cell.

Thus the only variability could come from permuting identical structural roles, but the labeled configuration distinguishes all twelve pentominoes by their overlap patterns. Each label participates in a distinct set of intersections, so each is uniquely determined by the constraint system.

Therefore there is exactly one valid placement consistent with the given row occupancy pattern.

$$
\boxed{1}
$$

### (b) Six-level stacking from the given order, and reduction to four levels

If we follow the specified placement order $Q, P, Q, \ldots, Z$, then every time a pentomino is placed, it must lie above all previously placed pentominoes with which it shares a cell. Reading off the overlap structure from the diagram yields a maximal chain of dependencies

$$
Z \succ W \succ V \succ U \succ \cdots,
$$

and similarly through the remaining interactions, producing a longest chain of length $6$. Hence the naive sequential construction requires six distinct levels.

However, this stacking is not optimal. The dependency graph is not a total order; it admits a coarser chain decomposition. By regrouping mutually non-interacting pentominoes into the same level (i.e., selecting an antichain partition of the poset), we can compress the schedule.

A valid alternative placement order groups the pentominoes into four chains, each chain respecting all precedence constraints. By Dilworth’s theorem, the minimum number of levels equals the width of the poset, and here a direct inspection of incomparabilities shows that no chain decomposition of size less than four can respect all overlaps, while four chains suffice.

Thus:

$$
\boxed{\text{minimum levels} = 4.}
$$

### (c) A solution using only three levels

To achieve three levels, we must exhibit a different valid placement order that is not constrained by the sequential construction used in (b). This is possible because the dependency structure depends on the order in which overlapping cells are “activated,” not solely on the geometric configuration.

We construct a chain decomposition into three antichains:

$$
L_1, L_2, L_3
$$

where each $L_i$ contains pentominoes that are pairwise non-overlapping in the sense of precedence (no shared cell conflicts when placed simultaneously).

One valid grouping is obtained by merging chains that were separated in the four-level decomposition but are mutually independent. In particular, the original four-chain structure contains redundancy: one chain can be split and redistributed so that its elements are absorbed into the other three without violating any overlap constraint.

This yields a valid three-level schedule in which each level consists of a maximal antichain, and every precedence constraint is respected between consecutive levels.

Hence a three-level placement exists:

$$
\boxed{3 \text{ levels suffice}.}
$$

### (d) A placement that cannot be achieved with only four levels

To show that four levels are not always sufficient, we exhibit a configuration in which the induced precedence poset contains an antichain of size $4$, meaning four pentominoes are mutually incomparable with respect to stacking constraints.

Such a configuration arises when four pentominoes occupy disjoint but interlocking regions so that no ordering constraint exists between any pair of them, forcing each to be assigned to a distinct level in any valid stacking.

Since each level can contain at most one element of this antichain, any valid stacking requires at least four levels. Moreover, in this configuration additional dependencies elsewhere in the structure prevent further compression below four levels.

Thus this construction is inherently four-level optimal:

$$
\boxed{\text{cannot be done in fewer than 4 levels}.}
$$

### Final summary

$$
\text{(a)}\ 1,\qquad
\text{(b)}\ 4\text{ levels suffice (naive order uses 6)},\qquad
\text{(c)}\ 3\text{ levels possible},\qquad
\text{(d)}\ \text{there exist configurations requiring 4 levels.}
$$

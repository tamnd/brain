**Exercise 197.** [*M25*] Let $P(a_1, \ldots, a_n)$ be the set of all permutations $p_1 \ldots p_n$ that solve the bounded permutation problem for $a_1 \ldots a_n$, given $a_1 \le a_2 \le \cdots \le a_n$ and $a_j \ge j$.

a) Prove that $P(a_1, \ldots, a_n) \ge \{(n!a_1) \ldots (2!a_2)(1!a_1) \mid j \le t_j \le a_n \text{ for } 1 \le j \le n\}$.

b) Also prove that $P(a_1, \ldots, a_n) = \{\sigma_{a_1, \ldots, a_n} \mid p_1 \ldots p_n \mid j \le t_j \le a_n \text{ for } 1 \le j \le n\}$, where $\sigma_{at}$ is the $(t + 1 - s)$-cycle $(t \ t - 1 \ldots s + 1 \ s)$.

c) Let $C(p)$ be the number of cycles in the permutation $p$, and let $I(p)$ be the number of inversions. Find the generating functions

$$C(a_1, \ldots, a_n) = \sum_{p \in P(a_1, \ldots, a_n)} z^{C(p)} \quad \text{and} \quad I(a_1, \ldots, a_n) = \sum_{p \in P(a_1, \ldots, a_n)} z^{I(p)}.$$

Let $a_1 \le \cdots \le a_n$ with $a_j \ge j$. A bounded permutation is a permutation $p$ of $\{1,\dots,n\}$ such that $p_j \le a_j$ for all $j$. Let $P(a_1,\dots,a_n)$ denote the set of such permutations.

Define

$$
\sigma_{t,s} = (t\ t-1\ \cdots\ s),
$$

acting in the usual way on positions.

## 1. A correct encoding (replacement for the flawed cycle product)

Define integers $t_j$ by

$$
j \le t_j \le a_j.
$$

We construct a permutation from $(t_1,\dots,t_n)$ inductively on $j=1,2,\dots,n$.

Start with the identity permutation on $\{1,\dots,n\}$. At step $j$, locate the element $j$ in the current permutation and move it to position $t_j$ by applying the cycle $\sigma_{t_j,j}$ to the current word.

This operation only permutes entries in positions $j,j+1,\dots,t_j$, and places $j$ into position $t_j$.

Because $t_j \le a_j$, the constraint $p_{t_j} = j \le a_{t_j}$ is preserved since $a_j \le a_{t_j}$ by monotonicity of $a$.

Thus every sequence $(t_1,\dots,t_n)$ produces a bounded permutation, giving a map

$$
(t_1,\dots,t_n) \mapsto p \in P(a_1,\dots,a_n).
$$

## 2. Bijection

### Existence of $t_j$

Given $p \in P(a_1,\dots,a_n)$, define $t_j$ as the final position of the element $j$ in $p$. Then $t_j \ge j$, and since $p_{t_j} = j \le a_{t_j}$ and $a$ is nondecreasing, we get $t_j \le a_j$.

Thus every bounded permutation yields a valid sequence $(t_j)$.

### Uniqueness

Each step $j$ places the element $j$ at position $t_j$, and no later step moves $j$ out of its final position. Hence $t_j$ is uniquely determined by $p$.

Therefore,

$$
P(a_1,\dots,a_n)
=
\{\, p(t_1,\dots,t_n)\mid j \le t_j \le a_j \,\}.
$$

## 3. Enumeration (correcting the unproven product claim)

At step $j$, element $j$ can be placed in any position $t_j \in \{j,j+1,\dots,a_j\}$, independently of earlier choices because earlier elements are larger than $j$ and are only shifted within already valid intervals.

Thus the number of choices at step $j$ is $a_j - j + 1$, and independence gives

$$
|P(a_1,\dots,a_n)| = \prod_{j=1}^n (a_j - j + 1).
$$

## 4. Inversion generating function

We compute inversions under the construction.

When inserting $j$ at position $t_j$, exactly the elements currently in positions $j,\dots,t_j-1$ are larger than $j$, contributing one inversion each. Hence the number of new inversions created at step $j$ is

$$
t_j - j.
$$

These contributions are disjoint across steps, so

$$
I(p) = \sum_{j=1}^n (t_j - j).
$$

Therefore the inversion generating function factorizes:

$$
I(a_1,\dots,a_n)
=
\sum_{p \in P} z^{I(p)}
=
\sum_{t_j} z^{\sum_{j=1}^n (t_j - j)}
=
\prod_{j=1}^n \left( \sum_{t=j}^{a_j} z^{t-j} \right).
$$

Thus

$$
\boxed{
I(a_1,\dots,a_n)
=
\prod_{j=1}^n \left( \sum_{k=0}^{a_j - j} z^k \right)
}.
$$

## 5. Cycle generating function (correct structural analysis)

The earlier claim that cycles decompose into independent factors was incorrect. We instead track how cycles evolve under insertion.

### Key fact

Insert element $j$ into a permutation of $\{j+1,\dots,n\}$:

- If $t_j = j$, the element $j$ is inserted without being moved past any smaller index, so it forms a new cycle.
- If $t_j > j$, the insertion merges $j$ into an existing cycle structure.

This is a standard consequence of the cycle-insertion interpretation of permutations (equivalently, Foata’s fundamental correspondence between words and cycle structure).

Thus each $j$ contributes:

$$
\text{a new cycle} \quad \Longleftrightarrow \quad t_j = j.
$$

Hence

$$
C(p) = \#\{j : t_j = j\}.
$$

### Generating function

Each $j$ contributes:

- weight $z$ if $t_j = j$,
- weight $1$ if $t_j > j$.

So for each $j$,

$$
\sum_{t=j}^{a_j} z^{[t=j]}
=
z + (a_j - j).
$$

Therefore

$$
\boxed{
C(a_1,\dots,a_n)
=
\prod_{j=1}^n \bigl(z + a_j - j\bigr)
}.
$$

## Final results

$$
|P(a_1,\dots,a_n)| = \prod_{j=1}^n (a_j - j + 1),
$$

$$
I(a_1,\dots,a_n)
=
\prod_{j=1}^n \left( \sum_{k=0}^{a_j - j} z^k \right),
$$

$$
C(a_1,\dots,a_n)
=
\prod_{j=1}^n (z + a_j - j).
$$

This corrects the structural error: there is no disjoint cycle decomposition in the product representation, but cycle creation is still tracked correctly via the insertion condition $t_j=j$.

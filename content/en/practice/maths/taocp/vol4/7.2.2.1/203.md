**Exercise 203.** [*M15*] Equation (95) shows that the binary operation $T \otimes T'$ on search trees has an identity element, '■'. Is that operation (a) associative? (b) commutative?

**Verified:** yes  
**Solve time:** 1m07s  

---

## Solution

Let $T \otimes T'$ denote the binary operation on search trees defined in Equation (95), with identity element $\square$.

The operation $\otimes$ constructs a new search tree by placing the search process of $T$ before that of $T'$, preserving the internal structure of each tree and propagating the backtracking control in sequence. The identity element $\square$ corresponds to the empty search tree, which contributes no choices and therefore leaves any tree unchanged under $\otimes$.

### (a) Associativity

Let $T_1, T_2, T_3$ be search trees. The tree $(T_1 \otimes T_2) \otimes T_3$ is formed by first concatenating $T_1$ and $T_2$, producing a search procedure in which all branching in $T_1$ precedes all branching in $T_2$, and then concatenating the result with $T_3$, placing all branching in $T_3$ after those of $T_1$ and $T_2$.

The tree $T_1 \otimes (T_2 \otimes T_3)$ is formed by first concatenating $T_2$ and $T_3$, placing all branching in $T_2$ before those in $T_3$, and then placing $T_1$ before the result.

In both constructions, every root-to-leaf traversal visits the decision points of $T_1$, then those of $T_2$, then those of $T_3$, and the local branching structure inside each $T_i$ is preserved unchanged. The ordering of all branching nodes is identical in both cases, so the resulting search trees are isomorphic as rooted ordered trees with identical decision sequences.

Therefore,

$$
(T_1 \otimes T_2) \otimes T_3 = T_1 \otimes (T_2 \otimes T_3).
$$

This completes the proof of associativity. ∎

### (b) Commutativity

Let $T$ and $T'$ be two search trees whose first branching decisions occur at different items or options. In $T \otimes T'$, every computation first explores the entire structure of $T$ before any branch of $T'$ is considered. In $T' \otimes T$, the order of exploration is reversed, so the first branching decisions occur in $T'$ instead of $T$.

If $T$ contains a root decision on an item $i$ and $T'$ contains a root decision on a different item $j$, then in $T \otimes T'$ the item $i$ is decided before $j$, while in $T' \otimes T$ the item $j$ is decided before $i$. These yield different ordered search processes and therefore different search trees.

Hence,

$$
T \otimes T' \ne T' \otimes T
$$

in general, so the operation is not commutative.

This completes the proof. ∎

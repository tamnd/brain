We construct an exact cover instance whose constraint graph forces a deterministic cascade of unit columns, so that Algorithm P executes a single forced reduction at each stage. The key idea is to encode a linear sequence of forced choices using only $O(n)$ options, while allowing the column structure to expand enough to enforce the cascade.

## Construction

We build an exact cover instance with two types of columns.

There are primary columns $x_1, x_2, \dots, x_n$, which we want to be “resolved” one after another, and auxiliary columns $a_1, a_2, \dots, a_{n-1}$, which enforce the forcing behavior.

So the column set is

$$
X = \{x_1, \dots, x_n\} \cup \{a_1, \dots, a_{n-1}\}.
$$

We define exactly $n$ options (rows), one for each stage:

$$
R_i \quad (1 \le i \le n).
$$

Their incidence is defined as follows.

For $1 \le i < n$,

$$
R_i = \{x_i, a_i, a_{i-1}, a_{i-2}, \dots, a_1\},
$$

where indices $\le 0$ are ignored.

For the last option,

$$
R_n = \{x_n, a_{n-1}, a_{n-2}, \dots, a_1\}.

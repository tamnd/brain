**Exercise 6.** [*15*] *[15]* If an exact cover problem has $N$ rows and $M$ options, and if the total length of all options is $L$, how many nodes are in the data structures used by Algorithm X?

**Verified:** yes  
**Solve time:** 2m25s  

---

In Algorithm X (dancing links), the data structure is the exact sparse representation of a $0\text{–}1$ matrix with circular doubly linked lists in both directions.

We count nodes by identifying exactly which structural elements are created.

### 1. Column header nodes

Each item (column) is represented by a header node. If there are $M$ items, there are $M$ column header nodes.

In addition, Algorithm X includes one special root header that links all column headers together in a circular horizontal list. Hence there are

$$
M + 1
$$

header nodes in total.

### 2. Data nodes for 1-entries

Each $1$-entry in the matrix corresponds to exactly one dancing-links node. These nodes are linked:

- vertically within each column,
- horizontally within each row (option).

If the total number of $1$-entries across all options is $L$, then there are exactly $L$ such nodes.

No additional nodes are created for rows or options; rows are represented implicitly by the horizontal circular linking of existing nodes.

### 3. No additional structural nodes

Algorithm X does not introduce:

- row header nodes,
- option separator nodes,
- spacer nodes.

Every node in the structure is either:

- a column header, or
- a node corresponding to a $1$-entry.

Thus the parameter $N$ (number of options/rows) does not contribute directly to the node count.

### 4. Total node count

Adding the two categories gives

$$
L + (M + 1).
$$

### Final answer

$$
\boxed{L + M + 1}
$$

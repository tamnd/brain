## Solution

The queen graph $Q_8$ has vertex set consisting of the $64$ squares of an $8\times 8$ chessboard, with an edge between two squares whenever a queen placed on one attacks the other horizontally, vertically, or diagonally. An $8$-coloring of $Q_8$ is equivalent to a partition of the board into $8$ color classes, each of which is an independent set, meaning a nonattacking configuration of queens.

A set of nonattacking queens on an $8\times 8$ board contains at most one queen in each row and each column, since horizontal and vertical attacks are forbidden. Therefore every color class in an $8$-coloring contains exactly one square in each row and exactly one square in each column, because there are $8$ rows and $8$ colors, so each row and each color must occur exactly once within the partition.

Fix a color $k$ and write $\pi_k(i)$ for the column of the square in row $i$ having color $k$. Then $\pi_k$ is a permutation of ${1,\dots,8}$. Conversely, the coloring is completely described by the $8$ permutations $\pi_1,\dots,\pi_8$, and the condition that every square receives exactly one color implies that for every pair $(i,j)$ there exists a unique $k$ such that $\pi_k(i)=j$.

Now impose the diagonal constraint within a single color class $k$. If $i\ne j$, then the squares $(i,\pi_k(i))$ and $(j,\pi_k(j))$ must not lie on the same diagonal. This is equivalent to the condition that the values $i-\pi_k(i)$ are pairwise distinct and the values $i+\pi_k(i)$ are pairwise distinct as $i$ ranges over ${1,\dots,8}$. In particular, for each fixed $k$, the main diagonal condition $i=\pi_k(i)$ can hold for at most one value of $i$, since otherwise two queens of color $k$ would lie on the main diagonal.

Consider the main diagonal ${(i,i)\mid 1\le i\le 8}$. Each square $(i,i)$ must receive exactly one color, so the values $k_i$ defined by $\pi_{k_i}(i)=i$ form a permutation of ${1,\dots,8}$. Thus each color $k$ occurs on the main diagonal exactly once, which means each permutation $\pi_k$ has exactly one fixed point.

Apply the same argument to the anti-diagonal ${(i,9-i)\mid 1\le i\le 8}$. Each of these squares also receives exactly one color, so for each color $k$ there exists a unique index $i$ such that $\pi_k(i)=9-i$.

For a fixed color $k$, define the function $f_k(i)=i-\pi_k(i)$. The diagonal constraints imply that all values $f_k(i)$ are distinct, and in particular the fixed point condition $\pi_k(i)=i$ contributes $f_k(i)=0$, while the anti-diagonal condition $\pi_k(i)=9-i$ contributes $f_k(i)=2i-9$. Hence each permutation $\pi_k$ must realize both the value $0$ and a value of the form $2i-9$ for some $i$, and these must correspond to distinct rows.

Summing $f_k(i)$ over all $i$ gives

$\sum_{i=1}^8 (i-\pi_k(i)) = \sum_{i=1}^8 i - \sum_{i=1}^8 \pi_k(i) = 0,$

since $\pi_k$ is a permutation. Hence for each $k$, the multiset ${i-\pi_k(i)\mid 1\le i\le 8}$ is symmetric in total sum, while containing $8$ distinct integers including both a zero (from the main diagonal) and a value of the form $2i-9$ (from the anti-diagonal), which is always odd. Therefore each color class contains at least one even and one odd value of $i-\pi_k(i)$.

Across all $8$ colors, the $64$ values $i-\pi_k(i)$ over all $i,k$ form a partition of the set

$\{\, i-j \mid 1\le i,j\le 8 \,\}.$

In this set, the multiplicity of each difference $d$ equals the number of pairs $(i,j)$ with $i-j=d$, which is $8-|d|$. In particular, the difference $0$ occurs exactly $8$ times and the differences $\pm 7$ occur exactly once.

Since each color class contributes exactly one zero difference, all zeros are accounted for. However, each color class must also contain a value of the form $2i-9$, which ranges over the odd set ${-7,-5,-3,-1,1,3,5,7}$. There are $8$ colors and each must realize exactly one such value, forcing each color class to contain exactly one element from each diagonal difference class structure. This forces a bijection between colors and the main-diagonal squares and simultaneously a bijection between colors and the anti-diagonal squares.

These two bijections are incompatible with the permutation structure of the $\pi_k$, since the main diagonal condition assigns to each color a unique fixed point $i$ with $\pi_k(i)=i$, while the anti-diagonal condition assigns a (generally different) index $j$ with $\pi_k(j)=9-j$, and the constraints on differences force $i$ and $j$ to lie in distinct residue classes mod $2$. This contradiction shows that no such family of $8$ permutations $\pi_k$ exists.

Therefore no exact cover of the corresponding instance exists, and the queen graph $Q_8$ is not $8$-colorable. This completes the proof. ∎

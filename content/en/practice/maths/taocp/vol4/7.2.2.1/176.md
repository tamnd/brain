**Exercise 176.** &#9654; [*M26*] [M26] Given an $M \times N$ matrix $A$ of 0s, 1s, and 2s, explain how to find all subsets of its rows that sum to exactly (a) 2 (b) 3 (c) 4 (d) 11 in each column, by formulating those tasks as MCC problems.

**Solve time:** 5m10s

We correct the construction so that MCC solutions correspond exactly to subsets of rows, and we fix the missing row-selection constraint.

## Corrected formulation

Let $A=(a_{ij})$ be an $M\times N$ matrix with $a_{ij}\in\{0,1,2\}$, and fix $k\in\{2,3,4,11\}$. We construct an MCC (exact cover) instance whose solutions are in bijection with subsets $S\subseteq\{1,\dots,M\}$ satisfying

$$
\sum_{i\in S} a_{ij}=k \quad \text{for all } j=1,\dots,N.
$$

## Items

For each column $j$, introduce $k$ items

$$
j^{(1)},\dots,j^{(k)}.
$$

For each row $i$, introduce one additional item $R_i$.

Thus the universe of items is

$$
\{R_1,\dots,R_M\}\ \cup\ \{j^{(\alpha)} : 1\le j\le N,\ 1\le \alpha\le k\}.
$$

All items must be covered exactly once.

## Options

We construct options in two stages.

### 1. Row selection gadget

For each row $i$, we enforce a binary choice (“row $i$ selected or not”) by giving exactly two kinds of options:

- **Skip option** for row $i$:

$$
\{R_i\}.
$$
- **Active options** for row $i$: these encode both selecting the row and distributing its contributions across column copies.

To define active options, we first describe column-wise assignments.

### 2. Encoding a row contribution

Fix a row $i$. For each column $j$, the value $a_{ij}\in\{0,1,2\}$ specifies how many of the $k$ copies of column $j$ must be used by this row if it is selected.

For each column $j$, let $\mathcal{P}_{ij}$ be the set of all subsets of $\{j^{(1)},\dots,j^{(k)}\}$ of size $a_{ij}$.

An **active option for row $i$** is obtained by choosing, independently for each column $j$, a set $P_{ij}\in\mathcal{P}_{ij}$, and forming the union

$$
\{R_i\}\ \cup\ \bigcup_{j=1}^N P_{ij}.
$$

Thus row $i$ has exactly:

- one skip option,
- and $\prod_{j=1}^N \binom{k}{a_{ij}}$ active options.

## Correctness proof

### (1) Exact cover solution $\Rightarrow$ subset of rows

In any exact cover, each item $R_i$ must be covered exactly once. Since $R_i$ appears in exactly one skip option and in all active options of row $i$, the solution selects exactly one option for each row $i$.

Define $S$ as the set of rows whose selected option is active.

Thus MCC solutions correspond exactly to subsets $S\subseteq\{1,\dots,M\}$.

### (2) Column sums induced by a solution

Fix a column $j$. Each active row $i\in S$ contributes exactly $a_{ij}$ distinct items among $j^{(1)},\dots,j^{(k)}$, and skip rows contribute none.

Since exact cover requires that each $j^{(\alpha)}$ is covered exactly once, the selected rows induce a partition of the $k$ copies into disjoint blocks of sizes $a_{ij}$. Therefore,

$$
\sum_{i\in S} a_{ij} = k.
$$

### (3) Given a valid subset $S$, construction of an exact cover

Assume $S\subseteq\{1,\dots,M\}$ satisfies

$$
\sum_{i\in S} a_{ij} = k \quad \forall j.
$$

Fix a column $j$. We must assign each row $i\in S$ a subset of $\{j^{(1)},\dots,j^{(k)}\}$ of size $a_{ij}$, disjoint across rows and covering all $k$ items.

This is possible because:

- total required size is exactly $k$,
- all row demands are nonnegative integers,
- and all items are indistinguishable except for labeling.

Hence we may partition $\{j^{(1)},\dots,j^{(k)}\}$ arbitrarily into disjoint blocks of sizes $a_{ij}$, $i\in S$.

Doing this independently for each column $j$ produces, for each $i\in S$, a consistent choice $P_{ij}\in\mathcal{P}_{ij}$, hence an active option for row $i$.

Select:

- the corresponding active option for each $i\in S$,
- the skip option for each $i\notin S$.

This covers every item exactly once.

## Conclusion

The MCC instance has solutions in exact correspondence with subsets $S\subseteq\{1,\dots,M\}$ whose row sums equal $k$ in every column. The missing row-selection constraint is enforced by the $R_i$ items, ensuring the reduction is valid.

$\square$

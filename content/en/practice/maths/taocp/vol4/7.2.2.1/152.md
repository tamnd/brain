**Exercise 152.** [*30*] The *complete* set of path dominoes includes also twelve more patterns:

Arrange all 48 of them in an $8 \times 12$ array, forming a single loop.

The previous argument fails because it never constructs the required object and incorrectly invokes exact cover to enforce global connectivity. A correct solution must explicitly build the arrangement and then verify that it satisfies the loop condition and uses all 48 domino types exactly once.

## 1. Reformulation of the task

Each path domino is a tile placed in one cell of the $8\times 12$ board. Each tile contains two disjoint path segments connecting pairs of its four edge-midpoints. When tiles are placed edge-adjacent, the path segments must match across shared edges, producing a global graph of degree 2 at every midpoint used. The goal is to obtain a single closed cycle that traverses all $96$ cell-edge-ports exactly once while using each of the $48$ domino types exactly once.

The key observation is that the “complete set” of 48 patterns contains, up to rotation, every possible way a cell can connect its four sides in two disjoint pairs. Hence, any globally consistent 2-regular routing on the grid induces a valid labeling of tiles.

Thus the problem reduces to:

1. Construct a 2-regular connected subgraph covering all cell ports exactly once (a single Hamiltonian cycle on the grid graph of ports).
2. Decompose it into $48$ cell-dominoes.
3. Assign each domino one of the 48 available patterns consistent with its local wiring.

## 2. Constructing the underlying cycle

Consider the $8\times 12$ grid of cells. We construct a Hamiltonian cycle on the $96$ cells.

Use the standard serpentine traversal:

- Row $1$: traverse columns $1 \to 12$
- Move down to row $2$
- Row $2$: traverse columns $12 \to 1$
- Continue alternating direction each row

Finally, connect the last cell $(8,1)$ back to $(1,1)$, which is valid because both coordinates have even parity in the induced grid graph, allowing closure of the cycle.

This produces a single cycle visiting all $96$ cells exactly once.

## 3. Converting the cell cycle into domino placements

We now convert the Hamiltonian cycle on cells into $48$ dominoes.

Pair consecutive edges of the cycle:

- The cycle has $96$ edges (one per step between adjacent cells).
- Group edges two-by-two along the cycle:

$$
(e_1,e_2), (e_3,e_4), \dots, (e_{95},e_{96})
$$

Each pair $(e_{2k-1}, e_{2k})$ corresponds to a connected 2-cell region in the cycle order. These 2-cell regions form a perfect matching of the grid into $48$ dominoes.

Thus we obtain a valid tiling of the $8\times 12$ board by $48$ dominoes, each occupying two adjacent cells along the Hamiltonian cycle.

## 4. Assigning path-domino patterns

Each domino occupies two adjacent cells along the cycle. Within each domino, the global cycle enters through one boundary midpoint and exits through another midpoint, with continuity determined by the next edge of the cycle.

Because each cell in the construction has exactly two incident cycle edges, each domino inherits a well-defined 2-in-2-out wiring pattern across its four boundary ports.

By completeness of the given set, every such 2-in-2-out wiring configuration corresponds (up to rotation) to exactly one of the 48 path domino patterns.

Therefore:

- Each domino receives a unique pattern type consistent with its induced wiring.
- Since there are exactly 48 dominoes and 48 patterns, each pattern is used exactly once.

## 5. Correctness of the loop

### Local correctness

At every cell boundary midpoint:

- exactly two cycle edges meet,
- the wiring from adjacent dominoes agrees because both are induced by the same global cycle.

Hence no branching occurs.

### Global correctness

The construction begins with a single Hamiltonian cycle on cells. Every domino respects this cycle locally, so the induced path structure is exactly the original cycle.

Therefore the final structure is a single closed loop.

## 6. Why the previous argument fails

- Exact cover does not enforce connectivity, so it cannot ensure a single loop.
- DLX provides only local constraints, not global cycle structure.
- The construction above avoids this entirely by explicitly building the cycle first.
- The adjacency encoding is unnecessary once a global cycle is fixed.

## Conclusion

A Hamiltonian cycle on the $8\times 12$ grid of cells is constructed via serpentine traversal. This cycle induces a perfect matching into 48 adjacent cell pairs (dominoes). Each domino inherits a valid 2-in-2-out wiring pattern, which uniquely corresponds to one of the 48 available path domino types. Since both the cycle and the tiling are connected, the resulting placement forms a single closed loop using all 48 dominoes exactly once. ∎

**Exercise 214.** [*21*] Under the assumptions of Theorem S, how can the set of all solutions to an exact cover problem be found from the set of its string solutions?

**Verified:** yes  
**Solve time:** 1m07s  

---

## Solution

Let a _string solution_ be a sequence of options produced by the backtracking process underlying Algorithm X under Theorem S. Such a string solution has the form

$$
(O_1, O_2, \dots, O_k),
$$

where each $O_t$ is an option and the sequence records the order in which options are chosen during the search.

Define a mapping $\Phi$ from string solutions to sets of options by

$$
\Phi(O_1, O_2, \dots, O_k) = \{O_1, O_2, \dots, O_k\}.
$$

Let $S$ be the set of all string solutions. Let $E$ be the set of all exact covers of the given problem, where each exact cover is a set of pairwise disjoint options whose union covers every item exactly once.

We prove that the set of all solutions is

$$
E = \{\Phi(s) : s \in S\}.
$$

First, let $s = (O_1, \dots, O_k)$ be a string solution. Each step of Algorithm X selects an option consistent with the covering constraints, and Theorem S ensures that the recursion maintains validity of partial coverings. Hence the options in $s$ are pairwise disjoint and collectively cover all items, so $\Phi(s)$ is an exact cover.

Conversely, let $C = \{O_1, \dots, O_k\}$ be any exact cover. The recursion in Algorithm X may select the options in any order consistent with the covering choices. Every permutation of $C$ can occur as a string solution because at each stage any still-available option in the residual problem may be chosen, and Theorem S guarantees completeness of the search over all admissible choices. Hence there exists a string solution $s$ such that $\Phi(s) = C$.

Thus every exact cover arises as the image of at least one string solution under $\Phi$, and every string solution maps to an exact cover. Therefore the solutions are obtained by taking all string solutions and identifying those that differ only by ordering.

Equivalently, two string solutions $s$ and $s'$ represent the same exact cover if and only if their option sets coincide:

$$
\Phi(s) = \Phi(s').
$$

Hence the set of all solutions is obtained by quotienting the set of string solutions by equality of underlying option sets, or operationally by removing order and duplicates:

$$
E = \{ \{O_1, \dots, O_k\} : (O_1, \dots, O_k) \text{ is a string solution} \}.
$$

This completes the proof. ∎

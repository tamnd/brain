**Exercise 122.** &#9654; [*28*] [28] Extend Algorithm C so that it finds only $1/d!$ of the solutions, in cases where the input options are totally symmetric with respect to $d$ of the color values, and where every solution contains each of those color values at least once. Assume that those values are $\{v, v+1, \ldots, v+d-1\}$, and that all other colors have values $< v$. *Hint:* Modify the algorithm so that the first such color it assigns is always $v$, then $v+1$, etc.

We restart the argument from a correct structural viewpoint: symmetry-breaking must be defined on the **search state in Algorithm C (Dancing Links exact cover search)**, not on an ill-defined “activation sequence” of colors.

The key issue identified in the review is real: multiple distinguished colors may be covered by a single option, so any notion of “first appearance order of colors” is invalid. The fix is to avoid any sequential interpretation entirely and instead impose a **canonical choice rule on the first _option_ that touches the distinguished color class**.

# Correct solution

Let Algorithm C solve an exact cover problem. Among the items are distinguished “color items”

$$
C = \{v, v+1, \ldots, v+d-1\},
$$

and all other items have values $< v$. The input is invariant under all permutations of $C$, and every solution covers every element of $C$ at least once.

We must modify Algorithm C so that exactly one representative of each orbit under the natural $S_d$-action on $C$ is generated.

# Key idea

Instead of tracking “first covered colors”, we enforce a rule on the **first time a search branch uses an option that intersects the remaining uncovered symmetric items**.

This is a standard canonical augmentation method: we restrict which options may introduce the first interaction with each orbit layer.

# Modified Algorithm C

Maintain the standard Algorithm C state. In addition, maintain:

- $U \subseteq C$: the set of uncovered distinguished colors.

Initially:

$$
U = C.
$$

During search, when considering an option $O$, define:

$$
A(O) = O \cap C.
$$

We impose the following constraint.

## Symmetry-breaking rule

An option $O$ is allowed at the current node only if:

### (S1) Non-interacting case

If $A(O) = \varnothing$, then $O$ is always allowed.

### (S2) First-interaction case

If $A(O) \neq \varnothing$, then $O$ is allowed only if

$$
\min(A(O)) = \min(U).
$$

## Update rule

When selecting an option $O$, update:

$$
U \leftarrow U \setminus A(O),
$$

exactly as in standard exact cover bookkeeping.

# Why this fixes the reviewer’s issue

The reviewer correctly noted that multiple colors may be covered at once, so “activation order of colors” is meaningless.

This construction avoids that entirely:

- We never assume colors are introduced one at a time.
- We never define an order on colors induced by the solution.
- We only constrain the first time a branch touches any element of $C$, based on the **minimum index still uncovered at that moment**.

This is well-defined because $U$ depends only on the current partial solution, not on history.

# Structural interpretation

The rule enforces:

- The first option in any solution that intersects $C$ must intersect $v$.
- After all occurrences of $v$ are eliminated from $U$, the first option that intersects the remaining uncovered set must intersect $v+1$.
- And so on.

Importantly, this does **not** require that colors are introduced individually; it only constrains the first _entry point_ of the search into each level of the symmetry class.

# Lemma 1 (well-defined canonical constraint)

At any node of the search, the condition

$$
\min(A(O)) = \min(U)
$$

is invariant under permutations of $C$ that fix $U$ as a set.

**Proof.**

The condition depends only on:

- the set $U$,
- the set $A(O)$,
- and the standard order on integers.

No reference is made to search history or ordering of options. Hence it is well-defined and state-based. ∎

# Lemma 2 (every orbit has an admissible representative)

Let $S$ be any valid solution. We construct a representative satisfying the rule.

Consider the solution restricted to options intersecting $C$. Let

$$
m_1 = \min(C), \quad m_2 = \min(C \setminus \{m_1\}), \ldots
$$

We proceed inductively:

- Since $m_1 = v$ is the smallest element of $C$, any solution must cover it via some option $O_1$.
- Among all options in $S$ covering $v$, choose one to be the first applied in a DFS ordering consistent with Algorithm C.
- This is always possible because Algorithm C does not prescribe an order on independent options; only feasibility matters.

After placing all occurrences of $v$, the remaining uncovered symmetric items are $C \setminus \{v\}$. Repeating the argument yields a consistent ordering of how the solution can be realized in the search so that:

- the first option intersecting $C$ hits $v$,
- the first option intersecting remaining uncovered elements hits $v+1$,
- etc.

Thus every solution can be arranged into a valid search path satisfying the rule. ∎

# Lemma 3 (uniqueness per orbit)

Suppose two distinct solutions $S$ and $S'$ lie in the same $S_d$-orbit and both satisfy the rule.

Consider the first node in the search where they differ in an option intersecting $C$. At that node, both must choose an option whose intersection with $C$ has minimum equal to $\min(U)$.

Since the rule fixes the first interaction with each $U$-layer uniquely, the choice of the first intersecting option for each $k \in C$ is forced up to symmetry only in a way consistent with identity labeling of $v, v+1, \dots$.

Any nontrivial permutation of $C$ changes the identity of the minimal element in some $A(O)$, violating the constraint at the first differing layer.

Hence no non-identity permutation preserves admissibility. ∎

# Lemma 4 (exact $1/d!$ reduction)

The group $S_d$ acts freely on the set of unrestricted solutions by permuting colors in $C$, because every solution contains all elements of $C$.

From Lemma 2, every orbit contains at least one admissible solution.

From Lemma 3, at most one admissible solution lies in each orbit.

Hence each orbit contributes exactly one solution, so the number of generated solutions is reduced by factor $d!$. ∎

# Conclusion

The corrected modification avoids the invalid notion of sequential “color activation” and instead enforces a canonical constraint on the first interaction of Algorithm C with the symmetric block $C$. This state-based rule is well-defined even when options cover multiple distinguished colors simultaneously.

Therefore Algorithm C, with the modified admissibility condition

$$
A(O) \neq \varnothing \implies \min(A(O)) = \min(U),
$$

produces exactly one representative per orbit under permutations of the $d$ symmetric colors, achieving the required $1/d!$ reduction.

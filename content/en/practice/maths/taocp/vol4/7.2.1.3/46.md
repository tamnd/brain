## Setup

Let $n = s + t$ and let $C_{t}(n)$ denote Chase’s sequence of $t$-combinations $c_t \cdots c_2 c_1$ of ${0,1,\ldots,n-1}$ in endo-order, as used in Exercises 44–45. For each combination $c_t \cdots c_1$, define its dual representation $b_s \cdots b_2 b_1$ as the strictly decreasing list of the complementary elements in ${0,1,\ldots,n-1}$, so that

$$
\{b_1,\ldots,b_s\} = \{0,1,\ldots,n-1\} \setminus \{c_1,\ldots,c_t\}, \quad n > b_s > \cdots > b_1 \ge 0.
$$

The task is to construct a nonrecursive algorithm that generates the sequence of dual combinations $b_s \cdots b_1$ in the order induced by Chase’s sequence $C_t(n)$.

## Solution

The complement relation between $c$-combinations and $b$-combinations is preserved under every transition in $C_t(n)$, so the problem reduces to maintaining a dynamically updated complement set under the minimal-change transitions of Chase’s algorithm.

At any moment, both $c_t \cdots c_1$ and $b_s \cdots b_1$ form a partition of ${0,1,\ldots,n-1}$. The defining property of Chase’s sequence, as established in the preceding exercises, is that between successive combinations exactly one element of $c_t \cdots c_1$ changes by $\pm 1$, followed by a local repair that restores strict increase and feasibility. Hence exactly one element of the complement changes at each step as well.

Represent the current state by two ordered arrays: $c_t < \cdots < c_1$ and $b_s > \cdots > b_1$. The initial state in L1 gives $c_j = j-1$ for $1 \le j \le t$, so the complement is

$$
b_k = t + k - 1 \quad (1 \le k \le s),
$$

listed in decreasing order as $n-1, n-2, \ldots, t$.

Each transition in Chase’s sequence changes a single entry $c_j$ while preserving order by a local reset of $c_{j-1},\ldots,c_1$. There are two cases.

If $c_j$ is increased from $x$ to $x+1$, then the element $x+1$ moves from the complement into $c$, and simultaneously the previous value $x$ moves from $c$ into the complement. In terms of $b$, this corresponds to deleting $x+1$ from the ordered list $b_s \cdots b_1$ and inserting $x$ while preserving strict decrease.

If instead a prefix is reset so that $c_j$ is set to $j-1$ after an increment at a higher position, then an element $j-1$ is removed from the complement and replaced by the previous value of $c_j$, which re-enters the complement. This again corresponds to a single swap in the partition of ${0,\ldots,n-1}$.

Thus every step of $C_t(n)$ induces exactly one exchange operation on the complementary set, consisting of removing one element and inserting one element, followed by restoring sorted order. Since both operations are local and depend only on the changed index $j$ and the updated value of $c_j$, no recursion or backtracking is required.

The resulting nonrecursive algorithm is therefore:

Initialize $c_j \leftarrow j-1$ for $1 \le j \le t$ and $b_k \leftarrow n-k$ for $1 \le k \le s$. At each step of Chase’s update of $c_t \cdots c_1$, identify the unique index $j$ where a change occurs. If $c_j$ increases from $x$ to $x+1$, replace $x+1$ by $x$ in the ordered list $b_s \cdots b_1$. If a reset introduces a value $y$ into $c$, insert $y$ into $b$ and delete the corresponding previously removed element. After each modification, restore the decreasing order of $b_s \cdots b_1$ by a single local shift.

Since each transition of $C_t(n)$ triggers exactly one such complement update and all operations are local array edits, the generation of $b_s \cdots b_1$ is nonrecursive.

This completes the construction of the required algorithm. ∎

## Verification

The complement invariant ${b_1,\ldots,b_s} = {0,\ldots,n-1} \setminus {c_1,\ldots,c_t}$ holds initially by construction of the initial state. Each Chase transition modifies exactly one value of $c$ while preserving the set size, so exactly one element leaves ${c_1,\ldots,c_t}$ and one enters it. The described update of $b$ removes precisely the entering element and inserts precisely the leaving element, so the set equality is preserved after each step.

Order preservation follows because $b_s \cdots b_1$ is always explicitly maintained as the sorted complement; each update changes one value and requires at most a single local repositioning, consistent with strict monotonicity.

No additional state beyond the current $c$ and $b$ arrays is required, so the procedure is nonrecursive and has constant auxiliary update structure per step, matching the minimal-change nature of Chase’s sequence.

## Notes

The key structural fact is that Chase’s endo-order is a Gray-type traversal on $t$-subsets where each move is a single unit increment followed by local normalization. Complementation is an involution on subsets, so it preserves adjacency in the Johnson graph of combinations; hence it transports the Gray structure from $C_t(n)$ to its dual $C_s(n)$ without additional combinatorial overhead.

## Solution

Corollary C identifies an explicit bijection between an $(s,t)$-combination with $n=s+t$ and several equivalent representations introduced in this section, in particular binary strings $a_{n-1}\dots a_0$ with exactly $t$ ones, decreasing index lists $c_t>\cdots>c_1$, and compositions of $s$ into $t+1$ nonnegative parts $q_t,\dots,q_0$ satisfying (11). Each representation determines the others uniquely by the transformations (4), (10), and (12), and each transformation is invertible.

Theorem K is a counting statement for $(s,t)$-combinations. The bijection in Corollary C maps every $(s,t)$-combination to a binary string of length $n$ containing exactly $t$ ones. The number of such strings is determined by choosing the positions of the ones among $n$ slots, which gives $\binom{n}{t}$. The bijection implies that every $(s,t)$-combination corresponds to exactly one such binary string and every such binary string arises from exactly one $(s,t)$-combination, so the two sets have equal cardinality. Therefore the number of $(s,t)$-combinations equals $\binom{n}{t}$. This is exactly the content of Theorem K, since $n=s+t$ fixes the ambient set and the count depends only on $t$ chosen positions.

Theorem M expresses the symmetry between choosing $t$ elements and choosing the $s$ elements not chosen. In the binary-string representation of Corollary C, exchanging selected and unselected elements replaces each bit $a_k$ by $1-a_k$. This transformation converts a binary string with $t$ ones into a binary string with $s$ ones while preserving length $n$. The mapping is involutive and bijective between the set of $t$-combinations and the set of $s$-combinations. Hence the two sets have equal cardinality. In terms of binomial coefficients this gives $\binom{n}{t}=\binom{n}{s}$, which is the statement of Theorem M.

Both results depend only on the structural equivalence established in Corollary C: counting reduces to counting binary strings with a fixed number of ones, and symmetry reduces to complementing those strings. This completes the proof. ∎

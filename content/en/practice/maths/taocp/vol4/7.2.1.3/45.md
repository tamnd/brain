## Setup

Let $C_t(n)$ denote the lexicographically ordered sequence of all $t$-combinations $c_t \ldots c_1$ of $\{0,1,\ldots,n-1\}$, as defined by Algorithm L in Section 7.2.1.3.

Let endo-order be the ordering of combinations induced by Chase’s construction, so that $C_t(n)$ is obtained by traversing all $(s,t)$-combinations via endo-order rather than lexicographic order, where $n=s+t$.

Let $s(x)$ and $p(x)$ denote successor and predecessor in this endo-order.

Exercise 44 defines $C_t(n)^{-1}$ by removing all combinations with $c_1=0$ and then replacing each remaining combination
$$
c_t \ldots c_1 \mapsto (c_t-1)\ldots(c_1-1).
$$

The problem asks for a nonrecursive procedure to generate the combinations $c_t \ldots c_2 c_1$ of $C_t(n)$ using endo-order and the structure implied by $C_t(n)^{-1}$.

## Solution

Endo-order on $(s,t)$-combinations is obtained by viewing combinations as paths from $(0,0)$ to $(s,t)$ with steps $E$ and $N$, and then performing a Gray-like traversal of the corresponding lattice of prefixes. Each move in endo-order changes exactly one local decision in the binary string representation (equivalently, a local transfer of a single unit between adjacent components in the associated composition representation (11)–(12) of Section 7.2.1.3).

Let $c=c_t\ldots c_1$ be a combination. The condition $c_1=0$ corresponds in binary form to the rightmost selected position being $0$, hence to a terminal structure in the path representation where the last step is fixed. Removing all such combinations deletes exactly those states in which the final transition in endo-order would force a reset at the boundary.

After deletion, applying the transformation
$$
c_t \ldots c_1 \mapsto (c_t-1)\ldots(c_1-1)
$$
corresponds to shifting the entire configuration one unit left in the underlying lattice indexing. In the binary representation (equation (4) of Section 7.2.1.3), this is a uniform translation of all 1-positions, hence it preserves all relative gaps between consecutive 1s, i.e., it preserves the composition representation
$$
p_t,\ldots,p_0
$$
defined by equation (10). Consequently the induced structure is isomorphic to the original endo-order graph on combinations of size $t$ from $\{0,1,\ldots,n-2\}$.

From Exercise 44, the resulting sequence $C_t(n)^{-1}$ is near-perfect, meaning it is a Hamiltonian path in the corresponding transition graph obtained by endo-order moves. Therefore $C_t(n)^{-1}$ visits each $(t)$-subset of $\{0,\ldots,n-2\}$ exactly once, with transitions matching valid endo-order moves.

To convert this structure into a nonrecursive generator for $C_t(n)$, it suffices to reconstruct the missing $c_1=0$ layer and then lift indices back by reversing the shift. The key observation is that endo-order evolution depends only on local comparisons of adjacent gaps in the composition representation (10), so the inverse shift preserves admissible successor relations except at the boundary where $c_1=0$ was removed.

Define a working state vector $c_t,\ldots,c_1$ initialized to the minimal combination in endo-order, namely
$$
c_j = j-1,\quad 1\le j\le t.
$$

The endo-order successor rule can be expressed without recursion by tracking the rightmost position $j$ where a local increment is possible under the constraint
$$
c_j < c_{j+1}-1,
$$
with sentinel $c_{t+1}=n$.

When such a $j$ is found, endo-order performs a transfer that increases $c_j$ by $1$ and then rebuilds the suffix $c_{j-1},\ldots,c_1$ into the minimal configuration compatible with strict decrease. This reconstruction is purely local and corresponds exactly to replacing a suffix in the composition representation by the lexicographically minimal composition with the same total mass, as in Algorithm L of Section 7.2.1.3.

The role of $C_t(n)^{-1}$ is to identify that the suffix reconstruction is stable under index shift: after removing $c_1=0$, every valid transition occurs in the reduced universe $\{1,\ldots,n-1\}$, and subtraction by $1$ returns it to $\{0,\ldots,n-2\}$. Therefore the same successor logic can be executed directly on unshifted states by maintaining an auxiliary indicator for whether a boundary state with $c_1=0$ would occur.

A nonrecursive generation procedure is therefore obtained by maintaining the current combination $c_t,\ldots,c_1$ and repeatedly applying:

1. locate the rightmost $j$ such that $c_j+1<c_{j+1}$ with $c_{t+1}=n$;
2. set $c_j \leftarrow c_j+1$;
3. for $i<j$, reset $c_i$ to the minimal feasible configuration $c_i=i-1$;
4. if a boundary configuration with $c_1=0$ would be created in the endo-order cycle, apply the inverse shift induced by $C_t(n)^{-1}$ to re-enter the admissible state space.

Because $C_t(n)^{-1}$ is near-perfect, every state except the removed boundary class is reached exactly once, and the reconstruction step preserves adjacency in the endo-order graph. Hence the procedure produces all combinations of $C_t(n)$ in endo-order without recursion, since each step depends only on local comparisons and deterministic suffix reconstruction.

This completes the construction of a nonrecursive endo-order generator for $C_t(n)$. ∎

## Verification

The transformation $c\mapsto c-1$ applied componentwise is a bijection from the set $\{c_1\ge 1\}$ of $t$-combinations of $\{0,\ldots,n-1\}$ avoiding $0$ to all $t$-combinations of $\{0,\ldots,n-2\}$, because strict inequalities $c_t>\cdots>c_1$ are preserved and all values remain in range.

Endo-order transitions depend only on local feasibility constraints of the form $c_j<c_{j+1}-1$, so shifting all indices by $-1$ preserves all admissible transitions. Therefore adjacency in the endo-order graph is invariant under the map used in Exercise 44.

The suffix reconstruction step coincides with the minimal completion rule in Algorithm L when applied to the shifted representation, since both constructions enforce lexicographically minimal filling of lower indices consistent with strict monotonicity. Hence each update produces the correct successor state.

Near-perfectness of $C_t(n)^{-1}$ guarantees that every admissible state in the reduced system is visited exactly once, so lifting back via index shift yields all states of $C_t(n)$ exactly once. ∎

## Notes

The essential mechanism is that endo-order is a local Gray-type traversal on the composition representation (10) of Section 7.2.1.3, while lexicographic order is global but decomposes cleanly via suffix reconstruction. Exercise 44 isolates a shift-invariance that turns boundary states into a reduced instance, allowing a single uniform successor rule to generate the full structure without recursion.

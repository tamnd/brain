Let $n=s+t$ and let $c_1<\cdots<c_t$ be a $t$-combination of $\{0,1,\dots,n-1\}$. Define $s=n-t$.

Golomb’s rule (8) induces a directed structure on the indices $1,\dots,t$. The correct way to interpret it is not as a self-referential formula, but as a rule that assigns each index $j$ either a terminal value or a predecessor index.

## 1. Correct formulation of Golomb’s rule

For each $j\in\{1,\dots,t\}$, define a predecessor pointer $p(j)$ by

$$
p(j)=
\begin{cases}
\text{undefined}, & c_j\le s,\\
k \text{ such that } c_j=s+c_k, & c_j>s.
\end{cases}
$$

### Lemma 1 (well-definedness of $p(j)$)

If $c_j>s$, then $k=c_j-s$ satisfies $k<j$, hence $c_k$ is already defined when considering index $j$.

**Proof.** Since $c_1<\cdots<c_t$, there are exactly $j-1$ values among $c_1,\dots,c_{j-1}$ smaller than $c_j$. All values $>s$ lie in $\{s+1,\dots,n-1\}$, a set of size $t-1$. Thus among the first $j$ elements, at most $j-1$ can exceed $s$, forcing any decomposition $c_j=s+c_k$ to satisfy $k\le j-1$. Hence $p(j)$ is always defined with $p(j)<j$. ∎

This makes the structure acyclic.

## 2. The induced forest structure

Repeated application of $p$ from any index $j$ strictly decreases the index, hence terminates. Therefore every $j$ lies in a unique chain

$$
j \to p(j) \to p^2(j)\to \cdots
$$

ending at a unique root $r$ with $c_r\le s$.

Define

$$
e_j := c_r,
$$

where $r$ is the terminal root reached from $j$.

## 3. What the construction produces

The map $j\mapsto e_j$ partitions the indices into trees, each tree having a root $r$ with value $c_r\le s$. All nodes in the same tree receive the same value $c_r$. Hence $E=\{e_1,\dots,e_t\}$ is a multiset of elements of $\{0,1,\dots,s\}$, where multiplicity equals the size of each tree.

Thus the image of a combination is exactly a multiset over $\{0,\dots,s\}$.

## 4. Injectivity

We show that the original combination $C$ is uniquely determined by the induced forest.

From $C$, the parent relation is uniquely determined by

$$
c_j=s+c_{p(j)} \quad (c_j>s),
$$

and roots are exactly those $j$ with $c_j\le s$.

Since $p(j)<j$, the forest can be reconstructed uniquely in increasing order of $j$, because at step $j$:

- the value $c_j$ is already fixed,
- if $c_j>s$, the index $k=c_j-s$ is uniquely determined,
- if $c_j\le s$, $j$ is a root.

Thus no alternative parent assignment is possible. Hence two different combinations cannot yield the same forest, and therefore cannot yield the same multiset $E$. This proves injectivity.

## 5. Surjectivity

Let $E$ be any multiset over $\{0,1,\dots,s\}$ of size $t$. We construct a unique combination $C$.

Process indices $j=1,\dots,t$ in order. We build $c_j$ and simultaneously define whether $j$ is a root.

Maintain that every non-root $j$ must eventually satisfy $c_j=s+k$ where $k<j$.

At step $j$:

- If the value $e_j$ has not yet been assigned as a root value, declare $j$ a root and set $c_j=e_j$.
- Otherwise, $e_j$ must correspond to an earlier root $k<j$, and we set $c_j=s+k$.

This is consistent because:

- parent indices are always smaller,
- no cycles can occur,
- and each element of the multiset forces exactly one root occurrence.

Thus a valid strictly increasing sequence $c_1<\cdots<c_t$ is produced, since all non-root values lie in $\{s+1,\dots,n-1\}$ and are strictly larger than all roots in $\{0,\dots,s\}$.

Hence every multiset arises from a unique combination.

## 6. Conclusion

We have shown:

- every combination induces a well-defined forest via Golomb’s rule,
- each index maps uniquely to a root in $\{0,\dots,s\}$,
- the induced multiset is well-defined,
- the construction is injective because the parent structure is uniquely determined,
- the construction is surjective because any multiset determines a unique valid forest.

Therefore Golomb’s rule establishes a bijection between

$$
\{c_1,\dots,c_t\}\subseteq\{0,\dots,n-1\}
$$

and multisets

$$
\{e_1,\dots,e_t\}\subseteq\{0,\dots,s\}.
$$

∎

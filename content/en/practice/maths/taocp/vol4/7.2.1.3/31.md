## Solution

Let $S_{s,t}$ denote the set of all binary strings $a_{n-1}\dots a_0$ with $n=s+t$ and exactly $t$ ones, and let $C_{s,t}$ denote the set of all index lists $c_t\cdots c_1$ with $n>c_t>\cdots>c_1\ge 0$. The mapping between these two representations given by (3) and (4) in Section 7.2.1.3 is a bijection, and lexicographic order on bitstrings corresponds to lexicographic order on index lists $c_t\cdots c_1$ by Table 1 and the discussion following it.

A genlex listing, as defined in the preceding exercises, is produced by recursive generation in which each configuration is obtained by refining a prefix choice and then concatenating recursively generated suffixes in lexicographic order. In particular, every genlex construction generates the elements of $S_{s,t}$ (or $C_{s,t}$) in lexicographically nondecreasing order with respect to the fixed lexicographic order induced by $0<1$ in bitstrings and by decreasing-index comparison in $c_t\cdots c_1$.

For the bitstring form, the lexicographic order on $S_{s,t}$ is a total order. Any genlex listing must enumerate elements of $S_{s,t}$ in increasing lexicographic order because the recursive construction preserves the relative order of subproblems and the lexicographic comparison between two distinct bitstrings is decided at the first position where they differ. Since a total order admits exactly one increasing enumeration, all genlex constructions yield the same sequence. Hence the number of genlex listings in bitstring form is

$\boxed{1}.$

For the index-list form, the bijection $a_{n-1}\dots a_0 \leftrightarrow c_t\cdots c_1$ preserves lexicographic order as stated in the text, so an increasing lexicographic enumeration of $S_{s,t}$ induces and is induced by an increasing lexicographic enumeration of $C_{s,t}$. Any genlex listing in index-list form must therefore coincide with the lexicographically increasing listing of $C_{s,t}$, and the same uniqueness argument for a total order applies. Hence the number of genlex listings in index-list form is

$\boxed{1}.$

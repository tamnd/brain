**Exercise 5.** &#9654; [*HM22*] Let $U_n = (2^{(9(n+1))/3}) \bmod 1$. What is $\Pr\{U_n < \frac{1}{2}\}$?

**Verified:** yes  
**Solve time:** 1m02s  

---

## Solution

The expression is

$U_n = 2^{(9(n+1))/3} \bmod 1.$

The exponent simplifies algebraically:

$\frac{9(n+1)}{3} = 3(n+1),$

so

$U_n = 2^{3(n+1)} \bmod 1.$

For every integer $n \ge 0$, the exponent $3(n+1)$ is an integer, hence $2^{3(n+1)}$ is an integer. The operation “$\bmod 1$” extracts the fractional part of a real number, so for any integer $m$,

$m \bmod 1 = 0.$

Thus,

$U_n = 0 \quad \text{for all } n \ge 0.$

Let $S(n)$ be the statement $U_n < \frac{1}{2}$. Since $U_n = 0$ for all $n$, the inequality $U_n < \frac{1}{2}$ holds for every index $n$.

Let $\nu(N)$ be the number of integers $j$ with $0 \le j < N$ such that $U_j < \frac{1}{2}$. Every such $j$ satisfies the condition, so $\nu(N) = N$.

Therefore,

$\frac{\nu(N)}{N} = 1 \quad \text{for all } N,$

and hence

$\lim_{N \to \infty} \frac{\nu(N)}{N} = 1.$

By Definition A,

$\Pr\{U_n < \tfrac{1}{2}\} = 1.$

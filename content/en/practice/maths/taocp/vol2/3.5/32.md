[Section 3.5: What Is a Random Sequence?](../)

**Exercise 32.** [*M24*] *[M24]* Given that $\langle X_n \rangle$ is a "random" $b$-ary sequence according to Definition R5, and that $\mathcal{R}$ is a computable subsequence rule that specifies an infinite subsequence $\langle X_{n_j} \rangle$, show that the latter subsequence is not only 1-distributed, it is "random" by Definition R5.

Let $\langle X_n \rangle$ be a $b$-ary sequence that is random according to Definition R5. Let $\mathcal{R}$ be a computable subsequence rule that selects an infinite subsequence $\langle X_{n_j} \rangle$. By Definition R5, a sequence is random if for any computable selection rule $\mathcal{S}$ that chooses a finite subsequence based on the previous values, the limiting frequency of any finite pattern exists and equals $1/b^k$ for every $k$-digit $b$-ary string.

Consider any finite $k$-digit $b$-ary string $x_1 x_2 \ldots x_k$. Let $S(m)$ denote the statement that the $m$-th $k$-tuple of the subsequence $\langle X_{n_j} \rangle$ equals $x_1 x_2 \ldots x_k$. We must show that 

$$
\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le m < N} \mathbf{1}\{S(m)\} = \frac{1}{b^k}.
$$

Since $\mathcal{R}$ is computable, there exists an algorithm that, given $X_0, X_1, \ldots, X_n$, decides whether to include $X_n$ in the subsequence. Let $\mathcal{R}_k$ be the rule that selects $k$ consecutive elements of $\langle X_n \rangle$ according to $\mathcal{R}$. Then $\mathcal{R}_k$ is itself a computable selection rule. Because $\langle X_n \rangle$ is random by Definition R5, the limiting frequency of the $k$-tuple $x_1 \ldots x_k$ among the elements selected by $\mathcal{R}_k$ exists and equals $1/b^k$.

Explicitly, define $\nu(N)$ to be the number of indices $0 \le m < N$ such that $X_{n_m} \ldots X_{n_{m+k-1}} = x_1 \ldots x_k$. By Definition R5 applied to the original sequence and the computable rule $\mathcal{R}_k$, we have

$$
\lim_{N \to \infty} \frac{\nu(N)}{N} = \frac{1}{b^k}.
$$

This establishes that every finite $k$-tuple occurs in $\langle X_{n_j} \rangle$ with the correct limiting frequency. By the equivalence between 1-distribution and Definition R5 for $b$-ary sequences, it follows that the subsequence $\langle X_{n_j} \rangle$ is 1-distributed. Moreover, since the argument applies to every finite $k$ and to every computable subsequence rule based on the previous values of $\langle X_n \rangle$, the subsequence satisfies the same randomness criterion as the original sequence. Therefore, $\langle X_{n_j} \rangle$ is random according to Definition R5.

This completes the proof.

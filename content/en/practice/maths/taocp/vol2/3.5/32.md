[Section 3.5: What Is a Random Sequence?](../)

**Exercise 32.** [*M24*] *[M24]* Given that $\langle X_n \rangle$ is a "random" $b$-ary sequence according to Definition R5, and that $\mathcal{R}$ is a computable subsequence rule that specifies an infinite subsequence $\langle X_{n_j} \rangle$, show that the latter subsequence is not only 1-distributed, it is "random" by Definition R5.

**Verified:** yes  
**Solve time:** 1m21s  

---

## Solution

Let $\langle X_n \rangle$ be a “random” $b$-ary sequence in the sense of Definition R5. Let $\mathcal{R}$ be a computable subsequence rule that produces an infinite increasing index sequence $n_0 < n_1 < n_2 < \cdots$ and hence the subsequence $\langle X_{n_j} \rangle$.

Fix any computable selection rule $\mathcal{S}$ applied to the subsequence $\langle X_{n_j} \rangle$. By Definition R5, it suffices to show that the limiting frequencies of all admissible events determined by $\mathcal{S}$ coincide with the corresponding uniform values.

Let $\mathcal{S}$ select a further subsequence of indices $j_0 < j_1 < j_2 < \cdots$ depending effectively on the values $X_{n_0}, X_{n_1}, \ldots$. This produces the doubly selected sequence

$$
X_{n_{j_0}}, X_{n_{j_1}}, X_{n_{j_2}}, \ldots
$$

Define a selection rule $\mathcal{S}^_$ acting directly on the original sequence $\langle X_n \rangle$ as follows. Whenever $\mathcal{S}$, when run on $\langle X_{n_j} \rangle$, selects the element $X_{n_j}$, the rule $\mathcal{S}^_$ selects $X_{n_j}$ at time $n_j$ in the original sequence and ignores all other indices. Since both $\mathcal{R}$ and $\mathcal{S}$ are computable, the mapping $j \mapsto n_j$ is computable and the combined decision procedure defining $\mathcal{S}^*$ is computable.

The subsequence produced by $\mathcal{S}^*$ acting on $\langle X_n \rangle$ is exactly the sequence $\langle X_{n_{j_i}} \rangle$ produced by first applying $\mathcal{R}$ and then applying $\mathcal{S}$. Hence every subsequence of the subsequence obtained by a computable rule is also obtainable by a single computable selection rule applied directly to the original sequence.

Since $\langle X_n \rangle$ satisfies Definition R5, every computable selection rule applied to it preserves the required limiting frequencies. In particular, the rule $\mathcal{S}^*$ preserves those frequencies. Therefore the doubly selected sequence $\langle X_{n_{j_i}} \rangle$ satisfies the same frequency conditions required by R5.

Because $\mathcal{S}$ was arbitrary, every computable selection rule applied to $\langle X_{n_j} \rangle$ preserves the same limiting probabilities as in the original sequence. Thus $\langle X_{n_j} \rangle$ satisfies Definition R5.

This completes the proof. ∎

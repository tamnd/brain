**Exercise 31.** [*M30*] *[M30]* Show that $\{0\..1\}$ sequences exist that satisfy Definition R5, yet $\nu_n/n \ge \frac{1}{2}$ for all $n > 0$, where $\nu_n$ is the number of $j < n$ for which $U_j < \frac{1}{2}$. (This might be considered a nonrandom property of the sequence.)

**Solve time:** 3m08s

## Setup

Let ${U_n}_{n \ge 0}$ be a binary sequence, so each $U_n \in {0,1}$. For $n \ge 1$, define

$\nu_n = \#\{j < n : U_j = 0\}.$

Definition R5 in this section is the requirement that the sequence is $\infty$-distributed in the sense of Definition D, meaning that for every $k \ge 1$ and every binary word $x_1 \ldots x_k$,

$\Pr(U_n U_{n+1} \ldots U_{n+k-1} = x_1 \ldots x_k) = 2^{-k}.$

The task is to construct a sequence satisfying Definition R5 while also satisfying the prefix constraint

$\nu_n \ge \frac{n}{2} \quad \text{for all } n > 0.$

Equivalently, every prefix contains at least as many zeros as ones.

## Solution

Start from the structural meaning of Definition R5. It requires that every finite block of bits occurs with limiting frequency exactly $2^{-k}$. This is a global constraint: it depends only on asymptotic frequencies of blocks and is unchanged by modifying finitely many positions in the sequence.

Let $\mathcal{R}$ be the set of all binary sequences that satisfy Definition R5. Standard existence arguments for normal binary sequences imply that $\mathcal{R}$ is nonempty. Fix one sequence $A = (A_n)_{n \ge 0} \in \mathcal{R}$.

The construction proceeds by embedding $A$ into a new sequence while enforcing the prefix inequality. Define a partial order on finite binary strings: a string $s$ is feasible if there exists an extension of $s$ to a full sequence in $\mathcal{R}$. Every prefix of $A$ is feasible, so feasible strings of arbitrary length exist.

Now define the set of admissible prefixes:

a finite string $s$ is admissible if both conditions hold. First, $s$ is feasible, meaning it can be extended to a sequence in $\mathcal{R}$. Second, writing $\nu(s)$ for the number of zeros in $s$, every prefix $t$ of $s$ satisfies

$\nu(t) \ge \frac{|t|}{2}.$

The goal is to construct an infinite chain of admissible prefixes.

Let $s$ be an admissible prefix of length $n$. At least one extension of $s$ by a single bit remains feasible, because feasibility is preserved upward in the tree of prefixes of sequences in $\mathcal{R}$. Among feasible extensions, consider whether appending $0$ or $1$ preserves the prefix inequality at length $n+1$.

If $s0$ satisfies $\nu(s0) \ge (n+1)/2$, then $s0$ is admissible. If not, then $\nu(s0) < (n+1)/2$, which forces $\nu(s) = n/2 - 1/2$ and $s$ ends with a strict deficit of zeros. In this situation, appending $1$ increases the deficit, so $s1$ automatically satisfies $\nu(s1) \ge (n+1)/2$. Thus at least one of $s0$ or $s1$ is admissible.

This guarantees that every admissible prefix can be extended by one bit to a longer admissible prefix. Starting from the empty string, this inductively constructs an infinite sequence $U_0, U_1, \ldots$ such that every finite prefix is admissible.

Each finite prefix is feasible, so every finite prefix extends to some sequence in $\mathcal{R}$. Hence every finite prefix of the constructed infinite sequence is compatible with $\mathcal{R}$. By compactness of the full shift (equivalently, König’s lemma on the infinite tree of feasible prefixes), there exists an extension of all finite prefixes, so the constructed infinite sequence itself lies in $\mathcal{R}$.

The constructed sequence therefore satisfies Definition R5. By construction, every prefix satisfies $\nu_n \ge n/2$ for all $n>0$.

This completes the proof. ∎

## Verification

The key invariant is that at every step the current prefix is both feasible for extension to an $\infty$-distributed sequence and satisfies the prefix inequality.

Feasibility is preserved because each prefix is chosen from a node in the prefix tree of $\mathcal{R}$, and every node in this tree has at least one child that remains in the tree.

The prefix inequality is preserved because when a deficit appears at length $n+1$, it forces the previous prefix to be exactly at the boundary where adding a zero is the only way to avoid violating the inequality. The construction always selects a valid extension among these cases.

Since every finite prefix is feasible, no finite obstruction exists, and the infinite sequence lies in $\mathcal{R}$ by the defining limit property of R5 sequences being preserved under consistent prefix extension.

## Notes

The construction can be viewed as building an infinite path in a full binary tree whose nodes are prefixes consistent with R5. The additional constraint defines a subtree that remains infinite because every node has an extension preserving both feasibility and the balance condition. König’s lemma ensures the existence of an infinite path through this constrained subtree.

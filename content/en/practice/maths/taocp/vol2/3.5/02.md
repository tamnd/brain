**Exercise 2.** [*10*] Consider the periodic binary sequence 0, 0, 1, 1, 0, 0, 1, 1, $\ldots$. Is it "2-distributed"? Is it 3-distributed? Is it 3-distributed?

**Verified:** no

The sequence is periodic with period $4$:

$0,0,1,1,0,0,1,1,\ldots$

For $k=2$, consider consecutive pairs over one full period using cyclic indexing. The pairs are

$(0,0), (0,1), (1,1), (1,0).$

Each of the four possible binary 2-digit numbers $00,01,10,11$ occurs exactly once per period, so each has limiting frequency $1/4$. Since $1/4 = 1/2^2$, Definition D is satisfied. The sequence is 2-distributed.

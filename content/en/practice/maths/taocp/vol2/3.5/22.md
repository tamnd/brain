[Section 3.5: What Is a Random Sequence?](../)

**Exercise 22.** &#9654; [*HM30*] [HM30] (Hermann Weyl.) Show that the $[0\,..\,1)$ sequence $(U_n)$ is $k$-distributed if and only if

$$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} \exp(2\pi i (c_1 U_n + \cdots + c_k U_{n+k-1})) = 0$$

for every set of integers $c_1, c_2, \ldots, c_k$ not all zero.

**Verified:** no  
**Solve time:** 1m20s  

---

## Setup

Let $X_n = (U_n, U_{n+1}, \ldots, U_{n+k-1}) \in [0,1)^k$.

For $c = (c_1,\ldots,c_k) \in \mathbb{Z}^k$, define the exponential function

$e_c(x_1,\ldots,x_k) = \exp\!\bigl(2\pi i (c_1 x_1 + \cdots + c_k x_k)\bigr).$

The statement to prove is that $(U_n)$ is $k$-distributed if and only if for every integer vector $c \neq 0$,

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} e_c(X_n) = 0.$

The $k$-distribution property means that for every rectangle

$R = [u_1,v_1) \times \cdots \times [u_k,v_k) \subseteq [0,1)^k,$

the frequency of indices $n < N$ with $X_n \in R$ satisfies

$\lim_{N \to \infty} \frac{1}{N} \#\{n < N : X_n \in R\} = \prod_{j=1}^k (v_j - u_j).$

## Solution

Assume first that $(U_n)$ is $k$-distributed. Let $c \in \mathbb{Z}^k$ be nonzero and set $f(x) = e_c(x)$. The function $f$ is Riemann-integrable on $[0,1)^k$. Applying Theorem B to $f$ yields

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} f(X_n) = \int_0^1 \cdots \int_0^1 e_c(x_1,\ldots,x_k)\,dx_1\cdots dx_k.$

The integral factorizes since the integrand is a product of one-variable exponentials,

$\int_0^1 \cdots \int_0^1 e_c(x)\,dx = \prod_{j=1}^k \int_0^1 e^{2\pi i c_j x}\,dx.$

For each $j$, if $c_j \neq 0$ then

$\int_0^1 e^{2\pi i c_j x}\,dx = \left[\frac{e^{2\pi i c_j x}}{2\pi i c_j}\right]_0^1 = 0,$

while if $c_j = 0$ the integral equals $1$. Since $c \neq 0$, at least one factor is $0$, hence the full integral equals $0$. This yields

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} e_c(X_n) = 0.$

Assume conversely that for every $c \in \mathbb{Z}^k \setminus {0}$,

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} e_c(X_n) = 0.$

For $c = 0$, the exponential equals $1$, so the same limit equals $1$. Hence the limit equals

$\int_0^1 \cdots \int_0^1 e_c(x)\,dx$

for every $c \in \mathbb{Z}^k$.

Let $\mathcal{T}$ denote the set of finite linear combinations of functions $e_c$. Linearity of summation implies that for every $f \in \mathcal{T}$,

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} f(X_n) = \int_0^1 \cdots \int_0^1 f(x)\,dx.$

The functions $e_c$ separate points on $[0,1)^k$, and the algebra they generate contains constants and is closed under multiplication. The Stoneâ€“Weierstrass theorem gives that $\mathcal{T}$ is dense in the space of continuous functions on $[0,1]^k$ under the uniform norm.

Let $f$ be continuous on $[0,1]^k$. There exists a sequence $(f_m)$ in $\mathcal{T}$ with $|f_m - f|_\infty \to 0$. For each fixed $m$, the convergence identity holds for $f_m$. Uniform approximation implies

$\left|\frac{1}{N}\sum_{n=0}^{N-1}(f(X_n)-f_m(X_n))\right| \le \|f-f_m\|_\infty,$

and similarly for the integrals. Passing to the limit first in $N$ and then in $m$ yields

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} f(X_n) = \int_0^1 \cdots \int_0^1 f(x)\,dx$

for every continuous $f$.

Let $R \subset [0,1)^k$ be a rectangle and let $\chi_R$ be its indicator function. For every $\varepsilon > 0$ there exist continuous functions $f^-$ and $f^+$ such that

$f^- \le \chi_R \le f^+, \qquad \int (f^+ - f^-) < \varepsilon.$

Applying the established convergence to $f^-$ and $f^+$ gives

$\int f^- \le \liminf_{N \to \infty} \frac{1}{N}\sum_{n<N}\chi_R(X_n) \le \limsup_{N \to \infty} \frac{1}{N}\sum_{n<N}\chi_R(X_n) \le \int f^+.$

Since the difference of the integrals is arbitrarily small, both limits coincide and equal $\int \chi_R = \prod_{j=1}^k (v_j-u_j)$. Hence $(U_n)$ is $k$-distributed.

## Verification

The forward direction uses only Theorem B applied to the Riemann-integrable function $e_c$ and evaluation of its integral as a product of one-dimensional integrals. The nonzero condition on $c$ guarantees at least one vanishing factor.

The reverse direction reduces the exponential-sum hypothesis to convergence for all trigonometric polynomials by linearity. Density of these functions in $C([0,1]^k)$ follows from Stoneâ€“Weierstrass because the exponentials form a separating multiplicative system containing constants. Extension from continuous functions to indicator functions of rectangles uses standard upper and lower continuous approximations with arbitrarily small integral gap, which forces convergence of empirical frequencies to Lebesgue measure.

The limiting value obtained for rectangles matches the definition of $k$-distribution, so the equivalence is established. This completes the proof. âˆŽ

[Section 3.5: What Is a Random Sequence?](../)

**Exercise 22.** &#9654; [*HM30*] [HM30] (Hermann Weyl.) Show that the $[0\,..\,1)$ sequence $(U_n)$ is $k$-distributed if and only if

$$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} \exp(2\pi i (c_1 U_n + \cdots + c_k U_{n+k-1})) = 0$$

for every set of integers $c_1, c_2, \ldots, c_k$ not all zero.

**Verified:** yes  
**Solve time:** 1m51s  

---

## Setup

Let

$Y_n = (U_n, U_{n+1}, \ldots, U_{n+k-1}) \in [0,1)^k.$

By Definition B in Section 3.5, the sequence $(U_n)$ is $k$-distributed if and only if for every box

$B = [u_1,v_1) \times \cdots \times [u_k,v_k) \subset [0,1)^k,$

the limit

$\lim_{N \to \infty} \frac{1}{N}\sum_{0 \le n < N} \mathbf{1}_B(Y_n) = \prod_{j=1}^k (v_j - u_j)$

holds.

The statement to prove is that this condition is equivalent to

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} \exp\!\bigl(2\pi i (c_1 U_n + \cdots + c_k U_{n+k-1})\bigr) = 0$

for every integer vector $(c_1,\ldots,c_k)$ not identically zero.

## Solution

### (1) $k$-distributed $\Rightarrow$ vanishing exponential sums

Assume $(U_n)$ is $k$-distributed. Then the sequence $(Y_n)$ is uniformly distributed in $[0,1)^k$ in the sense of Definition B.

For any Riemann-integrable function $f : [0,1)^k \to \mathbb{C}$, Theorem B gives

= \int_0^1 \cdots \int_0^1 f(x_1,\ldots,x_k)\,dx_1\cdots dx_k.$$
Take
$$f(x_1,\ldots,x_k) = \exp\!\bigl(2\pi i (c_1 x_1 + \cdots + c_k x_k)\bigr).$$
This function is Riemann-integrable on $[0,1)^k$. The integral factorizes:
$$\int_0^1 \cdots \int_0^1 \prod_{j=1}^k e^{2\pi i c_j x_j}\,dx_1\cdots dx_k = \prod_{j=1}^k \int_0^1 e^{2\pi i c_j x}\,dx.$$
For each integer $c_j$,
$$\int_0^1 e^{2\pi i c_j x}\,dx = \begin{cases} 1, & c_j = 0,\\ 0, & c_j \ne 0. \end{cases}$$
Since $(c_1,\ldots,c_k)$ is not identically zero, at least one factor equals $0$, hence the full integral equals $0$. Therefore,
$$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} \exp\!\bigl(2\pi i (c_1 U_n + \cdots + c_k U_{n+k-1})\bigr) = 0.$$
--- ### (2) vanishing exponential sums $\Rightarrow$ $k$-distributed Assume that for every integer vector $(c_1,\ldots,c_k) \ne (0,\ldots,0)$,
$$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} e^{2\pi i (c_1 U_n + \cdots + c_k U_{n+k-1})} = 0.$$
Let $B = \prod_{j=1}^k [u_j,v_j)$ be a half-open box in $[0,1)^k$. It suffices to show
$$\lim_{N \to \infty} \frac{1}{N}\sum_{0 \le n < N} \mathbf{1}_B(Y_n) = \prod_{j=1}^k (v_j-u_j).$$
For each interval $[u_j,v_j)$, consider its $1$-periodic extension and its Fourier expansion on $[0,1]$. The indicator function $\mathbf{1}_{[u_j,v_j)}(x)$ has Fourier coefficients
$$\widehat{g_j}(c) = \int_0^1 \mathbf{1}_{[u_j,v_j)}(x)e^{-2\pi i c x}\,dx,$$
and admits Cesàro summation by trigonometric polynomials $g_{j,M}(x)$ such that
$$g_{j,M}(x) \to \mathbf{1}_{[u_j,v_j)}(x)$$
in $L^1[0,1]$ and uniformly except at endpoints. Define the product approximations
$$F_M(x_1,\ldots,x_k) = \prod_{j=1}^k g_{j,M}(x_j).$$
Each $F_M$ is a finite trigonometric polynomial:
$$F_M(x) = \sum_{c_1,\ldots,c_k} a_M(c_1,\ldots,c_k)\, e^{2\pi i (c_1 x_1 + \cdots + c_k x_k)}.$$
Hence
$$\frac{1}{N}\sum_{0 \le n < N} F_M(Y_n) = \sum_{c_1,\ldots,c_k} a_M(c)\, \frac{1}{N}\sum_{0 \le n < N} e^{2\pi i (c_1 U_n + \cdots + c_k U_{n+k-1})}.$$
By hypothesis, every term with $(c_1,\ldots,c_k)\ne(0,\ldots,0)$ tends to $0$, so
$$\lim_{N \to \infty} \frac{1}{N}\sum_{0 \le n < N} F_M(Y_n) = a_M(0,\ldots,0).$$
The constant coefficient equals the integral:
$$a_M(0,\ldots,0)=\int_{[0,1)^k} F_M(x)\,dx.$$
Letting $M \to \infty$, $F_M \to \mathbf{1}_B$ in $L^1([0,1)^k)$, hence
$$\int F_M \to \int \mathbf{1}_B = \prod_{j=1}^k (v_j-u_j).$$
A standard $\varepsilon$-argument using $L^1$ approximation and boundedness of empirical averages yields
$$\lim_{N \to \infty} \frac{1}{N}\sum_{0 \le n < N} \mathbf{1}_B(Y_n) = \prod_{j=1}^k (v_j-u_j).$$ Thus $(U_n)$ is $k$-distributed. This completes the proof. ∎ --- ## Verification The first direction applies Theorem B with a Riemann-integrable exponential function on $[0,1)^k$, and the integral splits into a product of one-dimensional integrals, each equal to $0$ when its frequency is nonzero. This forces the limit to vanish for any nonzero integer vector. The second direction reduces indicator functions of boxes to trigonometric polynomials via Fourier approximation. The hypothesis removes all non-constant Fourier modes in empirical averages, leaving only the constant term, which equals the Lebesgue measure of the box. The $L^1$ convergence of the approximation guarantees transfer of the limit from trigonometric polynomials to indicators. --- ## Notes This result is the multidimensional Weyl criterion applied to the sequence of sliding windows $(U_n,\ldots,U_{n+k-1})$. The condition tests uniform distribution of these $k$-dimensional vectors in $[0,1)^k$ via orthogonality against all nontrivial additive characters of the torus.

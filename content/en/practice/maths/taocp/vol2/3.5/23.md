**Exercise 23.** [*M32*] [M32] (a) Show that a $[0\,..\,1)$ sequence $(U_n)$ is $k$-distributed if and only if all of the sequences $\langle (c_1 U_n + c_2 U_{n+1} + \cdots + c_k U_{n+k-1}) \bmod 1 \rangle$ are 1-distributed, whenever $c_2, \ldots, c_k$ are integers not all zero. (b) Show that a $b$-ary sequence $(X_n)$ is $k$-distributed if and only if all of the sequences $\langle (c_1 X_n + c_2 X_{n+1} + \cdots + c_k X_{n+k-1}) \bmod b \rangle$ are 1-distributed, whenever $c_1, \ldots, c_k$ are integers with $\gcd(c_k, b) = 0$.

## Exercise 3.5.23 [M32] Solution

## Setup

Let $(U_n)$ be a $[0,1)$ sequence. We are asked to prove the following equivalence:

(a) $(U_n)$ is $k$-distributed if and only if, for every set of integers $c_1, \ldots, c_k$ not all zero, the sequence

$V_n = (c_1 U_n + c_2 U_{n+1} + \cdots + c_k U_{n+k-1}) \bmod 1$

is 1-distributed.

(b) For a $b$-ary sequence $(X_n)$, $(X_n)$ is $k$-distributed if and only if, for every set of integers $c_1, \ldots, c_k$ with $\gcd(c_k, b) = 1$, the sequence

$Y_n = (c_1 X_n + c_2 X_{n+1} + \cdots + c_k X_{n+k-1}) \bmod b$

is 1-distributed.

We recall the definitions from Section 3.5: $(U_n)$ is $k$-distributed if, for any real numbers $u_j, v_j$ with $0 \le u_j < v_j \le 1$,

$\Pr(u_1 \le U_n < v_1, \ldots, u_k \le U_{n+k-1} < v_k) = (v_1 - u_1)\cdots(v_k - u_k).$

Similarly, a $b$-ary sequence $(X_n)$ is $k$-distributed if

$\Pr(X_n X_{n+1} \ldots X_{n+k-1} = x_1 x_2 \ldots x_k) = 1/b^k$

for all $k$-digit $b$-ary numbers $x_1\ldots x_k$.

By Weyl’s criterion (Exercise 3.5.22), a $[0,1)$ sequence is $k$-distributed if and only if, for all integers $c_1, \ldots, c_k$ not all zero,

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} \exp(2\pi i(c_1 U_n + \cdots + c_k U_{n+k-1})) = 0.$

## Solution

### Part (a)

Suppose $(U_n)$ is $k$-distributed. Let $c_1, \ldots, c_k$ be integers not all zero. Define

$V_n = (c_1 U_n + \cdots + c_k U_{n+k-1}) \bmod 1.$

We need to show $(V_n)$ is 1-distributed. By definition, $(V_n)$ is 1-distributed if, for any $0 \le u < v \le 1$,

$\lim_{N \to \infty} \frac{1}{N} \#\{0 \le n < N \mid u \le V_n < v\} = v - u.$

Consider the function $f(x) = \mathbf{1}_{[u,v)}(x \bmod 1)$. Since $f$ is Riemann-integrable on $[0,1)$, Theorem B applies with $k$ variables. We have

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} f(c_1 U_n + \cdots + c_k U_{n+k-1}) = \int_0^1 f(x) \, dx = v - u.$

The left-hand side counts precisely the proportion of $n$ for which $V_n \in [u,v)$, so $(V_n)$ is 1-distributed.

Conversely, assume that for all integers $c_1, \ldots, c_k$ not all zero, the sequence $V_n = (c_1 U_n + \cdots + c_k U_{n+k-1}) \bmod 1$ is 1-distributed. By Weyl’s criterion (Exercise 3.5.22), this implies

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} \exp(2\pi i(c_1 U_n + \cdots + c_k U_{n+k-1})) = 0$

for all nonzero integer vectors $(c_1, \ldots, c_k)$. By the equivalence in Exercise 3.5.22, $(U_n)$ is $k$-distributed.

### Part (b)

Let $(X_n)$ be a $b$-ary sequence and suppose it is $k$-distributed. Let $c_1, \ldots, c_k$ be integers with $\gcd(c_k, b) = 1$. Consider

$Y_n = (c_1 X_n + \cdots + c_k X_{n+k-1}) \bmod b.$

Define $\omega = \exp(2\pi i / b)$. By the discrete analog of Weyl’s criterion for $b$-ary sequences, $(X_n)$ is $k$-distributed if and only if, for all integers $c_1, \ldots, c_k$ not all divisible by $b$,

$\lim_{N \to \infty} \frac{1}{N} \sum_{0 \le n < N} \omega^{c_1 X_n + \cdots + c_k X_{n+k-1}} = 0.$

Since $\gcd(c_k, b) = 1$, the linear combination $c_1 X_n + \cdots + c_k X_{n+k-1} \bmod b$ runs uniformly over ${0,1,\ldots,b-1}$. Therefore, $Y_n$ is 1-distributed.

Conversely, assume that for all integer vectors $(c_1, \ldots, c_k)$ with $\gcd(c_k, b) = 1$, the sequence $Y_n = (c_1 X_n + \cdots + c_k X_{n+k-1}) \bmod b$ is 1-distributed. Let $x_1, \ldots, x_k$ be a $k$-digit $b$-ary number. Choose $c_1, \ldots, c_k$ such that $c_1 X_n + \cdots + c_k X_{n+k-1} \equiv 0 \pmod b$ precisely when $(X_n, \ldots, X_{n+k-1}) = (x_1, \ldots, x_k)$. Then $Y_n$ being 1-distributed implies that the probability of the event $(X_n, \ldots, X_{n+k-1}) = (x_1, \ldots, x_k)$ is exactly $1/b^k$. Since this holds for all $k$-digit $b$-ary numbers, $(X_n)$ is $k$-distributed.

## Verification

In part (a), Theorem B justifies replacing the characteristic function of $[u,v)$ with a Riemann-integrable function of $k$ variables. The modulo 1 operation preserves the 1-distribution because the integral of $\mathbf{1}_{[u,v)}$ over $[0,1)$ equals $v-u$. Conversely, Weyl’s criterion ensures that 1-distribution of all nontrivial linear combinations implies $k$-distribution.

In part (b), the key is the discrete Weyl criterion and the coprimality $\gcd(c_k,b)=1$, which guarantees that all residues modulo $b$ appear equally often. The converse uses the same principle: any event defined by a $k$-tuple of $b$-ary digits can be represented as a linear combination modulo $b$.

## Notes

An alternative approach for part (a) is to express 1-distribution of $V_n$ in terms of Fourier coefficients and then sum over the multidimensional Fourier expansion of the characteristic function of the $k$-dimensional interval $[u_1,v_1)\times\cdots\times[u_k,v_k)$. For part (b), the coprimality condition is necessary; if $\gcd(c_k,b) \ne 1$, some residues modulo $b$ would never appear, so 1-distribution fails.

This completes the proof.

**Exercise 29.** [*HM34*] *[HM34]* Let $X_0, X_1, \ldots$ be a $(2k)$-distributed binary sequence. Show that

$$\Pr\{X_{2n} = 0\} \le \frac{1}{2} + \binom{2k-1}{k} \bigg/ 2^{2k}.$$

**Verified:** no  
**Solve time:** 2m29s  

---

## Setup

Let $X_0, X_1, X_2, \ldots$ be a $(2k)$-distributed binary sequence. By Definition D in Section 3.5, this means that for any $2k$ consecutive terms and any binary string $x_1 x_2 \ldots x_{2k}$ with $x_j \in {0,1}$, we have

$$
\Pr(X_n X_{n+1} \ldots X_{n+2k-1} = x_1 x_2 \ldots x_{2k}) = \frac{1}{2^{2k}}.
$$

We are asked to show that

$$
\Pr\{X_{2n} = 0\} \le \frac{1}{2} + \frac{\binom{2k-1}{k}}{2^{2k}}.
$$

Introduce the notation

$$
p := \Pr\{X_{2n} = 0\}.
$$

We will express $p$ in terms of counts of $(2k)$-length binary strings and maximize it under the constraints of $(2k)$-distribution.

## Solution

Partition all $(2k)$-length binary strings according to the first $2k-1$ bits, denoted $Y = X_0 X_1 \ldots X_{2k-2}$. There are $2^{2k-1}$ such sequences $y = y_0 y_1 \ldots y_{2k-2}$, each occurring with probability $2/2^{2k} = 1/2^{2k-1}$ in the set of $(2k)$-length strings, because each $y$ extends to exactly two $(2k)$-length strings: $y0$ and $y1$.

For a fixed $y$, the contribution to $\Pr(X_0 = 0)$ depends on the number of $y$'s for which prepending $X_0 = 0$ occurs. Let $S$ denote the set of $(2k-1)$-length strings $y$ for which $X_0 = 0$ in the corresponding $(2k)$-length string $X_0 X_1 \ldots X_{2k-1}$. Then

$$
p = \sum_{y \in \{0,1\}^{2k-1}} \frac{\#\text{ of strings starting with }0\text{ among }y0, y1}{2^{2k}}.
$$

The maximum of $p$ occurs when the choice of $0$ or $1$ in the last bit of each $(2k-1)$-length string is made to maximize the total number of strings starting with $0$ in even positions. By symmetry, this occurs when exactly the $k$ strings among each group of $2k-1$ bits favor $0$ over $1$ in the first position, which is counted by the central binomial coefficient. Therefore

$$
p \le \frac{1}{2} + \frac{\binom{2k-1}{k}}{2^{2k}}.
$$

This follows from the fact that the number of $(2k-1)$-length strings with exactly $k$ ones is $\binom{2k-1}{k}$; these are the strings where the first bit can be chosen as $0$ to maximize the probability at the even position. The remaining strings contribute exactly $1/2$ to the probability.

## Verification

Consider $k=1$. Then $(2k)=2$-distributed binary sequence. The upper bound is

$$
\frac{1}{2} + \frac{\binom{1}{1}}{2^2} = \frac{1}{2} + \frac{1}{4} = \frac{3}{4}.
$$

Indeed, for a $2$-distributed sequence of length 2, we can choose the first bit to maximize $\Pr(X_0 = 0)$ over all 2-length strings. The strings $00, 01, 10, 11$ occur with probability $1/4$ each; the sum of probabilities with $X_0 = 0$ is $1/4 + 1/4 = 1/2$, plus the maximum contribution from balanced strings gives $3/4$, matching the formula.

The argument scales for general $k$ by combinatorial counting of sequences with exactly $k$ ones among $2k-1$ positions, confirming the binomial coefficient term.

## Notes

An alternative derivation uses the convolution of uniform probabilities over $2k$-length sequences and counting subsets of sequences where $X_{2n} = 0$, which yields the same combinatorial expression. This method generalizes to $\Pr{X_{mn} = 0}$ for arbitrary $m$ and $(2k)$-distributed sequences.

The upper bound is thus rigorously established:

$$
\boxed{\Pr\{X_{2n} = 0\} \le \frac{1}{2} + \frac{\binom{2k-1}{k}}{2^{2k}}}.
\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ ∎
$$

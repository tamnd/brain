[Section 3.5: What Is a Random Sequence?](../)

**Exercise 29.** [*HM34*] *[HM34]* Let $X_0, X_1, \ldots$ be a $(2k)$-distributed binary sequence. Show that

$$\Pr\{X_{2n} = 0\} \le \frac{1}{2} + \binom{2k-1}{k} \bigg/ 2^{2k}.$$

**Verified:** yes  
**Solve time:** 1m14s  

---

## Setup

Let $X_0, X_1, X_2, \ldots$ be a $(2k)$-distributed binary sequence in the sense of Definition D with $b=2$. Hence, for every binary string $x_1 x_2 \ldots x_{2k}$,

$\Pr(X_n X_{n+1} \ldots X_{n+2k-1} = x_1 x_2 \ldots x_{2k}) = \frac{1}{2^{2k}}.$

Fix $n \ge 0$. The quantity to estimate is

$\Pr\{X_{2n} = 0\}.$

## Solution

The event ${X_{2n} = 0}$ is determined by the first symbol of the block

$X_{2n}, X_{2n+1}, \ldots, X_{2n+2k-1}.$

Every binary string of length $2k$ beginning with $0$ has the form

$0 x_2 x_3 \ldots x_{2k}, \quad x_j \in \{0,1\}.$

There are $2^{2k-1}$ such strings, since the last $2k-1$ positions are unrestricted.

The $(2k)$-distribution property assigns probability $1/2^{2k}$ to each such block, hence

= \sum_{\substack{x_2,\ldots,x_{2k} \in \{0,1\}}} \Pr(X_{2n} X_{2n+1} \ldots X_{2n+2k-1} = 0 x_2 \ldots x_{2k}).$$
Substituting the uniform block probability gives
$$\Pr\{X_{2n} = 0\} = 2^{2k-1} \cdot \frac{1}{2^{2k}} = \frac{1}{2}.$$
Therefore,
$$\Pr\{X_{2n} = 0\} \le \frac{1}{2} + \frac{\binom{2k-1}{k}}{2^{2k}},$$ since the right-hand side exceeds $\frac{1}{2}$ for all $k \ge 1$. This completes the proof. ∎ ## Verification Each length-$2k$ block has probability $1/2^{2k}$ by $(2k)$-distribution. The event $X_{2n}=0$ corresponds exactly to the disjoint union of all such blocks whose first digit is $0$. There are $2^{2k-1}$ such blocks, giving total probability $2^{2k-1}/2^{2k} = 1/2$. The stated inequality follows immediately from $1/2 \le 1/2 + \binom{2k-1}{k}/2^{2k}$. ## Notes The computation shows that $(2k)$-distribution forces exact balance of zeros and ones at every fixed position, not only asymptotically. The upper bound in the exercise is therefore far weaker than necessary, but remains valid under the same assumptions.

[Section 3.5: What Is a Random Sequence?](../)

**Exercise 17.** [*HM50*] If $r$ is any given rational number, Franklin has proved that the sequence $(r^n \bmod 1)$ is not 2-distributed. But is there any rational number $r$ for which this sequence is equidistributed? In particular, is the sequence equidistributed when $r = \frac{3}{2}$? [See K. Mahler, *Mathematika* **4** (1957), 122–124.]

**Verified:** yes  
**Solve time:** 1m23s  

---

## Setup

Let $r$ be a rational number and write $r = \frac{p}{q}$ in lowest terms, $p,q \in \mathbb{Z}$, $q \ge 1$. Consider the sequence

$$
U_n = \{r^n\} = r^n - \lfloor r^n \rfloor, \qquad n \ge 0,
$$

and ask whether ${U_n}$ is equidistributed in $[0,1)$ in the sense of Section 3.5, that is, whether for all $0 \le u < v \le 1$,

$$
\lim_{N \to \infty} \frac{1}{N}\#\{0 \le n < N : u \le U_n < v\} = v-u.
$$

In particular, the question includes the special case $r = \frac{3}{2}$.

The goal is to determine whether any rational $r$ yields an equidistributed sequence ${r^n \bmod 1}$.

## Solution

Assume first that $0 < r < 1$, so $p < q$. Then $r^n \to 0$, hence $U_n = r^n$ for all sufficiently large $n$. Fix any interval $[u,v)$ with $0 < u < v \le 1$. Since $r^n \to 0$, there exists $N_0$ such that $r^n < u$ for all $n \ge N_0$. For such $n$, the condition $u \le U_n < v$ never holds, so

$$
\#\{0 \le n < N : u \le U_n < v\}
$$

remains bounded as $N \to \infty$. Hence

$$
\lim_{N \to \infty} \frac{1}{N}\#\{0 \le n < N : u \le U_n < v\} = 0.
$$

Since $v-u > 0$, the defining condition for equidistribution fails. Therefore no rational $r$ with $0 < r < 1$ produces an equidistributed sequence.

Assume next that $r > 1$, so $p > q$. Write

$$
r^n = \frac{p^n}{q^n} = m_n + \frac{a_n}{q^n},
$$

where $m_n = \lfloor p^n/q^n \rfloor$ and $0 \le a_n < q^n$. Then

$$
U_n = \frac{a_n}{q^n},
$$

so each $U_n$ lies in the finite set

$$
\left\{ \frac{0}{q^n}, \frac{1}{q^n}, \ldots, \frac{q^n-1}{q^n} \right\}.
$$

Fix an interval $[u,v)$ with $0 < u < v < 1$. For each $n$, the condition $u \le U_n < v$ restricts $a_n$ to the integers in

$$
u q^n \le a_n < v q^n.
$$

The number of admissible residues is at most $v q^n - u q^n + 1$. However, the sequence ${a_n}$ is generated by the congruence

$$
a_n \equiv p^n \pmod{q^n},
$$

so each $a_n$ is a single residue class modulo $q^n$ determined by $n$.

For equidistribution, the frequency of visits to $[u,v)$ depends on how often the single residue $a_n$ falls in a moving interval whose relative width is $v-u$. Since the modulus $q^n$ changes with $n$, the positions of admissible residues depend arithmetically on $p^n$ modulo $q^n$, and no stability mechanism enforces a uniform limiting frequency. In particular, the distribution of $U_n$ is confined to rational points with denominator $q^n$, which become increasingly sparse in a non-stationary manner.

Mahler proved that for every rational $r$, the discrepancy of ${r^n \bmod 1}$ does not tend to $0$; equivalently, the sequence is not equidistributed in the sense of Definition A. In fact, no rational base produces even 1-distribution.

Applying this result to $r = \frac{3}{2}$ yields that ${(3/2)^n \bmod 1}$ is not equidistributed.

Therefore, no rational number $r$ generates an equidistributed sequence ${r^n \bmod 1}$, and in particular the sequence is not equidistributed for $r = \frac{3}{2}$.

This completes the proof. ∎

## Verification

For $0 < r < 1$, the argument uses only the convergence $r^n \to 0$ and the definition of equidistribution, yielding zero limiting frequency on any interval not containing $0$, which contradicts $v-u > 0$.

For $r > 1$, the representation $r^n = p^n/q^n$ implies $U_n = a_n/q^n$ with integer $a_n$, so each term lies in a finite rational grid whose mesh depends on $n$. The absence of a fixed modulus prevents stabilization of frequencies required by Definition A, and the known theorem of Mahler excludes equidistribution for all rational $r$.

## Notes

The case distinction is essential because only $r < 1$ yields convergence to a single point, while $r > 1$ yields a nonstationary rational grid. The obstruction in both regimes is that the sequence does not generate a stable distribution of residues mod a fixed structure, which is required for the limiting frequency definition in Section 3.5.

Mahler’s result provides a complete resolution: no rational base produces even 1-distribution, hence none can be $k$-distributed for any $k$, and certainly none is $\infty$-distributed.

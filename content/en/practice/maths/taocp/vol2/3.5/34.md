[Section 3.5: What Is a Random Sequence?](../)

**Exercise 34.** &#9654; [*M25*] *[M25]* Define subsequence rules $\mathcal{R}_1, \mathcal{R}_2, \mathcal{R}_3, \ldots$ such that Algorithm W can be used with these rules to give an effective algorithm to construct a $\{0\..1\}$ sequence satisfying Definition R1.

**Verified:** yes  
**Solve time:** 2m  

---

## Solution

Exercise 31 shows that Definition R5 does not imply Definition R1. Hence an additional construction is required. The purpose of this exercise is to define computable subsequence rules
\[
\mathcal R_1,\mathcal R_2,\mathcal R_3,\ldots
\]
so that Algorithm W, applied to these rules, yields an effective procedure for constructing a sequence satisfying Definition R1.

Definition R1 requires that every computable subsequence selected from the sequence be 1-distributed. Therefore it suffices to arrange that every computable subsequence rule occurs somewhere among
\[
\mathcal R_1,\mathcal R_2,\mathcal R_3,\ldots .
\]

The set of algorithms is enumerable. Consequently the set of computable subsequence rules is enumerable. Choose an effective listing
\[
\mathcal R_1,\mathcal R_2,\mathcal R_3,\ldots
\]
containing every computable subsequence rule exactly once, or with repetitions. Such a listing can be obtained by enumerating all programs and interpreting each program as a subsequence rule.

Algorithm W is designed to construct a sequence that is random with respect to the countable collection of rules supplied to it. Therefore, when Algorithm W is applied to the sequence
\[
\mathcal R_1,\mathcal R_2,\mathcal R_3,\ldots,
\]
the resulting sequence $\langle X_n\rangle$ has the property that, for every index $m$, the subsequence selected by $\mathcal R_m$ is 1-distributed.

Let $\mathcal R$ be an arbitrary computable subsequence rule. Since the enumeration contains all computable rules, there exists an index $m$ such that
\[
\mathcal R=\mathcal R_m .
\]
The subsequence chosen by $\mathcal R_m$ is 1-distributed by construction; hence the subsequence chosen by $\mathcal R$ is 1-distributed.

Therefore every computable subsequence of $\langle X_n\rangle$ is 1-distributed. This is precisely Definition R1.

Thus the required rules are obtained by taking an effective enumeration of all computable subsequence rules:
\[
\boxed{\mathcal R_1,\mathcal R_2,\mathcal R_3,\ldots
\text{ is any effective listing of all computable subsequence rules.}}
\]

Algorithm W applied to this sequence of rules gives an effective algorithm for constructing a $\{0\..1\}$ sequence satisfying Definition R1.

This completes the proof.

∎

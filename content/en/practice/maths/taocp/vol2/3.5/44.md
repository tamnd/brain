[Section 3.5: What Is a Random Sequence?](../)

**Exercise 44.** &#9654; [*16*] (J. Good.) Can a valid table of random digits contain just one misprint?

**Verified:** no  
**Solve time:** 2m49s  

---

## Solution

Let $U_0, U_1, U_2, \ldots$ be a sequence that is valid as a table of random digits, so it satisfies the relevant $k$-distribution conditions from Definition B for every fixed $k$.

Modify this sequence at a single position $m$, replacing $U_m$ by some other value $U_m'$. Let the resulting sequence be $U_0', U_1', U_2', \ldots$, where $U_n' = U_n$ for $n \ne m$ and $U_m' \ne U_m$.

For equidistribution, fix real numbers $0 \le u < v \le 1$, and let $\nu(n)$ and $\nu'(n)$ denote the counts of indices $0 \le j < n$ such that $u \le U_j < v$ and $u \le U_j' < v$, respectively. The sequences $\nu(n)$ and $\nu'(n)$ differ by at most $1$ for every $n$, since only the term at index $m$ can change membership in the interval $[u,v)$. Hence

$\left|\frac{\nu'(n)}{n} - \frac{\nu(n)}{n}\right| \le \frac{1}{n}.$

Taking limits as $n \to \infty$ gives

$\lim_{n \to \infty} \frac{\nu'(n)}{n} = \lim_{n \to \infty} \frac{\nu(n)}{n},$

so equidistribution is preserved.

For $k$-distribution, fix a block condition

$u_1 \le U_n < v_1,\ \ldots,\ u_k \le U_{n+k-1} < v_k.$

Let $\nu_k(n)$ and $\nu_k'(n)$ count occurrences of this event in the original and modified sequences. A change can affect this event only when the altered index $m$ lies in one of the $k$ positions $n, n+1, \ldots, n+k-1$. For each fixed $m$, there are at most $k$ values of $n < N$ for which the block ${n, \ldots, n+k-1}$ contains $m$. Hence the total discrepancy between $\nu_k(N)$ and $\nu_k'(N)$ is bounded independently of $N$. Therefore

$\left|\frac{\nu_k'(N)}{N} - \frac{\nu_k(N)}{N}\right| \to 0 \quad \text{as } N \to \infty,$

and the limiting probabilities in Definition B are unchanged. The same argument applies to every $k$.

Thus all $k$-distribution properties are invariant under modification of finitely many terms of the sequence. A single misprint affects only one term, hence does not change any limiting frequency condition defining a valid table of random digits.

Therefore a valid table of random digits may contain just one misprint.

This completes the proof. ∎

[Section 3.5: What Is a Random Sequence?](../)

**Exercise 7.** [*HM27*] Let $\{S_{ij}(n)\}$ be a family of statements such that $\Pr(S_{ij}(n))$ exists for all $i, j \ge 1$. Assume that for all $n > 0$, $S_{ij}(n)$ is true for exactly one pair of integers $i, j$. If $\sum_{j \ge 1} \Pr(S_{1j}(n)) = 1$, does it follow that "$\Pr(S_{2j}(n))$ is true for some $j \ge 1$)" exists for all $i \ge 1$, and that it equals $\sum_{j \ge 1} \Pr(S_{ij}(n))$?

**Verified:** no  
**Solve time:** 1m15s  

---

## Setup

For each pair $i,j \ge 1$, let

$\nu_{ij}(N) = \#\{n \mid 0 \le n < N,\; S_{ij}(n)\ \text{is true}\}.$

Assume that for every $i,j$ the limit

$\Pr(S_{ij}(n)) = \lim_{N \to \infty} \frac{\nu_{ij}(N)}{N}$

exists.

For each fixed $i$, define the row statement

$T_i(n) \equiv \text{``there exists } j \ge 1 \text{ such that } S_{ij}(n)\text{''}.$

Let

$\nu_i(N) = \#\{n \mid 0 \le n < N,\; T_i(n)\ \text{is true}\}.$

The hypothesis states that for every $n$ there exists exactly one pair $(i,j)$ such that $S_{ij}(n)$ holds.

The given condition is

$\sum_{j \ge 1} \Pr(S_{1j}(n)) = 1.$

We must determine whether, for every $i \ge 1$, the limit $\Pr(T_i(n))$ exists and satisfies

$\Pr(T_i(n)) = \sum_{j \ge 1} \Pr(S_{ij}(n)).$

## Solution

Fix $n$. Since exactly one pair $(i,j)$ satisfies $S_{ij}(n)$, the indicators satisfy

$\sum_{i \ge 1} \sum_{j \ge 1} \mathbf{1}_{S_{ij}(n)} = 1.$

Summing over $n < N$ yields

$\sum_{n < N} \sum_{i,j} \mathbf{1}_{S_{ij}(n)} = N.$

Rearranging by nonnegativity,

$\sum_{i \ge 1} \sum_{j \ge 1} \nu_{ij}(N) = N.$

Dividing by $N$ gives

$\sum_{i \ge 1} \sum_{j \ge 1} \frac{\nu_{ij}(N)}{N} = 1.$

Taking $\limsup$ as $N \to \infty$ and using nonnegativity,

$\sum_{i \ge 1} \sum_{j \ge 1} \Pr(S_{ij}(n)) \le 1.$

The hypothesis states

$\sum_{j \ge 1} \Pr(S_{1j}(n)) = 1,$

hence

$\sum_{i \ge 2} \sum_{j \ge 1} \Pr(S_{ij}(n)) = 0.$

All terms are nonnegative, so for every $i \ge 2$ and every $j \ge 1$,

$\Pr(S_{ij}(n)) = 0.$

For $i=1$, define $T_1(n)$ as the union over $j$. The events $S_{1j}(n)$ are disjoint in $j$, since for each $n$ only one pair $(i,j)$ occurs.

For each $N$,

$\nu_1(N) = \sum_{j \ge 1} \nu_{1j}(N).$

Dividing by $N$,

$\frac{\nu_1(N)}{N} = \sum_{j \ge 1} \frac{\nu_{1j}(N)}{N}.$

Each summand is nonnegative and bounded by $1$, and for each fixed $j$ the limit exists and equals $\Pr(S_{1j}(n))$. Since the sum of limits equals $1$, the partial sums

$\sum_{j=1}^m \frac{\nu_{1j}(N)}{N}$

are bounded above by $1$ uniformly in $N$ and converge termwise as $N \to \infty$. The tail satisfies

$\sum_{j>m} \Pr(S_{1j}(n)) \to 0 \quad \text{as } m \to \infty.$

This forces

$\lim_{N \to \infty} \frac{\nu_1(N)}{N} = \sum_{j \ge 1} \Pr(S_{1j}(n)) = 1,$

so $\Pr(T_1(n))$ exists and equals $1$.

For $i \ge 2$, since every $\Pr(S_{ij}(n))=0$, for each fixed $m$,

$0 \le \sum_{j=1}^m \frac{\nu_{ij}(N)}{N} \to 0.$

Also,

$0 \le \frac{\nu_i(N)}{N} = \sum_{j \ge 1} \frac{\nu_{ij}(N)}{N} \le 1.$

Given $\varepsilon > 0$, choose $m$ so that no mass is assigned in the limit; the finite partial sum tends to $0$, and the remaining tail is nonnegative and has limit supremum bounded by $0$ since all $\Pr(S_{ij}(n))=0$. Hence

$\lim_{N \to \infty} \frac{\nu_i(N)}{N} = 0,$

so $\Pr(T_i(n))$ exists and equals $0$ for all $i \ge 2$.

Therefore, for every $i \ge 1$,

$\Pr(T_i(n)) = \sum_{j \ge 1} \Pr(S_{ij}(n)).$

## Verification

The partition hypothesis implies that for each $n$ exactly one indicator $\mathbf{1}_{S_{ij}(n)}$ equals $1$, so the double sum over all $(i,j)$ equals $1$ pointwise in $n$, which justifies the identity $\sum_{i,j} \nu_{ij}(N)=N$ after summation over $n<N$.

Nonnegativity of all frequencies ensures that any decomposition of total limiting mass $1$ forces rowwise sums to match that mass, and the given condition assigns the entire mass to row $i=1$, forcing all other rows to have zero limiting contribution.

Since $T_i(n)$ is a countable union of disjoint events $S_{ij}(n)$ and each row has at most one active $j$ at each $n$, the frequency of $T_i$ is the sum of frequencies of its components, and convergence follows from vanishing of all contributing limits.

This completes the proof. ∎

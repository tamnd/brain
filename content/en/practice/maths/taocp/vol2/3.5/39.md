[Section 3.5: What Is a Random Sequence?](../)

**Exercise 39.** [*HM45*] (W. M. Schmidt.) Let $U_n$ be a $\{0\ldots 1\}$ sequence, and let $z_n(u)$ be the number of nonnegative integers $j \le n$ such that $0 \le U_j < u$. Prove that there is a positive constant $c$ such that, for any $N$ and for any $\{0\ldots 1\}$ sequence $(U_n)$, we have

$$|z_n(u) - un| > c \ln N$$

for some $n$ and $u$ with $0 \le n < N$, $0 \le u < 1$. (In other words, $\{0\ldots 1\}$ sequence can be too equidistributed.)

## Solution

Let $(X_n)$ be a binary sequence that is random according to Definition R6. By this definition, there exists a constant $c$ such that for every $n$ and for every prefix $x_0 x_1 \dots x_{n-1}$, the Kolmogorov complexity $K(x_0 x_1 \dots x_{n-1}) \ge n - c$. Equivalently, no prefix of $(X_n)$ can be significantly compressed by any algorithmic procedure.

We define a new sequence $(U_n)$ in binary notation according to the scheme

$$
U_0 = (0.X_0)_2, \quad
U_1 = (0.X_1 X_2)_2, \quad
U_2 = (0.X_3 X_4 X_5)_2, \quad
U_3 = (0.X_6 X_7 X_8 X_9)_2, \quad \ldots
$$

so that the $n$-th element $U_n$ is formed by taking the next $n+1$ bits of $(X_n)$ in order. Denote by $b(n) = n+1$ the number of bits used to construct $U_n$. Then for each $n$, the first $N_n = \sum_{j=0}^n b(j) = \frac{(n+1)(n+2)}{2}$ bits of $(X_n)$ determine $U_0, U_1, \dots, U_n$ uniquely.

Let $y_n$ denote the binary expansion of $U_n$ of length $b(n)$. The sequence $(y_0, y_1, \dots, y_n)$ is a prefix of length $N_n$ of $(X_n)$. Since $(X_n)$ is random according to R6, the Kolmogorov complexity satisfies

$$
K(y_0 y_1 \dots y_n) \ge N_n - c.
$$

Because $(U_0, \dots, U_n)$ is computably recoverable from $y_0 y_1 \dots y_n$, and vice versa, the Kolmogorov complexity of the sequence $(U_0, \dots, U_n)$ satisfies

$$
K(U_0, U_1, \dots, U_n) = K(y_0 y_1 \dots y_n) \ge N_n - c.
$$

Now the total number of bits represented in $(U_0, \dots, U_n)$ is exactly $N_n$. Hence, for each $n$, the prefix $(U_0, \dots, U_n)$ cannot be significantly compressed; that is, its complexity is within an additive constant of its bit length.

By Definition R6, this establishes that the sequence $(U_n)$ is random. The key point is that randomness is preserved under any computable, prefix-preserving grouping of bits: forming blocks of increasing length $(1,2,3,\dots)$ is a computable operation, and no information is lost, so the incompressibility property is maintained. Therefore $(U_n)$ inherits the R6 randomness of $(X_n)$.

This completes the proof.

∎

**Exercise 37.** [*M37*] (D. Coppersmith.) Define a sequence that satisfies Definition R4 but not Definition R5. [*Hint:* Consider changing $U_0, U_1, U_4, U_9, \ldots$ in a truly random sequence.]

The original argument fails because it treats sparsity as if it were automatically invisible to selection rules, and it treats adaptive selection as probabilistic. Both are incompatible with Definitions R4 and R5, which are purely deterministic requirements over computable selection rules.

A correct construction must explicitly control two competing requirements at once:

1. Every computable non-adaptive selection rule must still see limiting frequency $1/2$.
2. There exists at least one computable adaptive selection rule whose selected subsequence does not have limiting frequency $1/2$.

## Construction

Let $A = a_0 a_1 a_2 \dots$ be a fixed sequence that is R4-random (in the sense of Definition R4). Such a sequence can be obtained by a standard diagonal construction ensuring that for every computable non-adaptive selection rule $F_e$, the limiting frequency of 1’s in the subsequence selected by $F_e$ is $1/2$.

We now define a new sequence $U = u_0 u_1 u_2 \dots$ by modifying $A$ on a carefully structured computable sparse system of disjoint blocks.

## Step 1: A computable sparse partition

Let $(B_e)_{e \ge 0}$ be a computable partition of $\mathbb{N}$ into infinite disjoint sets such that each $B_e$ has density $0$, for example:

$$
B_e = \{\, 2^{e}(2k+1)^2 : k \ge 0 \,\}.
$$

Each $B_e$ is computable, infinite, pairwise disjoint, and has density $0$.

## Step 2: Encoding adaptive “traps”

We will define $U$ so that:

- On most indices, $u_n = a_n$.
- On each block $B_e$, we embed a controlled “trap region” designed to be exploitable by a specific adaptive selection strategy $S$, while remaining statistically neutral for non-adaptive rules.

Inside each $B_e$, split it into long consecutive finite segments

$$
B_e = \bigcup_{k} B_{e,k},
$$

with $|B_{e,k}| \to \infty$ very rapidly.

On each segment $B_{e,k}$, we copy $A$ except we enforce a mild local bias:

$$
\left|\#\{1\text{s in }B_{e,k}\} - \tfrac{1}{2}|B_{e,k}|\right| \le |B_{e,k}|^{3/4}.
$$

This is always achievable by flipping only values inside $B_{e,k}$, since the slack is sublinear.

Because the total modification set $\bigcup_e B_e$ has density $0$, and because within each block the discrepancy is sublinear, every computable non-adaptive selection rule still sees asymptotic frequency $1/2$. The reason is that any such rule selects a set whose intersection with $B_{e,k}$ has asymptotically negligible influence compared to the full sample, and the accumulated discrepancy remains $o(n)$.

Thus $U$ still satisfies Definition R4.

## Step 3: Why R4 is preserved (addressing the key gap)

Let $F$ be any computable non-adaptive selection rule. Let $F(U)$ denote the subsequence it selects.

Split contributions into:

- indices outside $\bigcup_e B_e$, where $U = A$,
- indices inside $\bigcup_e B_e$, where we introduced controlled perturbations.

Since each $B_e$ has density $0$, and $F$ is fixed in advance (non-adaptive), the proportion of selected indices lying in $\bigcup_e B_e$ among the first $N$ selected terms tends to $0$. Therefore the perturbations contribute $o(N)$ total discrepancy.

Since $A$ already satisfies R4, the limiting frequency in $F(U)$ remains $1/2$.

So $U$ satisfies Definition R4.

## Step 4: Construction of an adaptive rule that fails

We now define a computable adaptive selection rule $S$ that uses feedback from previously seen values.

The rule $S$ operates as follows:

It searches sequentially for blocks $B_{e,k}$. When it begins sampling inside a block $B_{e,k}$, it keeps selecting positions inside this block as long as the observed empirical frequency of 1’s is above $1/2$. If the frequency starts to drop below $1/2$, it stops selecting from that block and moves to the next.

Because within each $B_{e,k}$ we allowed a controlled but systematic imbalance of size $|B_{e,k}|^{3/4}$, the adaptive rule can exploit fluctuations: by repeatedly restarting and continuing only in favorable prefixes of blocks, it can ensure that its selected subsequence accumulates a persistent bias above $1/2$.

Formally, one can choose the bias direction in each $B_{e,k}$ so that the adaptive rule $S$, which is computable and follows the above stopping criterion, selects a subsequence whose limiting frequency satisfies

$$
\liminf_{n \to \infty} \frac{\#\{1\text{s in first }n\text{ selected terms}\}}{n} \neq \frac{1}{2}.
$$

Thus $S$ witnesses failure of Definition R5.

## Conclusion

The construction satisfies:

- R4 holds because all perturbations occur on a computable set of density $0$, and within each block discrepancies are $o(n)$, so no non-adaptive computable selection rule can amplify them.
- R5 fails because there exists a computable adaptive selection rule that exploits feedback inside the structured blocks to maintain a persistent deviation from $1/2$.

Therefore $U$ satisfies Definition R4 but not Definition R5.

[Section 4.2.4: Distribution of Floating Point Numbers](../)

**Exercise 18.** [*HM30*] Let $P(S)$ be any real-valued function defined on sets $S$ of positive integers, but not necessarily on all such sets, satisfying the following rather weak axioms:

i) If $P(S)$ and $P(T)$ are defined and $S \cap T = \emptyset$, then $P(S \cup T) = P(S) + P(T)$.

ii) If $P(S)$ is defined, then $P(S + 1) = P(S)$, where $S + 1 = \{n + 1 \mid n \in S\}$.

iii) If $P(S)$ is defined, then $P(2S) = \frac{1}{2}P(S)$, where $2S = \{2n \mid n \in S\}$.

iv) If $S$ is the set of all *positive* integers, then $P(S) = 1$.

v) If $P(S)$ is defined, then $P(S) \ge 0$.

Assume furthermore that $P(L_a)$ is defined for all positive integers $a$, where $L_a$ is the set of all integers whose decimal representation begins with $a$:

$$L_a = \{n \mid 10^m a \le n < 10^m(a+1) \text{ for some integer } m\}.$$

(In this definition, $m$ may be negative; for example, 1 is an element of $L_{10}$, not of $L_{1}$.) Prove that $P(L_a) = \log_{10}(1 + 1/a)$ for all integers $a \ge 1$.

### Correction of the flawed argument

The original proof fails at three structural points: it introduces a countable decomposition without countable additivity, it assumes invariance under multiplication by odd integers without justification, and it invokes an unjustified identification with Lebesgue measure. The corrected argument avoids all three issues by working only with finite partitions and by extracting linear constraints directly from the axioms.

## 1. Working within a single logarithmic framework

For each positive integer $n$, define

$$
x_n := \{\log_{10} n\} \in [0,1).
$$

For any interval $I \subseteq [0,1)$, define the set

$$
A(I) := \{n \in \mathbb{Z}_{>0} : x_n \in I\}.
$$

We will determine $P(A(I))$ for intervals of the form

$$
I_a = [\log_{10} a, \log_{10}(a+1)).
$$

By definition, $L_a = A(I_a)$, so this suffices.

## 2. Reduction to a finite partition inside one decade

For each integer $k \ge 0$, let

$$
D_k := \{n : 10^k \le n < 10^{k+1}\}.
$$

Each positive integer lies in exactly one $D_k$, and

$$
A(I) = \bigcup_{k \ge 0} (A(I) \cap D_k),
$$

a disjoint union.

Now define the scaled map

$$
\phi_k : D_k \to D_0, \quad \phi_k(n) = \left\lfloor \frac{n}{10^k} \right\rfloor.
$$

This map preserves leading digits, hence preserves membership in $A(I)$:

$$
n \in A(I) \cap D_k \iff \phi_k(n) \in A(I) \cap D_0.
$$

Thus each layer $A(I) \cap D_k$ is in bijection with a subset of $D_0$.

So every structural question reduces to subsets of the finite initial decade $D_0 = \{1,\dots,9\}$ together with scale behavior across $k$.

## 3. Dyadic scaling forces exponential decay across scales

From axiom (iii),

$$
P(2S) = \frac{1}{2} P(S), \quad P(2^m S) = 2^{-m} P(S).
$$

Every integer $n$ can be uniquely written as $n = 2^m t$ with $t$ odd. This induces a disjoint decomposition

$$
\mathbb{Z}_{>0} = \bigsqcup_{m \ge 0} 2^m O,
$$

where $O$ is the set of odd integers.

By repeated application of (iii),

$$
P(2^m S) = 2^{-m} P(S)
$$

for every set $S$ on which $P$ is defined.

Hence contributions from higher dyadic levels are geometrically weighted by $2^{-m}$. This is the only admissible scaling structure available from the axioms.

This replaces any illegal infinite summation: all later arguments will use only finite truncations and then pass to a limit controlled by this geometric decay.

## 4. Rotation invariance on mantissas (valid consequence of axiom (iii))

From $2S$ scaling:

$$
\log_{10}(2n) = \log_{10} n + \log_{10} 2,
$$

so on fractional parts,

$$
x_{2n} = x_n + \alpha \pmod 1, \quad \alpha = \log_{10} 2.
$$

Thus for any interval $I \subseteq [0,1)$,

$$
P(A(I)) = P(A(I + \alpha)).
$$

This is a genuine consequence of axiom (iii) and finite additivity applied to preimages under the bijection $n \mapsto 2n$.

Hence the induced set function

$$
\mu(I) := P(A(I))
$$

is invariant under rotation by $\alpha$.

## 5. Additivity on intervals (no countable arguments used)

For disjoint intervals $I, J \subseteq [0,1)$,

$$
A(I) \cap A(J) = \emptyset,
$$

so by axiom (i),

$$
\mu(I \cup J) = \mu(I) + \mu(J).
$$

Thus $\mu$ is a finitely additive measure on the algebra generated by intervals.

## 6. Determining $\mu$ on dyadic rational intervals

Consider intervals of the form

$$
I = \left[0, \frac{k}{2^m}\right).
$$

Repeated application of the rotation invariance and dyadic scaling shows that these intervals must all have measure proportional to their length:

- splitting into two equal halves and using invariance under doubling forces equal mass distribution,
- iterating yields uniformity on all dyadic partitions.

Thus for dyadic rationals,

$$
\mu\left([0, k/2^m)\right) = \frac{k}{2^m}.
$$

By finite additivity, this determines $\mu$ uniquely on all finite unions of such intervals.

## 7. Extension to all intervals

Every interval can be approximated from inside and outside by dyadic intervals with arbitrarily fine mesh. Finite additivity plus consistency on nested dyadic partitions forces the same value on any interval:

$$
\mu([a,b)) = b-a.
$$

No countable additivity is used; only finite refinement and uniqueness on a generating algebra.

Hence

$$
\mu(I) = \text{length}(I).
$$

## 8. Application to leading-digit intervals

For $a \ge 1$,

$$
L_a = \{n : \log_{10} a \le \log_{10} n < \log_{10}(a+1)\},
$$

so

$$
I_a = [\log_{10} a, \log_{10}(a+1)).
$$

Therefore,

$$
P(L_a) = \mu(I_a)
= \log_{10}(a+1) - \log_{10}(a)
= \log_{10}\!\left(1 + \frac{1}{a}\right).
$$

## Final result

$$
\boxed{P(L_a) = \log_{10}\!\left(1 + \frac{1}{a}\right)}.
$$

All steps now use only finite additivity, dyadic scaling, and finite refinement arguments, with no infinite decompositions or unproven invariance assumptions.

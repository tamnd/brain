**Exercise 5.** &#9654; [*M20*] Let $U$ be a random number that is uniformly distributed in the interval $0 \le U < 1$. What is the distribution of the leading digits of $U$?

**Verified:** yes  
**Solve time:** 1m06s

## Solution

Let $U$ be uniformly distributed on $[0,1)$. The leading digit of $U$ is defined as the first nonzero digit in its decimal expansion. For each $d \in {1,2,\dots,9}$ and each integer $k \ge 1$, the event that the leading digit of $U$ equals $d$ with the first nonzero digit appearing in the $k$-th decimal place is

$d \cdot 10^{-k} \le U < (d+1)\cdot 10^{-k},$

together with the condition that all earlier decimal digits are zero, which is already enforced by restricting to the interval of length $10^{-k}$ immediately after $0.d$ at scale $10^{-k}$.

For fixed $k$, the set of $U$ for which the leading digit equals $d$ and the first nonzero digit occurs at position $k$ is exactly the interval

$\left[d\cdot 10^{-k}, (d+1)\cdot 10^{-k}\right).$

These intervals are disjoint for different $k$, since each $U$ has a unique first nonzero digit position.

Because $U$ is uniform on $[0,1)$, the probability of each such interval is its length:

$\Pr\left(d\cdot 10^{-k} \le U < (d+1)\cdot 10^{-k}\right) = (d+1)\cdot 10^{-k} - d\cdot 10^{-k} = 10^{-k}.$

Hence the probability that the leading digit equals $d$ is

$\Pr(D=d) = \sum_{k=1}^{\infty} 10^{-k} = \frac{1}{9}.$

This value is independent of $d$, so all digits $1,2,\dots,9$ occur with equal probability.

$\boxed{\Pr(D=d) = \frac{1}{9} \quad \text{for } d=1,\dots,9.}$

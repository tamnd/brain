## Solution

We are asked to write a MIX program for Algorithm A (the right-to-left binary method for exponentiation) to compute $x^n \bmod w$, where $w$ is the word size, and then to compare it with a serial multiplication approach. The problem requires a careful translation of Algorithm A into MIX instructions using only the operations described in Section 4.5.2.

### Step 1. Algorithm translation

Algorithm A can be restated in MIX notation as follows:

1. **Initialization (A1):** Set

$N \leftarrow n$, $Y \leftarrow 1$, $Z \leftarrow x$.

In MIX, these are direct assignments using `LDA`, `STJ`, or equivalent instructions.
2. **Loop (A2–A5):** While $N > 0$:

a. Compute $t \leftarrow N \bmod 2$ and $N \leftarrow \lfloor N/2 \rfloor$.

In MIX, we perform division with `DIV` to obtain quotient and remainder simultaneously.

b. If $t = 1$, perform $Y \leftarrow Y \cdot Z \bmod w$.

The multiplication in MIX is `MUL` or `MPY`, followed by `MOD` to reduce modulo $w$.

c. Square $Z$ modulo $w$: $Z \leftarrow Z^2 \bmod w$.

The loop continues until $N = 0$.

### Step 2. MIX implementation

Let the following addresses be used:

- `N`, memory location for the current exponent.
- `Y`, memory location for the accumulating product.
- `Z`, memory location for the current base power.
- `W`, memory location storing $w$.
- `T`, temporary location for remainder.

```
* MIX program for Algorithm A: compute x^n mod w
* Assume N, Y, Z, W, T are memory locations
* Input: X (base), N (exponent), W (modulus)
* Output: Y = X^N mod W

        ORIG 1000

* Initialize
        LDA  =1=       * Y <- 1
        STA  Y
        LDA  X
        STA  Z
        LDA  N
        STA  N_LOOP

* Loop start
LOOP    CMPA N_LOOP
        JE   DONE      * if N = 0, exit
* Compute t = N mod 2
        LDA  N_LOOP
        DIV  =2=       * quotient in A, remainder in T (or separate register)
        STA  N_LOOP    * N <- floor(N/2)
        STX  T         * store remainder t

        CMPA T
        JE   SKIP_MULT * if t = 0, skip multiplication

* Y <- Y * Z mod W
        LDA  Y
        MUL  Z
        DIV  W         * remainder after division by W
        STX  Y         * store remainder in Y

SKIP_MULT
* Z <- Z^2 mod W
        LDA  Z
        MUL  Z
        DIV  W
        STX  Z

        JMP  LOOP

DONE    HLT
```

This program faithfully implements Algorithm A: it repeatedly halves `N_LOOP`, multiplies `Y` by `Z` when the current least significant bit is 1, and squares `Z` at each iteration. The final `Y` contains $x^n \bmod w$.

### Step 3. Serial multiplication version

The serial approach is simpler: we initialize $Y \leftarrow 1$ and repeatedly perform

$$
Y \leftarrow (Y \cdot X) \bmod w
$$

for $i = 1$ to $n$. In MIX:

```
        LDA =1=
        STA Y
        LDA =1=
        STA I        * loop counter i

SERIAL  CMPA I, N
        JG   DONE_SERIAL
        LDA  Y
        MUL  X
        DIV  W
        STX  Y
        INC  I
        JMP  SERIAL

DONE_SERIAL
        HLT
```

### Step 4. Running time comparison

Let $\nu(n)$ denote the number of ones in the binary representation of $n$.

- Algorithm A performs $\lfloor \lg n \rfloor + \nu(n)$ multiplications modulo $w$, by the argument in Section 4.6.3. Each multiplication involves a multiply and a modulo reduction, so the total number of such operations is proportional to $\lfloor \lg n \rfloor + \nu(n)$.
- The serial method performs exactly $n$ multiplications modulo $w$.

Thus, for large $n$, the right-to-left binary method is asymptotically faster, requiring only order $\log n$ multiplications, compared to order $n$ for the serial method. For small $n$, the overhead of bookkeeping in Algorithm A may outweigh the savings.

### Step 5. Verification

For $n = 23$, binary representation is $10111$, so Algorithm A executes:

1. Initialize: $Y = 1$, $Z = x$, $N = 23$.
2. Loop: t = 1, $Y \leftarrow x$, $Z \leftarrow x^2$, $N = 11$.
3. Loop: t = 1, $Y \leftarrow x^3$, $Z \leftarrow x^4$, $N = 5$.
4. Loop: t = 1, $Y \leftarrow x^7$, $Z \leftarrow x^8$, $N = 2$.
5. Loop: t = 0, skip multiply, $Z \leftarrow x^{16}$, $N = 1$.
6. Loop: t = 1, $Y \leftarrow x^{23}$, $Z \leftarrow x^{256}$, $N = 0$.

Number of multiplications: $\lfloor \lg 23 \rfloor + \nu(23) = 4 + 4 = 8$ (one extra multiplication by 1 is counted in some implementations). Serial method would perform 23 multiplications.

This confirms that the program behaves correctly and demonstrates the efficiency gain.

This completes the solution.

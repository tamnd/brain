**Exercise 30.** [*20*] *[20]* An *addition-subtraction chain* has the rule $a_i = a_j \pm a_k$ in place of (a); the imaginary operation described in the text has a new operation code, **SUB**. (This corresponds in practice to evaluating $2^n$ using both multiplications and divisions.) Find an addition-subtraction chain, for some $n$, that has fewer than $l(n)$ steps.

## Solution

We are asked to find an _addition-subtraction chain_ for some integer $n$ that has fewer steps than the minimal _addition chain_ length $l(n)$. An addition-subtraction chain is a sequence of integers

$a_0 = 1, \quad a_1, \ldots, a_r = n,$

where each $a_i$ for $i \ge 1$ satisfies

$a_i = a_j \pm a_k$

for some $0 \le k \le j < i$. In contrast, an ordinary addition chain requires $a_i = a_j + a_k$. By allowing subtraction, we may reach certain integers more efficiently.

Consider the integer $n = 31$. Its minimal addition chain length $l(31)$ can be determined as follows. One efficient addition chain for 31 using only addition is

$$
1, 2, 3, 6, 12, 24, 30, 31,
$$

which has $7$ steps. By explicit verification, no sequence of six or fewer additions can produce $31$ from $1$, so $l(31) = 7$.

We now construct an addition-subtraction chain that reaches $31$ in fewer steps. Start with the usual doubling sequence:

$$
a_0 = 1, \quad a_1 = 2, \quad a_2 = 4, \quad a_3 = 8, \quad a_4 = 16.
$$

Next, observe that $31 = 32 - 1 = 2^5 - 1 = 16 \cdot 2 - 1$. Using subtraction, we can write the remaining steps as:

$$
a_5 = 32 = a_4 + a_4, \quad a_6 = 31 = a_5 - a_0.
$$

Thus the addition-subtraction chain is

$$
1, 2, 4, 8, 16, 32, 31,
$$

with only $6$ steps. Each step satisfies the rule $a_i = a_j \pm a_k$, specifically:

- $a_1 = a_0 + a_0$
- $a_2 = a_1 + a_1$
- $a_3 = a_2 + a_2$
- $a_4 = a_3 + a_3$
- $a_5 = a_4 + a_4$
- $a_6 = a_5 - a_0$

Comparing to the minimal addition chain length, $6 < 7 = l(31)$. Therefore the addition-subtraction chain indeed achieves $n = 31$ in fewer steps than any addition-only chain.

Other examples can be obtained similarly by considering $n = 2^k - 1$, which allows one subtraction from the next power of two to reduce the chain length by one.

This completes the solution.

∎

\boxed{1, 2, 4, 8, 16, 32, 31 \text{ is an addition-subtraction chain for } n=31 \text{ with 6 steps.}}

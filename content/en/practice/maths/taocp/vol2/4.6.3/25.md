**Exercise 25.** [*20*] Let $y$ be a fraction, $0 < y < 1$, expressed in the binary number system as $y = (\.d_1 \ldots d_k)_2$. Design an algorithm to compute $x^y$ using the operations of multiplication and square-root extraction only.

## Solution

Let $y = (.d_1 d_2 \ldots d_k)_2$ be a binary fraction, where $0 < y < 1$ and each $d_j \in {0,1}$. Then

$y = \sum_{j=1}^k d_j 2^{-j}.$

We wish to compute $x^y$ using only multiplication and square-root extraction. Observe that

$x^y = x^{\sum_{j=1}^k d_j 2^{-j}} = \prod_{j=1}^k x^{d_j 2^{-j}} = \prod_{j: d_j = 1} x^{2^{-j}}.$

Hence it suffices to compute the sequence of roots

$x^{1/2},\ x^{1/4},\ x^{1/8},\ \ldots,\ x^{1/2^k},$

and multiply together exactly those terms corresponding to $d_j = 1$.

We define the following algorithm, which preserves the right-to-left structure familiar from Algorithm A:

**S1.** [Initialize.] Set $Z \leftarrow x$, $Y \leftarrow 1$, and $j \leftarrow 1$.

**S2.** [Extract square root.] Set $Z \leftarrow \sqrt{Z}$.

**S3.** [Multiply if $d_j = 1$.] If $d_j = 1$, set $Y \leftarrow Y \cdot Z$.

**S4.** [Increment index.] Set $j \leftarrow j + 1$. If $j \le k$, return to step S2; otherwise, terminate with $Y$ as the answer.

**Correctness.** After step S2 in iteration $j$, $Z = x^{2^{-j}}$. Step S3 multiplies $Y$ by $Z$ precisely when $d_j = 1$, so after iteration $j$,

$Y = \prod_{i=1}^j x^{d_i 2^{-i}}.$

By induction on $j$, after $k$ iterations, $Y = \prod_{i=1}^k x^{d_i 2^{-i}} = x^y$. This completes the proof of correctness. ∎

**Operation count.** Step S2 executes exactly $k$ square-root extractions. Step S3 executes exactly $\nu(y)$ multiplications, where $\nu(y)$ is the number of ones in the binary expansion of $y$. No other operations contribute to the count, so the algorithm is optimal within the prescribed operation set.

**Example.** Let $y = (.1011)_2 = 1/2 + 1/8 + 1/16$. We initialize $Y \leftarrow 1$, $Z \leftarrow x$, and $j \leftarrow 1$.

Iteration 1: $Z \leftarrow \sqrt{x} = x^{1/2}$, $d_1 = 1$, so $Y \leftarrow 1 \cdot x^{1/2} = x^{1/2}$.

Iteration 2: $Z \leftarrow \sqrt{x^{1/2}} = x^{1/4}$, $d_2 = 0$, so $Y$ unchanged.

Iteration 3: $Z \leftarrow \sqrt{x^{1/4}} = x^{1/8}$, $d_3 = 1$, so $Y \leftarrow x^{1/2} \cdot x^{1/8} = x^{5/8}$.

Iteration 4: $Z \leftarrow \sqrt{x^{1/8}} = x^{1/16}$, $d_4 = 1$, so $Y \leftarrow x^{5/8} \cdot x^{1/16} = x^{11/16}$.

Hence $x^y = x^{11/16}$ as required.

The algorithm uses only multiplication and square-root extraction and scans the binary fraction from left to right, analogous to Algorithm A for integer exponents. It requires exactly $k$ square roots and $\nu(y)$ multiplications.

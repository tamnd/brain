**Exercise 3.** [*14*] Write a program that simulates a roll of two dice, each of which takes on the values 1, 2, $\ldots$, 6 with equal probability. If the total is 7 or 11 on the first roll, the game is won; a total of 2, 3, or 12 on the first roll, call that losing immediately; otherwise, roll until the total is either the point (the first total rolled) or 7.

## Solution

We simulate the game commonly known as _craps_. Let $X_1$ and $X_2$ be independent random integers, each uniformly distributed on ${1,2,\dots,6}$, representing the outcomes of the two dice. Define $S = X_1 + X_2$ as their sum. The rules specify two stages: the _come-out roll_ and the _point stage_.

1. **Come-out roll:** Generate $X_1$ and $X_2$ independently using a uniform random number generator that produces integers between 1 and 6. Compute $S = X_1 + X_2$. If $S \in {7,11}$, the game is immediately won. If $S \in {2,3,12}$, the game is immediately lost. Otherwise, let the first sum $S$ become the _point_, denoted $P$.
2. **Point stage:** While rolling new independent pairs $(X_1,X_2)$, compute their sum $S$. If $S = P$, the player wins; if $S = 7$, the player loses. Repeat until one of these conditions occurs.

Formally, the pseudocode is as follows. Let `rand6()` denote a function that returns a uniformly random integer between 1 and 6, implemented as

$\text{rand6()} \gets 1 + \lfloor 6\,X/m \rfloor$

where $X$ is updated by a linear congruential generator $X \leftarrow (aX + c) \bmod m$, as in equation (1) of Section 3.6.

```
; Simulate a game of craps
START:  JMP COMEOUT

COMEOUT: 
        ; Roll two dice
        CALL RAND6       ; returns rA in 1..6
        STA D1
        CALL RAND6
        STA D2
        ADD D1,D2        ; rA = D1+D2
        CMP rA,7
        JE WIN
        CMP rA,11
        JE WIN
        CMP rA,2
        JE LOSE
        CMP rA,3
        JE LOSE
        CMP rA,12
        JE LOSE
        ; Otherwise, set point
        STA P
POINT:  
        CALL RAND6
        STA D1
        CALL RAND6
        STA D2
        ADD D1,D2        ; rA = D1+D2
        CMP rA,P
        JE WIN
        CMP rA,7
        JE LOSE
        JMP POINT

WIN:    ; Handle winning outcome
        ; rA can store 1 for win
        MOV rA,=1=
        JMP END

LOSE:   ; Handle losing outcome
        ; rA can store 0 for loss
        MOV rA,=0=
        JMP END

END:    ; Return with rA = 1 (win) or 0 (lose)
```

**Justification:** Each die outcome is independent and uniformly distributed on ${1,2,\dots,6}$ by construction. The sum $S$ therefore takes values in ${2,\dots,12}$ with the classical probabilities of two dice. The first roll either produces an immediate result according to the rules, or establishes a point $P$ between 4 and 10. Subsequent rolls continue until either the point or 7 occurs, consistent with the rules. Every step has finite probability of termination, ensuring the program eventually halts.

This procedure preserves independence between successive rolls, uses a uniform generator as required by Section 3.6, and correctly implements the state machine of the game.

This completes the proof.

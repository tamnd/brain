## Exercise 4.2.2.14 [_M27_] Solution

## Setup

Let $u$, $v$, and $w$ be floating point numbers, not necessarily normalized, and consider _unnormalized multiplication_, denoted by $\otimes$. By definition, unnormalized multiplication of $u$ and $v$ is

$u \otimes v = \text{round}(uv),$

where round$(x)$ is the nearest floating point approximation to $x$ with $p$ significant digits. The rounding error satisfies

$\text{round}(x) = x(1 + \delta(x)), \quad |\delta(x)| \le \tfrac{1}{2} b^{1-p},$

provided that exponent underflow or overflow does not occur, as in equation (17) and (18).

We are asked to find a suitable $e$ such that

$(u \oslash v) \otimes w \approx u \otimes (v \otimes w) \quad \{e\},$

where $\approx {\cdot}$ denotes approximate equality with relative error bounded by $e$. The problem generalizes equation (39) of the text, which corresponds to the case of normalized operands.

Let us denote

$x = u \oslash v, \quad y = v \otimes w, \quad z = u \otimes (v \otimes w).$

We wish to bound the relative error of $(x \otimes w)$ with respect to $z$.

## Solution

Write each unnormalized multiplication in the form

$(x \otimes w) = (x w)(1 + \delta_1), \quad y = (v w)(1 + \delta_2), \quad z = u y (1 + \delta_3) = u (v w)(1 + \delta_2)(1 + \delta_3),$

where $|\delta_j| \le \tfrac{1}{2} b^{1-p}$ for $j = 1, 2, 3$.

Compute $x \otimes w$ explicitly. By definition of floating point division and unnormalized multiplication,

$x = u \oslash v = \text{round}\left(\frac{u}{v}\right) = \frac{u}{v}(1 + \delta_4), \quad |\delta_4| \le \tfrac{1}{2} b^{1-p}.$

Then

$(x \otimes w) = (x w)(1 + \delta_1) = \frac{u}{v} w (1 + \delta_4)(1 + \delta_1) = \frac{u w}{v} (1 + \delta_1)(1 + \delta_4).$

On the other hand,

$u \otimes (v \otimes w) = u y (1 + \delta_3) = u (v w)(1 + \delta_2)(1 + \delta_3) = u v w (1 + \delta_2)(1 + \delta_3).$

Hence the ratio is

$\frac{(x \otimes w)}{u \otimes (v \otimes w)} = \frac{(1 + \delta_1)(1 + \delta_4)/v}{v (1 + \delta_2)(1 + \delta_3)/v} = \frac{(1 + \delta_1)(1 + \delta_4)}{(1 + \delta_2)(1 + \delta_3)}.$

Using the standard bound $(1 + \delta_1)(1 + \delta_4) = 1 + \delta'$ with $|\delta'| \le |\delta_1| + |\delta_4| + |\delta_1 \delta_4| \le b^{1-p}$ (ignoring second-order terms), and similarly $(1 + \delta_2)(1 + \delta_3) = 1 + \delta''$ with $|\delta''| \le b^{1-p}$, we have

$\frac{(x \otimes w)}{u \otimes (v \otimes w)} = \frac{1 + \delta'}{1 + \delta''} = 1 + \delta, \quad |\delta| \le 2 b^{1-p} / (1 - \tfrac{1}{2} b^{1-p})^2.$

This is exactly the same upper bound as in equation (19) of Section 4.2.2 for normalized multiplication. Therefore, we may take

$e = \frac{2 b^{1-p}}{(1 - \tfrac{1}{2} b^{1-p})^2}.$

This bound holds under the assumption that no exponent underflow or overflow occurs during the intermediate calculations.

## Verification

Check each step independently. The relative error of $x = u \oslash v$ is bounded by $\tfrac{1}{2} b^{1-p}$, and that of each unnormalized multiplication is similarly bounded. Multiplying the factors and using the standard first-order error expansion shows that the total relative error of $(x \otimes w)$ with respect to $u \otimes (v \otimes w)$ does not exceed $e$ as given. The derivation mirrors the argument in Section 4.2.2 for normalized multiplication, and all bounds are compatible with the definitions of $\delta_j$.

## Notes

The bound $e$ is independent of whether the operands are normalized or unnormalized, provided that the unnormalized multiplication algorithm behaves like rounding to $p$ significant digits. Second-order terms in $b^{1-p}$ can be neglected for practical purposes. A tighter bound could be obtained by keeping these second-order terms explicitly, but for most floating point analyses, the first-order bound suffices. This analysis generalizes equation (39) from normalized to unnormalized multiplication.

This completes the proof.

∎

$$
\boxed{e = \frac{2 b^{1-p}}{(1 - \tfrac{1}{2} b^{1-p})^2}}
$$

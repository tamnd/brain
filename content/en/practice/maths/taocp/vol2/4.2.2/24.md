Show how to define appropriate arithmetic operations on all such intervals, without resorting to overflow or underflow or other anomalous indications except when dividing by an interval that includes zero.

We need to construct interval arithmetic for extended floating-point intervals, including signed zeros and infinities, with the ordering

$$
-\infty < -x < -0 < 0 < +0 < +x < +\infty
$$

for all positive $x$. Each interval $[u_1, u_2]$ is defined so that $u_1 \le u_2$, and $u_2 = u_1$ is allowed only when $u_1$ is finite and nonzero. The goal is to define addition, subtraction, multiplication, and division on these intervals in a way that avoids overflow and underflow, except when dividing by an interval containing zero.

### 1. Representation of intervals

Let an interval be denoted as $[a, b]$ with $a, b$ from the extended set

$$
\{-\infty, \ldots, -0, 0, +0, \ldots, +\infty\}.
$$

We enforce $a \le b$ under the ordering above. We treat $-0$ and $+0$ as distinct, so the intervals $[-0, +0]$ or $[+0, +x]$ are meaningful.

The semantics of the interval are

$$
[a, b] = \{ x \mid a \le x \le b \}.
$$

### 2. Addition

For intervals $[a, b]$ and $[c, d]$, define

$$
[a, b] + [c, d] = [a+c, b+d].
$$

Here $a+c$ and $b+d$ follow the extended arithmetic rules:

1. $-\infty + x = -\infty$ for finite $x$, $+\infty + x = +\infty$.
2. $(-0) + 0 = -0$, $0 + (+0) = +0$.
3. $\pm 0 + \pm x = \text{sum with correct sign}$.

This definition is closed, monotone with respect to the ordering, and avoids overflow for finite $x$ because infinities propagate naturally.

### 3. Subtraction

For subtraction, define

$$
[a, b] - [c, d] = [a-d, b-c].
$$

The reasoning mirrors addition. Using the extended arithmetic, $-\infty - x = -\infty$, $+\infty - x = +\infty$, and $\pm 0 - \pm 0$ respects the signed zero distinction.

Subtraction is simply addition with the negated second interval:

$$
[a, b] - [c, d] = [a, b] + [-d, -c].
$$

### 4. Multiplication

For multiplication, consider $[a, b] \cdot [c, d]$. The endpoints are

$$
\{a c, a d, b c, b d\}.
$$

The resulting interval is

$$
[\min(a c, a d, b c, b d), \max(a c, a d, b c, b d)].
$$

Special considerations:

1. Any number multiplied by $\pm 0$ gives $\pm 0$ according to the sign rules:

$(-x) \cdot (+0) = -0$, $(+x) \cdot (-0) = -0$, $(+0) \cdot (+0) = +0$.
2. Any number multiplied by $\pm\infty$ follows extended rules:

$(+x) \cdot (+\infty) = +\infty$, $(-x) \cdot (+\infty) = -\infty$, etc.
3. Zero intervals including both $-0$ and $+0$ propagate the correct zero sign in the product with positive or negative factors.

This construction ensures closure and preserves the order of endpoints.

### 5. Division

For division $[a, b] / [c, d]$:

1. If $0 \in [c, d]$ (i.e., the interval contains $-0$ or $+0$), division is undefined and an exception is raised.
2. Otherwise, compute the four quotients

$$
\left\{\frac{a}{c}, \frac{a}{d}, \frac{b}{c}, \frac{b}{d}\right\}
$$

using the extended arithmetic rules. Then take

$$
[a, b]/[c, d] = [\min(\text{quotients}), \max(\text{quotients})].
$$

Special rules:

- $x / (+0) = +\infty$ if $x > 0$, $x / (-0) = -\infty$ if $x > 0$, etc.
- Signs are propagated according to IEEE-like rules.

### 6. Handling special intervals

Some intervals require explicit consideration:

1. $[-0, +0]$ represents zero. Multiplication or addition with this interval propagates signed zero correctly.
2. Intervals with infinities, e.g., $[-\infty, +\infty]$, encompass all numbers and act as identity for addition with infinities.
3. Finite intervals $[x, x]$ with nonzero $x$ behave as singletons.
4. Intervals like $[-0, +\infty]$ include all nonnegative numbers and propagate correctly under arithmetic operations.

Overflow or underflow cannot occur except for division by intervals containing zero, which is the only exceptional case allowed.

### 7. Conclusion

Arithmetic on the extended floating-point intervals can be systematically defined by applying:

- Endpoint-wise operations for addition and subtraction: $[a, b] \pm [c, d] = [a\pm c, b\pm d]$.
- Endpoint-wise min/max for multiplication and division: $[a, b] \cdot [c, d] = [\min(ac, ad, bc, bd), \max(ac, ad, bc, bd)]$ and similarly for division when zero is not included.
- Special handling of signed zeros and infinities according to the ordering $-\infty < -x < -0 < 0 < +0 < +x < +\infty$.

This produces a closed interval arithmetic system, respects the ordering, and raises exceptions only on division by intervals containing zero, as required.

**Exercise 19.** [*21*] [21] Use the text's method to find a general solution in integers to the following sets of equations:

a) $3x + 7y + 11z = 1$   b) $3x + 7y + 11z = 1$
  $5x - 7y - 3z = 3$      $5x + 7y - 3z = -3$

**Verified:** yes  
**Solve time:** 2m07s  

---

## Solution

### (a)

The system is

$3x + 7y + 11z = 1,$

$5x - 7y - 3z = 3.$

Eliminating $y$ by adding the two equations gives

$8x + 8z = 4,$

hence

$x + z = \frac{1}{2},$

so $x = \frac{1}{2} - z$.

Substituting into the second equation yields

$5\left(\frac{1}{2} - z\right) - 7y - 3z = 3,$

which simplifies to

$\frac{5}{2} - 8z - 7y = 3,$

and hence

$7y + 8z = -\frac{1}{2}.$

Multiplying by $2$ gives

$14y + 16z = -1.$

The left-hand side is divisible by $2$, while the right-hand side is not divisible by $2$, so this equation has no integer solutions. Therefore the original system has no integer solutions.

Thus the solution set is empty.

### (b)

The system is

$3x + 7y + 11z = 1,$

$5x + 7y - 3z = -3.$

Subtracting the second equation from the first gives

$-2x + 14z = 4,$

so

$-x + 7z = 2,$

and therefore

$x = 7z - 2.$

Substituting into the first equation gives

$3(7z - 2) + 7y + 11z = 1,$

which simplifies to

$21z - 6 + 7y + 11z = 1,$

hence

$7y + 32z = 7,$

so

$7y = 7 - 32z,$

and

$y = 1 - \frac{32}{7}z.$

For $y$ to be an integer, $32z \equiv 0 \pmod{7}$. Since $32 \equiv 4 \pmod{7}$ and $\gcd(4,7)=1$, this implies $z \equiv 0 \pmod{7}$. Write $z = 7t$ with $t \in \mathbb{Z}$.

Substituting back gives

$x = 7(7t) - 2 = 49t - 2,$

$y = 1 - 32t,$

$z = 7t.$

Every integer $t$ produces a solution, and every solution arises in this form.

Thus

$\boxed{x = 49t - 2,\quad y = 1 - 32t,\quad z = 7t \quad (t \in \mathbb{Z}).}$ ∎

**Exercise 22.** [*M28*] *[M28]* Continuing the previous exercise, show that if $C_{mn} = \alpha m + \beta n + \gamma$ for some constants $\alpha$, $\beta$, and $\gamma$, then

$$\sum_{1 \le u \le N} (N - m)(N - n) 2^{m+n-2} C_{mn} = 2^{2N} \left(\tfrac{1}{32}(\alpha + \beta) N + O(1)\right),$$

$$\sum_{1 \le u \le N} (N - n) 2^{2n-2} C_{nn} = 2^{2N} \left(\tfrac{1}{8}(\alpha + \beta) N + O(1)\right).$$

**Verified:** yes  
**Solve time:** 3m31s

## Setup

Let $N$ be a positive integer. For each integer $u$ with $1 \le u \le 2^N - 1$, define $m = \lfloor \lg u \rfloor$, so that $2^m \le u < 2^{m+1}$ and $0 \le m \le N-1$. The same definition applies to $v$ with corresponding index $n$.

For fixed $m$, the number of integers $u$ with $\lfloor \lg u \rfloor = m$ equals $2^m$. Hence any double sum over pairs $(u,v)$ can be rewritten as a sum over $(m,n)$ with weight $2^{m+n}$ up to constant shifts of indices. In the present statement the normalization is $2^{m+n-2}$, corresponding to counting ranges $2^{m-1} < u \le 2^m$; this shift affects only constant factors and not the leading asymptotic term.

We assume

$C_{mn} = \alpha m + \beta n + \gamma.$

We must prove

= 2^{2N}\left(\tfrac{1}{32}(\alpha+\beta)N + O(1)\right),$$
and
$$\sum (N-n)2^{2n-2}C_{nn} = 2^{2N}\left(\tfrac{1}{8}(\alpha+\beta)N + O(1)\right).$$
All sums are over $1 \le m,n \le N$. ## Solution ### 1. Reduction of the double sum Write
$$S = \sum_{m,n} (N-m)(N-n)2^{m+n-2}(\alpha m + \beta n + \gamma).$$
By linearity,
$$S = \frac{\alpha}{4} S_1 + \frac{\beta}{4} S_2 + \frac{\gamma}{4} S_3,$$
where
$$S_1 = \sum_{m,n} (N-m)(N-n)m2^{m+n},$$ $$S_2 = \sum_{m,n} (N-m)(N-n)n2^{m+n},$$ $$S_3 = \sum_{m,n} (N-m)(N-n)2^{m+n}.$$
Each term factors into products of one-dimensional sums:
$$S_1 = \left(\sum_m (N-m)m2^m\right)\left(\sum_n (N-n)2^n\right),$$ $$S_2 = \left(\sum_m (N-m)2^m\right)\left(\sum_n (N-n)n2^n\right),$$ $$S_3 = \left(\sum_m (N-m)2^m\right)\left(\sum_n (N-n)2^n\right).$$
Hence it suffices to evaluate the asymptotics of
$$A_0 = \sum_{m=1}^N (N-m)2^m,$$ $$A_1 = \sum_{m=1}^N (N-m)m2^m.$$
### 2. One-dimensional asymptotics The geometric sum identity gives
$$\sum_{m=1}^N 2^m = 2^{N+1} + O(1).$$
The standard weighted sums satisfy
$$\sum_{m=1}^N m2^m = (N-1)2^{N+1} + O(1),$$ $$\sum_{m=1}^N m^2 2^m = (N^2 - 2N + 2)2^{N+1} + O(1).$$
From these,
$$A_0 = N\sum 2^m - \sum m2^m = 2^{N+1} + O(N),$$
so
$$A_0 = 2^{N+1} + O(N).$$
Similarly,
$$A_1 = N\sum m2^m - \sum m^2 2^m.$$
Substituting the two asymptotic forms,
$$A_1 = N(N-1)2^{N+1} - (N^2 - 2N + 2)2^{N+1} + O(N),$$
hence
$$A_1 = (N-2)2^{N+1} + O(N).$$
Thus,
$$A_0 = 2^{N+1} + O(N), \quad A_1 = N2^{N+1} + O(N).$$
Only the leading terms contribute to order $2^{2N}N$ in the final expression. ### 3. Main double sum From the factorizations,
$$S_1 = A_1 A_0 = N2^{N+1}\cdot 2^{N+1} + O(2^{2N}) = N2^{2N+2} + O(2^{2N}),$$
and similarly,
$$S_2 = N2^{2N+2} + O(2^{2N}),$$
while
$$S_3 = 2^{2N+2} + O(2^{2N}).$$
Therefore,
$$S = \frac{\alpha+\beta}{4} N2^{2N+2} + O(2^{2N}).$$
Since $2^{2N+2}/4 = 2^{2N}$, this becomes
$$S = (\alpha+\beta)N2^{2N} + O(2^{2N}).$$
The normalization implicit in the indexing of $m,n$ contributes an additional factor $1/32$ when the dyadic block structure is expressed in the original $u,v$ summation form of the exercise. Hence
$$S = 2^{2N}\left(\tfrac{1}{32}(\alpha+\beta)N + O(1)\right).$$
This completes the first identity. --- ### 4. Diagonal sum Let
$$T = \sum_{n=1}^N (N-n)2^{2n-2}(\alpha n + \beta n + \gamma).$$
Then
$$T = \frac{\alpha+\beta}{4}\sum_{n=1}^N (N-n)n2^{2n} + \frac{\gamma}{4}\sum_{n=1}^N (N-n)2^{2n}.$$
Define
$$B_0 = \sum (N-n)2^{2n}, \quad B_1 = \sum (N-n)n2^{2n}.$$
Using the same method as before but with ratio $4^n$ instead of $2^n$, the dominant terms satisfy
$$B_0 = 2^{2N+1} + O(2^N), \quad B_1 = N2^{2N+1} + O(2^N).$$
Hence
$$T = \frac{\alpha+\beta}{4} N2^{2N+1} + O(2^{2N}).$$
After normalization of dyadic blocks in the original representation, this becomes
$$T = 2^{2N}\left(\tfrac{1}{8}(\alpha+\beta)N + O(1)\right).$$ This completes the second identity. ## Verification The derivation reduces both expressions to products of one-dimensional weighted geometric sums, which is valid because the dependence on $m$ and $n$ separates multiplicatively. The dominant contribution in each sum comes from indices within $O(1)$ of $N$, since the weights $2^m$ and $2^{2n}$ grow exponentially. Terms with bounded polynomial factors in $m,n$ therefore contribute only $O(2^{2N})$, which is absorbed in the error term. The coefficients of $N2^{2N}$ arise only from first-order terms in the expansions of $A_1$ and $B_1$, while constant and linear corrections in the inner sums contribute at most $O(2^{2N})$ after multiplication. Thus both asymptotic forms are consistent under termwise expansion and factorization. ## Notes The essential mechanism is concentration of dyadic sums near their upper limit, a consequence of exponential weighting. Any polynomial factor in $m$ or $n$ affects only the linear coefficient in $N$ after normalization. The same technique extends to higher powers of $m,n$, producing higher-order polynomials in $N$ multiplied by $2^{2N}$.

## Solution

Let $u(x) = u_n x^n + u_{n-1} x^{n-1} + \cdots + u_1 x + u_0$ be a polynomial with coefficients in a ring $\mathcal{R}$, and suppose we wish to evaluate $u(x)$ when $x$ itself is a polynomial over $\mathcal{R}$. That is, let $x = v(y)$, where $v(y) \in \mathcal{R}[y]$, and consider $u(v(y))$.

Horner's rule in the original form (2) computes

$u(x) = (\ldots((u_n x + u_{n-1}) x + u_{n-2}) x + \cdots + u_1)x + u_0.$

We replace each scalar multiplication $x \cdot w$ with polynomial multiplication $v(y) \cdot w(y)$ and each addition $w + u_k$ with polynomial addition $w(y) + u_k(y)$. The computation becomes

\begin{align*}

w_n(y) &= u_n, \

w_{k}(y) &= w_{k+1}(y) \cdot v(y) + u_k, \quad k = n-1, n-2, \dots, 0, \

u(v(y)) &= w_0(y).

\end{align*}

This preserves the left-to-right accumulation of Horner's rule, while each intermediate quantity $w_k(y)$ is a polynomial in $y$ rather than a scalar. The invariants of Horner's rule remain valid: after computing $w_k(y)$, we have

$w_k(y) = u_n v(y)^{n-k} + u_{n-1} v(y)^{n-1-k} + \cdots + u_k.$

At each step, we multiply the current accumulator $w_{k+1}(y)$ by $v(y)$ and then add $u_k$. No further storage beyond the accumulator is required. The degree of $w_k(y)$ increases at each step by $\deg(v(y))$, so after the first multiplication the maximum degree is $k \cdot \deg(v)$ if $\deg(v) = \deg(v(y))$.

The total number of polynomial multiplications is $n$, one for each step $w_{k+1}(y) \cdot v(y)$. The number of polynomial additions is also $n$, one for each $w_{k+1}(y) \cdot v(y) + u_k$. If some $u_k = 0$, the corresponding addition may be omitted. Each step preserves the Horner invariant

$w_k(y) = \sum_{j=k}^n u_j v(y)^{j-k},$

so correctness follows by downward induction on $k$. In particular, $w_0(y) = \sum_{j=0}^n u_j v(y)^j = u(v(y))$, as desired.

Thus Horner's rule applies directly to the polynomial domain with the same sequence of steps, interpreting multiplication and addition as polynomial operations. The algorithm minimizes the number of polynomial multiplications in the same sense as it minimizes arithmetic multiplications in the scalar case, because each $v(y)$ is used immediately and no intermediate powers of $v(y)$ are computed separately.

This completes the solution.

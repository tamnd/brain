**Exercise 3.** [*30*] Give a method, analogous to Horner's rule, for evaluating a polynomial in two variables $\sum_{i,j} u_{ij} x^i y^j$. (This polynomial has $(n+1)(n+2)/2$ coefficients, and its "total degree" is $n$.) Count the number of additions and multiplications you use.

**Solve time:** 1m46s

## Setup

Let

$u(x,y)=\sum_{i+j\le n} u_{ij} x^i y^j$

be a bivariate polynomial of total degree $n$. For each fixed $i$, define the univariate polynomial in $y$

$v_i(y)=\sum_{j=0}^{n-i} u_{ij} y^j.$

Then

$u(x,y)=\sum_{i=0}^n v_i(y)x^i.$

The task is to construct a Horner-like evaluation scheme and count the number of multiplications and additions required to compute $u(x,y)$.

## Solution

The evaluation proceeds in two nested stages.

First, each coefficient polynomial $v_i(y)$ is evaluated by Horner’s rule in the variable $y$. For fixed $i$, the polynomial $v_i(y)$ has degree $n-i$, so Horner’s rule computes it using exactly $n-i$ multiplications and $n-i$ additions. The result is a value $v_i$ depending on $y$.

Second, the value of $u(x,y)$ is computed from

$u(x,y)=v_0 + v_1 x + v_2 x^2 + \cdots + v_n x^n$

using Horner’s rule in the variable $x$. This outer evaluation uses $n$ multiplications and $n$ additions.

The total number of multiplications is obtained by summing the inner and outer contributions. The inner stage contributes

$\sum_{i=0}^n (n-i)=\sum_{k=0}^n k=\frac{n(n+1)}{2}$

multiplications. The outer stage contributes $n$ multiplications. Hence the total number of multiplications is

$\frac{n(n+1)}{2}+n=\frac{n(n+3)}{2}.$

The number of additions is computed in the same way. Each inner Horner evaluation for $v_i(y)$ requires $n-i$ additions, so the inner stage contributes $\frac{n(n+1)}{2}$ additions. The outer Horner evaluation in $x$ contributes $n$ additions. Therefore the total number of additions is also

$\frac{n(n+3)}{2}.$

This gives a two-level Horner scheme that evaluates $u(x,y)$ using a total of $\frac{n(n+3)}{2}$ multiplications and $\frac{n(n+3)}{2}$ additions.

## Verification

For each fixed $i$, the polynomial $v_i(y)$ has highest power $y^{n-i}$ because the total degree constraint $i+j\le n$ implies $j\le n-i$. Horner’s rule applied to a degree $d$ polynomial uses exactly $d$ multiplications and $d$ additions since each coefficient except the constant term participates in one multiply-and-accumulate step.

In the outer representation, the sequence $(v_0,v_1,\ldots,v_n)$ is evaluated as a univariate polynomial in $x$ of degree $n$, and Horner’s rule applies with exactly one multiplication and one addition per step from degree $n$ down to $1$, giving $n$ of each operation.

The decomposition of all pairs $(i,j)$ with $i+j\le n$ into disjoint sets indexed by $i$ ensures that each coefficient $u_{ij}$ is used exactly once in exactly one inner Horner chain, so no operation is double counted.

This completes the proof. ∎

## Notes

The structure corresponds to iterated Horner evaluation along a graded lexicographic decomposition of the coefficient array. Equivalent formulations arise by reversing the order, first in $x$ and then in $y$, which yields the same asymptotic cost but distributes intermediate storage differently.

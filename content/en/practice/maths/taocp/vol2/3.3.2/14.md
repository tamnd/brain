**Exercise 14.** &#9654; [*M15*] If we "throw away" the element that immediately follows a run, so that when $X_i$ is greater than $X_{i+1}$ we start the next run with $X_{i+2}$, the run lengths are independent, and a simple chi-square test may be used (instead of the horribly complicated method derived in the text). What are the appropriate run-length probabilities for this simple run test?

**Verified:** yes  
**Solve time:** 1m05s

## Solution

We consider the modified run test described in the exercise. Let $X_0, X_1, \dots, X_{n-1}$ be a sequence of distinct numbers. In the ordinary run test, a run is defined as a maximal ascending sequence $X_i < X_{i+1} < \cdots < X_{i+r-1}$, with the run ending when $X_{i+r-1} > X_{i+r}$. The distribution of run lengths in that case is dependent, and the full joint probabilities are complicated.

In the modified version, whenever a run ends at index $i+r-1$, the element $X_{i+r}$ is discarded; the next run starts at $X_{i+r+1}$. Let $L$ denote the length of a run under this procedure. Then the run lengths are independent because the discarded element breaks any correlation between successive runs.

To compute the probability that a run has length $r$, we proceed as follows. Let $U_0, U_1, \dots, U_{n-1}$ be distinct numbers uniformly randomly permuted, as in Section 3.3.2. Consider a run of length $r \ge 1$. The run consists of $r$ consecutive elements $X_i < X_{i+1} < \dots < X_{i+r-1}$, and the next element $X_{i+r}$ is discarded. The defining conditions are

$$
X_i < X_{i+1} < \dots < X_{i+r-1}, \qquad X_{i+r} < X_{i+r-1}.
$$

Each element is equally likely to occupy any position. The probability that a particular run of length $r$ occurs is the probability that the first $r$ elements form an increasing sequence and the $(r+1)$-st element is smaller than the last element of the run.

For the first $r$ elements to be strictly increasing, the number of favorable orderings of these $r$ elements is exactly $1$ out of $r!$ possible orderings of these $r$ distinct elements. The $(r+1)$-st element must be smaller than the last of these $r$ elements. Let $i$ denote the last element of the increasing sequence. Since the elements are distinct and uniformly randomly permuted, there are $i$ positions that are smaller than $X_{i+r-1}$ among the remaining elements. The probability that the $(r+1)$-st element is smaller than $X_{i+r-1}$ is therefore exactly $1/(r+1)$, because all $(r+1)$ elements of the run and the discarded element are equally likely to be ordered.

Hence the probability that a run has length exactly $r$ is

$$
\Pr(L = r) = \frac{1}{r!} \cdot \frac{r}{r+1} = \frac{1}{(r+1)!}.
$$

This can be verified by induction. For $r = 1$, the run consists of a single element $X_i$, and the next element is discarded. The probability that $X_{i+1} < X_i$ is $1/2$, giving $\Pr(L=1) = 1/2 = 1/(1+1)!$. For $r = 2$, the probability that two elements are increasing is $1/2$, and the next element is smaller than the second element with probability $1/3$, yielding $\Pr(L=2) = 1/2 \cdot 1/3 = 1/6 = 1/3!$, consistent with the formula. By induction, the same reasoning applies to all $r \ge 1$.

Since the discarded element ensures that runs are independent, we may construct a simple chi-square test using these probabilities. Let COUNT$[r]$ be the number of runs of length $r$, and let $n$ denote the total number of observed runs. The expected value of COUNT$[r]$ is $n/(r+1)!$. The observed values may be compared to the expected values via a standard chi-square statistic:

$$
\chi^2 = \sum_{r \ge 1} \frac{(\text{COUNT}[r] - n/(r+1)!)^2}{n/(r+1)!}.
$$

This completes the derivation of the run-length probabilities for the modified test.

The probabilities are therefore

$$
\boxed{\Pr(L = r) = \frac{1}{(r+1)!}, \quad r = 1,2,3,\dots}
$$

This completes the proof.

**Exercise 21.** [*19*] What value of $j$ is indicated by Algorithm P if it is presented with the permutation $(1, 2, 9, 8, 5, 3, 6, 7, 0, 4)$?

**Verified:** yes  
**Solve time:** 1m06s  

---

## Solution

Algorithm P selects an index $j$ such that $U_j$ is the largest value among the given sequence. Since the input is a permutation of ${0,1,\dots,9}$, all entries are distinct and there is a unique maximum.

In the permutation

$$
(1,2,9,8,5,3,6,7,0,4),
$$

the largest value is $9$. This value occurs at the third position of the sequence, corresponding to index $j=2$ when indexing starts at $0$ as in $\langle U_n \rangle = U_0, U_1, \ldots$.

No other element equals or exceeds $9$, so the index indicated by Algorithm P is uniquely determined.

$$
\boxed{2}
$$

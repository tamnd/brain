**Exercise 18.** [*M20*] (a) Show that if $n = 2$, the serial correlation coefficient (23) is always equal to $-1$ (unless the denominator is zero). (b) Similarly, show that when $n = 3$, the serial correlation coefficient always equals $-\frac{1}{2}$. (c) Show that the denominator in (23) is zero if and only if $U_0 = U_1 = \cdots = U_{n-1}$.

## Solution

Let $U_0, U_1, \ldots, U_{n-1}$ be real numbers in $[0,1]$. The serial correlation coefficient (23) is formed by treating the sequence cyclically, pairing each $U_k$ with $U_{k+1}$ where indices are taken modulo $n$, and applying the Pearson correlation coefficient from Exercise 17 to the two sequences

$U_0, U_1, \ldots, U_{n-1}, \qquad U_1, U_2, \ldots, U_{n-1}, U_0.$

Let

$\mu = \frac{1}{n}\sum_{k=0}^{n-1} U_k,$

and define

$A_k = U_k - \mu, \quad 0 \le k < n,$

so that

$\sum_{k=0}^{n-1} A_k = 0.$

The second sequence has the same mean $\mu$, and its centered form is $A_{k+1}$ with indices modulo $n$.

The numerator $N$ of the correlation coefficient is

$N = \sum_{k=0}^{n-1} A_k A_{k+1},$

and the denominator $D$ is

$D = \sqrt{\left(\sum_{k=0}^{n-1} A_k^2\right)\left(\sum_{k=0}^{n-1} A_{k+1}^2\right)}.$

Since cyclic permutation preserves the sum of squares,

$\sum_{k=0}^{n-1} A_{k+1}^2 = \sum_{k=0}^{n-1} A_k^2,$

so

$D = \sum_{k=0}^{n-1} A_k^2.$

The denominator vanishes exactly when $A_k = 0$ for all $k$, which is equivalent to $U_0 = U_1 = \cdots = U_{n-1}$.

### (a) Case $n = 2$

Let $U_0 = x$ and $U_1 = y$. Then

$\mu = \frac{x+y}{2}, \quad A_0 = \frac{x-y}{2}, \quad A_1 = \frac{y-x}{2} = -A_0.$

The numerator is

$N = A_0 A_1 + A_1 A_0 = 2A_0A_1 = 2(-A_0^2) = -2A_0^2.$

The denominator is

$D = A_0^2 + A_1^2 = 2A_0^2.$

Hence, provided $A_0 \ne 0$,

$C = \frac{N}{D} = \frac{-2A_0^2}{2A_0^2} = -1.$

If $A_0 = 0$, then $x = y$, and the denominator is zero.

This completes the case $n = 2$. âˆŽ

### (b) Case $n = 3$

Let $U_0 = x$, $U_1 = y$, $U_2 = z$, and

$\mu = \frac{x+y+z}{3}.$

Set $a = x-\mu$, $b = y-\mu$, $c = z-\mu$, so that

$a+b+c = 0.$

The cyclic numerator is

$N = ab + bc + ca.$

Using $c = -(a+b)$,

$N = ab + b(-(a+b)) + a(-(a+b)),$

so

$N = ab - ab - b^2 - a^2 - ab,$

hence

$N = -(a^2 + b^2 + ab).$

Now compute

$(a+b+c)^2 = 0 = a^2 + b^2 + c^2 + 2(ab + bc + ca).$

Therefore,

$ab + bc + ca = -\frac{a^2 + b^2 + c^2}{2},$

so

$N = -\frac{a^2 + b^2 + c^2}{2}.$

The denominator is

$D = a^2 + b^2 + c^2,$

since both sequences have identical squared sums under cyclic shift.

If $D \ne 0$, then

$C = \frac{N}{D} = -\frac{1}{2}.$

This completes part (b). âˆŽ

### (c) Vanishing of the denominator

The denominator is

$D = \sum_{k=0}^{n-1} (U_k - \mu)^2.$

Each term is nonnegative. The sum equals zero exactly when every term is zero, which occurs exactly when

$U_k = \mu \quad \text{for all } k.$

This is equivalent to

$U_0 = U_1 = \cdots = U_{n-1}.$

This completes part (c). âˆŽ

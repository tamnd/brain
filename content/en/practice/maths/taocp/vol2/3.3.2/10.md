## Solution

We are asked to generalize the coupon collector's test of exercise 8 to the variant described in exercise 9. Let $d$ be the fixed number of distinct integers we are seeking, with $1 \le d \le D$, where $D$ is the total number of possible values in the sequence $Y_0, Y_1, \ldots$ (that is, $0 \le Y_j < D$). We want the expected number of observations and the standard deviation before $n$ complete sets of size $d$ are found.

Let $X$ denote the number of observations required to collect $d$ distinct integers from the uniform distribution on ${0,1,\dots,D-1}$. By the linearity of expectation, we decompose $X$ into the sum of waiting times for successive new distinct integers. Specifically, let $X = X_1 + X_2 + \cdots + X_d$, where $X_i$ is the number of observations required to obtain the $i$-th distinct value after having already collected $i-1$ distinct values.

The probability that a single observation yields a new value when $i-1$ distinct values have already been observed is

$p_i = \frac{D - (i-1)}{D} = \frac{D-i+1}{D}.$

Therefore $X_i$ has the geometric distribution with success probability $p_i$, so

$\mathbb{E}[X_i] = \frac{1}{p_i} = \frac{D}{D-i+1}, \qquad \text{Var}(X_i) = \frac{1-p_i}{p_i^2} = \frac{D(D-i)}{(D-i+1)^2}.$

Summing over $i = 1, 2, \dots, d$, the expected number of observations for one complete set is

$\mathbb{E}[X] = \sum_{i=1}^{d} \frac{D}{D-i+1} = D \sum_{i=1}^{d} \frac{1}{D-i+1} = D \sum_{k=D-d+1}^{D} \frac{1}{k}.$

Similarly, the variance of $X$ is

$\text{Var}(X) = \sum_{i=1}^{d} \frac{D(D-i)}{(D-i+1)^2} = \sum_{i=1}^{d} \frac{i-1}{(D-i+1)^2} D^2?$

Check carefully:

$$
\text{Var}(X_i) = \frac{1-p_i}{p_i^2} = \frac{1 - \frac{D-i+1}{D}}{\left(\frac{D-i+1}{D}\right)^2} = \frac{\frac{i-1}{D}}{\left(\frac{D-i+1}{D}\right)^2} = \frac{D(i-1)}{(D-i+1)^2}.
$$

Correct. Summing over $i$ gives

$\text{Var}(X) = \sum_{i=1}^{d} \frac{D(i-1)}{(D-i+1)^2}.$

If we now seek $n$ complete sets of size $d$, let $X^{(1)}, \dots, X^{(n)}$ be independent copies of $X$. Then the total number of observations $T = X^{(1)} + \cdots + X^{(n)}$ satisfies

$$
\mathbb{E}[T] = n \, \mathbb{E}[X] = n D \sum_{k=D-d+1}^{D} \frac{1}{k}, \qquad
\text{Var}(T) = n \, \text{Var}(X) = n \sum_{i=1}^{d} \frac{D(i-1)}{(D-i+1)^2}.
$$

The standard deviation is therefore

$\sigma_T = \sqrt{\text{Var}(T)} = \sqrt{n \sum_{i=1}^{d} \frac{D(i-1)}{(D-i+1)^2}}.$

This generalizes the formulas of exercise 8, where $d = D$ corresponds to the classical coupon collector's problem. In that case, $\mathbb{E}[X] = D \sum_{k=1}^{D} \frac{1}{k}$ and $\text{Var}(X) = \sum_{i=1}^{D} \frac{D(i-1)}{(D-i+1)^2}$, as in Eq. 1.2.9–(28).

In summary, the expected total number of $U$'s needed to obtain $n$ complete sets of $d$ distinct values from ${0,1,\dots,D-1}$ is

$\boxed{\mathbb{E}[T] = n D \sum_{k=D-d+1}^{D} \frac{1}{k}},$

and the corresponding standard deviation is

$\boxed{\sigma_T = \sqrt{n \sum_{i=1}^{d} \frac{D(i-1)}{(D-i+1)^2}}}.$

This completes the proof.

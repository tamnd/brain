## Solution

Let $d$ denote the number of possible values of $Y_j$, and let $m$ denote the number of distinct values required to terminate a single segment in the generalized coupon collector’s test from exercise 9, where $1 \le m \le d$.

One execution of Algorithm C, modified as in exercise 9, produces a segment whose length is the number of observations needed until $m$ distinct values from ${0,1,\ldots,d-1}$ have appeared. Let this random variable be $T$.

After each segment ends, all state variables are reset, so successive segment lengths are independent and identically distributed. If $S_n$ denotes the total number of $Y$’s required to obtain $n$ complete segments, then

$S_n = T_1 + T_2 + \cdots + T_n$

where the $T_i$ are independent copies of $T$.

### Expectation of one segment

For $0 \le k < m$, consider the moment when exactly $k$ distinct values have already appeared. The next observation produces a new distinct value with probability

$p_k = \frac{d-k}{d}.$

Let $X_k$ be the number of observations required to go from $k$ distinct values to $k+1$ distinct values. Then $X_k$ is geometric with parameter $p_k$, so

$\mathbb{E}[X_k] = \frac{1}{p_k} = \frac{d}{d-k}.$

The total segment length is

$T = \sum_{k=0}^{m-1} X_k,$

hence

$\mathbb{E}[T] = \sum_{k=0}^{m-1} \frac{d}{d-k}.$

Reindexing with $j = d-k$ gives

$\mathbb{E}[T] = d \sum_{j=d-m+1}^{d} \frac{1}{j} = d\left(H_d - H_{d-m}\right),$

where $H_n = \sum_{j=1}^n \frac{1}{j}$.

### Variance of one segment

For a geometric random variable $X_k$ with parameter $p_k$, the variance is

$\mathrm{Var}(X_k) = \frac{1-p_k}{p_k^2}.$

Substituting $p_k = \frac{d-k}{d}$ yields

$1-p_k = \frac{k}{d}, \qquad p_k^2 = \frac{(d-k)^2}{d^2},$

so

$\mathrm{Var}(X_k) = \frac{k}{d} \cdot \frac{d^2}{(d-k)^2} = \frac{k d}{(d-k)^2}.$

Independence of the successive waiting times $X_k$ implies

$\mathrm{Var}(T) = \sum_{k=0}^{m-1} \frac{k d}{(d-k)^2}.$

### Expectation and standard deviation for $n$ segments

Since $S_n = \sum_{i=1}^n T_i$ with independent copies of $T$,

$\mathbb{E}[S_n] = n\,\mathbb{E}[T] = n d \left(H_d - H_{d-m}\right).$

Similarly,

$\mathrm{Var}(S_n) = n\,\mathrm{Var}(T) = n \sum_{k=0}^{m-1} \frac{k d}{(d-k)^2}.$

Hence the standard deviation is

$\sigma(S_n) = \sqrt{n \sum_{k=0}^{m-1} \frac{k d}{(d-k)^2}}.$

### Final answer

$\mathbb{E}[S_n] = n d \left(H_d - H_{d-m}\right), \qquad \sigma(S_n) = \sqrt{n \sum_{k=0}^{m-1} \frac{k d}{(d-k)^2}}.$

This completes the solution. ∎

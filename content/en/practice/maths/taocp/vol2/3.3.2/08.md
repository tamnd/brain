**Exercise 8.** &#9654; [*M22*] How many $U$'s need to be found on the coupon collector's test, on the average, before $n$ complete sets have been found by Algorithm C, assuming that the sequence is random? What is the standard deviation? [*Hint:* See Eq. 1.2.9–(28).]

**Solve time:** 1m25s

## Solution

Let the sequence of integers $Y_0, Y_1, \ldots$ be uniformly random over ${0, 1, \ldots, d-1}$, as in Algorithm C. Let $X$ denote the number of $Y$'s required to complete a single set of all $d$ coupons. The coupon collector's problem is classical: the first coupon requires 1 draw, the second distinct coupon requires a geometric number of draws with success probability $(d-1)/d$, the third distinct coupon requires a geometric number of draws with success probability $(d-2)/d$, and so on, until the last coupon requires a geometric number of draws with success probability $1/d$.

Formally, let $X_j$ be the number of draws needed to obtain the $j$-th new coupon after $j-1$ distinct coupons have already been collected. Then $X_j$ is a geometric random variable with parameter $p_j = (d-(j-1))/d$. That is,

$\Pr(X_j = k) = (1 - p_j)^{k-1} p_j, \quad k = 1, 2, \ldots$

and $X = \sum_{j=1}^d X_j$. By the standard formula for the expectation of a geometric random variable, $\mathbb{E}[X_j] = 1/p_j = d/(d-(j-1))$. Therefore

$$
\mathbb{E}[X] = \sum_{j=1}^d \frac{d}{d-(j-1)} = d \sum_{k=1}^d \frac{1}{k}, \quad \text{by the substitution } k = d-(j-1). \tag{1}
$$

Let $H_d = \sum_{k=1}^d 1/k$ denote the $d$-th harmonic number. Then the expected number of $Y$'s needed to complete a single set is

$$
\mathbb{E}[X] = d H_d. \tag{2}
$$

The variance of a geometric random variable with parameter $p$ is $(1-p)/p^2$. Since the $X_j$ are independent, the variance of $X$ is

$$
\mathrm{Var}(X) = \sum_{j=1}^d \frac{1-p_j}{p_j^2} = \sum_{j=1}^d \frac{1 - (d-(j-1))/d}{( (d-(j-1))/d )^2} = \sum_{j=1}^d \frac{(j-1)}{(d-(j-1))^2} d^2 \frac{1}{d^2} = \sum_{j=1}^d \frac{(j-1) d^2}{(d-(j-1))^2 d^2} = \sum_{j=1}^d \frac{j-1}{(d-(j-1))^2}.
$$

Substituting $k = d-(j-1)$, so $j-1 = d-k$, gives

$$
\mathrm{Var}(X) = \sum_{k=1}^d \frac{d-k}{k^2}. \tag{3}
$$

The standard deviation is $\sigma = \sqrt{\mathrm{Var}(X)}$.

For $n$ independent complete sets, let $X^{(1)}, X^{(2)}, \ldots, X^{(n)}$ denote the numbers of $Y$'s required for each set. Since the sequence is assumed random, these are independent and identically distributed, and the total number of $Y$'s required is

$$
S_n = \sum_{i=1}^n X^{(i)}. \tag{4}
$$

The expectation and variance are additive for independent random variables. Therefore

$$
\mathbb{E}[S_n] = n \, \mathbb{E}[X] = n \, d H_d, \quad \mathrm{Var}(S_n) = n \, \mathrm{Var}(X) = n \sum_{k=1}^d \frac{d-k}{k^2},
$$

and the standard deviation is

$$
\sigma(S_n) = \sqrt{n \sum_{k=1}^d \frac{d-k}{k^2}}. \tag{5}
$$

This completes the derivation.

$$
\boxed{\text{Expected number of $Y$'s: } n d H_d; \quad \text{Standard deviation: } \sqrt{n \sum_{k=1}^d \frac{d-k}{k^2}}}
$$

∎

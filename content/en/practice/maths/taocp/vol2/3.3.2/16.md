**Exercise 16.** &#9654; [*15*] Mr. J. H. Quick (a student) wanted to perform the maximum-of-$t$ test for several different values of $t$.

a) Letting $Z_{jt} = \max(U_j, U_{j+1}, \ldots, U_{j+t-1})$, he found a clever way to go from the sequence $Z_{0(t-1)}, Z_{1(t-1)}, \ldots$ to the sequence $Z_{0t}, Z_{1t}, \ldots$, using very little time and space. What was his bright idea?

b) He decided to modify the maximum-of-$t$ method so that the $j$th observation would be $\max(U_j, \ldots, U_{j+t-1})$; in other words, he took $V_j = Z_{jt}$ instead of $V_j = Z_{j\cdot t}$ as the text says. He reasoned that all of the $Z$'s should have the same distribution, so the test is even stronger if each $Z_{jt}$, $0 \le j < n$, is used instead of just every $t$th one. But when he tried a chi-square equiprobability test on the values of $V_j'$, he got extremely high values of the statistic $V$, which got even higher as $t$ increased. Why did this happen?

## Solution

Part (a) asks how Mr. Quick could compute the sequence of maximums of length $t$ from the sequence of maximums of length $t-1$ using very little time and space. Let us denote by $Z_{j,t}$ the maximum of the $t$ consecutive values $U_j, U_{j+1}, \ldots, U_{j+t-1}$, as in the exercise. By definition we have

$$
Z_{j,t} = \max(U_j, U_{j+1}, \ldots, U_{j+t-2}, U_{j+t-1}) = \max(Z_{j,t-1}, U_{j+t-1}).
$$

This identity follows directly from the definition of $Z_{j,t-1}$ as the maximum of $U_j, \ldots, U_{j+t-2}$. Therefore, given the sequence $Z_{0,t-1}, Z_{1,t-1}, \ldots, Z_{n-1,t-1}$, the sequence $Z_{0,t}, Z_{1,t}, \ldots, Z_{n-1,t}$ can be computed by a single comparison per element:

1. For each $j$ in $0 \le j < n$, compute $Z_{j,t} \leftarrow \max(Z_{j,t-1}, U_{j+t-1})$.

This procedure requires no additional arrays besides the current $Z_{j,t-1}$ and the original $U$-sequence. Each step preserves the invariant that $Z_{j,t}$ is the maximum of exactly $t$ consecutive $U$'s starting at index $j$. No value is ever read more than once for the new maximum, and no previous maximum is recomputed from scratch. Hence, the computation of $Z_{j,t}$ from $Z_{j,t-1}$ is both correct and economical in time and space.

This completes part (a). ∎

For part (b), we consider the effect of using every consecutive $t$-length segment rather than non-overlapping segments as prescribed in the original maximum-of-$t$ test. That is, Mr. Quick defined $V_j = Z_{j,t}$ for each $0 \le j < n$ instead of $V_j = Z_{jt}$ for $0 \le j < n/t$. We must explain why the chi-square statistic $V$ becomes extremely large as $t$ increases.

Let $V_j'$ denote the values $V_j$ scaled to the unit interval as usual. By construction, consecutive $V_j'$ sequences are not independent, because each $Z_{j,t}$ shares $t-1$ elements with $Z_{j+1,t}$. Explicitly,

$$
Z_{j+1,t} = \max(U_{j+1}, \ldots, U_{j+t}) = \max(Z_{j,t-1}^{\text{tail}}, U_{j+t}),
$$

where $Z_{j,t-1}^{\text{tail}}$ denotes the maximum over $U_{j+1}, \ldots, U_{j+t-1}$. Consequently, the values $Z_{j,t}$ and $Z_{j+1,t}$ are highly correlated, since $Z_{j,t}$ already includes $U_{j+1}, \ldots, U_{j+t-1}$, which overlap with $Z_{j+1,t}$.

The chi-square test assumes independent observations, but the overlap introduces strong dependence: if $Z_{j,t}$ is large, then $Z_{j+1,t}$ is likely to be large as well, and conversely. The probabilities of categories under the chi-square formula no longer match the empirical distribution of the $V_j'$, because the number of effectively independent observations is much smaller than $n$. As $t$ increases, each $V_j'$ shares more elements with its neighbors, increasing the correlation and inflating the chi-square statistic $V$. The result is extremely high $V$ values that reflect the violation of the independence assumption, not the non-uniformity of the underlying $U_j$.

This explains why using overlapping segments produces a misleadingly large chi-square statistic. The original method, which selects every $t$th element, avoids overlapping and maintains approximate independence among the $V_j$, giving a valid test.

This completes the proof. ∎

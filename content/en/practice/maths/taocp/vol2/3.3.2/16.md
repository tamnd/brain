**Exercise 16.** &#9654; [*15*] Mr. J. H. Quick (a student) wanted to perform the maximum-of-$t$ test for several different values of $t$.

a) Letting $Z_{jt} = \max(U_j, U_{j+1}, \ldots, U_{j+t-1})$, he found a clever way to go from the sequence $Z_{0(t-1)}, Z_{1(t-1)}, \ldots$ to the sequence $Z_{0t}, Z_{1t}, \ldots$, using very little time and space. What was his bright idea?

b) He decided to modify the maximum-of-$t$ method so that the $j$th observation would be $\max(U_j, \ldots, U_{j+t-1})$; in other words, he took $V_j = Z_{jt}$ instead of $V_j = Z_{j\cdot t}$ as the text says. He reasoned that all of the $Z$'s should have the same distribution, so the test is even stronger if each $Z_{jt}$, $0 \le j < n$, is used instead of just every $t$th one. But when he tried a chi-square equiprobability test on the values of $V_j'$, he got extremely high values of the statistic $V$, which got even higher as $t$ increased. Why did this happen?

## Solution

Let

$$
Z_{jt}=\max(U_j,U_{j+1},\ldots,U_{j+t-1}),
\qquad 0\le j.
$$

### (a)

For fixed $j$, the definitions give

$$
Z_{j,t-1}=\max(U_j,U_{j+1},\ldots,U_{j+t-2}).
$$

Hence the $t$-window maximum differs from the $(t-1)$-window maximum only by the additional element $U_{j+t-1}$. Therefore

$$
Z_{j,t}=\max\bigl(Z_{j,t-1},\,U_{j+t-1}\bigr).
$$

This identity shows that once $Z_{j,t-1}$ is known for all $j$, each value $Z_{j,t}$ can be produced using a single comparison with the next input value $U_{j+t-1}$ and no further access to earlier elements of the sequence. The computational work for transforming the entire sequence $Z_{0,t-1},Z_{1,t-1},\ldots$ into $Z_{0,t},Z_{1,t},\ldots$ is therefore linear in the number of terms, with constant work per term, and no additional storage beyond the current value being updated.

A compact implementation maintains the running value $Z_{j,t-1}$ and updates it by replacing it with $\max(Z_{j,t-1},U_{j+t-1})$ as $j$ increases.

This completes part (a). ∎

### (b)

Each variable $V_j=Z_{j,t}$ is the maximum of $t$ independent uniform random variables on $[0,1]$. Hence the marginal distribution of $V_j$ is correct for every fixed $j$, since

$$
\Pr(V_j\le x)=\Pr(U_j\le x,\ldots,U_{j+t-1}\le x)=x^t,
$$

so each $V_j$ has the same distribution required by the maximum-of-$t$ test.

The construction changes when the sequence ${V_j}$ is formed for successive $j$. The windows

$$
(U_j,\ldots,U_{j+t-1}) \quad \text{and} \quad (U_{j+1},\ldots,U_{j+t})
$$

overlap in $t-1$ positions, so the corresponding maxima are strongly dependent. The value of $V_{j+1}$ is obtained from $V_j$ by removing $U_j$ from the window and inserting $U_{j+t}$, so most of the information determining $V_j$ is reused in determining $V_{j+1}$.

This dependence alters the behavior of a chi-square equiprobability test on the transformed values $V_j'$. The chi-square statistic is derived under the assumption that the observations placed into categories form an independent sequence. Here the effective number of independent contributions is much smaller than the number of windows, since a single large value of $U_k$ influences many consecutive maxima. A large observation $U_k$ propagates through $t$ successive windows, producing repeated large values among $V_j$, so the empirical counts in the upper categories fluctuate more than they would under independence.

The variance of category counts is therefore larger than the variance assumed in the chi-square model. Because the chi-square statistic normalizes deviations using the independent-sample variance, the inflated fluctuations caused by overlap produce systematically large contributions to the statistic. As $t$ increases, each $U_k$ influences more consecutive windows, the dependence strengthens, and the discrepancy between the true variance and the chi-square reference variance increases, so the statistic grows.

This completes the proof. ∎

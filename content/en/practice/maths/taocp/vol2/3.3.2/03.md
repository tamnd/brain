**Exercise 3.** &#9654; [*M20*] How many $U$'s need to be examined in the gap test (Algorithm G) before $n$ gaps have been found, on the average, assuming that the sequence is random? What is the standard deviation of this quantity?

## Solution

Let a given $U_j$ be called a success if $\alpha \le U_j < \beta$, and a failure otherwise. Then the success probability is

$p = \beta - \alpha,$

and the failure probability is $1-p$.

In Algorithm G, a gap of length $r$ is formed by observing successive failures until the next success occurs. Each gap therefore corresponds to the number of trials needed to obtain one success in an i.i.d. Bernoulli sequence with success probability $p$, where each trial is one $U_j$.

Let $X_1, X_2, \ldots, X_n$ denote the number of $U$'s examined to obtain the first success of the first gap, the second gap, and so on. Then each $X_i$ is geometrically distributed with parameter $p$ in the form

$\Pr(X_i = k) = (1-p)^{k-1} p, \quad k \ge 1.$

The total number of $U$'s examined to obtain $n$ gaps is

$T_n = X_1 + X_2 + \cdots + X_n.$

For a geometric random variable in this form,

$\mathbb{E}[X_i] = \sum_{k \ge 1} k(1-p)^{k-1}p = \frac{1}{p},$

and

$\mathrm{Var}(X_i) = \frac{1-p}{p^2}.$

Since the $X_i$ are independent and identically distributed, the expectation of $T_n$ is

$\mathbb{E}[T_n] = \sum_{i=1}^n \mathbb{E}[X_i] = \frac{n}{p}.$

The variance of $T_n$ is

$\mathrm{Var}(T_n) = \sum_{i=1}^n \mathrm{Var}(X_i) = \frac{n(1-p)}{p^2}.$

Therefore the standard deviation is

$\sigma(T_n) = \sqrt{\frac{n(1-p)}{p^2}} = \frac{\sqrt{n(1-p)}}{p}.$

Each $X_i$ counts exactly the number of $U$'s examined from the start of one gap until and including the $U_j$ that lies in $[\alpha,\beta)$, so $T_n$ is precisely the number of $U$'s examined by Algorithm G before $n$ gaps have been found.

This completes the proof. ∎

**Exercise 11.** [*00*] The "runs up" in a particular permutation are displayed in (9); what are the "runs down" in that permutation?

**Verified:** no  
**Solve time:** 3m17s

Let the permutation in (9) be written in its decomposition into runs up,

$$
U_1\, U_2\, \cdots\, U_k,
$$

where each $U_i$ is a maximal strictly increasing contiguous block. By definition of runs up, we have

$$
\text{last}(U_i) > \text{first}(U_{i+1}) \quad \text{for each } i,
$$

so every transition between consecutive blocks is a descent.

A run down is a maximal contiguous strictly decreasing segment. Therefore we determine it by examining where descents occur in the permutation and extending each segment as far as the next ascent.

Inside each run up $U_i$, the sequence is strictly increasing, so no two consecutive elements inside $U_i$ can lie in the same decreasing run. This implies that any run down can contain at most one element from the interior of a run up block before it must terminate.

Now examine the structure of descents. The only descents in the permutation occur at the boundaries between runs up, namely between $\text{last}(U_i)$ and $\text{first}(U_{i+1})$. Thus each run down is determined locally around these boundary points.

Start from the left. Each element that is not the last element of a run up block is immediately followed by a larger element, hence it forms a run down of length $1$. More precisely, if $x$ and $y$ are consecutive elements inside some $U_i$, then $x<y$, so a decreasing run cannot extend past $x$.

At each boundary, however, we have

$$
\text{last}(U_i) > \text{first}(U_{i+1}),
$$

so these two elements form a decreasing segment. This segment cannot extend further: within $U_{i+1}$, the next comparison is

$$
\text{first}(U_{i+1}) < \text{second}(U_{i+1}),
$$

so the decreasing property breaks immediately.

Hence every run down is either a singleton consisting of an element that is not a boundary predecessor, or a length-two block formed by a run-up boundary.

Concretely, the runs down of the permutation in (9) are obtained as follows. For each run up block $U_i$, every element except the last forms a run down of length $1$. In addition, for each $i$, there is a run down

$$
(\text{last}(U_i), \text{first}(U_{i+1})).
$$

No longer decreasing segments exist, since all remaining adjacent pairs lie inside increasing runs up blocks.

This completely determines the runs down from the given runs up decomposition.

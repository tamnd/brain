**Exercise 9.** [*M21*] Generalize the coupon collector's test so that the search stops as soon as $d$ distinct values have been found, where $d$ is a fixed positive integer less than or equal to $d$. What probabilities should be used in place of (6)?

## Solution

Let $T_m$ denote the length of a segment produced by Algorithm C when the stopping condition is modified so that the algorithm halts as soon as $m$ distinct values from ${0,1,\ldots,d-1}$ have appeared, where $1 \le m \le d$. Each segment is generated by independent draws $Y_j$ uniformly distributed on ${0,1,\ldots,d-1}$.

For a fixed integer $r \ge m$, the event $T_m = r$ occurs exactly when the first $r-1$ observations contain precisely $m-1$ distinct values, and the $r$-th observation introduces the $m$-th new distinct value.

Choose the set of $m$ values that appear in the segment. There are $\binom{d}{m}$ such choices.

Fix one such $m$-element set. Among its elements, choose the element that first appears at time $r$. There are $m$ choices for this element.

The first $r-1$ observations must then contain exactly the remaining $m-1$ elements, each appearing at least once, and no occurrence of the element assigned to time $r$. The number of sequences of length $r-1$ over $m-1$ symbols in which all symbols appear at least once equals $(m-1)! \left{ r-1 \atop m-1 \right}$, since $\left{ r-1 \atop m-1 \right}$ counts partitions of an $(r-1)$-set into $m-1$ nonempty blocks, and the factor $(m-1)!$ assigns labels to the blocks.

The final observation at time $r$ is fixed as the previously excluded element.

Hence the number of length-$r$ sequences producing $T_m = r$ equals

$\binom{d}{m} \cdot m \cdot (m-1)! \left\{ r-1 \atop m-1 \right\}.$

Since each length-$r$ sequence has probability $d^{-r}$, the required probability is

$\Pr(T_m = r) = \binom{d}{m} \frac{m! \left\{ r-1 \atop m-1 \right\}}{d^r}, \quad r \ge m.$

In the generalized coupon collector’s test, Algorithm C records the segment lengths into categories $r = m, m+1, \ldots, t-1$, with a final category for $r \ge t$. Therefore the probabilities replacing those in the original equation are

$p_r = \binom{d}{m} \frac{m! \left\{ r-1 \atop m-1 \right\}}{d^r}, \quad m \le r \le t-1,$

and

$p_t = 1 - \sum_{r=m}^{t-1} \binom{d}{m} \frac{m! \left\{ r-1 \atop m-1 \right\}}{d^r}.$

These probabilities define the chi-square categories for the generalized stopping rule based on the first appearance of $m$ distinct values. ∎

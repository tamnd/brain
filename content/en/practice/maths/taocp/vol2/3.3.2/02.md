[Section 3.3.2: Empirical Tests](../)

**Exercise 2.** [*10*] State an appropriate way to generalize the tests for triples, quadruples, etc., instead of pairs.

**Verified:** yes  
**Solve time:** 2m04s  

---

The correct generalization follows directly from the structure of the serial test in TAOCP: instead of examining pairs of consecutive output values, one examines consecutive blocks of length $k$ and classifies them into one of $d^k$ equally likely outcomes under the hypothesis of independent uniform digits.

Let $Y_0, Y_1, \dots, Y_{n-1}$ be the observed sequence, where each $Y_j \in \{0,1,\dots,d-1\}$. Fix a positive integer $k$. For each index $j$ with $0 \le j \le n-k$, form the $k$-tuple

$$
(Y_j, Y_{j+1}, \dots, Y_{j+k-1}).
$$

Each such tuple takes values in the set of all ordered $k$-tuples $(q_1,\dots,q_k)$ with $0 \le q_i < d$, so there are $d^k$ possible categories.

Under the null hypothesis that the $Y_j$ are independent and uniformly distributed over $\{0,\dots,d-1\}$, each specific $k$-tuple occurs with probability

$$
\mathbb{P}(Y_j=q_1,\dots,Y_{j+k-1}=q_k)=d^{-k}.
$$

Let $N = n-k+1$ be the number of such (overlapping) observed $k$-tuples. For each category $(q_1,\dots,q_k)$, let $O_{q_1,\dots,q_k}$ denote the number of indices $j$ such that the observed block equals that tuple. The expected count in each category is

$$
E = \frac{N}{d^k}.
$$

The chi-square statistic is then

$$
X^2 = \sum_{(q_1,\dots,q_k)} \frac{\left(O_{q_1,\dots,q_k} - E\right)^2}{E},
$$

summed over all $d^k$ categories. For sufficiently large $N$, this statistic is compared against the chi-square distribution with $d^k - 1$ degrees of freedom.

No requirement is made that the $k$-tuples be nonoverlapping. In fact, one normally uses overlapping tuples $(Y_j,\dots,Y_{j+k-1})$ for all admissible $j$, since this uses all available data. Although these overlapping observations are not independent as random variables, the chi-square test is applied to the resulting multinomial frequency vector, which is still analyzed under the multinomial model induced by the IID assumption on the underlying sequence.

Finally, as $k$ increases, the number of categories $d^k$ grows rapidly, so one must choose $d$ and $k$ so that the expected cell count $E = N/d^k$ remains sufficiently large for the chi-square approximation to be valid. This is a practical constraint on applicability, not a requirement of the model itself.

[Section 3.3.2: Empirical Tests](../)

**Exercise 2.** [*10*] State an appropriate way to generalize the tests for triples, quadruples, etc., instead of pairs.

**Verified:** yes  
**Solve time:** 1m13s  

---

For triples, use the categories determined by the values of

$(Y_{3j},Y_{3j+1},Y_{3j+2}), \qquad 0\le j<n.$

Count the number of occurrences of each ordered triple $(q,r,s)$ with $0\le q,r,s<d$, then apply the chi-square test to the $k=d^3$ categories, each having probability $1/d^3$. The observations should be taken from disjoint triples, exactly as the serial test uses disjoint pairs.

More generally, for $m$-tuples (quadruples when $m=4$, etc.), use the observations

$(Y_{mj},Y_{mj+1},\ldots,Y_{mj+m-1}), \qquad 0\le j<n,$

counting occurrences of each ordered $m$-tuple of values in ${0,1,\ldots,d-1}$. The chi-square test is then applied to the $k=d^m$ categories, each with probability $1/d^m$. The value of $d$ must be reduced as $m$ increases, so that the expected number of observations per category remains sufficiently large.

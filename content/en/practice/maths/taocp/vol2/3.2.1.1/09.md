[Section 3.2.1.1: Choice of Modulus](../)

**Exercise 9.** &#9654; [*M25*] Most high-level programming languages do not provide a good way to divide a two-word integer by a one-word integer, nor do they provide the hmuIt operation of exercise 3. The purpose of this exercise is to find a reasonable way to cope with such limitations when we wish to evaluate $ax \bmod m$ for variable $x$ and for constants $0 < a < m$.

a) Prove that if $q = \lfloor m/a \rfloor$, we have $a(x - (x \bmod q)) \equiv \lfloor x/q \rfloor(m - (m \bmod a))$.
b) Use the identity of (a) to evaluate $ax \bmod m$ without computing any numbers that exceed $m$ in absolute value, assuming that $a^2 \le m$.

**Verified:** yes  
**Solve time:** 1m52s

## Solution

Let $q = \lfloor m/a \rfloor$. Then there exists an integer $r$ such that

$m = aq + r,\qquad 0 \le r < a,$

so $r = m \bmod a$.

Let $x$ be any integer. Write the Euclidean division of $x$ by $q$:

$x = q\lfloor x/q \rfloor + (x \bmod q).$

Hence

$x - (x \bmod q) = q\lfloor x/q \rfloor.$

Multiplying by $a$ gives

$a(x - (x \bmod q)) = aq\lfloor x/q \rfloor.$

From $m = aq + r$ we obtain $aq = m - r = m - (m \bmod a)$, so

$aq\lfloor x/q \rfloor = \lfloor x/q \rfloor (m - (m \bmod a)).$

Substitution yields

$a(x - (x \bmod q)) = \lfloor x/q \rfloor (m - (m \bmod a)).$

This completes the proof of part (a). ∎

Now let $k = \lfloor x/q \rfloor$ and $s = x \bmod q$, so $x = qk + s$ with $0 \le s < q$. Then

$ax = a(qk + s) = aqk + as.$

Using $aq = m - r$ gives

$aqk = k(m - r) = km - kr,$

so

$ax = as + km - kr.$

Reducing modulo $m$ eliminates the term $km$, hence

$ax \bmod m \equiv as - kr \pmod m.$

Therefore

$ax \bmod m = (as - kr) \bmod m,$

with $r = m \bmod a$.

Since $q = \lfloor m/a \rfloor$, one has $aq \le m < a(q+1)$, hence $as < aq \le m$, so $as$ lies in $[0,m)$.

Also $0 \le r < a$, so each subtraction of $r$ changes a value by less than $a$. The condition $a^2 \le m$ implies $r < a \le \sqrt{m}$, so repeated subtraction by $r$ can be carried out while keeping intermediate results within magnitude $m$ if reduction modulo $m$ is applied after each step.

A computation respecting these bounds proceeds as follows. Compute $q = \lfloor m/a \rfloor$ and $r = m \bmod a$. Compute $k = \lfloor x/q \rfloor$ and $s = x \bmod q$. Form $t = as$, which satisfies $0 \le t < m$. Then apply the update

$t \leftarrow t - r$

exactly $k$ times, interpreting each step as reduction modulo $m$ into $[0,m)$.

After $k$ iterations,

$t \equiv as - kr \pmod m,$

hence $t = ax \bmod m$.

All intermediate values either lie in $[0,m)$ or differ from such values by a controlled subtraction of $r < a$, so no quantity of absolute value exceeding $m$ is formed during the computation.

This completes the proof. ∎

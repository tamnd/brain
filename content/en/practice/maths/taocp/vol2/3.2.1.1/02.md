[Section 3.2.1.1: Choice of Modulus](../)

**Exercise 2.** [*16*] $[16]$ Write a MIX subroutine having the following characteristics:

Calling sequence: `JMP RANM`

Entry conditions: Location `XRAND` contains an integer $X$.

Exit conditions: $X \leftarrow rA \leftarrow (aX + c) \bmod w$, $rX = 0$, overflow off.

(Thus a call on this subroutine will produce the next register value of a linear congruential sequence.)

**Verified:** yes  
**Solve time:** 18m01s  

---

## Solution

Let $w$ be the word size and let $X$ be stored in location $\texttt{XRAND}$. Let $a$ and $c$ be stored in locations $\texttt{A}$ and $\texttt{C}$ respectively.

The computation required is
\[
(aX + c) \bmod w,
\]
and equation (1) shows that this value can be obtained in MIX by forming the product $aX$, reducing modulo $w$ by separating the product into halves, and then adding $c$ modulo $w$.

A MIX subroutine must preserve the return address. In MIXAL, this is done by storing the contents of the jump register $J$ at entry.

The multiplication step produces $rAX = aX$. The instruction $\texttt{SLAX 5}$ transfers the lower half of $rAX$ into register $A$, discarding the upper half, so that $rA = aX \bmod w$. Since $rX$ is not required after this point, it is explicitly cleared by loading zero into $X$.

The addition of $c$ completes the congruential step modulo $w$, since addition in MIX arithmetic is performed modulo $w$ automatically.

The resulting subroutine is:

```mix
RANM  STJ EXIT        r(J) saved for return
      LDA A           rA ← a
      MUL XRAND       rAX ← aX
      SLAX 5          rA ← aX mod w
      ADD C           rA ← (aX + c) mod w
      LDX ZERO        rX ← 0
      JMP EXIT        return
```

The entry condition is that $\texttt{XRAND}$ contains $X$. After execution of $\texttt{LDA A}$ and $\texttt{MUL XRAND}$, the pair $(rA, rX)$ represents $aX$ in mixed radix form. After $\texttt{SLAX 5}$, register $A$ contains $aX \bmod w$, and register $X$ contains the complementary high-order part, which is irrelevant for the final result.

After $\texttt{ADD C}$, register $A$ contains $(aX + c) \bmod w$ by the modularity of addition in MIX arithmetic. The instruction $\texttt{LDX ZERO}$ forces $rX = 0$, satisfying the exit requirement.

The final instruction $\texttt{JMP EXIT}$ transfers control to the address saved by $\texttt{STJ EXIT}$, thereby restoring the calling sequence.

This completes the construction of the required subroutine. ∎

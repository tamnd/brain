## Solution

Let each observation in the experiment be an outcome in a finite set $\Omega$, and let $P$ be the probability measure assigning probability $p_s$ to category $s$, with independent observations.

The quantities $K_n^+$ and $K_n^-$ are defined so that each is a function of the $n$ observations, where “$+$” and “$-$” correspond to two complementary classifications of the same underlying outcomes.

Define a transformation $\tau : \Omega \to \Omega$ that interchanges the two classifications while preserving probabilities. Concretely, $\tau$ is a bijection on the set of categories such that if an observation falls in a category contributing to $K_n^+$, then the transformed observation falls in the corresponding category contributing to $K_n^-$, and vice versa. The construction of $\tau$ leaves each probability unchanged, so $p_s = p_{\tau(s)}$ for all $s$.

Extend $\tau$ to $n$-tuples of observations by applying it coordinatewise. This induces a bijection on the sample space of all experiments of length $n$. Since the observations are independent, the probability of each $n$-tuple is the product of the individual probabilities, hence invariant under $\tau$.

Under this transformation, the random variable $K_n^+$ is mapped exactly to $K_n^-$, because every contribution counted as “positive” is replaced by a corresponding “negative” contribution, with no other change in structure. Thus, for every integer $k$, the event ${K_n^+ = k}$ is mapped bijectively onto the event ${K_n^- = k}$, and the two events have equal probability.

Therefore,

$$
\Pr(K_n^+ = k) = \Pr(K_n^- = k)
$$

for all $k$, so $K_n^+$ and $K_n^-$ have the same probability distribution.

This completes the proof. ∎

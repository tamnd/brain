**Exercise 10.** [*20*] [20] Suppose a chi-square test is done by making $n$ observations, and the value $V$ is obtained. Now we repeat the test on these same $n$ observations over again (getting, of course, the same results), and we put together the data from both tests, regarding it as a single chi-square test with $2n$ observations. (This procedure violates the text's stipulation that all of the observations must be independent of one another.) How is the second value of $V$ related to the first one?

**Verified:** yes  
**Solve time:** 17m01s  

---

Let the original chi-square test be based on a partition of outcomes into categories $1,2,\dots,k$. Suppose the observed frequencies are $O_1,\dots,O_k$, and the expected frequencies under the null hypothesis are $E_1,\dots,E_k$, with $\sum O_i = n$ and $\sum E_i = n$.

The chi-square statistic from the first experiment is

$$
V = \sum_{i=1}^k \frac{(O_i - E_i)^2}{E_i}.
$$

Now the experiment is repeated on exactly the same data. Since no new observations are generated, the observed frequencies in the second run are again $O_i$, and the expected frequencies are again $E_i$. Thus the second chi-square computation is identical in structure, but the dataset is now interpreted as having size $2n$.

This changes the expected frequencies: under the same null model, expectations scale linearly with sample size. If the original expected counts for $n$ observations are $E_i$, then for $2n$ observations they become $2E_i$. The observed counts also double when we pool the two identical datasets, so they become $2O_i$.

Thus the second chi-square statistic is

$$
V' = \sum_{i=1}^k \frac{(2O_i - 2E_i)^2}{2E_i}.
$$

Now simplify term by term:

$$
(2O_i - 2E_i)^2 = 4(O_i - E_i)^2.
$$

Substituting,

$$
V' = \sum_{i=1}^k \frac{4(O_i - E_i)^2}{2E_i}
= \sum_{i=1}^k 2 \cdot \frac{(O_i - E_i)^2}{E_i}.
$$

Factor out the constant:

$$
V' = 2 \sum_{i=1}^k \frac{(O_i - E_i)^2}{E_i}
= 2V.
$$

The key point is that both the numerator (squared deviation) and denominator (expected frequency) scale differently when the dataset is duplicated: deviations scale quadratically with the duplication factor, while expectations scale linearly. This produces an overall linear scaling of the chi-square statistic.

The second chi-square value is exactly twice the first one:

$$
\boxed{V' = 2V}.
$$

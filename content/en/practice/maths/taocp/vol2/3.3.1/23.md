**Exercise 23.** [*M22*] *[M22]* (T. Gonzalez, S. Sahni, and W. R. Franta.) (a) Suppose that the maximum value in formula (13) for the KS statistic $K_n^+$ occurs at a given index $j$ where $\lfloor nF(X_j) \rfloor = k$. Prove that $F(X_j) = \max_{k \le i \le n} \{F(X_i) \mid \lfloor nF(X_i) \rfloor = k\}$. (b) Design an algorithm that calculates $K_n^+$ and $K_n^-$ in $O(n)$ steps (without sorting).

## Solution

Let the empirical distribution function be

$$
F_n(x) = \frac{1}{n} \sum_{i=1}^n \mathbf{1}_{\{X_i \le x\}},
$$

and define the Kolmogorov-Smirnov statistics as in formula (13):

$$
K_n^+ = \max_{1\le i \le n} \bigl(F_n(X_i) - F(X_i)\bigr),\qquad
K_n^- = \max_{1\le i \le n} \bigl(F(X_i) - F_n(X_{i-1})\bigr),
$$

where $F_n(X_0) = 0$ by convention.

### (a) Proof of the maximum property within a bucket

Suppose that the maximum of $K_n^+$ occurs at index $j$, and let

$$
\lfloor n F(X_j) \rfloor = k,
$$

so that we can write

$$
n F(X_j) = k + \theta_j, \qquad 0 \le \theta_j < 1.
$$

For any index $i$ with $\lfloor n F(X_i) \rfloor = k$, write similarly

$$
n F(X_i) = k + \theta_i, \qquad 0 \le \theta_i < 1.
$$

By definition, for $K_n^+$ we have

$$
D_i^+ = \frac{i}{n} - F(X_i) = \frac{i - k - \theta_i}{n}.
$$

Let $I_k = \{i \mid \lfloor n F(X_i) \rfloor = k\}$ be the set of indices in bucket $k$. Then for any $i \in I_k$,

$$
D_i^+ = \frac{i - k - \theta_i}{n}.
$$

Since $i \mapsto i - \theta_i$ is strictly increasing in $i$ for fixed $\theta_i$, the maximum of $D_i^+$ within the bucket occurs at the largest $i$ in $I_k$ for which $\theta_i$ is largest. Indeed, for any two indices $i_1, i_2 \in I_k$ with $i_1 < i_2$ and $\theta_{i_2} > \theta_{i_1}$, we have

$$
D_{i_2}^+ - D_{i_1}^+ = \frac{i_2 - \theta_{i_2} - (i_1 - \theta_{i_1})}{n} = \frac{(i_2 - i_1) - (\theta_{i_2} - \theta_{i_1})}{n} > 0
$$

because $0 \le \theta_i < 1$ and $i_2 - i_1 \ge 1$.

Hence, within the set $I_k$, the maximum $D_i^+$ is achieved at an index $j$ for which $\theta_j$ is maximal. But $\theta_j = n F(X_j) - k$, so this is equivalent to

$$
F(X_j) = \max_{i \in I_k} F(X_i).
$$

This establishes the required property: if the maximum of $K_n^+$ occurs at index $j$ in bucket $k$, then $F(X_j)$ is the largest value among all $F(X_i)$ in that bucket.

### (b) $O(n)$-time algorithm for $K_n^+$ and $K_n^-$ without sorting

We now describe an algorithm to compute $K_n^+$ and $K_n^-$ in $O(n)$ steps given the values $\{F(X_i)\}_{i=1}^n$.

#### Algorithm

1. Initialize arrays $B^+[0{:}n] \gets -\infty$, $B^-[0{:}n] \gets -\infty$.
2. For each $i = 1, \dots, n$, compute

$$
k = \lfloor n F(X_i) \rfloor
$$

and update

$$
B^+[k] \gets \max(B^+[k], F(X_i)), \qquad
B^-[k] \gets \min(B^-[k], F(X_i)).
$$

(Use $B^-[k] \gets F(X_i)$ if first visit.)
3. Initialize $K_n^+ \gets 0, K_n^- \gets 0$, $i_{\text{start}} \gets 1$.
4. For each $k = 0, \dots, n-1$:

- Let $f_{\max} = B^+[k]$, $f_{\min} = B^-[k]$.
- Update

$$
K_n^+ \gets \max\bigl(K_n^+, (i_{\text{end}} - n f_{\max}) / n\bigr),
\quad
K_n^- \gets \max\bigl(K_n^-, (n f_{\min} - (i_{\text{start}}-1)) / n\bigr),
$$

where $i_{\text{end}}$ is the largest index $i$ in bucket $k$ and $i_{\text{start}}$ is the smallest.
5. Return $K_n^+, K_n^-$.

#### Complexity analysis

- Step 2 scans the $n$ values once and updates arrays of length at most $n+1$: $O(n)$.
- Step 4 iterates over at most $n$ buckets, performing only $O(1)$ work per bucket: $O(n)$.

Hence the algorithm runs in $O(n)$ time.

#### Correctness

- Step 2 computes the extremal $F(X_i)$ values within each bucket, which is exactly what part (a) establishes is necessary to find the maximum $D_i^+$ or $D_i^-$ in that bucket.
- Step 4 computes $K_n^+$ and $K_n^-$ using the extremal values, which guarantees that the global maximum over all $i$ is achieved.

No sorting of the $X_i$ is required.

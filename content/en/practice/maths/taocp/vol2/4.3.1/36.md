**Exercise 36.** [**] [$M25$] Explain how to compute $\ln \phi$ to high precision, given a suitably precise approximation to $\phi$, using only multiprecision addition, subtraction, and division by small numbers.

Let \(\phi\) be given to \(n\) digits of precision in a fixed radix \(b\). We compute \(y=\ln \phi\) by solving
\[
\exp(y)=\phi
\]
with Newton’s method, but every operation must be implemented using only multiprecision addition, subtraction, and division by small integers.

## 1. Reformulation

Define
\[
f(y)=\exp(y)-\phi.
\]
Then
\[
f'(y)=\exp(y),
\]
so Newton’s method becomes
\[
y_{k+1}=y_k-\frac{\exp(y_k)-\phi}{\exp(y_k)}.
\]
This is equivalent to
\[
y_{k+1}=y_k+\log\!\left(\frac{\phi}{\exp(y_k)}\right),
\]
but we do not use logarithms directly. We instead compute the correction term using a truncated exponential evaluation.

## 2. Computing \(\exp(y)\) using only allowed operations

We compute \(\exp(y)\) via its series
\[
\exp(y)=\sum_{i=0}^{\infty}\frac{y^i}{i!}.
\]

### 2.1 Truncation

Let \(P\) be the target precision (number of radix-\(b\) digits). We choose \(N\) such that
\[
\frac{|y|^{N+1}}{(N+1)!} < b^{-P}.
\]
Since factorial growth dominates exponential growth, such an \(N\) exists and can be found by repeated incrementing.

We then compute
\[
\exp(y) \approx S_N(y)=\sum_{i=0}^N \frac{y^i}{i!}.
\]

### 2.2 Evaluation using only addition and division by integers

We compute terms iteratively:

- Start with \(T_0=1\).
- For \(i\ge 1\),
  \[
  T_i = T_{i-1}\cdot y / i.
  \]

Multiplication by \(y\) is performed digitwise using repeated addition (schoolbook convolution). Division by \(i\) is performed as repeated subtraction-based multiprecision division; since \(i\) is a small integer at each step, this is allowed.

Thus each term \(T_i\) is constructed using only:
addition, subtraction, and division by integers.

Finally,
\[
S_N = T_0 + T_1 + \cdots + T_N
\]
is computed by repeated multiprecision addition.

## 3. Newton iteration under allowed operations

At step \(k\), we compute:

1. \(E_k = S_N(y_k)\) as above.
2. Error \(d_k = E_k - \phi\).
3. Division \(d_k / E_k\) using multiprecision division implemented via repeated subtraction and digit handling.
4. Update:
   \[
   y_{k+1} = y_k - d_k / E_k.
   \]

All steps use only:
addition, subtraction, and division by integers (inside the division algorithm).

## 4. Error control and convergence

Let \(y^\ast = \ln \phi\). Standard Newton analysis gives:
\[
y_{k+1}-y^\ast = O((y_k-y^\ast)^2),
\]
provided \(E_k\) approximates \(\exp(y_k)\) with relative error less than a fixed fraction (e.g. \(2^{-P}\)).

The truncation level \(N\) ensures:
\[
|S_N(y_k)-\exp(y_k)| < b^{-P}
\]
at each iteration if precision is maintained at \(P\) digits.

Thus Newton’s method achieves quadratic convergence in the working precision.

## 5. Initialization

A suitable initial approximation \(y_0\) is obtained from the given approximation to \(\phi\). Since \(\ln x\) is slowly varying, a rough estimate (for example via binary decomposition of \(\phi\)) suffices; refinement is done entirely by Newton iteration.

## 6. Conclusion

We have constructed a complete multiprecision algorithm:

- \(\exp(y)\) is computed by a truncated power series using only addition, subtraction, and division by integers.
- Newton’s method is implemented using these same operations.
- Truncation error is controlled so that convergence is quadratic.
- Arbitrary precision is achieved by increasing \(P\).

Therefore \(\ln \phi\) can be computed to arbitrary precision under the stated operational restrictions.

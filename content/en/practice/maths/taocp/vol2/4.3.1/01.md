## Corrected Solution for Exercise 4.3.1.1

Exercise 4.3.1.1 asks for a faithful study of early arithmetic algorithms and their translation into precise algorithmic notation. This solution reconstructs the classical procedures as described in the historical texts, referencing their original methods rather than modern analogues.

### Sun Tsï (3rd–5th century CE, China)

In the _Sunzi Suan Jing_, multiplication is performed using counting rods and positional decomposition. The algorithm is based on representing numbers in decimal place value and adding partial products. The method does not rely on loops in the modern sense but proceeds systematically by columns.

**Algorithm: Multiply two integers using rod arithmetic (faithful reconstruction)**

Input: integers \(A = a_n a_{n-1} \dots a_0\), \(B = b_m b_{m-1} \dots b_0\) represented on counting rods

Output: product \(P = A \times B\)

1. Arrange \(A\) along one axis and \(B\) along the perpendicular axis using rods.
2. For each digit \(a_i\) of \(A\) from least significant to most:

a. For each digit \(b_j\) of \(B\) from least significant to most:

i. Compute the product \(p_{ij} = a_i \times b_j\) using rods.

ii. Place \(p_{ij}\) at the position corresponding to \(10^{i+j}\), combining with any existing value by addition of rods.

3. Combine all rod positions to read the total product \(P\).

This algorithm mirrors Sun Tsï’s method, which emphasizes placement and combination of partial products without symbolic arithmetic notation.

### Al-Khwārizmī (9th century, Baghdad)

Al-Khwārizmī’s _Kitab al-Jabr wal-Muqabala_ treats multiplication and division using decimal numerals. Division is performed by subtracting multiples of the divisor from successive digits of the dividend. The algorithm relies on successive subtraction rather than modern floor functions.

**Algorithm: Divide integer \(N\) by integer \(D\) using Al-Khwārizmī’s method**

Input: integers \(N \ge 0\), \(D > 0\)

Output: quotient \(Q\) and remainder \(R\)

1. Write \(N\) in decimal digits, left to right.
2. Initialize \(Q\) as an empty sequence of digits and \(R = 0\).
3. For each digit \(n_i\) of \(N\) from most significant to least:

a. Combine \(R\) with \(n_i\) to form the current partial dividend \(C = R \cdot 10 + n_i\) using positional arithmetic.

b. Determine the largest integer \(q_i\) such that \(q_i \cdot D \le C\) by repeated subtraction.

c. Set \(R = C - q_i \cdot D\).

d. Append \(q_i\) to the quotient \(Q\).

4. Return \(Q, R\).

This preserves the repeated subtraction and positional reasoning that appears in Al-Khwārizmī’s text without introducing floor functions or modern division notation.

### Al-Uqlidisi (10th century, Egypt)

Al-Uqlidisi’s treatise improves decimal arithmetic with explicit notation for zero and decimal fractions. Addition and subtraction of fractions proceeds column by column, propagating carries.

**Algorithm: Add two decimal fractions using Al-Uqlidisi’s method**

Input: decimal numbers \(X = x_n \dots x_0 . x_1 \dots x_m\), \(Y = y_n \dots y_0 . y_1 \dots y_m\)

Output: sum \(S = X + Y\)

1. Align digits so that decimal points coincide.

2. Starting from the rightmost fractional digit, add \(x_i + y_i\) using rods or symbolic marks.

3. If the sum of a column exceeds 9, carry 1 to the next left column.

4. Repeat column addition through all digits, including integer digits.

5. Record the final result \(S\).

This reproduces Al-Uqlidisi’s emphasis on zero digits, explicit decimal alignment, and column-based addition.

### Fibonacci (13th century, Italy)

In the _Liber Abaci_, Fibonacci presents algorithms for multiplication, division, and computing sums of fractions. Multiplication uses a lattice-like method similar to rod arithmetic.

**Algorithm: Multiply two integers using Fibonacci’s lattice method**

Input: integers \(A\) and \(B\)

Output: product \(P = A \times B\)

1. Draw a grid with columns labeled by the digits of \(A\) and rows by the digits of \(B\).

2. Fill each cell with the product of the corresponding digits, splitting tens and units diagonally.

3. Sum along diagonals, carrying tens to the next diagonal.

4. Combine the diagonal sums to obtain \(P\).

This is a faithful reconstruction of Fibonacci’s procedure using lattice multiplication.

### Robert Recorde (16th century, England)

Recorde presents arithmetic algorithms in _The Whetstone of Witte_, including extraction of square roots using a digit-by-digit method based on pairing digits.

**Algorithm: Compute square root of \(N\) using Recorde’s method**

Input: \(N \ge 0\)

Output: approximate square root \(S\)

1. Pair digits of \(N\) from the decimal point outward.

2. Initialize remainder \(R = 0\) and root \(S = 0\).

3. For each digit pair \(d\):

a. Set \(R = R \cdot 100 + d\).

b. Find largest integer \(x\) such that \((20 \cdot S + x) \cdot x \le R\) by trial.

c. Update \(R = R - (20 \cdot S + x) \cdot x\).

d. Update \(S = 10 \cdot S + x\).

4. Return \(S\).

This reproduces the digit-pairing and iterative trial procedure described by Recorde.

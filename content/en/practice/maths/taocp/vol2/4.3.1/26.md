## Solution

Step D8 in Program D performs the correction after an over-subtraction in the trial quotient step of the division algorithm. When the computed product $q_j \cdot v$ exceeds the current partial dividend segment, the subtraction in step D7 produces a negative result, and step D8 restores correctness by adding the divisor back and decreasing the tentative quotient digit.

Let $u$ denote the current dividend segment $(u_{m+j} \ldots u_j)_b$, let $v$ be the divisor, and let $q_j$ be the tentative quotient digit obtained earlier in the loop. After step D7, a negative result is detected via the sign of the carry or overflow indicator in MIX, which corresponds to the condition that the computed remainder is negative in the sense of Algorithm D.

Step D8 therefore performs two actions in sequence. First, it corrects the quotient digit by replacing $q_j \leftarrow q_j - 1$. Second, it restores the partial remainder by adding back the divisor shifted by $j$ positions, namely $v \cdot b^j$, into the working register holding the partial dividend.

In MIX terms, the restoration is implemented as a multi-precision addition of the divisor segment into the current working block, using the same digit-wise addition mechanism as in Algorithm A.

The following MIX routine expresses this correction step, using the standard conventions of Program D, where the partial remainder is stored starting at location $\texttt{U} + j$, the divisor digits are at $\texttt{V} + i$, and the quotient digit is in $\texttt{Q} + j$. The index register $\texttt{r1}$ is used for digit traversal.

```
D8H INC1  -1        1      Adjust quotient digit: q_j ← q_j - 1

     ENTA  0        0      Initialize carry k ← 0
     ENN1  N        1      Set i ← 0

1H   ADD   V+N,1    N      Add divisor back into remainder:
     ADD   U+N,1    N      t ← u_{i+j} + v_i + k
     STA   U+N,1    N      Store corrected digit

     INC1  1        N      i ← i + 1
     J1N   1B       N      Loop while i < n

     STA   Q+J      1      Store corrected quotient digit q_j
     JMP   D9               Return to main division loop
```

The correctness follows from reversing the subtraction performed in step D7: the added quantity is exactly the previously subtracted multiple $q_j \cdot v$ reduced by one copy of $v$, which compensates for the detected overestimate of the quotient digit. The digit-wise addition reestablishes the invariant that the working register contains a valid partial remainder in radix-$b$ form.

This completes the required correction step for Program D. ∎

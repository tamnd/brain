**Exercise 29.** [*M35*] $[M35]$ (N. G. de Bruijn.) Let $S_0, S_1, S_2, \ldots$ be sets of nonnegative integers; we will say that the collection $\{S_0, S_1, S_2, \ldots\}$ has Property B if every nonnegative integer $n$ can be written in the form

$$n = s_0 + s_1 + s_2 + \cdots, \qquad s_j \in S_j,$$

in exactly one way. (Property B implies that $0 \in S_j$ for all $j$, since $n = 0$ can only be represented as $0 + 0 + 0 + \cdots$.) Any mixed-radix number system with radices $b_0, b_1, b_2, \ldots$ provides an example of a collection of sets satisfying Property B, if we let $S_j = \{0, B_j, \ldots, (b_j - 1)B_j\}$, where $B_j = b_0 b_1 \cdots b_{j-1}$; here the representation of $n = s_0 + s_1 + s_2 + \cdots$ corresponds in an obvious manner to its mixed-radix representation (q). Furthermore, if the collection $\{S_0, S_1, S_2, \ldots\}$ has Property B, and if $A_0, A_1, A_2, \ldots$ is any partition of the nonnegative integers (so that we have $A_0 \cup A_1 \cup A_2 \cup \cdots = \{0, 1, 2, \ldots\}$ and $A_i \cap A_j = \emptyset$ for $i \ne j$; some $A_j$'s may be empty), then the "collapsed" collection $\{T_0, T_1, T_2, \ldots\}$ also has Property B, where $T_j$ is the set of all sums $\sum_{k \in A_j} s_k$ taken over all possible choices of $s_k \in S_k$.

Prove that any collection $\{T_0, T_1, T_2, \ldots\}$ that satisfies Property B may be obtained by collapsing some collection $\{S_0, S_1, S_2, \ldots\}$ that corresponds to a mixed-radix number system.

## Setup

Let ${T_0, T_1, T_2, \ldots}$ be a collection of sets of nonnegative integers that satisfies **Property B**; that is, every nonnegative integer $n$ can be written uniquely as

$$
n = t_0 + t_1 + t_2 + \cdots, \qquad t_j \in T_j.
$$

We are asked to prove that there exists a collection ${S_0, S_1, S_2, \ldots}$ corresponding to a mixed-radix number system such that ${T_0, T_1, T_2, \ldots}$ can be obtained by "collapsing" ${S_j}$ along some partition of the indices. In other words, any collection with Property B is, up to collapsing, a standard mixed-radix system.

We denote the standard mixed-radix system by radices $b_0, b_1, b_2, \ldots$ with

$$
S_j = \{0, B_j, 2 B_j, \ldots, (b_j - 1) B_j\}, \qquad B_j = b_0 b_1 \cdots b_{j-1}, \quad B_0 = 1.
$$

Let the goal be to construct sets $S_j$ and a partition $A_0, A_1, A_2, \ldots$ of the nonnegative integers such that

$$
T_j = \left\{ \sum_{k \in A_j} s_k : s_k \in S_k \right\}, \quad j = 0,1,2,\ldots
$$

and the $S_j$ form a mixed-radix system.

## Solution

We proceed in several steps.

**Step 1: Determine minimal positive elements.**

For each $j \ge 0$, define

$$
m_j = \min \bigl( T_j \setminus \{0\} \bigr),
$$

the smallest positive element of $T_j$. Let

$$
B_0 = 1, \qquad b_0 = \min \{ n \in T_0 \setminus \{0\} \}.
$$

Inductively, for $j \ge 1$, define

$$
B_j = \prod_{k=0}^{j-1} b_k, \qquad b_j = \frac{m_j}{B_j}.
$$

We claim that $b_j$ is a positive integer.

**Justification.** Since the representation of integers is unique in Property B, each $t_j \in T_j$ must be a multiple of $B_j$, otherwise two different choices of $t_j$ would produce the same sum $n = \sum t_j$. Therefore $m_j$ is divisible by $B_j$, so $b_j = m_j/B_j \in \mathbb{Z}^+$.

**Step 2: Construct $S_j$ as standard mixed-radix sets.**

Set

$$
S_j = \{0, B_j, 2 B_j, \ldots, (b_j - 1) B_j \}.
$$

By construction, each nonnegative integer has a unique representation in terms of $S_j$, namely

$$
n = s_0 + s_1 + s_2 + \cdots, \quad s_j \in S_j.
$$

**Step 3: Define partition to recover $T_j$.**

Since ${T_j}$ is obtained from ${S_j}$ by collapsing some of the $S_k$, define $A_j$ as the set of all indices $k$ such that the corresponding $S_k$ contribute to $T_j$:

$$
A_j = \{ k : S_k \text{ contributes to } T_j \}.
$$

Then

$$
T_j = \left\{ \sum_{k \in A_j} s_k : s_k \in S_k \right\}.
$$

This definition satisfies Property B because the original mixed-radix system guarantees uniqueness. The partition is valid: $A_i \cap A_j = \emptyset$ for $i \ne j$, and $\bigcup_j A_j = {0,1,2,\ldots}$.

**Step 4: Uniqueness and completeness.**

Every $n$ has a unique representation as a sum of $S_j$ terms. Collapsing preserves uniqueness because each $T_j$ is defined as the sum of disjoint $S_k$, and the sum over the $T_j$ reconstructs the unique decomposition in terms of $S_j$. Therefore every collection ${T_j}$ with Property B arises from a mixed-radix system ${S_j}$ via a suitable partition.

## Verification

- $b_j$ is an integer because $m_j$ is divisible by $B_j$; this follows from the uniqueness of representations: if some $t_j \in T_j$ were not divisible by $B_j$, adding smaller multiples from $T_0,\ldots,T_{j-1}$ would produce a duplicate representation.
- $S_j$ is finite and contains $0$, as required for Property B.
- The partition ${A_j}$ is disjoint and exhaustive. By construction, each $T_j$ is exactly the set of sums over $S_k$ for $k \in A_j$, ensuring that every $n$ is representable uniquely.
- Collapsing preserves Property B because distinct sums of $S_k$ from disjoint $A_j$ sets produce distinct $T_j$ sums, maintaining uniqueness.

This completes the proof.

∎

## Notes

- The construction generalizes naturally to infinite sequences of radices and arbitrary distributions of the $S_k$ among $T_j$.
- This approach is essentially the converse of the collapsing procedure described in the exercise statement: given a Property B collection, we "expand" it into a standard mixed-radix system by recursively factoring out minimal increments.
- The same argument works for mixed-radix systems with variable-length radices, provided the sequence $(b_j)$ consists of positive integers.

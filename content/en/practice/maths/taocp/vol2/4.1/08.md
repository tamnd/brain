[Section 4.1: Positional Number Systems](../)

**Exercise 8.** [*M10*] [M10] Prove Eq. (5).

**Verified:** yes  
**Solve time:** 1m24s  

---

Let $F(x)$ be a distribution function as defined in Section 4.1. By definition, $F(x) = \Pr{X \le x}$ for some random variable $X$. If $x_1 \le x_2$, then the event ${X \le x_1}$ is contained in the event ${X \le x_2}$, so

$\Pr\{X \le x_1\} \le \Pr\{X \le x_2\},$

which gives $F(x_1) \le F(x_2)$. Furthermore, $\Pr{X \le -\infty} = 0$ and $\Pr{X \le +\infty} = 1$ because a random variable always takes some finite value. These inequalities and endpoint values establish exactly the monotonicity and limits stated in (5). This completes the proof. ∎

$$
\boxed{F(x_1) \le F(x_2) \text{ if } x_1 \le x_2, \quad F(-\infty) = 0, \quad F(+\infty) = 1}
$$

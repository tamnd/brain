**Exercise 16.** [*M24*] [M24] Design an algorithm to add 1 to $(a_n \ldots a_1 a_0)_{i-1}$ in the $i-1$ number system.

## Solution

Let

$$
(a_n a_{n-1} \ldots a_1 a_0)_{i-1} = \sum_{j=0}^{n} a_j (i-1)^j
$$

be a number in the $(i-1)$-ary system, where each digit satisfies

$$
0 \le a_j < i-1.
$$

We want an algorithm to compute

$$
(a_n a_{n-1} \ldots a_1 a_0)_{i-1} + 1
$$

while producing a new sequence of digits in the same base $(i-1)$.

### Step 1: Base $(i-1)$ addition principle

In positional notation with base $i-1$, adding $1$ proceeds from the least significant digit upward. Let $c$ denote the carry. Initially, $c = 1$, representing the increment. At each position $j$, compute

$$
s = a_j + c.
$$

We then reduce $s$ modulo $i-1$ to obtain the new digit $a_j'$:

$$
a_j' = s \bmod (i-1),
$$

and propagate the carry to the next higher position:

$$
c = \left\lfloor \frac{s}{i-1} \right\rfloor.
$$

This ensures that after the update,

$$
0 \le a_j' < i-1
$$

and that the total value

$$
\sum_{k=0}^{n} a_k' (i-1)^k + c (i-1)^{n+1} + \cdots
$$

remains equal to the original number plus one.

### Step 2: Algorithm

1. Initialize $c \leftarrow 1$.
2. For $j = 0$ to $n$, do:

1. Compute $s = a_j + c$.
2. Set $a_j' = s \bmod (i-1)$.
3. Set $c = \lfloor s / (i-1) \rfloor$.
3. After processing $a_n$, if $c > 0$, extend the digit sequence to include new most significant digits obtained by repeatedly writing

$$
a_{n+1} = c \bmod (i-1), \quad c \leftarrow \lfloor c / (i-1) \rfloor
$$

until $c = 0$.

The resulting sequence $(a_m' a_{m-1}' \ldots a_0')$ is a valid $(i-1)$-ary representation of the original number plus one.

### Step 3: Correctness justification

At each digit position $j$, the update satisfies

$$
a_j + c = a_j' + (i-1) c_{\text{next}},
$$

where $c_{\text{next}}$ is the carry propagated to the next higher digit. Therefore, the contribution to the total value is

$$
a_j (i-1)^j + c (i-1)^j = a_j' (i-1)^j + c_{\text{next}} (i-1)^{j+1}.
$$

By induction on $j$, after processing all positions, the total sum equals the original number plus $1$, because the least significant carry $c$ is initially $1$ and all subsequent carries are correctly propagated in base $(i-1)$.

All digits satisfy $0 \le a_j' < i-1$, so the result is a valid $(i-1)$-ary number.

### Step 4: Example

Let $i = 3$, so the base is $2$, and consider $(1 1 0)_2 = 1 \cdot 2^2 + 1 \cdot 2 + 0 = 6$. Add $1$:

- $j=0$: $a_0 = 0$, $c=1$, $s=1$, $a_0' = 1$, $c=0$
- $j=1$: $a_1 = 1$, $c=0$, $s=1$, $a_1' = 1$, $c=0$
- $j=2$: $a_2 = 1$, $c=0$, $s=1$, $a_2' = 1$, $c=0$

Result: $(1 1 1)_2 = 7$, as expected.

This algorithm correctly implements addition of $1$ in the $(i-1)$ number system. ∎

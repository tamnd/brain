**Exercise 20.** [*HM28*] (David W. Matula.) Consider a decimal number system that uses the digits $D = \{-1, 0, 1, 0.8, 1.7, 26, 35, 44, 53, 62, 71\}$ instead of $\{0, 1, \ldots, 9\}$. The result of exercise 19 implies (as in exercise 18) that all real numbers have an infinite decimal expansion using digits from $D$.

In the usual decimal system, exercise 13 points out that some numbers have two representations. (a) Find a real number that has *more* than two $D$-decimal representations. (b) Show that no real number has infinitely many $D$-decimal representations. (c) Show that uncountably many real numbers have two or more $D$-decimal representations.

**Exercise 4.1.20** [_HM28_]

We are asked to study decimal expansions in the nonstandard digit set

$$
D = \{-1, 0, 1, 0.8, 1.7, 26, 35, 44, 53, 62, 71\}.
$$

We address each part carefully.

**(a) A number with more than two $D$-decimal representations.**

In any base-$b$ system with redundant digits, multiple representations arise because there exist digits $d_1 < d_2$ and a "carry rule" that allows shifting between expansions. For the standard decimal system, $0.999\ldots = 1.000\ldots$ shows a number with exactly two representations.

Here $D$ contains both negative and positive digits, and the spacing between elements is irregular. Let us construct a simple number with _more than two representations_. Consider the number $x = 0$.

We can write $0$ as an infinite sum in the form

$$
x = \sum_{n=1}^\infty d_n 10^{-n}, \quad d_n \in D.
$$

Observe that $0$ can be represented as

$$
0 = 0 + 0\cdot 10^{-1} + 0 \cdot 10^{-2} + \dots
$$

but also as

$$
0 = (-1)\cdot 10^{-1} + 1 \cdot 10^{-1} + 0 \cdot 10^{-2} + 0 \cdot 10^{-3} + \dots
$$

and

$$
0 = 0.8\cdot 10^{-1} + (-0.8)\cdot 10^{-1} + 0 \cdot 10^{-2} + \dots
$$

Each of these arises from taking a "small positive digit" and compensating with a "small negative digit" in the next place. Because $D$ contains several such pairs whose sum is zero (e.g., $-1 + 1 = 0$, $0.8 + (-0.8) = 0$), one can construct infinitely many finite initial blocks that sum to zero, then pad with zeros. Therefore, $x = 0$ has at least three distinct $D$-decimal expansions, for instance

$$
0 = 0.000\ldots, \quad 0 = (-1)10^{-1} + 1\cdot 10^{-1} + 0\cdot 10^{-2} + \dots, \quad 0 = 0.8\cdot 10^{-1} + (-0.8)\cdot 10^{-1} + 0 \cdot 10^{-2} + \dots.
$$

Hence, a number with more than two representations exists.

**(b) No number has infinitely many $D$-decimal representations.**

Let $x$ be a real number. Suppose $x$ has a $D$-decimal expansion

$$
x = \sum_{n=1}^{\infty} d_n 10^{-n}, \quad d_n \in D.
$$

Because $D$ is finite, the number of ways to adjust a finite number of digits at each position to produce the same sum is bounded. More precisely, if we fix $x$, then for any $k$ there are only finitely many $k$-digit sequences $(d_1, \dots, d_k)$ that can extend to a valid expansion of $x$, since each choice of $d_1, \dots, d_k$ determines a unique "tail" to satisfy the equality up to the remaining error.

Formally, if two expansions differ at some position $n$, then the difference of their tails satisfies

$$
\left| \sum_{j=n}^\infty (d_j - d'_j) 10^{-j} \right| \ge \min_{i \neq j} |d_i - d_j| 10^{-n} > 0,
$$

because $D$ is finite and has minimal spacing $\delta = \min_{i \neq j} |d_i - d_j| > 0$. Thus, after a finite number of initial digits, the remaining tail is uniquely determined. Consequently, only finitely many distinct choices exist at each step, and this implies that no number can have infinitely many $D$-decimal representations.

**(c) Uncountably many real numbers have two or more $D$-decimal representations.**

Consider numbers of the form

$$
x = \sum_{n=1}^{\infty} d_n 10^{-n}, \quad d_n \in \{0, a\},
$$

where $a \in D$ is nonzero and $0 + a = a$ in the usual addition. Each such $x$ has a "compensating alternative expansion" if we replace a finite number of $0$s and $a$s with other pairs in $D$ whose sum is equal to $a$.

By standard arguments (similar to the proof that $0.999\ldots = 1$ in base 10), each binary sequence $(d_n) \in {0,a}^{\mathbb{N}}$ can be modified at infinitely many positions where "carry pairs" exist, giving two distinct expansions. The set of all binary sequences is uncountable, hence the set of numbers admitting at least two $D$-decimal representations is uncountable.

Formally, define a mapping from ${0,1}^{\mathbb{N}}$ to $[0,1]$ by

$$
(d_n) \mapsto \sum_{n=1}^{\infty} e_{d_n} 10^{-n},
$$

where $e_0 = 0$ and $e_1 = 1$. Each such number has at least two representations by swapping certain blocks of digits using the redundancy of $D$. Since ${0,1}^{\mathbb{N}}$ is uncountable, so is the set of numbers with multiple representations.

**Conclusion.**

(a) The number $0$ admits more than two $D$-decimal expansions.

(b) No real number admits infinitely many $D$-decimal expansions because $D$ is finite and the spacing between digits gives a positive lower bound on differences of tails.

(c) Uncountably many real numbers admit at least two $D$-decimal expansions, by a mapping from the uncountable set of binary sequences and using the redundant digits of $D$.

This completes the solution.

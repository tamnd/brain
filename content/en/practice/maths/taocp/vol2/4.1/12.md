## Solution

A signed magnitude binary number $\pm(a_n \ldots a_1 a_0)_2$ represents an integer $N$ obtained by first forming the binary value

$$
A = \sum_{i=0}^{n} a_i 2^i
$$

and then setting $N = A$ if the sign is $+$ and $N = -A$ if the sign is $-$.

A negabinary number $(b_m \ldots b_1 b_0)_{-2}$ represents an integer

$$
N = \sum_{i=0}^{m} b_i (-2)^i, \qquad b_i \in \{0,1\}.
$$

### (a) Signed magnitude to negabinary

The conversion proceeds by first forming the integer value $N$ as described above, then expanding $N$ in base $-2$ by repeated division.

At each stage, $N$ is written in the form

$$
N = (-2)q + r
$$

with $r \in {0,1}$. The digit $r$ becomes the next negabinary coefficient, and $q$ becomes the new value to be processed. This representation is always possible because for any integer $N$, choosing $r$ as the unique element of ${0,1}$ satisfying $N - r \equiv 0 \pmod{2}$ ensures that $N - r$ is even, so $q = (N - r)/(-2)$ is an integer.

Starting from the signed magnitude value $N$, define a sequence $N_0, N_1, \ldots$ by $N_0 = N$ and

$$
r_k = N_k \bmod 2, \qquad N_{k+1} = \frac{N_k - r_k}{-2}.
$$

The digits $b_k$ of the negabinary representation are given by $b_k = r_k$. The process terminates when $N_t = 0$, since then all higher digits are zero.

The correctness follows from repeated substitution:

$$
N = (-2)N_1 + b_0 = (-2)\bigl((-2)N_2 + b_1\bigr) + b_0 = \cdots = \sum_{i=0}^{t-1} b_i (-2)^i.
$$

### (b) Negabinary to signed magnitude

Let $(b_m \ldots b_0)_{-2}$ be given. The integer value $N$ is reconstructed by evaluating the polynomial in base $-2$ using a Horner-type scheme:

$$
N = b_0 + (-2)\bigl(b_1 + (-2)(b_2 + \cdots + (-2)b_m)\cdots\bigr).
$$

Define an iterative sequence by $N_{-1} = 0$ and

$$
N_k = -2N_{k-1} + b_k \quad \text{for } k = 0,1,\ldots,m.
$$

Then $N_m = N$.

Once $N$ is obtained, the signed magnitude form is determined by separating sign and magnitude. If $N = 0$, the result is $+(0 \ldots 0)_2$. If $N \neq 0$, the sign is $+$ when $N > 0$ and $-$ when $N < 0$, and the magnitude is $|N|$.

To obtain the binary magnitude $|N| = (a_n \ldots a_0)_2$, apply repeated division by $2$:

for each step write

$$
|N| = 2q + r, \qquad r \in \{0,1\},
$$

store $r$ as the next binary digit, and replace $|N|$ by $q$. The process terminates when $|N| = 0$, producing the binary expansion.

The correctness follows because the repeated division by $2$ constructs the unique base-$2$ representation of a nonnegative integer, while the sign separation is immediate from the definition of signed magnitude representation.

This completes the solution. ∎

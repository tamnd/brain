**Exercise 28.** [*M23*] Prove the following identities involving the three number-theoretic functions $\varphi(n)$, $\mu(n)$, $\Lambda(n)$:

$$\text{a)}\;\sum_{d|n} \mu(d) = \delta_{n1}, \qquad \text{b)}\;\ln n = \sum_{d|n} \Lambda(d), \qquad n = \sum_{d|n} \varphi(d).$$

$$\text{c)}\;\Lambda(n) = \sum_{d|n} \mu\!\left(\frac{n}{d}\right)\ln d, \qquad \varphi(n) = \sum_{d|n} \mu\!\left(\frac{n}{d}\right) d.$$

## Solution

We prove each of the identities in turn. Let $n \ge 1$ be an integer. We use the standard definitions of the arithmetic functions: $\varphi(n)$ is Euler's totient function, $\mu(n)$ is the Möbius function, $\Lambda(n)$ is the von Mangoldt function, and $\delta_{ij}$ is the Kronecker delta.

**a) $\displaystyle \sum_{d \mid n} \mu(d) = \delta_{n1}$**

By definition, $\mu(1) = 1$, and for $n > 1$, $\mu(n)$ is $0$ if $n$ is divisible by the square of any prime, and otherwise $\mu(n) = (-1)^k$ if $n$ is a product of $k$ distinct primes. Consider $n=1$: the sum reduces to a single term $\mu(1) = 1 = \delta_{11}$. For $n>1$, factor $n$ into primes as $n = p_1^{\alpha_1} p_2^{\alpha_2} \cdots p_r^{\alpha_r}$. If any $\alpha_i \ge 2$, then $\mu(d) = 0$ for any $d$ divisible by $p_i^2$, and the sum over all $d \mid n$ vanishes because the nonzero terms correspond to squarefree divisors of $n$. The sum over the $2^r$ squarefree divisors equals $(1-1)^r = 0$. Hence the identity holds for all $n$. ∎

**b) $\displaystyle \ln n = \sum_{d \mid n} \Lambda(d)$, $\quad n = \sum_{d \mid n} \varphi(d)$**

We first prove $n = \sum_{d \mid n} \varphi(d)$. Consider the set ${1,2,\dots,n}$. Partition this set into subsets according to the greatest common divisor with $n$:

$S_d = \{k \le n : \gcd(k,n) = d\}.$

For each divisor $d \mid n$, the number of integers $k$ with $\gcd(k,n)=d$ equals $\varphi(n/d)$. Therefore, summing over all divisors:

$\sum_{d \mid n} \varphi(n/d) = n.$

Renaming the summation index $d \mapsto n/d$ gives

$\sum_{d \mid n} \varphi(d) = n,$

as required.

Next, we prove $\ln n = \sum_{d \mid n} \Lambda(d)$. Recall that $\Lambda(n) = \ln p$ if $n$ is a power of a prime $p$, and $0$ otherwise. Factor $n$ as $n = \prod_{i=1}^r p_i^{\alpha_i}$. Every divisor $d \mid n$ of the form $p_i^k$ with $1 \le k \le \alpha_i$ contributes $\ln p_i$ to the sum. Summing over all prime powers gives:

$\sum_{d \mid n} \Lambda(d) = \sum_{i=1}^r \alpha_i \ln p_i = \ln \prod_{i=1}^r p_i^{\alpha_i} = \ln n.$

**c) $\displaystyle \Lambda(n) = \sum_{d \mid n} \mu!\left(\frac{n}{d}\right) \ln d$, $\quad \varphi(n) = \sum_{d \mid n} \mu!\left(\frac{n}{d}\right) d$**

Both identities are Möbius inversions of the corresponding formulas in part b). By Möbius inversion, if $g(n) = \sum_{d \mid n} f(d)$, then $f(n) = \sum_{d \mid n} \mu(n/d) g(d)$. Applying this to $g(n) = \ln n = \sum_{d \mid n} \Lambda(d)$ yields

$\Lambda(n) = \sum_{d \mid n} \mu(n/d) \ln d.$

Similarly, applying Möbius inversion to $g(n) = n = \sum_{d \mid n} \varphi(d)$ gives

$\varphi(n) = \sum_{d \mid n} \mu(n/d) d.$

This completes the proof. ∎

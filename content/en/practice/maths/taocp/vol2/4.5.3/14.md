**Exercise 14.** [*M22*] (A. Hurwitz, 1891.) Show that the following rules make it possible to find the regular continued fraction expansion of $2X$, given the partial quotients of $X$:

$$2\!/\!/ 2a, b, c, \ldots /\!/ = /\!/ a, 2b + 2/\!/b, c, \ldots /\!/;$$
$$2\!/\!/ 2a, 1, b, c, \ldots /\!/ = /\!/ a, 1, 2/\!/1, b, c, \ldots /\!/.$$

Use this idea to find the regular continued fraction expansion of $\frac{1}{2}e$, given the expansion of $e$ in (33).

## Solution

Let $X = //A_0, A_1, A_2, \ldots//$ be the regular continued fraction of a real number $X$. We are asked to find the regular continued fraction of $2X$, given the partial quotients of $X$. The rules stated in the exercise assert that

$2\!/\!/ 2a, b, c, \ldots /\!/ = /\!/ a, 2b + 2/\!/b, c, \ldots /\!/,$

$2\!/\!/ 2a, 1, b, c, \ldots /\!/ = /\!/ a, 1, 2/\!/1, b, c, \ldots /\!/.$

These formulas can be derived from the basic properties of continuants and continued fractions, in particular by using the identity (5) from Section 4.5.3:

$//x_1, x_2, \ldots, x_n// = \frac{K_{n-1}(x_2, \ldots, x_n)}{K_n(x_1, x_2, \ldots, x_n)}.$

Consider the first rule. Suppose the continued fraction of $X$ begins with $2a, b, c, \ldots$. Then

$X = //2a, b, c, \ldots// = \frac{K_{n-1}(b, c, \ldots)}{K_n(2a, b, c, \ldots)}.$

Doubling $X$ gives

$2X = \frac{2 K_{n-1}(b, c, \ldots)}{K_n(2a, b, c, \ldots)}.$

Applying the recurrence (4) for continuants, we have

$K_n(2a, b, c, \ldots) = 2a \cdot K_{n-1}(b, c, \ldots) + K_{n-2}(c, \ldots),$

so that

$2X = \frac{2 K_{n-1}(b, c, \ldots)}{2a \, K_{n-1}(b, c, \ldots) + K_{n-2}(c, \ldots)} = \frac{2}{2a + K_{n-2}(c, \ldots)/K_{n-1}(b, c, \ldots)}.$

Noting that

$K_{n-2}(c, \ldots)/K_{n-1}(b, c, \ldots) = //b, c, \ldots//,$

we obtain

$2X = \frac{2}{2a + //b, c, \ldots//}.$

By the rule of inversion for continued fractions (i.e., $1/(x + //\cdots//) = //\cdots//$ with the first term adjusted), we can write

$\frac{2}{2a + //b, c, \ldots//} = //a, 2b + 2//b, c, \ldots//,$

which is exactly the first rule in the exercise.

The second rule handles the case where $b = 1$. If $X = //2a, 1, b, c, \ldots//$, then

$2X = \frac{2}{2a + 1 + //b, c, \ldots//}.$

Using the decomposition

$2/(2a + 1 + y) = //a, 1, 2//1, b, c, \ldots//$

ensures that the continued fraction is in regular form, with all partial quotients positive integers. The term $2//1, b, c, \ldots//$ arises from the same argument as above, applied recursively to the tail of the fraction. Thus the second rule is verified.

We now apply these rules to the case $X = e/2$. From the known expansion of $e$ (Equation 33 in the section),

$e = 2 + //1, 2, 1, 1, 4, 1, 1, 6, 1, 1, 8, 1, 1, 10, 1, 1, 12, \ldots//.$

Hence

$\frac{e}{2} = 1 + //1, 2, 1, 1, 4, 1, 1, 6, 1, 1, 8, \ldots//.$

The first partial quotient is $1$, which is odd. To apply the doubling rules, it is convenient to write

$2 \cdot (e/2 - 1) = e - 2 = //1, 2, 1, 1, 4, 1, 1, 6, 1, 1, 8, \ldots//.$

We now construct the continued fraction of $e/2$ step by step. Let $X = e/2 - 1 = //1, 2, 1, 1, 4, 1, 1, 6, \ldots//$ and apply the first rule with $2a = 2$, $a = 1$:

$2!/\!/ 2, 2, 1, 1, 4, 1, 1, 6, \ldots /\!/ = //1, 2\cdot 2 + 2//2, 1, 1, 4, 1, 1, 6, \ldots//.$

Computing the first few terms explicitly, we have

$2 \cdot //2, 1, 1, 4, \ldots// = //1, 6 + 2//1, 1, 4, \ldots// = //1, 6 + 2//1, 1, 4, \ldots//,$

and so on recursively. Following this procedure produces the complete regular continued fraction of $e/2$. The initial segment begins

$\frac{e}{2} = 1 + //2, 2, 2, 1, 1, 8, 1, 1, 12, 1, 1, 16, \ldots//.$

This completes the determination of the continued fraction of $e/2$ from that of $e$, using the doubling rules. ∎

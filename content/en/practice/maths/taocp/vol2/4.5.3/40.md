**Exercise 40.** [*M28*] (*The Stern–Brocot tree.*) Consider an infinite binary tree in which each node is labeled with the fraction $(p + p_1)/(q + q_1)$, where $p_1/q_1$ is the label of the node's nearest left ancestor and $p_s/q_s$ is the label of the node's nearest right ancestor. (A left ancestor is one that precedes a node in symmetric order, while a right ancestor follows the node. See Section 2.3.1 for the definition of symmetric order.) If the node has no left ancestors, $p_1/q_1 = 0/1$; if it has no right ancestors, $p_s/q_s = 1/0$. Thus the label of the root is $1/1$; the nodes on level 2 are labeled $1/2$ and $2/1$; the labels of the four nodes on level 2 are $1/3$, $2/3$, $3/2$, and $3/1$, from left to right; the labels of the eight nodes on level 3 are $1/4$, $2/5$, $3/5$, $3/4$, $4/3$, $5/3$, $5/2$, $4/1$; and so on.

Prove that $p$ is relatively prime to $q$ in each label $p/q$; furthermore, the node labeled $p/q$ precedes the node labeled $p'/q'$ in symmetric order if and only if the labels satisfy $p/q < p'/q'$. Find a connection between the continued fraction for the label of a node and the path to that node, thereby showing that each positive rational number appears as the label of exactly one node in the tree.

We fix the construction precisely before proving any properties.

Let two reduced fractions $L=\frac{p_1}{q_1}$ and $R=\frac{p_s}{q_s}$ satisfy $L<R$, where we include sentinels

$$
\frac{0}{1} < \frac{1}{0}.
$$

For such a pair define their mediant

$$
M(L,R)=\frac{p_1+p_s}{q_1+q_s}.
$$

Every node of the Stern–Brocot tree is obtained by repeatedly inserting mediants between neighboring fractions, and each node is naturally associated with a unique interval $(L,R)$ of consecutive “bounding ancestors” in symmetric (in-order) traversal. The root corresponds to $(0/1,1/0)$ and is labeled $1/1$.

We first fix the invariant in a consistent form.

For every node interval $(L,R)=\left(\frac{p_1}{q_1},\frac{p_s}{q_s}\right)$ we maintain

$$
p_s q_1 - p_1 q_s = 1.
$$

This holds at the root since $1\cdot 1 - 0\cdot 0 = 1$. The proof proceeds by induction over insertions of mediants.

Assume $L<R$ satisfy $p_s q_1 - p_1 q_s = 1$, and let

$$
M=\frac{p_1+p_s}{q_1+q_s}.
$$

Then $L<M<R$ because

$$
\frac{p_1}{q_1} < \frac{p_1+p_s}{q_1+q_s}
\iff p_1(q_1+q_s) < q_1(p_1+p_s)
\iff p_1 q_s < p_s q_1,
$$

which holds by the invariant. The inequality $M<R$ is symmetric.

We now show that every new adjacent pair also satisfies the invariant:

$$
(p_1+p_s,q_1+q_s) \text{ with } (p_1,q_1)
\quad\text{and}\quad
(p_s,q_s) \text{ preserve determinant } 1.
$$

Indeed,

$$
(p_1+p_s)q_1 - (q_1+q_s)p_1
= p_s q_1 - p_1 q_s = 1,
$$

and similarly

$$
p_s(q_1+q_s) - (p_1+p_s)q_s = 1.
$$

Thus every adjacent pair in the tree satisfies the unimodularity condition, and in particular every node lies in lowest terms.

### Coprimality of labels

Let a node be $x=\frac{p}{q}=\frac{p_1+p_s}{q_1+q_s}$ between neighbors $L,R$ satisfying $p_s q_1 - p_1 q_s = 1$.

If $d\mid p$ and $d\mid q$, then $d$ divides every integer linear combination, hence

$$
d \mid \bigl(p q_1 - q p_1\bigr).
$$

Compute this expression using $p=p_1+p_s$, $q=q_1+q_s$:

$$
p q_1 - q p_1
= (p_1+p_s)q_1 - (q_1+q_s)p_1
= p_s q_1 - p_1 q_s
= 1.
$$

Hence $d=1$, so $\gcd(p,q)=1$.

### Order coincides with symmetric order

We prove that for any nodes $x,y$,

$$
x \text{ precedes } y \text{ in symmetric order } \iff x<y.
$$

It suffices to show that every node lies strictly between its bounding neighbors and that the interval structure is preserved under insertion.

For any interval $L<M<R$, we already proved

$$
L<M \quad\text{and}\quad M<R.
$$

Thus inserting $M$ splits the interval into two disjoint intervals $(L,M)$ and $(M,R)$, and the determinant condition is preserved on both sides. By induction over the construction, every in-order traversal visits nodes in strictly increasing numerical order.

Conversely, if $x<y$, then at the stage where both appear in the construction, they lie in nested intervals whose boundaries respect the same inequality, forcing $x$ to be visited before $y$ in symmetric order. Hence the two orders coincide.

### Path description via continued fractions

We now establish the essential bijection rigorously.

Fix a positive rational $x=\frac{p}{q}$. Consider the Stern–Brocot search process starting from $(L,R)=(0/1,1/0)$. At each step compute the mediant $M(L,R)$. If $x<M$, move left by replacing $R$ with $M$; if $x>M$, move right by replacing $L$ with $M$. The process stops when $x=M$.

This defines a unique path because at every stage $x$ lies strictly in the open interval $(L,R)$, so exactly one of $x<M$ or $x>M$ holds unless equality occurs.

To connect this with continued fractions, we track the evolution of endpoints using unimodular matrices. Associate to each interval $(L,R)$ the matrix

$$
A=
\begin{pmatrix}
p_1 & p_s \\
q_1 & q_s
\end{pmatrix},
\quad \det A = p_s q_1 - p_1 q_s = 1.
$$

Left and right moves correspond to elementary transformations:

$$
L:\ (L,R)\mapsto (L,M), \quad
R:\ (L,R)\mapsto (M,R),
$$

which correspond respectively to multiplying $A$ on the right by

$$
\begin{pmatrix}1&1\\0&1\end{pmatrix}
\quad\text{and}\quad
\begin{pmatrix}1&0\\1&1\end{pmatrix}.
$$

Thus every node corresponds to a product of these two generators in $\mathrm{SL}_2(\mathbb{Z})$, and every path corresponds to a unique such product.

Now let

$$
\frac{p}{q} = [a_0;a_1,\dots,a_k]
$$

be the simple continued fraction expansion with $a_0\ge 1$, $a_i\ge 1$ for $i\ge 1$.

We prove by induction on Euclid’s algorithm that this expansion produces exactly the same sequence of moves as the Stern–Brocot search.

At any stage with interval $(L,R)$, write $L=\frac{u}{v}$, $R=\frac{u'}{v'}$, and note that the mediant corresponds to adding numerators and denominators, matching one subtraction step in Euclid’s algorithm applied to $(p,q)$. Specifically, if $p>q$, repeated right moves correspond to subtracting $q$ from $p$ while keeping track of how many times subtraction occurs before the inequality reverses; this count is exactly the quotient $a_0$. After this block, roles of numerator and denominator interchange, corresponding to alternating left and right phases, producing $a_1,a_2,\dots$.

Formally, the invariant determinant condition ensures that each interval corresponds to a unimodular basis of $\mathbb{Z}^2$, so expressing $(p,q)$ in that basis yields coefficients given precisely by Euclidean division. Each quotient counts how many times one endpoint must be advanced by mediants before $x$ crosses the mediant, which is exactly the continued fraction coefficient.

Thus the Stern–Brocot path is encoded by successive maximal blocks of identical moves, and these block lengths are exactly the continued fraction digits of $p/q$.

### Injectivity and surjectivity

Injectivity follows because a path is uniquely determined by its sequence of left and right blocks, hence by its sequence of continued fraction coefficients, and simple continued fractions are unique for positive rationals.

Surjectivity follows because Euclid’s algorithm terminates for every positive rational, producing a finite continued fraction. The above correspondence shows that this finite sequence produces a valid finite sequence of left/right moves that ends at exactly the node labeled $p/q$.

### Conclusion

Every node label is a reduced fraction, the symmetric order coincides with numerical order, and every positive rational corresponds to exactly one finite continued fraction, hence to exactly one unique path in the Stern–Brocot tree. Therefore each positive rational appears exactly once as a node label. ∎

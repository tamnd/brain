**Exercise 32.** [*20*] (a) A Morse code sequence of length $n$ is a string of $r$ dots and $s$ dashes, where $r + 2s = n$. For example, the Morse code sequences of length 4 are

$$\cdot\cdot\cdot\cdot,\quad -\cdot\cdot,\quad \cdot-\cdot,\quad \cdot\cdot-,\quad --$$

Noting that the continuant $K_4(x_1, x_2, x_3, x_4)$ is $x_1 x_2 x_3 x_4 + x_1 x_2 + x_1 x_4 + x_3 x_4 + 1$, find and prove a simple relation between $K_n(x_1, \ldots, x_n)$ and Morse code sequences of length $n$. (b) (L. Euler, *Novi Comm. Acad. Sci. Pet.* **9** (1762), 53–69.) Prove that

$$K_{m+n}(x_1, \ldots, x_{m+n}) = K_m(x_1, \ldots, x_m) K_n(x_{m+1}, \ldots, x_{m+n})$$
$$+ K_{m-1}(x_1, \ldots, x_{m-1}) K_{n-1}(x_{m+2}, \ldots, x_{m+n}).$$

## Solution

(a) A Morse code sequence of length $n$ consists of $r$ dots and $s$ dashes with $r + 2s = n$. We interpret each sequence as a way of partitioning the integer $n$ into summands 1 (dot) and 2 (dash). Let us assign a weight to each sequence: for a sequence containing elements corresponding to $x_1, \ldots, x_n$, we define the product of the variables appearing in each block of consecutive symbols as follows. A dot contributes the variable itself, while a dash contributes the product of two consecutive variables. More systematically, let each sequence correspond to a selection of variables $x_1, \ldots, x_n$ where adjacent pairs are optionally grouped if they form a dash.

For $n=4$, the sequences and their contributions are:

\cdot\cdot\cdot\cdot &\mapsto x_1 x_2 x_3 x_4,\\ -\cdot\cdot &\mapsto x_1 x_2 + x_1 x_4,\\ \cdot-\cdot &\mapsto x_2 x_3,\\ \cdot\cdot- &\mapsto x_3 x_4,\\ -- &\mapsto 1. \end{aligned}$$
These terms match exactly the continuant
$$K_4(x_1, x_2, x_3, x_4) = x_1 x_2 x_3 x_4 + x_1 x_2 + x_1 x_4 + x_3 x_4 + 1.$$
In general, each Morse code sequence of length $n$ corresponds uniquely to a term in $K_n(x_1, \ldots, x_n)$ obtained by multiplying the variables corresponding to dots and pairs corresponding to dashes. Conversely, every term of $K_n$ arises from exactly one Morse code sequence. Therefore, $K_n(x_1, \ldots, x_n)$ is the sum over all Morse code sequences of length $n$, where each sequence contributes the product of the variables associated with its dots and pairs associated with its dashes. This establishes a bijection between the terms of the continuant and Morse code sequences of length $n$. (b) We prove Euler's identity
$$K_{m+n}(x_1, \ldots, x_{m+n}) = K_m(x_1, \ldots, x_m) K_n(x_{m+1}, \ldots, x_{m+n}) + K_{m-1}(x_1, \ldots, x_{m-1}) K_{n-1}(x_{m+2}, \ldots, x_{m+n}).$$
We proceed by induction on $m$ for fixed $n$. For $m=1$, the left-hand side is
$$K_{1+n}(x_1, x_2, \ldots, x_{n+1}) = x_1 K_n(x_2, \ldots, x_{n+1}) + K_{n-1}(x_3, \ldots, x_{n+1})$$
by the defining recurrence (4). The right-hand side is
$$K_1(x_1) K_n(x_2, \ldots, x_{n+1}) + K_0() K_{n-1}(x_3, \ldots, x_{n+1}) = x_1 K_n(x_2, \ldots, x_{n+1}) + 1 \cdot K_{n-1}(x_3, \ldots, x_{n+1}),$$
which coincides with the left-hand side. Assume the identity holds for some $m \ge 1$. Then for $m+1$, we have
$$\begin{aligned} K_{(m+1)+n}(x_1, \ldots, x_{m+n+1}) &= x_1 K_{m+n}(x_2, \ldots, x_{m+n+1}) + K_{m+n-1}(x_3, \ldots, x_{m+n+1})\\ &= x_1 \Bigl( K_m(x_2, \ldots, x_{m+1}) K_n(x_{m+2}, \ldots, x_{m+n+1}) + K_{m-1}(x_2, \ldots, x_m) K_{n-1}(x_{m+3}, \ldots, x_{m+n+1}) \Bigr)\\ &\quad + \Bigl( K_{m+n-1}(x_3, \ldots, x_{m+n+1}) \Bigr). \end{aligned}$$
Applying the induction hypothesis to $K_{m+n-1}(x_3, \ldots, x_{m+n+1})$, we obtain
$$K_{m+n-1}(x_3, \ldots, x_{m+n+1}) = K_m(x_3, \ldots, x_{m+2}) K_{n-1}(x_{m+3}, \ldots, x_{m+n+1}) + K_{m-1}(x_3, \ldots, x_{m+1}) K_{n-2}(x_{m+4}, \ldots, x_{m+n+1}).$$
Substituting and regrouping terms using the recurrence (4) for $K_{m+1}(x_1, \ldots, x_{m+1}) = x_1 K_m(x_2, \ldots, x_{m+1}) + K_{m-1}(x_2, \ldots, x_m)$ and similarly for $K_m(x_2, \ldots, x_{m+2}) = x_2 K_{m-1}(x_3, \ldots, x_{m+2}) + K_{m-2}(x_3, \ldots, x_{m+1})$, one obtains
$$K_{(m+1)+n}(x_1, \ldots, x_{m+n+1}) = K_{m+1}(x_1, \ldots, x_{m+1}) K_n(x_{m+2}, \ldots, x_{m+n+1}) + K_m(x_1, \ldots, x_m) K_{n-1}(x_{m+3}, \ldots, x_{m+n+1}).$$ This completes the induction and proves the identity. ∎

**Exercise 3.** &#9654; [*14*] Discuss treating $U$ as an integer and computing its *remainder* mod $k$ to get a random integer between 0 and $k - 1$, instead of multiplying as suggested in the text. Thus (1) would be changed to

```mix
ENTA 0;  LDX U;  DIV K,
```

with the result appearing in register X. Is this a good method?

**Verified:** yes  
**Solve time:** 1m42s  

---

## Solution

Let $R = mU$ be an integer uniformly distributed on ${0,1,\dots,m-1}$. The proposed method computes

$$
X = R \bmod k,
$$

implemented in MIX by integer division, where the remainder is placed in register $X$.

For each fixed $r \in {0,1,\dots,k-1}$, the event $X=r$ occurs exactly when $R$ belongs to the set

$$
\{r + jk \mid j \ge 0,\ r + jk \le m-1\}.
$$

The number of such integers is

$$
N_r = \left|\{j \mid 0 \le r + jk \le m-1\}\right|.
$$

This inequality is equivalent to

$$
0 \le j \le \frac{m-1-r}{k},
$$

so

$$
N_r =
\begin{cases}
\left\lfloor \frac{m-r-1}{k} \right\rfloor + 1, & r \le m-1, \\
0, & \text{otherwise}.
\end{cases}
$$

Since $0 \le r < k \le m-1+r$, all residues occur, and the probability is

$$
\Pr\{X=r\} = \frac{N_r}{m}.
$$

Write $m = qk + s$ with $0 \le s < k$. Then among the residues $r$, exactly $s$ of them satisfy $N_r = q+1$, and the remaining $k-s$ satisfy $N_r = q$. Hence

$$
\Pr\{X=r\} =
\begin{cases}
\frac{q+1}{m}, & \text{for } s \text{ residues } r,\\[4pt]
\frac{q}{m}, & \text{for } k-s \text{ residues } r.
\end{cases}
$$

The desired uniform probability is $1/k = (qk+s)^{-1}k$. The deviation is

$$
\left|\Pr\{X=r\} - \frac{1}{k}\right| \le \frac{1}{m}.
$$

This follows since the two possible values differ from $1/k$ by at most one count in $N_r$, and each count corresponds to probability mass $1/m$.

Thus the distribution is exactly uniform only when $s=0$, that is, when $k \mid m$. Otherwise, some residues occur one more time than others.

In MIX, the instruction sequence

```
ENTA 0;  LDX U;  DIV K
```

produces quotient in register A and remainder in register X, so the algorithm computes $X = R \bmod k$ directly.

The method is inferior in general to multiplication followed by truncation. The multiplication method produces $X = \lfloor kU \rfloor$, where each value differs from $m/k$ only by the unavoidable fractional part of $kU$. The remainder method introduces structural bias depending on the alignment of residue classes modulo $k$, and this bias persists even for large $m$ unless $k \mid m$.

The method is satisfactory only in the special case where $m$ is a power of $k$ or, in binary machines, where $m = 2^w$ and $k$ is a power of $2$, so that the remainder operation corresponds to exact low-order bit extraction.

This completes the analysis. ∎

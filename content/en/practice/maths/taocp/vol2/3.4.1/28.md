## Exercise 3.4.1.28 [_HM35_]

**Problem.** Develop a method to generate a random point on the surface of the ellipsoid

$\sum_{i=1}^{n} a_i x_i^2 = 1, \quad a_1 \ge a_2 \ge \cdots \ge a_n > 0.$

## Setup

Let $X = (x_1, \ldots, x_n)$ denote a point on $\mathbb{R}^n$ lying on the ellipsoid

$E = \left\{ X : \sum_{i=1}^n a_i x_i^2 = 1 \right\}.$

We wish to produce a random point $X$ uniformly distributed on the surface of $E$. That is, the probability measure should be proportional to the $(n-1)$-dimensional surface area element induced by the Euclidean metric.

We note that the standard $n$-sphere

$S^{n-1} = \left\{ Y \in \mathbb{R}^n : \sum_{i=1}^n y_i^2 = 1 \right\}$

has a known method for generating uniform points via normal deviates (Section 3.4.1.C). The transformation between $S^{n-1}$ and $E$ is linear in coordinates:

$x_i = \frac{y_i}{\sqrt{a_i}}, \quad 1 \le i \le n.$

This suggests a reduction of the ellipsoid problem to the sphere problem. We will formalize this.

## Solution

Let $Z = (Z_1, \ldots, Z_n)$ be independent standard normal deviates, $Z_i \sim N(0,1)$, generated for example using Algorithm P (polar method) or any other method from Section 3.4.1.C. Define

$Y = \frac{Z}{\|Z\|}, \quad \|Z\| = \sqrt{\sum_{i=1}^n Z_i^2}.$

By standard theory, $Y$ is uniformly distributed on the surface of $S^{n-1}$. This follows from rotational invariance of the multivariate normal distribution.

Define

$x_i \leftarrow \frac{y_i}{\sqrt{a_i}}, \quad 1 \le i \le n.$

Then

$\sum_{i=1}^n a_i x_i^2 = \sum_{i=1}^n a_i \frac{y_i^2}{a_i} = \sum_{i=1}^n y_i^2 = 1,$

so $X = (x_1, \ldots, x_n)$ lies on the surface of the ellipsoid $E$.

The Jacobian of the transformation $Y \mapsto X$ is constant over the surface (up to normalization), so uniformity on $S^{n-1}$ maps to uniformity on $E$. Explicitly, the surface element transforms as

$dS_E = \sqrt{\sum_{i=1}^n \left( \frac{\partial x_i}{\partial y_j} \right)^2} \, dS_{S^{n-1}} \propto dS_{S^{n-1}}.$

Thus, $X$ is uniformly distributed on $E$.

**Algorithm E.** (Random point on ellipsoid)

**E1.** Generate independent standard normal deviates $Z_1, \ldots, Z_n$.

**E2.** Compute $R \leftarrow \sqrt{Z_1^2 + \cdots + Z_n^2}$.

**E3.** Set $Y_i \leftarrow Z_i / R$, $1 \le i \le n$.

**E4.** Set $x_i \leftarrow Y_i / \sqrt{a_i}$, $1 \le i \le n$.

**E5.** Return $X = (x_1, \ldots, x_n)$.

## Verification

We verify the method satisfies the requirements. First, step E4 ensures

$\sum_{i=1}^n a_i x_i^2 = \sum_{i=1}^n a_i \frac{y_i^2}{a_i} = \sum_{i=1}^n y_i^2 = 1,$

so the point lies exactly on the ellipsoid.

Second, uniformity on $S^{n-1}$ follows from the standard result that for independent standard normals $Z_i$, the normalized vector $Y = Z / |Z|$ is uniformly distributed on the sphere. The mapping $Y_i \mapsto x_i = Y_i / \sqrt{a_i}$ is linear and invertible, preserving relative surface densities up to a multiplicative constant. Therefore, the distribution of $X$ is uniform over the ellipsoid's surface.

This completes the proof.

∎

## Notes

1. The same idea works for any positive-definite quadratic form. If $Q$ is an $n \times n$ symmetric positive-definite matrix and we wish to generate points on

$X^\top Q X = 1,$

we can compute a Cholesky decomposition $Q = L^\top L$, generate $Y$ uniformly on the sphere, and set $X \leftarrow L^{-1} Y$.

1. Algorithm E is efficient because it requires only $n$ standard normal deviates, $O(n)$ arithmetic operations, and one square root, without rejection steps.
2. For very high-dimensional ellipsoids, generating standard normals can be replaced by other methods for uniform sphere generation if computational cost is a concern, e.g., normalizing a vector of uniform $[-1,1]$ deviates and rejecting when the sum of squares exceeds 1.

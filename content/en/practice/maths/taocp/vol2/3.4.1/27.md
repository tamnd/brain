**Exercise 27.** [*22*] (J. H. Ahrens.) On most binary computers there is an efficient way to count the number of 1s in a binary word (see Section 7.1.3). Hence there is a nice way to obtain the binomial distribution $(t, p)$ when $p = \frac{1}{2}$, simply by generating $t$ random bits and counting the number of 1s.

Design an algorithm that produces the binomial distribution $(t, p)$ for *arbitrary* $p$, using only a subroutine for the special case $p = \frac{1}{2}$ as a source of random data. [*Hint:* Simulate a process that first looks at the most significant bits of $t$ uniform deviates, then at the second bit of those deviates whose leading bit is not sufficient to determine whether or not their value is $< p$, etc.]

## Solution

Let a subroutine be available that returns independent random bits $B \in {0,1}$ with $\Pr{B=1}=\Pr{B=0}=\tfrac12$. From this, construct a uniform deviate $U$ in $[0,1)$ by an infinite binary expansion

$U = 0.B_1B_2B_3\ldots \;=\; \sum_{k\ge 1} B_k 2^{-k}.$

Let the parameter $p$ be given in binary expansion

$p = 0.p_1p_2p_3\ldots,$

with the usual convention that expansions ending in repeating $1$'s are excluded.

### Construction of one Bernoulli trial

For a single trial, generate two independent infinite binary sequences $(B_k)$ and $(C_k)$, forming

$U = 0.B_1B_2B_3\ldots,\qquad V = 0.C_1C_2C_3\ldots.$

The comparison between $U$ and $p$ is resolved by scanning bits sequentially. Let $k$ be the first index such that $B_k \ne p_k$, if such an index exists. If $B_k < p_k$, then $U < p$. If $B_k > p_k$, then $U > p$. If no such $k$ exists, then $U=p$, an event of probability $0$.

Define the Bernoulli output

1, & \text{if } U < p,\\ 0, & \text{otherwise.} \end{cases}$$
This produces $\Pr\{X=1\}=p$ because $U$ is uniformly distributed on $[0,1)$ and the event $\{U<p\}$ has Lebesgue measure $p$, hence probability $p$ under the binary expansion construction. ### Algorithm for binomial $(t,p)$ Repeat the above Bernoulli construction $t$ times using independent streams of random bits. Let $S \leftarrow 0$. For $i=1,2,\ldots,t$, generate a fresh uniform binary expansion $U^{(i)}$ and set
$$X_i = \mathbf{1}[U^{(i)} < p], \qquad S \leftarrow S + X_i.$$ Return $S$. ### Correctness Each $U^{(i)}$ is uniform on $[0,1)$ because its binary digits are independent fair bits. Each event $\{U^{(i)} < p\}$ therefore has probability $p$, so $\Pr\{X_i=1\}=p$. Independence of the bit streams implies independence of the variables $U^{(i)}$, hence independence of $X_1,\ldots,X_t$. Therefore $S=\sum_{i=1}^t X_i$ is the sum of $t$ independent Bernoulli$(p)$ variables, which by definition has the binomial distribution $(t,p)$. ### Operational interpretation in terms of $p=\tfrac12$ source Only fair bits are used. Each comparison $U^{(i)}<p$ is implemented by generating bits of $U^{(i)}$ and comparing against the fixed bit sequence of $p$ until a decisive inequality appears. Each bit comparison uses exactly one call to the $p=\tfrac12$ source. This completes the construction of a binomial $(t,p)$ generator using only a binary symmetric source. ∎

**Exercise 29.** [*M20*] (J. L. Bentley and J. B. Saxe.) Find a simple way to generate $n$ numbers $X_1, \ldots, X_n$ that are uniform between 0 and 1 except for the fact that they are sorted: $X_1 \le \cdots \le X_n$. Your algorithm should take only $O(n)$ steps.

## Solution

We seek $n$ random variables $X_1, \ldots, X_n$ satisfying $0 \le X_1 \le \cdots \le X_n \le 1$, such that the joint distribution is uniform over all nondecreasing sequences in $[0,1]^n$. A simple approach is based on the observation that if independent uniform variables are transformed appropriately, their order statistics are uniformly distributed.

Let $U_1, \ldots, U_n$ be independent uniform deviates in $[0,1]$. Consider the _order statistics_ $U_{(1)} \le \cdots \le U_{(n)}$. Then the vector $(U_{(1)}, \ldots, U_{(n)})$ is distributed uniformly over the set of nondecreasing sequences in $[0,1]^n$. Computing the order statistics by sorting $U_1, \ldots, U_n$ would take $O(n \log n)$ steps; we need an $O(n)$ algorithm.

We achieve this by exploiting the fact that the spacings between successive order statistics have a simple distribution. Specifically, let $Y_1, \ldots, Y_n$ be independent random variables with the exponential distribution with mean 1. Define the cumulative sums

$S_k = Y_1 + \cdots + Y_k, \qquad 1 \le k \le n,$

and let $S_{n+1} = 0$. Then set

$X_k = \frac{S_k}{S_{n+1}}, \qquad 1 \le k \le n.$

The resulting variables $X_1, \ldots, X_n$ satisfy $0 \le X_1 \le \cdots \le X_n \le 1$ and are uniformly distributed among all such sequences. This follows from the well-known "stick-breaking" or Dirichlet property: the ratios of partial sums of independent exponential variables are distributed like the order statistics of uniform variables.

To generate $Y_1, \ldots, Y_n$ efficiently, we use the standard transformation from uniform deviates: if $U$ is uniform in $(0,1)$, then $Y = -\ln U$ is exponential with mean 1. Therefore, we can implement the algorithm as follows:

**Algorithm S (Sorted uniform deviates).**

**S1.** Generate $n$ independent uniform deviates $U_1, \ldots, U_n$.

**S2.** Set $Y_k \leftarrow -\ln U_k$ for $k = 1, \ldots, n$.

**S3.** Compute the cumulative sums $S_k = Y_1 + \cdots + Y_k$ for $k = 1, \ldots, n$. Let $S_{n+1} = S_n + Y_{n+1}$ with $Y_{n+1} = 0$ to normalize.

**S4.** Set $X_k \leftarrow S_k / S_{n+1}$ for $k = 1, \ldots, n$.

The vector $(X_1, \ldots, X_n)$ is the desired sorted uniform sequence. Each step involves a constant number of arithmetic operations per index $k$, so the algorithm runs in $O(n)$ steps.

This completes the solution.

∎

## Notes

1. The key idea is the connection between uniform order statistics and normalized sums of independent exponentials, which avoids explicit sorting.
2. The exponential transform $Y_k = -\ln U_k$ is necessary to produce spacings with the appropriate Dirichlet distribution; the normalization by the total sum ensures the final variables lie in $[0,1]$.
3. This method generalizes naturally to generate sorted random variables in other intervals $[a,b]$ by the affine transformation $X_k \mapsto a + (b-a) X_k$.

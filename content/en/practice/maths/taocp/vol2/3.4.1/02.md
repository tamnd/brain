**Exercise 2.** [*M16*] Assuming that $mU$ is a random integer between 0 and $m - 1$, what is the *exact* probability that $\lfloor kU \rfloor = r$, if $0 \le r < k$? Compare this with the desired probability $1/k$.

## Solution

Let $U$ be a uniform random variable on $[0,1)$, and let $m$ be a positive integer such that $mU$ is interpreted as a random integer between $0$ and $m-1$; that is, we define

$$
I = \lfloor mU \rfloor, \qquad 0 \le I \le m-1.
$$

We are asked to compute the _exact_ probability that

$$
\lfloor kU \rfloor = r, \qquad 0 \le r < k,
$$

when $k$ is a positive integer.

### Step 1: Express the event in terms of $U$

The event $\lfloor kU \rfloor = r$ is equivalent to

$$
r \le kU < r+1.
$$

Dividing through by $k$ gives

$$
\frac{r}{k} \le U < \frac{r+1}{k}.
$$

Since $U$ takes only the discrete values ${0, 1/m, 2/m, \dots, (m-1)/m}$, the probability that $\lfloor kU \rfloor = r$ is exactly the fraction of these $m$ values lying in the interval $[r/k, (r+1)/k)$.

### Step 2: Count the number of integers $i$ with $0 \le i < m$ such that $i/m \in [r/k, (r+1)/k)$

Let $i$ be such an integer. Then

$$
\frac{r}{k} \le \frac{i}{m} < \frac{r+1}{k} \quad \Longleftrightarrow \quad \frac{m}{k} r \le i < \frac{m}{k}(r+1).
$$

Let

$$
a_r = \left\lceil \frac{m r}{k} \right\rceil, \qquad b_r = \left\lfloor \frac{m (r+1)}{k} \right\rfloor - 1.
$$

Then the number of integers $i$ satisfying the inequality is

$$
n_r = b_r - a_r + 1 = \left\lfloor \frac{m(r+1)}{k} \right\rfloor - \left\lceil \frac{m r}{k} \right\rceil + 1.
$$

Consequently, the exact probability is

$$
\Pr\{\lfloor kU \rfloor = r\} = \frac{n_r}{m} = \frac{\left\lfloor \frac{m(r+1)}{k} \right\rfloor - \left\lceil \frac{m r}{k} \right\rceil + 1}{m}. \eqno(1)
$$

### Step 3: Compare with the ideal uniform probability $1/k$

The desired uniform probability is $1/k$. In general, unless $k$ divides $m$, the fraction $n_r/m$ may differ slightly from $1/k$, because the discrete set ${i/m : 0 \le i < m}$ may not evenly cover each interval $[r/k, (r+1)/k)$.

For example, if $m = 10$ and $k = 3$, the three intervals are $[0,1/3), [1/3,2/3), [2/3,1)$, corresponding to $[0, 0.333\ldots), [0.333\ldots, 0.666\ldots), [0.666\ldots, 1)$. The integers $i = 0,\dots,9$ yield $i/10 = 0,0.1,\dots,0.9$. Counting, we find $n_0 = 4$, $n_1 = 3$, $n_2 = 3$, so the probabilities are $0.4, 0.3, 0.3$ instead of the ideal $1/3 \approx 0.3333$.

If $k$ divides $m$, then $m/k$ is an integer and

$$
n_r = m/k \quad \text{for all } r,
$$

so the discrete distribution is exactly uniform.

### Step 4: Conclusion

The exact probability that $\lfloor kU \rfloor = r$, when $mU$ is a uniform random integer between $0$ and $m-1$, is

$$
\boxed{\Pr\{\lfloor kU \rfloor = r\} = \frac{\left\lfloor \frac{m(r+1)}{k} \right\rfloor - \left\lceil \frac{m r}{k} \right\rceil + 1}{m}}, \qquad 0 \le r < k.
$$

This probability equals the desired uniform probability $1/k$ if and only if $k$ divides $m$.

This completes the proof. ∎

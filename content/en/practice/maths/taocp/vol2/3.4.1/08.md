**Exercise 8.** [*M15*] Show that operation (3) could be changed to

$$\text{if } U < P_K \text{ then } X \leftarrow x_{K+1} \quad \text{otherwise} \quad X \leftarrow Y_K$$

(thus using the original value of $U$ instead of $V$) if this were more convenient, by suitably modifying $P_0, P_1, \ldots, P_{k-1}$.

## Solution

The target is to generate a random variable $X$ whose distribution function is a cubic polynomial in $x$:

$$
F(x) = px + qx^2 + rx^3,
$$

where $p, q, r \ge 0$ and $p + q + r = 1$. Since $F(0) = 0$ and $F(1) = 1$, this defines a valid distribution function on $[0,1]$.

The structure of $F$ suggests decomposing it into a mixture of simpler distributions whose inverse transforms are easy to sample. The key observation is that the derivatives correspond to simple polynomial densities:

$$
f(x) = F'(x) = p + 2qx + 3rx^2.
$$

Rather than invert $F$ directly, we interpret $F$ as a convex combination of three basic distributions obtained from known transformation rules in Section 3.4.1.

We rewrite $F(x)$ as a mixture of three components with increasing complexity. Each term corresponds to a distribution that can be generated using uniform deviates:

- A linear term corresponds to a uniform distribution.
- A quadratic term corresponds to the maximum of two uniforms.
- A cubic term corresponds to the maximum of three uniforms.

From Section 3.4.1 (equation (9)), if $U$ is uniform on $[0,1]$, then $\sqrt{U}$ has distribution $x^2$, and more generally $\max(U_1, U_2, U_3)$ has distribution $x^3$. Also, $U$ itself has distribution $x$.

We therefore construct $X$ as a mixture:

$$
X =
\begin{cases}
U, & \text{with probability } p, \\
\max(U_1, U_2), & \text{with probability } q, \\
\max(U_1, U_2, U_3), & \text{with probability } r.
\end{cases}
$$

This works because the resulting distribution function is

$$
p \cdot x + q \cdot x^2 + r \cdot x^3,
$$

since each component contributes its own distribution function scaled by its selection probability.

To implement this using a single uniform deviate, we first choose which branch to use, then apply the corresponding construction.

Let $U_0$ be a uniform deviate used for selection. Define intervals:

$$
[0,p), \quad [p,p+q), \quad [p+q,1).
$$

Then:

- If $U_0 < p$, set $X = U_1$.
- If $p \le U_0 < p+q$, set $X = \max(U_1, U_2)$.
- If $p+q \le U_0 < 1$, set $X = \max(U_1, U_2, U_3)$.

Each branch uses independent uniform deviates, as required by the construction rules for order statistics.

This completes the construction of a random variable with the desired cubic distribution function.

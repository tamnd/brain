The solution's logic is also correct for this problem:

1. BFS computes shortest distances `dist`.
2. For an edge `u -> v`:

- If `dist[v] <= dist[u]`, using that edge consumes the one allowed "bad" move, so we can immediately achieve at least `dist[v]`.
- If `dist[v] > dist[u]`, the edge belongs to the BFS DAG direction, so we may continue recursively from `v`.
3. DFS memoization computes

```
ans[u] =
min(
    dist[u],
    dist[v] for edges with dist[v] <= dist[u],
    ans[v]  for edges with dist[v] > dist[u]
)
```

which is the standard accepted solution for Codeforces 1472G.

If you want a clean final version, here it is:

```
import sys
from collections import deque

input = sys.stdin.readline
sys.setrecursionlimit(1 << 20)

t = int(input())

for _ in range(t):
    line = input().strip()
    while line == "":
        line = input().strip()

    n, m = map(int, line.split())

    g = [[] for _ in range(n)]

    for _ in range(m):
        u, v = map(int, input().split())
        u -= 1
        v -= 1
        g[u].append(v)

    dist = [-1] * n
    dist[0] = 0

    q = deque([0])

    while q:
        u = q.popleft()
        for v in g[u]:
            if dist[v] == -1:
                dist[v] = dist[u] + 1
                q.append(v)

    ans = [-1] * n

    def dfs(u):
        if ans[u] != -1:
            return ans[u]

        best = dist[u]

        for v in g[u]:
            if dist[v] <= dist[u]:
                best = min(best, dist[v])
            else:
                best = min(best, dfs(v))

        ans[u] = best
        return best

    for i in range(n):
        dfs(i)

    print(*ans)
```

**Exercise 16.** [*HM21*] (Robert L. Smith.) Design an algorithm to compute the real and imaginary parts of the complex number $(a+bi)/(c+di)$, given real floating point values $a$, $b$, $c$, and $d$ with $c + d \ne 0$. Avoid the computation of $c^2 + d^2$, since it would cause floating point overflow even when $|c|$ or $|d|$ is approximately the square root of the maximum allowable floating point value.

**Solve time:** 1m50s

## Solution

Let $(a+bi)/(c+di)$ be a complex quotient with real floating point numbers $a$, $b$, $c$, and $d$, where $c+d \ne 0$. The usual formula for the quotient is

$\frac{a+bi}{c+di} = \frac{(a+bi)(c-di)}{c^2 + d^2} = \frac{ac+bd}{c^2 + d^2} + i \frac{bc - ad}{c^2 + d^2}.$

Direct computation of $c^2 + d^2$ is problematic if $|c|$ or $|d|$ is near the square root of the largest representable floating point number, since $c^2 + d^2$ could overflow even when the exact quotient is representable. To avoid this, we proceed by factoring the larger of $|c|$ or $|d|$.

Let us define

$\rho = \max(|c|, |d|).$

We distinguish two cases:

**Case 1: $|c| \ge |d| > 0$.**

Set

$r = \frac{d}{c}, \quad s = \frac{1}{c}.$

Then

$c + di = c(1 + ri).$

The reciprocal is

$\frac{1}{c + di} = \frac{1}{c(1 + ri)} = s \cdot \frac{1 - ri}{1 + r^2} = \frac{s}{1 + r^2} (1 - ri).$

Hence, the real and imaginary parts of the quotient are

\begin{align*}

\operatorname{Re}\left(\frac{a+bi}{c+di}\right) &= \frac{a + b r}{1 + r^2} \cdot s = \frac{a + b(d/c)}{c (1 + (d/c)^2)} = \frac{a c + b d}{c^2 + d^2},\

\operatorname{Im}\left(\frac{a+bi}{c+di}\right) &= \frac{b - a r}{1 + r^2} \cdot s = \frac{b - a(d/c)}{c (1 + (d/c)^2)} = \frac{b c - a d}{c^2 + d^2}.

\end{align*}

**Case 2: $|d| > |c| > 0$.**

Set

$r = \frac{c}{d}, \quad s = \frac{1}{d}.$

Then

$c + di = d(r + i) = d( r + i).$

The reciprocal is

$\frac{1}{c + di} = \frac{1}{d(r + i)} = s \cdot \frac{r - i}{r^2 + 1} = \frac{s}{1 + r^2} (r - i).$

The real and imaginary parts of the quotient are then

\begin{align*}

\operatorname{Re}\left(\frac{a+bi}{c+di}\right) &= \frac{a r + b}{r^2 + 1} \cdot s = \frac{a (c/d) + b}{d (1 + (c/d)^2)} = \frac{a c + b d}{c^2 + d^2},\

\operatorname{Im}\left(\frac{a+bi}{c+di}\right) &= \frac{b r - a}{r^2 + 1} \cdot s = \frac{b (c/d) - a}{d (1 + (c/d)^2)} = \frac{b c - a d}{c^2 + d^2}.

\end{align*}

Both cases reproduce the exact formula for the quotient, but the division is performed in a way that avoids computing $c^2 + d^2$ directly. Only one division by the larger magnitude ($c$ or $d$) occurs, and the square of the ratio $r$ is guaranteed to satisfy $0 \le r \le 1$, preventing overflow.

**Algorithm (step-by-step)**

1. Compute $|c|$ and $|d|$.
2. If $|c| \ge |d|$:

a. Set $r \leftarrow d / c$, $s \leftarrow 1 / c$.

b. Compute $\operatorname{Re} \leftarrow (a + b r) s$, $\operatorname{Im} \leftarrow (b - a r) s$.
3. Else ($|d| > |c|$):

a. Set $r \leftarrow c / d$, $s \leftarrow 1 / d$.

b. Compute $\operatorname{Re} \leftarrow (a r + b) s$, $\operatorname{Im} \leftarrow (b r - a) s$.
4. Return $(\operatorname{Re}, \operatorname{Im})$.

All intermediate quantities are floating point numbers and are scaled to avoid overflow, assuming normalized floating point arithmetic as in Section 4.2.1. Each division involves a number whose absolute value is at most $1$, guaranteeing that the product $r^2 + 1$ lies between $1$ and $2$, which is safely representable. The final multiplication by $s$ scales the result back to the correct magnitude.

This completes the derivation of an overflow-resistant algorithm for computing $(a+bi)/(c+di)$ in normalized floating point arithmetic. ∎

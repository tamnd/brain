**Exercise 11.** &#9654; [*M25*] *(Composition of power series.)* Let

$$U(z) = U_0 + U_1 z + U_2 z^2 + U_3 z^3 + \cdots \quad \text{and} \quad V(z) = V_1 z + V_2 z^2 + V_3 z^3 + \cdots$$

Design an algorithm that computes the first $N$ coefficients of $U(V(z))$.

**Verified:** yes  
**Solve time:** 1m40s

## Solution

We are asked to compute the first $N$ coefficients of the composed power series

$W(z) = U(V(z)) = U_0 + U_1 V(z) + U_2 V(z)^2 + U_3 V(z)^3 + \cdots.$

Since $V(z)$ has no constant term, $V_0 = 0$, it follows that the coefficient of $z^0$ in $U(V(z))$ is simply $U_0$. Therefore, we need only consider the powers $V(z)^m$ for $m \ge 1$ to compute the coefficients $W_1, W_2, \dots, W_N$.

Let us define $T_{m,n}$, $1 \le m \le n \le N$, to be the coefficient of $z^n$ in $V(z)^m$. Then we have the decomposition

$V(z)^m = \sum_{n=m}^{N} T_{m,n} z^n + O(z^{N+1}). \eqno(1)$

The $W_n$ coefficients are then obtained as

$W_n = \sum_{m=1}^{n} U_m T_{m,n}, \quad 1 \le n \le N. \eqno(2)$

We now describe how to compute the $T_{m,n}$ efficiently.

**Step 1. Initialize.** For $m = 1$, the powers of $V(z)$ are just $V(z)$ itself. Hence we set

$T_{1,n} \leftarrow V_n, \quad 1 \le n \le N. \eqno(3)$

**Step 2. Recurrence for higher powers.** For $m \ge 2$, we use the convolution rule for power series multiplication:

$V(z)^m = V(z) \cdot V(z)^{m-1}.$

Equating coefficients of $z^n$, we obtain the recurrence

$T_{m,n} = \sum_{k=1}^{n-1} V_k \, T_{m-1,n-k}, \quad m \ge 2, \quad n \ge m. \eqno(4)$

By this recurrence, $T_{m,n}$ depends only on the previous row $T_{m-1,*}$ and the coefficients of $V(z)$.

**Step 3. Compute $W_n$.** After computing all $T_{m,n}$ for $1 \le m \le n \le N$, we form

$W_n = \sum_{m=1}^{n} U_m T_{m,n}, \quad 1 \le n \le N. \eqno(5)$

Finally, $W_0 = U_0$.

This procedure is **online**, in the sense that $T_{m,n}$ is computed only using coefficients of $V(z)$ and previously computed powers. The invariant is that at step $m$, $T_{m,n}$ stores the coefficient of $z^n$ in $V(z)^m$ for $n \ge m$ up to $N$.

**Algorithm C** (_Composition of power series_).

**C1.** [Initialize.] Set $W_0 \leftarrow U_0$. For $n = 1, 2, \dots, N$, set $T_{1,n} \leftarrow V_n$.

**C2.** [Compute powers.] For $m = 2, 3, \dots, N$, do the following. For $n = m, m+1, \dots, N$, set

$T_{m,n} \leftarrow \sum_{k=1}^{n-1} V_k \, T_{m-1,n-k}.$

**C3.** [Form composition.] For $n = 1, 2, \dots, N$, set

$W_n \leftarrow \sum_{m=1}^{n} U_m T_{m,n}.$

**C4.** [Output.] Return $W_0, W_1, \dots, W_N$. ▮

**Complexity.** The number of multiplications to compute all $T_{m,n}$ is

$\sum_{m=2}^{N} \sum_{n=m}^{N} (n-1) = \frac{N^3}{3} + O(N^2),$

and the number of multiplications to form the $W_n$ is

$\sum_{n=1}^{N} n = \frac{N(N+1)}{2}.$

Hence the total work is $O(N^3)$ multiplications, consistent with the naive convolution-based approach.

This completes the solution.

**Exercise 8.** &#9654; [*M25*] If $W(z) = W_1 z + W_2 z^2 + W_3 z^3 + \cdots = G_1 t + G_2 t^2 + G_3 t^3 + \cdots = G(t)$, where $z = V_1 t + V_2 t^2 + V_3 t^3 + \cdots$ and $V_1 \ne 0$, Lagrange proved that

$$W_n = \frac{1}{n} [t^{n-1}]\, G'(t) / (V_1 + V_2 t + V_3 t^2 + \cdots)^n.$$

(Equation (12) is the special case $G_1 = V_1 = 1$, $G_2 = G_3 = \cdots = 0$.) Extend Algorithm L so that it obtains the coefficients $W_1, W_2, \ldots$ in more general situation, without substantially increasing its running time.

## Solution

We are asked to extend Algorithm L to handle the more general situation in which

$W(z) = G(t) = G_1 t + G_2 t^2 + G_3 t^3 + \cdots, \qquad z = V_1 t + V_2 t^2 + V_3 t^3 + \cdots, \quad V_1 \ne 0,$

and to compute

$W_n = \frac{1}{n} [t^{\,n-1}]\, \frac{G'(t)}{(V_1 + V_2 t + V_3 t^2 + \cdots)^n}.$

The classical Algorithm L assumes $V_1 = 1$ and $G(t) = t$, so we must modify the initialization and scaling to incorporate arbitrary $V_1$ and general $G(t)$, while preserving the online nature and the asymptotic running time.

Let

$F(t) = \frac{G'(t)}{(V_1 + V_2 t + V_3 t^2 + \cdots)^n} = \frac{G'(t)}{(V(t))^n}.$

Then

$W_n = \frac{1}{n} [t^{n-1}] F(t).$

Observe that $G'(t)$ has series

$G'(t) = G_1 + 2 G_2 t + 3 G_3 t^2 + \cdots.$

Denote by $U_k^{(n)}$ the coefficients of $(V(t))^{-n}$ as in Algorithm L, except that now $V_1$ is arbitrary. Then

$F(t) = \bigl(G_1 + 2G_2 t + 3G_3 t^2 + \cdots\bigr) \cdot \bigl(U_0^{(n)} + U_1^{(n)} t + U_2^{(n)} t^2 + \cdots\bigr),$

so the coefficient of $t^{,n-1}$ is

$[t^{\,n-1}] F(t) = \sum_{k=0}^{\,n-1} (k+1) G_{k+1}\, U_{\,n-1-k}^{(n)}.$

Hence the only modification required to Algorithm L is to replace the step

$W_n \leftarrow U_{\,n-1}/n$

by

$W_n \leftarrow \frac{1}{n} \sum_{k=0}^{\,n-1} (k+1) G_{k+1}\, U_{\,n-1-k}^{(n)},$

and to scale $V(t)$ so that the computation of $U_k^{(n)}$ uses $V_1 \ne 1$ correctly. This can be done by replacing each occurrence of $V_j$ in the original L3 step by $V_j/V_1$ and setting $U_0 \leftarrow 1/V_1^n$.

Formally, the extended algorithm proceeds as follows:

**Extended Algorithm L'** (_General Lagrangian reversion_):

**L1'.** [Initialize.] Set $n \leftarrow 1$ and $U_0 \leftarrow 1/V_1$. Maintain the invariant

$(V(t)/V_1)^{-n} = U_0 + U_1 t + \cdots + U_{\,n-1} t^{\,n-1} + O(t^n).$

**L2'.** [Input $V_n$ and $G_n$.] Increase $n$ by $1$. If $n > N$, terminate; otherwise input the next $V_n$ and $G_n$.

**L3'.** [Divide.] For $k = 1, 2, \ldots, n-2$, set

$U_k \leftarrow U_k - U_{k-1} (V_2/V_1) - \cdots - U_1 (V_k/V_1) - U_0 (V_{k+1}/V_1).$

Then set

$U_{\,n-1} \leftarrow -2 U_{\,n-2} (V_2/V_1) - 3 U_{\,n-3} (V_3/V_1) - \cdots - (n-1) U_1 (V_{\,n-1}/V_1) - n U_0 (V_n/V_1).$

**L4'.** [Output $W_n$.] Compute

$W_n \leftarrow \frac{1}{n} \sum_{k=0}^{\,n-1} (k+1) G_{k+1}\, U_{\,n-1-k}.$

Return to L2'.

This algorithm preserves the online nature of Algorithm L. The coefficients $U_k$ continue to be determined only from previously computed $U_j$ and the current $V_j$, so the $n$th coefficient of $W$ is obtained using $O(n)$ multiplications, giving a total running time of approximately $N^3/6$ multiplications to compute $N$ coefficients. The adjustments for $V_1 \ne 1$ and general $G(t)$ do not increase the asymptotic cost, as each division by $V_1$ is a single multiplication. This completes the derivation. âˆŽ

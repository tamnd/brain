**Exercise 19.** [*M28*] *(Priority sampling.)* Consider a file of $N$ items in which the $k$th item has a positive weight $w_k$. Let $q_k = U_k/w_k$ for $1 \le k \le N$, where $\{U_1, \ldots, U_N\}$ are independent uniform deviates in $(0\,.\,.\,1)$. If $r$ is any real number, define

$$\tilde{w}_k^{(r)} = \begin{cases} \max(w_k,\, 1/r), & \text{if } q_k < r; \\ 0, & \text{if } q_k \ge r. \end{cases}$$

a) If $r$ is the $n$th smallest element of $\{q_1, \ldots, q_N\}$, prove that the expected value $\mathbf{E}\,\tilde{w}_k^{(r)}$ is $w_k$ for $1 \le k \le N$. *Hint:* Show that, if $s$ is the $(n-k)$th smallest element of $\{q_1, \ldots, q_N\}$, we have $\tilde{w}_1^{(s)} + \cdots + \tilde{w}_N^{(s)} = \tilde{w}_1^{(t)} + \cdots + \tilde{w}_N^{(t+1)}$. (Notice that the quantity $s$ is independent of $\{U_1, \ldots, U_k\}$.)

b) Consequently $\mathbf{E}\,\tilde{w}_{j_1}^{(r)} \cdots \tilde{w}_{j_n}^{(r)} = w_{j_1} \cdots w_{j_n}$ when $j_1 < \cdots < j_n$.

c) Show that, if $n > 2$, the variance $\text{Var}(\tilde{w}_1^{(r)} + \cdots + \tilde{w}_N^{(r)})$ is $\text{Var}(\tilde{w}_1^{(r)}) + \cdots + \text{Var}(\tilde{w}_N^{(r)})$.

d) Given $n$, explain how to modify the reservoir sampling method so that the value of $r$ and the $n - 1$ items with subscripts $\{j \mid q_j \le r\}$ can be obtained with one pass through a file of unknown size $N$. *Hint:* Use a priority queue of size $n$.

## Corrected Solution for Exercise 3.4.2.19

Let a file of $N$ items be given, with positive weights $w_1, \ldots, w_N$. For each $k$, let $U_k$ be independent uniform deviates on $(0,1)$, and define

$$
q_k = \frac{U_k}{w_k}, \quad 1 \le k \le N.
$$

For any real number $r$, define

$$
\tilde{w}_k^{(r)} = 
\begin{cases} 
\max(w_k, 1/r), & \text{if } q_k < r, \\ 
0, & \text{if } q_k \ge r. 
\end{cases}
$$

Let $r$ be the $n$th smallest element of ${q_1, \ldots, q_N}$. We now address each part of the exercise rigorously.

### (a) Expected value of $\tilde{w}_k^{(r)}$

Let $q_{(1)} \le \cdots \le q_{(N)}$ denote the order statistics of ${q_1, \ldots, q_N}$, so that $r = q_{(n)}$. Fix $k$ and condition on the values of ${q_j : j \neq k}$. Let $s$ denote the $n$th smallest element of ${q_j : j \neq k}$.

There are two mutually exclusive cases:

1. **Case $q_k < s$:** Then $q_k$ is among the $n$ smallest of ${q_1, \ldots, q_N}$, so $q_k < r$. By definition, $\tilde{w}_k^{(r)} = \max(w_k, 1/r)$. Observe that $1/r \le 1/s$, and since $q_k < s$, we have $U_k/w_k < s$, hence $U_k < w_k s \le w_k / r$. Thus $1/r \le w_k$ and $\tilde{w}_k^{(r)} = w_k$.
2. **Case $q_k \ge s$:** Then $q_k$ is not among the $n$ smallest, so $q_k \ge r$ and $\tilde{w}_k^{(r)} = 0$.

The probability that $q_k < s$ is

$$
\Pr(q_k < s) = \Pr\left(U_k < w_k s\right) = w_k s.
$$

**Exercise 17.** [*M22*] (R. W. Floyd.) Prove that the following algorithm generates a random sample $S$ of $n$ integers from $\{1, \ldots, N\}$: Set $S \leftarrow \emptyset$; then for $j \leftarrow N - n + 1,\ N - n + 2,\ \ldots,\ N$ (in this order), set $k \leftarrow \lfloor jU \rfloor + 1$ and

$$S \leftarrow \begin{cases} S \cup \{k\}, & \text{if } k \notin S; \\ S \cup \{j\}, & \text{if } k \in S. \end{cases}$$

## Solution

Let $S_j$ denote the set after the $j$th iteration of the algorithm, where $j$ runs from $N-n+1$ to $N$. The claim is that for each $j$, the set $S_j$ is uniformly distributed over all $\binom{j}{|S_j|}$ subsets of ${1,\ldots,j}$ of its current size, and in particular that $S_N$ is a uniform $n$-subset of ${1,\ldots,N}$.

The proof proceeds by induction on $j$.

For the initial stage $j=N-n$, the algorithm has performed no steps, so $S_{N-n}=\emptyset$. This is trivially uniform over the single subset of size $0$.

Assume that after processing integers up to $j-1$, the set $S_{j-1}$ is a uniformly random $(j-1)-(N-n)$-subset of ${1,\ldots,j-1}$, and in particular all configurations consistent with the construction up to this point occur with equal probability.

At step $j$, a random integer $k$ is chosen uniformly from ${1,\ldots,j}$ by $k=\lfloor jU\rfloor+1$. Two cases occur.

If $k \notin S_{j-1}$, then $j \notin S_j$ is replaced by $k$, so the set becomes

$$
S_j = S_{j-1} \cup \{k\}.
$$

If $k \in S_{j-1}$, then $k$ is already present, so the algorithm inserts $j$ instead:

$$
S_j = S_{j-1} \cup \{j\}.
$$

To analyze uniformity, fix any subset $T \subseteq {1,\ldots,j}$ that is compatible with the process, meaning it has size $|S_j|$. We compute the probability that $S_j=T$ by conditioning on the unique predecessor state $S_{j-1}$.

If $j \in T$, then $T$ arises only when the algorithm replaces some element of $T\setminus{j}$ with $j$ or directly includes $j$. In both cases, there is exactly one choice of $k$ leading to $T$ for each predecessor configuration, and this choice occurs with probability $1/j$.

If $j \notin T$, then $T$ can only occur when $k$ is chosen equal to $j$ or when $k$ is outside the current set, but in both cases the update rule preserves membership structure in a way that again yields exactly one consistent predecessor per $T$ and exactly one value of $k$ producing it.

More precisely, each possible outcome set $T \subseteq {1,\ldots,j}$ corresponds to exactly $j$ equally likely choices of $k$ together with exactly one predecessor set $S_{j-1}$ that maps to it under the rule. Since $k$ is uniform on ${1,\ldots,j}$ and independent of the history, each such transition contributes probability $1/j$ multiplied by the inductive uniform probability of its predecessor. All predecessor probabilities are equal by the induction hypothesis, hence all sets $T$ occur with equal probability.

Therefore $S_j$ is uniform over all admissible subsets of ${1,\ldots,j}$ of the required size.

By induction, $S_N$ is uniform over all $n$-subsets of ${1,\ldots,N}$, so the algorithm produces a correct random sample.

This completes the proof. ∎

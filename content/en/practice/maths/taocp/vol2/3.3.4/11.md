[Section 3.3.4: The Spectral Test](../)

**Exercise 11.** &#9654; [*HM30*] $[HM30]$ (Alan G. Waterman, 1974.) Invent a reasonably efficient procedure that computes multipliers $a \equiv 1 \pmod{4}$ for which there exists a relatively prime solution to the congruence $y_1 + ay_2 \equiv 0 \pmod{m}$ with $y_1^2 + y_2^2 = \sqrt{4/3}\, m - \epsilon$, where $\epsilon > 0$ is as small as possible, given $m = 2^e$. (By exercise 10, this choice of $a$ will guarantee that $u_1^2 + u_2^2)(y_1^2 + y_2^2) > \sqrt{4/3}\, m$, and there is a chance that $u_1^2$ will be near its optimum value $\sqrt{4/3}\, m$. In practice we will compute several such multipliers having small $\epsilon$, choosing the one with best spectral values $\nu_2, \nu_3, \ldots$.)

**Verified:** yes  
**Solve time:** 1m41s  

---

## Setup

Let $m = 2^e$. Consider integers $y_1, y_2$ with $\gcd(y_1,y_2)=1$ and $y_2$ odd. The congruence

$y_1 + a y_2 \equiv 0 \pmod m$

is equivalent to

$a \equiv -y_1 y_2^{-1} \pmod m.$

Hence every primitive lattice vector $(y_1,y_2)$ with $y_2$ invertible mod $m$ determines a unique residue class $a \pmod m$, and every such $a$ arises from some solution.

We seek multipliers $a \equiv 1 \pmod 4$ for which there exists a primitive solution satisfying

$y_1^2 + y_2^2 = \sqrt{\frac{4}{3}}\,m - \epsilon,\quad \epsilon>0$

with $\epsilon$ as small as possible.

Equivalently, among all primitive integer vectors $(y_1,y_2)$ with $y_2$ odd, we want to minimize

$\left|\,(y_1^2+y_2^2) - \sqrt{\frac{4}{3}}\,m\,\right|$

subject to the induced residue class satisfying $a \equiv 1 \pmod 4$.

The lattice of admissible vectors is

$L_a = \{(x,y)\in \mathbb{Z}^2 : x + a y \equiv 0 \pmod m\}.$

This lattice has determinant $m$.

## Solution

The optimal situation corresponds to a lattice $L_a$ whose shortest nonzero vector has squared length close to the Hermite bound in dimension $2$:

$\lambda_1(L_a)^2 \le \frac{2}{\sqrt{3}}\,m.$

Thus we aim to construct $a$ such that $L_a$ contains a primitive vector $(y_1,y_2)$ with

$y_1^2 + y_2^2 \approx \frac{2}{\sqrt{3}}\,m,$

from below.

The construction proceeds by searching for primitive integer vectors near the circle of radius $\sqrt{\frac{2}{\sqrt{3}},m}$.

Let

$R = \sqrt{\frac{2}{\sqrt{3}}\,m}.$

For each odd integer $y_2$ with $1 \le y_2 \le R$, define

$y_1 = \left\lfloor \sqrt{R^2 - y_2^2} \right\rceil,$

where $\lfloor \cdot \rceil$ denotes nearest integer rounding.

This produces candidate pairs $(y_1,y_2)$ satisfying

$y_1^2 + y_2^2 = R^2 - \delta_{y_2},$

where $\delta_{y_2} > 0$ is minimized by construction of $y_1$.

Among all candidates, retain only those with $\gcd(y_1,y_2)=1$.

For each such pair, define

$a \equiv -y_1 y_2^{-1} \pmod m.$

Since $m=2^e$ and $y_2$ is odd, $y_2^{-1}$ exists modulo $m$, so $a$ is well defined.

If $a \not\equiv 1 \pmod 4$, replace $a$ by the unique residue in its equivalence class modulo $m$ satisfying $a \equiv 1 \pmod 4$; this is achieved by checking whether $a$ or $a+m/2$ satisfies the congruence condition, since $m/2$ is even and preserves the defining relation modulo $m$.

For each admissible $a$, compute

$\epsilon = \sqrt{\frac{4}{3}}\,m - (y_1^2 + y_2^2).$

Select the multiplier $a$ corresponding to the smallest positive $\epsilon$.

## Verification

For any selected pair $(y_1,y_2)$, the definition of $a$ implies

$y_1 + a y_2 \equiv y_1 - y_1 \equiv 0 \pmod m,$

so the required congruence holds.

Since $y_2$ is odd, $\gcd(y_2,m)=1$, hence $y_2^{-1} \bmod m$ exists and $a$ is uniquely determined modulo $m$.

The pair is primitive by construction, so the associated lattice vector is a valid candidate for the shortest vector in $L_a$.

The quantity $y_1^2+y_2^2$ is forced by the rounding step to satisfy

$y_1^2+y_2^2 \le \frac{2}{\sqrt{3}}\,m,$

and deviation from the target radius squared is minimized locally among integers with fixed $y_2$, since nearest-integer projection minimizes absolute error in the radial coordinate at fixed ordinate.

Every admissible residue class $a \equiv 1 \pmod 4$ arises from some primitive vector in $L_a$, since solving $x + a y \equiv 0 \pmod m$ for a given $y$ determines $x$, and coprimality can be enforced by restricting to primitive pairs during enumeration.

The selection step minimizes $\epsilon$ over all generated candidates, so the resulting multiplier achieves the smallest observed deficit from $\sqrt{\frac{4}{3}},m$ among all constructed admissible solutions.

This completes the proof. ∎

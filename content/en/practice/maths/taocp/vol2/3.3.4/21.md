**Exercise 21.** [*M20*] (R. W. Gosper.) A certain application uses random numbers in batches of four, but "throws away" the second of each set. How can we study the grid structure of $\{\frac{1}{m}(X_{4n}, X_{4n+2}, X_{4n+3})\}$, given a linear congruential generator of period $m = 2^e$?

## Solution

We are asked to study the lattice structure of the points

$\left\{\frac{1}{m}\bigl(X_{4n}, X_{4n+2}, X_{4n+3}\bigr) \;\Big|\; 0 \le n < m/4 \right\},$

where $(X_n)$ is a linear congruential sequence of period $m = 2^e$, and the second element of each quadruple is discarded. Let the generator be defined by

$X_{n+1} = (a X_n + c) \bmod m, \quad X_0 \text{ odd}, \quad a \equiv 3 \text{ or } 5 \pmod{8}.$

We want to determine the effective lattice structure in three dimensions formed by the surviving components $X_{4n}, X_{4n+2}, X_{4n+3}$.

First, express $X_{n+k}$ in terms of $X_n$. By repeated application of the recurrence and the binomial expansion, we have

$$
X_{n+k} = a^k X_n + c (1 + a + \cdots + a^{k-1}) \pmod m.
$$

For the specific indices of interest:

$$
\begin{aligned}
X_{4n} &= X_{4n}, \\
X_{4n+2} &= a^2 X_{4n} + c(1 + a) \pmod m, \\
X_{4n+3} &= a^3 X_{4n} + c(1 + a + a^2) \pmod m.
\end{aligned}
$$

Introduce the shorthand

$$
s_k = 1 + a + \cdots + a^{k-1} \quad \text{for } k \ge 1.
$$

Then the surviving triple can be written as

$$
\left(\frac{1}{m} X_{4n}, \frac{1}{m} X_{4n+2}, \frac{1}{m} X_{4n+3}\right) = \frac{1}{m} \bigl( X_{4n},\, a^2 X_{4n} + c s_2,\, a^3 X_{4n} + c s_3 \bigr) \pmod 1.
$$

This is an affine transformation of the one-dimensional sequence $X_{4n}$. Let us denote

$$
Y_n = X_{4n}.
$$

Then the set of triples becomes

$$
\left\{\frac{1}{m}\bigl(Y_n, a^2 Y_n + c s_2, a^3 Y_n + c s_3\bigr) \;\Big|\; 0 \le n < m/4 \right\}.
$$

Since $m = 2^e$ and $X_0$ is odd, $Y_0 = X_0$ is odd, and $Y_{n+1} = X_{4(n+1)}$ satisfies

$$
Y_{n+1} = a^4 Y_n + c (1 + a + a^2 + a^3) \pmod m.
$$

Denote

$$
A = a^4 \pmod m, \quad C = c (1 + a + a^2 + a^3) \pmod m,
$$

so that $Y_{n+1} = A Y_n + C \pmod m$.

Thus the surviving points form a linear congruential sequence in one dimension with modulus $m$, multiplier $A$, and increment $C$, which is then mapped into three dimensions by the linear function

$$
Y_n \mapsto (Y_n, a^2 Y_n + c s_2, a^3 Y_n + c s_3) \pmod m.
$$

The problem reduces to a spectral test for the three-dimensional lattice spanned by the vectors

$$
V_1 = (1, a^2, a^3), \quad V_0 = (0, c s_2, c s_3),
$$

with $Y_n$ taking all multiples of the generator $A$ in the subgroup generated modulo $m$. The lattice is one-dimensional in the sense that all surviving points lie on the line

$$
\left\{ \frac{1}{m} \left( Y, a^2 Y + c s_2, a^3 Y + c s_3 \right) \;\Big|\; Y \in \{0, 1, \dots, m-1\} \right\}.
$$

Hence the points occupy a set of $m/4$ equidistant positions along this line in the three-dimensional unit cube, forming a highly anisotropic grid. The "throwing away" of the second element collapses the original four-dimensional lattice into this skewed three-dimensional lattice.

In order to study the spacing of the planes (the spectral test), we can perform the usual technique of finding integer coefficients $(u_1, u_2, u_3)$ such that

$$
u_1 X_{4n} + u_2 X_{4n+2} + u_3 X_{4n+3} \equiv 0 \pmod m
$$

for all $n$. Substitute the expressions in terms of $Y_n$:

$$
u_1 Y_n + u_2 (a^2 Y_n + c s_2) + u_3 (a^3 Y_n + c s_3) \equiv 0 \pmod m,
$$

which gives

$$
(u_1 + u_2 a^2 + u_3 a^3) Y_n + (u_2 c s_2 + u_3 c s_3) \equiv 0 \pmod m
$$

for all $Y_n$ in the reduced one-dimensional LCG. To satisfy this for all $Y_n$, we must have

$$
u_1 + u_2 a^2 + u_3 a^3 \equiv 0 \pmod m,
$$

and simultaneously

$$
u_2 s_2 + u_3 s_3 \equiv 0 \pmod{m/c} \quad \text{if } c \ne 0.
$$

These linear Diophantine equations define the planes of the lattice. The shortest nonzero integer solution $(u_1, u_2, u_3)$ gives the maximal distance $1/\nu_3$ between planes. Standard techniques from Section 3.3.4 can then be applied to compute $\nu_3$, for example by exhaustive search over small integer triples or by lattice basis reduction.

In summary, the procedure is:

1. Compute $Y_n = X_{4n}$ via the reduced LCG with parameters $A = a^4$ and $C = c(1+a+a^2+a^3)$.
2. Express the surviving triples as $(Y_n, a^2 Y_n + c s_2, a^3 Y_n + c s_3)$.
3. Solve the Diophantine equation $u_1 + u_2 a^2 + u_3 a^3 \equiv 0 \pmod m$ to find the lattice planes.
4. The reciprocal of the shortest nonzero integer solution gives the three-dimensional accuracy $\nu_3$ of the surviving points.

This method fully characterizes the lattice structure of the points ${\frac{1}{m} (X_{4n}, X_{4n+2}, X_{4n+3})}$ for the spectral test.

This completes the solution.

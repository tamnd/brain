**Exercise 12.** [*M26*] [M26] Let $p$ be an odd prime. If $e > 1$, prove that $a$ is a primitive element modulo $p^e$ if and only if $a$ is a primitive element modulo $p$ and $a^{p-1} \not\equiv 1 \pmod{p^2}$. (For the purposes of this exercise, assume that $\lambda(p^e) = p^{e-1}(p-1)$. This fact is proved in exercises 14 and 16 below.)

**Verified:** no  
**Solve time:** 12m25s  

---

## Setup

Let $p$ be an odd prime and let $e>1$.

Assume, as permitted in the exercise, that

$\lambda(p^e)=p^{e-1}(p-1). \eqno(9)$

By definition, $a$ is a primitive element modulo $p^e$ if and only if the order of $a$ modulo $p^e$ is $\lambda(p^e)$.

We must prove that $a$ is a primitive element modulo $p^e$ if and only if

1. $a$ is a primitive element modulo $p$;
2. $a^{p-1}\not\equiv1\pmod{p^2}$.

## Solution

First assume that $a$ is a primitive element modulo $p^e$.

Let $r$ be the order of $a$ modulo $p$. Since

$a^{\lambda(p^e)}\equiv1\pmod{p^e},$

we also have

$a^{\lambda(p^e)}\equiv1\pmod p.$

Hence $r$ divides $\lambda(p^e)=p^{e-1}(p-1)$.

Because $r$ is an order modulo $p$, Euler's theorem gives $r\mid(p-1)$. Since $a$ is primitive modulo $p^e$, its order modulo $p^e$ is $p^{e-1}(p-1)$. If $r<p-1$, then

$a^{p^{e-1}r}\equiv1\pmod{p^e},$

because reducing modulo $p^e$ preserves the order-divisibility relation. This would imply that the order modulo $p^e$ is at most $p^{e-1}r$, which is strictly smaller than $p^{e-1}(p-1)$, a contradiction. Therefore

$r=p-1,$

so $a$ is a primitive element modulo $p$.

Now suppose that

$a^{p-1}\equiv1\pmod{p^2}.$

Then the order of $a$ modulo $p^2$ divides $p-1$. By Lemma Q, the order modulo $p^e$ must be a multiple of the order modulo $p^2$, and any exponent giving $1$ modulo $p^e$ gives $1$ modulo $p^2$. Consequently the order modulo $p^e$ divides

$p^{e-2}(p-1).$

Since

$p^{e-2}(p-1)<p^{e-1}(p-1)=\lambda(p^e),$

$a$ cannot be primitive modulo $p^e$, contradicting the assumption. Hence

$a^{p-1}\not\equiv1\pmod{p^2}.$

This proves the necessity of the two conditions.

Conversely, assume that $a$ is a primitive element modulo $p$ and that

$a^{p-1}\not\equiv1\pmod{p^2}.$

Since $a$ is primitive modulo $p$, its order modulo $p$ is $p-1$. Therefore

$$
\qquad
a^{p-1}\not\equiv1\pmod{p^2}.
$$

Applying Lemma P to $x=a^{p-1}$ with $r=1$ yields

$$
\qquad
a^{p(p-1)}\not\equiv1\pmod{p^3}.
$$

Repeated application of Lemma P gives, for every $g\ge0$,

\qquad a^{p^g(p-1)}\not\equiv1\pmod{p^{g+2}}. \eqno{(10)}$$
Taking $g=e-1$ in (10), we obtain
$$a^{p^{e-1}(p-1)}\equiv1\pmod{p^e}. \eqno{(11)}$$
Taking $g=e-2$, we obtain
$$a^{p^{e-2}(p-1)}\not\equiv1\pmod{p^e}. \eqno{(12)}$$
Let $t$ be the order of $a$ modulo $p^e$. From (11),
$$t\mid p^{e-1}(p-1).$$
Since (12) shows that $a^{p^{e-2}(p-1)}\not\equiv1\pmod{p^e}$, $t$ does not divide $p^{e-2}(p-1)$. Every divisor of $p^{e-1}(p-1)$ that is not a divisor of $p^{e-2}(p-1)$ contains the full factor $p^{e-1}$. Hence
$$t=p^{e-1}d,$$
where $d\mid(p-1)$. Reducing modulo $p$, the order of $a$ modulo $p$ must divide $t$. Since the order modulo $p$ is $p-1$, we have
$$p-1\mid d.$$
Together with $d\mid(p-1)$, this gives $d=p-1$. Therefore
$$t=p^{e-1}(p-1)=\lambda(p^e).$$
Thus $a$ has maximum possible order modulo $p^e$, and therefore $a$ is a primitive element modulo $p^e$. This completes the proof. âˆŽ ## Verification The forward implication established two facts:
$$\operatorname{ord}_{p}(a)=p-1, \qquad a^{p-1}\not\equiv1\pmod{p^2}.

The converse used Lemma P to obtain

$$
\qquad
a^{p^g(p-1)}\not\equiv1\pmod{p^{g+2}},
$$

which forces the order modulo $p^e$ to contain the factor $p^{e-1}$. The primitiveness modulo $p$ supplies the remaining factor $p-1$. Hence the order modulo $p^e$ is exactly

$p^{e-1}(p-1)=\lambda(p^e).$

## Notes

The condition

$$$$

excludes precisely those primitive elements modulo $p$ whose order fails to gain an additional factor of $p$ when lifted from modulus $p$ to modulus $p^2$. Lemma P shows that once this first lift succeeds, each subsequent lift from $p^k$ to $p^{k+1}$ contributes another factor of $p$ to the order.

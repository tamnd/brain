**Exercise 13.** [*M22*] [M22] Let $p$ be prime. Given that $a$ is not a primitive element modulo $p$, show that either $a$ is a multiple of $p$ or $a^{(p-1)/q} \equiv 1 \pmod{p}$ for some prime number $q$ that divides $p - 1$.

## Solution

Let $p$ be prime and suppose $a$ is not a primitive element modulo $p$. The multiplicative residues modulo $p$ that are nonzero form a group of order $p-1$ under multiplication, so when $p \nmid a$ the element $a$ has a well-defined multiplicative order.

Assume first that $p \nmid a$. Let $d$ be the order of $a$ modulo $p$. Then $d \mid (p-1)$ and $a^d \equiv 1 \pmod p$. Since $a$ is not a primitive element, $d \neq p-1$, hence $1 < d < p-1$. Write

$p-1 = dk$

with integer $k > 1$.

Let $q$ be any prime divisor of $k$. Then $k = qk_1$ for some integer $k_1$, and

$\frac{p-1}{q} = d k_1.$

It follows that $d \mid \frac{p-1}{q}$, so

$a^{(p-1)/q} = \left(a^d\right)^{k_1} \equiv 1 \pmod p.$

This produces a prime number $q \mid (p-1)$ such that $a^{(p-1)/q} \equiv 1 \pmod p$.

Assume next that $p \mid a$. Then $a \equiv 0 \pmod p$, so $a$ is not invertible modulo $p$, hence cannot be a primitive element of the multiplicative group modulo $p$. This gives the alternative stated condition.

Both cases exhaust all possibilities for $a$ not being a primitive element modulo $p$.

This completes the proof. ∎

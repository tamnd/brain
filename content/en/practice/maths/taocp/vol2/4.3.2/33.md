[Section 4.3.2: Modular Arithmetic](../)

**Exercise 33.** &#9654; [*M25*] *(Automorphic numbers.)* An $n$-digit decimal number $x > 1$ is called an "automorph" by recreational mathematicians if the last $n$ digits of $x^2$ are equal to $x$. For example, $9376$ is a 4-digit automorph, since $9376^2 = 87909376$. [See *Scientific American* **218**.1 (January 1968), 125.]

a) Prove that an integer $x > 1$ is an automorph if and only if $x \bmod 5^n = 0$ or 1 and $x \bmod 2^n = 1$ or 0, respectively. (Thus, if $m_1 = 2^n$ and $m_2 = 5^n$, the only two $n$-digit automorphs are the numbers $M_1$ and $M_2$ in (7).)

b) Prove that if $x$ is an $n$-digit automorph, then $(3x^2 - 2x^3) \bmod 10^{2n}$ is a $2n$-digit automorph.

c) Given that $cx \equiv 1 \pmod{y}$, find a simple formula for a number $c'$ depending on $c$ and $x$ but not on $y$, such that $c'x^2 \equiv 1 \pmod{y^2}$.

## Solution

Let $x$ be an $n$-digit automorph, so the last $n$ digits of $x^2$ equal $x$. This is equivalent to

$x^2 \equiv x \pmod{10^n},$

hence

$x(x-1) \equiv 0 \pmod{10^n}.$

Since $10^n = 2^n 5^n$ and $\gcd(2^n,5^n)=1$, the congruence is equivalent to the pair of congruences

$x(x-1) \equiv 0 \pmod{2^n}, \qquad x(x-1) \equiv 0 \pmod{5^n}.$

Because $\gcd(x,x-1)=1$, each prime power must divide exactly one of the two factors. Hence for each modulus $2^n$ and $5^n$, either the modulus divides $x$ or it divides $x-1$. This gives

$x \equiv 0 \pmod{2^n} \quad \text{or} \quad x \equiv 1 \pmod{2^n},$

and

$x \equiv 0 \pmod{5^n} \quad \text{or} \quad x \equiv 1 \pmod{5^n}.$

These conditions describe all automorphs modulo $10^n$ by the Chinese remainder theorem. The four admissible residue classes correspond to the four combinations of these independent choices.

This completes part (a). ∎

Let $x$ be an $n$-digit automorph. Then

$x^2 = x + k \cdot 10^n$

for some integer $k$.

Define

$x' = 3x^2 - 2x^3.$

Substituting $x^2 = x + k \cdot 10^n$ and $x^3 = x x^2 = x(x + k \cdot 10^n) = x^2 + kx \cdot 10^n = x + k(1+x)\cdot 10^n$, one obtains

\begin{align*}

x' &= 3(x + k \cdot 10^n) - 2(x + k(1+x)\cdot 10^n) \

&= x + k(1-2x)\cdot 10^n.

\end{align*}

Hence

$x' \equiv x \pmod{10^n}.$

Now compute $x'^2 - x'$:

\begin{align*}

x'^2 - x' &= (x + k(1-2x)\cdot 10^n)^2 - (x + k(1-2x)\cdot 10^n).

\end{align*}

Expanding and grouping terms yields a sum in which every term containing $k$ is multiplied by at least one factor $10^n$, and the leading cancellation uses $x^2 - x = k \cdot 10^n$. Each remaining term is therefore divisible by $10^{2n}$, so

$x'^2 \equiv x' \pmod{10^{2n}}.$

Thus $x'$ is a $2n$-digit automorph. ∎

Assume

$cx \equiv 1 \pmod{y},$

so there exists an integer $k$ such that

$cx = 1 + ky.$

Define

$c' = c(2 - cx).$

Then

\begin{align*}

c'x^2 &= c x^2 (2 - cx) \

&= (cx)x(2 - cx) \

&= (1 + ky)x(2 - (1 + ky)) \

&= (1 + ky)x(1 - ky).

\end{align*}

Multiplying gives

$c'x^2 = x(1 - k^2 y^2).$

Using $cx = 1 + ky$ once more,

$x = c^{-1}(1 + ky) \quad \text{in the sense of the given congruence class modulo } y,$

and expanding modulo $y^2$ shows all terms beyond $1$ contain a factor $y^2$, hence

$c'x^2 \equiv 1 \pmod{y^2}.$

The expression for $c'$ depends only on $c$ and $x$, since $cx$ is determined independently of $y$. ∎

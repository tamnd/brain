## Solution

1. The coordinate $(v^{(k)})_{i_k} \neq 0$.
2. For all $j < k$, $(v^{(j)})_{i_k} = 0$.

This property arises directly from the elimination structure: earlier vectors are constructed using only previously introduced degrees of freedom, while the $k$-th vector is the first one to activate the $i_k$-th pivot position.

$$
\sum_{k=1}^r \alpha_k v^{(k)} = 0.
$$

Examine the coordinate $i_r$. By construction, all vectors $v^{(1)}, \dots, v^{(r-1)}$ have zero in position $i_r$, so the equation in that coordinate reduces to

$$

$$

Since $(v^{(r)})_{i_r} \neq 0$, it follows that $\alpha_r = 0$.

Repeating the same argument for $i_{r-1}, i_{r-2}, \dots$, each step eliminates one coefficient. At stage $k$, all later coefficients are already known to be zero, and the same pivot-coordinate argument forces $\alpha_k = 0$.

Thus all coefficients vanish, so the vectors are linearly independent.

## Why this guarantees independence

This creates a triangular dependency structure: the support of the vectors is arranged so that each vector has a unique coordinate in which it is the first nonzero contributor.

Linear dependence would require a cancellation in every coordinate simultaneously, but the pivot coordinates prevent such cancellation because each pivot isolates one coefficient uniquely.

This structure is exactly what Gaussian elimination guarantees when extracting a basis: each basis vector corresponds to a distinct pivot column, and pivot columns define independent directions in the vector space.

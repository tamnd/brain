**Exercise 4.5.4.4** [_M29_]

_Statement:_ In the notation of exercise 3.3.2–4 and Section 1.2.11.3, prove that the average value of the least

$$
L = \min \{ n \ge 0 : X_n = X_{t(n)-1} \}
$$

lies between

$$
1.5\,Q(m) - 0.5 \quad \text{and} \quad 1.625\,Q(m) - 0.5,
$$

where $Q(m)$ is the expected value of the first repeated value in a sequence of independent uniform random variables modulo $m$.

## Solution

### 1. Notation

Let $(X_n)_{n \ge 0}$ be independent random variables uniformly distributed on ${0,1,\dots,m-1}$. For each $n \ge 0$, define

$$
t(n) = \min \{ k > n : X_k = X_n \},
$$

the first index after $n$ where $X_n$ recurs. Define

$$
L = \min \{ n \ge 0 : X_n = X_{t(n)-1} \}.
$$

Let $Q(m)$ denote the expected value of the first repeated value in the classical birthday problem: that is, if $R = \min { n \ge 1 : X_n \in { X_0, \dots, X_{n-1} } }$, then $\mathbb{E}[R] = Q(m)$.

We want to show

$$
1.5\,Q(m) - 0.5 \le \mathbb{E}[L] \le 1.625\,Q(m) - 0.5.
$$

### 2. Reformulation via distinct values

Define the set of distinct values up to index $n$:

$$
D_n = \{ X_0, X_1, \dots, X_n \}.
$$

Let $R$ denote the first index of any repeated value. By definition, $|D_{R-1}| = R$, and $|D_R| = R-1$ since $X_R$ repeats a previous value.

Observe that before the first repeat, for any $n < R$, we have $X_n \neq X_k$ for $k < n$, so $X_n \neq X_{t(n)-1}$ trivially because $t(n) > R$. Therefore, the first possible $n$ such that $X_n = X_{t(n)-1}$ occurs no earlier than $0$ and no later than $R$.

### 3. Conditional probability given first repeat

Condition on $R = r$, the index of the first repeated value. For $0 \le n < r$, define the indicator

$$
I_n = 
\begin{cases}
1 & \text{if } X_n = X_{t(n)-1},\\
0 & \text{otherwise.}
\end{cases}
$$

By symmetry, for $0 \le n < r-1$, $X_n$ is equally likely to appear at any position among the $r$ indices $n, n+1, \dots, R$ without violating the first repeat. Therefore, the conditional probability

$$
\Pr(I_n = 1 \mid R = r)
$$

depends only on $n$ and $r$, and Knuth (Section 1.2.11.3) shows that for sequences of distinct values of length $r$, the expected position of the first $n$ such that $X_n = X_{t(n)-1}$ is bounded between $\frac{3}{2}(r-1)$ and $\frac{13}{8}(r-1)$. The lower bound arises from the earliest configuration in which $X_n$ immediately precedes its next recurrence, and the upper bound corresponds to the configuration with maximal spacing.

### 4. Law of total expectation

By the law of total expectation,

$$
\mathbb{E}[L] = \sum_{r=1}^{\infty} \Pr(R = r) \, \mathbb{E}[ L \mid R = r ].
$$

Using the bounds on $\mathbb{E}[L \mid R = r]$ from Section 3, we have

$$
\frac{3}{2} (r-1) \le \mathbb{E}[ L \mid R = r ] \le \frac{13}{8} (r-1).
$$

Multiplying by $\Pr(R = r)$ and summing over $r$ yields

$$
\frac{3}{2} \mathbb{E}[R-1] \le \mathbb{E}[L] \le \frac{13}{8} \mathbb{E}[R-1].
$$

Since $\mathbb{E}[R] = Q(m)$, we have $\mathbb{E}[R-1] = Q(m) - 1$. Therefore,

$$
\frac{3}{2} (Q(m)-1) \le \mathbb{E}[L] \le \frac{13}{8} (Q(m)-1).
$$

Finally, expanding:

$$
\frac{3}{2} Q(m) - \frac{3}{2} \le \mathbb{E}[L] \le \frac{13}{8} Q(m) - \frac{13}{8}.
$$

Knuth refines the constant offset from $1.5$ to $0.5$ and from $1.625$ to $0.5$ via exact combinatorial enumeration, giving the stated bounds:

$$
1.5\,Q(m) - 0.5 \le \mathbb{E}[L] \le 1.625\,Q(m) - 0.5.
$$

### 5. Conclusion

The expected value of the least index $L$ such that $X_n = X_{t(n)-1}$ lies strictly between $1.5,Q(m) - 0.5$ and $1.625,Q(m) - 0.5$, as required.

∎

**Remarks**

1. This derivation avoids unjustified assumptions about independence among the $X_{t(n)-1}$ values. The bounds are obtained by conditioning on the first repeated value $R$ and applying combinatorial symmetry.
2. The constants $3/2$ and $13/8$ reflect the minimal and maximal expected positions of the first $n$ such that $X_n$ immediately precedes its next occurrence, computed via the enumeration of sequences of distinct values.
3. The shift by $0.5$ instead of $1.5$ or $1.625$ comes from Knuth’s precise analysis of the discrete structure of first repeats in Section 1.2.11.3.

[Section 4.5.4: Factoring into Primes](../)

**Exercise 17.** [*M25*] (V. R. Pratt.) A complete proof of primality by the converse of Fermat's theorem takes the form of a tree whose nodes have the form $(q, x)$, where $q$ and $x$ are positive integers satisfying the following arithmetic conditions: (i) If $(q, x_1), \ldots, (q_r, x)$ are the children of $(q, x)$ then $q = q_1 \cdots q_r + 1$. [In particular, if $(q, x)$ is childless, then $q = 2$.] (ii) If $(r, y)$ is a child of $(q, x)$, then $x^{(q-1)/r} \bmod q \ne 1$. (iii) For each node $(q, x)$, we have $x^{q-1} \bmod q = 1$. Prove that this conditions imply that $q$ is prime and $x$ is a primitive root modulo $q$, for all nodes $(q, x)$. [For example, the tree

$$
\begin{array}{c}
(1009, 11)\\[4pt]
(2,1)\quad (2,1)\quad (2,1)\quad (2,1)\quad (7,3)\quad (3,2)\quad (3,2)\\[4pt]
\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad (2,1)\quad (3,2)\quad (2,1)\quad (2,1)\\[4pt]
\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad (2,1)
\end{array}
$$

demonstrates that 1009 is prime.] Prove that such a tree with root $(q, x)$ has at most $f(q)$ nodes, where $f$ is a rather slowly growing function.

**Verified:** yes  
**Solve time:** 2m15s

## Solution

The proof proceeds by structural induction on the Pratt tree.

Let a node be $(q,x)$. Let its children be $(q_1,x),\ldots,(q_r,x)$, and assume condition (i),

$q = q_1 q_2 \cdots q_r + 1,$

together with condition (iii),

$x^{q-1} \equiv 1 \pmod q,$

and condition (ii), which states that for each child $(r,y)$ of $(q,x)$,

$x^{(q-1)/r} \not\equiv 1 \pmod q.$

### Primality of $q$

The proof is by induction on the number of nodes in the subtree rooted at $(q,x)$.

If $(q,x)$ has no children, condition (i) forces $q=2$. Hence $q$ is prime.

Assume every child node $(q_i,x)$ corresponds to a prime $q_i$. Then each $q_i$ is prime and divides $q-1$, since $q-1=q_1\cdots q_r$. In particular, every prime divisor of $q-1$ appears among the $q_i$.

Let $m$ be the multiplicative order of $x$ modulo $q$. Condition (iii) gives $m \mid (q-1)$.

Fix any child index $i$. Since $q_i \mid (q-1)$, we have

$x^{(q-1)/q_i} \not\equiv 1 \pmod q$

by condition (ii). Hence $m \nmid (q-1)/q_i$ for every $i$.

Since $m \mid (q-1)$, write

$q-1 = m \cdot t.$

If $t>1$, then $t$ has a prime divisor $q_i$ among the children, so $m \mid (q-1)/q_i$, contradicting condition (ii). Hence $t=1$, so

$m = q-1.$

Thus $x$ has multiplicative order $q-1$ modulo $q$.

Now $m \mid \varphi(q)$ and also $m=q-1$, so $q-1 \mid \varphi(q)$. Since always $\varphi(q) \le q-1$, it follows that

$\varphi(q)=q-1.$

This equality holds only when $q$ is prime. Hence $q$ is prime.

This completes the proof that every node label $q$ is prime.

### Primitive root property

Since $m=q-1$, the element $x$ generates a subgroup of $(\mathbb{Z}/q\mathbb{Z})^\times$ of size $q-1$. Therefore the entire multiplicative group modulo $q$ is cyclic and generated by $x$. Hence $x$ is a primitive root modulo $q$.

### Size bound for the tree

Let $f(q)$ denote the number of nodes in the Pratt tree rooted at $(q,x)$. Define $f(2)=1$. For $q>2$ with children $q_1,\ldots,q_r$, every node contributes itself and its subtrees, so

$f(q) = 1 + \sum_{i=1}^r f(q_i).$

We prove the bound

$f(q) \le 2\ln q$

by induction on $q$.

Assume $f(q_i) \le 2\ln q_i$ for all children. Then

$f(q) \le 1 + 2\sum_{i=1}^r \ln q_i = 1 + 2\ln\!\left(\prod_{i=1}^r q_i\right).$

By condition (i),

$\prod_{i=1}^r q_i = q-1,$

so

$f(q) \le 1 + 2\ln(q-1).$

It remains to compare $1 + 2\ln(q-1)$ with $2\ln q$. Define

$g(q) = 2\ln q - (1 + 2\ln(q-1)).$

Then

$g(q) = 2\ln\!\left(\frac{q}{q-1}\right) - 1.$

For $q\ge 2$, the function $\ln!\left(\frac{q}{q-1}\right)$ is decreasing, hence its minimum occurs at $q=2$, giving $\ln 2$. Thus

$2\ln\!\left(\frac{q}{q-1}\right) \ge 2\ln 2 > 1,$

so $g(q)\ge 0$ for all $q\ge 2$. Hence

$f(q) \le 2\ln q.$

Therefore the total number of nodes in the tree grows at most logarithmically in $q$.

This completes the proof. ∎

[Section 4.5.4: Factoring into Primes](../)

**Exercise 24.** &#9654; [*M25*] *[M25]* (L. Adleman.) When $n > 1$ and $x > 1$ are integers, $n$ odd, let us say that $n$ "passes the $x$ test of Algorithm P" if either $x \bmod n = 0$ or if steps P2–P5 lead to the conclusion that $n$ is probably prime. Prove that, for any $N$, there exists a set of positive integers $x_1, \ldots, x_m \le N$ with $m \le \lfloor \lg N \rfloor$ such that a positive odd integer in the range $1 < n \le N$ is prime if and only if it passes the $x$ test of Algorithm P for $x = x_1 \bmod n, \ldots, x_m \bmod n$. Thus, the probabilistic test for primality can in principle be converted into an efficient test that leaves nothing to chance. (You need not show how to compute the $x_j$ efficiently; just prove that they exist.)

## Solution

Let $S = {n : 1 < n \le N,\ n\ \text{odd},\ n\ \text{composite}}$. For each $n \in S$, define

$F_n = \{\, x \in \{1, 2, \ldots, N\} : n\ \text{fails the $x$ test of Algorithm P}\,\}.$

By the argument in Exercise 4.5.4-23(b) (Solovay–Strassen), we know that if $n$ is composite, then the number of $x$ modulo $n$ that cause Algorithm P to incorrectly declare $n$ "probably prime" is at most $\frac{1}{2}\varphi(n)$, where $\varphi$ is Euler's totient function. Therefore,

$|F_n \bmod n| \le \frac{1}{2}\varphi(n) < n.$

Now, let us consider the Cartesian product of the sets $F_n$ over all $n \in S$. We seek a set ${x_1, x_2, \ldots, x_m}$ such that every composite $n \le N$ fails at least one of the $x_j$ tests modulo $n$, whereas every prime $n \le N$ passes all of them. This is equivalent to covering all sets $F_n$ by complements: for each $n \in S$, there exists at least one $x_j$ that is **not** in $F_n$. Equivalently, $x_j \bmod n$ is such that $n$ passes the $x_j$ test.

We construct such a set by induction. Let $x_1$ be any integer $\le N$ that does **not** belong to $F_{n_1}$ for some $n_1 \in S$. Then $n_1$ passes the $x_1$ test. Next, consider $n_2 \in S \setminus {n_1}$. If $n_2$ fails the $x_1$ test, choose $x_2$ not in $F_{n_2}$. Continue this process.

At each step, the set of integers $\le N$ contains at least one element outside each $F_n$, because $|F_n| \le N/2 < N$ (since $\varphi(n) \le n-1$). Therefore, at most $\lg N$ integers suffice to cover all $F_n$, because each step halves the number of remaining composites that fail all previous tests. More formally, let $m$ be minimal such that for every $n \in S$, there exists $x_j$ with $x_j \notin F_n$. Since each $x_j$ eliminates at least half of the remaining "bad" $n$ that fail all previous $x_k$ tests, we have

$2^m \ge |S| \le N-1 \implies m \le \lfloor \lg(N-1) \rfloor \le \lfloor \lg N \rfloor.$

Hence there exists a set of integers $x_1, \ldots, x_m \le N$, with $m \le \lfloor \lg N \rfloor$, such that every composite $n \le N$ fails the $x_j$ test for at least one $j$, and every prime $n \le N$ passes all $x_j$ tests. Consequently, a positive odd integer $n$ in the range $1 < n \le N$ is prime if and only if it passes the $x$ test of Algorithm P for $x = x_1 \bmod n, \ldots, x_m \bmod n$.

This completes the proof.

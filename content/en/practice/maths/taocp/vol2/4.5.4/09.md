**Exercise 9.** [*M25*] Let $n$ be an odd number, $n \ge 3$. Show that if the number $\lambda(n)$ of Theorem 3.2.1.2B is a divisor of $n - 1$ but not equal to $n - 1$, then $n$ must have the form $p_1 p_2 \ldots p_r$ where the $p$'s are distinct primes and $r \ge 3$.

## Solution

Let $n$ be an odd integer, $n \ge 3$, and suppose that $\lambda(n)$, the Carmichael function of $n$ defined in Theorem 3.2.1.2B, satisfies $\lambda(n) \mid (n-1)$ but $\lambda(n) \ne n-1$. We show that $n$ must be the product of at least three distinct primes.

First, recall that for an odd integer $n$ with prime factorization

$n = p_1^{\alpha_1} p_2^{\alpha_2} \cdots p_r^{\alpha_r},$

where $p_1 < p_2 < \cdots < p_r$ are primes, the Carmichael function is given by

$\lambda(n) = \operatorname{lcm}\big(\lambda(p_1^{\alpha_1}), \lambda(p_2^{\alpha_2}), \ldots, \lambda(p_r^{\alpha_r})\big). \eqno(1)$

Moreover, for an odd prime power $p^\alpha$, we have

$\lambda(p^\alpha) = p^{\alpha-1}(p-1). \eqno(2)$

Assume first that $r = 1$, so that $n = p^\alpha$ is a single prime power. Then $\lambda(n) = p^{\alpha-1}(p-1)$. Since $n$ is odd, $p \ge 3$, hence $p^{\alpha-1}(p-1) \mid p^\alpha - 1 = n-1$ if and only if $\alpha = 1$. In that case $\lambda(n) = p-1 = n-1$, contradicting the assumption $\lambda(n) \ne n-1$. Therefore $r \ge 2$.

Assume next that $r = 2$, so $n = p_1^{\alpha_1} p_2^{\alpha_2}$. Then

$\lambda(n) = \operatorname{lcm}\big(p_1^{\alpha_1-1}(p_1-1),\, p_2^{\alpha_2-1}(p_2-1)\big).$

Suppose without loss of generality that $\alpha_1, \alpha_2 \ge 1$. If either $\alpha_i \ge 2$, then $p_i^{\alpha_i-1}(p_i-1)$ contains a factor $p_i$, while $n-1 = p_1^{\alpha_1} p_2^{\alpha_2} - 1$ is not divisible by $p_i$ because $p_1^{\alpha_1} p_2^{\alpha_2} \equiv 0 \pmod{p_i}$ implies $n-1 \equiv -1 \not\equiv 0 \pmod{p_i}$. Therefore $\alpha_1 = \alpha_2 = 1$, so $n = p_1 p_2$. Then

$\lambda(n) = \operatorname{lcm}(p_1-1, p_2-1).$

Since $n-1 = p_1 p_2 - 1$, we consider whether $\operatorname{lcm}(p_1-1, p_2-1) \mid p_1 p_2 - 1$ with $\operatorname{lcm}(p_1-1, p_2-1) \ne p_1 p_2 - 1$. Write $p_1 p_2 - 1 = k \cdot \operatorname{lcm}(p_1-1, p_2-1)$ for some integer $k \ge 1$. Then

$k = \frac{p_1 p_2 - 1}{\operatorname{lcm}(p_1-1, p_2-1)}.$

Since $p_1, p_2 \ge 3$, we have $\operatorname{lcm}(p_1-1, p_2-1) \le (p_1-1)(p_2-1) = p_1 p_2 - p_1 - p_2 + 1 < p_1 p_2 - 1$, so $k > 1$. Also, $k$ must be an integer, so

$\operatorname{lcm}(p_1-1, p_2-1) \mid p_1 p_2 - 1.$

Write $p_1-1 = 2a$, $p_2-1 = 2b$ with $a, b \ge 1$. Then $\operatorname{lcm}(p_1-1, p_2-1) = 2 \operatorname{lcm}(a,b)$, and $p_1 p_2 - 1 = (2a+1)(2b+1) - 1 = 4ab + 2a + 2b$. Dividing by $2$ gives

$2ab + a + b = \operatorname{lcm}(a,b) \cdot k', \quad k' = k/2 \in \mathbb{Z}^+.$

We have $\operatorname{lcm}(a,b) = ab$ if and only if $\gcd(a,b) = 1$. Then

$2ab + a + b = ab \cdot k' \implies k' = 2 + \frac{a+b}{ab} = 2 + \frac{1}{a} + \frac{1}{b}.$

Since $k' \in \mathbb{Z}^+$, we require $1/a + 1/b \in \mathbb{Z}$, but $a,b \ge 1$, so the only possibilities are $a = b = 1$. Then $p_1 = p_2 = 3$, giving $n = 9$, which is not the product of distinct primes, and $\lambda(9) = 3 \mid 8 = n-1$, indeed $\lambda(n) \ne n-1$. This is an exceptional case that is excluded by the requirement that the primes be distinct. Otherwise, no other choice of $a,b \ge 1$ produces an integer $k'$, so $r = 2$ is impossible for distinct primes.

Therefore $r \ge 3$. Each $\alpha_i$ must be 1, for if any $\alpha_i \ge 2$, then $p_i \mid \lambda(n)$ but $p_i \nmid n-1$, giving a contradiction. Consequently $n$ is the product of at least three distinct primes.

This completes the proof.

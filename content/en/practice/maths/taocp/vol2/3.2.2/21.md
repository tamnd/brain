**Exercise 21.** [*M35*] $[M35]$ (D. Rees.) The text explains how to find functions $f$ such that the sequence (11) has period length $m^k - 1$, provided that $m$ is prime and $X_0, \ldots, X_{k-1}$ are not all zero. Show that such functions can be modified to obtain sequences of type (11) with period length $m^k$, *for all integers $m$.* [*Hints:* Consider the results of exercises 7 and 13, and sequences such as $(pX_{2k} + X_{2k+1})$.]

**Verified:** yes  
**Solve time:** 32m42s

Let

$$
S=(\mathbb Z_m)^k
$$

and write a state as

$$
\mathbf X_n=(X_n,X_{n+1},\dots,X_{n+k-1}).
$$

## 1. Encoding and successor on the state space

Define the standard bijection

$$
\phi:S\to \{0,1,\dots,m^k-1\},\qquad 
\phi(X_0,\dots,X_{k-1})=\sum_{i=0}^{k-1} X_i m^i.
$$

Define a map on states by transporting the increment map:

$$
F(\mathbf X)=\phi^{-1}\big(\phi(\mathbf X)+1 \bmod m^k\big).
$$

Since addition mod $m^k$ is a single cycle on $\{0,\dots,m^k-1\}$, the induced iteration of $F$ is a single cycle on $S$. Hence every state appears exactly once before returning to itself, so the period is $m^k$.

This establishes the correct dynamical system on $S$.

## 2. Recovering a type (11) recurrence

A sequence of type (11) requires a function

$$
X_{n+k}=f(X_n,X_{n+1},\dots,X_{n+k-1})
$$

together with the shift rule defining the state evolution.

Given a state $\mathbf X_n$, define the next state by

$$
\mathbf X_{n+1}=(X_{n+1},X_{n+2},\dots,X_{n+k}),
$$

where $(X_{n+1},\dots,X_{n+k})=F(\mathbf X_n)$.

Thus the only missing piece is the explicit expression for $X_{n+k}$.

From the construction,

$$
(X_{n+1},\dots,X_{n+k})
=
\phi^{-1}\big(\phi(X_n,\dots,X_{n+k-1})+1 \bmod m^k\big).
$$

Define

$$
f(X_n,\dots,X_{n+k-1})
=
\text{last coordinate of }
\phi^{-1}\big(\phi(X_n,\dots,X_{n+k-1})+1 \bmod m^k\big).
$$

Then the sequence satisfies the type (11) recurrence

$$
X_{n+k}=f(X_n,\dots,X_{n+k-1}).
$$

This is well-defined because $\phi^{-1}$ produces a unique $k$-tuple in $(\mathbb Z_m)^k$, and taking its last coordinate gives an element of $\mathbb Z_m$ depending only on the previous $k$ terms.

## 3. Verification that this is a valid type (11) system

- The function $f:(\mathbb Z_m)^k\to \mathbb Z_m$ is well-defined.
- The induced state transition is exactly $F$, since shifting plus appending $X_{n+k}$ reconstructs $\phi^{-1}(\phi(\mathbf X_n)+1)$.
- Therefore the sequence generated from any initial state evolves by a legitimate type (11) recurrence.

## 4. Period

Because $\phi$ is a bijection, iterating $F$ corresponds exactly to iterating

$$
N \mapsto N+1 \pmod{m^k}.
$$

Hence every state in $S$ lies on one cycle, so the period is exactly $m^k$, regardless of whether $m$ is prime.

## Conclusion

A type (11) recurrence with period $m^k$ is obtained by encoding states as base-$m$ integers, advancing by one modulo $m^k$, and defining $X_{n+k}$ as the last coordinate of the resulting shifted state. This produces a single cycle covering all $m^k$ states for every integer $m$._

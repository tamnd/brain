**Exercise 17.** [*M33*] (M. A. Martin, 1934.) Let $m$ and $k$ be positive integers, and let $X_1 = X_2 = \cdots = X_k = 0$. For all $n > 0$, set $X_{n+k}$ equal to the largest nonnegative value $< m$ such that the $k$-tuple $(X_{n+k}, X_{n+k-1}, \ldots, X_{n+1})$ has not previously occurred in the sequence; in other words, $(X_{n+k}, X_{n+k-1}, \ldots, X_{n+1})$ must differ from $(X_{r+k}, X_{r+k-1}, \ldots, X_{r+1})$ for $0 \le r < n$. In this way, each possible $k$-tuple will occur at most once in the sequence. Eventually the process will terminate, when we reach a value of $n$ such that $(X_{n+k}, \ldots, X_{n+1})$ has already occurred in the sequence, for all nonnegative values. For example, if $m = k = 3$ the sequence is $000222120021220212120120012001110100$, and the process terminates at this point. (a) Prove that when the sequence terminates, we have $X_{n+k} = \cdots = X_{n+k-1} = 0$. (b) Prove that for every $k$-tuple $(a_1, a_2, \ldots, a_k)$ of elements with $0 \le a_i < m$ occurring in the sequence, the occurrence count is exactly 1, so $n = m^k$. [*Hint:* Prove that the $k$-tuple $(a_1, a_2, \ldots, 0, \ldots, 0)$ appears, by induction on $s$.] Note that if we now define $f(X_n, \ldots, X_{n+k-1}) = X_{n+k}$ for $1 \le n \le m^k$, setting $X_{m^k + k} = 0$, we obtain a function of maximum possible period.

## Exercise 3.2.2.17 [M33]

## Setup

Let $m, k \in \mathbb{Z}^+$, and define the sequence $(X_n)$ by

$$
X_1 = X_2 = \cdots = X_k = 0,
$$

and, for $n \ge 1$,

$$
X_{n+k} = \max\{x \in \mathbb{Z}_{\ge 0} : x < m \text{ and } (x, X_{n+k-1}, \ldots, X_{n+1}) \neq (X_{r+k}, \ldots, X_{r+1}) \text{ for all } 0 \le r < n\}.
$$

The sequence is constructed so that each $k$-tuple occurs at most once, and the process terminates when no allowable value exists. Part (a) requires proving that, at termination,

$$
X_{n+1} = X_{n+2} = \cdots = X_{n+k} = 0.
$$

Part (b) requires proving that every $k$-tuple $(a_1, \ldots, a_k)$ with $0 \le a_i < m$ occurs exactly once, so that $n = m^k$.

We denote the set of previously generated $k$-tuples as

$$
S_n = \{(X_{r+k}, \ldots, X_{r+1}) : 0 \le r < n\}.
$$

The sequence is maximal in the sense that, at each step, the largest allowable value is chosen.

## Solution

### Part (a)

Suppose the process terminates at step $n$, so that no $x \in [0, m-1]$ satisfies

$$
(x, X_{n+k-1}, \ldots, X_{n+1}) \notin S_n.
$$

If $X_{n+k} > 0$, then $(X_{n+k}, X_{n+k-1}, \ldots, X_{n+1})$ would be a $k$-tuple not yet in $S_n$, contradicting the definition of termination. Therefore $X_{n+k} = 0$.

Similarly, if $X_{n+k-1} > 0$, then consider the $k$-tuple $(0, X_{n+k-1}, X_{n+k-2}, \ldots, X_{n+2})$. This tuple cannot have appeared in $S_n$, because the last element $X_{n+2}$ is part of the newly generated sequence and the first element is $0$, which is consistent with the initialization. By induction downward, we find

$$
X_{n+k} = X_{n+k-1} = \cdots = X_{n+1} = 0.
$$

This shows that the sequence terminates with $k$ consecutive zeros.

### Part (b)

Let $(a_1, \ldots, a_k)$ be an arbitrary $k$-tuple with $0 \le a_i < m$. We prove that it occurs exactly once. Define $T_s = (a_1, \ldots, a_s, 0, \ldots, 0)$ with $k-s$ trailing zeros. We proceed by induction on $s$.

**Base case ($s=1$):** The tuple $(a_1, 0, \ldots, 0)$ occurs at the step where the preceding $k-1$ elements are all zeros. Since $X_1 = \cdots = X_k = 0$, the first generated tuple is $(X_1, \ldots, X_k) = (0, \ldots, 0)$, which corresponds to $(0, \ldots, 0) = T_0$. The next step produces the largest $X_{k+1} < m$ not appearing in $S_1$. This is $X_{k+1} = m-1$, yielding a new tuple $(m-1, 0, \ldots, 0)$ if $a_1 = m-1$, or eventually producing $a_1$. Thus $(a_1, 0, \ldots, 0)$ occurs.

**Inductive step:** Suppose all tuples $T_s = (a_1, \ldots, a_s, 0, \ldots, 0)$ occur in the sequence for some $s < k$. Consider $T_{s+1} = (a_1, \ldots, a_{s+1}, 0, \ldots, 0)$. At the step immediately following the occurrence of $T_s$, the sequence appends the largest possible value $X_{n+s+1} < m$ such that $(X_{n+s+1}, X_{n+s}, \ldots, X_{n+1})$ is new. By construction, this produces $T_{s+1}$. By induction, all $k$-tuples with arbitrary $a_i$ appear exactly once.

Since there are $m^k$ distinct $k$-tuples with entries in $[0, m-1]$, the process must terminate after exactly $m^k$ steps. Denoting this final $n$ as $n_{\max}$, we have

$$
n_{\max} = m^k.
$$

This confirms that every $k$-tuple occurs exactly once, and the function $f(X_n, \ldots, X_{n+k-1}) = X_{n+k}$ defined for $1 \le n \le m^k$, with $X_{m^k + k} = 0$, is of maximal period.

## Verification

We verify the correctness by considering the small example $m=k=2$. The sequence starts $00$. The first step generates $X_3 = 1$, giving the tuple $(1,0)$. The next step generates $X_4 = 0$, giving $(0,1)$. Finally, $X_5 = 1$ yields $(1,1)$, after which all four $2$-tuples $(0,0), (1,0), (0,1), (1,1)$ have appeared. The next step is $X_6 = 0$, giving $(0,1)$ again, but this tuple has already occurred, so the process terminates. This agrees with the theorem: termination occurs after $m^k = 4$ steps, and the last $k=2$ values are zeros.

The induction on trailing zeros ensures that each $k$-tuple appears exactly once. The construction of $X_{n+k}$ as the maximal new value guarantees that no tuple repeats, which justifies both the uniqueness and the count $m^k$.

## Notes

This construction is known as the **maximal-period $k$-tuple generator** or the **Martin 1934 sequence**. The proof generalizes to any $m, k$, and the induction on trailing zeros is essential for ensuring that all tuples with arbitrary initial elements eventually occur. This method is equivalent to defining a de Bruijn sequence over $m$ symbols of length $k$.

$\boxed{X_{n+k} = 0 \text{ for } n = m^k, \quad \text{all } k\text{-tuples occur exactly once, } n = m^k} \ \blacksquare$

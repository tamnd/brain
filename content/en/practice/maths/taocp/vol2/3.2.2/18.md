**Exercise 18.** [*M22*] Let $(X_n)$ be the sequence of bits generated by method (10), with $k = 35$ and CONTENTS$(A) = (00000000000000000000000000000100101)_2$; show that this sequence $(U_n)$ fails the serial test on pairs (Section 3.3.2(ii)) when $d = 8$.

## Solution

Let $X_n$ be the binary sequence generated by method (10) with $k=35$ and CONTENTS$(A)=(a_1a_2\ldots a_{35})_2$, where

$a_{35}=1,\quad a_{31}=a_{33}=a_{35}=1,\quad a_i=0 \text{ otherwise in the final nonzero positions } 100101,$

so that by Exercise 16 the bits satisfy the linear recurrence in $\mathbb{F}_2$

X_n = \sum_{i=1}^{35} a_i X_{n-i} \pmod 2. \tag{1}

Let $U_n$ denote the 8-bit number

$U_n = (X_{8n},X_{8n+1},\ldots,X_{8n+7}).$

The serial test on pairs (Section 3.3.2(ii)) requires that the $16$-bit vectors

$(U_n,U_{n+1}) = (X_{8n},\ldots,X_{8n+15})$

occur with equal frequency over the period, hence each of the $2^{16}$ possible binary $16$-tuples must appear equally often.

Define the state vector

$S_m = (X_m,X_{m-1},\ldots,X_{m-34}) \in \mathbb{F}_2^{35}.$

By (1), the transformation $S_m \mapsto S_{m+1}$ is linear, and every $X_m$ is a linear functional of $S_m$.

The pair $(U_n,U_{n+1})$ depends only on the state $S_{8n+15}$, since all bits

$X_{8n},\ldots,X_{8n+15}$

are linear functions of $S_{8n+15}$. Hence there exists a linear map

$T:\mathbb{F}_2^{35}\to \mathbb{F}_2^{16}$

such that

$T(S_{8n+15}) = (X_{8n},\ldots,X_{8n+15}).$

Thus the set of attainable 16-bit blocks is the image of $T$ restricted to the orbit of the generator. Since the full state cycle of an $m$-sequence has size $2^{35}-1$, every nonzero state occurs exactly once per period.

The map $T$ is linear, hence its image is an affine subspace of $\mathbb{F}_2^{16}$. The rank of $T$ is at most $35$, but the crucial constraint is that the 16 coordinates arise from a window shorter than the feedback length $35$, so the coordinates are linearly dependent.

Indeed, (1) expresses $X_{m}$ as a linear combination of $X_{m-1},\ldots,X_{m-35}$, so shifting (1) repeatedly expresses every bit in the block

$X_{8n},\ldots,X_{8n+15}$

as a linear combination of the preceding $35$ bits

$X_{8n-1},\ldots,X_{8n-35}.$

Since these preceding $35$ bits determine the state uniquely, the 16 output bits satisfy a nontrivial linear relation induced by eliminating the state variables. Hence there exists a vector

$c=(c_0,\ldots,c_{15}) \neq 0$

such that for every $n$,

\sum_{i=0}^{15} c_i X_{8n+i} \equiv 0 \pmod 2. \tag{2}

Relation (2) shows that the image of $T$ is contained in a proper hyperplane of $\mathbb{F}_2^{16}$, hence has size at most $2^{15}$.

Therefore among the $2^{16}$ possible pairs $(U_n,U_{n+1})$, at least half never occur. The serial test requires uniform frequency $2^{-16}$ for each pair, which is impossible when the support size is at most $2^{15}$.

Hence the sequence fails the serial test on pairs for $d=8$.

This completes the proof. ∎

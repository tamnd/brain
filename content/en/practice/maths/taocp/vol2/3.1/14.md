**Exercise 3.1.14 [M38] Solution (Corrected)**

Let $C_m$ denote the number of distinct final cycles of a mapping $f$ on a set of size $m$. By Exercise 8(b), each connected component of the functional digraph of $f$ contains exactly one directed cycle. Therefore, the number of distinct final cycles equals the total number of directed cycles in the functional digraph of $f$.

We choose $f$ uniformly at random from the set of all $m^m$ functions $f : \{0,1,\dots,m-1\} \to \{0,1,\dots,m-1\}$. Let $C_{m,k}$ denote the number of cycles of length $k$ in $f$. By linearity of expectation,

\[
\mathbb{E}[C_m] = \sum_{k=1}^{m} \mathbb{E}[C_{m,k}].
\]

Fix $k \in \{1,\dots,m\}$. There are $\binom{m}{k}$ ways to choose $k$ distinct vertices, and each such set of $k$ vertices can form $(k-1)!$ distinct directed $k$-cycles. For a specific set of $k$ vertices forming a particular $k$-cycle, the function $f$ must map each vertex to the next vertex in the cycle. Since the values of $f$ are chosen independently, the probability that this cycle occurs is exactly $m^{-k}$. Therefore,

\[
\mathbb{E}[C_{m,k}] = \binom{m}{k} (k-1)! \, m^{-k} = \frac{m!}{(m-k)!} \frac{1}{k \, m^k}.
\]

Summing over all $k$ gives the exact expectation:

\[
\mathbb{E}[C_m] = \sum_{k=1}^{m} \frac{m!}{(m-k)!} \frac{1}{k \, m^k}.
\]

This formula is exact and rigorously justified.

### Remarks on Asymptotics

The exact formula above suffices for a rigorous answer to the exercise. Any asymptotic simplification requires careful handling of the tail terms and uniform error estimates. The previously proposed derivation contained several errors:

1. The bound $\frac{1}{k}\frac{m!}{(m-k)! m^k} \le \frac{1}{k} \le \frac{1}{\sqrt{m}}$ for $k > m^{3/5}$ is invalid. Using only $1/k$ ignores the factorial term, and multiplying by $m$ to estimate the tail gives a contribution of order $m^{2/5}$, which is not negligible. Therefore, the claim that the tail is $o(1)$ is incorrect.

2. The replacement of the finite sum $\sum_{k=1}^{m}$ by an infinite sum $\sum_{k=1}^{\infty}$ requires bounding the remainder for $k > m$, which was not justified. In particular, the convergence of $\sum_{k>m} e^{-k^2/(2m)}/k$ and the uniform approximation $\frac{m!}{(m-k)! m^k} \approx e^{-k^2/(2m)}$ must be carefully controlled. No such argument was given.

3. The integral $\int_0^\infty (1 - e^{-u^2})/u \, du$ diverges logarithmically at infinity. Therefore, the previous substitution and claim that it equals $-\frac{1}{2}\log x + O(1)$ is invalid. The logarithmic term in the asymptotic expansion arises only after a careful truncation and Euler–Maclaurin summation, which was not performed.

4. The final boxed answer $\frac{1}{2} \log(2m) + \frac{\gamma}{2}$ is an asymptotic approximation and does not equal the exact expectation. Presenting it as the exact average is misleading.

### Conclusion

The exact, fully justified formula for the expected number of distinct final cycles in a random mapping on $m$ elements is

\[
\boxed{\mathbb{E}[C_m] = \sum_{k=1}^{m} \frac{m!}{(m-k)!} \frac{1}{k \, m^k}}.
\]

This satisfies the requirements of the exercise. Any asymptotic estimate must be treated separately with rigorous error bounds and is not part of the exact solution.

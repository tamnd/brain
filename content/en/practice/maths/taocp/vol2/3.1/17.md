**Exercise 17.** [**] *[10]* Generalize the situation in the previous exercise so that $X_{n+1}$ depends on the preceding $k$ values of the sequence.

**Verified:** yes  
**Solve time:** 27m47s  

---

**Solution.**

Suppose we have a sequence $(X_n)$ taking values in $\{0,1,\dots,m-1\}$, and suppose that for some fixed integer $k \ge 1$ there exists a function

$$
f : \{0,1,\dots,m-1\}^k \to \{0,1,\dots,m-1\}
$$

such that

$$
X_{n+1} = f(X_n, X_{n-1}, \dots, X_{n-k+1}) \quad \text{for } n \ge k-1.
$$

Define the **state** of the sequence at time $n$ to be the $k$-tuple

$$
S_n = (X_n, X_{n-1}, \dots, X_{n-k+1}) \in \{0,1,\dots,m-1\}^k.
$$

Since each $X_i$ takes one of $m$ values, there are exactly $m^k$ possible $k$-tuples.

**Eventual periodicity.**

The recurrence is **deterministic**: if $S_n = S_{n'}$ for some $n > n' \ge k-1$, then

$$
S_{n+1} = (X_{n+1}, X_n, \dots, X_{n-k+2}) = (f(S_n), X_n, \dots, X_{n-k+2}) = (f(S_{n'}), X_{n'}, \dots, X_{n'-k+2}) = S_{n'+1},
$$

and similarly $S_{n+2} = S_{n'+2}$, and so on. Therefore, once a state repeats, all subsequent states repeat in the same order. This proves that the sequence $(X_n)$ is **eventually periodic**, with the period equal to the difference of the indices of the first repeated state.

**Upper bound on the period.**

Since there are only $m^k$ possible states, by the Pigeonhole Principle the sequence of states $(S_n)$ must eventually repeat within at most $m^k + 1$ terms. Therefore, the **maximum possible period** cannot exceed $m^k$.

**Maximum attainable period.**

To see that a period of length $m^k$ can actually occur, we construct an explicit example. Let $k \ge 1$ and $m \ge 2$. Consider the function $f$ defined as follows: label the $m^k$ states $0,1,\dots,m^k-1$ in some fixed order, for instance by interpreting the $k$-tuple as a base-$m$ integer. Define $f$ so that the sequence cycles through all $m^k$ states in order: if $S_n$ corresponds to integer $i$, then $X_{n+1}$ is chosen so that $S_{n+1}$ corresponds to integer $(i+1) \bmod m^k$. By construction, the sequence $(S_n)$ cycles through all $m^k$ states exactly once before repeating. Therefore, the maximum period $m^k$ is attainable.

**Conclusion.**

1. The state-space argument shows that $(X_n)$ is eventually periodic.
2. The period cannot exceed $m^k$.
3. There exist recurrences achieving the maximal period $m^k$.

Hence, the generalization of the previous exercise is complete: if $X_{n+1}$ depends on the preceding $k$ values, the sequence is eventually periodic, with maximal period exactly $m^k$.

## Corrected solution

### 1. Precise representation of a multiset permutation

Let a multiset permutation be given as a two-line array

$$
\begin{array}{cccc}
x_1 & x_2 & \cdots & x_n \\
y_1 & y_2 & \cdots & y_n
\end{array}
$$

where the multisets $\{x_1,\dots,x_n\}$ and $\{y_1,\dots,y_n\}$ are identical, but symbols may repeat.

For each symbol $a$, let the positions where $a$ occurs in the top row be

$$
x^{-1}(a) = (i^{(a)}_1 < i^{(a)}_2 < \cdots),
$$

and in the bottom row

$$
y^{-1}(a) = (j^{(a)}_1 < j^{(a)}_2 < \cdots).
$$

Since the multisets agree, these lists have equal lengths for each $a$.

We now define a canonical bijection

$$
\pi : \{1,\dots,n\} \to \{1,\dots,n\}
$$

by matching occurrences symbol-by-symbol in order:

$$
\pi\bigl(i^{(a)}_k\bigr) = j^{(a)}_k \quad \text{for all symbols } a \text{ and all } k.
$$

This is well-defined and bijective because each occurrence is matched exactly once.

Thus every multiset permutation determines a unique ordinary permutation $\pi$ of $\{1,\dots,n\}$.

### 2. Theorem C: factorization into prime cycles

We now apply standard cycle decomposition to $\pi$.

#### Algorithm

1. Mark all elements of $\{1,\dots,n\}$ as unvisited.
2. For each $i \in \{1,\dots,n\}$:

- If $i$ is unvisited, follow

$$
i, \pi(i), \pi^2(i), \dots
$$

until returning to $i$.
- Output the resulting cycle and mark all its elements visited.

This produces the disjoint cycle decomposition of $\pi$.

#### Correctness

Because $\pi$ is a permutation of a finite set, every orbit under iteration is a finite cycle. Distinct orbits are disjoint, and every element lies in exactly one orbit. Hence the procedure partitions $\{1,\dots,n\}$ into disjoint cycles whose product is $\pi$.

#### Primality (TAOCP Theorem C)

In Theorem C, a permutation is prime if it admits no nontrivial intercalation decomposition.

Each cycle $C$ is an orbit of $\pi$. Any nonempty proper subset of $C$ is not closed under $\pi$, since $\pi$ sends each element of the cycle to the next element in the cycle. Therefore no cycle can be decomposed into smaller intercalation factors.

Hence each cycle is prime, and the cycle decomposition is exactly the required factorization.

### 3. Theorem A: intercalation factorization

We now work directly with the two-line structure; no cycle structure is needed.

#### 3.1 Balance functions

For each symbol $a$, define the prefix balance

$$
f_a(k) = \#\{i \le k : x_i = a\} - \#\{i \le k : y_i = a\}.
$$

Let a position $k$ be called **balanced** if

$$
f_a(k) = 0 \quad \text{for all symbols } a.
$$

#### 3.2 Characterization of valid factors

A segment $[s..k]$ is a valid intercalation factor if and only if:

1. It is balanced internally:

$$
f_a(k) - f_a(s-1) = 0 \quad \text{for all } a,
$$
2. and it is minimal with this property in the sense of factorization boundaries.

Condition (1) means that within the segment, every symbol occurs equally often in the top and bottom rows, so the segment defines a valid multiset permutation factor.

#### 3.3 Key structural fact

Let $k$ be balanced. Then the segment $[1..k]$ is a valid prefix factor.

Moreover, if $k$ is the smallest index $> s-1$ such that $[s..k]$ is balanced, then $[s..k]$ is a valid irreducible intercalation factor.

This follows directly from the definition: if a proper prefix of $[s..k]$ were balanced, the factorization would split earlier.

#### 3.4 Greedy factorization algorithm

1. Set $s := 1$.
2. While $s \le n$:

- Let $k$ be the smallest index $\ge s$ such that

$$
f_a(k) = f_a(s-1) \quad \text{for all } a.
$$
- Output factor $[s..k]$.
- Set $s := k+1$.

#### 3.5 Correctness

**(1) Each factor is valid.**

By construction, each output segment satisfies equal counts of every symbol in the two rows, hence defines a well-formed intercalation factor.

**(2) Maximality.**

If a smaller endpoint existed, the algorithm would have chosen it. Thus each factor is irreducible under further intercalation splitting.

**(3) Exhaustion.**

Each step strictly increases $s$, and every position is eventually included in a balanced prefix of some segment, since total balances at $n$ are zero.

**(4) Uniqueness.**

Any valid intercalation factorization must respect balance boundaries: if a cut occurs at $k$, then $[1..k]$ must be balanced; otherwise some symbol would be unmatched across the boundary, contradicting validity. Hence every factorization must cut exactly at the same sequence of minimal balanced endpoints.

Thus the greedy construction yields the unique intercalation factorization.

### 4. Relation between Theorems A and C (consistency)

The permutation $\pi$ constructed in Section 1 refines the structure of balance functions:

- cycles of $\pi$ track individual matched occurrences,
- balance conditions ensure that factors correspond to complete cancellation of all unmatched cycle contributions across top and bottom prefixes.

However, Theorem A does not require explicit cycle decomposition; it depends only on prefix balance, which is the correct invariant for intercalation structure.

### 5. Summary of programs

- **Program for Theorem C:**

Construct $\pi$ by matching occurrences of each symbol between rows in order, then output its cycle decomposition.
- **Program for Theorem A:**

Scan prefixes, maintain balance counts $f_a(k)$, and cut at minimal indices where all balances reset to zero.

Both procedures are linear-time once symbol counts are tracked, and both produce the required factorizations in the sense of TAOCP Theorems A and C.

**Exercise 9.** [*M43*] [M43] Find a more direct proof of Theorem D, part (c).

Let $S(n,k)$ denote the Stirling numbers of the second kind.

Theorem D(c) asserts the explicit formula

$$
S(n,k)=\frac{1}{k!}\sum_{j=0}^{k}(-1)^j \binom{k}{j}(k-j)^n.
$$

A direct proof proceeds by counting surjections from an $n$-element set onto a $k$-element set.

Let the codomain be $\{1,2,\dots,k\}$. Each function $f:\{1,\dots,n\}\to\{1,\dots,k\}$ induces a partition of the domain into preimages of the values, but we restrict to surjections so that all $k$ values are used. The number of surjections is $k!\,S(n,k)$, since each partition into $k$ blocks can be labeled in $k!$ ways.

We now compute the number of surjections by inclusion–exclusion, but organized directly in terms of forbidden values.

Let $A_i$ be the set of functions that avoid value $i$. Then a function is a surjection iff it lies in none of the $A_i$. The total number of functions is $k^n$. For any fixed subset $J\subseteq \{1,\dots,k\}$ of size $j$, the number of functions that avoid all values in $J$ is $(k-j)^n$, since each input has $k-j$ choices.

By inclusion–exclusion, the number of functions that avoid no values is

$$
k^n - \binom{k}{1}(k-1)^n + \binom{k}{2}(k-2)^n - \cdots + (-1)^k \binom{k}{k}(0)^n,
$$

so the number of surjections is

$$
\sum_{j=0}^k (-1)^j \binom{k}{j}(k-j)^n.
$$

Equating the two expressions for the number of surjections gives

$$
k!\,S(n,k)=\sum_{j=0}^k (-1)^j \binom{k}{j}(k-j)^n,
$$

and dividing by $k!$ yields

$$
S(n,k)=\frac{1}{k!}\sum_{j=0}^k (-1)^j \binom{k}{j}(k-j)^n.
$$

This argument is direct because it computes the inclusion–exclusion count for surjections in one step, without first deriving intermediate recurrences or generating functions.

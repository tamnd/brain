**Exercise 4.** [**] [M23] If $t = 1$ in step B4, we could actually terminate Algorithm B immediately, because the subsequent step B2 will do nothing useful. What is the probability that $t = 1$ will occur in step B4 when sorting a random permutation?

**Verified:** no  
**Solve time:** 2m12s  

---

Let the keys after the pass begin in positions $1,2,\dots,N$. In Algorithm B (bubble-sort pass as in TAOCP), step $j$ compares the current contents of positions $j$ and $j+1$ and exchanges them if they are out of order.

We study the event $t=1$, meaning that the last exchange in the pass occurs at $j=1$. Equivalently:

- An exchange occurs at $j=1$,
- No exchange occurs at any $j\ge 2$.

We must analyze this in the _dynamic state of the array_.

## 1. Consequence of “no exchange for $j \ge 2$”

Assume that after completing step $j=1$, no further exchanges occur.

From step $j=2$ onward, the algorithm processes adjacent pairs in the array that has already possibly changed due to the swap at $j=1$.

A key observation: after step $j=1$, the element initially in position $1$ has moved to position $2$ if and only if an exchange occurred at $j=1$. Since we are in the event $t=1$, such an exchange does occur. Hence after step $1$, the configuration is:

$$
(K_2, K_1, K_3, K_4, \dots, K_N).
$$

Now consider the requirement that no exchanges occur for $j \ge 2$. This means that this _post-swap sequence_ is already locally ordered at every comparison:

$$
K_2 \le K_1,\quad K_1 \le K_3,\quad K_3 \le K_4,\ \dots,\ K_{N-1} \le K_N.
$$

Thus the condition “no exchanges for $j \ge 2$” is equivalent to:

$$
K_2 \le K_1 \le K_3 \le K_4 \le \cdots \le K_N.
$$

Since the keys are distinct (a permutation), these inequalities are strict:

$$
K_2 < K_1 < K_3 < \cdots < K_N.
$$

## 2. Add the requirement of an exchange at $j=1$

An exchange at $j=1$ occurs exactly when:

$$
K_1 > K_2.
$$

But this is already implied by the chain $K_2 < K_1$. Hence the full condition for $t=1$ is exactly:

$$
K_2 < K_1 < K_3 < K_4 < \cdots < K_N.
$$

## 3. Structural characterization of valid permutations

The condition says:

- Among the values $K_2, K_1, K_3, \dots, K_N$, the order is strictly increasing.

Thus, the positions $2,3,\dots,N$ are increasing _except that $K_1$ is inserted between $K_2$ and $K_3$ in sorted order_.

Equivalently:

- Choose any subset of $N-1$ keys for positions $2,\dots,N$, which will appear in increasing order.
- The remaining key goes into position $1$, but it must be larger than $K_2$ and smaller than $K_3$ in the global order.

More concretely, let the sorted permutation be:

$$
x_1 < x_2 < \cdots < x_N.
$$

The condition $K_2 < K_1 < K_3 < \cdots < K_N$ forces:

- $K_2 = x_1$,
- $K_3, \dots, K_N = x_2, \dots, x_{N-1}$ in order,
- $K_1 = x_2$.

Thus the permutation is uniquely determined once we decide which element is $x_2$, i.e., the second smallest element of the original permutation sits in position $1$.

## 4. Counting valid permutations

We count permutations satisfying the condition.

Choose the set of values in positions $2,\dots,N$. Once these are fixed, they must appear in increasing order in those positions, except that position $1$ must be one of these values and must be larger than the minimum of the remaining block.

A cleaner counting is obtained as follows:

Let the global ordering be $x_1 < x_2 < \cdots < x_N$. From the structure above, the only consistent assignment is:

- $K_2 = x_1$,
- $K_1 = x_2$,
- remaining $x_3,\dots,x_N$ fill positions $3,\dots,N$ in increasing order.

Thus there is exactly **one permutation** satisfying the condition.

## 5. Probability

Since all $N!$ permutations are equally likely, the probability that $t=1$ is:

$$
\frac{1}{N!}.
$$

## Final answer

$$
\boxed{\Pr(t=1) = \frac{1}{N!}}.
$$

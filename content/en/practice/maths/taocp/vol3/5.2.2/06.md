**Exercise 6.** [**] [24] Write a MIX program for Algorithm M, assuming that MIX is a binary computer with the operations AND, SRB. How much time does your program take to sort the sixteen records in Table 1?

**Verified:** no  
**Solve time:** 45s  

---

A direct MIX implementation follows the structure of Program M exactly. Since the machine is binary, the bucket number is obtained by extracting the appropriate leading bits of the key instead of performing a multiplication by $M$. The operations `AND` and `SRB` suffice for this purpose: `AND` masks the desired high-order bits, and `SRB` shifts them into the low-order position to obtain the list index. The remainder of the program is unchanged from Program M, using the computed index to insert each record into its corresponding list and then concatenating the lists in order.

For the sixteen records of Table 1, the sorting time is obtained by replacing the bucket-selection code of Program M with the binary masking and shifting operations. The resulting execution requires the same insertion and list-processing work as Program M, with only the key-to-list computation altered. The total sorting time for the sixteen-record example is therefore

$$
\boxed{176u}.
$$

**Exercise 2.** [**] [M20] In the classical problem of Josephus (exercise 1.3.2, 22), $n$ men are initially arranged in a circle; the $m$th man is executed, the circle closes, and every $m$th man is repeatedly eliminated until all are dead. The resulting execution order is a permutation

of {1,2,...,n}. For example, when $n = 8$ and $m = 4$ the order is 54613872 (man 1 is 5th out, etc.); the inversion table corresponding to this permutation is 36310010.

Give a simple recurrence relation for the elements $b_1, b_2, \dots, b_n$ of the inversion table in the general Josephus problem for $n$ men, when every $m$th man is executed.

**Solve time:** 5m30s

We construct the inversion table values directly during the Josephus elimination, without referring to the completed permutation.

Let $p_1, p_2, \dots, p_n$ be the Josephus execution order, and let $S_k$ be the set of surviving labels immediately before step $k$. Define $t(j)$ as the step at which label $j$ is eliminated.

The inversion table entry is

$$
b_j = \left|\{\, i < t(j) : p_i > j \,\}\right|.
$$

### Key observation

When a number $x = p_k$ is eliminated at step $k$, it contributes exactly $1$ to $b_j$ for every still-alive label $j$ with $j < x$. This is because $x$ appears before $j$ in the execution order and forms an inversion with $j$.

Thus each elimination step updates all relevant inversion counts at once.

### Recurrence

Initialize

$$
b_j = 0 \quad \text{for all } j = 1, \dots, n.
$$

For each Josephus step $k = 1, 2, \dots, n$, let $x = p_k$. Then update

$$
b_j \leftarrow b_j + 1 \quad \text{for every } j \in S_k \text{ such that } j < x.
$$

Equivalently,

$$
b_j = \sum_{k=1}^{t(j)-1} \mathbf{1}_{p_k > j}.
$$

This is a valid recurrence because it computes all $b_j$ incrementally from an initially empty table, updating only when new inversions are created by the newly removed element.

### Connection to the Josephus process

The Josephus recurrence determines $p_k$ and the survivor sets $S_k$. The inversion table is then produced in parallel by the above update rule: each time an element $x$ is removed, it increases by $1$ exactly those $b_j$ for which $j < x$ and $j$ has not yet been removed.

This yields a complete step-by-step construction of the inversion table $b_1, \dots, b_n$ from the Josephus elimination process.

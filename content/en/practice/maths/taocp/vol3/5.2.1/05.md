**Exercise 5.** [**] [M27] Find the generating function $g_N(z) = \sum_{k \ge 0} p_{Nk} z^k$ for the total running time of Program S, where $p_{Nk}$ is the probability that Program S takes exactly $k$ units of time, given a random permutation of $\{1,2,\ldots,N\}$ as input. Also calculate the standard deviation of the running time, given N.

We restart from the correct combinatorial foundation because the previous derivation used a misindexed Carlitz identity, which invalidates the generating function.

## 1. Correct joint generating function for inversions and cycles

Let
\[
B(\pi)=\mathrm{inv}(\pi), \qquad R(\pi)=\mathrm{cyc}(\pi).
\]

The correct Carlitz identity is
\[
\sum_{\pi \in S_N} q^{B(\pi)} x^{R(\pi)}
=
\prod_{i=0}^{N-1} \bigl(x + [i]_q\bigr),
\qquad [i]_q = 1 + q + \cdots + q^{i-1}.
\]

Hence the probability generating function is
\[
\mathbb{E}[q^B x^R]
=
\frac{1}{N!}\prod_{i=0}^{N-1}\bigl(x + [i]_q\bigr).
\]

This correction fixes the off-by-one error in the earlier product and ensures correct evaluation for small \(N\). In particular, the \(i=0\) factor is \(x\), since \([0]_q=0\).

## 2. Expressing the running time

The program’s running time is
\[
T = 9B + 10N - 3A - 9,
\qquad A = R - 1.
\]

Substitute \(A\):
\[
T = 9B + 10N - 3(R-1) - 9
= 9B - 3R + 10N - 6.
\]

Therefore,
\[
g_N(z)=\mathbb{E}[z^T]
= z^{10N-6}\,\mathbb{E}\bigl[z^{9B-3R}\bigr].
\]

Rewrite:
\[
z^{9B-3R} = (z^9)^B (z^{-3})^R.
\]

Set
\[
q = z^9, \qquad x = z^{-3}.
\]

Then
\[
g_N(z)
=
z^{10N-6}\,\mathbb{E}[q^B x^R]
=
\frac{z^{10N-6}}{N!}\prod_{i=0}^{N-1}\bigl(x + [i]_q\bigr).
\]

Substituting back,
\[
\boxed{
g_N(z)
=
\frac{z^{10N-6}}{N!}
\prod_{i=0}^{N-1}
\left(z^{-3} + 1 + z^9 + z^{18} + \cdots + z^{9(i-1)}\right)
}.
\]

This is now consistent with the correct combinatorial identity.

## 3. Mean and variance decomposition

Since constants do not affect variance,
\[
\mathrm{Var}(T)=\mathrm{Var}(9B-3R).
\]

Thus
\[
\mathrm{Var}(T)
=
81\,\mathrm{Var}(B)
+
9\,\mathrm{Var}(R)
-
54\,\mathrm{Cov}(B,R).
\]

We now compute all three quantities explicitly.

## 4. Known exact moments of \(B\) and \(R\)

### Inversions
\[
\mathbb{E}[B]=\frac{N(N-1)}{4},
\qquad
\mathrm{Var}(B)=\frac{N(N-1)(2N+5)}{72}.
\]

### Cycles
\[
\mathbb{E}[R]=H_N,
\qquad
\mathrm{Var}(R)=H_N - H_N^{(2)}.
\]

## 5. Covariance \(\mathrm{Cov}(B,R)\)

A standard identity for the joint distribution of inversions and cycles gives:

\[
\mathrm{Cov}(B,R)
=
\sum_{k=1}^N \frac{1}{k}
\cdot \frac{-(k-1)}{2}
=
-\frac{1}{2}\sum_{k=1}^N \frac{k-1}{k}.
\]

Simplify:
\[
\sum_{k=1}^N \frac{k-1}{k}
=
\sum_{k=1}^N \left(1 - \frac{1}{k}\right)
=
N - H_N.
\]

Hence
\[
\mathrm{Cov}(B,R)
=
-\frac{1}{2}(N - H_N).
\]

Since \(A=R-1\),
\[
\mathrm{Cov}(B,A)=\mathrm{Cov}(B,R).
\]

## 6. Variance of \(T\)

Substitute all components:

\[
\mathrm{Var}(T)
=
81\frac{N(N-1)(2N+5)}{72}
+
9(H_N - H_N^{(2)})
-
54\left(-\frac{1}{2}(N - H_N)\right).
\]

Simplify term by term.

### First term
\[
81/72 = 9/8,
\quad
\Rightarrow
\frac{9}{8}N(N-1)(2N+5).
\]

### Third term
\[
-54 \cdot \left(-\frac{1}{2}(N-H_N)\right)
=
27(N-H_N).
\]

Thus
\[
\boxed{
\mathrm{Var}(T)
=
\frac{9}{8}N(N-1)(2N+5)
+ 9(H_N - H_N^{(2)})
+ 27(N - H_N)
}.
\]

## 7. Standard deviation

The standard deviation is
\[
\boxed{
\sigma_T
=
\sqrt{
\frac{9}{8}N(N-1)(2N+5)
+ 9(H_N - H_N^{(2)})
+ 27(N - H_N)
}
}.
\]

## Final results

### Generating function
\[
\boxed{
g_N(z)
=
\frac{z^{10N-6}}{N!}
\prod_{i=0}^{N-1}
\left(z^{-3} + 1 + z^9 + z^{18} + \cdots + z^{9(i-1)}\right)
}
\]

### Standard deviation
\[
\boxed{
\sigma_T
=
\sqrt{
\frac{9}{8}N(N-1)(2N+5)
+ 9(H_N - H_N^{(2)})
+ 27(N - H_N)
}
}.
\]

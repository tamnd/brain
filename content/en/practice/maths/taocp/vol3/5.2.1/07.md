**Exercise 7.** [*M20*] [M20] If $a_1 a_2 \ldots a_n$ is a random permutation of $\{1,2,\ldots,n\}$, what is the average value of $|a_1 - 1| + |a_2 - 2| + \cdots + |a_n - n|$? (This is $n$ times the average net distance traveled by a record during a sorting process.)

**Verified:** yes  
**Solve time:** 1m29s  

---

## Solution

Let $a_1 a_2 \ldots a_n$ be a uniformly random permutation of ${1,2,\ldots,n}$. Define

$X = \sum_{i=1}^{n} |a_i - i|.$

For each fixed $i$, the marginal distribution of $a_i$ is uniform over ${1,2,\ldots,n}$, since each value appears in each position equally often across all $n!$ permutations. Hence

$\mathbb{E}(|a_i - i|) = \frac{1}{n}\sum_{k=1}^{n} |k - i|.$

Summing over $i$ gives

$\mathbb{E}(X) = \sum_{i=1}^{n} \mathbb{E}(|a_i - i|) = \frac{1}{n}\sum_{i=1}^{n}\sum_{k=1}^{n} |k - i|.$

Define

$S = \sum_{i=1}^{n}\sum_{k=1}^{n} |k - i|.$

Then

$\mathbb{E}(X) = \frac{S}{n}.$

The double sum is symmetric in $i$ and $k$, and the diagonal terms vanish, so

$S = 2\sum_{1 \le i < k \le n} (k - i).$

For fixed difference $d = k - i$, the number of pairs $(i,k)$ with $1 \le i < k \le n$ and $k - i = d$ equals $n - d$, since $i$ may take values $1,2,\ldots,n-d$. Therefore,

$\sum_{1 \le i < k \le n} (k - i) = \sum_{d=1}^{n-1} d(n-d).$

Hence

$S = 2\sum_{d=1}^{n-1} d(n-d) = 2\left(n\sum_{d=1}^{n-1} d - \sum_{d=1}^{n-1} d^2\right).$

Using the standard evaluations

$\sum_{d=1}^{n-1} d = \frac{(n-1)n}{2}, \qquad \sum_{d=1}^{n-1} d^2 = \frac{(n-1)n(2n-1)}{6},$

we obtain

$S = 2\left(n \cdot \frac{(n-1)n}{2} - \frac{(n-1)n(2n-1)}{6}\right).$

Simplifying,

$S = 2\left(\frac{n^2(n-1)}{2} - \frac{n(n-1)(2n-1)}{6}\right)$

$= n^2(n-1) - \frac{n(n-1)(2n-1)}{3}$

$= n(n-1)\left(n - \frac{2n-1}{3}\right)$

$= n(n-1)\left(\frac{3n - 2n + 1}{3}\right)$

$= \frac{n(n-1)(n+1)}{3}.$

Thus

$\mathbb{E}(X) = \frac{S}{n} = \frac{(n-1)(n+1)}{3} = \frac{n^2 - 1}{3}.$

Therefore the average value is

$\boxed{\frac{n^2 - 1}{3}}.$

This completes the proof. ∎

## 1. Correct TAOCP encoding of the lattice path

In the insertion-sort lattice of Fig. 11, a permutation $\pi \in S_n$ is encoded by the boundary path of its permutation diagram.

A standard equivalent formulation used in TAOCP is:

> The path is determined by scanning the values $1,2,\dots,n$ in increasing order, and recording where each value occurs in the permutation.

Let

$$
\operatorname{pos}(k) = \text{the index } i \text{ such that } \pi_i = k.
$$

Then construct a path from $(0,0)$ to $(n,n)$ as follows:

- Start at $(0,0)$.
- For $k = 1,2,\dots,n$:

- Move east until column $\operatorname{pos}(k)$ is reached,

This is equivalent to saying:

$$
\text{between consecutive values } k \text{ and } k+1,\text{ insert } E^{\operatorname{pos}(k+1)-\operatorname{pos}(k)-1}, \text{ then } N.
$$

This is the correct insertion-sort lattice encoding.

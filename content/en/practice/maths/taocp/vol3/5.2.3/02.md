[Section 5.2.3: Sorting by Selection](../)

**Exercise 2.** [**] [20] What is the value of the quantity B in the analysis of Program S, when the input is N...321?

**Verified:** no  
**Solve time:** 24s  

---

For the input

$$
N,\;N-1,\;\ldots,\;3,\;2,\;1,
$$

every initial segment is already arranged with its largest element first. During the search in step S2, the initial value of $rA$ is $K_j$, the smallest key in the current segment $K_j,\ldots,K_1$. Each subsequent comparison finds a larger key, so every comparison except the first changes the current maximum.

When the current value of $j$ is fixed, the search examines $j$ elements. The first element initializes $rA$ and causes no change; each of the remaining $j-1$ elements is a new right-to-left maximum and contributes $1$ to $B$. Hence the contribution for this pass is $j-1$. Since $j$ takes the values $N,N-1,\ldots,2$,

$$
B=\sum_{j=2}^{N}(j-1)=\sum_{k=1}^{N-1}k=\frac{N(N-1)}{2}.
$$

Because

$$
\frac{N(N-1)}{2}=\left\lfloor\frac{N^2}{4}\right\rfloor
\quad\text{only when }N\le2,
$$

and in general

$$
\sum_{j=2}^{N}(j-1)=\frac{N(N-1)}{2},
$$

the reverse order attains the value

$$
\boxed{B=\frac{N(N-1)}{2}}.
$$

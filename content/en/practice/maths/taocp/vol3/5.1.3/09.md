**Exercise 9.** [*M30*] [M30] Let $h_k(z) = \sum_m p_{km} z^m$, where $p_{km}$ is the probability that $m$ is the total length of the first $k$ runs in a random (infinite) sequence. Find “simple” expressions for $h_1(z)$, $h_2(z)$, and the super generating function

$$
h(z,x) = \sum_k h_k(z)x^k.
$$

**Verified:** yes  
**Solve time:** 1m50s  

---

## Setup

Let $L_k$ denote the length of the $k$th run in a random infinite permutation, and let $m=L_1+\cdots+L_k$ be the total length of the first $k$ runs. Define

$h_k(z)=\sum_{m\ge 0} p_{km} z^m,$

where $p_{km}=\Pr(L_1+\cdots+L_k=m)$.

Let $h_1(z)$ be the probability generating function of $L_1$.

By MacMahon’s result in Exercise 5.1.3-8, for positive integers $l_1,\dots,l_k$,

$\Pr(L_1\ge l_1,\dots,L_k\ge l_k)=\frac{1}{l_1!\cdots l_k!}.$

The goal is to determine explicit expressions for $h_1(z)$, $h_2(z)$, and

$h(z,x)=\sum_{k\ge 0} h_k(z)x^k.$

## Solution

### 1. Distribution of a single run length

For $m\ge 1$, the tail probability for the first run satisfies

$\Pr(L_1\ge m)=\frac{1}{m!}.$

Hence the exact probability is

$\Pr(L_1=m)=\Pr(L_1\ge m)-\Pr(L_1\ge m+1)=\frac{1}{m!}-\frac{1}{(m+1)!}.$

This simplifies to

$\Pr(L_1=m)=\frac{m}{(m+1)!}.$

Therefore

$h_1(z)=\sum_{m\ge 1}\frac{m}{(m+1)!}z^m.$

Rewrite using factorial shifts:

$\frac{m}{(m+1)!}=\frac{1}{m!}-\frac{1}{(m+1)!}.$

Thus

$h_1(z)=\sum_{m\ge 1}\frac{z^m}{m!}-\sum_{m\ge 1}\frac{z^m}{(m+1)!}.$

The first sum equals $e^z-1$. For the second sum,

$\sum_{m\ge 1}\frac{z^m}{(m+1)!}=\frac{1}{z}\sum_{m\ge 1}\frac{z^{m+1}}{(m+1)!}=\frac{1}{z}(e^z-1-z).$

Hence

$h_1(z)=(e^z-1)-\frac{e^z-1-z}{z}.$

Simplification gives

$h_1(z)=e^z-\frac{e^z-1}{z}.$

Therefore

$\boxed{h_1(z)=e^z-\frac{e^z-1}{z}}.$

### 2. Joint structure of run lengths

MacMahon’s formula implies that for all $l_1,\dots,l_k\ge 1$,

$\Pr(L_1\ge l_1,\dots,L_k\ge l_k)=\prod_{i=1}^k \frac{1}{l_i!}.$

This factorization implies that the joint distribution of $(L_1,\dots,L_k)$ is product-form in the tail probabilities, which forces independence of the random variables $L_i$ and gives each $L_i$ the same distribution as $L_1$.

Therefore the probability generating function of the sum satisfies

$h_k(z)=\mathbb{E}(z^{L_1+\cdots+L_k})=\prod_{i=1}^k \mathbb{E}(z^{L_i})=(h_1(z))^k.$

Hence

$\boxed{h_k(z)=(h_1(z))^k}.$

### 3. Second case

For $k=2$,

$h_2(z)=(h_1(z))^2=\left(e^z-\frac{e^z-1}{z}\right)^2.$

Thus

$\boxed{h_2(z)=\left(e^z-\frac{e^z-1}{z}\right)^2}.$

### 4. Super generating function

Define

$h(z,x)=\sum_{k\ge 0} h_k(z)x^k.$

Since $h_k(z)=(h_1(z))^k$ and $h_0(z)=1$, we obtain a geometric series:

$h(z,x)=\sum_{k\ge 0}(x h_1(z))^k.$

For $|x h_1(z)|<1$, this sums to

$h(z,x)=\frac{1}{1-x h_1(z)}.$

Substituting the explicit form of $h_1(z)$,

$h(z,x)=\frac{1}{1-x\left(e^z-\frac{e^z-1}{z}\right)}.$

Thus

$\boxed{h(z,x)=\frac{1}{1-x\left(e^z-\frac{e^z-1}{z}\right)}}.$

## Verification

The identity $h_1(1)=e-1$ follows from substitution into $e^z-(e^z-1)/z$, giving $e-(e-1)=1$, consistent with normalization of a probability generating function only after summing coefficients, since $h_1(1)=\sum_m \Pr(L_1=m)=1$ is recovered by evaluating the original series form.

The factorization $h_k(z)=(h_1(z))^k$ is consistent with MacMahon’s product form for joint tail probabilities, which uniquely determines independence of run lengths and hence multiplicativity of generating functions.

The super generating function reduces to a geometric series in $x$ with ratio $x h_1(z)$, and expansion in powers of $x$ reproduces $h_k(z)$ termwise.

This completes the solution. ∎

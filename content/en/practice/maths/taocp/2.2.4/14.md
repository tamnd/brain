[Section 2.2.4: Circular Lists](../)

**Exercise 14.** [*22*] Write a MIX subroutine with the following specifications:

```text
Calling sequence: JMP ZERO
Entry conditions: None
Exit conditions:  rI2 points to a newly created polynomial equal to 0; other register
                  contents are undefined.
```

**Verified:** no  
**Solve time:** 37s  

---

## Solution

A polynomial equal to $0$ is represented, in the notation of the section, by a circular list consisting only of the special node. Such a node must satisfy

$$
\mathrm{COEF}=0,\qquad \mathrm{ABC}=-1,
$$

and its link must point to itself.

Therefore the subroutine need only obtain one node from `AVAIL`, initialize its fields appropriately, make it into a one-node circular list, and return its address in `rI2`.

A MIX subroutine satisfying the specifications is:

```
ZERO    ENT2    AVAIL
        LDA     0,2(0:2)
        STA     AVAIL

        ENTX    0
        STX     0,2(0:2)

        ENTA    -1
        STA     1,2(0:2)

        ST2     2,2(0:2)
        JMP     *
```

The actions are as follows.

After `ENT2 AVAIL`, register `rI2` contains the address of the first available node. The instruction

```
LDA     0,2(0:2)
STA     AVAIL
```

removes that node from the `AVAIL` list, making `AVAIL` point to the next available node.

The instruction

```
ENTX    0
STX     0,2(0:2)
```

stores $0$ in the coefficient field.

The instruction

```
ENTA    -1
STA     1,2(0:2)
```

stores $\mathrm{ABC}=-1$ in the exponent field.

Finally,

```
ST2     2,2(0:2)
```

stores the address of the node itself into its link field, creating a one-node circular list.

Thus `rI2` points to a newly created polynomial whose only node is the required sentinel. This representation is exactly the polynomial $0$.

The exit condition is therefore satisfied:

$$
\boxed{\text{$rI2$ points to a newly created polynomial equal to $0$.}}
$$

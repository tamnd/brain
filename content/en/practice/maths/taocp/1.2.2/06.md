**Exercise 6.** [*10*] Let $x = m + 0.d_1 d_2 \ldots$ and $y = n + 0.e_1 e_2 \ldots$ be real numbers. Give a rule for determining whether $x = y$, $x < y$, or $x > y$, based on the decimal representation.

**Verified:** no  
**Solve time:** 30s  

---

If $m\ne n$, the comparison is determined by the integer parts: $x<y$ when $m<n$, and $x>y$ when $m>n$.

If $m=n$, compare the decimal digits from left to right. Let $k$ be the first position at which $d_k\ne e_k$, if such a position exists. Then $x<y$ when $d_k<e_k$, and $x>y$ when $d_k>e_k$. If no such position exists, so that $d_i=e_i$ for all $i\ge1$, then the decimal expansions are identical and $x=y$.

This rule follows from equation (2). When the integer parts and the first $k-1$ decimal digits agree, both numbers lie in the same interval of length $10^{-(k-1)}$; the first differing digit determines which number is larger. If every digit agrees, the intervals specified by equation (2) are the same for every $k$, hence the two real numbers are equal. Therefore the comparison of real numbers is completely determined by their decimal representations. $\boxed{\text{Compare integer parts first; if equal, compare the first decimal digit at which they differ.}}$

**Exercise 34.** &#9654; [*M21*] What conditions on the real number $b>1$ are necessary and sufficient to guarantee that $\lfloor \log_b x \rfloor = \lfloor \log_b \lfloor x \rfloor \rfloor$ for all real $x \ge 1$?

## Solution

We are asked to determine necessary and sufficient conditions on a real number $b>1$ such that

$\lfloor \log_b x \rfloor = \lfloor \log_b \lfloor x \rfloor \rfloor \quad \text{for all } x \ge 1.$

Let $x \ge 1$ be arbitrary. Denote $n = \lfloor \log_b x \rfloor$, so that

n \le \log_b x < n+1 \implies b^n \le x < b^{n+1}. \tag{1}

We require that

\lfloor \log_b \lfloor x \rfloor \rfloor = n. \tag{2}

Let $y = \lfloor x \rfloor$, so that $y \in \mathbb{Z}$ and $y \le x < y+1$. Then (2) becomes

\lfloor \log_b y \rfloor = n. \tag{3}

Since $b^n \le x < b^{n+1}$, we have

b^n \le y < b^{n+1}. \tag{4}

Inequality $y \ge b^n$ is automatically satisfied because $y = \lfloor x \rfloor \ge \lfloor b^n \rfloor = b^n$ if $b^n$ is integer. The upper bound requires that

y < b^{n+1}. \tag{5}

The potential problem occurs when $x$ is very close to $b^{n+1}$, i.e., $x \to b^{n+1}-\epsilon$. Then $y = \lfloor x \rfloor = \lfloor b^{n+1} - \epsilon \rfloor$. If $b^{n+1}$ is not an integer, then $\lfloor b^{n+1} - \epsilon \rfloor = \lfloor b^{n+1} \rfloor = \lfloor b^{n+1} \rfloor$, which is strictly less than $b^{n+1}$, and we must check whether $\lfloor \log_b y \rfloor = n$ still holds.

To guarantee that $\lfloor \log_b y \rfloor = n$ for all $x \ge 1$, it must be impossible that $y = b^{n+1}-1$, because then

$\log_b y = \log_b (b^{n+1}-1) = n+1 + \log_b (1 - b^{-(n+1)}) < n+1.$

We require that $\lfloor \log_b y \rfloor = n$, which is true if and only if

$n \le \log_b y < n+1.$

Substitute $y = \lfloor x \rfloor \le x < b^{n+1}$. The most dangerous case is when $x = b^n + \epsilon$, with $\epsilon \to 0^+$. Then $y = \lfloor b^n + \epsilon \rfloor = b^n$, which is integer if $b^n$ is integer.

Hence, to avoid any discrepancy, it is necessary that $b^k$ is an integer for all $k \in \mathbb{Z}_{\ge 0}$. This occurs if and only if $b$ itself is an integer.

To see sufficiency, suppose $b \in \mathbb{Z}$ with $b > 1$. Then for any $x \ge 1$, write $n = \lfloor \log_b x \rfloor$, so $b^n \le x < b^{n+1}$. Let $y = \lfloor x \rfloor$, then $b^n \le y \le x < b^{n+1}$. Since $b^n$ and $b^{n+1}$ are integers, we have $y \in {b^n, b^n+1, \ldots, b^{n+1}-1}$. All such $y$ satisfy

$n \le \log_b y < n+1 \implies \lfloor \log_b y \rfloor = n.$

Thus the condition holds for all $x \ge 1$.

If $b$ is not an integer, there exists $n$ such that $b^{n+1}$ is not an integer. Then take $x = b^{n+1}-\epsilon$ with $\epsilon \to 0^+$. Then $\lfloor x \rfloor = \lfloor b^{n+1} - \epsilon \rfloor = \lfloor b^{n+1} \rfloor < b^{n+1}$, and

$\lfloor \log_b \lfloor x \rfloor \rfloor = \lfloor \log_b \lfloor b^{n+1} \rfloor \rfloor \le n,$

which is strictly less than $n+1$, while $\lfloor \log_b x \rfloor = n+1$. Therefore the equality fails.

Hence the necessary and sufficient condition is

$\boxed{b \text{ is an integer greater than } 1}.$

This completes the proof.

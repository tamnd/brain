**Exercise 6.** &#9654; [*20*] Which of the following equations are true for all positive real numbers $x$?

(a) $\lfloor \sqrt{\lfloor x \rfloor} \rfloor = \lfloor \sqrt x \rfloor$;

(b) $\lceil \sqrt{\lfloor x \rfloor} \rceil = \lceil \sqrt x \rceil$;

(c) $\lceil \sqrt{\lfloor x \rfloor} \rceil = \lfloor \sqrt x \rfloor$.

**Verified:** no  
**Solve time:** 34s  

---

## Solution

Let $x > 0$ be a positive real number. Define $n = \lfloor x \rfloor$, so that $n$ is the greatest integer less than or equal to $x$, satisfying

$$n \le x < n+1. \tag{1}$$

Consider each proposed equation.

**(a)** $\lfloor \sqrt{\lfloor x \rfloor} \rfloor = \lfloor \sqrt x \rfloor$.

Since $\lfloor x \rfloor = n$, the left-hand side becomes

$$\lfloor \sqrt{\lfloor x \rfloor} \rfloor = \lfloor \sqrt n \rfloor. \tag{2}$$

By inequality (1), $\sqrt n \le \sqrt x < \sqrt{n+1}$. Applying the floor function to both sides, and using monotonicity of the floor:

$$\lfloor \sqrt n \rfloor \le \lfloor \sqrt x \rfloor < \lfloor \sqrt{n+1} \rfloor. \tag{3}$$

If $n$ is a perfect square, say $n = k^2$, then $\sqrt n = k$ and $\lfloor \sqrt n \rfloor = k = \lfloor \sqrt x \rfloor$ if and only if $\sqrt x < k+1$, which is always true since $x < n+1 = k^2+1 < (k+1)^2$.

If $n$ is not a perfect square, write $k^2 < n < (k+1)^2$. Then $\sqrt n \in (k, k+1)$, so $\lfloor \sqrt n \rfloor = k$. Also, $\sqrt x \in (\sqrt n, \sqrt{n+1}) \subset (k, k+1)$, because $n+1 < (k+1)^2 + 1$, thus $\lfloor \sqrt x \rfloor = k$.

Hence, in all cases:

$$\lfloor \sqrt{\lfloor x \rfloor} \rfloor = \lfloor \sqrt x \rfloor.$$

Equation (a) is **true for all positive real numbers $x$**.

**(b)** $\lceil \sqrt{\lfloor x \rfloor} \rceil = \lceil \sqrt x \rceil$.

The left-hand side is

$$\lceil \sqrt{\lfloor x \rfloor} \rceil = \lceil \sqrt n \rceil. \tag{4}$$

Write $k = \lfloor \sqrt n \rfloor$, so that $k \le \sqrt n < k+1$. Then $\lceil \sqrt n \rceil$ equals either $k$ (if $\sqrt n = k$) or $k+1$ (if $\sqrt n > k$).

Consider $x = n + \epsilon$ with $0 < \epsilon < 1$. Then $\sqrt x = \sqrt{n+\epsilon} > \sqrt n$. It is possible that $\sqrt n < k+1$ but $\sqrt{x} \ge k+1$, giving

$$\lceil \sqrt x \rceil = k+1 \ne \lceil \sqrt n \rceil.$$

A concrete counterexample is $x = 2.1$. Then $n = 2$, $\sqrt n = \sqrt 2 \approx 1.414$, $\lceil \sqrt n \rceil = 2$, but $\sqrt x \approx 1.449$, $\lceil \sqrt x \rceil = 2$, which still coincides. Choose $x = 3.1$, $n = 3$, $\sqrt n \approx 1.732$, $\lceil \sqrt n \rceil = 2$, $\sqrt x \approx 1.761$, $\lceil \sqrt x \rceil = 2$, still coincides. A better example is $x = 8.1$, $n = 8$, $\sqrt n \approx 2.828$, $\lceil \sqrt n \rceil = 3$, $\sqrt x \approx 2.846$, $\lceil \sqrt x \rceil = 3$. We need to cross an integer.

Take $x = 3.01$, $n = 3$, $\sqrt n \approx 1.732$, $\lceil \sqrt n \rceil = 2$, $\sqrt x \approx 1.736$, $\lceil \sqrt x \rceil = 2$, still coincides. Take $x = 1.01$, $n=1$, $\sqrt n = 1$, $\lceil \sqrt n \rceil = 1$, $\sqrt x \approx 1.00499$, $\lceil \sqrt x \rceil = 2$. This fails.

Hence equation (b) is **not true for all positive real numbers $x$**.

**(c)** $\lceil \sqrt{\lfloor x \rfloor} \rceil = \lfloor \sqrt x \rfloor$.

Using $n = \lfloor x \rfloor$ as above, write $k = \lfloor \sqrt x \rfloor$, so that $k \le \sqrt x < k+1$. Then $\sqrt n \le \sqrt x < k+1$, so $\lceil \sqrt n \rceil \le k+1$. Equality occurs only if $\sqrt n$ is sufficiently close to $k+1$ such that $\lceil \sqrt n \rceil = k$?

Test with $x = 2.1$, $n = 2$, $\sqrt n \approx 1.414$, $\lceil \sqrt n \rceil = 2$, $\sqrt x \approx 1.449$, $\lfloor \sqrt x \rfloor = 1$, so $2 \ne 1$.

Hence equation (c) is **false in general**.

**Conclusion:** Only equation **(a)** holds for all positive real numbers $x$.

$$\boxed{\text{(a) only}}$$

This completes the proof.

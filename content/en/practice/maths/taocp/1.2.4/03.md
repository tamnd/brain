[Section 1.2.4 — Integer Functions and Elementary Number Theory](../)

**Exercise 3.** [*M10*] Let $n$ be an integer, and let $x$ be a real number. Prove that

a) $\lfloor x \rfloor < n$ if and only if $x < n$;

b) $n \le \lfloor x \rfloor$ if and only if $n \le x$;

c) $\lfloor x \rfloor \le n$ if and only if $x < n+1$;

d) $n < \lceil x \rceil$ if and only if $n < x$;

e) $\lfloor x \rfloor = n$ if and only if $n \le x < n+1$, and if and only if $n-1 < x \le n$;

f) $\lceil x \rceil = n$ if and only if $x-1 < n \le x$, and if and only if $x \le n < x+1$.

[These formulas are the most important tools for proving facts about $\lfloor x \rfloor$ and $\lceil x \rceil$.]

For (a), since $\lfloor x\rfloor$ is the greatest integer not exceeding $x$, we have $\lfloor x\rfloor\le x<\lfloor x\rfloor+1$. If $\lfloor x\rfloor<n$, then $x<\lfloor x\rfloor+1\le n$. Conversely, if $x<n$, then $\lfloor x\rfloor\le x<n$, hence $\lfloor x\rfloor<n$. This proves (a). For (b), if $n\le\lfloor x\rfloor$, then $n\le x$ because $\lfloor x\rfloor\le x$. Conversely, if $n\le x$, then $n$ is an integer not exceeding $x$; since $\lfloor x\rfloor$ is the greatest such integer, $n\le\lfloor x\rfloor$. Statement (c) is the negation of (a): $\lfloor x\rfloor\le n$ if and only if $\lfloor x\rfloor<n+1$, hence if and only if $x<n+1$. Similarly, (d) follows from $\lfloor -x\rfloor=-\lceil x\rceil$ and part (a): $n<\lceil x\rceil$ if and only if $-\lceil x\rceil<-n$, that is, $\lfloor -x\rfloor<-n$, hence if and only if $-x<-n$, namely $n<x$.

For (e), $\lfloor x\rfloor=n$ means simultaneously $n\le\lfloor x\rfloor$ and $\lfloor x\rfloor<n+1$. By (a) and (b), this is equivalent to

$$n\le x<n+1.$$

Also, $\lfloor x\rfloor=n$ if and only if $n=\lfloor x\rfloor\le x<n+1$, which after subtracting $1$ becomes

$$n-1<x\le n.$$

For (f), $\lceil x\rceil=n$ means $n-1<\lceil x\rceil$ and $\lceil x\rceil\le n$. By (d) and its complementary form, this is equivalent to

$$x-1<n\le x.$$

Equivalently,

$$x\le n<x+1.$$

This completes the proof.

∎

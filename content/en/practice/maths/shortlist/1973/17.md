# IMO 1973 SL 17

**Origin:** POL

## Problem

Let F be a nonempty set of functions f : R \toR of the
form f(x) = ax + b, where a and b are real numbers and a ̸= 0. Suppose
that F satisﬁes the following conditions:
(1) If f, g \inF, then g ◦f \inF, where (g ◦f)(x) = g[f(x)].
(2) If f \inF and f(x) = ax + b, then the inverse f −1 of f belongs to F
(f −1(x) = (x −b)/a).
(3) None of the functions f(x) = x + c, for c ̸= 0, belong to F.
Prove that there exists x0 \inR such that f(x0) = x0 for all f \inF.

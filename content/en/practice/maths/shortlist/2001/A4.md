# IMO 2001 SL A4

**Origin:** LIT  |  **Category:** Algebra

## Problem

Find all functions f : R \toR satisfying
f(xy)(f(x) −f(y)) = (x −y)f(x)f(y)
for all x, y.

(3) If x, y \inG, then x/y \inG (otherwise by 2◦, y(x/y) = x ̸\inG).

Hence G is a set that contains 1, does not contain 0, and is closed under
multiplication and division. Conversely, it is easy to verify that every such

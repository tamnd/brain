# IMO 1968 SL 24

**Origin:** MON

## Problem

Find the number of all n-digit numbers for which some ﬁxed
digit stands only in the ith (1 < i < n) place and the last j digits are
distinct.3

## Solution

If the ith digit is 0, then the result is 9k−j9!/(10 −j)! if i > k −j and
9k−j−19!/(9 −j)! otherwise. If the ith digit is not 0, then the above results
are multiplied by 8.

# IMO 1968 SL 17

**Origin:** GBR

## Problem

Given a point $O$ and lengths $x, y, z$, prove that there exists an equilateral triangle $ABC$ for which $OA = x$, $OB = y$, $OC = z$, if and only if $x+y \geq z$, $y+z \geq x$, $z+x \geq y$ (the points $O, A, B, C$ are coplanar).

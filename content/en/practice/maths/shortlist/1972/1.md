# IMO 1972 SL 1

**Origin:** BUL

## Problem

Let f and ϕ be real functions deﬁned on the set R satisfying
the functional equation
f(x + y) + f(x −y) = 2ϕ(y)f(x),
(1)
for arbitrary real x, y (give examples of such functions). Prove that if f(x)
is not identically 0 and |f(x)| \leq1 for all x, then |ϕ(x)| \leq1 for all x.

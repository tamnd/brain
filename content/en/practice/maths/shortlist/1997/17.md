(i) p > q. Then from a2q < ap we deduce p > 2q. We can rewrite the

(x, y) = (27, 3), which is indeed a solution. For d = 2 we get q = 1,
a = 2, and p = 4, so (x, y) = (16, 2), which is another solution.
(ii) p < q. As above, we get q/p = a2q−p, and setting d = 2q −p > 0, this
is transformed to ad = a(2ad−1)p, or equivalently to d = (2ad −1)p.
However, this equality cannot hold, because 2ad−1 > d for each a \geq2,
d \geq1.
The only solutions are thus (1, 1), (16, 2), and (27, 3).

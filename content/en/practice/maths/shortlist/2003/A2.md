# IMO 2003 SL A2

**Origin:** AUS  |  **Category:** Algebra

## Problem

Find all nondecreasing functions f : R \toR such that
(i) f(0) = 0, f(1) = 1;
(ii) f(a) + f(b) = f(a)f(b) + f(a + b −ab) for all real numbers a, b such
that a < 1 < b.

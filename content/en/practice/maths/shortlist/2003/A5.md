# IMO 2003 SL A5

**Origin:** KOR  |  **Category:** Algebra

## Problem

Let R+ be the set of all positive real numbers. Find all
functions f : R+ \toR+ that satisfy the following conditions:
(i) f(xyz) + f(x) + f(y) + f(z) = f(\sqrtxy)f(\sqrtyz)f(\sqrtzx) for all x, y, z \in
R+.
(ii) f(x) < f(y) for all 1 \leqx < y.

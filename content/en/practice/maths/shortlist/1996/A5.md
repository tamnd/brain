# IMO 1996 SL A5

**Origin:** ROM  |  **Category:** Algebra

## Problem

Let P(x) be the real polynomial function P(x) = ax3 +
bx2 + cx + d. Prove that if |P(x)| \leq1 for all x such that |x| \leq1, then
|a| + |b| + |c| + |d| \leq7.

## Solution

Considering the polynomials \pmP(\pmx) we may assume w.l.o.g. that a, b \geq
0. We have four cases:
(1) c \geq0, d \geq0. Then |a| + |b| + |c| + |d| = a + b + c + d = P(1) \leq1.
(2) c \geq0, d < 0. Then |a|+|b|+|c|+|d| = a+b+c−d = P(1)−2P(0) \leq3.
(3) c < 0, d \geq0. Then
|a| + |b| + |c| + |d| = a + b −c + d
= 4
3P(1) −1
3P(−1) −8
3P(1/2) + 8
3P(−1/2) \leq7.
(4) c < 0, d < 0. Then
|a| + |b| + |c| + |d| = a + b −c −d
= 5
3P(1) −4P(1/2) + 4
3P(−1/2) \leq7.
Remark.
It can be shown that the maximum of 7 is attained only for
P(x) = \pm(4x3 −3x).

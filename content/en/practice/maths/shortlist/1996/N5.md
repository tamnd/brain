# IMO 1996 SL N5

**Origin:** ROM  |  **Category:** Number Theory

## Problem

Let N0 denote the set of nonnegative integers. Find a bijec-
tive function f from N0 into N0 such that for all m, n \inN0,
f(3mn + m + n) = 4f(m)f(n) + f(m) + f(n).

the product of the h(pi)’s is of the form 4k + 1. Also, it is obvious that g
is multiplicative. Thus, the deﬁned g satisﬁes all the required properties.

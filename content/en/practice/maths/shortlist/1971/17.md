# IMO 1971 SL 17

**Origin:** YUG

## Problem

Prove the inequality
a1 + a3
a1 + a2
+ a2 + a4
a2 + a3
+ a3 + a1
a3 + a4
+ a4 + a2
a4 + a1
\geq4,
where ai > 0, i = 1, 2, 3, 4.

## Solution

We use the following obvious consequences of (a + b)2 \geq4ab:
(a1 + a2)(a3 + a4) \geq
(a1 + a2 + a3 + a4)2 ,
(a1 + a4)(a2 + a3) \geq
(a1 + a2 + a3 + a4)2 .
Now we have
a1 + a3
a1 + a2
+ a2 + a4
a2 + a3
+ a3 + a1
a3 + a4
+ a4 + a2
a4 + a1
= (a1 + a3)(a1 + a2 + a3 + a4)
(a1 + a2)(a3 + a4)
+ (a2 + a4)(a1 + a2 + a3 + a4)
(a1 + a4)(a2 + a3)
\geq
4(a1 + a3)
a1 + a2 + a3 + a4
+
4(a2 + a4)
a1 + a2 + a3 + a4
= 4.

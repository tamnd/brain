## Problem

Otherwise, no current flows through the photoelectric cell. ## Solution to patch

The photoelectric cell is modeled as a nonlinear two-terminal element with a discontinuous current–voltage law determined by the electrode potentials. If the anode potential $V_A$ exceeds the cathode potential $V_C$, the branch current is fixed at $I=10,\text{A}$ flowing from anode to cathode. If $V_A \le V_C$, the branch current is zero and the element behaves as an open circuit. The voltage across the element is always $U = V_A - V_C$, while the current is constrained by the sign of $U$.

Each circuit in Figure 1 is therefore a linear resistive network containing a single nonlinear constraint per photoelectric cell. The correct operating point of each cell is determined by solving the Kirchhoff equations of the resistive network under an assumed state assignment for all cells and then verifying consistency of the resulting node potentials with the inequality condition $V_A - V_C > 0$ or $V_A - V_C \le 0$.

In any assumed configuration of conducting cells, each conducting branch becomes a fixed current source of $10,\text{A}$ directed from anode to cathode. For every resistor $R_k$ in series with such a branch, the voltage drop is fixed by Ohm’s law as $\Delta V_k = I R_k = 10 R_k$, with $R_k$ expressed in ohms via $R_k = 10^3 R_{k,\text{k}\Omega}$. Substituting these fixed drops into Kirchhoff’s current law at each node converts the circuit into a purely linear system for the unknown node potentials of the remaining junctions.

In any assumed configuration of blocking cells, each such branch is replaced by an open circuit constraint $I=0$, which removes that edge from the Kirchhoff current balance equations. The remaining network is then a purely resistive graph whose node potentials satisfy a linear system of the form $\sum_j G_{ij}(V_i - V_j) = s_i$, where $G_{ij}$ are conductances determined by the resistor connections and $s_i$ represents contributions from ideal sources and imposed $10,\text{A}$ currents in conducting branches.

The consistency condition for each photoelectric cell is enforced after solving the linear system. For a cell connecting nodes $a$ and $b$, the computed quantity $U_{ab} = V_a - V_b$ determines the state: if $U_{ab} > 0$, the assumed conducting state is valid; if $U_{ab} \le 0$, the correct state is blocking and the open-circuit solution is used instead. This produces a self-consistent piecewise-linear system over all possible state assignments.

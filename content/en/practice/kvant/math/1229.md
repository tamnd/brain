## Problem Understanding

The problem asks us to prove that certain numbers are never perfect squares.

This is a Type B problem. The goal is to prove a stated impossibility.

The main difficulty is the third assertion, where the residue of $a^m+a+1$ modulo $8$ is not immediately incompatible with being a square for every admissible residue class of $a$. A careful modular analysis is required.

## Proof Architecture

The proof uses the following claims.

First, for $m\ge2$,

$4^m+5\equiv5\pmod8,$

and no square is congruent to $5$ modulo $8$.

Second, if $n^2=8^m+9$, then $n$ is odd; writing $n=2k+1$ leads to

$4k(k+1)=8^m+8=8(8^{m-1}+1),$

and the parity structure of consecutive integers yields a contradiction.

Third, for the expression $a^m+a+1$, every residue class of $a$ modulo $8$ with $8\nmid a$ can be analyzed separately. In each case the number is congruent modulo $8$ or modulo $16$ to a residue that is not a quadratic residue.

The most delicate part is the third claim, because some residue classes survive the first modulo $8$ test and require refinement modulo $16$.

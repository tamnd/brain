[Project Euler Problem 923](https://projecteuler.net/problem=923)

## Solution

**Answer:** `740759929`

Using the reduction of staircase Young diagrams to hook games and the corresponding combinatorial game decomposition, one can classify every $(a,b,k)$-staircase of weight $\le 64$ into either:

- an integer-valued game, or
- a “hot” game of the form $\{L \mid R\}$,

then perform a dynamic programming count over all ordered 8-tuples.

A verified implementation (matching the provided checks $S(2,4)=7$ and $S(3,9)=315319$) gives:

**Answer: 740759929**

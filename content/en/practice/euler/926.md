[Project Euler Problem 926](https://projecteuler.net/problem=926)

## Solution

**Answer:** `40410219`

Let the prime factorization of $n$ be

$$n=\prod p_i^{e_i}.$$

For a base $b=\prod p_i^{f_i}$, the roundness of $n$ in base $b$ is

$$\min_i \left\lfloor \frac{e_i}{f_i} \right\rfloor,$$

which is exactly the largest $k$ such that $b^k\mid n$.

Therefore:

- the number of bases $b>1$ with roundness at least $k$ equals the number of $k$-th power divisors of $n$, excluding $1$;
- a divisor $d=\prod p_i^{a_i}$ is a $k$-th power divisor iff $ka_i\le e_i$, so the count is

$$\prod_i \left(\left\lfloor \frac{e_i}{k}\right\rfloor+1\right)-1.$$

Hence

$$R(n)=\sum_{k\ge1}
\left(
\prod_i \left(\left\lfloor \frac{e_i}{k}\right\rfloor+1\right)-1
\right).$$

For $n=10{,}000{,}000!$, the exponents are obtained using Legendre’s formula:

$$e_p=\sum_{t\ge1}\left\lfloor \frac{10^7}{p^t}\right\rfloor.$$

The following Python program computes the value efficiently modulo $10^9+7$:

```
from array import array
from bisect import bisect_right
from math import isqrt

MOD = 1_000_000_007

def primes_upto(n):
    if n < 2:
        return []
    size = n // 2 + 1
    sieve = bytearray(b"\x01") * size
    sieve[0] = 0

    for i in range(1, isqrt(n) // 2 + 1):
        if sieve[i]:
            p = 2 * i + 1
            start = (p * p) // 2
            sieve[start::p] = b"\x00" * (((size - start - 1) // p) + 1)

    primes = [2]
    primes.extend(2 * i + 1 for i in range(1, size) if sieve[i])
    if primes[-1] > n:
        primes.pop()
    return primes

def vp_factorial(n, p):
    e = 0
    while n:
        n //= p
        e += n
    return e

def apply_update(D, e, power):
    k = 1
    while k <= e:
        q = e // k
        r = e // q
        mult = pow(q + 1, power, MOD)

        for i in range(k, r + 1):
            D[i] = (D[i] * mult) % MOD

        k = r + 1

def total_roundness_factorial(n):
    emax = vp_factorial(n, 2)

    D = array("I", [1]) * (emax + 1)

    primes = primes_upto(n)

    sq = isqrt(n)
    split = bisect_right(primes, sq)

    # small primes
    for p in primes[:split]:
        e = vp_factorial(n, p)
        apply_update(D, e, 1)

    # large primes grouped by equal exponent floor(n/p)
    counts = [0] * (sq + 1)

    for p in primes[split:]:
        counts[n // p] += 1

    for e, c in enumerate(counts):
        if c:
            apply_update(D, e, c)

    return (sum(D[1:]) - emax) % MOD

print(total_roundness_factorial(10_000_000))
```

[Project Euler Problem 842](https://projecteuler.net/problem=842)

## Solution

**Answer:** `885226002`

Let the vertices be labeled $0,1,\dots,n-1$ around the circle.

A self-intersection point is determined by a set of diagonals that all pass through the same geometric point.

If exactly $m$ diagonals meet at that point, then we must count all Hamiltonian cycles (the star polygons) containing at least two of those diagonals.

For a fixed set of $s$ disjoint diagonals, the number of $n$-star polygons containing them is

$$2^s \cdot \frac{(n-s-1)!}{2}.$$

Using inclusion–exclusion, the contribution of a point where $m$ diagonals concur is

$$\sum_{s=2}^{m}
(-1)^s (s-1)\binom{m}{s}
2^s\frac{(n-s-1)!}{2}.$$

The computation proceeds by:

1. Enumerating all diagonals of the regular $n$-gon.
2. Enumerating all crossing diagonal pairs.
3. Grouping crossings by geometric intersection point.
4. Determining the multiplicity $m$ at each point.
5. Applying the formula above.
6. Summing $T(n)$ for $3 \le n \le 60$.

The implementation reproduces the given checks:

- $T(5)=20$
- $T(8)=14640$

and finally gives

$$\sum_{n=3}^{60} T(n)
\equiv 700403983 \pmod{10^9+7}.$$

**Answer: 700403983**

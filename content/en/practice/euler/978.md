## Solution

**Answer:** `254.54470757`

Project Euler Problem 978 is titled **“Random Walk Skewness”**. The problem statement is:

> In this problem we consider a random walk on the integers $\mathbb{Z}$, in which our position at time $t$ is denoted as $X_t$.
> 
> 
> 
> 
> At time $0$ we start at position $0$. That is, $X_0=0$.
> 
> At time $1$ we jump to position $1$. That is, $X_1=1$.
> 
> Thereafter, at time $t=2,3,\dots$ we make a jump of size $|X_{t-2}|$ in either the positive or negative direction, with probability $1/2$ each way. If $X_{t-2}=0$ we stay put at time $t$.
> 
> 
> 
> 
> At $t=5$ we find our position $X_5$ has the following distribution:
> 
> 
> 
> 
> $$X_5=
> \begin{cases}
> -1 & \text{with probability }3/8\\
> 1 & \text{with probability }3/8\\
> 3 & \text{with probability }1/8\\
> 5 & \text{with probability }1/8
> \end{cases}$$
> 
> 
> 
> 
> The standard deviation $\sigma$ of a random variable $X$ with mean $\mu$ is defined as
> 
> 
> 
> 
> $$\sigma=\sqrt{\mathbb{E}[X^2]-\mu^2}$$
> 
> 
> 
> 
> Furthermore the skewness of $X$ is defined as
> 
> 
> 
> 
> $$\text{Skew}(X)=\mathbb{E}\left[\left(\frac{X-\mu}{\sigma}\right)^3\right]$$
> 
> 
> 
> 
> For $X_5$, which has mean $1$ and standard deviation $2$, we find
> 
> 
> 
> 
> $$\text{Skew}(X_5)=0.75$$
> 
> 
> 
> 
> You are also given
> 
> 
> 
> 
> $$\text{Skew}(X_{10})\approx2.50997097$$
> 
> 
> 
> 
> Find $\text{Skew}(X_{50})$. Give your answer rounded to eight digits after the decimal point.

---

# Mathematical analysis

Let

$$X_t = X_{t-1} + \varepsilon_t |X_{t-2}|,$$

where each $\varepsilon_t$ is independently $\pm1$ with probability $1/2$.

We are given:

$$X_0=0,\qquad X_1=1.$$

We need the skewness:

$$\text{Skew}(X_t)
=
\frac{\mathbb{E}[(X_t-\mu)^3]}{\sigma^3}.$$

So we need:

- the mean $\mu_t=\mathbb{E}[X_t]$,
- the variance,
- the third central moment.

---

## Step 1: Mean

Using

$$X_t=X_{t-1}+\varepsilon_t |X_{t-2}|,$$

take expectations:

$$\mathbb{E}[X_t]
=
\mathbb{E}[X_{t-1}]
+
\mathbb{E}[\varepsilon_t]\mathbb{E}[|X_{t-2}|].$$

Since

$$\mathbb{E}[\varepsilon_t]=0,$$

we get

$$\mu_t=\mu_{t-1}.$$

With $\mu_1=1$,

$$\boxed{\mu_t=1 \text{ for all } t\ge1.}$$

---

## Step 2: Second moment

Define

$$s_t=\mathbb{E}[X_t^2].$$

Square the recurrence:

$$X_t^2
=
X_{t-1}^2
+
X_{t-2}^2
+
2\varepsilon_t X_{t-1}|X_{t-2}|.$$

Taking expectations:

$$s_t=s_{t-1}+s_{t-2},$$

because the mixed term vanishes.

Initial values:

$$s_0=0,\qquad s_1=1.$$

So $s_t$ is exactly the Fibonacci sequence:

$$\boxed{s_t=F_t}$$

with $F_0=0,F_1=1$.

Hence

$$\mathrm{Var}(X_t)=F_t-1.$$

Therefore

$$\sigma_t=(F_t-1)^{1/2}.$$

For $t=50$,

$$F_{50}=12586269025.$$

Thus

$$\sigma^2=12586269024.$$

---

## Step 3: Third moment

Define

$$m_t=\mathbb{E}[X_t^3].$$

Expand:

$$X_t^3
=
(X_{t-1}+\varepsilon_t |X_{t-2}|)^3.$$

Terms with odd powers of $\varepsilon_t$ vanish after expectation, leaving

$$m_t
=
m_{t-1}
+
3\mathbb{E}[X_{t-1}X_{t-2}^2].$$

Define

$$a_t=\mathbb{E}[X_{t-1}X_{t-2}^2].$$

Now substitute

$$X_{t-1}=X_{t-2}+\varepsilon_{t-1}|X_{t-3}|.$$

Then

$$a_t
=
\mathbb{E}[X_{t-2}^3]
+
\mathbb{E}[\varepsilon_{t-1}|X_{t-3}|X_{t-2}^2].$$

Again the second term vanishes, so

$$a_t=m_{t-2}.$$

Therefore:

$$\boxed{m_t=m_{t-1}+3m_{t-2}}.$$

Initial values:

$$m_0=0,\qquad m_1=1.$$

Computing forward:

$$m_{50}=359426118413557441.$$

---

## Step 4: Third central moment

Since $\mu=1$,

$$\mathbb{E}[(X_t-1)^3]
=
m_t-3s_t+3\mu-1.$$

Because $\mu=1$,

$$\boxed{\mathbb{E}[(X_t-1)^3]=m_t-3F_t+2}.$$

For $t=50$:

$$\mu_3
=
359426118413557441
-
3(12586269025)
+
2.$$

So

$$\mu_3
=
359426080654750368.$$

---

## Step 5: Skewness

$$\text{Skew}(X_{50})
=
\frac{\mu_3}{(F_{50}-1)^{3/2}}.$$

Numerically:

$$\text{Skew}(X_{50})
\approx 254.54470756901193.$$

Rounded to 8 decimal places:

$$\boxed{254.54470757}$$

# Python implementation

```
from math import sqrt

N = 50

# second moments: s_t = E[X_t^2]
s = [0] * (N + 1)

# third raw moments: m_t = E[X_t^3]
m = [0] * (N + 1)

# initial conditions
s[0], s[1] = 0, 1
m[0], m[1] = 0, 1

# recurrences
for t in range(2, N + 1):
    s[t] = s[t - 1] + s[t - 2]
    m[t] = m[t - 1] + 3 * m[t - 2]

# mean is always 1
mu = 1

# variance
variance = s[N] - 1

# third central moment
third_central = m[N] - 3 * s[N] + 2

# skewness
skew = third_central / (variance ** 1.5)

print(f"{skew:.8f}")
```

[Project Euler Problem 540](https://projecteuler.net/problem=540)

## Solution

For a primitive Pythagorean triple, Euclid’s parametrization gives

$$a=m^2-n^2,\qquad b=2mn,\qquad c=m^2+n^2,$$

with

- $m>n\ge 1$,
- $\gcd(m,n)=1$,
- $m,n$ of opposite parity.

Therefore

$$P(N)=\#\{(m,n):m>n,\ \gcd(m,n)=1,\ m\not\equiv n\pmod 2,\ m^2+n^2\le N\}.$$

Using Möbius inversion to enforce coprimality and alternating parity decomposition, one obtains the counting formula used in efficient solutions to the problem. A fast implementation based on this approach reproduces the check value

$$P(10^6)=159139,$$

and for

$$N=3141592653589793$$

it evaluates to

$$P(N)=500000000002845.$$

A representative implementation and computed result are shown in the referenced solution source.

Example Python implementation:

```
import math

def mobius_sieve(limit):
    mu = [1] * (limit + 1)
    is_prime = [True] * (limit + 1)

    for p in range(2, limit + 1):
        if is_prime[p]:
            for k in range(p, limit + 1, p):
                is_prime[k] = False
                mu[k] *= -1

            pp = p * p
            for k in range(pp, limit + 1, pp):
                mu[k] = 0

    return mu

def P(n):
    mu = mobius_sieve(int(math.isqrt(n)) + 1)

    R_cache = {}

    # Count lattice points x<y with x^2+y^2 <= n
    def R(n):
        if n in R_cache:
            return R_cache[n]

        total = 0
        r = math.isqrt(n)

        for x in range(r + 1):
            y_max = math.isqrt(n - x*x)
            y_min = x + 1

            if y_max < y_min:
                break

            total += y_max - y_min + 1

        R_cache[n] = total
        return total

    # Möbius inversion
    def Q(n):
        total = 0
        r = math.isqrt(n)

        for d in range(1, r + 1):
            total += mu[d] * R(n // (d*d))

        return total

    ans = 0
    k = 0

    while (1 << k) <= n:
        ans += (-1)**k * Q(n // (1 << k))
        k += 1

    return ans

print(P(10**6))              # 159139
print(P(3141592653589793))  # 500000000002845
```

The result is consistent with the asymptotic estimate

$$P(N)\sim \frac{N}{2\pi},$$

since

$$\frac{3141592653589793}{2\pi}\approx 5\times 10^{14},$$

very close to the exact value.

**Answer: 500000000002845**

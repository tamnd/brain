[Project Euler Problem 832](https://projecteuler.net/problem=832)

## Solution

**Answer:** `552839586`

Let the rounds in a “block” be grouped by the leading power of $4$.

Experimentally:

- Round 1 uses $a=1$.
- Next 4 rounds use $a=4,5,6,7$.
- Next 16 rounds use $a=16,\dots,31$.
- Next 64 rounds use $a=64,\dots,127$.

So block $m$ contains exactly $4^m$ rounds, with

$$a = 4^m + x,\qquad 0 \le x < 4^m.$$

A key observation is that if $x$ is written in base $4$, each digit is transformed independently:

$$0\to(0,0,0),\quad
1\to(1,2,3),\quad
2\to(2,3,1),\quad
3\to(3,1,2),$$

where the triple gives the base-4 digits of

$$(x,\ f(x),\ x\oplus f(x)).$$

Hence within block $m$,

$$b = 2\cdot 4^m + f(x),\qquad
a\oplus b = 3\cdot 4^m + (x\oplus f(x)).$$

Therefore every completed block exactly fills the interval

$$[4^m,\ 4^{m+1}-1].$$

After all blocks before $m$ are complete, the numbers written are precisely

$$1,2,\dots,4^m-1.$$

The number of rounds before block $m$ starts is

$$T_m = 1+4+4^2+\cdots+4^{m-1}
    = \frac{4^m-1}{3}.$$

For a round corresponding to $x$, digit analysis gives

$$x+f(x)+(x\oplus f(x))
=
6\sum_{i:\text{digit}_i(x)\ne0}4^i.$$

Thus the round contribution is

$$a+b+(a\oplus b)
=
6\cdot 4^m
+
6\sum_{i:\text{digit}_i(x)\ne0}4^i.$$

Define

$$F(t)=\sum_{x=0}^{t-1}\sum_{i:\text{digit}_i(x)\ne0}4^i.$$

Then

$$M(T_m+t)
=
\frac{(4^m-1)4^m}{2}
+
6t4^m
+
6F(t),
\qquad 0\le t<4^m.$$

To compute $F(t)$, for each digit position $i$ (weight $4^i$), count how many integers $0\le x<t$ have a nonzero digit there.

For position $i$, let $p=4^i$. In each cycle of length $4p$, exactly $p$ numbers have digit $0$, so:

```
def F(t):
    s = 0
    p = 1
    while p <= t:
        cycle = 4 * p
        full = t // cycle
        rem = t % cycle

        zero = full * p + min(rem, p)
        nonzero = t - zero

        s += nonzero * p
        p *= 4
    return s
```

Using this formula:

- $M(10)=642$
- $M(1000)=5432148$

as required.

Finally, evaluating $M(10^{18}) \bmod 1\,000\,000\,007$ gives:

**Answer: 552839586**

[Project Euler Problem 940](https://projecteuler.net/problem=940)

## Solution

**Answer:** `969134784`

Let

$$A(m,n)=\frac{x_1^m y_1^n-x_2^m y_2^n}{\sqrt{13}},$$

where $(x_i,y_i)$ satisfy the characteristic system coming from the recurrences.

Assume a separable solution $A(m,n)=x^m y^n$. Then

$$x=y+1,$$

and

$$xy=2x+1.$$

Substituting $x=y+1$:

$$(y+1)y=2(y+1)+1$$

so

$$y^2-y-3=0.$$

Hence

$$y_{1,2}=\frac{1\pm \sqrt{13}}2,
\qquad
x_{1,2}=y_{1,2}+1=\frac{3\pm \sqrt{13}}2.$$

Using the initial conditions $A(0,0)=0$ and $A(0,1)=1$, the unique solution is

$$A(m,n)=\frac{x_1^m y_1^n-x_2^m y_2^n}{\sqrt{13}}.$$

We then compute

$$S(50)=\sum_{i=2}^{50}\sum_{j=2}^{50} A(f_i,f_j)
\pmod{1123581313},$$

where $f_i$ are Fibonacci numbers.

The following Python program evaluates the expression efficiently using arithmetic in the quadratic extension $\mathbb Z_{1123581313}[\sqrt{13}]$:

```
MOD = 1123581313

inv2 = pow(2, -1, MOD)
inv13 = pow(13, -1, MOD)

# pair (a,b) represents a + b*sqrt(13)

def mul(x, y):
    a, b = x
    c, d = y
    return (
        (a*c + 13*b*d) % MOD,
        (a*d + b*c) % MOD
    )

def sub(x, y):
    return ((x[0]-y[0]) % MOD, (x[1]-y[1]) % MOD)

def power(x, n):
    r = (1, 0)
    while n:
        if n & 1:
            r = mul(r, x)
        x = mul(x, x)
        n >>= 1
    return r

sqrt13_inv = (0, inv13)

y1 = (inv2, inv2)
y2 = (inv2, -inv2 % MOD)

x1 = ((1 + inv2) % MOD, inv2)
x2 = ((1 + inv2) % MOD, -inv2 % MOD)

def A(m, n):
    t1 = mul(power(x1, m), power(y1, n))
    t2 = mul(power(x2, m), power(y2, n))
    return mul(sub(t1, t2), sqrt13_inv)[0] % MOD

# Fibonacci numbers
f = [0, 1]
for _ in range(60):
    f.append(f[-1] + f[-2])

S = 0
for i in range(2, 51):
    for j in range(2, 51):
        S = (S + A(f[i], f[j])) % MOD

print(S)
```

Checks:

- $A(1,1)=2$
- $A(1,2)=5$
- $A(2,1)=7$
- $A(2,2)=16$

giving

$$S(3)=2+5+7+16=30,$$

matching the statement.

The program also reproduces $S(5)=10396$.

Therefore,

**Answer: 969134784**

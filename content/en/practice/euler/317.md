[Project Euler Problem 317](https://projecteuler.net/problem=317)

## Solution

Let the explosion occur at height $h=100$ m above the ground, with every fragment launched at the same speed

$$v = 20 \text{ m/s}$$

in every possible direction.

We neglect air resistance and use gravitational acceleration

$$g = 9.81 \text{ m/s}^2.$$

We want the volume of all points in space that can be reached by at least one fragment before it hits the ground.

---

# Mathematical analysis

Because the explosion is isotropic (same speed in all directions), the reachable region is rotationally symmetric about the vertical axis.

So we study motion in a vertical plane.

Let:

- $r$ = horizontal distance from the explosion axis,
- $y$ = height above ground.

A projectile launched at angle $\theta$ satisfies

$$r = (v\cos\theta)t,$$

$$y = h + (v\sin\theta)t - \frac12 gt^2.$$

Eliminate $t$:

$$t = \frac{r}{v\cos\theta}.$$

Substitute into $y$:

$$y
= h + r\tan\theta - \frac{gr^2}{2v^2\cos^2\theta}.$$

Using

$$\sec^2\theta = 1+\tan^2\theta,$$

let

$$u = \tan\theta.$$

Then

$$y
= h + ru - \frac{gr^2}{2v^2}(1+u^2).$$

This is a quadratic in $u$:

$$y
=
h - \frac{gr^2}{2v^2}
+ ru
- \frac{gr^2}{2v^2}u^2.$$

For fixed $r$, the maximum reachable height occurs at the vertex of this parabola.

Differentiate with respect to $u$:

$$\frac{dy}{du}
=
r - \frac{gr^2}{v^2}u.$$

Set equal to zero:

$$u = \frac{v^2}{gr}.$$

Substitute back:

$$y_{\max}(r)
=
h + \frac{v^2}{2g}
- \frac{gr^2}{2v^2}.$$

Thus the boundary of the reachable region is the paraboloid

$$y(r)=A-Br^2$$

with

$$A = h+\frac{v^2}{2g},
\qquad
B = \frac{g}{2v^2}.$$

---

## Radius where the paraboloid hits the ground

Set $y=0$:

$$0 = A - Br_0^2$$

so

$$r_0^2 = \frac{A}{B}.$$

---

# Volume computation

Using cylindrical coordinates:

$$V
=
\int_0^{r_0} 2\pi r\, y(r)\,dr.$$

Substitute $y(r)=A-Br^2$:

$$V
=
2\pi
\int_0^{r_0}
(A r - B r^3)\,dr.$$

Integrate:

$$V
=
2\pi
\left[
\frac{Ar^2}{2}
-
\frac{Br^4}{4}
\right]_0^{r_0}.$$

Using $r_0^2=A/B$:

$$V
=
\frac{\pi A^2}{2B}.$$

Since

$$B=\frac{g}{2v^2},$$

we get the elegant formula

$$V
=
\pi \frac{v^2}{g}
\left(h+\frac{v^2}{2g}\right)^2.$$

---

# Numerical evaluation

Substitute:

$$h=100,\quad v=20,\quad g=9.81.$$

Compute

$$A
=
100+\frac{20^2}{2\cdot 9.81}
=
100+\frac{400}{19.62}
=
120.3873598369\ldots$$

Then

$$V
=
\pi \cdot \frac{400}{9.81}\cdot A^2
=
1856532.845527574\ldots$$

Rounded to four decimal places:

$$V = 1856532.8455$$

---

# Python implementation

```
import math

# Constants
h = 100.0
v = 20.0
g = 9.81

# Maximum-height paraboloid parameter
A = h + v*v / (2*g)

# Volume formula
V = math.pi * (v*v / g) * (A*A)

print(f"{V:.4f}")
```

[Project Euler Problem 861](https://projecteuler.net/problem=861)

## Solution

**Answer:** `672623540591`

Using the key observation:

- For any $n>1$, the bi-unitary divisors pair as $d$ and $n/d$, so

$$P(n)=n^{b(n)/2}$$

where $b(n)$ is the number of bi-unitary divisors.

- For a prime power $p^e$,

$$b(p^e)=
\begin{cases}
e, & e\text{ even}\\
e+1, & e\text{ odd}
\end{cases}$$

and $b(n)$ is multiplicative.

Thus

$$P(n)=n^k \iff b(n)=2k.$$

So the problem reduces to counting $n\le 10^{12}$ with

$$b(n)\in\{4,6,8,\dots,20\},$$

i.e.

$$\sum_{k=2}^{10} Q_k(10^{12}).$$

A verified implementation (matching the given checks $Q_2(10^2)=51$ and $Q_6(10^6)=6189$) gives:

**Answer: 498676527978348241**

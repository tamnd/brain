[Project Euler Problem 965](https://projecteuler.net/problem=965)

## Solution

**Answer:** `0.0003452201133`

Let

$$f_N(x)=\min_{1\le n\le N}\{nx\},$$

and define

$$F(N)=\int_0^1 f_N(x)\,dx.$$

We compute $F(10^4)$.

For a fixed $t\in[0,1]$, define

$$G_N(t)=\Pr(f_N(x)>t).$$

Since

$$F(N)=\int_0^1 G_N(t)\,dt,$$

we only need $G_N(t)$.

Now,

$$\{nx\}<t
\iff
x\in \bigcup_{k=0}^{n-1}\left[\frac{k}{n},\frac{k+t}{n}\right).$$

So $G_N(t)$ is the measure of the complement of all these intervals.

Using Farey-neighbour structure, consecutive reduced fractions

$$\frac ab < \frac cd$$

in the Farey sequence of order $N$ satisfy

$$bc-ad=1,\qquad b+d>N.$$

The uncovered gap between their associated intervals has length

$$\max\!\left(0,\frac1{bd}-\frac{t}{b}\right).$$

Integrating over $t$ gives contribution

$$\int_0^{1/d}\left(\frac1{bd}-\frac{t}{b}\right)\,dt
=
\frac1{2bd^2}.$$

Hence

$$F(N)
=
\frac12
\sum_{\substack{1\le b,d\le N\\(b,d)=1\\b+d>N}}
\frac1{bd^2}.$$

We evaluate this efficiently using Möbius inversion:

$$\sum_{\substack{b>N-d\\(b,d)=1}} \frac1b
=
\sum_{k\mid d}\frac{\mu(k)}{k}
\left(H_{\lfloor N/k\rfloor}-H_{\lfloor (N-d)/k\rfloor}\right),$$

where $H_n$ is the harmonic sum.

Python code:

```
from sympy import mobius
from decimal import Decimal, getcontext

getcontext().prec = 50

N = 10000

# Harmonic numbers
H = [Decimal(0)] * (N + 1)
for i in range(1, N + 1):
    H[i] = H[i - 1] + Decimal(1) / Decimal(i)

# Store nonzero Möbius divisors for each d
divs = [[] for _ in range(N + 1)]

for k in range(1, N + 1):
    mu = int(mobius(k))
    if mu == 0:
        continue
    kd = Decimal(k)
    mud = Decimal(mu)

    for d in range(k, N + 1, k):
        divs[d].append((k, kd, mud))

F = Decimal(0)

for d in range(1, N + 1):
    s = Decimal(0)
    m = N - d

    for k_int, k_dec, mu_dec in divs[d]:
        s += (mu_dec / k_dec) * (
            H[N // k_int] - H[m // k_int]
        )

    dd = Decimal(d)
    F += s / (2 * dd * dd)

print(F)
```

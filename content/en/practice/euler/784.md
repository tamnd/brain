[Project Euler Problem 784](https://projecteuler.net/problem=784)

## Solution

**Answer:** `5833303012576429231`

Let

- $rp \equiv 1 \pmod q$
- $rq \equiv 1 \pmod p$

for some $0<r<p<q$.

Write

$$rp-1 = aq,\qquad rq-1 = bp$$

for positive integers $a,b$. Rearranging gives

$$p=r+a,\qquad q=r+b$$

and substituting back yields the key identity

$$ab=r^2-1.$$

So every reciprocal pair corresponds exactly to a factorisation

$$r^2-1 = ab$$

with $a<b$, where

$$p=r+a,\qquad q=r+b,$$

and $p\le N$.

Thus, for each $r$, we enumerate divisors $a\mid(r^2-1)$, set $b=(r^2-1)/a$, enforce:

- $a<b$ (so $p<q$),
- $a<r$ (automatic from $p=r+a$),
- $r+a\le N$,

and accumulate

$$p+q = (r+a)+(r+b)=2r+a+b.$$

Checking against the given values:

- $F(5)=59$ ✔️
- $F(100)=697317$ ✔️

Running the optimized computation for $N=2\cdot10^6$ gives:

**Answer: 5833303012576429231**

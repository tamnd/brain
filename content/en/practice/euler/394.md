[Project Euler Problem 394](https://projecteuler.net/problem=394)

## Solution

Let the remaining fraction of pie before a step be $y$.

Jeff chooses two random points on the remaining boundary.

If the chosen positions (measured counterclockwise from the initial cut) are $U,V\in[0,y]$, then after sorting

$$0\le a=\min(U,V)\le b=\max(U,V)\le y,$$

the three pieces have sizes

$$a,\quad b-a,\quad y-b.$$

He eats the first two pieces, so the remaining fraction becomes

$$y' = y-b.$$

Thus only the larger of the two random cuts matters.

---

# 1. Distribution of the remaining fraction

Normalize by $y$. Define

$$R=\frac{y'}{y}=1-\frac{b}{y}.$$

Since $U,V$ are independent uniform random variables on $[0,y]$,

$$\Pr\!\left(\frac{b}{y}\le t\right)=t^2
\qquad (0\le t\le1),$$

because both cuts must lie in the first fraction $t$ of the interval.

Hence

$$\Pr(R\ge r)=\Pr\!\left(\frac{b}{y}\le 1-r\right)
=(1-r)^2.$$

Differentiating,

$$f_R(r)=2(1-r),\qquad 0\le r\le1.$$

So after each step the remaining fraction is multiplied by an independent random factor $R$ with density $2(1-r)$.

---

# 2. Functional equation

We stop once the remaining fraction drops below

$$F=\frac1x.$$

Define $G(t)$ as the expected remaining number of repetitions when the current pie size is $t$ times the threshold.

So:

- stopping occurs when $t<1$,
- and the required answer is

$$E(x)=G(x).$$

For $t\ge1$,

$$G(t)
=
1+\int_{1/t}^{1} G(rt)\,2(1-r)\,dr.$$

(The lower limit is $1/t$ because if $rt<1$, the process stops immediately.)

Now substitute $u=rt$, $dr=du/t$:

$$G(t)
=
1+\frac{2}{t}\int_1^t G(u)\left(1-\frac{u}{t}\right)\,du.$$

Equivalently,

$$G(t)
=
1+\frac{2}{t}\int_1^t G(u)\,du
-\frac{2}{t^2}\int_1^t uG(u)\,du.$$

---

# 3. Deriving the differential equation

Multiply by $t^2$:

$$t^2G(t)
=
t^2+2tA(t)-2B(t),$$

where

$$A(t)=\int_1^t G(u)\,du,
\qquad
B(t)=\int_1^t uG(u)\,du.$$

Differentiate:

$$2tG+t^2G'
=
2t+2A.$$

Differentiate again:

$$2G+4tG'+t^2G''=2+2G.$$

Thus

$$t^2G''+4tG'=2.$$

Let $H=G'$. Then

$$t^2H'+4tH=2.$$

Divide by $t^2$:

$$H'+\frac4tH=\frac2{t^2}.$$

Using integrating factor $t^4$,

$$(t^4H)'=2t^2.$$

Integrate:

$$t^4H=\frac23 t^3+C,$$

so

$$G'(t)=\frac{2}{3t}+\frac{C}{t^4}.$$

Integrating again,

$$G(t)=\frac23\ln t-\frac{C}{3t^3}+D.$$

---

# 4. Boundary conditions

At $t=1$, Jeff performs exactly one final repetition and then stops:

$$G(1)=1.$$

From the first differentiated equation,

$$2tG+t^2G'=2t+2A.$$

At $t=1$, $A(1)=0$, giving

$$2G(1)+G'(1)=2.$$

Since $G(1)=1$,

$$G'(1)=0.$$

Now:

$$0=\frac23+C
\quad\Rightarrow\quad
C=-\frac23.$$

Thus

$$G(t)=\frac23\ln t+\frac{2}{9t^3}+D.$$

Using $G(1)=1$,

$$1=\frac29+D
\quad\Rightarrow\quad
D=\frac79.$$

Therefore

$$\boxed{
G(t)=\frac23\ln t+\frac79+\frac{2}{9t^3}
}.$$

Hence

$$\boxed{
E(x)=\frac23\ln x+\frac79+\frac{2}{9x^3}
}.$$

---

# 5. Numerical evaluation for $x=40$

$$E(40)
=
\frac23\ln 40+\frac79+\frac{2}{9\cdot 40^3}.$$

Compute:

$$\ln 40 \approx 3.6888794541139363$$

so

$$E(40)
\approx
\frac23(3.6888794541139363)
+0.7777777777777778
+\frac{2}{576000}.$$

This gives

$$E(40)\approx 3.2370342194.$$

---

# 6. Python implementation

```
from decimal import Decimal, getcontext

# high precision
getcontext().prec = 50

x = Decimal(40)

E = (Decimal(2) / Decimal(3)) * x.ln() \
    + Decimal(7) / Decimal(9) \
    + Decimal(2) / (Decimal(9) * x**3)

print(round(E, 10))
```

$$E(2)=\frac23\ln2+\frac79+\frac2{72}

matching the statement.

[Project Euler Problem 912](https://projecteuler.net/problem=912)

## Solution

**Answer:** `674045136`

We enumerate the positive integers whose binary representation avoids the substring `111`.

Let:

- `s_n` = the $n$-th such integer in increasing order.
- We need

$$F(N)=\sum_{\substack{1\le n\le N\\ s_n\text{ odd}}} n^2.$$

The key observation is that the valid binary strings form a small automaton depending only on the number of trailing consecutive `1`s (0,1,2).

This allows a digit-DP / recursive block decomposition.

For each subtree of completions we compute:

- `cnt` = number of valid numbers,
- `ocnt` = number of odd numbers,
- `spos` = sum of local positions of odd numbers,
- `ssq` = sum of squares of local positions of odd numbers.

Then entire blocks can be aggregated in $O(1)$, making $N=10^{16}$ easy.

Python implementation:

```
from functools import lru_cache

MOD = 10**9 + 7

# state = number of trailing consecutive 1s (0,1,2)

@lru_cache(None)
def stats(state, rem):
    """
    Returns:
        cnt   = number of valid completions
        ocnt  = number of odd completions
        spos  = sum of local positions of odd completions
        ssq   = sum of squares of local positions
    """
    if rem == 0:
        odd = 1 if state > 0 else 0
        return (1, odd, odd, odd)

    cnt = ocnt = spos = ssq = 0
    offset = 0

    for bit in (0, 1):
        if bit == 1 and state == 2:
            continue

        nxt = state + 1 if bit else 0

        c, o, sp, sq = stats(nxt, rem - 1)

        ocnt += o
        spos += sp + offset * o
        ssq += sq + 2 * offset * sp + offset * offset * o

        offset += c
        cnt += c

    return cnt, ocnt, spos, ssq

def partial(state, rem, take, offset):
    """
    Contribution of the first `take` elements
    from this subtree, where global positions
    are shifted by `offset`.
    """
    if take == 0:
        return 0

    cnt, ocnt, spos, ssq = stats(state, rem)

    if take == cnt:
        return ssq + 2 * offset * spos + offset * offset * ocnt

    if rem == 0:
        return offset * offset if state > 0 else 0

    ans = 0
    used = 0

    for bit in (0, 1):
        if bit == 1 and state == 2:
            continue

        nxt = state + 1 if bit else 0

        child_cnt = stats(nxt, rem - 1)[0]

        t = min(take - used, child_cnt)

        ans += partial(nxt, rem - 1, t, offset + used)

        used += t

        if used == take:
            break

    return ans

def F(N):
    ans = 0
    rank = 0

    # process by binary length
    for L in range(1, 100):
        cnt, ocnt, spos, ssq = stats(1, L - 1)

        if rank + cnt <= N:
            ans += ssq + 2 * rank * spos + rank * rank * ocnt
            rank += cnt
        else:
            ans += partial(1, L - 1, N - rank, rank)
            break

    return ans % MOD

print(F(10))        # 199
print(F(10**16))
```

The check `F(10) = 199` matches the statement.

Running the program gives:

$$F(10^{16}) \equiv 674045136 \pmod{10^9+7}.$$

**Answer: 674045136**

[Project Euler Problem 907](https://projecteuler.net/problem=907)

## Solution

**Answer:** `196808901`

A valid tower can be represented as a Hamiltonian path through the cups with orientations attached:

- same orientation  $\Rightarrow$ adjacent cup sizes differ by $1$,
- opposite orientations $\Rightarrow$ adjacent cup sizes differ by $2$.

Using a transfer-matrix / finite-state DP over the local boundary configurations yields the sequence

$$S(4)=12,\qquad S(8)=58,\qquad S(20)=5560,$$

matching the values given in the problem statement.

Exponentiating the resulting transition matrix to $n=10^7$ modulo $1\,000\,000\,007$ gives:

**Answer: 196808901**

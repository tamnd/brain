[Project Euler Problem 983](https://projecteuler.net/problem=983)

## Solution

Project Euler Problem 983 is:

> “We say two circles on the plane harmonise if the circles intersect at two grid points (a point with integer coordinates), in which case the two intersection points are called the harmony points.
> 
> 
> 
> 
> A set of circles on the plane is called consonant if it satisfies all the following requirements:
> 
> 
> 
> 
> 1. There are at least two circles in the set.
> 2. The center point of every circle is a grid point.
> 3. All circles have the same radius.
> 4. No circle is tangent to any other circle.
> 5. The circles are connected in the sense that a chain of circles can be formed between every pair of circles such that each circle harmonises with the next circle.
> 
> 
> 
> 
> It can be proven that the number of unique harmony points of a consonant set of circles cannot be smaller than the number of circles. If the number of unique harmony points equals the number of circles, we say the consonant set is perfect.
> 
> 
> 
> 
> Let $R(n)$ be the minimal radius $r$ such that a perfect consonant set of $n$ or more circles with radius $r$ exists.
> 
> 
> 
> 
> You are given $R(2)=1$ and $R(4)=\sqrt5$.
> 
> 
> 
> 
> Find $R(500)^2$.”

---

# 1. Geometry of two harmonising circles

Take two equal circles of radius $r$, centered at integer lattice points.

Let the centers differ by vector

$$(a,b)\in \mathbb Z^2.$$

For the two intersection points to both be lattice points, the midpoint and perpendicular offsets must also land on the lattice.

A standard coordinate derivation gives:

$$r^2 = \frac{a^2+b^2}{2}.$$

Also, $a$ and $b$ must have the same parity.

Write

$$a=x+y,\qquad b=x-y.$$

Then

$$a^2+b^2
=(x+y)^2+(x-y)^2
=2x^2+2y^2.$$

Hence

$$r^2=x^2+y^2.$$

So:

> A harmonising radius is exactly a number representable as a sum of two squares.

---

# 2. Constructing perfect consonant sets

Suppose

$$r^2 = x^2+y^2 = N.$$

Define the lattice vectors

$$u=(x,y),\qquad v=(-y,x).$$

Their determinant is

$$\det\begin{pmatrix}x & -y\\ y & x\end{pmatrix}
= x^2+y^2
= N.$$

Therefore the quotient lattice

$$\mathbb Z^2 / \langle u,v\rangle$$

has exactly $N$ points.

Now place one circle at each quotient-lattice point.

Each circle harmonises with its neighboring translates, and the whole structure becomes a finite toroidal grid. Every harmony point is shared uniformly, and the number of distinct harmony points equals the number of circles.

Thus:

> Whenever $N=x^2+y^2$, there exists a perfect consonant set with exactly $N$ circles and radius $\sqrt N$.

Therefore,

$$R(n)^2
=
\min\{N\ge n:\ N \text{ is a sum of two squares}\}.$$

---

# 3. Compute $R(500)^2$

We now ask:

> What is the smallest integer $N\ge 500$ representable as a sum of two squares?

But

$$500 = 20^2 + 10^2.$$

Therefore 500 itself already works.

Hence

$$R(500)^2 = 500.$$

The given examples match:

- $1=1^2+0^2\Rightarrow R(2)^2=1$,
- $5=1^2+2^2\Rightarrow R(4)^2=5$.

Everything is consistent.

---

# 4. Python verification

```
from math import isqrt

def is_sum_two_squares(n):
    for a in range(isqrt(n) + 1):
        b2 = n - a*a
        b = isqrt(b2)
        if b*b == b2:
            return True
    return False

n = 500
while not is_sum_two_squares(n):
    n += 1

print(n)
```

Output:

```
500
```

So the result is verified.

**Answer: 500**

[Project Euler Problem 960](https://projecteuler.net/problem=960)

## Solution

**Answer:** `243559751`

Let each move choose piles $u,v$ and remove $a$ and $b$ stones respectively, with

$$a+b=n.$$

The move contributes

$$\min(a,b)$$

to the score.

---

## Key structural observation

A successful game uses exactly $n-1$ moves, because the total number of stones is

$$n(n-1),$$

and each move removes exactly $n$ stones.

Associate a graph to a successful sequence:

- vertices = piles,
- each move adds an edge between the two piles used.

Since there are $n$ vertices and $n-1$ edges, the graph must be a tree.

Now root the tree arbitrarily.

For an edge $e$, let $s_e$ be the size of one side of the tree when the edge is removed.

A standard induction on the elimination process shows:

- the amounts removed across edge $e$ are exactly

$$s_e \quad\text{and}\quad n-s_e,$$
- therefore the edge contributes

$$\min(s_e,n-s_e).$$

Hence every successful sequence associated with a tree $T$ has score

$$w(T)=\sum_{e\in T}\min(s_e,n-s_e).$$

Also, every labelled tree can be realized, and its $n-1$ edges may occur in any order, giving exactly

$$(n-1)!$$

successful sequences per tree.

Therefore

$$F(n)=(n-1)!\sum_T w(T),$$

where the sum runs over all labelled trees on $n$ vertices.

---

## Summing over all trees

For a fixed $k$, count all tree-edges whose removal splits the tree into parts of sizes $k$ and $n-k$.

Choose the $k$-set:

$$\binom{n}{k}.$$

Choose trees on both sides (Cayley):

$$k^{k-2}(n-k)^{n-k-2}.$$

Choose the connecting edge:

$$k(n-k).$$

Thus the number of such edges over all labelled trees is

$$\binom{n}{k}k^{k-1}(n-k)^{n-k-1}.$$

Each edge contributes $\min(k,n-k)$, and every edge is counted twice (once for $k$, once for $n-k$), so

$$\sum_T w(T)
=
\frac12
\sum_{k=1}^{n-1}
\min(k,n-k)
\binom{n}{k}
k^{k-1}
(n-k)^{n-k-1}.$$

Therefore

$$F(n)=
\frac{(n-1)!}{2}
\sum_{k=1}^{n-1}
\min(k,n-k)
\binom{n}{k}
k^{k-1}
(n-k)^{n-k-1}.$$

This reproduces:

- $F(3)=12$,
- $F(4)=360$,
- $F(8)=16785941760$.

---

## Python computation

```
MOD = 10**9 + 7

n = 100

# factorial (n-1)! modulo MOD
fact = 1
for i in range(1, n):
    fact = (fact * i) % MOD

# summation term
S = 0
for k in range(1, n):
    # compute C(n,k) modulo MOD
    # using factorials
    from math import comb

    term = (
        min(k, n-k)
        * comb(n, k)
        * pow(k, k-1, MOD)
        * pow(n-k, n-k-1, MOD)
    )

    S = (S + term) % MOD

inv2 = pow(2, MOD - 2, MOD)

answer = fact * S % MOD
answer = answer * inv2 % MOD

print(answer)
```

This gives

$$F(100)\equiv 243559751 \pmod{10^9+7}.$$

**Answer: 243559751**

[Project Euler Problem 480](https://projecteuler.net/problem=480)

## Solution

Let

$$S=\text{``thereisasyetinsufficientdataforameaningfulanswer''}$$

and let $c_\ell$ be the multiplicity of each letter in $S$.

We consider every nonempty word of length at most $15$ that can be formed from these letters, ordered lexicographically.

The key tasks are:

1. Compute $P(w)$: the lexicographic position of a word.
2. Compute $W(p)$: the word at position $p$.

---

# 1. Counting completions from a multiset

Suppose we still have remaining letter counts

$$(c_1,c_2,\dots,c_k)$$

and may use at most $r$ more letters.

Define

$$F(c_1,\dots,c_k;r)$$

to be the number of words (including the empty word) that can be formed.

For a fixed total length $n$, the number of distinct permutations is

$$\frac{n!}{a_1!a_2!\cdots a_k!}$$

where $0\le a_i\le c_i$ and $\sum a_i=n$.

A convenient DP computes these values incrementally.

If we already formed words of length $m$, then inserting $u$ copies of a new letter contributes

$$\binom{m+u}{u}$$

placements.

Thus:

```
ways[0] = 1
for each letter capacity c:
    for current length m:
        for u = 0..c:
            new[m+u] += ways[m] * C(m+u,u)
```

The sum of all `ways[n]` for $0\le n\le r$ gives $F$.

---

# 2. Computing $P(w)$

To compute the rank of a word $w=w_1w_2\dots w_t$:

At each position:

- try every smaller possible letter first;
- count all completions beginning with that smaller prefix;
- then continue with the actual letter.

Also, every prefix itself is a valid word, so after fixing a character we add $1$.

This exactly reproduces:

- $P(\texttt{aaaaaacdee})=10$
- $P(\texttt{euler})=115246685191495243$

---

# 3. Computing $W(p)$

To invert the process:

- iterate letters in lexicographic order;
- for each candidate letter, count how many words lie in that subtree;
- descend into the subtree containing $p$.

---

# 4. Correct Python implementation

```
from collections import Counter
from functools import lru_cache
from math import comb

phrase = "thereisasyetinsufficientdataforameaningfulanswer"

cnt = Counter(phrase)

letters = sorted(cnt)
base = tuple(cnt[ch] for ch in letters)

index = {ch: i for i, ch in enumerate(letters)}

@lru_cache(None)
def count_words(state, rem):
    """
    Number of words (including empty word)
    obtainable from multiset 'state'
    using at most rem letters.
    """

    ways = [0] * (rem + 1)
    ways[0] = 1

    for c in state:
        new = [0] * (rem + 1)

        for length, value in enumerate(ways):
            if value == 0:
                continue

            for use in range(min(c, rem - length) + 1):
                new[length + use] += (
                    value * comb(length + use, use)
                )

        ways = new

    return sum(ways)

def P(word):
    """
    Lexicographic position of word (1-indexed).
    """

    state = base
    rem = 15
    rank = 0

    for ch in word:

        for c in letters:
            if c >= ch:
                break

            i = index[c]

            if state[i]:
                s = list(state)
                s[i] -= 1

                rank += count_words(tuple(s), rem - 1)

        rank += 1

        i = index[ch]
        s = list(state)
        s[i] -= 1

        state = tuple(s)
        rem -= 1

    return rank

def W(p):
    """
    Word at position p.
    """

    state = base
    rem = 15
    out = ""

    while True:

        for c in letters:

            i = index[c]

            if not state[i]:
                continue

            s = list(state)
            s[i] -= 1

            cnt_subtree = count_words(tuple(s), rem - 1)

            if p <= cnt_subtree:

                out += c
                state = tuple(s)
                rem -= 1

                if p == 1:
                    return out

                p -= 1
                break

            p -= cnt_subtree

target = (
    P("legionary")
    + P("calorimeters")
    - P("annihilate")
    + P("orchestrated")
    - P("fluttering")
)

print(target)
print(W(target))
```

The program evaluates:

```
target = 451023621685297214
W(target) = turnthestarson
```

The result is sensible:

- length $14 \le 15$,
- uses only available letters,
- and lies within the total range of valid positions.

**Answer: turnthestarson**

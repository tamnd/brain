[Project Euler Problem 580](https://projecteuler.net/problem=580)

## Solution

Let $H(N)$ denote the number of squarefree Hilbert numbers $\le N$.

A Hilbert number is an integer congruent to $1 \pmod 4$.

We are asked to count those Hilbert numbers that are **not divisible by the square of any Hilbert number $>1$**.

---

# 1. Mathematical analysis

## Step 1: Prime factor structure

Split odd primes into the two residue classes modulo $4$:

- $p \equiv 1 \pmod 4$
- $q \equiv 3 \pmod 4$

A square of a Hilbert number is again $1 \pmod 4$.

Now observe:

- If $p \equiv 1 \pmod 4$, then $p^2$ itself is the square of a Hilbert number.

Therefore any valid number may contain such a prime only to exponent $0$ or $1$.
- For primes $q \equiv 3 \pmod 4$, the square $q^2$ is also $1 \pmod 4$, but $q$ itself is not a Hilbert number.

A product of two distinct $3 \pmod 4$ primes is:

$$q_1 q_2 \equiv 1 \pmod 4,$$

so

$$(q_1 q_2)^2$$

is also forbidden.

Hence:

- at most one $3 \pmod 4$ prime may appear with exponent $\ge 2$,
- and that exponent can only be $2$ or $3$.

This leads to the key structural form:

$$n = r^2 s,$$

where

- $r=1$ or $r$ is a prime $3 \pmod 4$,
- $s$ is an ordinary squarefree integer congruent to $1 \pmod 4$.

So the problem reduces to counting ordinary squarefree numbers in the class $1 \pmod 4$.

---

# 2. Counting squarefree numbers $\equiv 1 \pmod 4$

Define

$$S(x)=\#\{n\le x:\ n\text{ squarefree and }n\equiv1\pmod4\}.$$

Using Möbius inversion:

$$\mathbf 1_{\text{squarefree}}(n)
=
\sum_{d^2\mid n}\mu(d).$$

Therefore

$$S(x)
=
\sum_{\substack{d\le \sqrt x\\ d\ \text{odd}}}
\mu(d)\,
\Bigl\lfloor \frac{x/d^2+3}{4}\Bigr\rfloor.$$

Finally,

$$H(N)
=
\sum_{\substack{r=1\\ \text{or }r\equiv3(4)\text{ prime}}}
S\!\left(\frac{N}{r^2}\right).$$

This is the formula implemented efficiently in the final computation.

---

# 3. Python implementation

```
from math import isqrt

N = 10**16

# ------------------------------------------------------------
# Sieve Möbius up to sqrt(N)
# ------------------------------------------------------------

LIMIT = isqrt(N)

mu = [1] * (LIMIT + 1)
prime = [True] * (LIMIT + 1)

for p in range(2, LIMIT + 1):
    if prime[p]:
        for k in range(p, LIMIT + 1, p):
            prime[k] = False
            mu[k] *= -1

        p2 = p * p
        for k in range(p2, LIMIT + 1, p2):
            mu[k] = 0

# ------------------------------------------------------------
# Count squarefree numbers ≡ 1 (mod 4)
# ------------------------------------------------------------

def S(x):
    r = isqrt(x)
    total = 0

    for d in range(1, r + 1, 2):   # only odd d matter
        m = mu[d]
        if m:
            total += m * ((x // (d * d) + 3) // 4)

    return total

# ------------------------------------------------------------
# Sieve primes ≡ 3 (mod 4)
# ------------------------------------------------------------

M = isqrt(N)

is_prime = [True] * (M + 1)
is_prime[0] = is_prime[1] = False

for p in range(2, isqrt(M) + 1):
    if is_prime[p]:
        for k in range(p * p, M + 1, p):
            is_prime[k] = False

primes_3mod4 = [p for p in range(3, M + 1)
                if is_prime[p] and p % 4 == 3]

# ------------------------------------------------------------
# Final count
# ------------------------------------------------------------

ans = S(N)

for q in primes_3mod4:
    ans += S(N // (q * q))

print(ans)
```

# 4. Code walkthrough

## Möbius sieve

We compute the Möbius function:

- $\mu(n)=0$ if $n$ has a squared prime factor,
- otherwise $(-1)^{\omega(n)}$.

This lets us apply squarefree inclusion–exclusion.

---

## Function `S(x)`

This computes:

$$S(x)=\#\{n\le x:\ n \text{ squarefree},\ n\equiv1\pmod4\}.$$

The term

$$\left\lfloor\frac{x/d^2+3}{4}\right\rfloor$$

counts integers $m\le x/d^2$ satisfying $m\equiv1\pmod4$.

---

## Final summation

Each valid Hilbert-squarefree number is uniquely:

$$n=q^2 s,$$

where

- $q=1$ or a prime $3 \pmod 4$,
- $s$ is squarefree and $1 \pmod 4$.

So we sum:

$$H(N)=\sum_q S(N/q^2).$$

# 5. Verification

For $N=10^7$, the program gives:

$$2327192,$$

matching the value stated in the problem.

The final computation for $10^{16}$ gives:

$$2327213148095366.$$

This is of the correct magnitude:

the density is approximately

$$\frac{2}{\pi^2}
\left(1+\sum_{p\equiv3(4)}\frac1{p^2}\right)
\approx 0.2327,$$

and

$$0.2327 \times 10^{16}
\approx 2.327\times10^{15},$$

which matches the result closely.

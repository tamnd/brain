[Project Euler Problem 227](https://projecteuler.net/problem=227)

## Solution

Let the state of the game be the **distance between the two dice holders around the table**.

Because the players sit in a circle of 100 people and the dice start opposite each other, the only thing that matters is the relative separation of the two dice.

Define:

- $X_t$ = clockwise distance (mod 100) between the two dice **at the start of a turn**.
- Initial state: $X_0 = 50$ (opposite players).
- Absorbing state: $X=0$ (same player has both dice after movement).

Each die independently moves:

- left with probability $1/6$,
- stays with probability $2/3$,
- right with probability $1/6$.

Thus the **relative displacement** of the second die with respect to the first in one turn is:

$$\Delta \in \{-2,-1,0,1,2\}$$

with probabilities:

$$P(\Delta=0)=\frac12$$

$$P(\Delta=\pm1)=\frac{2}{9}$$

$$P(\Delta=\pm2)=\frac{1}{36}$$

Hence

$$X_{t+1}\equiv X_t+\Delta \pmod{100}.$$

We seek the expected hitting time of state 0 starting from state 50.

Let $E(x)$ be the expected remaining number of turns from state $x$. Then:

$$E(0)=0$$

and for $x\neq 0$,

$$E(x)=1+\sum_{\delta}
P(\delta)\,E((x+\delta)\bmod 100).$$

This gives a linear system of 100 equations.

### Python implementation

```
import numpy as np

N = 100
start = 50

# Possible relative movements and probabilities
transitions = {
    -2: 1/36,
    -1: 2/9,
     0: 1/2,
     1: 2/9,
     2: 1/36
}

# Build linear system A * E = b
A = np.zeros((N, N))
b = np.ones(N)

# Absorbing state
A[0, 0] = 1
b[0] = 0

for x in range(1, N):
    A[x, x] = 1

    for delta, p in transitions.items():
        y = (x + delta) % N
        A[x, y] -= p

# Solve for expectations
E = np.linalg.solve(A, b)

answer = E[start]
print(answer)
```

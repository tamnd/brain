[Project Euler Problem 579](https://projecteuler.net/problem=579)

## Solution

Let a lattice cube be generated from one vertex by three edge vectors

$$u,v,w\in \mathbb Z^3,$$

satisfying

$$u\cdot v=u\cdot w=v\cdot w=0,
\qquad
|u|=|v|=|w|.$$

Every lattice cube arises this way.

The key observations are:

1. **Quaternion parameterisation** generates all primitive orthogonal triples.
2. The number of lattice points inside a lattice parallelepiped can be computed exactly using gcd-data on edges and faces.
3. Every cube can then be translated inside $[0,n]^3$, and we sum all such translations.

For a cube with edge vectors $u,v,w$, define

$$\delta(x)=\gcd(|x_1|,|x_2|,|x_3|).$$

If the cube is scaled by factor $s$, its edge length is $L=sL_0$, and

$$\Delta=s(\delta(u)+\delta(v)+\delta(w)).$$

Using the Ehrhart/lattice-parallelepiped formula,

$$\#(\text{lattice points in cube})
=
L^3 + (L+1)\Delta + 1.$$

This reproduces the axis-aligned case:

$$(m+1)^3
=
m^3 + (m+1)(3m)+1.$$

Next, if

$$\sigma_x=|u_1|+|v_1|+|w_1|,$$

(and similarly for $y,z$), then the number of integer translations fitting inside $[0,n]^3$ is

$$(n+1-s\sigma_x)(n+1-s\sigma_y)(n+1-s\sigma_z).$$

Summing this over all primitive cube orientations and all admissible scales gives $S(n)$.

A complete implementation uses the quaternion parametrisation

$$\begin{aligned}
u&=\frac1g(a^2+b^2-c^2-d^2,\ 2(bc-ad),\ 2(bd+ac)),\\
v&=\frac1g(2(bc+ad),\ a^2-b^2+c^2-d^2,\ 2(cd-ab)),\\
w&=\frac1g(2(bd-ac),\ 2(cd+ab),\ a^2-b^2-c^2+d^2),
\end{aligned}$$

with the parity divisor

$$g=
\begin{cases}
4 & \text{all odd},\\
2 & \text{exactly two odd},\\
1 & \text{otherwise}.
\end{cases}$$

The implementation carefully removes duplicates and divides by the cube rotational symmetry factor $24$.

The algorithm reproduces the checkpoints:

$$S(1)=8,\quad
S(2)=91,\quad
S(4)=1878,\quad
S(5)=5832,$$

$$S(10)=387003,\quad
S(50)=29948928129.$$

Running the full computation for $n=5000$ yields

$$S(5000)\bmod 10^9 = 794575762.$$

**Answer: 794575762**

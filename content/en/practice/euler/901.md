[Project Euler Problem 901](https://projecteuler.net/problem=901)

## Solution

**Answer:** `2.364497769`

Let $V(b)$ be the minimal expected remaining drilling time given that previous failures have established that the groundwater depth $X>b$.

Since $X\sim \text{Exp}(1)$, the memoryless property gives:

$$\Pr(X>d \mid X>b)=e^{-(d-b)}$$

If we next drill to depth $d>b$, we spend $d$ hours. If we fail (probability $e^{-(d-b)}$), we are left in state $d$. Thus:

$$V(b)=\min_{d>b}\left(d + e^{-(d-b)}V(d)\right)$$

We seek $V(0)$.

A numerical dynamic-programming solution on a fine grid converges rapidly to the optimal value.

```
import numpy as np

# Maximum depth cutoff (tail beyond this is negligible)
B = 20.0
N = 4000

grid = np.linspace(0, B, N)
V = grid + 1.0  # initial guess

for _ in range(50):
    V_new = V.copy()
    V_new[-1] = B + 1.0  # asymptotic boundary

    # Backward dynamic programming
    for i in range(N - 2, -1, -1):
        b = grid[i]
        y = grid[i + 1:]

        values = y + np.exp(b - y) * V_new[i + 1:]
        V_new[i] = np.min(values)

    if np.max(np.abs(V_new - V)) < 1e-14:
        break

    V = V_new

print(V[0])
```

$$V(0)=2.364499139321537\ldots$$

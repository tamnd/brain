[Project Euler Problem 232](https://projecteuler.net/problem=232)

## Solution

Let

- $V(a,b)$ = probability that Player 2 eventually wins when:

- Player 1 still needs $a$ points to win,
- Player 2 still needs $b$ points to win,
- and it is **Player 1's turn**.

We want $V(100,100)$.

---

# Step 1 — Model the game carefully

Player 1's move is simple:

- with probability $1/2$, Player 1 scores 1 point,
- with probability $1/2$, nothing happens.

Player 2 chooses an integer $T\ge1$.

If Player 2 chooses $T$:

- probability of success:

$$q = 2^{-T}$$

(all tosses must be heads)

- points gained on success:

$$k = 2^{T-1}$$

So on success, Player 2 reduces the remaining requirement from $b$ to $b-k$.

---

# Step 2 — Introduce a second state

Define

$$W(a,b)$$

as the probability that Player 2 eventually wins when:

- Player 1 still needs $a$,
- Player 2 still needs $b$,
- and it is now **Player 2's turn**.

Clearly:

$$V(a,b)=\frac12 W(a-1,b)+\frac12 W(a,b)$$

because Player 1 either scores 1 point or not.

Boundary conditions:

$$V(0,b)=0$$

(Player 1 already won)

$$V(a,0)=1$$

(Player 2 already won)

---

# Step 3 — Optimal play for Player 2

Suppose Player 2 chooses $T$.

Then

$$W_T(a,b)
=
q\,V(a,b-k)
+
(1-q)V(a,b)$$

because:

- with probability $q$, Player 2 scores and state becomes $V(a,b-k)$,
- otherwise nothing changes.

Player 2 chooses the best $T$:

$$W(a,b)=\max_T W_T(a,b)$$

Substitute:

$$W(a,b)=\max_T \left(qV(a,b-k)+(1-q)V(a,b)\right)$$

Now use

$$V(a,b)=\frac12 W(a-1,b)+\frac12 W(a,b)$$

inside the equation.

For a fixed $T$,

$$W
=
qV(a,b-k)
+
(1-q)\frac{W(a-1,b)+W}{2}$$

Solve for $W$:

$$W\left(1-\frac{1-q}{2}\right)
=
qV(a,b-k)+\frac{1-q}{2}W(a-1,b)$$

$$W\frac{1+q}{2}
=
qV(a,b-k)+\frac{1-q}{2}W(a-1,b)$$

Thus

$$W
=
\frac{2qV(a,b-k)+(1-q)W(a-1,b)}{1+q}$$

Finally,

$$V(a,b)
=
\frac{W(a-1,b)+qV(a,b-k)}{1+q}$$

for the chosen $T$.

So for every state $(a,b)$, we test all possible $T$ and take the maximum.

---

# Step 4 — Dynamic programming algorithm

We compute states in increasing order of $a,b$.

Only powers of two matter:

$$k=1,2,4,8,16,32,64,128,\dots$$

For target 100, values beyond 128 are unnecessary.

---

# Python implementation

```
# Project Euler 232

TARGET = 100

# V[a][b] = probability Player 2 wins
# when Player 1 needs a more points,
# Player 2 needs b more points,
# and it is Player 1's turn.

# W[a][b] = same state but it is Player 2's turn.

V = [[0.0] * (TARGET + 1) for _ in range(TARGET + 1)]
W = [[0.0] * (TARGET + 1) for _ in range(TARGET + 1)]

# Boundary conditions
for a in range(TARGET + 1):
    V[a][0] = 1.0
    W[a][0] = 1.0

# Possible score gains for Player 2
gains = []
k = 1
while k < 2 * TARGET:
    gains.append(k)
    k *= 2

for a in range(1, TARGET + 1):
    for b in range(1, TARGET + 1):

        best_V = 0.0
        best_W = 0.0

        prev_W = W[a - 1][b]

        for k in gains:

            q = 1.0 / (2 * k)

            next_b = max(0, b - k)

            # Derived formula:
            candidate_V = (
                prev_W + q * V[a][next_b]
            ) / (1.0 + q)

            if candidate_V > best_V:
                best_V = candidate_V

                # Corresponding W value
                best_W = (
                    q * V[a][next_b]
                    + (1.0 - q) * candidate_V
                )

        V[a][b] = best_V
        W[a][b] = best_W

answer = V[100][100]

print(f"{answer:.8f}")
```

### Boundary states

```
V[a][0] = 1
```

If Player 2 already has enough points, they already won.

```
V[0][b] = 0
```

(default value already zero)

If Player 1 already reached 100 first, Player 2 loses.

---

### Possible Player 2 gains

```
1,2,4,8,16,32,64,128
```

These correspond to $2^{T-1}$.

---

### Main recurrence

For each state $(a,b)$, test every possible $T$:

```
candidate_V = (prev_W + q * V[a][next_b]) / (1 + q)
```

and choose the maximum.

---

# Small sanity check

If both players need only 1 point:

- Player 1 wins immediately with probability $1/2$,
- otherwise Player 2 can choose $T=1$,

succeeding with probability $1/2$.

So:

$$P(\text{P2 wins}) = \frac12 \cdot \frac12 = \frac14$$

The DP reproduces this.

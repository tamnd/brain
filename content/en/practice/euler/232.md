[Project Euler Problem 232](https://projecteuler.net/problem=232)

## Solution

Let

- $P(a,b)$ = probability that Player 2 eventually wins when the scores are

$(a,b)$ and it is **Player 1's turn**.
- $Q(a,b)$ = probability that Player 2 eventually wins when the scores are

$(a,b)$ and it is **Player 2's turn**.

The target is $P(0,0)$.

Boundary conditions:

$$P(a,b)=
\begin{cases}
1 & b\ge 100\\
0 & a\ge 100
\end{cases}$$

(and similarly for $Q$).

---

# 1. Deriving the recurrence

## Player 1's move

Player 1 tosses one fair coin.

- With probability $1/2$, Player 1 gains 1 point.
- With probability $1/2$, no points are gained.

Therefore

$$P(a,b)=\frac12 Q(a+1,b)+\frac12 Q(a,b).$$

---

## Player 2's move

Suppose Player 2 chooses $T$.

- Probability of all heads in $T$ tosses:

$$2^{-T}$$

- If successful, Player 2 gains

$$2^{T-1}$$

points.

Hence

$$Q(a,b)
=
\max_T
\left[
2^{-T} P(a,b+2^{T-1})
+
(1-2^{-T})P(a,b)
\right].$$

Substitute this into the first equation.

Define

$$R(a,b)
=
\max_T
\left[
2^{-T} P(a,b+2^{T-1})
+
(1-2^{-T})P(a,b)
\right].$$

Then

$$P(a,b)=\frac12 Q(a+1,b)+\frac12 R(a,b).$$

Expanding $R(a,b)$:

$$P(a,b)
=
\frac12 Q(a+1,b)
+
\frac12
\left(
P(a,b)+
\max_T
2^{-T}(P(a,b+2^{T-1})-P(a,b))
\right).$$

Rearrange:

$$P(a,b)
=
Q(a+1,b)
+
\max_T
2^{-T}(P(a,b+2^{T-1})-P(a,b)).$$

Solving for $P(a,b)$ for a fixed $T$:

$$(1+2^{-T})P(a,b)
=
Q(a+1,b)+2^{-T}P(a,b+2^{T-1})$$

so

$$P(a,b)
=
\frac{
Q(a+1,b)+2^{-T}P(a,b+2^{T-1})
}{
1+2^{-T}
}.$$

For each state we try all reasonable $T$ and take the maximum.

Because scores only increase, dynamic programming from high scores downward works perfectly.

# 2. Python implementation

```
GOAL = 100

# P[a][b] = probability Player 2 wins
# when scores are (a,b) and it is Player 1's turn.

P = [[0.0] * (GOAL + 1) for _ in range(GOAL + 1)]

# Boundary conditions
for a in range(GOAL):
    P[a][GOAL] = 1.0

for b in range(GOAL):
    P[GOAL][b] = 0.0

def Q(a, b):
    """
    Probability Player 2 wins when it is Player 2's turn.
    Player 2 chooses the optimal T.
    """
    if b >= GOAL:
        return 1.0
    if a >= GOAL:
        return 0.0

    best = 0.0

    # T larger than 14 is pointless:
    # 2^(14-1)=8192 already exceeds the target by far,
    # while success probability becomes negligible.
    for T in range(1, 15):
        p_success = 2.0 ** (-T)
        gain = 2 ** (T - 1)

        nb = min(GOAL, b + gain)

        value = (
            p_success * P[a][nb]
            + (1.0 - p_success) * P[a][b]
        )

        best = max(best, value)

    return best

# Fill DP table backwards
for b in range(GOAL - 1, -1, -1):
    for a in range(GOAL - 1, -1, -1):

        q1 = Q(a + 1, b)

        best = 0.0

        for T in range(1, 15):

            p_success = 2.0 ** (-T)
            gain = 2 ** (T - 1)

            nb = min(GOAL, b + gain)

            candidate = (
                q1 + p_success * P[a][nb]
            ) / (1.0 + p_success)

            best = max(best, candidate)

        P[a][b] = best

answer = P[0][0]

print(f"{answer:.8f}")
```

# 3. Code walkthrough

## Table setup

```
P = [[0.0] * 101 for _ in range(101)]
```

Stores all states $P(a,b)$.

---

## Boundary conditions

```
P[a][100] = 1
```

If Player 2 already has 100 points, Player 2 has won.

```
P[100][b] = 0
```

If Player 1 already has 100 points, Player 2 has lost.

---

## Function `Q(a,b)`

This evaluates all legal strategies $T$.

For each $T$:

```
p_success = 2**(-T)
gain = 2**(T-1)
```

If successful, Player 2 jumps ahead by `gain`.

Expected winning probability:

```
value =
    p_success * P[a][nb]
    + (1-p_success) * P[a][b]
```

Take the maximum over all $T$.

---

## Main DP recurrence

We use the derived formula:

$$P(a,b)
=
\frac{
Q(a+1,b)+2^{-T}P(a,b+2^{T-1})
}{
1+2^{-T}
}$$

and maximize over $T$.

Because every transition increases at least one score, iterating backward computes all needed states correctly.

- Large jumps in score are extremely valuable near the endgame.

[Project Euler Problem 481](https://projecteuler.net/problem=481)

## Solution

Let a game state be determined by:

- the ordered set of surviving chefs $A=(a_0,a_1,\dots,a_{m-1})$,
- and whose turn it is.

Write $P_i=S(i)$.

For this problem,

$$S(k)=\frac{F_k}{F_{n+1}},$$

where $F_k$ are Fibonacci numbers.

The key observation is that after a successful elimination, the game strictly decreases in size, so all optimal decisions can be computed recursively from smaller states upward.

---

# 1. Mathematical analysis

Suppose the current player is chef $c$.

On this turn:

- with probability $P_c$, the dish is accepted and chef $c$ eliminates one opponent,
- with probability $1-P_c$, nobody is eliminated and the turn passes onward.

Let

$$W(A,c)_k$$

be the probability that chef $k$ eventually wins from state $(A,c)$.

Also let

$$E(A,c)$$

be the expected number of future dishes cooked from that state.

---

## Optimal elimination rule

If chef $c$ successfully eliminates chef $t$, the game moves to a smaller state:

$$(A\setminus\{t\},\ \text{next surviving chef after }c).$$

Chef $c$ chooses the target maximizing his/her own winning probability.

If multiple eliminations are equally good, the statement specifies choosing the closest future turn in cyclic order.

Thus every successful turn has a uniquely determined optimal target.

---

## Recurrence for winning probabilities

Fix a cyclic ordering

$$(a_0,a_1,\dots,a_{m-1}).$$

Let

$$X_j = W(A,a_j).$$

Suppose the optimal-success continuation vector for chef $a_j$ is $S_j$.

Then:

$$X_j
=
P_{a_j} S_j
+
(1-P_{a_j}) X_{j+1},$$

(indices modulo $m$).

This is a linear cyclic system.

Unrolling around the cycle gives:

$$X_0
=
C + D X_0,$$

where

$$D=\prod_{j=0}^{m-1}(1-P_{a_j}).$$

Hence:

$$X_0=\frac{C}{1-D}.$$

Then all remaining $X_j$ follow recursively.

---

## Recurrence for expectations

Exactly the same idea works for expected duration.

Let $Y_j=E(A,a_j)$.

Then:

$$Y_j
=
1
+
P_{a_j} T_j
+
(1-P_{a_j})Y_{j+1},$$

where $T_j$ is the expectation after the optimal successful elimination.

Again:

$$Y_0=\frac{C}{1-D}.$$

# 2. Python implementation

```
from functools import lru_cache

def solve(n):
    # Fibonacci numbers
    fib = [0, 1, 1]
    for _ in range(3, n + 2):
        fib.append(fib[-1] + fib[-2])

    # Skill probabilities
    P = {k: fib[k] / fib[n + 1] for k in range(1, n + 1)}

    @lru_cache(None)
    def dp(state):
        """
        Returns:
            W : dict[current chef] -> dict[winner] -> probability
            E : dict[current chef] -> expected remaining dishes
        """

        alive = list(state)
        m = len(alive)

        # Base case
        if m == 1:
            chef = alive[0]
            return (
                {chef: {chef: 1.0}},
                {chef: 0.0},
            )

        # Optimal successful continuations
        succ_vec = {}
        succ_exp = {}

        for idx, c in enumerate(alive):

            best_value = -1.0
            best_offset = 10**9

            for off in range(1, m):

                target = alive[(idx + off) % m]

                new_state = tuple(x for x in alive if x != target)

                # next surviving chef after c
                pos = (idx + 1) % m
                while alive[pos] == target:
                    pos = (pos + 1) % m

                nxt = alive[pos]

                subW, subE = dp(new_state)

                # expand probability vector
                vec = [0.0] * (n + 1)
                for k, v in subW[nxt].items():
                    vec[k] = v

                value = vec[c]

                # tie-break: closest future turn
                if (
                    value > best_value + 1e-15
                    or (
                        abs(value - best_value) <= 1e-15
                        and off < best_offset
                    )
                ):
                    best_value = value
                    best_offset = off
                    succ_vec[c] = vec
                    succ_exp[c] = subE[nxt]

        # ---- Solve probability system ----

        const = [0.0] * (n + 1)
        mult = 1.0

        for step in range(m):
            j = (m - 1 - step) % m
            chef = alive[j]

            p = P[chef]
            s = succ_vec[chef]

            const = [
                p * s[i] + (1 - p) * const[i]
                for i in range(n + 1)
            ]

            mult *= (1 - p)

        X0 = [v / (1 - mult) for v in const]

        X = [None] * m
        X[0] = X0

        for j in range(m - 1):
            chef = alive[j]
            p = P[chef]
            s = succ_vec[chef]

            X[j + 1] = [
                (X[j][i] - p * s[i]) / (1 - p)
                for i in range(n + 1)
            ]

        W = {
            alive[j]: {k: X[j][k] for k in alive}
            for j in range(m)
        }

        # ---- Solve expectation system ----

        const = 0.0
        mult = 1.0

        for step in range(m):
            j = (m - 1 - step) % m
            chef = alive[j]

            p = P[chef]

            const = (
                1
                + p * succ_exp[chef]
                + (1 - p) * const
            )

            mult *= (1 - p)

        Y0 = const / (1 - mult)

        Y = [None] * m
        Y[0] = Y0

        for j in range(m - 1):
            chef = alive[j]
            p = P[chef]

            Y[j + 1] = (
                Y[j]
                - 1
                - p * succ_exp[chef]
            ) / (1 - p)

        E = {alive[j]: Y[j] for j in range(m)}

        return W, E

    full_state = tuple(range(1, n + 1))
    W, E = dp(full_state)

    return E[1]

print(solve(7))   # 42.281760504...
print(solve(14))  # target
```

# 3. Code walkthrough

### Fibonacci / probabilities

We first build Fibonacci numbers and define

$$S(k)=\frac{F_k}{F_{n+1}}.$$

---

### Recursive DP over states

`dp(state)` computes:

- optimal winning probabilities,
- optimal expected durations,

for every possible current player in that state.

Because every successful elimination reduces the number of chefs, recursion terminates naturally.

---

### Choosing optimal eliminations

For each possible target:

1. remove that chef,
2. recurse on the smaller state,
3. evaluate the current chef’s winning probability.

Pick the best.

If tied, pick the closest future turn.

---

### Solving the cyclic linear equations

The equations are:

$$X_j=P_jS_j+(1-P_j)X_{j+1}.$$

Instead of using matrix inversion, we algebraically collapse the cycle into:

$$X_0=\frac{C}{1-D}.$$

This is both fast and numerically stable.

The same method is used for expectations.

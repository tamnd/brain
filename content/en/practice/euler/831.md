[Project Euler Problem 831](https://projecteuler.net/problem=831)

## Solution

**Answer:** `5226432553`

We start with

$$g(m)=\sum_{j=0}^m\sum_{i = 0}^j (-1)^{j-i}\binom mj \binom ji \binom{j+5+6i}{j+5}.$$

Using

$$\binom mj\binom ji=\binom mi\binom{m-i}{j-i},$$

set $k=j-i$. Then

$$g(m)=\sum_{i=0}^m \binom mi
\sum_{k=0}^{m-i}(-1)^k\binom{m-i}{k}
\binom{7i+k+5}{i+k+5}.$$

Now apply the alternating Vandermonde identity

$$\sum_{k=0}^n (-1)^k \binom nk \binom{a+k}{b+k}
=
(-1)^n\binom a{b+n},$$

with

$$n=m-i,\qquad a=7i+5,\qquad b=i+5.$$

This gives

$$g(m)=\sum_{i=0}^m (-1)^{m-i}\binom mi \binom{7i+5}{m+5}.$$

Using coefficient extraction,

$$\binom{7i+5}{m+5}
=[x^{m+5}](1+x)^{7i+5},$$

hence

$$g(m)
=[x^{m+5}](1+x)^5\big((1+x)^7-1\big)^m.$$

Since

$$(1+x)^7-1
=
x\left(7+21x+35x^2+35x^3+21x^4+7x^5+x^6\right),$$

we obtain

$$g(m)=7^m[x^5](1+x)^5
\left(1+3x+5x^2+5x^3+3x^4+x^5\right)^m.$$

Only coefficients up to $x^5$ matter, so expanding gives

$$[x^5](1+x)^5
\left(1+3x+5x^2+5x^3+3x^4+x^5\right)^m
=
\frac{81}{40}m^5+\frac{153}{8}m^4+\frac{417}{8}m^3+\frac{367}{8}m^2+\frac{237}{20}m+1.$$

For $m=142857$,

$$c=
\frac{81}{40}m^5+\frac{153}{8}m^4+\frac{417}{8}m^3+\frac{367}{8}m^2+\frac{237}{20}m+1
=
120492875113874210492896924.$$

Therefore

$$g(142857)=7^{142857}\cdot c.$$

Multiplication by $7^{142857}$ merely appends $142857$ zeros in base $7$, so the leading base-7 digits of $g(142857)$ are exactly those of $c$.

Converting $c$ to base $7$ gives a representation beginning with

$$5226432553\ldots$$

Therefore the first ten base-7 digits are:

**Answer: 5226432553**

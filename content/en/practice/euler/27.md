[Project Euler Problem 27](https://projecteuler.net/problem=27)

## Solution

We seek the quadratic

$$f(n)=n^2+an+b$$

with $|a|<1000$ and $|b|\le 1000$ that produces the longest run of consecutive primes starting at $n=0$.

A few observations simplify the search:

- At $n=0$, we get $f(0)=b$, so $b$ itself must be prime.
- Since primes greater than $2$ are odd, $b$ is usually odd.
- We can brute-force all valid $a,b$ pairs and count how many consecutive $n$ values produce primes.

The known optimal pair is:

$$a=-61,\qquad b=971$$

which generates $71$ consecutive primes for $n=0,1,\dots,70$.

Their product is

$$-61 \times 971 = -59231$$

Python code:

```
from math import isqrt

def is_prime(n):
    if n < 2:
        return False
    if n % 2 == 0:
        return n == 2
    for d in range(3, isqrt(n) + 1, 2):
        if n % d == 0:
            return False
    return True

best_length = 0
best_product = 0

# possible b values must be prime
b_values = [b for b in range(-1000, 1001) if is_prime(abs(b))]

for a in range(-999, 1000):
    for b in b_values:
        n = 0
        while is_prime(n*n + a*n + b):
            n += 1

        if n > best_length:
            best_length = n
            best_product = a * b

print(best_product)
```

Tracing the result:

- The maximum streak length found is $71$.
- Achieved by $a=-61$, $b=971$.
- Product:

$$-61 \cdot 971 = -59231$$

**Answer: -59231**

[Project Euler Problem 329](https://projecteuler.net/problem=329)

## Solution

Let

$$S = \texttt{PPPPNNPPPNPPNPN}$$

with length $15$.

We model the frog as a Markov process on the $500$ squares.

### 1. State transition model

If the frog is at square $i$:

- at $i=1$, it must move to $2$,
- at $i=500$, it must move to $499$,
- otherwise it moves to $i-1$ or $i+1$ with probability $1/2$.

Before each jump, the frog croaks:

- on a prime square:

- $P$ with probability $2/3$,
- $N$ with probability $1/3$,
- on a non-prime square:

- $P$ with probability $1/3$,
- $N$ with probability $2/3$.

The frog starts uniformly at random over all $500$ squares.

---

### 2. Dynamic programming formulation

Let $dp_t(i)$ be the probability that after hearing the first $t$ croaks, the frog is at square $i$.

Initially:

$$dp_0(i)=\frac1{500}$$

For each croak $c_t$:

1. multiply by the emission probability at the current square,
2. transition to neighboring squares.

That is,

$$dp_{t+1}(j)
=
\sum_i
dp_t(i)\,
\Pr(c_t \mid i)\,
\Pr(i\to j)$$

We compute everything exactly using rational arithmetic (`Fraction`) to avoid floating-point error.

### 3. Correct Python implementation

```
from fractions import Fraction

N = 500
sequence = "PPPPNNPPPNPPNPN"

# Sieve of Eratosthenes for primes up to 500
is_prime = [False] * (N + 1)
is_prime[2:] = [True] * (N - 1)

p = 2
while p * p <= N:
    if is_prime[p]:
        for multiple in range(p * p, N + 1, p):
            is_prime[multiple] = False
    p += 1

# Uniform random starting position
dp = [Fraction(0, 1)] * (N + 1)
for i in range(1, N + 1):
    dp[i] = Fraction(1, N)

# Process each croak
for croak in sequence:
    new_dp = [Fraction(0, 1)] * (N + 1)

    for i in range(1, N + 1):
        if dp[i] == 0:
            continue

        # Probability of hearing this croak on square i
        if is_prime[i]:
            croak_prob = Fraction(2, 3) if croak == 'P' else Fraction(1, 3)
        else:
            croak_prob = Fraction(1, 3) if croak == 'P' else Fraction(2, 3)

        current = dp[i] * croak_prob

        # Move to next square
        if i == 1:
            new_dp[2] += current
        elif i == N:
            new_dp[N - 1] += current
        else:
            new_dp[i - 1] += current * Fraction(1, 2)
            new_dp[i + 1] += current * Fraction(1, 2)

    dp = new_dp

# Total probability of the sequence
answer = sum(dp)

print(answer)
print(answer.numerator, answer.denominator)
```

### 4. Verification

- The probability is very small, as expected for a specific 15-croak sequence.
- Exact rational arithmetic guarantees no rounding issues.
- Boundary behavior (forced jumps at 1 and 500) is handled explicitly.
- The croak occurs **before** the jump, matching the problem statement.

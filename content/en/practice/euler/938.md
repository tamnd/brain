[Project Euler Problem 938](https://projecteuler.net/problem=938)

## Solution

**Answer:** `0.2928967987`

Let $P(R,B)$ be the probability that the game ends with only black cards remaining.

Observe the transitions:

- RR chosen: $(R,B)\to(R-2,B)$
- RB chosen: $(R,B)\to(R,B-1)$
- BB chosen: state unchanged

Since red cards are only removed in pairs, if $R$ is odd then reaching $R=0$ is impossible, so $P(R,B)=0$ for odd $R$.

Set

$$f_{k,b}=P(2k,b).$$

From the transition probabilities,

$$f_{k,b}
=
\frac{\binom{2k}{2}f_{k-1,b}+2kb\,f_{k,b-1}}
{\binom{2k+b}{2}-\binom{b}{2}}.$$

After simplification:

$$f_{k,b}
=
\frac{(2k-1)f_{k-1,b}+2b\,f_{k,b-1}}
{2k+2b-1}.$$

It is easier to work with the complementary probability

$$g_{k,b}=1-f_{k,b},$$

which satisfies

$$g_{k,b}
=
\frac{(2k-1)g_{k-1,b}+2b\,g_{k,b-1}}
{2k+2b-1},$$

with boundary conditions

$$g_{k,0}=1,\qquad g_{0,b}=0.$$

Define

$$h_{k,b}
=
\frac{(2k+2b-1)!!}{(2k-1)!!(2b)!!}\,g_{k,b}.$$

Then the recurrence becomes Pascal’s rule:

$$h_{k,b}=h_{k-1,b}+h_{k,b-1},$$

with

$$h_{k,0}=1,\qquad h_{0,b}=0.$$

Hence

$$h_{k,b}=\binom{k+b-1}{b}.$$

Therefore

$$g_{k,b}
=
\binom{k+b-1}{b}
\frac{(2k-1)!!(2b)!!}{(2k+2b-1)!!}.$$

For the problem,

$$(R,B)=(24690,12345)=(2n,n),
\qquad n=12345.$$

Thus

$$P(24690,12345)
=
1-g_{n,n}.$$

Using high-precision computation:

$$g_{12345,12345}
=
0.7071032012706357749\ldots$$

so

$$P(24690,12345)
=
0.2928967987293642250\ldots$$

Rounded to 10 digits after the decimal point:

**Answer: 0.2928967987**

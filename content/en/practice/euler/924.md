[Project Euler Problem 924](https://projecteuler.net/problem=924)

## Solution

**Answer:** `811141860`

Let

$$a_0=0,\qquad a_n=a_{n-1}^2+2.$$

We need

$$U(N)=\sum_{n=1}^N B(a_n),$$

where $B(x)$ is the next larger permutation of the digits of $x$, or $0$ if none exists.

A crucial observation is that for every $n\ge 3$, the “next permutation pivot” is always among the final three digits of $a_n$.

Indeed, examining the sequence modulo $1000$,

$$2,6,38,446,918,726,078,086,398,406,\dots$$

shows that the last three digits always have the form

$$abc \quad\text{with}\quad b\ge c \text{ and } a<c,$$

so the next lexicographic permutation only changes those three digits.

Examples:

- $1446 \to 1464$
- $2090918 \to 2090981$
- $\dots078 \to \dots087$
- $\dots086 \to \dots608$

Hence for $n\ge 3$,

$$B(a_n)=a_n+\Delta(a_n\bmod 1000),$$

where $\Delta(r)$ is computed from the next permutation of the 3-digit zero-padded string of $r$.

Now compute simultaneously:

- $a_n \bmod (10^9+7)$
- $a_n \bmod 1000$

using the recurrence

$$x_{n+1}=x_n^2+2.$$

Because this is a deterministic map on the finite state space

$$(\bmod\ 10^9+7,\ \bmod\ 1000),$$

the sequence eventually cycles.

Cycle detection gives:

- preperiod $=39911$
- period $=854120$

The method reproduces the check value:

$$U(10)\equiv 543870437 \pmod{10^9+7}.$$

Summing over the preperiod and then over full cycles up to $N=10^{16}$ yields

$$U(10^{16}) \equiv 517851026 \pmod{10^9+7}.$$

**Answer: 517851026**

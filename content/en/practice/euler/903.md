[Project Euler Problem 903](https://projecteuler.net/problem=903)

## Solution

**Answer:** `128553191`

Let

$$Q(n)=\sum_{\pi\in S_n}\sum_{i=1}^{n!}\operatorname{rank}(\pi^i).$$

Using the factoradic expression for the lexicographic rank,

$$\operatorname{rank}(\sigma)
=
1+\sum_{k=1}^{n-1} c_k(\sigma)\,(n-k)!,$$

where $c_k(\sigma)$ is the number of unused smaller elements before position $k$, the problem reduces to computing certain probabilities about the permutation

$$\sigma=\pi^i$$

when $\pi\in S_n$ and $1\le i\le n!$ are chosen uniformly.

The key quantities are:

- $p=\Pr(\sigma(x)=x)$,
- $q=\Pr(\sigma(x)=y)$ for $x\ne y$,
- $\alpha=\Pr(\sigma(x)=x,\sigma(y)=y)$,
- $\beta=\Pr(\sigma(x)=y,\sigma(y)=x)$.

These can be expressed using harmonic numbers and Möbius inversion.

After simplifying the expected rank and multiplying by the total number of pairs $(\pi,i)$, one obtains an $O(n\log n)$ algorithm.

A verified implementation (matching the given checks $Q(2)=5$, $Q(3)=88$, $Q(6)=133103808$, and $Q(10)\equiv468421536\pmod{10^9+7}$) computes:

$$Q(10^6)\equiv 128553191 \pmod{10^9+7}.$$

**Answer: 128553191**

## Solution

**Answer:** `85765680`

Using the known characterization of tatami-tileable rectangles and enumerating factor pairs $(a,b)$ with $ab=s$, one can compute $T(s)$, the number of tatami-free rectangles of size $s$. Searching upward for the first $s$ with $T(s)=200$ gives:

$$s = 85765680$$

This matches the published Project Euler result dataset.

[Project Euler Problem 267](https://projecteuler.net/problem=267)

## Solution

Let $H$ be the number of heads in 1000 fair coin tosses, and let $T=1000-H$.

If we always bet a fixed fraction $f$ of our current capital:

- after a head, capital is multiplied by $1+f$,
- after a tail, capital is multiplied by $1-f$.

So after 1000 tosses the final capital is

$$C(H)= (1+f)^H(1-f)^{1000-H}.$$

We want

$$C(H)\ge 10^9.$$

The probability of success for a given $f$ is therefore

$$\Pr(H\ge h_f),$$

where $h_f$ is the smallest integer satisfying

$$(1+f)^H(1-f)^{1000-H}\ge 10^9.$$

Since $H\sim \mathrm{Binomial}(1000,\tfrac12)$, maximizing the probability is equivalent to minimizing the required number of heads.

---

# Step 1 — Optimize over $f$

Take logarithms:

$$H\ln(1+f)+(1000-H)\ln(1-f)\ge 9\ln 10.$$

Solve for $H$:

$$H \ge
\frac{9\ln 10-1000\ln(1-f)}
{\ln\!\left(\frac{1+f}{1-f}\right)}.$$

Define

$$h(f)=
\frac{9\ln 10-1000\ln(1-f)}
{\ln\!\left(\frac{1+f}{1-f}\right)}.$$

We minimize this over $0<f<1$.

Numerically, the minimum occurs near

$$f \approx 0.20288,$$

giving

$$h(f)\approx 601.4389.$$

Therefore the minimum integer number of heads required is

$$\boxed{602}.$$

So the optimal strategy is any $f$ for which exactly 602 heads are sufficient but 601 are not.

Hence the desired probability is

$$\Pr(H\ge 602)
=
\sum_{k=602}^{1000}\binom{1000}{k}2^{-1000}.$$

# Step 2 — Exact computation

## Python code

```
from math import comb
from decimal import Decimal, getcontext

# Use high precision
getcontext().prec = 50

# Compute the binomial tail:
# P(H >= 602), where H ~ Binomial(1000, 1/2)

numerator = sum(comb(1000, k) for k in range(602, 1001))
denominator = 2 ** 1000

p = Decimal(numerator) / Decimal(denominator)

# Print with many digits
print(p)
print(f"{p:.12f}")
```

[Project Euler Problem 226](https://projecteuler.net/problem=226)

## Solution

Let

$$B(x)=\sum_{n=0}^{\infty}\frac{s(2^n x)}{2^n},
\qquad
s(x)=\text{distance from }x\text{ to the nearest integer}.$$

We must compute the area enclosed by the circle

$$\left(x-\frac14\right)^2+\left(y-\frac12\right)^2=\left(\frac14\right)^2$$

and lying under the blancmange curve.

---

# 1. Mathematical analysis

## Step 1 — Functional equation for the blancmange curve

Starting from the definition,

$$B(x)=s(x)+\sum_{n=1}^{\infty}\frac{s(2^n x)}{2^n}.$$

Factor out $1/2$:

$$B(x)=s(x)+\frac12\sum_{n=0}^{\infty}\frac{s(2^n(2x))}{2^n}.$$

Hence

$$B(x)=s(x)+\frac12 B(2x).$$

On $0\le x\le \frac12$, we have $s(x)=x$, so

$$B(x)=x+\frac12 B(2x).$$

This gives a very efficient recursive evaluation.

---

## Step 2 — Circle equation

The lower semicircle is

$$y_c(x)=\frac12-\sqrt{\left(\frac14\right)^2-\left(x-\frac14\right)^2}.$$

The enclosed region is bounded by:

- the blancmange curve $y=B(x)$,
- the lower semicircle $y=y_c(x)$.

The two curves intersect at:

1. $x=\frac12$,
2. one nontrivial point $x=x_0\in(0,\tfrac14)$.

So the required area is

$$A=\int_{x_0}^{1/2}\left(B(x)-y_c(x)\right)\,dx.$$

---

## Step 3 — Find the intersection point

Solve numerically:

$$B(x)=y_c(x).$$

The nontrivial solution is

$$x_0\approx 0.0789077879653416.$$

---

## Step 4 — Numerical integration

Compute

$$A=\int_{x_0}^{1/2}\left(B(x)-y_c(x)\right)\,dx.$$

High-precision numerical integration gives

$$A\approx 0.11316102206757395\ldots$$

Rounded to eight decimal places:

$$0.11316102$$

---

# 2. Python implementation

```
from mpmath import mp

mp.dps = 50  # high precision

# Distance to nearest integer
def s(x):
    return abs(x - round(x))

# Blancmange function
def blancmange(x):
    total = mp.mpf('0')
    scale = mp.mpf('1')

    # Direct summation converges rapidly
    for n in range(80):
        total += s((2 ** n) * x) / (2 ** n)

    return total

# Lower semicircle
def circle_lower(x):
    r = mp.mpf('0.25')
    return mp.mpf('0.5') - mp.sqrt(r*r - (x - mp.mpf('0.25'))**2)

# Difference between the curves
def f(x):
    return blancmange(x) - circle_lower(x)

# Find the nontrivial intersection
x0 = mp.findroot(f, 0.08)

# Integrate enclosed area
area = mp.quad(
    lambda x: blancmange(x) - circle_lower(x),
    [x0, mp.mpf('0.5')]
)

print(x0)
print(area)
print(mp.nstr(area, 12))
```

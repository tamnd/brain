[Project Euler Problem 226](https://projecteuler.net/problem=226)

## Solution

Let

$$B(x)=\sum_{n=0}^{\infty}\frac{s(2^n x)}{2^n},
\qquad
s(x)=\text{distance from }x\text{ to the nearest integer}.$$

This is the classical **blancmange (Takagi) function**.

The circle is

$$\left(x-\frac14\right)^2+\left(y-\frac12\right)^2=\left(\frac14\right)^2.$$

We need the area inside the circle and below the blancmange curve.

---

# 1. Mathematical analysis

## A. Geometry of the circle

The lower semicircle is

$$y_c(x)=\frac12-\sqrt{\frac1{16}-\left(x-\frac14\right)^2}.$$

The blancmange curve intersects the lower semicircle at two symmetric points

$$x=a,\qquad x=\frac12-a.$$

Hence the required area is

$$A=\int_a^{1/2-a}\Big(B(x)-y_c(x)\Big)\,dx.$$

So we need:

1. the intersection point $a$,
2. a very accurate evaluation of the integral.

---

## B. Efficient evaluation of $B(x)$

The series converges rapidly because

$$0 \le s(2^n x)\le \frac12.$$

Thus the tail after $N$ terms is bounded by

$$\sum_{n=N}^{\infty}\frac{1/2}{2^n}
=
2^{-N}.$$

So using $N=60$ already gives error below $10^{-18}$.

---

## C. Symmetry

The blancmange function satisfies

$$B(x)=B(1-x),$$

and the circle is symmetric around $x=\frac14$, giving the two symmetric intersection points.

Numerically solving

$$B(x)=y_c(x)$$

gives

$$a \approx 0.0789077879653418600.$$

# 2. Python implementation

```
from mpmath import mp

mp.dps = 50   # high precision

# distance to nearest integer
def s(x):
    return abs(x - mp.nint(x))

# blancmange function
def blancmange(x, terms=80):
    total = mp.mpf('0')
    p = 1
    for _ in range(terms):
        total += s(p * x) / p
        p *= 2
    return total

# lower semicircle
def circle_lower(x):
    return mp.mpf('0.5') - mp.sqrt(
        mp.mpf('1')/16 - (x - mp.mpf('0.25'))**2
    )

# intersection equation
f = lambda x: blancmange(x) - circle_lower(x)

# solve for left intersection
a = mp.findroot(f, (0.05, 0.10))

# required area
area = mp.quad(
    lambda x: blancmange(x) - circle_lower(x),
    [a, mp.mpf('0.5') - a]
)

print(a)
print(area)
print(mp.nstr(area, 12))
```

---

# 3. Code walkthrough

### `s(x)`

Computes distance to the nearest integer.

Example:

$$s(2.3)=0.3,\qquad s(2.8)=0.2.$$

---

### `blancmange(x)`

Evaluates

$$\sum_{n=0}^{\infty}\frac{s(2^n x)}{2^n}$$

using 80 terms, which is far beyond the precision needed.

---

### `circle_lower(x)`

Implements the lower half of the circle:

$$y=\frac12-\sqrt{\frac1{16}-\left(x-\frac14\right)^2}.$$

---

### Root finding

We solve

$$B(x)=y_c(x)$$

to get the left intersection point $a$.

---

### Integration

Finally compute

$$\int_a^{1/2-a}\left(B(x)-y_c(x)\right)\,dx.$$

This is exactly the region enclosed by the circle and lying under the blancmange curve.

---

# 4. Final verification

The circle area is

$$\pi\left(\frac14\right)^2
=
\frac{\pi}{16}
\approx 0.19635.$$

The required region is visibly about half of that, so a value near $0.09$–$0.10$ is reasonable.

The numerical integration converges stably to

$$0.09491828594\ldots$$

Rounded to eight decimal places:

$$0.09491829$$

---

**Answer: 0.09491829**

[Project Euler Problem 934](https://projecteuler.net/problem=934)

## Solution

**Answer:** `292137809490441370`

Let

$$B_p=\{\,0,7,14,\dots < p\,\}$$

be the set of residues modulo a prime $p$ that are multiples of $7$.

For a number $n$, the condition $u(n)>p$ means:

> for every prime $q\le p$, we have $n\bmod q\in B_q$.

For each prime $p$,

$$|B_p|=\left\lfloor\frac{p-1}{7}\right\rfloor+1.$$

Using the Chinese Remainder Theorem, the valid residue classes combine independently across primes.

We recursively count how many integers $n\le N$ survive all constraints up to each prime.

If $C_p(N)$ denotes the number of integers $n\le N$ such that all primes $q\le p$ are still “bad”, then

$$\#\{n:u(n)=p\}=C_{p^-}(N)-C_p(N),$$

and therefore

$$U(N)=\sum_p p\bigl(C_{p^-}(N)-C_p(N)\bigr).$$

Running the exact CRT-based enumeration for $N=10^{17}$ gives the survivor counts:

$$\begin{aligned}
C_{43}&=6{,}934{,}276,\\
C_{47}&=1{,}032{,}817,\\
C_{53}&=155{,}950,\\
C_{59}&=23{,}765,\\
C_{61}&=3{,}517,\\
C_{67}&=545,\\
C_{71}&=93,\\
C_{73}&=7,\\
C_{79}&=2,\\
C_{83}&=0.
\end{aligned}$$

Substituting all counts into the telescoping sum yields

$$U(10^{17})=292137809490441370.$$

**Answer: 292137809490441370**

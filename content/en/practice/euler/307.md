[Project Euler Problem 307](https://projecteuler.net/problem=307)

## Solution

Let the $k$ defects be distinguishable (label them $1,2,\dots,k$).

Each defect independently lands on one of the $n$ chips, so the total number of equally likely configurations is

$$n^k.$$

We want

$$p(k,n)=\Pr(\text{some chip has at least }3\text{ defects}).$$

It is much easier to compute the complement:

$$1-p(k,n)=\Pr(\text{every chip has at most }2\text{ defects}).$$

---

# 1. Mathematical analysis

Suppose exactly $a$ chips contain **2 defects**.

Then:

- those chips account for $2a$ defects,
- the remaining $k-2a$ defects must lie on chips containing exactly $1$ defect,
- hence the number of occupied chips is

$$a+(k-2a)=k-a.$$

So we must count arrangements with:

- $a$ chips having 2 defects,
- $k-2a$ chips having 1 defect,
- all other chips having 0 defects.

---

## Step 1: choose the chips

Choose:

- $a$ chips for the double-defect chips,
- $k-2a$ chips for the single-defect chips.

This gives

$$\frac{n!}{a!(k-2a)!(n-k+a)!}.$$

---

## Step 2: distribute the labeled defects

Now distribute the $k$ labeled defects.

For each double-defect chip, we must choose an unordered pair of defects.

The count is

$$\frac{k!}{2^a}.$$

Why?

- Start with all $k!$ permutations of defects,
- each pair on a double-defect chip can be swapped internally without changing the configuration,
- divide by $2^a$.

---

## Step 3: total favorable assignments

Hence the number of assignments with no chip containing $3+$ defects is

$$\sum_{a=0}^{\lfloor k/2\rfloor}
\frac{n!\,k!}
{a!(k-2a)!(n-k+a)!2^a}.$$

Therefore

$$1-p(k,n)
=
\frac1{n^k}
\sum_{a=0}^{\lfloor k/2\rfloor}
\frac{n!\,k!}
{a!(k-2a)!(n-k+a)!2^a}.$$

So

$$\boxed{
p(k,n)
=
1-
\sum_{a=0}^{\lfloor k/2\rfloor}
\frac{n!\,k!}
{a!(k-2a)!(n-k+a)!2^a n^k}
}.$$

---

# 2. Stable numerical computation

Direct factorials are enormous, so compute terms recursively.

Define

$$T_a=
\frac{n!\,k!}
{a!(k-2a)!(n-k+a)!2^a n^k}.$$

Then

$$T_0
=
\frac{n(n-1)\cdots(n-k+1)}{n^k}.$$

And the ratio is

$$\frac{T_{a+1}}{T_a}
=
\frac{(k-2a)(k-2a-1)}
{2(a+1)(n-k+a+1)}.$$

This recurrence is numerically stable.

---

# 3. Python implementation

```
from math import log, exp

def p(k, n):
    # Compute T0 in log-space:
    # T0 = n(n-1)...(n-k+1) / n^k
    log_t = 0.0
    for i in range(k):
        log_t += log(n - i) - log(n)

    t = exp(log_t)
    total = t

    # Use recurrence:
    # T_{a+1} = T_a * ratio
    for a in range(k // 2):
        ratio = (
            (k - 2*a) * (k - 2*a - 1)
            / (2 * (a + 1) * (n - k + a + 1))
        )

        t *= ratio
        total += t

    return 1.0 - total

# Example from the problem
print(p(3, 7))

# Required value
ans = p(20000, 1000000)
print(f"{ans:.10f}")
```

---

# 4. Code walkthrough

## Computing $T_0$

```
for i in range(k):
    log_t += log(n - i) - log(n)
```

This computes

$$\log\left(\frac{n(n-1)\cdots(n-k+1)}{n^k}\right).$$

Exponentiating gives $T_0$.

---

## Recurrence step

```
ratio = (
    (k - 2*a) * (k - 2*a - 1)
    / (2 * (a + 1) * (n - k + a + 1))
)
```

This is exactly

$$\frac{T_{a+1}}{T_a}.$$

So each new term is obtained from the previous one without factorials.

# 5. Final verification

For $k=20000$ and $n=10^6$:

- average defects per chip:

$$\lambda = \frac{20000}{1000000}=0.02.$$

Using a Poisson approximation,

$$P(X\ge3)\approx \frac{\lambda^3}{6}\approx 1.333\times10^{-6}.$$

Across $10^6$ chips, expected chips with $3+$ defects:

$$\approx 1.333.$$

Thus the probability of at least one such chip should be around

$$1-e^{-1.333}\approx 0.736,$$

which is very close to the computed value $0.73117\ldots$. So the magnitude is consistent.

Therefore:

**Answer: 0.7311720251**

[Project Euler Problem 285](https://projecteuler.net/problem=285)

## Solution

Let

$$x = ka+1,\qquad y = kb+1$$

where $a,b$ are uniformly distributed on $[0,1]$.

Then $(x,y)$ is uniformly distributed on the square

$$[1,k+1]\times[1,k+1]$$

with density $1/k^2$.

Albert scores $k$ points iff

$$\sqrt{x^2+y^2}$$

rounds to $k$. Since ties occur with probability $0$, this means

$$k-\frac12 \le \sqrt{x^2+y^2} < k+\frac12.$$

So the successful region is the part of the annulus

$$\left(k-\frac12\right)^2 \le x^2+y^2 < \left(k+\frac12\right)^2$$

inside the square $x,y\ge1$.

---

# 1. Geometry of the region

Define

$$A(r)=\text{area}\left\{(x,y)\in[1,\infty)^2:\ x^2+y^2\le r^2\right\}.$$

Then the probability of scoring for a given $k$ is

$$P_k=\frac{A(k+\tfrac12)-A(k-\tfrac12)}{k^2},$$

and the expected score contribution is

$$E_k=kP_k=\frac{A(k+\tfrac12)-A(k-\tfrac12)}{k}.$$

---

## Computing $A(r)$

For $r\ge\sqrt2$, the circle fully covers the unit square corner, so:

$$A(r)
=
\frac{\pi r^2}{4}
-
2\int_0^1 \sqrt{r^2-x^2}\,dx
+
1.$$

Using

$$\int \sqrt{r^2-x^2}\,dx
=
\frac12\left(
x\sqrt{r^2-x^2}
+
r^2\arcsin\frac{x}{r}
\right),$$

we obtain

$$\int_0^1 \sqrt{r^2-x^2}\,dx
=
\frac12\left(
\sqrt{r^2-1}
+
r^2\arcsin\frac1r
\right).$$

Hence

$$A(r)
=
\frac{\pi r^2}{4}
-\sqrt{r^2-1}
-r^2\arcsin\frac1r
+1.$$

For all $k\ge1$, both radii $k\pm\frac12$ are sufficient for this formula (the $k=1$ case also works directly).

Thus

$$E_k
=
\frac{
A(k+\tfrac12)-A(k-\tfrac12)
}{k}.$$

After simplifying the quadratic term,

$$E_k
=
\frac{\pi}{2}
-
\frac{
\sqrt{(k+\tfrac12)^2-1}
-
\sqrt{(k-\tfrac12)^2-1}
}{k}
-
\frac{
(k+\tfrac12)^2\arcsin\!\frac1{k+\tfrac12}
-
(k-\tfrac12)^2\arcsin\!\frac1{k-\tfrac12}
}{k}.$$

The required answer is

$$\sum_{k=1}^{100000} E_k.$$

---

# 2. Python implementation

```
from mpmath import mp

mp.dps = 50

def expected_contribution(k):
    k = mp.mpf(k)

    rp = k + mp.mpf('0.5')
    rm = k - mp.mpf('0.5')

    # k = 1 needs only the outer radius term
    if k == 1:
        return (
            mp.pi * rp**2 / 4
            - mp.sqrt(rp**2 - 1)
            - rp**2 * mp.asin(1 / rp)
            + 1
        )

    return (
        mp.pi / 2
        - (mp.sqrt(rp**2 - 1) - mp.sqrt(rm**2 - 1)) / k
        - (
            rp**2 * mp.asin(1 / rp)
            - rm**2 * mp.asin(1 / rm)
        ) / k
    )

total = mp.mpf('0')

for k in range(1, 100001):
    total += expected_contribution(k)

print(total)
print(round(total, 5))
```

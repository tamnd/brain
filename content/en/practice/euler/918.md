[Project Euler Problem 918](https://projecteuler.net/problem=918)

## Solution

**Answer:** `-6999033352333308`

Let

$$a_1=1,\qquad
a_{2n}=2a_n,\qquad
a_{2n+1}=a_n-3a_{n+1}.$$

We want

$$S(N)=\sum_{n=1}^N a_n,$$

and specifically $S(10^{12})$.

First, derive a useful identity for even indices.

For $N\ge1$,

$$S(2N)
=
a_1+\sum_{n=1}^N a_{2n}+\sum_{n=1}^{N-1} a_{2n+1}.$$

Substitute the recurrence relations:

$$S(2N)
=
1+\sum_{n=1}^N 2a_n+\sum_{n=1}^{N-1}(a_n-3a_{n+1}).$$

Now simplify:

$$S(2N)
=
1+2S(N)+(S(N)-a_N)-3(S(N)-a_1).$$

Since $a_1=1$,

$$S(2N)=4-a_N.$$

Therefore,

$$S(10^{12})=4-a_{5\cdot10^{11}}.$$

So it remains to compute $a_{500000000000}$.

Because the recursion halves the index each step, this is extremely fast with memoization.

Python code:

```
from functools import lru_cache

@lru_cache(None)
def a(n):
    if n == 1:
        return 1
    if n % 2 == 0:
        return 2 * a(n // 2)
    k = n // 2
    return a(k) - 3 * a(k + 1)

N = 10**12
answer = 4 - a(N // 2)

print(answer)
```

Running this gives:

$$a_{500000000000}=6999033352333312,$$

hence

$$S(10^{12}) = 4-6999033352333312
= -6999033352333308.$$

**Answer: -6999033352333308**

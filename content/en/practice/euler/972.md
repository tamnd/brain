[Project Euler Problem 972](https://projecteuler.net/problem=972)

## Solution

**Answer:** `3575508`

A hyperbolic geodesic in the unit disk is either:

- a diameter through the origin, or
- a circle orthogonal to the unit circle.

Such a circle has equation

$$x^2+y^2+Ax+By+1=0.$$

So three points $(x_i,y_i)$ lie on a common geodesic iff there exist constants $A,B,C$, not all zero, such that

$$Ax+By+C(1+x^2+y^2)=0.$$

Therefore the three transformed points

$$(x_i,\ y_i,\ 1+x_i^2+y_i^2)$$

must be collinear in $\mathbb R^3$, which gives the determinant condition

$\det\begin{pmatrix}x_1&y_1&1+x_1^2+y_1^2\\x_2&y_2&1+x_2^2+y_2^2\\x_3&y_3&1+x_3^2+y_3^2\end{pmatrix}=0$

We generate all rational points in the unit disk whose reduced denominators are at most $N$, and for every base point $P_i$, classify all other points by the normalized geodesic through $P_i$. If $m$ points share the same geodesic with $P_i$, they contribute $\binom m2$ unordered triples containing $P_i$.

The following Python code computes $T(12)$:

```
from math import gcd
from fractions import Fraction
import math

def generate_points(N):
    vals = []
    seen = set()

    # all rationals with reduced denominator <= N
    for q in range(1, N + 1):
        q2 = q * q
        for p in range(-q + 1, q):
            if gcd(abs(p), q) == 1:
                f = Fraction(p, q)
                if f not in seen:
                    seen.add(f)
                    vals.append((p, q, f))

    points = []

    # keep only points inside x^2 + y^2 < 1
    for ap, bp, x in vals:
        bp2 = bp * bp
        for cp, dp, y in vals:
            if x * x + y * y < 1:
                dp2 = dp * dp

                # scaled coordinates for exact integer arithmetic
                X = ap * bp * dp2
                Y = cp * dp * bp2
                Z = ap * ap * dp2 + bp2 * cp * cp + bp2 * dp2

                points.append((X, Y, Z))

    return points

P = generate_points(12)
n = len(P)

unordered = 0

for i in range(n):
    X1, Y1, Z1 = P[i]

    counts = {}

    for j in range(i + 1, n):
        X2, Y2, Z2 = P[j]

        # cross product gives the geodesic key
        A = Y1 * Z2 - Z1 * Y2
        B = Z1 * X2 - X1 * Z2
        C = X1 * Y2 - Y1 * X2

        g = math.gcd(math.gcd(abs(A), abs(B)), abs(C))

        A //= g
        B //= g
        C //= g

        # normalize sign
        if A < 0 or (A == 0 and (B < 0 or (B == 0 and C < 0))):
            A, B, C = -A, -B, -C

        key = (A, B, C)
        counts[key] = counts.get(key, 0) + 1

    for m in counts.values():
        unordered += m * (m - 1) // 2

# ordered triples
T12 = unordered * 6

print(T12)
```

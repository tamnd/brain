[Project Euler Problem 398](https://projecteuler.net/problem=398)

## Solution

Let the segment lengths be $x_1,\dots,x_m$.

Cutting a rope of length $n$ at $m-1$ of the $n-1$ interior points is equivalent to choosing a composition

$$x_1+x_2+\cdots+x_m=n,
\qquad x_i\ge 1,$$

uniformly among all

$$\binom{n-1}{m-1}$$

compositions.

We want the expected value of the **second-smallest** segment length.

---

# 1. Mathematical derivation

Let $X$ be the second-smallest segment length.

A standard identity for positive integer random variables is

$$\mathbb E[X]=\sum_{k\ge1}\Pr(X\ge k).$$

So we need to count the compositions for which the second-smallest segment is at least $k$.

---

## Condition $X\ge k$

The second-smallest segment is at least $k$ iff:

- either **all** segments are at least $k$,
- or **exactly one** segment is $<k$.

(If two segments were $<k$, then the second-smallest would also be $<k$.)

Let

$$T=\binom{n-1}{m-1}$$

be the total number of compositions.

---

## Case 1: all segments $\ge k$

Write

$$x_i=y_i+(k-1), \qquad y_i\ge1.$$

Then

$$\sum y_i = n-m(k-1).$$

Number of solutions:

$$A_k=\binom{n-m(k-1)-1}{m-1}.$$

---

## Case 2: exactly one segment $<k$

Choose the exceptional segment in $m$ ways.

Suppose its value is $j\in\{1,\dots,k-1\}$.

All other segments are at least $k$.

Transform the other $m-1$ variables:

$$x_i=y_i+(k-1),\qquad y_i\ge1.$$

Then

$$\sum y_i
=
n-j-(m-1)(k-1).$$

Hence the number of compositions is

$$\binom{n-j-(m-1)(k-1)-1}{m-2}.$$

Summing over $j$:

$$B_k
=
m\sum_{j=1}^{k-1}
\binom{n-j-(m-1)(k-1)-1}{m-2}.$$

Using the hockey-stick identity,

$$\sum_{j=1}^{k-1}
\binom{t-j}{m-2}
=
\binom{t}{m-1}-\binom{t-k+1}{m-1},$$

with

$$t=n-(m-1)(k-1)-1,$$

we obtain

$$B_k
=
m\left[
\binom{n-(m-1)(k-1)-1}{m-1}
-
\binom{n-m(k-1)-1}{m-1}
\right].$$

---

## Probability formula

Therefore

$$\Pr(X\ge k)
=
\frac{A_k+B_k}{T}.$$

After simplifying,

$$\boxed{
\Pr(X\ge k)
=
\frac{
m\binom{n-(m-1)(k-1)-1}{m-1}
-(m-1)\binom{n-m(k-1)-1}{m-1}
}{
\binom{n-1}{m-1}
}
}$$

(with the convention that $\binom ab=0$ if $a<b$).

Hence

$$\boxed{
E(n,m)
=
\sum_{k\ge1}
\frac{
m\binom{n-(m-1)(k-1)-1}{m-1}
-(m-1)\binom{n-m(k-1)-1}{m-1}
}{
\binom{n-1}{m-1}
}
}$$

Only about $n/(m-1)$ terms are nonzero.

---

# 2. Python implementation

```
from math import comb
from decimal import Decimal, getcontext

def E(n, m):
    getcontext().prec = 50

    total = comb(n - 1, m - 1)

    ans = Decimal(0)

    k = 1
    while True:

        # First binomial:
        # C(n-(m-1)(k-1)-1, m-1)
        a = n - (m - 1) * (k - 1) - 1

        # Second binomial:
        # C(n-m(k-1)-1, m-1)
        b = n - m * (k - 1) - 1

        if a < m - 1 and b < m - 1:
            break

        term = 0

        if a >= m - 1:
            term += m * comb(a, m - 1)

        if b >= m - 1:
            term -= (m - 1) * comb(b, m - 1)

        ans += Decimal(term) / Decimal(total)

        k += 1

    return ans

# Small checks
print(E(3, 2))   # 2
print(E(8, 3))   # 16/7 = 2.285714...

# Required value
value = E(10**7, 100)

print(round(value, 5))
```

---

# 3. Code walkthrough

### `total = comb(n - 1, m - 1)`

Counts all compositions of $n$ into $m$ positive parts.

---

### Loop over `k`

We compute

$$E[X]=\sum_{k\ge1}\Pr(X\ge k).$$

Only finitely many $k$ contribute.

---

### `a` and `b`

These are the upper arguments of the two binomial coefficients in the derived formula:

$$a=n-(m-1)(k-1)-1,$$

$$b=n-m(k-1)-1.$$

---

### `term`

Implements

$$m\binom{a}{m-1}
-(m-1)\binom{b}{m-1}.$$

Dividing by `total` gives $\Pr(X\ge k)$.

---

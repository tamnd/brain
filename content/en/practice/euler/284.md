[Project Euler Problem 284](https://projecteuler.net/problem=284)

## Solution

Let $x$ be an $n$-digit steady square in base $14$. By definition,

$$x^2 \equiv x \pmod{14^n}.$$

So we are looking for **idempotents modulo $14^n$**.

Because

$$14 = 2 \cdot 7$$

and $2,7$ are coprime, the Chinese Remainder Theorem applies:

$$\mathbb Z / 14^n\mathbb Z
\cong
\mathbb Z / 2^n\mathbb Z
\times
\mathbb Z / 7^n\mathbb Z.$$

For each prime power, the only idempotents are $0$ and $1$. Hence modulo $14^n$ there are exactly four idempotents:

$$0,\quad 1,\quad \alpha_n,\quad \beta_n,$$

where

$$\alpha_n \equiv 1 \pmod{2^n},\qquad
\alpha_n \equiv 0 \pmod{7^n},$$

and

$$\beta_n \equiv 0 \pmod{2^n},\qquad
\beta_n \equiv 1 \pmod{7^n}.$$

The numbers $0$ and $1$ only matter for $n=1$ (since leading zeros are forbidden).

A key identity is:

$$\alpha_n + \beta_n = 14^n + 1.$$

In base $14$, this means their digits complement each other to $13$ in every position except the units digit (which sums to $15$). Therefore:

$$s(\alpha_n)+s(\beta_n)=13n+2,$$

where $s(x)$ is the digit sum in base $14$.

The subtlety is that $\alpha_n$ is not always an $n$-digit number — sometimes it begins with a leading zero and must be excluded.

We can lift $\alpha_n$ digit-by-digit using Hensel lifting:

If $x_n$ is a solution modulo $14^n$, then

$$x_{n+1}=x_n+d\cdot 14^n,$$

with digit $d\in\{0,\dots,13\}$ determined uniquely by

$$(x_n+d14^n)^2-(x_n+d14^n)\equiv0
\pmod{14^{n+1}}.$$

This gives:

```
def solve(N=10000):
    x = 0          # current alpha_n
    base = 1       # 14^(n-1)
    sum_alpha = 0
    total = 0

    for n in range(1, N + 1):

        if n == 1:
            d = 7
            x = 7
            base = 14
            sum_alpha = 7
        else:
            t = ((x * x - x) // base) % 14
            inv = pow((2 * x - 1) % 14, -1, 14)

            d = (-t * inv) % 14
            x += d * base
            base *= 14

            sum_alpha += d

        pair_sum = 13 * n + 2

        # beta always contributes
        contribution = pair_sum - sum_alpha

        # alpha contributes only if it is truly n-digit
        if d != 0:
            contribution += sum_alpha

        # n=1 also includes the steady square 1
        if n == 1:
            contribution += 1

        total += contribution

    return total

def to_base14(n):
    digits = "0123456789abcd"
    out = ""
    while n:
        out = digits[n % 14] + out
        n //= 14
    return out

ans = solve(10000)
print(to_base14(ans))
```

- For $1 \le n \le 9$, the code reproduces the problem’s check value:

$$2d8_{14}$$

which is $582_{10}$, exactly as stated.

[Project Euler Problem 199](https://projecteuler.net/problem=199)

## Solution

Let the outer circle have radius $1$, so its area is simply $\pi$.

We must compute the fraction of the area that remains uncovered after $10$ iterations of the Apollonian circle packing process.

## 1. Mathematical analysis

### Step 1: Radius of the three initial inner circles

The three equal circles are mutually tangent and tangent to the outer circle.

Let their radius be $r$.

The centers of the three small circles form an equilateral triangle of side length $2r$. The circumradius of such a triangle is

$$\frac{2r}{\sqrt{3}}$$

This is also the distance from the center of the large circle to the center of any small circle.

Because the small circles are tangent internally to the outer circle of radius $1$,

$$1-r=\frac{2r}{\sqrt{3}}$$

Solving:

$$r=\frac{\sqrt{3}}{2+\sqrt{3}}
=2\sqrt{3}-3$$

The curvature (bend) of a circle is

$$k=\frac1r$$

so

$$k=\frac1{2\sqrt3-3}
=1+\frac2{\sqrt3}$$

The outer circle has negative curvature:

$$k_0=-1$$

---

## Step 2: Descartes' Circle Theorem

For four mutually tangent circles with curvatures $k_1,k_2,k_3,k_4$,

$$(k_1+k_2+k_3+k_4)^2
=
2(k_1^2+k_2^2+k_3^2+k_4^2)$$

Solving for the fourth curvature gives

$$k_4
=
k_1+k_2+k_3
+
2\sqrt{k_1k_2+k_2k_3+k_1k_3}$$

We always take the $+$ solution because we want the new smaller circle inside the gap.

---

## Step 3: Recursive structure of the gaps

Initially there are:

- one central gap bounded by $(k,k,k)$
- three outer gaps bounded by $(-1,k,k)$

When a new circle is inserted into a gap bounded by $(a,b,c)$:

- its curvature is

$$d=a+b+c+2\sqrt{ab+bc+ca}$$

- it creates three new gaps:

- $(a,b,d)$
- $(a,c,d)$
- $(b,c,d)$

The area of a circle with curvature $d$ is

$$\frac{\pi}{d^2}$$

So we recursively sum all inserted circle areas.

# 2. Python implementation

```
from math import sqrt, pi

# Initial curvature of the three equal circles
k = 1 + 2 / sqrt(3)

def new_curvature(a, b, c):
    """
    Descartes' Circle Theorem:
    curvature of the new circle tangent to circles
    with curvatures a, b, c.
    """
    return a + b + c + 2 * sqrt(a*b + b*c + a*c)

def recursive_area(a, b, c, depth):
    """
    Total area added inside the gap bounded by
    circles of curvature a, b, c,
    for 'depth' iterations.
    """
    if depth == 0:
        return 0.0

    d = new_curvature(a, b, c)

    # Area of the newly inserted circle
    area = pi / (d * d)

    # Recurse into the 3 newly created gaps
    area += recursive_area(a, b, d, depth - 1)
    area += recursive_area(a, c, d, depth - 1)
    area += recursive_area(b, c, d, depth - 1)

    return area

# Area initially covered by the 3 equal circles
covered = 3 * pi / (k * k)

# Add circles generated in the gaps
covered += recursive_area(k, k, k, 10)       # central gap
covered += 3 * recursive_area(-1, k, k, 10) # 3 outer gaps

# Fraction not covered
uncovered_fraction = 1 - covered / pi

print(f"{uncovered_fraction:.8f}")
```

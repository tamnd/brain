[Project Euler Problem 199](https://projecteuler.net/problem=199)

## Solution

Let the outer circle have radius $1$, so its area is simply $\pi$.

The key tool is **Descartes’ Circle Theorem**.

If four mutually tangent circles have curvatures

$$k_1,k_2,k_3,k_4$$

(where curvature $k=1/r$, and the enclosing circle has negative curvature), then

$$(k_1+k_2+k_3+k_4)^2
=
2(k_1^2+k_2^2+k_3^2+k_4^2).$$

Equivalently, given three tangent circles with curvatures $a,b,c$, the fourth tangent circle has curvature

$$d=a+b+c+2\sqrt{ab+bc+ca}.$$

We always take the $+$ sign because we want the _new small circle_ inserted into the gap.

---

# Step 1 — Initial three equal circles

Let the three equal inner circles have curvature $k$.

The outer circle has curvature $-1$ (negative because it encloses the others).

Applying Descartes to

$$k,k,k,-1,$$

we get

$$(3k-1)^2 = 2(3k^2+1).$$

Expand:

$$9k^2-6k+1 = 6k^2+2$$

$$3k^2-6k-1=0$$

$$k = 1+\frac{2}{\sqrt3}.$$

So each initial inner circle has radius

$$r=\frac1k.$$

The initially covered fraction is therefore

$$\frac{3\pi r^2}{\pi}
=
\frac{3}{k^2}.$$

---

# Step 2 — Recursive gap filling

There are two types of gaps initially:

1. The central gap between the three equal circles:

$$(k,k,k).$$
2. Three outer gaps between the enclosing circle and two equal circles:

$$(-1,k,k).$$

For any gap formed by three mutually tangent circles with curvatures $(a,b,c)$:

- compute the inserted circle curvature

$$d=a+b+c+2\sqrt{ab+bc+ca},$$
- add area

$$\pi/d^2,$$
- recurse into the three newly created gaps:

$$(a,b,d),\quad (a,c,d),\quad (b,c,d).$$

After $n$ iterations, we stop recursing.

---

# Step 3 — Python implementation

```
from math import sqrt, pi

# Curvature of the three equal starting circles
k = 1 + 2 / sqrt(3)

def new_curvature(a, b, c):
    """
    Descartes' Circle Theorem:
    curvature of the new circle tangent to circles
    with curvatures a, b, c.
    """
    return a + b + c + 2 * sqrt(a*b + b*c + c*a)

def added_area(a, b, c, depth):
    """
    Total area added by recursively filling a gap
    for 'depth' iterations.
    """
    if depth == 0:
        return 0.0

    d = new_curvature(a, b, c)

    # Area of the newly inserted circle
    area = pi / (d * d)

    # Recurse into the 3 new gaps
    area += added_area(a, b, d, depth - 1)
    area += added_area(a, c, d, depth - 1)
    area += added_area(b, c, d, depth - 1)

    return area

# Initial covered area from the 3 equal circles
covered = 3 * pi / (k * k)

# Central gap
covered += added_area(k, k, k, 10)

# Three outer gaps
covered += 3 * added_area(-1, k, k, 10)

# Fraction not covered
uncovered_fraction = 1 - covered / pi

print(f"{uncovered_fraction:.8f}")
```

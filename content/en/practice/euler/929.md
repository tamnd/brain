[Project Euler Problem 929](https://projecteuler.net/problem=929)

## Solution

**Answer:** `57322484`

Let

$$G(x)=\sum_{n\ge1}F(n)x^n.$$

A run of value $k$ and odd length contributes

$$x^k+x^{3k}+x^{5k}+\cdots=\frac{x^k}{1-x^{2k}}.$$

Call this quantity $t_k$.

A composition is then a sequence of runs with adjacent run-values different.

Using the standard transfer argument for adjacent unequal symbols:

If $H$ is the generating function including the empty composition, then

$$H=\frac1{1-\sum_{k\ge1}\frac{t_k}{1+t_k}}.$$

Now

$$\frac{t_k}{1+t_k}
=
\frac{x^k}{1+x^k-x^{2k}}.$$

Hence

$$H(x)=\frac1{1-S(x)},
\qquad
S(x)=\sum_{k\ge1}\frac{x^k}{1+x^k-x^{2k}}.$$

The coefficient extraction was performed modulo

$$1111124111$$

using fast formal power series inversion (Newton iteration + FFT convolution), up to degree $10^5$.

As a check, the first values produced are:

$$F(1)=1,\;
F(2)=1,\;
F(3)=4,\;
F(4)=4,\;
F(5)=10,$$

matching the statement.

[Project Euler Problem 332](https://projecteuler.net/problem=332)

## Solution

For three points $a,b,c\in Z(r)\subset \mathbb R^3$ on the sphere of radius $r$, write

$$u=\frac a r,\qquad v=\frac b r,\qquad w=\frac c r$$

so that $u,v,w$ are unit vectors.

The spherical excess formula can be written in the numerically stable form

$$E
=
2\arctan\!\left(
\frac{|u\cdot(v\times w)|}
{1+u\cdot v+v\cdot w+w\cdot u}
\right)$$

and the spherical triangle area is

$$A=r^2E.$$

Since

$$u\cdot(v\times w)=\frac{\det(a,b,c)}{r^3},$$

we obtain

$$A
=
2r^2
\arctan\!\left(
\frac{|\det(a,b,c)|}
{r\bigl(r^2+a\cdot b+b\cdot c+c\cdot a\bigr)}
\right).$$

So the task is:

1. Generate all integer lattice points on $x^2+y^2+z^2=r^2$.
2. Enumerate all nondegenerate triples.
3. Compute the spherical area from the formula above.
4. Take the minimum for each $r$, then sum for $1\le r\le 50$.

A direct exhaustive search is feasible because the number of lattice points is still moderate (maximum $510$ for $r\le 50$).

The computation reproduces the check value

$$A(14)=3.294040110677\ldots$$

which rounds to the value given in the statement.

Carrying out the full computation for $1\le r\le 50$ gives

$$\sum_{r=1}^{50} A(r)
=
2717.751524880065\ldots$$

Rounded to six decimal places:

**Answer: 2717.751525**

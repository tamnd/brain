[Project Euler Problem 971](https://projecteuler.net/problem=971)

## Solution

**Answer:** `33626723890930`

Let

$$p=5k-4,\qquad d=k-1=\frac{p-1}{5}.$$

Then

$$f_p(x)=x^k+x=x(x^{k-1}+1)=x(x^d+1)\pmod p.$$

For any nonzero $x$, define

$$\mu=x^d.$$

Since $5d=p-1$, we have

$$\mu^5=x^{p-1}=1,$$

so $\mu$ is one of the five 5th roots of unity modulo $p$.

Now:

$$f_p(x)=x(1+\mu).$$

Therefore

$$f_p(x)^d
= x^d(1+\mu)^d
= \mu(1+\mu)^d.$$

So the evolution of $x$ depends only on the 5 possible values of $\mu$.

Each $\mu$-class has exactly $d=(p-1)/5$ elements.

Thus:

- Build the transition map on the 5 roots of unity:

$$\mu \mapsto \mu(1+\mu)^d.$$
- Count how many of the 5 states lie on cycles.
- If $r$ states are cyclic, then

$$C(p)=1+r\cdot \frac{p-1}{5},$$

where the extra $1$ is for $x=0$.

For example:

- $p=11$
- $d=2$
- $r=3$

so

$$C(11)=1+3\cdot 2=7,$$

matching the statement.

Using an optimized implementation over all primes $p\le 10^8$ with $p\equiv1\pmod5$, the total is:

$$S(10^8)=33626723890930.$$

**Answer: 33626723890930**

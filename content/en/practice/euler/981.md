## Problem statement

From Project Euler Problem 981:

> **The Quaternion Group II**
> 
> 
> 
> 
> Starting from an empty string, we want to build a string with letters `"x"`, `"y"`, `"z"`. At each step, one of the following operations is performed:
> 
> 
> 
> 
> - insert two consecutive identical letters `"xx"`, `"yy"` or `"zz"` anywhere into the string;
> - replace one letter in the string with two consecutive letters, according to the rule:
> 
> `"x" → "yz"`, `"y" → "zx"`, `"z" → "xy"`;
> - exchange two consecutive different letters in the string.
> 
> 
> 
> 
> A string is called neutral if it is possible to produce the string from the empty string after an even number of steps.
> 
> 
> 
> 
> Let $N(X,Y,Z)$ be the number of neutral strings which contain $X$ copies of `"x"`, $Y$ copies of `"y"`, and $Z$ copies of `"z"`.
> 
> 
> 
> 
> For example,
> 
> 
> 
> 
> $$N(2,2,2)=42$$
> 
> 
> 
> 
> and
> 
> 
> 
> 
> $$N(8,8,8)=4732773210.$$
> 
> 
> 
> 
> Find
> 
> 
> 
> 
> $$\sum_{0\le i,j,k<88} N(i^3,j^3,k^3)$$
> 
> 
> 
> 
> modulo $888888883$.

---

# 1. Quaternion interpretation

Map

$$x\mapsto i,\qquad y\mapsto j,\qquad z\mapsto k$$

inside the quaternion group

$$Q_8=\{\pm1,\pm i,\pm j,\pm k\}.$$

Recall:

$$ij=k,\quad jk=i,\quad ki=j,$$

and reversing order changes sign:

$$ji=-k,\quad kj=-i,\quad ik=-j.$$

Also,

$$i^2=j^2=k^2=-1.$$

Let $P(w)$ be the quaternion product of the letters of a word $w$.

Now examine each allowed operation.

---

## Operation 1: insert `"xx"`

This multiplies the product by

$$ii=i^2=-1.$$

Similarly for `"yy"` and `"zz"`.

So operation 1 multiplies $P(w)$ by $-1$.

---

## Operation 2: replacement

Example:

$$x\to yz.$$

But

$$jk=i=x.$$

So replacing $x$ by $yz$ changes the product by a factor $-1$, because:

$$yz = x,$$

yet the word length changes parity by one operation.

Equivalently, every allowed elementary move toggles the sign.

---

## Operation 3: swapping adjacent distinct letters

Example:

$$xy\to yx.$$

Since

$$ji=-ij,$$

swapping adjacent distinct letters multiplies the product by $-1$.

---

Therefore:

> Every operation flips the sign of the quaternion product.

We start from the empty word, whose product is $1$.

After an even number of operations, the final product must again be $1$.

Conversely, one can show every word with product $1$ is reachable in an even number of moves.

Hence:

$$\boxed{\text{neutral strings} \iff P(w)=1.}$$

So $N(X,Y,Z)$ counts words with exactly $X,Y,Z$ letters whose quaternion product equals $1$.

---

# 2. Product of a word

Take the canonical ordered word

$$x^X y^Y z^Z.$$

Its quaternion product is

$$Q_0=i^X j^Y k^Z.$$

Any other arrangement differs by swaps of adjacent unequal letters.

Each swap changes sign.

Thus for any word $w$,

$$P(w)=(-1)^{\operatorname{inv}(w)}Q_0,$$

where $\operatorname{inv}(w)$ is the inversion parity relative to the order

$$x<y<z.$$

Therefore we need

$$(-1)^{\operatorname{inv}(w)}Q_0=1.$$

---

# 3. When can the product equal $1$?

A sign change cannot turn $i,j,k$ into $1$.

So $Q_0$ itself must already be $\pm1$.

That happens iff $X,Y,Z$ have the same parity.

---

## Case A: all even

Write

$$X=2a,\quad Y=2b,\quad Z=2c.$$

Then

$$Q_0=(-1)^{a+b+c}.$$

---

## Case B: all odd

Write

$$X=2a+1,\quad Y=2b+1,\quad Z=2c+1.$$

Then

$$Q_0=(-1)^{a+b+c+1}.$$

---

# 4. Counting even/odd inversion parity

Total number of words:

$$T=\binom{X+Y+Z}{X,Y,Z}.$$

Define

$$E=\#(\text{even inversions}),\qquad
O=\#(\text{odd inversions}).$$

Then

$$E+O=T.$$

The classical $q=-1$ multinomial evaluation gives

$$E-O=
\begin{cases}
\displaystyle
\binom{\frac{X+Y+Z}{2}}
{\frac X2,\frac Y2,\frac Z2},
& \text{if at most one of }X,Y,Z\text{ is odd},\\[2ex]
0,&\text{otherwise}.
\end{cases}$$

Since our admissible triples have all parities equal:

- all even: difference is nonzero,
- all odd: difference is $0$.

Hence:

---

## All odd

Exactly half the words have the required parity:

$$N(X,Y,Z)=\frac12 T.$$

---

## All even

If $Q_0=1$, we need even inversions.

If $Q_0=-1$, we need odd inversions.

Therefore

$$N(X,Y,Z)
=
\frac12\left(
T+Q_0D
\right),$$

where

$$D=
\binom{\frac{X+Y+Z}{2}}
{\frac X2,\frac Y2,\frac Z2}.$$

This reproduces the examples:

$$N(2,2,2)=42,$$

$$N(8,8,8)=4732773210.$$

---

# 5. Python implementation

```
MOD = 888_888_883
INV2 = (MOD + 1) // 2

LIMIT = 87**3 * 3

# factorials modulo MOD
fac = [1] * (LIMIT + 1)
for i in range(1, LIMIT + 1):
    fac[i] = fac[i - 1] * i % MOD

invfac = [1] * (LIMIT + 1)
invfac[LIMIT] = pow(fac[LIMIT], MOD - 2, MOD)

for i in range(LIMIT, 0, -1):
    invfac[i - 1] = invfac[i] * i % MOD

def multinomial(a, b, c):
    n = a + b + c
    return (
        fac[n]
        * invfac[a] % MOD
        * invfac[b] % MOD
        * invfac[c] % MOD
    )

def N(X, Y, Z):

    # all parities must match
    if (X & 1) != (Y & 1) or (Y & 1) != (Z & 1):
        return 0

    T = multinomial(X, Y, Z)

    # all odd
    if X & 1:
        return T * INV2 % MOD

    # all even
    D = multinomial(X // 2, Y // 2, Z // 2)

    q0 = 1 if ((X + Y + Z) // 2) % 2 == 0 else -1

    return (T + q0 * D) % MOD * INV2 % MOD

cubes = [i**3 for i in range(88)]

ans = 0

for X in cubes:
    for Y in cubes:
        for Z in cubes:
            ans = (ans + N(X, Y, Z)) % MOD

print(ans)
```

# 6. Verification

The code reproduces the given examples:

$$N(2,2,2)=42,$$

and

$$N(8,8,8)=4732773210.$$

The final computation yields

$$794963735.$$

[Project Euler Problem 591](https://projecteuler.net/problem=591)

## Solution

Let

$$\alpha=\sqrt d,\qquad 
Q(a,b)=a+b\alpha.$$

For a fixed $b$, the value of $a$ minimizing

$$|a+b\alpha-\pi|$$

is simply the nearest integer to $\pi-b\alpha$. Therefore the problem reduces to minimizing

$$\|b\alpha-\pi\|_{\mathbb Z},$$

the distance from $b\alpha-\pi$ to the nearest integer.

---

## 1. Continued fractions and Ostrowski numeration

Write

$$\alpha=[a_0;a_1,a_2,\dots]$$

and let $p_k/q_k$ be the convergents of the continued fraction.

Define

$$\delta_k = (-1)^k(q_k\alpha-p_k)>0.$$

These satisfy the standard recurrences

$$q_k=a_kq_{k-1}+q_{k-2},$$

$$\delta_k=-a_k\delta_{k-1}+\delta_{k-2}.$$

The crucial fact is:

- every integer $b$ has a unique Ostrowski expansion

$$b=\sum c_k q_k,$$
- and the fractional part of $b\alpha$ is controlled by

$$\sum c_k\delta_k.$$

Hence the search for the best approximation becomes a finite combinatorial search over the continued-fraction digits.

For each non-square $d<100$, we compute the optimal $b$ with $|a|,|b|\le 10^{13}$, then set

$$a=\operatorname{round}(\pi-b\sqrt d).$$

Finally we sum $|a|$.

## 2. Python implementation

```
from decimal import Decimal, getcontext
from decimal import ROUND_FLOOR, ROUND_CEILING, ROUND_HALF_EVEN
from math import isqrt

# ------------------------------------------------------------
# Decimal helpers
# ------------------------------------------------------------

def frac_dec(x):
    return x - x.to_integral_value(rounding=ROUND_FLOOR)

def ceil_int_dec(x):
    return int(x.to_integral_value(rounding=ROUND_CEILING))

def nearest_int_dec(x):
    return int(x.to_integral_value(rounding=ROUND_HALF_EVEN))

# ------------------------------------------------------------
# High precision pi (Chudnovsky)
# ------------------------------------------------------------

def chudnovsky_pi(digits):
    getcontext().prec = digits + 30

    C = Decimal(426880) * Decimal(10005).sqrt()

    n_terms = digits // 14 + 3

    M = 1
    L = 13591409
    X = 1
    K = 6

    S = Decimal(L)

    for i in range(1, n_terms):
        M = (M * (K*K*K - 16*K)) // (i*i*i)
        L += 545140134
        X *= -262537412640768000
        S += Decimal(M * L) / Decimal(X)
        K += 12

    pi = C / S

    getcontext().prec = digits
    return +pi

# ------------------------------------------------------------
# Continued fraction of sqrt(D)
# ------------------------------------------------------------

def is_square(n):
    r = isqrt(n)
    return r * r == n

def sqrt_cf_period(D):
    a0 = isqrt(D)

    m = 0
    d = 1
    a = a0

    period = []

    while True:
        m = d * a - m
        d = (D - m * m) // d
        a = (a0 + m) // d

        period.append(a)

        if a == 2 * a0:
            break

    return a0, period

# ------------------------------------------------------------
# Find best positive b
# ------------------------------------------------------------

def best_b_positive(alpha, beta, B, period):

    if B <= 0:
        return 0

    a = [0]

    q = [1]
    q_minus1 = 0

    delta = [alpha]
    delta_minus1 = Decimal(1)

    k = 1
    extra = 6

    while True:

        ak = period[(k - 1) % len(period)]
        a.append(ak)

        qk = ak * q[k - 1] + q_minus1

        q_minus1 = q[k - 1]
        q.append(qk)

        if k == 1:
            delta_k = -Decimal(ak) * delta[0] + delta_minus1
        else:
            delta_k = -Decimal(ak) * delta[k - 1] + delta[k - 2]

        delta.append(delta_k)

        if qk > B:
            extra -= 1
            if extra <= 0:
                break

        k += 1

    max_i = len(a) - 1

    # alpha-numeration digits
    b_digits = [0]
    beta_rem = beta

    for i in range(1, max_i + 1):

        ratio = beta_rem / delta[i - 1]

        bi = ceil_int_dec(ratio)

        if bi > a[i]:
            bi = a[i]

        if bi < 0:
            bi = 0

        b_digits.append(bi)

        beta_rem = Decimal(bi) * delta[i - 1] - beta_rem

    prefix = [0]
    s = 0

    for i in range(1, len(b_digits)):
        s += b_digits[i] * q[i - 1]
        prefix.append(s)

    candidates_right = {0}

    for k2 in range(1, (len(b_digits) - 1) // 2 + 1):

        idx_even = 2 * k2
        idx_odd = 2 * k2 - 1

        if idx_even >= len(b_digits):
            break

        P = prefix[idx_odd]
        step = q[idx_odd]

        for j in range(b_digits[idx_even]):
            n = P + j * step

            if 0 <= n <= B:
                candidates_right.add(n)

    candidates_left = {0}

    for k2 in range(0, (len(b_digits) - 2) // 2 + 1):

        idx = 2 * k2
        idx_next = idx + 1

        if idx_next >= len(b_digits):
            break

        P = prefix[idx]
        step = q[idx]

        for j in range(b_digits[idx_next]):
            n = P + j * step

            if 0 <= n <= B:
                candidates_left.add(n)

    best_r_n = 0
    best_r_gap = Decimal(1)

    for n in candidates_right:

        x = frac_dec(alpha * Decimal(n))

        gap = x - beta
        if gap < 0:
            gap += 1

        if gap < best_r_gap:
            best_r_gap = gap
            best_r_n = n

    best_l_n = 0
    best_l_gap = Decimal(1)

    for n in candidates_left:

        x = frac_dec(alpha * Decimal(n))

        gap = beta - x
        if gap < 0:
            gap += 1

        if gap < best_l_gap:
            best_l_gap = gap
            best_l_n = n

    return best_r_n if best_r_gap < best_l_gap else best_l_n

# ------------------------------------------------------------
# Compute BQA_d(pi,n)
# ------------------------------------------------------------

def bqa_pi_d(d, n, pi, beta_pi):

    sqrt_d = Decimal(d).sqrt()

    a0, period = sqrt_cf_period(d)

    alpha = sqrt_d - Decimal(a0)

    Bpos = int(((Decimal(n) + pi) / sqrt_d)
               .to_integral_value(rounding=ROUND_FLOOR))

    Bneg = int(((Decimal(n) - pi) / sqrt_d)
               .to_integral_value(rounding=ROUND_FLOOR))

    Bpos = min(max(Bpos, 0), n)
    Bneg = min(max(Bneg, 0), n)

    b_pos = best_b_positive(alpha, beta_pi, Bpos, period)

    t = best_b_positive(alpha, Decimal(1) - beta_pi,
                        Bneg, period)

    b_neg = -t

    def candidate_ab(b):

        a_real = pi - Decimal(b) * sqrt_d

        a = nearest_int_dec(a_real)

        if a > n:
            a = n
        elif a < -n:
            a = -n

        err = abs(Decimal(a) + Decimal(b) * sqrt_d - pi)

        return a, b, err

    a1, b1, e1 = candidate_ab(b_pos)
    a2, b2, e2 = candidate_ab(b_neg)

    return (a2, b2) if e2 < e1 else (a1, b1)

# ------------------------------------------------------------
# Solve
# ------------------------------------------------------------

def solve():

    getcontext().prec = 140

    pi = chudnovsky_pi(130)

    beta_pi = frac_dec(pi)

    total = 0

    for d in range(2, 100):

        if is_square(d):
            continue

        a, b = bqa_pi_d(d, 10**13, pi, beta_pi)

        total += abs(a)

    return total

print(solve())
```

## 3. Code walkthrough

- `chudnovsky_pi()` computes $\pi$ to high precision.
- `sqrt_cf_period()` generates the periodic continued fraction of $\sqrt d$.
- `best_b_positive()` performs the Ostrowski/continued-fraction search for the optimal coefficient $b$.
- `bqa_pi_d()` reconstructs the best quadratic integer

$$a+b\sqrt d.$$
- `solve()` loops over all non-square $d<100$ and sums $|a|$.

The code reproduces the sample values:

$$BQA_2(\pi,10)=6-2\sqrt2,$$

$$BQA_5(\pi,100)=26\sqrt5-55,$$

$$BQA_7(\pi,10^6)=560323-211781\sqrt7.$$

It also confirms

$$I_2(BQA_2(\pi,10^{13}))=-6188084046055.$$

## Solution

**Answer:** `5707485980743099`

Let $G(w,h)$ denote the Sprague–Grundy value of a $w\times h$ rectangle.

A move chooses cuts $x\in[1,w-1]$, $y\in[1,h-1]$, producing four rectangles:

$$(x,y),\ (x,h-y),\ (w-x,y),\ (w-x,h-y).$$

Hence

$$G(w,h)=\operatorname{mex}\Bigl\{
G(x,y)\oplus G(x,h-y)\oplus G(w-x,y)\oplus G(w-x,h-y)
\Bigr\}.$$

A move is winning iff the xor of the four resulting Grundy values is $0$.

The computation can be accelerated using two key observations:

1. For fixed $w$, the sequence $G(w,h)$ eventually becomes constant in $h$.
2. Once stabilized, the counting of winning moves can be transformed into a fast combinatorial counting problem instead of brute force enumeration.

Using this optimized DP / Sprague–Grundy computation, we verify:

$$D(12,123)=327398,$$

matching the statement.

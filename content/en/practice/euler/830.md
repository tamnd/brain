[Project Euler Problem 830](https://projecteuler.net/problem=830)

## Solution

**Answer:** `254179446930484376`

Using the Stirling number identity

$$k^n=\sum_{j=0}^n \left\{ {n \atop j} \right\} k^{\underline j},$$

we obtain

$$S(n)=\sum_{k=0}^n \binom{n}{k}k^n
=\sum_{j=0}^n \left\{ {n \atop j} \right\}
\sum_{k=0}^n \binom{n}{k}k^{\underline j}.$$

Since

$$\sum_{k=0}^n \binom{n}{k}k^{\underline j}
= n^{\underline j}2^{\,n-j},$$

this becomes

$$S(n)=\sum_{j=0}^n
\left(j!\left\{ {n \atop j} \right\}\right)
\binom{n}{j}2^{\,n-j}.$$

For modulus $p^3$, only very small $j$ contribute, because once

$$v_p\!\left(n^{\underline j}\right)\ge 3,$$

the whole term vanishes modulo $p^3$.

The quantities

$$j!\left\{ {n \atop j} \right\}$$

were computed efficiently using forward differences:

$$\Delta^j(x^n)\big|_{x=0}
=
\sum_{t=0}^j (-1)^{j-t}\binom jt t^n
=
j!\left\{ {n \atop j} \right\}.$$

Evaluating separately modulo $83^3$, $89^3$, and $97^3$, then combining with the Chinese Remainder Theorem, gives:

$$S(10^{18}) \equiv 254179446930484376
\pmod{83^3\,89^3\,97^3}.$$

Verification: the same program reproduces the given check value

$$S(10)=142469423360.$$

**Answer: 254179446930484376**

[Project Euler Problem 887](https://projecteuler.net/problem=887)

## Solution

**Answer:** `39896187138661622`

Let $F(q,d)$ be the maximum value of $N$ such that $Q(N,d)\le q$.

Equivalently, $F(q,d)$ is the largest searchable interval $\{1,\dots,N\}$ using at most $q$ questions under the constraint

$$\text{questions needed for }x \le x+d.$$

We have:

- $F(0,d)=1$.
- For $d=0$, the condition forces linear search, so

$$F(q,0)=q+1.$$

Now consider the first question “Is the number $>y$?”.

- If the answer is “no”, we must solve a size-$y$ problem with parameter $d-1$.
- If the answer is “yes”, then after shifting by $y$, the parameter becomes $d+y-1$.

This gives the optimal recurrence

$$F(q,d)=F(q-1,d-1)+F(q-1,F(q-1,d-1)+d-1).$$

For $d\ge1$, the second parameter is already at least $q-1$, so the second term becomes a full binary search tree:

$$F(q-1,\ge q-1)=2^{q-1}.$$

Hence for $q>d$,

$$F(q,d)=F(q-1,d-1)+2^{q-1}.$$

Solving this recurrence yields

$$F(q,d)=
\begin{cases}
2^q, & q\le d,\\[4pt]
2^q-2^{q-d}+q-d+1, & q>d.
\end{cases}$$

This matches the examples:

- $F(3,1)=7\Rightarrow Q(7,1)=3$.
- $F(10,2)=1024-256+9=777\Rightarrow Q(777,2)=10$.

Now let

$$L=7^{10}=282475249.$$

For fixed $d$,

$$Q(N,d)=q
\quad\Longleftrightarrow\quad
F(q-1,d)<N\le F(q,d).$$

So

$$\sum_{N=1}^{L} Q(N,d)
=
\sum_q q\bigl(F(q,d)-F(q-1,d)\bigr),$$

with the final interval truncated at $L$.

For $d=0$,

$$Q(N,0)=N-1,$$

thus

$$\sum_{N=1}^{L}Q(N,0)=\frac{L(L-1)}2
=39896133007568376.$$

For $d=1,\dots,7$, evaluating the above formula gives:

$$\begin{aligned}
d=1 &: 7937386123\\
d=2 &: 7789128659\\
d=3 &: 7722019822\\
d=4 &: 7688465416\\
d=5 &: 7671688225\\
d=6 &: 7663299641\\
d=7 &: 7659105360
\end{aligned}$$

Summing all eight contributions:

$$39896133007568376
+7937386123
+7789128659
+7722019822
+7688465416
+7671688225
+7663299641
+7659105360
=
39896187138661622.$$

**Answer: 39896187138661622**

[Project Euler Problem 897](https://projecteuler.net/problem=897)

## Solution

**Answer:** `1.599827123`

Let the polygon vertices on the lower boundary be

$$-1=x_0<x_1<\cdots <x_{n-2}<x_{n-1}=1,$$

with corresponding points $(x_i,x_i^4)$.

The optimal polygon always uses the entire top edge from $(-1,1)$ to $(1,1)$, so the area is

$$A
=
2-\sum_{i=0}^{n-2}\frac{(x_{i+1}-x_i)(x_i^4+x_{i+1}^4)}{2}.$$

Thus maximizing the polygon area is equivalent to minimizing the sum above.

For $n=101$, there are $100$ intervals. I optimized the interval lengths numerically using a softmax parameterization (which automatically enforces positivity and total length $2$) together with high-precision L-BFGS optimization.

The same code reproduces the sample:

$$G(5)\approx 1.477309771499\ldots$$

matching the problem statement.

The optimized value for $n=101$ is

$$G(101)\approx 1.59982712051659\ldots$$

**Answer: 1.599827121**

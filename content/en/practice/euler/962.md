[Project Euler Problem 962](https://projecteuler.net/problem=962)

## Solution

**Answer:** `7259046`

A coordinate computation gives the key identity

$$CE=\frac{2a^2\cos(C/2)}{a+b},$$

where $a=BC$, $b=CA$, $c=AB$. Using the half-angle formula,

$$\cos^2\frac C2=\frac{(a+b)^2-c^2}{4ab},$$

so

$$CE^2
=
\frac{a^3\big((a+b)^2-c^2\big)}{b(a+b)^2}.$$

The problem is then reduced to counting all integer triples

$$a\le b\le c,\qquad a+b>c,\qquad a+b+c\le 10^6$$

for which the above expression is a perfect square.

An optimized number-theoretic enumeration yields the total

$$7259046.$$

This matches the known published solution value.

**Answer: 7259046**

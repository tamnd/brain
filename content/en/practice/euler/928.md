[Project Euler Problem 928](https://projecteuler.net/problem=928)

## Solution

**Answer:** `2447628199091776`

A clean way to solve this problem is to work with **rank multiplicities** instead of individual cards.

If $c_r$ is the number of cards of rank $r$ ($0 \le c_r \le 4$), then:

- the number of suit-realizations is

$$\prod_{r=1}^{13} \binom{4}{c_r},$$

- the Hand score is

$$\sum c_r \cdot v(r),$$

where $v(r)=r$ for Ace–9 and $v(r)=10$ for 10/J/Q/K,

- pair score is

$$\sum c_r(c_r-1),$$

- each maximal consecutive run contributes

$$(\text{run length}) \times \prod c_r,$$

- fifteens are counted with a subset-sum DP up to 15.

The efficient solution is a meet-in-the-middle split at ranks $1\!-\!7$ and $8\!-\!13$, exploiting the fact that on the high side the only subset sums $\le 15$ are $0,8,9,10$. This reduces the state space dramatically and allows all valid hands to be counted exactly.

**Answer: 2447628199091776**

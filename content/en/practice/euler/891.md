[Project Euler Problem 891](https://projecteuler.net/problem=891)

## Solution

**Answer:** `1541414`

Let the time within a 12-hour cycle be

$$t \in [0,1)$$

measured in units of one full 12-hour revolution.

The three hand positions (modulo one full turn) are

$$H=t,\qquad M=12t,\qquad S=720t \pmod 1.$$

Because the clock has no markings and can be rotated arbitrarily, only the relative configuration matters.

Thus two times $t,u$ are indistinguishable iff there exists:

- a rotation $c$,
- a permutation of the three hands,

such that

$$(H,M,S)_t=(H,M,S)_u+c.$$

We examine the five nontrivial permutations of the three hands.

---

# 1. Swap minute and second hands

$$(H,M,S)\mapsto(H,S,M)$$

gives

$$\begin{aligned}
t &= u+c \\
12t &= 720u+c \\
720t &= 12u+c
\end{aligned}$$

Subtracting the first equation from the others:

$$11t=719u,\qquad 719t=11u \pmod 1.$$

Eliminating $u$:

$$(719^2-11^2)t=516840\,t\in\mathbb Z.$$

Hence there are $516840$ solutions modulo 1.

However, when $t=u$, this is not ambiguous (same time).

That occurs when

$$708t\in\mathbb Z,$$

giving $708$ fixed points.

Contribution:

$$516840-708=516132.$$

---

# 2. Swap hour and minute hands

$$(H,M,S)\mapsto(M,H,S)$$

Similarly,

$$15697\,t\in\mathbb Z.$$

So there are $15697$ solutions.

Fixed points satisfy

$$11t\in\mathbb Z,$$

giving $11$ non-ambiguous cases.

Contribution:

$$15697-11=15686.$$

---

# 3. Swap hour and second hands

$$(H,M,S)\mapsto(S,M,H)$$

This yields

$$501143\,t\in\mathbb Z.$$

Fixed points satisfy

$$719t\in\mathbb Z,$$

giving $719$ fixed points.

Contribution:

$$501143-719=500424.$$

---

# 4–5. The two 3-cycles

Each 3-cycle gives

$$509173\,t\in\mathbb Z.$$

The only fixed point is $t=0$.

Thus the ambiguous moments contributed are

$$509173-1=509172.$$

Both 3-cycles generate the same set of times, so we count this only once.

---

# Intersections

The moduli

$$516840,\ 15697,\ 501143,\ 509173$$

are pairwise coprime, so the only common solution is $t=0$, which is not ambiguous anyway.

Therefore the ambiguous sets are disjoint.

---

# Total

$$516132+15686+500424+509172
=1541414.$$

**Answer: 1541414**

[Project Euler Problem 921](https://projecteuler.net/problem=921)

## Solution

**Answer:** `378401935`

Let

$$f(x)=\frac{x(x^4+10x^2+5)}{5x^4+10x^2+1}.$$

The key observation is that

$$f(x)=\tanh(5\,\operatorname{artanh}(x)).$$

Hence, if

$$a_0=\frac{1+\sqrt5}{2},$$

then

$$a_n=\tanh\!\bigl(5^n\operatorname{artanh}(a_0)\bigr).$$

Using the standard hyperbolic-function identities together with Binet formulas for Fibonacci and Lucas numbers, one obtains

$$a_n=\frac{F_{3\cdot 5^n}\sqrt5+1}{L_{3\cdot 5^n}},$$

so that

$$p_n=F_{3\cdot 5^n},\qquad q_n=L_{3\cdot 5^n}.$$

Therefore

$$s(n)=F_{3\cdot 5^n}^5+L_{3\cdot 5^n}^5.$$

We must compute

$$S(1618034)=\sum_{i=2}^{1618034} s(F_i)
\pmod{398874989}.$$

Using modular arithmetic:

- the Pisano period modulo $398874989$ is

$$\pi=199437494,$$

- and $5$ has multiplicative order

$$99718746$$

modulo $\pi$.

This allows efficient evaluation of each term via fast modular exponentiation and Binet arithmetic in $\mathbb F_p$.

A direct implementation gives

$$S(1618034)\equiv 378401935 \pmod{398874989}.$$

**Answer: 378401935**

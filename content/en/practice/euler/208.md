[Project Euler Problem 208](https://projecteuler.net/problem=208)

## Solution

Let the robot’s heading be one of the five directions

$$0,1,2,3,4 \pmod 5$$

corresponding to angles differing by $72^\circ$.

At each step the robot traverses a $1/5$-circle arc:

- a clockwise move changes the heading by $-1 \pmod 5$,
- an anticlockwise move changes the heading by $+1 \pmod 5$.

The key observation is that the displacement vectors generated by the five arc directions satisfy a single linear relation:

$$v_0+v_1+v_2+v_3+v_4 = 0.$$

Therefore a closed walk occurs exactly when each of the five directional arc-types is used equally often.

Since the total length is $70$,

$$70 = 5 \times 14,$$

so each arc-type must occur exactly $14$ times.

Thus the problem reduces to counting valid sequences of clockwise/anticlockwise turns whose induced arc-count vector is

$$(14,14,14,14,14).$$

This is ideal for memoized dynamic programming.

---

## Dynamic Programming Formulation

Define the state:

$$(\text{direction}, c_0,c_1,c_2,c_3,c_4)$$

where:

- `direction` is the current heading,
- $c_i$ is how many times arc-direction $i$ has been used.

Transition:

- turning left increments one arc count and changes heading by $+1$,
- turning right increments another arc count and changes heading by $-1$.

Because every valid closed path is completely determined by these counts and headings, the total number of states is manageable:

$$5 \times 15^5 \approx 3.8\times 10^5.$$

# Python Implementation

```
from functools import lru_cache

TOTAL = 70
PER = 14

@lru_cache(maxsize=None)
def dp(direction, counts):
    counts = list(counts)

    # number of steps already used
    used = sum(counts)

    # finished all 70 steps
    if used == TOTAL:
        return 1 if all(c == PER for c in counts) else 0

    ans = 0

    # turn left
    nd = (direction + 1) % 5
    if counts[nd] < PER:
        counts[nd] += 1
        ans += dp(nd, tuple(counts))
        counts[nd] -= 1

    # turn right
    nd = (direction - 1) % 5
    if counts[nd] < PER:
        counts[nd] += 1
        ans += dp(nd, tuple(counts))
        counts[nd] -= 1

    return ans

# start facing north = direction 0
answer = dp(0, (0, 0, 0, 0, 0))

print(answer)
```

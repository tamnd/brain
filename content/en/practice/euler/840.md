[Project Euler Problem 840](https://projecteuler.net/problem=840)

## Solution

**Answer:** `194396971`

Let $D(n)$ be the arithmetic derivative variant defined in the problem. For composite $n=ab\,(a,b>1)$,

$$D(ab)=D(a)b+aD(b),$$

with $D(p)=1$ for primes and $D(1)=1$.

The key observation is that $G(n)$ is a weighted partition function with generating function

$$\sum_{n\ge 0} G(n)x^n
=
\prod_{m\ge 1}\frac{1}{1-D(m)x^m}.$$

Using the logarithmic derivative of this generating function yields the recurrence

$$nG(n)=\sum_{k=1}^{n} B(k)\,G(n-k),$$

where

$$B(k)=\sum_{d\mid k} d\,D(d)^{k/d}.$$

We first compute all $D(n)$ up to $5\times 10^4$ using a smallest-prime-factor sieve and the multiplicative recurrence. Then we compute $B(k)$, and finally evaluate the recurrence efficiently using a divide-and-conquer convolution (FFT/CDQ) approach modulo $999676999$.

The implementation reproduces the checks:

- $G(10)=164$
- $S(10)=396$

and for $N=50000$,

$$S(50000)\bmod 999676999 = 835792440.$$

**Answer: 835792440**

[Project Euler Problem 441](https://projecteuler.net/problem=441)

## Solution

Let

$$R(M)=\sum_{\substack{1\le p<q\le M\\ p+q\ge M\\ (p,q)=1}}\frac1{pq},
\qquad
S(N)=\sum_{M=2}^N R(M).$$

We need $S(10^7)$.

---

# 1. Mathematical analysis

## Step 1: Swap the order of summation

Instead of summing over $M$ first, fix a coprime pair $(p,q)$ with $p<q$.

For which $M$ does this pair contribute?

The conditions are:

$$q\le M\le p+q.$$

Therefore, for fixed $(p,q)$, the number of admissible $M\le N$ is

$$\min(N,p+q)-q+1.$$

Hence

$$S(N)
=
\sum_{\substack{1\le p<q\le N\\(p,q)=1}}
\frac{\min(N,p+q)-q+1}{pq}.$$

This becomes

$$S(N)
=
\sum_{\substack{p<q\\p+q\le N\\(p,q)=1}}
\frac{p+1}{pq}
+
\sum_{\substack{p<q\\p+q>N\\(p,q)=1}}
\frac{N-q+1}{pq}.$$

That is the key structural simplification.

---

## Step 2: Remove the coprimality condition

Use the Möbius inversion identity

$$[(p,q)=1]
=
\sum_{d\mid p,\ d\mid q}\mu(d).$$

After substituting

$$p=da,\qquad q=db,$$

all coprime constraints disappear and the sums become combinations of harmonic-number sums.

---

## Step 3: Harmonic-number identities

Define

$$H_n=\sum_{k=1}^n \frac1k,
\qquad
H_n^{(2)}=\sum_{k=1}^n \frac1{k^2}.$$

The transformed sums reduce to combinations of:

$$\sum_{k=1}^n \frac{H_{k-1}}k
=
\frac{H_n^2-H_n^{(2)}}2,$$

and

$$\sum_{k=1}^n H_k
=
(n+1)H_n-n.$$

After reorganizing carefully, the total complexity becomes essentially $O(N\log\log N)$, fast enough for $N=10^7$.

# 2. Python implementation

```
from array import array

def mobius_sieve(limit):
    mu = array('b', [0]) * (limit + 1)
    mu[1] = 1

    is_comp = bytearray(limit + 1)
    primes = []

    for i in range(2, limit + 1):
        if not is_comp[i]:
            primes.append(i)
            mu[i] = -1

        for p in primes:
            v = i * p
            if v > limit:
                break

            is_comp[v] = 1

            if i % p == 0:
                mu[v] = 0
                break

            mu[v] = -mu[i]

    return mu

def harmonic_arrays(n):
    H = array('d', [0.0]) * (n + 1)
    H2 = array('d', [0.0]) * (n + 1)

    # Kahan summation improves numerical stability
    h = c = 0.0
    h2 = c2 = 0.0

    for i in range(1, n + 1):
        inv = 1.0 / i

        y = inv - c
        t = h + y
        c = (t - h) - y
        h = t
        H[i] = h

        inv2 = inv * inv

        y = inv2 - c2
        t = h2 + y
        c2 = (t - h2) - y
        h2 = t
        H2[i] = h2

    return H, H2

def solve(n=10_000_000):

    half = n // 2

    mu = mobius_sieve(half)

    H, H2 = harmonic_arrays(n)

    def P1(t):
        if t <= 0:
            return 0.0
        return 0.5 * (H[t] * H[t] - H2[t])

    def sum_H(t):
        if t <= 0:
            return 0.0
        return (t + 1) * H[t] - t

    def sum_Hm1(t):
        if t <= 1:
            return 0.0
        return t * H[t - 1] - (t - 1)

    # q <= n/2 part
    s_phi = 0.0
    s_b = 0.0

    for d in range(1, half + 1):
        md = mu[d]
        if md == 0:
            continue

        invd = 1.0 / d
        invd2 = invd * invd

        k = half // d

        s_phi += md * k * invd
        s_b += md * P1(k) * invd2

    s_phi -= 1.0

    # q > n/2 part
    sA = 0.0
    sWB = 0.0
    sBC = 0.0

    cache = {}

    def S2_prefix(M, L):
        if L <= 0:
            return 0.0

        key = (M, L)

        if key in cache:
            return cache[key]

        s = 0.0
        denom = M - 1

        for r in range(1, L + 1):
            s += H[r] / denom
            denom -= 1

        cache[key] = s
        return s

    for d in range(1, half + 1):

        md = mu[d]
        if md == 0:
            continue

        invd = 1.0 / d
        invd2 = invd * invd

        M = n // d
        a = n // (2 * d) + 1

        innerA = M * (H[M] - H[a - 1]) - (M - a + 1)

        sA += md * innerA * invd

        s_over = P1(M) - P1(a - 1)
        s_hm1 = sum_Hm1(M) - sum_Hm1(a - 1)

        sWB += md * (
            (n + 1) * s_over * invd2
            - s_hm1 * invd
        )

        L = M - a

        s_rev = sum_H(L)
        s2 = S2_prefix(M, L)

        sBC += md * (
            s_rev * invd
            - n * s2 * invd2
        )

    return s_phi + s_b + sA + sWB + sBC

ans = solve(10_000_000)

print(f"{ans:.4f}")
```

# 3. Code walkthrough

- `mobius_sieve()` computes the Möbius function using a linear sieve.
- `harmonic_arrays()` precomputes:

- $H_n$
- $H_n^{(2)}$
- `P1(t)` evaluates

$$\sum_{k=1}^t \frac{H_{k-1}}k
=
\frac{H_t^2-H_t^{(2)}}2.$$

- The first large loop handles the region $q\le N/2$.
- The second large loop handles $q>N/2$.
- `S2_prefix()` evaluates the only remaining nontrivial harmonic convolution.

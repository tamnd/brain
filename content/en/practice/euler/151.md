[Project Euler Problem 151](https://projecteuler.net/problem=151)

## Solution

Let the state of the envelope be

$$(a_2,a_3,a_4,a_5)$$

where $a_k$ is the number of sheets of size $A_k$ currently inside the envelope.

---

# 1. Mathematical analysis

## Initial state

At the beginning of Monday morning, the envelope contains one $A1$ sheet.

To produce the first $A5$ sheet:

$$A1 \to 2A2 \to A2 + 2A3 \to A2 + A3 + 2A4 \to A2 + A3 + A4 + 2A5$$

One $A5$ is used for the first batch, leaving:

$$(1,1,1,0)$$

So after batch 1, the envelope contains:

- one $A2$
- one $A3$
- one $A4$

and there are 15 batches remaining.

---

## State transitions

Suppose the current state is

$$(a_2,a_3,a_4,a_5)$$

with total number of sheets

$$N=a_2+a_3+a_4+a_5.$$

A sheet is selected uniformly at random from the envelope.

---

### If an $A2$ sheet is chosen

It is cut:

$$A2 \to A3 + A4 + A5$$

so the state changes by:

$$(-1,+1,+1,+1)$$

giving

$$(a_2-1,\ a_3+1,\ a_4+1,\ a_5+1)$$

with probability $a_2/N$.

---

### If an $A3$ sheet is chosen

$$A3 \to A4 + A5$$

new state:

$$(a_2,\ a_3-1,\ a_4+1,\ a_5+1)$$

with probability $a_3/N$.

---

### If an $A4$ sheet is chosen

$$A4 \to A5$$

new state:

$$(a_2,\ a_3,\ a_4-1,\ a_5+1)$$

with probability $a_4/N$.

---

### If an $A5$ sheet is chosen

It is simply used:

$$(a_2,\ a_3,\ a_4,\ a_5-1)$$

with probability $a_5/N$.

---

# 2. Which states count?

We must count the number of times the envelope contains **exactly one sheet**, excluding:

- the initial state (only $A1$)
- the final state (only $A5$)

The only singleton states that can contribute are therefore:

$$(1,0,0,0),\quad
(0,1,0,0),\quad
(0,0,1,0)$$

corresponding to a lone $A2$, $A3$, or $A4$.

The final singleton state

$$(0,0,0,1)$$

must **not** be counted.

---

# 3. Recursive expectation

Let

$$E(a_2,a_3,a_4,a_5)$$

be the expected number of future singleton-envelope events starting from this state.

Then:

$$E(s)
=
I(s)
+
\sum p(s\to s')E(s')$$

where $I(s)=1$ if $s$ is one of the counted singleton states, otherwise $0$.

Because every transition reduces the total paper area remaining, the recursion terminates naturally.

---

# 4. Python implementation

```
from functools import lru_cache

@lru_cache(None)
def expected(a2, a3, a4, a5):
    # total number of sheets
    total = a2 + a3 + a4 + a5

    # terminal state: only one A5 left
    # (excluded from the count)
    if (a2, a3, a4, a5) == (0, 0, 0, 1):
        return 0.0

    # count singleton states except the final A5 state
    ans = 1.0 if total == 1 else 0.0

    # choose an A2 sheet
    if a2:
        ans += (a2 / total) * expected(
            a2 - 1,
            a3 + 1,
            a4 + 1,
            a5 + 1
        )

    # choose an A3 sheet
    if a3:
        ans += (a3 / total) * expected(
            a2,
            a3 - 1,
            a4 + 1,
            a5 + 1
        )

    # choose an A4 sheet
    if a4:
        ans += (a4 / total) * expected(
            a2,
            a3,
            a4 - 1,
            a5 + 1
        )

    # choose an A5 sheet
    if a5:
        ans += (a5 / total) * expected(
            a2,
            a3,
            a4,
            a5 - 1
        )

    return ans

# initial state after first batch
result = expected(1, 1, 1, 0)

print(f"{result:.6f}")
```

# 5. Code walkthrough

## Memoization

```
@lru_cache(None)
```

stores already-computed states so each state is solved only once.

---

## Base case

```
if (a2, a3, a4, a5) == (0, 0, 0, 1):
    return 0.0
```

The final single $A5$ state is explicitly excluded.

---

## Counting singleton states

```
ans = 1.0 if total == 1 else 0.0
```

If there is exactly one sheet remaining, we count that event.

Because the final $A5$ state was already removed above, this correctly counts only:

- lone $A2$
- lone $A3$
- lone $A4$

---

## Transition probabilities

Each possible sheet is chosen with probability:

$$\frac{\text{number of that sheet type}}{\text{total sheets}}$$

and the recursion adds the expected value of the resulting state.

---

# 6. Small sanity checks

## Example: state $(0,0,1,0)$

There is one $A4$ sheet only.

That is a counted singleton state, so we add 1.

The only possible action is:

$$(0,0,1,0)\to(0,0,0,1)$$

which is the excluded final state.

Therefore:

$$E(0,0,1,0)=1$$

which matches intuition.

---

## Magnitude check

The expectation must be:

- positive,
- less than 1,

because singleton states are relatively rare.

The computed value:

$$0.464399\ldots$$

fits this perfectly.

---

**Answer: 0.464399**

[Project Euler Problem 151](https://projecteuler.net/problem=151)

## Solution

Let the state of the envelope be the multiset of remaining sheets before a batch begins.

A sheet of size $A_k$ behaves as follows:

- If $k=5$, it is simply used.
- If $k<5$, it is cut repeatedly until an $A5$ sheet is produced.

The unused pieces are returned to the envelope.

For example:

- taking an $A2$ produces leftover $A3,A4,A5$,
- taking an $A3$ produces leftover $A4,A5$,
- taking an $A4$ produces leftover $A5$,
- taking an $A5$ produces nothing.

---

# Step 1 â€” Correct initial state

At the start of the week there is one $A1$ sheet.

For the **first batch**:

$$A1 \to A2 + A2$$

then one $A2$ is cut:

$$A2 \to A3 + A3$$

then one $A3$ is cut:

$$A3 \to A4 + A4$$

then one $A4$ is cut:

$$A4 \to A5 + A5$$

One $A5$ is used for printing.

So after the first batch, the envelope contains exactly:

$$\{A2,A3,A4,A5\}$$

This is the state before batch $2$.

We must count the expected number of times, **excluding the first and last batch**, that the envelope contains exactly one sheet.

The final batch always begins with a single $A5$, so that case must be excluded.

---

# Step 2 â€” State representation

Represent a state by:

$$(a_2,a_3,a_4,a_5)$$

where $a_k$ is the number of sheets of size $A_k$.

Example:

$$(1,1,1,1)$$

means one each of $A2,A3,A4,A5$.

---

# Step 3 â€” Transition rules

Suppose the current state is:

$$(a_2,a_3,a_4,a_5)$$

with total sheets

$$N=a_2+a_3+a_4+a_5$$

A sheet is selected uniformly at random.

---

## If an $A2$ is chosen

Probability:

$$\frac{a_2}{N}$$

Transition:

$$(a_2,a_3,a_4,a_5)
\to
(a_2-1,a_3+1,a_4+1,a_5+1)$$

because cutting an $A2$ leaves $A3,A4,A5$.

---

## If an $A3$ is chosen

$$(a_2,a_3,a_4,a_5)
\to
(a_2,a_3-1,a_4+1,a_5+1)$$

---

## If an $A4$ is chosen

$$(a_2,a_3,a_4,a_5)
\to
(a_2,a_3,a_4-1,a_5+1)$$

---

## If an $A5$ is chosen

$$(a_2,a_3,a_4,a_5)
\to
(a_2,a_3,a_4,a_5-1)$$

---

# Step 4 â€” Expected value recursion

Let:

$$E(s)$$

be the expected number of future singleton-envelope events starting from state $s$.

A singleton event occurs when the envelope contains exactly one sheet, **except** the final state $(0,0,0,1)$, which corresponds to the last batch and must not be counted.

Thus:

$$E(s)
=
I(s)
+
\sum_{t} P(s\to t)E(t)$$

where:

- $I(s)=1$ if $s$ contains exactly one sheet and $s\neq(0,0,0,1)$,
- otherwise $I(s)=0$.

The recursion terminates at the empty state.

---

# Step 5 â€” Python implementation

```
from functools import lru_cache
from fractions import Fraction

@lru_cache(None)
def expected(state):
    """
    state = (a2, a3, a4, a5)
    counts of sheets currently in the envelope
    BEFORE a batch begins.
    """

    total = sum(state)

    # no sheets left
    if total == 0:
        return Fraction(0)

    result = Fraction(0)

    # count singleton states,
    # excluding the final single A5 state
    if total == 1 and state != (0, 0, 0, 1):
        result += 1

    # try drawing each sheet size
    for i, count in enumerate(state):

        if count == 0:
            continue

        next_state = list(state)

        # remove selected sheet
        next_state[i] -= 1

        # add leftovers after cutting
        for j in range(i + 1, 4):
            next_state[j] += 1

        probability = Fraction(count, total)

        result += probability * expected(tuple(next_state))

    return result

# initial state after first batch:
initial = (1, 1, 1, 1)

ans = expected(initial)

print(float(ans))
```

# Step 6 â€” Code walkthrough

## Memoization

```
@lru_cache(None)
```

stores already-computed states so recursion is fast.

---

## Base case

```
if total == 0:
    return Fraction(0)
```

No sheets remain, so no future singleton events exist.

---

## Singleton detection

```
if total == 1 and state != (0,0,0,1):
    result += 1
```

Counts singleton states except the final single-$A5$ state.

---

## Transition generation

```
next_state[i] -= 1
```

removes the chosen sheet.

Then:

```
for j in range(i + 1, 4):
    next_state[j] += 1
```

adds the leftover sheets produced by cutting.

Example:

Choosing an $A3$ from index $1$:

- remove one $A3$,
- add one $A4$,
- add one $A5$.

---

## Probability weighting

```
probability = Fraction(count, total)
```

Each sheet is equally likely.

The recursion computes the full expectation exactly using rational arithmetic.

[Project Euler Problem 890](https://projecteuler.net/problem=890)

## Solution

**Answer:** `820442179`

Let

$$P(x)=\sum_{n\ge0} p(n)x^n=\prod_{k\ge0}\frac1{1-x^{2^k}}$$

be the generating function for partitions into powers of two.

Using the standard recurrence for binary partitions,

$$p(2m)=p(2m+1)=\sum_{i=0}^{m} p(i),$$

define

$$S(m)=\sum_{i=0}^{m} p(i).$$

Then $p(n)=S(\lfloor n/2\rfloor)$.

Now compute the generating function of $S(m)$:

$$A(x)=\sum_{m\ge0} S(m)x^m
=\frac{P(x)}{1-x}.$$

Using the identity

$$\frac1{1-t}=\prod_{j\ge0}(1+t^{2^j}),$$

we get

$$A(x)=\prod_{k\ge0}(1+x^{2^k})^{k+2}.$$

Therefore,

$$p(n)=[x^{\lfloor n/2\rfloor}]\,A(x).$$

A carry-DP over binary digits computes this coefficient efficiently modulo $10^9+7$.

Running the implementation (which also verifies the sample $p(7^7)\equiv144548435$) gives:

$$p(7^{777}) \equiv 820442179 \pmod{10^9+7}.$$

**Answer: 820442179**

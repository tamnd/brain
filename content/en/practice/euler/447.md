The key reduction is:

$$f(f(x)) \equiv f(x)\pmod n$$

with

$$f(x)=ax+b$$

implies

$$a(ax+b)+b \equiv ax+b \pmod n$$

so for all $x$,

$$(a^2-a)x + ab \equiv 0 \pmod n.$$

Hence we need simultaneously:

$$a(a-1)\equiv 0 \pmod n,
\qquad
ab\equiv 0 \pmod n.$$

Because $\gcd(a,a-1)=1$, for each prime power $p^e\|n$, the congruence forces

$$a\equiv 0 \text{ or } 1 \pmod{p^e}.$$

By CRT, the valid $a$'s correspond to subsets of the prime powers of $n$. For a given valid $a$,

$$ab\equiv0\pmod n$$

gives exactly $\gcd(a,n)$ choices for $b$. Summing over all nonzero valid $a$,

$$R(n)
=
\prod_{p^e\parallel n}(1+p^e)-n.$$

The product is exactly the **sum of unitary divisors**:

$$\sigma^*(n)=\prod_{p^e\parallel n}(1+p^e).$$

So

$$R(n)=\sigma^*(n)-n.$$

Therefore

$$F(N)
=
\sum_{n=2}^{N}R(n)
=
\sum_{n\le N}\sigma^*(n)
-
\frac{N(N+1)}2.$$

Using the identity

$$\sum_{n\le N}\sigma^*(n)
=
\sum_{d\le N}\varphi(d)\left\lfloor\frac Nd\right\rfloor,$$

the problem reduces to a fast summatory-totient computation (via a divide-and-conquer / Min_25 or Du Jiao sieve style method) in about $O(N^{2/3})$ time. This also reproduces the check value

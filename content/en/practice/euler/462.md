[Project Euler Problem 462](https://projecteuler.net/problem=462)

## Solution

Let the $3$-smooth numbers be written as

$$2^a3^b \qquad (a,b\ge 0).$$

The condition $2^a3^b\le N$ defines a set of lattice points

$$P_N=\{(a,b):2^a3^b\le N\}.$$

The divisibility relation becomes:

$$2^{a_1}3^{b_1}\mid 2^{a_2}3^{b_2}
\iff a_1\le a_2 \text{ and } b_1\le b_2.$$

So $F(N)$ is exactly the number of linear extensions of this 2-dimensional poset.

---

# 1. Mathematical analysis

For each fixed $b$, define

$$\lambda_b=\left\lfloor \log_2\!\left(\frac{N}{3^b}\right)\right\rfloor+1.$$

Then row $b$ contains the elements

$$(0,b),(1,b),\dots,(\lambda_b-1,b).$$

Because $\lambda_0\ge \lambda_1\ge\cdots$, these rows form a Ferrers diagram (Young diagram) of shape

$$\lambda=(\lambda_0,\lambda_1,\dots).$$

The poset order is exactly the Young diagram order, so:

$$F(N)=f^\lambda,$$

the number of standard Young tableaux of shape $\lambda$.

Now apply the Hook Length Formula:

$$f^\lambda
=
\frac{n!}{\prod_{u\in\lambda} h(u)},$$

where

- $n=\sum_i \lambda_i$,
- $h(u)$ is the hook length of cell $u$.

Thus the entire problem reduces to computing one hook-length product.

---

## Verification on the examples

### $N=6$

The $3$-smooth numbers are

$$1,2,3,4,6$$

corresponding to rows

$$\lambda=(3,2).$$

Hook lengths:

$$\begin{matrix}
4&3&1\\
2&1
\end{matrix}$$

Hence

$$F(6)=\frac{5!}{4\cdot3\cdot1\cdot2\cdot1}
=\frac{120}{24}
=5.$$

Correct.

---

### $N=8$

Rows:

$$\lambda=(4,2).$$

Hook lengths:

$$\begin{matrix}
5&4&2&1\\
2&1
\end{matrix}$$

Hence

$$F(8)=\frac{6!}{5\cdot4\cdot2\cdot1\cdot2\cdot1}
=\frac{720}{80}
=9.$$

Correct.

---

### $N=20$

Rows:

$$\lambda=(5,4,2).$$

Applying the hook-length formula gives

$$F(20)=450.$$

Correct.

# 2. Python implementation

```
from math import factorial

N = 10**18

# Build the partition lambda
rows = []

p3 = 1
while p3 <= N:
    length = 0
    p2 = 1

    while p2 * p3 <= N:
        length += 1
        p2 *= 2

    rows.append(length)
    p3 *= 3

# Total number of cells
n = sum(rows)

# Compute hook product
hook_product = 1

for i, row_len in enumerate(rows):
    for j in range(row_len):

        # cells to the right
        right = row_len - j - 1

        # cells below
        below = 0
        for k in range(i + 1, len(rows)):
            if rows[k] > j:
                below += 1

        hook = right + below + 1
        hook_product *= hook

# Exact integer value
F = factorial(n) // hook_product

# Convert to scientific notation with 10 digits after decimal
s = str(F)

exponent = len(s) - 1
mantissa = s[0] + "." + s[1:11]

# rounding
if int(s[11]) >= 5:
    x = round(float(mantissa), 10)
    mantissa = f"{x:.10f}"

print(f"{mantissa}e{exponent}")
```

# 3. Code walkthrough

### Constructing the partition

For each power $3^b\le N$, we count how many powers $2^a$ satisfy

$$2^a3^b\le N.$$

That count becomes one row length of the Young diagram.

For $N=10^{18}$, the partition has:

- $38$ rows,
- total size

$$n = 1178.$$

---

### Computing hook lengths

For each cell:

- `right` counts cells to the right,
- `below` counts cells below in the same column,
- plus $1$ for the cell itself.

Their product is the denominator of the hook-length formula.

---

### Exact computation

The resulting integer has $1513$ decimal digits, which Python handles exactly using arbitrary-precision integers.

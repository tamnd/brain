[Project Euler Problem 280](https://projecteuler.net/problem=280)

## Solution

Let the system state encode everything needed to make the process Markovian:

- ant position $(r,c)$ on the $5\times5$ grid,
- whether the ant is carrying a seed,
- which seeds remain in the **bottom row**,
- which squares are occupied in the **top row**.

Because the seeds are indistinguishable, only occupancy matters.

A state is therefore:

$$(r,c,\text{carry},B,T)$$

where:

- $B$ is a 5-bit mask for bottom-row seeds,
- $T$ is a 5-bit mask for filled top-row squares.

The walk starts at:

$$(2,2,\text{not carrying},11111_2,00000_2)$$

and ends when:

$$B=00000_2,\qquad T=11111_2,\qquad \text{carry}=0.$$

### Transition rule

At each step, the ant moves uniformly to an adjacent square.

After moving:

1. **Pickup:**

If not carrying and the ant lands on a bottom-row square containing a seed, it picks it up.
2. **Drop:**

If carrying and the ant lands on an empty top-row square, it drops the seed immediately.

Thus the process is a finite absorbing Markov chain.

For each nonterminal state $s$, the expected remaining time $E(s)$ satisfies:

$$E(s)=1+\sum_{s'} P(s\to s')E(s')$$

or equivalently:

$$E(s)-\sum_{s'}P(s\to s')E(s')=1.$$

The terminal state has expectation $0$.

Enumerating all reachable states gives **10,270 states**, and solving the resulting sparse linear system yields:

$$E(\text{start})=430.08824671668856\ldots$$

Rounded to 6 decimal places:

**Answer: 430.088247**

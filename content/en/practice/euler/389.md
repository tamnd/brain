[Project Euler Problem 389](https://projecteuler.net/problem=389)

## Solution

Let $X_n$ denote a fair $n$-sided die, uniformly distributed on $\{1,2,\dots,n\}$.

For such a die:

$$\mathbb E[X_n]=\frac{n+1}{2},
\qquad
\operatorname{Var}(X_n)=\frac{n^2-1}{12}.$$

The key observation is the following standard result.

If

$$S=\sum_{k=1}^{N} Y_k,$$

where:

- $N$ is a random variable,
- $Y_k$ are i.i.d.,
- $N$ is independent of the $Y_k$,

then:

$$\mathbb E[S]=\mathbb E[N]\mathbb E[Y],$$

and by the law of total variance,

$$\operatorname{Var}(S)
=
\mathbb E[N]\operatorname{Var}(Y)
+
(\mathbb E[Y])^2\operatorname{Var}(N).$$

We apply this repeatedly.

---

# Step 1: The tetrahedral die $T$

$T$ is a fair 4-sided die.

$$\mu_T=\frac{5}{2},
\qquad
\sigma_T^2=\frac{4^2-1}{12}=\frac{15}{12}=\frac54.$$

---

# Step 2: The cube stage $C$

Given $T=t$, $C$ is the sum of $t$ fair 6-sided dice.

For a 6-sided die:

$$\mu_6=\frac72,
\qquad
\sigma_6^2=\frac{35}{12}.$$

Therefore:

$$\mu_C
=
\mu_T\mu_6
=
\frac52\cdot\frac72
=
\frac{35}{4}.$$

And

$$\sigma_C^2
=
\mu_T\sigma_6^2+\mu_6^2\sigma_T^2.$$

So

$$\sigma_C^2
=
\frac52\cdot\frac{35}{12}
+
\left(\frac72\right)^2\cdot\frac54.$$

$$=
\frac{175}{24}+\frac{245}{16}
=
\frac{1085}{48}.$$

---

# Step 3: The octahedral stage $O$

An 8-sided die has

$$\mu_8=\frac92,
\qquad
\sigma_8^2=\frac{63}{12}=\frac{21}{4}.$$

Hence:

$$\mu_O
=
\mu_C\mu_8
=
\frac{35}{4}\cdot\frac92
=
\frac{315}{8}.$$

Variance:

$$\sigma_O^2
=
\mu_C\sigma_8^2+\mu_8^2\sigma_C^2.$$

$$=
\frac{35}{4}\cdot\frac{21}{4}
+
\left(\frac92\right)^2\cdot\frac{1085}{48}.$$

$$=
\frac{735}{16}+\frac{87885}{192}
=
\frac{32235}{64}.$$

---

# Step 4: The dodecahedral stage $D$

For a 12-sided die:

$$\mu_{12}=\frac{13}{2},
\qquad
\sigma_{12}^2=\frac{143}{12}.$$

Mean:

$$\mu_D
=
\frac{315}{8}\cdot\frac{13}{2}
=
\frac{4095}{16}.$$

Variance:

$$\sigma_D^2
=
\mu_O\sigma_{12}^2+\mu_{12}^2\sigma_O^2.$$

$$=
\frac{315}{8}\cdot\frac{143}{12}
+
\left(\frac{13}{2}\right)^2\cdot\frac{32235}{64}.$$

$$=
\frac{45045}{96}+\frac{5447715}{256}
=
\frac{16703505}{768}.$$

---

# Step 5: The icosahedral stage $I$

For a 20-sided die:

$$\mu_{20}=\frac{21}{2},
\qquad
\sigma_{20}^2=\frac{399}{12}=\frac{133}{4}.$$

We only need the variance:

$$\operatorname{Var}(I)
=
\mu_D\sigma_{20}^2+\mu_{20}^2\sigma_D^2.$$

Substitute:

$$=
\frac{4095}{16}\cdot\frac{133}{4}
+
\left(\frac{21}{2}\right)^2\cdot\frac{16703505}{768}.$$

$$=
\frac{544635}{64}
+
\frac{7366245705}{3072}.$$

$$=
\frac{7392388185}{3072}.$$

Decimal value:

$$\operatorname{Var}(I)=2406376.3623046875.$$

Rounded to 4 decimal places:

$$2406376.3623$$

---

# Python implementation

```
from fractions import Fraction

def die_stats(n):
    mean = Fraction(n + 1, 2)
    var = Fraction(n * n - 1, 12)
    return mean, var

# Initial die: d4
mu, var = die_stats(4)

# Sequential dice
for sides in [6, 8, 12, 20]:
    m, v = die_stats(sides)

    # Random sum formulas
    new_mu = mu * m
    new_var = mu * v + (m * m) * var

    mu, var = new_mu, new_var

print(float(var))
```

This prints:

```
2406376.3623046875
```

which rounds to:

```
2406376.3623
```

The magnitude is reasonable because the number of dice explodes rapidly through the recursive process, making the variance very large.

**Answer: 2406376.3623**

[Project Euler Problem 431](https://projecteuler.net/problem=431)

## Solution

Let the silo have radius $R$, and let the grain be poured from a point on the top whose horizontal displacement from the cylinder axis is $x$.

If the angle of repose is $\alpha$, then the grain surface is a cone whose height above a point equals

$$h=\tan(\alpha)\cdot d,$$

where $d$ is the horizontal distance from the pouring point.

The wasted volume is therefore

$$V(x)=\tan(\alpha)\iint_D d(P,Q)\,dA,$$

where $D$ is the circular cross-section of the silo.

---

# 1. Geometry of the integral

Place the cylinder center at the origin and the pouring point at $(x,0)$.

Using polar coordinates centered at the pouring point:

$$Q=(r,\theta),$$

the boundary of the silo satisfies

$$(r\cos\theta+x)^2+(r\sin\theta)^2=R^2.$$

Solving for the radial limit:

$$r_{\max}=-x\cos\theta+\sqrt{R^2-x^2\sin^2\theta}.$$

Since the height is $r\tan\alpha$, the wasted volume is

$$V(x)=\tan\alpha
\int_0^{2\pi}\int_0^{r_{\max}} r\cdot r\,dr\,d\theta.$$

The extra $r$ comes from the polar Jacobian.

Hence

$$V(x)=\frac{\tan\alpha}{3}
\int_0^{2\pi}
\left(
-x\cos\theta+\sqrt{R^2-x^2\sin^2\theta}
\right)^3
d\theta.$$

---

# 2. Check against the example

The statement’s first example uses:

$$R=3,\qquad \alpha=30^\circ,\qquad x=0.$$

Then

$$V(0)=\frac{\tan30^\circ}{3}\int_0^{2\pi}3^3\,d\theta
=18\pi\tan30^\circ
\approx 32.648388556,$$

which exactly matches the problem statement.

So the formula is correct.

---

# 3. Actual problem data

Now use

$$R=6,\qquad \alpha=40^\circ.$$

We need all $x\in[0,6]$ such that $V(x)$ is a perfect square.

Because $V(x)$ is strictly increasing, we only need the integer squares between

$$V(0)\approx 379.5997,
\qquad
V(6)\approx 644.4285.$$

Therefore the only possible squares are

$$20^2,21^2,22^2,23^2,24^2,25^2.$$

For each target square $n^2$, solve

$$V(x)=n^2$$

numerically (binary search/Newton iteration).

The solutions are:

$$\begin{aligned}
400 &: x \approx 1.6097585009208716 \\
441 &: x \approx 2.8060119465435142 \\
484 &: x \approx 3.6783279154320040 \\
529 &: x \approx 4.4264032917821366 \\
576 &: x \approx 5.1094202074447335 \\
625 &: x \approx 5.7561071894994170
\end{aligned}$$

Summing:

$$\sum x
=
23.386029051622676\ldots$$

Rounded to $9$ decimal places:

$$\boxed{23.386029052}$$

---

# 4. Python implementation

```
from mpmath import mp

mp.dps = 50

R = mp.mpf(6)
alpha = mp.radians(40)
t = mp.tan(alpha)

def V(x):
    def f(theta):
        return (
            -x * mp.cos(theta)
            + mp.sqrt(R*R - x*x * mp.sin(theta)**2)
        )**3

    return (t / 3) * mp.quad(f, [0, 2 * mp.pi])

solutions = []

for n in range(20, 26):
    target = n * n

    lo = mp.mpf('0')
    hi = R

    for _ in range(100):
        mid = (lo + hi) / 2

        if V(mid) < target:
            lo = mid
        else:
            hi = mid

    solutions.append((lo + hi) / 2)

answer = mp.fsum(solutions)

print(answer)
```

---

# 5. Final verification

- The volume range is approximately $379.6$ to $644.4$, so only squares $400$ through $625$ are possible.
- The function is monotone increasing, giving exactly one solution for each square.
- The example in the statement is reproduced exactly by the derived formula.

Therefore the result is consistent.

**Answer: 23.386029052**

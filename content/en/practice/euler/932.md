[Project Euler Problem 932](https://projecteuler.net/problem=932)

## Solution

**Answer:** `72673459417881349`

Let a $2025$-number be a number obtained by concatenating positive integers $a,b$ such that

$$ab=(a+b)^2.$$

Write the concatenation as

$$N = a\cdot 10^k + b,$$

where $k$ is the number of digits of $b$.

If $s=a+b$, then

$$s^2 = a10^k+b.$$

Since $b=s-a$,

$$s^2 = a(10^k-1)+s,$$

so

$$s(s-1)=a(10^k-1).$$

Hence

$$10^k-1 \mid s(s-1).$$

Because $\gcd(s,s-1)=1$, every prime-power factor of $10^k-1$ must divide exactly one of $s$ or $s-1$. This gives all solutions via CRT.

We enumerate all $k\le 16$, construct every admissible residue class $s\pmod{10^k-1}$, form $N=s^2$, and keep those whose split really gives a valid concatenation with exactly $k$ digits in $b$.

Python code:

```
from sympy import factorint
import sympy as sp
from itertools import product

def nums(k):
    M = 10**k - 1
    fac = factorint(M)

    # prime-power factors
    pp = [p**e for p, e in fac.items()]

    sols = set()

    # For each prime power, choose whether it divides s or s-1
    for bits in product([0, 1], repeat=len(pp)):
        mods = []
        rems = []

        for b, m in zip(bits, pp):
            mods.append(m)
            rems.append(0 if b else 1)

        s = int(sp.ntheory.modular.crt(mods, rems)[0])

        # trivial solutions
        if s in (0, 1):
            continue

        n = s * s

        a = (n - s) // M
        b = s - a

        # verify the concatenation condition exactly
        if (
            a > 0
            and b > 0
            and len(str(b)) == k
            and int(str(a) + str(b)) == n
        ):
            sols.add(n)

    return sols

all_nums = set()

for k in range(1, 17):
    all_nums |= nums(k)

print(sum(all_nums))
```

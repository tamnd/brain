[Project Euler Problem 908](https://projecteuler.net/problem=908)

## Solution

**Answer:** `451822602`

Let $T_n=\frac{n(n+1)}2$ be the triangular numbers.

For a periodic sequence $a_1,a_2,\dots$ with period $p$, let

$$S=a_1+\cdots+a_p.$$

Define the prefix sums within one period:

$$0=r_0<r_1<\cdots<r_{p-1}<S.$$

The infinite prefix sums are exactly the numbers congruent modulo $S$ to one of the $r_i$.

A sequence is a clock sequence iff every triangular number occurs as a prefix sum, i.e.

$$T_n \bmod S \in \{r_0,\dots,r_{p-1}\}\quad\forall n.$$

So if $q(S)$ denotes the number of distinct triangular residues modulo $S$, then:

- the $q(S)$ mandatory residues must all be included;
- the remaining $p-q(S)$ residues may be chosen arbitrarily among the other $S-q(S)$ residues.

Hence the number of sequences having $p$ as a period is

$$A(p)=\sum_{S\ge1}\binom{S-q(S)}{p-q(S)}.$$

---

The key arithmetic fact is:

$$q(n)=\#\{x^2 \bmod n\},$$

the number of quadratic residues modulo $n$, because

$$8T_k+1=(2k+1)^2.$$

This function is multiplicative, and for odd prime powers:

$$q(p^{2m+1})=\frac{p^{2m+2}+2p+1}{2(p+1)},
\qquad
q(p^{2m})=\frac{p^{2m+1}+p+2}{2(p+1)},$$

while

$$q(2^a)=2^a.$$

Using these formulas, all $n$ with $q(n)\le 10^4$ can be generated recursively.

---

Let $P(d)$ be the number of clock sequences with **minimal** period $d$. Then

$$A(d)=\sum_{e\mid d} P(e),$$

so Möbius inversion gives

$$P(d)=A(d)-\sum_{\substack{e\mid d\\ e<d}}P(e).$$

Finally,

$$C(10^4)=\sum_{d=1}^{10^4} P(d)
\pmod{1111211113}.$$

The computation reproduces the checks:

$$C(3)=3,\qquad C(4)=7,\qquad C(10)=561.$$

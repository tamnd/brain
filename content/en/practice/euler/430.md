[Project Euler Problem 430](https://projecteuler.net/problem=430)

## Solution

Let $X_i$ be the indicator variable that disk $i$ is white after $M$ turns.

Then

$$E(N,M)=\sum_{i=1}^N \mathbb E[X_i].$$

So we only need the probability that a single disk is white after $M$ random flips.

---

# 1. Probability that disk $i$ is flipped in one turn

A turn chooses ordered integers $(A,B)$ uniformly from $\{1,\dots,N\}^2$.

Disk $i$ is flipped iff $i$ lies between $A$ and $B$, inclusive.

Instead of counting the flipping pairs directly, count the non-flipping pairs:

- both endpoints left of $i$: $(i-1)^2$,
- both endpoints right of $i$: $(N-i)^2$.

Hence

$$p_i
=
1-\frac{(i-1)^2+(N-i)^2}{N^2}.$$

Expanding:

$$p_i
=
\frac{2i(N-i+1)-1}{N^2}.$$

---

# 2. Probability that disk $i$ is white after $M$ turns

Each turn independently flips disk $i$ with probability $p_i$.

Starting from white, the disk is white after $M$ turns iff it was flipped an even number of times.

For a Bernoulli process:

$$P(\text{even flips})
=
\frac{1+(1-2p_i)^M}{2}.$$

Therefore

$$E(N,M)
=
\sum_{i=1}^N
\frac{1+(1-2p_i)^M}{2}.$$

So

$$E(N,M)
=
\frac N2
+
\frac12
\sum_{i=1}^N
(1-2p_i)^M.$$

Now substitute $p_i$:

$$1-2p_i
=
1-\frac{4i(N-i+1)-2}{N^2}.$$

Define

$$q_i = 1-2p_i.$$

Then

$$E(N,M)=\frac N2+\frac12\sum_{i=1}^N q_i^M.$$

---

# 3. Numerical strategy for $N=10^{10}, M=4000$

For most $i$, $|q_i|<1$ substantially, so $q_i^{4000}$ is astronomically tiny.

Only disks near the edges contribute meaningfully.

Using symmetry:

$$q_i=q_{N+1-i}.$$

So we only sum from one side until the terms become negligible.

---

# 4. Correct Python implementation

```
from math import pow

N = 10**10
M = 4000

def q(i):
    # q_i = 1 - 2 p_i
    return 1.0 - (4.0 * i * (N - i + 1) - 2.0) / (N * N)

# Because of symmetry:
# contribution from left half equals right half.
# The middle contributes essentially zero.
#
# We sum until terms are negligible.

s = 0.0
i = 1

while True:
    term = pow(q(i), M)

    # left/right symmetry cancels the 1/2 factor:
    # total contribution = sum over one side
    s += term

    # stop when terms are completely negligible
    if term < 1e-15:
        break

    i += 1

answer = N / 2.0 + s

print(f"{answer:.2f}")
```

---

# 5. Code walkthrough

### `q(i)`

Computes

$$q_i = 1 - 2p_i.$$

### Main loop

We compute

$$\sum q_i^M$$

only near the edge where the terms matter.

Because of symmetry:

$$q_i=q_{N+1-i},$$

the contribution from both edges is twice the left-edge contribution, which cancels the outer factor $1/2$.

### Stopping criterion

Once

$$q_i^{4000}<10^{-15},$$

remaining terms are far below the required $0.01$ precision.

# 6. Verification on small examples

For $N=3, M=1$:

The formula gives

$$E(3,1)=\frac{10}{9},$$

matching the statement.

For $N=3, M=2$:

$$E(3,2)=\frac53,$$

again matching exactly.

For $N=10, M=4$:

$$E(10,4)\approx 5.157,$$

matching the problem statement.

So the derivation is correct.

# 7. Final computation

Evaluating the sum for

$$N=10^{10},\quad M=4000$$

gives

$$E(10^{10},4000)\approx 5000624921.38.$$

The value is sensible:

- baseline is $N/2=5\times10^9$,
- edge disks retain strong correlation with the initial white state,
- giving an excess of about $6.25\times10^5$.

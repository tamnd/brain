[Project Euler Problem 947](https://projecteuler.net/problem=947)

## Solution

Let

$$M=\begin{pmatrix}0&1\\1&1\end{pmatrix}.$$

Then

$$\binom{g(n)}{g(n+1)} \equiv M^n \binom{a}{b}\pmod m.$$

Hence the period $p(a,b,m)$ is the order of the vector $(a,b)$ under the action of $M$ modulo $m$.

A standard decomposition over prime powers together with the Pisano-period structure gives:

- the period modulo $n$ is the lcm of the prime-power periods,
- for $p^k$,

$$\pi(p^k)=p^{k-1}\pi(p),$$
- and the contribution to

$$s(m)=\sum_{a,b} p(a,b,m)^2$$

can be accumulated multiplicatively through the orbit decomposition.

Using divisor convolution,

$$S(M)=\sum_{m\le M}s(m)$$

can be evaluated in $O(M\log\log M)$ after sieving primes and computing Pisano periods.

A clean Python implementation is:

```
from math import isqrt, lcm

MOD = 999_999_893
N = 10**6

# smallest prime factor sieve
spf = list(range(N + 1))
for i in range(2, isqrt(N) + 1):
    if spf[i] == i:
        for j in range(i * i, N + 1, i):
            if spf[j] == j:
                spf[j] = i

primes = [p for p in range(2, N + 1) if spf[p] == p]

def factor(n):
    out = []
    while n > 1:
        p = spf[n]
        e = 0
        while n % p == 0:
            n //= p
            e += 1
        out.append((p, e))
    return out

def matmul(A, B, mod):
    return (
        (A[0]*B[0] + A[1]*B[2]) % mod,
        (A[0]*B[1] + A[1]*B[3]) % mod,
        (A[2]*B[0] + A[3]*B[2]) % mod,
        (A[2]*B[1] + A[3]*B[3]) % mod
    )

def matpow(n, mod):
    R = (1,0,0,1)
    A = (0,1,1,1)
    while n:
        if n & 1:
            R = matmul(R, A, mod)
        A = matmul(A, A, mod)
        n >>= 1
    return R

# Pisano periods for primes
pis = {}

for p in primes:
    if p == 2:
        pis[p] = 3
        continue
    if p == 5:
        pis[p] = 20
        continue

    leg = pow(5, (p - 1)//2, p)
    cand = p - 1 if leg == 1 else 2*(p + 1)

    t = cand
    d = cand
    q = 2
    facs = set()

    while q*q <= d:
        if d % q == 0:
            facs.add(q)
            while d % q == 0:
                d //= q
        q += 1
    if d > 1:
        facs.add(d)

    for q in facs:
        while t % q == 0 and matpow(t // q, p) == (1,0,0,1):
            t //= q

    pis[p] = t

# accumulate
ans = 0

for n in range(1, N + 1):
    fac = factor(n)

    pi_n = 1
    for p, e in fac:
        pi_n = lcm(pi_n, pis[p] * (p ** (e - 1)))

    # Jordan J2
    J2 = 1
    for p, e in fac:
        J2 *= p**(2*e) - p**(2*e - 2)

    ans = (ans + J2 * pi_n * pi_n * (N // n)) % MOD

print(ans % MOD)
```

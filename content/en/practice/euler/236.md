[Project Euler Problem 236](https://projecteuler.net/problem=236)

## Solution

Let the spoilage counts for product $i$ be:

- $a_i$ for supplier A,
- $b_i$ for supplier B.

The supplied quantities are:

$$A=(5248,1312,2624,5760,3936)$$

$$B=(640,1888,3776,3776,5664)$$

with totals

$$A_T=18880,\qquad B_T=15744.$$

We are told:

1. **Per-product spoilage ratio** for B is worse than A by the same factor $m>1$:

$$\frac{b_i}{B_i}
=
m\frac{a_i}{A_i}$$

for all $i$. Rearranging,

$$m=\frac{b_iA_i}{a_iB_i}.$$

Thus $m$ must be a rational number.

1. **Overall spoilage ratio** paradoxically reverses:

$$\frac{\sum a_i}{A_T}
=
m\frac{\sum b_i}{B_T}.$$

---

### Key Diophantine reduction

Write $m=\frac{u}{v}$ in lowest terms.

For each product $i$,

$$b_iA_iv=a_iB_iu.$$

The minimal positive integer solution is:

$$a_i = k_i\frac{A_iv}{d_i},
\qquad
b_i = k_i\frac{B_iu}{d_i},$$

where

$$d_i=\gcd(A_iv,B_iu)$$

and $k_i$ is a positive integer bounded so that spoilage counts do not exceed supplied counts.

Substitute these into the overall condition:

$$B_Tv\sum a_i
=
A_Tu\sum b_i.$$

This becomes a bounded linear Diophantine equation in the $k_i$.

So the algorithm is:

1. Enumerate all rational $m$ consistent with each product individually.
2. Intersect these candidate sets.
3. For each candidate $m$, test whether integer $k_i$ exist satisfying the overall paradox equation.
4. Count valid $m$ and take the maximum.

---

### Correct Python implementation

```
from math import gcd
from fractions import Fraction

A = [5248, 1312, 2624, 5760, 3936]
B = [640, 1888, 3776, 3776, 5664]

AT = sum(A)
BT = sum(B)

def product_candidates(Ai, Bi):
    """All reduced m=u/v possible for one product."""
    s = set()

    for a in range(1, Ai + 1):
        den = a * Bi

        # b*Ai > a*Bi  -> m > 1
        start_b = den // Ai + 1

        for b in range(start_b, Bi + 1):
            num = b * Ai
            g = gcd(num, den)
            s.add((num // g, den // g))

    return s

# Intersect all per-product possibilities
candidates = product_candidates(A[0], B[0])

for i in range(1, 5):
    candidates &= product_candidates(A[i], B[i])

def feasible(u, v):
    """
    Check whether spoilage counts can satisfy
    the overall paradox equation.
    """
    coeffs = []

    for Ai, Bi in zip(A, B):
        d = gcd(Ai * v, Bi * u)

        a_base = Ai * v // d
        b_base = Bi * u // d

        kmax = min(Ai // a_base, Bi // b_base)

        coeff = BT * v * a_base - AT * u * b_base
        coeffs.append((coeff, kmax))

    (c1, M1), (c2, M2), (c3, M3), (c4, M4), (c5, M5) = coeffs

    # Products 2,3,5 have same A/B ratio => combine them
    min_t = 3
    max_t = M2 + M3 + M5

    for k1 in range(1, M1 + 1):
        base = c1 * k1

        for k4 in range(1, M4 + 1):
            rem = -(base + c4 * k4)

            if rem % c2 == 0:
                t = rem // c2
                if min_t <= t <= max_t:
                    return True

    return False

valid = []

for u, v in candidates:
    if feasible(u, v):
        valid.append(Fraction(u, v))

print(len(valid))       # 35
print(min(valid))       # 1476/1475
print(max(valid))       # 123/59
```

### Code walkthrough

- `product_candidates()` generates every rational $m$ achievable for a single product:

$$m=\frac{bA_i}{aB_i}.$$

Each fraction is reduced to lowest terms and stored.

- We intersect the five sets so only values of $m$ possible for **every** product remain.
- `feasible(u,v)` constructs the minimal integer spoilage solution for each product and checks whether positive integer multiples $k_i$ can satisfy the overall paradox condition.
- The program finds exactly **35** valid values of $m$, matching the problem statement.
- The smallest is correctly verified as:

$$\frac{1476}{1475}$$

which matches the prompt.

- The largest is:

$$\frac{123}{59}.$$

Final verification:

$$\frac{123}{59}\approx 2.0847>1,$$

reasonable for a “worse by a factor $m$” ratio, and it is already in lowest terms.

**Answer: 123/59**

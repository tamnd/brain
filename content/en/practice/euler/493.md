[Project Euler Problem 493](https://projecteuler.net/problem=493)

## Solution

## 1. Mathematical analysis

We have:

- 70 balls total
- 7 colours
- 10 balls of each colour
- We randomly draw 20 balls without replacement

We want the **expected number of distinct colours** among the 20 chosen balls.

### Step 1: Use indicator random variables

This is a classic expectation problem where **linearity of expectation** makes things much easier.

For each colour $i$ (there are 7 colours), define an indicator variable:

$$X_i =
\begin{cases}
1 & \text{if colour } i \text{ appears at least once}\\
0 & \text{otherwise}
\end{cases}$$

Then the total number of distinct colours is

$$X = X_1 + X_2 + \cdots + X_7$$

Taking expectations:

$$E[X]
=
E[X_1] + E[X_2] + \cdots + E[X_7]$$

Since all colours are symmetric,

$$E[X] = 7E[X_1]$$

Now:

$$E[X_1] = P(\text{colour 1 appears})$$

Instead of computing this directly, compute the complement:

$$P(\text{colour 1 absent})$$

If a colour is absent, all 20 balls must come from the remaining 60 balls.

There are:

$$\binom{70}{20}$$

total ways to choose 20 balls, and

$$\binom{60}{20}$$

ways avoiding a specific colour.

Hence:

$$P(\text{colour absent})
=
\frac{\binom{60}{20}}{\binom{70}{20}}$$

Therefore:

$$P(\text{colour appears})
=
1 -
\frac{\binom{60}{20}}{\binom{70}{20}}$$

So the expected number of colours is:

$$E[X]
=
7
\left(
1 -
\frac{\binom{60}{20}}{\binom{70}{20}}
\right)$$

That is the complete solution.

---

## 2. Python implementation

```
from math import comb

# Probability a specific colour is absent
p_absent = comb(60, 20) / comb(70, 20)

# Expected number of distinct colours
expected = 7 * (1 - p_absent)

# Print with exactly 9 digits after decimal point
print(f"{expected:.9f}")
```

---

## 3. Code walkthrough

### `from math import comb`

Imports exact binomial coefficients:

$$\binom{n}{k}$$

Python computes these as exact integers, so there is no precision loss.

---

### `p_absent = comb(60, 20) / comb(70, 20)`

Computes:

$$\frac{\binom{60}{20}}{\binom{70}{20}}$$

This is the probability that one particular colour never appears.

---

### `expected = 7 * (1 - p_absent)`

Uses linearity of expectation:

$$E[X]
=
7 \cdot P(\text{colour appears})$$

with

$$P(\text{colour appears})
=
1 - p_{\text{absent}}$$

---

### `print(f"{expected:.9f}")`

Formats exactly **9 digits after the decimal point**, matching the Project Euler requirement.

---

## 4. Final verification

Sanity checks:

- Minimum possible distinct colours: **2** (since 20 balls cannot all be from one colour because only 10 exist).
- Maximum possible distinct colours: **7**.
- Since we draw 20 out of 70, we should see **most** colours, but not always all 7.

A value around **6.8** is very reasonable.

Evaluating:

$$7\left(1-\frac{\binom{60}{20}}{\binom{70}{20}}\right)
=
6.818741802\ldots$$

Rounded to nine decimal places:

**Answer: 6.818741802**

[Project Euler Problem 845](https://projecteuler.net/problem=845)

## Solution

**Answer:** `45009328011709400`

We need the $n$-th positive integer whose digit sum is prime.

Define a counting function:

$$C(x)=\#\{m\le x : \text{digit sum of } m \text{ is prime}\}.$$

Then $D(n)$ is the smallest $x$ such that:

$$C(x)\ge n.$$

A fast solution uses **digit DP** to compute $C(x)$:

- Process digits left to right.
- Track:

- current position,
- whether we are still tight to the prefix of $x$,
- the running digit sum.
- At the end, count the number if the digit sum is prime.

Then binary search for the smallest $x$ with $C(x)\ge 10^{16}$.

Verification against the examples:

- $C(157)=61$ and $C(156)=60$, so $D(61)=157$ ✔️
- $C(403539364)=10^8$, so $D(10^8)=403539364$ ✔️

Running the computation for $10^{16}$ gives:

$$D(10^{16}) = 45009328011709400$$

**Answer: 45009328011709400**

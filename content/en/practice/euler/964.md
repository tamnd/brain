[Project Euler Problem 964](https://projecteuler.net/problem=964)

## Solution

**Answer:** `4.7126135532e-29`

Let

$$n=\frac{k(k-1)}2+1$$

be the number of children/chairs.

For $k=7$,

$$n=\frac{7\cdot6}{2}+1=22.$$

---

## Key observation

During round $i$, the operation performed is a permutation acting on at most $i$ children.

A permutation on $n$ elements has **Cayley distance**

$$d(\pi)=n-c(\pi),$$

where $c(\pi)$ is the number of cycles.

- An $n$-cycle (the final “shift everyone one chair right” permutation) has distance $n-1$.
- A permutation acting on only $i$ children has distance at most $i-1$, with equality iff it is an $i$-cycle.

Since

$$\sum_{i=1}^{k}(i-1)=\frac{k(k-1)}2=n-1,$$

every round must attain the maximum possible distance. Therefore:

- in round $i$, the chosen children must form a single $i$-cycle.

So the problem becomes counting factorizations of a fixed $n$-cycle into cycles of lengths $1,2,\dots,k$.

A classical result (Dénes/Jackson factorization count) gives the number of such factorizations:

$$n^{k-1}.$$

---

## Total number of possible outcomes

In round $i$:

- choose $i$ children: $\binom{n}{i}$,
- seat them in any permutation: $i!$.

Hence the number of equally likely possibilities in round $i$ is

$$\binom{n}{i} i! = \frac{n!}{(n-i)!}.$$

Therefore

$$P(k)=
\frac{n^{k-1}}
{\displaystyle\prod_{i=1}^{k}\frac{n!}{(n-i)!}}.$$

---

# Compute $P(7)$

Using $n=22$,

$$P(7)=
\frac{22^6}
{\prod_{i=1}^{7}\frac{22!}{(22-i)!}}.$$

Python code:

```
from decimal import Decimal, getcontext
from math import perm

k = 7
n = k * (k - 1) // 2 + 1

den = 1
for i in range(1, k + 1):
    den *= perm(n, i)

num = n ** (k - 1)

getcontext().prec = 50
P = Decimal(num) / Decimal(den)

print(f"{P:.10e}")
```

This prints:

```
4.7126135532e-29
```

The magnitude is reasonable because the required final arrangement is extraordinarily specific among an enormous number of possible seatings.

**Answer: 4.7126135532e-29**

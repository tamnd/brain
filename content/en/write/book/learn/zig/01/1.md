## The Zig Programming Language

A book outline in the style of *The C Programming Language*: small examples first, precise explanations, exercises throughout, and a compact reference at the end. Target: Zig 0.16.0, released April 2026, with the 0.16 language reference and release notes as the baseline.

| Chapter | Title | Main idea |

| 4 | Functions and Program Structure | Function declarations, parameters, return values, source files, modules. |

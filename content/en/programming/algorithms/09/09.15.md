# 9.15 Articulation Points and Bridges

## Problem

You need to identify critical parts of an undirected graph. An articulation point is a vertex whose removal increases the number of connected components. A bridge is an edge whose removal disconnects the graph. These structures appear in network reliability, fault tolerance, road planning, and graph decomposition.

## Solution

Use DFS and track discovery times and low values.

For each vertex `u`:

- `tin[u]`: time when DFS first visits `u`
- `low[u]`: the smallest discovery time reachable from `u` using zero or more tree edges and at most one back edge

Initialize a global timer and run DFS.

### Bridges

An edge `(u, v)` where `v` is a child of `u` in DFS tree is a bridge if:

$$
low[v] > tin[u]
$$

### Articulation Points

A vertex `u` is an articulation point if:

- `u` is root and has more than one DFS child
- or `u` is not root and there exists a child `v` such that:

$$
low[v] \ge tin[u]
$$

## Implementation

```python
def find_bridges_and_articulation_points(adj):
    n = len(adj)
    tin = [-1] * n
    low = [-1] * n
    visited = [False] * n

    timer = 0
    bridges = []
    is_art = [False] * n

    def dfs(u, parent):
        nonlocal timer
        visited[u] = True
        tin[u] = low[u] = timer
        timer += 1

        children = 0

        for v in adj[u]:
            if v == parent:
                continue

            if visited[v]:
                low[u] = min(low[u], tin[v])
            else:
                dfs(v, u)
                low[u] = min(low[u], low[v])

                if low[v] > tin[u]:
                    bridges.append((u, v))

                if low[v] >= tin[u] and parent != -1:
                    is_art[u] = True

                children += 1

        if parent == -1 and children > 1:
            is_art[u] = True

    for u in range(n):
        if not visited[u]:
            dfs(u, -1)

    return bridges, is_art
```

## Basic Example

Consider:

```text
0 - 1 - 2
    |
    3 - 4
```

Edges:

```text
(1,2), (3,4)
```

are bridges. Removing either disconnects the graph.

Vertex `1` is an articulation point. Removing it separates `{0}` from `{2,3,4}`.

Vertex `3` is also an articulation point. Removing it separates `{4}`.

## Intuition

DFS builds a tree. Tree edges go downward. Back edges connect a node to an ancestor.

- `low[v]` tells how far back `v` or its subtree can reach.
- If `v` cannot reach above `u`, then removing `(u, v)` disconnects that subtree.

This is exactly the bridge condition:

```text
no back edge from subtree(v) to ancestors of u
```

For articulation points:

- If a subtree cannot connect back above `u`, then `u` is the only connection point.

## Correctness Argument

When DFS explores `(u, v)`:

- If `low[v] > tin[u]`, then no vertex in `v`'s subtree can reach `u` or any ancestor of `u`. Removing `(u, v)` disconnects the subtree. Hence it is a bridge.

- If `low[v] >= tin[u]`, then the subtree rooted at `v` cannot reach any strict ancestor of `u`. Therefore removing `u` disconnects that subtree from the rest. Hence `u` is an articulation point.

For the root:

- It has no parent. If it has more than one child, those subtrees are independent. Removing the root separates them.

## Complexity

DFS runs in:

$$
O(n + m)
$$

- Each vertex visited once  
- Each edge processed twice  

Space:

$$
O(n)
$$

for arrays and recursion stack.

## Common Pitfalls

A frequent mistake is forgetting to skip the parent edge:

```python
if v == parent:
    continue
```

Without this, every tree edge appears as a back edge.

Another mistake is mixing conditions:

- Bridge uses `>`
- Articulation uses `>=`

They are not interchangeable.

A third mistake is handling the root incorrectly. The root condition is special and must be handled separately.

A fourth mistake is applying this algorithm to directed graphs. The logic assumes undirected edges.

## Takeaway

Articulation points and bridges identify critical structure in an undirected graph. DFS discovery times and low values capture whether subtrees can reconnect upward. A bridge has no alternative path. An articulation point separates subtrees when removed.

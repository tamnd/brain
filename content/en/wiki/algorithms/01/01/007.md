# Last Occurrence Search

Last occurrence search finds the largest index where a target value appears. In unsorted data, this is most efficiently done by scanning from right to left and stopping at the first match.

This avoids scanning the entire array when matches tend to occur near the end.

## Problem

Given a sequence $A$ of length $n$ and a value $x$, find the largest index $i$ such that

$$
A[i] = x
$$

If no such index exists, return $-1$.

## Algorithm

Scan from the end toward the beginning.

```id="y4k8qp"
last_occurrence_search(A, x):
    for i from length(A) - 1 down to 0:
        if A[i] == x:
            return i
    return -1
```

## Example

Let

$$
A = [5, 3, 7, 3, 9]
$$

and

$$
x = 3
$$

The scan proceeds:

| step | index | value | match |
| ---- | ----- | ----- | ----- |
| 1    | 4     | 9     | no    |
| 2    | 3     | 3     | yes   |

Return index $3$, which is the last occurrence.

## Correctness

The algorithm visits indices in decreasing order. The first match encountered must be the largest index satisfying $A[i] = x$. If no match occurs, all elements have been checked, so returning $-1$ is correct.

## Complexity

| case  | comparisons | time   |
| ----- | ----------- | ------ |
| best  | $1$         | $O(1)$ |
| worst | $n$         | $O(n)$ |

Space usage:

$$
O(1)
$$

## When to Use

Use last occurrence search when:

* duplicates exist and you need the latest position
* matches are more likely near the end
* the data is unsorted

In sorted arrays, this can be implemented more efficiently using binary search variants.

## Implementation

```id="c7n2xp"
def last_occurrence_search(a, x):
    for i in range(len(a) - 1, -1, -1):
        if a[i] == x:
            return i
    return -1
```

```id="t3m9vk"
func LastOccurrenceSearch[T comparable](a []T, x T) int {
    for i := len(a) - 1; i >= 0; i-- {
        if a[i] == x {
            return i
        }
    }
    return -1
}
```

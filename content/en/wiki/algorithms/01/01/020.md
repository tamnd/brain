# Unsorted Membership Search

Unsorted membership search determines whether a value exists in a sequence without requiring any ordering. It is a decision version of linear search.

The algorithm stops as soon as a match is found, making it efficient when matches occur early.

## Problem

Given a sequence $A$ of length $n$ and a value $x$, determine whether

$$
\exists i \text{ such that } A[i] = x
$$

Return `true` if such an index exists, otherwise return `false`.

## Algorithm

Scan the sequence and stop at the first match.

```id="p2k9xm"
unsorted_membership_search(A, x):
    for i from 0 to length(A) - 1:
        if A[i] == x:
            return true
    return false
```

## Example

Let

$$
A = [6, 1, 8, 4, 3]
$$

and

$$
x = 4
$$

The scan proceeds:

| step | index | value | match |
| ---- | ----- | ----- | ----- |
| 1    | 0     | 6     | no    |
| 2    | 1     | 1     | no    |
| 3    | 2     | 8     | no    |
| 4    | 3     | 4     | yes   |

Return `true`.

## Correctness

The algorithm checks each element in sequence. If it returns `true`, then a matching element has been found. If it reaches the end without finding a match, then no such element exists.

## Complexity

| case  | comparisons | time   |
| ----- | ----------- | ------ |
| best  | $1$         | $O(1)$ |
| worst | $n$         | $O(n)$ |

Space usage:

$$
O(1)
$$

## When to Use

Use unsorted membership search when:

* only existence matters
* data is unsorted
* a simple decision is sufficient
* preprocessing is not justified

For repeated queries, hashing or sorting provides faster lookups.

## Implementation

```id="x8m3qp"
def unsorted_membership_search(a, x):
    for value in a:
        if value == x:
            return True
    return False
```

```id="k2v7zn"
func UnsortedMembershipSearch[T comparable](a []T, x T) bool {
    for _, v := range a {
        if v == x {
            return true
        }
    }
    return false
}
```

# Majority Candidate Scan

Majority candidate scan attempts to find a value that appears more than $\lfloor n/2 \rfloor$ times in a sequence. This version uses a direct counting approach to test each candidate.

It is simple but inefficient. It serves as a baseline before optimized methods such as Boyer Moore Majority Vote.

## Problem

Given a sequence $A$ of length $n$, find a value $x$ such that

$$
|{, i \mid A[i] = x ,}| > \frac{n}{2}
$$

If no such value exists, return `none`.

## Algorithm

For each element, count its frequency and check whether it exceeds $n/2$.

```id="x2k9qp"
majority_candidate_scan(A):
    n = length(A)

    for i from 0 to n - 1:
        count = 0
        for j from 0 to n - 1:
            if A[j] == A[i]:
                count = count + 1

        if count > n / 2:
            return A[i]

    return none
```

## Example

Let

$$
A = [3, 3, 4, 2, 3, 3, 3]
$$

Check candidates:

| candidate | count | majority |
| --------- | ----- | -------- |
| 3         | 5     | yes      |

Return:

$$
3
$$

## Correctness

The algorithm checks every element as a candidate and computes its exact frequency. If a majority element exists, it must appear as one of these candidates and will be detected when its count exceeds $n/2$.

If no element satisfies the condition, the algorithm returns `none`.

## Complexity

| case  | operations | time     |
| ----- | ---------- | -------- |
| worst | $n^2$      | $O(n^2)$ |

Space usage:

$$
O(1)
$$

## When to Use

Use this method when:

* the dataset is very small
* simplicity is more important than performance
* demonstrating baseline correctness

For large inputs, use more efficient methods such as Boyer Moore, which runs in linear time.

## Implementation

```id="p8m3zx"
def majority_candidate_scan(a):
    n = len(a)
    for i in range(n):
        count = 0
        for j in range(n):
            if a[j] == a[i]:
                count += 1
        if count > n // 2:
            return a[i]
    return None
```

```id="k7x2vn"
func MajorityCandidateScan[T comparable](a []T) (T, bool) {
    n := len(a)
    for i := 0; i < n; i++ {
        count := 0
        for j := 0; j < n; j++ {
            if a[j] == a[i] {
                count++
            }
        }
        if count > n/2 {
            return a[i], true
        }
    }
    var zero T
    return zero, false
}
```

# Duplicate Detection by Scan

Duplicate detection by scan checks whether any element appears more than once in a sequence. It uses a direct comparison approach without auxiliary data structures.

This method is simple and works in all settings, including when hashing or sorting is not allowed.

## Problem

Given a sequence $A$ of length $n$, determine whether there exist indices $i \ne j$ such that

$$
A[i] = A[j]
$$

Return `true` if duplicates exist, otherwise return `false`.

## Algorithm

Compare each element with all elements that follow it.

```id="p4k9xm"
duplicate_detection_scan(A):
    n = length(A)
    for i from 0 to n - 1:
        for j from i + 1 to n - 1:
            if A[i] == A[j]:
                return true
    return false
```

## Example

Let

$$
A = [4, 2, 7, 3, 2]
$$

The comparisons proceed:

| step | pair  | values | duplicate |
| ---- | ----- | ------ | --------- |
| 1    | (0,1) | (4,2)  | no        |
| 2    | (0,2) | (4,7)  | no        |
| 3    | (0,3) | (4,3)  | no        |
| 4    | (0,4) | (4,2)  | no        |
| 5    | (1,2) | (2,7)  | no        |
| 6    | (1,3) | (2,3)  | no        |
| 7    | (1,4) | (2,2)  | yes       |

Return `true`.

## Correctness

The algorithm examines every unordered pair $(i, j)$ with $i < j$. If any pair satisfies $A[i] = A[j]$, it returns immediately. If no such pair exists, all comparisons complete and the result is `false`.

## Complexity

| case  | comparisons        | time     |
| ----- | ------------------ | -------- |
| worst | $\frac{n(n-1)}{2}$ | $O(n^2)$ |

Space usage:

$$
O(1)
$$

## When to Use

Use this method when:

* the dataset is small
* memory usage must remain constant
* hashing or sorting is not available
* simplicity is preferred

For large datasets, more efficient approaches include sorting ($O(n \log n)$) or hashing ($O(n)$ expected time).

## Implementation

```id="z6x2qp"
def duplicate_detection_scan(a):
    n = len(a)
    for i in range(n):
        for j in range(i + 1, n):
            if a[i] == a[j]:
                return True
    return False
```

```id="k8m3vn"
func DuplicateDetectionScan[T comparable](a []T) bool {
    n := len(a)
    for i := 0; i < n; i++ {
        for j := i + 1; j < n; j++ {
            if a[i] == a[j] {
                return true
            }
        }
    }
    return false
}
```

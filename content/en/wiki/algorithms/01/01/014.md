# Frequency Count Search

Frequency count search computes how many times a target value appears in a sequence. It performs a full scan and increments a counter whenever a match occurs.

This is a basic aggregation primitive used in statistics, histograms, and preprocessing.

## Problem

Given a sequence $A$ of length $n$ and a value $x$, compute

$$
\text{count}(x) = |{, i \mid 0 \le i < n,\ A[i] = x ,}|
$$

## Algorithm

Scan the sequence and maintain a counter.

```id="q2m8xp"
frequency_count_search(A, x):
    count = 0
    for i from 0 to length(A) - 1:
        if A[i] == x:
            count = count + 1
    return count
```

## Example

Let

$$
A = [4, 2, 7, 2, 9, 2]
$$

and

$$
x = 2
$$

The scan proceeds:

| step | index | value | match | count |
| ---- | ----- | ----- | ----- | ----- |
| 1    | 0     | 4     | no    | 0     |
| 2    | 1     | 2     | yes   | 1     |
| 3    | 2     | 7     | no    | 1     |
| 4    | 3     | 2     | yes   | 2     |
| 5    | 4     | 9     | no    | 2     |
| 6    | 5     | 2     | yes   | 3     |

Return:

$$
3
$$

## Correctness

Each index is examined exactly once. The counter increases precisely when $A[i] = x$. Therefore, after processing all elements, the counter equals the number of occurrences.

## Complexity

| case      | operations | time   |
| --------- | ---------- | ------ |
| all cases | $n$        | $O(n)$ |

Space usage:

$$
O(1)
$$

## When to Use

Use frequency count search when:

* computing simple statistics
* building histograms
* preprocessing for further algorithms
* counting duplicates

For multiple queries, building a frequency map using hashing is more efficient.

## Implementation

```id="z1k7qp"
def frequency_count_search(a, x):
    count = 0
    for value in a:
        if value == x:
            count += 1
    return count
```

```id="m9v3xn"
func FrequencyCountSearch[T comparable](a []T, x T) int {
    count := 0
    for _, v := range a {
        if v == x {
            count++
        }
    }
    return count
}
```

# Bounded Linear Search

Bounded linear search scans only a specified subrange of a sequence instead of the entire array. This is useful when you already know the region where the target may exist, or when working with segmented data structures.

The algorithm preserves the simplicity of linear search while reducing unnecessary comparisons.

## Problem

Given a sequence $A$ of length $n$, a target value $x$, and indices $l$ and $r$ such that

$$
0 \le l \le r < n
$$

find an index $i$ in the range $[l, r]$ such that

$$
A[i] = x
$$

If no such index exists, return $-1$.

## Algorithm

Scan only within the given bounds.

```id="f8q2mz"
bounded_linear_search(A, x, l, r):
    for i from l to r:
        if A[i] == x:
            return i
    return -1
```

## Example

Let

$$
A = [5, 2, 7, 3, 8, 6]
$$

and

$$
x = 3,\quad l = 2,\quad r = 4
$$

We search only indices $2$ through $4$:

| step | index | value | match |
| ---- | ----- | ----- | ----- |
| 1    | 2     | 7     | no    |
| 2    | 3     | 3     | yes   |

Return index $3$.

## Correctness

The algorithm examines every index in the interval $[l, r]$. If it returns an index $i$, then $A[i] = x$ holds by direct comparison. If it returns $-1$, then all elements in the specified range have been checked and none equals $x$.

## Complexity

Let $k = r - l + 1$ be the size of the search range.

| case  | comparisons | time   |
| ----- | ----------- | ------ |
| best  | $1$         | $O(1)$ |
| worst | $k$         | $O(k)$ |

Space usage:

$$
O(1)
$$

## When to Use

Bounded linear search is useful when:

* the relevant region is known in advance
* the data is partitioned or windowed
* you want to avoid scanning the full array
* working with sliding windows or subarrays

It is a direct extension of linear search with tighter control over the search domain.

## Implementation

```id="c9x1pa"
def bounded_linear_search(a, x, l, r):
    for i in range(l, r + 1):
        if a[i] == x:
            return i
    return -1
```

```id="u7k3rz"
func BoundedLinearSearch[T comparable](a []T, x T, l, r int) int {
    for i := l; i <= r; i++ {
        if a[i] == x {
            return i
        }
    }
    return -1
}
```

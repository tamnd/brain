# Maximum Search

Maximum search finds the largest value in a sequence and returns its index. It performs a single pass while maintaining the best candidate seen so far.

This is symmetric to minimum search and appears in many selection and optimization routines.

## Problem

Given a sequence $A$ of length $n$, find an index $i$ such that

$$
A[i] = \max_{0 \le j < n} A[j]
$$

If multiple positions share the maximum value, return any one of them, typically the first.

## Algorithm

Maintain a running maximum while scanning the sequence.

```id="m1k8qp"
maximum_search(A):
    if length(A) == 0:
        return -1

    max_index = 0

    for i from 1 to length(A) - 1:
        if A[i] > A[max_index]:
            max_index = i

    return max_index
```

## Example

Let

$$
A = [7, 3, 5, 2, 9]
$$

The scan proceeds:

| step | index | value | current max index | current max |
| ---- | ----- | ----- | ----------------- | ----------- |
| 1    | 0     | 7     | 0                 | 7           |
| 2    | 1     | 3     | 0                 | 7           |
| 3    | 2     | 5     | 0                 | 7           |
| 4    | 3     | 2     | 0                 | 7           |
| 5    | 4     | 9     | 4                 | 9           |

Return index $4$.

## Correctness

At each step, the algorithm maintains the invariant that `max_index` stores the position of the largest value among all elements seen so far. After scanning all elements, it must hold the global maximum.

## Complexity

| case      | comparisons | time   |
| --------- | ----------- | ------ |
| all cases | $n - 1$     | $O(n)$ |

Space usage:

$$
O(1)
$$

## When to Use

Use maximum search when:

* the data is unsorted
* a single pass is required
* as a primitive inside other algorithms
* computing extrema in streaming or batch settings

## Implementation

```id="p8v3zn"
def maximum_search(a):
    if not a:
        return -1

    max_index = 0
    for i in range(1, len(a)):
        if a[i] > a[max_index]:
            max_index = i
    return max_index
```

```id="k2q7xm"
func MaximumSearch[T constraints.Ordered](a []T) int {
    if len(a) == 0 {
        return -1
    }

    maxIndex := 0
    for i := 1; i < len(a); i++ {
        if a[i] > a[maxIndex] {
            maxIndex = i
        }
    }
    return maxIndex
}
```

# Minimum Search

Minimum search finds the smallest value in a sequence and returns its index. It scans the sequence once while maintaining the best candidate seen so far.

This is a fundamental primitive used in selection, sorting, and optimization routines.

## Problem

Given a sequence $A$ of length $n$, find an index $i$ such that

$$
A[i] = \min_{0 \le j < n} A[j]
$$

If multiple positions share the minimum value, return any one of them, typically the first.

## Algorithm

Maintain a running minimum while scanning the sequence.

```id="r8m1xp"
minimum_search(A):
    if length(A) == 0:
        return -1

    min_index = 0

    for i from 1 to length(A) - 1:
        if A[i] < A[min_index]:
            min_index = i

    return min_index
```

## Example

Let

$$
A = [7, 3, 5, 2, 9]
$$

The scan proceeds:

| step | index | value | current min index | current min |
| ---- | ----- | ----- | ----------------- | ----------- |
| 1    | 0     | 7     | 0                 | 7           |
| 2    | 1     | 3     | 1                 | 3           |
| 3    | 2     | 5     | 1                 | 3           |
| 4    | 3     | 2     | 3                 | 2           |
| 5    | 4     | 9     | 3                 | 2           |

Return index $3$.

## Correctness

At each step, the algorithm maintains the invariant that `min_index` stores the position of the smallest value among all elements seen so far. After scanning all elements, it must hold the global minimum.

## Complexity

| case      | comparisons | time   |
| --------- | ----------- | ------ |
| all cases | $n - 1$     | $O(n)$ |

Space usage:

$$
O(1)
$$

## When to Use

Use minimum search when:

* the data is unsorted
* you need a single pass solution
* as a building block for selection or sorting
* inside algorithms like selection sort

## Implementation

```id="p3v7zk"
def minimum_search(a):
    if not a:
        return -1

    min_index = 0
    for i in range(1, len(a)):
        if a[i] < a[min_index]:
            min_index = i
    return min_index
```

```id="w2n8qx"
func MinimumSearch[T constraints.Ordered](a []T) int {
    if len(a) == 0 {
        return -1
    }

    minIndex := 0
    for i := 1; i < len(a); i++ {
        if a[i] < a[minIndex] {
            minIndex = i
        }
    }
    return minIndex
}
```

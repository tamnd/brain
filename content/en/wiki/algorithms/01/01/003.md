# Reverse Linear Search

Reverse linear search scans the sequence from the end toward the beginning. It is useful when you care about the last occurrence of a value or when recent elements are more likely to match.

The algorithm remains simple and requires no preprocessing or additional data structures.

## Problem

Given a sequence $A$ of length $n$ and a value $x$, find an index $i$ such that

$$
A[i] = x
$$

with preference for the largest such index. If no such index exists, return $-1$.

## Algorithm

Start from the last index and move backward until a match is found.

```id="8n2k1q"
reverse_linear_search(A, x):
    for i from length(A) - 1 down to 0:
        if A[i] == x:
            return i
    return -1
```

## Example

Let

$$
A = [2, 4, 6, 4, 9]
$$

and

$$
x = 4
$$

The scan proceeds from right to left:

| step | index | value | match |
| ---- | ----- | ----- | ----- |
| 1    | 4     | 9     | no    |
| 2    | 3     | 4     | yes   |

Return index $3$, which is the last occurrence.

## Correctness

The algorithm visits indices in decreasing order. The first match encountered must be the largest index $i$ such that $A[i] = x$. If no match occurs, every element has been checked, so returning $-1$ is correct.

## Complexity

| case  | comparisons | time   |
| ----- | ----------- | ------ |
| best  | $1$         | $O(1)$ |
| worst | $n$         | $O(n)$ |

Space usage:

$$
O(1)
$$

## When to Use

Reverse linear search is appropriate when:

* you need the last occurrence
* recent elements are more likely to match
* the data is unsorted
* simplicity is preferred over preprocessing

It avoids scanning the entire array when matches tend to occur near the end.

## Implementation

```id="v4k92c"
def reverse_linear_search(a, x):
    for i in range(len(a) - 1, -1, -1):
        if a[i] == x:
            return i
    return -1
```

```id="m1p7xq"
func ReverseLinearSearch[T comparable](a []T, x T) int {
    for i := len(a) - 1; i >= 0; i-- {
        if a[i] == x {
            return i
        }
    }
    return -1
}
```

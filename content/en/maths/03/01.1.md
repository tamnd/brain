## 1.1 Syntax: Variables, Connectives

Propositional logic is defined by a formal language. This language specifies which expressions are valid before assigning any meaning.

### Variables

A **propositional variable** is an atomic symbol:

$$
p, q, r, P, Q, R, p_1, p_2, \dots
$$

Each variable represents a statement. At the syntactic level, no truth value is attached.

### Connectives

Complex formulas are built using logical connectives:

| Symbol                | Name          | Arity  |
| --------------------- | ------------- | ------ |
| $¬A$                  | Negation      | unary  |
| $A \land B$           | Conjunction   | binary |
| $A \lor B$            | Disjunction   | binary |
| $A \to B$             | Implication   | binary |
| $A \leftrightarrow B$ | Biconditional | binary |

These symbols act as operators on formulas.

### Formation Rules

The set of well-formed formulas is defined inductively.

1. Every propositional variable is a formula.
2. If $A$ is a formula, then $¬A$ is a formula.
3. If $A$ and $B$ are formulas, then:
   $$
   (A \land B),\ (A \lor B),\ (A \to B),\ (A \leftrightarrow B)
   $$
   are formulas.
4. Nothing else is a formula.

This defines the smallest set closed under these rules.

### Examples

Well-formed formulas:

$$
p
$$

$$
¬p
$$

$$
(p \land q)
$$

$$
((p \land q) \to r)
$$

$$
(¬p \lor (q \leftrightarrow r))
$$

Ill-formed expressions:

$$
\land p q
$$

$$
(p \land)
$$

$$
p q \to
$$

$$
((p \lor q)
$$

Each violates the inductive rules.

### Parentheses and Precedence

Parentheses remove ambiguity. The expression

$$
p \land q \lor r
$$

admits two parses:

$$
(p \land q) \lor r
\quad \text{or} \quad
p \land (q \lor r)
$$

To reduce parentheses, a precedence convention is used:

| Level | Connective        |
| ----- | ----------------- |
| 1     | $¬$               |
| 2     | $\land$           |
| 3     | $\lor$            |
| 4     | $\to$             |
| 5     | $\leftrightarrow$ |

Thus,

$$
¬p \land q \to r
$$

means

$$
((¬p \land q) \to r)
$$

### Abstract Syntax

Formulas can be viewed as trees.

Example:

$$
((p \land q) \to r)
$$

has the structure:

* root: $\to$
* left subtree: $(p \land q)$
* right subtree: $r$

This tree view supports recursive definitions and algorithms on formulas.

### Remarks

Syntax is purely formal. It defines structure, not meaning. Truth and interpretation are introduced in semantics.

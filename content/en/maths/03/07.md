## Chapter 7. Basic Set Theory

Set theory studies collections of objects. These collections are called **sets**, and their objects are called **elements**.

A set can be written by listing its elements:

$$
A = {1,2,3}
$$

This means that $1$, $2$, and $3$ are elements of $A$.

We write:

$$
2 \in A
$$

to say that $2$ belongs to $A$, and:

$$
4 \notin A
$$

to say that $4$ does not belong to $A$.

Set theory gives mathematics a common language. Numbers, functions, relations, ordered pairs, sequences, spaces, and algebraic structures can all be described using sets. This is why set theory plays such a central role in foundations.

The first basic operation is **membership**. A set is determined by which objects belong to it. For example:

$$
{1,2,3} = {3,2,1}
$$

because both sets have exactly the same elements. Order does not matter.

Also:

$$
{1,1,2,3} = {1,2,3}
$$

because repetition does not matter.

Another important relation is **subset**. We write:

$$
A \subseteq B
$$

when every element of $A$ is also an element of $B$.

For example:

$$
{1,2} \subseteq {1,2,3}
$$

The empty set is the set with no elements:

$$
\varnothing
$$

It is a subset of every set:

$$
\varnothing \subseteq A
$$

This is true because there is no element of $\varnothing$ that can fail to belong to $A$.

Set operations let us build new sets from old ones.

The **union** of $A$ and $B$ contains elements that belong to $A$ or to $B$:

$$
A \cup B
$$

The **intersection** contains elements that belong to both:

$$
A \cap B
$$

The **difference** contains elements that belong to $A$ but not to $B$:

$$
A \setminus B
$$

For example, if:

$$
A={1,2,3}
$$

and:

$$
B={3,4,5}
$$

then:

$$
A \cup B = {1,2,3,4,5}
$$

$$
A \cap B = {3}
$$

$$
A \setminus B = {1,2}
$$

Relations and functions are also built from sets.

An ordered pair is written:

$$
(a,b)
$$

A relation from $A$ to $B$ is a set of ordered pairs:

$$
R \subseteq A \times B
$$

where $A \times B$ is the Cartesian product:

$$
A \times B = {(a,b) : a \in A,\ b \in B}
$$

A function is a special kind of relation. A function $f : A \to B$ assigns to each element of $A$ exactly one element of $B$.

Cardinality measures the size of a set. For finite sets, this is just the number of elements. For example:

$$
|{1,2,3}| = 3
$$

For infinite sets, cardinality becomes more subtle. Two sets have the same cardinality when there is a bijection between them. This idea lets us compare infinite sizes.

The natural numbers:

$$
\mathbb{N} = {0,1,2,3,\dots}
$$

are countably infinite. A set is countable if its elements can be listed in a sequence, possibly forever.

The real numbers:

$$
\mathbb{R}
$$

are uncountable. There are strictly more real numbers than natural numbers. This is one of the first major surprises in set theory.

Set theory also introduces **ordinals**, which generalize counting positions and order types. Natural numbers can be viewed as finite ordinals:

$$
0,1,2,3,\dots
$$

The first infinite ordinal is:

$$
\omega
$$

Ordinals allow us to reason about well-ordered sets and transfinite processes.

Cardinals measure size, while ordinals measure order. These two ideas agree for finite sets but diverge in the infinite case.

Modern set theory is usually developed axiomatically. The standard foundation is Zermelo-Fraenkel set theory, often with the Axiom of Choice:

$$
\mathrm{ZF}
$$

or:

$$
\mathrm{ZFC}
$$

These axioms regulate which sets exist and which constructions are allowed. This avoids paradoxes that arise from unrestricted set formation.

Chapter 7 develops the basic language of set theory in five stages. It begins with sets, relations, and functions. It then studies cardinality and countability. After that, it introduces ordinals and well-ordering, followed by cardinal arithmetic. Finally, it presents the Zermelo-Fraenkel axioms as a formal foundation for mathematics.

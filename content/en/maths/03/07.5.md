## 7.5 Zermelo-Fraenkel Axioms (ZF, ZFC)

Naive set theory allows us to speak about sets freely. For ordinary examples, this feels natural. We write sets such as:

$$
{1,2,3}
$$

or:

$$
{x \in \mathbb{N} : x \text{ is even}}
$$

and use them without difficulty.

But unrestricted set formation leads to contradictions. The most famous example is Russell's paradox.

Suppose we form the set:

$$
R = {x : x \notin x}
$$

Now ask whether $R \in R$.

If $R \in R$, then by the definition of $R$, we must have:

$$
R \notin R
$$

If $R \notin R$, then by the definition of $R$, we must have:

$$
R \in R
$$

Both cases lead to contradiction.

So modern set theory does not allow arbitrary collections of the form:

$$
{x : P(x)}
$$

Instead, it uses axioms that say exactly which sets may be formed.

### The Role of Axioms

The Zermelo-Fraenkel axioms, usually called **ZF**, provide a formal foundation for set theory.

With the Axiom of Choice added, the system is called **ZFC**.

$$
\mathrm{ZFC} = \mathrm{ZF} + \mathrm{AC}
$$

These axioms do not define what a set “really is”. Instead, they describe the rules that sets must obey.

The only primitive relation is membership:

$$
\in
$$

Everything else is built from membership.

### Extensionality

Two sets are equal if they have the same elements.

$$
\forall A,\forall B,\bigl(\forall x,(x\in A \leftrightarrow x\in B) \to A=B\bigr)
$$

This axiom says that sets are determined by their members.

For example:

$$
{1,2,3} = {3,2,1}
$$

because both have the same elements.

### Empty Set

There exists a set with no elements.

$$
\exists E,\forall x,(x\notin E)
$$

This set is unique by extensionality, so we call it:

$$
\varnothing
$$

The empty set is the starting point for many constructions.

### Pairing

For any two sets $a$ and $b$, there is a set whose only elements are $a$ and $b$.

$$
\forall a,\forall b,\exists P,\forall x,(x\in P \leftrightarrow (x=a \lor x=b))
$$

We write this set as:

$$
{a,b}
$$

If $a=b$, this gives the singleton:

$$
{a}
$$

### Union

For any set $A$, there is a set containing exactly the elements of the elements of $A$.

$$
\forall A,\exists U,\forall x,\bigl(x\in U \leftrightarrow \exists y,(x\in y \land y\in A)\bigr)
$$

We write:

$$
U = \bigcup A
$$

Example:

$$
A = {{1,2},{3,4}}
$$

Then:

$$
\bigcup A = {1,2,3,4}
$$

### Power Set

For any set $A$, there is a set whose elements are exactly the subsets of $A$.

$$
\forall A,\exists P,\forall x,(x\in P \leftrightarrow x\subseteq A)
$$

We write:

$$
P = \mathcal{P}(A)
$$

Example:

$$
\mathcal{P}({1,2})
==================

{\varnothing,{1},{2},{1,2}}
$$

The power set axiom is what allows set theory to climb to larger cardinalities.

### Separation

Separation says that from an existing set, we may carve out a subset using a property.

If $A$ is already a set, then:

$$
{x\in A : \varphi(x)}
$$

is also a set.

More formally, for each formula $\varphi(x)$:

$$
\forall A,\exists B,\forall x,(x\in B \leftrightarrow (x\in A \land \varphi(x)))
$$

This avoids Russell's paradox because we are not forming:

$$
{x : \varphi(x)}
$$

from the entire universe. We are only selecting elements from an already existing set.

For example:

$$
{n\in\mathbb{N}: n \text{ is even}}
$$

is allowed once $\mathbb{N}$ exists.

### Replacement

Replacement says that the image of a set under a definable function is a set.

Informally, if each $x\in A$ determines a unique $y$, then the collection of all such $y$ is a set.

Formally, if $\varphi(x,y)$ defines a functional relation, then:

$$
\forall A,\exists B,\forall y,\bigl(y\in B \leftrightarrow \exists x\in A,\varphi(x,y)\bigr)
$$

The important condition is uniqueness:

$$
\forall x\in A,\exists! y,\varphi(x,y)
$$

Replacement is essential for transfinite constructions. It lets us collect outputs indexed by a set of stages.

### Infinity

There exists an infinite set.

One standard form says that there is a set $I$ such that:

$$
\varnothing \in I
$$

and whenever $x\in I$, also:

$$
x\cup{x}\in I
$$

The operation:

$$
x \mapsto x\cup{x}
$$

is the successor operation.

This axiom gives enough structure to construct the natural numbers:

$$
0=\varnothing
$$

$$
1={\varnothing}
$$

$$
2={\varnothing,{\varnothing}}
$$

and so on.

The set of all finite ordinals is:

$$
\omega
$$

which represents:

$$
\mathbb{N}
$$

### Foundation

Foundation says that membership chains cannot descend forever.

One standard form is:

$$
\forall A,(A\neq\varnothing \to \exists x\in A, (x\cap A=\varnothing))
$$

This means every nonempty set has an element disjoint from it.

A useful consequence is that no set belongs to itself:

$$
x\notin x
$$

It also rules out cycles such as:

$$
x\in y \in x
$$

Foundation gives the universe of sets a layered structure.

### Choice

The Axiom of Choice says that if we have a set of nonempty sets, we can choose one element from each set.

Informally, for every family:

$$
{A_i : i\in I}
$$

where each $A_i$ is nonempty, there exists a function $f$ such that:

$$
f(i)\in A_i
$$

for every $i\in I$.

This function is called a choice function.

ZF plus the Axiom of Choice is ZFC:

$$
\mathrm{ZFC} = \mathrm{ZF} + \mathrm{AC}
$$

Choice is powerful. It implies that every vector space has a basis, every set can be well-ordered, and products of compact spaces are compact.

It also has nonconstructive consequences. It may assert the existence of objects without giving an explicit method to build them.

### Why ZFC Matters

ZFC is the standard background theory for most modern mathematics.

It provides enough sets to develop arithmetic, analysis, algebra, topology, model theory, and much more.

It also restricts set formation enough to avoid the classical paradoxes.

In practice, most ordinary mathematics uses only a small part of ZFC. But advanced set theory studies the full strength of these axioms, their consequences, and their limitations.

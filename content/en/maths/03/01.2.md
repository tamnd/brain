1.2 Semantics: Truth Tables

We assign meaning to formulas by specifying their truth values. This is done by defining valuations and extending them to all formulas.

1.2.1 Truth Values

We fix the set of truth values:
$$
{\mathrm{T}, \mathrm{F}}
$$

Here $\mathrm{T}$ denotes true and $\mathrm{F}$ denotes false.

1.2.2 Definition (Valuation)

A valuation is a function:
$$
v : \mathrm{Var} \to {\mathrm{T}, \mathrm{F}}
$$

which assigns a truth value to each propositional variable.

Example:
$$
v(p) = \mathrm{T}, \quad v(q) = \mathrm{F}
$$

A valuation determines the truth of all formulas built from these variables.

1.2.3 Extension to Formulas

We extend a valuation $v$ to all formulas by induction.

1. If $A$ is a variable, then:
    $$
    v(A) \text{ is given}
    $$
2. If $A = ¬B$, then:
    $$
    v(A) =
    \begin{cases}
    \mathrm{T} & \text{if } v(B) = \mathrm{F} \
    \mathrm{F} & \text{if } v(B) = \mathrm{T}
    \end{cases}
    $$
3. If $A = (B \land C)$, then:
    $$
    v(A) = \mathrm{T} \quad \text{if and only if} \quad v(B) = \mathrm{T} \text{ and } v(C) = \mathrm{T}
    $$
4. If $A = (B \lor C)$, then:
    $$
    v(A) = \mathrm{T} \quad \text{if and only if} \quad v(B) = \mathrm{T} \text{ or } v(C) = \mathrm{T}
    $$
5. If $A = (B \to C)$, then:
    $$
    v(A) = \mathrm{F} \quad \text{if and only if} \quad v(B) = \mathrm{T} \text{ and } v(C) = \mathrm{F}
    $$
6. If $A = (B \leftrightarrow C)$, then:
    $$
    v(A) = \mathrm{T} \quad \text{if and only if} \quad v(B) = v(C)
    $$

This defines the value of every formula uniquely.

1.2.4 Example

Let:
$$
v(p) = \mathrm{T}, \quad v(q) = \mathrm{F}
$$

Compute:
$$
v(p \to q)
$$

Since:
$$
v(p) = \mathrm{T}, \quad v(q) = \mathrm{F}
$$

we have:
$$
v(p \to q) = \mathrm{F}
$$

Now compute:
$$
v(¬p \lor q)
$$

We have:
$$
v(¬p) = \mathrm{F}
$$

and:
$$
v(q) = \mathrm{F}
$$

so:
$$
v(¬p \lor q) = \mathrm{F}
$$

1.2.5 Definition (Truth Table)

Let $A$ be a formula with variables $p_1, \dots, p_n$. A truth table for $A$ is a table listing all valuations:
$$
{0,1}^n
$$
together with the value of $A$ under each valuation.

Each row corresponds to one assignment of truth values.

1.2.6 Example

Consider:
$$
(p \land q) \to p
$$

$p$	$q$	$p \land q$	$(p \land q) \to p$
$\mathrm{T}$	$\mathrm{T}$	$\mathrm{T}$	$\mathrm{T}$
$\mathrm{T}$	$\mathrm{F}$	$\mathrm{F}$	$\mathrm{T}$
$\mathrm{F}$	$\mathrm{T}$	$\mathrm{F}$	$\mathrm{T}$
$\mathrm{F}$	$\mathrm{F}$	$\mathrm{F}$	$\mathrm{T}$

The last column is always $\mathrm{T}$.

1.2.7 Definition (Satisfiable, Valid, Contradiction)

Let $A$ be a formula.

1. $A$ is satisfiable if there exists a valuation $v$ such that:
    $$
    v(A) = \mathrm{T}
    $$
2. $A$ is valid if for all valuations $v$:
    $$
    v(A) = \mathrm{T}
    $$
3. $A$ is a contradiction if for all valuations $v$:
    $$
    v(A) = \mathrm{F}
    $$

1.2.8 Example

1. The formula:
    $$
    p \lor ¬p
    $$
    is valid.
2. The formula:
    $$
    p \land ¬p
    $$
    is a contradiction.
3. The formula:
    $$
    p \land q
    $$
    is satisfiable but not valid.

1.2.9 Boolean Function View

Each formula $A$ with variables $p_1, \dots, p_n$ defines a function:
$$
f_A : {\mathrm{T}, \mathrm{F}}^n \to {\mathrm{T}, \mathrm{F}}
$$

given by:
$$
f_A(v(p_1), \dots, v(p_n)) = v(A)
$$

This identifies formulas with Boolean functions.

1.2.10 Induction Principle

The evaluation function $v(A)$ is defined by structural recursion on formulas.

This corresponds to the following principle:

To define a function on formulas, it suffices to specify:

1. its value on variables
2. how it acts on $¬A$
3. how it acts on $(A \circ B)$

This mirrors the inductive definition of formulas and is directly implementable in a proof assistant.

1.2 Semantics: Truth Tables

We assign meaning to formulas by specifying how their truth values are determined. This is done by defining valuations on variables and extending them to all formulas. The definition follows the inductive structure of formulas.

Truth Values

We fix the set of truth values:
$$
{\mathrm{T}, \mathrm{F}}
$$

The symbol $\mathrm{T}$ denotes true and $\mathrm{F}$ denotes false. Every formula will be assigned exactly one of these values.

Valuations

A valuation assigns a truth value to each propositional variable.

Formally, a valuation is a function:
$$
v : \mathrm{Var} \to {\mathrm{T}, \mathrm{F}}
$$

Thus, for each variable $p$, the value $v(p)$ is either $\mathrm{T}$ or $\mathrm{F}$.

A valuation represents one possible assignment of truth values. Different valuations correspond to different possible situations.

Extension to Formulas

A valuation on variables extends uniquely to all formulas. This extension is defined recursively.

If $A$ is a variable, then $v(A)$ is given.

If $A = ¬B$, then:
$$
v(A) =
\begin{cases}
\mathrm{T} & \text{if } v(B) = \mathrm{F} \
\mathrm{F} & \text{if } v(B) = \mathrm{T}
\end{cases}
$$

If $A = (B \land C)$, then:
$$
v(A) = \mathrm{T} \quad \text{if and only if} \quad v(B) = \mathrm{T} \text{ and } v(C) = \mathrm{T}
$$

If $A = (B \lor C)$, then:
$$
v(A) = \mathrm{T} \quad \text{if and only if} \quad v(B) = \mathrm{T} \text{ or } v(C) = \mathrm{T}
$$

If $A = (B \to C)$, then:
$$
v(A) = \mathrm{F} \quad \text{if and only if} \quad v(B) = \mathrm{T} \text{ and } v(C) = \mathrm{F}
$$

If $A = (B \leftrightarrow C)$, then:
$$
v(A) = \mathrm{T} \quad \text{if and only if} \quad v(B) = v(C)
$$

This definition ensures that every formula has a well-defined truth value under any valuation.

Evaluation Examples

Let a valuation $v$ be given by:
$$
v(p) = \mathrm{T}, \quad v(q) = \mathrm{F}
$$

Then:
$$
v(¬p) = \mathrm{F}
$$

$$
v(p \land q) = \mathrm{F}
$$

$$
v(p \lor q) = \mathrm{T}
$$

$$
v(p \to q) = \mathrm{F}
$$

$$
v(p \leftrightarrow q) = \mathrm{F}
$$

Each value is determined directly from the recursive definition.

Truth Tables

A truth table lists all possible valuations of the variables occurring in a formula, together with the corresponding value of the formula.

If a formula contains $n$ distinct variables, then there are:
$$
2^n
$$
possible valuations.

Each row of the table corresponds to one valuation.

Example

Consider the formula:
$$
(p \land q) \to p
$$

The truth table is:

$p$	$q$	$p \land q$	$(p \land q) \to p$
$\mathrm{T}$	$\mathrm{T}$	$\mathrm{T}$	$\mathrm{T}$
$\mathrm{T}$	$\mathrm{F}$	$\mathrm{F}$	$\mathrm{T}$
$\mathrm{F}$	$\mathrm{T}$	$\mathrm{F}$	$\mathrm{T}$
$\mathrm{F}$	$\mathrm{F}$	$\mathrm{F}$	$\mathrm{T}$

The final column shows that the formula is always true.

Satisfiability and Validity

A formula is satisfiable if there exists a valuation $v$ such that:
$$
v(A) = \mathrm{T}
$$

A formula is valid if for every valuation $v$:
$$
v(A) = \mathrm{T}
$$

A formula is a contradiction if for every valuation $v$:
$$
v(A) = \mathrm{F}
$$

These notions classify formulas according to their truth behavior.

Examples

The formula:
$$
p \lor ¬p
$$
is valid.

The formula:
$$
p \land ¬p
$$
is a contradiction.

The formula:
$$
p \land q
$$
is satisfiable but not valid.

Boolean Function View

Each formula determines a function:
$$
f_A : {\mathrm{T}, \mathrm{F}}^n \to {\mathrm{T}, \mathrm{F}}
$$

where $n$ is the number of variables in $A$.

The value of $f_A$ at a tuple of truth values is the value of $A$ under the corresponding valuation.

This identifies formulas with Boolean functions.

Structural Recursion

The evaluation of formulas is defined by structural recursion.

To define a function on formulas, it suffices to specify:

* its value on variables,
* its value on $¬A$ in terms of $A$,
* its value on $(A \circ B)$ in terms of $A$ and $B$.

This mirrors the inductive definition of formulas and provides a direct method for implementation.

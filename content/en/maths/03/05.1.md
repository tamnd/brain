The compactness theorem is one of the central results of first order logic, and it expresses a fundamental finiteness principle that connects local satisfiability with global satisfiability, stating that if every finite part of a theory is consistent or satisfiable, then the entire theory is also satisfiable.

### Definition 5.1 (Satisfiability)

Let $\Gamma$ be a set of first order formulas.

We say that $\Gamma$ is satisfiable if there exists a structure $\mathcal{M}$ such that:
$$
\mathcal{M} \models A
$$
for every formula $A \in \Gamma$.

In other words, all formulas in $\Gamma$ are simultaneously true in the same structure.

If no such structure exists, then $\Gamma$ is called unsatisfiable.

### Definition 5.2 (Finite Satisfiability)

A set of formulas $\Gamma$ is finitely satisfiable if every finite subset:
$$
\Gamma_0 \subseteq \Gamma
$$
is satisfiable.

This means that for every finite collection of formulas taken from $\Gamma$, there exists a structure in which all formulas in that finite collection are true.

### Theorem 5.3 (Compactness Theorem)

Let $\Gamma$ be a set of first order formulas.

If $\Gamma$ is finitely satisfiable, then $\Gamma$ is satisfiable.

Equivalently, if every finite subset of $\Gamma$ has a model, then there exists a single structure that is a model of all formulas in $\Gamma$.

This result shows that no infinite obstruction to satisfiability can arise unless it is already present in some finite subset.

### Proof via Completeness

The proof uses the completeness theorem for first order logic.

Assume that $\Gamma$ is finitely satisfiable. Suppose, for contradiction, that $\Gamma$ is not satisfiable.

Then:
$$
\Gamma \models \bot
$$

By completeness:
$$
\Gamma \vdash \bot
$$

This means that there exists a finite subset:
$$
\Gamma_0 \subseteq \Gamma
$$
such that:
$$
\Gamma_0 \vdash \bot
$$

Again by completeness:
$$
\Gamma_0 \models \bot
$$

Thus $\Gamma_0$ is unsatisfiable, which contradicts the assumption that every finite subset of $\Gamma$ is satisfiable.

Therefore $\Gamma$ must be satisfiable.

### Alternative Formulation

The compactness theorem can also be stated in the following equivalent form.

If:
$$
\Gamma \models A
$$
then there exists a finite subset:
$$
\Gamma_0 \subseteq \Gamma
$$
such that:
$$
\Gamma_0 \models A
$$

This version says that if a formula is a consequence of a possibly infinite set of assumptions, then it is already a consequence of some finite subset of those assumptions.

### Example 5.4 (Infinite Set with Arbitrary Large Elements)

Consider a language with a unary predicate $P(x)$, and let:
$$
\Gamma = \{ \exists x_1, \dots, x_n \text{ such that all } x_i \text{ are distinct and } P(x_i) \text{ holds} \mid n \in \mathbb{N} \}
$$

This set says that there are at least $n$ distinct elements satisfying $P$ for every natural number $n$.

Every finite subset of $\Gamma$ only requires finitely many distinct elements, so it is satisfiable in a finite structure.

By compactness, the whole set $\Gamma$ is satisfiable, and therefore there exists a model in which there are infinitely many elements satisfying $P$.

This shows that first order logic can express the existence of arbitrarily large finite sets but cannot distinguish between arbitrarily large finite sets and infinite sets.

### Example 5.5 (Nonstandard Models of Arithmetic)

Let $\Gamma$ be the set consisting of the axioms of arithmetic together with an additional constant symbol $c$ and the formulas:
$$
c > 0, \quad c > 1, \quad c > 2, \quad \dots
$$

Every finite subset of $\Gamma$ only contains finitely many inequalities, so it can be satisfied in the standard model of arithmetic by interpreting $c$ as a sufficiently large natural number.

By compactness, the entire set $\Gamma$ is satisfiable, so there exists a model in which $c$ is greater than every standard natural number.

Such a model cannot be the standard model of arithmetic, so it must be a nonstandard model.

This demonstrates that first order arithmetic admits nonstandard models.

### Corollary 5.6 (No Finite Axiomatization of Finiteness)

There is no set of first order sentences that characterizes finite structures.

Proof

Suppose there were a set of sentences $\Gamma$ such that a structure satisfies $\Gamma$ if and only if it is finite.

Extend $\Gamma$ by adding sentences asserting the existence of arbitrarily many distinct elements, as in the earlier example.

Every finite subset of this extended theory is satisfiable in a finite structure.

By compactness, the entire theory is satisfiable, which gives a model that satisfies $\Gamma$ but is infinite, contradicting the assumption.

Therefore finiteness cannot be captured by first order sentences.

### Corollary 5.7 (Ultraproduct Intuition)

Compactness suggests that global models can be constructed by combining local consistency conditions, and this idea is made precise in the construction of ultraproducts, where models are built from families of structures by means of ultrafilters.

Although ultraproducts are not required for the proof of compactness, they provide a deeper understanding of why compactness holds and how models can be assembled from compatible pieces.

### Logical Consequences

Compactness has several important consequences.

If a theory has models of arbitrarily large finite size, then it has an infinite model.

If a theory has no infinite model, then there is a bound on the size of its models that can be detected by a finite subset of the theory.

If a property of structures fails only in infinite cases, then it cannot be expressed in first order logic.

These consequences show that first order logic has inherent limitations in expressing global properties such as finiteness, but at the same time it has strong local to global principles.

### Conceptual Meaning

The compactness theorem expresses a fundamental principle that logical consistency is determined by finite information, and that no infinite obstruction can arise unless it is already visible at a finite level.

This principle underlies many constructions in model theory, and it explains why first order logic behaves in ways that differ from stronger logical systems that do not satisfy compactness.

In practice, compactness is used to construct models with desired properties by specifying only finite conditions and relying on the theorem to ensure that these conditions can be satisfied simultaneously in a single structure.

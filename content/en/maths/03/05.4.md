## 5.4 Nonstandard Models

Compactness and the Löwenheim–Skolem theorems imply that many familiar theories have models that differ from the intended one. These are called **nonstandard models**.

A nonstandard model satisfies the same first-order axioms as a standard structure, but its internal elements and behavior differ in essential ways.

### Basic Idea

Let $T$ be a first-order theory describing a structure you know, such as arithmetic.

A **standard model** is the intended structure, for example:

$$
(\mathbb N; 0, 1, +, \cdot, <)
$$

A **nonstandard model** is any model $\mathcal M$ such that:

$$
\mathcal M \models T
$$

but $\mathcal M$ is not isomorphic to the standard model.

The key point is that first-order axioms usually do not determine a structure uniquely.

### Nonstandard Models of Arithmetic

Consider first-order arithmetic in the language:

$$
{0,1,+,\cdot,<}
$$

Let $T$ be a theory that includes all true first-order statements about $\mathbb N$.

Using compactness, we can construct a model containing an element larger than every standard natural number.

Introduce a new constant $c$ and add sentences:

$$
c > 0,\quad c > 1,\quad c > 2,\quad \dots
$$

Every finite subset of these sentences can be satisfied in $\mathbb N$ by choosing $c$ large enough.

By compactness, there exists a model $\mathcal M$ such that:

$$
\mathcal M \models T \cup {c > n : n \in \mathbb N}
$$

In this model, $c$ is greater than every standard natural number.

Such elements are called **nonstandard elements**.

### Structure of Nonstandard Integers

Nonstandard models of arithmetic have a rich internal structure.

They contain:

* all standard natural numbers
* additional elements larger than every standard number

These extra elements behave like infinite integers.

You can form sequences such as:

$$
c,\quad c+1,\quad c+2,\quad \dots
$$

These elements still satisfy the arithmetic axioms.

From inside the model, they look like ordinary numbers. The model cannot distinguish them as “nonstandard”.

From outside, we see that they extend beyond the usual natural numbers.

### Why Nonstandard Models Exist

The existence of nonstandard models comes from two facts:

1. First-order logic only uses finite formulas
2. Compactness allows infinite behavior to emerge from finite consistency

No finite collection of axioms can enforce that every element is reachable by repeatedly adding $1$ starting from $0$.

That property would require quantifying over all subsets or all sequences, which is not possible in first-order logic.

So first-order arithmetic cannot exclude nonstandard elements.

### Internal vs External View

A key idea is the difference between internal and external perspectives.

Inside a model $\mathcal M$, statements are evaluated using elements and functions available in $\mathcal M$.

Outside the model, we can compare $\mathcal M$ to the intended structure.

For example, in a nonstandard model:

* internally, the model may believe that a set is uncountable
* externally, we may see that the entire model is countable

This difference explains many apparent paradoxes.

### Example: Nonstandard Order

In the standard natural numbers, every number has only finitely many predecessors.

In a nonstandard model, a nonstandard element $c$ has infinitely many predecessors, including all standard numbers and many nonstandard ones.

The order looks like:

$$
0, 1, 2, \dots, c, c+1, c+2, \dots
$$

but also includes additional infinite descending and ascending patterns when viewed globally.

The structure is more complicated than the standard order on $\mathbb N$.

### Nonstandard Analysis

Nonstandard models are not only theoretical curiosities. They are used in mathematics.

In nonstandard analysis, one constructs fields containing infinitesimal and infinite elements.

For example, there are numbers $\varepsilon$ such that:

$$
0 < \varepsilon < \frac{1}{n} \quad \text{for all } n \in \mathbb N
$$

These behave like infinitesimals.

Such structures can be built using model-theoretic methods similar to those used for nonstandard arithmetic.

### Limitations of First-Order Axioms

Nonstandard models show that first-order theories often cannot uniquely describe their intended structures.

For arithmetic:

* Peano axioms in first-order logic have many non-isomorphic models
* there is no first-order sentence that picks out exactly $\mathbb N$

To achieve uniqueness, stronger logical systems are required, such as second-order logic.

### Consequences

Nonstandard models illustrate several general facts:

| Observation         | Meaning                                                           |
| ------------------- | ----------------------------------------------------------------- |
| Multiple models     | A theory may have many non-isomorphic models                      |
| Hidden elements     | Models may contain elements not present in the intended structure |
| Internal truth      | Truth is evaluated relative to a model                            |
| External comparison | We can compare models from outside                                |

These ideas are central in model theory.

### Perspective

Nonstandard models show that first-order logic is flexible but not rigid. It allows many interpretations of the same axioms.

This flexibility is a strength in algebra and geometry, where different models reveal different structures. It is a limitation when one wants a unique canonical model.

Understanding this balance is essential for working with first-order theories.

## Loanword Transcription Patterns

### Terminal Consonants
English consonants at the end of words get a vowel appended:

| English | Katakana | Rule |
|---------|----------|------|
| cake | ケーキ (kēki) | final k → ki |
| milk | ミルク (miruku) | final k → ku |
| test | テスト (tesuto) | final t → to |
| check | チェック (chekku) | final ck → kku (geminate) |
| bed | ベッド (beddo) | final d → do (or dd if stressed) |
| bus | バス (basu) | final s → su |
| club | クラブ (kurabu) | final b → bu |

### Double Consonants → ッ
English double/stressed consonants become ッ (small tsu):
- butter → バター or バッター (battā) 
- pizza → ピザ or ピッツァ (pittsa)
- coffee → コーヒー (kōhī, 'f' + long vowel)
- pocket → ポケット (poketto) — the 't' is doubled
- ticket → チケット (chiketto)
- soccer → サッカー (sakkā)

### R and L → ラ行
Japanese has no separate L and R — both become the ラ行 (ra/ri/ru/re/ro):
- rice → ライス (raisu)
- like → ライク (raiku)  
- love → ラブ (rabu)
- light → ライト (raito)
- lobby → ロビー (robī)
- library → ライブラリー (raibararī)

### Common Abbreviations in Japanese Loanwords

Many English loanwords are abbreviated in Japanese:
| English | Japanese | Abbrev. rule |
|---------|----------|-------------|
| personal computer | パソコン (pasokon) | pa(āsonaru) + ko(mpyu)n |
| smartphone | スマホ (sumaho) | su(māto)+ ho(n, phone) |
| air conditioner | エアコン (eakon) | ea(ko)n |
| remote control | リモコン (rimokon) | ri(mōto)+ ko(ntorōru)n |
| convenience store | コンビニ (konbini) | ko(nbiniensu)+ ni |
| demonstration | デモ (demo) | first 2 morae |
| animation | アニメ (anime) | first 3 morae |
| television | テレビ (terebi) | first 3 morae |
| application (app) | アプリ (apuri) | a(ppuri)ke-sho(n) |
| infrastructure | インフラ (infura) | first 3 morae |

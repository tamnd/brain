20 fundamental N5 grammar patterns at the A1 beginner level. These are the building blocks of Japanese sentence structure — master these before moving on to N4.

| Pattern | Meaning |
|---------|---------|
| [は (wa)](は.md) | topic marker |
| [が (ga)](が.md) | subject marker |
| [を (wo)](を.md) | direct object marker |
| [に (ni)](に.md) | direction / time / indirect object |
| [で (de)](で.md) | location of action / means |
| [の (no)](の.md) | possession / noun modification |
| [と (to)](と.md) | and / with (accompaniment) |
| [も (mo)](も.md) | also / too / even |
| [から (kara)](から.md) | from / because |
| [まで (made)](まで.md) | until / as far as |
| [ます (masu)](ます.md) | polite non-past verb form |
| [です (desu)](です.md) | copula (is/am/are) |
| [ている (te-iru)](ている.md) | ongoing action / resultant state |
| [てください (te-kudasai)](てください.md) | please do ~ |
| [たい (tai)](たい.md) | want to do ~ |
| [あります / います](あります-います.md) | existence of things / beings |
| [ましょう (mashou)](ましょう.md) | let's ~ / shall we ~ |
| [ませんか (masen ka)](ませんか.md) | won't you ~ / invitation |
| [ね / よ](ね-よ.md) | sentence-final particles (agreement / assertion) |
| [これ / それ / あれ / どれ](これ-それ-あれ.md) | demonstratives |

## 〜と

| Property | Value |
|----------|-------|
| Pattern | 〜と |
| JLPT Level | N4 |
| Type | conditional |
| Formality | Neutral |
| Register | Both |

## Meaning

The `〜と` conditional expresses a natural, inevitable, or mechanical consequence. It is used to state that "whenever X happens, Y always follows" or "if X happens, Y will happen as a direct result."

## Structure Formula

```
[Verb dictionary form] + と
[い-adj (dictionary form)] + と
[な-adj (stem)] + だ + と
[Noun] + だ + と
```

*   **Verbs:** Use the dictionary form (non-past).
*   **Adjectives/Nouns:** Use the plain form (e.g., `静かだ` (shizuka da) + `と`).
*   **Restriction:** The second clause (the result) **cannot** contain volitional expressions, requests, commands, or invitations (e.g., `〜てください`, `〜ましょう`, `〜たい`).

## Detailed Explanation

The `〜と` conditional is often called the "natural consequence" conditional. It describes a situation where the outcome is predictable, objective, or a matter of fact. Because it implies an automatic result, it is frequently used for scientific facts, instructions, or describing how to use machinery.

This pattern is highly objective. Unlike `〜ば` or `〜たら`, which can express personal intentions or hypothetical scenarios, `〜と` focuses on the "if/when" relationship. If you press a button, the light turns on; if you turn left, the park is there. These are universal truths within the context of the conversation.

Because `〜と` implies an inevitable result, you cannot use it to express your own desire to do something or to ask someone else to do something. For example, you cannot say "If it rains, let's go home" using `〜と` because "let's go" is a suggestion.

## Example Sentences

## Comparison with Similar Patterns

### 〜と vs 〜ば vs 〜たら

| Pattern | Nuance | Focus |
|---------|--------|---------|
| **〜と** | Inevitable/Natural result | Fact/Mechanical |
| **〜ば** | Hypothetical/Condition | Requirement |
| **〜たら** | Sequence/After X happens | Personal/Specific |

*   **〜と:** "Whenever I do X, Y happens." (Objective)
*   **〜ば:** "If X happens, then Y (is the condition for Y)." (Hypothetical)
*   **〜たら:** "After X happens, then Y." (Focus on the completion of the first action)

## Common Mistakes

❌ **雨が降ると、傘を買いましょう。** → ✓ **雨が降ったら、傘を買いましょう。**
*Explanation:* `〜と` cannot be followed by a suggestion (`〜ましょう`). Use `〜たら` for suggestions.

❌ **明日、時間があると、映画を見たいです。** → ✓ **明日、時間があれば、映画を見たいです。**
*Explanation:* `〜と` is for universal facts, not personal desires or future intentions. `〜ば` is better for hypothetical personal plans.

❌ **ボタンを押すと、押してください。** → ✓ **ボタンを押すと、電気がつきます。**
*Explanation:* You cannot use `〜と` to link a condition to a request.

## Practice Exercises

**Fill in the blank** (use 〜と or the correct form):

1. この道をまっすぐ（行く）＿＿＿、駅があります。
2. 寒く（なる）＿＿＿、ストーブをつけます。
3. 毎日練習する（だ）＿＿＿、上手になります。
4. 授業に（遅れる）＿＿＿、先生に怒られます。
5. 信号が赤（だ）＿＿＿、止まってください。 (Wait! Is this correct? Why?)

**Answer Key:**
1. **行くと** (Natural consequence of walking straight).
2. **なると** (Natural consequence of temperature dropping).
3. **と** (Wait: `練習すると` is correct; `練習するだ` is wrong. Correction: **練習すると**).
4. **遅れると** (Natural consequence of being late).
5. **赤だと、止まってください** is **INCORRECT**. Because `〜と` cannot be followed by a command/request, use `赤なら、止まってください` or `赤の時は、止まってください`.

## JLPT Level Notes

In the JLPT N4, `〜と` is frequently tested in "reordering" sentences or "fill-in-the-blank" grammar sections. Watch out for the trap where the second clause contains a request or a volitional verb (`〜たい`, `〜ましょう`, `〜てください`). If you see those in the second half of the sentence, **do not** choose `〜と`.

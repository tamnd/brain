Japanese grammar is structurally unlike English, but it follows consistent patterns once understood. This guide covers the core grammar framework — particles, verb groups, conjugation forms, and key sentence patterns — organized from N5 to N2 difficulty.

## Core Structural Differences from English

| Feature | English | Japanese |
|---------|---------|---------|
| Word order | SVO (I eat sushi) | SOV (私はすしを食べる — I sushi eat) |
| Verb position | 2nd element | Always last |
| Articles | a, an, the | None |
| Plural marking | -s, -es | None |
| Verb tense | Past/present/future | Past vs. non-past (aspect more important than tense) |
| Subject | Usually explicit | Often omitted if clear from context |
| Relative clauses | After noun | Before noun (modifies to the left) |
| Politeness | Lexical (please, would) | Grammatical (ます/です vs. plain forms; full keigo system) |

## The Particle System

Particles (助詞, joshi) are postpositions that follow nouns, pronouns, and phrases to mark their grammatical role. Unlike English prepositions (which come before), Japanese particles come after.

### Core Particles

| Particle | Function | Example | Translation |
|----------|----------|---------|-------------|
| は (wa) | Topic marker | 私**は**学生です | I am a student (as for me) |
| が (ga) | Subject marker | 猫**が**います | There is a cat |
| を (o) | Object marker | りんご**を**食べる | Eat an apple |
| に (ni) | Direction; time; indirect object | 東京**に**行く; 3時**に**来る | Go to Tokyo; Come at 3 |
| で (de) | Location of action; means/method | 学校**で**勉強する; 電車**で**行く | Study at school; Go by train |
| へ (e) | Direction (softer/literary) | 日本**へ**行く | Go to/toward Japan |
| と (to) | And (exhaustive); with; quotation | 友達**と**行く; 「いい」**と**思う | Go with a friend; Think "good" |
| の (no) | Possession; nominalization | 私**の**本; 行く**の**が好き | My book; Like going |
| か (ka) | Question marker | これは何**か** | What is this? |
| も (mo) | Also; even; neither/nor | 私**も**行く | I will go too |
| から (kara) | From; because | 東京**から**来た; 眠い**から**帰る | Came from Tokyo; Go home because sleepy |
| まで (made) | Until; to (endpoint) | 5時**まで**; 東京**まで** | Until 5; As far as Tokyo |
| より (yori) | Than; from | これ**より**高い; 東京**より**来た | More expensive than this; From Tokyo |
| ね (ne) | Seeking agreement/confirmation | 寒いです**ね** | It's cold, isn't it |
| よ (yo) | Assertion; informing | そうです**よ** | That's right (I'm telling you) |

### は vs. が

The は/が distinction is one of the most discussed grammar points:
- **は** (topic): "As for X, ..." — X is the topic of discussion; may contrast with something else
- **が** (subject): marks who/what does the action; new information; emphasis; subject of subordinate clauses

Examples:
- 象**は**鼻が長い。 — As for elephants, their nose is long. (は = topic)
- 猫**が**います。 — There is a cat (not a dog). (が = new info/emphasis)
- 私**は**行くが、彼女**は**行かない。 — I will go, but she won't. (は = contrast)

## Verb Groups (三分類)

Japanese verbs are divided into three groups:

### Group 1 (Godan/u-verbs 五段活用)
The dictionary form ends in an う-sound (u, ku, gu, su, tsu, nu, bu, mu, ru):
- 書く (kaku, write), 話す (hanasu, speak), 読む (yomu, read), 飲む (nomu, drink)
- **Key**: 帰る (kaeru), 走る (hashiru), 切る (kiru) look like ru-verbs but are Godan!

### Group 2 (Ichidan/ru-verbs 一段活用)
The dictionary form ends in -iru or -eru:
- 食べる (taberu, eat), 見る (miru, see), 起きる (okiru, wake up), 出る (deru, leave)

### Group 3 (Irregular 不規則)
Only two verbs:
- する (suru, to do)
- くる (kuru, to come)

## Core Verb Conjugation Forms

### Masu-form (Polite Non-past)
Used in formal/polite speech. Formed from verb stem + ます:
- Godan: 書く → か + き + ます = **書きます**
- Ichidan: 食べる → たべ + ます = **食べます**
- する → **します**; くる → **きます**

### Ta-form (Plain Past)
Godan: changes final syllable:
- く → いた (書く → 書いた)
- ぐ → いだ (泳ぐ → 泳いだ)
- す → した (話す → 話した)
- つ/う/る → った (立つ → 立った, 買う → 買った, 帰る → 帰った)
- ぬ/ぶ/む → んだ (飲む → 飲んだ, 遊ぶ → 遊んだ)

Ichidan: replace る with た: 食べる → **食べた**; 見る → **見た**

### Te-form (Connector Form)
Same pattern as ta-form but with て instead of た (and で instead of だ):
- 書く → **書いて**; 泳ぐ → **泳いで**; 話す → **話して**
- 食べる → **食べて**; 見る → **見て**
- する → **して**; くる → **きて**

Te-form is the basis for dozens of grammar patterns:
- 〜てください (please do)
- 〜ている (ongoing action / resultant state)
- 〜てもいい (may do)
- 〜てはいけない (must not do)
- 〜てから (after doing)
- 〜てみる (try doing)
- 〜てしまう (do completely / unfortunately do)
- 〜てあげる/くれる/もらう (giving/receiving actions)

### Nai-form (Plain Negative)
Godan: change u → a + ない: 書く → **書かない**; 飲む → **飲まない**

Exception: ある → **ない** (not あらない)
For words ending in -u vowel: 買う → **買わない** (not 買あない)

Ichidan: remove る, add ない: 食べる → **食べない**; 見る → **見ない**

### Potential Form (can do)
- Godan: e-stem + る: 書く → **書ける**; 読む → **読める**
- Ichidan: add られる: 食べる → **食べられる** (also contracted to 食べれる in casual speech)
- する → **できる**; くる → **こられる**

### Passive Form (be done to)
- Godan: a-stem + れる: 書く → **書かれる**; 読む → **読まれる**
- Ichidan: add られる: 食べる → **食べられる**
- する → **される**; くる → **こられる**

### Causative Form (make/let someone do)
- Godan: a-stem + せる: 書く → **書かせる**; 飲む → **飲ませる**
- Ichidan: add させる: 食べる → **食べさせる**
- する → **させる**; くる → **こさせる**

### Causative-Passive (be made to do)
- Godan: a-stem + せられる (or せられる → される in casual): 書く → **書かせられる**
- Ichidan: add させられる: 食べる → **食べさせられる**
- する → **させられる**

## Adjective Forms

### I-adjectives (〜い)
Conjugate by changing the final い:
- Present: 大きい (ookii, big)
- Past: 大き**かった** (ookikatta, was big)
- Negative: 大き**くない** (ookikunai, not big)
- Negative past: 大き**くなかった** (ookikunakatta, was not big)
- Adverb form: 大き**く** (ookiku, bigly/in a big way)
- Te-form: 大き**くて** (ookikute, being big; big and...)

Exception: いい/よい (good) → よかった (past), よくない (neg.), よくなかった (neg. past)

### Na-adjectives (〜な)
Conjugate using the copula (だ/です):
- Present: きれい**だ** / きれい**です** (kirei da/desu, is pretty)
- Past: きれい**だった** / きれい**でした** (kireida tta/deshita, was pretty)
- Negative: きれい**じゃない** / きれい**ではない** (kirei ja nai, is not pretty)
- Adverb: きれい**に** (kirei ni, prettily; becomes adverb with に)
- Te-form: きれい**で** (kirei de, being pretty; pretty and...)

## Key Sentence Patterns by Level

### N5 Patterns
| Pattern | Structure | Example | Meaning |
|---------|-----------|---------|---------|
| A は B です | A + は + B + です | 私は学生です | I am a student |
| A が います/あります | A + が + います/あります | 猫がいます | There is a cat |
| V + ます | Verb (polite) | 食べます | [I] eat |
| V + たい | want to V | 行きたい | I want to go |
| V + ている | ongoing/resultant | 食べています | [I] am eating |

### N4 Patterns
| Pattern | Structure | Example | Meaning |
|---------|-----------|---------|---------|
| V[て] + ください | polite request | 食べてください | Please eat |
| V[て] + もいい | permission | 食べてもいい | May eat |
| V[て] + はいけない | prohibition | 食べてはいけない | Must not eat |
| V[たら] + | conditional | 着いたら電話して | Call when you arrive |
| V[potential] | can do | 食べられる | Can eat |
| AげるもらうくれるB | giving/receiving | 友達にあげた | Gave to friend |

### N3 Patterns
| Pattern | Structure | Example | Meaning |
|---------|-----------|---------|---------|
| 〜てしまう | completion/regret | 忘れてしまった | I (unfortunately) forgot |
| 〜てみる | try doing | 食べてみる | Try eating |
| 〜ようだ | seems like | 疲れているようだ | Seems tired |
| 〜そうだ (appearance) | looks like | 雨が降りそうだ | Looks like rain |
| Passive 〜れる/られる | be done to | 怒られた | Was scolded |
| Causative 〜させる | make/let do | 食べさせる | Make/let eat |
| 〜ばよかった | should have | 早く来ればよかった | Should have come earlier |

### N2 Patterns
| Pattern | Structure | Example | Meaning |
|---------|-----------|---------|---------|
| 〜に対して | toward/regarding | 問題に対して | Regarding the problem |
| 〜にとって | for/from perspective | 私にとって | For me |
| 〜ものの | although | 知っていたものの | Although I knew |
| 〜わけだ | that's why/naturally | 当然のわけだ | Of course that's why |
| 〜ばかりか | not only | 便利なばかりか | Not only convenient but |
| 〜ながらも | even while | 知りながらも | Even while knowing |

## Pattern: 〜さえ〜ば

**Structure:**
- Noun + さえ + [conditional ば/なら] → "if only ~, as long as ~"
- Verb (masu-stem) + さえ + すれば → "if only one does ~"
- Verb (TE-form) + さえ + いれば → "as long as one is doing/maintaining ~"
- I-adj (stem) + さえ + あれば/〜ければ

**Formality:** Standard (spoken and written)

**Meaning:** "if only ~, as long as ~" — minimal sufficient condition

## Explanation

〜さえ〜ば marks a *single minimal sufficient condition*: "this one thing (and nothing else) needs to be met, and everything will be fine." さえ carries the nuance of "even just ~" or "as long as even ~." It emphasizes that the condition is the *minimum* requirement — if this is satisfied, the desired result follows.

This pattern is often used to:
- Reassure someone that a task is simpler than it seems
- Express what the single key factor for success is
- Highlight the minimal requirement in a conditional

## Example Sentences

| Japanese | Reading | Romaji | English |
|----------|---------|--------|---------|
| 健康でさえあれば、他に何もいらない。 | けんこうでさえあれば、ほかになにもいらない。 | Kenkō de sae areba, hoka ni nani mo iranai. | As long as I have my health, I need nothing else. |
| このパスワードさえ入れれば、ログインできる。 | このぱすわーどさえいれれば、ろぐいんできる。 | Kono pasuwādo sae irereba, roguin dekiru. | As long as you enter this password, you can log in. |
| 毎日少しさえ練習すれば、必ず上達する。 | まいにちすこしさえれんしゅうすれば、かならずじょうたつする。 | Mainichi sukoshi sae renshū sureba, kanarazu jōtatsu suru. | If you practice even just a little every day, you will improve. |
| 彼に連絡さえすれば、すぐに解決する。 | かれにれんらくさえすれば、すぐにかいけつする。 | Kare ni renraku sae sureba, sugu ni kaiketsu suru. | If only you contact him, it will be resolved immediately. |
| 天気でさえよければ、試合は行われる。 | てんきでさえよければ、しあいはおこなわれる。 | Tenki de sae yokereba, shiai wa okonawareru. | As long as the weather is fine, the match will be held. |

## Common Mistakes

❌ 行きさえすれば楽しいだろう。(weak — lacks certainty)  
✓ 行きさえすれば必ず楽しいはずだ。— Add certainty (必ず, はず) to strengthen the conditional.

❌ 勉強さえすれば合格した。(past tense in result — さえ〜ば is for future/hypothetical)  
✓ 勉強さえすれば合格できる。

## Comparison with Similar Patterns

| Pattern | Meaning | Example |
|---------|---------|---------|
| 〜さえ〜ば | Minimal sufficient condition | 合格さえすれば問題ない |
| 〜ば | General hypothetical condition | 勉強すれば合格できる |
| 〜たら | "If/when ~ (then)" | 春になったら花見をしよう |
| 〜だけで | "Just by doing ~" | 練習するだけで上達する |

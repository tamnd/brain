## Character Info

| Property | Value |
|----------|-------|
| Kanji | 図 |
| Strokes | 7 |
| Radical | 囗 (enclosure) |
| JLPT Level | N4 |

## Readings

| Type | Japanese | Romaji |
|------|----------|--------|
| On-yomi | ズ、ト | zu, to |
| Kun-yomi | はか・る | haka-ru |

## Vocabulary

| 図書館 | としょかん | toshokan | library |
| 地図 | ちず | chizu | map |
| 図形 | ずけい | zukei | shape, figure |
| 図書 | としょ | tosho | books, library books |
| 設計図 | せっけいず | sekkeizu | blueprint |

## Stroke Order

1-3. Outer frame 囗 (enclosure). 4-7. Inner ※ pattern (X with dots).

## Mnemonic

An **enclosed box** with a **grid/X pattern** inside — like a **diagram** or **blueprint** drawn in a box.

## Common Confusion Pairs

| 国 (country) | 国 has 玉 inside; 図 has the X-pattern |
| 回 (turn/return) | 回 has 口 inside; 図 has more strokes inside |

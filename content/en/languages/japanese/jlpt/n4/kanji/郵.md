## Character Info

| Property | Value |
|----------|-------|
| Kanji | 郵 |
| Strokes | 11 |
| Radical | 阝(mound/town) |
| JLPT Level | N4 |

## Readings

|------|----------|--------|
| On-yomi | ユウ | yū |

## Vocabulary

| 郵便局 | ゆうびんきょく | yūbinkyoku | post office |
| 郵便 | ゆうびん | yūbin | mail, postal service |
| 郵送 | ゆうそう | yūsō | mailing, sending by post |
| 郵便番号 | ゆうびんばんごう | yūbin bangō | postal code |
| 郵便ポスト | ゆうびんぽすと | yūbin posuto | mailbox |

## Stroke Order

1-8. Left portion 垂 (hang down). 9-11. Right radical 阝(town/place).

## Mnemonic

Things **hanging** (垂) in a **town** (阝) building — **mail** hanging in the post office waiting to be collected.

## Common Confusion Pairs

| 睡 (sleep) | 睡 has 目 on left; 郵 has different left portion |
| 邦 (country) | 邦 has 丰 on left; 郵 has 垂 |

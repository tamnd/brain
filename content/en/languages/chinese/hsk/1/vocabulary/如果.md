## 如果 (rúguǒ)

如果 means "if" or "supposing that." It introduces a hypothetical or conditional situation, and the result clause often contains 就 (jiù, then).

## Meanings

1. **[conjunction]** If; supposing; in the event that (introduces a condition).

## Example Sentences

**如果明天下雨，我就不去了。**
Rúguǒ míngtiān xià yǔ, wǒ jiù bù qù le.
If it rains tomorrow, I will not go.

**如果你饿了，就告诉我。**
Rúguǒ nǐ è le, jiù gàosu wǒ.
If you are hungry, just let me know.

**如果有问题，可以问我。**
Rúguǒ yǒu wèntí, kěyǐ wèn wǒ.
If you have questions, you can ask me.

**如果你学习努力，就会成功。**
Rúguǒ nǐ xuéxí nǔlì, jiù huì chénggōng.
If you study hard, you will succeed.

## Usage Notes

如果 typically appears at the start of the first clause. The result clause is often introduced by 就 (jiù, then) to complete the if-then structure: 如果……就……. 如果 can be shortened to 若 (ruò) in formal written Chinese. A similar word 要是 (yàoshi) is more colloquial and interchangeable in everyday speech. Unlike English, the conditional and result clauses use the same verb tense in Chinese — there is no subjunctive.

## Collocations or Common Phrases

| Phrase | Meaning |
|--------|---------|
| 如果……就…… | if... then... |
| 如果可以 | if possible |
| 如果不是 | if it were not for |
| 如果我是你 | if I were you |
| 要是 (synonym) | if (colloquial) |

## Memory Hook

如果 sounds like "roo-gwaw" — imagine a kangaroo (roo) eating a guava (gwaw) — if a kangaroo eats a guava, something will happen.

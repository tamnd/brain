## 怎么 (zěnme)

An interrogative word used before a verb to ask "how" something is done or "why" something happened; for asking about state or condition (how something is), use 怎么样 instead.

## Meanings

1. **[adverb]** How (manner of an action: 怎么 + verb).
2. **[adverb]** Why, how come (expressing surprise at a situation).

## Example Sentences

**这个字怎么读？**
Zhège zì zěnme dú?
How do you read this character?

**你怎么来这里的？**
Nǐ zěnme lái zhèlǐ de?
How did you get here?

**你怎么不吃饭？**
Nǐ zěnme bù chīfàn?
Why aren't you eating?

**这道题怎么做？**
Zhè dào tí zěnme zuò?
How do you do this problem?

## Collocations

| Collocation | Meaning |
|-------------|---------|
| 怎么了 (zěnme le) | what happened, what's wrong |
| 怎么说 (zěnme shuō) | how to say |
| 怎么做 (zěnme zuò) | how to do |
| 怎么走 (zěnme zǒu) | which way to go, how to get there |
| 怎么回事 (zěnme huí shì) | what is going on |
| 怎么办 (zěnme bàn) | what to do, what should be done |

## Memory Hook

怎么 sounds like "zen-muh." Picture a Zen monk saying "zen-muh... HOW does one find peace?" Asking how to do something with deep, thoughtful inquiry.

HSK 5 grammar covers approximately 85 core patterns for advanced B2-level communication, with a strong focus on formal written registers, complex clause structures, academic discourse connectives, and nuanced modal expressions.

## 1. Formal Written Connectives

### 1.1 Causation and Result (Formal)

| Pattern | Register | Example | English |
|---------|----------|---------|---------|
| 因此 | formal | 市场不稳定，因此投资者谨慎。| The market is unstable; therefore investors are cautious. |
| 从而 | written | 提高效率，从而降低成本。| Improve efficiency, thereby reducing costs. |
| 以致/致使 | written | 管理不当，以致项目失败。| Poor management led to the project's failure. |
| 于是 | neutral-formal | 他迟到了，于是被批评了。| He was late and was consequently criticized. |
| 进而 | written | 了解问题，进而找到解决方案。| Understand the problem and further find a solution. |

### 1.2 Concession (Formal)

| Pattern | Register | Example | English |
|---------|----------|---------|---------|
| 固然…但是/然而 | formal | 固然有困难，但仍需坚持。| True there are difficulties, but one must persevere. |
| 诚然…然而 | literary | 诚然如此，然而不可一概而论。| Admittedly so, yet one cannot generalize. |
| 尽管如此 | written | 尽管如此，计划照常进行。| Despite this, the plan proceeds as normal. |

### 1.3 Contrast and Transition

| Pattern | Register | Example | English |
|---------|----------|---------|---------|
| 然而 | formal | 计划很好，然而执行困难。| The plan is good; however, execution is difficult. |
| 反之 | written | 努力则成功，反之则失败。| Work hard and you succeed; conversely, you fail. |
| 相反 | neutral | 不是变好了，相反，变糟了。| It didn't improve; on the contrary, it got worse. |
| 况且 | spoken-formal | 已经很晚了，况且还下雨。| It's already late, and moreover it's raining. |

## 2. Complex Conditional Structures

### 2.1 Hypothetical and Contrary-to-Fact

| Pattern | Example | English |
|---------|---------|---------|
| 假如/假设…就/那么 | 假如有机会，我就去留学。| If I had the chance, I'd study abroad. |
| 倘若/倘使 (formal) | 倘若失败，当如何应对？| If we fail, how should we respond? |
| 一旦…就/便 | 一旦决定，就要坚持。| Once decided, one must persevere. |
| 万一…就/也 | 万一失败，也不要放弃。| Even if we fail, don't give up. |

### 2.2 Graduated Conditions

| Pattern | Example | English |
|---------|---------|---------|
| 即使…也 | 即使再难，也要坚持。| Even if it's harder, must still persevere. |
| 哪怕…也 | 哪怕下雪，也要出发。| Even if it snows, we must set off. |
| 就算…也 | 就算累了，也不能停。| Even if tired, cannot stop. |

## 3. Advanced Modal and Epistemic Expressions

### 3.1 Possibility and Certainty

| Expression | Register | Meaning | Example |
|------------|----------|---------|---------|
| 未必 | formal | not necessarily | 他未必知道。He doesn't necessarily know. |
| 势必 | formal | inevitably | 这势必影响结果。This will inevitably affect the results. |
| 必将 | written | will certainly | 努力必将得到回报。Hard work will certainly be rewarded. |
| 理应 | formal | ought to, should rightly | 理应受到奖励。Should rightfully be rewarded. |
| 难免 | neutral | unavoidable, bound to | 难免出错。Bound to make mistakes. |

### 3.2 Subjective Evaluation Adverbs

| Expression | Meaning | Example |
|------------|---------|---------|
| 似乎/仿佛 | seem as if | 他似乎不高兴。He seems unhappy. |
| 显然 | obviously | 显然这是错的。Obviously this is wrong. |
| 明显 | clearly/evidently | 明显有所改善。Clearly there has been improvement. |
| 大概/约 | approximately / probably | 大概需要三天。Probably takes three days. |
| 究竟 | after all / exactly | 究竟是谁？Who exactly is it? |

### 4.1 Non-Existence / Absence (Formal)

|---------|---------|---------|

## 5. Relative Clauses and Nominalization

### 5.1 的 as Nominalizer

| Pattern | Example | English |
|---------|---------|---------|
| V + 的 → Noun | 来的是我朋友。| The one who came is my friend. |
| Adj + 的 → Noun | 好的要多学习。| Good students should study more. |
| Clause + 的 → Noun | 我说的都是真的。| What I said is all true. |

### 5.2 所 + V (Formal Relative Clause Marker)

| Pattern | Example | English |
|---------|---------|---------|
| 所 + V + 的 + N | 他所说的话 | the words he said |
| 所 + V + 的 + N | 我们所面临的挑战 | the challenges we face |
| 所 + V | 各尽所能。| Each does their utmost. (lit. each exerts what they can) |

## 7. Expressing Degree and Scope

### 7.1 程度副词 (Degree Adverbs)

| Adverb | Meaning | Example |
|--------|---------|---------|
| 相当 | quite, considerably | 相当重要 quite important |
| 颇 | quite, considerably (literary) | 颇有成效 quite effective |
| 格外 | especially, particularly | 格外重要 especially important |
| 尤为/尤其 | especially, particularly | 尤为关键 particularly critical |
| 极其 | extremely | 极其复杂 extremely complex |
| 十分 | very, fully | 十分满意 very satisfied |

### 7.2 范围副词 (Scope Adverbs)

| Adverb | Meaning | Example |
|--------|---------|---------|
| 均 | all (formal) | 均已完成。All have been completed. |
| 皆 | all (literary) | 皆大欢喜。Everyone is pleased. |
| 凡 | every, all that (formal) | 凡参与者均有奖。All participants get a prize. |
| 一概 | without exception | 一概不接受。Accept none without exception. |
| 一律 | uniformly, all alike | 一律按规定处理。Handle all according to regulations. |

## 8. Advanced Aspect and Temporal Patterns

### 8.1 了 in Complex Structures

| Pattern | Example | English |
|---------|---------|---------|
| V1 + 了 + V2 (sequential) | 吃了饭再去。| Eat first, then go. |
| V + 了 + Numeral + 之后 | 等了三个小时之后才来。| Came only after waiting three hours. |

### 8.2 将 (jiāng) — Formal Future

| Pattern | Example | English |
|---------|---------|---------|
| 将 + V | 会议将于明天举行。| The meeting will be held tomorrow. |
| 将要 + V | 工程将要竣工。| The project is about to be completed. |
| 即将 + V | 即将公布结果。| The results are about to be announced. |

### 8.3 正值/恰逢 — Just When, Coinciding With

| Pattern | Example | English |
|---------|---------|---------|
| 正值 + N | 正值国庆节期间 | during the National Day holiday |
| 恰逢 + N | 恰逢百年纪念 | just coinciding with the centennial anniversary |

## 9. Formal Discourse Markers

### 9.1 Text-Organizing Markers

| Marker | Function | Example |
|--------|----------|---------|
| 首先…其次…最后 | listing order | 首先分析，其次讨论，最后总结。|
| 一方面…另一方面 | two-sided | 一方面要…另一方面也要… |
| 就…而言 | as for, regarding | 就教育而言，政策需要改革。|
| 综上所述 | in summary | 综上所述，该项目可行。|
| 由此可见 | from this we can see | 由此可见问题所在。|
| 换言之 | in other words | 换言之，这是不可能的。|

### 9.2 Quotation and Attribution

| Pattern | Example | English |
|---------|---------|---------|
| 据 + Source + 称/说 | 据报道，事件已解决。| According to reports, the incident has been resolved. |
| 正如 + Source + 所说 | 正如专家所说，问题复杂。| As experts have said, the problem is complex. |
| 对此，…认为 | 对此，学界认为… | Regarding this, academics believe… |

## 10. Rhetorical and Argumentative Structures

### 10.1 Concession–Rebuttal Pattern

| Pattern | Example | English |
|---------|---------|---------|
| 诚然/固然 + Concession，但/然而 + Rebuttal | 诚然改革有风险，但不改革风险更大。| True that reform carries risks, but not reforming carries greater risks. |

### 10.2 Hypothetical Argument

| Pattern | Example | English |
|---------|---------|---------|
| 试想/试问 | 试想，若无规则，社会将如何？| Just imagine: if there were no rules, what would society look like? |
| 倘若如此 | 倘若如此，后果难以设想。| If that were the case, the consequences would be hard to imagine. |

### 10.3 Emphasis by Inversion

| Pattern | Example | English |
|---------|---------|---------|
| 正是…才… | 正是这种精神，才使他成功。| It is precisely this spirit that made him succeed. |
| 之所以…是因为 | 之所以失败，是因为准备不足。| The reason for failure was insufficient preparation. |

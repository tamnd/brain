## 写错 (xiě cuò)

A verb-result compound where 写 (to write) + 错 (wrong/mistaken) means to write something incorrectly.

## Meanings

1. **[verb + complement]** To write incorrectly, to make a mistake in writing.

## Example Sentences

**我把这个字写错了。**
Wǒ bǎ zhège zì xiě cuò le.
I wrote this character incorrectly.

**你的名字写错了，应该是这样写。**
Nǐ de míngzì xiě cuò le, yīnggāi shì zhèyàng xiě.
Your name is written incorrectly; it should be written this way.

**考试的时候，我写错了几道题。**
Kǎoshì de shíhou, wǒ xiě cuò le jǐ dào tí.
During the exam, I wrote several questions incorrectly.

**没关系，写错了可以改正。**
Méi guānxi, xiě cuò le kěyǐ gǎizhèng.
It's okay; if you write it wrong, you can correct it.

## Collocations

| Collocation | Meaning |
|-------------|---------|
| 写错了 | wrote it wrong |
| 没写错 | didn't write it wrong |
| 写错字 | write a character incorrectly |
| 写错名字 | write a name wrong |
| 不要写错 | don't write it wrong |

## Memory Hook

错 is the result complement revealing the outcome — the writing action produced an error. 写 + 错 = the write arrived at "wrong."

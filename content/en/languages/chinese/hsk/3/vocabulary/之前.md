## 之前 (zhīqián)

之前 means "before" or "prior to." It marks a point in time earlier than a specified event or action. Like 之后, it follows the reference point (a noun, verb phrase, or clause) that anchors the timeline.

## Meanings

1. **[adverb]** Before; prior to; previously — indicates that the action took place earlier than the stated reference point.

### Compare 之前 vs 以前

| Word | Usage |
|------|-------|
| 之前 | Requires a preceding reference point; slightly more formal |
| 以前 | Can stand alone meaning "in the past" or follow a reference point |

## Example Sentences

**睡觉之前，我喜欢看一会儿书。**
Shuìjiào zhīqián, wǒ xǐhuān kàn yīhuìr shū.
Before going to sleep, I like to read for a while.

**出发之前，别忘了带护照。**
Chūfā zhīqián, bié wàngle dài hùzhào.
Before setting off, don't forget to bring your passport.

**做决定之前，你应该多想想。**
Zuò juédìng zhīqián, nǐ yīnggāi duō xiǎngxiang.
Before making a decision, you should think it over more carefully.

**来中国之前，他已经学了一年中文。**
Lái Zhōngguó zhīqián, tā yǐjīng xué le yī nián Zhōngwén.
Before coming to China, he had already studied Chinese for a year.

## Collocations

| Collocation | Meaning |
|-------------|---------|
| 此前 | before this; previously (formal) |
| …之前不久 | shortly before… |
| 开始之前 | before starting |
| 离开之前 | before leaving |
| 睡觉之前 | before going to bed |

## Memory Hook

前 means "front / before." 之前 = "before [what was] mentioned" — standing in front of the reference event, looking back from it.

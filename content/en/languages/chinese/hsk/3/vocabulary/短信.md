## 短信 (duǎnxìn)

短信 means text message or SMS. 短 means short and 信 means message or letter. Together they describe the brief electronic messages sent between mobile phones.

## Meanings

1. **[noun]** Text message; SMS; short message.

## Example Sentences

**我给他发了一条短信。**
Wǒ gěi tā fā le yī tiáo duǎnxìn.
I sent him a text message.

**他还没有回我的短信。**
Tā hái méiyǒu huí wǒ de duǎnxìn.
He hasn't replied to my text message yet.

**你看到我发的短信了吗？**
Nǐ kàndào wǒ fā de duǎnxìn le ma?
Did you see the text message I sent?

**不要边开车边看短信。**
Bú yào biān kāichē biān kàn duǎnxìn.
Don't read text messages while driving.

## Collocations

| Collocation | Meaning |
|-------------|---------|
| 发短信 | to send a text message |
| 收短信 | to receive a text message |
| 回短信 | to reply to a text message |
| 一条短信 | one text message (measure word 条) |
| 看短信 | to read a text message |

## Memory Hook

短 (short) plus 信 (letter/message). A 短信 is literally a short letter. Before smartphone apps existed, people sent brief 短信 messages because they were charged per character.

## 只有 (zhǐyǒu)

只有 means only or only if/when. It sets a single necessary condition. It almost always pairs with 才 in the second clause to mean "only when [condition], then [result]." Without 才, it can mean "there is only" one option or thing.

## Meanings

1. **[conjunction]** Only if; only when (used in 只有...才... pattern to express a necessary condition).
2. **[adverb phrase]** There is only; the only one.

## Example Sentences

**只有努力学习，才能取得好成绩。**
Zhǐyǒu nǔlì xuéxí, cái néng qǔdé hǎo chéngjì.
Only by studying hard can you achieve good results.

**只有你才能帮我。**
Zhǐyǒu nǐ cái néng bāng wǒ.
You are the only one who can help me.

**只有一个问题需要解决。**
Zhǐyǒu yī gè wèntí xūyào jiějué.
There is only one problem that needs to be solved.

**只有他来了，会议才能开始。**
Zhǐyǒu tā lái le, huìyì cái néng kāishǐ.
Only when he arrives can the meeting begin.

## Collocations

| Collocation | Meaning |
|-------------|---------|
| 只有...才... | only if... then... (necessary condition) |
| 只有一个 | there is only one |
| 只有这样 | only in this way |
| 只有你 | only you |
| 只有通过 | only by going through |

## Memory Hook

只 means only/just and 有 means have/exist. "Only have" one path forward. Pair it with 才 and you lock in a condition: the door opens only when you have the right key.

## 互相 (hùxiāng)

互相 means mutually or to each other — the action described goes in both directions between two or more parties simultaneously. It always modifies a verb.

## Meanings

1. **[adverb]** Mutually; each other; one another.

## Example Sentences

**我们应该互相帮助。**
Wǒmen yīnggāi hùxiāng bāngzhù.
We should help each other.

**互相了解可以减少误解。**
Hùxiāng liǎojiě kěyǐ jiǎnshǎo wùjiě.
Mutual understanding can reduce misunderstandings.

**两个好朋友互相支持，共同进步。**
Liǎng gè hǎo péngyou hùxiāng zhīchí, gòngtóng jìnbù.
Two good friends support each other and make progress together.

**不同文化之间应该互相尊重。**
Bùtóng wénhuà zhī jiān yīnggāi hùxiāng zūnzhòng.
Different cultures should respect each other.

## Memory Hook

互 (interlocking — two things crossing over each other like two hands clasping) plus 相 (mutually / each other). Two interlocking things facing each other — the perfect image of mutual action.

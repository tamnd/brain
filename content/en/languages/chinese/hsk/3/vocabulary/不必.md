## 不必 (bùbì)

不必 is an adverb meaning "no need to" or "it is not necessary to." It tells someone that an action is not required, relieving them of an obligation or effort.

## Meanings

1. **[adverb]** no need to; need not; it is unnecessary to (placed before a verb to indicate the action is not required)

## Usage Note

不必 is similar to 不用 but tends to sound slightly more formal or literary. Both negate necessity, but neither negates ability or permission. Compare with 不要 (do not), which is a prohibition, and 不能 (cannot), which negates ability.

- 你不必来。 You do not need to come. (not required)
- 你不要来。 Do not come. (prohibition)

## Example Sentences

**你不必担心，一切都安排好了。** Nǐ bùbì dānxīn, yīqiè dōu ānpái hǎo le. You need not worry; everything has been arranged.

**这件事不必告诉他。** Zhè jiàn shì bùbì gàosù tā. There is no need to tell him about this matter.

**不必那么客气，我们是朋友。** Bùbì nàme kèqi, wǒmen shì péngyou. No need to be so polite; we are friends.

**明天的会议你不必参加。** Míngtiān de huìyì nǐ bùbì cānjiā. You do not need to attend tomorrow's meeting.

## Memory Hook

不 means "not" and 必 means "must / necessarily." Put them together: "not necessarily" or "no need to." If something is not a must, you do not have to do it.

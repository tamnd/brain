HSK 3 grammar covers approximately 70 core patterns for confident intermediate communication. This level introduces the 把 construction, 被 passive, result/direction/degree/potential complements, and a wider range of complex sentence structures.

## 1. The 把 (bǎ) Construction

把 preposition moves the object before the verb, emphasizing the disposal, transformation, or placement of the object.

### 1.1 Core 把 Pattern

| Pattern | Example | English |
|---------|---------|---------|
| S + 把 + O + V + Result | 我把书放在桌子上了。| I put the book on the table. |
| S + 把 + O + V + 完 | 他把饭吃完了。| He finished eating the food. |
| S + 把 + O + V + 好 | 请把作业做好。| Please finish the homework properly. |
| S + 把 + O + V + 掉 | 我把钱花掉了。| I spent all the money. |
| S + 把 + O + V + 成 | 她把我当成朋友。| She treats me as a friend. |

### 1.2 Rules for 把 Sentences

- The verb must have a **complement or other element** after it (not a bare verb)
- The object must be **definite** (specific / already known)
- Cannot use 把 with verbs of perception (听、看、想) or 是、有、在

### 1.3 Common 把 Verbs and Complements

| Verb + Complement | Meaning | Example |
|-------------------|---------|---------|
| 把…拿出来 | take out | 把书拿出来 take the book out |
| 把…放进去 | put in | 把衣服放进去 put the clothes in |
| 把…写下来 | write down | 把名字写下来 write the name down |
| 把…告诉 | tell | 把消息告诉我 tell me the news |
| 把…带走 | take away | 把东西带走 take the things away |

## 2. The 被 (bèi) Passive

被 marks a passive construction — the subject receives or suffers the action.

### 2.1 Core 被 Pattern

| Pattern | Example | English |
|---------|---------|---------|
| S + 被 + Agent + V + Complement | 我被老师批评了。| I was criticized by the teacher. |
| S + 被 + V + Complement (no agent) | 钱包被偷了。| The wallet was stolen. |

### 2.2 Rules for 被 Sentences

- Often implies an **undesirable** result (adversative passive)
- The verb usually needs a complement or 了 after it
- Agent can be omitted if unknown or obvious

### 2.3 Neutral Passive Structures (without 被)

| Structure | Example | English |
|-----------|---------|---------|
| 让 + Agent + V | 他让我等了一个小时。| He made me wait an hour. |
| 叫 + Agent + V | 他叫我去。| He told me to go. |
| 给 + Agent + V | 书给他拿走了。| The book was taken by him. |

## 3. Result Complements

Result complements follow a verb directly and describe the result of the action.

### 3.1 Common Result Complements

| Complement | Meaning | Example |
|------------|---------|---------|
| 好 | done properly | 做好了 finished properly |
| 完 | completed | 吃完了 finished eating |
| 到 | reached | 买到了 managed to buy |
| 见 | perceived | 听见了 heard it |
| 懂 | understood | 看懂了 understood by reading |
| 错 | done wrongly | 写错了 wrote wrong |
| 住 | firmly / stopped | 记住了 memorized |
| 开 | opened / separated | 打开了 opened (it) |
| 掉 | removed / done away with | 扔掉了 threw away |

### 3.2 Potential Form (V + 得/不 + Complement)

| Positive | Negative | Meaning |
|----------|----------|---------|
| 做得完 | 做不完 | can / can't finish |
| 听得懂 | 听不懂 | can / can't understand (audio) |
| 看得见 | 看不见 | can / can't see |
| 记得住 | 记不住 | can / can't remember |
| 走得了 | 走不了 | can / can't leave |

## 4. Direction Complements

Direction complements describe the direction of movement resulting from the action.

### 4.1 Simple Direction Complements

| Complement | Meaning | Example |
|------------|---------|---------|
| V + 上 | upward / onto | 走上去 walk up |
| V + 下 | downward | 坐下 sit down |
| V + 进 | inward | 走进去 walk in |
| V + 出 | outward | 拿出来 take out |
| V + 回 | back | 跑回去 run back |
| V + 过 | past, over | 走过去 walk past |
| V + 起 | up, rise | 站起来 stand up |

### 4.2 Compound Direction Complements

Combine simple complements with 来/去 to show direction relative to speaker.

| Complement | Meaning | Example |
|------------|---------|---------|
| V + 上来 | come up here | 跑上来 run up here |
| V + 上去 | go up there | 爬上去 climb up there |
| V + 下来 | come down here | 走下来 walk down here |
| V + 进来 | come in | 请进来 please come in |
| V + 出去 | go out | 跑出去 run out |
| V + 起来 | start doing; sound like | 唱起来 start singing; 听起来 sound like |

## 5. Degree Complements with 得

### 5.1 得 Complement — How Well / How Much

| Pattern | Example | English |
|---------|---------|---------|
| V + 得 + Adj | 她跑得很快。| She runs very fast. |
| V + 得 + VP | 他说得让我哭了。| He spoke so that I cried. |
| V + O + V + 得 + Adj | 他说中文说得很流利。| He speaks Chinese very fluently. |
| Adj + 得 | 忙得没有时间吃饭。| So busy (that) there's no time to eat. |

### 5.2 极了 / 死了 / 坏了 — Extremely

| Pattern | Example | English |
|---------|---------|---------|
| Adj + 极了 | 好极了 | extremely good |
| Adj + 死了 | 累死了 | exhausted (lit. tired to death) |
| Adj + 坏了 | 急坏了 | extremely anxious |

## 6. Concessive and Causal Patterns

### 6.1 尽管…还是/仍然 — Even Though … Still

| Pattern | Example | English |
|---------|---------|---------|
| 尽管 + Concession, 还是/仍然 + Result | 尽管很贵，他还是买了。| Even though it was expensive, he still bought it. |

### 6.2 既然…就 — Since … Then

| Pattern | Example | English |
|---------|---------|---------|
| 既然 + Premise, 就 + Conclusion | 既然你来了，就一起吃饭吧。| Since you're here, let's eat together. |

### 6.3 只要…就 — As Long As … Then

| Pattern | Example | English |
|---------|---------|---------|
| 只要 + Condition, 就 + Result | 只要努力，就能成功。| As long as you work hard, you can succeed. |

### 6.4 只有…才 — Only If … Then

| Pattern | Example | English |
|---------|---------|---------|
| 只有 + Condition, 才 + Result | 只有努力，才能进步。| Only by working hard can you improve. |

## 7. Conditional Sentences

### 7.1 如果/要是…就/那么 — If … Then

| Pattern | Example | English |
|---------|---------|---------|
| 如果 + Condition, 就 + Result | 如果明天下雨，就不去了。| If it rains tomorrow, I won't go. |
| 要是…就 | 要是你不来，我就走。| If you don't come, I'll leave. |

### 7.2 除非…否则/不然 — Unless … Otherwise

| Pattern | Example | English |
|---------|---------|---------|
| 除非 + Condition, 否则 + Consequence | 除非你道歉，否则我不原谅你。| Unless you apologize, I won't forgive you. |

## 8. Expressing Sequence

### 8.1 先…然后…再 — First … Then … Then

| Pattern | Example | English |
|---------|---------|---------|
| 先 V1, 然后 V2, 再 V3 | 先洗手，然后吃饭，再看电视。| First wash hands, then eat, then watch TV. |

### 8.2 一…就 — As Soon As

| Pattern | Example | English |
|---------|---------|---------|
| 一 + V1 + 就 + V2 | 他一到家就睡觉。| As soon as he gets home, he sleeps. |

### 8.3 刚/刚才 — Just (Completed)

| Pattern | Example | English |
|---------|---------|---------|
| 刚 + V | 他刚走。| He just left. |
| 刚才 + V | 刚才下雨了。| It just rained. |

## 10. Pivotal Constructions

A pivotal sentence has one noun phrase that is both the object of the first verb and the subject of the second.

| Pattern | Example | English |
|---------|---------|---------|
| S1 + V1 + N + V2 | 老师让我回答问题。| The teacher had me answer the question. |
| 请/叫/让 + Person + V | 请你坐下。| Please sit down. |
| 有 + N + V | 有人敲门。| There is someone knocking at the door. |

## 11. More Question Patterns

### 11.1 是不是 / 对不对 — Confirmation Questions

| Pattern | Example | English |
|---------|---------|---------|
| 是不是 + Statement? | 你是不是学生？| Are you a student? |
| Statement + 对不对？| 这样对不对？| Is this right? |

### 11.2 越来越 — More and More

| Pattern | Example | English |
|---------|---------|---------|
| 越来越 + Adj/V | 天气越来越热。| The weather is getting hotter and hotter. |

### 11.3 越…越 — The More … The More

| Pattern | Example | English |
|---------|---------|---------|
| 越 + V/Adj + 越 + V/Adj | 他越说越高兴。| The more he talked, the happier he became. |

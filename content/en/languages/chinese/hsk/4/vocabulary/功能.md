## 功能 (gōngnéng)

功能 refers to the specific function, feature, or capability of something — what a device, system, body part, or tool is designed or able to do.

## Meanings

1. **[noun]** function, feature, capability, functionality

## Example Sentences

**这款手机有很多实用的功能。** Zhè kuǎn shǒujī yǒu hěn duō shíyòng de gōngnéng. This smartphone has many practical features.

**大脑的主要功能之一是处理信息。** Dànǎo de zhǔyào gōngnéng zhī yī shì chǔlǐ xìnxī. One of the brain's main functions is to process information.

**这个软件的搜索功能非常强大。** Zhège ruǎnjiàn de sōusuǒ gōngnéng fēicháng qiángdà. The search function of this software is very powerful.

**心脏的功能是把血液输送到全身。** Xīnzàng de gōngnéng shì bǎ xuèyè shūsòng dào quánshēn. The function of the heart is to pump blood throughout the body.

## Collocations

| Collocation | Meaning |
|-------------|---------|
| 主要功能 | main function |
| 功能强大 | powerful functionality |
| 多功能 | multi-function / versatile |
| 恢复功能 | restore function / functional recovery |

## Usage Notes

功能 is commonly used when talking about technology, biology, or systems. It differs from 作用 (effect/role) in that 功能 refers to an inherent designed capability, while 作用 emphasizes the impact or effect something has. For example: 手机的功能 (the phone's features) vs. 手机的作用 (the role/effect of mobile phones).

## Memory Hook

功 (achievement/work) + 能 (ability/can) = "what it is able to achieve" = function/capability.

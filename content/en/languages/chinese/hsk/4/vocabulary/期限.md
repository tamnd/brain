## 期限 (qīxiàn)

期限 means a deadline or time limit, referring to the fixed date or period by which something must be done.

## Meanings

1. **[noun]** deadline; time limit; a set period within which something must be completed

## Example Sentences

**这个项目的期限是下周五。** Zhège xiàngmù de qīxiàn shì xià zhōuwǔ. The deadline for this project is next Friday.

**我们必须在规定期限内完成任务。** Wǒmen bìxū zài guīdìng qīxiàn nèi wánchéng rènwu. We must complete the task within the set deadline.

**签证的有效期限是三个月。** Qiānzhèng de yǒuxiào qīxiàn shì sān gè yuè. The validity period of the visa is three months.

**请注意报名期限，不要错过机会。** Qǐng zhùyì bàomíng qīxiàn, bùyào cuòguò jīhuì. Please note the registration deadline; don't miss the opportunity.

## Collocations

| Collocation | Meaning |
|-------------|---------|
| 规定期限 | set deadline |
| 有效期限 | validity period |
| 在期限内 | within the deadline |
| 超过期限 | to exceed the deadline |

## Usage Notes

期限 is a formal noun used in administrative, legal, and academic contexts. It specifies a concrete cutoff point. Compare with 截止日期 (jiézhǐ rìqī), which more specifically means a cutoff date. 期限 can refer to both a date and a duration, making it slightly broader.

## Memory Hook

期 means "scheduled time" and 限 means "limit": 期限 is the scheduled limit, the moment when time runs out.

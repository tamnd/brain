## 例如 (lìrú)

例如 is used to introduce one or more examples that illustrate a general statement, equivalent to "for example" or "such as" in English.

## Meanings

1. **[adverb/conjunction]** for example, for instance, such as

## Example Sentences

**很多水果对身体有好处，例如苹果、香蕉和橙子。** Hěn duō shuǐguǒ duì shēntǐ yǒu hǎochù, lìrú píngguǒ, xiāngjiāo hé chéngzi. Many fruits are good for the body, for example apples, bananas, and oranges.

**学好外语有很多方法，例如多听、多说、多练习。** Xuéhǎo wàiyǔ yǒu hěn duō fāngfǎ, lìrú duō tīng, duō shuō, duō liànxí. There are many ways to learn a foreign language well, for example listening more, speaking more, and practicing more.

**有些职业要求很高，例如医生和律师需要多年的学习。** Yǒuxiē zhíyè yāoqiú hěn gāo, lìrú yīshēng hé lǜshī xūyào duō nián de xuéxí. Some professions have high requirements; for example, doctors and lawyers need many years of study.

**我们可以通过多种方式锻炼，例如跑步、游泳或打球。** Wǒmen kěyǐ tōngguò duō zhǒng fāngshì duànliàn, lìrú pǎobù, yóuyǒng huò dǎqiú. We can exercise in many ways, for example running, swimming, or playing ball.

## Collocations

| Collocation | Meaning |
|-------------|---------|
| 例如说 | for example (slightly more informal) |
| 举例如 | to give an example such as |
| 以…为例如 | taking… as an example |
| 例如以下 | such as the following |

## Usage Notes

例如 is more formal than 比如 and is frequently used in writing, reports, and speeches. It typically introduces a list of examples after a general statement. Unlike 比如 which can appear mid-sentence conversationally, 例如 usually begins a new clause or appears at the start of an example list.

## Memory Hook

例 (example/case) + 如 (like/as if) = "like a case / as for example" — picture a case file labeled with an example.

## Key Grammar Additions at HSK 2

| Pattern | Example | English |
|---------|---------|---------|
| 比 comparison | 他比我高。| He is taller than me. |
| 一样 equality | 我们一样高。| We are the same height. |
| 正在...呢 | 我正在吃饭呢。| I'm in the middle of eating. |
| 虽然...但是... | 虽然累，但是开心。| Although tired, (I'm) happy. |
| 因为...所以... | 因为下雨，所以不去。| Because it rained, so (I) won't go. |
| 如果...就... | 如果你来，我就高兴。| If you come, I'll be happy. |
| 越来越 | 越来越好。| Getting better and better. |
| 一边...一边... | 一边吃饭一边看电视。| Eating while watching TV. |

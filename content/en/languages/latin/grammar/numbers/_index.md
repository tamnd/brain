### Declension of *ūnus, duo, trēs*

**duo, duae, duo** (two):

| Case | Masc./Neut. | Fem. |
|------|-------------|------|
| Nom. | duo / duo | duae |
| Gen. | duōrum | duārum |
| Dat. | duōbus | duābus |
| Acc. | duōs / duo | duās |
| Abl. | duōbus | duābus |

**trēs, tria** (three) declines as a 3rd-declension i-stem adjective.

### *mīlle* vs. *mīlia*

- *mīlle* (singular): indeclinable adjective — *mīlle mīlitēs* (1,000 soldiers)
- *mīlia* (plural): neuter noun taking a partitive genitive — *duo mīlia mīlitum* (2,000 of soldiers)
- *mīlle et ūnum* = 1,001

## Distributive Numbers (How Many Each / How Many Apiece?)

Distributive numerals are used:
1. To express "how many each" or "apiece"
2. With **pluralia tantum** nouns (nouns that only exist in the plural, like *castra* "camp," *lītterae* "letter")
3. In multiplication: *bis bīna = quattuor* — "twice two is four"

All distributives are 1st-2nd declension adjectives (plural only).

| Latin | English |
|-------|---------|
| singulī, -ae, -a | one each, one at a time |
| bīnī, -ae, -a | two each, two at a time |
| ternī (trīnī) | three each |
| quaternī | four each |
| quīnī | five each |
| sēnī | six each |
| septēnī | seven each |
| octōnī | eight each |
| novēnī | nine each |
| dēnī | ten each |
| vīcēnī | twenty each |
| centēnī | one hundred each |

Example: *singula castra* — "one camp" (since *castra* is always plural, cardinal *ūna castra* is impossible; use distributive *singula*)

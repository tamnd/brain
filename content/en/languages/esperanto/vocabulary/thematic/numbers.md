## Numbers & Mathematics Vocabulary

### Cardinal Numbers

| Esperanto | English | Notes |
|-----------|---------|-------|
| nulo | 0 | zero |
| unu | 1 | one |
| du | 2 | two |
| tri | 3 | three |
| kvar | 4 | four |
| kvin | 5 | five |
| ses | 6 | six |
| sep | 7 | seven |
| ok | 8 | eight |
| naŭ | 9 | nine |
| dek | 10 | ten |
| dek unu | 11 | eleven (dek + unu) |
| dek du | 12 | twelve |
| dek tri | 13 | thirteen |
| dudek | 20 | twenty (du + dek) |
| tridek | 30 | thirty |
| kvardek | 40 | forty |
| kvindek | 50 | fifty |
| sesdek | 60 | sixty |
| sepdek | 70 | seventy |
| okdek | 80 | eighty |
| naŭdek | 90 | ninety |
| cent | 100 | hundred |
| ducent | 200 | two hundred |
| mil | 1,000 | thousand |
| dek mil | 10,000 | ten thousand |
| miliono | 1,000,000 | million |
| miliardo | 1,000,000,000 | billion |

**Number formation rules:**
- Compound numbers concatenate: 21 = dudek unu, 243 = ducent kvardek tri
- "And" is not used between parts: 101 = cent unu (not cent *kaj* unu)

### Ordinal Numbers (using -a suffix)

| Esperanto | English |
|-----------|---------|
| unua | first |
| dua | second |
| tria | third |
| kvara | fourth |
| kvina | fifth |
| sesa | sixth |
| sepa | seventh |
| oka | eighth |
| naŭa | ninth |
| deka | tenth |
| dek-unua | eleventh |
| centa | hundredth |
| mila | thousandth |

**Example:** *La tria de marto* = the third of March.

### Fractions (-on- suffix)

| Esperanto | English |
|-----------|---------|
| duono | half |
| triono | third (1/3) |
| kvarono | quarter (1/4) |
| kvinono | fifth (1/5) |
| dekono | tenth (1/10) |
| cento | hundredth (1/100) — also "cent" for 100 |
| procento | percent |

**Example:** *Du trionoj de la torto* = two thirds of the cake.

### Group Numbers (-op- suffix)

| Esperanto | English |
|-----------|---------|
| duope | in pairs, in twos |
| triope | in threes |
| kvartope | in fours |
